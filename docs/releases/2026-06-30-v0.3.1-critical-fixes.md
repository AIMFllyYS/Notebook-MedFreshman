# Gailvlun v0.3.1 桌面端关键修复刷新

发布标签：`v0.3.1`

本次刷新保持版本号 `0.3.1` 不变，重新覆盖 Windows 桌面端安装包和便携包，重点修复当前 exe 中影响使用的关键回归。

## 已修复

- 修复桌面端右侧面板白屏：AI、视频、交互演示、浏览器等 tab 现在都有 loading、错误态和重试入口，异常时不再静默白屏。
- 修复自定义 API 思考块展示：兼容 `reasoning_content`、`reasoning`、`reasoning_text`、`reasoning_details` 等流式字段，并补充不同 provider 的 thinking 请求风格。
- 修复 HTML 交互演示生成：生成时固定使用用户当前选中的文本模型；如果当前选择的是生图模型，会明确提示不支持 HTML 交互组件生成，不再错误调用 `/api/artifact`。
- 修复 SVG 生成稳定性：提示词强制要求完整 `<svg ...>...</svg>` 根标签，渲染端也会容错包裹裸 `<path>`、`<rect>`、`<text>` 等 SVG 子节点。
- 修复生图模式链路：选择生图模型后，聊天会强制走 `generateImage` 和 `/api/image-gen`，并使用当前选中的生图模型，不再被默认模型覆盖。
- 修复自定义 OpenAI 格式生图兼容：OpenAI-compatible image API 与 SiliconFlow-like image API 会按配置和自动识别走正确请求格式。
- 修复对话上下文机制：同一会话保持稳定上下文预算，圆环展示当前标签页总上下文，展示 cache hit/miss；超过 80% 后继续允许输入，并自动只向模型发送最近消息，同时持续警示。
- 修复本地测试发现范围：排除生成后的桌面端构建产物，避免旧 packaged artifact 污染 TypeScript/单测验证。

# Gailvlun v0.3.1 原始发布说明

## [0.3.1] - 2026-06-29

本版本在 v0.3.0 基础上完成对话存储性能重构、画布运行时统一、AI 图片生成端到端落地，并新增大学物理第一章完整内容与计费仪表盘。动效层面引入折叠动画、3D 翻卡片、独占手风琴与画布 AI 修订框架，整体交互平滑度显著提升。

### 性能重构（Storage v2 与虚拟化）

- **对话存储迁移至 Storage v2**：聊天历史按会话拆分写入 IndexedDB，替换旧的单一 blob 持久化；附 v2 key helpers 与 debounce 测试，迁移过程可持久化、可回滚
- **ChatThread 虚拟化**：基于 `@tanstack/react-virtual` 实现长会话窗口化渲染，万条消息滚动保持稳定帧率
- **订阅隔离与延迟挂载**：聊天 hook 拆分订阅源，浮窗未可见时延迟挂载，避免后台无效渲染
- **侧边栏与例题懒加载**：例题正文按需加载，侧边栏导航树瘦身，首屏 JS 显著下降
- **搜索引擎升级**：向量检索改为堆 top-K 扫描，BM25 走 chunks-meta 预计算，检索吞吐量提升

### 画布运行时统一重构

- **类型化画布块运行时**：新增 typed canvas block runtime，函数图、SVG、HTML 图表、化学式统一走同一渲染管线
- **统一 AI 修订框架**：画布块支持 AI 修订（shared AI revision frame + unified canvas revision API），修订结果按完整模式持久化
- **渲染器拆分**：canvas frame 与各渲染器解耦，便于扩展新画布类型
- **表达式诊断**：绘图表达式渲染前先诊断，失败时降级为可读错误卡片，不再白屏
- **遗留 SVG 修复路径下线**：旧的 SVG repair 通路移除，统一到新运行时

### AI 图片生成（端到端落地）

- **多组自定义模型与图片生成数据结构**：支持多 API 分组管理自定义模型，新增图片生成专用数据结构
- **图片生成 API 路由**：新增 `/api/image-gen`，复用 OpenAI 兼容端点
- **generateImage 工具与 useImageGen hook**：AI 对话可调用图片生成工具，前端通过 hook 订阅生成状态
- **图片生成 UI 组件**：独立查看器、设置内联编辑表单、模型菜单整合
- **会话持久化与多窗口**：图片生成会话写入 IndexedDB，关闭后可重开；多窗口查看器协同
- **图片搜索配额与画廊**：单会话图片搜索硬上限，超额降级；新增画廊视图汇总历史
- **模型路由归一化**：以 provider apiModelId 与 registryId 双轨寻址，自定义模型按 API 分组作用域隔离，避免跨组串扰

### 计费仪表盘

- **IndexedDB 长期统计**：token 与图片生成计费按会话实时记录，持久化至 IndexedDB
- **API 分组定价模型**：设置面板支持按 API 分组配置定价，含美元汇率换算
- **BillingDashboard**：虚拟化表格 + 3D 翻卡片摘要 + 趋势图表，支持窗口管理器注册与最小化任务栏召回
- **美元汇率设置**：范围校验（clamp）下沉至 store action 单一源，UI 与迁移层共用
- **三项关键修复**：自定义图片模型按 registryId 计费并归类（修正 0 元与错误归类）、趋势图跨年同月冲突按年分桶并切换月聚合、美元汇率 0/NaN 兜底
- **图标复用**：任务栏与溢出菜单复用计费饼图与图片 ImagePlus 图标

### 集中化快捷键系统

- **统一快捷键中枢**：全局快捷键集中注册，支持逐项启停
- **设置面板独占手风琴**：设置分区改为独占手风琴，展开一个自动收起其余，配合折叠动画过渡更顺滑

### 折叠动画与 UI 细节

- **AnimatedCollapse 折叠组件**：SettingsSection 接入平滑的展开收起动画，避免高度突变
- **多彩主题学科配色扩展**：subject accent 由笔记区扩展到整 UI（primary、primary-container、on-primary-container），亮/暗主题分别调色
- **官方品牌 SVG 图标**：替换占位图标，统一视觉
- **ContentPageClient 清理**：移除冗余 data-subject（已由 AppShell 统一注入）

### 大学物理第一章（完整内容上线）

- **OCR 源文件与配图**：第一章课堂录音 OCR 与章节配图入库
- **1.1–1.4 详解**：四节详尽笔记，含 KaTeX 公式与例题
- **例题按小节重分类**：例题与各小节对齐，便于检索
- **题库**：第一章题库上线，含客观题与计算题
- **内容树与侧边栏注册**：大学物理第一章接入内容树与侧边栏导航

### 大学物理（考前模拟与录音例题）

- **考前模拟真题**：收录大学物理考前模拟试卷，附 callout 渲染修复
- **录音例题（rec-01–27）**：27 套录音例题全部完成，含 SVG 图与 LaTeX 转义修复
- **录音例题测验重写**：rec-01–06 测验题重写，覆盖图中渲染场景

### 其他修复与测试

- **AI 伪思考流剔除**：路由层将伪思考内容剔出渲染流，避免污染对话
- **公式步骤 Markdown 复用**：公式步骤走统一 Markdown 渲染，消除渲染偏差
- **会话替换加载与 manifest 写入收敛**：v2 迁移后替换会话正确加载，manifest 写入节流
- **中文命名例题 SSR 加载**：中文文件名例题改走 SSR，规避构建期路径解析
- **笔记选区复制**：浮窗按钮 + Ctrl+C 拦截恢复选区复制
- **图片灯箱点击与拖拽**：图片消息支持点击放大与拖拽到输入框
- **SVG 修复表单收敛**：修复表单限制在 fallback 卡片内
- **测试覆盖**：流式节流与 API payload 窗口性能护栏、虚拟线程渲染、上游错误处理、模型端点校验、quiz 图渲染等

### 安全

- 自定义 API 密钥与图片生成端点均仅服务端处理，不进入前端包；图片生成会话与计费数据本地 IndexedDB，不上传
