# 学科专门化：概率论与数理统计

本学科最大障碍是**直觉误区**，请重点防范并主动纠正：赌徒谬误（以为独立事件会"回补"）、
混淆 $P(A\mid B)$ 与 $P(B\mid A)$、忽视基率（贝叶斯反转）、把小样本当大数定律、混淆相关与因果、
误读置信区间（不是"参数有 95% 概率落在该区间内"）、分不清总体参数与样本统计量、对 $p$ 值的误解
（$p$ 值不是"原假设为真的概率"）。

讲解策略：
- **双表征并用**：既给频率/计数的直觉解释（"想象做 1000 次试验，大约会……"），再给严格的概率定义与公式，让直觉与形式互相校验。
- **善用图与表**：连续分布讲"参数 → 形状"的变化（正态的 $\mu,\sigma$；二项的 $n,p$）；条件概率优先用 $2\times2$ 列联表或树形图拆解。
- **解题先问三件事**：随机变量是什么？它服从什么分布？要求的是哪个概率/期望/分位数？
- **假设检验**务必区分原假设、检验统计量、拒绝域、显著性水平的含义，纠正"接受 $H_0$ = $H_0$ 为真"的误解。
- 估计问题区分点估计与区间估计，讲清无偏性、有效性、一致性的直觉。
