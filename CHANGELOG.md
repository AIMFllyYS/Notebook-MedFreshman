# 更新日志

本项目所有重要变更记录于此。版本遵循语义化版本（SemVer）。

## [0.3.0] - 2026-06-29

### 🪟 统一窗口管理系统（架构重构）

- **统一窗口基础设施**：浮窗全面迁移至统一窗口框架，支持任意层级子窗口管理、生命周期统一调度
- **全局任务栏**：所有浮窗与 Artifact 均可从页面底部任务栏一键召回或切换；面板切换时自动停止生成，避免无效请求
- **全屏状态追踪**：窗口全屏状态统一追踪；Artifact Viewer 全局挂载，跨页面导航不中断
- **全局搜索入口**：任务栏新增全局搜索快捷入口

### 🎨 全局外观定制

- **外观设置面板**：字体大小、行间距、界面主题均可在设置中调节，通过 CSS 层实时生效
- **设置持久化**：外观配置写入 localStorage，刷新后保留

### 💰 Token 看板 & 自定义 API 多模型管理

- **浮窗独立 Token 看板**：每个浮窗单独显示当前会话的 Token 用量与上下文估算圆环，圆环颜色随用量动态变化
- **自定义 API 多模型管理**：在设置中可添加任意 OpenAI 兼容 API 端点和模型，与内置模型统一在模型下拉中选择
- **内置模型定价补全**：所有内置模型均已录入 Token 定价信息，Token 成本估算精确显示

### 🖼 AI 图片搜索（Unsplash 集成）

- **智能图片搜索**：AI 对话中可通过工具调用搜索 Unsplash 高质量图片并插入回复，自动附加版权归属
- **按需启用**：配置 `UNSPLASH_ACCESS_KEY` 即可激活；未配置时自动降级为 Demo 模式（每小时 50 次）

### 🖥 画布与渲染全面修复

- **KaTeX 公式 SVG 白屏根治**：修复公式渲染后画布内容消失的问题
- **SvgDiagram 统一画布**：函数图、SVG 图、SMILES 化学式、HTML-in-canvas 四类内容统一走同一画布渲染路径
- **HTML 图表增强**：AI 生成的 HTML 图表支持下载；思考框跟随画布滚动；新增全屏控件
- **FormulaSteps 解析修复**：公式步骤组件从 rehype-raw 路径迁移至组件解析器，消除渲染偏差
- **AI 消息标签解析统一化**：所有自定义组件标签走统一解析入口，FollowUp 移出渲染流

### 📝 复习板升级

- **划词记录多模式流式处理**：记录板块升级为「解释 / 步骤拆解 / 联系知识点」三种模式，逐字流式输出
- **预览浮窗 UI 全面升级**：统一布局结构、样式规范和模式切换按钮

### 🎯 答题系统

- **答题会话持久化**：答题进度写入 IndexedDB，退出后重新进入自动恢复现场
- **测试题渲染修复**：修复 Markdown 自定义标签在测验组件中不渲染的问题

### 📚 内容扩充：大学英语（CET-4 难度）

- **英语练习板块上线**：新增独立英语科目，Unit 1–8 全部完成
- 每单元包含：原文精读、精讲例题（阅读理解 / 翻译 / 词汇）、随堂测验

### 📚 内容扩充：有机化学

- **测试题全面重写**：全部 14 章测试题由 10 题扩充至 34 题/章，难度梯度更合理
- **华科历年真题**：收录华中科技大学 2021-2022、2023-2024 学年有机化学期末考试真题
- 真题采用卡片式布局，附分子结构图，支持考前模拟训练

### 🗂 导航体验

- **同级文件面板（SiblingFilesPanel）**：笔记页新增同目录文件快速切换面板，浏览相关内容无需返回目录
- **TOC 性能优化**：目录渲染性能提升，修复竞态条件导致的目录跳动问题
- **AI 头部自动隐藏**：发送第一条消息后，AI 面板顶部工具栏自动收起，阅读区更大

### 🎨 视觉细节

- **划词高亮橙色蜡笔风格**：文本划词高亮改为橙色蜡笔风格，视觉辨识度更高
- **Tooltip 优化**：单行 Tooltip 不再截断，圆环图标增加浅色背景轨道
- **会话标题生成优化**：自动标题生成改为轻量方案，不再调用大模型，响应更快

### 安全

- 自定义 API 密钥与内置密钥均仅在服务端处理，不进入前端包；Unsplash Access Key 仅服务端持有

## [0.2.1] - 2026-06-26

### 修复（重要）

- **桌面端数据重启丢失（P0）**：旧版本桌面应用每次启动用随机端口,导致页面 origin 每次都变,
  浏览器按 origin 隔离的 IndexedDB 因此每次重启都从空开始——**对话记录 / 导入的技能 / 复习卡全部丢失**
  (API 密钥因单独加密存盘不受影响)。现固定为本地端口 **35349**(网页与桌面同源),数据从此跨重启持久;
  并对端口占用做安全等待与清晰报错,绝不静默改端口而丢数据。

### 新增

- **划词 AI 深度重构**：划词「解释 / 追问」浮窗不再是单独的弱实现，而是**复用完整的右侧 AI 对话引擎**——真实支持工具调用、交互演示、检索引用、图片附件、FollowUp 等全部能力；模型 / 深度思考 / 联网搜索统一在窗口底部，发送↔停止与主对话一致。
- **划词对话本地持久化**：每个划词浮窗就是一段持久会话，关窗不丢；可随时还原。
- **历史记录分栏**：右侧「历史记录」升级为「**对话 / 划词**」两栏；划词栏点击即**还原**旧浮窗。
- **导出所有聊天数据**：AI 设置新增一键导出（主对话 + 划词）为本地 JSON 备份，纯本地、无外发。
- **应用 / 安装器图标**：打包应用沿用站点书本图标；安装器配套精致 LOGO；附 publisher / copyright 元数据。
- **基础代码签名**：对 portable / installer 做自签名 + RFC3161 时间戳（非 CA 受信，仍建议按 SHA256 校验）。

### 优化

- **删除二次确认**：历史记录中删除会话需「删除？/取消」二次确认，防误删。
- **拖拽 / 缩放更顺**：划词浮窗几何改用 rAF 直写样式（零逐帧重渲），调整大小不卡顿。

### 安全

- 划词与主对话同走**服务端** `/api/chat`，API Key 仅后端；导出为本地 Blob，绝不外发；签名私钥仅存本机、不进仓库。

## [0.2.0] - 2026-06-25

### 新增

- **记忆系统（记忆卡片）**：在正文 / AI 输出处划词或右键「记录」，DeepSeek-V4-Flash 后台把*知识点*转成**填空闪卡**、把*题目*转成**问答卡**，自动带章节出处；**先存原文再异步成卡**，绝不丢失。
- **复习板**（`/[subject]/review`）：翻卡复习、左右导航、跳转选卡、删卡、下载（本科目 / 全部）JSON 备份；板内可继续划词记录。
- **首页书架**：根路径改为书架视图，每科一本书，悬浮「开始学习 / 开始复习」；左栏新增「首页」入口。
- **记录预览浮窗**：解析中实时反馈，生成后可保留 / 重做 / 丢弃。
- 划词菜单「举例」替换为「**记录**」；用户消息与全部 AI 输出支持**右键菜单**（复制 / 引用 / 追问 / 记录）。
- AI 对话输入框支持 **Ctrl+V 粘贴 / 拖拽图片**。

### 优化

- **跳卡数字面板**：复习板「选第几张」由下拉框改为**向上弹出的数字方格面板**。
- **卡片可折叠**：预览窗原文区、卡片背面解析支持展开 / 收起（复用统一伸缩组件）。
- **预览浮窗可拖拽缩放**（右下角手柄）。
- **AI 修订卡片**：对已生成的卡片输入一句话指令让 AI 改写，**修订失败自动回滚原卡**（绝不丢内容）。
- **书架封面每科专属配色** + 装饰 SVG 水印（告别统一同色）。

### 安全

- 成卡 / 修订均走**服务端**，API Key 仅后端，绝不进前端包或公开产物。
