# 更新日志

本项目所有重要变更记录于此。版本遵循语义化版本（SemVer）。

## [0.2.1] - 2026-06-26

### 新增

- **划词 AI 深度重构**：划词「解释 / 追问」浮窗不再是单独的弱实现，而是**复用完整的右侧 AI 对话引擎**——真实支持工具调用、交互演示、检索引用、图片附件、FollowUp 等全部能力；模型 / 深度思考 / 联网搜索统一在窗口底部，发送↔停止与主对话一致。
- **划词对话本地持久化**：每个划词浮窗就是一段持久会话，关窗不丢；可随时还原。
- **历史记录分栏**：右侧「历史记录」升级为「**对话 / 划词**」两栏；划词栏点击即**还原**旧浮窗。
- **导出所有聊天数据**：AI 设置新增一键导出（主对话 + 划词）为本地 JSON 备份，纯本地、无外发。
- **应用 / 安装器图标**：打包应用沿用站点书本图标；安装器配套精致 LOGO；附 publisher / copyright 元数据。
- **基础代码签名**：对 portable / installer 做自签名 + RFC3161 时间戳（非 CA 受信，仍建议按 SHA256 校验）。

### 优化

- **删除二次确认**：历史记录中删除会话需「删除？/取消」二次确认，防误删。
- **拖拽 / 缩放更顺**：划词浮窗几何改用 rAF 直写样式（零逐帧重渲），调整大小不卡顿。

### 安全

- 划词与主对话同走**服务端** `/api/chat`，API Key 仅后端；导出为本地 Blob，绝不外发；签名私钥仅存本机、不进仓库。

## [0.2.0] - 2026-06-25

### 新增

- **记忆系统（记忆卡片）**：在正文 / AI 输出处划词或右键「记录」，DeepSeek-V4-Flash 后台把*知识点*转成**填空闪卡**、把*题目*转成**问答卡**，自动带章节出处；**先存原文再异步成卡**，绝不丢失。
- **复习板**（`/[subject]/review`）：翻卡复习、左右导航、跳转选卡、删卡、下载（本科目 / 全部）JSON 备份；板内可继续划词记录。
- **首页书架**：根路径改为书架视图，每科一本书，悬浮「开始学习 / 开始复习」；左栏新增「首页」入口。
- **记录预览浮窗**：解析中实时反馈，生成后可保留 / 重做 / 丢弃。
- 划词菜单「举例」替换为「**记录**」；用户消息与全部 AI 输出支持**右键菜单**（复制 / 引用 / 追问 / 记录）。
- AI 对话输入框支持 **Ctrl+V 粘贴 / 拖拽图片**。

### 优化

- **跳卡数字面板**：复习板「选第几张」由下拉框改为**向上弹出的数字方格面板**。
- **卡片可折叠**：预览窗原文区、卡片背面解析支持展开 / 收起（复用统一伸缩组件）。
- **预览浮窗可拖拽缩放**（右下角手柄）。
- **AI 修订卡片**：对已生成的卡片输入一句话指令让 AI 改写，**修订失败自动回滚原卡**（绝不丢内容）。
- **书架封面每科专属配色** + 装饰 SVG 水印（告别统一同色）。

### 安全

- 成卡 / 修订均走**服务端**，API Key 仅后端，绝不进前端包或公开产物。

## [0.1.0] - 2026-06-24

### 新增

- 首个 **Windows 桌面版**（Electron）：多学科笔记 / 正文、AI 助教、Manim 视频、交互演示、内置真实浏览器、题库，本地离线可用。
- 划词追问多开悬浮窗；桌面端真实 Chromium 浏览器（保留登录态 / Cookie）；上下文看板（rAF 拖动 + 实时估算）。
- 3 个 API 密钥用户首启填写，Windows DPAPI 加密存本机，**绝不进包**。
