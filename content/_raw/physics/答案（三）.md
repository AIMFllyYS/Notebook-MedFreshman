**7. B**　安培环路定理中，B 是环路上各点的总磁感应强度，由空间一切电流共同产生；但 B 沿闭合环路的线积分 ∮B·dl 只与环路所包围的电流有关。环路内无电流仅说明 ∮B·dl = 0，并不意味着环路上各点 B 为零，故 A、C、D 均错。

**8. B**　麦克斯韦位移电流假说的核心：变化的电场（位移电流）和传导电流一样能激发磁场，从而使电磁场理论自洽。A 项描述的是法拉第电磁感应（变化磁场激发涡旋电场）。

**9. A**　按“沿入射光方向为正”的符号规则，位于折射面前方（入射一侧）的实物，其物距 u 取正值；实像距 v 也为正。

**三、简答题**

**21.**　自发辐射（2 分）：处于高能级的原子在不受外来光子作用时，自发地跃迁到低能级并放出一个光子，这种辐射称为自发辐射。各原子的自发辐射彼此独立，发出的光子在相位、方向、偏振上都是随机的，故为非相干光（普通光源即如此）。

受激辐射（2 分）：处于高能级的原子在频率满足 hν = E₂ − E₁ 的外来光子诱发下，跃迁到低能级并放出一个与入射光子完全相同的光子。

主要区别（1 分）：受激辐射放出的光子与入射光子在频率、相位、传播方向和偏振态上完全相同（全同光子），因而是相干的；自发辐射则是随机、非相干的。受激辐射在粒子数反转条件下可实现光放大，是激光产生的物理基础。

**22.**　玻尔氢原子的三条基本假设：

① 定态假设（2 分）：原子只能处于一系列具有不连续能量的稳定状态（定态），处于定态的电子绕核做圆周运动但不辐射电磁波，能量保持不变。

② 跃迁假设（2 分）：当原子从能量为 Eₙ 的定态跃迁到能量为 Eₘ 的定态时，会吸收或辐射一个光子，其频率满足 hν = |Eₙ − Eₘ|。

③ 角动量量子化假设（1 分）：电子做圆周运动时，其角动量只能取 ℏ 的整数倍，即 L = n·(h/2π)（n = 1, 2, 3 …）。

**四、计算题**

**23.（8 分）**

(1) 方均根速率 v<sub>rms</sub> = √(3RT/μ) = √(3 × 8.31 × 300 / 32×10⁻³)

= √(7479 / 0.032) = √(2.34×10⁵) ≈ 4.83×10² m/s（约 483 m/s）。（2 分）

(2) 平均平动动能 ε̄ = (3/2)kT = 1.5 × 1.38×10⁻²³ × 300 ≈ 6.21×10⁻²¹ J。（2 分）

(3) 刚性双原子分子自由度 i = 5，1 mol 内能 U = (i/2)RT = (5/2) × 8.31 × 300 ≈ 6.23×10³ J。（2 分）

(4) 分子数密度 n = p/(kT) = 1.0×10⁵ / (1.38×10⁻²³ × 300) ≈ 2.42×10²⁵ m⁻³；

平均自由程 λ̄ = 1 / (√2 π d² n) = 1 / (1.414 × 3.14 × (3.6×10⁻¹⁰)² × 2.42×10²⁵)

= 1 / (√2 × 3.14 × 1.296×10⁻¹⁹ × 2.42×10²⁵) ≈ 7.2×10⁻⁸ m。（2 分）

**24.（8 分）**

(1) 环上每个电荷元 dq 到轴上 P 点的距离都相等，r = √(R² + x²)，故各元电势可直接相加：

V(x) = (1/4πε₀) · q / √(R² + x²)。

代入数据：√(R² + x²) = √(0.20² + 0.15²) = √(0.04 + 0.0225) = √0.0625 = 0.25 m，

V<sub>P</sub> = 9.0×10⁹ × 2.0×10⁻⁸ / 0.25 ≈ 7.2×10² V（720 V）。（3 分）

(2) 由对称性，轴线上电场只有沿轴分量：E(x) = (1/4πε₀) · qx / (R² + x²)<sup>3/2</sup>。

E<sub>P</sub> = 9.0×10⁹ × 2.0×10⁻⁸ × 0.15 / (0.25)³ = 27 / 0.015625 ≈ 1.7×10³ V/m，

方向沿轴线由环心指向 P（即沿 x 轴正向，设 q > 0）。（3 分）

(3) 令 dE/dx = 0，可得轴线上场强在 x = ±R/√2 处取最大值（此处 |x| = 0.20/√2 ≈ 0.14 m）。（2 分）

25.（8 分）由对称性，取与电缆同轴、半径为 r 的圆为安培环路，B 沿环路切向且大小处处相等，∮B·dl = B·2πr = μ₀ I<sub>enc</sub>。

① r &lt; R₁（内导体内）：电流均匀分布，I<sub>enc</sub> = I·(r²/R₁²)，

B = μ₀ I r / (2π R₁²)，随 r 线性增大。

② R₁ &lt; r &lt; R₂（内外导体之间）：I<sub>enc</sub> = I，

B = μ₀ I / (2π r)。

③ R₂ < r < R₃（外导体内）：外导体反向电流在该处包围量为 I·(r²−R₂²)/(R₃²−R₂²)，

I<sub>enc</sub> = I − I·(r²−R₂²)/(R₃²−R₂²)，B = μ₀ I<sub>enc</sub> / (2π r)，随 r 增大而减小。

④ r &gt; R₃（电缆外）：I<sub>enc</sub> = I − I = 0，B = 0。（6 分）

(2) r = 2.0 mm 处于 R₁ < r < R₂ 区域：

B = μ₀ I / (2π r) = (4π×10⁻⁷ × 5.0) / (2π × 2.0×10⁻³) = (2×10⁻⁷ × 5.0) / (2.0×10⁻³) = 5.0×10⁻⁴ T = 0.50 mT。（2 分）

**26.（8 分）**

(1) 单球面折射：n₁/u + n₂/v = (n₂ − n₁)/R。凸面朝向入射光、曲率中心在出射侧，R > 0。

1.0/0.30 + 1.5/v = (1.5 − 1.0)/0.10 = 5.0，

1.5/v = 5.0 − 3.33 = 1.67 ⟹ v = 1.5/1.67 ≈ 0.90 m = 90 cm。

v > 0，像在折射面后方（玻璃中），为实像。（4 分）

(2) 近视眼远点在 0.50 m 处，需用凹（发散）透镜把无穷远物成像于其远点。对无穷远物 u → ∞，要求像距 v = −0.50 m（虚像，与物同侧）：

1/f = 1/v − 1/u = 1/(−0.50) − 0 = −2.0 m⁻¹，

即焦度 Φ = 1/f = −2.0 D（应配戴 −2.0 屈光度的凹透镜，相当于 200 度近视镜）。（4 分）

**27.（8 分）**

(1) ① 衰变常数 λ = ln2 / T = 0.693 / (8.0 × 86400 s) = 0.693 / (6.91×10⁵ s) ≈ 1.0×10⁻⁶ s⁻¹。（2 分）

② 24 天 = 3 个半衰期，N = N₀ · (1/2)³ = 1.0×10²⁰ / 8 = 1.25×10¹⁹ 个。（2 分）

③ 初始活度 A₀ = λN₀ = 1.0×10⁻⁶ × 1.0×10²⁰ = 1.0×10¹⁴ Bq。（2 分）

(2) 质量亏损 Δm = (m<sub>p</sub> + m<sub>n</sub>) − m<sub>d</sub> = (1.0073 + 1.0087) − 2.0136 = 0.0024 u；

结合能 E<sub>b</sub> = Δm × 931.5 MeV/u = 0.0024 × 931.5 ≈ 2.2 MeV（比结合能约 1.1 MeV/核子）。（2 分）
