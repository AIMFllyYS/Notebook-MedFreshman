**【详细解析】**

1.【B】

带电球面（总电荷Q，半径R），用高斯定理：取球外同心球面S（半径r > R），∮E·dS = Q/ε₀，E × 4πr² = Q/ε₀，故 E = Q/(4πε₀r²)。球内（r < R）E = 0。

2.【C】

高斯定理 ∮E·dS = ΣQₑₙcₗₒₛₑd/ε₀，右边只计高斯面内部的电荷总量（代数和），与面外电荷无关（面外电荷对面上各点E有贡献，但积分抵消）。

3.【A】

安培环路定理：∮B·dl = μ₀Iₑₙc，对无限长直导线（线电流I），取距导线r的圆形安培环路，B × 2πr = μ₀I，故 B = μ₀I/(2πr)。

4.【B】

安培环路定理：环路上各点的B由空间中所有电流（包括环路外的电流）决定（场的叠加原理），但积分 ∮B·dl 只等于穿过环路的电流（代数和）× μ₀。选B。

5.【A】

矩形线框在匀强磁场中匀速转动，磁通量 Φ = NBAcosωt，感应电动势 ε = -dΦ/dt = NBAω sinωt。
当 sinωt = 1（最大值）时，线框平面平行于B（线框法向垂直于B），此时电动势最大。选A。

6.【B】

杨氏双缝干涉亮纹间距 Δx = Dλ/d，与D成正比（注意非D²），与λ成正比，与d成反比。选B。

7.【A】

等厚干涉相邻亮纹条件：2ne = kλ 和 2ne' = (k+1)λ，故 Δe = e' - e = λ/(2n)。相邻纹间厚度差为 λ/(2n)。选A。

8.【C】

光栅方程 d sinθ = kλ，k\_max = d/λ = 2×10⁻⁶/(500×10⁻⁹) = 4，但 sinθ ≤ 1，k = 4 时 sinθ = 1（θ = 90°，边缘，实际刚好能看到），故最多到 k = ±4（实际可见 0, ±1, ±2, ±3）。选C。

9.【C】

马吕斯定律全过程：自然光通过起偏器P₁，强度变为 I₁ = I₀/2（只取一个振动分量）；再通过检偏器P₂（与P₁夹角θ），I₂ = I₁ cos²θ = (I₀/2) cos²θ。选C。

10.【B】

麦克斯韦预言位移电流：变化的电场（电位移D随时间变化，即 ∂D/∂t）等效为一种'电流'（位移电流），会产生磁场。这正是电磁波能在真空中传播的理论基础。选B。

**三、简答题答案**

1. 麦克斯韦方程组（5分）：

（积分形式，国际单位制）

① 高斯电场定律：∮E·dS = ΣQ/ε₀
   意义：电场线起于正电荷止于负电荷，电荷是电场的源（有源场）。

② 磁高斯定律：∮B·dS = 0
   意义：磁感线为封闭曲线，不存在磁单极子，磁场是无源场。

③ 法拉第电磁感应定律：∮E·dl = -dΦ\_B/dt
   意义：变化的磁场产生（感应）电场（涡旋电场），这是电磁感应的本质。

④ 安培-麦克斯韦定律：∮B·dl = μ₀(I\_传导 + ε₀ dΦ\_E/dt)
   意义：传导电流和变化的电场（位移电流）都能产生磁场。位移电流是麦克斯韦的核心创新。

这四个方程完整描述了电磁场的宏观规律，预言了电磁波的存在（c = 1/√(μ₀ε₀) = 3×10⁸ m/s）。

2. 光的干涉相干条件与薄膜增透（5分）：

【相干条件（三个）】：
① 频率相同（同频）
② 振动方向相同（同向/偏振方向不垂直）
③ 初相差恒定（初相差 Δφ = const，或相位差稳定）

【杨氏双缝中稳定干涉条纹的原因】：
双缝由同一光源S照射，两缝的光来自同一波前（分波前法），保证了频率相同、初相差恒定的条件，满足相干条件，因此产生稳定的干涉条纹。

【薄膜增透原理（医学光学应用）】：
近视眼镜镜片镀增透膜（如 MgF₂，n\_膜 ≈ 1.38）：
• 光在空气/膜界面反射（较疏→较密，有半波损失），在膜/玻璃界面也反射（依折射率大小而定）。
• 选取膜厚 e = λ/(4n\_膜)（四分之一波长），使两反射光光程差 = λ/2，发生相消干涉，减小反射（增大透射）。
• 对入射光波长（如 λ = 550 nm 绿光）增透效果最佳，提高光学仪器透光率。

**四、计算题详细解答（共50分）**

计算题1 解答（10分）：

已知：λ = 2×10⁻⁸ C/m，ε₀ = 8.85×10⁻¹²C²/(N·m²)

（1）用高斯定理求E（r = 0.1 m）：
  取半径 r = 0.1 m、长度 L 的柱形高斯面（侧面）：
  E × 2πrL = λL/ε₀
  E = λ/(2πε₀r) = 2×10⁻⁸ / (2π × 8.85×10⁻¹² × 0.1)
  E = 2×10⁻⁸ / (5.565×10⁻¹²) ≈ 3596 N/C ≈ 3.6×10³ N/C  ……[3分]

（2）电势（以r₀ = 1 m为零点）：
  V(r) = -∫[r₀→r] E dr = -∫[1→r] λ/(2πε₀r) dr
  V(r) = -λ/(2πε₀) × ln(r/r₀) = λ/(2πε₀) × ln(r₀/r)
  V(0.1) = 2×10⁻⁸ / (2π × 8.85×10⁻¹²) × ln(1/0.1)
  V(0.1) = 3596/10 × 2.303 ≈ 3596 × ln10 / 2π... 
  更简洁：V = (λ/2πε₀) ln(r₀/r) = 3596×0.1 × ln(10) ≈ 359.6 × 2.303 ≈ 828 V  ……[3分]

（3）静电场力做功：
  W = q[V(r₁) - V(r₂)] = q × (λ/2πε₀) × ln(r₂/r₁)
  V(0.1) ≈ 828 V，V(0.2) = (λ/2πε₀)ln(1/0.2) ≈ 359.6 × ln5 ≈ 359.6 × 1.609 ≈ 579 V
  W = q[V(r₁) - V(r₂)] = 2×10⁻⁹ × (828 - 579) = 2×10⁻⁹ × 249 ≈ 4.98×10⁻⁷ J  ……[4分]

计算题2 解答（10分）：

已知：a = 0.1 m，b = 0.2 m，R = 2 Ω，B = 1 T，ω = 100π rad/s，t = 0时Φ = 0

（1）磁通量 Φ(t)：
  线框面积 S = ab = 0.1 × 0.2 = 0.02 m²
  t = 0时平面平行于B（法线⊥B），Φ = 0；随旋转：Φ(t) = BS sinωt = 1×0.02×sin(100πt)
  Φ(t) = 0.02 sin(100πt) Wb  ……[2分]

（2）感应电动势 ε(t)：
  ε(t) = -dΦ/dt = -0.02 × 100π × cos(100πt)
  ε(t) = -2π cos(100πt) V  ……[2分]

（3）最大值和有效值：
  ε\_m = 0.02 × 100π = 2π ≈ 6.28 V  ……[3分]
  ε\_rms = ε\_m/√2 = 2π/√2 = π√2 ≈ 4.44 V

（4）电流有效值：
  I\_rms = ε\_rms/R = π√2/2 ≈ 2.22 A  ……[3分]

计算题3 解答（10分）：

（1）空气中 Δx = Dλ/d = 1.0 × 589.3×10⁻⁹ / (0.2×10⁻³) = 2.946×10⁻³ m ≈ 2.95 mm  ……[3分]

（2）液体中有效波长 λ' = λ/n = 589.3/1.33 ≈ 443 nm
  Δx' = Dλ'/ d = 1.0 × 443×10⁻⁹ / (0.2×10⁻³) ≈ 2.215×10⁻³ m ≈ 2.22 mm  ……[3分]
  (或直接：Δx' = Δx/n = 2.95/1.33 ≈ 2.22 mm)

（3）插入玻璃片后，一条缝光程增加 (n'-1)e。
  零级亮纹移到第一级亮纹位置，条纹移动量等于Δx，对应光程差变化为λ：
  (n'-1)e = λ
  e = λ/(n'-1) = 589.3×10⁻⁹/(1.5-1) = 589.3×10⁻⁹/0.5 = 1178.6×10⁻⁹ m ≈ 1.18 μm  ……[4分]

计算题4 解答（10分）：

（1）光栅方程：d sinθ = kλ（k = 0, ±1, ±2, ...）
  d = 1.67×10⁻⁶ m，λ = 589.3 nm = 589.3×10⁻⁹ m
  k = ±1：sinθ₁ = λ/d = 589.3×10⁻⁹/1.67×10⁻⁶ = 0.353 → θ₁ ≈ 20.7°
  k = ±2：sinθ₂ = 0.706 → θ₂ ≈ 44.9°
  k = ±3：sinθ₃ = 1.058 > 1，不存在（超过90°）  ……[4分]
  故可见级次为 k = 0, ±1, ±2（最大级次 k\_max = ±2）

（2）总缝数 N = L/d = 0.03/(1.67×10⁻⁶) ≈ 18000 缝
  第一级分辨本领：A = kN = 1 × 18000 = 18000  ……[3分]

（3）能否分辨钠双线 Δλ = 0.6 nm？
  最小可分辨波长差：δλ = λ/A = 589.3/18000 ≈ 0.033 nm
  由于 δλ = 0.033 nm < Δλ = 0.6 nm，可以分辨（分辨本领足够）。  ……[3分]

计算题5 解答（10分）：

（1）自然光（I₀）经P₁：I₁ = I₀/2  ……[2分]

（2）P₁与P₂夹角 θ₁₂ = 30°，经P₂（马吕斯定律）：
  I₂ = I₁ cos²30° = (I₀/2) × (√3/2)² = (I₀/2) × (3/4) = 3I₀/8  ……[2分]

（3）插入P₃，P₃与P₁夹角 30°，P₃与P₂夹角也是 30°：
  经P₁后：I₁ = I₀/2
  经P₃（与P₁夹角30°）：I₃ = I₁ cos²30° = (I₀/2)(3/4) = 3I₀/8
  经P₂（与P₃夹角30°）：I\_final = I₃ cos²30° = (3I₀/8)(3/4) = 9I₀/32  ……[4分]

（4）直接P₁和P₃（P₁与P₃夹角=P₁与P₂夹角+P₂与P₃夹角=60°）：
  I' = (I₀/2) cos²60° = (I₀/2)(1/2)² = I₀/8  ……[2分]
  比较：I₃ = 9I₀/32 > I' = I₀/8 = 4I₀/32
  原因：中间插入P₃后，每次只旋转30°，光能损失较少；直接经过60°时损失更大。
