**运动学基础知识**

**参考系与坐标系**

参考系的选择：研究物体运动需选择参考系，参考系选择具有任意性，但应根据研究目的简化研究。不同参考系对物体运动的描述不同，如火车上的人相对对面的人静止，相对路旁的人运动。

坐标系的建立：为定量描述物体位置和运动，需在参考系中建立坐标系，常用直角坐标系，满足右手定则。

**质点的概念**

质点的定义：质点是一个抽象概念，指不考虑物体大小和形状，只有质量的几何点。当物体大小对研究问题影响不大时，可将其看成质点，如研究火车从武汉到北京的运动、地球在银河系中的运动等。

质点的应用：在某些情况下，物体大小和形状不能忽略，可将其分解为多个小体积基元，每个基元按质点处理。

**位置矢量与运动方程**

位置矢量的表示：位置矢量简称位矢或矢径，可表示为从原点到物体位置的矢量。在直角坐标系中，可分解为三个坐标轴上的分量，也可用长度和与坐标轴的夹角表示。

运动方程：位置矢量随时间变化的函数称为运动方程，消去时间后得到的方程为轨迹方程。

**位移与路程**

位移的定义：位移是位置矢量的移动，用 \$\Delta\vec{r}\$ 表示，是矢量，方向从起点指向终点。

路程的定义：路程是物体运动轨迹的长度，是标量。一般情况下，位移和路程不相等，只有在物体做直线运动且不回头时，两者相等。当 \$\Delta t\$ 趋近于 0 时，位移和路程相等，但不等于 \$\Delta r\$（位置矢量模长的变化）。

**速度与加速度**

速度的定义：速度是位移对时间的导数，平均速度是一段时间内的位移除以时间，瞬时速度是 \$\Delta t\$ 趋近于 0 时平均速度的极限。速度是矢量，方向沿运动轨迹的切向。

加速度的定义：加速度是速度对时间的导数，平均加速度是一段时间内速度的变化量除以时间，瞬时加速度是 \$\Delta t\$ 趋近于 0 时平均加速度的极限。加速度方向与速度变化量的方向相同，指向轨迹的凹侧。

**圆周运动**

角速度与线速度：角速度是矢量，方向垂直于运动平面，用右手定则确定。线速度大小等于角速度乘以半径，方向与角速度和半径的叉乘方向相同。

角加速度：角加速度是角位置对时间的二阶导数，方向沿角速度方向。

切向加速度与法向加速度：切向加速度是线速度大小对时间的导数，法向加速度是速度方向变化产生的加速度，在匀速圆周运动中，切向加速度为 0，法向加速度大小为 \$v^2/r\$ 或 \$\omega^2r\$。

**运动学问题的求解方法**

微分问题：已知位移求速度和加速度，通过求导得到。

积分问题：已知加速度求速度和位移，通过积分得到，积分时需根据初始条件确定积分上下限。

**例题分析**

圆周运动例题：已知质点运动方程 \$x = r\cos\omega t\$，求位移、速度、加速度、轨迹方程和运动情况。通过求导得到速度和加速度，利用三角函数关系消去时间得到轨迹方程，证明位矢和速度垂直，计算特定时间内的位移、路程、平均速度和平均速率。

炮弹运动例题：以与地面成 \$\theta\_0\$ 角度、初始速度为 \$V\_0\$ 发射炮弹，考虑空气阻力，分析其运动状态，求轨迹方程。通过受力分析得到加速度，分解到 \$x\$ 和 \$y\$ 方向，利用分离变量法积分求解速度和位移，消去时间得到轨迹方程。

[45:41](https://1037solo.feishu.cn/minutes/obcn7w2yuiy15o9yw9cluhj9?t=2741000)  **质点概念、抽象原因及参考系选择探讨**

<table><tr><td><p>本章节主要介绍了质点的概念，它是物理上抽象出的有质量但无大小形状的几何点，现实中一般不存在。引入质点是为方便研究，当物体大小对研究问题不重要时可将其看成质点，如火车从武汉到北京、地球在银河系中的运动；若物体大小形状不能忽略，则需分解处理，同时选择是否看成质点要依据研究内容。</p></td></tr></table>

[54:49](https://1037solo.feishu.cn/minutes/obcn7w2yuiy15o9yw9cluhj9?t=3289000)  **物体运动位置矢量的表示及随时间变化情况**

<table><tr><td><p>本章节主要围绕描述物体运动所需内容展开。以物体沿曲线运动为例，介绍位置矢量 r，为区分手写与打印体，矢量加小箭头。位矢有两种表示方式，一是分解到直角坐标系的三个单位矢量上，二是用长度及与坐标轴夹角表示，且位矢会随时间变化。</p></td></tr></table>

[57:59](https://1037solo.feishu.cn/minutes/obcn7w2yuiy15o9yw9cluhj9?t=3479000)  **质点运动方程与轨迹方程的概念及关系**

<table><tr><td><p>本章节主要介绍了质点运动相关的两个重要概念。一是运动方程，质点在空间位置随时间变化的函数，可写成矢量或标量形式，体现不同时间位置矢量的变化；二是轨迹方程，若将运动方程中的时间消掉，得到只含x、y、z的方程，如y=x²所示的抛物线，它不含时间信息，反映质点运动轨迹。</p></td></tr></table>

[59:18](https://1037solo.feishu.cn/minutes/obcn7w2yuiy15o9yw9cluhj9?t=3558000)  **位移、路程与Δr的概念、区别及特殊情况**

<table><tr><td><p>本章节主要讨论运动轨迹中的位移、路程和 Δr。位移用 Δr 表示，是质点位置变化，可投影到直角坐标系；路程是质点运动的弧长，是标量。一般位移和路程不相等，Δr 既不等于位移也不等于路程，只有走直线不回头时三者相等，当 Δt 趋近于 0，位移和路程相等，但仍不等于 Δr。</p></td></tr></table>

[01:03:44](https://1037solo.feishu.cn/minutes/obcn7w2yuiy15o9yw9cluhj9?t=3824000)  **速度定义、方向、计算及直角坐标系分解**

<table><tr><td><p>本章节主要讲解速度相关知识。速度是向量，定义为位移除以时间，平均速度是一段时间内位移除以该时间，方向沿Δr方向；瞬时速度是t趋近0时平均速度的极限，方向沿运动轨迹切向，大小是dr/dt取模。在直角坐标系中，速度可分解为x、y、z方向计算。速度还可用模长加夹角表示，物理意义是位移变化快慢。</p></td></tr></table>

<table><tr><td><p>本章节主要讲解加速度相关知识。加速度是速度对时间的变化，表示速度变化快慢，可求平均和瞬时加速度，其方向与速度变化方向相同且指向轨迹凹处。还提到加速度在直角坐标系的表示方式。此外，探讨了位移的三阶导，指出因加速度与力相关，人能感受加速度实则是感受力。</p></td></tr></table>

[01:24:36](https://1037solo.feishu.cn/minutes/obcn7w2yuiy15o9yw9cluhj9?t=5076000)  **运动第一章微分与积分问题及解题关键**

<table><tr><td><p>本章节介绍了运动中的两类问题，即微分和积分问题。微分问题是已知位移求速度、已知速度求加速度；积分问题则相反，已知加速度求速度、已知速度求位移。解题关键在于积分时需根据题目找初始条件作为积分上下限，而微分时不需要考虑初始条件。</p></td></tr></table>
