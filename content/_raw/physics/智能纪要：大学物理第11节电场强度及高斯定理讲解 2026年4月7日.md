**带电圆环电场强度**

x 方向电场分量推导：由于对称性，垂直于 x 轴的电场为 0，x 方向电场分量等于 \$B\_1\$ 乘以 \$\cos\theta\$，其中 \$\cos\theta\$ 为 \$x\$ 除以 \$r\$（\$r\$ 为 \$x^2 + R^2\$ 的开方）。对 \$dx\$ 求积分，将 \$dq\$ 积分后得到总电量 \$q\$，最终电场强度表达式为 \$q x / (x^2 + R^2)^{3/2}\$，方向沿 x 方向。

特殊情况讨论：当 \$x\$ 远远大于 \$R\$ 时，可将圆环看成点电荷；当 \$x = 0\$ 时，电场强度为 0；电场强度最大值出现在 \$x = \frac{\sqrt{2}}{2}R\$ 处。

**带电圆盘电场强度**

微元法求解：将圆盘视为许多小圆环，取宽度为 \$\Delta r\$ 的圆环微元，其带电量 \$dq = \sigma \times 2\pi r \times dr\$。利用圆环电场强度公式，将 \$q\$ 替换为 \$dq\$，对 \$r\$ 从 0 到 \$R\$ 求积分。通过换元法（令 \$u = r^2 + x^2\$）可简化积分计算。

特殊情况讨论：当 \$x\$ 远远大于 \$R\$ 时，可将圆盘看成点电荷；当 \$R\$ 远远大于 \$x\$ 时，圆盘可视为无限大平面，产生垂直于平面的均匀电场，电场强度为 \$\sigma / (2\varepsilon\_0)\$。

**带电细棒电场强度**

电场分量分析：取电荷元 \$dy\$，其产生的电场在 y 方向场强抵消，合场强只沿 x 方向。\$dE\_x\$ 与 \$\cos\alpha\$ 有关，\$\cos\alpha = x / \sqrt{x^2 + y^2}\$。对 \$dy\$ 从 0 到 \$\frac{l}{2}\$ 积分（考虑对称性），可得到电场强度表达式。

特殊情况讨论：当 \$l\$ 趋近于无穷大或 \$l\$ 远远大于 \$x\$ 时，电场强度为 \$\lambda / (2\pi\varepsilon\_0 x)\$，方向沿 x 方向。若 \$\lambda > 0\$，电场方向指向外面；若 \$\lambda < 0\$，电场方向沿镜像向内。

**带电平面电场强度**

微元法求解：平面可看成无数条宽度为 \$dy\$ 的细线组成，线电荷密度 \$\lambda = \sigma \times dy\$。利用细线电场强度公式，对 \$dy\$ 从 0 到正无穷积分（考虑对称性），可得到电场强度表达式为 \$\sigma / (2\pi\varepsilon\_0)\$，方向垂直于平面。

**电场线与电通量**

**电场线概念**

方向表示：电场线上每一点的切线方向表示该电场的强度方向，用于形象描述电场。

密度表示：电场线中的任意一点处，通过该点垂直于电场单位面积上的电场线的根数等于电场的强度大小。电场线稀疏的地方场强较小，电场线密集的地方场强较大。

特征：静电场的电场线起于正电荷，止于负电荷，不会在无电荷处中断，也不是闭合回路；在没有电荷的空间里，任何两条电场线不会相交。

**电通量概念**

定义：通过电场中任意给定面的电场线的总数就是该面的电通量 \$\varPhi\_E\$。

计算方法：对于均匀电场，若平面 \$S\$ 垂直于电场方向，电通量 \$\varPhi\_E = E \times S\$；若平面法线与电场方向成夹角 \$\theta\$，则 \$\varPhi\_E = E \times S \times \cos\theta\$。对于非均匀电场，需取微元 \$dS\$，电通量微元 \$d\varPhi\_E = E \times dS \times \cos\theta\$，总电通量通过对面积求积分得到。

正负判断：当 \$\theta < 90^{\circ}\$ 时，\$\varPhi\_E > 0\$；当 \$\theta = 90^{\circ}\$ 时，\$\varPhi\_E = 0\$；当 \$\theta > 90^{\circ}\$ 时，\$\varPhi\_E < 0\$。电通量单位为牛·米²/库仑。

**电通量实例**：以三棱柱放置在均匀电场中为例，前后和下面与电场平行，电通量为 0；左和右两面，通过向量相乘计算电通量，最终整个三棱柱的电场强度通量为 0。

**高斯定理**

**定理内容**：真空中通过任意闭合曲面的电通量等于该曲面所包围的所有电荷的代数和除以 \$\varepsilon\_0\$。

**证明方法**：以点电荷在真空中的电场为例，取半径为 \$r\$ 的球面 \$S\$，电场强度方向与球面法线方向平行，通过对球面求积分 \$E \cdot dS\$ 计算电通量，最终得到电通量等于 \$q / \varepsilon\_0\$，证明了高斯定理。
