**运动学基础**

**直角坐标与矢量表示**

矢量分解便于解题：引入直角坐标后，可定量描述运动。位置矢量、位移矢量、速度等矢量写成 \$\mathrm{i}\mathrm{j}\mathrm{k}\$ 形式，如 \$\mathrm{x}\mathrm{i}、 \mathrm{Y}\mathrm{j}、 \mathrm{Z}\mathrm{k}\$，可将运动看成 \$\mathrm{I}\mathrm{J}\mathrm{K}\$ 三个坐标系上线性运动的叠加，方便后续用标量解题。例如在处理速度矢量时，可分别对 \$\mathrm{i}、\mathrm{j}\$ 方向按线性运动计算，最后再合成矢量。

圆周运动与矢量叉乘：圆周运动中涉及矢量叉乘，其大小为两矢量绝对值相乘再乘以 \$\sin\theta\$，与点乘（\$\cos\theta\$）不同，方向按右手定则确定，食指往内弯，大拇指方向即为叉乘后向量的方向。

**运动学解题方法**

微分与积分应用：运动学解题中，已知位置矢量求速度用微分，已知加速度求速度、位置或轨迹用积分。积分时需注意两个问题，一是从题目中找初始条件，二是当加速度与速度有关而非与时间有关时，不能直接对 \$t\$ 求积分，需用分离变量法。如在炮弹轨迹问题中，炮弹加速度与速度有关，通过分解加速度到 \$\mathrm{i}、\mathrm{j}\$ 方向，用分离变量法求积分得到 \$x、y\$ 方向的运动方程，再消去 \$t\$ 得到轨迹方程。

例题分析：以炮弹发射和质点加速运动为例，详细讲解解题思路。炮弹发射问题考虑了空气阻力，其轨迹不是抛物线；质点加速运动问题根据加速度与速度的关系，用分离变量法求出速度与时间、位置与时间的函数关系，还讨论了加速度与位置有关时引入 \$\mathrm{d}x\$ 进行分离变量求解的方法。

**相对运动**

**相对运动原理**

参考系选择影响运动描述：选择不同的参考系，对物体运动的描述不同。如坐在火车里，选地面为参考系火车匀速直线运动，选火车为参考系火车里的人相对静止。

绝对、相对与牵连速度关系：绝对速度是物体相对于静止参考系的速度，相对速度是物体相对于运动参考系的速度，牵连速度是运动参考系相对于静止参考系的速度，绝对速度等于相对速度加上牵连速度，对速度求导得到的加速度也满足此关系，即绝对加速度等于牵连加速度加上相对加速度。

**相对运动例题**

雨滴下落问题：在无风下雨天，火车匀速前进，车内旅客看到窗外雨滴与铅锤垂直方向成 \$75^{\circ}\$ 夹角下落。以地面为绝对参考系，火车为牵连参考系，通过建立矢量三角形，已知车对地的速度和夹角，可求出雨对地的速度和雨对车的速度。

**子弹击中靶问题**

传统解法：要使子弹击中靶，需满足时间和位置都在同一点。子弹与靶加速度都为重力加速度 \$g\$ 且垂直向下，根据运动方程 \$R\_0 = V\_T + \frac{1}{2}at^2\$，子弹初始位置为 \$0\$，初始速度为 \$v\_0\$，靶初始位置为 \$R\_{OT}\$，初始速度为 \$0\$，当 \$R\_P = R\_T\$ 时可约掉部分项，得到 \$V\_{OP}t = R\_{OT}\$，说明 \$V\_{OT}\$ 与 \$R\_{OT}\$ 矢量方向平行才能击中，进而求出时间。

转换参考系解法：从猴子视角看，子弹与靶同时做自由落体运动，子弹相对于靶子做匀速直线运动，一定会打到靶子上。此问题也可用高中知识求解，通过计算横向移动时间、靶掉落距离和子弹在 \$y\$ 方向移动距离，使其相等可求出击中时间。

**牛顿运动定律**

**牛顿运动定律内容**

牛顿第一定律（惯性定律）：物体在不受外力或受平衡力作用时，总是保持静止或匀速直线运动，此定律成立的参考系为惯性系。

牛顿第二定律：物体受到外力会改变运动状态，加速度与所受合力成正比，方向沿合外力方向。高中课本定义为 \$F = ma\$，但牛顿最早写成合外力等于动量的变化率，在低速宏观下 \$m\$ 不变可得到 \$F = ma\$，在高速相对论力学中 \$m\$ 会变化。力和加速度同时产生，动量有方向，\$F\$ 是矢量，大学物理常将其写成分量形式，如 \$F\_X = ma\_X\$、\$F\_Y = ma\_Y\$，方便在直角坐标系中用标量计算。

牛顿第三定律：两个物体之间的作用力和反作用力大小相等、方向相反，虽看似简单，但在很多问题中有重要应用。

**牛顿运动定律应用条件**

惯性系要求：牛顿第二定律必须在惯性系中成立，惯性系可大致认为是静止或匀速直线运动的参考系，如不考虑地球自转公转时，地球可看作惯性系。

积分与变量构造：\$F = ma\$ 是微分方程，当 \$F\$ 是关于 \$t\$ 的函数时可直接两边求积分，当 \$F\$ 是 \$v\$ 或 \$r\$ 的函数时，可能需构造变量用分离变量法求解。

**牛顿运动定律解题步骤**

确定研究对象与分析状态：明确研究对象，列出已知量和未知量，定性分析研究对象的运动状态，包括轨迹、速度和加速度。

隔离物体与受力分析：隔离每一个物体，分析其受力情况，这是牛顿第二定律应用的关键。

选择参考系与建立坐标：选择合适的参考系并建立坐标，有时选择非惯性参考系可简化计算。

列方程与求解讨论：将质量、加速度、力按牛顿第二定律或分量形式列出方程，求解方程并对结果进行讨论。

牛顿运动定律解题步骤

**非惯性系与惯性力**

**非惯性系概念**：若选择的参考系中，物体不受力却加速或减速运动，说明该参考系不是惯性系，可能是参考系本身在加速或减速，如加速运动的房间或公交车。

**惯性力引入原因**：在非惯性系中牛顿第二定律不成立，为使其适用，需引入惯性力。惯性力是虚构力，无施力物体和反作用力。

**惯性力形式与计算**

加速平动参考系：质量为 \$m\$ 的质点，在真实外力 \$F\$ 作用下相对于惯性系 \$S\$ 产生加速度 \$a'\$，另一参考系 \$S\_1'\$ 相对于惯性系以加速度 \$A\_0\$ 沿直线加速平动，质点在 \$S\_1'\$ 参考系中的加速度为 \$A\_1'\$。根据矢量叠加原理，绝对加速度等于相对加速度加上牵连加速度，即 \$a' = A\_1' + A\_0\$，两边同乘 \$m\$ 可得 \$F = mA\_1' + mA\_0\$，引入惯性力 \$F\_{惯} = -mA\_0\$，则在 \$S\_1'\$ 参考系中有 \$F' = F + F\_{惯} = mA\_1'\$，满足牛顿第二定律形式。

匀速转动参考系：在匀速转动的水平圆盘上，方块因摩擦力相对静止，以地面为参考系，方块做匀速圆周运动，向心力等于摩擦力；以转盘为参考系，方块加速度为 \$0\$，合力应为 \$0\$，但实际受向心力，所以存在一个与向心力方向相反的惯性力，即高中学的离心力，离心力在生活中有很多应用，如衣服甩干和医学离心机。

**惯性力应用例题**

电梯问题：以地面为参考系，小孩受重力和支持力，满足牛顿第二定律；以电梯为参考系，小孩加速度为 \$0\$，原等式不成立，需引入惯性力使牛顿第二定律适用。从宇航员角度看，宇宙飞船做圆周运动时，重力提供向心力，同时存在离心力，当两者相等时宇航员感觉失重。

光滑三角形问题：光滑三角形和小块物体在无摩擦情况下，小物体下滑会推动大三角形向右加速运动。以大三角形为非惯性参考系，分析小物体受力，除重力和支持力外，还受与大三角形加速度方向相反的惯性力。通过建立坐标，分别列出小物体和大三角形在 \$x、y\$ 方向的运动方程，将动力学问题转化为静力学问题处理，最后求解出小物体和大三角形相对于地面的加速度。
