**一、单项选择题答案（共20分）**

1.B  2.A  3.C  4.B  5.A  6.B  7.C  8.B  9.B  10.B

**【详细解析】**

1.【B】

匀速圆周运动速率恒定（大小不变），但方向不断改变，因此速度是矢量且在变化。加速度（向心加速度）大小 a = v²/R，方向始终指向圆心（不为零）。故选B。

2.【A】

由牛顿第二定律：F = ma，a = F/m = 3t²/2 = 1.5t² m/s²
由 dv = a·dt，积分得：v = ∫₀ᵗ 1.5t² dt = 0.5t³ m/s
t = 2 s时：v = 0.5 × 2³ = 0.5 × 8 = 4 m/s。故选A。

3.【C】

简谐振动中，当质点在平衡位置时，位移x = 0，弹性恢复力 F = -kx = 0，故加速度 a = F/m = 0。
此时动能最大即速度最大（能量守恒：½mv² = ½kA²，速度最大 v\_max = Aω）。故选C。

4.【B】

已知沿x轴正方向传播，角频率ω = 4π，波速u = 2 m/s，则波数 k = ω/u = 4π/2 = 2π rad/m。
波函数一般形式：y = A cos(ωt - kx + φ₀)。
t = 0，x = 0时：y₀ = A cosφ₀ = 0 → φ₀ = ±π/2
速度 v = -Aω sinφ₀ > 0 → sinφ₀ < 0 → φ₀ = -π/2
故 y = A cos(4πt - 2πx - π/2) = A sin(4πt - 2πx)。选B。

5.【A】

理想气体分子间无相互作用势能，内能只是所有分子动能之和。由能均分定理，每个自由度的平均动能为 kT/2，内能仅是温度T的函数，与体积、压强无关。故选A。

6.【B】

麦克斯韦速率分布律给出三种特征速率：
最概然速率 vₚ = √(2RT/M)
平均速率 v̄ = √(8RT/πM)
均方根速率 v\_rms = √(3RT/M)
数值比：vₚ : v̄ : v\_rms = 1 : 1.128 : 1.225
故 vₚ < v̄。选B。

7.【C】

由连续方程 S₁v₁ = S₂v₂，截面减小→流速增大。
由伯努利方程 p + ½ρv² + ρgh = const（水平管道，h不变），流速增大→压强减小。故选C。

8.【B】

液体的表面张力系数（表面张力）随温度升高而减小。高温时分子热运动加剧，分子间距增大，液面收缩力（表面张力）减小。例：沸腾时水的表面张力近乎为零。故选B。

9.【B】

两相干波相遇时，路程差 Δr = 3λ/2 = (2×1+1)λ/2，满足干涉相消条件 Δr = (2k+1)λ/2（k = 1），故该点干涉减弱（振幅最小）。选B。

10.【B】

多普勒效应公式：f' = f × (u+v\_R)/(u-v\_S)
波源不动(v\_S = 0)，观测者向波源运动(v\_R > 0)：
f' = f × (u+v\_R)/u > f，观测频率增大。选B。

**三、简答题答案（共10分）**

1. 简谐振动特征（5分）：

【特征】：回复力与位移成正比且方向相反：F = -kx（弹性恢复力），做功无阻尼，振动能量守恒。

【运动方程】：对弹簧振子：ma = -kx，即 d²x/dt² + ω²x = 0，其中 ω² = k/m。

【一般解】：x(t) = A cos(ωt + φ₀)
其中：
• 振幅 A：质点离开平衡位置的最大距离，由初始条件决定，A = √(x₀² + v₀²/ω²)。
• 角频率 ω：描述振动快慢，ω = √(k/m)（弹簧振子），单位 rad/s，ω = 2π/T。
• 初相位 φ₀：决定 t = 0 时质点的状态（位置和速度方向），由初始条件决定：cosφ₀ = x₀/A，sinφ₀ = -v₀/(Aω)。

2. 热力学第一定律（5分）：

【内容】：热力学系统从外界吸收的热量，一部分用于增加系统内能，另一部分用于对外做功。

【数学表达式】：ΔU = Q - W，或 dU = δQ - δW
其中：Q 为系统吸收的热量，W 为系统对外做的功，ΔU 为内能变化。

四种准静态过程分析（理想气体，ν = 1 mol，双原子 i = 5）：
┌──────┬────────┬────────────┬────────────────────────┐
│ 过程 │ ΔU     │ W          │ Q                      │
├──────┼────────┼────────────┼────────────────────────┤
│ 等温 │ 0      │ νRT ln(V₂/V₁)│ = W（从外界吸热）       │
│ 等压 │ νCᵥΔT  │ pΔV = νRΔT │ νCₚΔT（Cₚ = Cᵥ + R）   │
│ 等容 │ νCᵥΔT  │ 0          │ = ΔU = νCᵥΔT          │
│ 绝热 │ νCᵥΔT  │ = -ΔU = -νCᵥΔT│ 0                 │
└──────┴────────┴────────────┴────────────────────────┘

**四、计算题详细解答（共50分）**

计算题1 解答（10分）：

已知：a = 6t m/s²，x₀ = 2 m，v₀ = 1 m/s

（1）求速度 v(t)：
  由 a = dv/dt，积分：
  v(t) = v₀ + ∫₀ᵗ a dt = 1 + ∫₀ᵗ 6t dt = 1 + 3t² m/s  ……[3分]

（2）求位置 x(t)：
  由 v = dx/dt，积分：
  x(t) = x₀ + ∫₀ᵗ v dt = 2 + ∫₀ᵗ (1 + 3t²) dt = 2 + t + t³ m  ……[3分]

（3）t = 2 s时：
  v(2) = 1 + 3×2² = 1 + 12 = 13 m/s  ……[2分]
  x(2) = 2 + 2 + 2³ = 2 + 2 + 8 = 12 m  ……[2分]

计算题2 解答（10分）：

已知：k = 50 N/m，m = 0.5 kg，x₀ = 0.1 m，v₀ = 0.5 m/s（正方向）

（1）角频率和周期：
  ω = √(k/m) = √(50/0.5) = √100 = 10 rad/s  ……[2分]
  T = 2π/ω = 2π/10 ≈ 0.628 s  ……[1分]

（2）振幅 A：
  由能量守恒：½kA² = ½kx₀² + ½mv₀²
  A = √(x₀² + v₀²/ω²) = √(0.1² + 0.5²/10²)
  A = √(0.01 + 0.0025) = √0.0125 = 0.112 m ≈ 0.112 m  ……[3分]

（3）初相位和振动方程：
  由 x₀ = A cosφ₀：cosφ₀ = 0.1/0.112 = 0.894 → φ₀ ≈ ±26.6°
  由 v₀ = -Aω sinφ₀ > 0：sinφ₀ < 0 → φ₀ = -26.6° = -0.464 rad  ……[2分]
  x(t) = 0.112 cos(10t - 0.464) m  ……[2分]

计算题3 解答（10分）：

已知：f = 1 Hz，u = 10 m/s，λ = u/f = 10 m，S₁S₂ = 15 m，S₂P = 3 m（P在延长线上）

（1）P到S₁的距离：S₁P = S₁S₂ + S₂P = 15 + 3 = 18 m
  路程差：Δr = S₁P - S₂P = 18 - 3 = 15 m  ……[3分]

（2）判断干涉情况：
  Δr/λ = 15/10 = 1.5 = 3/2（半整数）
  满足相消条件 Δr = (2k+1)λ/2（k = 1），故P点干涉减弱（极小点）  ……[3分]

（3）合振幅：
  两波源同相（初相相同），路程差为奇数半波长时，合振幅为 |A₁ - A₂|。
  若 A₁ = A₂ = A，则合振幅 = |A - A| = 0  ……[4分]

计算题4 解答（10分）：

已知：p₁ = 2×10⁵ Pa，V₁ = 10 L = 0.01 m³，V₂ = 20 L = 0.02 m³
p₃ = p₁/2 = 1×10⁵ Pa，V₃ = V₂ = 0.02 m³

（1）A→B等温膨胀（T\_A = T\_B）：
  p₁V₁ = p₂V₂ → p₂ = p₁V₁/V₂ = 2×10⁵ × 0.01/0.02 = 1×10⁵ Pa
  W\_AB = νRT ln(V₂/V₁) = p₁V₁ ln(V₂/V₁)
  W\_AB = 2×10⁵ × 0.01 × ln(2) = 2000 × 0.693 = 1386 J ≈ 1.39 kJ  ……[3分]

（2）B→C等容（V不变）：
  T\_B = p₂V₂/νR = p₁V₁/νR（等温，T\_B = T\_A）
  T\_C = p₃V₃/νR = (p₁/2)V₂/νR
  T\_A = p₁V₁/νR；T\_C = (p₁/2)(2V₁)/νR = p₁V₁/νR = T\_A
  故 T\_C = T\_A = T\_B（温度相同！）
  ΔU\_BC = νCᵥ(T\_C - T\_B) = 0  ……[3分]
  （等容过程 W = 0，Q\_BC = ΔU\_BC = 0，说明B与C温度相同）

（3）整个循环A→B→C：
  A到C总内能变化 ΔU\_total = νCᵥ(T\_C - T\_A) = 0
  总功 W\_total = W\_AB + W\_BC = 1386 + 0 = 1386 J
  由热力学第一定律：Q\_total = ΔU\_total + W\_total = 0 + 1386 ≈ 1.39 kJ  ……[4分]

计算题5 解答（10分）：

已知：R = 2 mm = 0.002 m（圆管内半径），L = 0.5 m，Δp = 200 Pa，η = 3×10⁻³ Pa·s

（1）泊肃叶定律公式：Q = πR⁴Δp/(8ηL)  ……[2分]
  Q = π × (0.002)⁴ × 200 / (8 × 3×10⁻³ × 0.5)
  Q = π × 1.6×10⁻¹¹ × 200 / (1.2×10⁻²)
  Q = π × 3.2×10⁻⁹ / 1.2×10⁻²
  Q = π × 2.667×10⁻⁷ ≈ 8.38×10⁻⁷ m³/s ≈ 0.838 mL/s  ……[3分]

（2）管中心流速最大值（抛物面顶点）：
  v\_max = R²Δp/(4ηL) = (0.002)² × 200 / (4 × 3×10⁻³ × 0.5)
  v\_max = 4×10⁻⁶ × 200 / (6×10⁻³)
  v\_max = 8×10⁻⁴ / 6×10⁻³ = 0.133 m/s  ……[3分]

（3）管壁处（r = R）流速：
  由抛物面速度分布公式 v(r) = (Δp/4ηL)(R² - r²)
  当 r = R：v(R) = 0（无滑移边界条件）  ……[2分]
  速度分布规律：v(r) = v\_max(1 - r²/R²)，形成旋转抛物面分布，中心流速最大，管壁流速为零。
