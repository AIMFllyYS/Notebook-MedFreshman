**【详细解析】**

1.【C】

全反射发生在光密介质→光疏介质时。当入射角恰好等于临界角 θ\_c 时，折射角 = 90°（折射光沿界面传播）。sinθ\_c = n₂/n₁（n₂ < n₁）。选C。

2.【A】

薄透镜公式 1/u + 1/v = 1/f（使用符号规定：实物u > 0，实像v > 0，凸透镜f > 0）。当实物经凸透镜成实像：u > f，则 v > 0，满足A的条件。选A。

3.【C】

正常人眼的明视距离（最近清晰视距，即近点）约为 25 cm（0.25 m），这是视觉系统的重要参数，与眼镜度数计算相关。选C。

4.【B】

氢原子能级：Eₙ = -13.6/n² eV
E₄ = -13.6/16 = -0.85 eV，E₂ = -13.6/4 = -3.4 eV
ΔE = E₄ - E₂ = -0.85 - (-3.4) = 2.55 eV（可见光波段，λ ≈ 486 nm，属巴尔末系Hβ线）。选B。

5.【B】

截止频率（红限频率）定义：当光子能量 hν₀ 恰好等于逸出功 W 时，光电子以零初动能逸出，即 hν₀ = W。低于此频率，无论光强多大都不产生光电效应。选B。

6.【A】

λ = h/(m\_e v) = 6.63×10⁻³⁴ / (9.11×10⁻³¹ × 10⁶) = 6.63×10⁻³⁴ / (9.11×10⁻²⁵) = 7.28×10⁻¹⁰ m ≈ 0.73 nm。选A。

7.【B】

放射性衰变 N = N₀ e^(-λt)，当 N = N₀/2 时，e^(-λT₁/₂) = 1/2，-λT₁/₂ = ln(1/2) = -ln2，故 T₁/₂ = ln2/λ = 0.693/λ。选B。

8.【C】

X射线（X ray）是波长很短（0.001\~10 nm）的电磁波，本质是高频光子，由高速电子轰击靶材时产生（韧致辐射和特征辐射）。选C。

9.【A】

粒子数反转（population inversion）：在激光增益介质中，高能级（激发态）的粒子数 N₂ 大于低能级（基态）的粒子数 N₁（N₂ > N₁），这是产生受激辐射放大（激光）的必要条件。选A。

10.【B】

CT值（Hounsfield Unit，HU）：HU = (μ\_组织 - μ\_水)/μ\_水 × 1000。水的CT值定义为 0 HU；空气约 -1000 HU；致密骨约 +1000 HU。选B。

**三、简答题答案**

1. X射线产生机制（5分）：

X射线在X射线管中通过高速电子轰击靶（通常为钨靶）产生，包含两种机制：

【连续X射线（韧致辐射，Bremsstrahlung）】：
高速电子（由管电压U加速至 v = √(2eU/m)）在靶核库仑场中减速（制动），将部分动能转变为X光子。X光子能量 E = hν 不确定（取决于每次减速程度），因此形成连续光谱。最短波长（对应最大能量）：λ\_min = hc/(eU)，即全部动能转为一个光子时。

【特征X射线（特征辐射，Characteristic X-ray）】：
当入射电子的能量足够大，能将靶原子内层（如K层）电子轰出，产生空穴。外层（L层、M层）电子跃迁填补这个空穴，多余能量以X光子形式辐射。由于能级差固定（ΔE = E\_L - E\_K等），辐射的X光子频率确定，形成叠加在连续谱上的特征峰（K\_α、K\_β线等）。

【医学应用】：
• 诊断成像（X射线透视、DR/CR/CT）：利用X射线穿透力和不同组织对X射线吸收系数不同（骨骼μ大，软组织μ小），产生对比影像。
• 放射治疗：高能X射线用于肿瘤放疗（损伤DNA）。

2. 放射性衰变三种方式（5分）：

【α衰变】：
本质：母核放出氦核（⁴₂He，α粒子），质量数减少4，电荷数减少2。
方程：ᴬ\_ZX → ᴬ⁻⁴\_Z₋₂Y + ⁴₂He + Q
α粒子穿透力弱（空气中射程几厘米），电离能力强。

【β衰变】：
β⁻衰变：中子→质子，发射电子和反中微子；质量数不变，电荷数+1。
  ᴬ\_ZX → ᴬ\_Z+1Y + e⁻ + ν̄\_e
β⁺衰变：质子→中子，发射正电子和中微子；电荷数-1。
  ᴬ\_ZX → ᴬ\_Z-1Y + e⁺ + ν\_e

【γ衰变】：
核从激发态跃迁到基态，发射高能γ光子；质量数和电荷数不变，只是核能量降低。
穿透力最强，可穿透人体，医学成像利用此特性。

【医学应用】：
• ¹³¹I（碘-131）：β⁻衰变 + γ辐射，T₁/₂ = 8.03天。甲状腺摄碘特性使其在甲状腺显像和甲亢/甲癌治疗中广泛应用。
• ⁹⁹ᵐTc（锝-99m）：纯γ辐射（140 keV），T₁/₂ = 6 h（短暂减小辐射剂量），是SPECT显像最常用核素。由⁹⁹Mo/⁹⁹ᵐTc发生器（奶牛）获得。
• ¹⁸F（氟-18）：β⁺衰变，T₁/₂ = 110 min。以FDG（氟代脱氧葡萄糖）形式用于PET肿瘤代谢显像。正电子湮灭产生两个511 keV γ光子（符合探测）。

计算题1 解答（10分）：

已知：n₁ = 1.0（空气），n₂ = 1.336（眼球），R = 5.6 mm = 0.0056 m

（1）单球面折射公式：n₁/u + n₂/v = (n₂-n₁)/R
  无穷远物体（u = ∞），n₁/∞ = 0：
  0 + n₂/v = (n₂-n₁)/R
  v = n₂R/(n₂-n₁) = 1.336 × 5.6 / (1.336 - 1.0) = 7.48 / 0.336 ≈ 22.3 mm  ……[3分]
  （正常眼睛的视网膜约在角膜后22 mm处，与结果相符）

（2）调节量：
  看无穷远时，像距 v∞ = 22.3 mm（眼睛放松）
  看25 cm = 250 mm处时：n₁/u + n₂/v₂₅ = (n₂-n₁)/R
  1.0/250 + 1.336/v₂₅ = 0.336/5.6 = 0.06 mm⁻¹
  1.336/v₂₅ = 0.06 - 0.004 = 0.056 mm⁻¹
  v₂₅ ≈ 1.336/0.056 ≈ 23.86 mm
  眼球通过睫状肌调节增大了等效焦距，调节量（屈光度增加）= Δ(1/f') ≈ 4 D  ……[3分]

（3）近视患者矫正：
  近视患者远点为 0.5 m = 50 cm（最远只能看清50 cm处的物体）
  需要凹透镜将无穷远物体的像成在50 cm处（虚像）：
  1/f\_lens = 1/v - 1/u = 1/(-50) - 1/∞ = -0.02 cm⁻¹ = -2 m⁻¹
  D = -2 屈光度 = -200度（近视200度）  ……[4分]

计算题4 解答（10分）：

计算题5 解答（10分）：

已知：ν = 8×10¹⁴ Hz，U₀ = 1.5 V，h = 6.63×10⁻³⁴ J·s，e = 1.6×10⁻¹⁹ C

（1）最大初动能：
  Eₖ\_max = eU₀ = 1.6×10⁻¹⁹ × 1.5 = 2.4×10⁻¹⁹ J = 1.5 eV  ……[2分]

（2）逸出功 W：
  由爱因斯坦光电效应方程：hν = W + Eₖ\_max
  W = hν - Eₖ\_max = 6.63×10⁻³⁴ × 8×10¹⁴ - 2.4×10⁻¹⁹
  W = 5.304×10⁻¹⁹ - 2.4×10⁻¹⁹ = 2.904×10⁻¹⁹ J ≈ 1.815 eV  ……[3分]

（3）截止频率 ν₀：
  hν₀ = W → ν₀ = W/h = 2.904×10⁻¹⁹ / 6.63×10⁻³⁴ = 4.38×10¹⁴ Hz  ……[2分]

（4）入射光频率加倍 ν' = 2ν = 1.6×10¹⁵ Hz，W不变：
  Eₖ\_max' = hν' - W = 6.63×10⁻³⁴ × 1.6×10¹⁵ - 2.904×10⁻¹⁹
  Eₖ\_max' = 1.061×10⁻¹⁸ - 2.904×10⁻¹⁹ = 7.704×10⁻¹⁹ J
  Eₖ\_max' = 7.704×10⁻¹⁹ / 1.6×10⁻¹⁹ ≈ 4.815 eV  ……[3分]
  （注：频率加倍，但动能远不止加倍，因为逸出功W是固定消耗。）
