## 杂环化合物

构成环系的原子除碳原子外还有其它原子是，叫做杂环。杂环化合物种类多，数目庞大，是有机化合物中数目最庞大的一类。杂环化合物的成环规律和碳环一样，最稳定和最常见的杂环也是五元或六元的，有的环只含一个杂原子，有的含有多个杂原子，最常见从杂原子有N,O,S。

五元杂环化合物及五元杂环苯并化合物吡咯、呋喃、噻吩；吲哚、苯并呋喃、苯并噻吩

六元杂环化合物及六元杂环苯并化合物吡啶、喹啉、异喹啉

噻吩

上述三个杂环化合物中，碳原子与杂原子均呈 $S { \boldsymbol { \mathsf { p } } } ^ { 2 }$ 杂化，并在一个平面上，碳原子和杂原子的p轨道中的6个π电子形成一个环状封闭的共轭体系，符合4n+2规则，具有芳香性。因此这些杂环化合物的性质与苯类似，故称芳香杂环化合物。

由于五元杂环环上π电子云密度较高，属于富电子体系，易发生亲电取代反应。
