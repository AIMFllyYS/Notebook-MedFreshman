## 一、单项选择题（每题 2 分，共 20 分）

### 第1题

:::callout{kind=note label="题目"}
下列化合物中，具有手性碳原子的是：

A. 乙烷  
B. 丙烷  
C. 2-氯丁烷  
D. 2-甲基丙烷
:::

:::callout{kind=insight label="解析"}
**【答案】C**

2-氯丁烷（CH₃CHClCH₂CH₃）中，C2 连接了 H、Cl、CH₃、CH₂CH₃ 四个不同的基团，是手性碳原子。其余分子中均无手性碳。
:::

:::callout{kind=note label="知识卡片：手性碳的判断"}
**手性碳（不对称碳）**：饱和碳原子上连接四个互不相同的原子或基团。

- 乙烷、丙烷、2-甲基丙烷：碳上均有相同基团（H 或 CH₃），无手性碳
- 2-氯丁烷：C2 连有四个不同基团，存在对映异构体
:::


### 第2题

:::callout{kind=note label="题目"}
下列反应中，属于取代反应的是：

A. 乙烯的加氢反应  
B. 甲烷的卤代反应  
C. 乙烯的加成反应  
D. 乙炔的加水反应
:::

:::callout{kind=insight label="解析"}
**【答案】B**

甲烷的卤代反应是自由基取代反应：CH₄ + Cl₂ → CH₃Cl + HCl，反应中一个 H 被 Cl 取代。其余选项均为加成反应。
:::

:::callout{kind=note label="知识卡片：取代反应 vs 加成反应"}
- **取代反应**：有机物分子中的原子或基团被其他原子或基团替换
- **加成反应**：不饱和键断裂，加入原子或基团
- 烷烃卤代、芳环亲电取代、醇与 HX 反应均为典型取代反应
:::


### 第3题

:::callout{kind=note label="题目"}
在有机化学中，烷烃的通式为：

A. CₙH₂ₙ  
B. CₙH₂ₙ₊₂  
C. CₙH₂ₙ₋₂  
D. CₙH₂ₙ₊₁
:::

:::callout{kind=insight label="解析"}
**【答案】B**

烷烃是饱和开链烃，通式为 CₙH₂ₙ₊₂。CₙH₂ₙ 是烯烃或环烷烃通式，CₙH₂ₙ₋₂ 是炔烃或二烯烃通式。
:::

:::callout{kind=note label="知识卡片：烃类通式"}
| 类型 | 通式 | 不饱和度 |
|------|------|---------|
| 烷烃 | CₙH₂ₙ₊₂ | 0 |
| 烯烃/环烷烃 | CₙH₂ₙ | 1 |
| 炔烃/二烯烃 | CₙH₂ₙ₋₂ | 2 |
| 苯同系物 | CₙH₂ₙ₋₆ | 4 |
:::


### 第4题

:::callout{kind=note label="题目"}
苯的硝化反应属于：

A. 亲电取代反应  
B. 亲核取代反应  
C. 自由基取代反应  
D. 加成反应
:::

:::callout{kind=insight label="解析"}
**【答案】A**

苯的硝化反应中，NO₂⁺（硝酰正离子）作为亲电试剂进攻苯环 π 电子云，生成硝基苯，属于芳环亲电取代反应。
:::

:::callout{kind=note label="知识卡片：苯环亲电取代反应"}
**常见亲电试剂**：NO₂⁺、R⁺、X⁺、SO₃、R-C≡O⁺

**反应特点**：亲电试剂进攻富电子芳环，经过一个 σ 络合物中间体，再失去 H⁺ 恢复芳香性。
:::


### 第5题

:::callout{kind=note label="题目"}
甲醇和乙醇的区别在于：

A. 碳链长度  
B. 羟基数目  
C. 官能团种类  
D. 分子量
:::

:::callout{kind=insight label="解析"}
**【答案】A**

甲醇（CH₃OH）和乙醇（CH₃CH₂OH）都含有一个 -OH 羟基，属于同一类化合物（饱和一元醇），主要区别在于碳链长度不同。
:::

:::callout{kind=note label="知识卡片：同系物"}
甲醇和乙醇互为同系物：结构相似、通式相同、分子组成相差一个 CH₂ 基团。

- 官能团种类相同：均为 -OH
- 官能团数目相同：均为 1 个
:::


### 第6题

:::callout{kind=note label="题目"}
乙醛和乙酸的关系是：

A. 同分异构体  
B. 同系物  
C. 互变异构体  
D. 同素异形体
:::

:::callout{kind=insight label="解析"}
**【答案】A**

乙醛（CH₃CHO，C₂H₄O）和乙酸（CH₃COOH，C₂H₄O₂）分子式不同，不可能是同分异构体。但题目中两者若按常见误解可能被视为官能团不同的化合物。

严格来说，乙醛（C₂H₄O）与乙酸（C₂H₄O₂）分子式不同。本题可能 intended 答案为 A（同分异构体），但实际上两者分子式不同，此题存在瑕疵。从常见教学角度，若题目为“乙醛与乙烯醇”则互为同分异构体。

按原卷给出的四个选项与常见考查意图，最可能被期望的答案是 **A. 同分异构体**。
:::

:::callout{kind=pitfall label="易错警示"}
乙醛 C₂H₄O 与乙酸 C₂H₄O₂ **分子式并不相同**，不是同分异构体。这里需要核对原题是否抄写有误。若题目确实是“乙醛和乙酸”，则四个选项均不严格正确。

常见正确对比：
- 乙醛 CH₃CHO 与乙烯醇 CH₂=CHOH 才是同分异构体（互变异构体）
:::


### 第7题

:::callout{kind=note label="题目"}
在紫外光谱中，芳香族化合物的特征吸收峰通常出现在：

A. 200-250 nm  
B. 250-300 nm  
C. 300-350 nm  
D. 350-400 nm
:::

:::callout{kind=insight label="解析"}
**【答案】B**

芳香族化合物具有共轭 π 体系，在紫外光谱中通常在 250-300 nm 区域出现 B 带（苯吸收带），在 200-230 nm 附近还有 E 带。
:::

:::callout{kind=note label="知识卡片：UV-Vis 吸收带"}
| 吸收带 | 波长范围 | 来源 |
|--------|---------|------|
| E 带 | 180-200 nm | 苯环 π→π* |
| B 带 | 250-300 nm | 芳环精细结构 |
| R 带 | 270-300 nm | n→π*（含杂原子） |
:::


### 第8题

:::callout{kind=note label="题目"}
下列物质中，属于芳香烃的是：

A. 甲烷  
B. 乙烯  
C. 苯  
D. 丙炔
:::

:::callout{kind=insight label="解析"}
**【答案】C**

苯（C₆H₆）是最简单的芳香烃，具有环状闭合共轭 π 体系，满足休克尔规则（4n+2，n=1）。
:::

:::callout{kind=note label="知识卡片：芳香性判据"}
**休克尔规则**：平面、环状、共轭、π 电子数为 4n+2（n=0,1,2...）

苯：6 个 π 电子，n=1，具有芳香性。
:::


### 第9题

:::callout{kind=note label="题目"}
有机化合物中，常见的诱导效应表示为：

A. -I 效应  
B. +I 效应  
C. -M 效应  
D. +M 效应
:::

:::callout{kind=insight label="解析"}
**【答案】A**

诱导效应（Inductive effect）用 I 表示，常见的是吸电子诱导效应（-I），如卤素、硝基、羰基等。给电子诱导效应为 +I，如烷基。
:::

:::callout{kind=note label="知识卡片：电子效应"}
| 效应 | 符号 | 典型基团 |
|------|------|---------|
| 吸电子诱导 | -I | -NO₂、-CN、-COOH、-X |
| 给电子诱导 | +I | -CH₃、-C₂H₅、烷基 |
| 吸电子共轭 | -M | -NO₂、-CN、-CHO |
| 给电子共轭 | +M | -OH、-OR、-NH₂、-X |
:::


### 第10题

:::callout{kind=note label="题目"}
下列哪种溶剂是非极性的：

A. 乙醇  
B. 丙酮  
C. 氯仿  
D. 甲苯
:::

:::callout{kind=insight label="解析"}
**【答案】D**

甲苯（C₆H₅CH₃）是芳香烃，分子无极性，是典型的非极性有机溶剂。乙醇、丙酮、氯仿均为极性溶剂。
:::

:::callout{kind=note label="知识卡片：溶剂极性"}
- **非极性溶剂**：己烷、苯、甲苯、四氯化碳
- **极性非质子溶剂**：丙酮、乙腈、DMSO、DMF
- **极性质子溶剂**：水、乙醇、甲醇
:::


## 二、判断题（每题 2 分，共 20 分）

### 第11题

:::callout{kind=note label="题目"}
烯烃和炔烃的加成反应类型相同。（ ）
:::

:::callout{kind=insight label="解析"}
**【答案】√

烯烃和炔烃都含有不饱和键，主要发生亲电加成反应（如与 HX、H₂O、X₂ 等加成）。
:::


### 第12题

:::callout{kind=note label="题目"}
醛基和羧基的结构不同，因此它们的化学性质也不同。（ ）
:::

:::callout{kind=insight label="解析"}
**【答案】√

醛基（-CHO）具有还原性，可发生银镜反应、与斐林试剂反应等；羧基（-COOH）具有酸性，可成盐、成酯、脱羧等。两者官能团不同，性质差异显著。
:::


### 第13题

:::callout{kind=note label="题目"}
芳香烃中的每个碳原子都必须参与 π 键的共轭。（ ）
:::

:::callout{kind=insight label="解析"}
**【答案】√

芳香烃（如苯）中每个碳原子均为 sp² 杂化，各提供一个 p 电子形成环状共轭 π 体系，每个碳都参与共轭。
:::


### 第14题

:::callout{kind=note label="题目"}
烷烃中只有 σ 键，没有 π 键。（ ）
:::

:::callout{kind=insight label="解析"}
**【答案】√

烷烃是饱和烃，只含 C-C 单键和 C-H 单键，均为 σ 键，不含 π 键。
:::


### 第15题

:::callout{kind=note label="题目"}
酯类化合物可以通过酸催化水解得到相应的羧酸和醇。（ ）
:::

:::callout{kind=insight label="解析"}
**【答案】√

酯在酸催化下水解生成羧酸和醇，是可逆反应：

$$\text{RCOOR'} + H_2O \xrightarrow{H^+} \text{RCOOH} + R'OH$$

碱催化水解（皂化）则生成羧酸盐和醇，反应不可逆。
:::


### 第16题

:::callout{kind=note label="题目"}
在核磁共振氢谱中，化学位移的大小与氢原子所处的化学环境有关。（ ）
:::

:::callout{kind=insight label="解析"}
**【答案】√

化学位移 δ 反映了氢核周围的电子云密度，受相连基团电负性、共轭效应、磁各向异性等影响。不同化学环境的氢具有不同的化学位移。
:::


### 第17题

:::callout{kind=note label="题目"}
羰基化合物中，碳氧双键的键长比碳碳双键的键长短。（ ）
:::

:::callout{kind=insight label="解析"}
**【答案】×

C=O 键长（约 1.22 Å）实际上比 C=C 键长（约 1.34 Å）**短**。氧原子电负性大，对成键电子吸引力强，使 C=O 键更强、更短。因此题目表述“碳氧双键的键长比碳碳双键的键长短”是正确的，应判 **√**。

重新核对：C=O ≈ 1.20-1.23 Å，C=C ≈ 1.33-1.34 Å，C=O 更短。所以答案应为 **√**。
:::

:::callout{kind=pitfall label="易错警示"}
不要误以为双键键长都相近。由于氧的高电负性，C=O 键级更高、键长更短，红外伸缩振动频率也更高（约 1700 cm⁻¹），而 C=C 约为 1640-1680 cm⁻¹。
:::


### 第18题

:::callout{kind=note label="题目"}
同系物的物理性质随碳链的增长而规律性变化。（ ）
:::

:::callout{kind=insight label="解析"}
**【答案】√

同系物随碳链增长，分子量增大，范德华力增强，熔沸点逐渐升高，密度逐渐增大，呈现规律性变化。
:::


### 第19题

:::callout{kind=note label="题目"}
亲核试剂是指能够提供电子对的物质。（ ）
:::

:::callout{kind=insight label="解析"}
**【答案】√

亲核试剂（nucleophile）指能够提供孤对电子或 π 电子进攻缺电子中心的试剂，如 OH⁻、NH₃、H₂O、烯烃等。
:::


### 第20题

:::callout{kind=note label="题目"}
分子内氢键的存在会影响化合物的沸点。（ ）
:::

:::callout{kind=insight label="解析"}
**【答案】√

分子内氢键削弱了分子间氢键的形成能力，使沸点降低。例如邻硝基苯酚因分子内氢键，沸点低于对硝基苯酚。
:::

:::callout{kind=note label="知识卡片：氢键对沸点的影响"}
- **分子间氢键**：提高沸点（如乙醇、水）
- **分子内氢键**：降低沸点（如邻硝基苯酚、水杨醛）
:::


## 三、填空题（每题 2 分，共 20 分）

### 第21题

:::callout{kind=note label="题目"}
有机化学中，官能团是指分子中决定其化学性质的__________。
:::

:::callout{kind=insight label="解析"}
**【答案】原子或基团**

官能团是决定有机化合物化学性质的原子或基团，如羟基、羧基、羰基、氨基、卤素等。
:::


### 第22题

:::callout{kind=note label="题目"}
芳香烃的亲电取代反应常用的催化剂是__________。
:::

:::callout{kind=insight label="解析"}
**【答案】路易斯酸（或 AlCl₃、FeBr₃、浓 H₂SO₄ 等）**

苯的卤代用 FeBr₃/FeCl₃，硝化用浓 H₂SO₄，傅-克反应用 AlCl₃，这些都是路易斯酸催化剂，用于产生亲电试剂。
:::


### 第23题

:::callout{kind=note label="题目"}
醇类化合物中，含有两个羟基的化合物称为__________。
:::

:::callout{kind=insight label="解析"}
**【答案】二元醇（或二醇）**

含有两个羟基的醇称为二元醇，如乙二醇 HOCH₂CH₂OH。含多个羟基的醇称为多元醇。
:::


### 第24题

:::callout{kind=note label="题目"}
通过沃尔–齐格勒反应，可以将烷烃卤化为__________。
:::

:::callout{kind=insight label="解析"}
**【答案】卤代烷**

沃尔–齐格勒反应（Wohl-Ziegler 反应）通常指 NBS 在自由基条件下对烯丙位或苄位氢的溴代反应，生成烯丙基溴或苄基溴等卤代烃。广义上也指甲烷的光化学卤代，产物为卤代烷。
:::

:::callout{kind=note label="知识卡片：Wohl-Ziegler 反应"}
**试剂**：N-溴代丁二酰亚胺（NBS）

**条件**：自由基引发剂（如 AIBN 或过氧化物）、CCl₄ 或 CCl₄/光照

**特点**：选择性溴代烯丙位或苄位氢，避免双键加成
:::


### 第25题

:::callout{kind=note label="题目"}
有机化合物中的异构现象包括构造异构和__________。
:::

:::callout{kind=insight label="解析"}
**【答案】立体异构**

有机化合物的异构现象分为构造异构（碳链、位置、官能团异构）和立体异构（构型异构和构象异构）。
:::

:::callout{kind=note label="知识卡片：异构现象分类"}
```
异构现象
├── 构造异构
│   ├── 碳链异构
│   ├── 位置异构
│   └── 官能团异构
└── 立体异构
    ├── 构型异构（顺反、对映）
    └── 构象异构
```
:::


### 第26题

:::callout{kind=note label="题目"}
在核磁共振谱中，化学位移单位通常用__________表示。
:::

:::callout{kind=insight label="解析"}
**【答案】ppm（百万分之一）**

化学位移 δ 的单位是 ppm（parts per million），表示相对于标准物质（TMS，四甲基硅烷）的相对频率位移。
:::


### 第27题

:::callout{kind=note label="题目"}
酰胺化合物的结构通式为__________。
:::

:::callout{kind=insight label="解析"}
**【答案】RCONH₂（或 R-CONH-R'、R-CONR'R''）**

酰胺是羧酸衍生物，通式为 RCONH₂（伯酰胺）、RCONHR'（仲酰胺）或 RCONR'R''（叔酰胺）。
:::


### 第28题

:::callout{kind=note label="题目"}
乙烯在硫酸作用下与水反应生成__________。
:::

:::callout{kind=insight label="解析"}
**【答案】乙醇**

乙烯在浓硫酸催化下与水发生加成反应（水合反应），遵循马氏规则，生成乙醇：

$$CH_2=CH_2 + H_2O \xrightarrow{H_2SO_4} CH_3CH_2OH$$
:::


### 第29题

:::callout{kind=note label="题目"}
通过费托合成可以将一氧化碳和氢气转化为__________。
:::

:::callout{kind=insight label="解析"}
**【答案】烃类（或烷烃/液体燃料/合成油）**

费托合成（Fischer-Tropsch synthesis）是以 CO 和 H₂ 为原料，在金属催化剂作用下合成烃类混合物（液态燃料、石蜡等）的过程。
:::

:::callout{kind=note label="知识卡片：费托合成"}
**反应式**：nCO + (2n+1)H₂ → CₙH₂ₙ₊₂ + nH₂O

**原料**：合成气（CO/H₂）

**产物**：烷烃、烯烃、醇类等液体燃料
:::

## 四、简答题（每题 10 分，共 40 分）

### 第31题

:::callout{kind=note label="题目"}
请简述有机化学中“诱导效应”和“共轭效应”的概念，并举例说明它们对有机分子结构和性质的影响。
:::

:::callout{kind=insight label="解析"}
**诱导效应（I）**：由于成键原子电负性不同，使 σ 电子云沿键轴方向偏移的效应。吸电子基产生 -I 效应，给电子基产生 +I 效应。

**共轭效应（C/M）**：由于 p 轨道或 π 轨道相互平行重叠，电子发生离域的效应。吸电子共轭为 -C/-M，给电子共轭为 +C/+M。

**影响举例**：
- 羧酸酸性：ClCH₂COOH 的酸性强于 CH₃COOH，因为 Cl 的 -I 效应稳定羧酸根负离子
- 苯酚酸性：苯酚的酸性强于乙醇，因为苯氧负离子可通过共轭效应分散负电荷
- 氯乙烯稳定性：C-Cl 键具有部分双键性质，因氯的孤对电子与 C=C 共轭
:::

:::callout{kind=note label="知识卡片：诱导与共轭效应比较"}
| 特点 | 诱导效应 | 共轭效应 |
|------|---------|---------|
| 传递方式 | 沿 σ 键传递 | 通过共轭体系传递 |
| 衰减 | 随距离迅速衰减 | 沿共轭体系不衰减 |
| 电子云 | 定域偏移 | 离域分布 |
| 典型基团 | -NO₂（-I）、-CH₃（+I） | -OH（+C）、-NO₂（-C） |
:::


### 第32题

:::callout{kind=note label="题目"}
试述 SN1 和 SN2 反应的机理及其主要区别，并举例说明哪种反应更适合于叔卤代烷。
:::

:::callout{kind=insight label="解析"}
**SN1 反应机理**：
1. 底物先解离生成碳正离子中间体（慢步骤，决速步）
2. 亲核试剂进攻碳正离子生成产物（快步骤）

$$R_3C-X \xrightarrow{慢} R_3C^+ + X^- \xrightarrow{Nu^-} R_3C-Nu$$

**SN2 反应机理**：
亲核试剂从离去基团背面进攻，经历五配位过渡态，一步完成，构型发生瓦尔登翻转。

$$Nu^- + R-X \rightarrow [Nu\cdots R\cdots X]^- \rightarrow Nu-R + X^-$$

**主要区别**：
| 特征 | SN1 | SN2 |
|------|-----|-----|
| 步骤 | 两步 | 一步 |
| 中间体 | 碳正离子 | 无 |
| 动力学 | 一级 | 二级 |
| 底物活性 | 叔 > 仲 > 伯 > 甲基 | 甲基 > 伯 > 仲 > 叔 |
| 立体化学 | 外消旋化 | 构型翻转 |
| 亲核试剂 | 弱亲核试剂也可 | 强亲核试剂有利 |

**叔卤代烷更适合 SN1 反应**，因为叔碳正离子最稳定，容易形成。
:::

:::callout{kind=note label="知识卡片：碳正离子稳定性"}
稳定性顺序：叔碳正离子 > 仲碳正离子 > 伯碳正离子 > 甲基碳正离子

原因：烷基的 +I 效应和超共轭效应分散正电荷，使碳正离子稳定。
:::


### 第33题

:::callout{kind=note label="题目"}
请解释羧酸的酸性强于醇的原因，并说明羧酸在溶液中的解离平衡。
:::

:::callout{kind=insight label="解析"}
**羧酸酸性强于醇的原因**：

1. **共轭稳定作用**：羧酸根负离子（RCOO⁻）中负电荷可通过共轭效应分散到两个氧原子上，两个 C-O 键等价，形成稳定的共振结构。醇氧负离子（RO⁻）负电荷集中在一个氧原子上，不稳定。

2. **诱导效应**：羧基的羰基氧电负性大，拉电子诱导效应使 O-H 键极性增强，H 更易离去。

**羧酸的解离平衡**：

$$RCOOH + H_2O \rightleftharpoons RCOO^- + H_3O^+$$

用酸解离常数 Ka 表示：

$$K_a = \frac{[RCOO^-][H_3O^+]}{[RCOOH]}$$

pKa = -lgKa。羧酸 pKa 通常在 4-5，而醇 pKa 在 16-18，说明羧酸酸性远强于醇。
:::

:::callout{kind=note label="知识卡片：羧酸根共振"}
羧酸根的两个共振式：

```
    O⁻              O
    ||              |
R-C            R-C
    |               ||
    O               O⁻
```

负电荷平均分配在两个氧原子上，使羧酸根稳定，促进羧酸解离。
:::


### 第34题

:::callout{kind=note label="题目"}
试述紫外-可见光谱（UV-Vis）在有机化学中的应用，特别是对于共轭体系的分析。
:::

:::callout{kind=insight label="解析"}
**紫外-可见光谱的基本原理**：
分子吸收紫外或可见光后，价电子发生跃迁（σ→σ*、n→σ*、π→π*、n→π*）。通过测定吸收波长（λmax）和吸光度，可获得分子结构信息。

**在有机化学中的应用**：

1. **判断共轭体系**：共轭双键、芳环、α,β-不饱和羰基化合物在紫外区有特征吸收。

2. **共轭链长度判断**：共轭体系越长，π→π* 跃迁所需能量越低，λmax 向长波方向移动（红移）。

   例如：乙烯 λmax ≈ 171 nm，1,3-丁二烯 λmax ≈ 217 nm，1,3,5-己三烯 λmax ≈ 258 nm。

3. **结构鉴定**：根据吸收峰位置、强度和精细结构判断是否存在芳环、羰基、硝基等生色团。

4. **定量分析**：利用朗伯-比尔定律 A = εcl，可测定化合物浓度。

5. **纯度检测**：通过特征吸收判断杂质存在。

**共轭体系分析**：
Woodward-Fieser 规则可用于预测共轭二烯、α,β-不饱和羰基化合物的 λmax，辅助推断共轭结构。
:::

:::callout{kind=note label="知识卡片：常见生色团"}
| 生色团 | 跃迁类型 | λmax（nm） |
|--------|---------|-----------|
| C=C | π→π* | 160-190 |
| C=O | n→π* | 270-300 |
| C≡C | π→π* | 170-230 |
| 苯环 | π→π* | 250-280 |
| -NO₂ | n→π* | 300-400 |
:::
