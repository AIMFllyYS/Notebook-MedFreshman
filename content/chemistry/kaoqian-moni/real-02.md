## 一、选择题（每题2分，共24分）

### 第1题

:::callout{kind=note label="题目"}
最容易发生SN2反应的是

A. CH₃Br    B. (CH₃)₂CHBr    C. (CH₃)₃CBr    D. PhCl
:::

:::callout{kind=insight label="解析"}
**【答案】A**

SN2受位阻控制，甲基卤代烷最快；芳基氯通常不发生SN2。

:::

:::callout{kind=note label="SN2反应活性"}
**活性顺序**：CH₃X > 1° > 2° > 3° > 芳基卤

**原因**：
- 背面进攻需要空间
- 位阻越大，SN2越不利
- 芳基卤：C-X键有双键性质，难断裂

**对比SN1**：3° > 2° > 1° > CH₃X（与SN2相反）
:::


### 第2题

:::callout{kind=note label="题目"}
下列离去基团能力由强到弱正确的是

A. F⁻ > Cl⁻ > Br⁻ > I⁻    B. I⁻ > Br⁻ > Cl⁻ >> F⁻    C. Cl⁻ > Br⁻ > I⁻ > F⁻    D. Br⁻ > I⁻ > Cl⁻ > F⁻
:::

:::callout{kind=insight label="解析"}
**【答案】B**

卤离子稳定性和C-X键强度决定离去能力：I⁻最好，F⁻很差。

:::

:::callout{kind=insight label="离去基团能力"}
**判断依据**：
1. C-X键强度（越弱越好）
2. 卤离子稳定性（越大越好）

**顺序**：I⁻ > Br⁻ > Cl⁻ >> F⁻

**原因**：
- 键能：C-F(485) > C-Cl(339) > C-Br(285) > C-I(213) kJ/mol
- 离子半径：I⁻ > Br⁻ > Cl⁻ > F⁻
- 电荷分散：大离子更稳定

**注意**：F⁻是极差的离去基团！
:::


### 第3题

:::callout{kind=note label="题目"}
下列条件最有利于叔丁基氯发生SN1水解的是

A. 无水丙酮/KI    B. 水/乙醇混合溶剂    C. 强碱叔丁醇钾/DMSO    D. 低极性CCl₄
:::

:::callout{kind=insight label="解析"}
**【答案】B**

SN1需稳定碳正离子和离去离子，极性质子溶剂有利。

:::

:::callout{kind=note label="SN1反应条件"}
**有利条件**：
- 极性质子溶剂（H₂O、EtOH）
- 3°底物（稳定碳正离子）
- 弱亲核试剂
- 加热

**不利条件**：
- 强碱（促进E2消除）
- 非极性溶剂（不稳定离子）
- 强亲核试剂（促进SN2）

**叔丁基氯SN1**：
- (CH₃)₃CCl → (CH₃)₃C⁺ + Cl⁻
- 极性质子溶剂稳定离子
:::


### 第4题

:::callout{kind=note label="题目"}
2-溴丁烷在KOH/乙醇、加热条件下主产物是

A. 1-丁醇    B. 2-丁醇    C. 反-2-丁烯    D. 1-丁烯
:::

:::callout{kind=insight label="解析"}
**【答案】C**

醇溶液强碱加热促进E2，按Zaitsev和反式消除，以反-2-丁烯为主。

:::

:::callout{kind=insight label="SN2/E2竞争"}
**2-溴丁烷 + KOH/EtOH**：

**分析**：
- 底物：2°卤代烷
- 试剂：强碱/醇溶剂/加热
- 条件：有利于E2

**产物**：
- 主产物：反-2-丁烯（Zaitsev产物，更稳定）
- 副产物：顺-2-丁烯、1-丁烯

**注意**：E2要求反式共面消除
:::


### 第5题

:::callout{kind=note label="题目"}
Lucas试剂能最快产生浑浊的是

A. 1-丁醇    B. 2-丁醇    C. 叔丁醇    D. 乙醇
:::

:::callout{kind=insight label="解析"}
**【答案】C**

叔醇易形成稳定碳正离子，SN1最快。

:::

:::callout{kind=note label="Lucas试剂反应"}
**试剂**：ZnCl₂ + 浓HCl

**反应机理**：SN1（醇先质子化，然后离去H₂O）

**反应速率**：
- 叔醇：立即浑浊（3°碳正离子最稳定）
- 仲醇：5-10分钟浑浊
- 伯醇：室温不反应，需加热

**应用**：鉴别1°、2°、3°醇
:::


### 第6题

:::callout{kind=note label="题目"}
下列化合物酸性最强的是

A. 乙醇    B. 苯酚    C. 对硝基苯酚    D. 对甲苯酚
:::

:::callout{kind=insight label="解析"}
**【答案】C**

硝基强吸电子，稳定酚氧负离子，酸性增强。

:::

:::callout{kind=insight label="酚的酸性影响因素"}
**pKa值**：
- 对硝基苯酚：7.15
- 苯酚：10.0
- 对甲苯酚：10.3
- 乙醇：16

**吸电子基团增强酸性**：
- -NO₂（强）
- -Cl（中等）
- -Br（中等）

**给电子基团减弱酸性**：
- -CH₃（弱）
- -OCH₃（弱，但邻位有氢键例外）
:::


### 第7题

:::callout{kind=note label="题目"}
能与I₂/NaOH发生碘仿反应的是

A. 3-戊酮    B. 苯甲醛    C. 2-戊酮    D. 苯甲酸
:::

:::callout{kind=insight label="解析"}
**【答案】C**

碘仿反应要求CH₃CO-或能被氧化为该结构的片段。

:::

:::callout{kind=note label="碘仿反应"}
**反应条件**：I₂/NaOH（或NaOI）

**底物要求**：
- CH₃CO-R（甲基酮）
- CH₃CH(OH)-R（可被氧化为甲基酮的醇）
- 乙醇、乙醛等

**2-戊酮**：CH₃COCH₂CH₂CH₃（含CH₃CO-）

**3-戊酮**：CH₃CH₂COCH₂CH₃（不含CH₃CO-）

**产物**：CHI₃（黄色沉淀）+ 羧酸盐
:::


### 第8题

:::callout{kind=note label="题目"}
可用于鉴别多数醛和脂肪族甲基酮的白色晶体反应是

A. NaHSO₃加成    B. FeCl₃显色    C. NaHCO₃放气    D. 溴水褪色
:::

:::callout{kind=insight label="解析"}
**【答案】A**

醛、脂肪族甲基酮和小环酮可与NaHSO₃形成加成晶体。

:::

:::callout{kind=note label="NaHSO₃加成反应"}
**反应**：RCHO + NaHSO₃ → RCH(OH)SO₃Na（白色晶体）

**适用范围**：
- 醛（大部分）
- 脂肪族甲基酮
- 小环酮（C<8）

**不适用**：
- 芳香酮（位阻大）
- 大环酮

**应用**：
- 鉴别醛和酮
- 分离提纯（加成物可水解恢复）
:::


### 第9题

:::callout{kind=note label="题目"}
苯甲醛与CH₃MgBr后酸化的产物是

A. 苯乙酮    B. 1-苯乙醇    C. 苯甲醇    D. 乙苯
:::

:::callout{kind=insight label="解析"}
**【答案】B**

格氏试剂对醛羰基亲核加成，苯甲醛给仲醇1-苯乙醇。

:::

:::callout{kind=note label="格氏试剂与醛酮反应"}
**反应**：RMgX + R'CHO → R-CH(R')-OH（仲醇）

**产物类型**：
- 甲醛：伯醇
- 其他醛：仲醇
- 酮：叔醇

**苯甲醛 + CH₃MgBr**：
- 产物：C₆H₅CH(OH)CH₃（1-苯乙醇）
- 类型：仲醇
:::


### 第10题

:::callout{kind=note label="题目"}
羧酸衍生物水解活性由强到弱通常为

A. 酰胺 > 酯 > 酸酐 > 酰氯    B. 酰氯 > 酸酐 > 酯 > 酰胺    C. 酯 > 酰氯 > 酸酐 > 酰胺    D. 酸酐 > 酰胺 > 酯 > 酰氯
:::

:::callout{kind=insight label="解析"}
**【答案】B**

羰基正电性和离去基团能力共同决定，酰氯最活泼，酰胺最不活泼。

:::

:::callout{kind=insight label="羧酸衍生物活性"}
**活性顺序**：酰氯 > 酸酐 > 酯 > 酰胺

**原因**：
1. **羰基碳正电性**：Cl吸电子 > O > OR > NR₂
2. **离去基团能力**：Cl⁻ > RCOO⁻ > RO⁻ > NH₂⁻

**反应条件**：
- 酰氯：室温，不需催化剂
- 酸酐：室温或微热
- 酯：需酸或碱催化，加热
- 酰胺：需强酸/强碱，加热回流
:::


### 第11题

:::callout{kind=note label="题目"}
乙酸与SOCl₂反应的主要产物是

A. 乙酸乙酯    B. 乙酰氯    C. 乙酸酐    D. 乙酰胺
:::

:::callout{kind=insight label="解析"}
**【答案】B**

SOCl₂将羧酸转化为酰氯。

:::

:::callout{kind=note label="羧酸转化为酰氯"}
**试剂**：SOCl₂（氯化亚砜）

**反应**：RCOOH + SOCl₂ → RCOCl + SO₂↑ + HCl↑

**优点**：
- 产物纯净（气体副产物）
- 不需分离
- 反应条件温和

**其他方法**：
- PCl₃、PCl₅
- (COCl)₂（草酰氯）
:::


### 第12题

:::callout{kind=note label="题目"}
酯与过量格氏试剂反应后酸化通常得到

A. 醛    B. 酮    C. 叔醇    D. 羧酸
:::

:::callout{kind=insight label="解析"}
**【答案】C**

酯先转为酮，酮继续与格氏试剂加成，最终得叔醇。

:::

:::callout{kind=insight label="酯与格氏试剂反应"}
**反应过程**：
1. RMgX + R'COOR'' → R'C(=O)R（酮）+ R''OMgX
2. RMgX + R'C(=O)R → R'C(OH)R₂（叔醇）

**特点**：
- 需过量格氏试剂（2当量以上）
- 第一步生成酮，但酮更活泼，立即继续反应
- 最终产物：叔醇

**对比**：
- 酰氯 + 1mol RMgX → 酮（可控制）
- 酰胺 + RMgX → 酮（活性低，可控制）
:::

## 二、填空题（每空1分，共8分）

### 第1题

:::callout{kind=note label="题目"}
SN2反应的立体化学特征是 __________ 进攻和构型 __________。
:::

:::callout{kind=insight label="解析"}
**答案**：背面；翻转

**解析**：SN2为背面进攻，导致构型翻转（Walden翻转）。

:::

:::callout{kind=note label="SN2立体化学"}
**背面进攻**：亲核试剂从离去基团背面进攻

**Walden翻转**：
- 构型完全翻转（如R→S）
- 类似雨伞被风吹翻

**对比SN1**：
- 碳正离子平面结构
- 两面进攻，外消旋化
:::


### 第2题

:::callout{kind=note label="题目"}
SN1反应中若底物为手性中心，产物常出现 __________ 化。
:::

:::callout{kind=insight label="解析"}
**答案**：外消旋

**解析**：SN1经历碳正离子中间体（平面结构），亲核试剂从两面进攻概率相等，得外消旋化产物。

:::

:::callout{kind=note label="SN1的立体化学"}
**外消旋化原因**：
- 碳正离子sp²平面结构
- 亲核试剂从两面进攻
- 产物为R和S的等量混合物

**注意**：
- 并非完全外消旋（离子对效应）
- 通常部分构型保持+部分翻转
:::


### 第3题

:::callout{kind=note label="题目"}
浓KOH醇溶液通常促进卤代烃发生 __________ 反应；NaOH水溶液通常促进 __________ 反应。
:::

:::callout{kind=insight label="解析"}
**答案**：消除；取代

**解析**：浓KOH/醇/加热 → E2消除；NaOH/H₂O → SN2取代。

:::

:::callout{kind=insight label="消除 vs 取代"}
| 条件 | 主要反应 | 原因 |
|------|----------|------|
| 浓KOH/醇/加热 | E2消除 | 强碱/高温/醇溶剂 |
| NaOH/H₂O | SN2取代 | 亲核试剂/水溶剂 |
| 弱碱/低温 | SN2 | 亲核取代占优 |
| 强碱/3°底物 | E2 | 位阻大，不利SN2 |

**竞争关系**：
- 1°底物：SN2为主
- 2°底物：SN2和E2竞争
- 3°底物：E2为主
:::


### 第4题

:::callout{kind=note label="题目"}
醇与PBr₃反应可转化为相应的 __________。
:::

:::callout{kind=insight label="解析"}
**答案**：溴代烃

**解析**：PBr₃将醇转化为溴代烃，机理类似SN2（伯醇）或SN1（叔醇）。

:::

:::callout{kind=note label="醇转化为卤代烃"}
**试剂**：
- PBr₃：R-OH → R-Br
- PCl₃：R-OH → R-Cl
- SOCl₂：R-OH → R-Cl
- HX：R-OH → R-X（酸催化）

**PBr₃机理**：
1. 醇与PBr₃形成亚磷酸酯
2. Br⁻背面进攻（SN2，伯醇）
3. 构型翻转

**优点**：不重排（对比HX）
:::


### 第5题

:::callout{kind=note label="题目"}
苯酚遇FeCl₃溶液常出现 __________ 反应。
:::

:::callout{kind=insight label="解析"}
**答案**：显色

**解析**：苯酚与FeCl₃形成紫色络合物，用于鉴别酚类。

:::

:::callout{kind=note label="酚的显色反应"}
**反应**：6 C₆H₅OH + FeCl₃ → [Fe(OC₆H₅)₆]³⁻ + 6 H⁺ + 3 Cl⁻

**颜色**：紫色（苯酚）

**其他酚**：
- 邻苯二酚：绿色
- 对苯二酚：暗绿色
- 1,2,3-苯三酚：红色

**应用**：鉴别酚类化合物
:::


### 第6题

:::callout{kind=note label="题目"}
醛酮的典型反应类型是羰基上的 __________ 加成。
:::

:::callout{kind=insight label="解析"}
**答案**：亲核

**解析**：醛酮羰基碳带部分正电荷，易受亲核试剂进攻。

:::

:::callout{kind=note label="羰基亲核加成"}
**亲核试剂**：
- HCN（氰醇）
- NaHSO₃（亚硫酸氢钠加成）
- RMgX（格氏试剂）
- RLi（有机锂）
- NH₂Y（氨衍生物：肼、羟胺等）

**反应机理**：
1. 亲核试剂进攻羰基碳
2. 氧负离子质子化
3. 生成加成产物
:::


### 第7题

:::callout{kind=note label="题目"}
羟醛缩合要求至少一方醛酮具有 __________ 氢。
:::

:::callout{kind=insight label="解析"}
**答案**：α

**解析**：羟醛缩合需要α-H形成碳负离子（烯醇负离子）。

:::

:::callout{kind=note label="羟醛缩合"}
**反应**：RCH₂CHO + R'CHO → RCH₂CH(OH)CH(R')CHO

**条件**：
- 稀碱（NaOH、KOH）
- 至少一方有α-H

**产物**：β-羟基醛（或酮）

**脱水**：加热得α,β-不饱和醛（酮）

**交叉羟醛缩合**：两种不同醛酮反应
:::


### 第8题

:::callout{kind=note label="题目"}
酰氯、酸酐、酯、酰胺都可看作 __________ 的衍生物。
:::

:::callout{kind=insight label="解析"}
**答案**：羧酸

**解析**：羧酸衍生物是羧酸分子中羟基被取代后的产物。

:::

:::callout{kind=note label="羧酸衍生物"}
**结构通式**：R-CO-L

| 衍生物 | L | 特点 |
|--------|---|------|
| 酰氯 | -Cl | 最活泼 |
| 酸酐 | -OCOR | 较活泼 |
| 酯 | -OR | 较稳定 |
| 酰胺 | -NH₂ | 最稳定 |

**转化关系**：
羧酸 → 酰氯 → 酸酐 → 酯 → 酰胺（活性递减）
:::

## 三、判断题（每题1分，共8分）

### 第1题

:::callout{kind=note label="题目"}
叔卤代烷通常比伯卤代烷更有利于SN2反应。（　　）
:::

:::callout{kind=insight label="解析"}
**【答案】×**

解析：叔卤代烷位阻大，不利于背面进攻，通常SN1/E1更有利。

:::

:::callout{kind=note label="SN2位阻效应"}
**活性顺序**：CH₃X > 1° > 2° > 3°

**原因**：
- 背面进攻需要空间
- 3°碳周围三个烷基阻挡
- 过渡态拥挤

**对比SN1**：
- 3° > 2° > 1°（碳正离子稳定性）
:::


### 第2题

:::callout{kind=note label="题目"}
强碱、加热、醇溶剂通常会增强消除反应比例。（　　）
:::

:::callout{kind=insight label="解析"}
**【答案】√**

解析：E2需要碱夺β-H，醇溶剂和加热常使消除占优。

:::

:::callout{kind=insight label="消除反应条件"}
**有利条件**：
- 强碱（KOH、NaOEt、KOtBu）
- 高温
- 醇溶剂（极性非质子不利SN2）
- 3°底物

**不利条件**：
- 弱碱/水溶剂
- 低温
- 1°底物
:::


### 第3题

:::callout{kind=note label="题目"}
苄基卤代烃和烯丙基卤代烃因中间体可共振稳定，反应活性较高。（　　）
:::

:::callout{kind=insight label="解析"}
**【答案】√**

解析：SN1/SN2过渡态或中间体可由π体系稳定。

:::

:::callout{kind=note label="苄基/烯丙基卤活性"}
**苄基卤（PhCH₂X）**：
- SN1：苄基碳正离子p-π共轭稳定
- SN2：过渡态p-π共轭稳定

**烯丙基卤（CH₂=CHCH₂X）**：
- SN1：烯丙基碳正离子p-π共轭稳定
- SN2：过渡态p-π共轭稳定

**活性**：苄基/烯丙基 > 3° > 2° > 1° > CH₃
:::


### 第4题

:::callout{kind=note label="题目"}
叔醇在Lucas试剂中通常加热后才出现浑浊。（　　）
:::

:::callout{kind=insight label="解析"}
**【答案】×**

解析：叔醇通常立即浑浊；伯醇需加热。

:::

:::callout{kind=note label="Lucas试剂反应速率"}
**反应速率**：叔醇 > 仲醇 > 伯醇

**现象**：
- 叔醇：立即浑浊（3°碳正离子最稳定）
- 仲醇：5-10分钟浑浊
- 伯醇：室温不反应，加热才浑浊

**原因**：SN1机理，碳正离子稳定性决定速率
:::


### 第5题

:::callout{kind=note label="题目"}
NaBH₄通常可以把醛酮还原为醇。（　　）
:::

:::callout{kind=insight label="解析"}
**【答案】√**

解析：NaBH₄选择性还原醛酮羰基。

:::

:::callout{kind=note label="NaBH₄ vs LiAlH₄"}
| 试剂 | 还原能力 | 选择性 |
|------|----------|--------|
| NaBH₄ | 较弱 | 只还原醛酮 |
| LiAlH₄ | 强 | 还原醛酮、酯、酰胺、羧酸 |

**NaBH₄**：
- 还原醛 → 伯醇
- 还原酮 → 仲醇
- 不还原酯、酰胺、羧酸

**LiAlH₄**：
- 还原几乎所有含羰基化合物
- 需无水条件（遇水分解）
:::


### 第6题

:::callout{kind=note label="题目"}
苯甲醛能发生银镜反应，也能迅速使菲林试剂产生红色沉淀。（　　）
:::

:::callout{kind=insight label="解析"}
**【答案】×**

解析：芳香醛可发生银镜，但斐林反应远慢于脂肪醛，常用于区别。

:::

:::callout{kind=note label="醛的鉴别"}
**Tollens试剂（银氨溶液）**：
- 所有醛：银镜反应
- 酮：不反应

**Fehling试剂**：
- 脂肪醛：红色Cu₂O沉淀（快）
- 芳香醛：反应慢或不反应
- 酮：不反应

**区别**：
- 脂肪醛：Tollens和Fehling都阳性
- 芳香醛：Tollens阳性，Fehling阴性或弱阳性
- 酮：两者都阴性
:::


### 第7题

:::callout{kind=note label="题目"}
缩醛在碱性条件下较稳定，在酸性水溶液中可水解。（　　）
:::

:::callout{kind=insight label="解析"}
**【答案】√**

解析：缩醛是羰基保护基，酸水解脱保护。

:::

:::callout{kind=note label="缩醛的保护与脱保护"}
**生成**：醛/酮 + 2ROH（酸催化，除水）

**稳定性**：
- 碱性条件：稳定（可用于保护羰基）
- 酸性条件：水解恢复羰基

**应用**：
- 保护醛酮（对碱、还原剂稳定）
- 后续反应后酸性水解恢复
:::


### 第8题

:::callout{kind=note label="题目"}
酰胺中氮的孤对电子与羰基共轭，因此酰胺碱性远弱于胺。（　　）
:::

:::callout{kind=insight label="解析"}
**【答案】√**

解析：孤对电子离域降低结合质子能力。

:::

:::callout{kind=note label="酰胺的弱碱性"}
**原因**：
- p-π共轭：N孤对电子离域到羰基
- 氮上电子云密度极低
- 难以结合质子

**碱性顺序**：
季铵碱 > 脂肪胺 > 氨 > 苯胺 > 酰胺

**酰亚胺**：
- 两个羰基夹一个氮
- N-H显酸性（如邻苯二甲酰亚胺pKa~8）
:::

## 四、完成反应式（每题4分，共32分）

写出主要有机产物；若有立体化学、区域选择性或重排，请在产物旁注明。

### 反应式1

(S)-2-溴丁烷 + HO⁻/DMSO →

:::callout{kind=insight label="解析"}
**产物**：(R)-2-丁醇为主

**解析**：强亲核试剂、偶极非质子溶剂，SN2背面进攻，构型翻转。

:::

:::callout{kind=note label="SN2构型翻转"}
**条件**：HO⁻/DMSO（强亲核/极性非质子）

**机理**：SN2

**立体化学**：
- 背面进攻
- 构型翻转（S→R）

**注意**：
- 若部分SN1竞争，可能少量外消旋化
- 纯SN2时完全翻转
:::


### 反应式2

叔丁基氯 + H₂O/EtOH →

:::callout{kind=insight label="解析"}
**产物**：叔丁醇（伴少量异丁烯）

**解析**：三级卤代烷在极性质子溶剂中SN1水解为主，E1副反应可生成烯烃。

:::

:::callout{kind=note label="SN1/E1竞争"}
**条件**：H₂O/EtOH（极性质子溶剂）

**主产物**：叔丁醇（SN1）

**副产物**：异丁烯（E1）

**比例**：
- 低温：SN1为主
- 高温：E1增加
:::


### 反应式3

2-溴丁烷 + KOH/EtOH, Δ →

:::callout{kind=insight label="解析"}
**产物**：反-2-丁烯为主，少量顺-2-丁烯和1-丁烯

**解析**：E2，Zaitsev产物为2-丁烯；反式烯烃更稳定。

:::

:::callout{kind=note label="E2消除的立体化学"}
**反式共面消除**：
- H和Br必须在反式共面
- 过渡态中部分双键形成

**产物**：
- 主产物：反-2-丁烯（更稳定）
- 副产物：顺-2-丁烯、1-丁烯

**Zaitsev规则**：取代基多的烯烃为主产物
:::


### 反应式4

1-溴丁烷 + NaCN/DMF →

:::callout{kind=insight label="解析"}
**产物**：戊腈（CH₃CH₂CH₂CH₂CN）

**解析**：CN⁻发生SN2，链增长一个碳。

:::

:::callout{kind=note label="腈的合成与应用"}
**合成**：R-X + NaCN → R-CN（SN2）

**转化**：
- R-CN + H₃O⁺, Δ → R-COOH（羧酸）
- R-CN + LiAlH₄ → R-CH₂NH₂（伯胺）
- R-CN + H₂/Ni → R-CH₂NH₂（伯胺）

**应用**：增碳、制备羧酸和胺
:::


### 反应式5

环己醇 + PCC（或Collins试剂）→

:::callout{kind=insight label="解析"}
**产物**：环己酮

**解析**：二级醇选择性氧化为酮。

:::

:::callout{kind=note label="温和氧化剂"}
**PCC（吡啶氯铬酸盐）**：
- 氧化1°醇 → 醛（不进一步氧化）
- 氧化2°醇 → 酮

**Collins试剂**：
- CrO₃·吡啶络合物
- 类似PCC，温和氧化

**Jones试剂**：
- CrO₃/H₂SO₄/丙酮
- 氧化1°醇 → 羧酸
- 氧化2°醇 → 酮
:::


### 反应式6

苄醇 + HBr →

:::callout{kind=insight label="解析"}
**产物**：苄基溴

**解析**：苄位碳正离子/过渡态共振稳定，取代活性高。

:::

:::callout{kind=note label="苄醇的取代"}
**活性**：苄醇 > 烯丙醇 > 3°醇 > 2°醇 > 1°醇

**原因**：
- 苄基碳正离子p-π共轭稳定
- SN1和SN2都有利

**应用**：
- 苄基保护基（易上易下）
- 催化氢解脱苄基
:::


### 反应式7

苯乙酮 + NaBH₄/MeOH →

:::callout{kind=insight label="解析"}
**产物**：1-苯乙醇（外消旋）

**解析**：平面羰基两面进攻，生成新的手性中心，非手性条件下外消旋。

:::

:::callout{kind=note label="羰基还原的立体化学"}
**非手性条件**：
- NaBH₄从两面进攻概率相等
- 生成外消旋混合物

**手性条件**：
- 手性还原剂（如(R)-或(S)-BINAL-H）
- 一面进攻优先
- 生成对映体过量产物
:::


### 反应式8

乙酸 ① SOCl₂ ② NH₃（过量）→

:::callout{kind=insight label="解析"}
**产物**：乙酰胺

**解析**：羧酸先变乙酰氯，再与氨氨解生成酰胺。

:::

:::callout{kind=note label="羧酸→酰胺的转化"}
**路线**：RCOOH → RCOCl → RCONH₂

**步骤**：
1. RCOOH + SOCl₂ → RCOCl + SO₂ + HCl
2. RCOCl + NH₃ → RCONH₂ + HCl

**注意**：
- NH₃需过量（中和HCl）
- 也可用(RCO)₂O + NH₃
:::

## 五、鉴别/排序题（每题5分，共10分）

### 第1题

:::callout{kind=note label="题目"}
鉴别1-丁醇、2-丁醇、叔丁醇。
:::

:::callout{kind=insight label="解析"}
**答案**：用Lucas试剂：叔丁醇立即浑浊；2-丁醇数分钟后浑浊；1-丁醇室温下不明显，加热后才浑浊。

**解析**：浑浊来自生成不溶卤代烃；反应快慢反映碳正离子稳定性和SN1倾向。

:::

:::callout{kind=note label="Lucas试剂鉴别醇"}
**试剂**：ZnCl₂/浓HCl

**现象**：
| 醇类型 | 反应时间 | 现象 |
|--------|----------|------|
| 叔醇 | 立即 | 浑浊 |
| 仲醇 | 5-10分钟 | 浑浊 |
| 伯醇 | 加热后 | 浑浊 |

**原理**：SN1机理，碳正离子稳定性
:::


### 第2题

:::callout{kind=note label="题目"}
鉴别乙酸、乙醛、丙酮、乙酸乙酯。
:::

:::callout{kind=insight label="解析"}
**答案**：先加NaHCO₃：放CO₂气泡者为乙酸。余者加Tollens试剂：银镜者为乙醛。剩余两者加I₂/NaOH：黄色碘仿沉淀者为丙酮；无明显反应者为乙酸乙酯。

**解析**：酸与碳酸氢钠放气；醛可被银氨氧化；甲基酮发生碘仿反应。

:::

:::callout{kind=insight label="系统鉴别策略"}
**第一步：NaHCO₃**
- 放CO₂：羧酸（乙酸）

**第二步：Tollens试剂**
- 银镜：醛（乙醛）

**第三步：I₂/NaOH**
- 黄色沉淀：甲基酮（丙酮）
- 无反应：酯（乙酸乙酯）

**注意**：鉴别顺序很重要！
:::

## 六、合成题（8分）

由1-溴丙烷合成丁酸，写出关键步骤。

:::callout{kind=insight label="解析"}
**路线**：

1-溴丙烷 --(NaCN/DMF)--> 丁腈 CH₃CH₂CH₂CN --(H₃O⁺, Δ 或 OH⁻, Δ 后酸化)--> 丁酸。

**解析**：伯卤代烷与CN⁻发生SN2，增加一个碳；腈水解得到相应羧酸。

:::

:::callout{kind=note label="腈水解条件"}
**酸性水解**：
R-CN + 2H₂O + H⁺ → R-COOH + NH₄⁺

**碱性水解**：
R-CN + OH⁻ + H₂O → R-COO⁻ + NH₃
- 然后酸化得羧酸

**特点**：
- 增加一个碳原子
- 产物为羧酸
:::

## 七、推断题（10分）

A、B、C均为C₄H₈O，均能与2,4-二硝基苯肼生成沉淀。A不发生银镜反应但有碘仿反应；B发生银镜反应，氧化得丁酸；C发生银镜反应，氧化得2-甲基丙酸。推断A、B、C。

:::callout{kind=insight label="解析"}
**A = 2-丁酮**；**B = 正丁醛**；**C = 2-甲基丙醛**

**解析**：2,4-DNP阳性说明均为醛/酮。A不银镜为酮，且有CH₃CO-，故为2-丁酮。B/C为醛；氧化产物分别保留碳骨架为丁酸和2-甲基丙酸。

:::

:::callout{kind=insight label="推断题分析"}
**线索分析**：
1. C₄H₈O，不饱和度1 → 醛或酮
2. 2,4-DNP阳性 → 含羰基
3. 银镜反应 → 区分醛和酮

**A（酮，碘仿阳性）**：
- 含CH₃CO-结构
- 2-丁酮：CH₃COCH₂CH₃

**B（醛，氧化得丁酸）**：
- 碳骨架不变
- 正丁醛：CH₃CH₂CH₂CHO

**C（醛，氧化得2-甲基丙酸）**：
- 碳骨架不变
- 2-甲基丙醛：(CH₃)₂CHCHO

**验证**：
- 正丁醛氧化：CH₃CH₂CH₂COOH（丁酸）
- 2-甲基丙醛氧化：(CH₃)₂CHCOOH（2-甲基丙酸）
:::

## 复习总结

:::callout{kind=tip label="本卷高频失分点"}
1. **SN1/SN2/E1/E2**：不要只看底物，还要看试剂、溶剂、温度
2. **醇与卤代烃转化**：常通过"质子化羟基/好离去基团"理解
3. **羰基反应**：先判断亲核加成、α-H反应还是氧化还原
4. **羧酸衍生物**：活性顺序和离去基团能力是合成题核心
5. **鉴别题**：系统分析，按顺序鉴别
6. **推断题**：利用氧化产物推断碳骨架
:::
