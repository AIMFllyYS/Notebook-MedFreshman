## 一、选择题（每小题2分，共20分）

### 第1题

:::callout{kind=note label="题目"}
下列基团中，诱导效应为吸电子的是（　　）

A. -CH₃　　B. -C(CH₃)₃　　C. -NO₂　　D. -C₂H₅
:::

:::callout{kind=insight label="解析"}
**【答案】C**

-NO₂是强吸电子基团，既有吸电子诱导效应，又有吸电子共轭效应（π-π共轭将电子拉向硝基）。-CH₃、-C(CH₃)₃和-C₂H₅均为给电子基团（超共轭效应/诱导效应）。

:::

:::callout{kind=note label="诱导效应判断"}
**吸电子基团（-I）**：
- 卤素（-F、-Cl、-Br、-I）
- 硝基（-NO₂）
- 氰基（-CN）
- 羰基（-CHO、-COR、-COOH）
- 三氟甲基（-CF₃）

**给电子基团（+I）**：
- 烷基（-CH₃、-C₂H₅、-CH(CH₃)₂、-C(CH₃)₃）
- 烷氧基（-OR）
- 氨基（-NH₂）

**注意**：诱导效应沿σ键传递，随距离增加而衰减
:::


### 第2题

:::callout{kind=note label="题目"}
1,3-丁二烯与HBr在低温（-80°C）下加成的主要产物是（　　）

A. 1,4-加成产物（热力学产物）　　B. 1,2-加成产物（动力学产物）

C. 两者等量　　D. 不反应
:::

:::callout{kind=insight label="解析"}
**【答案】B**

低温(-80°C)条件下，反应受动力学控制，碳正离子直接在离去基团最近的位置（1,2位）结合负离子，生成1,2-加成产物（80%）。高温时受热力学控制，共振导致电荷转移，生成更稳定的1,4-加成产物。老师重点讲解了"1,2-加成是动力学产物，1,4-加成是热力学产物"。

:::

:::callout{kind=insight label="动力学控制 vs 热力学控制"}
**1,2-加成（动力学产物）**：
- 低温（-80°C）
- 速率快，直接结合
- 产物能量较高

**1,4-加成（热力学产物）**：
- 高温（40°C以上）
- 经过共振稳定化
- 产物更稳定（共轭二烯）

**机理**：
H⁺进攻形成烯丙基碳正离子（共振：C2或C4带正电）
- 低温：Br⁻快速结合C2（1,2-加成）
- 高温：达到平衡，更稳定的1,4-产物为主
:::


### 第3题

:::callout{kind=note label="题目"}
下列胺的碱性最强的是（　　）

A. 苯胺（C₆H₅NH₂）　　B. 二苯胺（(C₆H₅)₂NH）

C. 甲胺（CH₃NH₂）　　D. 三苯胺（(C₆H₅)₃N）
:::

:::callout{kind=insight label="解析"}
**【答案】C**

甲胺的甲基为给电子基团（推电子效应），使氮上电子云密度增大，碱性增强。苯胺、二苯胺、三苯胺中苯环与氮形成p-π共轭，分散了氮上孤对电子，碱性减弱。苯环越多碱性越弱。碱性顺序：甲胺 > 氨 > 苯胺 > 二苯胺 > 三苯胺。

:::

:::callout{kind=insight label="胺类碱性比较"}
**碱性顺序**：脂肪胺 > 氨 > 苯胺 > 二苯胺 > 三苯胺 > 酰胺

**原因分析**：
- **脂肪胺**：烷基给电子，氮上电子云密度高
- **氨**：无取代基效应
- **苯胺**：p-π共轭分散孤对电子
- **二苯胺/三苯胺**：共轭效应更强，碱性更弱
- **酰胺**：孤对电子参与共轭，几乎无碱性

**注意**：气相中碱性顺序可能不同（溶剂化效应影响）
:::


### 第4题

:::callout{kind=note label="题目"}
下列卤代烃发生E2消除反应的活性最高的是（　　）

A. (CH₃)₃CBr　　B. (CH₃)₂CHBr　　C. CH₃CH₂Br　　D. CH₃Br
:::

:::callout{kind=insight label="解析"}
**【答案】A**

E2消除反应活性主要受底物烷基结构影响。叔卤代烷最容易发生E2消除反应：①叔碳上直接连接的β-氢最多；②过渡态中形成的部分双键取代基多，更稳定；③位阻大不利于SN2竞争。甲基卤代烷无β-氢，不能发生消除。

:::

:::callout{kind=note label="E2消除反应活性"}
**活性顺序**：3° > 2° > 1°（与SN2相反）

**原因**：
- 过渡态中形成部分双键
- 双键上取代基越多，过渡态越稳定
- 叔卤代烷形成的烯烃取代基最多

**E2反应条件**：
- 强碱（NaOH、KOH、NaOEt、KOtBu）
- 极性非质子溶剂有利
- 加热促进消除

**注意**：E2与SN2竞争，强碱/高温/位阻大有利于E2
:::


### 第5题

:::callout{kind=note label="题目"}
下列羧酸酸性最强的是（　　）

A. CH₃COOH　　B. ClCH₂COOH　　C. Cl₂CHCOOH　　D. Cl₃CCOOH
:::

:::callout{kind=insight label="解析"}
**【答案】D**

α-碳上连的吸电子基团（Cl）越多，通过诱导效应使羧酸负离子越稳定，酸性越强。酸性顺序：Cl₃CCOOH > Cl₂CHCOOH > ClCH₂COOH > CH₃COOH。每多一个氯原子，吸电子效应叠加，pKa逐级降低。

:::

:::callout{kind=insight label="取代基对羧酸酸性的影响"}
**吸电子基团增强酸性**（稳定羧酸根负离子）：
- -Cl、-Br、-I（诱导效应）
- -NO₂、-CN（诱导+共轭）
- -CF₃（强吸电子）

**给电子基团减弱酸性**：
- -CH₃、-C₂H₅（超共轭效应）

**pKa值对比**：
- CH₃COOH：4.76
- ClCH₂COOH：2.86
- Cl₂CHCOOH：1.35
- Cl₃CCOOH：0.70
- CF₃COOH：0.23（超强酸）
:::


### 第6题

:::callout{kind=note label="题目"}
霍夫曼降解反应可以实现（　　）

A. 增碳反应　　B. 减碳反应　　C. 碳数不变　　D. 加碳两个
:::

:::callout{kind=insight label="解析"}
**【答案】B**

霍夫曼降解反应是将酰胺在强碱（NaOH）和氧化剂（Br₂）条件下转化为少一个碳的伯胺。如苯甲酰胺→苯胺，碳数从7减为6。这是课堂上强调的"减碳反应包括霍夫曼降解和羧酸脱羧"。

:::

:::callout{kind=note label="霍夫曼降解反应"}
**反应式**：R-CONH₂ + Br₂ + 4NaOH → R-NH₂ + Na₂CO₃ + 2NaBr + 2H₂O

**特点**：
- 酰胺 → 伯胺（减少1个碳）
- 经历异氰酸酯中间体（R-N=C=O）
- 重排机理（类似Curtius重排）

**应用**：
- 由羧酸制备少一个碳的胺
- 制备不能由硝基还原得到的胺
:::


### 第7题

:::callout{kind=note label="题目"}
关于糖的变旋现象，下列说法正确的是（　　）

A. 所有糖都有变旋现象　　B. 蔗糖有变旋现象

C. 单糖一般都有变旋现象　　D. 多糖都有变旋现象
:::

:::callout{kind=insight label="解析"}
**【答案】C**

变旋现象是指单糖溶于水后，其比旋光度随时间变化最终达到平衡值的现象，原因是α-和β-构型之间的互变平衡。单糖一般都有变旋现象；双糖是否有变旋取决于苷羟基是否保留（蔗糖无变旋，麦芽糖有变旋）；多糖基本无变旋现象。

:::

:::callout{kind=note label="变旋现象与还原性"}
**变旋现象**：α-构型 ↔ β-构型 ↔ 开链式（互变异构）

**有变旋现象**：
- 单糖（葡萄糖、果糖、甘露糖等）
- 还原性二糖（麦芽糖、乳糖）

**无变旋现象**：
- 非还原性二糖（蔗糖）
- 糖苷（甲基葡萄糖苷）
- 多糖

**判断依据**：是否有游离半缩醛/半缩酮羟基
:::


### 第8题

:::callout{kind=note label="题目"}
下列化合物与亚硝酸（NaNO₂/HCl）反应后能生成重氮盐的是（　　）

A. 二乙胺　　B. 苯胺　　C. 三甲胺　　D. 乙酰胺
:::

:::callout{kind=insight label="解析"}
**【答案】B**

芳香伯胺（如苯胺）与NaNO₂/HCl在低温(0-5°C)下反应生成重氮盐(ArN₂⁺Cl⁻)。脂肪伯胺生成的重氮盐不稳定会分解。仲胺生成N-亚硝基胺，叔胺不与NaNO₂反应（脂肪叔胺）或在环上发生亲电取代。

:::

:::callout{kind=insight label="胺与亚硝酸的反应"}
| 胺类型 | 产物 | 稳定性 |
|--------|------|--------|
| 芳香伯胺 | 重氮盐 | 稳定（低温） |
| 脂肪伯胺 | 重氮盐 | 不稳定，立即分解（N₂↑） |
| 仲胺 | N-亚硝基胺 | 稳定（黄色油状） |
| 叔胺 | 不反应或环上取代 | — |

**重氮盐应用**：
- Sandmeyer反应：ArN₂⁺ + CuX → Ar-X
- 偶合反应：ArN₂⁺ + ArOH → Ar-N=N-Ar（偶氮染料）
- 水解：ArN₂⁺ + H₂O → Ar-OH
:::


### 第9题

:::callout{kind=note label="题目"}
酯在强碱条件下发生的缩合反应称为（　　）

A. 羟醛缩合　　B. 克莱森缩合　　C. 康尼扎罗反应　　D. 傅克反应
:::

:::callout{kind=insight label="解析"}
**【答案】B**

克莱森缩合是两分子酯在强碱（如NaOC₂H₅）作用下，一个酯的α-氢被拔掉形成碳负离子，进攻另一个酯的羰基碳，经亲核加成-消除得到β-酮酸酯。羟醛缩合是醛或酮的反应，康尼扎罗是无α-氢醛的歧化反应。

:::

:::callout{kind=note label="克莱森缩合反应"}
**反应式**：2 RCH₂COOR' + NaOEt → RCH₂COCH₂COOR' + R'OH

**机理**：
1. 碱夺取α-H形成碳负离子
2. 碳负离子进攻另一分子酯的羰基
3. 消除烷氧基（RO⁻）
4. 生成β-酮酸酯

**注意**：
- 需要至少一个α-H
- 产物β-酮酸酯有活泼亚甲基
- 可进一步发生烷基化、酰基化
:::


### 第10题

:::callout{kind=note label="题目"}
下列关于氨基酸的叙述，错误的是（　　）

A. 氨基酸为两性化合物　　B. 人体有8种必需氨基酸

C. 蛋白质的主键是肽键　　D. 蛋白质的副键全是非共价键
:::

:::callout{kind=insight label="解析"}
**【答案】D**

蛋白质的副键不全是非共价键。副键包括氢键、离子键、疏水键等非共价键，但也包括二硫键（-S-S-），二硫键是共价键。老师课堂上明确指出"副键不见得都是非共价键，如二硫键是共价键"。

:::

:::callout{kind=note label="蛋白质结构层次"}
**一级结构**：氨基酸序列（肽键连接）

**二级结构**：α-螺旋、β-折叠（氢键维持）

**三级结构**：整体三维构象（多种副键）：
- 氢键（非共价）
- 离子键（非共价）
- 疏水键（非共价）
- 范德华力（非共价）
- **二硫键（共价键）**

**四级结构**：亚基组装（非共价键）
:::

## 二、填空题（每空2分，共20分）

### 第1题

:::callout{kind=note label="题目"}
取代基效应由_______效应和_______效应两部分构成，总电子效应有推电子和吸电子之分。
:::

:::callout{kind=insight label="解析"}
**答案**：诱导；共轭

**解析**：取代基效应=诱导效应+共轭效应。诱导效应通过σ键传递（沿碳链衰减），共轭效应需在共轭体系中存在。两种效应之和决定基团的总电子效应。

:::

:::callout{kind=note label="诱导效应 vs 共轭效应"}
| 特征 | 诱导效应 | 共轭效应 |
|------|----------|----------|
| 传递方式 | σ键 | π键或p轨道 |
| 衰减性 | 随距离衰减 | 不衰减（共轭体系内） |
| 存在条件 | 所有分子 | 需共轭体系 |
| 电子移动 | 沿σ键 | 离域 |

**总电子效应** = 诱导效应 + 共轭效应
- 同向：效应叠加（如-NO₂：-I和-C都是吸电子）
- 反向：效应抵消（如-Cl：-I吸电子，+C给电子）
:::


### 第2题

:::callout{kind=note label="题目"}
共轭二烯烃1,3-丁二烯与HBr加成，低温主要得到_______加成产物（动力学控制），高温主要得到_______加成产物（热力学控制）。
:::

:::callout{kind=insight label="解析"}
**答案**：1,2-；1,4-

**解析**：低温时动力学控制，直接结合碳正离子最近的负电中心（1,2-位），速率快但产物能量较高。高温时热力学控制，体系有足够能量达到热力学平衡，1,4-加成产物内能更低，更稳定，为主要产物。

:::

:::callout{kind=insight label="动力学控制 vs 热力学控制"}
**动力学控制**：
- 低温、短时间
- 产物由反应速率决定
- 1,2-加成产物（速率快）

**热力学控制**：
- 高温、长时间
- 产物由稳定性决定
- 1,4-加成产物（共轭二烯更稳定）

**能量图**：
1,2-产物：活化能低，但产物能量较高
1,4-产物：活化能高，但产物能量更低
:::


### 第3题

:::callout{kind=note label="题目"}
羧酸衍生物的反应活性顺序为：_______＞_______＞酯＞酰胺。
:::

:::callout{kind=insight label="解析"}
**答案**：酰卤（酰氯）；酸酐

**解析**：活性由羰基碳的电正性和L基团的离去能力决定。酰卤中卤素电负性强，使羰基碳正电性最大，且Cl⁻是好的离去基团；酸酐次之；酯再次之；酰胺中N的孤对电子与羰基形成p-π共轭，使羰基碳正电性减小，活性最低。

:::

:::callout{kind=note label="羧酸衍生物活性顺序"}
**活性顺序**：酰卤 > 酸酐 > 酯 > 酰胺

**原因**：
1. **羰基碳正电性**：Cl吸电子 > O > OR > NR₂
2. **离去基团能力**：Cl⁻ > RCOO⁻ > RO⁻ > NH₂⁻

**反应特点**：
- 酰卤：最活泼，不需催化剂
- 酸酐：较活泼，常用于酰基化
- 酯：需碱或酸催化
- 酰胺：最稳定，需强条件
:::


### 第4题

:::callout{kind=note label="题目"}
合成中加碳一个碳原子常用_______（试剂），加碳两个碳原子常用_______与格氏试剂反应实现。
:::

:::callout{kind=insight label="解析"}
**答案**：NaCN（氰化钠/氰基）；环氧乙烷

**解析**：这是老师课堂上总结的加碳方法。多一个碳用氰基(CN⁻)：R-X + NaCN → R-CN → R-CH₂NH₂(催化氢化)或R-COOH(水解)。多两个碳用环氧乙烷：R-MgBr + 环氧乙烷 → R-CH₂CH₂OH。

:::

:::callout{kind=insight label="有机合成增碳策略"}
| 增碳数 | 试剂/方法 | 产物 |
|--------|-----------|------|
| +1 | NaCN → 水解 | 羧酸 |
| +1 | NaCN → 还原 | 胺 |
| +1 | CO₂ + 格氏试剂 | 羧酸 |
| +2 | 环氧乙烷 + 格氏试剂 | 伯醇 |
| +R | RCHO + 格氏试剂 | 仲醇 |
| +R | RCOR + 格氏试剂 | 叔醇 |
| +R | RX + NaCN | 腈 → 多种产物 |
:::


### 第5题

:::callout{kind=note label="题目"}
伯胺与亚硝酸反应可生成_______，进一步可发生_______反应（如Sandmeyer反应）得到多种取代产物。
:::

:::callout{kind=insight label="解析"}
**答案**：重氮盐（ArN₂⁺）；取代（如Sandmeyer反应、Schiemann反应）

**解析**：芳香伯胺在低温下与NaNO₂/HCl反应生成重氮盐。重氮盐可发生多种取代反应：Sandmeyer反应（CuX）生成卤代苯，Schiemann反应（HBF₄）生成氟代苯，与CuCN生成氰基苯，还可生成酚（加热水解）和偶氮化合物。

:::

:::callout{kind=note label="重氮盐的反应"}
**取代反应**：
| 试剂 | 产物 |
|------|------|
| CuCl/HCl | 氯代苯 |
| CuBr/HBr | 溴代苯 |
| CuCN/KCN | 氰基苯 |
| HBF₄ | 氟代苯（Schiemann） |
| H₂O/加热 | 苯酚 |
| H₃PO₂ | 苯（还原去氨基） |

**偶合反应**：
ArN₂⁺ + ArOH（弱碱）→ 偶氮化合物（染料）
:::

## 三、判断题（每小题2分，共10分，对的画√，错的画×）

### 第1题

:::callout{kind=note label="题目"}
诱导效应沿碳链传递，一般传递三个碳以上就可以忽略不计。（　　）
:::

:::callout{kind=insight label="解析"}
**【答案】√**

解析：诱导效应的特点是沿σ键传递并逐渐衰减。一般经过三个碳原子后，诱导效应已经很弱，可以忽略不计。这与共轭效应不同，共轭效应沿共轭体系传递，理论上不衰减。

:::

:::callout{kind=note label="诱导效应的衰减"}
**衰减规律**：每经过一个碳原子，效应衰减为原来的约1/3

**示例**：
Cl-CH₂-CH₂-CH₂-COOH
- α位（Cl直接连的碳）：效应最强
- β位：效应减弱
- γ位：效应很弱
- δ位以上：可忽略

**对比共轭效应**：
- 共轭体系中电子离域，不衰减
- 如苯环上取代基效应传递到对位仍明显
:::


### 第2题

:::callout{kind=note label="题目"}
端基炔烃的三键碳上氢有一定酸性，可与银氨溶液反应。（　　）
:::

:::callout{kind=insight label="解析"}
**【答案】√**

解析：端基炔碳为sp杂化，电负性较大（sp > sp² > sp³），使C-H键上的氢具有一定酸性(pKa≈25)。可与银氨溶液反应生成白色炔基银沉淀（R-C≡CAg↓），也可与铜盐反应生成红色炔基铜。此反应可用于端基炔的鉴定。

:::

:::callout{kind=note label="端基炔的酸性"}
**酸性顺序**：
羧酸 > 碳酸 > 苯酚 > 水 > 醇 > 端基炔 > 氨 > 烷烃

**pKa值**：
- 端基炔：~25
- 氨：~35
- 烷烃：~50

**鉴定反应**：
- 银氨溶液：白色沉淀（炔银）
- 铜氨溶液：红色沉淀（炔铜）
- 非端基炔无此反应
:::


### 第3题

:::callout{kind=note label="题目"}
查依采夫规则指出，消除反应的主要产物是双键上取代基最多的烯烃（更稳定的烯烃）。（　　）
:::

:::callout{kind=insight label="解析"}
**【答案】√**

解析：查依采夫规则（Zaitsev规则）指出：在消除反应中，主要产物是双键上取代基更多的烯烃（即更稳定的烯烃）。如2-丁基溴消除主要生成2-丁烯而非1-丁烯。但霍夫曼消除是例外，生成取代基较少的烯烃。

:::

:::callout{kind=note label="消除反应取向规则"}
**Zaitsev规则（查依采夫）**：
- 主要产物：双键上取代基多的烯烃（更稳定）
- 适用于：E1、E2（一般情况）

**Hofmann规则（霍夫曼）**：
- 主要产物：双键上取代基少的烯烃
- 适用于：季铵碱消除、 bulky base（如t-BuOK）

**原因**：
- Zaitsev：过渡态中部分双键，取代基多更稳定
- Hofmann：位阻大的碱优先进攻位阻小的β-H
:::


### 第4题

:::callout{kind=note label="题目"}
蔗糖没有还原性，因为它的苷羟基已参与形成糖苷键。（　　）
:::

:::callout{kind=insight label="解析"}
**【答案】√**

解析：蔗糖是由葡萄糖和果糖通过1,2-糖苷键连接的非还原性二糖。两个单糖的苷羟基（异头碳上的-OH）都参与了糖苷键的形成，不能开环生成开链含醛基形式，因此无还原性，也无变旋现象。

:::

:::callout{kind=note label="糖的还原性判断"}
**还原糖**：含游离半缩醛/半缩酮羟基
- 单糖（葡萄糖、果糖、甘露糖等）
- 还原性二糖（麦芽糖、乳糖）

**非还原糖**：无游离半缩醛/半缩酮羟基
- 蔗糖（两个苷羟基都参与糖苷键）
- 糖苷（甲基葡萄糖苷等）
- 多糖（淀粉、纤维素等）

**鉴定方法**：
- Tollens试剂：银镜反应
- Fehling试剂：红色Cu₂O沉淀
- Benedict试剂：砖红色沉淀
:::


### 第5题

:::callout{kind=note label="题目"}
酰胺的碱性比胺强，因为酰胺中也含有氮原子。（　　）
:::

:::callout{kind=insight label="解析"}
**【答案】×**

解析：酰胺的碱性远弱于胺，甚至可以说酰胺几乎没有碱性。原因是酰胺中氮上的孤对电子与羰基形成p-π共轭，电子离域到羰基上，使氮上电子云密度大大降低。课堂上强调碱性顺序：季铵碱 > 有机胺 > 氨 > 苯胺 > 酰胺。

:::

:::callout{kind=insight label="酰胺的弱碱性"}
**原因**：
- 氮孤对电子参与p-π共轭
- 电子离域到羰基氧上
- 氮上电子云密度极低

**对比**：
- 胺：氮孤对电子可结合质子（碱性）
- 酰胺：氮孤对电子被共轭"占用"（几乎无碱性）
- 酰亚胺：两个羰基夹一个氮，N-H显酸性（如邻苯二甲酰亚胺）

**应用**：
- 酰胺可用作非碱性溶剂（DMF、NMP）
- 蛋白质中肽键的酰胺键稳定
:::

## 四、反应题（每小题5分，共25分。写出主要产物或反应条件）

### 第1题

:::callout{kind=note label="题目"}
间二硝基苯 + Na₂S（过量）——→ ？
:::

:::callout{kind=insight label="解析"}
**答案**：间二氨基苯（1,3-二氨基苯）

**解析**：Na₂S是还原条件，可以将硝基(-NO₂)还原为氨基(-NH₂)。过量的Na₂S可以将两个硝基都还原成氨基。如果Na₂S不过量（如1mol），则只还原一个硝基，生成间氨基硝基苯。老师课堂上以此为例讲解了反应条件对产物的决定作用。

:::

:::callout{kind=note label="硝基还原方法"}
| 还原剂 | 条件 | 产物 |
|--------|------|------|
| Fe/HCl | 酸性 | 胺 |
| Sn/HCl | 酸性 | 胺 |
| H₂/Pd | 催化氢化 | 胺 |
| Na₂S | 碱性 | 部分还原 |
| LiAlH₄ | 无水 | 胺 |

**选择性还原**：
- Na₂S（1mol）：只还原一个硝基
- Na₂S（过量）：还原所有硝基
- SnCl₂/HCl：选择性还原（多硝基中只还原一个）
:::


### 第2题

:::callout{kind=note label="题目"}
苯胺 + NaNO₂/HCl（0-5°C）——→ ？　进一步 + CuCN ——→ ？
:::

:::callout{kind=insight label="解析"}
**第一步**：C₆H₅NH₂ + NaNO₂/HCl(0-5°C) → C₆H₅N₂⁺Cl⁻（苯重氮盐）

**第二步**：C₆H₅N₂⁺Cl⁻ + CuCN → C₆H₅CN（苯腈/氰基苯）+ N₂↑ + CuCl

**解析**：苯胺在低温下与NaNO₂/HCl反应生成重氮盐。重氮盐与CuCN反应是Sandmeyer反应的一种变体，N₂离去的同时CN取代进入苯环。苯腈可进一步水解得苯甲酸。

:::

:::callout{kind=insight label="Sandmeyer反应"}
**反应类型**：重氮基被卤素、氰基等取代

**反应机理**：
1. CuX与重氮盐形成络合物
2. 电子转移，N₂离去
3. X⁻或CN⁻进入芳环

**应用**：
- 由氨基引入卤素、氰基
- 合成不能直接卤代/氰化的芳烃
- 制备酚（水解重氮盐）
:::


### 第3题

:::callout{kind=note label="题目"}
CH₃COOC₂H₅ + NaOC₂H₅ ——(克莱森缩合)——→ ？
:::

:::callout{kind=insight label="解析"}
**答案**：CH₃COCH₂COOC₂H₅（乙酰乙酸乙酯）+ C₂H₅OH

**解析**：克莱森缩合反应：一分子酯的α-H被强碱NaOC₂H₅拔掉形成碳负离子（亲核试剂），进攻另一分子酯的羰基碳，发生亲核加成，然后乙氧基作为离去基团离去，生成β-酮酸酯——乙酰乙酸乙酯。

:::

:::callout{kind=note label="乙酰乙酸乙酯的合成应用"}
**结构特点**：
- 含活泼亚甲基（-CH₂-夹在两个羰基之间）
- pKa ~11（比普通C-H酸性更强）

**应用**：
1. **酮式分解**：稀碱水解后酸化加热 → 酮
2. **酸式分解**：浓碱水解 → 羧酸（少一个碳）
3. **亚甲基烷基化**：
   - 碱夺取α-H
   - 与RX反应引入烷基
   - 再酮式分解得取代酮

**示例**：合成2-戊酮
CH₃COCH₂COOEt →(NaOEt)→ CH₃COCH⁻COOEt →(CH₃I)→ CH₃COCH(CH₃)COOEt →(酮式分解)→ CH₃COCH₂CH₃
:::


### 第4题

:::callout{kind=note label="题目"}
端基炔 R-C≡CH + AgNO₃(NH₃·H₂O) ——→ ？
:::

:::callout{kind=insight label="解析"}
**答案**：R-C≡CAg↓（白色炔银沉淀）+ NH₄NO₃

**解析**：端基炔上的氢有一定酸性，可以被银氨溶液中的银离子取代，生成白色的炔基银沉淀。这是端基炔的特征反应，可用于鉴定端基炔。非端基炔不发生此反应。

:::

:::callout{kind=note label="端基炔的鉴定"}
**反应**：R-C≡CH + [Ag(NH₃)₂]⁺ → R-C≡CAg↓ + NH₄⁺ + NH₃

**现象**：白色沉淀（炔银）

**注意事项**：
- 端基炔才有此反应
- 非端基炔（R-C≡C-R'）无反应
- 沉淀可溶于稀硝酸

**类似反应**：
- Cu₂Cl₂/NH₃·H₂O：红色沉淀（炔铜）
- 用于区分端基炔和非端基炔
:::


### 第5题

:::callout{kind=note label="题目"}
(CH₃)₃CBr + C₂H₅O⁻ ——(乙醇)——→ ？（判断反应类型并写出主要产物）
:::

:::callout{kind=insight label="解析"}
**答案**：(CH₃)₂C=CH₂（2-甲基丙烯/异丁烯）+ C₂H₅OH + Br⁻

**反应类型**：E2消除反应

**解析**：叔卤代烃空间位阻大，不利于SN2反应。C₂H₅O⁻是强碱但亲核性因位阻不够强，乙醇为非极性溶剂不利于SN1。综合判断，主要发生E2消除反应。β-碳上的H与Br反式共面消除，生成烯烃。

:::

:::callout{kind=insight label="SN1/SN2/E1/E2 竞争分析"}
**底物**：(CH₃)₃CBr（3°卤代烷）

**试剂**：C₂H₅O⁻/EtOH（强碱/质子溶剂）

**分析**：
- SN2：3°位阻大，不利
- SN1：EtOH质子溶剂有利，但C₂H₅O⁻强碱会促进消除
- E2：强碱 + 3°底物（有利）
- E1：质子溶剂有利，但E2更快

**结论**：E2消除为主

**产物**：遵循Zaitsev规则，生成取代基多的烯烃
:::

## 五、合成题（每小题7.5分，共15分。写出合成路线及所用试剂和条件）

### 第1题

:::callout{kind=note label="题目"}
以溴代环戊烷为原料合成环戊基甲胺（碳链增加一个碳的胺）。

（提示：利用氰基实现增碳）
:::

:::callout{kind=insight label="解析"}
**路线**：

① 环戊基溴 + NaCN ——(SN2)——→ 环戊基氰（环戊腈）

② 环戊基氰 ——(LiAlH₄或H₂/Ni)——→ 环戊基甲胺（环戊基-CH₂NH₂）

**解析**：

第一步利用CN⁻的亲核取代反应（SN2），将溴代环戊烷转化为环戊腈，实现碳链增长一个碳。

第二步将氰基(-CN)催化氢化还原为氨甲基(-CH₂NH₂)。

这是课堂上讲解的"三个碳的溴代烷合成四个碳的胺"的经典方法，利用氰基实现增碳。

:::

:::callout{kind=note label="由卤代烷合成胺"}
**路线**：R-X → R-CN → R-CH₂NH₂

**步骤**：
1. **亲核取代**：R-X + NaCN → R-CN（SN2）
2. **还原**：R-CN + LiAlH₄ → R-CH₂NH₂

**替代方法**：
- Gabriel合成：R-X + 邻苯二甲酰亚胺钾 → R-NH₂（纯伯胺）
- 霍夫曼重排：R-CONH₂ + Br₂/NaOH → R-NH₂（少一个碳）

**注意**：
- LiAlH₄还原腈得伯胺
- 催化氢化（H₂/Ni）也可还原腈
:::


### 第2题

:::callout{kind=note label="题目"}
由苯甲酸合成苯胺。（提示：需要减碳反应）
:::

:::callout{kind=insight label="解析"}
**路线**：

① C₆H₅COOH + SOCl₂ ——→ C₆H₅COCl（苯甲酰氯）

② C₆H₅COCl + NH₃ ——→ C₆H₅CONH₂（苯甲酰胺）

③ C₆H₅CONH₂ + Br₂/NaOH ——(霍夫曼降解)——→ C₆H₅NH₂（苯胺）+ Na₂CO₃

**解析**：

苯甲酸(7C)变为苯胺(6C)，需要减少一个碳。利用霍夫曼降解反应：酰胺在强碱和溴的条件下，脱去羰基碳生成少一个碳的伯胺。这是老师课堂上重点强调的减碳方法。

先将羧酸转化为酰卤（SOCl₂），再氨解生成酰胺，最后进行霍夫曼降解。

:::

:::callout{kind=insight label="减碳策略"}
**减碳方法**：
1. **霍夫曼降解**：R-CONH₂ → R-NH₂（-1C）
2. **脱羧反应**：R-COOH → R-H（-1C）
3. **Ruff降解**：糖化学中减碳

**霍夫曼降解特点**：
- 酰胺 → 伯胺
- 减少一个碳（羰基碳）
- 重排机理（异氰酸酯中间体）
- 构型保持（若R为手性中心）

**对比Curtius重排**：
R-CO-N₃ → R-N=C=O → R-NH₂（同样减碳）
:::

## 六、推断题（10分）

化合物A(C₄H₉N)是一个含氮有机化合物，A与过量CH₃I反应生成B(季铵盐)，B用湿Ag₂O处理后加热发生消除反应生成C(C₄H₈)和三甲胺。

**(1) 写出A、B、C的结构式。（4分）**

**(2) 写出A→B→C各步反应方程式。（3分）**

**(3) C经酸性KMnO₄氧化后的产物是什么？（3分）**

:::callout{kind=insight label="解析"}
**(1) A、B、C的结构式**

A(C₄H₉N)含氮有机化合物，不饱和度=(2×4+2+1-9)/2=0，为饱和胺。

A与过量CH₃I反应生成季铵盐B，说明A为伯胺或仲胺。

最终消除得到C(C₄H₈)和三甲胺N(CH₃)₃。

A = 丁胺（CH₃CH₂CH₂CH₂NH₂）或 2-丁胺，根据消除产物C₄H₈推断：

如果A=正丁胺，甲基化得季铵盐，碱处理后霍夫曼消除生成1-丁烯(C₄H₈)+N(CH₃)₃。

因此：
- **A** = CH₃CH₂CH₂CH₂NH₂（正丁胺）
- **B** = CH₃CH₂CH₂CH₂N⁺(CH₃)₃ I⁻（季铵盐）
- **C** = CH₂=CHCH₂CH₃（1-丁烯）

（4分）

**(2) 各步反应方程式**

① A→B：CH₃CH₂CH₂CH₂NH₂ + 3CH₃I → CH₃CH₂CH₂CH₂N⁺(CH₃)₃ I⁻ + 2HI

（甲基化反应，过量CH₃I使伯胺变为季铵盐）

② B经Ag₂O(湿)处理：季铵碘化物 → 季铵氢氧化物（季铵碱）

③ 季铵碱加热消除：CH₃CH₂CH₂CH₂N⁺(CH₃)₃ OH⁻ →(加热)→ CH₂=CHCH₂CH₃ + N(CH₃)₃ + H₂O

（霍夫曼消除反应，生成取代基较少的烯烃）

（3分）

**(3) C经酸性KMnO₄氧化**

CH₂=CHCH₂CH₃ →(KMnO₄/H⁺,加热)→ CO₂↑ + CH₃CH₂COOH（丙酸）

**解析**：酸性KMnO₄是强氧化条件，双键断裂。=CH₂端无烷基，氧化生成CO₂；另一端连有乙基，氧化生成丙酸。

（3分）

:::

:::callout{kind=insight label="霍夫曼消除反应"}
**反应条件**：
1. 伯胺/仲胺 + 过量CH₃I → 季铵盐
2. 季铵盐 + Ag₂O/H₂O → 季铵碱
3. 季铵碱加热 → 烯烃 + 三甲胺

**产物特点**：
- 生成取代基较少的烯烃（Hofmann产物）
- 与Zaitsev规则相反
- 原因：位阻大的碱（OH⁻）优先进攻位阻小的β-H

**应用**：
- 由胺制备烯烃
- 测定胺的结构（通过消除产物推断）
- 与Cope消除（胺氧化物）互补
:::

:::callout{kind=tip label="本卷高频考点"}
1. **电子效应**：诱导效应 vs 共轭效应，吸电子/给电子基团
2. **动力学/热力学控制**：1,2-加成 vs 1,4-加成
3. **胺类碱性**：脂肪胺 > 氨 > 苯胺 > 酰胺
4. **消除反应**：E2活性、Zaitsev/Hofmann规则
5. **羧酸酸性**：吸电子取代基增强酸性
6. **重氮盐**：Sandmeyer反应、偶合反应
7. **糖化学**：变旋现象、还原性判断
8. **合成策略**：增碳（CN⁻、环氧乙烷）、减碳（霍夫曼降解）
9. **推断题**：霍夫曼消除、烯烃氧化产物分析
:::
