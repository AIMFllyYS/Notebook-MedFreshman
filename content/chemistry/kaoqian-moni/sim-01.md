## 一、选择题（每小题2分，共20分）

### 第1题

:::callout{kind=note label="题目"}
下列化合物中，碳原子采取sp²杂化的是（　　）

A. CH₄　　B. CH₂=CH₂　　C. CH≡CH　　D. CH₃OH
:::

:::callout{kind=insight label="解析"}
**【答案】B**

CH₂=CH₂（乙烯）中碳原子形成一个σ键和一个π键，采用sp²杂化。CH₄为sp³杂化，CH≡CH为sp杂化，CH₃OH中碳为sp³杂化。

:::

:::callout{kind=note label="杂化轨道判断要点"}
- **sp³杂化**：碳原子形成4个σ键（如烷烃、醇）
- **sp²杂化**：碳原子形成3个σ键+1个π键（如烯烃、芳烃）
- **sp杂化**：碳原子形成2个σ键+2个π键（如炔烃）
- 判断口诀：数σ键，4→sp³，3→sp²，2→sp
:::


### 第2题

:::callout{kind=note label="题目"}
根据Brønsted酸碱理论，下列物质中碱性最强的是（　　）

A. CH₃COO⁻　　B. C₂H₅O⁻　　C. C₆H₅O⁻　　D. Cl⁻
:::

:::callout{kind=insight label="解析"}
**【答案】B**

根据Brønsted酸碱理论，碱是接受质子的物质。共轭酸碱对中，共轭酸的酸性越弱，其共轭碱的碱性越强。乙醇的酸性(pKa≈16)远弱于醋酸(pKa≈4.7)和苯酚(pKa≈10)，故C₂H₅O⁻碱性最强。

:::

:::callout{kind=insight label="共轭酸碱对碱性判断"}
**核心规律**：共轭酸的酸性越弱 → 共轭碱的碱性越强

常见有机酸酸性顺序：
- 羧酸 pKa ≈ 4~5（如醋酸 pKa ≈ 4.7）
- 苯酚 pKa ≈ 10
- 醇 pKa ≈ 16~18
- 水 pKa ≈ 15.7

因此碱性顺序：C₂H₅O⁻ > C₆H₅O⁻ > CH₃COO⁻ > Cl⁻（HCl为强酸，Cl⁻几乎无碱性）
:::


### 第3题

:::callout{kind=note label="题目"}
下列化合物的系统命名正确的是（　　）

A. 2,2-二甲基丁烷（新己烷）　　B. 2-甲基-3-乙基戊烷

C. 3,3-二甲基戊烷　　　　　　　D. 2-乙基丁烷
:::

:::callout{kind=insight label="解析"}
**【答案】C**

A选项"新己烷"只是俗称不是系统命名；B选项应选含取代基最多的最长碳链，正确命名应为3-甲基己烷等；D选项中乙基不应出现在2号位需重新选主链。C选项3,3-二甲基戊烷命名规范正确。

:::

:::callout{kind=note label="系统命名规则要点"}
1. **选主链**：含取代基最多的最长碳链
2. **编号**：使取代基位次之和最小
3. **命名**：先小取代基后大取代基，相同取代基合并用中文数字
4. **常见错误**：2-乙基丁烷（主链选错，应为3-甲基戊烷）
:::


### 第4题

:::callout{kind=note label="题目"}
丙烯与HBr在过氧化物存在下的加成产物主要是（　　）

A. 2-溴丙烷　　B. 1-溴丙烷　　C. 1,2-二溴丙烷　　D. 2,3-二溴丙烷
:::

:::callout{kind=insight label="解析"}
**【答案】B**

在过氧化物存在下，丙烯与HBr发生自由基加成反应（Kharasch反应），遵循反马氏规则。溴自由基先进攻双键端基碳（含氢较多的碳），生成更稳定的二级碳自由基中间体，最终主要产物为1-溴丙烷。

:::

:::figure{src="/chemistry/kaoqian-moni/images/sim-01-丙烯-C=CC.png" alt="丙烯" caption="丙烯"}
:::

:::callout{kind=insight label="马氏规则 vs 反马氏规则"}
**马氏规则**（亲电加成）：H加在含H较多的碳上，适用于HX（无过氧化物）

**反马氏规则**（自由基加成）：Br加在含H较多的碳上，仅适用于HBr+过氧化物

**关键识别点**：
- 看到"过氧化物"→ 反马氏（Kharasch反应）
- 看到"酸催化"→ 马氏规则
- 看到"硼氢化-氧化"→ 反马氏+顺式加成
:::
### 第5题

:::callout{kind=note label="题目"}
烯烃经硼氢化-氧化反应所得醇的特点是（　　）

A. 马氏规则、顺式加成　　B. 反马氏规则、顺式加成

C. 马氏规则、反式加成　　D. 反马氏规则、反式加成
:::

:::callout{kind=insight label="解析"}
**【答案】B**

硼氢化-氧化反应中，硼原子进攻电子云密度较高（含氢较多）的碳原子，形成四元环过渡态，属于顺式加成。氧化后羟基取代硼的位置，总结果为反马氏规则的顺式加成。

:::

:::figure{src="/chemistry/kaoqian-moni/images/sim-01-硼烷-B.png" alt="硼烷" caption="硼烷"}
:::

:::callout{kind=insight label="硼氢化-氧化反应机理"}
**反应特点**：反马氏 + 顺式加成

**机理分步**：
1. **硼氢化**：BH₃中B-H键对双键顺式加成，B连在位阻小的碳上
2. **氧化**：H₂O₂/NaOH将C-B键氧化为C-OH键，构型保持

**应用**：由末端烯烃制备伯醇（反马氏产物）
:::
### 第6题

:::callout{kind=note label="题目"}
下列卤代烃进行SN2反应的活性顺序正确的是（　　）

A. CH₃Br > (CH₃)₂CHBr > (CH₃)₃CBr　　B. (CH₃)₃CBr > (CH₃)₂CHBr > CH₃Br

C. (CH₃)₂CHBr > CH₃Br > (CH₃)₃CBr　　D. CH₃Br > (CH₃)₃CBr > (CH₃)₂CHBr
:::

:::callout{kind=insight label="解析"}
**【答案】A**

SN2反应为背面进攻的一步反应，对底物的空间位阻非常敏感。烷基上取代基越多，位阻越大，SN2反应越不利。因此活性顺序为：甲基卤代烷 > 伯卤代烷 > 仲卤代烷 > 叔卤代烷。

:::

:::callout{kind=note label="SN2反应活性规律"}
**活性顺序**：CH₃X > 1° > 2° > 3°（位阻效应主导）

**关键特征**：
- 背面进攻，构型翻转（Walden翻转）
- 一步协同反应，无中间体
- 亲核试剂浓度影响反应速率（二级动力学）

**对比SN1**：SN1活性顺序相反：3° > 2° > 1° > CH₃X
:::


### 第7题

:::callout{kind=note label="题目"}
醛酮与NaHSO₃发生亲核加成反应时，下列哪种底物反应活性最低？（　　）

A. HCHO　　B. CH₃CHO　　C. CH₃COCH₃　　D. (C₆H₅)₂CO
:::

:::callout{kind=insight label="解析"}
**【答案】D**

醛酮与NaHSO₃的亲核加成反应受羰基周围位阻的影响很大。(C₆H₅)₂CO（二苯酮）两侧都有体积较大的苯基，位阻最大，反应活性最低。一般规律：醛 > 甲基酮 > 其他酮 > 位阻大的酮。

:::

:::callout{kind=note label="NaHSO₃加成反应活性"}
**活性规律**：醛 > 甲基酮 > 其他脂肪酮 > 芳香酮

**原因**：
- 醛的羰基碳正电性最强，位阻最小
- 甲基酮次之
- 芳香酮因苯环共轭使羰基碳正电性降低，且位阻大

**应用**：利用NaHSO₃加成可分离醛和酮（醛反应，酮不反应）
:::


### 第8题

:::callout{kind=note label="题目"}
判断卤代烷亲核取代反应按SN1还是SN2机理进行的"四个抓手"不包括（　　）

A. 烷基结构　　B. 离去基团　　C. 反应温度　　D. 溶剂
:::

:::callout{kind=insight label="解析"}
**【答案】C**

老师明确指出判断卤代烷SN反应机理的"四个抓手"为：①卤代烷的烷基结构、②离去基团、③亲核试剂的亲核性、④溶剂。反应温度不在这四个因素之中。

:::

:::callout{kind=insight label="SN1 vs SN2 判断"}
**四个抓手**：
1. **烷基结构**：3°→SN1；1°/CH₃→SN2；2°→两者竞争
2. **离去基团**：好的离去基团（I⁻ > Br⁻ > Cl⁻）有利于SN1和SN2
3. **亲核试剂**：强亲核试剂（I⁻、CN⁻、RS⁻）有利于SN2；弱亲核试剂（H₂O、ROH）有利于SN1
4. **溶剂**：极性质子溶剂（H₂O、EtOH）有利于SN1；极性非质子溶剂（DMSO、DMF）有利于SN2

**反应温度**：温度升高通常同时加速SN1和SN2，不是判断依据
:::


### 第9题

:::callout{kind=note label="题目"}
下列杂环化合物中碱性最强的是（　　）

A. 吡咯　　B. 吡啶　　C. 苯胺　　D. 呋喃
:::

:::callout{kind=insight label="解析"}
**【答案】B**

吡啶的氮上孤对电子不参与芳香共轭体系（吡啶型氮），可以接受质子，碱性较强。吡咯的氮上孤对电子参与芳香共轭（吡咯型氮），不显碱性。苯胺因苯环与氮的p-π共轭削弱了碱性。呋喃含氧无碱性。碱性顺序：吡啶 > 苯胺 > 吡咯。

:::

:::figure{src="/chemistry/kaoqian-moni/images/sim-01-吡啶-c1ccncc1.png" alt="吡啶" caption="吡啶"}
:::
:::figure{src="/chemistry/kaoqian-moni/images/sim-01-吡咯-c1cc[nH]c1.png" alt="吡咯" caption="吡咯"}
:::

:::callout{kind=insight label="杂环化合物碱性比较"}
**碱性顺序**：吡啶 > 苯胺 > 吡咯 > 呋喃

**原因分析**：
- **吡啶**：氮上孤对电子在sp²杂化轨道，不参与芳香共轭，可接受质子
- **苯胺**：氮孤对电子与苯环p-π共轭，电子云密度降低
- **吡咯**：氮孤对电子参与6π芳香体系，不能结合质子（N-H反而显弱酸性）
- **呋喃**：氧孤对电子参与芳香共轭，无碱性
:::

### 第10题

:::callout{kind=note label="题目"}
下列关于吡咯和吡啶的叙述，错误的是（　　）

A. 吡咯的氮上氢具有酸性　　B. 吡啶的氮上孤对电子具有碱性

C. 吡咯的亲电取代比苯容易　　D. 吡啶的亲电取代比苯容易
:::

:::callout{kind=insight label="解析"}
**【答案】D**

吡啶环上氮原子的吸电子效应使环上电子云密度降低，因此吡啶的亲电取代反应比苯更难发生，亲电取代发生在β位。而吡咯、呋喃、噻吩等五元杂环的亲电取代比苯环容易。

:::

:::callout{kind=note label="五元杂环 vs 六元杂环亲电取代"}
**五元杂环**（吡咯、呋喃、噻吩）：
- 富电子芳香体系（6π电子中2个来自杂原子）
- 亲电取代比苯容易，优先在α位

**六元杂环**（吡啶）：
- 缺电子芳香体系（6π电子全部来自碳）
- 亲电取代比苯困难，在β位
- 但亲核取代比苯容易（在α、γ位）
:::

## 二、填空题（每空2分，共20分）

### 第1题

:::callout{kind=note label="题目"}
共价键断裂有两种方式：_______和_______。均裂产生_______，异裂产生碳正离子或碳负离子。
:::

:::callout{kind=insight label="解析"}
**答案**：均裂（homolysis）；异裂（heterolysis）；自由基（碳自由基）

**解析**：均裂是共价键断裂时每个原子各得一个电子，产生自由基；异裂是两个电子都归一个原子，产生正离子和负离子。

:::

:::callout{kind=note label="均裂 vs 异裂"}
| 类型 | 电子分配 | 产物 | 典型反应 |
|------|----------|------|----------|
| 均裂 | 各得1个电子 | 自由基 | 自由基取代、自由基加成 |
| 异裂 | 一方得2个电子 | 离子 | SN1、SN2、E1、E2 |
:::


### 第2题

:::callout{kind=note label="题目"}
烯烃与冷稀碱性KMnO₄反应生成_______（弱氧化），与酸性KMnO₄反应则双键断裂生成_______（强氧化）。
:::

:::callout{kind=insight label="解析"}
**答案**：顺式邻二醇（邻二醇/顺二醇）；羧酸

**解析**：弱氧化条件（冷稀碱性/中性KMnO₄）不断键，烯烃氧化为顺式邻二醇；强氧化条件（酸性KMnO₄加热）双键断裂，含氢一端生成羧酸，完全取代的碳也生成羧酸。

:::

:::callout{kind=insight label="烯烃氧化反应条件对比"}
| 氧化条件 | 产物 | 特点 |
|----------|------|------|
| 冷稀碱性KMnO₄ | 顺式邻二醇 | 双键保留，顺式加成 |
| 酸性KMnO₄（加热） | 羧酸/酮 | 双键断裂 |
| O₃ + Zn/H₂O | 醛/酮 | 双键断裂，还原条件 |
| O₃ + H₂O₂ | 羧酸 | 双键断裂，氧化条件 |
| 稀冷OsO₄ | 顺式邻二醇 | 立体专一性更好 |
:::


### 第3题

:::callout{kind=note label="题目"}
SN2反应为_______分子参与的一步反应，产物构型_______（保持/翻转）；SN1反应经历_______中间体，产物构型_______。
:::

:::callout{kind=insight label="解析"}
**答案**：双（二）；翻转（Walden翻转）；碳正离子；外消旋化（构型保持和翻转各约50%）

**解析**：SN2为双分子亲核取代，亲核试剂从离去基团背面进攻，导致构型翻转；SN1经历碳正离子中间体（平面构型），两面进攻概率相等，得外消旋化产物。

:::

:::callout{kind=note label="SN1 vs SN2 立体化学"}
| 特征 | SN2 | SN1 |
|------|-----|-----|
| 反应级数 | 二级（双分子） | 一级（单分子） |
| 中间体 | 无 | 碳正离子（平面） |
| 立体化学 | 构型翻转（Walden翻转） | 外消旋化（±各50%） |
| 重排 | 无 | 可能有碳正离子重排 |
:::


### 第4题

:::callout{kind=note label="题目"}
醛酮与HCN反应生成_______，水解后可得α-羟基酸，属于_______反应（增碳/减碳）。
:::

:::callout{kind=insight label="解析"}
**答案**：α-羟基腈（氰醇）；增碳

**解析**：HCN中的CN⁻作为亲核试剂进攻醛酮的羰基碳，生成α-羟基腈。该反应使碳链增加一个碳原子，水解后得α-羟基酸。

:::

:::callout{kind=note label="氰醇反应"}
**反应式**：RCHO + HCN → RCH(OH)CN（氰醇）

**特点**：
- 碱催化（CN⁻是亲核试剂）
- 增碳反应（增加1个碳）
- 水解后得α-羟基酸
- 可进一步转化为α-氨基酸

**应用**：Strecker氨基酸合成法的基础
:::


### 第5题

:::callout{kind=note label="题目"}
吡咯氮上孤对电子参与环的共轭，故称_______氮，无碱性；吡啶氮上孤对电子不参与共轭，故称_______氮，有碱性。
:::

:::callout{kind=insight label="解析"}
**答案**：吡咯（型）；吡啶（型）

**解析**：这是课堂重点强调的概念。吡咯型氮的孤对电子参与形成6π芳香体系，不能再接受质子；吡啶型氮的孤对电子处于sp²杂化轨道上，不参与芳香共轭，可以接受质子显碱性。

:::

:::callout{kind=insight label="吡咯型氮 vs 吡啶型氮"}
| 类型 | 孤对电子 | 芳香性 | 碱性 |
|------|----------|--------|------|
| 吡咯型 | 参与6π体系 | 有 | 无（N-H显弱酸性） |
| 吡啶型 | 不参与，在sp²轨道 | 有 | 有 |

**判断方法**：
- 看氮是否参与共轭环的π电子计数
- 吡咯：6π = 4C贡献4π + N贡献2π（孤对电子）
- 吡啶：6π = 5C贡献5π + N贡献1π（p轨道电子），孤对电子在sp²轨道
:::

## 三、判断题（每小题2分，共10分，对的画√，错的画×）

### 第1题

:::callout{kind=note label="题目"}
尿素的合成是有机化学发展史上的重要事件，它推翻了"生命力学说"。（　　）
:::

:::callout{kind=insight label="解析"}
**【答案】√**

解析：1828年维勒用无机物氰酸铵合成了有机物尿素，首次证明有机物可以由无机物合成，动摇并最终推翻了"生命力学说"。这是有机化学发展史上的重要里程碑。

:::

:::callout{kind=note label="有机化学发展史"}
**1828年 维勒合成尿素**：
- 反应：NH₄OCN → CO(NH₂)₂
- 意义：首次由无机物合成有机物，打破"生命力学说"
- 生命力学说：认为有机物只能由生物体产生，含有"生命力"
:::


### 第2题

:::callout{kind=note label="题目"}
马氏规则适用于所有不对称烯烃的亲电加成反应。（　　）
:::

:::callout{kind=insight label="解析"}
**【答案】×**

解析：马氏规则的适用范围是双键碳上连有给电子基团的不对称烯烃。当双键碳上连有强吸电子基团（如-CF₃、-CN、-COOH、-NO₂）时，加成遵循反马氏规则。因此马氏规则不适用于所有不对称烯烃。

:::

:::callout{kind=note label="马氏规则的例外"}
**反马氏规则的情况**：
1. **自由基加成**：HBr + 过氧化物 → 反马氏产物
2. **硼氢化-氧化**：反马氏 + 顺式加成
3. **吸电子基团**：CF₃CH=CH₂ + HBr → CF₃CH₂CH₂Br（反马氏）

**原因**：吸电子基团使邻近碳带部分正电荷，亲核试剂优先进攻该碳
:::


### 第3题

:::callout{kind=note label="题目"}
烯烃与溴的加成是反式加成，经历环溴鎓离子中间体。（　　）
:::

:::callout{kind=insight label="解析"}
**【答案】√**

解析：烯烃与溴加成时，溴分子靠近双键发生动态诱导极化，带正电的溴先进攻双键形成环溴鎓离子中间体（三元环），然后溴负离子从另一侧进攻，最终产物为反式加成产物。

:::

:::callout{kind=note label="环溴鎓离子机理"}
**反应机理**：
1. Br₂靠近双键，发生极化
2. Br⁺进攻双键，形成环溴鎓离子（三元环，Br带正电）
3. Br⁻从背面进攻，开环
4. 结果：反式加成（两个Br在双键两侧）

**立体化学**：反式烯烃 → 内消旋产物；顺式烯烃 → 外消旋产物
:::


### 第4题

:::callout{kind=note label="题目"}
SN1反应速率与亲核试剂的浓度无关。（　　）
:::

:::callout{kind=insight label="解析"}
**【答案】√**

解析：SN1反应的决速步是底物自行电离生成碳正离子，该步骤只涉及底物一种分子，速率方程为v=k[RX]，与亲核试剂浓度无关。亲核试剂在第二步快速结合碳正离子。

:::

:::callout{kind=note label="SN1反应动力学"}
**速率方程**：v = k[RX]（一级反应）

**决速步**：RX → R⁺ + X⁻（慢）
**快步骤**：R⁺ + Nu⁻ → RNu（快）

**特点**：
- 反应速率只与底物浓度有关
- 亲核试剂浓度不影响速率
- 但亲核试剂影响产物比例（与消除反应竞争）
:::


### 第5题

:::callout{kind=note label="题目"}
醛酮与醇反应生成缩醛/缩酮的反应是不可逆的。（　　）
:::

:::callout{kind=insight label="解析"}
**【答案】×**

解析：醛酮与醇生成缩醛/缩酮的反应是可逆反应。在酸催化下可以正向生成缩醛/缩酮（用于保护羰基），也可以在酸性水溶液中逆向水解使羰基游离出来。但醛酮与硫醇生成缩硫酮的反应是不可逆的。

:::

:::callout{kind=note label="缩醛/缩酮的保护与脱保护"}
**生成条件**：醛/酮 + 2当量醇 + 酸催化（干燥条件）

**水解条件**：酸性水溶液（H₃O⁺）

**应用**：
- 保护羰基（对碱、还原剂、亲核试剂稳定）
- 后续反应完成后，酸性水解恢复羰基

**缩硫酮**：与硫醇反应，不可逆，可用Raney Ni脱硫还原为亚甲基
:::

## 四、反应题（每小题5分，共25分。写出主要产物或反应条件）

### 第1题

:::callout{kind=note label="题目"}
CH₃CH=CH₂ + HBr ——(过氧化物)——→ ？
:::

:::callout{kind=insight label="解析"}
**答案**：CH₃CH₂CH₂Br（1-溴丙烷）

**解析**：过氧化物作为自由基引发剂，使反应按自由基加成机理进行。溴自由基先进攻双键端碳（含氢多的碳），生成二级碳自由基（更稳定），再结合H·生成1-溴丙烷。这是反马氏规则产物。反应条件是关键识别点。

:::

:::callout{kind=insight label="反马氏加成机理"}
**Kharasch反应机理**：
1. 过氧化物分解产生RO·自由基
2. RO· + HBr → ROH + Br·
3. Br· + CH₂=CHCH₃ → ·CH₂-CHBr-CH₃（二级自由基，更稳定）
4. ·CH₂-CHBr-CH₃ + HBr → CH₃CH₂CH₂Br + Br·

**关键点**：自由基稳定性决定加成方向
:::


### 第2题

:::callout{kind=note label="题目"}
环己烯 + O₃ → Zn/H₂O → ？
:::

:::callout{kind=insight label="解析"}
**答案**：OHC-(CH₂)₄-CHO（己二醛）

**解析**：臭氧化反应先将烯烃与臭氧生成不稳定的三氧五元环中间体（二级臭氧化物），再用Zn/H₂O还原处理。双键断裂，每个原来双键碳上生成一个羰基。环己烯双键断开后，两端均有氢，生成二醛。

:::

:::callout{kind=note label="臭氧化反应"}
**反应条件与产物**：
| 后处理 | 产物 |
|--------|------|
| Zn/H₂O 或 (CH₃)₂S | 醛/酮 |
| H₂O₂ | 羧酸（醛被进一步氧化） |

**应用**：
- 由烯烃制备醛/酮
- 由产物推断烯烃结构（逆推双键位置）
:::


### 第3题

:::callout{kind=note label="题目"}
CH₃CH₂CH₂Br + NaCN ——(SN2)——→ ？
:::

:::callout{kind=insight label="解析"}
**答案**：CH₃CH₂CH₂CN（丁腈）+ NaBr

**解析**：CN⁻是强亲核试剂，从溴原子的背面进攻伯碳，一步完成取代。该反应使碳链增长一个碳原子（增碳反应），丁腈水解可得丁酸。这是合成题中常用的加碳方法。

:::

:::callout{kind=note label="氰基在合成中的应用"}
**增碳反应**：R-X + NaCN → R-CN（增加1个碳）

**后续转化**：
- R-CN + H₃O⁺, Δ → R-COOH（羧酸）
- R-CN + LiAlH₄ 或 H₂/Ni → R-CH₂NH₂（伯胺）

**应用**：
- 增加一个碳原子
- 制备羧酸、胺、醛等
:::


### 第4题

:::callout{kind=note label="题目"}
CH₃CHO + CH₃MgBr → H₃O⁺ → ？
:::

:::callout{kind=insight label="解析"}
**答案**：CH₃CH(OH)CH₃（2-丙醇/异丙醇）

**解析**：格氏试剂（CH₃MgBr）中的碳负离子进攻醛的羰基碳，发生亲核加成反应。加成产物经酸性水解（质子化）后得到仲醇。醛与格氏试剂反应生成仲醇，酮与格氏试剂反应生成叔醇。

:::

:::callout{kind=insight label="格氏试剂反应"}
**格氏试剂**：RMgX（R = 烷基，X = Br/Cl/I）

**与醛酮反应**：
- 甲醛 → 伯醇（增加1个碳）
- 其他醛 → 仲醇
- 酮 → 叔醇

**注意事项**：
- 无水无氧条件（格氏试剂遇水、醇、酸分解）
- 溶剂：无水乙醚或THF
- 后处理：H₃O⁺水解
:::


### 第5题

:::callout{kind=note label="题目"}
呋喃 + Br₂ ——→ ？（写出主要产物及取代位置）
:::

:::callout{kind=insight label="解析"}
**答案**：2-溴呋喃（α-溴代呋喃）

**解析**：五元杂环化合物（吡咯、呋喃、噻吩）的亲电取代反应比苯环容易，主要产物在α位（2位）。呋喃溴代在α位发生，生成2-溴呋喃。老师课堂上明确讲到"呋喃溴代在α位"。

:::

:::callout{kind=note label="五元杂环亲电取代定位"}
**活性顺序**：吡咯 > 呋喃 > 噻吩 > 苯

**定位规则**：
- α位（2位）优先于β位（3位）
- 原因：α位取代的σ络合物共振稳定更多

**反应条件**：
- 吡咯：非常活泼，不需催化剂，室温即可
- 呋喃：较活泼，低温温和条件
- 噻吩：较稳定，需要催化剂
:::

## 五、合成题（每小题7.5分，共15分。写出合成路线及所用试剂和条件）

### 第1题

:::callout{kind=note label="题目"}
以丙烯为原料合成1-丙醇。（提示：需得到反马氏规则产物）
:::

:::callout{kind=insight label="解析"}
**路线**：CH₂=CHCH₃ →(B₂H₆)→ (CH₃CH₂CH₂)₃B →(H₂O₂/NaOH)→ CH₃CH₂CH₂OH

**解析**：

① **硼氢化反应**：丙烯与乙硼烷（B₂H₆）加成，硼原子进攻含氢较多的碳（反马氏），顺式加成。

② **氧化反应**：三烷基硼在碱性双氧水中氧化，硼被羟基取代。

**总结**：硼氢化-氧化反应得到反马氏规则的伯醇。这与酸催化水合得到马氏产物（2-丙醇）互补。

**评分标准**：路线正确4分，试剂条件正确2分，解释/说明1.5分。

:::

:::callout{kind=insight label="由烯烃制备醇的方法对比"}
| 方法 | 产物 | 特点 |
|------|------|------|
| 酸催化水合 | 马氏产物（仲醇/叔醇） | 碳正离子重排可能 |
| 硼氢化-氧化 | 反马氏产物（伯醇） | 顺式加成，无重排 |
| 羟汞化-脱汞 | 马氏产物 | 不发生重排 |
| Oxymercuration-demercuration | 马氏产物 | 温和条件 |
:::


### 第2题

:::callout{kind=note label="题目"}
以甲苯为原料合成苯甲酸。
:::

:::callout{kind=insight label="解析"}
**路线**：C₆H₅CH₃ →(KMnO₄/H⁺, 加热)→ C₆H₅COOH

**解析**：

甲苯苯环上的烷基侧链在酸性KMnO₄强氧化条件下被氧化成羧基，无论烷基侧链多长都氧化成-COOH。这是芳香烃苯环侧链氧化的典型反应。

也可用：C₆H₅CH₃ →(CrO₃/H₂SO₄)→ C₆H₅COOH

**评分标准**：路线正确4分，条件正确2分，解释1.5分。

:::

:::callout{kind=note label="苯环侧链氧化"}
**反应特点**：
- 无论侧链多长，最终都氧化为-COOH
- 需要α-H（与苯环直接相连的碳上有H）
- 强氧化条件：KMnO₄/H⁺、CrO₃/H₂SO₄、K₂Cr₂O₇/H⁺

**例外**：
- 叔丁基苯（无α-H）不被氧化
- 硝基苯不被氧化（硝基钝化）
:::

## 六、推断题（10分）

某化合物A的分子式为C₅H₁₀，A经臭氧化-还原水解后得到丙酮和乙醛两种产物。

**(1) 推断A的结构式（4分）**

**(2) A与HBr加成的主要产物是什么？说明理由。（3分）**

**(3) A经冷稀碱性KMnO₄氧化的产物是什么？（3分）**

:::callout{kind=insight label="解析"}
**(1) A的结构式**

A的结构式为：CH₃C(=CH)CH₃ → 即2-甲基-2-丁烯（(CH₃)₂C=CHCH₃）

**推断过程**：A的分子式C₅H₁₀，不饱和度=(2×5+2-10)/2=1，含一个双键。臭氧化-还原水解后得到丙酮(CH₃COCH₃)和乙醛(CH₃CHO)，说明双键从中间断开，左边生成丙酮（双键碳连两个甲基），右边生成乙醛（双键碳连一个甲基和一个氢）。因此A为2-甲基-2-丁烯。

（4分：结构式2分，推断过程2分）

**(2) A与HBr加成的主要产物**

主要产物为2-溴-2-甲基丁烷 (CH₃)₂CBrCH₂CH₃

**理由**：遵循马氏规则，H加在含氢较多的双键碳上（3位碳），Br加在含氢较少的双键碳上（2位碳），生成更稳定的三级碳正离子中间体。

（3分：产物1.5分，理由1.5分）

**(3) A经冷稀碱性KMnO₄氧化的产物**

(CH₃)₂C(OH)-CH(OH)CH₃（2-甲基-2,3-丁二醇，顺式邻二醇）

弱氧化条件下双键不断裂，两个羟基加到双键两侧碳上，且为顺式加成。

（3分：产物2分，解释1分）

:::

:::callout{kind=insight label="推断题解题策略"}
**由臭氧化产物推断烯烃结构**：
- 将产物中的C=O还原为C=C
- 两个羰基碳原来是双键的两个碳
- 注意：O₃/Zn,H₂O得醛/酮；O₃/H₂O₂得羧酸

**不饱和度计算**：
Ω = (2C + 2 + N - H - X) / 2

**常见推断线索**：
- 能使Br₂/CCl₄褪色 → 含C=C或C≡C
- 能使KMnO₄褪色 → 含不饱和键或易被氧化基团
- 与Na反应放H₂ → 含-OH或-COOH
- 银镜反应 → 含-CHO
:::
