# 1.3 酸碱理论与有机反应基础

:::insight{label=一切有机反应的起点：旧键怎么断}
有机化学既然是化学，它的实质就是**旧键的断裂与新键的生成**。那么旧键到底是怎么断的？一根共价键由一对共用电子组成，断裂方式自然要看这对电子怎么分。沿着这条线索，我们会依次遇到三种活性中间体、各类官能团，以及两套酸碱理论和共振论——它们是后面所有反应机理的"语法"。这一节就把这套语法打牢。
:::

## 一、共价键的断裂：均裂与异裂

:::definition{label=均裂与异裂}
一根共价键的两个共用电子，断裂时有两种分配方式：

- **均裂**：把键打断时，两个电子**各随一个原子**。生成的带单电子物种叫**自由基**（其实质就是一个中性原子）。一个共价单键均裂产生**两个自由基**。
- **异裂**：把键打断时，这**一对电子全部跟随某一个原子**。以碳为中心看有两种情况：
  - 电子对**全部离开碳**：生成**碳正离子**，另一原子带走电子对成负离子；
  - 电子对**全部被碳留下**：生成**碳负离子**，另一原子以正离子形式离开。
:::

所以共价键的断裂可归为"两大类、三种"：均裂只有一种（一刀切，各拿一个电子）；异裂有两种（生成碳正离子或碳负离子）。自由基、碳正离子、碳负离子都是反应过程中生成的**活性中间体**，后续许多反应都通过它们进行。

## 二、三种活性中间体的结构：杂化的关键例子

:::insight{label=敲掉一根键，剩下的键会"舒展"}
以甲烷为例。甲烷是高度对称的正四面体，键角 109.5°——这是四根键上的电子互相排斥、"谁也不打架"达到的妥协平衡。现在把其中一根键敲断，剩下三根键还会乖乖保持 109.5° 吗？打个比方：四人间宿舍住满时各睡各的床；要是有一个人长期不在，那张床慢慢会被其他人的东西堆满。键也一样——少了一根键的约束，剩下的键会重新舒展到新的平衡。
:::

:::theorem{label=碳自由基、碳正、碳负的构型}
对甲基（$\ce{CH3}$）类活性中间体：

- **碳自由基**：均裂后该碳变成中性原子、此处无电荷，剩下三根键舒展成**平面型**，属 **sp² 构型**。
- **碳正离子**：该处不仅没有负电荷甚至带正电，也变成**平面三角形**，属 **sp² 构型**。
- **碳负离子**：异裂时是把另一原子以正电荷形式拉走，**那一对电子仍留在这个碳上**——相当于那根键其实没真正断开，所以碳负离子仍保持原来的 **sp³ 构型**。

一句话记住：**碳自由基和碳正离子都是平面型（sp²），碳负离子仍是 sp³ 型。**
:::

:::pitfall{label=易错点：sp²/sp³ 由结构决定，不由"成了什么键"决定}
不要以为"成双键才是 sp²"。判断杂化看的是**几何结构**：只要这个碳成了平面三角形，它就是 sp² 碳——碳正离子、碳自由基没有双键却也是 sp²。反过来，碳负离子虽是活性中间体，却因电子对仍在、保持四面体而是 sp³。
:::

## 三、官能团分类总览

下面这些官能团是后续每一章的主角，先打个照面：

:::definition{label=主要官能团一览}
- **烷烃**：无官能团，所有键饱和。
- **烯烃**：碳碳双键（烯键）。
- **炔烃**：碳碳三键（炔键）。
- **卤代烃**：官能团是卤素 / 碳卤键，是本课的一个"枢纽"——许多转化都以卤代烃为中心发生。
- **醇与酚**：都带羟基 $\ce{-OH}$；直接连在苯环上的叫**酚**，连其他结构的叫**醇**（所以苯环连羟基叫苯酚而非苯醇）。
- **硫醇与硫酚**：把醇/酚的氧换成硫；作为基团时这个集团叫**巯基** $\ce{-SH}$。
- **醚**：含醚键。
- **醛**：含醛基。
- **酮**：羰基两边都连碳。
- **羧酸**：官能团是羧基。
- **胺**：含氨基 $\ce{-NH2}$ 的有机胺。（注意：作取代基/集团时写气字头的"氨基"，作一类化合物时写肉字旁的"胺"。）
:::

## 四、布朗斯特（质子）酸碱理论

高中认为"能电离出 $\ce{H+}$ 的是酸、能电离出 $\ce{OH-}$ 的是碱"，碱被局限为氢氧根，太狭隘。大学要把碱的范围大大拓展。

:::definition{label=布朗斯特（质子）酸碱理论}
- **酸**：凡能**给出质子**的物质（与高中一致）。
- **碱**：凡能**接受质子**的物质（不再局限于氢氧根——质子缺电子，只要有共用电子就能结合）。
- 酸给出质子后生成它的**共轭碱**；碱接受质子后生成它的**共轭酸**：

$$
\ce{HA + B <=> A- + HB+}
$$

其中 $\ce{A-}$ 是 $\ce{HA}$ 的共轭碱，$\ce{HB+}$ 是 $\ce{B}$ 的共轭酸。
:::

:::insight{label=酸碱是"角色"，不是"身份"——水的两性}
在质子酸碱理论里，**不能说某物质"是酸"或"是碱"，必须放到它参与的具体反应里看**它给出还是接受质子。最典型的就是水：
- $\ce{HCl + H2O -> H3O+ + Cl-}$：水接受质子，**水作碱**，$\ce{H3O+}$ 是共轭酸。
- $\ce{CH3COOH + OH- -> CH3COO- + H2O}$：乙酸是酸、$\ce{OH-}$ 是碱，生成的水是**共轭酸**（即水扮演酸）。
- $\ce{H2O + NH3 -> NH4+ + OH-}$：水给出质子，**水作酸**，$\ce{NH4+}$ 是共轭酸、$\ce{OH-}$ 是共轭碱。

同一个水，一会儿是碱一会儿是酸——这就是水的两性。所以判断题"根据布朗斯特酸碱理论，水是酸"**一定是错的**，因为脱离了具体反应。
:::

强度关系是此消彼长的：酸给质子能力越强、酸性越强，其共轭碱碱性越弱；碱接受质子能力越强、碱性越强，其共轭酸酸性越弱。

:::example{label=例1：同一个丙酮，既能作碱又能作酸}
**问题：** 丙酮 $\ce{(CH3)2CO}$ 在下面两个反应里各扮演什么角色？

$$
\ce{(CH3)2CO + H2SO4 -> (CH3)2C=OH+ + HSO4-}
$$
$$
\ce{(CH3)2CO + CH3O- -> {}^-CH2COCH3 + CH3OH}
$$

**解答：**
- 上式：丙酮**结合了质子**（羰基氧被质子化），所以**丙酮作碱**，硫酸是酸；质子化丙酮是共轭酸，$\ce{HSO4-}$ 是共轭碱。
- 下式：丙酮**给出了质子**（甲基上一个氢被拔走，生成碳负离子），所以**丙酮作酸**，甲氧基负离子 $\ce{CH3O-}$ 是碱；甲醇是共轭酸。

**为什么丙酮能两面派？** 关键在羰基的极性。碳氧双键中 π 键比 σ 键更易变形，电子云整体朝电负性大的氧流动，于是**氧上富集负电荷、羰基碳带部分正电**。
- 氧富集负电 → 质子（带正电）来了优先粘到氧上 → 酸性条件下**羰基氧被质子化，丙酮作碱**。
- 与羰基相邻的碳叫 **α-碳**，其上的氢叫 **α-氢**；α 碳氢键与碳氧双键 π 电子云有一定重叠（**超共轭**），羰基的极性诱导使这根 α-碳氢键比别的碳氢键更易极化。遇到甲氧基负离子这种强碱时，α-氢就被"一叫就走"→ 碱性条件下**丙酮作酸**。
:::

## 五、路易斯（电子）酸碱理论

有机转化中更常用的是以电子对得失来定义的路易斯理论。

:::definition{label=路易斯（电子）酸碱理论}
- **路易斯酸**：能**接受电子对**的物质（有空轨道）。
- **路易斯碱**：能**给出电子对**的物质（有孤对电子或可共享的 π 电子）。
- 二者结合不再生成共轭酸碱，而是生成一个**酸碱加合物**（形成配位键）。

例如乙醚的氧有孤对电子、三氟化硼的硼有空轨道：

$$
\ce{(C2H5)2O + BF3 -> (C2H5)2O\bond{->}BF3}
$$

此处乙醚是路易斯碱、$\ce{BF3}$ 是路易斯酸，生成酸碱加合物。
:::

:::theorem{label=路易斯酸、碱的范围}
**路易斯酸**（确定的，不必看具体反应）：
1. 能接受电子的分子，如 $\ce{BF3}$、$\ce{AlCl3}$、氯化锡、$\ce{ZnCl2}$、$\ce{FeCl3}$（含硼或过渡金属，有空轨道）；
2. 金属离子（本身有空轨道）；
3. 带正电的离子，如碳正离子、溴正离子、硝基正离子、$\ce{H+}$。

**路易斯碱**：
1. 含未成键孤对电子的分子，如含氮/氧/硫的氨、醇、醚、醛、酮及含巯基化合物；
2. 带负电的负离子，如溴离子、氢氧根、烷氧基负离子、巯基负离子、$\ce{R-}$（碳负离子）；
3. 烯烃或芳香化合物（有可流动的 π 电子，遇强路易斯酸时可共享出去作碱）。
:::

:::pitfall{label=易错点：两套理论的关键区别}
- **路易斯酸碱是"确定"的，布朗斯特酸碱要看反应。** 路易斯理论由电子对决定，什么是酸什么是碱明确，不必代入具体反应；而布朗斯特理论中同一物质（如水、丙酮）可酸可碱。
- **路易斯碱与布朗斯特碱差别不大，但路易斯酸范围大得多。** 布朗斯特酸必须含质子，而许多金属离子、金属盐（不含可电离质子）都能作路易斯酸。
- **产物不同**：布朗斯特酸碱反应生成共轭酸+共轭碱；路易斯酸碱反应生成一个**加合物**（新物质）。
:::

## 六、共振论与极限式

:::insight{label=为什么要写"极限式"}
苯环本是一个均匀的大 π 键，并没有哪两个电子专属哪两个碳。可价键理论要求"一根键必须是两个电子"。如果硬要在某一瞬间把高速运转的大 π 键"喊停"、并把每两个电子都分配到两个碳上，就会出现两种极限情况（凯库勒式中 1,2 号碳之间一种是双键、一种是单键）。这两种写法都代表苯环，它们之间通过**共振**变换。
:::

:::definition{label=共振论与极限式（共振结构）}
**共振结构（极限式）**表达的是某一分子、离子或自由基**真实电子结构**所采用的两个或两个以上的经典共价键结构。它是一种**化学书写**，**并不代表分子的真实情况**，而是代表真实情况的某种"极限"。为符合价键理论（一根键=两个电子），我们用这些极限式来描述，它们之间的变换关系就叫**共振**。

> 注意：共振论不要求重点掌握，但后续会用它解释一些现象。
:::

:::example{label=例2：乙酸根负离子的两个等价共振式}
**问题：** 乙酸失去质子成乙酸根 $\ce{CH3COO-}$，那个负电荷到底在哪个氧上？

$$
\ce{CH3COO- <-> CH3COO-}
$$

**解答：** 我们书写时习惯把负电荷画在某一个氧上，但它完全可以共振到另一个氧上——一个氧的电子成键，就把另一对电子逼到另一个氧上。这两个共振式**结构相同、能量相同**，对共振杂化体的贡献也相同。

**真实情况：** 拔掉羧酸质子后，并不存在"一个双键氧、一个单键氧"，**每个氧各占 1/2 电荷、两个氧完全等价**，负电荷在两氧间快速摆动。实验也证明乙酸根两个碳氧键键长相等、电荷分布相等。只因 1/2 电荷不符合价键书写，才写成两个极限式之一。
:::

:::example{label=例3：质子化丙酮——两个不等价的共振式}
**问题：** 丙酮羰基氧被质子化后（氧带正电），是否也存在共振？两个共振式地位相同吗？

$$
\ce{(CH3)2C=OH+ <-> (CH3)2C+-OH}
$$

**解答：** 会发生共振——双键 π 电子可以把正电荷"中和"、让正电荷转移到碳上。但与苯环、乙酸根不同，这两个共振式**结构不同、能量不同**，因而对共振杂化体的**贡献也不同**：以**碳氧双键形式存在的结构较稳定，贡献较大**。

**规律：** 当两个共振极限式结构相同（苯环、乙酸根）时，能量相同、贡献相同；当两个极限式结构不同（质子化丙酮）时，能量不同、**越稳定的那个贡献越大**。
:::

:::pitfall{label=易错点：共振 ≠ 真实存在的两种分子}
- **极限式是书写，不是真实分子。** 乙酸根不是"一半时间长这样、一半时间长那样"地来回跳，而是真实电子结构是介于两者之间的杂化体；写成两个式子只是受价键理论书写所限。
- **箭头要分清。** 共振用双箭头 $\ce{<->}$ 表示同一物种的不同极限式，**不是**化学平衡（$\ce{<=>}$）、更不是反应（$\ce{->}$）。
- **等价 vs 不等价。** 不要把所有共振式都当成"贡献相同"——只有结构相同的极限式才等贡献；结构不同时，稳定者贡献大。
:::

:::note{label=本节小结}
- 共价键断裂分**两大类三种**：**均裂**→两个自由基；**异裂**→碳正离子或碳负离子。三者都是**活性中间体**。
- 结构与杂化：**碳自由基、碳正离子为平面型 sp²；碳负离子仍为 sp³**（电子对仍在，键未真断）。杂化由几何结构决定。
- 官能团总览：烷/烯/炔/卤代烃/醇·酚/硫醇·硫酚/醚/醛/酮/羧酸/胺；酚=连苯环的羟基，巯基 $\ce{-SH}$。
- **布朗斯特酸碱**：给质子=酸、接受质子=碱；生成共轭碱/共轭酸；酸碱是"角色"非"身份"，**水具两性**，丙酮可酸可碱（羰基极性 + α-氢超共轭）。
- **路易斯酸碱**：接受电子对=酸、给出电子对=碱，生成**加合物**；酸碱身份确定；路易斯酸范围远大于布朗斯特酸。
- **共振论**：极限式是符合价键理论的书写、不代表真实；等价极限式（乙酸根、苯环）贡献相同，不等价极限式（质子化丙酮）中**越稳定者贡献越大**。
:::
