:::insight{label=从"诱导效应衰减快"到"共轭效应贯穿全局"}
上一节我们学到，诱导效应沿 σ 键传递，3 个键以后就基本可以忽略。但有机化学中有许多现象无法用诱导效应解释：

- 苯酚 $\ce{C6H5OH}$ 的 $\text{p}K_a = 9.99$，而环己醇 $\ce{C6H11OH}$ 的 $\text{p}K_a = 17$。两者都有 $\ce{-OH}$，为什么苯酚酸性比环己醇强近 1 亿倍？
- 对硝基苯酚 $\text{p}K_a = 7.16$，比苯酚强 70 倍。$\ce{-NO2}$ 在对位，距 $\ce{-OH}$ 隔了 4 个键，按诱导效应应该影响很弱，为什么实际影响这么大？

答案就是**共轭效应**（conjugated effect）：通过 π 键的电子离域，效应可以**贯穿整个共轭体系而不衰减**。$\ce{-NO2}$ 虽然离 $\ce{-OH}$ 远，但通过苯环的 π 体系，它的吸电子效应可以"传递"到 $\ce{-O^-}$，稳定酚氧负离子，增强酸性。

本节我们就来学习共轭效应的定义、特点、分类（π-π、p-π、p-p 共轭），以及它与诱导效应的协同与竞争。
:::

## 共轭效应的定义

:::definition{label=共轭体系与共轭效应}
**共轭体系**（conjugated system）：单双键交替出现的体系（或类似结构）。

**共轭效应**（conjugated effect，符号 C）：在共轭体系中，由于原子间的相互影响使体系内的 π 电子分布发生变化的电子效应。

共轭效应的本质是**电子的离域**（delocalization）：两个 π 键以单键相连接后，一个经典 π 轨道中的电子会运动到另一个经典 π 轨道中去。π 电子的离域能够**降低体系的能量**，使分子更稳定。

典型共轭体系：
- 1,3-丁二烯 $\ce{CH2=CH-CH=CH2}$（π-π 共轭）
- 苯 $\ce{C6H6}$（π-π 共轭，闭合环状）
- 苯酚 $\ce{C6H5OH}$（p-π 共轭）
- 苯胺 $\ce{C6H5NH2}$（p-π 共轭）
:::

:::theorem{label=共轭效应的键长平均化}
共轭效应的一个重要实验证据是**键长平均化**：

- 1,3-丁二烯中，C2-C3 单键的键长为 147 pm（正常 C—C 单键 154 pm），变短了
- C1=C2 和 C3=C4 双键的键长为 137 pm（正常 C=C 双键 134 pm），变长了
- 单键变短、双键变长 → 键长趋向"平均"

这说明 π 电子不再局域于"双键"位置，而是离域到整个共轭体系，使单键和双键的差别减小。
:::

:::insight{label=共轭效应 vs 诱导效应的根本区别**
| 特性 | 诱导效应（I） | 共轭效应（C） |
|------|--------------|--------------|
| 传递方式 | 沿 σ 键 | 沿 π 键（或 p 轨道） |
| 衰减 | 迅速衰减（3 个键后可忽略） | **不衰减**，贯穿整个共轭体系 |
| 影响范围 | 局部 | 整个共轭体系 |
| 电子运动 | 电子云"偏移"（不离域） | 电子"离域"（在整个共轭体系运动） |

共轭效应的"不衰减"特性，使它能影响远距离的原子——这是诱导效应做不到的。
:::

## 共轭效应引起电子对偏移——苯酚的例子

:::theorem{label=苯酚的 p-π 共轭}
苯酚 $\ce{C6H5OH}$ 中，O 的 p 轨道（含孤对电子）与苯环的 π 体系发生 **p-π 共轭**：

- O 的孤对电子离域到苯环 π 体系
- 这使 $\ce{-OH}$ 的电子云向苯环偏移
- 同时使 O—H 键的电子云向 O 偏移
- H 更易以 $\ce{H+}$ 形式离去 → 酸性增强

对比：
- 苯酚 $\text{p}K_a = 9.99$（p-π 共轭 + 诱导效应）
- 环己醇 $\text{p}K_a = 17$（只有诱导效应，无共轭）

苯酚酸性比环己醇强近 1 亿倍，主要归因于 p-π 共轭稳定了酚氧负离子（$\ce{C6H5O^-}$ 的负电荷通过共轭分散到整个苯环）。
:::

## 吸电子共轭效应（-C）与给电子共轭效应（+C）

:::definition{label=-C 与 +C}
**吸电子共轭效应（-C）**：能够降低共轭体系 π 电子云密度的基团。常见 -C 基团：

$$
\ce{-NO2},\ \ce{-CN},\ \ce{-COOH},\ \ce{-CHO},\ \ce{-COR}
$$

这些基团都有 π 键（或 π 体系），能从共轭体系"拉"电子。

**给电子共轭效应（+C）**：能够增加共轭体系 π 电子云密度的基团。常见 +C 基团：

$$
\ce{-NR2},\ \ce{-OR},\ \ce{-OH},\ \ce{-F},\ \ce{-Cl},\ \ce{-Br},\ \ce{-I}
$$

这些基团都有孤对电子（p 轨道），可以向共轭体系"给"电子（p-π 共轭）。
:::

:::insight{label=为什么卤素是 +C 但 -I**
卤素（$\ce{-F}$、$\ce{-Cl}$、$\ce{-Br}$、$\ce{-I}$）是一个有趣的"矛盾"基团：

- **-I（吸电子诱导）**：卤素电负性大，吸引 σ 键电子云 → -I
- **+C（给电子共轭）**：卤素有孤对电子（p 轨道），可以向共轭体系给电子 → +C

所以卤素同时具有 -I 和 +C，方向相反！总效应取决于哪个更强：

- 对 **$\ce{-OH}$** 的酸性影响：$\ce{-F}$ 的 -I 强（电负性大），+C 也强（2p-2p 共轭好），但 -I 略占优势 → 总体吸电子（弱）
- $\ce{-Cl}$ 的 -I 弱于 F，+C 也弱（3p-2p 共轭差，轨道大小不匹配）→ 总体吸电子（弱）

这就是为什么对氯苯酚的酸性比苯酚强（$\text{p}K_a$ 9.38 vs 9.99）——Cl 的 -I 主导，+C 较弱。
:::

:::theorem{label=共轭效应的特点}
1. **只能在共轭体系中传递**：没有共轭体系就没有共轭效应
2. **无论共轭体系有多大，共轭效应能够贯穿整个共轭体系**（不衰减）

这是共轭效应与诱导效应最本质的区别。
:::

## 共轭效应对苯酚酸性的影响——硝基苯酚的例子

:::theorem{label=硝基苯酚的酸性}
不同位置硝基苯酚的 $\text{p}K_a$：

| 化合物 | $\text{p}K_a$ | 与苯酚比较 |
|--------|---------------|-----------|
| 苯酚 | 9.99 | 标准 |
| 邻硝基苯酚 | 7.23 | 强 60 倍 |
| 间硝基苯酚 | 8.40 | 强 6 倍 |
| 对硝基苯酚 | 7.16 | 强 70 倍 |

**分析**：
- $\ce{-NO2}$ 是吸电子基（-I 和 -C 都有）
- **邻位**：-I 强（距离近）+ -C 强（共轭贯穿）→ 酸性增强显著
- **间位**：-I 较弱（距离远）+ -C 不能直接作用（间位无共轭路径）→ 酸性增强较弱
- **对位**：-I 弱（距离最远）+ -C 强（共轭贯穿整个苯环）→ 酸性增强显著

间位的 -C 不能直接作用，因为间位没有直接的共轭路径连接 $\ce{-NO2}$ 和 $\ce{-O^-}$。这是"共轭效应只在共轭体系中传递"的体现。
:::

:::insight{label=邻位 vs 对位硝基苯酚**
邻硝基苯酚 $\text{p}K_a = 7.23$，对硝基苯酚 $\text{p}K_a = 7.16$，对位酸性略强。为什么？

- **邻位**：-I 最强（距离最近），但 -C 较弱（邻位有空间位阻，$\ce{-NO2}$ 与苯环不能完全共平面，共轭程度降低）
- **对位**：-I 最弱（距离最远），但 -C 最强（$\ce{-NO2}$ 与苯环完全共平面，共轭充分）

对位的 -C 充分发挥，弥补了 -I 的减弱，最终酸性略强于邻位。这完美展示了 -I 和 -C 的"分工"。
:::

:::example{label=硝基苯酚酸性分析}
**题目**：解释为什么对硝基苯酚比间硝基苯酚酸性强。

**分析**：$\ce{-NO2}$ 在对位和间位时，-I 都较弱（距离远），主要差别在 -C。

**解**：
- **对硝基苯酚**：$\ce{-NO2}$ 与 $\ce{-O^-}$ 通过苯环 π 体系共轭（4 个碳的共轭路径），$\ce{-NO2}$ 的 -C 可以稳定 $\ce{-O^-}$ → 酸性强（$\text{p}K_a = 7.16$）
- **间硝基苯酚**：$\ce{-NO2}$ 与 $\ce{-O^-}$ 之间没有直接的共轭路径（间位阻断），-C 不能直接作用 → 只有 -I 影响 → 酸性较弱（$\text{p}K_a = 8.40$）

差异说明：**共轭效应只在共轭体系中传递**，间位无共轭路径，所以 -C 失效。
:::

## 氯代苯酚——-I 与 +C 的竞争

:::theorem{label=氯代苯酚的酸性}
不同位置氯代苯酚的 $\text{p}K_a$：

| 化合物 | $\text{p}K_a$ |
|--------|---------------|
| 苯酚 | 9.99 |
| 邻氯苯酚 | 8.48 |
| 间氯苯酚 | 9.02 |
| 对氯苯酚 | 9.38 |

**分析**：
- $\ce{-Cl}$ 同时具有 -I（吸电子）和 +C（给电子）
- 氯代苯酚酸性比苯酚强，说明 **-I 主导**（+C 不足以抵消 -I）
- **邻位 -I 最强**（距离最近）→ 酸性最强（8.48）
- **间位 -I 次之** → 酸性次之（9.02）
- **对位 -I 最弱**，且 +C 显现 → 酸性最弱（9.38）

注意：对位氯苯酚酸性仍强于苯酚，说明即使 +C 显现，-I 仍占主导。
:::

:::insight{label=为什么 Cl 的 +C 比 -I 弱**
Cl 的 +C（3p 轨道与苯环 2p 轨道共轭）比较弱，因为 3p 轨道比 2p 轨道大，重叠效率低（轨道大小不匹配）。而 F 的 +C 较强（2p-2p 重叠好），但仍弱于 -I。

所以卤素的总体效应都是吸电子（-I 主导），只是程度不同：
- $\ce{-F}$：-I 强，+C 也强，但 -I > +C → 总体弱吸电子
- $\ce{-Cl}$：-I 中等，+C 弱 → 总体弱吸电子
- $\ce{-Br}$、$\ce{-I}$：-I 更弱，+C 更弱 → 总体弱吸电子
:::

## 总电子效应

:::theorem{label=总电子效应}
**总电子效应 = 诱导效应 + 共轭效应**（在共轭体系中）

判断基团是给电子还是吸电子，要看总结果：

- 大部分取代基的诱导效应和共轭效应**一致**（如 $\ce{-NO2}$：-I + -C = 强吸电子）
- 有些取代基的诱导效应与共轭效应**相反**（如 $\ce{-Cl}$：-I + +C），这时要看总的结果
- 总结果是给电子的 → 给电子基
- 总结果是吸电子的 → 吸电子基

**注意**：共轭效应**只在共轭体系中存在**！非共轭体系中只考虑诱导效应。
:::

:::theorem{label=常见基团的总电子效应（共轭体系中）}
**给电子基团**（总 +E）：

$$
\ce{-OH},\ \ce{-N(CH3)2},\ \ce{-NH2},\ \ce{-OR},\ \ce{-R}
$$

**吸电子基团**（总 -E）：

$$
\ce{-NO2},\ \ce{-CN},\ \text{含羰基基团（醛基、酮基、酰基、酯基、羧基）},\ \ce{-X}
$$

注意：
- $\ce{-OH}$、$\ce{-NH2}$、$\ce{-OR}$：+C > -I → 总给电子
- $\ce{-R}$（烷基）：+I，无共轭 → 总给电子
- $\ce{-X}$（卤素）：-I > +C → 总吸电子（弱）
- $\ce{-NO2}$、$\ce{-CN}$、羰基：-I + -C → 总吸电子（强）
:::

:::example{label=判断总电子效应}
**题目**：判断 $\ce{-OH}$、$\ce{-Cl}$、$\ce{-NO2}$、$\ce{-CH3}$ 在共轭体系中的总电子效应。

**解**：
- **$\ce{-OH}$**：-I（O 电负性大）+ +C（O 孤对电子给电子）。+C > -I → **总给电子**
- **$\ce{-Cl}$**：-I（Cl 电负性大）+ +C（Cl 孤对电子给电子，但 3p-2p 重叠差）。-I > +C → **总吸电子（弱）**
- **$\ce{-NO2}$**：-I（N 电负性大）+ -C（$\ce{N=O}$ π 键吸电子）。-I + -C 同向 → **总吸电子（强）**
- **$\ce{-CH3}$**：+I（烷基给电子），无共轭 → **总给电子**

规律：含孤对电子的杂原子基团（$\ce{-OH}$、$\ce{-NH2}$、$\ce{-OR}$）+C 主导 → 给电子；含 π 键的基团（$\ce{-NO2}$、$\ce{-CN}$、羰基）-C 主导 → 吸电子；卤素特殊，-I 主导 → 弱吸电子。
:::

## 几种特征的共轭体系

:::definition{label=三种特征共轭体系}
**1. π-π 共轭**：两个或两个以上的双键（或叁键）以单键相连时所发生的 π 电子离域作用。

典型例子：
- 1,3-丁二烯 $\ce{CH2=CH-CH=CH2}$
- 苯乙烯 $\ce{C6H5-CH=CH2}$
- 丙烯腈 $\ce{CH2=CH-CN}$
- 丙烯酸甲酯 $\ce{CH2=CH-COOR}$

**2. p-π 共轭**：π 轨道与 p 轨道之间的共轭离域作用。

两种情况：
- (a) O、N、S、卤素（X）直接与简单 π 体系相连时，p 电子和 π 电子在 p 轨道和 π 轨道间离域
  - 苯胺 $\ce{C6H5-NH2}$
  - 苯甲醚 $\ce{C6H5-OCH3}$
  - 硫酚 $\ce{C6H5-SH}$
- (b) 烯丙基正离子 $\ce{CH2=CH-CH2+}$、苄基正离子 $\ce{C6H5-CH2+}$，或烯丙基自由基、苄基自由基可形成 p-π 共轭（这种情况在碳正离子或自由基化学中很重要）

**3. p-p 共轭**：一个缺电子的 p 轨道与含有一对电子（n 电子）的 p 轨道之间的共轭离域作用。杂原子 O、N、S 或卤素（X）与碳正离子相连可形成 p-p 共轭。
:::

:::insight{label=为什么烯丙基/苄基正离子特别稳定**
烯丙基正离子 $\ce{CH2=CH-CH2+}$ 和苄基正离子 $\ce{C6H5-CH2+}$ 比普通碳正离子稳定得多，原因就是 p-π 共轭：

- 烯丙基正离子：$\ce{CH2=CH-CH2+}$ 中，正电荷碳的空 p 轨道与 $\ce{C=C}$ 的 π 键共轭，正电荷离域到 $\ce{C=C}$ 的两个碳上 → 电荷分散 → 稳定
- 苄基正离子：$\ce{C6H5-CH2+}$ 中，正电荷碳的空 p 轨道与苯环 π 体系共轭，正电荷离域到苯环 → 稳定

共振结构表示：
- 烯丙基正离子：$\ce{CH2=CH-CH2+ <-> +CH2-CH=CH2}$（两个等价共振式）
- 苄基正离子：正电荷可以分布到苯环的邻、对位

这种 p-π 共轭稳定效应是理解碳正离子反应（如 $\mathrm{S_N1}$、E1）的关键。
:::

:::pitfall{label=把 p-π 共轭和 p-p 共轭混淆**
- **p-π 共轭**：p 轨道（含电子）与 π 体系共轭。如 $\ce{C6H5-NH2}$（N 的孤对电子与苯环 π 共轭）
- **p-p 共轭**：两个 p 轨道之间的共轭，通常指缺电子 p 轨道（如碳正离子的空 p）与含电子 p 轨道（如 O、N 的孤对电子）共轭。如 $\ce{CH3-O+-CH2}$（氧鎓离子中 O 的 p 轨道与 $\ce{CH2+}$ 的空 p 共轭）

两者都涉及 p 轨道，但 p-π 是"p 与 π 体系"，p-p 是"p 与 p"。考试时不要混用。
:::

:::note{label=本节小结}
- **共轭效应**：共轭体系中 π 电子离域，降低体系能量，使键长平均化；
- **共轭效应 vs 诱导效应**：
  - 诱导效应沿 σ 键传递，迅速衰减
  - 共轭效应沿 π 键（或 p 轨道）传递，**不衰减**，贯穿整个共轭体系
- **-C（吸电子共轭）**：$\ce{-NO2}$、$\ce{-CN}$、含羰基基团
- **+C（给电子共轭）**：$\ce{-NR2}$、$\ce{-OR}$、$\ce{-OH}$、卤素
- **卤素特殊**：-I + +C，-I 主导 → 总体弱吸电子
- **总电子效应 = 诱导 + 共轭**（共轭体系中）；非共轭体系只考虑诱导；
- **三种共轭体系**：
  - π-π：双键以单键相连（1,3-丁二烯、苯乙烯）
  - p-π：p 轨道与 π 体系（苯胺、苯酚、烯丙基正离子）
  - p-p：缺电子 p 与含电子 p（碳正离子与 O、N 相连）
- 下一节我们将学习超共轭效应、动态诱导极化效应和立体效应——三种更精细的电子效应。
