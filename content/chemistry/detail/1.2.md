# 1.2 共价键与分子结构基础

:::insight{label=为什么有机物数量这么庞大}
上一节我们看到有机化合物数量庞大到一周就新增两万多种，可它们的组成元素却很少。这"少元素、多种类"的反差，根源就在**成键方式**。所有物质的成键无非三种主要形式：离子键、金属键、共价键。前两种不是有机化学的方式——而**有机化学的成键方式是共价键**，而且碳能形成**四个**共价键。一个原子四面出键，再去做排列组合，就像四面都能拼接的乐高积木，能搭出的结构数量自然非常庞大。所以要理解有机世界，必须先把共价键这件事彻底吃透。
:::

## 一、三类化学键与共价键

:::definition{label=三类基本化学键}
- **离子键**：原子或分子得失电子形成阴阳离子，离子间通过**静电引力**结合。如氯化钠晶体 $\ce{NaCl}$ 及许多无机盐。
- **金属键**：金属原子最外层价电子脱落，在正离子产生的场中**自由运动**，自由电子与正离子相互吸引使原子紧密堆积成金属晶格。如铁、铝。
- **共价键**：两个或多个原子通过**共用电子对**产生的化学键。这是有机化学唯一的成键方式。
:::

## 二、价键理论与成键方式

价键理论由**路易斯于 1916 年**提出，1927 年得到量子力学印证，1931 年**鲍林**引入杂化轨道理念，最终形成现代价键理论。它的要点可概括为：

:::theorem{label=价键理论要点}
1. **配对成键**：原子 A、B 各有一个自旋方向相反的未成对电子，即可配对形成共价单键；各有两个或三个未成对电子，则形成双键或三键。若 A 有两个未成对电子、两个 B 各有一个，则 A 可与两个 B 成两根键（如 $\ce{H2O}$）。
2. **饱和性**：一个电子与另一自旋相反的电子配对后，就不能再与第三个电子配对。如碳与四个氢成四根 $\ce{C-H}$ 键后即饱和。
3. **方向性**：原子轨道重叠越多，键越稳定，因此成键沿电子云密度最大的方向叠加。球形的 1s 轨道两球相互靠近；哑铃形的 2p 轨道"头碰头"叠加形成 **σ 键**，"肩并肩"叠加形成 **π 键**。
4. **可杂化**：能量相近的原子轨道可以进行杂化。
:::

:::insight{label=电子是个"顽皮的小孩"}
有一个会反复用到的模型：把电子想象成一个破坏性很强的顽皮小孩。把它关进狭小空间，它动静很大、能量很高；把它放到操场上，它就相对平和。轨道重叠越多、电子能离域运动的"场域"越大，电子能量就越低，形成的物种活性越低、越稳定。这就是为什么"轨道重叠越多，键越稳定"。
:::

## 三、杂化轨道：sp³、sp²、sp

:::insight{label=为什么需要"杂化"这个概念}
价键理论到这里遇到了一个解释不了的难题：甲烷里一个碳和四个氢成键，氢贡献的是 s 轨道，碳贡献的是 s 和 p 轨道，可为什么最后成出来的四根 $\ce{C-H}$ 键完全一模一样？答案就是**杂化**。打个比方：杂化就像和面——抓一把 s、抓三把 p，揉到一起，再切成四等份，每一份里都是 1/4 的 s 和 3/4 的 p，于是四份完全相同。这就是 sp³ 杂化的直觉。
:::

:::definition{label=三种常见杂化轨道}
- **sp³ 杂化**：由 1 个 s 轨道与 3 个 p 轨道杂化而成，呈**正四面体**分布，键角 **109.5°**。典型是甲烷 $\ce{CH4}$。
- **sp² 杂化**：由 1 个 s 轨道与 2 个 p 轨道杂化而成，呈**平面型**，键角 **120°**。典型是乙烯 $\ce{C2H4}$ 中的两个碳，形成碳碳双键。
- **sp 杂化**：由 1 个 s 轨道与 1 个 p 轨道杂化而成，呈**直线型**，键角 **180°**。典型是乙炔 $\ce{C2H2}$、炔烃。
:::

需要补充的是**分子轨道理论**：价键理论只能解释两个原子间的成键，多原子分子要用分子轨道理论。其大意是，多少个原子轨道组合就生成多少个分子轨道，分为**成键轨道**和**反键轨道**；每个轨道只容纳两个自旋相反的电子，电子优先填充能量最低的轨道。原子轨道组成有效分子轨道需满足三个条件：**能量相近、电子云最大重叠、对称性匹配**。

## 四、共价键的四个参数

考察一个共价键，要看四个特征：**键长、键角、键能、键的极性**。

### 1. 键长

:::definition{label=键长}
键长是两个成键原子之间键的长度。由于两原子永远在振动，键长有最长和最短时刻，所以**键长定义为一个振动周期内的平均值**。

单位：过去常用埃（$1\ \text{Å}=10^{-10}\ \text{m}$），现在很多书用皮米（$1\ \text{pm}=10^{-12}\ \text{m}$）。
:::

键长与键型、成键原子轨道的杂化类型有关：原子半径越小键长越短；相同原子之间，**单键最长、双键次之、三键最短**。从录音里的数据看：

- **碳氢键**：在烷烃中约 **109 pm** 基本不变，到乙烯、乙炔中略微缩短，整体在 **106~109 pm** 之间，变化幅度不大。
- **碳碳键**：乙烷 $\ce{C2H6}$ 中碳碳单键约 **154 pm**；乙烯中碳碳双键缩到约 **134 pm**（缩了近 20 pm）；乙炔中碳碳三键又缩了约 20 pm。

:::derivation{label=为什么单键>双键>三键}
从单键到双键再到三键，键中 **s 轨道的成分越来越多**。而 s 轨道是离原子核最近的轨道，s 成分越多，电子云越贴近核，两原子核被拉得越近，于是键长越来越短。这就解释了**键长：单键 > 双键 > 三键**。
:::

### 2. 键角

:::definition{label=键角}
键角是键与键之间形成的夹角，它确定了分子的几何形状。
:::

例如甲烷中 $\ce{C-H}$ 键夹角是正四面体的 **109.5°**；而乙烯中 $\ce{C-H}$ 键之间的夹角是 **121.3°**（标准值 120°，实测 121.3°）。可见同样由碳氢构成的键，夹角会随分子结构不同而不同。

:::note{label=补充：两种分子模型}
有机化学常用两种模型：**球棍模型**把共价键"亮"成一根棍；**比例模型**按原子实际大小作比例。真实分子更接近"胖嘟嘟"的**比例模型**——分子中并不存在像棍一样的化学键，原子是被高速运动的电子结合在一起的。
:::

### 3. 键能

:::definition{label=键能}
**键能**是将一个**气态分子**分解成组成它的全部原子时所需的能量。例如把一个气态甲烷分子的四根 $\ce{C-H}$ 键全部打断所需的能量，再除以 4，即为甲烷中碳氢键的键能。

由于逐个敲断时所需能量并不相同（越到后面越高），所以这里得到的仍是**平均键能**。
:::

键能是化学键强度的主要标志之一：**键能越高，键越强、越不容易被打断**。这一点对有机化学极其重要——

:::insight{label=有机反应的本质就是断键与成键}
有机化学反应的实质是**化学键的断裂与重构**：从 A 分子变成 B 分子，本质就是旧键断、新键成。所以"键能"就是打断一根化学键需要付出的能量代价，理解键能，才能理解反应为什么这样发生、需要多少能量。
:::

### 4. 键的极性

:::definition{label=键的极性}
共价键的实质是两个原子共享一对电子。
- **非极性共价键**：两个**相同**原子成键时，双方对共用电子对的控制力相等，电子对始终在中间、不偏向任何一方。
- **极性共价键**：两个**不同**原子成键时，吸电子能力强的一方使电子云偏向自己，该端显负电、另一端感应出正电，键就有了极性（正电端与负电端）。
:::

:::insight{label=从非极性键到离子键是连续渐变的}
把极性继续加强到极限，吸电子能力强的原子干脆把这对电子完全要过去，键就断了、变成正离子和负离子——这就是离子键。所以**从非极性共价键 → 极性共价键 → 离子键，是一个连续的、从量变到质变的过程**，中间并没有截然的界线，是程度的不同。
:::

## 五、电负性：判断键极性的定量工具

:::definition{label=电负性}
原子核与内层（非价）电子组成的实体称为**原子实**，**原子实对价电子的吸引能力就叫电负性**。电负性越大，说明它越牢牢控制外层电子、越不容易被夺走。

常见原子电负性（归一化数值，据录音）：氢 2.2，碳 2.5，氮 3.1，氧 3.5，氟 4.0，氯 3.2，溴 3.0。
:::

:::theorem{label=用电负性差判断键型}
设两成键原子电负性之差为 $\Delta\chi$：

- $\Delta\chi = 0 \sim 0.6$：**非极性共价键**（偏转很小，有机化学中视为非极性）；
- $\Delta\chi = 0.6 \sim 1.7$：**典型的极性共价键**（数值越大极性越大、电子云偏转越大）；
- $\Delta\chi > 1.7$：相差太大，电子被直接夺走，形成**离子键**。
:::

定量衡量键极性还可用偶极矩 $\mu$，它等于电量乘以两电荷中心之间的距离：

$$
\mu = q \cdot d
$$

:::example{label=例1：碳氢键与碳氟键到底是不是极性键}
**问题：** 碳氢键 $\ce{C-H}$ 和碳氟键 $\ce{C-F}$，分别属于哪种键？

**解答：**
- $\ce{C-H}$：碳 2.5、氢 2.2，$\Delta\chi = 0.3$。虽然碳、氢是不同原子，理论上应有极性，但 $0.3 < 0.6$，电子云变形并不强，所以在有机化学中把碳氢键**归为非极性共价键**。
- $\ce{C-F}$：碳 2.5、氟 4.0，$\Delta\chi = 1.5$。接近 1.7 但尚未断键，所以是**极性共价键**。聚四氟乙烯（实验室常见的硬白塑料塞）里就全是这种碳氟键。

**结论：** 不能只凭"两原子是否相同"来判断，要用电负性差这把"尺子"。碳氢键是非极性键、碳氟键是极性键，二者差别正源于电负性差的大小。
:::

## 六、分子的极性：键极性的"向量和"

键有极性，由键组成的分子就可能有极性——但**有极性键不等于分子就有极性**。

:::theorem{label=分子极性的判据}
判断一个多原子分子是否有极性，**既要看组成它的共价键是否有极性，又要看分子的几何结构**。多原子分子的极性由分子中各极性键的**向量和（矢量和）**决定：

- 若各极性键的向量和不为零，分子有极性；
- 若各极性键因对称而相互抵消（向量和为零），分子无极性。

对**双原子分子**，分子极性与键的极性一致（如 $\ce{H2}$、$\ce{O2}$ 非极性，$\ce{HCl}$ 极性）。
:::

:::example{label=例2：四氯化碳 vs 一氯甲烷}
**问题：** $\ce{CCl4}$ 由四个极性的碳氯键组成，它是极性分子吗？把三个氯换成氢得到 $\ce{CH3Cl}$ 呢？

**解答：**
- **四氯化碳 $\ce{CCl4}$**：碳氯键是非常典型的极性键，但分子是正四面体对称结构，四个键的极性向量**互相抵消**，向量和为零，所以 $\ce{CCl4}$ 是**非极性分子**。
- **一氯甲烷 $\ce{CH3Cl}$**：只有一个氯，对称性被破坏。虽然碳氢键近似非极性，但整个分子的电子云被电负性大的氯"像抽水一样"往氯端拉，向量和不为零，所以 $\ce{CH3Cl}$ 是**极性分子**，极性朝向氯。

**类比：** 甲醇 $\ce{CH3OH}$ 也是极性分子——氧电负性大，电子云朝氧偏移，氧端富集负电荷（电子云图上显红），极性朝向氧。
:::

:::pitfall{label=易错点}
- **"由非极性键组成的分子一定是非极性分子""含极性键的分子一定有极性"——这两句都是错的（凡说"一定"基本都不一定对）。** 反例就是 $\ce{CCl4}$：四个极性键却是非极性分子。
- **判断分子极性不能只数键，要算向量和。** 别看四氯化碳有四个极性键就断定它有极性；高度对称会让极性相互抵消。
- **碳氢键在有机化学里按非极性处理，但它并非"严格"非极性。** 它仍有微小变形（$\Delta\chi=0.3$），这一点在后面讲 α-氢酸性、超共轭时会派上用场——只是程度小，通常按非极性看待。
- **杂化构型由结构决定，不是由"成了什么键"决定。** sp²、sp³ 是看这个碳是平面三角形还是四面体，而不是看它有没有双键——这一点在 1.3 讲碳正/碳负离子时尤其关键。
:::

:::note{label=本节小结}
- 三类化学键：离子键、金属键、共价键；有机化学只用**共价键**，碳成**四键**是有机物数量庞大的根源。
- 价键理论（路易斯1916→鲍林1931）四要点：配对成键、饱和性、方向性（σ头碰头、π肩并肩）、可杂化。
- 三种杂化：**sp³ 正四面体 109.5°（甲烷）、sp² 平面 120°（乙烯）、sp 直线 180°（乙炔）**。
- 共价键四参数：**键长**（振动平均值，单键>双键>三键，因 s 成分递增）、**键角**（定几何形状）、**键能**（拆成原子的平均能量，反应即断键成键）、**键的极性**。
- **电负性** = 原子实对价电子的吸引力；电负性差 0~0.6 非极性键、0.6~1.7 极性键、>1.7 离子键；$\mu=q\cdot d$。
- **分子极性 = 各极性键的向量和**，既看键极性又看几何结构：$\ce{CCl4}$ 对称抵消为非极性，$\ce{CH3Cl}$、$\ce{CH3OH}$ 为极性。
:::
