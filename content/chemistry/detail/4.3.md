# 4.3 烷烃的自由基取代与环烷烃开环

## 一、从一个高中实验说起

烷烃是饱和烃，化学性质总体上很不活泼：通常不与酸、碱、氧化剂反应。但在高温或光照下，它能发生几类反应，其中最核心、机理最清晰的，就是**自由基取代反应**。

先回忆一个高中就见过的实验：把甲烷和氯气放进一个密闭瓶子，置于阳光下照射，会观察到黄绿色（氯气的颜色）渐渐变淡，并有液体生成。

$$\ce{CH4 + Cl2 ->[h\nu] CH3Cl + HCl}$$

高中老师只告诉你"生成什么、看到什么"，记住即可。但大学有机化学要追问：**这个反应是怎么发生的？为什么一定要光照或高温？产物是怎么一步步冒出来的？** 这就是"机理"——它告诉我们反应从哪来、怎么走、中间经过了哪些步骤。把这个机理学透，是理解整个自由基化学的钥匙。

:::definition{label=自由基取代反应}
从反应类型看，甲烷的氢被氯取代，是一个**取代反应**；"自由基"则指这个反应是**由自由基引发并通过自由基传递**的。

回顾第一章：共价键的**均裂**（两个成键电子各分一个）就产生**自由基**——带有一个未成对单电子、极不稳定、急于配对的活性微粒。
:::

## 二、链反应机理：引发、增长、终止

我们沿着甲烷氯代反应，把机理拆成三步。

### 链引发

光照（或高温）把比较弱的氯氯键均裂，产生两个氯自由基。这一步的特征是**自由基的产生**。

$$\ce{Cl2 ->[h\nu] 2 Cl^.}$$

### 链增长（链转移）

氯自由基极活泼，它那个孤电子总想找人配对。此时体系里全是甲烷，它"一出门就撞上甲烷"，从甲烷的碳氢键上**夺走一个氢**，自己配对成 HCl 而变稳定，同时把"不稳定性"传给了甲烷，生成一个**甲基自由基**（碳自由基）：

$$\ce{Cl^. + CH4 -> HCl + CH3^.}$$

甲基自由基也不稳定，转而去抢氯气分子上的一个氯，生成第一个产物一氯甲烷，**同时又放出一个新的氯自由基**：

$$\ce{CH3^. + Cl2 -> CH3Cl + Cl^.}$$

:::insight{label=自由基"借鸡生蛋"}
把链增长两步合起来看：消耗掉一个氯自由基，又生成一个新的氯自由基——**自由基的浓度不降反持续维持**，真正被消耗的是甲烷和氯气这两种反应物。这一步的特征就是"一个自由基消失、另一个自由基产生"。新生的氯自由基再去找甲烷，循环往复，产物（一氯甲烷）不断累积。
:::

### 链终止

当甲烷快被消耗光时，氯自由基出门很难再撞到甲烷，于是自由基之间相互碰撞、耦合，生成稳定分子，活性消失，反应结束：

$$\ce{Cl^. + Cl^. -> Cl2}\qquad \ce{CH3^. + CH3^. -> CH3CH3}\qquad \ce{CH3^. + Cl^. -> CH3Cl}$$

:::theorem{label=自由基取代的三阶段机理}
1. **链引发**：弱键（如 $\ce{Cl-Cl}$）在光照/高温下均裂产生自由基；特征——**自由基的产生**。
2. **链增长（链转移）**：自由基夺取反应物的氢/原子，自身配对稳定的同时生成新自由基；特征——**一个自由基消失、另一个自由基产生**，不断消耗反应物。
3. **链终止**：反应物耗尽后，自由基相互耦合生成稳定化合物，反应结束；特征——**自由基消失**。

引发条件是**光照或高温**；凡是有利于自由基产生的因素，都有利于该反应进行。
:::

:::insight{label=一个自由基能引发几千次反应}
一个氯原子平均能引发约 **5000 次**链反应——这就是自由基的可怕之处：只要产生一点点，它就在循环中被反复利用，引发一连串反应。著名的氟利昂破坏臭氧层正是这个原理：氟利昂在高空紫外照射下产生氯自由基，一个氯自由基能催毁大量臭氧，所以人类释放的少量氟利昂能造成大面积臭氧空洞。
:::

:::pitfall{label=条件决定方向：取代还是开环}
后面会看到环丙烷的反应：同样是和氯气反应，**光照**下走的是自由基**取代**（取代到环上、环不开），而在**路易斯酸（如 $\ce{FeCl3}$）催化**下走的是**开环加成**。所以有机反应的条件极其重要——写反应若不写条件，方向都可能反掉，只能得一半分。
:::

## 三、不同氢的活性：三级 > 二级 > 一级

甲烷的 4 个氢化学环境完全相同，不存在选择性问题。但换成更大的烷烃，分子里就有了不同种类的氢，它们被取代的难易**不一样**。

:::example{label=丙烷的氯代——二级氢活性是一级氢的4倍}
丙烷有 6 个一级氢、2 个二级氢。氯代产物中约 **43%** 取代了一级氢、**57%** 取代了二级氢。把氢的数目折算进去：

$$\frac{\text{二级氢活性}}{\text{一级氢活性}} = \frac{57\%/2}{43\%/6} \approx 4$$

即**每个二级氢的活性约为每个一级氢的 4 倍**。
:::

:::example{label=异丁烷的氯代——三级氢活性是一级氢的5倍}
异丁烷有 9 个一级氢、1 个三级氢。折算后，**三级氢的活性约为一级氢的 5 倍**。
:::

:::theorem{label=氢的活性顺序}
不同种类氢发生自由基取代的活性顺序为：

$$\text{三级氢} > \text{二级氢} > \text{一级氢} > \text{甲烷的氢}$$

对氯代而言，叔氢 : 仲氢 : 伯氢的相对活性约为 **5 : 4 : 1**。
:::

## 四、氯与溴的选择性差异

同样的活性次序，溴代表现得比氯代**悬殊得多**。

:::example{label=溴代的高选择性}
用溴对丙烷取代：一级氢取代仅约 2%、二级氢约 98%；对异丁烷三级氢取代高达约 99%（一级氢几乎不被取代）。折算后溴代时三级 : 二级 : 一级活性相差极大，达到约 **1600 : 82 : 1** 的量级。
:::

:::theorem{label=溴代选择性高于氯代}
对比两组数据：
- 氯代 叔 : 仲 : 伯 ≈ 5 : 4 : 1（活性差别小，选择性低）；
- 溴代 叔 : 仲 : 伯 ≈ 1600 : 82 : 1（活性差别大，选择性高）。

**溴代的选择性远高于氯代。**
:::

:::insight{label=有机反应要的是选择性，不是"雨露均沾"}
若想专一地在丙烷 2 位取代得到 2-溴丙烷，该用氯还是溴？答案是**溴**。有机反应追求的不是"各个位点都反应一点"，而是高活性、高选择性——这为后续分离纯化提供极大便利。所以实验中做取代多用溴代试剂。

卤素整体的反应活性从氟到碘依次降低：**氟**太活泼、不受控，一般不用；**氯**活性适中但选择性差；**溴**活性略低但选择性好，最常用；**碘**活性太低、基本不反应（虽然选择性高，但难以发生）。
:::

## 五、为什么叔 > 仲 > 伯：自由基稳定性与超共轭

无论氯代还是溴代，活性次序永远是叔 > 仲 > 伯。这背后是统一的本质原因——**中间体碳自由基的稳定性**。

### 从中间体稳定性看反应难易

自由基取代要先夺氢、生成一个碳自由基。**生成的中间体越稳定，释放能量越多，反应越容易进行。** 在不同位置夺氢，生成的是不同的碳自由基：甲烷生成甲基自由基，其他烷烃在哪个位置反应就生成哪个位置的碳自由基。

回顾第一章：碳自由基的中心碳从 $\ce{sp^3}$ 变为 $\ce{sp^2}$，是平面型，单电子占据一个 p 轨道。沿甲基 → 乙基 → 异丙基 → 叔丁基自由基这条线，中心碳始终是 $\ce{sp^2}$，变的是周围碳氢键的数目越来越多。

:::insight{label=电子"住房"的比喻}
电子像顽皮的小孩：把它关在狭小空间里、活动范围受限时能量很高、破坏性大；把它放到开阔的操场上，活动范围一宽，反而安分了、能量低了。碳自由基上的单电子也一样——能"串门"的房间越多，它越稳定。
:::

### σ-p 超共轭

那么"房间"是怎么变多的？关键在于相邻碳氢 σ 键与碳自由基 p 轨道之间的关系。

:::definition{label=σ-p 超共轭}
相邻 $\ce{C-H}$ σ 键的电子云与碳自由基中心碳上的 p 轨道，二者**不完全平行**，但存在一定夹角下的部分交叉重叠。这种不完全的共轭，使 σ 键电子有机会向 p 轨道**离域**。

由于是 σ 键与 p 轨道之间形成的共轭，称为 **σ-p 超共轭**。
:::

:::derivation{label=超共轭数目越多，自由基越稳定}
数一数每种碳自由基相邻的 $\ce{C-H}$ σ 键能提供几个超共轭：

- 甲基自由基：无相邻 $\ce{C-H}$ 可参与（电子住"单间"）；
- 乙基自由基：1 个超共轭（"一室一厅"）；
- 异丙基自由基：2 个超共轭（"两室一厅"）；
- 叔丁基自由基：3 个超共轭（"三室一厅"）。

超共轭越多，单电子离域范围越大，电荷分布越分散，能量越低、越稳定。于是碳自由基稳定性：

$$\text{叔丁基} > \text{异丙基} > \text{乙基} > \text{甲基}$$
:::

:::theorem{label=自由基稳定性解释活性次序}
碳自由基稳定性 **三级 > 二级 > 一级 > 甲基**。中间体越稳定，对应位置的氢越容易被夺取，所以氢的取代活性 **三级 > 二级 > 一级 > 甲烷**。

这就从中间体稳定性的角度，统一解释了为什么不论氯代还是溴代，活性次序恒为叔 > 仲 > 伯——实验数据背后是同一个超共轭稳定化机制。
:::

## 六、环烷烃的开环：小环的特殊性

环烷烃里，**小环（三元环、四元环）有一个独特性质：容易开环加成**，而五元环及以上则稳定、不易开环，化学性质与一般烷烃相似。

### 真实结构：键角与"香蕉键"

为什么小环这么容易开环？历史上有过一个著名的（但不完全正确的）解释，再有正确的解释，我们分别来看。

:::note{label=拜尔张力学说的评价}
**拜尔张力学说**当年假设所有环状化合物都是**平面**结构，并定义"偏转角"：

$$\text{偏转角} = \frac{109°28' - \text{正多边形内角}}{2}$$

偏转角越大，张力越大、越不稳定。按此算出各环偏转角约为 24°、9°、0°、-5°、-9°、-12°（取绝对值）。

**问题在哪？** 由此会推出六元环张力也不小——这与事实矛盾（六元环其实最稳定）。错因是它的前提"所有环都是平面"就错了（如 4.2 所述，环己烷是椅式而非平面）。

**正确的部分**：用"键角偏离正常 $\ce{sp^3}$ 键角（109°28′）的程度"来衡量张力，这个思路是对的；**错误的部分**：认为所有环都共平面、且张力随环增大又增大，则全错。所以叫"学说"——它能部分解释现象，但并不正确，只是被科学史保留下来。
:::

:::insight{label=三元环的真相——香蕉键}
真正测得的环丙烷键角约为 **105.3°**，比标准的 109.5° 只偏离约 4°，并不像画成正三角形时想象的 60°（若真是 60° 要偏离将近 50°）。

也就是说，三元环为了**尽量维持键角**，牺牲的是**电子云的重叠程度**：成键电子云不能正对正地重叠，而是向外弯曲成弧形，形似香蕉，故称**香蕉键**。电子云重叠程度越小，键越弱、越容易断裂——这才是小环容易开环的真正原因。
:::

:::note{label=四元环与五元环的构象}
- **环丁烷**：四个碳不共平面，呈**折叠型（蝶式）**构象并快速翻转，以尽量保持 $\ce{sp^3}$ 构型，构象转化能约 $\pu{6.3 kJ/mol}$。
- **环戊烷**：呈**信封状**构象，五个碳中有一个碳不与其余四个共平面。
:::

### 开环活性的比较

催化氢化的开环条件，直观反映了开环的难易：

:::example{label=开环活性随环增大而骤降}
- 环丙烷 + $\ce{H2}$：1 atm、约 40 ℃、镍催化即可开环 → 丙烷；
- 环丁烷 + $\ce{H2}$：需约 100 ℃（条件更苛刻）；
- 环戊烷、环己烷：常态下很难开环，需约 300 ℃ 且要用昂贵的铂（Pt）催化。

$$\ce{C3H6 (环丙烷) + H2 ->[Ni][40^\circ C] CH3CH2CH3}$$

催化条件由易到难，正说明开环活性 **环丙烷 > 环丁烷 ≫ 环戊烷、环己烷**。
:::

## 七、环丙烷的开环加成转化

除催化氢化外，小环还能与多种试剂发生开环加成，这是从烷烃出发制备其他化合物的途径。

:::theorem{label=环丙烷的开环加成}
以环丙烷为例（环丁烷类似，但常需加热）：

- 与**卤素**（室温下与 $\ce{Br2}$）开环，溴加到两端碳上，生成 1,3-二溴丙烷：
$$\ce{C3H6 + Br2 -> BrCH2CH2CH2Br}$$

- 与**卤化氢**（如 $\ce{HBr}$）开环加成，生成一卤代丙烷（1-溴丙烷）：
$$\ce{C3H6 + HBr -> CH3CH2CH2Br}$$

- 与**水**（浓硫酸酸催化下）开环加成，相当于把一分子水加到环上（一头氢、一头羟基），生成丙醇：
$$\ce{C3H6 + H2O ->[H2SO4] CH3CH2CH2OH}$$

- 与**氯气**在**路易斯酸（$\ce{FeCl3}$）催化**下开环加成。
:::

:::pitfall{label=同一对反应物，条件不同结果迥异}
环丙烷与氯气：
- **光照** ⇒ 自由基**取代**（取代到环上，不开环）；
- **$\ce{FeCl3}$（路易斯酸）催化** ⇒ **开环加成**。

这是有机反应"条件至上"的经典例证。考试时若只画产物不写条件，最多得一半分。
:::

:::insight{label=学转化要分清碳数变化}
做有机合成、设计路线时，先看起始物与目标物的碳数关系，把转化分三类：
- **碳数不变的转化**（如环丙烷 → 丙醇，仍三个碳）；
- **增碳的转化**；
- **减碳的转化**。

"变有变的转法，不变有不变的转法"——这是后续设计合成路线的基本思路起点。
:::

## 八、本节小结

:::note{label=本节小结}
1. 烷烃的核心反应是**自由基取代**（光照/高温引发），机理分**链引发（生成自由基）→ 链增长（自由基此消彼长、消耗反应物）→ 链终止（自由基耦合）** 三阶段。一个自由基可引发约 5000 次链反应。
2. 不同氢的取代活性 **三级 > 二级 > 一级 > 甲烷**：氯代 叔:仲:伯 ≈ 5:4:1，溴代 ≈ 1600:82:1。
3. **溴代选择性远高于氯代**，故合成中常用溴；卤素活性 F > Cl > Br > I，但综合可控性与选择性，溴最常用。
4. 活性次序的本质是**碳自由基稳定性**（叔 > 仲 > 伯 > 甲基），由 **σ-p 超共轭**数目（叔丁基 3 个、异丙基 2 个、乙基 1 个、甲基 0 个）决定——中间体越稳定，反应越易、氢越易被取代。
5. **小环（三、四元环）易开环加成**：三元环真实键角约 105.3°（仅偏离 4°），靠弯曲的**香蕉键**维持键角、牺牲电子云重叠，故键弱易断；开环活性 环丙烷 > 环丁烷 ≫ 环戊烷、环己烷。
6. 环丙烷可与 $\ce{H2}$、$\ce{Br2}$、$\ce{HBr}$、$\ce{H2O}$（酸催化）、$\ce{Cl2}$（$\ce{FeCl3}$ 催化）发生开环加成；**条件决定取代还是开环**。
7. **拜尔张力学说**：用键角偏离 109°28′ 衡量张力的思路正确，但"所有环共平面、张力随环增大又增大"的结论错误（与六元环最稳定矛盾）——是部分正确、被保留的"学说"。
:::
