# 12.1 羰基的亲核加成

## 从一只蚊子说起：羰基为什么这么"招惹"试剂

夏天里蚊子专挑你叮，并不是因为它"看见"了你，而是闻到了你身上的气味。一项 *Nature* 的研究把人体汗液里占比最高的两种气味分子拎了出来——癸醛和甲基庚烯酮，它们恰恰都是醛、酮分子。蚊子的大脑里有专门的神经区域，闻到这类醛酮就被点亮。换句话说，醛和酮是一类"会说话"的分子：肉桂的香来自肉桂醛，黑胡椒的辛辣味里也有酮类分子，连雄性激素和雌性激素的差别，往往也只是一个羟基与一个羰基之间的细微变化。

为什么这样一类分子如此活泼、又如此"善于交际"？答案全都藏在它们共同的核心结构——**羰基**里。这一节我们要把羰基的电子特征讲透，然后顺着同一条逻辑主线，看它如何与氢氰酸、亚硫酸氢钠、醇、伯胺、仲胺等一系列亲核试剂发生**亲核加成反应**。你会发现，这些花样繁多的反应，内核其实只有一句话：亲核试剂进攻羰基碳、碳氧双键打开。

:::definition{label=羰基与羰基化合物}
羰基是碳氧双键 $\ce{C=O}$ 这个结构单元。羰基的一端连烃基、另一端连氢，就是**醛**；两端都连烃基，就是**酮**。

要特别强调：羰基本身并不是一个独立的官能团，它是构成许多官能团的"零件"。羰基连不同的基团会得到不同的官能团——连羟基 $\ce{-OH}$ 是羧酸，连烷氧基 $\ce{-OR}$ 是酯，连氨基 $\ce{-NH2}$ 是酰胺，连卤素是酰卤。醛和酮，是羰基连烃基或氢这两类最"裸露"的羰基化合物。
:::

## 羰基的结构：一根容易断的 π 键，加上电荷分离

羰基是碳氧双键，和我们熟悉的碳碳双键一样，由一根 $\sigma$ 键和一根 $\pi$ 键组成。$\sigma$ 键键能强、很难被打断；$\pi$ 键是两个 p 轨道并肩重叠形成的，流动性好、键能弱，容易被打开。所以**醛酮的反应，大多数时候是让碳氧 π 键发生断裂**——这是理解后面所有反应的第一把钥匙。

第二把钥匙是电荷分离。氧的电负性比碳大得多，碳氧双键形成以后，无论从诱导效应（对 $\sigma$ 键）的角度，还是从 $\pi$ 键的角度，电子云都被拉向氧端。结果是：

- 氧端带上部分负电荷 $\delta^-$；
- 碳端带上部分正电荷 $\delta^+$。

$$\ce{C^{\delta+}=O^{\delta-}}$$

:::insight{label=一个分子，两个反应位点}
正是这种电荷分离，给了羰基两个截然不同的反应位点：

**位点一：羰基碳。** 它带部分正电荷，是个"缺电子中心"，能接受带负电荷的试剂——也就是**亲核试剂**——的进攻。

**位点二：α 碳。** α 位上的 C–H 键能够与碳氧双键形成 σ-碳超共轭，使这个 C–H 发生部分极化，让 α-氢带上一点酸性。在强碱作用下，α-氢可以被拔走，生成一个碳负离子。这个碳负离子旁边就是羰基，能发生共振：负电荷既能在碳上、也能在氧上，但真正反应时往往是负电荷停在碳上去进攻别人（氧本身电负性高，再背一个负电荷反而更不稳定）。

此外，氧端的部分负电荷加上氧上的孤对电子，使氧可以被**质子化**。一旦质子化，碳氧双键极化得更厉害，羰基碳更"缺电子"，就更容易挨亲核进攻——这正是许多反应要用酸催化的原因。
:::

## 为什么是"加成"，不是"取代"

把羰基和卤代烷放在一起对比，能讲清醛酮的"特征反应"为什么是加成。卤代烷里，亲核试剂进攻碳，结果是 C–X 键断裂、卤素带着电子离去——这是**取代**。而在醛酮里，亲核试剂进攻羰基碳后，断的是碳氧双键中的那根 $\pi$ 键，氧并不离去，而是接住一对电子变成氧负离子，再质子化成羟基。试剂"加"上去了，没有谁离开——这就是**亲核加成**。

:::theorem{label=亲核加成是醛酮的特征反应}
羰基碳受亲核试剂进攻后，发生的是碳氧双键中 π 键的断裂，生成亲核加成产物（而非取代产物）。亲核加成因此成为醛酮最核心、最具特征性的一类反应。它的通用图景是：

$$\ce{R2C=O + Nu^- -> R2C(Nu)-O^- ->[H+] R2C(Nu)-OH}$$

其中 $\ce{Nu^-}$ 是亲核试剂。后面所有反应，第一步都是这个"进攻 + π 键打开"，区别只在后续的演化。
:::

## 影响亲核加成快慢的三个因素

亲核试剂要冲向那个带正电的羰基碳，能不能冲得进去、冲得快不快，看三件事——和我们当年分析亲核取代时的思路如出一辙。

:::note{label=三因素}
**1. 诱导效应。** 羰基上连吸电子基时，羰基碳电子云密度下降、正电性增强，更容易受亲核进攻；连给电子基则相反，活性下降。

**2. 共轭效应。** 与羰基直接相连的基团若能与羰基共轭，会分担掉氧的吸电子性。例如连一个苯环（如苯甲醛、苯乙酮），苯环与羰基共轭，羰基碳的正电性被削弱，反应活性下降。

**3. 立体效应（位阻效应）。** 羰基两边的基团太大，亲核试剂就难以靠近。位阻是决定醛酮能否顺利发生亲核加成的关键之一。
:::

这三条里，位阻在后面的反应中反复出现：醛（一头是小小的氢）几乎总能反应；脂肪族酮要是甲基酮才行；环酮要 8 个碳以下。记住这个"敏感性梯度"，后面就顺了。

## 反应一：与氢氰酸加成——增碳，得 α-羟基腈

氢氰酸 $\ce{HCN}$ 离解出氰基负离子 $\ce{CN-}$，它体积小、又带负电，是个极好的亲核试剂。

:::derivation{label=与 HCN 加成的电子流向}
1. $\ce{CN-}$ 以碳上的孤对电子进攻羰基碳，一对电子从 $\ce{CN-}$ 流向缺电子的羰基碳，形成新的 C–C 键；
2. 与此同时，碳氧双键的 π 电子被"挤"到氧上，碳氧双键打开，氧变成氧负离子；
3. 氧负离子从体系中夺取一个质子，变成羟基。

$$\ce{R2C=O ->[CN^-] R2C(CN)-O^- ->[H+] R2C(CN)-OH}$$

产物是 **α-羟基腈**。
:::

这个反应的价值在于"增碳"——氰基自带一个碳，上去就把碳链延长了一格。而且 α-羟基腈的氰基在酸性或碱性条件下很容易水解成羧基，整条线下来就得到一个 α-羟基酸。再加热，由于一个碳上同时挂着羟基和羧基（两个含氧基团）很不稳定，会脱水生成 $\alpha,\beta$-不饱和酸。

:::example{label=工业上的甲基丙烯酸甲酯}
丙酮与氰化钠（因 $\ce{HCN}$ 剧毒，工业上用盐 $\ce{NaCN}$ 代替）反应生成 α-羟基腈，再水解成羧酸，经酯化、加热脱水，得到**甲基丙烯酸甲酯**。它聚合后就是聚甲基丙烯酸甲酯——也就是我们买相框时常说的"亚克力玻璃"。

反应一般在催化量的碱条件下进行：碱把质子中和掉，能离解出更多 $\ce{CN-}$，有利于反应。
:::

这个反应对位阻很敏感：醛没问题；脂肪族甲基酮没问题（甲基够小）；8 个碳以下的环酮也可以；但甲基一换成乙基，活性立刻掉下来。

## 反应二：与亚硫酸氢钠加成——鉴别醛酮

亚硫酸氢钠 $\ce{NaHSO3}$ 中硫上的孤对电子充当亲核试剂，进攻羰基碳，加成后生成一种**白色晶体盐**。

:::insight{label=用"看得见的位阻"鉴别}
这个反应在合成上没什么用，价值全在**鉴别**：把 $\ce{NaHSO3}$ 加进醛酮里，只要羰基位阻不太大，就生成白色沉淀；位阻太大则不反应、无沉淀。于是从"有没有白色晶体"就能反推羰基附近的位阻情况。

收率把位阻效应量化得很直观：
- 两边都是甲基的丙酮，收率约 56%；
- 一头换成乙基，收率约 36%（在有机合成里完全可以接受）；
- 8 个碳以下的环酮约 35%（成椅式后羰基仍易被进攻）；
- 两边都换成乙基，掉到约 2%。

所以"能与 $\ce{NaHSO3}$ 生成白色晶体"基本说明：这是醛、甲基酮，或 8 个碳以下的环酮。
:::

## 反应三：与醇加成——半缩醛/缩醛与羰基保护

醇的羟基氧也是亲核试剂。但醇的亲核性不算强，所以醛酮与醇的加成要在**酸催化**下进行，并伴随脱水。

:::definition{label=半缩醛(酮)与缩醛(酮)}
1 分子醛酮可与 2 分子醇反应。只反应掉 1 分子醇时，得到的产物一个碳上同时带羟基和一个 $\ce{-OR}$，叫**半缩醛**（醛来的）或**半缩酮**（酮来的）；再上去第 2 分子醇、把那个羟基也换成 $\ce{-OR}$，就成了**缩醛**或**缩酮**。

一句话区分：底物有氢（来自醛）叫"缩醛"，两边都是烃基（来自酮）叫"缩酮"。
:::

:::derivation{label=成缩醛的机理（两次亲核加成）}
1. 酸先把羰基氧质子化，羰基碳更缺电子、更易受进攻；
2. 第一分子醇的氧进攻羰基碳，再脱去质子，得到**半缩醛/半缩酮**（已完成第一次亲核加成）；
3. 半缩物上的羟基被质子化后离去，驱动力是邻位氧的孤对电子去进攻碳、把水顶走，重新形成一个碳氧双键（此时氧被烃基化，带正电）；
4. 第二分子醇再进攻这个碳，脱质子，得到**缩醛/缩酮**。

整个过程是醛酮在酸催化下与醇发生了**两次**亲核加成。反应式用**双箭头**——它是**可逆**的：

$$\ce{RCHO + 2 R'OH <=>[H+] RCH(OR')2 + H2O}$$
:::

:::insight{label=真正的用途：保护羰基}
缩醛、半缩醛本身在合成上没什么价值，它们的价值在于**保护羰基**。羰基容易被氧化，当你要在分子别处做强氧化反应、又不想动到羰基时，可以先把羰基做成缩醛/缩酮"封存"起来，做完氧化反应后，再在酸中加热，可逆地把羰基游离出来。

例如：1 分子丁醛 + 2 分子甲醇，酸催化生成丁醛缩二甲醇；环己酮 + 乙二醇类似地生成环状缩酮。都是为了保护羰基，事后能在酸性条件下复原。
:::

:::note{label=两个补充}
**硫醇成缩硫酮**：硫醇也能像醇那样与醛酮加成生成缩硫酮，但这个反应**不可逆**。它的妙用在于：生成缩硫酮后用催化氢化，可把那两个 C–S 键直接打断、还原成 $\ce{CH2}$。于是整条路把羰基**定向还原成了 $\ce{CH2}$**（而不是停在羟基），是一条把羰基"打回原形"的合成路线。

**与水的加成**：水也是亲核试剂，醛酮可与水加成，但产物一般不稳定。例外是三氯乙醛——它与水加成得到稳定的水合三氯乙醛，是一种助眠药物。
:::

## 反应四：与伯胺加成——亚胺与多孔材料

氨和有机胺也是亲核试剂。其中最有价值的是**伯胺**与醛酮的反应，生成**亚胺**。

从总包看，相当于醛酮的羰基氧和伯胺上的两个氢一起被"切下来"、脱去一分子水，生成碳氮双键。但实际过程仍是亲核加成一步步走出来的。

:::derivation{label=成亚胺的机理}
1. 伯胺的氮以孤对电子进攻羰基碳，碳氧 π 键打开；
2. 中间体结合质子，生成一个碳上同时带羟基和氨基的加成物；
3. 羟基与氮上的氢脱去一分子水，生成碳氮双键。

$$\ce{R2C=O + R'NH2 -> R2C=NR' + H2O}$$

产物（碳氮双键结构）就是**亚胺**，也叫**席夫碱**。
:::

:::example{label=亚胺反应与 COF 多孔材料}
伯胺成亚胺是化学当量反应，定向、高效、几乎没有副产物，因而在新材料合成中极常见。

MOF（Metal-Organic Framework，金属有机框架）、COF（Covalent Organic Framework，共价有机框架）、HOF（Hydrogen-bonded Organic Framework，氢键有机框架）都是**多孔材料**，区别在连接方式：MOF 靠金属配位、HOF 靠氢键、COF 靠共价键连接。

与醛酮亲核加成直接相关的是 **COF**。把一个多醛（如三醛苯类）与一个二胺（如对苯二胺）放在一起，醛基与氨基脱水生成刚性的碳氮双键，两头都能继续反应，于是无限聚合成网状结构，网格中间就是纳米尺度的孔。这种构筑用的亚胺反应就是**席夫碱反应**，是搭建多孔材料最有用的"有机砌块"之一。想要更大的孔，就换更大的砌块（多连几个苯环），孔径可调。多孔材料可用于吸附、分离、装载分子等。
:::

:::pitfall{label=亚胺挑胺，且能被水解}
- **挑胺**：脂肪族伯胺生成的亚胺不稳定，容易可逆回去；构筑材料一般用**芳香族胺（芳胺）**，生成的亚胺较稳定，可分离得到。
- **可水解**：亚胺在稀酸性条件下也会发生水解，并非"永久"。

顺带记住命名：伯胺给亚胺（席夫碱）；**羟胺** $\ce{NH2OH}$ 给**肟**；**肼**给**腙**（如与苯肼反应得苯腙）。它们都是氨基与羰基脱水的产物。
:::

## 反应五：与仲胺加成——烯胺与 α-烷基化

仲胺（二级胺）的氮上原本带一个 N–H。它与醛酮反应，产物不是亚胺，而是**烯胺**。

:::derivation{label=成烯胺的机理}
1. 酸催化下羰基质子化，仲胺的氮进攻羰基碳，生成加成产物；
2. 脱质子，得到一个碳上既带羟基、又带氨基的分子（不稳定，必继续）；
3. 继续酸催化：羟基质子化后离去，同时拔去邻位（α 位）的一个氢，在该处生成一个**碳碳双键**。

最终氮仍是饱和的氮。烯胺的特点是：氨基与一个碳碳双键直接相连。

$$\ce{R2CH-C(=O)-R'' + R'2NH ->[H+] R2C=C(NR'2)-R'' + H2O}$$
:::

注意烯胺与亚胺的本质区别：亚胺是在碳、氮之间形成双键；烯胺则多走了一步，生成的是**碳碳双键**加一个氨基。

:::example{label=烯胺作诱导基——环己酮 α 位引入甲基}
烯胺本身用处不大，但它是个出色的**诱导基团**。要在环己酮的 α 位引入一个甲基，可以借道烯胺：

1. 环己酮先与仲胺反应，生成烯胺；
2. 用碘甲烷 $\ce{CH3I}$ 进攻，甲基正好被引到 α 位；
3. 再用酸性条件把烯胺水解掉（所有与胺的反应都能在酸性条件下可逆游离）。

这样就完成了环己酮 α 位的甲基化——烯胺充当了"诱导基"，引导反应在指定位置发生。
:::

## 例题

:::example{label=例题：判断产物与位点}
**题目：** 下列各组反应，分别写出主要产物的结构类型，并说明依据。
（1）丁醛 + 2 分子甲醇，酸催化；
（2）丙酮 + $\ce{NaHSO3}$；
（3）苯甲醛 + 苯胺（一种芳香伯胺）。

**解答：**
（1）醇与醛在酸催化下发生两次亲核加成、脱水，得**缩醛**（丁醛缩二甲醇）。反应可逆，用于保护羰基。
（2）丙酮是两边甲基的脂肪族甲基酮，位阻不大，可与 $\ce{NaHSO3}$ 加成生成**白色晶体盐**（收率约 56%），可用于鉴别。
（3）苯甲醛无 α-氢、苯胺为芳香伯胺，二者脱水生成稳定的**亚胺（席夫碱）**，碳氮双键结构，可分离得到。

**点拨：** 抓住"亲核试剂是谁、底物位阻多大、产物含哪些键"三条，绝大多数加成产物都能判准。
:::

## 易错点

:::pitfall{label=四个常见坑}
1. **把羰基当成单独官能团。** 羰基只是"零件"，连不同基团才形成醛、酮、羧酸、酯、酰胺、酰卤等不同官能团。

2. **混淆加成与取代。** 醛酮亲核加成断的是碳氧 π 键、氧不离去；不要套用卤代烷"C–X 断裂、卤素离去"的取代思路。

3. **忽略位阻的"敏感性梯度"。** 与 $\ce{HCN}$、$\ce{NaHSO3}$ 的加成只对醛、甲基酮、8 碳以下环酮顺利；两边都是较大烃基时（如二乙基酮）几乎不反应。

4. **分不清亚胺与烯胺。** 伯胺→亚胺（C=N）；仲胺→烯胺（C=C + 氨基）。另外缩醛反应可逆、可水解，缩硫酮反应不可逆。
:::

:::note{label=本节小结}
- **核心结构**：羰基 $\ce{C=O}$ 由强 σ 键与易断 π 键组成；氧拉电子使碳带 $\delta^+$、氧带 $\delta^-$，并可被质子化。
- **两个位点**：羰基碳受亲核进攻（位点一）；α 碳在强碱下失氢生成碳负离子、可作亲核试剂（位点二）。
- **特征反应**：亲核加成——亲核试剂进攻羰基碳、碳氧 π 键打开，产物为加成物而非取代物。
- **三因素**：吸电子基增活性、共轭降活性、位阻降活性；敏感性梯度为醛 > 甲基酮/8碳以下环酮 > 大位阻酮。
- **六类亲核试剂**（本节五类）：$\ce{HCN}$（α-羟基腈，增碳）；$\ce{NaHSO3}$（白色晶体，鉴别）；醇（半缩醛/缩醛，保护羰基，可逆；硫醇成缩硫酮可定向还原为 $\ce{CH2}$）；伯胺（亚胺/席夫碱，构筑 COF 多孔材料）；仲胺（烯胺，作诱导基实现 α-烷基化）。
- 所有反应的内核相同：**亲核进攻 + 碳氧双键打开**，后续演化不同，便得到形形色色的产物。
