# ppt-9.3 消除反应——E1、E2、Zaitsev 规则与反式消除

:::insight{label=从"取代还是消除"说起}
把 2-溴丙烷分别加入 $\ce{NaOH}$ 的**水溶液**和**醇溶液**中加热，会得到完全不同的产物：

- **水溶液**：主要得到 2-丙醇（**取代产物**，SN2）
- **醇溶液**：主要得到丙烯（**消除产物**，E2）

同样的底物，同样的碱，只是溶剂不同，产物就从"取代"变成了"消除"！为什么？

因为亲核试剂（$\ce{OH^-}$）有两种"进攻方式"：
- 进攻 $\alpha$-碳 → **取代**（SN）
- 进攻 $\beta$-氢 → **消除**（E）

两种反应**竞争**进行，谁占主导取决于底物结构、试剂、溶剂、温度等因素。本节我们学习消除反应的机理（E1、E2）、区域选择性（Zaitsev 规则）、立体选择性（反式消除），以及消除与取代的竞争。
:::

## 消除反应的定义

:::definition{label=消除反应}
**消除反应**（elimination reaction，符号 E）：在一个有机分子中消去两个原子或基团的反应。

**类型**（按消除位置）：
- **1,1-消除**（$\alpha$-消除）：同一碳上消除两个基团
- **1,2-消除**（$\beta$-消除）：相邻两个碳上各消除一个基团——**最常见**
- **1,3-消除**（$\gamma$-消除）：相隔一个碳的消除

**卤代烃的 $\beta$-消除**：卤素和 $\beta$-碳上的氢失去一分子卤化氢，生成烯烃。

$$
\ce{R-CH2-CHX-R' ->[碱] R-CH=CH-R' + HX}
$$

（$\alpha$-碳连 X，$\beta$-碳连 H，消除 $\ce{HX}$ 生成双键）
:::

## E1 机理——单分子消除

:::definition{label=E1反应机理}
**E1 反应**（unimolecular elimination）：只有一种分子参与了决速步的消除反应，分步完成。

**机理**（两步）：

**第 1 步**（慢，决速步）：C—X 键异裂，形成碳正离子

$$
\ce{R-CH2-CHX-R' ->[慢] R-CH2-\overset{+}{CH}-R' + X^-}
$$

**第 2 步**（快）：碱夺取 $\beta$-H，形成双键

$$
\ce{R-CH2-\overset{+}{CH}-R' + B^- ->[快] R-CH=CH-R' + HB}
$$

**速率方程**：

$$
v = k[\ce{RX}]
$$

一级反应，速率只与底物浓度有关。
:::

:::theorem{label=E1反应的特点}
1. **单分子反应**，一级反应
2. **分步完成**，形成碳正离子中间体（决速步）
3. **C—X 键异裂是决速步**
4. **可能有碳正离子重排**（仲→叔等）
5. **与 SN1 竞争**：碳正离子既可以被亲核试剂进攻（SN1），也可以失去 $\beta$-H（E1）
:::

:::theorem{label=SN1和E1中的碳正离子重排}
E1（和 SN1）反应中，碳正离子中间体可能发生**重排**，形成更稳定的碳正离子：

**新戊基溴代烷** $\ce{(CH3)3C-CH2Br}$ 的反应：
1. $\ce{C-Br}$ 键异裂 → 伯碳正离子 $\ce{(CH3)3C-\overset{+}{CH2}}$（不稳定）
2. 1,2-甲基迁移 → 叔碳正离子 $\ce{(CH3)2\overset{+}{C}-CH2CH3}$（更稳定）
3. 重排后的碳正离子：
   - 失去 $\beta$-H → **消除产物**
   - 被亲核试剂进攻 → **取代产物**

**关键**：E1 和 SN1 共享碳正离子中间体，重排后两者都可能发生。
:::

## E2 机理——双分子消除

:::definition{label=E2反应机理}
**E2 反应**（bimolecular elimination）：有两种分子参与了决速步的消除反应，一步完成。

**机理**（一步协同）：

$$
\ce{B^- + H-\beta C - \alpha C - X ->[慢] [B\bond{...}H\bond{...}C\bond{...}C\bond{...}X]^- -> B-H + C=C + X^-}
$$

- 碱 $\ce{B^-}$ 进攻 $\beta$-H
- $\beta$-H 离去，$\ce{C-H}$ 键电子形成 $\pi$ 键
- 同时 $\ce{C-X}$ 键断裂，$\ce{X^-}$ 离去
- 一步协同完成，经过一个**四中心过渡态**

**速率方程**：

$$
v = k[\ce{RX}][\ce{B^-}]
$$

二级反应，速率与底物和碱浓度都有关。
:::

:::theorem{label=E2反应的特点}
1. **双分子反应**，二级反应
2. **一步完成**，协同进行
3. **碱进攻 $\beta$-H**（不是 $\alpha$-C）
4. **反式消除**：$\beta$-H 和 X 必须处于**反式**（对位交叉）位置
5. **与 SN2 竞争**：碱既可以进攻 $\beta$-H（E2），也可以进攻 $\alpha$-C（SN2）
:::

:::insight{label=E2与SN2的竞争——进攻位点不同}
E2 和 SN2 都是双分子协同反应，但进攻位点不同：
- **SN2**：亲核试剂进攻 $\alpha$-C（带 X 的碳）
- **E2**：碱进攻 $\beta$-H（$\alpha$-C 相邻碳上的 H）

两者同时发生，竞争进行。谁占主导取决于：
- 强亲核性、弱碱性试剂 → SN2 占优
- 强碱性、弱亲核性试剂 → E2 占优
- 高温 → E2 占优（消除熵增有利）
- 叔卤代烷 + 强碱 → E2 占优（位阻大，SN2 困难）
:::

## Zaitsev 规则——区域选择性

:::theorem{label=Zaitsev规则}
**Zaitsev 规则**（扎依采夫规则）：消除反应的主要产物是脱去**含氢较少**的 $\beta$-碳原子上的氢，生成双键碳原子上连有**最多烃基**的烯烃（更稳定）。

**例**：2-溴丁烷消除 $\ce{HBr}$：

$$
\ce{CH3-CHBr-CH2-CH3 ->[碱] \underset{81\%, \text{Zaitsev}}{CH3-CH=CH-CH3} + \underset{19\%, \text{反Zaitsev}}{CH2=CH-CH2-CH3}}
$$

- **Zaitsev 产物**：2-丁烯（双键在中间，连 2 个烷基）→ 81%
- **反 Zaitsev 产物**：1-丁烯（双键在端位，连 1 个烷基）→ 19%

**原因**：双键上烷基越多，烯烃越稳定（超共轭效应使双键稳定）。
:::

:::insight{label=为什么Zaitsev产物更稳定}
烯烃的稳定性与双键上的烷基数目有关：

$$
\ce{R2C=CR2 > R2C=CHR > R2C=CH2 > RCH=CH2 > CH2=CH2}
$$

**原因**：烷基的 C—H σ 键与 $\ce{C=C}$ π 键发生 σ-π 超共轭，使体系稳定。烷基越多，超共轭越多，烯烃越稳定。

所以消除反应倾向于生成**双键上烷基最多**的烯烃——这就是 Zaitsev 规则的本质。
:::

:::theorem{label=反Zaitsev产物——Hofmann消除}
在某些情况下，消除反应主要生成**反 Zaitsev 产物**（双键上烷基较少的烯烃）：

1. **大体积碱**（如 $\ce{(CH3)3CO^-}$）：位阻使碱优先夺取位阻小的 $\beta$-H（端碳的 H），生成反 Zaitsev 产物
2. **季铵碱消除**（Hofmann 消除）：主要生成反 Zaitsev 产物

**例**：2-溴-2-甲基丁烷用 $\ce{(CH3)3CO^-}$ 消除，主要得到 2-甲基-1-丁烯（反 Zaitsev），而非 2-甲基-2-丁烯（Zaitsev）。
:::

## 反式消除——立体选择性

:::theorem{label=E2的反式消除}
**E2 反应要求 $\beta$-H 和 X 处于反式（对位交叉）位置**。

**原因**：
- E2 过渡态要求 $\ce{C-H}$ 键和 $\ce{C-X}$ 键的电子云**反平行**（在同一平面，方向相反）
- 这样 $\ce{C-H}$ 键断裂时电子可以"流入"$\ce{C-C}$ 之间形成 $\pi$ 键，同时 $\ce{C-X}$ 键断裂
- 只有**对位交叉构象**满足这一要求

**对位交叉式**：空间位阻小，优势构象，反式消除
**重叠式**：空间位阻大，非优势构象，顺式消除（不利）
:::

:::insight{label=为什么E2要反式消除}
想象 E2 过渡态是一个"四元环"：$\ce{B...H-C-C...X}$。要让 $\ce{C-H}$ 键的电子流向 $\ce{C-C}$ 形成 $\pi$ 键，同时 $\ce{C-X}$ 键断裂，需要 $\ce{C-H}$ 和 $\ce{C-X}$ 的电子云**反平行**——就像两个人背靠背，一个"推"一个"拉"。

如果 $\ce{H}$ 和 $\ce{X}$ 在同侧（顺式），它们的电子云"重叠"，不能形成有效的 $\pi$ 键——所以顺式消除不利。

**类比**：开门时，门把手和铰链要在门的两侧（反式），才能顺利开门。如果都在一侧（顺式），门就"卡住"了。
:::

:::theorem{label=E2消除的立体选择性}
由于反式消除的要求，E2 反应具有**立体选择性**：

- 从一种构象出发，只能得到特定立体化学的烯烃
- $(E)$-型烯烃（大基团反式）通常比 $(Z)$-型烯烃（大基团顺式）更稳定，是主产物

**例**：2-溴丁烷消除，反式消除得到的主产物是 **$(E)$-2-丁烯**（61%），$(Z)$-2-丁烯较少（20%）。
:::

:::example{label=E2反式消除的产物预测}
**题目**：预测 2-溴丁烷在 $\ce{NaOEt/EtOH}$ 中消除的主产物。

**分析**：
1. 2-溴丁烷 $\ce{CH3-CHBr-CH2-CH3}$ 有两种 $\beta$-H：
   - $\ce{C1}$ 上的 H（3 个）→ 消除得 1-丁烯（反 Zaitsev）
   - $\ce{C3}$ 上的 H（2 个）→ 消除得 2-丁烯（Zaitsev）
2. Zaitsev 产物（2-丁烯）更稳定，是主产物
3. 2-丁烯有 $(E)$ 和 $(Z)$ 两种构型，$(E)$ 更稳定

**解**：

主产物为 **$(E)$-2-丁烯**（约 61%），副产物为 $(Z)$-2-丁烯（约 20%）和 1-丁烯（约 19%）。

**解释**：
- 区域选择性：Zaitsev 产物（2-丁烯）> 反 Zaitsev 产物（1-丁烯），因为 2-丁烯双键上烷基多，更稳定
- 立体选择性：$(E)$-2-丁烯 > $(Z)$-2-丁烯，因为 $(E)$ 型大基团反式，位阻小，更稳定
- 反式消除：$\beta$-H 和 Br 必须处于对位交叉构象
:::

:::example{label=构象分析预测消除产物}
**题目**：新戊基溴代烷 $\ce{(CH3)3C-CH2Br}$ 能否发生 E2 消除？

**分析**：E2 消除要求 $\beta$-碳上有 H。新戊基溴代烷的 $\beta$-碳是 $\ce{(CH3)3C-}$（叔碳），其上**没有 H**（$\beta$-碳上全是甲基，但 $\beta$-碳本身无 H）。

**解**：新戊基溴代烷**不能发生 E2 消除**，因为 $\beta$-碳上没有 H 可消除。

它只能发生 SN2（伯卤代烷，但位阻大，慢）或 SN1/E1（先重排为叔碳正离子，再反应）。
:::

## 消除与取代的竞争

:::theorem{label=影响消除与取代竞争的因素}
卤代烷结构、试剂、溶剂、反应温度均对 SN 和 E 反应的活性有影响。

**1. 卤代烷结构**：
- **伯卤代烷 + 强亲核试剂**：主要 SN2（91% SN : 9% E）
- **仲卤代烷**或 $\beta$-碳上有支链的伯卤代烷：位阻削弱 SN2，增强 E2（60% SN : 40% E）
- **叔卤代烷**：
  - 无强碱：主要 SN1（81% SN : 19% E）
  - 有强碱：主要 E2（3% SN1 : 97% E2）

**2. 进攻试剂**：
- 亲核性强、碱性弱的试剂 → **SN 有利**
- 亲核性弱、碱性强的试剂 → **E 有利**
- 伯卤代烷 + 强亲核试剂 → SN2
- 叔卤代烷 + 强碱性试剂 → E2
- 仲卤代烷 → 介于两者之间
- 强碱存在时 → 主要 E2

**3. 溶剂**：
- 增加极性 → 对单分子反应（SN1, E1）有利（稳定碳正离子）
- 对双分子反应（SN2, E2）不利，但影响不大

**4. 温度**：
- 提高温度 → 有利于**消除反应**（消除熵增 $\Delta S > 0$，高温 $T\Delta S$ 项使 $\Delta G$ 更负）
- 例：某卤代烷消除产物比例，25°C 时 19%，55°C 时 28%
:::

:::insight{label=为什么高温有利于消除**
消除反应：$\ce{RX + B^- -> 烯烃 + HB + X^-}$

反应物 2 个分子 → 产物 3 个分子（+1 个分子），**熵增**（$\Delta S > 0$）。

根据 $\Delta G = \Delta H - T\Delta S$：
- $\Delta S > 0$ 时，$-T\Delta S < 0$，且温度越高 $-T\Delta S$ 越负
- 高温使 $\Delta G$ 更负，反应更有利

而取代反应：$\ce{RX + Nu^- -> RNu + X^-}$

反应物 2 个分子 → 产物 2 个分子，**熵变约 0**。

所以高温对消除反应的"加成"更大，有利于消除。
:::

:::example{label=消除与取代竞争的判断}
**题目**：预测下列反应的主要产物类型（SN 还是 E）。

(1) $\ce{CH3CH2Br + NaOH}$（水溶液，室温）
(2) $\ce{(CH3)3CBr + NaOEt}$（乙醇，加热）
(3) $\ce{CH3CH2CH2Br + KOH}$（乙醇，加热）
(4) $\ce{(CH3)2CHBr + H2O}$（水溶液）

**解**：

**(1)** 伯卤代烷 + 强亲核试剂（$\ce{OH^-}$）+ 水溶液 + 室温 → **SN2**（取代产物 2-丙醇为主）

**(2)** 叔卤代烷 + 强碱（$\ce{EtO^-}$）+ 乙醇 + 加热 → **E2**（消除产物 2-甲基丙烯为主，约 97%）

**(3)** 伯卤代烷 + 强碱（$\ce{OH^-}$）+ 醇溶液 + 加热 → **E2**（醇溶液 + 加热有利于消除，消除产物 1-丙烯为主）

**(4)** 仲卤代烷 + 弱亲核试剂（$\ce{H2O}$）+ 水溶液 → **SN1/E1**（碳正离子机理，混合产物）

**关键规律**：
- 伯卤代烷 + 水溶液 → SN2
- 伯卤代烷 + 醇溶液 + 加热 → E2
- 叔卤代烷 + 强碱 → E2
- 叔卤代烷 + 弱亲核试剂 → SN1/E1
:::

:::pitfall{label=混淆SN/E竞争条件}
**常见错误**：以为强碱一定导致消除，强亲核试剂一定导致取代。

**正确做法**：
- **强亲核 + 弱碱**（如 $\ce{I^-}$、$\ce{CN^-}$、$\ce{RS^-}$）→ SN2 占优
- **强碱 + 弱亲核**（如 $\ce{(CH3)3CO^-}$）→ E2 占优
- **强亲核 + 强碱**（如 $\ce{OH^-}$、$\ce{RO^-}$）→ 看底物和溶剂
  - 伯卤代烷 + 水溶液 → SN2
  - 叔卤代烷 → E2
  - 醇溶液 + 加热 → E2

**口诀**：水溶液取代、醇溶液消除；低温取代、高温消除；伯卤代烷取代、叔卤代烷消除。
:::

:::note{label=本节小结}
- **消除反应**：分子中消去两个原子或基团，最常见的是 $\beta$-消除（生成烯烃）
- **E1 机理**：
  - 单分子，一级反应 $v = k[\ce{RX}]$
  - 分步进行，碳正离子中间体（决速步）
  - 可能重排
  - 与 SN1 竞争
- **E2 机理**：
  - 双分子，二级反应 $v = k[\ce{RX}][\ce{B^-}]$
  - 一步协同，四中心过渡态
  - **反式消除**（H 和 X 对位交叉）
  - 与 SN2 竞争
- **Zaitsev 规则**：主要产物是双键上烷基最多的烯烃（更稳定）
- **反 Zaitsev**：大体积碱或 Hofmann 消除时
- **立体选择性**：$(E)$-烯烃 > $(Z)$-烯烃（更稳定）
- **SN/E 竞争**：
  - 伯卤代烷 + 强亲核 + 水溶液 → SN2
  - 叔卤代烷 + 强碱 → E2
  - 醇溶液 + 加热 → E2
  - 高温有利于消除（熵增）
- 下一节我们将学习不饱和卤代烃和金属有机反应
:::
