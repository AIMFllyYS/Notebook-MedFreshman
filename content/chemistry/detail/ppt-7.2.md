:::insight{label=从"为什么苯不发生加成而发生取代"说起}
上一节我们学到，苯有 124-150 kJ/mol 的离域能，特别稳定。如果苯像烯烃一样发生加成反应，$\pi$ 电子参与成键，离域大 $\pi$ 键就被破坏了——丢失离域能，能量上"不划算"。

那苯环上的氢怎么被替换呢？答案是**亲电取代反应**：亲电试剂 $\ce{E+}$ 进攻苯环，先与一个碳形成 $\sigma$ 键（中间体为 $\sigma$-络合物，保留部分芳香性），然后该碳上的 H 以 $\ce{H+}$ 形式离去，恢复芳香环。整个过程**保留芳香性**，能量上"划算"。

本节我们学习苯环上的五大亲电取代反应：**卤代、硝化、磺化、傅-克烷基化、傅-克酰基化**，并理解它们的机理和特点。
:::

## 亲电取代反应的通式与机理

:::definition{label=苯的亲电取代反应}
**苯的亲电取代反应**（electrophilic aromatic substitution, EAS）：亲电试剂 $\ce{E+}$ 进攻苯环，取代环上的氢，生成取代苯。

$$
\ce{C6H6 + E+ -> C6H5E + H+}
$$

常见的亲电取代反应及对应的亲电试剂 E：

| 反应 | 亲电试剂 $\ce{E+}$ | 引入基团 |
|------|---------------------|----------|
| 卤代 | $\ce{X+}$（$\ce{X = Cl, Br}$） | $\ce{-X}$ |
| 硝化 | $\ce{NO2+}$ | $\ce{-NO2}$ |
| 磺化 | $\ce{SO3}$ | $\ce{-SO3H}$ |
| 傅-克烷基化 | $\ce{R+}$ | $\ce{-R}$ |
| 傅-克酰基化 | $\ce{RCO+}$ | $\ce{-COR}$ |
:::

:::theorem{label=亲电取代反应机理}
亲电取代反应分两步进行：

**第 1 步**（慢，决速步）：亲电试剂 $\ce{E+}$ 进攻苯环，形成 **$\sigma$-络合物**（又称芳基正离子或 Wheland 中间体）

$$
\ce{C6H6 + E+ -> [C6H6E]+}
$$

此时被进攻的碳变为 $\mathrm{sp^3}$，芳香性被局部破坏（只剩 5 个碳的共轭体系，4 个 $\pi$ 电子），能量升高。这是反应的**能垒**。

**第 2 步**（快）：$\sigma$-络合物失去 $\ce{H+}$，恢复芳香性

$$
\ce{[C6H6E]+ -> C6H5E + H+}
$$

被进攻的碳重新变为 $\mathrm{sp^2}$，6 个 $\pi$ 电子完全离域，芳香性恢复，能量大幅降低。

**总结果**：$\ce{E}$ 取代了 $\ce{H}$，芳香环保留。
:::

:::insight{label=为什么是取代而不是加成}
对比两条路径的能量变化：

**加成路径**：$\ce{C6H6 + E-Nu -> C6H6E-Nu}$（破坏芳香性，丢失离域能 150 kJ/mol）
- 产物能量高（无芳香性）
- 不利

**取代路径**：$\ce{C6H6 + E+ -> C6H5E + H+}$（保留芳香性）
- 中间体 $\sigma$-络合物能量较高，但只是过渡态
- 产物恢复芳香性，能量低
- 有利

所以苯环"选择"了取代路径——这是芳香性"驱动"的结果。
:::

## 卤代反应

:::definition{label=苯的卤代反应}
**卤代反应**：苯在 Lewis 酸（$\ce{FeX3}$ 或 $\ce{AlX3}$）催化下与 $\ce{X2}$（$\ce{X = Cl, Br}$）反应，生成卤代苯。

$$
\ce{C6H6 + Br2 ->[FeBr3] C6H5Br + HBr}
$$

**机理**：
1. **Lewis 酸活化**：$\ce{Br2}$ 与 $\ce{FeBr3}$ 配位，使 $\ce{Br-Br}$ 键异裂，产生 $\ce{Br+}$

   $$\ce{Br2 + FeBr3 -> Br+ - FeBr4^-}$$

2. **亲电进攻**：$\ce{Br+}$ 进攻苯环，形成 $\sigma$-络合物（慢）
3. **去质子**：$\sigma$-络合物失去 $\ce{H+}$，恢复芳香性（快），$\ce{FeBr4^-}$ 接受 $\ce{H+}$ 生成 $\ce{HBr}$ + $\ce{FeBr3}$（催化剂再生）
:::

:::theorem{label=卤代反应的特点}
1. **需要 Lewis 酸催化**：$\ce{FeCl3}$、$\ce{FeBr3}$、$\ce{AlCl3}$ 等。无催化剂时苯不与 $\ce{Br2}$ 反应（与烯烃不同）
2. **只发生单取代**：溴苯的活性比苯低（$\ce{-Br}$ 是致钝基团），所以不易继续卤代
3. **氯代和溴代可行**；碘代需要特殊条件（$\ce{I2}$ 亲电性弱）
4. **产率高**：溴代产率约 87%
:::

:::insight{label=为什么苯需要Lewis酸催化而烯烃不需要}
烯烃的 $\pi$ 电子云"暴露"在外，$\ce{Br2}$ 自己就能被极化进攻（形成溴鎓离子）。但苯的 6 个 $\pi$ 电子完全离域，"分散"在 6 个碳上，每个碳的电子云密度比烯烃低，$\ce{Br2}$ 单独进攻"力度不够"。

Lewis 酸 $\ce{FeBr3}$ 的作用是"拉走" $\ce{Br2}$ 的一端，使另一端成为更强的亲电试剂 $\ce{Br+}$，才能"突破"苯环的电子云。
:::

## 硝化反应

:::definition{label=苯的硝化反应}
**硝化反应**：苯与浓硝酸/浓硫酸混合物（混酸）在 50-60°C 反应，生成硝基苯。

$$
\ce{C6H6 + HNO3 ->[H2SO4][50-60°C] C6H5NO2 + H2O}
$$

**亲电试剂**：硝基正离子 $\ce{NO2+}$，由混酸生成：

$$
\ce{HNO3 + 2H2SO4 -> NO2+ + H3O+ + 2HSO4^-}
$$

（$\ce{H2SO4}$ 质子化 $\ce{HNO3}$，脱水生成 $\ce{NO2+}$）
:::

:::theorem{label=硝化反应的特点}
1. **温度控制重要**：50-60°C 为宜
   - 温度过低：反应慢
   - 温度过高：生成多硝基产物（间二硝基苯、1,3,5-三硝基苯）
2. **多硝化需要更强条件**：硝基是致钝基团，第二次硝化需要更高温度（100°C）和发烟硝酸
3. **主要产物**：硝基苯（淡黄色油状液体）
:::

:::example{label=多硝化产物的形成}
**题目**：苯在发烟硝酸/发烟硫酸、110°C 反应 5 天，主产物是什么？

**分析**：高温、发烟条件使硝化连续进行。但硝基 $\ce{-NO2}$ 是**间位定位基**且**致钝**，所以第二个硝基进入第一个硝基的间位。

**解**：连续硝化得到 **1,3,5-三硝基苯**（间位定位，三个硝基相互处于间位）。

中间过程：
- 苯 → 硝基苯（第一硝化）
- 硝基苯 → 1,3-二硝基苯（第二硝化，间位定位，约 80%）
- 1,3-二硝基苯 → 1,3,5-三硝基苯（第三硝化，间位定位）

每一步都需要更剧烈的条件，因为每加一个硝基，环就更不活泼。
:::

## 磺化反应

:::definition{label=苯的磺化反应}
**磺化反应**：苯与浓硫酸在 75°C 反应，生成苯磺酸。

$$
\ce{C6H6 + H2SO4 ->[75°C] C6H5SO3H + H2O}
$$

**亲电试剂**：$\ce{SO3}$（由 $\ce{H2SO4}$ 脱水生成）

**机理**：
1. $\ce{SO3}$ 进攻苯环，形成 $\sigma$-络合物（慢）
2. $\sigma$-络合物失去 $\ce{H+}$，恢复芳香性（快）
:::

:::theorem{label=磺化反应的可逆性}
磺化反应是**可逆反应**——这是五大亲电取代中唯一可逆的！

$$
\ce{C6H6 + H2SO4 <=>[\Delta] C6H5SO3H + H2O}
$$

- **正向**（磺化）：浓硫酸、加热
- **逆向**（脱磺酸基）：稀硫酸、高温蒸汽，或酸性水溶液加热

**应用**：
1. **保护基**：磺酸基可作为占位基，引导后续反应进入特定位置，反应后再水解脱除
2. **分离提纯**：磺酸基使化合物水溶性增大，可用于分离
:::

:::insight{label=磺化可逆性的妙用}
磺化的可逆性在有机合成中非常有用。例如，想在甲苯的某个特定位置引入基团，但该位置不易直接反应，可以：
1. 先磺化，把磺酸基"占"在不想要的位置
2. 再进行目标反应，基团进入想要的位置
3. 最后水解脱除磺酸基

这种"占位-反应-脱保护"的策略是有机合成的重要技巧。
:::

## 傅-克反应——烷基化与酰基化

:::definition{label=傅-克烷基化反应}
**傅-克烷基化反应**（Friedel-Crafts alkylation）：苯在 Lewis 酸（$\ce{AlCl3}$）催化下与卤代烃反应，引入烷基。

$$
\ce{C6H6 + R-Cl ->[AlCl3] C6H5-R + HCl}
$$

**机理**：
1. $\ce{R-Cl}$ 与 $\ce{AlCl3}$ 作用，异裂生成 $\ce{R+}$：

   $$\ce{R-Cl + AlCl3 -> R+ + AlCl4^-}$$

2. $\ce{R+}$ 进攻苯环，形成 $\sigma$-络合物（慢）
3. $\sigma$-络合物失去 $\ce{H+}$，恢复芳香性（快）

**常用烷基化试剂**：卤代烃、烯烃（质子酸催化）、醇
:::

:::theorem{label=傅-克烷基化的三大特点}
1. **重排产物生成**：碳正离子易发生 1,2-迁移，重排为更稳定的碳正离子

   例如正丙基氯与苯反应，$\ce{CH3CH2CH2+}$（伯）重排为 $\ce{(CH3)2CH+}$（仲），主产物为**异丙苯**而非正丙苯

2. **多取代产物生成**：烷基是致活基团，烷基苯比苯更活泼，容易继续烷基化

   例如苯 + $\ce{CH3Cl}$ → 甲苯 → 二甲苯 → 三甲苯 → ...

   需要控制烷基化试剂用量，或苯过量，才能减少多取代

3. **可逆反应**：烷基化可逆，可发生脱烷基化

**限制**：苯环上有强吸电子基（如 $\ce{-NO2}$、$\ce{-COR}$）时，不发生傅-克反应（环太不活泼）。
:::

:::example{label=傅-克烷基化的重排}
**题目**：苯与 1-氯丙烷 $\ce{CH3CH2CH2Cl}$ 在 $\ce{AlCl3}$ 催化下反应，主产物是什么？

**分析**：
1. $\ce{CH3CH2CH2Cl + AlCl3 -> CH3CH2CH2+ + AlCl4^-}$（伯碳正离子）
2. 伯碳正离子不稳定，发生 1,2-氢迁移：$\ce{CH3CH2CH2+ -> (CH3)2CH+}$（仲碳正离子，更稳定）
3. 仲碳正离子进攻苯环

**解**：主产物为**异丙苯**（约 70%），副产物为正丙苯（约 30%）。

$$
\ce{C6H6 + CH3CH2CH2Cl ->[AlCl3] \underset{70\%}{C6H5-CH(CH3)2} + \underset{30\%}{C6H5-CH2CH2CH3}}
$$

**启示**：傅-克烷基化中，碳正离子总往更稳定的方向重排。要引入直链烷基，应该用酰基化反应（不重排），再还原羰基。
:::

:::definition{label=傅-克酰基化反应}
**傅-克酰基化反应**（Friedel-Crafts acylation）：苯在 Lewis 酸（$\ce{AlCl3}$）催化下与酰氯或酸酐反应，引入酰基。

$$
\ce{C6H6 + RCOCl ->[AlCl3] C6H5-COR + HCl}
$$

**机理**：
1. $\ce{RCOCl + AlCl3 -> RCO+ + AlCl4^-}$（酰基正离子）
2. $\ce{RCO+}$ 进攻苯环，形成 $\sigma$-络合物（慢）
3. $\sigma$-络合物失去 $\ce{H+}$，恢复芳香性（快）

**常用酰基化试剂**：酰氯 $\ce{RCOCl}$、酸酐 $\ce{(RCO)2O}$
:::

:::theorem{label=酰基化反应的优势}
与烷基化相比，酰基化有三个优势：

1. **不重排**：酰基正离子 $\ce{RCO+}$ 中正电荷在 C 上，但与 O 共轭（$\ce{R-C#O+ <-> R-C+=O}$），稳定，不发生重排
2. **不多取代**：酰基 $\ce{-COR}$ 是致钝基团（间位定位基），酰基化后环变不活泼，不再继续酰基化 → 单一产物
3. **区域选择性好**：产物纯度高

**应用**：要引入直链烷基，先用酰基化引入酰基，再用 Clemmensen 或 Wolff-Kishner 还原羰基为 $\ce{CH2}$。
:::

:::example{label=由苯合成正丁苯}
**题目**：如何由苯合成正丁苯 $\ce{C6H5-CH2CH2CH2CH3}$？

**分析**：如果直接用 1-氯丁烷进行傅-克烷基化，会发生碳正离子重排，得到仲丁苯（主产物）。要避免重排，应该用酰基化-还原法。

**解**：

**步骤 1**：傅-克酰基化
$$
\ce{C6H6 + CH3CH2CH2COCl ->[AlCl3] C6H5-CO-CH2CH2CH3}
$$
（正丁酰苯，不重排）

**步骤 2**：Clemmensen 还原（$\ce{Zn-Hg/HCl}$）或 Wolff-Kishner 还原（$\ce{N2H4/KOH}$）
$$
\ce{C6H5-CO-CH2CH2CH3 ->[Zn-Hg, HCl] C6H5-CH2CH2CH2CH3}
$$
（正丁苯）

**关键**：酰基化不重排，还原后羰基变为 $\ce{CH2}$，得到直链烷基苯。
:::

:::pitfall{label=傅-克反应的限制条件}
**常见错误**：在含 $\ce{-NO2}$、$\ce{-CN}$、$\ce{-COR}$ 等强吸电子基的苯环上进行傅-克反应。

**正确做法**：傅-克反应**要求苯环上有足够的电子密度**。当苯环上有强吸电子基（致钝基团）时，环太不活泼，**不发生**傅-克反应。

**不发生傅-克反应的基团**：$\ce{-NO2}$、$\ce{-CN}$、$\ce{-COR}$、$\ce{-COOH}$、$\ce{-SO3H}$、$\ce{-CF3}$ 等

**记忆**：所有间位定位基都使苯环不发生傅-克反应。
:::

:::note{label=本节小结}
- 苯的亲电取代反应机理：亲电试剂进攻 → $\sigma$-络合物 → 失去 $\ce{H+}$ 恢复芳香性
- **卤代**：$\ce{X2/FeX3}$，引入 $\ce{-X}$，需要 Lewis 酸催化
- **硝化**：$\ce{HNO3/H2SO4}$，50-60°C，亲电试剂 $\ce{NO2+}$
- **磺化**：$\ce{H2SO4}$，75°C，亲电试剂 $\ce{SO3}$，**可逆反应**（唯一可逆的亲电取代）
- **傅-克烷基化**：$\ce{RCl/AlCl3}$，引入 $\ce{-R}$
  - 特点：重排、多取代、可逆
  - 限制：环上有强吸电子基时不反应
- **傅-克酰基化**：$\ce{RCOCl/AlCl3}$，引入 $\ce{-COR}$
  - 优势：不重排、不多取代、区域选择性好
  - 用于引入直链烷基（先酰基化再还原）
- 下一节我们将学习亲电取代反应的定位规律
:::
