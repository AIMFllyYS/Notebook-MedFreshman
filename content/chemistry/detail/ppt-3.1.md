:::insight{label=从"化学键"到"分子间力"}
前两章我们一直在讨论化学键——原子之间 200-600 kJ/mol 的强相互作用。但化学键只能解释"分子内部"的结构，无法解释"分子之间"为什么会有吸引力。为什么水在常温下是液体而甲烷是气体？为什么 DNA 能形成精确的双螺旋？为什么肥皂能去污？为什么油水不相溶？

这些问题的答案都指向**分子间的弱相互作用**——分子与分子之间 5-100 kJ/mol 的吸引力。虽然比化学键弱得多，但它作用范围广、累加效应强，决定了几乎所有有机物的物理性质（沸点、熔点、溶解性）和生物功能（DNA 配对、蛋白质折叠、药物结合）。

法国化学家 Jean-Marie Lehn 因在**超分子化学**（supramolecular chemistry）领域的开创性工作获得 1987 年诺贝尔化学奖——超分子化学研究的正是"分子间弱相互作用如何组装出复杂结构"。本节我们就来学习三种主要的弱相互作用：**范德华力**、**氢键**、**疏水亲脂作用**。
:::

## 弱相互作用的强度对比

:::theorem{label=弱相互作用与化学键的强度对比}
| 相互作用类型 | 强度（kJ/mol） | 作用范围 |
|-------------|---------------|----------|
| 化学键（共价键） | 200-600 | 短（原子间） |
| 氢键 | 20-30 | 较短 |
| 范德华力 | < 100 | 较广 |

**特点**：分子间作用力比化学键弱，但作用范围比化学键广。
:::

## 范德华作用力

:::definition{label=范德华力}
**范德华力**（van der Waals force）又称"分子间作用力"，是分子间弱相互作用的总称，由荷兰物理学家范德华（1837-1923）首先提出。其特点：

- 分子间弱相互作用；
- 分子接近一定距离时产生；
- 随分子间距离的增加**迅速减弱**。

范德华力包含三种成分：**色散力、诱导力、取向力**。
:::

### 色散力（伦敦力）

:::definition{label=色散力}
**色散力**（dispersion force）是由**瞬时偶极**产生的分子间相互作用力。

非极性分子虽然平均电中性，但在每一瞬间，电子云的分布并不均匀，会产生瞬时偶极。一个分子的瞬时偶极会诱导邻近分子产生相反的偶极，从而相互吸引。

色散力存在于**所有分子之间**：
- 非极性分子-非极性分子
- 极性分子-非极性分子
- 极性分子-极性分子
:::

:::insight{label=色散力是"普遍存在"的}
色散力是三种范德华力中**最普遍**的——任何分子都有电子云，任何电子云都会产生瞬时偶极。即使是完全非极性的 $\ce{CH4}$、$\ce{CO2}$、$\ce{N2}$，也能靠色散力凝聚成液体或固体。色散力的大小主要取决于分子的**极化率**（电子云变形能力），分子越大、电子越多，色散力越强。这就是为什么同系物中分子量越大，沸点越高（甲烷气态、辛烷液态、三十烷固态）。
:::

### 诱导力

:::definition{label=诱导力}
**诱导力**（induction force）是**固有偶极**与**诱导偶极**之间的相互作用力。

当极性分子靠近非极性分子时，极性分子的固有偶极会"拉扯"非极性分子的电子云，使其产生诱导偶极，从而相互吸引。

诱导力存在于：
- 极性分子-非极性分子之间
- 极性分子-极性分子之间（互相诱导）

**必须有极性分子存在时才有诱导力**。
:::

### 取向力

:::definition{label=取向力}
**取向力**（orientation force）是极性分子之间**固有偶极的取向**所产生的吸引力。

当两个极性分子靠近时，它们的固有偶极会自动取向——正极指向负极，负极指向正极——从而相互吸引。

取向力**只存在于极性分子之间**。
:::

:::theorem{label=三种范德华力的存在范围}
| 分子类型 | 色散力 | 诱导力 | 取向力 |
|---------|--------|--------|--------|
| 非极性-非极性 | ✓ | ✗ | ✗ |
| 极性-非极性 | ✓ | ✓ | ✗ |
| 极性-极性 | ✓ | ✓ | ✓ |

记忆口诀：**色散力人人有，诱导力需极性，取向力需双极性**。
:::

:::pitfall{label=以为非极性分子之间没有作用力}
非极性分子之间**有色散力**！很多人误以为非极性分子（如 $\ce{CH4}$、$\ce{N2}$）之间没有任何作用力，所以无法解释为什么甲烷能液化、氮气能凝固。事实上色散力是普遍存在的，只是较弱。非极性分子的沸点高低主要由色散力决定——分子越大，色散力越强，沸点越高。
:::

:::example{label=判断范德华力类型}
**题目**：判断下列分子对之间存在哪些范德华力。

(a) $\ce{CH4}$ 与 $\ce{CH4}$  (b) $\ce{H2O}$ 与 $\ce{CH4}$  (c) $\ce{H2O}$ 与 $\ce{H2O}$

**解**：
- (a) $\ce{CH4}$ 是非极性分子 → 只有**色散力**
- (b) $\ce{H2O}$ 极性，$\ce{CH4}$ 非极性 → **色散力 + 诱导力**
- (c) $\ce{H2O}$ 极性，$\ce{H2O}$ 极性 → **色散力 + 诱导力 + 取向力**（外加氢键，见下文）
:::

## 氢键

:::definition{label=氢键}
**氢键**（hydrogen bond）是分子中与电负性很大的元素相结合的 H 原子，与另一分子中电负性很大的原子之间产生的结合力。表示为：

$$
\ce{X-H \cdots Y}
$$

其中 X、Y 代表 F、O、N 等电负性大的原子。

**氢键的本质是静电作用**：$\ce{X-H}$ 中 X 电负性大，吸引电子云，使 H 带部分正电（$\delta^+$）；Y 有孤对电子，带部分负电（$\delta^-$）；$\delta^+$ 的 H 与 $\delta^-$ 的 Y 相互吸引形成氢键。

氢键键能约 20-30 kJ/mol，介于化学键和范德华力之间。
:::
::figure{src="/images/chemistry/svg/ch03/hydrogen-bond.svg" caption="氢键形成示意图：X—H···Y，X/Y 为 F/O/N，H 带 δ+ 正电，静电吸引" alt="X-H共价键与Y之间的氢键示意图，标注δ-/δ+电荷分布、孤对电子、近180°直线构型"}

:::theorem{label=氢键的方向性与饱和性}
氢键具有**方向性**和**饱和性**（与共价键类似，但比共价键弱）：

1. **方向性**：H 受体 Y 与供体 X 之间的角度接近 $180^\circ$，即 X-H...Y 接近直线。这是因为直线构型使 X、Y 之间的静电斥力最小。
2. **饱和性**：一般情况下，一个 H 受体只与一个 H 原子形成氢键。

氢键的几何构型一般为直线型或稍有弯曲。
:::

:::insight{label=氢键的方向性决定生物大分子结构}
氢键的高度方向性使被氢键连接的两个基团维持特有的几何排列。**DNA 双螺旋结构精准的三维结构即得益于氢键的方向性**：

- 腺嘌呤（A）与胸腺嘧啶（T）之间形成 **2 个氢键**
- 鸟嘌呤（G）与胞嘧啶（C）之间形成 **3 个氢键**

G-C 配对比 A-T 配对更稳定（多 1 个氢键）。这种精确的氢键配对是 DNA 复制、转录的信息基础——只有正确配对才能形成最稳定的氢键网络。
:::

:::theorem{label=氢键键能的影响因素}
氢键键能随 X、Y 原子的电负性增大或半径减小而增大：

$$
\ce{F-H \cdots F} > \ce{O-H \cdots O} > \ce{O-H \cdots N} > \ce{N-H \cdots N}
$$

一般规律：
- F 电负性最大（4.0），半径小 → $\ce{F-H \cdots F}$ 最强
- O 电负性 3.5 → $\ce{O-H \cdots O}$ 次之
- N 电负性 3.1 → $\ce{N-H \cdots N}$ 较弱
:::

### 分子内氢键与分子间氢键

:::definition{label=分子内/分子间氢键}
- **分子内氢键**（intramolecular hydrogen bond）：同一分子内的 H 供体与受体形成的氢键
- **分子间氢键**（intermolecular hydrogen bond）：不同分子之间的 H 供体与受体形成的氢键

典型例子：
- **邻羟基苯甲醛（水杨醛）**：沸点 197 ℃（形成分子内氢键，分子间作用力相对弱）
- **对羟基苯甲醛**：沸点 246 ℃（只能形成分子间氢键，分子间作用力强）
:::
::figure{src="/images/chemistry/svg/ch03/ortho-para-hydroxybenzaldehyde.svg" caption="邻羟基苯甲醛（分子内氢键五元环，沸点197°C）vs 对羟基苯甲醛（分子间氢键，沸点246°C）" alt="邻位：O-H与C=O形成五元环分子内氢键；对位：OH与相邻分子形成分子间氢键链，导致更高沸点"}

:::insight{label=分子内氢键 vs 分子间氢键对沸点的影响}
分子内氢键"消耗"了 H 供体和受体，使分子间无法再形成氢键，因此**分子内氢键化合物的沸点低于分子间氢键化合物**。

例如邻羟基苯甲醛（分子内氢键）沸点 197 ℃，比对羟基苯甲醛（分子间氢键）的 246 ℃ 低 49 ℃。这个差异完全来自氢键类型的不同。

类似地，邻硝基苯酚能形成分子内氢键，对硝基苯酚只能形成分子间氢键。结果邻硝基苯酚沸点 100 ℃（0.009 MPa），对硝基苯酚沸点 194 ℃，间硝基苯酚沸点更高（分解）。
:::

:::example{label=氢键对沸点的影响}
**题目**：解释为什么乙醇（$\ce{CH3CH2OH}$，分子量 46）沸点 78.3 ℃，而甲醚（$\ce{CH3OCH3}$，分子量 46）沸点只有 -24 ℃。

**分析**：两者分子量相同，但乙醇有 $\ce{-OH}$ 可形成分子间氢键，甲醚没有 $\ce{-OH}$ 不能形成氢键。

**解**：
- 乙醇：$\ce{-OH}$ 之间形成分子间氢键（$\ce{O-H \cdots O}$），需额外能量打破氢键才能汽化 → 沸点高（78.3 ℃）
- 甲醚：只有范德华力（色散力 + 诱导力 + 取向力），无氢键 → 沸点低（-24 ℃）

沸点差异 102 ℃ 完全来自氢键的存在与否。这印证了"氢键显著影响物理性质"的结论。
:::

:::pitfall{label=以为所有含 H 的化合物都能形成氢键}
形成氢键需要满足两个条件：
1. **H 供体**：H 必须与高电负性原子（F、O、N）直接相连，使 H 带 $\delta^+$
2. **H 受体**：必须有高电负性原子带孤对电子（F、O、N）

例如 $\ce{CH4}$ 虽然含 H，但 H 与 C 相连（C 电负性 2.5，不够大），H 不带明显 $\delta^+$，无法形成氢键。$\ce{CHCl3}$ 的 H 也与 C 相连，但 C 上连 3 个 Cl 使 C 电负性增强，H 带 $\delta^+$，可以形成弱氢键。
:::

## 疏水亲脂作用

:::definition{label=疏水亲脂作用}
**疏水亲脂作用**（hydrophobic lipophilic interaction）是有机分子在水中表现出的两种相关作用：

- **疏水作用**：有机分子溶于水后，水分子要保持原有结构而**排斥**有机分子的倾向
- **亲脂作用**：有机分子之间的**范德华吸引力**

**胶束的形成**：表面活性剂分子（一端亲水、一端疏水）在水中因疏水亲脂作用聚集形成**胶束**——亲水头朝外与水接触，疏水尾朝内互相靠拢。这就是肥皂去污的原理。
:::

:::insight{label=疏水作用的本质不是"排斥"而是"水的自我保持"}
很多人误以为疏水作用是有机分子"互相排斥水"，其实恰恰相反——是**水分子为了保持自身的氢键网络而"挤走"有机分子**。有机分子一旦进入水中，会破坏水分子之间的氢键网络，水分子为了"恢复秩序"，会尽量减少与有机分子的接触面积，把有机分子"推"到一起。这就是为什么油水分离——不是油分子之间有特殊吸引力，而是水分子"嫌弃"油分子。

胶束的形成正是这种效应的体现：表面活性剂的疏水尾被水"挤"到一起，亲水头与水保持接触，形成球状结构。
:::

:::theorem{label=疏水亲脂作用的应用}
疏水亲脂作用在化学和生物中有广泛应用：

1. **去污**：肥皂、洗涤剂形成胶束，疏水尾包裹油污，亲水头与水结合，把油污"拉"入水中
2. **药物传递**：脂质体利用疏水作用包裹疏水性药物
3. **光催化**：王锋教授团队利用胶束体系在水中进行光催化产氢，胶束的疏水内核浓缩有机催化剂，提高反应效率（TON 从 4.1 提高到 134）

组装体系的优点：
- 省去复杂的合成工作
- 水中质子迁移速率提高
- 局部催化剂浓度增强效应
:::

:::note{label=本节小结}
- 分子间弱相互作用（< 100 kJ/mol）比化学键（200-600 kJ/mol）弱，但作用范围广；
- **范德华力**三种成分：
  - 色散力（瞬时偶极，所有分子间都有）
  - 诱导力（极性分子诱导非极性分子，需有极性分子）
  - 取向力（极性分子间固有偶极取向，只在极性分子间）
- **氢键** $\ce{X-H \cdots Y}$（X、Y = F、O、N）：
  - 键能 20-30 kJ/mol，本质是静电作用
  - 有方向性（接近 $180^\circ$）和饱和性
  - 决定 DNA 双螺旋、水的高沸点等
  - 分子内氢键降低沸点，分子间氢键升高沸点
- **疏水亲脂作用**：水排斥有机分子（疏水）+ 有机分子间范德华力（亲脂）；导致胶束形成、油水分离；
- 下一节我们将学习这些弱相互作用如何影响有机物的沸点、熔点、溶解性。
:::
