# 6.2 亲电加成与马氏规则

## 从"谁去亲谁"说起

上一节的催化加氢，是在催化剂表面悄悄完成的顺式加成，没有"亲电不亲电"的问题。这一节进入烯烃反应真正的主战场——**亲电加成**。这是烯烃反应里机理最丰富、考点最密集的一块，几乎所有"为什么这样加、为什么那样加"的疑问都在这里得到回答。

理解亲电加成，先要把两个名词的字面意思拆开。在有机化学里，试剂进攻底物无非两种取向：**亲核**与**亲电**。

:::definition{label=亲核与亲电}
- **亲核**：字面是"亲近原子核"。原子核带正电，所以亲核试剂自身要带**负电**——要么本身就是负离子（如烷氧基负离子 $\ce{RO-}$），要么是中性分子里因极化而带部分负电的那一端。
- **亲电**：正好相反，"亲近带电的（负电）部分"。亲电试剂自身要提供一个**正电中心**——要么本身就是正离子（如碳正离子），要么是中性分子里因极化而带部分正电的那一端。
:::

烯烃的双键是富电子的、带负电性的区域，它天生就吸引带正电的亲电试剂。**由亲电试剂进攻不饱和键、最终引起加成的反应，就叫亲电加成。** 能跟烯烃发生亲电加成的试剂很多：质子酸（硫酸、氢卤酸、有机酸）、水，以及看似中性、却能被诱导出瞬时偶极的卤素单质。

## 一、亲电加成的通式与两步机理

### 1. 两步走，慢一步、快一步

:::theorem{label=亲电加成的两步机理}
烯烃的亲电加成一般**分两步**进行：
- **第一步（决速步，慢）**：亲电试剂的正电端去进攻富电子的双键。双键的这对 $\pi$ 电子去结合正电中心，迫使亲电试剂原有的键发生**异裂**——键断开后，一对电子随负电端离去，正电端则接到双键的一个碳上。结果双键打开，生成一个**碳正离子**中间体。这一步要断键、要拆开稳定的双键，最耗能，是整个反应的**决速步**。
- **第二步（快）**：上一步离去的负离子（或负电端）很快回过头来，结合到碳正离子上。正电荷和负电荷的结合非常迅速，几乎没有悬念。
:::

通式可表示为（以 $\ce{HX}$ 为例）：

$$
\ce{C=C + H+ -> \overset{+}{C}-C-H ->[\ce{X-}] X-C-C-H}
$$

:::insight{label=碳正离子是理解一切的钥匙}
回忆烷烃那一章：自由基取代的中间体是碳自由基，碳自由基的稳定性决定了反应的取向。亲电加成完全类似，只是中间体换成了**碳正离子**。烯烃亲电加成的许多性质——区域选择性（马氏规则）、重排、立体化学——归根到底都来自"第一步生成了什么样的碳正离子、它有多稳定"。抓住碳正离子，这一整节就提纲挈领了。
:::

### 2. 亲电加成是一台"转化机器"

烯烃和不同的亲电试剂反应，能转化成五花八门的有机物，这正是亲电加成在合成上极其重要的原因：

:::example{label=同一个烯烃，转出多种产物}
- 与**氢卤酸** $\ce{HX}$ 加成 → **卤代烷**；
- 与**硫酸**加成 → 硫酸氢酯，但此酯不稳定，**一水解就变成醇**，所以与硫酸加成实质相当于与水加成，最终得**醇**；
- 与**有机酸**（羧酸的羟基）加成 → **酯**；
- 与**醇**（羟基）加成 → **醚**。

可见烯烃是一个"转化枢纽"：从一个烯烃出发，能得到卤代烷、醇、酯、醚等常见小分子。烯烃又是石油炼化得到的大宗工业原料，所以这类转化在工业上意义重大。
:::

## 二、马氏规则：现象与本质

### 1. 一个让人困惑的实验事实

把烯烃和氢卤酸加成时，如果烯烃是**不对称**的（双键两端取代情况不同），就会出现一个问题：氢和卤素到底各加到哪个碳上？

以丙烯 $\ce{CH3-CH=CH2}$ 与 $\ce{HBr}$ 加成为例，理论上有两种加法。实验给出的数据很悬殊：溴加到含氢较少的中间碳、氢加到端基碳这种方式约占 80%，反过来只占 20%。换成取代更多的不对称烯烃，选择性更强，主产物甚至能达到 90%、100%。这说明反应绝不是随机的，而是有明显**区域选择性**。

:::definition{label=马尔科夫尼科夫规则}
不对称烯烃与氢卤酸等极性试剂发生亲电加成时，酸中的**氢原子加到含氢较多的双键碳上**，其余的原子或基团加到含氢较少的双键碳上。简称"**氢上加氢**"。这一经验规律由俄国化学家马尔科夫尼科夫总结，简称**马氏规则**，符合它的产物叫**马氏产物**，相反的叫**反马氏（反马）产物**。
:::

马氏当年只是从大量实验里总结出这条经验规律，**并不知道为什么**。今天我们能用碳正离子机理把它彻底讲清楚。

### 2. 用电子效应 + 碳正离子稳定性解释马氏规则

理解马氏规则，要先记住录音里反复强调的一点：双键上的取代基，主要影响的是**活泼的 $\pi$ 电子云**的流动方向。

:::insight{label=取代基"吸/推"的是π电子}
$\sigma$ 键束缚电子很紧、极性变化小；$\pi$ 键柔韧、电子云容易被基团"推"或"拉"。
- 烷基（如甲基）是**弱推电子基**：它把 $\pi$ 电子往远端推，于是**远端（含氢较多的端基碳）带上部分负电、近端带部分正电**。
- 没有取代基时，$\pi$ 电子是对称分布的，无所谓哪端富电。
:::

现在沿着两步机理走一遍：

:::derivation{label=为什么"氢上加氢"是主路}
**第一步看进攻取向（电子效应）。** 甲基把 $\pi$ 电子推向端基碳，使端基碳成为电子云密度更高的负电中心。亲电的质子 $\ce{H+}$ 总是去找电子最多的地方，所以它**优先进攻端基碳**——这就在第一步就定下了优势路线。

**第二步看中间体稳定性（碳正离子）。** 质子加到端基碳后，正电荷（碳正离子）就落在了中间那个碳上，而这个碳是**取代基更多**的碳。回忆碳正离子稳定性规律：取代基越多越稳定（更多的 $\sigma$-p 超共轭、离域范围更广），三级 > 二级 > 一级。所以走这条路生成的碳正离子也**更稳定**。

**两个因素指向同一条路。** 无论从"进攻时底物哪端更富电"还是从"生成的碳正离子哪个更稳定"分析，质子进攻端基碳都是优势选择。生成更稳定的碳正离子后，卤负离子再结合上去，就得到主要的"氢上加氢"马氏产物。另一条路也存在，只是不占主导，给出少量反马产物。
:::

:::note{label=两个判据其实是一回事}
"进攻富电子端"和"生成更稳定碳正离子"在马氏体系里**总是一致地**指向同一个产物。这不是巧合：推电子基让端基碳富电，质子加上去后，正电荷自然落到取代多的碳上，而取代多正是碳正离子稳定的原因。记住核心一句：**亲电加成的决速步要生成尽可能稳定的碳正离子。**
:::

## 三、吸电子基下的"规则反转"

马氏规则是**经验规律**，经验规律的特点是——有时候靠不住。一旦双键上的取代基从推电子基换成强吸电子基，规则就要反过来。

:::theorem{label=吸电子基使马氏规则反转}
把丙烯的甲基换成三氟甲基 $\ce{-CF3}$（强吸电子基），$\pi$ 电子的流向就**反过来**了：电子被吸向 $\ce{-CF3}$ 一端，于是**靠近 $\ce{-CF3}$ 的碳带部分负电、端基碳带部分正电**。质子先去找负电中心，进攻取向整个反转，最终得到的是**反马氏产物**。

所以：**马氏规则只在双键碳连有给电子基团时成立；连有强吸电子基（如 $\ce{-CF3}$、$\ce{-CN}$、$\ce{-NO2}$、$\ce{-COOH}$）时，规则反着来。**
:::

:::insight{label=动力学与热力学的视角}
录音里用了个生动的比方：动力学像"单脚跳谁跳得远"，热力学像"跳得远还要稳得住"。
- 当电子效应**明显**时（如强吸电子基），第一步进攻的取向被电子效应牢牢决定，反应沿这条**最快**的动力学路径走下去，给出**动力学产物**——此时取向反转。
- 当没有明显的电子效应竞争时，**热力学**才出来起作用，生成更稳定结构的那条路成为主导。

吸电子基把动力学取向掰反了，所以反马产物成了主产物。
:::

:::pitfall{label=卤原子是个"两面派"，结果仍是马氏}
不要一看到吸电子基就喊"反马"。**卤原子（如 $\ce{Cl}$）连在双键上时,加成产物仍是马氏产物。** 原因：氯虽有吸电子的诱导效应，但当氢按马氏方式加成、碳正离子生成在与氯相连的碳上时，氯能用自身孤对电子的 p 轨道与碳正离子形成 **$\pi$-$\pi$ 共轭**，把这个碳正离子稳定下来。这个共轭稳定作用主导了产物方向，所以最终还是"氢上加氢"。同理，连有饱和氧（如乙氧基 $\ce{-OC2H5}$）时，氧也能提供 p 轨道形成共轭，产物方向同样由共轭效应决定。判断时务必把诱导效应和共轭效应**通盘考虑**，而不是只记"吸电子=反马"这种表象。
:::

## 四、碳正离子重排

既然亲电加成的中间体是碳正离子，就要时刻警惕它会"篡位"——发生重排。

:::definition{label=碳正离子重排}
碳正离子生成后，相邻碳上的一个基团（如甲基、氢）连同一对电子迁移到带正电的碳上，从而把正电荷"挪"到相邻碳上，生成一个新的、**更稳定**的碳正离子。这种正电荷在相邻碳之间的迁移叫**碳正离子重排**。
:::

:::theorem{label=重排发生的三个条件}
1. **只在相邻碳之间**发生——不会远端迁移，迁移基团必须就在碳正离子的隔壁碳上。
2. **重排后必须更稳定**——重排后的碳正离子要比重排前更稳定（如二级 → 三级），这是重排的驱动力。
3. **只重排一次**——排成一个更稳定的就停下，不会接连重排。
:::

:::example{label=甲基迁移导致"篡位"}
某个 4 碳不对称烯烃（双键在 1、2 号碳之间）在酸性条件下与 $\ce{HCl}$ 加成，主产物的氯竟然落在了**3 号碳**上（约 83%），而正常的马氏产物（氯在 2 号碳）只占约 17%。过程是这样的：
1. 质子进攻 1 号碳，碳正离子先生成在 2 号碳上（**二级**碳正离子）——氯若此刻结合，就是那 17% 的马氏产物；
2. 但相邻 3 号碳上的一个甲基连着电子迁移过来（**甲基迁移**），正电荷转到 3 号碳，生成更稳定的**三级**碳正离子；
3. 氯负离子结合到 3 号碳上 → 这就是 83% 的重排主产物。
:::

:::pitfall{label=见到碳正离子就要"长个心眼"}
录音里反复叮嘱：以后凡是机理涉及碳正离子的反应（氢卤酸加成、水合，乃至后面会学到的醇脱水、傅克烷基化），都要**多长一个心眼**——只要满足"相邻碳重排一次能得到更稳定的碳正离子"这个条件，重排就会发生，而且**重排产物往往成为主产物**。同样要记住逻辑：是先在实验里发现了那个"反常"的多数产物，才**推测**发生了重排，不是先规定它重排。
:::

## 五、与水的加成（水合）

水合是亲电加成大家族里和氢卤酸加成"如出一辙"的一员，只是产物不同。

:::definition{label=烯烃的酸催化水合}
在酸催化下，烯烃与水加成生成醇。机理仍是亲电加成两步：质子先发起亲电进攻生成碳正离子，水分子用孤对电子结合碳正离子，再脱去一个质子，最终得到醇。
:::

$$
\ce{CH3-CH=CH2 ->[\ce{H2O}][\ce{H+}] CH3-CH(OH)-CH3}
$$

:::note{label=水合的三个特征}
1. **遵守马氏规则**：氢加到含氢较多的碳上，羟基落在取代多的碳上（如丙烯水合得 2-丙醇）。
2. **中间体是碳正离子**：因此**同样可能发生重排**，主产物可能是重排醇。
3. 这是工业上由烯烃制醇的重要方法。
:::

## 六、与卤素的加成：环鎓离子与反式加成

烯烃和卤素单质（如 $\ce{Br2}$）加成生成邻二卤代烷，但它的机理和立体化学都很特别。

### 1. 中性卤素怎么变成亲电试剂？

卤素分子本是非极性的，靠什么进攻富电子的双键？答案是**动态诱导极化**：当 $\ce{Br2}$ 慢慢靠近双键时，双键的电子云使它发生极化，靠近双键的那头感应出正电荷、远端感应出负电荷，于是正电端就成了亲电进攻端。

### 2. 不是碳正离子，而是环鎓离子

:::theorem{label=环溴鎓离子与反式加成}
卤素加成的第一步**不是**生成开放的碳正离子，而是溴"一头扎进"双键、与两个碳同时成键，形成一个**三元环的环溴鎓离子**，另一个溴以 $\ce{Br-}$ 形式离去——这是决速步。第二步，离去的 $\ce{Br-}$ 只能从环鎓离子的**背面**进攻（正面被三元环挡住了），于是两个溴最终落在双键平面的**两侧**，得到**反式加成**产物。
:::

$$
\ce{\text{环己烯} ->[\ce{Br2}] \text{反式-1,2-二溴环己烷}}
$$

:::insight{label=为什么唯独这里要谈"顺反"}
录音里点破了一个常被忽略的逻辑：催化加氢谈顺反、硼氢化谈顺反、卤素加成谈顺反，**唯独碳正离子型的亲电加成（氢卤酸、水合）不谈顺反**。原因是——碳正离子是**平面**结构，第二步负离子从平面上下两侧进攻**机会均等**，产物无所谓顺反。而卤素加成走的是环鎓离子，背面进攻的几何限制强制产生了反式，所以这里顺反就有意义了。一句话：**有平面碳正离子就没顺反，有环鎓离子就一定反式。**
:::

### 3. 分步进行的实验证据

:::example{label=氯化钠水溶液里的"插足"现象}
在含 $\ce{NaCl}$ 的水溶液中让乙烯和 $\ce{Br2}$ 加成，除了得到 1,2-二溴产物，还检出了**含氯**的加成产物（一个溴、一个氯）。这怎么解释？正因为反应**分两步**：第一步先生成环溴鎓离子，第二步才由负离子进攻。体系里既有 $\ce{Br-}$ 也有 $\ce{Cl-}$，$\ce{Cl-}$ 同样能去进攻环鎓离子，于是"插足"生成了氯代产物。这个结果反过来**证明了加成是分步进行的**——若是一步同时加两个溴，就不会给氯任何机会。
:::

卤素与烯烃的反应活性一般为 $\ce{Cl2 > Br2 > I2}$（亲电性次序），氟则过于剧烈、难以可控加成。

## 七、硼氢化反应：四元环过渡态、反马、顺式

亲电加成生成的醇都是马氏醇。如果我们想要**反马氏的醇**怎么办？答案是硼氢化氧化——它借助乙硼烷，实现"反马 + 顺式"的水合。

:::definition{label=硼氢化氧化}
烯烃先与乙硼烷 $\ce{B2H6}$（甲硼烷常以氢桥键二聚成乙硼烷形式存在）加成，再用**碱性双氧水** $\ce{H2O2/OH-}$ 氧化处理，总效果相当于给烯烃加上一分子水生成醇，但得到的是**反马氏的醇**。
:::

$$
\ce{R-CH=CH2 ->[\ce{B2H6}][1] ->[\ce{H2O2,\ OH-}][2] R-CH2-CH2-OH}
$$

:::derivation{label=为什么是反马、又是顺式}
关键在硼氢键的极性。查电负性：硼约 2.04，氢约 2.1，**氢比硼略大**。所以硼氢键里**氢是吸电子端（带部分负电）、硼是缺电子端（带部分正电）**——这和碳氢键正好相反。

把硼氢键的极性和不对称烯烃的极性"耦合"起来（正找负、负找正），硼会去接富电子端、氢会去接另一端。结果是**氢加到了含氢较少的碳上**——这正是反马氏取向。

而且这一步是经过一个**四元环过渡态**的**协同**加成：硼和氢同时、从双键的同一侧加上去，所以是**顺式加成**。最后一步碱性双氧水把"硼那部分"直接氧化成羟基，羟基就停在了硼原来所在的位置，于是得到反马氏、且立体上顺式的醇。
:::

:::note{label=硼氢化的两种收尾}
第一步（硼氢化、过四元环过渡态、顺式）完全相同，第二步条件不同：
- **碱性 $\ce{H2O2}$ 氧化** → 硼变羟基 → 得**醇**（反马氏、顺式），这是"硼氢化氧化"，由烯制醇；
- **用酸处理（烷基化）** → 硼那部分变成质子 → 得**烷烃**，这是"硼氢化烷基化"，由烯制烷（此时加的是两个氢，不存在马氏不马氏的问题）。
:::

## 八、合成视角：互补的加成路线

把这些反应配成对，烯烃就成了灵活的合成起点：

:::example{label=制醇的两条互补路线}
- 要**马氏醇**（羟基在取代多的碳上）→ 用**酸催化水合**（亲电加成）。
- 要**反马氏醇**（羟基在取代少的碳上）→ 用**硼氢化氧化**。
两条路在区域选择性上正好互补，给你哪种醇你就选哪条路——这就是"合成设计"。
:::

## 本节小结

:::note{label=本节小结}
1. **亲电加成两步走**：第一步亲电试剂进攻双键、生成碳正离子（**决速步、慢**）；第二步负离子结合（**快**）。一切性质都从碳正离子来。
2. **马氏规则**："氢上加氢"。本质 = 电子效应让质子进攻富电子端 + 生成更稳定的碳正离子，两个判据指向同一产物。
3. **规则反转**：双键连**强吸电子基**（$\ce{-CF3}$、$\ce{-CN}$、$\ce{-NO2}$）时取向反转 → 反马氏产物。但**卤原子、饱和氧**因能与碳正离子形成 $\pi$-$\pi$ 共轭，产物仍是马氏。
4. **碳正离子重排**：相邻碳迁移一次得到更稳定碳正离子时发生，重排产物常为主产物；见碳正离子就要警惕。
5. **水合**：酸催化、马氏、可能重排，由烯制醇。
6. **卤素加成**：经**环溴鎓离子**、背面进攻 → **反式加成**；分步机理由"$\ce{Cl-}$ 插足"实验佐证。
7. **硼氢化氧化**：硼氢键极性反常（氢带负电），经**四元环过渡态**协同加成 → **反马氏 + 顺式**的醇；改用酸处理则得烷烃。
8. **顺反口诀**：碳正离子（平面）不谈顺反；环鎓离子一定反式；硼氢化一定顺式。
下一节我们看烯烃的另一条加成路线——自由基加成，以及烯烃的氧化反应。
:::
