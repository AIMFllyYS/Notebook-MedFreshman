# ppt-18.2 多肽与蛋白质

:::insight{label=从"胰岛素为什么能拿诺贝尔奖"说起}
1955 年，Frederick Sanger 测定了牛胰岛素的氨基酸序列——这是人类第一次测出蛋白质的一级结构，他因此获得 1958 年诺贝尔化学奖。1965 年，中国科学家首次人工合成牛胰岛素，这是人类第一次人工合成蛋白质。为什么测定氨基酸序列如此重要？因为**一级结构决定高级结构，高级结构决定功能**——胰岛素的 51 个氨基酸排列顺序，决定了它能调节血糖。

蛋白质是生命的基础物质，从酶、激素、抗体到血红蛋白，都是蛋白质。本节我们学习多肽的肽键、一级结构测定、蛋白质的四级结构和性质。
:::

## 多肽

:::definition{label=多肽与肽键}
**肽**（peptide）：多个氨基酸相互之间脱水，以**酰胺键**连接的形式形成的链状缩合物。

**肽键**（peptide bond）：一个氨基酸的氨基和另一个氨基酸的羧基失水形成的**酰胺键**：

$$
\ce{-CO-NH-}
$$

**氨基酸残基**：肽分子中的氨基酸重复单元（已脱水的部分）。

**多肽**：由 2-50 个左右氨基酸以肽键结合起来的链状分子。

**末端**：
- **N-端**：有游离氨基的一端（通常是肽链的"开始"）
- **C-端**：有游离羧基的一端（通常是肽链的"结束"）

**命名**：从 N-端开始，依次写出氨基酸残基，最后加"肽"。如：
- 甘氨酰-甘氨酸（Gly-Gly）：二肽
- 丙氨酰-甘氨酰-丝氨酸（Ala-Gly-Ser）：三肽
:::

:::theorem{label=肽键的结构特点}
**肽键的平面性**：

肽键中 $\ce{C-N}$ 键长介于正常的 $\ce{C-N}$ 单键（$147 \text{ pm}$）和 $\ce{C=N}$ 双键（$127 \text{ pm}$）之间，约为 **$132 \text{ pm}$**。

**原因**：肽键中的氮原子与羰基形成 **$\mathrm{p-}\pi$ 共轭**，使 $\ce{C-N}$ 键具有部分双键特征。

**后果**：
1. 肽键 $\ce{C-N}$ 键**不能自由旋转**（双键特征）
2. 肽单元（$\ce{-CO-NH-}$）具有**刚性平面结构**——肽键平面
3. 肽键平面上的各原子可出现**顺反异构**现象
4. 一般**反式构型较稳定**（$\ce{C=O}$ 和 $\ce{N-H}$ 在两侧）

**意义**：肽键的平面性限制了多肽链的构象，是蛋白质高级结构形成的基础。
:::

:::example{label=肽的命名与结构}
**题目**：写出三肽 Ala-Gly-Ser 的结构，指出肽键、N-端和 C-端。

**解**：

**结构**：

$$
\ce{H2N-CH(CH3)-CO-NH-CH2-CO-NH-CH(CH2OH)-COOH}
$$

- **N-端**：丙氨酸（Ala），有游离氨基 $\ce{H2N-}$
- **C-端**：丝氨酸（Ser），有游离羧基 $\ce{-COOH}$
- **肽键 1**：Ala 与 Gly 之间的 $\ce{-CO-NH-}$
- **肽键 2**：Gly 与 Ser 之间的 $\ce{-CO-NH-}$

**命名**：丙氨酰-甘氨酰-丝氨酸

**关键**：肽键是酰胺键，有部分双键性质，不能自由旋转，形成刚性平面。
:::

## 多肽一级结构的测定

:::definition{label=多肽一级结构}
**多肽的一级结构**（primary structure）：各种氨基酸在多肽链中的**排列顺序**。

**测定内容**：
1. 组成多肽的氨基酸**种类**（酸性水解后分析）
2. 组成多肽氨基酸的**数目**
3. 氨基酸在肽链中的**排列顺序**（末端残基分析法、部分水解法）
:::

:::theorem{label=多肽一级结构测定方法}
**1. 氨基酸组成分析**：
- **酸性水解**：用 $6 \text{ mol/L}$ $\ce{HCl}$，$110°C$，水解 24 小时，将肽链完全水解为游离氨基酸
- 用氨基酸分析仪（离子交换色谱）测定各氨基酸的种类和数量

**2. 末端残基分析法**：
- **N-端分析**：
  - **Sanger 法**：2,4-二硝基氟苯（DNFB）与 N-端氨基反应，水解后用纸色谱鉴定
  - **Edman 降解**：苯异硫氰酸酯（PITC）与 N-端氨基反应，依次切下 N-端氨基酸，可循环测定多个残基
- **C-端分析**：
  - **羧肽酶法**：羧肽酶从 C-端依次切下氨基酸

**3. 部分水解法**：
- 用酶或酸部分水解肽链，得到较小的肽片段
- 测定各片段的序列
- 通过片段的"重叠"推断完整序列

**Edman 降解的优势**：可循环进行，每次切下一个 N-端氨基酸，是自动化测序的基础。
:::

## 蛋白质

:::definition{label=蛋白质}
**蛋白质**（protein）：由氨基酸通过酰胺键（肽键）连接而成的**聚酰胺类高分子**，相对分子量从一万到数千万。

**元素组成**：$\ce{C}$、$\ce{H}$、$\ce{N}$、$\ce{O}$、$\ce{S}$、$\ce{P}$、$\ce{Fe}$、$\ce{Cu}$、$\ce{Zn}$、$\ce{Mn}$、$\ce{I}$ 等。

**意义**：蛋白质是生命的物质基础，是生物体内极为重要的功能大分子，是所有细胞的重要组成部分。
:::

:::definition{label=蛋白质的分子结构}
**一级结构**（primary structure）：
- 多肽链中 α-氨基酸残基的**排列顺序**
- **主键**：肽键
- **副键**：氢键、离子键、疏水键、二硫键、酯键等

**二级结构**（secondary structure）：
- 蛋白质分子中的肽链段**盘绕折叠**成不同的形状，即不同的空间构象
- **维持键**：氢键（主要副键）
- **主要类型**：
  - **α-螺旋**（α-helix）：多肽链主链骨架围绕一个轴螺旋上升，形成螺旋式构象
  - **β-折叠**（β-pleated sheet）：较伸展的锯齿形主链构象，两条 β-折叠平行排布，以氢键作用构成 β-折叠片

**三级结构**（tertiary structure）：
- 在二级结构基础上，多肽链在三维空间的**卷曲、折叠、盘绕**形成的紧密高级结构
- 作用力是氨基酸残基侧链的相互作用
- 两条或多条肽链构成的特定三级结构称为**亚基**

**四级结构**（quaternary structure）：
- 少数**亚基**之间通过非共价键相互作用缔合在一起形成蛋白质的四级结构
- 亚基的缔合或组织方式称为蛋白质的四级结构
:::

:::theorem{label=α-螺旋与β-折叠}

| 特征 | α-螺旋 | β-折叠 |
|------|--------|--------|
| **形状** | 螺旋上升 | 锯齿形伸展 |
| **氢键** | 链内（第 i 个残基与第 i+4 个残基之间） | 链间（不同肽链之间） |
| **紧密度** | 紧凑 | 较伸展 |
| **典型例子** | 毛发角蛋白 | 丝心蛋白 |

**α-螺旋特点**：
- 每圈螺旋含 3.6 个氨基酸残基
- 螺距 $0.54 \text{ nm}$
- 氢键方向与螺旋轴平行
- 右手螺旋（L-氨基酸形成）

**β-折叠特点**：
- 肽链伸展成锯齿形
- 两条或多条肽链平行或反平行排列
- 氢键在链间形成
- 反平行 β-折叠更稳定
:::

:::example{label=蛋白质结构层次}
**题目**：简述蛋白质四个结构层次及其维持力。

**解**：

1. **一级结构**：氨基酸的排列顺序
   - 维持力：**肽键**（共价键）+ 二硫键

2. **二级结构**：多肽链主链的局部构象（α-螺旋、β-折叠）
   - 维持力：**氢键**（主链 $\ce{C=O}$ 与 $\ce{N-H}$ 之间）

3. **三级结构**：整条肽链在三维空间的折叠
   - 维持力：**侧链相互作用**（疏水键、氢键、离子键、二硫键、范德华力）

4. **四级结构**：多个亚基的缔合方式
   - 维持力：**亚基间非共价键**（疏水键、氢键、离子键）

**关键**：一级结构是共价键（肽键）决定；二级、三级、四级结构主要是非共价键决定。一级结构决定高级结构，高级结构决定功能。
:::

## 蛋白质的性质

:::definition{label=蛋白质的等电点与两性}
**蛋白质的等电点**（pI）：当蛋白质所带的正、负电荷数相等，**净电荷为零**时，此时溶液的 pH 值即为蛋白质的等电点。

**不同蛋白质具有不同的等电点**（因氨基酸组成不同）。

**pH 与蛋白质电荷**：
- $\text{pH} < \text{pI}$：蛋白质带正电（阳离子）
- $\text{pH} = \text{pI}$：蛋白质净电荷为零（在电场中不泳动）
- $\text{pH} > \text{pI}$：蛋白质带负电（阴离子）

**等电点时的性质**：
- 蛋白质容易**聚集形成沉淀**析出
- 溶解度最小
- **电泳分离**：不同 pI 的蛋白质在电场中迁移方向和速度不同
:::

:::definition{label=蛋白质的胶体性质与沉淀}
**胶体性质**：蛋白质是高分子化合物，具有胶体性质（如**丁达尔现象**——光通过蛋白质溶液时出现光路）。

**蛋白质的沉淀方法**：
1. **盐析**：加入高浓度中性盐（如 $\ce{(NH4)2SO4}$），破坏蛋白质水化膜，使其沉淀（**可逆**，透析后可恢复）
2. **生物碱试剂沉淀**：加入生物碱试剂（如三氯乙酸、钨酸），与蛋白质结合沉淀
3. **重金属离子沉淀**：加入重金属盐（如 $\ce{CuSO4}$、$\ce{Pb(Ac)2}$），与蛋白质结合沉淀
4. **有机溶剂沉淀**：加入乙醇、丙酮等有机溶剂，破坏水化膜

**蛋白质的变性**：
- 受热、高压、紫外线、强酸强碱、重金属盐、生物碱等因素使蛋白质**空间结构破坏**，失去生理活性
- **不可逆**（一级结构不变，但高级结构破坏）
- **应用**：高温消毒（杀菌）、低温保存（防变性）
:::

:::definition{label=蛋白质的颜色反应}
**茚三酮反应**：蛋白质与茚三酮水溶液共热，生成**蓝紫色**化合物，用于**鉴别蛋白质**（因蛋白质含 α-氨基酸单元）。

**其他显色反应**：
- **双缩脲反应**：蛋白质与 $\ce{CuSO4}$/NaOH 反应显紫红色（肽键反应）
- **黄色反应**：含芳香族氨基酸的蛋白质遇浓硝酸显黄色（黄蛋白反应）
:::

:::pitfall{label=混淆蛋白质沉淀与变性}
**常见错误**：以为所有沉淀都使蛋白质变性。

**正确做法**：
- **盐析**：沉淀但**不变性**（可逆，透析后恢复活性和溶解度）
- **重金属盐、强酸强碱、加热**：沉淀**且变性**（不可逆，活性丧失）

**记忆**：盐析"温和"可逆；变性"剧烈"不可逆。
:::

:::pitfall{label=混淆蛋白质结构层次}
**常见错误**：把二级结构与三级结构混淆。

**正确做法**：
- **二级结构**：肽链**主链**的局部构象（α-螺旋、β-折叠），由**氢键**维持
- **三级结构**：**整条肽链**（包括侧链）在三维空间的折叠，由**侧链相互作用**维持
- **四级结构**：**多条肽链（亚基）**之间的缔合方式

**层次关系**：一级（序列）→ 二级（局部构象）→ 三级（整链折叠）→ 四级（亚基缔合）
:::

:::note{label=本节小结}
- **肽**：氨基酸以酰胺键（肽键）连接的链状缩合物
- **肽键**：$\ce{-CO-NH-}$，有部分双键性质（$\mathrm{p-}\pi$ 共轭），不能自由旋转
- **肽键平面**：刚性平面结构，一般反式构型稳定
- **N-端**（游离氨基）、**C-端**（游离羧基）；命名从 N-端开始
- **一级结构**：氨基酸排列顺序；主键肽键，副键氢键/离子键/疏水键/二硫键/酯键
- **一级结构测定**：
  - 氨基酸组成：酸性水解 + 色谱分析
  - 末端分析：Sanger 法（N-端）、Edman 降解（N-端循环）、羧肽酶（C-端）
  - 部分水解：片段重叠推断序列
- **蛋白质**：氨基酸通过肽键连接的聚酰胺高分子，分子量一万到数千万
- **二级结构**：肽链主链局部构象（α-螺旋、β-折叠），氢键维持
- **三级结构**：整条肽链三维折叠，侧链相互作用维持
- **四级结构**：亚基间非共价键缔合
- **α-螺旋**：链内氢键，每圈 3.6 残基，螺距 0.54 nm，右手螺旋
- **β-折叠**：链间氢键，锯齿形伸展，反平行更稳定
- **等电点**：净电荷为零时的 pH；pI 时溶解度最小，易沉淀
- **胶体性质**：丁达尔现象
- **沉淀方法**：盐析（可逆）、生物碱试剂、重金属离子、有机溶剂
- **变性**：高温/高压/紫外/强酸强碱/重金属盐 → 空间结构破坏，活性丧失，不可逆
- **颜色反应**：茚三酮（蓝紫色）、双缩脲（紫红色）、黄蛋白反应（黄色）
- 本章完，有机化学课程全部结束
:::
