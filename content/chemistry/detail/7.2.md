# 7.2 炔烃的氧化与有机金属衍生物

## 从"把分子剪开"与"把分子接长"两个方向看炔烃

上一节我们把炔烃的结构、酸性和加成讲完了。这一节我们换两个视角继续认识炔烃。

第一个视角是**氧化**——也就是"把分子从三键处剪开"。氧化既能让我们认识炔烃的反应性，更妙的是它能反过来**当侦探用**：把氧化得到的碎片拼回去，就能推断原来三键长在哪里。

第二个视角是**有机金属衍生物**——这是真正让有机化学"长出骨架"的关键。上一节我们已经见过端基炔被强碱夺氢生成炔化物，这一节我们把炔基钠/锂、格氏试剂、烷基锂这一整套"提供碳负离子"的试剂讲透，看它们如何**构筑新的碳碳键**，把小分子拼成大分子。一个"剪"，一个"接"，正好是合成化学的两只手。

为完整起见，我们也补上炔烃的硼氢化与亲核加成两种加成方式，把上一节的加成谱系收尾。

## 一、炔烃的硼氢化反应

炔烃也能发生硼氢化反应——和乙硼烷 $\ce{B2H6}$（有时写成硼烷 $\ce{BH3}$，二者等效）反应。和烯烃一样，硼氢化有两种走向。

:::definition{label=炔烃的硼氢化}
炔烃与乙硼烷加成，硼氢键跨过三键形成四元环过渡态，硼加到位阻较小（含氢较多）的炔碳上，氢加到另一碳，**只加成一次**停在烯基硼上。随后：
- **硼氢化—烷基化**（用酸处理）：最终生成**顺式烯烃**；
- **硼氢化—氧化**（用 $\ce{H2O2}$/碱处理）：先得到**烯醇**结构，再互变异构成羰基，最终生成**醛或酮**。
:::

:::insight{label=硼氢化氧化与水合的互补}
炔烃硼氢化氧化和直接水合都"经烯醇式互变成醛/酮"，但**区域选择性相反**：
- 直接水合（酸/汞盐）按**马氏**加成，端基炔 $\ce{R-C#CH}$ 得到**甲基酮**；
- 硼氢化氧化是**反马**取向（硼加在含氢多的端碳，羟基最终落在端碳），端基炔得到**醛**而非酮。

录音里点得很清楚："炔也能发生硼氢化氧化……是个顺式加成，而且是反马的加成，所以会生成的是一个醛，而不是酮。"所以想由端基炔制醛用硼氢化氧化，想制甲基酮用直接水合——这是一对互补方法。
:::

## 二、炔烃的亲核加成

炔烃除了亲电加成，在碱催化下还能发生**亲核加成**——由带负电的亲核试剂主动发起进攻。

:::definition{label=炔烃的亲核加成}
在碱催化下，乙炔（或带活泼氢的端基炔）可与含羟基、巯基、氨基、氰基、羧基等基团的化合物发生加成，生成含双键的烯烃产物，中间体是碳负离子。
:::

:::insight{label=亲核加成的机理：是"进攻碳"而不是"拔氢"}
以烷氧基亲核试剂（如乙醇钠提供的 $\ce{RO-}$）与乙炔为例：
1. 烷氧负离子 $\ce{RO-}$ **直接进攻**三键碳（不是去拔炔氢！），用自身的负电荷把一根碳碳 $\pi$ 键异裂；
2. 异裂后在另一碳上生成碳负离子；
3. 该碳负离子再结合一个质子。

净结果相当于把一分子醇 $\ce{ROH}$ 的两部分（$\ce{RO-}$ 与 $\ce{H}$）加到三键上，生成乙烯基醚类产物。
:::

:::pitfall{label=亲核加成 vs 夺氢，看试剂碱性}
同样是负离子和端基炔，结果可能完全不同，关键看试剂**碱性够不够强去拔氢**：
- **碳负离子**（炔钠、RLi、RMgX）碱性极强，足以把炔氢拔下来 → 发生**酸碱反应（生成炔化物）**；
- **氧负离子**（$\ce{RO-}$）碱性没那么强，不去拔氢，而是**直接进攻三键碳** → 发生**亲核加成**。

录音强调："注意它没有拔氢……这个时候是一个氧负离子，氧负离子直接进攻这个碳。"别把这两类混为一谈。
:::

## 三、炔烃的氧化

炔烃能被氧化，且**凡能氧化烯烃的条件一般也能氧化炔烃**，只是产物不同。同样地，强氧化条件断键，弱氧化条件温和。

### 1. 高锰酸钾氧化

:::definition{label=高锰酸钾氧化炔烃}
- **强氧化条件**（酸性、加热的 $\ce{KMnO4}$）：三键**断键**，原三键碳处生成**羧基**（变成羧酸）；若是末端的端基炔碳，则被氧化成**二氧化碳** $\ce{CO2}$。
- **弱氧化条件**（中性/弱碱性、稀的 $\ce{KMnO4}$，如水、pH≈7.5）：不断键，生成**二酮**结构。
:::

强氧化断键示意：
$$
\ce{R-C#C-R' ->[\ce{KMnO4}, H+][\Delta] R-COOH + R'-COOH}
$$

:::insight{label=双键、三键并存时谁先被氧化}
若分子里同时有双键和三键，1 摩尔氧化剂**优先氧化双键**（双键活性高于三键）。例如用三氧化铬 $\ce{CrO3}$ 作氧化剂，限量时先把双键断开生成醛或酮，三键保留。这与加成时"限量试剂先动双键"是同一个道理。
:::

### 2. 臭氧化反应

:::example{label=臭氧化断键定位}
炔烃与臭氧 $\ce{O3}$ 反应后断键，三键处生成相应的**羧酸**。这同样可用于**确定三键的位置**。
$$
\ce{R-C#C-R' ->[1.\ \ce{O3}][2.\ \text{还原}] R-COOH + R'-COOH}
$$
:::

:::insight{label=氧化反应当"侦探"——逆推三键位置}
这是氧化反应最有价值的用法。把断键得到的羧酸碎片在断口处"接回去"，就能复原出原来炔烃的骨架、定出三键的位置。录音里的推断题正是这套路：$\ce{C5H8}$ 化合物 b 用热 $\ce{KMnO4}$ 氧化得到**乙酸和丙酸**，说明三键把分子切成了 $\ce{CH3-}$（→乙酸 $\ce{CH3COOH}$，2 个碳）和 $\ce{CH3CH2-}$（→丙酸 $\ce{CH3CH2COOH}$，3 个碳）两段，接回去就是 $\ce{CH3-C#C-CH2CH3}$（2-戊炔）。碎片的碳数 + 断口位置 = 原三键位置。
:::

### 3. 偶联反应（了解）

:::note{label=炔烃的偶联反应}
炔烃还能发生偶联（工业、加压条件，了解即可）：
- 端基炔在 $\ce{CuCl}$、$\ce{NH4Cl}$ 催化下，一分子炔的碳氢键加到另一分子炔的三键上发生偶联；
- 在镍 $\ce{Ni}$ 催化下，**三分子**乙炔可偶联成苯环，**四分子**乙炔可偶联成环辛四烯。

这些反应实验室不常用，了解其存在即可。
:::

## 四、有机金属试剂：碳负离子的"碳碳键工厂"

现在进入这一节的重头戏。前面反复出现"碳负离子"这个中间体，而**有机金属试剂正是碳负离子的稳定来源**。它们是有机合成里构筑碳碳键最重要的一类工具。

### 1. 什么是有机金属化合物

:::definition{label=有机金属化合物}
**同时含有有机部分和金属部分**的化合物，称为有机金属化合物。其特征是金属直接与碳相连，由于金属电负性比碳小，电子偏向碳，使这个碳带上**部分负电荷乃至成为碳负离子**——这正是它强碱性、强亲核性的来源。
:::

### 2. 炔基钠/锂

:::example{label=炔基钠、炔基锂}
端基炔与金属钠（甲苯/THF，无水）、氨基钠（液氨）或烷基锂反应，炔氢被夺走，生成**炔基钠 $\ce{R-C#C-Na}$** 或**炔基锂 $\ce{R-C#C-Li}$**。
$$
\ce{R-C#CH + NaNH2 ->[\text{液氨}] R-C#C-Na + NH3}
$$
这个炔基负离子是 SP 杂化的碳负离子，既是强碱，也是好的亲核试剂，可进一步去构筑碳碳键。
:::

### 3. 烷基锂

:::definition{label=烷基锂试剂}
**烷基锂 $\ce{R-Li}$** 相当于一个碳负离子 $\ce{R-}$ 和一个锂离子 $\ce{Li+}$ 的组合，碳上带强负电、碱性极强。它能把酸性氢（如炔氢、醇氢）拔下来：例如丁基锂拔走炔氢后，自身变成丁烷 $\ce{C4H10}$，炔基则把锂置换上去成为炔基锂。
:::

### 4. 格氏试剂

:::definition{label=格氏（格利雅）试剂}
**格氏试剂 $\ce{R-MgX}$** 由卤代烷与金属镁 $\ce{Mg}$ 反应制得——金属镁"插入"到碳卤键之间形成。
$$
\ce{R-X + Mg ->[\text{无水乙醚}] R-MgX}
$$
用化合价分析：镁 $+2$ 价、卤素 $-1$ 价，则有机部分 $\ce{R}$ 整体显 $-1$ 价，且这个负电荷集中在与镁相连的碳上，所以**它本质上提供一个碳负离子**。
:::

:::insight{label=三大碳负离子试剂的统一理解}
炔基钠/锂、烷基锂、格氏试剂虽然金属不同，本质都是**提供碳负离子的强碱/强亲核试剂**。它们都能：
- **拔走酸性氢**（炔氢、醇氢、$\ce{N-H}$、$\ce{O-H}$……）——这是上一节端基炔酸性的延伸；
- **作为亲核试剂去进攻缺电子的碳**，从而**接出一根新的碳碳键**。

录音里把这点讲得很透："这两个试剂（烷基锂和格氏试剂）其实是类似的，都是一个提供碳负离子的试剂。"理解了这个统一性，三种试剂就不必各记一套。
:::

### 5. 用炔化物构筑碳碳键

把炔基负离子和卤代烷放在一起，就能把两段碳骨架"焊"成一根更长的链——这是合成上极重要的增碳手段。

:::example{label=炔化物烷基化——碳链增长}
炔基钠（或炔基锂）作为亲核试剂，进攻卤代烷（伯卤代烷为佳）中与卤素相连的碳，把卤素挤走，形成一根新的碳碳键：
$$
\ce{R-C#C-Na + R'-CH2-X -> R-C#C-CH2-R' + NaX}
$$
原来的端基炔（$n$ 个碳）就接上了 $\ce{R'-CH2-}$ 这一段，链变长了。配合上一节的部分加氢（林德拉→顺式、钠/液氨→反式），还能进一步把新生成的三键变成指定构型的双键。这条"端基炔取氢→烷基化增碳→部分加氢控构型"的路线，是用小分子搭建复杂烯烃骨架的经典策略。
:::

:::pitfall{label=有机金属试剂遇活泼氢就"失效"}
格氏试剂、烷基锂、炔化物都是强碱，遇到水、醇、端基炔等任何含活泼氢的物质，会优先去**夺氢**而不是做你想要的亲核加成/取代。所以：
- 制备和使用必须在**无水、无活泼氢**的体系（格氏试剂常用无水乙醚或 THF）；
- 录音里格氏试剂"和所有带活泼氢的试剂反应时，会把氢转移给格氏试剂"，正是这个道理。设计反应时务必先检查体系里有没有活泼氢"偷走"你的试剂。
:::

## 五、例题

:::example{label=用氧化碎片逆推三键位置}
**题：** 某炔烃 $\ce{C5H8}$（a）能与银氨溶液生成白色沉淀；用热的酸性高锰酸钾氧化，a 得到"它和它"（即两份相同的碎片，题面提示）。推断 a 的结构。

**解：**
- 能与银氨生成沉淀 → a 是**端基炔**，含 $\ce{-C#CH}$。
- 强氧化断键，端基炔碳 → $\ce{CO2}$，另一段 → 羧酸。$\ce{C5H8}$ 端基炔骨架为 $\ce{HC#C-CH2-CH2-CH3}$（1-戊炔）等。结合"对称碎片"提示与分子式，定出三键在链端。
- 验证不饱和度：$\ce{C5H8}$ 不饱和度 $=2$，恰为一根三键，符合。

**点拨：** 银氨 → 定"端基炔"；高锰酸钾断键碎片 → 定"三键位置"，两个信息叠加锁定结构。
:::

:::example{label=区域选择性：醛还是酮}
**题：** 端基炔 $\ce{CH3CH2-C#CH}$（1-丁炔）分别经 (1) 稀硫酸/汞盐水合，(2) 硼氢化—氧化，得什么？

**解：**
(1) 马氏水合，羟基落在内侧炔碳，互变后 → $\ce{CH3CH2-CO-CH3}$（丁酮，**酮**）。
(2) 硼氢化—氧化为反马取向，羰基落在端碳 → $\ce{CH3CH2CH2-CHO}$（丁醛，**醛**）。

**点拨：** 同一端基炔，水合给甲基酮、硼氢化氧化给醛，二者**区域互补**。
:::

## 六、易错点小结

:::pitfall{label=常见错误清单}
1. **硼氢化氧化与水合取向记反。** 直接水合=马氏（端基炔→甲基酮）；硼氢化氧化=反马（端基炔→醛）。
2. **亲核加成误当成拔氢。** 氧负离子碱性不足以拔炔氢，是**直接进攻三键碳**做亲核加成；只有碳负离子才拔氢成炔化物。
3. **氧化产物记错。** 强氧化断键得羧酸（端碳→CO₂）；弱氧化（稀、中性/弱碱）得二酮，不断键。
4. **忘了限量试剂先动双键。** 双键、三键并存，1 摩尔氧化剂优先氧化双键。
5. **有机金属试剂体系带活泼氢。** 格氏/烷基锂/炔化物遇水、醇、端基炔会被夺氢失效，必须无水无活泼氢。
6. **逆推三键位置时碎片碳数算错。** 断键碳变羧基（保留该碳），端基炔端碳变 CO₂（也保留一个碳），接回时不要漏碳。
:::

:::note{label=本节小结}
**补充加成：**
- 硼氢化：只加一次。烷基化→顺式烯烃；氧化→经烯醇式互变成**醛/酮**（反马，端基炔得醛，与直接水合互补）。
- 亲核加成：碱催化下 $\ce{RO-}$ 等**直接进攻三键碳**（不拔氢），经碳负离子得乙烯基醚类。

**氧化（剪开分子）：**
- $\ce{KMnO4}$：强氧化（酸/热）断键生羧酸、端碳→CO₂；弱氧化（稀/中性弱碱）得二酮。
- 臭氧化：断键生羧酸。
- 双键、三键并存时限量氧化剂先动双键。
- **核心用途：氧化碎片逆推三键位置**（碎片接回原骨架）。

**有机金属衍生物（接长分子）：**
- 炔基钠/锂、烷基锂、格氏试剂本质都是**提供碳负离子**的强碱/强亲核试剂；格氏试剂 = $\ce{R-X + Mg}$。
- 两大用途：拔走酸性氢、作亲核试剂构筑**碳碳键**。
- 炔化物 + 卤代烷 → 碳链增长，再配合部分加氢控构型，是搭建复杂烯烃骨架的经典策略。
- 全程**无水、无活泼氢**。

一"剪"一"接"——氧化让我们读懂并逆推分子，有机金属试剂让我们搭建分子。下一节进入共轭二烯烃与 Diels-Alder 反应。
:::
