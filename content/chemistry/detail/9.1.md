# 9.1 对称性与手性、手性碳原子

## 从两只手说起：什么是手性

把你的左手和右手并排放在桌面上，它们看起来几乎一模一样：同样数目的手指、同样的关节、同样的纹路。可是当你试图把左手"放进"右手的位置——也就是让它们完全重合时，你会发现无论怎么转、怎么翻，都做不到。它们是彼此的镜像，却不能互相重叠。这种"实物与镜像不能重合"的性质，就叫做**手性**（chirality），手性这个词本身就来自希腊文的"手"。

这个直觉非常重要，请把它牢牢记住：手性的本质不是某个原子，也不是某种特殊的键，而是一种几何关系——**实物和它的镜像无法叠在一起**。大到山川河流、海螺的螺壳，小到一个分子，只要满足这个条件，它就是手性的。本节我们要做的，就是把这种几何直觉翻译成判断分子手性的可操作工具，并由此引出"手性碳原子"这个核心概念。

## 用对称因素判断手性

既然手性是"镜像不能重合"，那么一个物体在什么情况下镜像能够和自己重合呢？答案是：当它具有足够的**对称性**时。一个高度对称的物体（比如一个光滑的球），它的镜像和自己根本就是同一个东西。所以判断手性，我们可以换一个等价但更好操作的问法：**这个分子有没有对称因素？** 如果有，它就没有手性；如果一个对称因素都没有，它就是手性的。

在我们这门课的范围内，需要关注的对称因素有两个：对称面和对称中心。

:::definition{label=对称面与对称中心}
**对称面**：假想有一个平面，把分子"切"成两半，如果这一刀切下去，平面两侧的原子或基团互成镜像、彼此对称，那么这个平面就是分子的对称面。

**对称中心**：分子中（或分子所占据的空间里）存在这样一个点，从这个点出发，沿任意方向走出去，在等距离的位置上总能找到相同的原子或基团，那么这个点就是分子的对称中心。
:::

关于对称中心，有一处特别容易被忽略的细节：**这个点不一定落在分子的某个原子上，它可以是分子所形成的空间里的一个虚拟的点**。比如反式-1,3-二氯环丁烷，它的对称中心就位于环的中央那个虚空处——通过这个点，沿任意方向等距离都能找到相同的原子或基团。所以找对称中心时，不要只盯着原子，要把分子整体所占的空间一起考虑进来。

:::insight{label=判断手性的核心法则}
判断一个分子有没有手性，就去找它有没有对称面、有没有对称中心。**只要具备其中任意一样**，这个分子就没有手性；**只有当这两种对称因素都不存在时**，分子才是手性的。
:::

这里要补充一句说明：对于某些更复杂的结构（例如一些螺旋形的分子），还存在所谓的"轴手性"，它们有一根手性轴或对称轴，左旋还是右旋由这根轴决定。这部分内容已经超出我们这门课的范围。在我们这门课所接触的分子里，只需用"有没有对称面、有没有对称中心"这两个对称因素来判断就足够了。

## 手性是怎么产生的：从乙烷到二卤代物

光有定义还不够，我们要亲眼看一看手性是怎么一步步"长出来"的。最干净的办法，是拿一个完全没有手性的分子，逐步改造它，观察它在哪一步获得了手性。

我们从乙烷 $\ce{CH3-CH3}$ 开始。乙烷是手性分子吗？显然不是——它有三个对称面，对称性很高，镜像和自己完全重合。

**第一步：取代一个氢。** 把乙烷中的一个氢换成溴，得到 $\ce{CH3-CH2Br}$。这时它还有没有对称因素？还有。原来的三个对称面现在虽然减少到只剩一个——也就是由甲基、那个碳以及溴所共同决定的那个平面仍然是对称面——但只要还有这一个对称面，分子就仍然没有手性。

**第二步：再取代另一个氢。** 把另一个氢换成氯，得到一个二卤代的乙烷（一个碳上是甲基方向，另一个碳上连着溴、氯、氢）。现在再来找：还能切出一个对称面吗？切不出来了。还有对称中心吗？也没有了。**对称面和对称中心都消失了，于是手性就在这一刻产生了。** 这个分子的实物和镜像不再能重合，它成了手性分子。

:::insight{label=手性产生的瞬间}
乙烷 → 一溴乙烷 → 二卤代乙烷的这条改造链告诉我们一件事：手性不是凭空出现的，而是在分子**失去最后一个对称因素**的那一刻产生的。当某个碳原子上连接的四个原子（或基团）彼此都不相同时，围绕这个碳的局部环境就再也找不到对称面、也找不到对称中心了，手性随之诞生。
:::

## 手性碳原子（不对称碳原子）

上面的改造过程把规律暴露得很清楚：当一个碳原子上连有**四个互不相同**的原子或基团时，这个碳的实物与镜像就无法重合，局部失去了全部对称因素，手性便产生在它身上。我们给这种碳一个专门的名字。

:::definition{label=手性碳原子}
当一个碳原子上连接**四个互不相同**的原子或基团时，这个碳原子就称为**手性碳原子**，也叫做**不对称碳原子**。由于手性正是从它身上产生的，所以它又被称为分子的**手性中心**或**不对称中心**。书写时，习惯在该碳上打一个星号，记作 $\ce{C}^{*}$。
:::

请特别注意这个判据里"四个互不相同"这一要求是不可松动的：哪怕四个取代基里有两个相同，碳上就会出现一个对称面（穿过这个碳和那两个相同基团对称分布的平面），手性就被破坏了。所以"四个不同"既是手性碳原子的定义，也是它能成为手性中心的根本原因。

为什么一个碳上连四个不同基团就会失去对称性？我们可以反过来想：碳是 $sp^3$ 杂化，四个键指向四面体的四个顶点。如果四个顶点上的基团两两都不一样，那么无论你假想在哪里放一面镜子去切它，镜子两侧都不可能凑出完全对称的图样；同样，也找不到一个中心点让等距离处总有相同基团。对称面和对称中心一齐落空，手性就此确立。

## 例题：乳酸为什么有手性

:::example{label=乳酸的手性}
乳酸是最经典的含一个手性碳原子的分子。观察它的 $\alpha$ 碳，这个碳上连接着四个不同的基团：羟基、羧基、氢、甲基。

按定义，四个基团互不相同 → 这是一个手性碳原子 → 围绕它失去了全部对称因素 → 乳酸是手性分子。它和它的镜像是两个不同的分子，无法重合，化学组成完全一样，区别仅在于碳上四个基团的空间排列顺序不同。这样一对互为镜像、不能重合的分子，称为**对映异构体**。

实验上可以测出，其中一种乳酸使偏振光向右旋（标作正号），另一种向左旋（标作负号）。
:::

这里要点出一个常被混淆的地方：**手性是几何性质（镜像能否重合），旋光性是物理现象（能否、以及向哪个方向偏转偏振光）。** 二者紧密相关——手性分子才会有旋光性——但它们不是一回事。是否手性，靠的是找对称因素这套几何判断；是左旋还是右旋、偏转多少度，则必须靠实验去测，无法单凭结构"算"出方向。下一节我们会看到，正号、负号这些旋光符号和分子的构型标记之间并没有固定对应关系。

:::pitfall{label=三个常见误区}
1. **把"手性碳原子"当成"手性"本身。** 手性碳原子只是产生手性的一种原因，不能直接画等号。本节先建立"四个不同基团→手性碳"的直觉，但后续会看到含两个以上手性碳、或根本不含手性碳的分子，必须回到对称因素去判断。

2. **找对称中心时只盯着原子。** 对称中心可以是分子空间里一个虚拟的点（如反式-1,3-二氯环丁烷的环心），它不必落在任何一个原子上。

3. **把手性和旋光混为一谈。** 手性是"镜像不能重合"的几何关系，可由对称因素判断；旋光方向（左旋/右旋）是实验测出来的，不能从结构直接推断。
:::

:::note{label=本节小结}
- **手性的本质**：实物与其镜像无法重合，如同左右手。判断对象大到物体、小到分子皆适用。
- **判断方法**：找对称因素——对称面与对称中心。只要具备其中任意一个，分子就没有手性；两者都没有，才是手性分子。对称中心可以是空间中的虚拟点。
- **手性的产生**：通过乙烷→一溴乙烷→二卤代乙烷的逐步取代可以看到，当某碳连接四个互不相同的基团时，局部失去全部对称因素，手性随之产生。
- **手性碳原子**：连接四个不同原子/基团的碳，记作 $\ce{C}^{*}$，又称不对称碳原子、手性中心、不对称中心。"四个不同"这一条件不可松动。
- **手性 ≠ 旋光**：手性靠对称因素几何判断；旋光方向靠实验测定。如乳酸（$\alpha$ 碳为手性碳）有一对对映异构体，分别右旋（+）、左旋（−）。
:::
