# 13.1 羧酸的酸性与化学性质

## 一、从一个生活直觉说起

烧水壶里水垢厚了，抓一把柠檬酸煮一煮，水垢很快就化掉。柠檬酸正是从柠檬里提取出来的有机酸，它是一个三元酸，还额外带着一个羟基。这件再日常不过的事，背后藏着本章的主角——**羧酸**：分子中含有羧基 $\ce{-COOH}$ 的有机化合物。

羧基写出来是 $\ce{-COOH}$，由一个**羰基**和一个**羟基**拼在一起。这两个部件合在同一个碳上之后，并不是简单的"羰基 + 羟基"的算术叠加，而是产生了一种新的、彼此影响的电子结构——正是这种结构，决定了羧酸能显酸性，也决定了它后续一长串反应的脾气。

按羧基连接的烃基不同，羧酸可分为：脂肪族羧酸（最常见的乙酸、丙酸、丁酸）、芳香族酸（要求羧基**直接**连在芳环上，如苯甲酸）、脂环族羧酸。按羧基个数又分一元酸、二元酸：草酸（乙二酸）、对苯二甲酸都是二元酸，柠檬酸是三元酸。

:::definition{label=羧酸与羧基}
**羧酸**是分子中含有羧基 $\ce{-COOH}$ 的有机化合物。**羧基**由一个羰基（$\ce{C=O}$）和一个羟基（$\ce{-OH}$）构成，是羧酸的特征官能团。羧基上的质子非常容易离去，使羧酸显酸性。
:::

## 二、为什么羧酸会显酸性：共振与键长等长化

羧酸的酸性，根子在于"质子离去之后形成了什么"。当羧基 $\ce{-COOH}$ 脱去一个质子，剩下的 $\ce{-COO^-}$ 并不是把负电荷孤零零地压在某一个氧上，而是两个氧之间形成了非常好的**共轭**。

:::insight{label=羧酸根的共振稳定}
负电荷离去后形成的羧酸根，两个碳氧键发生**共振**，导致键长的平均化、等长化。真正的羧基负离子，它的两个碳氧键都是 1.27 埃（即 127 皮米），被完全等长化了——一个负电荷均匀地分散在 O、C、O 三个原子之间。
:::

我们在纸面上书写时，为了方便，最终还是会把负电荷固定写在某一个氧上，画成"一个碳羰基 + 一个氧负离子"的形式。但要记住：这只是书写的妥协，真实的羧酸根是一个负电荷均布、两键等长的对称结构。正因为离去之后的产物这么稳定，质子才"愿意"走，羧酸才显酸性。

顺带做一个键长的横向比较：醛酮羰基的碳氧键和羧基里羰基的碳氧键基本一样长；但对于羧基来说，醇里的那种碳氧单键要比羧基里的碳氧键更长一些。这说明羧基内部的电子分布已经被共轭"重新揉过一遍"了。

:::note{label=布朗斯特酸碱的呼应}
按布朗斯特酸碱理论：较强的酸脱出质子，生成更弱的共轭碱；酸性越强，对应的共轭碱越弱。羧酸根之所以稳定（是弱碱），正对应羧酸是一个能稳定电离出质子的酸。
:::

## 三、诱导效应如何调节酸性：吸电子增强、给电子减弱

羧酸的酸性强弱不是固定的，可以通过在 $\alpha$ 位引入取代基来精细调节。乙酸的 $\mathrm{p}K_a$ 是 4.74，是一个中等强度的酸。我们以乙酸为基准，在 $\alpha$ 碳上不断引入氯原子：

| 化合物 | 取代情况 | $\mathrm{p}K_a$ |
|---|---|---|
| 乙酸 | — | 4.74 |
| 氯乙酸 | 一个 $\ce{Cl}$ | 2.86 |
| 二氯乙酸 | 两个 $\ce{Cl}$ | 1.26 |
| 三氯乙酸 | 三个 $\ce{Cl}$ | 0.64 |

取代一个氯，$\mathrm{p}K_a$ 就变小了将近两个单位，相当于酸性强了约 100 倍；取代到三个氯时，三氯乙酸已经是一个强酸了。

:::derivation{label=吸电子基为何增强酸性}
1. 羧酸电离后变成羧酸**负离子**，带一个负电荷。
2. 氯的**吸电子效应**（诱导效应）会把这个负电荷往自己方向"拉拽"，让负电荷被更好地分散、稳住。
3. 负离子越稳定，电离的"代价"越小，质子就越容易离去。
4. 所以**吸电子基使羧酸负离子稳定性增加 → 酸性增强**；氯越多，吸电子越强，酸性越强。
:::

反方向也成立。丁酸可以看成乙酸的 $\alpha$ 位被一个乙基取代，乙基是一个**弱给电子基团**，有微弱的推电子效果，它会让羧酸负离子的稳定性略微降低，于是丁酸的 $\mathrm{p}K_a$ 比乙酸稍微大了一点点，酸性略减弱。

:::insight{label=取代基效应的总规律}
要让**酸性变强**，连**吸电子基**；连**给电子基**则酸性减弱。（碱性化合物正好相反：给电子基使碱性增强，吸电子基使碱性减弱——这一点等学到含氮化合物时会再印证。）
:::

羧酸的酸性还体现在它能与无机碱发生酸碱中和，也能与碱性的盐（如碳酸氢钠）反应放出二氧化碳。

## 四、酯化反应：一个被高中"简化"过的经典反应

### 4.1 高中的版本与遗留的疑问

羧酸和醇在酸催化（或碱催化）下脱水生成羧酸酯，这个成酯反应高中就学过。最经典的例子是乙酸和乙醇生成乙酸乙酯。当时记的口诀是：**脱出的那一分子水，羧酸贡献羟基，醇贡献氢**。

但问题来了：是不是**所有**酯化反应都这样分配？答案是——**不一定**。高中只学了一个最典型的例子。要弄清楚到底谁脱什么，必须看机理。

### 4.2 酸催化的亲核加成-消除机理

酯化反应和醛酮的亲核加成是"同一家人"，本质仍是亲核加成的过程，全程在酸催化下进行：

:::derivation{label=酯化的亲核加成-消除机理}
1. **质子化**：酸提供的质子先质子化羧基的羰基氧，使羰基碳的电正性升高，更容易被进攻。
2. **亲核加成**：醇作为亲核试剂，其氧上的孤对电子进攻羰基碳，加上去，碳氧双键打开。
3. **质子交换**：脱除/转移一个质子，把质子换到羟基上，形成一个容易离去的基团。
4. **消除脱水**：脱去一分子水，氧重新形成双键，得到一个质子化的酯。
5. **去质子**：再脱除质子，最终生成羧酸酯和水。

总账：**羧酸脱掉的是整个羟基，醇脱掉的是羟基上的氢**。因为先发生了亲核加成、又消除了一分子水，故称"亲核加成-消除"机理。
:::

### 4.3 同位素标记实验：怎么证明谁脱了谁

机理是推出来的，怎么证明？用 $^{18}\mathrm{O}$ **同位素标记**。

:::example{label=18O 标记验证酯化机理}
用 $^{18}\mathrm{O}$ 标记的**醇**去做酯化反应，结果发现 $^{18}\mathrm{O}$ 进入到了**酯分子**当中，而**没有**进入水分子。

这说明：醇脱掉的是羟基上的**氢**（氧留在了酯里），而羧酸脱掉的是羧基上的整个**羟基**（变成水中的氧）。实验结果与亲核加成-消除机理完全吻合。

检测手段：用**高分辨质谱**。$^{18}\mathrm{O}$ 标记的乙酸乙酯，测出来比正常分子多 2 个质量单位；若 $^{18}\mathrm{O}$ 跑进水里，则会测到质量为 20 的水。质谱精度可达小数点后 4～5 位，必须与理论值小数点后多位相符才算实锤。（有机结构分析的四大方法：核磁、质谱、红外、紫外。）
:::

同位素实验很有说服力，但很贵——做一个标记的产物，价钱可能翻好多倍。

### 4.4 位阻效应：为什么大分子酯化慢

只要反应里有亲核加成这一步，就一定会受位阻影响：既有羰基附近的位阻，也有亲核试剂自身的位阻。

:::insight{label=位阻降低酯化速率}
当羧酸或醇的 $\alpha$ 碳上烃基增大时，空间位阻明显增加，酯化速率明显下降。对羧酸而言乙酸最快，$\alpha$ 碳越拥挤越慢；对醇而言体积小的醇最快，烃基越大越慢。
:::

### 4.5 叔醇酯化：碳正离子路径（另一种机理）

但"大酸 + 大醇就不能成酯"是不对的。"羧酸脱羟基、醇脱氢"只是酯化的**一种**机制。换一类醇，机制会换。

:::example{label=叔丁醇酯化的 18O 去向}
用 $^{18}\mathrm{O}$ 标记的**叔丁醇**和乙酸在酸催化下酯化，结果发现 $^{18}\mathrm{O}$ 进入了**水**中。

这说明醇脱掉的是**整个羟基**！如果还是走亲核加成-消除，醇上的氧本该留在酯里——现在它跑到水里去了，说明机制变了。
:::

:::derivation{label=叔醇酯化的碳正离子机理}
1. 叔丁醇位阻太大，氧带着这么大一个烃基去进攻羰基，进攻性不足，走不动亲核加成-消除。
2. 改走 $\ce{SN1}/\ce{E1}$ 思路：先把**醇羟基**质子化。
3. "自断臂膀"——水离去，生成一个比较稳定的**叔碳正离子**。$^{18}\mathrm{O}$ 此刻已随水离去。
4. 叔碳正离子作为试剂，与羧酸上羟基氧的孤对电子结合。
5. 再脱一个质子，生成酯。

总账：此时是**醇提供整个羟基、酸只提供羟基上的氢**，仍脱一分子水，但全程不涉及羰基碳，是碳正离子与氧孤对电子的结合，不是亲核进攻羰基。
:::

:::pitfall{label=不要以为酯化只有一条路}
高中学的"羧酸脱羟基、醇脱氢"只是酯化的一条路径。当醇结构变化（如换成叔醇）时，酯照样生成，但路径反过来：醇脱羟基、酸脱氢。判断 $^{18}\mathrm{O}$ 去向时，必须先判断走的是哪条机理。
:::

## 五、羧酸生成各类衍生物

羧酸羟基氧与羰基共轭，电子密度较小、不易接受质子，因此氢卤酸**不能**直接对羧酸做卤代。要用卤原子取代羟基生成酰卤，得换专门的卤代试剂。

:::example{label=羧酸生成酰氯}
用 $\ce{PCl3}$、$\ce{PCl5}$ 或二氯亚砜 $\ce{SOCl2}$ 作卤代试剂，羧基上的羟基被卤原子取代，生成第一类衍生物——酰氯：

$$\ce{RCOOH + PCl5 -> RCOCl + POCl3 + HCl}$$
$$\ce{RCOOH + SOCl2 -> RCOCl + SO2 ^ + HCl ^}$$
:::

**与卤代烷成酯（$\ce{SN2}$ 路径）**：成酯不一定非得醇与羧酸脱水。羧酸**负离子**本身就是亲核试剂，可与卤代烷发生 $\ce{SN2}$ 亲核取代，顶掉卤离子成酯，此时没有醇参与。

:::pitfall{label=羧酸负离子是"差"的亲核试剂}
羧酸负离子虽带负电荷，但负电荷高度分散，亲核能力不强。所以与卤代烷成酯只适用于**伯、仲卤代烷**，叔卤代烷一般很难反应（位阻又拖后腿）。可用 HMPA（六甲基磷酰胺）等促进。此外：羧酸银盐与卤代烷借卤化银沉淀也可成酯；重氮甲烷可专门制羧酸**甲**酯（相当于与甲醇反应的效果）。
:::

**生成酸酐（成环效应）**：两分子羧酸在脱水剂（如 $\ce{P2O5}$）作用下分子间脱水生成酸酐——一分子贡献羟基、一分子贡献羟基氢。

:::note{label=酸酐的命名与成环}
乙酸酐由两分子乙酸脱水而来，含 **4** 个碳（不是 2 个）。两分子相同羧酸成的酸酐叫**单酐**（对称），不同羧酸成的叫**混酐**。含 4、5 个碳的二元酸特别容易**分子内脱水**，形成稳定的五元环或六元环酸酐：邻苯二甲酸 → 邻苯二甲酸酐（五元环）；丁烯二酸 → 丁烯二酸酐（五元环）；戊二酸 → 戊二酸酐（六元环）。成环效应再次印证五、六元环最稳定。
:::

**生成酰胺**：羧酸与氨或胺先形成铵盐，铵盐加热脱水生成酰胺。但此反应可逆、产率低，实际制备酰胺一般不走这条路。

## 六、羧酸的还原

羧酸是有机含氧官能团里的**最高氧化态**，可被还原。

:::theorem{label=LiAlH4 还原羧酸}
在强还原剂氢化铝锂 $\ce{LiAlH4}$ 作用下，羧酸被直接还原成**醇**（一级醇），不会停在醛的程度：
$$\ce{RCOOH ->[LiAlH4] RCH2OH}$$
$\ce{LiAlH4}$、$\ce{NaBH4}$ 属同一类试剂，能选择性还原含氧官能团（醛、羧酸等），同时**保留**分子中的碳碳双键、三键。乙硼烷（硼氢化）同样能把羧酸还原成醇。
:::

## 七、羧酸的脱羧反应

:::definition{label=脱羧反应}
**脱羧**：当羧酸的 $\alpha$ 碳上连有**强吸电子基**或**不饱和基团**时，加热条件下整个羧基 $\ce{-COOH}$ 以二氧化碳形式脱去。脱羧属于**减碳**反应，分子会少一个碳。
:::

:::example{label=两类典型脱羧}
- 乙二酸：羧基旁边就连着另一个羧基（羧基本身是强吸电子基），加热脱羧生成甲酸（$\ce{HOOC-COOH ->[\Delta] HCOOH + CO2 ^}$）。
- $\alpha$ 位连不饱和基团者，加热也可脱羧，例如脱羧生成苯乙烯和二氧化碳。
:::

**二元酸加热的脱羧/脱水规律**（都朝着生成五元环或六元环的方向走）：

| 二元酸（碳数） | 加热行为 | 产物 |
|---|---|---|
| 乙二酸（2） | 脱羧 | 甲酸 |
| 丙二酸（3） | 脱羧 | — |
| 丁二酸（4） | 脱水 | 五元环酸酐 |
| 戊二酸（5） | 脱水 | 六元环酸酐 |
| 己二酸（6）[源材料口述为"7 个碳"，按碳数序列应为 6] | 又脱羧又脱水 | 五元环环戊酮 |
| 庚二酸级（8 个碳口述） | 又脱羧又脱水 | 六元环环己酮 |
| 9 个碳及以上 | 链太长，失去相互作用 | 各反应各的 |

:::insight{label=脱羧脱水的统一逻辑}
前面两个（乙二酸、丙二酸）因成不了环只能脱羧；从能成环开始，一律朝**有利于生成五元环/六元环**的方向反应（脱水成环酐，或既脱羧又脱水成环酮）。九碳及以上链太长，相互作用消失，不再成环。
:::

**汉斯狄克反应与柯奇反应**（两个脱羧卤化/失羧卤化反应）：

:::theorem{label=汉斯狄克反应与柯奇反应}
- **汉斯狄克反应（Hunsdiecker）**：羧酸先与碱性氧化银生成银盐，银盐在溴单质、加热下脱羧，$\ce{-COOH}$ 直接脱掉变成溴代烷，同时生成溴化银。
- **柯奇反应（Kochi）**：羧酸与四乙酸铅，在卤化锂（氯化锂/溴化锂）作用下脱羧，生成氯代烷或溴代烷。

两者类型相同（由羧酸生成**少一个碳**的卤代烷，减碳反应），只是条件不同：汉斯狄克用 $\ce{Br2}$ + 氧化银（银盐），柯奇用四乙酸铅 + 卤化锂。
:::

## 八、本节小结

:::note{label=本节小结}
1. **酸性根源**：羧酸脱质子后生成羧酸根，两个碳氧键共振、等长化（均为 127 pm），负电荷均布在 O—C—O 三原子，负离子稳定 → 显酸性。
2. **诱导效应调酸性**：吸电子基（如 $\ce{Cl}$）稳定负离子、增强酸性（三氯乙酸 $\mathrm{p}K_a$ 0.64 已是强酸）；给电子基（如乙基）减弱酸性（丁酸 $\mathrm{p}K_a$ 略大于乙酸 4.74）。
3. **酯化两条路**：常规走酸催化**亲核加成-消除**（羧酸脱羟基、醇脱氢，$^{18}\mathrm{O}$ 进酯）；叔醇等大位阻醇走**碳正离子**路径（醇脱整个羟基、酸脱氢，$^{18}\mathrm{O}$ 进水）。位阻越大越慢。
4. **成酯补充**：羧酸负离子 + 卤代烷走 $\ce{SN2}$（仅伯、仲卤代烷）；重氮甲烷专做甲酯。
5. **生成衍生物**：$\ce{SOCl2}/\ce{PCl5}$ 等成酰氯；$\ce{P2O5}$ 脱水成酸酐（4、5 碳二元酸易成五/六元环酐）；铵盐加热成酰胺（可逆、低产率）。
6. **还原**：$\ce{LiAlH4}$ 把羧酸还原成一级醇，保留 C=C/C≡C。
7. **脱羧**：$\alpha$ 位有强吸电子基/不饱和基 + 加热即脱羧（减碳）；二元酸按碳数走脱羧/脱水成环；汉斯狄克、柯奇是两个失羧卤化反应（生成少一碳的卤代烷）。
:::
