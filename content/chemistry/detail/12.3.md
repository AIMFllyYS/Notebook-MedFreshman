# 12.3 醛酮的氧化还原与羟醛缩合

## 站在氧化还原中间态的醛酮

醛酮这类含氧官能团，恰好处在氧化还原的**中间态**：往下走可以被还原成醇、甚至还原成亚甲基 $\ce{CH2}$；往上走，醛可以被氧化成羧酸。正因为它"上能上、下能下"，围绕醛酮的氧化还原方法格外丰富。

这一节分三块讲：先把**还原**的各路条件理顺（催化加氢、$\ce{NaBH4}$/$\ce{LiAlH4}$、克莱门森、黄鸣龙等），再讲**氧化**（醛的银镜/铜镜与强氧化、酮的拜耳-维利格氧化），最后落到本章分量最重的**羟醛缩合**——它从亲核加成长出来，是人工合成 $\alpha,\beta$-不饱和醛酮的看家本领。

## 一、醛酮的还原

### 催化加氢：把氢加到碳氧双键上

:::definition{label=催化加氢}
在活泼金属催化下，醛酮与氢反应，把羰基直接还原成羟基。可视为把一分子 $\ce{H2}$ 加到碳氧双键上的**加成反应**：

$$\ce{RCHO ->[H2][催化剂] RCH2OH}\qquad \ce{R2C=O ->[H2][催化剂] R2CH-OH}$$

- 醛还原得**伯醇**；
- 酮还原得**仲醇**。
:::

催化加氢是醛酮与催化剂结合后、从某个面接收氢的过程，所以它**与位阻有关**，而且是**顺式加成**（从同一个面加上去）。对刚性环状底物，氢可从位阻较小的一面（如 β 面）或另一面加成，得到羟基处于直立键或平伏键的不同产物——这一现象本身也印证了它是单面进攻的顺式加成。

### 硼氢化钠与氢化铝锂：实验室的选择性还原

催化加氢在实验室一般不用。实验室更常用**硼氢化钠 $\ce{NaBH4}$** 或 **氢化铝锂 $\ce{LiAlH4}$**——它们提供的是**负氢**，具有还原性。

:::insight{label=负氢试剂的选择性}
$\ce{NaBH4}$ 和 $\ce{LiAlH4}$ 都能**选择性地**把醛酮羰基还原成醇羟基，而不动其他不饱和基团。比如分子里同时有碳碳双键和羰基时，用 $\ce{LiAlH4}$ 还原，碳碳双键可以保留，只把羰基选择性地还原成醇。

负氢还原也能体现位阻效应。以某取代环己酮为例：
- 若负氢从位阻小的一面进攻，羟基会落到另一侧——**这是位阻（动力学）因素**主导，进攻方向取决于哪边更好接近；
- 但若取代基离羰基较远（如 4 位）、已构不成位阻，则进攻方向由**产物稳定性（热力学）因素**决定——大基团处于平伏键更稳定的那种产物占主导（如某例平伏键产物占 88%、直立键仅 12%）。

记住这对孪生判据：**位阻不可忽略时看动力学（位阻）；位阻已消失时看热力学（产物稳定性，大基团居平伏键更稳）。**
:::

:::note{label=其他还原条件}
- **麦尔外因-彭道夫（Meerwein-Ponndorf）还原**：用**异丙醇铝**在异丙醇溶液中还原醛酮，可保留双键、选择性地只把羰基还原成羟基。
- **金属还原成醇 / 偶联成邻二醇**：金属钠、铁、镁及镁汞齐、铝汞齐等也能把醛酮还原成醇；特别地，酮在**非质子溶剂**（如苯）中被金属镁、钠、汞齐等还原、再水解，可得到**双分子偶联的邻二醇**。这是继烯烃弱氧化、醇章之后，又一条生成邻二醇的途径，而邻二醇接下来还可能发生重排。
:::

### 把羰基一路还原成亚甲基 $\ce{CH2}$

有时我们不想停在醇，而要把羰基彻底"打回原形"成 $\ce{CH2}$。有三条路：

:::theorem{label=羰基→亚甲基的三条路线}
**1. 克莱门森（Clemmensen）还原**：酮在**锌汞齐 + 酸性**条件下加热，直接把羰基氧"敲掉"，还原成 $\ce{CH2}$。
$$\ce{R2C=O ->[Zn(Hg),\ HCl][\Delta] R2CH2}$$

**2. 沃尔夫-基希涅尔-黄鸣龙（Wolff-Kishner-黄鸣龙）还原**：最初（沃尔夫、基希涅尔）是在**高温高压**下用**肼和金属钾**把羰基还原成 $\ce{CH2}$；黄鸣龙的改进是把溶剂换成**二缩乙二醇（二甘醇）**，使原本要高温高压的反应能在常温常压下进行，大幅降低工业成本。这是中国第一个人名反应。
$$\ce{R2C=O ->[N2H4][KOH,\ 二甘醇,\ \Delta] R2CH2}$$

**3. 缩硫酮路线（见 12.1）**：醛酮先与硫醇生成缩硫酮，再催化氢化打断 C–S 键，也能把羰基定向还原成 $\ce{CH2}$。
:::

:::insight{label=酸性还是碱性？克莱门森 vs 黄鸣龙}
两者目标相同（羰基→$\ce{CH2}$）但条件互补：**克莱门森在酸性**（锌汞齐/盐酸），**黄鸣龙在碱性**（肼/碱/二甘醇）。所以底物里若含有怕酸的基团，选黄鸣龙；含有怕碱的基团，选克莱门森。这一"酸碱二选一"是合成选路的常见考点。
:::

## 二、醛酮的氧化

总原则：**酮很难再被氧化，而醛基可以继续被氧化成羧基。**

### 醛的氧化：强氧化与"镜面"鉴别

:::note{label=醛的氧化}
- **高锰酸钾酸性溶液**可把醛氧化成羧酸；
- **三氧化铬 $\ce{CrO3}$ 酸性溶液**也可把醛氧化成羧酸；
- **银镜反应、铜镜反应**：醛在银氨溶液或铜氨溶液中被氧化，生成银镜或相应沉淀（本质是氧化还原）。这类有明显现象的反应**在合成上没有价值，但在鉴别上有价值**——能区分醛（反应）与酮（一般不反应）。
:::

### 拜耳-维利格氧化：把酮氧化成酯

酮虽难被一般氧化剂氧化，却有一个特殊反应能把它氧化成**酯**。

:::definition{label=拜耳-维利格（Baeyer-Villiger）反应}
酮在**过氧酸（过酸）**作用下，在羰基旁边**插入一个氧**，结果酮变成了一种酯（因为多了一个氧，这也是一个氧化反应）。

$$\ce{R-C(=O)-R' ->[过氧酸] R-C(=O)-O-R'}$$
:::

:::insight{label=不对称酮的迁移优先级}
对不对称的酮，是哪一边的基团"迁移"到氧上、从而被插氧，是有选择性的。迁移（口语里讲"啃")的优先级为：

$$\text{叔烷基} > \text{仲烷基} > \text{环己基} > \text{苯基} > \text{烷基} > \text{伯烷基}$$

优先级高的那一边先在它旁边插氧。
:::

此外，酮在**硝酸**作用下也可发生**开环氧化**，生成二元羧酸。

:::note{label=氧化还原总览}
醛酮反应可归为两大类：一类是**亲核加成**（见 12.1、12.2），另一类就是**氧化还原**。还原有众多条件（成醇，或直接成 $\ce{CH2}$）；氧化一般针对醛（酮很难氧化），而酮有两个特例：拜耳-维利格氧化（成酯）、硝酸开环（成二元羧酸）。
:::

## 三、羟醛缩合：从亲核加成长出的"造烯"利器

本章开头见过的那些醛酮——香料里的、激素里的——大多带有一个共同片段：**$\alpha,\beta$-不饱和醛酮**，即在醛基或酮羰基的 $\alpha,\beta$ 位有一个碳碳双键。这个片段极其重要。天然产物能提供它，而我们要人工合成它，靠的就是**羟醛缩合**。

### 反应实质：自身生成的亲核试剂

:::definition{label=羟醛缩合}
羟醛缩合本质上仍从**亲核加成**开始，只不过亲核试剂不是外加的，而是醛酮分子在**强碱**作用下，由自身 α-氢被拔走生成的碳负离子。该碳负离子进攻另一分子醛酮的羰基，加成后质子化得到 **β-羟基醛/β-羟基酮**；再在酸或碱催化下脱水，生成 $\alpha,\beta$-不饱和醛酮。

"羟醛缩合"里的"羟"字，正是因为加成那一步生成过一个羟基；虽然最终脱水产物里看不到它，但名字保留了这段历史。
:::

:::derivation{label=羟醛缩合的完整机理（以丁醛为例）}
1. 两分子醛酮，其中一分子先被强碱拔去 α-氢，生成碳负离子（可写其共振式，但真正参与反应的是碳上带负电的那个）；
2. 碳负离子作为亲核试剂，进攻另一分子醛酮的羰基碳——一对电子从碳负离子流向缺电子的羰基碳，**生成一根新的碳碳键**，同时碳氧 π 键打开，氧变成氧负离子；
3. 氧负离子质子化，生成 **β-羟基醛**（β 从羰基碳开始数）；
4. 进一步酸碱催化下，β-羟基与 α-氢之间脱去一分子水，生成 **$\alpha,\beta$-不饱和醛**（若为酮则得不饱和酮）。

$$\ce{2\,CH3CH2CH2CHO ->[OH^-] CH3CH2CH2CH(OH)CH(CH2CH3)CHO ->[-H2O] CH3CH2CH2CH=C(CH2CH3)CHO}$$
:::

### α-氢给体与羰基受体：用"插积木"看清拆分

:::insight{label=给体与受体}
羟醛缩合中，一分子充当亲核试剂（需强碱生成碳负离子），另一分子充当被进攻的底物。我们用"插积木"的方式拆解结果（仅为记忆，并非真实切法）：相当于一个分子贡献两个 $\alpha$ 氢、另一个分子贡献羰基，拼接起来。

- 提供 α-氢的，叫 **α-氢给体**；
- 提供羰基的，叫 **羰基受体**。

找准给体的 α 位、受体的羰基，连起来就是产物骨架。
:::

### 交叉缩合：如何避免"四产物"乱局

同类醛酮缩合（如两分子丁醛）不必区分给体受体。但不同醛酮之间也能缩合，问题就来了。

:::pitfall{label=交叉缩合会生成 4 种产物}
乙醛与丙醛一起做羟醛缩合，会得到**至少 4 种**产物：
1. 乙醛做受体、丙醛做给体；
2. 丙醛做受体、乙醛做给体；
3. 乙醛自身缩合；
4. 丙醛自身缩合。

产物太杂，**在合成上没有价值**。
:::

:::theorem{label=让交叉缩合产物单一的办法}
选一个**没有 α-氢的醛或酮**参与，它只能做**羰基受体**（无法提供碳负离子），从而砍掉一部分交叉，让产物趋于单一。常用的无 α-氢受体：
- **甲醛**——无 α-氢，只能做受体；
- **苯甲醛**——无 α-氢，也只能做受体。

由另一分子提供碳负离子去进攻它们，加成后脱水成双键，产物就干净多了。
:::

:::example{label=逆合成应用：合成肉桂醛}
**题目：** 用羟醛缩合合成肉桂醛（咖啡里常加的香料分子，属 $\alpha,\beta$-不饱和醛）。

**解答（逆合成）：** 看到肉桂醛这个 $\alpha,\beta$-不饱和醛，把 $\alpha,\beta$-双键处"切开"，反推出两块原料：
- **苯甲醛**（$\ce{C6H5CHO}$，无 α-氢）只能作**羰基受体**；
- **乙醛**（$\ce{CH3CHO}$）提供 α-氢，作 **α-氢给体**。

二者在强碱下缩合、脱水：

$$\ce{C6H5CHO + CH3CHO ->[OH^-] C6H5CH=CH-CHO + H2O}$$

由于苯甲醛无 α-氢、只能做受体，乙醛只能做给体，交叉方向唯一，产物单一，正是肉桂醛。

**点拨：** 设计交叉缩合时，先在两种醛酮里挑出"无 α-氢"的那个钉死为受体，方向就定了。
:::

### 康尼扎罗反应：无 α-氢醛的"自身氧化还原"

既然无 α-氢的醛不能做 α-氢给体，那它在强碱里会怎样？

:::definition{label=康尼扎罗（Cannizzaro）反应}
**没有 α-氢的醛**（如甲醛、苯甲醛）在**浓强碱**作用下，不能进行羟醛缩合，转而发生**歧化**（自身氧化还原）：两分子醛中，一分子被氧化成**羧酸（盐）**，另一分子被还原成**醇**。

$$\ce{2\,C6H5CHO ->[浓\ NaOH] C6H5COO^-Na+ + C6H5CH2OH}$$

这正与羟醛缩合形成互补：**有 α-氢的醛在强碱下走羟醛缩合；无 α-氢的醛在浓强碱下走康尼扎罗歧化。**
:::

### 分子内羟醛缩合：成环

:::insight{label=分子内缩合成环}
当同一分子中既有 α-氢、又有可作受体的羰基，且两者间隔合适时，碳负离子可进攻**自身另一端**的羰基，发生**分子内羟醛缩合**，脱水后生成**环状的 $\alpha,\beta$-不饱和酮**。这是构筑五元、六元环的重要手段——把"扩链造烯"的本领收拢到一个分子里，就成了"闭环造环"。
:::

## 易错点

:::pitfall{label=六个常见坑}
1. **醛酮还原级别记反。** 醛→伯醇、酮→仲醇；要彻底成 $\ce{CH2}$ 才用克莱门森/黄鸣龙/缩硫酮路线。

2. **克莱门森与黄鸣龙的酸碱条件混淆。** 克莱门森酸性（锌汞齐/酸），黄鸣龙碱性（肼/碱/二甘醇）；按底物耐酸还是耐碱选路。

3. **以为酮也能被一般氧化剂氧化成酸。** 酮很难被氧化；酮的特殊氧化只有拜耳-维利格（成酯）和硝酸开环（成二元羧酸）。

4. **拜耳-维利格产物写成酸。** 它把酮氧化成**酯**（羰基旁插氧），不是酸；不对称酮按"叔>仲>环己基>苯基>烷基>伯"的迁移优先级决定插氧位置。

5. **羟醛缩合误以为产物含羟基。** "羟"指加成步生成过 β-羟基，最终脱水产物是 $\alpha,\beta$-不饱和醛酮，看不到羟基。

6. **交叉缩合不控产物 / 混淆康尼扎罗。** 不加控制会得 4 种产物；要单一就选无 α-氢的醛酮做受体。而无 α-氢的醛在浓强碱下走的是康尼扎罗歧化，不是缩合。
:::

:::note{label=本节小结}
- **还原**：催化加氢（醛→伯醇、酮→仲醇，顺式加成、看位阻）；$\ce{NaBH4}$/$\ce{LiAlH4}$ 选择性还原羰基、保留 C=C；麦尔外因-彭道夫（异丙醇铝）保留双键；金属还原可成醇或偶联成邻二醇；克莱门森（锌汞齐/酸）、黄鸣龙（肼/碱/二甘醇）、缩硫酮路线把羰基彻底还原成 $\ce{CH2}$。
- **氧化**：酮难氧化；醛可被高锰酸钾、$\ce{CrO3}$ 氧化成羧酸，银镜/铜镜用于鉴别；酮的特例为拜耳-维利格（成酯，迁移有优先级）与硝酸开环（成二元羧酸）。
- **羟醛缩合**：强碱下自身 α-氢成碳负离子，进攻另一分子羰基构筑新碳碳键，经 β-羟基醛酮脱水得 $\alpha,\beta$-不饱和醛酮。区分 α-氢给体与羰基受体；交叉缩合用"无 α-氢受体"（甲醛、苯甲醛）控产物；无 α-氢醛在浓强碱下走康尼扎罗歧化；间隔合适时可分子内缩合成环。
:::
