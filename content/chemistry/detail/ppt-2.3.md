# ppt-2.3 环烷烃与烯炔烃的命名

:::insight{label=从"链"到"环"再到"双键"}
上一节我们学会了给烷烃命名，但有机分子远不止开链烷烃。当碳链首尾相连成环，就得到**环烷烃**；当碳碳之间形成双键或叁键，就得到**烯烃**或**炔烃**。这些分子的命名有两个新挑战：

1. **环的处理**：环上连什么基团时以环为母体？什么时候又把环当取代基？多环化合物（桥环、螺环）怎么命名？
2. **立体化学标记**：双键不能自由旋转，所以会产生顺反异构。如何用"顺/反"或"Z/E"准确描述这种空间关系？

本节我们就来解决这两个问题。学完之后，你就能给从环丙烷到桥环壬烷、从乙烯到带 5 个取代基的复杂烯烃——任何分子起一个唯一名字。
:::

## 单环化合物的命名

:::theorem{label=单环化合物命名规则}
单环化合物命名时，"环是母体还是取代基"取决于环上连接的基团类型：

**以环为母体**：环上连有**简单烷基、硝基、亚硝基、卤素**时，以环为母体命名。

- 甲基环戊烷
- 1-甲基-2-硝基环己烷
- 2-氯甲苯（卤素连在苯环上）

**环作为取代基**：环上连有**复杂烷基（烯烃、炔烃基团）、氨基、羟基、醛基、羧基**等时，环作为取代基命名。

- 环丙基甲酸（环丙基作取代基，甲酸为母体）
- 2-甲基环己醇（环己醇为母体，甲基为取代基）
:::

:::insight{label=母体选择的本质——优先级}
"以谁为母体"的本质是**官能团优先级**：高优先级官能团作母体，低优先级作取代基。一般优先级顺序（从高到低）：羧酸 > 酯 > 酰卤 > 酰胺 > 腈 > 醛 > 酮 > 醇 > 胺 > 醚 > 烯 > 炔 > 烷。

所以：
- 环 + 简单烷基/卤素/硝基 → 烷基/卤素/硝基优先级低 → **环为母体**
- 环 + 羧基/羟基/醛基 → 这些官能团优先级高 → **环为取代基**

记住这个优先级，就不用死记"什么情况以环为母体"了。
:::

:::pitfall{label=混淆"环己醇"和"环己基醇"}
- **环己醇**：$\ce{-OH}$ 直接连在环上，环为母体
- **环己基甲醇**：$\ce{-CH2OH}$ 通过 $\ce{CH2}$ 连到环上，环作取代基，甲醇为母体

两者结构完全不同，命名也完全不同。判断时要看 $\ce{-OH}$ 是直接连环还是经过 $\ce{CH2}$。
:::

## 多环烷烃：桥环与螺环

:::definition{label=桥环与螺环}
**多环烷烃**是含有两个或多个环的烷烃，按环与环的连接方式分两类：

- **桥环**（bridged ring）：两个环共用**两个或多个**碳原子
- **螺环**（spiro ring）：两个环共用**一个**碳原子（螺原子）
:::

:::theorem{label=多环烷烃命名规则}
**桥环命名**：用"二环[X.Y.Z]某烷"格式
- X、Y：两条主桥两侧的碳原子数，**先多后少**
- Z：主桥（第三条桥）的碳原子数
- 总碳数 = X + Y + Z + 2（2 个桥头碳）

例如：
- **二环[1.1.0]丁烷**：X=1, Y=1, Z=0，总碳数 = 1+1+0+2 = 4（丁烷）
- **二环[3.2.1]辛烷**：X=3, Y=2, Z=1，总碳数 = 3+2+1+2 = 8（辛烷）

**螺环命名**：用"螺[X.Y]某烷"格式
- X、Y：两个环各自的碳原子数（不含螺原子），**从小到大**
- 总碳数 = X + Y + 1（1 个螺原子）

例如：
- **螺[4.5]癸烷**：X=4, Y=5，总碳数 = 4+5+1 = 10（癸烷）
- **螺[5.5]十一烷**：X=5, Y=5，总碳数 = 5+5+1 = 11（十一烷）
:::

:::pitfall{label=混淆桥环和螺环的方括号规则}
- **桥环**：方括号内**3 个数字** [X.Y.Z]，**先多后少**（X ≥ Y ≥ Z）
- **螺环**：方括号内**2 个数字** [X.Y]，**从小到大**（X ≤ Y）

记忆：桥环有 2 个桥头碳共享，所以多 1 个数字；螺环只有 1 个螺原子，少 1 个数字。数字顺序：桥环"先大后小"（像桥的"主桥-次桥"），螺环"先小后大"（像螺的"小环-大环"）。
:::

:::example{label=识别多环烷烃}
**题目**：判断下列化合物是桥环还是螺环，并命名。

(a) 两个环共用 1 个碳原子，两环分别有 4 个和 5 个碳（不含共用碳）
(b) 两个环共用 2 个碳原子，三条桥分别有 3、2、1 个碳

**解**：
- (a) 共用 1 个碳 → **螺环**。两环碳数 4 和 5，从小到大 → **螺[4.5]癸烷**（总碳数 4+5+1=10）
- (b) 共用 2 个碳 → **桥环**。三条桥 3、2、1，先多后少 → **二环[3.2.1]辛烷**（总碳数 3+2+1+2=8）
:::

## 环烷烃的顺反异构

:::definition{label=环烷烃的顺反异构}
环烷烃环上有两个取代基时，可用"**顺/反**"表明取代基与环平面的位置关系：

- **顺（cis-）**：两个取代基在环的**同侧**
- **反（trans-）**：两个取代基在环的**异侧**

例如：
- 顺-1,4-二氯环己烷：两个 Cl 在环同侧
- 反-1,4-二氯环己烷：两个 Cl 在环异侧
:::

:::insight{label=为什么环烷烃有顺反而烷烃没有}
环烷烃的环限制了碳碳单键的自由旋转——一旦成环，环上的取代基就"锁"在了环的某一侧，无法翻到另一侧（除非破坏环）。所以环烷烃会产生顺反异构。而开链烷烃的 $\ce{C-C}$ 单键可以自由旋转，取代基可以随时翻到另一侧，所以没有顺反异构。

这也解释了为什么环烷烃的化学性质比开链烷烃更复杂——环限制了构象自由度。
:::

## 烯烃与炔烃的命名

:::theorem{label=烯烃/炔烃系统命名法}
1. **选主链**：选择含双键/叁键的最长碳链为主链，按主链碳原子数命名为"某烯"/"某炔"；
2. **编号**：从**靠近双键/叁键一端**开始给主链编号，使双键/叁键所在碳原子位号最小；
3. **写名称**：把双键/叁键的较小数字及取代基的位次、数目和名称写在"某烯/某炔"之前。

例如：
- $\ce{CH2=CH-CH2-CH(CH3)-CH2-CH3}$：双键在 1 位，甲基在 4 位 → **4-甲基-1-己烯**
- $\ce{CH3-CH=CH-CH(CH3)-CH3}$：双键在 2 位，甲基在 4 位 → **4-甲基-2-戊烯**
:::

:::example{label=烯烃命名}
**题目**：命名 $\ce{CH2=CH-CH(CH3)-CH3}$（含 4 个碳，双键在 1 位，3 位有甲基）。

**解**：
- 主链 4 个碳，含双键 → 丁烯
- 从靠近双键一端编号：双键在 1 位，甲基在 3 位
- 名称：**3-甲基-1-丁烯**
:::

## 烯烃的顺反异构与 Z/E 标记

:::definition{label=顺反异构与 Z/E 标记}
**顺反标记**（适用于双键碳上有相同基团时）：
- **顺（cis-）**：两个相同基团在双键同侧
- **反（trans-）**：两个相同基团在双键异侧

例如：
- 顺-2-丁烯：两个 $\ce{CH3}$ 在双键同侧
- 反-2-丁烯：两个 $\ce{CH3}$ 在双键异侧

**Z/E 标记**（适用于双键碳上无相同基团时）：
- 按**顺序规则**比较每个双键碳上的两个基团，找出"大基团"
- **Z（zusammen，同侧）**：两个大基团在双键同侧
- **E（entgegen，异侧）**：两个大基团在双键异侧
:::

:::insight{label=顺反和 Z/E 的关系}
"顺反"和"Z/E"是两套独立的标记系统：

- **顺反**只适用于双键碳上有**相同基团**的情况（如 2-丁烯两端都有 $\ce{CH3}$ 和 H）
- **Z/E**适用于**任何**双键，无论是否有相同基团

它们之间没有必然对应关系：顺可能是 Z 也可能是 E，取决于基团大小关系。例如顺-2-丁烯中两个大基团 $\ce{CH3}$ 在同侧，所以也是 (Z)-2-丁烯；但如果把 $\ce{CH3}$ 换成更小的基团，"顺"对应的可能是 E。

**通用做法**：能用 Z/E 就用 Z/E，因为它适用于所有情况，不会产生歧义。
:::

:::example{label=Z/E 命名}
**题目**：命名 $\ce{CHBr=CHCl}$（1-溴-1-氯乙烯）的两个异构体。

**分析**：
- 左碳：连 Br 和 H，Br > H（Br 原子序数 35 > H 1）
- 右碳：连 Cl 和 H，Cl > H（Cl 17 > H 1）

**解**：
- **(Z)-1-溴-1-氯乙烯**：两个大基团（Br 和 Cl）在双键同侧
- **(E)-1-溴-1-氯乙烯**：两个大基团（Br 和 Cl）在双键异侧

注意：这个分子两端没有相同基团，**不能用顺反命名**，只能用 Z/E。
:::

:::example{label=1-溴丙烯的 Z/E}
**题目**：命名 $\ce{CH3-CH=CHBr}$ 的两个异构体。

**分析**：
- 左碳（C1）：连 $\ce{CH3}$ 和 H，$\ce{CH3}$ > H
- 右碳（C2）：连 Br 和 H，Br > H

**解**：
- **(E)-1-溴丙烯**：大基团 $\ce{CH3}$ 和 Br 在双键异侧
- **(Z)-1-溴丙烯**：大基团 $\ce{CH3}$ 和 Br 在双键同侧

判断步骤：① 找每个双键碳上的两个基团；② 用顺序规则确定每个碳上的"大基团"；③ 看两个大基团在双键同侧（Z）还是异侧（E）。
:::

:::pitfall{label=把"顺"等同于"Z"}
"顺"和"Z"不是一回事！

- 顺-2-丁烯 = (Z)-2-丁烯（巧合一致）
- 但如果分子是 $\ce{CH3-CH=CH-CH2Br}$，左碳大基团是 $\ce{CH3}$，右碳大基团是 $\ce{CH2Br}$（Br > H）。若 $\ce{CH3}$ 和 $\ce{CH2Br}$ 在同侧，是 Z；但此时"顺"无法定义（两端无相同基团）。

**结论**：顺反只适用于两端有相同基团的特殊情况，Z/E 是通用规则。考试时优先用 Z/E。
:::

:::note{label=本节小结}
- **单环命名**：环 + 简单烷基/卤素/硝基 → 环为母体；环 + 复杂基团/含氧含氮官能团 → 环为取代基；
- **多环烷烃**：
  - 桥环：二环[X.Y.Z]某烷，X≥Y≥Z，总碳数 = X+Y+Z+2
  - 螺环：螺[X.Y]某烷，X≤Y，总碳数 = X+Y+1
- **环烷烃顺反**：环限制了单键旋转，产生顺反异构（同侧=顺，异侧=反）；
- **烯烃/炔烃命名**：选含双键/叁键的最长链为主链，编号使双键/叁键位号最小；
- **Z/E 标记**：用顺序规则确定每个双键碳上的"大基团"，同侧 Z，异侧 E；Z/E 是通用规则，顺反只适用于两端有相同基团的特殊情况；
- 下一节我们将学习含官能团化合物（卤代烃、醇、醛酮、羧酸、胺）的命名，以及多官能团化合物的命名策略。
:::
