:::insight{label=从"为什么叔卤代烷和伯卤代烷反应机理不同"说起}
把 1-溴丁烷（伯卤代烷）和 2-溴-2-甲基丙烷（叔卤代烷）分别与 $\ce{NaOH}$ 水溶液反应，会发现：

- **伯卤代烷**：反应速率随 $\ce{[RX]}$ 和 $\ce{[OH^-]}$ **两者**浓度变化 → 二级反应，机理为 **SN2**
- **叔卤代烷**：反应速率只随 $\ce{[RX]}$ 浓度变化，与 $\ce{[OH^-]}$ 浓度**无关** → 一级反应，机理为 **SN1**

为什么会有这种差异？因为两种机理走的"路径"完全不同：
- SN2 是**一步协同**：亲核试剂从背面进攻，离去基团同时离去，构型翻转
- SN1 是**两步分步**：先断裂 C—X 键形成碳正离子（慢），再亲核试剂进攻（快），外消旋化

本节我们就来深入理解 SN1 和 SN2 两种机理，以及它们在立体化学上的差异。
:::

## 卤代烃概述

:::definition{label=卤代烃}
**卤代烃**（halohydrocarbon）：烃分子中的氢原子被卤素原子取代生成的化合物，通式 $\ce{RX}$（$\ce{X = F, Cl, Br, I}$）。

**分类**：
- 按 $\ce{R}$ 结构：饱和卤代烃、不饱和卤代烃、芳香卤代烃
- 按 $\ce{X}$ 数目：一卤代烃、二卤代烃、三卤代烃
- 按 $\alpha$-碳级数：
  - **伯卤代烷**（一级，1°）：$\ce{RCH2X}$
  - **仲卤代烷**（二级，2°）：$\ce{R2CHX}$
  - **叔卤代烷**（三级，3°）：$\ce{R3CX}$

**应用**：
- 溶剂：$\ce{CH2Cl2}$、$\ce{CHCl3}$、$\ce{CCl4}$
- 制冷剂：氟利昂 $\ce{CHF2Cl}$、$\ce{CH3Cl}$
- 麻醉剂：$\ce{CHCl3}$
- 干洗剂：$\ce{CCl4}$
:::

:::theorem{label=碳卤键的性质}
**键能**（$\ce{CH3-X}$，kJ/mol）：

| 卤素 | F | Cl | Br | I |
|------|---|----|----|---|
| 键能 | 451.9 | 351.8 | 292.9 | 221.8 |
| 键长 (Å) | 1.385 | 1.784 | 1.929 | 2.139 |

**稳定性**：$\ce{C-F > C-Cl > C-Br > C-I}$（键能越大越稳定）

**反应活性**：$\ce{R-F < R-Cl < R-Br < R-I}$（键能越小越易断裂，反应活性越高）

**碳卤键的极性**：$\ce{C^{\delta+}-X^{\delta-}}$，碳带部分正电荷，是亲核试剂进攻的"靶子"。
:::

## 亲核取代反应的定义

:::definition{label=亲核取代反应}
**亲核取代反应**（nucleophilic substitution，符号 SN）：有机化合物分子中的原子（或基团）被**亲核试剂**取代的反应。

$$
\ce{Nu^- + R-X -> Nu-R + X^-}
$$

- **底物**：$\ce{R-X}$（待反应的分子）
- **亲核试剂**：$\ce{Nu^-}$（富电子，进攻带正电的碳）
- **离去基团**：$\ce{X^-}$（带电子离去）
- **产物**：$\ce{Nu-R}$

**关键**：亲核试剂进攻 $\alpha$-碳（带 $\delta+$ 的碳），离去基团带着电子对离去。
:::

:::insight{label=亲核取代的"推-拉"模型**
亲核取代反应就像"换座位"：
- 亲核试剂 $\ce{Nu^-}$（富电子）"推"向 $\alpha$-碳（缺电子）
- 离去基团 $\ce{X^-}$（带电子）"拉"走电子，离开碳

整个过程是"新键形成"和"旧键断裂"的协同或分步过程。根据协同还是分步，分为 SN2 和 SN1 两种机理。
:::

## SN2 反应机理——双分子亲核取代

:::definition{label=SN2反应机理}
**SN2 反应**（bimolecular nucleophilic substitution）：有两种分子参与了决速步的亲核取代反应，是一步完成的**协同反应**。

**机理**（一步完成）：

$$
\ce{Nu^- + R-X ->[慢] [Nu\bond{...}R\bond{...}X]^- -> Nu-R + X^-}
$$

- 亲核试剂 $\ce{Nu^-}$ 从**离去基团的背面**进攻 $\alpha$-碳
- 新键 $\ce{Nu-C}$ 形成的同时，旧键 $\ce{C-X}$ 断裂
- 经过一个**五配位碳的过渡态**（$\ce{Nu}$ 和 $\ce{X}$ 同时部分连接到 C）
- 一步完成，无中间体

**速率方程**：

$$
v = k[\ce{RX}][\ce{Nu^-}]
$$

二级反应，速率与底物和亲核试剂浓度都有关。
:::

::figure{src="/images/chemistry/svg/ch09/sn2-mechanism-backside-attack.svg" caption="SN2机理——背面进攻，五配位过渡态，Walden构型翻转" alt="Nu-从X背面180°进攻α碳，五配位三角双锥过渡态，三个基团翻面，构型R→S翻转"}

:::theorem{label=SN2反应的特点}
1. **双分子反应**，二级反应（$v = k[\ce{RX}][\ce{Nu^-}]$）
2. **一步完成**，协同进行（新键生成与旧键断裂同时发生）
3. **进攻位点在离去基团的背面** → **构型翻转**（Walden 翻转）
4. **无中间体**，只有一个过渡态
:::

:::insight{label=为什么SN2是背面进攻}
想象 $\alpha$-碳是一个"球"，离去基团 $\ce{X}$ 在球的"前方"（带电子准备离开）。亲核试剂 $\ce{Nu^-}$ 想进攻这个球，但如果从"前方"（$\ce{X}$ 那侧）进攻，会被 $\ce{X}$ 的电子云"挡住"——同号电荷相斥。

所以 $\ce{Nu^-}$ 只能从"后方"（$\ce{X}$ 的对面）进攻。这种背面进攻导致其他三个基团"翻面"——就像雨伞被风吹翻——这就是 **Walden 翻转**。
:::

:::theorem{label=Walden翻转}
**Walden 翻转**（构型翻转）：SN2 反应中，亲核试剂从背面进攻，使 $\alpha$-碳的构型发生翻转——R 变 S，S 变 R。

**历史**：Paul Walden（1863-1957）首次发现这一现象，故命名为 Walden 翻转。

**例**：(S)-2-溴辛烷经 SN2 反应转化为 (R)-2-辛硫醇（构型翻转）。
:::

:::example{label=SN2反应的立体化学}
**题目**：(S)-2-溴丁烷与 $\ce{NaSH}$ 反应，产物的构型是什么？

**分析**：$\ce{SH^-}$ 是亲核试剂，2-溴丁烷是仲卤代烷，反应为 SN2 机理。SN2 是背面进攻，构型翻转。

**解**：

(S)-2-溴丁烷 $\xrightarrow{\ce{NaSH}}$ **(R)-2-丁硫醇**

构型从 S 翻转为 R，这是 SN2 反应的标志性特征。

**注意**：如果反应是 SN1 机理，则会发生外消旋化（R 和 S 各 50%），而不是构型翻转。
:::

## SN1 反应机理——单分子亲核取代

:::definition{label=SN1反应机理}
**SN1 反应**（unimolecular nucleophilic substitution）：只有一种分子参与了决速步的亲核取代反应，是**分步完成**的。

**机理**（两步）：

**第 1 步**（慢，决速步）：C—X 键异裂，形成碳正离子

$$
\ce{R-X ->[慢] R+ + X^-}
$$

**第 2 步**（快）：亲核试剂进攻碳正离子

$$
\ce{R+ + Nu^- ->[快] R-Nu}
$$

**速率方程**：

$$
v = k[\ce{RX}]
$$

一级反应，速率只与底物浓度有关，与亲核试剂浓度无关。
:::

::figure{src="/images/chemistry/svg/ch09/sn1-mechanism-racemization.svg" caption="SN1机理——两步分步，sp²碳正离子中间体，Nu-从两侧进攻外消旋化" alt="叔卤代烷C-X异裂形成sp2碳正离子，Nu-从上下两侧等概率进攻，产物50:50外消旋"}

:::theorem{label=SN1反应的特点}
1. **单分子反应**，一级反应（$v = k[\ce{RX}]$）
2. **分步完成**，形成**碳正离子中间体**（决速步）
3. **亲核试剂从碳正离子两侧进攻**（机会均等）→ **外消旋化**
4. **可能有碳正离子重排**（仲→叔等）
:::

:::insight{label=为什么SN1会外消旋化**
碳正离子 $\ce{R+}$ 是 $\mathrm{sp^2}$ 杂化，平面三角形结构——三个基团在同一平面，p 轨道垂直于平面（空的）。

亲核试剂 $\ce{Nu^-}$ 可以从这个平面的**上方**或**下方**进攻，机会均等：
- 从上方进攻 → 得到一种构型
- 从下方进攻 → 得到另一种构型（对映异构体）

所以产物是两种构型的混合物——**外消旋化**（如果底物是手性的）。

但实际上往往不是严格的 50:50，因为离去基团 $\ce{X^-}$ 还在一侧"挡着"，从另一侧进攻略多——这叫"离子对效应"。
:::

:::example{label=SN1反应的立体化学}
**题目**：(R)-3-溴-3-甲基己烷在水中反应（SN1），产物的立体化学是什么？

**分析**：3-溴-3-甲基己烷是叔卤代烷，水是弱亲核试剂，反应为 SN1 机理。

**解**：

1. **第 1 步**：$\ce{C-Br}$ 键异裂，形成平面型碳正离子 $\ce{R+}$（$\mathrm{sp^2}$）
2. **第 2 步**：$\ce{H2O}$ 从碳正离子平面的上方或下方进攻，机会均等
3. **产物**：(R)-3-羟基-3-甲基己烷 和 (S)-3-羟基-3-甲基己烷 的混合物（外消旋体）

**注意**：实际产物可能略偏向某一构型（离子对效应），但总体是外消旋化。
:::

## SN1 与 SN2 的对比

:::theorem{label=SN1与SN2的全面对比}

| 特征 | SN2 | SN1 |
|------|-----|-----|
| **分子数** | 双分子 | 单分子 |
| **速率方程** | $v = k[\ce{RX}][\ce{Nu^-}]$ | $v = k[\ce{RX}]$ |
| **反应级数** | 二级反应 | 一级反应 |
| **反应步骤** | 一步完成 | 多步完成 |
| **中间体** | 无（只有过渡态） | 碳正离子 |
| **进攻位点** | 离去基团背面 C | $\ce{C+}$ 两侧 |
| **立体化学** | 构型转化（Walden 翻转） | 外消旋化 |
| **重排** | 不重排 | 可能重排 |
| **底物偏好** | 伯卤代烷 > 仲 > 叔 | 叔卤代烷 > 仲 > 伯 |
| **亲核试剂** | 强亲核试剂有利 | 亲核性影响小 |
| **溶剂** | 极性溶剂影响不大 | 极性溶剂有利 |
:::

:::insight{label=如何判断SN1还是SN2}
判断机理的"四看"：

1. **看底物**（$\alpha$-碳级数）：
   - 伯卤代烷 → SN2
   - 叔卤代烷 → SN1
   - 仲卤代烷 → 看其他条件

2. **看亲核试剂**：
   - 强亲核试剂（$\ce{OH^-}$、$\ce{RO^-}$、$\ce{CN^-}$）→ SN2
   - 弱亲核试剂（$\ce{H2O}$、$\ce{ROH}$）→ SN1

3. **看溶剂**：
   - 极性溶剂（水、醇）→ SN1
   - 偶极溶剂（丙酮、DMF）→ SN2

4. **看立体化学**：
   - 构型翻转 → SN2
   - 外消旋化 → SN1
:::

:::example{label=判断反应机理}
**题目**：判断下列反应是 SN1 还是 SN2。

(1) $\ce{CH3Br + OH^- -> CH3OH + Br^-}$（水溶液）
(2) $\ce{(CH3)3CBr + H2O -> (CH3)3COH + HBr}$
(3) $\ce{CH3CH2Br + CN^- -> CH3CH2CN + Br^-}$（DMSO）

**解**：

**(1)** $\ce{CH3Br}$（伯卤代烷，实际是甲基）+ $\ce{OH^-}$（强亲核试剂）→ **SN2**
- 甲基无位阻，强亲核试剂，符合 SN2 条件

**(2)** $\ce{(CH3)3CBr}$（叔卤代烷）+ $\ce{H2O}$（弱亲核试剂）→ **SN1**
- 叔卤代烷易形成稳定碳正离子，弱亲核试剂，符合 SN1 条件

**(3)** $\ce{CH3CH2Br}$（伯卤代烷）+ $\ce{CN^-}$（强亲核试剂）+ DMSO（偶极溶剂）→ **SN2**
- 伯卤代烷 + 强亲核试剂 + 偶极溶剂，全部指向 SN2
:::

:::pitfall{label=桥头卤代烃不发生SN1也不发生SN2}
**常见错误**：以为所有卤代烃都能发生 SN1 或 SN2。

**正确做法**：桥头卤代烃（如 1-溴双环[2.2.1]庚烷）既不发生 SN1 也不发生 SN2：
- **不能 SN1**：桥环体系的刚性，无法形成平面型碳正离子
- **不能 SN2**：C—X 键的背面是环系，空间位阻大，亲核试剂无法背面进攻

**记忆**：桥头碳上的卤素"卡死"了，亲核取代反应难以进行。
:::

::figure{src="/images/chemistry/svg/ch09/bridgehead-haloalkane.svg" caption="桥头卤代烃——刚性笼状结构阻止SN1和SN2（Bredt规则）" alt="1-溴降冰片烷桥头碳上Br，刚性笼阻止背面进攻(SN2)，桥头碳无法平面化(SN1)"}

:::note{label=本节小结}
- **卤代烃**：$\ce{R-X}$，按 $\alpha$-碳级数分为伯/仲/叔卤代烷
- **碳卤键活性**：$\ce{R-F < R-Cl < R-Br < R-I}$（键能越小越活泼）
- **亲核取代反应** $\ce{SN}$：亲核试剂进攻 $\alpha$-碳，离去基团带电子离去
- **SN2 机理**：
  - 双分子，二级反应 $v = k[\ce{RX}][\ce{Nu^-}]$
  - 一步协同，背面进攻
  - **构型翻转**（Walden 翻转）
  - 伯卤代烷优先
- **SN1 机理**：
  - 单分子，一级反应 $v = k[\ce{RX}]$
  - 分步进行，形成碳正离子中间体（决速步）
  - **外消旋化**（两侧进攻机会均等）
  - 叔卤代烷优先
- **判断机理**：看底物（伯/叔）、亲核试剂（强/弱）、溶剂（极性/偶极）、立体化学（翻转/外消旋）
- **桥头卤代烃**：既不 SN1 也不 SN2（刚性结构 + 位阻）
- 下一节我们将学习影响亲核取代反应的因素
:::
