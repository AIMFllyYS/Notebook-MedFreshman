# 11.3 醚·环氧乙烷·硫醇硫醚

## 一、直觉：把醇的氢换成烃基，得到一个"惰性溶剂"

醇里那个活泼的羟基氢，要是被一个烃基顶替掉，会怎样？答案是——分子立刻变"佛系"了。这就是**醚**。

醚可以看成醇羟基里的氢被烃基取代（$\ce{R-O-H} \to \ce{R-O-R'}$），也可以看成水分子的两个氢都被烃基取代（$\ce{H-O-H} \to \ce{R-O-R'}$）。无论从哪个角度看，氧仍然是个饱和氧、$\mathrm{sp^3}$ 杂化，上面还带着两对孤对电子。

但少了那个活泼氢，醚就失去了醇的大半反应能力：它不和碱反应，不和氧化剂、还原剂反应，化学性质相当稳定。正因为"惰性 + 溶解性好 + 便宜"，醚成了实验室里最常用的一类有机溶剂。

可是凡事总有例外。本节的主角之一**环氧乙烷**，结构上是醚，性质上却活泼得离谱——因为它被掰成了一个张力极大的三元环。看懂"稳定的链状醚"与"活泼的三元环醚"之间的反差，是本节的关键。

:::definition{label=醚}
**醚**是两个烃基通过一个氧原子相连的化合物，通式 $\ce{R-O-R'}$。氧为 $\mathrm{sp^3}$ 杂化的饱和氧。两个烃基相同者为**对称醚**（单纯醚），不同者为**混合醚**。
:::

## 二、常见的醚与命名

醚普遍带特殊气味（"茴香醚"一看就知是从茴香里提取的）。几个要点：

- **乙醚** $\ce{(CH3CH2)2O}$：最熟悉的醚，有麻醉性。注意——乙醚含 **4 个碳**，不是 2 个；
- **甲醚（二甲醚）**、**甲乙醚**、**甲基叔丁基醚**、**苯醚**（芳香醚）、**茴香醚（苯甲醚）**；
- **环醚**：四氢呋喃（五元环，一处被氧取代）、四氢吡喃、1,4-二氧六环——后三者都是常用溶剂；还有环氧乙烷、环氧丙烷。

:::insight{label=做溶剂要具备什么}
能当大宗有机溶剂的醚，需同时满足三条：**化学性质稳定**（不参与反应、不干扰体系）、**溶解性好**、**便宜**（大量使用，成本必须低）。四氢呋喃、1,4-二氧六环正是如此。
:::

命名上，简单的叫"某醚"；结构复杂时不再以醚为母体，而把 $\ce{R-O-}$ 当作**烃氧基**（如甲氧基）来命名，例如二甲氧基戊烷、1,2-二甲氧基乙烷等。

:::note{label=冠醚}
**冠醚**也是一类醚，含多个氧的大环，如 12-冠-4、18-冠-6。18-冠-6 能很好地络合钾离子，属于主客体识别、超分子识别——这是分子间相互作用那条线里的内容。
:::

## 三、醚的稳定性与一个安全隐患：过氧化物

醚虽稳定，长期放置却会在 **$\alpha$ 位**（紧邻氧的碳上）被空气缓慢氧化，生成**过氧化物**。这些过氧化物积累多了，受热或撞击时**容易爆炸**。

:::pitfall{label=用醚前必须先检测}
使用放置过的醚（尤其乙醚、四氢呋喃）**必须先用淀粉-碘化钾试纸检验有无过氧化物**：若有，先加还原剂除掉过氧化物再用。长期存放的醚，务必鉴定后方可使用——这是实验室安全的硬规矩。
:::

## 四、醚键的反应：质子化与酸性断裂

醚的化学活性主要体现在**醚键（碳氧键）**上，而这一切都从氧的孤对电子开始。

**质子化生成𬭩盐：** 醚氧有孤对电子，在酸性条件下能被质子化，生成的正离子叫 **𬭩盐（佯盐）**。质子化后碳氧键被更好地极化，键变得更易反应。

**与路易斯酸加合：** 醚氧同样能与三氟化硼（$\ce{BF3}$）、三氯化铝（$\ce{AlCl3}$）发生酸碱加合——醚提供孤对电子，路易斯酸提供空轨道，生成酸碱加合物（$\ce{R2O \cdot BF3}$ 一类）。

**氢卤酸下的醚键断裂：**

:::example{label=醚的酸性断裂}
在氢卤酸条件下，质子先把醚氧质子化成𬭩盐，剩下的卤离子便去进攻与氧相连的碳，发生亲核取代，把醚键打断，生成一分子醇 + 一分子卤代烷：
$$
\ce{R-O-R' + HX -> R-X + R'-OH}
$$
活性顺序仍是 $\ce{HI > HBr > HCl}$（碘离子亲核性最强、氢碘酸酸性最强）。

不过这反应**在合成上价值不大**——用醚去制醇/卤代烷并不划算，通常只作为醚键性质的说明。
:::

## 五、威廉森合成（醚的制备）

> *说明：威廉森合成在 rec-16 录音中未单独展开（课程在环氧乙烷开环后即结束），但其原理已在 11.2"酚氧负离子 + 卤代烷成醚"中体现。以下据有机化学通识补全框架，不编造录音以外的具体反应实例。*

:::definition{label=威廉森醚合成}
**威廉森合成**是制备醚（尤其是混合醚）最常用、最可靠的方法：用一个**烷氧基（或酚氧基）负离子**作亲核试剂，去进攻**卤代烷**（或磺酸酯）的 $\alpha$ 碳，按 $\mathrm{S_N2}$ 机理发生亲核取代，形成 $\ce{C-O-C}$ 键成醚。
$$
\ce{R-O^- + R'-X ->[S_N2] R-O-R' + X^-}
$$
:::

:::insight{label=选料的智慧}
既然走 $\mathrm{S_N2}$，**卤代烷一端应尽量选伯卤代烷**（位阻小、利于背面进攻），把空间庞大的一端放到烷氧基负离子那边。若误用叔卤代烷，则消除（$\ce{E2}$）会大量竞争、生成烯烃而非醚。这与 11.1 中"$\mathrm{S_N2}$ 由底物碳位阻决定"的原则一脉相承。
:::

回看 11.2：酚氧负离子与硫酸二甲酯/卤代烷成醚，本质就是威廉森合成；醇钠与卤代烷成醚亦然。这也再次印证了"铁三角"里官能团互换的思路。

## 六、环氧乙烷：被环张力"激活"的醚

我们一直说醚惰性、做溶剂，**环氧乙烷却是个大例外**。它结构上是环醚，性质上却异常活泼。

:::insight{label=环张力从哪来}
环氧乙烷由两个碳、一个氧组成三元环，氧的键角被压到约 **60°**，几乎是个标准正三角形——键角变形极大，**环张力远大于环丙烷**。（环丙烷虽也画成三角形，但成键是"香蕉键"，键角变形没那么剧烈；环氧乙烷的键角变形则货真价实。）正是这股要把环"撑开"的张力，让环氧乙烷极易**开环**。它通常是无色、有异味的气体。
:::

### 环氧乙烷的合成价值

环氧乙烷在酸或碱催化下，能与一大批试剂开环反应：

| 试剂 | 产物片段 |
| --- | --- |
| 水 $\ce{H2O}$ | 乙二醇（再聚合得聚乙二醇） |
| 醇 $\ce{ROH}$ | 一端羟基、一端烷氧基 |
| 氨 $\ce{NH3}$ | 氨基醇（氨基 + 羟基） |
| 氯化氢 $\ce{HCl}$ | 氯醇（氯 + 羟基） |
| 氰化钠 $\ce{NaCN}$ | 引入氰基 |
| 格氏试剂 $\ce{RMgX}$ | 引入烃基（增长两个碳） |

:::insight{label=为什么环氧乙烷这么值钱}
除聚合外，这些反应都生成一个固定的 **$\ce{-CH2CH2OH}$ 片段**（由开环提供），并在另一端引入各式各样的基团（氨基、氯、氰基……）。

一个分子上同时挂着**两种不同官能团**，在合成上极有价值——一头做一种反应、另一头做另一种反应。这类双官能团化合物身价倍增：乙二醇本身很便宜，可一旦把一个羟基换成氯、氨基，价格能差出几百倍。所以环氧乙烷是工业与合成上极其重要的中间体：天生带羟基，再按试剂引入第二个官能团。
:::

### 区域选择性：酸催化与碱催化进攻位置相反

这是本节最精彩、也最易错的对比。对一个**不对称（取代的）环氧乙烷**，酸、碱催化开环得到的产物方向**正好相反**。

:::derivation{label=碱催化——进攻位阻小的碳}
**碱催化**（如乙醇钠 $\ce{NaOEt}$ 条件）是典型的 **$\mathrm{S_N2}$**。体系中有强亲核试剂（乙氧基负离子 $\ce{EtO^-}$），它直接进攻——而 $\mathrm{S_N2}$ 由**底物碳的位阻**决定，故进攻**位阻较小的碳**（取代少的那个，如 1 号碳），打断该处碳氧键，羟基留在另一碳（2 号碳）上。

口诀：**碱催 → $\mathrm{S_N2}$ → 进攻位阻小的碳。**
:::

:::derivation{label=酸催化——进攻位阻大、取代多的碳}
**酸催化**条件下没有强亲核试剂（不存在烷氧基负离子，只有质子和中性醇）。所以：
1. 先发生**氧的质子化**，形成环上的氧正离子；
2. 亲核性较弱的中性醇分子靠孤对电子去进攻。进攻谁？看断裂时**哪个碳上生成的碳正离子更稳定**——取代基较多的碳上正电荷累积更稳定，故进攻**取代多、位阻大的那个碳**，羟基生成在另一端。

口诀：**酸催 → 进攻取代多的碳**（由"类三级碳正离子"的稳定性决定）。

为何叫"类似"三级碳正离子？因为并未真正生成自由碳正离子——若真生成，产物会外消旋化，而实验未见外消旋。只是在过渡态中那个碳氧键变弱时，取代多的碳上正电荷累积更多，对中性亲核试剂的吸引更强，反应便朝这个方向走。
:::

:::pitfall{label=酸碱开环方向千万别记反}
- **碱催化（$\mathrm{S_N2}$）**：进攻**位阻小**的碳——由底物位阻决定；
- **酸催化**：进攻**位阻大 / 取代多**的碳——由碳正离子稳定性决定。

两者产物结构不同、机理不同。这是考试高频陷阱，务必分清"谁决定方向"：碱看位阻、酸看碳正离子稳定性。
:::

### 磷霉素：环氧结构的药物应用

:::note{label=磷霉素}
**磷霉素（Fosfomycin）** 是一种含**环氧（三元环）结构**的抗生素。其药理正是利用环氧环的高反应活性——开环后与细菌细胞壁合成相关酶的亲核基团共价结合，使酶失活，从而抑制细菌细胞壁合成。这是"环张力驱动开环"在药物分子设计中的真实应用。（具体酶学机制超出本章范围，此处仅点明环氧活性的应用价值，不展开 rec-16 以外的细节。）
:::

## 七、硫醇、硫酚与硫醚

> *说明：硫醇、硫酚、硫醚在 rec-16 录音中未展开讲解。以下据有机化学通识简要搭建框架（含氧化合物把氧换成硫的对应关系），不编造录音以外的具体反应数据。*

把醇、酚、醚里的氧换成同族的硫，就得到硫醇（$\ce{R-SH}$）、硫酚（$\ce{Ar-SH}$）、硫醚（$\ce{R-S-R'}$）。硫与氧同主族，性质既相似又有几点关键差异。

:::definition{label=含硫类似物}
- **硫醇**：$\ce{R-SH}$（巯基 $\ce{-SH}$ 连饱和碳），对应醇；
- **硫酚**：$\ce{Ar-SH}$，对应酚；
- **硫醚**：$\ce{R-S-R'}$，对应醚。
:::

### （1）酸性更强

:::insight{label=为什么硫醇比醇酸}
硫原子半径比氧大、电负性比氧小，$\ce{S-H}$ 键比 $\ce{O-H}$ 键更弱、更易断裂；同时生成的 $\ce{RS^-}$（硫负离子）半径大、负电荷分散得更开、更稳定。两点叠加，使**硫醇的酸性明显强于对应的醇**，硫酚也强于酚。因此硫醇能与较弱的碱（乃至部分重金属氢氧化物）成盐。
:::

### （2）二硫键与烫发原理

:::example{label=巯基的氧化——二硫键}
两个硫醇在温和氧化下可脱氢偶联，形成**二硫键（$\ce{-S-S-}$）**：
$$
\ce{2\,R-SH ->[{[O]}] R-S-S-R + H2O}
$$
该反应**可逆**——用还原剂又能把二硫键打回两个巯基。

这正是**烫发（卷发）的化学原理**：头发角蛋白中含大量半胱氨酸残基的巯基/二硫键。烫发时先用还原剂把原有二硫键打断（让头发可塑形），重新定型后再用氧化剂在新位置重新形成二硫键，使发型固定。冷烫、热烫都建立在"二硫键的断—接"之上。
:::

### （3）易被氧化

硫比氧更易被氧化，硫醇/硫醚的氧化比对应的醇/醚更容易、产物更丰富：

- 硫醇温和氧化 → 二硫化物（如上）；强氧化则可一路到磺酸 $\ce{R-SO3H}$；
- **硫醚**可被氧化为**亚砜**（$\ce{R-S(=O)-R'}$），进一步氧化为**砜**（$\ce{R-SO2-R'}$）。

（以上为含硫化合物的通识框架；具体试剂条件与产物比例 rec-16 未涉及，此处不臆造细节。）

## 八、易错点清单

:::pitfall{label=常见陷阱}
1. **乙醚含 4 个碳**，不是 2 个；命名复杂醚时改用"烃氧基"作取代基。
2. **放置过的醚先验过氧化物再用**——有爆炸风险，淀粉-碘化钾试纸检测、还原剂处理。
3. **威廉森合成走 $\mathrm{S_N2}$**：卤代烷一端宜用伯卤代烷，叔卤代烷会大量消除成烯烃。
4. **环氧乙烷开环方向**：碱催化（$\mathrm{S_N2}$）进攻**位阻小**的碳；酸催化进攻**取代多/位阻大**的碳——别记反，区分"谁决定方向"。
5. **环氧乙烷产物总带 $\ce{-CH2CH2OH}$ 片段**（开环固定贡献）。
6. **硫醇酸性强于对应的醇**（$\ce{S-H}$ 弱、$\ce{RS^-}$ 稳定），别想当然套用醇的弱酸性。
7. **二硫键可逆地断—接**是烫发原理；硫醚氧化得亚砜、砜。
:::

:::note{label=本节小结}
醚是醇羟基氢被烃基取代的产物，氧仍为 $\mathrm{sp^3}$ 饱和氧，因缺活泼氢而性质惰性，是常用溶剂（需化学稳定、溶解好、便宜）；但 $\alpha$ 位易生成过氧化物，用前须检测。醚的反应集中在醚键：氧可质子化成𬭩盐、与路易斯酸加合，氢卤酸下醚键断裂成醇 + 卤代烷（$\ce{HI>HBr>HCl}$，合成价值不大）。制备醚的主力是威廉森合成——烷氧/酚氧负离子对（伯）卤代烷做 $\mathrm{S_N2}$。环氧乙烷是惰性规律的例外：三元环 60° 键角带来巨大张力、极易开环，能与水/醇/氨/HCl/CN⁻/格氏试剂反应，固定生成 $\ce{-CH2CH2OH}$ 片段并引入第二官能团，合成价值极高；其区域选择性是核心考点——碱催化 $\mathrm{S_N2}$ 进攻位阻小的碳，酸催化进攻取代多的碳，磷霉素即环氧活性的药物应用。把氧换成硫，硫醇/硫酚酸性更强（$\ce{S-H}$ 弱、$\ce{RS^-}$ 稳），巯基可逆地氧化成二硫键（烫发原理），硫醚易氧化为亚砜、砜。其中威廉森合成与含硫部分在 rec-16 中未展开，本节据通识补全框架而未编造具体细节。
:::
