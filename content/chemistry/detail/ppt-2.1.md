:::insight{label=从"碎片"开始学命名}
学命名时，很多同学一上来就背"系统命名法六步走"，结果遇到 $\ce{(CH3)2CHCH2-}$ 这种基团就懵了：这叫"异丁基"还是"2-甲基丙基"？为什么 $\ce{CH2=CH-CH2-}$ 叫"烯丙基"而不叫"2-丙烯基"？

问题的根源在于：**有机物的名字是由"基团"和"母体"两部分拼起来的**。就像中文姓名有"姓"和"名"，有机物的名字也是"基团名 + 母体名"的组合。如果你不先认识常见的"基团"，就没办法读懂任何一个稍复杂分子的名字。本节我们就先把最常见的基团——烷基、烯基、炔基、芳基、环烃基、含杂原子基团——系统地过一遍，为后续的系统命名法打好基础。
:::

## 基团的分类

:::definition{label=基团}
**基团**（group / radical）是有机分子中去掉一个或几个原子后剩余的原子团，在命名中作为取代基出现。常见基团分为两大类：

- **烃基**：烷基、烯基、炔基、环烃基、芳基
- **含杂原子基团**：卤素（X）、含氧（O）基团、含氮（N）基团、含硫（S）基团
:::

## 烷基的命名

:::definition{label=烷基}
**烷基**（alkyl group）是烷烃分子中去掉一个氢原子后剩余的基团，通式为 $\ce{-C_nH_{2n+1}}$，用 R- 表示。

常见烷基（按碳数递增）：

| 名称 | 结构 | 备注 |
|------|------|------|
| 甲基 | $\ce{CH3-}$ | 最简单的烷基 |
| 乙基 | $\ce{CH3CH2-}$ | |
| 正丙基 | $\ce{CH3CH2CH2-}$ | 直链 |
| 异丙基 | $\ce{(CH3)2CH-}$ | 链端第二位有支链 |
| 正丁基 | $\ce{CH3CH2CH2CH2-}$ | 直链 |
| 异丁基 | $\ce{(CH3)2CHCH2-}$ | 链端第二位有 1 个甲基支链 |
| 仲丁基 | $\ce{CH3CH2CH(CH3)-}$ | 取代基连在仲碳上 |
| 叔丁基 | $\ce{(CH3)3C-}$ | 链端第二位有 2 个甲基支链 |
| 异戊基 | $\ce{(CH3)2CHCH2CH2-}$ | |
| 新戊基 | $\ce{(CH3)3CCH2-}$ | |
| 叔戊基 | $\ce{CH3C(CH3)(CH2CH3)-}$ | |
:::

:::insight{label=正/异/新/仲/叔的规律}
四个前缀的本质都是描述**支链位置**：

- **正（n-）**：直链，无支链
- **异（iso-）**：链端第二位碳上有 1 个甲基支链
- **新（neo-）**：链端第二位碳上有 2 个甲基支链
- **仲（sec-）**：取代基连接在仲碳（2°C）上
- **叔（tert-）**：取代基连接在叔碳（3°C）上

注意"正/异/新"描述的是**链的整体形状**，"仲/叔"描述的是**取代基连接点的碳类型**。两者是不同维度的描述。
:::

:::pitfall{label=混淆"异丁基"和"仲丁基"}
$\ce{(CH3)2CHCH2-}$ 是**异丁基**（链端有 1 个甲基支链，连接点在伯碳），$\ce{CH3CH2CH(CH3)-}$ 是**仲丁基**（连接点在仲碳）。两者分子式都是 $\ce{C4H9-}$，但结构完全不同。记忆口诀："异"看链端支链，"仲"看连接位置。
:::

:::example{label=识别丁基的四种异构}
**题目**：写出 $\ce{C4H9-}$ 的四种异构基团及名称。

**解**：$\ce{C4H9-}$ 共有 4 种异构体：

1. **正丁基**：$\ce{CH3CH2CH2CH2-}$（直链，连接点是伯碳）
2. **异丁基**：$\ce{(CH3)2CHCH2-}$（链端第二位有甲基支链，连接点是伯碳）
3. **仲丁基**：$\ce{CH3CH2CH(CH3)-}$（连接点是仲碳）
4. **叔丁基**：$\ce{(CH3)3C-}$（连接点是叔碳，链端第二位有 2 个甲基支链）

记忆要点：先看连接点碳的类型（伯/仲/叔），再看链端支链情况。
:::

## 烯基与炔基

:::definition{label=烯基与炔基}
**烯基**（alkenyl）是烯烃去掉一个氢后剩余的基团；**炔基**（alkynyl）是炔烃去掉一个氢后剩余的基团。

常见烯基：

| 名称 | 结构 |
|------|------|
| 乙烯基 | $\ce{CH2=CH-}$ |
| 丙烯基 | $\ce{CH3CH=CH-}$（双键在 1,2 位） |
| 烯丙基 | $\ce{CH2=CH-CH2-}$（双键在 2,3 位，连接点饱和） |
| 异丙烯基 | $\ce{CH2=C(CH3)-}$ |
| 1,3-丁二烯基 | $\ce{CH2=CH-CH=CH-}$ |

常见炔基：

| 名称 | 结构 |
|------|------|
| 乙炔基 | $\ce{HC#C-}$ |
| 1-丙炔基 | $\ce{CH3-C#C-}$ |
| 2-丙炔基 | $\ce{HC#C-CH2-}$（炔丙基） |
:::

:::pitfall{label=混淆"丙烯基"和"烯丙基"}
这两个名字最容易混：

- **丙烯基** $\ce{CH3-CH=CH-}$：连接点就在双键碳上（不饱和）
- **烯丙基** $\ce{CH2=CH-CH2-}$：连接点在双键旁的饱和碳上

记忆：**"烯丙基"是"烯丙"——双键+丙基的连接点**，连接点离双键隔了一个 $\ce{CH2}$；而"丙烯基"连接点直接在双键碳上。两者化学性质差异很大：烯丙基上的反应更活泼，因为可以形成烯丙基正离子/自由基（p-π 共轭稳定）。
:::

## 含苯环基团与环烃基

:::definition{label=芳基与环烃基}
**芳基**（aryl group）是芳香烃去掉一个氢后剩余的基团，最常见的是**苯基** $\ce{C6H5-}$（Ph-）。

常见含苯环基团：

| 名称 | 结构 |
|------|------|
| 苯基 | $\ce{C6H5-}$（Ph-） |
| 对甲苯基 | $\ce{p-CH3C6H4-}$ |
| 苄基 | $\ce{C6H5CH2-}$（Bn-，苯环+ $\ce{CH2}$） |
| 三苯甲基 | $\ce{(C6H5)3C-}$ |
| 苯乙烯基 | $\ce{C6H5CH=CH-}$ |

常见环烃基：

| 名称 | 结构 |
|------|------|
| 环丙基 | $\ce{cyclo-C3H5-}$ |
| 2-甲基环丙基 | 2-甲基取代的环丙基 |
| 环己基 | $\ce{cyclo-C6H11-}$ |
| 3-环戊烯基 | 含 1 个双键的环戊烯基 |
| 2,4-环戊二烯基 | 含 2 个双键的环戊二烯基 |
:::

:::pitfall{label=混淆"苯基"和"苄基"}
- **苯基** $\ce{C6H5-}$：直接连在苯环碳上
- **苄基** $\ce{C6H5CH2-}$：通过 $\ce{CH2}$ 连到苯环

记忆：**"苄"字带"艹"头**，比"苯"多一个 $\ce{CH2}$（"草"下加"亚甲基"）。两者化学性质差别巨大：苄基的 $\ce{CH2}$ 上的氢很活泼（易形成苄基自由基/正离子，p-π 共轭稳定），而苯基直接连时没有这种活性。
:::

## 含杂原子基团

:::definition{label=含杂原子基团}
含杂原子的基团按杂原子种类分类：

**卤素**：$\ce{-F}$（氟）、$\ce{-Cl}$（氯）、$\ce{-Br}$（溴）、$\ce{-I}$（碘）

**含氧基团**：

| 名称 | 结构 |
|------|------|
| 羟基 | $\ce{-OH}$ |
| 甲氧基 | $\ce{CH3O-}$ |
| 乙氧基 | $\ce{CH3CH2O-}$ |
| 苄氧基 | $\ce{C6H5CH2O-}$ |
| 乙酰氧基 | $\ce{CH3COO-}$ |
| 羰基 | $\ce{>C=O}$ |
| 甲酰基 | $\ce{-CHO}$ |
| 乙酰基 | $\ce{CH3CO-}$ |
| 苯甲酰基 | $\ce{C6H5CO-}$ |
| 羧基 | $\ce{-COOH}$ |
| 甲氧羰基 | $\ce{CH3OOC-}$ |
| 氯甲酰基 | $\ce{ClCO-}$ |

**含氮基团**：$\ce{-NH2}$（氨基）、$\ce{-NHCH3}$（甲氨基）、$\ce{-N(CH3)2}$（二甲氨基）、$\ce{-NO2}$（硝基）、$\ce{-NO}$（亚硝基）、$\ce{-CN}$（氰基）

**含硫基团**：$\ce{-SH}$（巯基）、$\ce{-SCH3}$（甲硫基）、$\ce{-SO3H}$（磺酸基）、$\ce{-SO2CH3}$（甲砜基）、$\ce{-SOCH3}$（甲亚砜基）
:::

:::insight{label=为什么要记这么多基团名}
有机物的系统命名法本质是"主链 + 取代基"，而取代基就是各种基团。如果你不认识"乙酰基" $\ce{CH3CO-}$，看到"苯乙酮"这个名字就反应不过来它其实是 $\ce{C6H5COCH3}$；不认识"苄基" $\ce{C6H5CH2-}$，看到"苄醇"就不知道是 $\ce{C6H5CH2OH}$。

记忆策略：
1. **按杂原子归类**：先记含氧、含氮、含硫的"特征基团"
2. **按碳骨架归类**：再记烷基、烯基、芳基的常见变体
3. **多写多画**：每个基团都画一遍结构，建立"名字-结构"的双向映射
:::

:::example{label=从结构识别基团}
**题目**：识别下列基团的名称。

(a) $\ce{(CH3)3C-}$  (b) $\ce{CH2=CH-CH2-}$  (c) $\ce{C6H5CH2-}$  (d) $\ce{CH3CO-}$  (e) $\ce{-NO2}$  (f) $\ce{CH3OOC-}$

**解**：
- (a) **叔丁基**：连接点是叔碳，链端第二位有 2 个甲基
- (b) **烯丙基**：双键+ $\ce{CH2-}$，连接点饱和
- (c) **苄基**：苯环+ $\ce{CH2-}$
- (d) **乙酰基**：甲基+羰基 $\ce{CH3CO-}$
- (e) **硝基**：$\ce{-NO2}$
- (f) **甲氧羰基**：$\ce{CH3O-CO-}$（甲氧基+羰基）
:::

:::pitfall{label=把"羰基"和"酰基"混淆}
- **羰基** $\ce{>C=O}$：是结构单元，两端都连碳时叫"酮羰基"，一端连 H 时叫"醛羰基"
- **酰基** $\ce{RCO-}$：是基团，必须有一端是"自由连接点"

例如 $\ce{CH3CO-}$ 是**乙酰基**（一种酰基），而 $\ce{>C=O}$ 是**羰基**（结构描述）。考试时不要写"乙酰基是羰基"，应说"乙酰基含有一个羰基"。
:::

:::note{label=本节小结}
- 基团分两大类：**烃基**（烷基/烯基/炔基/环烃基/芳基）和**含杂原子基团**（卤素/含氧/含氮/含硫）；
- 烷基前缀：正（直链）、异（链端 1 个甲基支链）、新（链端 2 个甲基支链）、仲（连仲碳）、叔（连叔碳）；
- 易混对：
  - 异丁基 vs 仲丁基（前者看支链，后者看连接点）
  - 丙烯基 vs 烯丙基（前者连双键碳，后者连饱和碳）
  - 苯基 vs 苄基（前者直接连苯环，后者多一个 $\ce{CH2}$）
  - 羰基 vs 酰基（前者是结构单元，后者是基团）
- 下一节我们将学习烷烃的系统命名法——把基团名和主链名拼起来，给任何烷烃一个唯一名字。
:::
