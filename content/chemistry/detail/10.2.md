# 10.2 消除反应 E1/E2 与扎伊采夫规则

## 一、直觉：取代之外还冒出了一个烯烃

上一节我们把卤代烷在亲核试剂环境里的反应讲成了"亲核取代"，生成醇。但做实验时会发现一件怪事：除了醇，反应瓶里还冒出了**烯烃**。

这个烯烃从哪来？仔细看：从卤代烷到烯烃，相当于卤代烷上的一个卤原子，和它旁边碳上的一个氢原子，被一起拔走，剩下的两个碳之间补上一根双键。

$$\ce{R-CH2-CH2-X ->[\text{碱}] R-CH=CH2 + HX}$$

也就是说，**同一个卤代烷，在同样的环境里，既会发生取代、又会发生消除**——它们是一对竞争的反应。本节就来认识消除反应。

## 二、定义与分类

:::definition{label=消除反应}
在一个有机分子中**消去两个原子或基团**的反应，称为消除反应，记作 **E**。相邻碳上的消除往往生成不饱和键（碳碳双键或三键）。

例：
- $\ce{CH3-CHBr-CH2Br ->[\text{消除}] CH3-CH=CH2 + HBr}$（脱一分子卤化氢得丙烯）
- 醇脱一分子水也能得烯烃
- 二卤代烷连续两次消除可得炔
:::

消除反应不是卤代烷专属，很多有机分子都能发生；而且不只在相邻碳之间发生。按消去的两个基团所在位置，分为三类：

:::definition{label=α、β、γ 消除}
对卤代烷而言，连卤素的碳是 α 碳，旁边是 β 碳。
- **α 消除（1,1 消除）**：同一个碳上的氢和卤被消去，生成活性中间体卡宾，不常见。
- **β 消除（1,2 消除）**：相邻两碳上的氢和卤被消去生成烯烃，**最常见**，是本节重点。
- **γ 消除（1,3 消除）**：1,3 位消去，可生成三元环，不常见。
:::

β 消除一般在**强碱**条件下完成，例如在氢氧化钠的**醇溶液**中。和取代一样，消除也有两种机制：单分子消除 **E1** 和双分子消除 **E2**。

## 三、E1 机制：单分子消除

E1 的第一步与 $S_N1$ 完全相同：叔卤代烷的碳卤键异裂，生成碳正离子和离去的卤负离子。区别在第二步。

:::theorem{label=E1 机理}
- **第一步（决速步、耗能步）**：碳卤键异裂，生成平面碳正离子，与 $S_N1$ 一致。
- **第二步**：体系中的强碱去进攻碳正离子旁边 β 碳上的氢，把 β 氢拔走，电子回填生成双键，得到烯烃。
- 与 $S_N1$ 共享第一步，因此同样**分步进行、可能伴随碳正离子重排**。
:::

:::insight{label=为何 E1 不太常见}
有机化学是先看到现象再用理论解释。碳正离子一旦生成，面前是一个赤裸的正电荷。以 $\ce{OH^-}$ 为例：它去进攻碳，表现的是**亲核性**（生成取代产物）；它去拔 β 氢，表现的是**碱性**（生成消除产物）。

一个正电荷明晃晃摆在那，$\ce{OH^-}$ 何必费劲绕到旁边去拔那个带电性弱得多的 β 氢？直接连上碳不香吗？所以**碳正离子一旦生成，更容易发生 $S_N1$ 取代**，E1 概率较低。但"概率低"不等于"不发生"——一大群分子里总有少数 $\ce{OH^-}$ 去拔了 β 氢，于是仍有少量 E1 消除产物。
:::

:::example{label=E1 实例与重排}
新戊基型溴代烷在醇溶液中按单分子机制反应：先生成碳正离子，该碳正离子有重排机会（甲基迁移）重排成更稳定的碳正离子。此后：
- 醇分子作亲核试剂进攻碳、再脱质子 → 醚（$S_N1$ 取代产物，主）。
- 若有碱拔 β 氢 → 烯烃（E1 消除产物，少量）。
由于该条件下缺乏足够强碱，主要得取代产物，但不能排除少量消除产物。
:::

## 四、E2 机制：双分子消除（重中之重）

消除反应**绝大多数走 E2**，所以 E2 最重要。

E2 中的 2 同样指**决速步双分子参与**。对消除而言，参与的两个分子是：底物卤代烷 + 碱（如 $\ce{OH^-}$）。

:::derivation{label=E2 协同过程}
1. 碱进攻 β 碳上的氢，开始拔氢，β 位 **C–H 键弱化**、逐渐断裂。
2. 与此同时，α 碳上的 **C–X 键也弱化**、逐渐断裂。
3. 与此同时，两碳之间的 **π 键开始形成、强化**。

整个反应共断**两根键**（一根 C–H、一根 C–X），并生成一根 π 键——这与亲核取代只断一根 C–X 键不同。三件事在同一步同步发生，没有中间体，这就是**协同过程**。
$$\ce{B^- + H-CH2-CH2-X -> B-H + CH2=CH2 + X^-}$$
:::

:::theorem{label=E2 机理特征}
- **双分子**：决速步由碱和卤代烷共同决定过渡态，速率 $\propto$ [底物][碱]。
- **一步完成**：拔氢、断 C–X、生成双键一气呵成（协同）。
- **反式消除**：被消除的 β 氢与离去卤原子需处于纽曼投影式的**反位**（下面详述）。
- 断两根键、生成一根 π 键。
:::

:::insight{label=进攻位点决定取代还是消除}
$\ce{OH^-}$ 既是碱又是亲核试剂——同一个物种，进攻位置不同，反应类型就不同：
- 进攻 **α 碳** → 亲核取代（$S_N$）。
- 进攻 **β 氢** → 消除（E）。

所以消除反应里一定伴随取代反应的竞争，因为消除用的碱本身就是亲核试剂。这就是"取代—消除竞争"的根源。
:::

## 五、扎伊采夫规则与区域选择性

当卤代烷有不止一种 β 氢可选时，消除往哪边走？

考虑 2-位连溴的分子：1,2 之间消除得端基双键，2,3 之间消除得中间双键。实验结果**不是随机的**——2,3 位双键占 81%，1,2 位只占 19%。

为什么？从产物稳定性看：位于碳链中间、取代基更多的那个烯烃对称性更好、更稳定。消除倾向于生成更稳定的烯烃。

:::theorem{label=扎伊采夫规则}
β 消除时若有多个 β 碳可选，主产物是消除了**含氢较少的那个 β 碳**上的氢——这样生成的双键上取代基更多、产物更稳定。

对照记忆：烯烃加成时的**马氏规则**讲"氢加到含氢较多的碳上"；消除时的**扎伊采夫规则**讲"拔走含氢较少的 β 碳上的氢"。
:::

:::definition{label=区域选择性}
烯烃最终生成在碳链端基还是中间，这种位置上的选择叫**区域选择性**。卤代烷消除的区域选择性可用扎伊采夫规则预测。
:::

## 六、立体选择性与反式消除

区域选择性定下了"81% 在 2,3 位"，但 2,3 位双键还分顺反——这又是一重选择。

实验：81% 的产物里，**反式（E 型）占 61%，顺式占 20%**。反式更稳定，是多数。

:::definition{label=立体选择性}
消除产物是顺式还是反式，这种顺反上的选择叫**立体选择性**。通常生成更稳定的反式（E 型）产物为主。
:::

于是卤代烷消除生成烯烃有两重选择：**区域选择性**（双键在哪个位置，扎伊采夫规则预测）和**立体选择性**（顺式还是反式，通常反式为主）。

### 反式消除的构象解释

为什么反式产物占多数？要从 E2 的立体要求说起。

:::derivation{label=为什么是反式消除}
回忆烷烃那章：像乙烷这样的分子，稳定构象是**交叉式**（六个原子互成交叉排列），重叠式能垒太高、几乎不存在。

E2 要同时消去 β 碳上的氢和 α 碳上的溴。两种极限构象里，重叠式不可能（能垒太高），只能走交叉式。在交叉式中，当**被脱除的 β 氢与被脱除的卤原子恰好处于反位（对位交叉）**时，是能量最低、最优势的构象。

所以 E2 走的是**反式消除**：被消除的两个原子处在未来生成烯烃那个平面的反位上。
:::

:::derivation{label=优势构象决定反式烯烃为主}
对一个两端都带大基团（甲基）的底物，能让 H 与 X 处于反位的构象有两种：
- 一种是两个甲基也处于对位交叉的反位；
- 另一种是转 60°，H 与 X 仍反位，但两个甲基不再都处于反位。

显然让**所有大基团都处于反位**的那个构象更稳定。稳定构象意味着活化能低、反应速度快、走这条通道的分子最多。沿此优势构象反式消除，自然生成**反式烯烃**；另一种构象消除则给出顺式烯烃。这就从构象上解释了反式烯烃为何是多数产物。
:::

:::insight{label=认识方向与学习方向相反}
我们学的时候说"因为反式消除，所以生成更多 E 型烯烃"；但真实的认识顺序是：实验上**先发现**大多数消除产物是 E 型烯烃，于是**反推**它走的是反式消除机制。把这个逻辑链搞清楚，才算真懂。
:::

### 消除产物的整体分布

对一个典型卤代烷，消除会有三个产物：一个端基烯烃（无顺反）、一个顺式产物、一个反式产物。

- 那个 19% 的端基烯烃叫**反扎伊采夫产物**。
- 顺式和反式都属于**扎伊采夫产物**（81%）。
- 扎伊采夫产物内部用区域选择性区分位置；扎伊采夫产物内部的顺反，由反式消除机制决定**反式（E 型）为多数**。

## 七、例题

:::example{label=判断消除速度——谁的 β 氢多}
在**浓氢氧化钾醇溶液**中（加热）脱除溴化氢，比较一组卤代烷的反应速度。

分析：浓碱的醇溶液加热是典型消除条件，脱 HBr 即消除。不管 E1 还是 E2 都要进攻 β 氢，所以**谁的 β 氢多、消除几率就大**。题目没限定消到哪个位置，消哪都行。故周围 β 氢最多者速度最大。
:::

:::example{label=二级卤代烷的取代/消除与消除方向}
一个二级卤代烷在强碱存在下：因取代有位阻，容易发生消除。若两侧都是 β 氢（共 4 个 β 氢），按反式消除机制，消除的是环平面（或纽曼式）反位上的那个氢。

若 β 消除有共轭方向可选（如往某一侧消能生成共轭烯烃），则倾向往共轭方向消除——共轭性导致了这里的区域选择性。
:::

## 八、易错点

:::pitfall{label=四个常见陷阱}
1. **把"反式消除"当口号背**：它的真实含义是——被消除的 β 氢与卤原子要处于纽曼投影式的**反位**，不是随便记四个字。
2. **混淆扎伊采夫与马氏规则**：马氏是加成（氢加到含氢多的碳），扎伊采夫是消除（拔走含氢少的 β 碳上的氢）。方向相反，别记串。
3. **以为 E1 常见**：碳正离子面前更易走取代，E1 概率很低；卤代烷生成烯烃绝大多数走 E2。
4. **答题不标主产物**：消除有区域选择性和立体选择性时，要点明主产物（含氢少 β 碳消除、反式为主），把取代和消除都写出却不标主产物是错的。
:::

:::note{label=本节小结}
- 消除反应 E：分子中消去两个原子/基团；按位置分 α(1,1)、β(1,2)、γ(1,3)，**β 消除最常见**，强碱（如 NaOH 醇溶液）下进行，得烯烃 + HX。
- **E1**：与 $S_N1$ 共享第一步（碳卤键异裂生碳正离子），分步、可重排；因碳正离子面前更易取代，故 E1 不常见。
- **E2**：双分子、一步协同、断两键生一根 π 键、**反式消除**（H 与 X 处纽曼式反位）；是消除的主流机制。
- 进攻 α 碳→取代，进攻 β 氢→消除，同一个碱/亲核试剂导致取代—消除竞争。
- **区域选择性**：扎伊采夫规则——消除含氢较少 β 碳上的氢，生成取代基更多、更稳定的烯烃。
- **立体选择性**：反式消除机制决定反式（E 型）烯烃为主产物，因其优势构象活化能最低。
:::
