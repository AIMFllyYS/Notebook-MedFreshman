# 14.1 硝基苯还原·重氮盐·偶氮化合物

含氮有机化合物这一章里，最值得反复琢磨的并不是某一个孤立的反应，而是一条贯穿始终的"合成主干线"：从苯出发，经硝化、还原、重氮化，最后兵分几路去做卤代、酚化、脱氮、偶联。把这条线在脑子里画通了，苯环上很多看似无从下手的取代产物，都能找到一条体面的合成路径。本节就沿着这条主干线走一遍。

## 一、直觉：为什么要绕道重氮盐

在芳香化合物那一章里，我们已经学过苯环上的亲电取代——卤代、硝化、磺化、傅克反应。那么为什么还要再学一条新路线？

原因在于：亲电取代受定位规则约束，有些取代产物你用亲电取代根本做不出来，或者做出来位置不对。比如想在苯环上直接引入一个碘、一个氟，或者想把某个位置变回氢、想接上一段染料发色团，亲电取代往往力不从心。这时候我们换一个思路：先在苯环上做一个氨基，把氨基转化成重氮盐（一个 $\ce{-N2+}$ 的氮正离子官能团），再让这个重氮盐去做各种各样的下游反应。重氮盐就像一个"万能转接头"，接上它之后，溴、氯、氟、碘、氰基、羟基、氢、偶氮基——想要什么基本都能换上去。

:::insight{label=这条主干线是从苯开始的}
往前回溯：苯胺由硝基苯还原而来，硝基苯由苯硝化而来。所以整条路线总体是"从苯开始"：

$$\ce{苯 ->[硝化] 硝基苯 ->[还原] 苯胺 ->[亚硝酸,低温] 重氮盐 ->[多路转化] 各种取代产物}$$

它给了我们制备苯的取代产物的"第二条路"，与直接亲电取代互为补充。注意：这套打法**只在芳香环上有合成价值**。在脂肪链上，重氮盐活性太高，产生的碳正离子会发生大量副反应，合成上没有意义。
:::

## 二、第一步：硝基苯的还原

要走这条线，第一件事是把硝基还原成氨基。录音里给了两类典型条件：

:::definition{label=硝基还原成氨基的条件}
- **铁 + 酸**（酸性条件的铁）：把苯环上的硝基还原成氨基。
- **硫化钠**（$\ce{Na2S}$）：也是把硝基还原成氨基的还原条件，而且它是一个可以**当量还原**的条件。

$$\ce{C6H5-NO2 ->[Fe/HCl 或 Na2S] C6H5-NH2}$$
:::

硫化钠最大的特点是"按当量办事"，这一点在多硝基底物上非常关键：

:::example{label=间二硝基苯的选择性还原}
间二硝基苯加硫化钠：如果**没写"过量"**，就当作一当量，只还原**一个**硝基；如果写了**过量**，就把**两个**硝基都还原掉。

$$\ce{1,3\text{-}(O2N)2C6H4 ->[Na2S(1\,当量)] 3\text{-}硝基苯胺}$$
$$\ce{1,3\text{-}(O2N)2C6H4 ->[Na2S(过量)] 间苯二胺}$$

铁加酸也能实现硝基到氨基的转化。
:::

:::insight{label=硝基→氨基还有"定位效应转化"的价值}
把硝基变成氨基，除了为重氮化做准备，本身在合成上还有一层用处：**改变定位方向**。硝基是间位定位基，氨基是邻对位定位基。所以拿到一个硝基苯，要再引入一个取代基时：

- 想引到**间位**：硝基不用动，直接利用它的间位定位；
- 想引到**邻、对位**：先把硝基还原成氨基（邻对位定位基），引入新基团后，必要时再把氨基变回硝基或转走。

苯环上仅凭一个硝基，就能衍生出很多花样。
:::

## 三、第二步：苯胺的重氮化

拿到苯胺后，下一步是低温下用亚硝酸把它变成重氮盐。

:::definition{label=重氮盐的生成}
苯胺与亚硝酸在酸性、**低温**条件下反应，生成重氮盐。条件一般写成两种形式：

- 直接用亚硝酸 $\ce{HNO2}$（但亚硝酸较毒）；
- 实验中常用**亚硝酸盐 + 无机酸**（如 $\ce{NaNO2 + H2SO4}$ 或 $\ce{NaNO2 + HCl}$）替代。

$$\ce{C6H5-NH2 ->[NaNO2,\ HCl][0\text{--}5^\circ C] C6H5-N2+\ Cl-}$$

重氮盐是一个**氮二正**（$\ce{-N2+}$），是带电荷的离子/基团。用盐酸得到 $\ce{-N2+\ Cl-}$，用硫酸得到对应的硫酸氢盐。
:::

:::pitfall{label=重氮盐一定要写对}
录音里特别点名：有时候明明知道是重氮盐，却把它写错了。要记住它是 $\ce{Ar-N2+}$ 这个氮二正结构，并且**带着配对的负离子**（$\ce{Cl-}$ 或 $\ce{HSO4-}$）。另外，重氮化是**低温反应**，温度高了重氮盐会分解，这一点条件不能漏。
:::

## 四、第三步：重氮盐的多路转化

重氮盐做好之后，下游分成几条岔路。把这几条岔路连起来记，是本节的骨架。

:::theorem{label=重氮盐的转化路线（取代型）}
**桑德迈尔反应（Sandmeyer）**：在**亚铜的卤化物**条件下，把重氮基换成卤素。可得到溴、氯取代产物，也能换上氰基；氰基又能进一步转化（水解成羧基、还原成氨甲基，详见 14.2 与合成题）。

$$\ce{Ar-N2+ ->[CuCl 或 CuBr] Ar-X}\qquad \ce{Ar-N2+ ->[CuCN] Ar-CN}$$

**席曼反应（Schiemann）**：把重氮盐转化成**氟**和**碘**的取代产物。

$$\ce{Ar-N2+ ->[Schiemann] Ar-F\ (或\ Ar-I)}$$

这样卤素一族（F、Cl、Br、I）都能通过重氮盐接到苯环上。
:::

:::theorem{label=重氮盐的转化路线（脱氮型与酚化）}
**次磷酸还原**：用次磷酸（$\ce{H3PO2}$）把重氮基直接**去掉**，该位置回到一个氢。这相当于"把氨基/重氮基所占的位置抹平"，在合成上常用来借助氨基的定位效应排好其他基团后，再把它本身脱掉。

$$\ce{Ar-N2+ ->[H3PO2] Ar-H}$$

**还原成苯酚**：重氮盐在亚硫酸钠/亚硫酸氢钠、酸性条件下，先还原成苯酚的盐，再在碱中处理即得苯酚。

$$\ce{Ar-N2+ ->[Na2SO3/H+][然后\ OH-] Ar-OH}$$
:::

## 五、第四步：偶联反应与偶氮化合物

这是本节的重头戏，也是重氮盐"不丢掉氮"的那条路。前面几条路都是把重氮基替换掉；偶联反应里重氮基**不替换**，而是带着它的氮去和另一分子芳环成键。

### 偶联的本质：苯环上的亲电取代

:::insight{label=重氮盐就是一个带正电的亲电试剂}
学亲电取代时，硝化、磺化、傅克反应的共同点是：在路易斯酸催化下先生成一个带正电的亲电试剂（硝基正、磺酸基、碳正离子），再去进攻苯环。而重氮盐 $\ce{Ar-N2+}$ 本身就是个带正电的试剂。所以它能直接充当亲电试剂，去进攻另一分子芳环，发生亲电取代，生成偶氮化合物。

**偶氮**指的是结构中间有一个氮氮双键 $\ce{-N=N-}$，两端各连一个芳环。
:::

:::definition{label=重氮组分与偶联组分}
偶联反应需要两个角色：

- **重氮组分**：提供重氮盐 $\ce{Ar-N2+}$ 的那部分（亲电试剂）；
- **偶联组分**：受进攻的那部分芳环。

由于氮二正的亲电性**不算很强**，偶联组分必须是**亲电受体性好**的芳环，即上面要带**给电子基**——一般用**酚（苯酚、萘酚）或芳胺（苯胺类）**。给电子基把环上电子云抬高，环才容易被氮正离子进攻。
:::

### 偶联遵守定位规则

:::example{label=酚和氨基上的偶联位置}
偶联仍然遵守芳香环那一章的定位规则——你就把它当作"在酚或芳胺上做一次亲电取代"来定位：

- **酚类**：羟基是强活化的邻对位定位基。若环上同时还有甲基这类弱活化基，由**强活化的羟基做主**。对位若被占据，就发生在羟基的邻位。
- **芳胺类**：氨基也是邻对位活化基，反应在邻、对位。邻、对位都空时，主产物在**对位**（位阻小）。例如重氮盐与 $N,N$-二甲基苯胺偶联，主产物在对位，邻位很少。

$$\ce{Ar-N2+ + C6H5-NMe2 -> Ar-N=N-C6H4-NMe2\ (对位为主)}$$
:::

:::example{label=萘环上的偶联}
萘比苯更容易发生亲电取代（多并了一个苯环，相当于又加了一个给电子基）。萘酚上的偶联是"同环"的邻对位取代：

- **$\alpha$-萘酚**（羟基在 $\alpha$ 位）：偶联发生在同环对位（4 位）；
- **$\beta$-萘酚**（羟基在 $\beta$ 位）：偶联发生在它的邻位。

萘酚与重氮盐偶联可得到红色染料**苯胺红**。用重氮盐和 $\beta$-萘酚偶联，则可在纸上写出**苏丹红**的结构（苏丹红正是一个偶氮基化合物——这只用于讲合成，绝不能用于食品）。
:::

## 六、为什么偶氮化合物是染料

偶氮化合物都有非常鲜艳的颜色，是有机染料的主要来源。这背后的道理在于**大共轭体系**与**分子轨道**。

:::insight{label=共轭、HOMO/LUMO 与吸收色}
偶氮分子从一端的取代基（如甲基、磺酸基）开始，经苯环、氮氮双键、另一苯环，形成一个非常大的共轭体系。共轭程度越大，分子的最高占有轨道（HOMO）和最低空轨道（LUMO）之间的能差越小。

- **HOMO**（Highest Occupied Molecular Orbital）：被电子占据的最高轨道；
- **LUMO**（Lowest Unoccupied Molecular Orbital）：最低的空轨道。

分子吸光时，HOMO 上的电子被激发跳到 LUMO，这一能差对应被吸收光的波长。可见光大约 380–680 nm（赤橙黄绿青蓝紫）。被吸收的那段光"挖掉"，剩下透出来的光就是我们看到的颜色——这叫**吸收色**；若激发态电子跳回时发光，则是**发光色**（荧光）。
:::

:::derivation{label=由吸收波长估算能差}
吸收波长 $\lambda$（纳米）与跃迁能 $E$（电子伏特）的近似关系：

$$E\ (\mathrm{eV}) \approx \dfrac{1240}{\lambda\ (\mathrm{nm})}$$

例如吸收 560 nm 的光：$E \approx 1240/560 \approx 2.2\ \mathrm{eV}$。560 nm 大致在绿—橙范围，被吸收后透出的颜色就偏向其互补色。

共轭程度越大 → HOMO/LUMO 间隙越窄 → 吸收越偏向低能量（红光）→ 透出色越偏紫。苯本身 HOMO 与 LUMO 间隙很大，激发约需 280 nm 紫外光，肉眼看不见吸收，所以苯是无色透明的；共轭程度不断提高后，分子开始吸收可见光，溶液就显出颜色来。
:::

## 七、染料实例与光异构

:::example{label=甲基橙作 pH 指示剂}
甲基橙是偶氮染料，可作 pH 指示剂：$\mathrm{pH} < 3.1$ 显红色，$\mathrm{pH} > 4.4$ 显黄色。

原理：溶液中质子浓度升高时，甲基橙在某个位置结合质子，**打破了原有的 $p\text{-}\pi$ 共轭**，分子共轭程度改变即结构改变，吸收的颜色随之改变，于是显出不同颜色。
:::

:::insight{label=偶氮化合物的光异构}
偶氮化合物极易发生**光异构**：在可见光照射下由顺式向反式异构，在紫外光照射下反式又能回到顺式。

放置较久、保护不好的苏丹 3 染料，里面其实是顺、反两种异构体的混合物，所以做薄层色谱（TLC）点板时会出现**两个点**；若是非常新的样品，可能只点出一个点。
:::

:::example{label=用 TLC 区分顺反式}
点板时跑得快慢由两种作用力较劲决定：分子极性与硅胶的作用（极性大、被硅胶抓得牢、跑得慢）；分子溶解性与展开剂的作用（溶解好、带得快）。多数情况下展开剂对待测分子溶解都好，所以**主要由极性决定**。

偶氮苯类化合物：**反式是非极性（极性小）的，跑得快；顺式极性大，跑得慢**。所以一块小小的薄层板既能判断样品坏没坏（坏了会有杂质点），又能凭极性判断顺反式结构。

光异构现象也使偶氮苯在材料领域有应用：把偶氮苯单元装进材料，用光驱动它在顺/反之间切换，材料的结晶态随之改变，从而实现**光控材料开关**。
:::

:::note{label=本节小结}
1. **主干线**：苯 →（硝化）硝基苯 →（铁/酸 或 硫化钠还原）苯胺 →（亚硝酸、低温）重氮盐 → 多路转化。这是制备苯环取代产物的第二条路，只对芳香环有合成价值。
2. **硝基还原**：铁加酸、硫化钠均可把硝基还原成氨基；硫化钠是当量还原，可在多硝基底物上选择性地只还原一个（不写"过量"即按一当量）。硝基→氨基还能实现间位定位基到邻对位定位基的转化。
3. **重氮盐**是氮二正 $\ce{Ar-N2+}$，低温酸性下由苯胺与亚硝酸（盐）生成，书写时务必带配对负离子。
4. **下游转化**：桑德迈尔（Cu 催化得 Cl/Br/CN）、席曼（得 F/I）、次磷酸（脱氮还氢）、亚硫酸盐还原（得苯酚）、偶联（得偶氮化合物）。
5. **偶联**本质是苯环上的亲电取代，重氮盐作亲电试剂，偶联组分须为带给电子基的酚或芳胺，位置遵守定位规则。
6. **偶氮化合物**因大共轭体系使 HOMO/LUMO 间隙变窄而显色，是染料主要来源；甲基橙作 pH 指示剂靠结合质子改变共轭，偶氮苯易光异构（顺反互变），TLC 中反式跑得快、顺式跑得慢。
:::
