:::insight{label=从"为什么果糖也能还原托伦试剂"说起}
托伦试剂是用来鉴别醛的——酮通常不反应。但果糖是酮糖，却也能被托伦试剂氧化！为什么？因为在弱碱条件下，果糖能通过**烯二醇中间体**异构化为葡萄糖和甘露糖（都是醛糖），进而被氧化。这就是"差向异构化"——酮糖与醛糖在碱催化下通过烯二醇互变。

单糖的化学性质丰富多彩：环式结构的苷羟基可形成糖苷；开链式的醛基/酮基可被氧化、还原；羰基可与苯肼成脎；多个羟基可成酯成醚；浓酸下脱水生成糠醛。本节我们系统学习这些反应，并扩展到双糖。
:::

## 单糖的化学性质

:::definition{label=糖苷的生成}
醛可以与两分子醇在酸作用下形成缩醛，但糖只能与**一分子醇**形成缩醛，称为**糖苷**（glycoside）。这是因为糖的醛基已与分子内的一个羟基形成环状半缩醛结构。

$$
\ce{α-D-葡萄糖 + CH3OH ->[无水, H+] α-D-葡萄糖甲苷 + H2O}
$$

**糖苷的特点**：
- 是一种**缩醛或缩酮**
- 在**碱中稳定**
- 在**稀酸或酶**的条件下可水解为原来的糖和醇
- 形成糖苷后，**无变旋现象**（因 C-1 已被固定为缩醛）
- 能与醇、酚、$\ce{-SH}$、$\ce{-NH2}$ 脱水形成缩醛型结构

**苷羟基**：糖的半缩醛羟基（C-1 上的 OH），是糖形成糖苷的反应位点。
:::

:::definition{label=单糖的氧化反应}
单糖的氧化反应取决于氧化剂的强弱：

**弱氧化剂**（托伦试剂、班乃德试剂、斐林试剂）：
- 氧化糖的醛基为羧基
- **所有单糖**（无论醛糖还是酮糖）都是**还原糖**
- 酮糖能在弱碱条件下异构化为醛糖，因此也能被氧化

$$
\ce{RCHO + 2[Ag(NH3)2]OH -> RCOONH4 + 2Ag v + 3NH3 + H2O}
$$

$$
\ce{RCHO + 2Cu(OH)2 + NaOH -> RCOONa + Cu2O v + 3H2O}
$$

**溴水**（温和氧化剂）：
- 只氧化醛糖为糖酸，**不氧化酮糖**
- 用于**鉴别醛糖和酮糖**

$$
\ce{醛糖 + Br2/H2O -> 糖酸}
$$

**稀硝酸**（强氧化剂）：
- 氧化醛糖的两端（醛基和 $\ce{-CH2OH}$）为**糖二酸**
- 酮糖在稀 $\ce{HNO3}$ 下发生**断链**

**酶催化氧化**：只氧化 $\ce{-CH2OH}$ 端，得**糖醛酸**
:::

:::theorem{label=差向异构化与还原糖}
**差向异构体**（epimer）：含有多个手性碳原子的旋光异构体中，若**只有一个手性碳原子的构型不同**，其他构型完全相同，这样的旋光异构体被称为差向异构体。

**差向异构化**（在弱碱条件下）：

$$
\ce{D-葡萄糖 <=>[烯二醇中间体] D-甘露糖 <=> D-果糖}
$$

**机理**：碱夺去 α-H，形成**烯二醇**中间体，再重新质子化时，可以从不同方向加成，得到葡萄糖、甘露糖或果糖。

**意义**：
- 解释了为什么酮糖（果糖）也能被弱氧化剂氧化（异构化为醛糖）
- 葡萄糖和甘露糖是 C-2 差向异构体

**还原糖**：凡是能被托伦试剂、班乃德试剂、斐林试剂氧化的糖。所有单糖（无论醛糖还是酮糖）都是还原糖。
:::

:::example{label=鉴别醛糖和酮糖}
**题目**：如何用化学方法鉴别葡萄糖（醛糖）和果糖（酮糖）？

**分析**：两者都是还原糖，都能与托伦试剂、斐林试剂反应。需用只氧化醛糖不氧化酮糖的试剂——溴水。

**解**：

分别加入**溴水**（$\ce{Br2/H2O}$）：
- **葡萄糖**（醛糖）：溴水**褪色**（醛基被氧化为羧基，生成葡萄糖酸）
- **果糖**（酮糖）：溴水**不褪色**（酮基不被溴水氧化）

$$
\ce{CH2OH(CHOH)4CHO + Br2 + H2O -> CH2OH(CHOH)4COOH + 2HBr}
$$

**注意**：不能用托伦试剂或斐林试剂鉴别，因为两者都反应（果糖通过差向异构化为醛糖被氧化）。
:::

:::definition{label=单糖的还原反应}
单糖的羰基可被还原为羟基，生成**糖醇**：

$$
\ce{D-葡萄糖 ->[NaBH4 或 H2/Ni] D-山梨糖醇}
$$

**D-山梨糖醇**：葡萄糖的还原产物，用作甜味剂、保湿剂。

**注意**：L-山梨糖醇是维生素C工业合成的重要中间体。
:::

:::definition{label=成脎反应}
单糖的羰基与**苯肼**反应，先生成苯腙，再与第二分子苯肼反应，生成**糖脎**（osazone）：

$$
\ce{D-葡萄糖 + 3PhNHNH2 -> D-葡萄糖脎 + 3NH3 + 2H2O}
$$

**特点**：
- **D-葡萄糖、D-甘露糖、D-果糖**生成**相同的糖脎**（因 C-1 和 C-2 都被"标准化"为脎结构，三者只在 C-1、C-2 不同）
- 糖脎是黄色晶体，有特定熔点，可用于**鉴定糖**
- 反应消耗 3 分子苯肼
:::

:::definition{label=单糖的脱水与显色反应}
**脱水反应**：糖类化合物有 β-羟基羰基结构，在酸性条件下可发生 β-羟基与 α-H 的脱水，生成 α,β-不饱和羰基化合物。强酸作用下分子内脱水生成**糠醛**（呋喃甲醛）或其衍生物。

**显色反应**（鉴别糖类）：

1. **莫力许反应**（Molisch）：糖的水溶液 + $\alpha$-萘酚乙醇溶液 + 沿管壁加浓硫酸 → 两层液面间形成**紫色环**。**所有糖均反应**，鉴别糖的常用方法。

2. **西列凡诺夫反应**（Seliwanoff）：酮糖在浓 $\ce{HCl}$ 下与间苯二酚反应生成**红色**物质；醛糖同样条件下两分钟内不显色。用于**鉴别酮糖和醛糖**。

3. **皮阿耳反应**（Bial）：戊糖在浓 $\ce{HCl}$ 下与 5-甲基间苯酚反应生成**绿色**物质。用于**鉴别戊糖和己糖**。

4. **狄斯克反应**（Dische）：脱氧核糖在乙酸和硫酸混合液中与二苯胺共热，生成**蓝色**物质。用于**鉴别脱氧核糖**。
:::

:::definition{label=单糖的成酯和成醚反应}
**成酯反应**：
- 与生物酶作用：$\alpha$-D-葡萄糖 → 1-磷酸-α-D-葡萄糖 → 1,6-二磷酸-α-D-葡萄糖（生物体内能量代谢）
- 与乙酸酐反应：$\alpha$-D-葡萄糖 → $\alpha$-D-葡萄糖五乙酸酯

**成醚反应**：糖苷在碱性条件下与烷基化试剂（如 $\ce{CH3I/NaOH}$）反应，所有羟基被甲基化为醚。
:::

## 双糖

:::definition{label=双糖}
**双糖**（disaccharide）：由两个糖单元构成的糖，是一个糖分子中的**半缩醛羟基**（苷羟基）和另一个单糖分子中的羟基失水得到的糖。两个单糖可以相同，也可以不同。

**脱水形式**：
- **(I) 两个单糖分子都以苷羟基脱水**：非还原糖，**无变旋现象**
- **(II) 一个糖分子贡献苷羟基，另一个糖分子贡献醇羟基**：还原糖，**有变旋现象**

**典型双糖**：
- **还原糖**：麦芽糖、纤维二糖、乳糖
- **非还原糖**：蔗糖
:::

:::theorem{label=常见双糖的结构与性质}

| 双糖 | 分子式 | 水解产物 | 苷键类型 | 还原性 | 变旋 |
|------|--------|---------|---------|--------|------|
| 麦芽糖 | $\ce{C12H22O11}$ | 2 分子 D-葡萄糖 | α-1,4 苷键 | 还原糖 | 有 |
| 纤维二糖 | $\ce{C12H22O11}$ | 2 分子 D-葡萄糖 | β-1,4 苷键 | 还原糖 | 有 |
| 乳糖 | $\ce{C12H22O11}$ | 半乳糖 + 葡萄糖 | β-1,4 苷键 | 还原糖 | 有 |
| 蔗糖 | $\ce{C12H22O11}$ | 葡萄糖 + 果糖 | α,β-1,2 苷键 | 非还原糖 | 无 |

**麦芽糖**：淀粉酶水解淀粉的产物，α-1,4 苷键连接两分子葡萄糖，有游离苷羟基，是还原糖。

**纤维二糖**：纤维素水解的产物，β-1,4 苷键连接两分子葡萄糖，有游离苷羟基，是还原糖。

**乳糖**：哺乳动物乳汁中的糖，β-1,4 苷键连接半乳糖和葡萄糖，有游离苷羟基，是还原糖。

**蔗糖**：葡萄糖和果糖通过 α,β-1,2 苷键连接（两个苷羟基都参与成键），**无游离苷羟基**，是非还原糖，无变旋现象。
:::

:::example{label=鉴别蔗糖和麦芽糖}
**题目**：如何用化学方法鉴别蔗糖和麦芽糖？

**分析**：蔗糖是非还原糖（无游离苷羟基），麦芽糖是还原糖（有游离苷羟基）。

**解**：

分别加入**托伦试剂**（或斐林试剂），加热：
- **麦芽糖**：出现**银镜**（或红色 $\ce{Cu2O}$ 沉淀）——还原糖
- **蔗糖**：**无变化**——非还原糖

**或**测变旋现象：
- **麦芽糖**：新配溶液有变旋现象（比旋光度随时间变化）
- **蔗糖**：无变旋现象（比旋光度恒定）

**关键**：是否有游离苷羟基决定了双糖的还原性和变旋性。
:::

:::pitfall{label=混淆还原糖与非还原糖}
**常见错误**：以为所有双糖都是还原糖。

**正确做法**：
- **还原糖**：有游离苷羟基的双糖（麦芽糖、纤维二糖、乳糖）+ 所有单糖
- **非还原糖**：无游离苷羟基的双糖（蔗糖——两个苷羟基都参与成键）

**判断方法**：看双糖中是否还有游离的半缩醛羟基（C-1 上的 OH）
- 有 → 还原糖（能被托伦/斐林氧化，有变旋）
- 无 → 非还原糖（不能被氧化，无变旋）
:::

:::pitfall{label=混淆醛糖和酮糖的氧化}
**常见错误**：以为酮糖不能被弱氧化剂氧化。

**正确做法**：
- **弱氧化剂**（托伦、斐林、班乃德）：酮糖**能**被氧化（通过差向异构化为醛糖）
- **溴水**：酮糖**不能**被氧化（只氧化醛糖）

**鉴别醛糖和酮糖**：用**溴水**（不是托伦/斐林），醛糖褪色，酮糖不褪色。
:::

:::note{label=本节小结}
- **糖苷**：糖 + 1 分子醇（酸催化）→ 缩醛；碱稳定，酸/酶水解；无变旋
- **苷羟基**：C-1 上的半缩醛羟基，是糖苷形成位点
- **氧化反应**：
  - 弱氧化剂（托伦、斐林、班乃德）：所有单糖都反应（还原糖）
  - 溴水：只氧化醛糖，不氧化酮糖（鉴别醛糖/酮糖）
  - 稀硝酸：醛糖→糖二酸；酮糖断链
  - 酶催化：糖醛酸
- **差向异构化**：弱碱下酮糖↔醛糖（经烯二醇中间体）；葡萄糖、甘露糖、果糖互为差向异构体（C-2）
- **还原糖**：所有单糖 + 有游离苷羟基的双糖
- **还原反应**：$\ce{NaBH4}$ 或 $\ce{H2/Ni}$ → 糖醇（葡萄糖→山梨糖醇）
- **成脎反应**：3 分子苯肼 → 糖脎（黄色晶体）；葡萄糖、甘露糖、果糖得相同糖脎
- **脱水**：强酸 → 糠醛（呋喃甲醛）
- **显色反应**：莫力许（所有糖，紫色环）、西列凡诺夫（酮糖，红色）、皮阿耳（戊糖，绿色）、狄斯克（脱氧核糖，蓝色）
- **成酯/成醚**：酶催化磷酸化；乙酸酐得五乙酸酯；糖苷碱性甲基化
- **双糖**：
  - 麦芽糖：α-1,4 苷键，2 葡萄糖，还原糖
  - 纤维二糖：β-1,4 苷键，2 葡萄糖，还原糖
  - 乳糖：β-1,4 苷键，半乳糖+葡萄糖，还原糖
  - 蔗糖：α,β-1,2 苷键，葡萄糖+果糖，非还原糖，无变旋
- 下一节我们将学习多糖与脂类化合物
:::
