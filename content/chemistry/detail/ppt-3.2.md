# ppt-3.2 弱相互作用对物理性质的影响

:::insight{label=从"为什么有沸点差异"说起}
看一组数据：乙烷（$\ce{C2H6}$，分子量 30）沸点 -88.5 ℃，乙醇（$\ce{C2H6O}$，分子量 46）沸点 78.3 ℃，乙胺（$\ce{C2H7N}$，分子量 45）沸点 17 ℃。三者分子量相近，沸点却差了 100 多度——为什么？

答案就在上一节学的弱相互作用里：乙烷只有色散力，乙胺有色散力 + 诱导力 + 取向力 + 弱氢键（$\ce{N-H \cdots N}$），乙醇有色散力 + 诱导力 + 取向力 + 强氢键（$\ce{O-H \cdots O}$）。**氢键的强弱差异直接造成了沸点的巨大差异**。

本节我们就来系统学习弱相互作用如何决定有机物的三大物理性质：**沸点、熔点、溶解性**。掌握这些规律，你就能从分子结构预测物理性质，反过来也能从物理性质反推分子间作用力。
:::

## 弱相互作用与沸点

:::theorem{label=沸点的影响因素}
沸点是液体分子克服分子间作用力进入气相的温度。**分子间作用力越强，沸点越高**。主要影响因素：

1. **分子量（碳原子数）**：碳原子数增加 → 分子链变长 → 分子间接触面增加 → 范德华力（特别是色散力）增强 → 沸点升高
2. **支链**：直链 > 支链；支链越多，沸点越低（支链使分子形状"球化"，接触面减小）
3. **极性**：极性分子 > 非极性分子（极性分子有诱导力和取向力）
4. **氢键**：能形成氢键的分子（含 $\ce{-OH}$、$\ce{-NH}$、$\ce{-FH}$）沸点显著高于不能形成氢键的分子
5. **氢键类型**：分子间氢键 > 分子内氢键（分子内氢键"消耗"了 H 供体/受体，分子间作用力反而弱）
:::

### 直链烷烃沸点随碳数增加

:::theorem{label=直链烷烃沸点递增规律}
直链烷烃随碳原子数增加，沸点升高：

| 化合物 | 碳数 | 沸点（℃） |
|--------|------|-----------|
| 甲烷 $\ce{CH4}$ | 1 | -161.6 |
| 丙烷 $\ce{C3H8}$ | 3 | -42.1 |
| 戊烷 $\ce{C5H12}$ | 5 | 36.1 |
| 庚烷 $\ce{C7H16}$ | 7 | 98.4 |
| 壬烷 $\ce{C9H20}$ | 9 | 150.7 |

**原因**：碳原子增加、分子链变长，分子与分子之间的接触面增加——范德华作用力（色散力）增强。
:::

:::insight{label=为什么甲烷接触面小而戊烷大}
甲烷是球形小分子，两个甲烷分子之间只能"点接触"，色散力作用面积小。戊烷是长链分子，两个戊烷分子可以"面接触"——整条链并排贴合，色散力作用面积大得多。所以戊烷的色散力远强于甲烷，沸点高出近 200 ℃。

这个规律可以推广：**分子越大、链越长、接触面越大，色散力越强，沸点越高**。
:::

### 支链使沸点降低

:::theorem{label=支链对沸点的影响}
同分异构体中，**直链烷烃的沸点大于支链烷烃**，支链越多，沸点越低。

戊烷异构体的沸点：
- 正戊烷（直链）：36 ℃
- 异戊烷（1 个支链）：28 ℃
- 新戊烷（2 个支链，球状）：9.5 ℃

**原因**：支链使分子形状趋向"球化"，分子间接触面积减小，色散力减弱。
:::

### 顺式烯烃沸点高于反式

:::theorem{label=顺反异构体的沸点差异}
顺式烯烃的沸点**高于**反式烯烃。

例如：
- 顺-2-丁烯：沸点 3.7 ℃
- 反-2-丁烯：沸点 0.9 ℃

**原因**：顺式烯烃的偶极矩大（两个烷基在同侧，偶极不能抵消），分子间取向力强；反式烯烃偶极矩小（两个烷基对称分布，偶极抵消），取向力弱。
:::

### 极性对沸点的影响

:::theorem{label=极性分子与非极性分子的沸点差异}
分子量相当时：
- **醚 > 烷烃**（醚是极性分子，有诱导力和取向力）
- **醛、酮 > 烷烃**（醛酮是极性分子，分子间作用力更强）
- **醇 > 醚**（醇有氢键，醚没有）

具体例子：
- 丙烷（$\ce{C3H8}$，分子量 44）：-42.1 ℃
- 甲醚（$\ce{C2H6O}$，分子量 46）：-24 ℃
- 乙醇（$\ce{C2H6O}$，分子量 46）：78.3 ℃
- 丙酮（$\ce{C3H6O}$，分子量 58）：56 ℃
- 丁烷（$\ce{C4H10}$，分子量 58）：-0.5 ℃
- 丙醛（$\ce{C3H6O}$，分子量 58）：48 ℃
:::

:::example{label=沸点比较}
**题目**：比较下列化合物的沸点高低：丙烷、甲醚、乙醇、丙醛、丙酮、丁烷。

**分析**：先看氢键，再看极性，最后看分子量。

**解**：沸点从高到低：
1. **乙醇**（78.3 ℃）：有 $\ce{-OH}$，形成 $\ce{O-H \cdots O}$ 氢键，最强
2. **丙醛**（48 ℃）：极性分子，有 $\ce{C=O}$，无氢键
3. **丙酮**（56 ℃）：极性分子，有 $\ce{C=O}$，无氢键（注：丙酮沸点 56 ℃ 高于丙醛 48 ℃，因丙酮分子对称性高、极性大；但都比烷烃高）

修正排序：乙醇（78.3）> 丙酮（56）> 丙醛（48）> 甲醚（-24）> 丁烷（-0.5）> 丙烷（-42.1）

**规律**：醇（氢键）> 醛酮（极性，无氢键）> 醚（弱极性）> 烷烃（非极性）
:::

### 羧酸的二缔合体

:::theorem{label=羧酸的高沸点}
羧酸因分子间形成**氢键二缔合体**（两个羧酸分子通过两个氢键形成环状二聚体），沸点比相近分子量的醇、酯还高。

$$
\ce{2 R-COOH <=> [R-COOH \cdots HOOC-R]\text{（二缔合体）}}
$$

二缔合体相当于分子量"翻倍"，需要更多能量才能汽化，所以羧酸沸点显著高于同碳数的醇。
:::

### 胺的沸点规律

:::theorem{label=胺的沸点}
**碳原子数相同的胺**：沸点顺序为 **伯胺 > 仲胺 > 叔胺**

例如：
- 二乙胺（仲胺，分子量 73）：56 ℃
- 丁胺（伯胺，分子量 73）：77.8 ℃

**原因**：
- 伯胺 $\ce{R-NH2}$ 有 2 个 N-H，可形成分子间氢键
- 仲胺 $\ce{R2NH}$ 有 1 个 N-H，可形成氢键但少
- 叔胺 $\ce{R3N}$ 无 N-H，**不能**形成分子间氢键

**同碳数烷烃、伯胺、醇的沸点**：**醇 > 伯胺 > 烷烃**

例如：
- 乙烷（-88.5 ℃）< 乙胺（17 ℃）< 乙醇（78.3 ℃）

**原因**：氢键 $\ce{N-H \cdots N}$ 比 $\ce{O-H \cdots O}$ 弱（N 电负性 3.1 < O 3.5）
:::

:::example{label=胺的沸点比较}
**题目**：比较乙烷、乙胺、乙醇的沸点。

**分析**：
- 乙烷：非极性，只有色散力
- 乙胺：极性 + $\ce{N-H \cdots N}$ 氢键（弱）
- 乙醇：极性 + $\ce{O-H \cdots O}$ 氢键（强）

**解**：沸点顺序：**乙醇（78.3 ℃）> 乙胺（17 ℃）> 乙烷（-88.5 ℃）**

差异主要来自氢键强弱：$\ce{O-H \cdots O}$（20-30 kJ/mol）> $\ce{N-H \cdots N}$（约 10 kJ/mol）> 无氢键。
:::

### 分子内 vs 分子间氢键对沸点的影响

:::theorem{label=氢键类型与沸点}
**分子内氢键化合物 < 分子间氢键化合物**（沸点）

硝基苯酚异构体的沸点（0.009 MPa）：
- 邻硝基苯酚（分子内氢键）：100 ℃
- 间硝基苯酚（分子间氢键）：194 ℃
- 对硝基苯酚（分子间氢键）：分解（更高）

**原因**：邻位硝基苯酚形成分子内氢键（$\ce{-OH}$ 与 $\ce{-NO2}$ 的 O 之间），"消耗"了 H 供体和受体，分子间不能再形成氢键，所以分子间作用力弱，沸点低。间位和对位无法形成分子内氢键，只能形成分子间氢键，分子间作用力强，沸点高。
:::

## 弱相互作用与熔点

:::theorem{label=熔点的影响因素}
熔点不仅取决于分子间作用力，还取决于**分子的对称性**：

1. **对称性越好，熔点越高**（对称分子在晶格中排列更紧密，晶格能更高）
2. **分子间作用力越大，熔点越高**

注意：熔点与沸点的规律**不完全一致**——沸点主要由分子间作用力决定，熔点还要考虑晶格堆积效率。
:::

:::theorem{label=烷烃熔点的奇偶效应}
直链烷烃的熔点随碳原子数增加而升高，但有一个有趣的**奇偶效应**：

- **含偶数碳原子的烷烃熔点升高幅度 > 含奇数碳原子的升高幅度**
- 偶数碳烷烃的对称性优于奇数碳烷烃，导致其熔点高于相邻奇数碳烷烃

例如：
- 正戊烷（5C）：熔点 -130 ℃，沸点 36 ℃
- 异戊烷（5C）：熔点 -160 ℃，沸点 28 ℃
- 新戊烷（5C）：熔点 -17 ℃，沸点 9.5 ℃

注意：新戊烷虽然沸点最低（球状，色散力弱），但**熔点最高**（-17 ℃），因为它的对称性最好（四面体对称），在晶格中堆积紧密。
:::

:::insight{label=沸点 vs 熔点的差异}
新戊烷是"沸点最低、熔点最高"的典型例子，完美展示了沸点和熔点规律的差异：

- **沸点**：分子间作用力决定 → 新戊烷球状，色散力弱 → 沸点最低
- **熔点**：分子间作用力 + 对称性决定 → 新戊烷四面体对称，晶格堆积紧密 → 熔点最高

类似地，顺-2-丁烯沸点（3.7 ℃）高于反-2-丁烯（0.9 ℃），但熔点（-139 ℃）低于反-2-丁烯（-105.5 ℃）——反式对称性更好，熔点更高。
:::

:::pitfall{label=用沸点规律直接套熔点}
不要以为"沸点高熔点也高"。新戊烷、反-2-丁烯都是反例：沸点低但熔点高，因为对称性在熔点中起重要作用。判断熔点时一定要**同时考虑分子间作用力和对称性**。
:::

## 弱相互作用与溶解性

:::definition{label=相似相溶}
**相似相溶**（like dissolves like）：极性相似的化合物之间具有良好的相互溶解能力；极性差别很大的化合物之间相互溶解能力差。

具体规律：
- 极性分子溶于极性溶剂（如水、乙醇）
- 非极性分子溶于非极性溶剂（如己烷、苯、乙醚）
- 极性分子难溶于非极性溶剂，反之亦然
:::

:::insight{label=相似相溶的本质}
"相似相溶"的本质是**分子间作用力的匹配**：

- 极性溶剂（如水）分子间有强相互作用（氢键、取向力）。如果溶质也是极性的或能形成氢键，溶质分子可以"融入"溶剂的相互作用网络，溶解度高。
- 非极性溶质无法参与水的氢键网络，反而会破坏水的氢键结构（疏水效应），所以难溶于水。
- 非极性溶剂（如己烷）分子间只有色散力。非极性溶质可以靠色散力与溶剂分子相互作用，溶解度高。极性溶质在非极性溶剂中无法形成强相互作用，溶解度低。

这就解释了：乙醇（有 $\ce{-OH}$）与水任意比例互溶，而己烷（非极性）与水几乎不溶。
:::

:::example{label=溶解性预测}
**题目**：预测下列物质的溶解性。

(a) $\ce{CH3CH2OH}$ 在水中  (b) $\ce{CH3(CH2)4CH3}$（己烷）在水中  (c) $\ce{CH3COOH}$ 在水中  (d) $\ce{CH3Cl}$ 在水中

**解**：
- (a) 乙醇有 $\ce{-OH}$，能与水形成氢键 → **任意比例互溶**
- (b) 己烷是非极性分子，与水极性差异大 → **难溶于水**
- (c) 乙酸有 $\ce{-COOH}$，能与水形成氢键 → **易溶于水**（且能电离）
- (d) 一氯甲烷是极性分子，但极性较弱，且不能形成氢键 → **微溶于水**

**规律**：含 $\ce{-OH}$、$\ce{-NH2}$、$\ce{-COOH}$ 等能形成氢键的基团 → 易溶于水；纯烃基 → 难溶于水。
:::

:::note{label=本节小结}
- **沸点**主要由分子间作用力决定：
  - 碳数 ↑ → 色散力 ↑ → 沸点 ↑
  - 支链 ↑ → 接触面 ↓ → 沸点 ↓
  - 极性 > 非极性（顺式 > 反式）
  - 氢键显著提高沸点：醇 > 醚，伯胺 > 仲胺 > 叔胺
  - 羧酸形成二缔合体，沸点特别高
  - 分子内氢键 < 分子间氢键（沸点）
- **熔点**由分子间作用力 + 对称性共同决定：
  - 对称性越好，熔点越高
  - 偶数碳烷烃熔点 > 相邻奇数碳烷烃
  - 新戊烷：沸点最低、熔点最高（对称性好）
- **溶解性**遵循"相似相溶"：
  - 极性分子溶于极性溶剂
  - 非极性分子溶于非极性溶剂
  - 能形成氢键的物质易溶于水

:::
