# 9.2 对映异构、非对映、内消旋与外消旋

## 一对镜像分子能引出多少概念

上一节我们建立了一个核心直觉：手性就是"实物与镜像不能重合"，而判断手性要靠找对称面、对称中心这两个对称因素。本节我们把镜头拉远，从"一个手性碳"逐步走到"两个手性碳"，再走到"根本没有手性碳"，沿途会遇到一串容易混淆的名词：对映异构体、非对映异构体、内消旋体、外消旋体。它们之间的关系，全都可以用上一节那把"对称因素"的尺子量清楚。

提前点明一条贯穿全节的主线：**手性碳原子和分子手性之间没有必然关系。** 含手性碳的分子不一定有手性，有手性的分子也不一定含手性碳。手性碳只是产生手性的一种原因，最终是否表现为手性，仍要回到分子整体有没有对称因素。

## 含一个手性碳：一定有手性

我们先把最简单的情形钉死。回顾乳酸、以及上一节那个二卤代乙烷，它们的共同特点是：分子里**只含一个手性碳原子**。

:::theorem{label=单手性碳定理}
**只含有一个手性碳原子的分子，一定是手性分子。** 这个"一定"是确定无疑的，没有例外。
:::

为什么如此干脆？因为当你在分子里造出一个手性碳时，就等于造出了一个手性中心，而分子的其余部分本身并没有手性、没有能与之"抵消"的对称结构。这个孤立的手性中心就会原原本本地反映到整个分子上，使分子失去对称面和对称中心。于是含一个手性碳的分子必有手性。

这样一个手性分子和它的镜像，构成一对**对映异构体**：它们组成相同、原子连接的种类也相同，只是空间排列互为镜像、不能重合，如同双胞胎——长得极像，却是两个不同的个体。

## 外消旋体：等量混合后旋光抵消

既然一对对映异构体一个右旋（+）、一个左旋（−），那么把它们**等量**混在一起会怎样？两者的旋光方向相反、大小相等，恰好互相抵消，混合物整体不再显示旋光性。

:::definition{label=外消旋体}
**外消旋体**是一对对映异构体的**等量混合物**，用 $(\pm)$ 表示。它必须同时满足两个条件：
1. 由**一对对映异构体**构成；
2. 两者**等量**混合（不等量则旋光抵消不完全，仍有净旋光）。

由于它由两种不同的分子组成，**外消旋体是混合物**，而不是纯净物。
:::

:::pitfall{label=外消旋体是混合物，不是纯净物}
课堂上有同学凭"看起来像一种东西"答成纯净物——这是用初中的纯净物/混合物概念误套了立体化学。外消旋体里实实在在地含有两种不同的分子（一对对映异构体），它们是两个不同的个体，所以外消旋体必然是混合物。注意：旋光性抵消，不等于分子变成了同一种。
:::

## 含两个不同手性碳：四种异构体，对映与非对映并存

把分子复杂化一步。如果一个分子含有**两个互不相同**的手性碳原子，每个手性碳上的基团都有各自的空间取向，组合下来会产生**四种**旋光异构体。我们把它们记作 a、b、c、d。

要厘清这四个分子的关系，可以在分子中间画一条虚线当作"镜子"，逐对比较：

- **a 与 b**：互为实物和镜像 → 它们是一对**对映异构体**。
- **c 与 d**：同样互为镜像 → 也是一对**对映异构体**。
- **a 与 c（以及 b 与 d）**：这两组之间**没有**镜像关系。

先判断单个分子的手性。以 a 为例：它含两个不同的手性碳，找对称面——任何一刀都切不出两侧对称的样子，没有；找对称中心——因为两个手性碳连的基团不一样，也没有。所以 a 是手性分子。a 有手性，作为它镜像的 b 自然也有手性，同理 c、d 都有手性。这四个全是手性分子。

那么"既不互为镜像、又都是手性"的 a 与 c 是什么关系？

:::definition{label=非对映异构体}
**非对映异构体**：两个分子的化学组成相同，但它们之间**没有对映（镜像）关系**。

在含两个不同手性碳的体系里，a 与 c、b 与 d 就是非对映异构体的关系——同一个分子骨架可以组成两组对映异构体（a/b 和 c/d），而这两组之间互为非对映异构。
:::

:::insight{label=对映 vs 非对映，用镜子一刀切开}
- **对映异构体**：互为实物与镜像、不能重合（a–b、c–d）。它们的熔点、沸点、旋光大小、折光率、极性等物理性质**相同**，只有在手性环境下（如偏振光照射、手性的化学/生理环境、手性容器中）才表现出理化性质的差异。
- **非对映异构体**：彼此**没有**镜像关系（a–c、b–d）。它们的物理性质**完全不同**，所有性质都不一样，仅仅是化学组成相同而已。

判别诀窍：先问"它俩是不是镜像？"——是 → 对映；不是、但组成相同 → 非对映。
:::

至此可见，含两个手性碳的分子**也能有手性**。我们已有两条线索：含一个手性碳的分子一定有手性；含两个（不同的）手性碳时仍可能有手性。但下面这个例子会打破"含手性碳就有手性"的惯性。

## 含两个相同手性碳：内消旋体登场

换一个分子——酒石酸。它也含两个手性碳，但这两个手性碳是**相同的手性碳原子**：所谓相同，是指两个手性碳上各自连接的四个基团完全一样（上面那个碳连着羟基、羧基、氢、以及另一含手性碳的支链；下面那个碳连着羟基、氢、羧基、以及对应的支链——这一堆和那一堆是一样的）。

含两个相同手性碳的分子，能产生几种异构体？仍然用对称因素来判断手性：

- 写成某一种排布时，找不到对称面，也找不到对称中心 → 这种排布是**手性的**，它有一个对映异构体。
- 但如果把其中一个手性碳上的羟基和氢对调一下（即让两个手性碳一个是 S、一个是 R 那样的排布），分子内部就**出现了一个对称面**——在中间切一刀，上下两半互成镜像、彼此对称。一旦有了对称面，这种排布就**没有手性**了。

:::definition{label=内消旋体}
**内消旋体**：含有两个（或多个）相同手性碳原子的分子，由于分子**内部存在一个对称面**而使整个分子不具手性、没有旋光性。它的"消旋"来自分子**内部结构的对称**，故称内消旋。
:::

:::theorem{label=相同手性碳的两种可能}
**含有两个相同手性碳原子的分子，有可能有手性，也有可能没有手性。** 是否有手性，取决于具体排布下分子内是否存在对称面——这正是上一节"用对称因素判断手性"那把尺子的直接应用。
:::

这个例子彻底说明：**手性碳原子和分子手性没有必然关系。** 内消旋体含两个手性碳，分子却没有手性。所以遇到含两个及以上手性碳的分子，绝不能数着手性碳就下结论，必须回到"有没有对称面、有没有对称中心"去判断。

## 内消旋体 vs 外消旋体：一张对照表

两种都"消旋"（整体不旋光），但来路完全不同，必须分清。

:::insight{label=四个概念的辨析}
| | 是纯净物还是混合物 | 是否手性分子 | 是否旋光 | 消旋来源 |
|---|---|---|---|---|
| **内消旋体** | 纯净物（单一化合物） | 否 | 否 | 分子内部对称面 |
| **外消旋体** | 混合物（一对对映体等量混合） | 含两种手性分子 | 否（互相抵消） | 等量混合，旋光相消 |

- **对映异构体**：物理性质（熔点、沸点、旋光、折光、极性等）相同，仅在手性环境下显出差异。典型例子如沙利度胺（反应停）的左旋体与右旋体。
- **非对映异构体**：彼此无对映关系，物理性质完全不同，仅化学组成相同。
- **外消旋体**：一对对映异构体的等量混合物，无旋光性，其物理性质与单一对映体不同（单一对映体有旋光，外消旋后无旋光）。
- **内消旋体**：单一化合物，非手性分子，无旋光性；其物理性质与对映体、非对映体及外消旋体都不一样。
:::

:::pitfall{label=内消旋 ≠ 外消旋}
- 内消旋体是**一种**纯净的、单一的化合物，靠分子**内部**的对称面而不旋光；它本身就**不是**手性分子。
- 外消旋体是**两种**对映体**等量混合**的混合物，靠两个分子的旋光**互相抵消**而不旋光；其中每个组分本身都是手性分子。
- 它们只是"整体不旋光"这一表观相同，本质（纯净物/混合物、是否手性）完全不同。
:::

## 没有手性碳，照样可以有手性

最后，我们打破第二个惯性认知：**有手性的分子，不一定含手性碳。** 判断手性最终仍回到"有没有对称因素"。下面几类结构都不含手性碳，却能产生手性。

:::example{label=联苯型分子}
联苯由两个苯环用一个 $\sigma$ 键连接。当连接处都是氢时，两个苯环共平面；但当端位取代上不同的大基团（如硝基等）时，两个苯环平面会因位阻而**正交**（互相垂直）。此时：找对称中心——两平面正交，找不到；找对称面——任取一面切下去，两侧基团都不一样，也没有。于是分子有手性，可它**没有任何手性碳原子**。其实物与镜像不能重合。

改造思路：若把一端的硝基换成与另一端相同的基团（如都变成羧基），就能沿某个面切出两侧对称的结构，对称面恢复，手性随之消失。可见联苯型分子的手性来自**两端取代基不同**——它是"有可能产生手性"的一类结构，具体要看取代情况。
:::

:::example{label=联二萘酚}
联二萘酚是联苯型的一种：两个萘环各带一个羟基。若两萘环共平面，两个羟基会因位阻"打架"，所以它们错开一定角度、不共平面。两环不共平面时，同样找不到对称面、也找不到对称中心，因而有手性，实物与镜像无法重合。
:::

:::example{label=丙二烯型（累积二烯）}
累积二烯是一个碳上同时连两个双键的结构。两个双键各由 $sp^2$ 碳形成一个平面，而这两个平面以**正交**方式存在（一个端基伸向纸面外、另一个伸向纸面里，中间部分与纸面平行）。这样的分子既无对称中心、也无对称面，因而有手性，且不含手性碳。其实物与镜像不能重合。手性同样来自两端连接的基团不同；若把两端基团改成相同（如两端都是苯环或都是萘环），切一刀两侧对称，手性消失。
:::

:::example{label=螺环类分子}
螺环的螺原子是一个 $sp^3$ 碳，它所携带的两个环面必然正交。在这种正交结构下也可能产生手性：若一端是氨基、另一端是甲基，怎么切两侧都对不上，既无对称面也无对称中心，分子有手性。但若把氨基也换成甲基（使一端的取代基相同），手性即消失。
:::

:::insight{label=杂原子也能成为手性中心}
手性中心不一定是碳。只要某原子连接了**四个互不相同的基团**，它就可能成为手性中心；若分子中只含这一个手性中心，则一定是手性分子。

- **手性氮、手性磷**：季铵盐、季磷盐中，氮/磷连接四个不同基团时，可产生手性。
- **亚砜（$\ce{S=O}$）**：硫上带一对孤对电子，由于分子翻转困难，这对孤对电子相当于一根键（可看作连了一个"0 号原子"）。于是硫上的苯基、氧、孤对电子等相当于四个不同基团，分子有手性，且能稳定地分离出来。
- **氨分子为何一般不算手性**：氨的氮上也有孤对电子，理论上连接三个不同基团时也构成手性氮。但氨的翻转极快，两个对映体在瞬间互相转化、无法分离，整体对外表现为外消旋，手性"固定不下来"。亚砜则因翻转很慢，手性能被固定，故有手性。
:::

:::note{label=本节小结}
- **含一个手性碳**：分子一定有手性，其镜像构成一对**对映异构体**（一个右旋 +，一个左旋 −）。
- **外消旋体**：一对对映异构体的**等量混合物**，记作 $(\pm)$，旋光抵消、整体不旋光，是**混合物**。
- **含两个不同手性碳**：产生四种旋光异构体；a–b、c–d 互为**对映异构体**，a–c、b–d 互为**非对映异构体**（无镜像关系，物理性质完全不同）。
- **含两个相同手性碳**：可能有手性、也可能没有；当分子内出现**对称面**时为**内消旋体**——单一纯净物、非手性、不旋光，消旋来自内部对称。
- **内消旋 vs 外消旋**：前者纯净物、靠内部对称面、本身非手性；后者混合物、靠等量相消、组分各自手性。
- **手性碳与手性无必然关系**：含手性碳不一定有手性（内消旋体），有手性不一定含手性碳（联苯、联二萘酚、丙二烯型、螺环、手性氮/磷/硫）。最终判据始终是：**有没有对称面、有没有对称中心。**
:::
