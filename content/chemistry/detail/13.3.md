# 13.3 与格氏试剂的反应及克莱森缩合

## 一、直觉：把"温柔的进攻"换成"带电的进攻"

上一节里，水解、醇解、氨解用到的亲核试剂——水、醇、氨——都是**中性分子**，我们利用的只是它们氧或氮上的孤对电子去进攻羰基碳。孤对电子虽能进攻，但毕竟"温柔"。

那如果换一个**直接带负电荷**的亲核试剂呢？反应会不会更猛、更彻底？答案是肯定的。本节的两大主角——**格氏试剂**和**克莱森缩合里的碳负离子**——都是这种"带电的进攻者"。更妙的是，它们进攻的结果不再是官能团之间打转，而是**真正延长了分子的碳链**。

:::insight{label=本节为什么和前面不一样}
前面学醛酮的反应，多是官能团之间的转化，**不涉及碳链变化**。而格氏试剂、克莱森缩合通过亲核加成-消除，把一个新的碳骨架接到了羰基碳上——这是**增碳/增长碳链**的反应。这是本节的核心价值。
:::

格氏试剂就是卤代烷和金属镁反应后生成的一个带负电荷的亲核试剂，相当于提供一个**烃基碳负离子**。羧酸衍生物碰上它，很容易反应。

## 二、酰氯与格氏试剂：温度决定停在酮还是冲到醇

酰氯活性最高，遇上格氏试剂这种强亲核试剂，反应非常剧烈，剧烈到"刹不住车"。要把产物控制在酮，还是放任它冲到醇，关键是**温度**。

:::theorem{label=酰氯 + 格氏试剂：温度控制}
- **低温（−78 ℃）→ 停在酮**：正常温度下反应太剧烈，会一路冲到醇。把温度降到 −78 ℃"hold 着点"，让反应克制，产物停在**醛/酮**阶段。
- **常温 → 直接到醇**：温度高时，第一次加成后中间体迅速被质子化，失去"回来变双键、把氯赶走再被第二次进攻"的驱动力前的克制，最终被第二分子格氏试剂继续进攻，**直接干成醇**。
:::

:::derivation{label=为什么低温能停在酮}
1. 格氏试剂的烃基碳负离子进攻酰氯羰基碳，亲核加成。
2. 消除：氧负回来变双键，把氯赶走——此刻得到的是**酮**（酰基上接了新烃基）。
3. 酮本身仍含羰基，还能被第二分子格氏试剂继续进攻 → 一旦继续就冲到醇。
4. **低温**抑制了第二步进攻的发生（反应太剧烈，需降温克制），于是停在酮；**常温**则拦不住，直奔醇。
:::

用酰氯 + 格氏试剂制醛酮是一个**增碳**反应，而且增碳自由度很大——你想在那个位置接什么烃基，就设计相应的格氏试剂接上去。

:::insight{label=位阻也能"刹车"}
如果分子本身有**空间位阻**，即便不低温，反应也会停在生成酮的阶段——因为位阻决定了反应活性，把第二次进攻挡住了。所以"停在酮"有两条路：**低温**或**靠位阻**。
:::

## 三、酯与格氏试剂：直奔叔醇

酯的活性比酰氯低，但碰上格氏试剂时反而**很难停在酮**，会直接冲到醇。

:::theorem{label=酯/甲酸酯 + 格氏试剂}
- **一般酯 + 格氏试剂 → 叔醇**：酯与苯基格氏试剂反应，最终生成一个**三苯基取代的叔醇**（羰基碳上接了两个来自格氏试剂的相同烃基）。
- **甲酸酯 + 格氏试剂 → 仲醇**：因为甲酸酯羰基碳上原本是个氢，加成两次后得到的是**仲醇**。
:::

:::derivation{label=为什么酯停不下来、走到叔醇}
1. 格氏试剂碳负离子进攻酯羰基碳，亲核加成、消除掉烷氧基 → 中间生成一个**酮**。
2. 但这个酮活性仍高，立刻被**第二分子**格氏试剂进攻、加成。
3. 第二次加成没有可消除的离去基团了，质子化后得到**叔醇**（羰基碳上挂了两个相同的新烃基）。
4. 甲酸酯因羰基碳上本是氢，两次加成后变成**仲醇**。
:::

这些反应都能由羧酸衍生物制备相应醛酮/醇，并**延长分子碳链**——正是它区别于普通醛酮官能团转化之处。

## 四、酰胺的特殊性与其他金属有机试剂

:::pitfall{label=含氢酰胺会"毁掉"格氏试剂}
酰胺**氮上有氢**时，这个氢酸性较大，会先把格氏试剂分解掉（格氏试剂是强碱，遇活泼氢即被破坏），所以**含 N—H 的酰胺一般很难与格氏试剂反应**。
:::

:::example{label=无氢酰胺与其他金属有机试剂}
- **氮上无氢的酰胺**（如酰基咪唑——也是酰胺，但氮上没有氢）可与格氏试剂反应，生成相应产物。
- **芳基锂**与酰胺反应：芳基碳负离子进攻后把相应部分脱掉，在原位引入醛基（生成**醛**）。
- **二烷基铜锂**可把酰氯变成相应的**酮**。
- **烯基铜锂**可把氯换掉、把烯基引到底物上。

这些金属有机试剂同样提供碳负离子，走亲核加成-消除，给底物引入不同烃基。
:::

## 五、克莱森缩合：羧酸酯版本的"缩合"

### 5.1 先回忆"缩合"是什么

:::definition{label=缩合}
**缩合**指两分子反应、通过脱除某些小分子，生成一个**更长链分子**的反应。羟醛缩合是两分子醛（酮）经碳负离子进攻、缩合成带 $\alpha,\beta$-不饱和结构的更大分子；克莱森缩合则把"提供 $\alpha$ 碳负离子"的角色从醛酮换成了**羧酸酯**。
:::

### 5.2 酯 α-氢的酸性与碳负离子

:::theorem{label=克莱森缩合的定义}
**克莱森缩合（Claisen 缩合）**：羧酸衍生物（主要指**羧酸酯**）中，与羰基直接相连的 **$\alpha$ 氢**也具有微弱酸性。在**强碱**作用下，$\alpha$ 氢被拔掉，生成一个**碳负离子**，这个碳负离子作为亲核试剂去进攻另一分子羰基，发生一个与羟醛缩合类似的过程。
:::

:::insight{label=碳负离子提供者被"拓展"了}
羟醛缩合时，只有醛或酮能提供 $\alpha$ 碳负离子。到了羧酸衍生物这里，**羧酸酯只要 $\alpha$ 位有氢**，强碱下也能提供碳负离子。而且这个碳负离子能进攻一切含羰基的分子——和醛能反应、和酮能反应、**和酯自身也能反应**。这是知识的迁移与拓展。
:::

### 5.3 与羟醛缩合的关键区别：消除残基 vs 脱水

:::derivation{label=克莱森缩合的机理}
1. **生成碳负离子**：强碱拔掉酯 $\alpha$ 位的氢，得到 $\alpha$ 碳负离子。
2. **亲核加成**：碳负离子进攻另一分子酯的羰基碳，一步进攻、把碳氧双键打开（与羟醛缩合的进攻完全一样）。
3. **消除残基**：对羧酸衍生物而言，紧接着氧负回来变双键，把被进攻那分子酯上原来连接的**残基（烷氧基）脱除掉**。

这里就是与羟醛缩合的**分水岭**：
- **羧酸衍生物（克莱森）**：加成后**消除残基**（赶走烷氧基），不脱水。
- **醛酮（羟醛缩合）**：加成后没有可脱的残基，生成的 **$\beta$-羟基会消除一分子水**。

但两者的进攻方式完全一致——都是一次性亲核加成、打开双键。
:::

### 5.4 例：两分子酯缩合成 β-酮酯

:::example{label=两分子乙酸乙酯的克莱森缩合}
两分子酯在强碱作用下缩合，最终生成 **乙酰乙酸乙酯**（一个 $\beta$-酮酯）。

怎么理解这个产物？把乙酰乙酸乙酯里"乙酰基"那部分挡住，剩下的本身就是**乙酸乙酯**——只不过乙酸乙酯的 **$\alpha$ 位被乙酰化了**。

机理串一遍：一分子乙酸乙酯被强碱拔掉 $\alpha$-H 成碳负离子 → 进攻另一分子乙酸乙酯的羰基碳（亲核加成）→ 氧负回来变双键、把那分子的**乙氧基（$\ce{-OC2H5}$）赶走**（消除残基）→ 得到 $\beta$-酮酯乙酰乙酸乙酯。
:::

:::pitfall{label=别把克莱森当成羟醛}
两者机理"进攻"那步一模一样，极易混。区别牢记：
1. **谁提供碳负离子**：羟醛是醛/酮的 $\alpha$ 位；克莱森是**酯**的 $\alpha$ 位（需酯 $\alpha$ 位有氢）。
2. **加成之后**：羟醛**脱水**（生成 $\alpha,\beta$-不饱和醛酮）；克莱森**消除烷氧基残基**（生成 $\beta$-酮酯）。
3. 两者都必须有**强碱**来拔 $\alpha$-H。
:::

## 六、本节小结

:::note{label=本节小结}
1. **带电进攻者更猛**：格氏试剂提供烃基碳负离子（卤代烷 + Mg 而来），比中性的水/醇/氨更易进攻羰基碳；这类反应能**延长碳链**（区别于普通醛酮的官能团转化）。
2. **酰氯 + 格氏**：**低温（−78 ℃）停在酮**，**常温直接到醇**；有**位阻**时不低温也可停在酮。是自由度很大的增碳法。
3. **酯 + 格氏**：很难停在酮，直接到**叔醇**（如三苯基取代叔醇）；**甲酸酯 + 格氏 → 仲醇**。
4. **酰胺特殊**：N—H 的氢酸性大，会分解格氏试剂，含氢酰胺难反应；**无氢酰胺**（如酰基咪唑）可反应。其他金属有机：芳基锂 + 酰胺引入醛基（生成醛），二烷基铜锂使酰氯→酮，烯基铜锂引入烯基。
5. **克莱森缩合**：强碱拔掉**酯 $\alpha$-H** 成碳负离子，进攻另一分子羰基；与羟醛缩合的关键区别——羧酸衍生物加成后**消除烷氧基残基**（不脱水），产物是 **$\beta$-酮酯**（如两分子乙酸乙酯 → 乙酰乙酸乙酯）。
6. 一条主线：**碳负离子（或孤对电子）→ 亲核加成 →（消除残基或脱水）**，把羟醛缩合、克莱森缩合、格氏反应统一在亲核加成的逻辑下。
:::
