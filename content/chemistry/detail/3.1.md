# 3.1 范德华力与氢键

## 从"超分子"说起：弱相互作用的舞台

上一章我们讨论的是分子内部的化学键——共价键如何把原子连成分子。本章把镜头拉远一格，去看**分子与分子之间**的相互作用。它们没有强到能成为共价键，所以统称为"弱相互作用"。正是这些弱相互作用，让一个个独立的分子聚集、组装成"超分子"——具有单个分子所不具备的、超越单一分子的崭新性质的聚集态。

:::definition{label=超分子与弱相互作用}
**弱相互作用**：分子之间存在的、强度不足以构成共价键的相互作用力。
**超分子**：分子之间通过弱相互作用形成的聚集态或组装体，它具有超越单一分子的新性质。
:::

本节先认识两位最基础的"主角"：**范德华作用力**和**氢键**。理解了它们的本质（都是静电作用），后面整章关于物性、组装、识别的内容都会顺理成章。

## 一、范德华作用力

### 1. 三种力共同构成范德华力

:::definition{label=范德华作用力}
**范德华作用力**由三种力共同构成：**色散力**、**诱导力**、**取向力**。这三种力一起作用于相互靠近的分子之间，统称范德华作用力。
:::

范德华力有一个重要的几何特征：它是**当分子接近到一定距离时才产生的**，随着分子间距离的增大会迅速减小甚至消失。换句话说，它是一种"近身才有效"的力。请把这一点牢牢记住——本章后面解释一切物性规律，前提都是"分子靠得足够近"。

### 2. 瞬时偶极：非极性分子也会"瞬间有极性"

在第一章我们就区分过极性分子（如 $\ce{HCl}$）和非极性分子（如 $\ce{H2}$）。这里要追问一句：非极性分子是不是**时时刻刻**都没有极性？

:::insight{label=瞬时极性的来由}
分子里的原子在不停地振动，电子云也在不停地运动。从**一段时间的平均值**看，非极性分子的电子云是均匀对称分布的，所以它确实是非极性分子。但在**某一瞬间**，电子云不一定恰好对称——当这对电子没有运动到两原子正中间时，分子就在那一瞬间产生了极性，这叫**瞬时极性**（瞬时偶极）。
:::

这是理解色散力的钥匙：没有任何分子能保证每一瞬间电子云都完美对称，所以瞬时偶极对任何分子都会发生。

### 3. 色散力——不挑分子极性，处处都有

:::definition{label=色散力}
由**瞬时偶极**产生的分子间弱相互作用，称为**色散力**。
:::

设想一个非极性分子，平均看电子云均匀；但某一瞬间它总会出现一处电子云密度高的区域和一处密度低的区域，于是这一瞬间它就有了一个正电端和一个负电端。当另一个分子在同一瞬间也出现了正、负电端，两个分子又离得很近时，它们之间就发生静电吸引——正电端去配对邻近分子的负电端，这种偶极配对就是色散力。

色散力存在于哪些分子之间？

:::insight{label=色散力无处不在}
- 非极性分子 ↔ 非极性分子：**有**
- 非极性分子 ↔ 极性分子：**有**
- 极性分子 ↔ 极性分子：**有**

**色散力不挑分子的极性，三种情况全都有。** 因为只要有那么一瞬间产生了瞬时偶极，色散力就发生了，而瞬时偶极对所有分子都会出现。
:::

### 4. 诱导力——必须有极性分子"带头"

:::definition{label=诱导力}
分子的**诱导偶极**与**固有偶极**之间产生的相互作用，称为**诱导力**。
:::

什么是诱导偶极？当一个非极性分子靠近一个极性分子时，它自身的电子云会被极性分子已有的偶极"诱导"而发生变化，从而产生出一个诱导偶极。这个诱导偶极一旦出现，两分子之间又有了偶极—偶极相互作用，这就是诱导力。

:::insight{label=诱导力的前提}
诱导力**必须有极性分子存在**（要有一个固有偶极去做"诱导源"）。所以诱导力存在于：
- 极性分子 ↔ 非极性分子
- 极性分子 ↔ 极性分子

两个非极性分子之间**没有**诱导力（没有固有偶极去诱导），它们之间只有色散力。范围比色散力小了一圈。
:::

### 5. 取向力——只在极性分子之间

:::definition{label=取向力}
两个极性分子的**固有偶极**之间产生的相互作用，称为**取向力**。
:::

两个极性分子离得较远时彼此并无偶极作用（再次强调：弱相互作用一定发生在很近的时候）；当它们靠近，固有偶极之间发生取向配对，就是取向力。

:::insight{label=取向力的范围}
**取向力只存在于极性分子之间。** 范围比诱导力又小了一圈。
:::

### 6. 范德华力的实质：静电作用

把三种力放在一起看，它们的范围像三个套叠的圈：

:::note{label=三种力的范围对比}
| 力 | 非极性↔非极性 | 极性↔非极性 | 极性↔极性 | 根源 |
|---|---|---|---|---|
| 色散力 | ✓ | ✓ | ✓ | 瞬时偶极 |
| 诱导力 | ✗ | ✓ | ✓ | 诱导偶极—固有偶极 |
| 取向力 | ✗ | ✗ | ✓ | 固有偶极—固有偶极 |
:::

:::theorem{label=范德华力的本质}
范德华作用力虽可拆成色散、诱导、取向三种力，但它的**实质是一种静电相互吸引**。因此实际讨论时我们一般不再细分到底是哪一种力——两个分子一旦靠近，总会存在静电作用（再不济也总有第一种色散力）。

所以：**两个分子只要相互靠近，总会产生范德华作用力。** 范德华作用理论上存在于所有有机分子之间。
:::

## 二、氢键

### 1. 氢键无处不在，却很"强大"

第二种弱相互作用是我们早已多次提到的**氢键**。它的身影遍布生命与自然：DNA 双螺旋中碱基的配对，主要靠氢键；水中一个水分子的氢可以与另一个水分子的氧形成氢键。

:::insight{label=水为什么是液态}
水的分子量只有 18，非常小、非常轻——按这个分子量本"应该"是气态。可它在常温下是液态，正是因为水分子之间存在**非常强大的氢键网络**把它们维系在一起。这是氢键威力的最好例证。
:::

### 2. 氢键的键能：弱在单键，强在成网

:::insight{label=键能的定量比较}
- 氢键键能：约 **20～30 千焦/摩尔**
- 一般共价键键能：约 **200～600 千焦/摩尔**

一个氢键至少只是典型共价键约 **1/10** 的键能。
:::

那氢键凭什么这么"管用"？关键在于它和共价键的作用方式不同：

:::insight{label=弱而广}
共价键只能作用于**两个原子之间**；而氢键（弱相互作用）**可以连成网**。所以氢键虽然单根很弱，但作用范围广——就像很多人手拉手，团结起来就是力量。
:::

### 3. 氢键的定义与本质

以水中的氢键为例理解它的来由：水分子里氧与氢的电负性差距很大，是极性分子，电子云偏向氧，于是氧端感应出负电荷、氢端感应出正电荷。另一个水分子靠近时，它的氢自然去找这个水分子上带负电的氧，氢键就形成了。

:::theorem{label=氢键的本质}
氢键的本质仍然是**静电作用**，与前面学的偶极作用同源。它的键能（20～30 千焦/摩尔）与理论计算的偶极、偶极—离子静电作用基本相当，反过来印证了它确实是一种静电相互作用。
:::

那为什么单独叫"氢键"？因为**一定要有氢参与**。

:::definition{label=氢键}
分子中与一个**电负性很大的元素**相结合的**氢原子**，还能与**另一个电负性很大的原子**之间产生结合力，所形成的键，称为**氢键**。

记作 $\ce{X-H\bond{...}Y}$：
- $\ce{X}$ —— 与氢直接相连、电负性很大的原子（决定了氢带正电，使其成为**氢给体**）
- $\ce{Y}$ —— 另一处电负性大的原子（**氢受体**）
- 弱相互作用一般用**虚线**表示
:::

:::insight{label=给体的"硬要求"，受体的"宽要求"}
- **氢给体**：那个氢原子**自身必须与一个电负性很大的原子相连**，这样它才带正电荷、才能当给体。
- **氢受体**：只要含有电负性很大的原子即可，**受体里不一定要有氢**。
- 氢键**不要求**是相同分子之间形成——一个水分子完全可以和一个醇分子里的氧形成氢键。
:::

### 4. 氢键的特征：方向性与饱和性

:::insight{label=键能随电负性与半径变化}
不同类型氢键的键能：随两个电负性大原子的**电负性增大而增大**，随**原子半径增大而减小**。电负性越大越容易成键；半径越大、越难靠近重叠，氢键就越弱。
:::

:::theorem{label=方向性}
氢键的几何构型一般为**直线型或稍有弯曲**，以使 $\ce{X}$ 与 $\ce{Y}$ 之间的斥力最小。只有当 $\ce{X-H\bond{...}Y}$ 三个原子大致在同一直线上时，两个大电负性原子之间的距离才最远、斥力才最小——这决定了氢键具有**方向性**。

正因为有方向性，水结冰时排出的冰晶才是六边形（或五边形）的规则结构。
:::

:::theorem{label=饱和性}
一个氢原子与某个氧原子形成氢键后，就**无法再与另一个氧原子**形成氢键了——氢键是**一对一配对**的，这就是氢键的**饱和性**。
:::

### 5. 实例：DNA 碱基配对

回到 DNA。在腺嘌呤（A）与胸腺嘧啶（T）配对中，T 上氮—氢键里的氢是**氢给体**（这个氢直接连在电负性大的氮上），它去找 A 上作为**氢受体**的氮。另一对配对中，氢给体是氨基，氢受体是**羰基氧**——羰基氧电负性大，是常见的氢受体。这些都严格符合上面的定义。

## 三、分子内氢键 vs 分子间氢键

氢键能成在一个分子**内部**，也能成在**两个分子之间**，二者对物性的影响截然不同。最经典的对照是邻、对位的取代苯。

:::example{label=邻羟基苯甲醛 vs 对羟基苯甲醛}
两个分子都是苯环上同时有羟基和醛基，只是相对位置不同，沸点却相差将近 **50 ℃**：邻位的沸点**低**，对位的沸点**高**。怎么解释？

先弄清沸点的物理含义：在饱和大气压下把液态变成气态所需的温度。液态是连续相、气态是单分子相，加热提供能量打断的是**分子间的弱相互作用**。

**邻羟基苯甲醛（水杨醛，沸点低）**：羟基的氢（氢给体）与同一分子里醛基的羰基氧（氢受体）距离很近，容易形成**分子内氢键**。为什么"愿意"成分子内氢键？一是离得近；二是有原理支撑——有机成环结构里六元环、五元环最稳定，这个分子内氢键闭合后恰好形成一个**稳定的六元环**。结果：分子内部"自给自足"，分子之间就只剩范德华作用、没有氢键，所以沸点低。

**对羟基苯甲醛（沸点高）**：羟基和醛基隔得远，**够不着**，无法形成分子内氢键，只能转而形成**分子间氢键**。分子间氢键把一个个分子连接起来，分子间的弱相互作用大大增强，所以沸点高。
:::

:::insight{label=口诀}
**分子内氢键 → 分子间作用力变弱 → 沸点（及熔点等）偏低；**
**分子间氢键 → 分子间作用力变强 → 沸点偏高。**
邻位常因距离近、易成五/六元环而走"分子内"，对位因距离远只能走"分子间"。
:::

:::pitfall{label=常见误区}
1. **以为色散力只存在于非极性分子之间。** 恰恰相反，色散力对所有分子（含极性分子之间）都存在，它"不挑极性"。挑极性的是诱导力（要有极性分子）和取向力（只在极性分子之间）。
2. **以为只要有电负性大的原子（如醚里的氧）就能当氢给体。** 错。氢给体的关键是那个**氢**要直接连在电负性大的原子上。连在碳上的氢（$\ce{C-H}$ 视作非极性键）不能当氢给体。
3. **把"分子内氢键使沸点升高"记反。** 分子内氢键是分子"自己抱自己"，反而削弱了分子间作用，使沸点**降低**。
4. **忘记"靠近"这个前提。** 范德华力和氢键都只在分子足够靠近时才有效，距离一大就迅速消失。
:::

:::note{label=本节小结}
1. **范德华作用力**由色散力、诱导力、取向力构成：色散力源于瞬时偶极，对所有分子都有；诱导力需极性分子存在（极性↔非极性、极性↔极性）；取向力只在极性分子之间。三者实质都是**静电吸引**，故两分子一旦靠近总有范德华力，它存在于一切有机分子之间。
2. **氢键** $\ce{X-H\bond{...}Y}$ 的本质也是**静电作用**，但要求有氢参与：氢给体的氢须连在电负性大的原子上，氢受体含电负性大的原子即可（不必有氢）。
3. 氢键单根弱（20～30 kJ/mol，约共价键 1/10），但**弱而广，可连成网**；具有**方向性**（近直线型）和**饱和性**（一对一）；键能随电负性增大而增大、随半径增大而减小。
4. **分子内氢键**削弱分子间作用、降低沸点（邻位、易成六元环）；**分子间氢键**增强分子间作用、升高沸点（对位）。
5. 范德华力与氢键二者**共同的本质都是静电相互作用**——这是贯穿全章的主线。
:::
