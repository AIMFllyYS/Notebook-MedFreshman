# 14.3 杂环化合物与含氮天然产物

讲完链状和苯环上的含氮化合物，有机化学最主要的大类就基本介绍完了。接下来这部分都与天然产物相关——杂环、糖、氨基酸与蛋白质，自然界里最多的有机分子恰恰是杂环化合物。本节抓三件事：吡啶与五元杂环（吡咯/呋喃/噻吩）的结构、碱性与亲电取代规律；糖类的变旋现象与还原性；氨基酸的两性、肽键与蛋白质的结构与化学键。

## 一、直觉：什么是芳香杂环

:::definition{label=芳香性与杂原子}
**芳香性**不是指味道，而是一类化学性质：高度不饱和、**易取代、难加成**。判断用**休克尔规则**，三条同时满足：共平面、能形成环状共轭体系、$\pi$ 电子数满足 $4n+2$。

**杂环**指成环骨架中含有除碳以外的原子。除碳之外的氮、氧、硫、磷等都叫**杂原子**。本节所学杂环都具芳香性、都符合休克尔规则。
:::

杂环广泛存在于自然界与人体中：血红素（铁卟啉）、叶绿素（镁卟啉）、维生素 $\ce{B12}$（钴）等看似复杂，构成卟啉环的单元其实都是含杂原子的共轭杂环。它们能螯合金属，正是因为杂环氮上的孤对电子可与金属空轨道配位，形成路易斯酸碱配合物。

## 二、杂环氮的碱性：吡啶氮与吡咯氮

判断杂环上某个氮有没有碱性，关键看它的孤对电子**有没有参与到成环的共轭体系里**。

:::theorem{label=参与环共轭则无碱性，不参与则显碱性}
- **吡咯**：满足休克尔规则——五个原子共平面，两个碳碳双键各提供 2 个 $\pi$ 电子，**氮上的孤对电子也参与进环平面共轭**，共 6 个 $\pi$ 电子（$4n+2$），有芳香性。但氮的孤对电子既已离域进大 $\pi$ 体系，就"只能顾一头"，无法再去结合质子。所以**吡咯环上的氮没有碱性**。

- **吡啶**：相当于苯环的一个碳被氮取代。碳碳、碳氮共轭凑成 6 个 $\pi$ 电子循环排布、有芳香性；但**氮的孤对电子没有参与成环共轭，仍露在环外**，所以**吡啶有碱性**（弱碱）。正因如此，吡啶常作金属配合物的配体——孤对电子与金属空轨道成键。
:::

:::definition{label=吡咯氮与吡啶氮}
我们用这两个名字给杂环氮分类（而不是回答"$\mathrm{sp^2}$ 氮 / $\mathrm{sp^3}$ 氮"）：

- **吡咯氮**：孤对电子参与环共轭、**不显碱性**的氮（通常是带氢的那个）；
- **吡啶氮**：孤对电子露在环外、**显碱性**的氮。
:::

:::example{label=咪唑中的两种氮}
咪唑里两个氮**不是同一类型**：带氢、孤对参与共轭的那个是**吡咯氮**（不显碱性）；另一个露在外面的是**吡啶氮**（显碱性）。回答这类题，只能答"吡啶氮 / 吡咯氮"。
:::

:::example{label=把杂环放进碱性大排序}
甲胺、吡咯、吡啶、氨、苯胺由强到弱：

$$\text{甲胺} > \text{氨} > \text{吡啶} > \text{苯胺} > \text{吡咯}$$

有机胺（甲胺）最强；无机氨次之；吡啶氮孤对直接露出、显碱性排第三；苯胺氮与苯环部分 $\pi\text{-}\pi$ 共轭（不完全），更弱；吡咯氮完全参与环共轭，碱性最弱。

吡啶环上若带取代基比碱性，就是比取代基的电子效应：氨基、甲基给电子使碱性增强，氰基吸电子使碱性减弱。
:::

## 三、吡啶的反应特性

:::insight{label=吡啶氮吸电子，亲电取代困难}
吡啶氮的吸电子效应使环平面电子密度**降低**，所以亲电取代活性比苯环**更低**，需要更剧烈的条件（如更高温度）。这与五元杂环恰好相反（见下）。
:::

:::theorem{label=吡啶的取代位置}
吡啶有 $\alpha$、$\beta$、$\gamma$ 三个位置：

- **亲电取代**（如硝化）发生在 **$\beta$ 位**；
- **亲核取代**发生在 **$\alpha$ 位**；
- **$\gamma$ 位**取代需先把吡啶氮变成 **$N$-氧化物**才能发生。

吡啶本身显碱性，能与酸反应生成**吡啶盐**——这是它碱性的直接体现。
:::

:::example{label=喹啉的氧化}
喹啉（吡啶并苯环）用高锰酸钾氧化时，被氧化掉的是**苯环**部分，**保留吡啶环**。这与"苯并杂环上反应究竟落在杂环还是苯环"的选择问题相呼应。
:::

## 四、五元芳香杂环：吡咯、呋喃、噻吩

:::definition{label=结构与芳香性}
吡咯（含 $\ce{N-H}$）、呋喃（含 O）、噻吩（含 S）是同一系列，区别仅在杂原子。三者所有原子（含杂原子）都 $\mathrm{sp^2}$ 杂化、五原子共平面，杂原子孤对电子参与封闭环状共轭，形成 6 个 $\pi$ 电子体系（$4n+2$），都有芳香性。呋喃被氢化即四氢呋喃（一种环醚）。
:::

:::insight{label=五元杂环为何比苯更易亲电取代}
以苯为基准：苯是 6 个碳占有 6 个 $\pi$ 电子。五元杂环只有 5 个成环原子，$\pi$ 电子数却仍是 6——环变小、电子数不变，环上 $\pi$ 电子云密度比苯**更高**，所以亲电取代比苯**更容易**。这与吸电子的吡啶正好相反。

亲电取代活性顺序：$\text{吡咯} > \text{呋喃} > \text{噻吩} > \text{苯环}$。
:::

:::theorem{label=取代位置在 α 位}
位置标定：杂原子为 1 位（不取代），相邻为 2 位（$\alpha$ 位），依次 2、3、4、5。和杂原子相邻的两个是 $\alpha$ 位，外侧两个是 $\beta$ 位。

吡咯、呋喃、噻吩的亲电取代主产物在 **$\alpha$ 位**。例如呋喃的溴代发生在 $\alpha$ 位；重氮盐与吡咯偶联也是亲电取代，且发生在吡咯环上（吡咯电子密度更高、比并连的苯环更易反应），位置在 $\beta$ 位 [按录音]。
:::

:::example{label=吡咯的弱酸性与吲哚}
吡咯氮上带一个氢，这是它与呋喃、噻吩的唯一区别。因为有这个氢，吡咯在强碱下显**一点酸性**，氢被拿走生成**吡咯盐**。

碱性方面：仲胺（环胺）> 吡咯 > 吲哚。吲哚是苯并吡咯，与苯环并连后共轭程度更大、电子离域更远，碱性比吡咯更小（吡咯 $\mathrm{p}K_b$ 约 13.6，吲哚约 17.6，数值以板书为准）。
:::

:::example{label=呋喃甲醛当"无 α 氢的芳香醛"看}
呋喃甲醛可整体看作一个没有 $\alpha$ 氢的芳香醛，类似苯甲醛。所以：与氢氧化钠反应（无 $\alpha$ 氢无法自身羟醛缩合）发生**康尼扎罗反应**（歧化：一分子氧化成酸、一分子还原成醇）；若与有 $\alpha$ 氢的乙醛混合，则呋喃甲醛作羰基提供者、乙醛作碳负离子提供者，发生**羟醛缩合**。
:::

## 五、糖类：变旋与还原性

:::definition{label=苷羟基与变旋现象}
糖是多羟基醛或多羟基酮。醛糖成环时形成**半缩醛**结构，成环碳上新生成的羟基叫**苷羟基**。

**变旋现象**：单糖都有变旋；多糖都没有。双糖看缩合时两分子糖**是否都用掉了苷羟基**——若都用掉则**无变旋、为非还原糖**；若**保留了苷羟基**，则**有变旋、为还原糖**。
:::

:::example{label=判断变旋与还原性}
- **蔗糖**：两分子单糖的苷羟基都用于缩合，**无变旋、非还原糖**。
- **纤维二糖**：有一个单元的苷羟基**保留**，**有变旋、还原糖**。
- **甲基葡萄糖苷**（糖苷）：苷羟基已被醚化成醚，**无变旋、非还原糖**。一看到"糖苷"就知是非还原糖。
- **脱氧核糖**：苷羟基仍保留，**有变旋**。
:::

:::insight{label=糖脎与差向异构体}
糖在前两个碳上与**三分子苯肼**（需过量）反应生成**糖脎**：一分子与醛反应、一分子用于氧化、第三分子用于氧化后的反应。由于反应只动前两个碳，所以**前两个碳以下的手性结构相同**的糖会生成**相同的糖脎**。

对同一个糖脎，一般对应**三种不同的糖**：原糖、它在某一手性碳上的**差向异构体**，以及把成脎位置另一种异构氧化成醛后的糖。差向异构体不是对映异构体——它是主体构型保持、只有一个碳互成对映关系的两个**完全不同**的化合物。
:::

## 六、含氮天然产物：氨基酸、肽与蛋白质

:::definition{label=氨基酸的两性}
氨基酸既含氨基又含羧基，是**两性**化合物，在分子内常以**两性离子**（内盐，$\ce{-NH3+}$ 与 $\ce{-COO-}$ 并存）形式存在。氨基能发生的反应它都能发生，羧基能发生的反应它也能发生。
:::

:::example{label=氨基酸的典型反应}
- **酯化**：羧基与醇成酯。
- **酯的水解**：氨基酸酯在酸性条件下水解，生成氨基酸和相应的醇。
- **酸酐的氨解**：氨基酸（酯）的氨基与酸酐反应，生成酰胺——这也是多肽合成中**氨基保护**的一种方法。
- **与过量亚硝酸**：$\alpha$-氨基酸在过量亚硝酸作用下，氨基变成羟基，生成 **$\alpha$-羟基酸**。
:::

:::definition{label=肽键与蛋白质一级结构}
**肽键**实质就是**酰胺键**，是蛋白质唯一的**主键**。**蛋白质一级结构**指氨基酸在多肽链中的**排列顺序**。

测定多肽结构要测三样：氨基酸的**种类、数目、排列顺序**，测全这三项结构即确定。测定方法主要两种：**末端残基分析法**与**部分水解**。
:::

:::insight{label=蛋白质的主键与副键——不都是非共价键}
- **主键**只有一个：**肽键（酰胺键）**，共价键，强度高、难打断。
- **副键**有多种：氢键、离子键、疏水键、二硫键、酯键等。

关键提醒：**副键不见得都是非共价键**。共价键靠共用电子对、强度高（如身体与手臂，难断）；非共价键是弱相互作用，如**氢键**（如同手拉手，力量不小却易被打断）。离子键、疏水键（范德华作用，疏水分子在水中抱团）也属弱相互作用。但**二硫键（硫硫键）是共价键**——所以蛋白质副键里既有非共价键，也有共价键。
:::

:::pitfall{label=必需氨基酸与几个易错点}
- 要记住 **8 种必需氨基酸**的结构。
- 变旋：单糖都有、多糖都没有、双糖看苷羟基；"糖苷"一定是非还原糖。
- 一个糖脎一般对应三种糖（含差向异构体）；差向异构体 ≠ 对映异构体。
- 杂环氮：孤对**参与**环共轭则**不显碱性**（吡咯氮），**不参与**则**显碱性**（吡啶氮）——别说反。
:::

:::note{label=本节小结}
1. **吡啶**：氮为吡啶型（孤对露在环外、不参与环共轭），显碱性、可与酸成吡啶盐；氮吸电子使环电子密度降低，亲电取代困难、条件剧烈，亲电取代在 $\beta$ 位、亲核取代在 $\alpha$ 位、$\gamma$ 位需经 $N$-氧化物。
2. **五元杂环（吡咯、呋喃、噻吩）**：杂原子孤对参与环共轭，6 个 $\pi$ 电子有芳香性；环小电子密度高，亲电取代比苯易（吡咯 > 呋喃 > 噻吩 > 苯），主产物在 $\alpha$ 位；吡咯氮不显碱性但有弱酸性（生成吡咯盐）。
3. **糖类**：单糖都变旋、多糖都不变旋、双糖看苷羟基是否保留（保留则有变旋、为还原糖）；糖脎由前两碳与三分子苯肼生成，一个糖脎对应三种糖（含差向异构体）。
4. **氨基酸/蛋白质**：氨基酸两性、以两性离子存在，可酯化、水解、氨解、与过量亚硝酸生成 $\alpha$-羟基酸；肽键即酰胺键，是蛋白质唯一主键；一级结构是排列顺序；副键多样，且**不全是非共价键**（二硫键是共价键）。
:::
