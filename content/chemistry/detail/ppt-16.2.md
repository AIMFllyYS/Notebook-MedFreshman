# ppt-16.2 六元杂环化合物

:::insight{label=从"为什么尼古丁让人上瘾"说起}
尼古丁（nicotine）是烟草中的成瘾物质，其结构包含一个吡啶环和一个吡咯烷环。吡啶环是六元含氮杂环的代表——与五元杂环不同，吡啶氮的孤对电子不参与共轭，使吡啶既是弱碱（$\mathrm{p}K_b = 8.8$），又是吸电子基团，使环上电子云密度降低。这导致吡啶的化学性质与五元杂环"相反"：亲电取代困难，亲核取代容易。

本节我们学习吡啶及其稠环（喹啉、异喹啉）的结构和反应，对比五元杂环理解"富电子 vs 缺电子"杂环的差异。
:::

## 吡啶的结构与性质

:::definition{label=吡啶的结构}
**吡啶**（pyridine）：$\ce{C5H5N}$，氮杂苯，是有机碱。

**结构**：
- 5 个碳和 1 个氮均为 $\mathrm{sp^2}$ 杂化，所有原子在同一平面上
- 每个原子的 $\mathrm{p}$ 轨道有 1 个电子，形成 6 个 π 电子的环状共轭体系
- 符合 4n+2 规则，具有**芳香性**
- **氮的孤对电子在 $\mathrm{sp^2}$ 杂化轨道中**，不参与共轭 → 显碱性

**编号**：氮为 1 位，依次 2、3、4、5、6；2、6 位又称 **α 位**，3、5 位又称 **β 位**，4 位又称 **γ 位**。

**缺电子性**：氮的电负性大于碳，氮"吸"电子，使环上电子云密度比苯低 → **缺电子体系** → 亲电取代反应困难，亲核取代反应容易。
:::

:::theorem{label=吡啶的碱性}
吡啶氮上有孤对电子（未参与共轭），能接受质子，显**弱碱性**：

$$
\ce{Py + H+ -> PyH+}
$$

$\mathrm{p}K_b = 8.8$，碱性比脂肪胺弱（因氮为 $\mathrm{sp^2}$ 杂化，电负性大于 $\mathrm{sp^3}$ 氮），但比苯胺强（无共轭分散）。

**应用**：吡啶常作为有机反应的**碱性溶剂**和**酸吸收剂**（如酰化反应中吸收 $\ce{HCl}$）。
:::

## 吡啶的化学反应

:::theorem{label=吡啶的亲电取代反应}
吡啶环上的亲电取代反应（硝化、卤化、磺化）一般发生在 **β 位**（3 位）。

**原因**：
- 氮的吸电子效应使吡啶环电子云密度降低
- 亲电试剂进攻 α 位或 γ 位时，中间体正离子有一个共振结构正电荷在氮上（极不稳定）
- 进攻 β 位时，正电荷不落在氮上，相对稳定

**反应条件**：吡啶环不易发生亲电取代反应，需**剧烈条件**：
- 硝化：浓 $\ce{HNO3}$ + 浓 $\ce{H2SO4}$，$300°C$
- 磺化：浓 $\ce{H2SO4}$，$100°C$
- 卤化：较难

**亲电取代活性**：**吡咯、呋喃、噻吩 > 苯 > 吡啶**

**注意**：吡啶在强酸中会质子化为吡啶鎓离子 $\ce{PyH+}$，使环更加缺电子，亲电取代更难。
:::

:::theorem{label=吡啶的亲核取代反应}
吡啶环的特点是能发生**亲核取代反应**，一般发生在 **α 位**（2 位）。

**原因**：
- 氮吸电子，使 α 位和 γ 位碳带较多正电荷
- 亲核试剂进攻 α 位时，中间体负离子有共振结构负电荷在氮上（电负性大的氮稳定负电荷）

**典型反应**：吡啶与 $\ce{NaNH2}$ 反应（Chichibabin 反应），生成 2-氨基吡啶：

$$
\ce{Py + NaNH2 ->[回流] 2\text{-}氨基吡啶 + NaH}
$$

**总结**：
- 亲电取代：β 位（缺电子环，难）
- 亲核取代：α 位（缺电子环，易）
:::

:::example{label=吡啶的亲电取代与亲核取代对比}
**题目**：吡啶硝化和氨基化分别发生在什么位置？为什么？

**解**：

**硝化**（亲电取代）：发生在 **β 位**（3 位）
- 吡啶是缺电子环，亲电取代困难
- 进攻 β 位时，中间体正离子的共振结构中正电荷不落在氮上，相对稳定
- 需剧烈条件：浓 $\ce{HNO3}$ + 浓 $\ce{H2SO4}$，$300°C$

**氨基化**（亲核取代，Chichibabin 反应）：发生在 **α 位**（2 位）
- 亲核试剂 $\ce{NH2^-}$ 进攻 α 位
- 中间体负离子的共振结构中负电荷可落在氮上（电负性大，稳定）
- 条件：$\ce{NaNH2}$，回流

**对比**：
- 亲电取代 → β 位（避免正电荷在 N 上）
- 亲核取代 → α 位（负电荷可被 N 稳定）

**记忆**：吡啶"亲电找 β，亲核找 α"。
:::

:::definition{label=吡啶的氧化与还原}
**氧化反应**：吡啶环较稳定，一般氧化剂难氧化。但用过氧酸（如 $\ce{mCPBA}$、$\ce{H2O2}$）可氧化为 **N-氧化吡啶** $\ce{Py^+-O^-}$：

$$
\ce{Py + [O] -> N\text{-}氧化吡啶}
$$

**N-氧化吡啶的特点**：
- 氧带负电，氮带正电
- 进行亲电取代反应比吡啶**容易**，主要发生在 **γ 位**（4 位）
- 因为 N-O 的共轭效应使 γ 位电子云密度增大

**还原反应**：吡啶催化加氢得**哌啶**（六氢吡啶）：

$$
\ce{Py + 3H2 ->[Pt] 哌啶}
$$

哌啶是仲胺（$\mathrm{sp^3}$ 氮），碱性比吡啶强。
:::

## 喹啉与异喹啉

:::definition{label=喹啉与异喹啉}
**喹啉**（quinoline）：苯并吡啶，氮在 1 位。
**异喹啉**（isoquinoline）：苯并吡啶，氮在 2 位。

**结构**：由苯环和吡啶环稠合而成，具有芳香性。

**编号**：
- 喹啉：N 为 1 位，沿吡啶环编号 2、3、4，再沿苯环 5、6、7、8
- 异喹啉：N 为 2 位，吡啶环 1、3、4，苯环 5、6、7、8
:::

:::theorem{label=喹啉和异喹啉的亲电取代}
喹啉和异喹啉的亲电取代反应主要发生在**苯环上**（因吡啶环缺电子，苯环相对富电子）。

**喹啉**：主要在 **5 位和 8 位**取代
**异喹啉**：主要在 **5 位**取代

**反应条件**：相对温和（比吡啶容易，因苯环部分活性较高）

**例**：
- 喹啉硝化：5-硝基喹啉（50%）+ 8-硝基喹啉（48%）
- 异喹啉硝化：5-硝基异喹啉（72%）+ 8-硝基异喹啉（8%）
:::

:::theorem{label=喹啉和异喹啉的亲核取代}
喹啉和异喹啉的亲核取代主要发生在**杂环（吡啶环）上**。

**喹啉**：主要在 **2 位**取代
**异喹啉**：主要在 **1 位**取代

**典型反应**：
- 与 $\ce{NaNH2}$ 反应（氨基化）
- 与烷基锂反应（烷基化）
- 2-位或 4-位有卤原子时，该卤原子可被亲核试剂取代（如 $\ce{NaOH}$、$\ce{NaOR}$）

**条件**：硝基苯溶剂，加热
:::

:::example{label=喹啉的亲电取代与亲核取代}
**题目**：喹啉硝化和氨基化分别发生在什么位置？

**解**：

**硝化**（亲电取代）：发生在**苯环的 5 位和 8 位**
- 苯环比吡啶环富电子，亲电取代优先在苯环
- 5 位和 8 位是苯环上电子云密度较高的位置

**氨基化**（亲核取代）：发生在**吡啶环的 2 位**
- 吡啶环缺电子，亲核取代优先在吡啶环
- 2 位（α 位）的中间体负离子可被 N 稳定

**对比**：
- 亲电取代 → 苯环（5、8 位）
- 亲核取代 → 吡啶环（2 位）

**规律**：稠杂环中，亲电取代找"富电子环"（苯环），亲核取代找"缺电子环"（吡啶环）。
:::

:::definition{label=喹啉和异喹啉的氧化与还原}
**氧化**：喹啉和异喹啉不被大多数氧化剂氧化，但**高锰酸钾**和**过氧化物**能氧化苯环部分，生成**吡啶二甲酸**（喹啉氧化得喹啉酸）。

**还原**：
- 催化加氢：1,2-二氢喹啉 → 四氢喹啉 → 十氢喹啉
- 吡啶环先被还原（缺电子环更易加氢）
- 还原产物十氢喹啉是饱和双环化合物
:::

:::pitfall{label=混淆吡啶亲电取代与亲核取代的位置}
**常见错误**：以为吡啶亲电取代和亲核取代都发生在同一位置。

**正确做法**：
- **亲电取代**：发生在 **β 位**（3 位）——避免中间体正电荷落在氮上
- **亲核取代**：发生在 **α 位**（2 位）——中间体负电荷可被氮稳定

**记忆**：吡啶"亲电找 β（远避氮），亲核找 α（亲近氮）"。
:::

:::pitfall{label=混淆喹啉亲电取代与亲核取代的环}
**常见错误**：以为喹啉亲电取代和亲核取代都在同一个环上。

**正确做法**：
- **亲电取代**：在**苯环**上（5、8 位）——苯环富电子
- **亲核取代**：在**吡啶环**上（2 位）——吡啶环缺电子

**规律**：亲电找富电子环，亲核找缺电子环。
:::

:::note{label=本节小结}
- **吡啶**：氮杂苯，$\mathrm{sp^2}$，6 π 电子，芳香性；氮孤对电子未参与共轭 → 显碱性（$\mathrm{p}K_b = 8.8$）
- **缺电子环**：氮吸电子，环电子云密度 < 苯 → 亲电取代难，亲核取代易
- **亲电取代**：β 位（3 位），需剧烈条件（浓酸、高温）
- **亲核取代**：α 位（2 位），如 Chichibabin 反应（$\ce{NaNH2}$ → 2-氨基吡啶）
- **N-氧化吡啶**：过氧酸氧化得 $\ce{Py^+-O^-}$；亲电取代比吡啶容易，发生在 γ 位（4 位）
- **还原**：催化加氢得哌啶（六氢吡啶，仲胺，碱性更强）
- **喹啉**（苯并吡啶，N 在 1 位）：
  - 亲电取代：苯环 5、8 位
  - 亲核取代：吡啶环 2 位
  - 氧化：高锰酸钾氧化苯环 → 吡啶二甲酸
  - 还原：吡啶环先还原 → 十氢喹啉
- **异喹啉**（苯并吡啶，N 在 2 位）：
  - 亲电取代：苯环 5 位
  - 亲核取代：吡啶环 1 位
- **规律**：亲电取代找富电子环（苯环），亲核取代找缺电子环（吡啶环）

:::
