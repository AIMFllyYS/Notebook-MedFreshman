# 12.2 金属有机试剂与 Wittig 反应

## 有机合成真正想做的事：把碳和碳接起来

做有机合成，绝大多数时候并不是要把一个卤原子换成羟基——那种"官能团转化"的小分子大多能直接买到，没必要自己做。我们真正想做的，是**搭出新的骨架**，也就是**构筑新的碳碳键**，让分子长大、长出我们设计的形状。

上一节里，亲核试剂们（$\ce{CN-}$、醇、胺）进攻羰基碳时，加上去的要么是含氮、含氧的基团，要么只增一个碳。这一节出场的两类试剂——**金属有机试剂**和**叶立德**——则不同：它们送上羰基碳的是一个**碳负离子**，加成的瞬间就生成一根崭新的碳碳键，分子的碳骨架被实打实地扩长。这正是它们在合成中地位崇高的原因。我们先看格氏试剂等金属有机试剂如何"造醇增碳"，再看 Wittig 反应如何"造烯增碳"，最后用逆合成分析把这两件武器装进思维工具箱。

## 金属有机试剂：随身携带的碳负离子

:::definition{label=金属有机试剂与碳负离子}
金属有机试剂是碳直接与金属相连的化合物，常见的有格氏试剂（$\ce{RMgX}$）、有机锂试剂，以及炔基金属试剂（炔基钠、炔基锂）。

它们共同的本质特征是：碳-金属键高度极化，碳一端相当于带一个**碳负离子** $\ce{R^-}$。碳负离子是极强的亲核试剂，一旦遇到带正电的羰基碳，便迅速进攻、生成一根新的碳碳键。
:::

:::theorem{label=金属有机试剂加醛酮——构筑碳碳键、扩碳成醇}
碳负离子进攻羰基碳，碳氧 π 键打开生成氧负离子，再经酸化（质子化）得到一个**碳数增多的醇**。这是有机合成中构筑碳碳键、给分子扩链的主力反应之一。

$$\ce{R-[M] + R'2C=O -> R'2C(R)-O^-[M]^+ ->[H3O+] R'2C(R)-OH}$$
:::

## 加在不同底物上，得到不同级别的醇

碳负离子加到什么底物上，决定了生成几级醇。这一点要和上一章"醇的制备"对照着记——那时我们正是用这个反应由醛酮造出了醇。

:::derivation{label=加醛→仲醇，加酮→叔醇的电子流向}
以苯甲醛和乙基格氏试剂为例（一般在无水乙醚中进行，再酸化）：

1. 格氏试剂相当于一个乙基碳负离子 $\ce{CH3CH2^-}$，这一对电子进攻苯甲醛的羰基碳，形成新的 C–C 键；
2. 碳氧双键的 π 电子被挤到氧上，氧变成氧负离子；
3. 酸化时氧负离子结合质子，生成羟基。

$$\ce{C6H5CHO ->[CH3CH2MgBr][无水乙醚] ->[H3O+] C6H5CH(OH)CH2CH3}$$

**底物决定醇的级别：**
- 加**甲醛**（羰基碳上两个氢），生成**伯醇**；
- 加其他**醛**（羰基碳上一个氢、一个烃基），生成**仲醇**；
- 加**酮**（羰基碳上两个烃基），生成**叔醇**。
:::

:::insight{label=与环氧乙烷反应——一次增两个碳}
格氏试剂不只对羰基有效。它还能进攻**环氧乙烷**：碳负离子进攻张力很大的三元环碳，使环开裂，得到一个比格氏试剂多两个碳的**伯醇**。

$$\ce{R-MgX + \underset{环氧乙烷}{(CH2-CH2)O} -> R-CH2CH2-O-MgX ->[H3O+] R-CH2CH2-OH}$$

要点：加醛酮是增碳生成醇且醇的级别随底物变化；而加环氧乙烷则是稳定地**一次延长两个碳**、生成伯醇。两条路都通向"碳链增长 + 生成醇"。
:::

## 炔基钠/炔基锂：把碳碳三键带进来

端基炔（末端为 $\ce{C#CH}$ 的炔烃）的炔氢有一定酸性，在活泼金属作用下可生成**炔基钠、炔基锂**这类带碳负离子的试剂。

:::example{label=炔基负离子加醛酮}
炔基负离子同样去进攻醛酮的羰基碳，结果也是生成醇——而且把一个**碳碳三键**引入到了产物中：

$$\ce{R-C#C^- + R'2C=O ->[][H3O+] R'2C(OH)-C#C-R}$$

这就在生成醇的同时，给分子装上了可继续衍生的三键，是构筑复杂分子的有力一招。
:::

:::note{label=金属有机试剂这一类的合成"信号"}
在合成路线里，只要你看到一个反应同时具备这两个特征——**碳链增长**，并且**生成一个醇**——就应当立刻想到：这是金属有机试剂（格氏试剂、有机锂、炔基钠/锂）与醛酮（或环氧乙烷）的亲核加成。

反过来用更重要：拿到一个目标醇，看羟基所在的碳，把它周围的某根 C–C 键"切开"，就能拆出"金属有机试剂 + 醛酮"这一对原料——这正是逆合成分析的雏形，下文详谈。
:::

## Wittig 反应：把羰基氧"换成"碳碳双键

醛酮还能与**叶立德试剂**发生亲核加成，最终把羰基变成碳碳双键，得到烯烃。这就是大名鼎鼎的 **Wittig（维蒂希）反应**。

### 叶立德试剂是怎么制备的

:::definition{label=叶立德（磷叶立德）}
**第一步**，三苯基膦 $\ce{(C6H5)3P}$ 与卤代烷发生亲核取代，得到一个**季鏻盐**：

$$\ce{(C6H5)3P + R-CH2-X -> (C6H5)3P^+-CH2R \cdot X^-}$$

**第二步**，用强碱（如烷基锂试剂）拔去 α 碳（与磷相连那个碳）上的氢，得到一种内盐——**磷叶立德**：

$$\ce{(C6H5)3P^+-CH2R \cdot X^- ->[RLi] (C6H5)3P^+-CHR^- <=> (C6H5)3P=CHR}$$

磷碳之间既可写成离子式（负电荷在碳、正电荷在磷），也可写成磷碳双键。这就是叶立德的通式：它的一部分由三苯基膦提供，另一部分由卤代烷提供。
:::

:::pitfall{label=制备叶立德对卤代烷的限制}
- 卤代烷一般用**伯卤代烷或仲卤代烷**；
- **三级卤代烷、卤代烯烃、卤代芳烃制备不出叶立德试剂**；
- 卤代烷分子中可以含碳碳双键、碳碳三键，这些不影响叶立德的制备。
:::

### Wittig 反应的机理与结论

叶立德里又出现了一个**碳负离子**（碳上带负电的那部分），它就是进攻羰基的亲核试剂。

:::derivation{label=Wittig 反应机理：四元环中间体}
1. 叶立德的碳负离子进攻醛酮的羰基碳，形成新的碳碳键；
2. 体系经过一个**过渡的四元环态**（磷、碳、碳、氧组成的四元环，常称氧磷杂环丁烷）；
3. 这个四元环不稳定，很快断裂——分解成一分子**烯烃**和一分子**三苯基氧膦**（三价磷的氧化物 $\ce{(C6H5)3P=O}$）。

$$\ce{(C6H5)3P=CHR + R'2C=O -> R'2C=CHR + (C6H5)3P=O}$$

反应的"交换点"很清楚：原来醛酮的羰基氧，跑到叶立德那一侧（与磷结合成 $\ce{P=O}$）去了；而叶立德贡献的那部分烃基，被插到了原来羰基氧所在的位置，形成碳碳双键。
:::

:::theorem{label=Wittig 反应的结论}
醛酮与叶立德反应，相当于把叶立德中**由卤代烷提供的那部分烃基**，直接插到醛酮羰基氧原来的位置上，形成一个碳碳双键。

因此整个反应：
- **从结果看，是一个制备烯烃的反应**；
- **原料来源有两个**：一个是醛酮，另一个是卤代烷（卤代烷先被转化成叶立德，再把它的残基经叶立德转移到醛酮上）；
- **碳链是增长的**——这与消除反应制烯烃截然不同。消除是在原有骨架上脱去小分子造双键，碳数不增；Wittig 则像"插积木"，把两块拼到一起，造出碳链更长的烯烃。
:::

## 逆合成分析：从产物反推原料

学会了这两件武器，最值钱的不是正着背反应，而是**倒着想**——拿到一个目标分子，把它"切开"成可买、可做的原料。这就是逆合成分析。

:::insight{label=逆合成的两把"剪刀"}
**剪刀一（金属有机试剂 + 醛酮）：** 看到目标分子是一个**醇**，把羟基碳周围的某根 C–C 键切开。切开后，一侧补上金属（成格氏试剂/有机锂/炔基负离子），另一侧的碳带着氧补成醛或酮。
- 切开后氧所在碳上还剩一个氢 → 原料是醛 → 正反应给仲醇；
- 剩两个烃基 → 原料是酮 → 正反应给叔醇；
- 若羟基碳与相邻碳之间是"乙基桥"形式，可考虑由环氧乙烷提供两个碳。

**剪刀二（Wittig 反应）：** 看到目标分子是一个**碳碳双键**（烯烃），就把双键从中间切开。切开后，一侧补成醛或酮（带羰基氧），另一侧补成卤代烷（再经三苯基膦与强碱转化为叶立德）。
:::

:::example{label=例题：用逆合成设计两条路线}
**题目：**
（1）以"苯甲醛 + 适当格氏试剂"为思路，合成 1-苯基-1-丙醇 $\ce{C6H5CH(OH)CH2CH3}$。
（2）用 Wittig 反应思路，分析合成 $\ce{C6H5CH=CH2}$（苯乙烯）可用的"醛酮 + 卤代烷"原料对。

**解答：**
（1）目标是一个**仲醇**，羟基碳上连着苯基、氢、乙基。把羟基碳与乙基之间的 C–C 键切开：羟基碳一侧带氧、且还剩一个氢和一个苯基 → 还原为**苯甲醛** $\ce{C6H5CHO}$；乙基一侧补金属 → **乙基格氏试剂** $\ce{CH3CH2MgBr}$。
$$\ce{C6H5CHO ->[CH3CH2MgBr][无水乙醚] ->[H3O+] C6H5CH(OH)CH2CH3}$$
因底物是醛，生成仲醇，与目标一致。

（2）目标是烯烃 $\ce{C6H5CH=CH2}$，把双键切开。一种拆法：醛酮一侧取**苯甲醛** $\ce{C6H5CHO}$（提供 $\ce{C6H5CH=}$ 中的碳与原羰基氧），叶立德一侧由**卤甲烷**经三苯基膦与强碱转化的叶立德 $\ce{(C6H5)3P=CH2}$ 提供 $\ce{=CH2}$。
$$\ce{C6H5CHO + (C6H5)3P=CH2 -> C6H5CH=CH2 + (C6H5)3P=O}$$

**点拨：** 切键时务必分清"谁提供羰基氧、谁提供烃基残基"。叶立德提供的永远是原卤代烷的烃基残基。
:::

## 易错点

:::pitfall{label=四个常见坑}
1. **醇的级别记反。** 加甲醛→伯醇、加其他醛→仲醇、加酮→叔醇。判据是"原羰基碳上还剩几个氢/烃基"。

2. **环氧乙烷增几个碳记错。** 格氏试剂加环氧乙烷一次增**两个**碳、生成伯醇，不是增一个。

3. **叶立德原料用错卤代烷。** 三级卤代烷、卤代烯烃、卤代芳烃都做不出叶立德；只能用伯、仲卤代烷（其中可含 C=C、C≡C）。

4. **把 Wittig 当消除。** Wittig 制烯烃碳链增长、且原料是"醛酮 + 卤代烷"两块拼接；消除制烯烃碳数不增。逆合成切双键时，别忘了切出来的是这两块原料。
:::

:::note{label=本节小结}
- **金属有机试剂**（格氏 $\ce{RMgX}$、有机锂、炔基钠/锂）本质是携带碳负离子，进攻羰基碳构筑新碳碳键，扩碳生成醇。
- **醇的级别由底物定**：甲醛→伯醇，其他醛→仲醇，酮→叔醇；加**环氧乙烷**一次增两个碳生成伯醇；炔基负离子加成在生成醇的同时引入碳碳三键。
- **Wittig 反应**：三苯基膦 + 伯/仲卤代烷成季鏻盐，强碱拔 α-氢成**叶立德**；叶立德碳负离子进攻羰基，经**四元环中间体**断裂，生成**碳链增长的烯烃** + 三苯基氧膦。
- **逆合成两把剪刀**：见醇切羟基碳旁 C–C 键 → 金属有机试剂 + 醛酮；见烯烃切双键 → 醛酮 + 卤代烷（经叶立德）。叶立德始终提供原卤代烷的烃基残基。
:::
