# 10.3 取代与消除的竞争 · 金属有机化合物

## 一、直觉：一个卤代烷，四条路在抢

到这里，我们已经认识了卤代烷的两大类反应、四种机制：取代里有 $S_N1$、$S_N2$，消除里有 E1、E2。把它们放在一起看，对一个卤代烷在碱性条件下的体系，**有四条路径同时在竞争**：$S_N2$、$S_N1$、E2、E1。

为什么会"四条路抢"？根子在一个事实——

:::insight{label=亲核试剂与碱往往是同一个物种}
带负电荷的试剂（比如 $\ce{OH^-}$，或有机里常用的烷氧基负离子 $\ce{RO^-}$）既能当亲核试剂，又能当碱。区别只在它"结合谁"：
- 结合**质子**时，它是**碱**——拔走质子后自己变成醇。
- 结合 **α 碳**时，它是**亲核试剂**——发生取代。

连醇分子（有孤对电子）都能既作弱亲核试剂进攻 α 碳、又作弱碱拔质子。所以只要体系里有这种带负电的试剂，它就有可能进攻卤代烷的两个位置。
:::

:::definition{label=两个进攻位点决定两类反应}
- **进攻 α 碳**：碳卤键极化使 α 碳带部分正电，能接受负电试剂进攻，发生 $S_N$（亲核取代）。
- **进攻 β 氢**：碳卤键的极性通过**诱导效应**传到旁边的 C–H 上，使 β 氢带一定酸性（部分正电），容易被碱拔走，发生 E（消除）。诱导效应虽是短程效应，但传到相邻碳的 C–H 上仍有威力。

取代产物五花八门（看亲核试剂是什么），消除产物却唯一（就是烯烃）。
:::

## 二、谁赢？四个调控因素

到底走取代还是消除，由**底物结构、试剂（亲核性/碱性）、溶剂、温度**共同决定。

### 因素一：试剂——亲核性强还是碱性强

:::theorem{label=试剂对取代/消除竞争的影响}
- **亲核性强、碱性弱的试剂 → 有利取代（$S_N$）**。典型代表是 $\ce{I^-}$：碱性不强，但可极化性（变形能力）强。因为亲核试剂要挤进被基团包围、有位阻的 α 碳，可极化性很关键；而碱拔的质子是裸露在外、无取代基的，只看碱性。
- **碱性强、亲核性弱的试剂 → 有利消除（E）**。典型代表是 $\ce{OH^-}$：碱性很强。
:::

这正好呼应上一节的辨析：碱性只是亲核性的一个分量，亲核性还要叠加可极化性。碘负离子"碱性弱、可极化性强"，所以偏取代；氢氧根"碱性强"，所以偏消除。

### 因素二：底物结构

:::example{label=例1——伯卤代烷 + 强亲核试剂}
一溴丁烷（伯卤代烷）+ 烷氧基负离子：产物 91% 是醚（取代）、仅 9% 消除。

伯卤代烷位阻小，容易走双分子 $S_N2$。结论：**伯卤代烷 + 强亲核试剂 → $S_N2$（取代为主）**，有明确取向性。
:::

:::example{label=例2——β 位有位阻的伯卤代烷}
仍是伯卤代烷，但 β 碳上有了支链（位阻）。$S_N2$ 对位阻极其敏感，α 碳和 β 碳的位阻都会削弱它。于是相同条件下取代比例下降、消除比例上升。

这恰恰显示了竞争性：α 碳进攻受阻，碱"调转头"去进攻 β 氢，照样完成转化。结论：**仲卤代烷、以及 β 碳带支链的伯卤代烷**，位阻削弱 $S_N2$、增大 E2，得取代与消除的混合物，没有强烈倾向。
:::

:::example{label=例3——叔卤代烷}
叔卤代烷先发生单分子异裂生成碳正离子（单分子过程），再看条件：
- **无强碱时 → $S_N1$**：面对一个正电荷，不论带负电还是中性的亲核试剂都易结合；中性试剂更没机会去拔 β 氢，只能和正电荷结合。
- **有强碱时 → E2**：强碱去拔 β 氢，同时 C–X 键也更易断（E2 断两键），走双分子消除。

数据示意：某体系取代 81%、消除 19%；换条件后取代 3%、消除 97%。
:::

:::insight{label=叔卤代烷有强碱时为何不走 E1}
碳正离子一旦生成，面前明摆着一个正电荷——不论来的是亲核试剂还是碱，都更愿意直接结合这个正电荷，谁还绕去进攻旁边带电性弱的 β 氢？所以 **E1 概率极低**。叔卤代烷若有强碱并生成烯烃，绝大多数走 **E2**（拔氢与断 C–X 同步的双分子过程），而非 E1。
:::

### 因素三：溶剂

溶剂主要影响的是**机制（单分子 vs 双分子）**，而非取代/消除的类型本身。

:::theorem{label=溶剂效应}
- 增加溶剂极性 **有利单分子机制**（主要是 $S_N1$，E1 可暂时忽略）：因为单分子要生成碳正离子，极性溶剂能稳定它。
- 增加溶剂极性 **对双分子机制影响不大**。

现象层面：同一结构卤代烷，在极性较小的醇中倾向消除、在水中倾向取代；但本质上溶剂左右的是单/双分子机制。
:::

### 因素四：温度

:::theorem{label=温度效应——升温有利消除}
某叔卤代烷生成烯烃：25 ℃ 时消除占 19%，55 ℃ 时占 28%。**提高温度有利于消除**。

原因：消除要同时断两根键（C–H 与 C–X），是更耗能的反应，升温为这种耗能反应提供了更多能量，所以更有利。
:::

### 竞争规律总表

:::note{label=取代—消除竞争总小结}
对一个卤代烷，理论上四条路径 $S_N1$/$S_N2$/E1/E2：
- **直链伯卤代烷**：$S_N2$ 易行，消除很少；只有强碱存在时才发生消除。
- **仲卤代烷、β 碳带支链的伯卤代烷**：$S_N2$ 变慢。低极性溶剂 + 强亲核试剂 → $S_N2$；低极性溶剂 + 强碱 → E2。
- **叔卤代烷**：$S_N2$ 很慢；高浓度强碱 → E2；无强碱时得 $S_N1$ 与 E1 的混合产物。
- 一句话口诀：伯 + 强亲核试剂 → $S_N2$；叔 + 强碱 → E2；强碱当道 → E2；升温利消除；极性溶剂利单分子。

提醒：取代和消除不止针对卤代烷，后续学的醇、羧酸衍生物等也有这些过程，机制大同小异（双分子都要发起进攻、单分子都生成碳正离子）。
:::

## 三、卤代烷与金属的反应：金属有机化合物

卤代烷还有一类很重要的反应——与活泼（还原性）金属直接化合，形成**碳原子与金属原子直接相连**的化合物，称为**金属有机化合物**（有机物里含金属原子）。本课介绍三种：格氏试剂、有机锂试剂、二烷基铜锂。它们的共同灵魂是——都能提供**碳负离子**。

### 格氏试剂

:::definition{label=格利雅试剂（格氏试剂）}
由卤代烷与金属镁在**醚**中反应生成，脂肪族和芳香族卤代烷都能生成。例：
$$
\ce{CH3CH2Br + Mg ->[\text{醚}] CH3CH2-MgBr}
$$
在 $\ce{CH3CH2MgBr}$ 中，镁 +2 价、溴 −1 价，所以烷基那端带一个负电荷——可视作一个**碳负离子**与正离子结合，而非纯共价。
:::

:::insight{label=格氏试剂是碳负离子，怕水}
既然烷基端是碳负离子，它就很容易与含**活泼氢**的化合物反应（把活泼氢拔走）。所以格氏试剂**一见水就失活**：
$$
\ce{CH3CH2-MgBr + H2O -> CH3CH3 + Mg(OH)Br}
$$
碳负离子拔走水中质子，烷基变回烷烃——相当于"卤代烷 → 格氏试剂 → 烷烃"。同理，格氏试剂能与卤化氢、醇的羟基氢、炔的端基氢（都有活泼性/酸性）反应生成相应的烃。所以格氏试剂本质是一个**碳负离子供体**。
:::

### 有机锂试剂与二烷基铜锂

:::definition{label=有机锂与二烷基铜锂}
- 卤代烷与活泼碱金属**锂**反应，卤素直接掉下，生成**有机锂试剂**，烷基端带负电荷。
- 有机锂再与**碘化亚铜**反应，两个烷基结合、配一个铜一个锂，生成**二烷基铜锂**，同样带负电荷。

格氏试剂、有机锂、二烷基铜锂，都是有机化学中提供碳负离子的金属有机试剂。
:::

## 四、碳负离子的真正用途：构筑碳碳键

:::insight{label=碳负离子是为造碳碳键而生}
碳负离子若只是去和水、醇反应，那等于费劲做出卤代烷又变回了烷烃，毫无意义。它真正的价值在于**构筑碳碳键**——这是有机化学最重要的能力之一。

有机反应按碳数变化分三类：
1. **碳数不变**：取代、消除，只是化合物类型变化。
2. **碳数减少**：需敲断碳键（后续学）。
3. **碳数增加**：最重要的一类，靠**构筑碳碳键**实现。
:::

:::example{label=用碳负离子增长碳链}
碳负离子本身就是亲核试剂，拿它去进攻碘代烷：
$$
\ce{R-CH2^{-} + R'-CH2-I -> R-CH2-CH2-R' + I^-}
$$
原来 5 个碳的进攻后变 6 个碳，构筑了一根碳碳键、碳链增长。

二甲基铜锂与卤苯反应同样构筑碳碳键、增加碳数；二烷基铜锂还能与苯环上的卤原子、双键上的位置反应——它**不挑位置**，是灵活的碳碳键构筑试剂。
:::

:::theorem{label=三种金属有机试剂的共性}
格氏试剂、有机锂、二烷基铜锂的共同特征：都能提供**碳负离子**；碳负离子作为亲核试剂，可生成更长的碳链，用于构筑碳碳键。合成上若起始物碳数少、目标物碳数多，就要想到这些试剂来增长碳链。
:::

## 五、卤代烷的还原（脱卤）

卤代烷可被还原——有机化学中通常认为"加氢"即被还原。卤代烷一般用以下条件还原成相应的烷烃：

:::theorem{label=卤代烷还原脱卤的条件}
- **氢化铝锂 $\ce{LiAlH4}$**：其中氢为 −1 价（负氢，带负电的氢），是强还原剂，可把卤代烷直接还原为烷烃。这个还原剂后续会反复用到。
- **羧酸 + 锌**：在羧酸、锌催化下，卤代烷也可被还原为烷烃。
- **催化加氢**：用催化加氢方式同样能把卤代烷还原为烷烃。
$$
\ce{R-X ->[\ce{LiAlH4}] R-H}
$$
:::

## 六、例题

:::example{label=取代还是消除——条件决定方向}
同样是伯卤代烷，两种条件：
- **水的碱溶液**：取代为主，生成相应的醇。
- **碱的醇溶液 + 加热**：消除为主，生成烯烃（不加热也会大概率取代，加热后才往消除走）。

结论：一级卤代烷在水溶液中易取代，在碱的醇溶液中（尤其加热）易消除。答题须标主产物，不可两者都写却不分主次。
:::

:::example{label=金属有机反应辨析}
- 卤代烷 + Mg/醚 → 格氏试剂。
- 卤代烷在苯环章学过的**傅克烷基化**：会经碳正离子并可能重排（如一级碳正离子重排为二级再发生亲电取代）——学了后面别忘前面。
- 格氏试剂的烷基化：一部分被烷基化，另一部分发生交换。
:::

## 七、易错点

:::pitfall{label=四个常见陷阱}
1. **混淆亲核试剂与碱**：同一物种结合 α 碳是取代、结合质子是消除；$\ce{I^-}$ 偏取代（可极化性强）、$\ce{OH^-}$ 偏消除（碱性强）。
2. **把温度对消除的促进当成对取代**：升温有利消除（断两键耗能），别记反。
3. **以为叔卤代烷有强碱时走 E1**：碳正离子面前更易结合正电荷，E1 概率极低，叔卤代烷生成烯烃绝大多数走 E2。
4. **忘记格氏试剂怕水**：格氏试剂含碳负离子，遇水/醇/卤化氢/炔端氢等活泼氢即失活变烷烃，制备和使用须无水。
:::

:::note{label=本节小结}
- 卤代烷在碱性体系有四条竞争路径：$S_N1$、$S_N2$、E1、E2；亲核试剂与碱常是同一物种，进攻 α 碳→取代、进攻 β 氢→消除。
- **竞争四因素**：①试剂——亲核性强碱性弱（如 $\ce{I^-}$）利取代，碱性强（如 $\ce{OH^-}$）利消除；②结构——伯+强亲核试剂→$S_N2$，叔+强碱→E2，仲居中；③溶剂——极性利单分子机制（主要 $S_N1$）；④温度——升温利消除。
- **三种金属有机试剂**：格氏试剂（卤代烷+Mg/醚）、有机锂（卤代烷+Li）、二烷基铜锂（有机锂+CuI），共性是提供**碳负离子**，用于**构筑碳碳键、增长碳链**；格氏试剂怕活泼氢（怕水）。
- **还原脱卤**：$\ce{LiAlH4}$、羧酸+锌、催化加氢，均可把卤代烷还原为烷烃。
:::
