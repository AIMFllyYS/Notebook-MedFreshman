# 7.1 炔烃的结构、酸性与加成

## 从"再多一根键"说起

上一章我们花了很大力气去理解烯烃那根活泼的碳碳双键。现在把不饱和度再往上推一格——在两个碳之间塞进三根键，就得到了**炔烃**。最简单的炔烃是乙炔 $\ce{C2H2}$，它的结构看上去比乙烯还要"紧凑"。

很自然地，你会以为炔烃只是"加强版的烯烃"：既然双键活泼，三键应该更活泼才对。但有机化学里有意思的地方恰恰在这——炔烃既继承了烯烃的脾气（能加成、能氧化），又长出了一些烯烃完全没有的独门性质。其中最特别的两件事，一是**端基炔上那个氢居然有酸性**，二是**炔烃首次加成反而比烯烃更难**。这一节我们就从结构出发，把这两件"反直觉"的事讲清楚，再看炔烃形形色色的加成反应。

## 一、炔烃的结构与不饱和度

### 1. 三键由什么组成

炔烃和烯烃的共同点是都含有不饱和的碳碳键，但炔烃比烯烃的不饱和度又增加了——它有两个不饱和的碳碳键。换句话说，碳碳三键里有一根 $\sigma$ 键，外加**两根** $\pi$ 键。

:::definition{label=碳碳三键的结构}
炔烃中成三键的碳原子采取 **SP 杂化**。两个 SP 杂化碳之间形成一根"头碰头"重叠的 $\sigma$ 键；剩下两个相互垂直的 p 轨道，分别"肩并肩"侧面重叠，形成两根互相垂直的 $\pi$ 键。这两根 $\pi$ 键的电子云叠加起来，像一个圆柱形的电子云筒，把中间的碳碳 $\sigma$ 键整个包裹起来。由于是 SP 杂化，三键以及与三键相连的原子排成**直线形**，键角约 $180°$。
:::

所以乙炔 $\ce{H-C#C-H}$ 是一根笔直的棍子：四个原子共线，中间是一根被圆柱形 $\pi$ 电子云包裹着的碳碳单键骨架。

### 2. 乙烷、乙烯、乙炔的结构对照

把这三个最典型的二碳分子放在一起比较，能看出 s 成分的变化怎样一步步改变分子。

:::insight{label=从SP³到SP，键在缩短、电负性在升高}
从饱和的 $\ce{C-C}$（乙烷，SP³ 杂化），到不饱和的 SP² 杂化（乙烯），再到更不饱和的 SP 杂化（乙炔），随着杂化轨道中 **s 成分逐渐增加**，原子核对外层电子的吸引力增强，于是：
- 碳碳键在不断**缩短**：乙烷的 $\ce{C-C}$ 单键最长，乙烯的 $\ce{C=C}$ 次之，乙炔的 $\ce{C#C}$ 最短；
- 碳氢键也随之缩短；
- 碳原子的**电负性**升高，顺序为：SP 杂化碳 > SP² 杂化碳 > SP³ 杂化碳。

s 轨道比 p 轨道更靠近原子核，s 成分越多，电子被"拉"得越近，键越短、电负性越大。这条规律是后面理解炔氢酸性的关键。
:::

### 3. 不饱和度的角度

一根碳碳三键贡献两个不饱和度。比如分子式 $\ce{C5H8}$，按不饱和度公式算出来是 $2$ 个不饱和度，这两个不饱和度可以是"一根三键"，也可以是"一根双键加一个环"，还可以是"两根双键"。录音里的推断题正是抓住这一点：$\ce{C5H8}$ 的三个同分异构体都能使溴的四氯化碳溶液褪色，说明都含不饱和的碳碳键，再用别的反应去区分它们到底是炔、是双烯还是环烯。

## 二、炔烃的总体反应特点

在逐个看反应之前，先建立一个总判断，能帮你预判很多结果。

:::theorem{label=炔烃首次加成比烯烃难，第二次反而容易}
炔烃发生加成时，**第一次加成（三键→双键）比烯烃困难，第二次加成（双键→单键）则比第一次容易。**

原因在中间体：按经典亲电加成机理，第一步要先打断一根 $\pi$ 键生成碳正离子。炔烃第一次加成生成的碳正离子坐落在 **SP 杂化碳**上，而 SP 碳电负性最大、最"吝啬"电子，在它上面安放一个正电荷非常不稳定，所以炔烃首次加成的能垒高、比烯烃难。可一旦突破第一步、炔烃变成了烯烃，后续加成就回到了烯烃的"容易"轨道上。
:::

:::insight{label=这个特点的两个实用推论}
1. **想停在烯烃这一步是有挑战的。** 因为第二次加成更容易，反应往往一冲到底直接加成两次。要想"克制"住、只加一次停在烯烃，需要刻意选择温和的条件（如下面的林德拉催化剂、低温控制）。
2. **同分子里既有双键又有三键时，1 摩尔试剂优先和双键反应。** 因为双键的（亲电）加成、氧化活性都高于三键。录音里反复用这一点出题：分子里同时存在双键和三键，限量试剂总是先动双键。
:::

## 三、炔烃的酸性

这是炔烃区别于烯烃最有"个性"的一条性质：**端基炔上直接连在三键碳上的那个氢，具有一定的酸性。**

### 1. 有机化学里怎么判断"酸性"

:::note{label=有机酸性的判断标准}
有机化学谈"酸性"，不像无机化学那样非要在水里电离出质子才算。这里判断一个氢有没有酸性，常看的是：**在强碱条件下，能不能把这个氢以质子形式"拽"下来。** 用有机强碱能拽下来，就说明它有一定酸性；连强碱都拽不动（比如饱和烃里的氢），就认为它没有酸性。
:::

### 2. 用 pKa 量化比较

更精确的办法是看 pKa 值（pKa 越小，酸性越强）。

:::insight{label=炔氢酸性的定位}
几类典型 C–H、O–H、N–H 的 pKa 大致是：
- 乙烷的 $\ce{C-H}$：约 **50**——基本没有酸性；
- 乙烯的 $\ce{=C-H}$：比乙烷降了一点，但降得不多；
- 乙炔的 $\ce{#C-H}$：约 **25**——已经有了一定的酸性；
- 醇的 $\ce{O-H}$：约 **16～18**；
- 氨/有机胺的 $\ce{N-H}$：比醇更弱。

所以**端基炔氢的酸性介于醇羟基氢和氨（胺）氮氢之间**：比醇弱，但比氨强。

为什么炔氢相对"酸"？回到上面的结构结论：炔碳是 SP 杂化、电负性最大，它能更好地容纳拽走质子后留下的那对负电荷（碳负离子），负电荷越稳定，质子越容易离开，酸性就越强。SP 碳负离子比 SP²、SP³ 碳负离子稳定得多，这正是 pKa 从 50→25 大幅下降的根本原因。
:::

### 3. 与活泼金属、强碱反应生成炔化物

既然端基炔氢有酸性，它就能被活泼金属或强碱夺走，生成一类同时含有机部分和金属部分的化合物——**有机金属化合物（金属炔化物）**。

:::example{label=生成炔化物的几种方式}
**与金属钠：** 乙炔在甲苯/THF 混合溶剂、严格无水的条件下与金属钠反应放出氢气，炔氢被拽下，生成 SP 杂化的碳负离子（炔钠）。
$$
\ce{HC#CH + Na ->[\text{甲苯/THF, 无水}] HC#C-Na + 1/2 H2 ^}
$$

**与氨基钠：** 在液氨中，乙炔与氨基钠 $\ce{NaNH2}$ 发生类似置换的过程，炔氢被夺走、生成一分子氨，同时得到炔基钠。
$$
\ce{HC#CH + NaNH2 ->[\text{液氨}] HC#C-Na + NH3}
$$

**与有机金属试剂（强碱）：** 烷基锂、格氏试剂本质上都提供强碱性的碳负离子，它们也能夺走炔氢。例如丁基锂夺氢后自身变成丁烷 $\ce{C4H10}$，把锂"让"给炔基；格氏试剂 $\ce{RMgX}$ 夺氢后自身变成烷烃，生成炔基格氏试剂。
$$
\ce{R-C#CH + R'MgX -> R-C#C-MgX + R'H}
$$
:::

:::pitfall{label=无水！这一步很危险}
炔钠、氨基钠这类碳负/氮负离子是极强的碱，遇水会立刻被水的质子"喂饱"而失效甚至剧烈放气。录音里特别强调乙炔和钠反应"没有水——不能有水，很危险"。凡是涉及强碱性碳负离子的反应，无水是底线。
:::

### 4. 端基炔的鉴定反应

炔氢的酸性还有一个非常实用的用途——**鉴定端基炔**。

:::example{label=银氨溶液与亚铜的鉴定}
- **与亚铜（一价铜）：** 端基炔与一价铜反应生成不溶的炔化亚铜，是**红褐色沉淀**。
- **与银氨溶液：** 端基炔与银氨溶液反应生成炔化银，是**白色沉淀**。

$$
\ce{R-C#CH ->[\text{Ag(NH3)2+}] R-C#C-Ag v\ (\text{白色})}
$$

两种试剂都靠生成有色/不溶的炔化物来"显形"，因此都可用来鉴定分子中是否含有端基炔。注意：**只有端基炔（三键在链端、带活泼氢）才有这个反应**，非端基炔（三键在链中间、没有炔氢）不反应。
:::

:::insight{label=鉴定题的标准套路}
录音里的鉴别题给出"烷、烯、端基炔"三种烃：先用**溴的四氯化碳溶液**，不褪色的是烷烃（先排除）；褪色的是烯和炔；再用**银氨溶液**，生成沉淀的就是端基炔，剩下的是烯烃。先用加成褪色分出饱和/不饱和，再用银氨分出端基炔，是这类题的固定思路。
:::

## 四、炔烃的催化加氢与选择性

炔烃同样能在金属催化下加氢。如果不加控制、条件给足，三键会被彻底还原成烷烃——这反而是容易的。真正的挑战是：**能不能让它只还原到烯烃这一步，而且还能控制生成顺式还是反式烯烃？**

### 1. 林德拉催化剂——顺式烯烃

:::definition{label=林德拉催化剂}
**林德拉（Lindlar）催化剂**是被"毒化"过的钯催化剂：把钯 $\ce{Pd}$ 负载在碳酸钙 $\ce{CaCO3}$（或硫酸钡 $\ce{BaSO4}$）上，再加少量抑制剂削弱其活性。碳酸钙、硫酸钡在这里相当于钯的**毒化剂**，把钯过强的催化活性降下来，让加氢"刹得住车"，停在烯烃这一步。
:::

$$
\ce{R-C#C-R' ->[\ce{H2}][\text{Lindlar (Pd/CaCO3)}] \text{顺式-}R-CH=CH-R'}
$$

:::insight{label=为什么是顺式}
林德拉催化加氢的机理和烯烃催化加氢的顺式加成一样：氢气先在催化剂表面被固定、活化成氢原子，炔烃"贴"在催化剂表面，两个氢原子从三键的**同一侧**加上去。结果两个原来的大基团被顶到双键的同一侧，得到**顺式烯烃**。
:::

### 2. 钠/液氨——反式烯烃

要得到反式烯烃，换一套条件。

:::definition{label=金属钠/液氨（溶解金属还原）}
把金属钠分散在液氨中（或用氢化铝锂 $\ce{LiAlH4}$ 处理内炔），同样能让炔停在烯烃这一步，但得到的是**反式烯烃**。
:::

$$
\ce{R-C#C-R' ->[\ce{Na}][\text{液氨}] \text{反式-}R-CH=CH-R'}
$$

:::insight{label=反式的来由（了解即可）}
钠/液氨体系中，金属钠先把一个电子交给体系。这个电子先与三键作用生成碳负离子（自由基负离子），碳负离子再去结合氨的氢……如此一步步把氢从两侧分别接上去，最终生成**反式**产物。机理细节不要求记忆，但要记住结论：**钠/液氨给反式，林德拉给顺式。**
:::

:::theorem{label=炔烃→烯烃的几何选择性总结}
通过选择不同的部分加氢条件，既能让炔烃停留在烯烃这一步，又能控制几何构型：
- **林德拉催化剂（Pd/CaCO₃ 或 Pd/BaSO₄）→ 顺式烯烃**；
- **金属钠/液氨（或 LiAlH₄）→ 反式烯烃**。
:::

## 五、炔烃的亲电加成

烯烃能亲电加成，炔烃当然也能，只是因为前面说的"首次加成更难"，结果常有不同。

### 1. 与卤素加成

:::example{label=与卤素：生成邻二卤代烯或四卤代烷}
炔烃与卤素的加成一般很难停在烯烃这一步——一旦生成烯烃，它与卤素的加成比炔烃更容易，所以常温下"一冲到底"，加上两分子卤素生成**四卤代烷**。

想控制只加一次怎么办？**降温即可。** 在较低温度下可把反应控制在生成**邻二卤代烯**（一分子卤素加成产物）。
$$
\ce{R-C#C-R' ->[\ce{Br2}][\text{低温}] \text{邻二溴代烯} ->[\ce{Br2}] \text{四溴代烷}}
$$
:::

注意：炔烃与卤素加成**不像烯烃那样讨论环鎓离子**——炔烃这里是"直接加到底"生成偕/邻二卤代结构。若分子里同时有双键和三键，1 摩尔卤素**优先**加到双键（烯烃亲电加成活性高于三键）。

### 2. 与氢卤酸加成（马氏规则）

:::example{label=与HX：两次马氏加成生成偕二卤代烷}
炔烃与氢卤酸加成同样遵守**马氏规则**（氢加到含氢较多的碳上），且也难以控制在第一步，往往连续进行两次。两次加成的卤素都加到**同一个碳**上，生成**偕二卤代烷**（一个碳上带两个卤原子）。
$$
\ce{HC#CH ->[\ce{HBr}] CH2=CHBr ->[\ce{HBr}] CH3-CHBr2}
$$

氢卤酸对炔烃的反应活性顺序：$\ce{HI} > \ce{HBr} > \ce{HCl}$；同时炔烃的反应活性比烯烃小。
:::

:::pitfall{label=邻二卤代烷 vs 偕二卤代烷}
两个卤原子在**相邻两碳**上，叫**邻二卤代烷**（来自卤素加成的两次分别加在不同碳）；两个卤原子在**同一碳**上，叫**偕二卤代烷**（来自 HX 两次马氏加成，氢都加到含氢多的碳、卤都集中到另一个碳）。一字之差，来源和结构完全不同，做题时要看清是哪种试剂、几次加成。
:::

### 3. 与水加成——经烯醇式互变成醛酮

这是炔烃与烯烃差别最大的一个反应。

:::theorem{label=炔烃水合：烯醇式互变异构}
炔烃在（酸/汞盐催化下）与水加成时，第一分子水按马氏规则加上去，先生成一个**烯醇式**结构——双键碳上连着一个羟基 $\ce{C=C-OH}$。烯醇式很不稳定，会迅速发生**互变异构**，转变为更稳定的**酮式**（羰基）结构。羰基一旦生成，就**失去再与第二分子水加成的活性**，所以炔烃水合只加成一次，最终产物是**醛或酮**，而不是二醇。
:::

$$
\ce{R-C#CH + H2O ->[\text{马氏}] [R-C(OH)=CH2] ->[\text{互变异构}] R-CO-CH3}
$$

:::insight{label=醛还是酮，看哪个碳}
第一步加水仍遵守马氏规则：氢加到含氢较多的炔碳，羟基加到另一个炔碳。互变异构后，原来连羟基的那个碳变成羰基。
- 乙炔水合：羰基碳上还连着一个氢 → 生成**乙醛**（醛）；
- 末端为 $\ce{R-C#CH}$ 型的端基炔水合（马氏加成）→ 羰基落在内侧碳 → 生成**甲基酮**（酮）。
所以"炔烃 + 水"是制备醛、酮的一条重要途径。
:::

:::pitfall{label=别把炔的水合当成烯的水合}
烯烃 + 水 → 醇（停在羟基）；炔烃 + 水 → 醛或酮（经烯醇式互变，不是二醇）。很多同学惯性地以为炔加两次水变二醇，这是错的——烯醇式一旦互变成羰基就"封口"了。这是炔、烯亲电加成最大的一处差别，务必记牢。
:::

## 六、例题

:::example{label=综合判断炔烃的加成产物}
**题：** 某炔烃 $\ce{CH3-C#CH}$（丙炔）分别在下列条件下反应，写出主要产物：
(1) 1 摩尔 $\ce{HBr}$；(2) 过量 $\ce{HBr}$；(3) $\ce{H2O}$（酸/汞盐催化）；(4) 林德拉催化剂 + $\ce{H2}$；(5) $\ce{Na}$/液氨 + $\ce{H2}$（设为内炔时讨论）。

**解：**
(1) 一次马氏加成：氢加到端基 $\ce{=CH}$ 上（含氢多），溴加到中间碳 → $\ce{CH3-CBr=CH2}$（2-溴丙烯）。
(2) 两次马氏加成：两个溴都集中到中间碳 → $\ce{CH3-CBr2-CH3}$（2,2-二溴丙烷，偕二卤代烷）。
(3) 马氏加水经烯醇式互变 → 羰基落在中间碳 → $\ce{CH3-CO-CH3}$（丙酮）。
(4) 林德拉部分加氢，丙炔为端基炔，得 $\ce{CH3-CH=CH2}$（丙烯）。
(5) 对内炔，钠/液氨给反式烯烃（端基炔无顺反之分）。

**点拨：** 抓住三条线——HX/卤素遵守马氏且易加两次（偕二卤代烷）；水合走烯醇式互变成醛酮；催化加氢用条件控制顺/反。
:::

## 七、易错点小结

:::pitfall{label=常见错误清单}
1. **以为三键比双键更容易加成。** 恰恰相反，炔烃**首次**加成比烯烃难（SP 碳正离子不稳定），但第二次容易。
2. **限量试剂搞错优先级。** 同分子里双键、三键并存时，1 摩尔试剂**先动双键**。
3. **炔水合写成二醇。** 应是经烯醇式互变得**醛或酮**，只加一次。
4. **偕/邻二卤代烷不分。** HX 两次加成 → 偕二卤代烷（同碳）；卤素加成 → 邻二卤代（异碳）。
5. **顺反烯烃条件记反。** 林德拉给顺式，钠/液氨给反式。
6. **强碱炔化物遇水。** 炔钠、格氏试剂等必须无水操作。
:::

:::note{label=本节小结}
**结构：** 炔烃含碳碳三键（SP 杂化，直线形），三键 = 1 根 $\sigma$ + 2 根 $\pi$，圆柱形 $\pi$ 电子云包裹骨架。从 SP³→SP，键缩短、碳电负性升高（SP > SP² > SP³）。一根三键贡献 2 个不饱和度。

**总特点：** 首次加成比烯烃难、第二次容易；双键与三键并存时限量试剂先动双键。

**酸性：** 端基炔氢 pKa ≈ 25，介于醇（16～18）与氨之间；能与 Na、NaNH₂、RLi、RMgX 反应生成炔化物（无水！）；银氨（白色沉淀）、亚铜（红褐色沉淀）可鉴定端基炔。

**加成：**
- 催化加氢选择性：林德拉(Pd/CaCO₃)→顺式烯烃；Na/液氨→反式烯烃。
- 与卤素、HX：马氏规则，易加两次，分别生成邻二卤代烯/四卤代烷、偕二卤代烷；降温可控制只加一次。
- 与水：马氏加成经烯醇式互变 → 醛或酮（只加一次）。

:::
