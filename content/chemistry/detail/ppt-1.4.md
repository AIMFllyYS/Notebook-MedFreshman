:::insight{label=从"分子怎么反应"到"分子是酸还是碱"}
学完了化学键和键参数，我们已经能描述一个分子的"静态"结构。但有机化学的核心魅力在于"动态"——分子如何反应、如何转化。要理解反应，必须先回答三个问题：

1. **反应的键怎么断？** 共价键断裂有两种基本方式：均裂和异裂，对应自由基反应和离子型反应；
2. **反应在哪里发生？** 有机分子中真正决定化学性质的"活性位点"叫官能团，掌握官能团就掌握了有机化学的"地图"；
3. **反应的驱动力是什么？** 大量有机反应本质上是酸碱反应——质子的转移或电子对的给予/接受。Brönsted 和 Lewis 两种酸碱理论是理解有机反应机理的基石。

本节我们就来逐一回答这三个问题，并最后引入"共振理论"——一种描述电子离域的强大工具。
:::

## 共价键的断裂方式与反应中间体

:::definition{label=均裂与异裂}
共价键断裂有两种基本方式：

- **均裂（homolysis）**：共用电子对均等地分给两个原子，每个原子各得一个电子，生成**自由基**（radical）。

$$
\ce{A:B -> A^. + B^.}
$$

- **异裂（heterolysis）**：共用电子对完全归其中一个原子，生成**碳正离子**（carbocation）或**碳负离子**（carbanion）。

$$
\ce{A:B -> A+ + :B-} \quad \text{或} \quad \ce{A:B -> A:- + B+}
$$
:::

:::theorem{label=三种碳反应中间体的结构}
- **甲基碳自由基** $\ce{.CH3}$：碳为 **sp² 杂化**，平面构型，未成对电子占据垂直平面的 p 轨道；
- **甲基碳正离子** $\ce{CH3+}$：碳为 **sp² 杂化**，平面构型，空的 p 轨道垂直于平面；
- **甲基碳负离子** $\ce{CH3-}$：碳为 **sp³ 杂化**，近似四面体构型（一对孤对电子占据一个 sp³ 轨道）。
:::
::figure{src="/images/chemistry/svg/ch1/carbocation-radical-carbanion.svg" caption="三种碳活性中间体构型：自由基、碳正离子、碳负离子" alt="甲基自由基sp²平面含单电子；甲基碳正离子sp²空p轨道；甲基碳负离子sp³四面体含孤对电子"}

:::insight{label=为什么碳正离子是 sp² 而碳负离子是 sp³}
碳正离子 $\ce{CH3+}$ 只有 3 个 σ 键 + 1 个空 p 轨道，所以采取 sp² 杂化（3 个等价轨道成键，1 个 p 空着），平面构型有利于空 p 轨道接受亲核试剂进攻。碳负离子 $\ce{CH3-}$ 有 3 个 σ 键 + 1 对孤对电子，共 4 个电子对，采取 sp³ 杂化（与 $\ce{NH3}$ 类似），近似四面体。这种结构差异直接决定了它们在反应中的不同行为。
:::

:::pitfall{label=把自由基和离子混淆}
自由基是电中性的（带未成对电子），不是带电的；碳正离子带正电，碳负离子带负电。考试时一定要写清楚：自由基用 $\ce{^.}$ 表示，离子用 $\ce{+}$ 或 $\ce{-}$ 表示。例如 $\ce{CH3^.}$（自由基）、$\ce{CH3+}$（碳正离子）、$\ce{CH3-}$（碳负离子）是三个完全不同的物种。
:::

## 有机化合物的主要官能团

:::definition{label=官能团}
**官能团**（functional group）是有机分子中决定其化学性质的特定原子或原子团。含有相同官能团的化合物具有相似的化学性质，因此官能团是有机化合物分类和反应预测的基础。
:::

:::theorem{label=常见官能团及对应化合物类型}
| 化合物类型 | 官能团 | 实例 |
|-----------|--------|------|
| 烷烃 | 无 | $\ce{CH4}$ |
| 烯烃 | $\ce{C=C}$（烯键） | $\ce{CH2=CH2}$ |
| 炔烃 | $\ce{-C#C-}$（炔键） | $\ce{HC#CH}$ |
| 卤代烃 | $\ce{-X}$（卤素） | $\ce{CH3Br}$、$\ce{CHCl3}$ |
| 醇、酚 | $\ce{-OH}$（羟基） | $\ce{CH3OH}$、$\ce{C6H5OH}$ |
| 硫醇、硫酚 | $\ce{-SH}$（巯基） | $\ce{CH3SH}$ |
| 醚 | $\ce{R-O-R}$（醚键） | $\ce{CH3OCH3}$ |
| 醛 | $\ce{-CHO}$（醛基） | $\ce{CH3CHO}$ |
| 酮 | $\ce{>C=O}$（酮基） | $\ce{CH3COCH3}$ |
| 羧酸 | $\ce{-COOH}$（羧基） | $\ce{CH3COOH}$ |
| 胺 | $\ce{-NH2}$（氨基） | $\ce{CH3NH2}$ |
:::

:::insight{label=官能团是"反应活性位点"}
有机分子的反应几乎都发生在官能团上，而不是碳骨架上。例如 $\ce{CH3CH2OH}$（乙醇）和 $\ce{CH3OH}$（甲醇）都能与钠反应放出氢气，因为它们都有 $\ce{-OH}$；而 $\ce{CH3CH2CH3}$（丙烷）不能，因为它没有官能团。掌握官能团 = 掌握有机化学的"地图"。
:::

## Brönsted-Lowry 酸碱理论（质子理论）

:::definition{label=Brönsted-Lowry酸碱}
**质子理论**：凡是能给出质子（$\ce{H+}$）的物质是**酸**，凡能接受质子的物质是**碱**。酸给出质子后形成该酸的**共轭碱**，碱接受质子后形成该碱的**共轭酸**。

$$
\ce{HA + B <=> A^- + BH+}
$$

其中 $\ce{HA}$ 与 $\ce{A^-}$ 是一对共轭酸碱，$\ce{B}$ 与 $\ce{BH+}$ 是另一对共轭酸碱。
:::

:::theorem{label=酸碱强弱与共轭关系}
- 给出质子的能力越强，**酸性越强**，其**共轭碱的碱性越弱**；
- 接受质子的能力越强，**碱性越强**，其**共轭酸的酸性越弱**。

例如 $\ce{HCl}$ 是强酸，其共轭碱 $\ce{Cl^-}$ 是极弱碱；$\ce{CH4}$ 是极弱酸，其共轭碱 $\ce{CH3^-}$ 是极强碱。
:::

:::pitfall{label=把"酸"或"碱"当成物质的固有属性}
在 Brönsted 理论中，一个化合物是酸还是碱**取决于具体反应**。例如 $\ce{H2O}$：
- 与 $\ce{HCl}$ 反应时，$\ce{H2O}$ 接受 $\ce{H+}$，是**碱**：$\ce{HCl + H2O -> H3O+ + Cl^-}$
- 与 $\ce{NH3}$ 反应时，$\ce{H2O}$ 给出 $\ce{H+}$，是**酸**：$\ce{H2O + NH3 -> OH^- + NH4+}$

所以 $\ce{H2O}$ 既是酸又是碱（两性物质）。判断酸碱要看在具体反应中扮演的角色。
:::

:::example{label=识别共轭酸碱对}
**题目**：在反应 $\ce{CH3COOH + H2O <=> CH3COO^- + H3O+}$ 中，指出两对共轭酸碱。

**解**：
- 共轭酸碱对 1：$\ce{CH3COOH}$（酸）↔ $\ce{CH3COO^-}$（共轭碱）
- 共轭酸碱对 2：$\ce{H3O+}$（共轭酸）↔ $\ce{H2O}$（碱）

$\ce{CH3COOH}$ 给出 $\ce{H+}$ 后变成 $\ce{CH3COO^-}$；$\ce{H2O}$ 接受 $\ce{H+}$ 后变成 $\ce{H3O+}$。乙酸酸性弱，所以其共轭碱 $\ce{CH3COO^-}$ 有一定碱性。
:::

## Lewis 酸碱理论（电子理论）

:::definition{label=Lewis酸碱}
**电子理论**：凡能**接受电子对**的物质是 Lewis 酸，凡能**给出电子对**的物质是 Lewis 碱。酸碱反应是生成**配位键**的过程，产物称为**酸碱加合物**。

$$
\ce{A + :B -> A:B}
$$

Lewis 酸有外层空轨道，Lewis 碱可提供孤对电子。
:::

:::theorem{label=Lewis酸碱的常见类型}
**Lewis 酸**（电子对受体）：
1. 可接受电子的分子：$\ce{BF3}$、$\ce{AlCl3}$、$\ce{SnCl4}$、$\ce{ZnCl2}$、$\ce{FeCl3}$
2. 金属离子：$\ce{Li+}$、$\ce{Ag+}$、$\ce{Cu^2+}$
3. 正离子：$\ce{C+}$、$\ce{Br+}$、$\ce{NO2+}$、$\ce{H+}$

**Lewis 碱**（电子对给体）：
1. 含未共享电子对的化合物：$\ce{NH3}$、$\ce{RNH2}$、$\ce{ROH}$、$\ce{ROR}$、$\ce{RCHO}$、$\ce{RSH}$
2. 负离子：$\ce{X^-}$、$\ce{HO^-}$、$\ce{RO^-}$、$\ce{R^-}$
3. 烯烃或芳香化合物（π 电子可给出）
:::

:::insight{label=Lewis 与 Brönsted 的关系}
- **Lewis 碱**与 **Brönsted 碱**没有太大区别：都是能给出电子对的物质（Brönsted 碱给出电子对去结合 $\ce{H+}$）；
- **Lewis 酸**的范围比 **Brönsted 酸**广：Brönsted 酸指能给出 $\ce{H+}$ 的物质（如 $\ce{HCl}$），而按 Lewis 定义，$\ce{HCl}$ 本身是酸碱加合物（$\ce{H+}$ 是 Lewis 酸，$\ce{Cl^-}$ 是 Lewis 碱）。

在有机化学中，**亲核试剂**（nucleophile）= Lewis 碱，**亲电试剂**（electrophile）= Lewis 酸。这一对应关系是理解有机反应机理的关键。
:::

:::example{label=Lewis酸碱加合物}
**题目**：$\ce{BF3}$ 与 $\ce{CH3OCH3}$（乙醚）反应，谁是酸谁是碱？

**分析**：$\ce{BF3}$ 中 B 外层只有 6 个电子，有空的 p 轨道；乙醚中 O 有孤对电子。

**解**：

$$
\ce{BF3 + :O(CH3)2 -> F3B-O(CH3)2}
$$

- $\ce{BF3}$：接受电子对 → **Lewis 酸**
- $\ce{CH3OCH3}$：给出电子对 → **Lewis 碱**
- 产物 $\ce{F3B-O(CH3)2}$：**酸碱加合物**，B 与 O 之间形成配位键

这个反应解释了为什么 $\ce{BF3}$（催化剂）常与乙醚一起保存——它们形成稳定的加合物。
:::

## 共振理论

:::definition{label=共振结构与共振杂化体}
**共振结构**（resonance structures）是表达某一分子、离子、自由基真实电子结构时所采用的两个或两个以上的经典共价键结构。真实的分子是这些共振结构的**共振杂化体**（resonance hybrid），即各共振结构的"加权平均"。

单个共振结构（又称**极限式**）并不真实存在，真实存在的只有杂化体。
:::

:::theorem{label=共振结构的贡献规则}
1. **能量越低的共振结构越稳定，对杂化体的贡献越大**；
2. **等价共振结构**（结构相同、能量相同）对杂化体的贡献相同（如苯的两个 Kekulé 式）；
3. **不等价共振结构**：较稳定者贡献较大（如乙酸根的两个共振式中，负电荷平均分布在两个 O 上的形式更稳定）；
4. 共振结构中**所有原子须满足八隅体**（除 H 外）的结构贡献大；电荷分离的结构贡献小。
:::

:::example{label=苯的共振结构}
**题目**：写出苯的共振结构，并说明其意义。

**解**：苯有两种主要的 Kekulé 共振式（单双键交替的两种排列）：

$$
\ce{ \text{式1：单-双-单-双-单-双} \leftrightarrow \text{式2：双-单-双-单-双-单} }
$$

两个共振式**完全等价**（能量相同），对杂化体贡献相同。因此真实苯中所有 C—C 键长完全相等（139 pm），介于单键（154 pm）和双键（134 pm）之间。这就是为什么苯不像普通烯烃那样容易发生加成反应——它的电子高度离域，体系能量低，特别稳定。
:::
::figure{src="/images/chemistry/svg/ch1/benzene-kekule.svg" caption="苯的两个Kekulé共振结构（C-C键长139 pm）" alt="苯环两个Kekulé结构，单双键交替排列不同，双向箭头连接，C-C键长139pm介于单键154pm和双键134pm之间"}

:::example{label=乙酸根的共振}
**题目**：写出乙酸根 $\ce{CH3COO^-}$ 的共振结构。

**解**：乙酸根有两种共振式：

$$
\ce{CH3-C(=O)-O^- <-> CH3-C(-O^-)=O}
$$

两种共振式中，负电荷分别位于两个 O 上，结构等价，对杂化体贡献相同。因此实际乙酸根中两个 C—O 键长完全相等（127 pm），介于 C=O（120 pm）和 C—O（143 pm）之间，负电荷**平均分布**在两个 O 上。

这种共振离域使乙酸根比普通的烷氧基负离子更稳定，因此乙酸具有较强的酸性（$\text{p}K_a = 4.76$）。
:::

:::pitfall{label=把共振结构当成"快速互变"}
共振结构**不是**分子在不同结构间的快速互变（那是互变异构 tautomerism）。共振结构是同一分子的不同"描述方式"，真实分子是它们的杂化体，从未真正"存在"于任何一个单一共振式中。打个比方：犀牛的"杂化体"既不是"独角兽"也不是"马"，而是真实存在的犀牛本身——独角兽和马只是描述它的两种"极限式"。
:::

:::note{label=本节小结}
- 共价键断裂方式：
  - **均裂** → 自由基（sp²，电中性）
  - **异裂** → 碳正离子（sp²，正电）/ 碳负离子（sp³，负电）
- 官能团决定有机物化学性质，是分类与反应预测的核心；
- **Brönsted 酸碱**：质子给体/受体，强调 $\ce{H+}$ 转移；酸越强，共轭碱越弱；
- **Lewis 酸碱**：电子对受体/给体，范围更广；亲核试剂 = Lewis 碱，亲电试剂 = Lewis 酸；
- **共振理论**：用多个极限式描述电子离域的真实分子；等价共振式贡献相同，稳定者贡献大；共振结构不是互变，而是同一分子的不同描述。

:::
