# 6.3 自由基加成与氧化反应

## 从"换条路、换个结果"说起

上一节我们看到，烯烃和氢溴酸在酸催化下走亲电加成，老老实实给出马氏产物。但同一对反应物——烯烃和 $\ce{HBr}$——只要换一个"引发条件"，加成的结果竟然会**翻个个儿**。这就是这一节要讲的第一件事：**自由基加成**。它和亲电加成构成一对漂亮的"互补"，让我们能自由控制溴加在哪个碳上。

随后我们转向烯烃的另一大类反应——**氧化**。烯烃除了"易加成"，还有一个鲜明特点是"易氧化"。高锰酸钾、臭氧、过氧酸都能在双键上做文章，有的温和地"补两个羟基"，有的干脆把双键"一刀切断"。这些断键反应不仅是制备手段，更是推断分子里双键位置的利器。

## 一、过氧化物引发的自由基加成

### 1. 现象：过氧化物让 HBr 加成变成反马氏

:::definition{label=自由基加成}
烯烃与氢溴酸加成时，若不用酸催化、而是加入**过氧化物**作为自由基引发剂，反应就不再走亲电加成的路线，而是走**自由基机理**。对不对称烯烃，主产物变成**反马氏产物**（氢加到含氢较少的碳上）。
:::

$$
\ce{R-CH=CH2 ->[\ce{HBr}][\text{过氧化物}] R-CH2-CH2-Br}
$$

对照一下就一目了然：

- 酸催化（亲电加成）→ **马氏**产物：$\ce{R-CHBr-CH3}$
- 过氧化物引发（自由基加成）→ **反马氏**产物：$\ce{R-CH2-CH2Br}$

:::insight{label=一对互补的加成}
于是我们对"烯烃 + HBr"就有了两套可控手段：
- 要**马氏卤代烷** → 选**酸性条件**（氢卤酸 + 酸催化，亲电加成）；
- 要**反马氏卤代烷** → 选**过氧化物**（自由基引发）。
两条路在区域选择性上正好互补。注意：这种"过氧化效应"主要对 $\ce{HBr}$ 显著。
:::

### 2. 机理：为什么反马氏？

自由基反应"谈不上亲电、亲核"，那它凭什么也有选择性、而且方向恰好相反？关键要一步步看清楚谁先进攻。

:::derivation{label=自由基加成的链式机理}
**引发：** 过氧化物的 $\ce{O-O}$ 键很弱，受热（或光照）容易**均裂**，生成两个烷氧自由基 $\ce{RO·}$。

**夺氢生溴自由基：** 烷氧自由基与 $\ce{HBr}$ 发生自由基交换——它**夺走氢**变成稳定的醇，同时把溴变成**溴自由基** $\ce{Br·}$。注意：这一步生成的活泼进攻物种是**溴自由基**。

**溴自由基进攻双键（决定取向的一步）：** $\ce{Br·}$ 去和双键反应，有两种加法：加到端基碳上，碳自由基就落在中间碳（**二级**碳自由基）；加到中间碳上，碳自由基落在端基碳（**一级**碳自由基）。碳自由基的稳定性规律和碳正离子一致：**三级 > 二级 > 一级**。所以溴优先加到端基碳、生成更稳定的二级碳自由基——这是优势中间体。

**夺氢成产物：** 这个碳自由基再从 $\ce{HBr}$ 上夺一个氢，生成稳定产物，同时再生出一个 $\ce{Br·}$ 继续链传递。

数一数：溴落在了**端基碳**（含氢较多的碳）上，氢落在了**中间碳**（含氢较少的碳）上——这正是**反马氏**！
:::

### 3. 和亲电加成到底差在哪

:::theorem{label=自由基加成 vs 亲电加成：关键在"谁先进攻"}
两者都是往双键上加一个氢、一个溴，差别有两点，但**最关键的只有一点**：
1. **中间体不同**：亲电加成是**碳正离子**，自由基加成是**碳自由基**。但这不是反马的根本原因——两者稳定性规律一模一样（三级>二级>一级）。
2. **先进攻的试剂不同（根本原因）**：亲电加成里**先进攻的是质子 $\ce{H+}$**，氢先加，碳正离子生成在取代多的碳上 → 马氏；自由基加成里**先进攻的是溴自由基 $\ce{Br·}$**，溴先加，碳自由基生成在取代多的碳上 → 反马氏。

一句话：**都是为了生成更稳定的中间体，但因为"先加的人"不同（一个先加氢、一个先加溴），最终的区域取向就反了过来。**
:::

:::pitfall{label=自由基加成不谈顺反}
和碳正离子一样，**碳自由基也是平面结构**，生成后不存在顺反选择性。所以自由基加成**不谈顺反**。回忆顺反口诀：催化加氢（顺）、卤素加成（反式）、硼氢化（顺）要谈顺反；碳正离子型亲电加成、自由基加成**都不谈**——因为中间体是平面的。
:::

## 二、与高锰酸钾的氧化

烯烃易氧化。高锰酸钾是常用氧化剂，它有"温和"和"剧烈"两套条件，给出完全不同的产物。这里我们只关注**条件与结果**（金属催化的机理不展开）。

:::theorem{label=高锰酸钾的两套条件}
**温和（弱氧化）条件**——冷、稀、碱性或中性 $\ce{KMnO4}$：
双键**不断裂**，在原双键的两个碳上各加一个羟基，生成**邻二醇**，而且是**顺式**邻二醇（两个羟基落在原双键平面的同一侧）。

**剧烈（强氧化）条件**——酸性 $\ce{KMnO4}$（或加热）：
双键**被打断**。断开处若原来连氢，氧化成**羧基**（生成羧酸）。
:::

:::insight{label="冷稀碱"就是温和信号}
看到题目条件里出现"**冷、稀、碱性或中性**"这些字，就意味着温和氧化——产物是顺式邻二醇、双键保留。写产物时一定要把**顺式**这个立体结构画出来（两个羟基同侧）。看到"**酸性**（或加热）"就是强氧化——双键断、生成羧基。
:::

:::example{label=丙烯的两种氧化结局}
- 冷稀碱性 $\ce{KMnO4}$：$\ce{CH3-CH=CH2 -> CH3-CH(OH)-CH2OH}$（顺式邻二醇，双键保留）。
- 酸性 $\ce{KMnO4}$：双键断开，端基 $\ce{=CH2}$ 一侧被彻底氧化，含氢的双键碳氧化成羧基，可由产物结构反推原烯烃的双键位置。
:::

:::note{label=强氧化的"反推"价值}
强氧化断键后，把生成的羧基（或酮）在断点处"弥合"回去，就能倒推出原来烯烃双键的位置。这一点和臭氧化一样，是结构鉴定的常用思路。
:::

## 三、臭氧化反应：断键成醛酮、定双键位置

:::definition{label=臭氧化}
烯烃先与**臭氧 $\ce{O3}$** 反应，生成一个五元环的臭氧化物中间体（不稳定，通常不专门写出），再在**锌 + 水**（还原剂）处理下，把双键**切断**，每个双键碳变成一个羰基。
- 原来双键碳上若**连有氢** → 生成**醛**；
- 原来双键碳上已被**完全取代**（无氢）→ 生成**酮**。
:::

$$
\ce{R2C=CHR' ->[\ce{O3}][1] ->[\ce{Zn,\ H2O}][2] R2C=O + R'CHO}
$$

通俗记法（录音原话）："就是把双键切开，你给它添俩氧就行了。"切开处是醛还是酮，只看原来那个双键碳上**有没有连氢**。

:::example{label=臭氧化的产物反推双键位置}
某烯烃经臭氧化—还原后得到**一分子甲醛**和**一分子乙醛**。把两个羰基在断点处"对接"回去（甲醛的 $\ce{CH2=}$ 和乙醛的 $\ce{=CHCH3}$ 拼合），即可还原出原烯烃的双键所在——这就是用臭氧化做**结构鉴定**的基本玩法。
:::

:::insight{label=断键氧化是"拼图游戏"}
对于天然产物等复杂分子，你常常不知道双键藏在哪里。把它臭氧化（或强氧化）断成醛、酮、羧酸这些"小积木"，再测出这些小分子的结构，就能像拼图一样把断点"拼回去"，定位原来的双键。如果断键后只得到**一个**产物（而不是两个），往往说明双键在**环内**——分子内断键，断开的两端还连在同一条链上。
:::

:::pitfall{label=区分两套断键氧化的产物}
**臭氧化（$\ce{O3}$，然后 $\ce{Zn/H2O}$）**：断键后双键碳上**有氢就成醛、无氢就成酮**（还原性后处理，停在醛/酮）。
**酸性高锰酸钾（强氧化）**：断键后双键碳上**有氢的会进一步氧化成羧酸**（无氢的成酮）。
两者都断键，但"有氢端"的归宿不同：臭氧化止于醛，酸性 $\ce{KMnO4}$ 推到羧酸。审题时务必看清条件。
:::

## 四、环氧化与聚合（了解）

### 1. 环氧化

烯烃的双键还可以被氧化成**环氧化物**——在双键的两个碳之间架上一个氧，形成一个三元的环氧环（产物保留两个碳，只是多了个"氧桥"）。这与高锰酸钾的邻二醇、臭氧化的断键不同：环氧化**既不断键、也不直接成两个羟基**，而是把一个氧"嵌"进双键上方，得到环氧乙烷类的结构。它是后续制备邻二醇等含氧化合物的重要中间体。

:::note{label=三类含氧化的对照}
同样是把含氧基团接到双键上，结果各异：
- **冷稀碱 $\ce{KMnO4}$**：不断键 → **顺式邻二醇**（两个 $\ce{-OH}$）；
- **环氧化**：不断键 → **环氧化物**（一个氧桥架在双键上）；
- **酸性 $\ce{KMnO4}$ / 臭氧化**：**断键** → 羧酸 / 醛、酮。
是否断键、加的是几个氧、加在哪——这是区分它们的三把尺子。
:::

### 2. 聚合反应

用更宏大的视角看，许多双键的**聚合**也可以看作烯烃的加成——烯烃与烯烃之间通过双键打开，互相加成、串成长链。

:::example{label=两种重要的聚合}
- 乙烯 → **聚乙烯**（聚乙烯是日常塑料的主要成分）；
- 丙烯 → **聚丙烯**。
这都是塑料化工的基础反应。本节作了解即可。
:::

## 五、把"加成 + 氧化"的全貌串起来

到这里，烯烃围绕双键的两大类反应（加成、氧化）就讲全了。我们把它们拢成一棵知识树：

:::note{label=本节小结}
**自由基加成：**
1. 条件：$\ce{HBr}$ + **过氧化物**（自由基引发）。中间体是**碳自由基**，特点是**反马氏**。
2. 机理：过氧化物均裂 → 烷氧自由基夺氢生 $\ce{Br·}$ → $\ce{Br·}$ 先加双键、生成更稳定的碳自由基 → 夺氢成产物。
3. 反马氏的根本原因：**先进攻的是溴自由基**（亲电加成里先进攻的是质子），所以取向反转。它与酸催化的马氏加成**互补**。自由基加成**不谈顺反**（碳自由基平面）。

**氧化反应：**
4. **高锰酸钾**——冷稀碱性（弱）：不断键 → **顺式邻二醇**；酸性/加热（强）：**断键** → 羧酸（有氢端）/酮（无氢端）。
5. **臭氧化**（$\ce{O3}$，再 $\ce{Zn/H2O}$）：**断键** → 双键碳有氢成**醛**、无氢成**酮**；是定位双键、做结构鉴定的利器。
6. **环氧化**：不断键 → 在双键上架氧桥，得**环氧化物**。
7. **聚合**：乙烯→聚乙烯、丙烯→聚丙烯，了解。

**两条核心线索：**
- 加成看"谁先进攻、生成什么中间体（碳正离子/环鎓离子/碳自由基）"——决定马氏/反马、顺/反；
- 氧化看"条件温和还是剧烈"——决定断键还是保留，进而决定产物是邻二醇、环氧、还是醛/酮/羧酸。
有机化学反应"一看条件定产物"，这一章把这句话演绎得淋漓尽致。
:::
