# 13.2 羧酸衍生物的结构与反应活性序列

## 一、直觉：为什么酰氯一开瓶就冒白烟

把一瓶乙酰氯打开，潮湿空气里立刻冒出白烟，伴着强烈的刺激性气味——它已经在和空气里的水反应了，生成羧酸和氯化氢。而旁边一瓶酰胺，常温放着稳如泰山，多数还是白色固体。同样都是"羧酸的衍生物"，活性怎么差这么多？

这背后是一条贯穿整章的主线：**酰氯 > 酸酐 > 酯 > 酰胺**。本节就要把这条活性序列从结构层面讲透，并用一个统一的机理把它们的水解、醇解、氨解全部串起来。

## 二、什么是羧酸衍生物：羟基被取代的四兄弟

:::definition{label=羧酸衍生物}
**羧酸衍生物**具体指四类化合物，记住这个固定顺序：**酰卤、酸酐、酯、酰胺**。它们之所以叫"羧酸衍生物"，是因为都可以看成羧酸的**羟基**被不同基团取代：

- **酰卤**：羟基被卤原子取代（实际几乎只有酰氯——酰溴很少，基本无酰碘、酰氟）。
- **酸酐**：羟基被一个酰氧基（带羰基的氧）取代。
- **酯**：羟基被一个烷氧基取代。
- **酰胺**：羟基被一个氨基取代。
:::

:::note{label=和羧酸的真实关系}
它们叫"羧酸衍生物"，但和羧酸其实没太大直接关系——唯一的联系是四类都**可以**通过羧酸制备（虽然实际制备往往另走他法）。把它们统一看成"羧酸羟基被取代"，是为了抓住共同的结构骨架：都保留了 $\ce{R-C(=O)-}$ 这个酰基，只是连在羰基碳上的那个基团（记作离去基团 $L$）不同。
:::

## 三、四类衍生物的物理性质速览

物理性质的差异，已经在暗示活性差异：

| 衍生物 | 沸点（对比羧酸） | 溶解性 | 备注 |
|---|---|---|---|
| **酰卤** | 比羧酸**低**（无法形成分子间氢键） | 不溶于水，**遇水分解** | 相对密度 > 1（卤原子重）；活性极高，开瓶冒白烟，强刺激性气味 |
| **酸酐** | 比羧酸**高**（分子量大） | 微溶于水、易溶于有机溶剂 | 无色液体，有刺激性气味（高级的无味） |
| **酯** | 一般比羧酸**低**，易挥发 | 微溶于水、易溶于有机溶剂 | 有**芳香气味**（成酯实验做成功满屋飘香） |
| **酰胺** | 比羧酸**高** | — | 除甲酰胺外多为固体（白色晶体） |

:::insight{label=酰胺为什么放在最后}
酰胺键是构成生命物质（如蛋白质）的重要结构。这么重要的东西，绝不能一见水就反应——所以酰胺的活性最低、最稳定，序列里排在最后。从结构看：酰胺氮上的孤对电子与羰基发生很好的 $\pi$-$\pi$ 共轭，使 C—N 键带上一定的双键性质，难以打断。
:::

## 四、活性序列的电子学根源

四类衍生物的反应活性位点都是**羰基碳**。羰基碳要被反应，通常是被**亲核试剂**进攻，所以关键看羰基碳的**电正性**：电正性越大，越容易被亲核进攻。

而羰基碳电正性的大小，由旁边那个 $L$ 基团决定（酰基那半边都一样）：

:::derivation{label=活性序列的电子推导}
- **酰氯**：旁边是卤原子，**强吸电子**，把羰基碳拉得很正 → 最易被进攻 → **活性最高**，一见水就反应。
- **酸酐**：旁边是酰氧基（饱和氧 + 一个羰基），氧电负性大、有吸电子性，但整体吸电子没有氯强 → **排第二**。
- **酯**：旁边是烷氧基，有一定吸电子效应，但不如带羰基的酰氧基 → **活性再降**。
- **酰胺**：旁边是氨基，主要表现为 $\pi$-$\pi$ 共轭的**给电子效应**，把电子推向羰基碳、降低其电正性 → **活性最低、最稳定**。

故反应活性：$$\ce{酰氯 > 酸酐 > 酯 > 酰胺}$$
:::

## 五、统一机理：亲核加成-消除

四类衍生物所有的"××解"反应，本质都是**同一个机理**——亲核加成-消除。这正是上一节酯化反应里讲过的那套机理的推广。

:::theorem{label=羧酸衍生物的亲核加成-消除通式}
设衍生物为 $\ce{R-C(=O)-L}$（$L$ 为离去基团），亲核试剂为 $\ce{Nu}$：

1. **亲核加成**：$\ce{Nu}$ 进攻羰基碳，碳氧双键打开，氧变成氧负，形成四面体中间体（一步完成，进攻上去即算加成）。
2. **消除**：氧负回来重新形成双键，把原来的 $L$ 基团赶走（离去）。

$$\ce{R-C(=O)-L + Nu -> R-C(=O)-Nu + L}$$
:::

整个过程有两个关键关卡，它们恰好**指向同一个结论**：

:::insight{label=两步都偏向同一序列}
**第一关——亲核加成**：由羰基碳电正性决定。$L$ 的给电子能力 氨基 > 烷氧基 > 酰氧基 > 氯，反过来吸电子能力 氯 > 酰氧基 > 烷氧基 > 氨基，所以加成的难易是 酰氯 > 酸酐 > 酯。

**第二关——消除（$L$ 离去）**：$L$ 越容易离去、离去后越稳定，越好。离去能力与 $L$ 的碱性强弱有关——**碱性越弱越容易离去**。而碱性大小排序与上面一致：氯的共轭碱（氯离子）碱性最弱、最易离去。

两关结论一致：**酰氯活性最高 → 酸酐 → 酯 → 酰胺**。酰胺基本可视为惰性、很难反应（特定强条件下仍可反应）。
:::

:::pitfall{label=别把加成-消除拆成两个无关的反应}
亲核加成和消除是连在一起的一套机理，不是先做完一个加成产物、再单独搞一个消除。氧负"回来变双键"的同时就把 $L$ 顶走了。判断产物时，关键是认准：**谁进攻上去（新连上的 Nu 不走）、谁被赶走（原来的 L 离去）**。
:::

## 六、反应一：水解——全员参加，统一生成羧酸

:::theorem{label=水解反应}
所有羧酸衍生物都能水解，水分子的氧作为亲核试剂进攻羰基碳。水解产物高度一致：**所有衍生物水解都生成相应的羧酸**，只是另一分子各不相同：

- 酰氯水解 → 羧酸 + **氯化氢**
- 酸酐水解 → 羧酸 + **羧酸**
- 酯水解 → 羧酸 + **醇**
- 酰胺水解 → 羧酸 + **胺（氨）**

水解活性序列：$$\ce{酰氯 > 酸酐 > 酯 > 酰胺}$$
:::

这个序列在实验条件上看得很直观：

:::insight{label=水解条件的难易梯度}
- **酰氯**：不需加热，自己就反应（所以遇水分解、开瓶冒白烟）。
- **酸酐**：基本也不需加热。
- **酯**：水解需要一定的催化。
- **酰胺**：需要更强的催化条件。

活性越低，越需要苛刻的条件去"逼"它反应——这正是电子效应活性序列的实验体现。
:::

## 七、反应二：醇解——生成酯（酰胺除外）

羧酸衍生物与醇发生**醇解**。由于酰胺已经比较惰性，**醇解里只有酰氯、酸酐、酯参加**，酰胺不参与。醇解产物都是**酯**。

机理照旧：醇羟基氧的孤对电子作为亲核端进攻羰基碳，先亲核加成、再消除掉离去基团 $L$，生成酯。这个酯一半来自衍生物自带的酰基，另一半来自醇。

:::example{label=乙酰氯与乙醇的醇解全过程}
乙酰氯 + 乙醇：

1. **亲核加成**：乙醇的羟基氧进攻羰基碳（碳正、氧负），进攻上去、双键打开，加成一步完成。
2. **消除**：氧负要回来变双键，碳上此刻同时连着新进来的醇氧和原来的氯。新连上的醇氧不走，于是**氯带着一对电子离开**——消除完成，生成乙酸乙酯。

换成酸酐，离开的就是整个酰氧基；换成酯，离开的就是烷氧基。
:::

醇解后产物都是酯，另一分子各异：

| 醇解反应 | 主产物 | 另一分子 |
|---|---|---|
| 酰氯醇解 | 酯 | 氯化氢 |
| 酸酐醇解 | 酯 | 羧酸 |
| 酯醇解 | 酯 | 另一分子醇 |
| 酰胺 | **不发生醇解** | — |

### 醇解的合成应用

:::example{label=阿司匹林、普鲁卡因与酯交换}
- **阿司匹林**：把苯酚看作"醇"，苯酚羟基进攻乙酸酐，经亲核加成-消除把酚羟基酯化，产物即阿司匹林。本质是酸酐的一个醇解反应。
- **普鲁卡因**：一个酯与羟基发生醇解，氧进攻把这部分接上去、$\ce{C2H5O-}$ 离开，生成普鲁卡因（一种药）。
- **酯交换反应**：两个不同的酯，例如苯环上连的乙酸酯与苯甲酸甲酯，和甲醇醇解时——苯甲酸甲酯那边"解来解去还是甲酯"相当于不变；乙酸酯那边解出来把对应的醇游离出来，实现酯之间的交换。
:::

## 八、反应三：氨解——生成酰胺（酰胺除外）

第三类是**氨解**。机理可由通式直接写出：进攻的亲核端由醇解里的"羟基氧"换成**氨基里的氮**（氨基氮的孤对电子）。氨分子就用氮直接进攻；有机胺则是氮上连了烃基后，孤对电子进攻；之后照样消除。

:::theorem{label=氨解反应}
氨解 = 氨/胺的氮作亲核试剂，对羧酸衍生物做亲核加成-消除，产物为**酰胺**，另一分子各不相同。由于酰胺连醇解都难发生，氨解就更难，所以**酰胺仍不参与**氨解。氨解是由羧酸衍生物制备酰胺的一种方法。
:::

:::example{label=氨解的合成应用}
- 酰氯 + 二级胺，氨解生成一个酰胺。
- 酸酐氨解可制得一种治感冒的药（药名待核）。
:::

## 九、三类反应统一对照

:::insight{label=水解/醇解/氨解一张表看懂}
全部都是**亲核加成-消除**，区别只在"谁来进攻"和"生成什么"：

| 反应 | 亲核试剂 | 产物主体 | 参与者 |
|---|---|---|---|
| **水解** | 水的氧 | 羧酸 | 四类全参加 |
| **醇解** | 醇的羟基氧 | 酯 | 酰氯、酸酐、酯（酰胺不参与） |
| **氨解** | 氨/胺的氮 | 酰胺 | 酰氯、酸酐、酯（酰胺不参与） |

所以："水解制羧酸、醇解制酯、氨解制酰胺"，而酰胺由于最惰性，除了水解外基本不参加。
:::

:::pitfall{label=三个易混点}
1. **水解人人都生成羧酸**，但"另一分子"不同（HCl / 羧酸 / 醇 / 胺）——别只记产物主体。
2. **醇解和氨解里酰胺都不参与**（它太惰性），只有水解才四类全到齐。
3. **离去基团认错就全错**：醇解时新进来的醇氧不走，离开的是原来的 $L$（酰氯→氯、酸酐→酰氧基、酯→烷氧基）。
:::

## 十、本节小结

:::note{label=本节小结}
1. **四兄弟**：酰卤、酸酐、酯、酰胺，都看成羧酸羟基被取代（卤原子 / 酰氧基 / 烷氧基 / 氨基）。
2. **活性序列**：$\ce{酰氯 > 酸酐 > 酯 > 酰胺}$。根源是 $L$ 基团对羰基碳电正性的调控——氯强吸电子使碳最正、活性最高；氨基 $\pi$-$\pi$ 给电子使碳电正性最低、酰胺最惰性最稳定。
3. **统一机理**：亲核加成-消除。两个关卡（加成看碳电正性、消除看 $L$ 离去/碱性）结论一致，共同支撑同一序列；$L$ 碱性越弱越易离去。
4. **水解**：四类全参加，统一生成羧酸；活性序列同上，条件上酰氯/酸酐不需加热、酯需催化、酰胺需强催化。
5. **醇解**：生成酯，酰胺不参与；应用有阿司匹林（酸酐醇解）、普鲁卡因、酯交换。
6. **氨解**：生成酰胺，酰胺不参与，更难发生。
7. 一句话收口：**水解制羧酸、醇解制酯、氨解制酰胺，本质全是亲核加成-消除**。
:::
