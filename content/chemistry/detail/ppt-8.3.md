:::insight{label=从"如何画出手性分子的三维结构"说起}
手性分子的关键在于四个基团在三维空间中的排列。但纸面是二维的，怎么表示三维关系？化学家发明了多种方法：

1. **楔形式**：用楔形键（实心 ▲）表示伸向纸面前方，虚线键（▼）表示伸向纸面后方，普通键（—）表示在纸面内
2. **Fischer 投影式**：把三维分子"投影"到纸面上，用横竖线表示前后关系
3. **D/L 标记法**：早期人为规定的相对构型，以甘油醛为标准
4. **R/S 标记法**：基于"顺序规则"的绝对构型，是现代有机化学的标准方法

本节我们学习这四种方法，重点掌握 R/S 标记法——它是国际通用的构型标记系统，能唯一确定手性分子的空间排列。
:::

## Fischer 投影式

:::definition{label=Fischer 投影式}
**Fischer 投影式**：将手性分子的三维结构投影到纸面上的表示方法。

**规则**：
1. **碳链尽量在垂直方向**，氧化态高的基团在上，氧化态低的基团在下
2. **垂直方向**的碳链应伸向**纸面后方**（远离观察者）
3. **水平方向**的基团伸向**纸面前方**（朝向观察者）
4. 用**横线与竖线的交叉点**表示手性碳原子

**转换关系**：
- 楔形式中，伸向纸面前方的键 → Fischer 式中的水平键
- 楔形式中，伸向纸面后方的键 → Fischer 式中的垂直键
:::

:::theorem{label=Fischer 投影式的操作规则}
Fischer 投影式有一些重要的操作规则：

1. **可以在纸面上旋转 180°** → 得到相同分子（构型不变）
2. **不能在纸面上旋转 90° 或 90° 的奇数倍** → 得到原分子的**对映异构体**
3. **不能向纸面外旋转 180°**（即翻面）→ 得到原分子的**对映异构体**
4. **交换任意两个基团** → 得到对映异构体
5. **交换三个基团**（相当于交换两次）→ 构型不变

**核心**：Fischer 投影式只能在纸面上旋转 180°，其他操作都会改变构型。
:::

:::insight{label=Fischer投影式的"前后"约定}
Fischer 投影式的关键约定是：**水平键朝前，垂直键朝后**。这就像你站在十字路口，水平方向（左右）的路朝你走来，垂直方向（上下）的路离你远去。

记住这个约定，就能从 Fischer 式想象出三维结构：水平键的基团"凸"出纸面，垂直键的基团"凹"进纸面。
:::
::figure{src="/images/chemistry/svg/ch08/fischer-projection-convention.svg" caption="Fischer投影式约定与楔形式转换——水平键朝前，垂直键朝后" alt="甘油醛楔形式转Fischer投影：水平键实心楔朝前，垂直键虚线朝后，操作规则"}

:::example{label=甘油醛的Fischer投影式}
**题目**：写出 D-甘油醛 $\ce{HOCH2-CH(OH)-CHO}$ 的 Fischer 投影式。

**解**：

D-甘油醛有一个手性碳（$\ce{C2}$），连有 $\ce{-CHO}$、$\ce{-CH2OH}$、$\ce{-OH}$、$\ce{-H}$ 四个基团。

按 Fischer 投影式规则：
1. 碳链垂直：$\ce{-CHO}$ 在上（氧化态高），$\ce{-CH2OH}$ 在下（氧化态低）
2. 水平键朝前：$\ce{-OH}$ 和 $\ce{-H}$ 在水平方向

D-甘油醛的 Fischer 投影式：

```
   CHO
    |
H — C — OH
    |
  CH2OH
```

（$\ce{-OH}$ 在右侧，$\ce{-H}$ 在左侧）

**D 型约定**：$\ce{-OH}$ 在右侧为 D 型，$\ce{-OH}$ 在左侧为 L 型。
:::

:::pitfall{label=Fischer投影式旋转错误}
**常见错误**：把 Fischer 投影式在纸面上旋转 90°，以为构型不变。

**正确做法**：
- **可以**在纸面上旋转 **180°** → 构型不变
- **不能**在纸面上旋转 **90°**（或 90° 的奇数倍）→ 得到对映异构体
- **不能**翻面（向纸面外旋转 180°）→ 得到对映异构体

**记忆**：Fischer 式只能"原地转半圈"（180°），不能"转四分之一圈"（90°），也不能"翻面"。
:::

## D/L 构型标记法——相对构型

:::definition{label=D/L构型标记法}
**D/L 构型标记法**：早期（Fischer 时代）在未知绝对构型的情况下，人为规定的**相对构型**。

**约定**：
- 人为规定右旋 $(+)$-甘油醛为 **D 构型**，指定其 Fischer 投影式中 $\ce{-OH}$ 在右侧
- 相应地，其对映异构体 $(-)$-甘油醛为 **L 构型**，$\ce{-OH}$ 在左侧
- 其他化合物的构型通过与甘油醛的化学反应关联确定

**注意**：D/L 是**相对构型**，与旋光方向 $(+)$/$(-)$ **没有必然联系**。例如 D-甘油醛是 $(+)$，但 D-乳酸是 $(-)$。
:::

:::theorem{label=D/L标记法的局限性}
D/L 标记法的局限：
1. **相对构型**：只能通过与甘油醛的化学关联确定，不能直接反映分子的真实空间排列
2. **适用范围有限**：主要用于糖类和氨基酸，对其他化合物不方便
3. **不能处理多手性中心**：对于含多个手性碳的分子，D/L 标记不明确

**现代有机化学**主要使用 R/S 标记法（绝对构型），D/L 标记法在糖类和氨基酸化学中仍有使用。
:::

:::insight{label=D/L与d/l的区别}
注意区分：
- **D/L**（大写）：构型标记，基于甘油醛的相对构型
- **d/l**（小写）：旋光方向，d = 右旋（dextrorotatory），l = 左旋（levorotatory）
- **$(+)$/$(-)$**：旋光方向，$(+)$ = 右旋，$(-)$ = 左旋

D/L 与 $(+)$/$(-)$ 没有必然联系！例如：
- D-(+)-甘油醛：D 构型，右旋
- D-(-)-乳酸：D 构型，左旋
- L-(+)-乳酸：L 构型，右旋
:::

## R/S 构型标记法——绝对构型

:::definition{label=R/S构型标记法}
**R/S 构型标记法**（Cahn-Ingold-Prelog，CIP 规则）：根据手性碳原子上四个不同原子或基团在"顺序规则"中的先后顺序确定的**绝对构型**，能真实反映分子的空间排列。

**R**（rectus，拉丁语"右"）：顺时针
**S**（sinister，拉丁语"左"）：逆时针
:::

:::theorem{label=顺序规则}
**顺序规则**（CIP 规则）用于排列手性碳上四个基团的优先级：

**规则 1**：单原子取代基按**原子序数**大小排列，原子序数大，顺序在前；原子序数小，顺序在后。

$$
\ce{I > Br > Cl > S > P > F > O > N > C > D > H}
$$

**规则 2**：第一个原子相同，比较与它相连的**其他原子**——有大则先。逐层比较，直到分出大小。

例如 $\ce{-CH3}$ vs $\ce{-CH2CH3}$：第一个原子都是 C，比较 C 上连的原子：
- $\ce{-CH3}$：C 连 (H, H, H)
- $\ce{-CH2CH3}$：C 连 (C, H, H)
- C > H，所以 $\ce{-CH2CH3 > -CH3}$

**规则 3**：含双键、三键的基团，可认为连有**两个或三个相同**的原子。

例如 $\ce{-CHO}$ 中的 C 连 (O, O, H)（双键 O 算两个 O）；$\ce{-CN}$ 中的 C 连 (N, N, N)（三键 N 算三个 N）

**规则 4**：若参与原子的键不到四个，补充适量**原子序数为 0**的假想原子。

例如 $\ce{-NH2}$ 中的 N 连 (C, H, 0)（N 只有 3 个键，补一个 0）
:::

:::theorem{label=R/S构型的确定法则}
**R/S 构型确定步骤**：

1. 将手性碳上四个基团按**顺序规则**排列大小，得到优先级 1 > 2 > 3 > 4
2. 把**最小的基团**（优先级 4，通常是 $\ce{-H}$）放在**离眼睛最远**的位置（伸向纸面后方）
3. 观察其他三个基团（1, 2, 3）按**由大到小**的旋转方向：
   - **顺时针** → **R 构型**
   - **逆时针** → **S 构型**
:::

:::insight{label=R/S确定的"方向盘"模型**
想象你坐在手性碳上，最小的基团（H）在"后方"（你背后），其他三个基团在前方。从最大的基团（1）开始，看向次大的（2），再看向第三大的（3）：
- 如果是顺时针（像方向盘向右转）→ R（rectus，右）
- 如果是逆时针（像方向盘向左转）→ S（sinister，左）

**关键**：必须把最小基团放在"后方"才能直接判断。如果最小基团在"前方"，判断结果要**反转**（顺时针变 S，逆时针变 R）。
:::
::figure{src="/images/chemistry/svg/ch08/rs-configuration-steering-wheel.svg" caption="R/S构型判断方向盘模型——H在后顺时针为R，H在前需反转" alt="H在后时1→2→3顺时针为R(绿)，逆时针为S；H在前时结果反转；CIP优先级规则"}

:::example{label=甘油醛的R/S构型}
**题目**：D-甘油醛的 Fischer 投影式中 $\ce{-OH}$ 在右侧，确定其 R/S 构型。

**解**：

D-甘油醛 $\ce{HOCH2-CH(OH)-CHO}$ 的手性碳 $\ce{C2}$ 连有：
- $\ce{-OH}$（O，原子序数 8）
- $\ce{-CHO}$（C 连 (O, O, H)）
- $\ce{-CH2OH}$（C 连 (O, H, H)）
- $\ce{-H}$（H，原子序数 1）

**顺序规则排序**：
1. $\ce{-OH}$（O 直接连）→ 优先级 1
2. $\ce{-CHO}$（C 连 (O, O, H)）→ 优先级 2（双键 O 算两个 O）
3. $\ce{-CH2OH}$（C 连 (O, H, H)）→ 优先级 3
4. $\ce{-H}$ → 优先级 4（最小）

**Fischer 投影式中**：
- $\ce{-CHO}$ 在上（垂直，朝后）
- $\ce{-CH2OH}$ 在下（垂直，朝后）
- $\ce{-OH}$ 在右（水平，朝前）
- $\ce{-H}$ 在左（水平，朝前）

**判断**：最小基团 $\ce{-H}$ 在水平方向（朝前），需要反转判断结果。

观察 1 → 2 → 3（$\ce{-OH}$ → $\ce{-CHO}$ → $\ce{-CH2OH}$）：
- $\ce{-OH}$ 在右，$\ce{-CHO}$ 在上，$\ce{-CH2OH}$ 在下
- 从右 → 上 → 下，是**逆时针**
- 但 $\ce{-H}$ 朝前，需要反转 → **顺时针** → **R 构型**

**结论**：D-甘油醛 = **(R)-甘油醛** = D-(+)-甘油醛
:::

:::example{label=苯基乙胺的R/S构型}
**题目**：确定 S-(-)-苯基乙胺 $\ce{C6H5-CH(NH2)-CH3}$ 的构型。

**解**：

手性碳 $\ce{C}$ 连有：
- $\ce{-NH2}$（N，原子序数 7）
- $\ce{-C6H5}$（C 连 (C, C, H)）
- $\ce{-CH3}$（C 连 (H, H, H)）
- $\ce{-H}$（H，原子序数 1）

**顺序规则排序**：
1. $\ce{-NH2}$（N 直接连）→ 优先级 1
2. $\ce{-C6H5}$（C 连 (C, C, H)）→ 优先级 2
3. $\ce{-CH3}$（C 连 (H, H, H)）→ 优先级 3
4. $\ce{-H}$ → 优先级 4（最小）

**S 构型**：最小基团 $\ce{-H}$ 在后方时，1 → 2 → 3 为**逆时针**。
:::

:::example{label=2,3-二羟基丁二酸的构型}
**题目**：写出 (2S,3S)-2,3-二羟基丁二酸（即 L-(+)-酒石酸）的构型。

**解**：

2,3-二羟基丁二酸 $\ce{HOOC-CH(OH)-CH(OH)-COOH}$ 有两个手性碳 $\ce{C2}$ 和 $\ce{C3}$。

对 $\ce{C2}$：
- 连有 $\ce{-OH}$、$\ce{-COOH}$、$\ce{-CH(OH)COOH}$、$\ce{-H}$
- 顺序：$\ce{-OH}$ > $\ce{-COOH}$ > $\ce{-CH(OH)COOH}$ > $\ce{-H}$
- (S) 构型：1 → 2 → 3 逆时针（H 在后）

对 $\ce{C3}$：同理为 (S) 构型。

所以 (2S,3S)-酒石酸是 L-(+)-酒石酸（右旋）。

**注意**：D-(-)-酒石酸是 (2R,3R)，meso-酒石酸是 (2R,3S)。
:::

:::pitfall{label=Fischer式中R/S判断的"反转"陷阱}
**常见错误**：在 Fischer 投影式中直接判断 R/S，忘记考虑最小基团的位置。

**正确做法**：
- 如果最小基团（H）在**垂直方向**（朝后）→ 直接判断，1→2→3 顺时针为 R，逆时针为 S
- 如果最小基团（H）在**水平方向**（朝前）→ 判断结果**反转**，顺时针为 S，逆时针为 R

**口诀**：H 在竖直直接判，H 在水平要反转。
:::

:::theorem{label=生物活性与构型}
手性分子的生物活性与构型密切相关：

**青霉素**（penicillin）：含有 3 个手性碳，活性构型为 (2S, 5R, 6R)。其他构型无活性或活性大幅降低。

**氯霉素**（chloramphenicol）：含有 2 个手性碳，活性构型为 (1R, 2R)。其他构型无抗菌活性。

**启示**：药物分子必须与生物受体（手性环境）匹配，"锁与钥匙"关系要求特定的立体构型。这就是为什么现代药物研发重视手性合成——只生产有效构型，避免无效或有害构型。
:::

:::note{label=本节小结}
- **Fischer 投影式**：水平键朝前，垂直键朝后；可在纸面旋转 180°，不能旋转 90° 或翻面
- **D/L 标记法**：相对构型，以甘油醛为标准（$\ce{-OH}$ 在右为 D，在左为 L）；与旋光方向无关
- **R/S 标记法**（CIP 规则）：绝对构型，基于顺序规则
  - **顺序规则**：原子序数 > 逐层比较 > 重键重复 > 补假想原子
  - **判断**：最小基团在后方，1→2→3 顺时针为 R，逆时针为 S
  - **Fischer 式中技巧**：H 在竖直直接判，H 在水平要反转
- **D-甘油醛 = (R)-甘油醛 = D-(+)-甘油醛**（D/L 与 R/S 的对应关系，但不是普遍规律）
- **生物活性**：药物分子的构型必须与受体匹配，如青霉素 (2S,5R,6R)、氯霉素 (1R,2R)
- 至此，第八章对映异构的内容学习完毕
:::
