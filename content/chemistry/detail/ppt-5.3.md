:::insight{label=从"经典共轭"到"边缘效应"}
前两节我们学了诱导效应和共轭效应——这是有机化学中最经典的两种电子效应。但还有一些"边缘"效应，虽然比经典效应弱，却在许多反应中起关键作用：

1. **超共轭效应**：σ 键（C—H 或 C—C）与 π 键或 p 轨道的"部分重叠"。它解释了为什么自由基和碳正离子的稳定性顺序是 $\ce{3^{\circ} > 2^{\circ} > 1^{\circ} > CH3}$——这不是共轭效应，而是 σ-p 超共轭！
2. **动态诱导极化效应**：非极性分子在极性分子作用下产生瞬时偶极，改变反应性。它解释了为什么溶剂极性会改变反应产物。
3. **立体效应**：基团的空间位阻影响分子结构和反应性。它解释了为什么大基团会改变构象、减慢反应速率。

本节我们就来学习这三种"精细"效应，补全取代基效应的完整图景。
:::

## 超共轭效应

:::definition{label=超共轭效应}
**超共轭效应**（hyperconjugation），又称 **σ-π** 或 **σ-p** 共轭，主要是指由一个 C—H 键的 σ 电子云与邻近的 π 电子云或空 2p 轨道之间相互交盖而产生电子部分离域的一种共轭现象。

超共轭效应分两类：
- **σ-π 超共轭**：C—H σ 键与 π 键（如 $\ce{C=C}$、$\ce{C=O}$）的部分重叠
- **σ-p 超共轭**：C—H σ 键与 p 轨道（如自由基的半满 p、碳正离子的空 p）的部分重叠
:::

### σ-π 超共轭

:::theorem{label=σ-π 超共轭}
**σ-π 超共轭**：丙烯 $\ce{CH3-CH=CH2}$ 中，甲基的 C—H σ 键与 $\ce{C=C}$ π 键发生部分重叠，使 σ 电子云部分离域到 π 体系。

效应：
- 使 π 键电子云向另一端偏移
- 使 σ 键电子云部分离域
- 整体上使体系能量降低，分子稳定

**α-H 的酸性**：羰基化合物（醛、酮）的 α-H 有一定酸性，原因之一就是 C—H 与 $\ce{C=O}$ 之间有 σ-π 超共轭效应，使 C—H 键变弱，H 更易离去。

$$
\ce{R-CH2-CO-R' ->[\text{碱}] R-CH^--CO-R' + H+}
$$

形成的碳负离子也可以通过 σ-π 超共轭（或更准确地说，p-π 共轭）稳定。
:::

:::insight{label=σ-π 超共轭与 α-H 酸性**
为什么醛、酮的 α-H 比普通烷烃的 H 酸性强得多？

- 普通烷烃 $\ce{R-CH3}$：$\text{p}K_a \approx 50$
- 醛/酮 α-H $\ce{R-CH2-CHO}$：$\text{p}K_a \approx 17$

原因：
1. **σ-π 超共轭**：C—H σ 键与 $\ce{C=O}$ π 键部分重叠，使 C—H 键变弱
2. **碳负离子稳定**：失去 α-H 后形成的碳负离子 $\ce{R-CH^- -CHO}$，其 p 轨道与 $\ce{C=O}$ π 体系共轭（p-π 共轭），负电荷离域到 O 上 → 稳定

这两个因素共同使 α-H 酸性大大增强。这是醛、酮能发生羟醛缩合、烷基化等反应的基础。
:::

### σ-p 超共轭——自由基与碳正离子的稳定

:::theorem{label=σ-p 超共轭与自由基稳定性}
**σ-p 超共轭**：碳自由基的 p 轨道（含未成对电子）可与相邻碳上的 C—H σ 轨道发生部分重叠，使 σ 键的电子云部分离域。这种作用使电荷分散，体系变得稳定。

自由基稳定性顺序：

$$
\ce{(CH3)3C^. > (CH3)2CH^. > CH3CH2^. > ^.CH3}
$$

即：**叔自由基 > 仲自由基 > 伯自由基 > 甲基自由基**

**原因**：σ-p 超共轭的数目决定稳定性：
- 甲基自由基 $\ce{^.CH3}$：0 个相邻 C—H → 0 个超共轭 → 最不稳定
- 乙基自由基 $\ce{CH3-^.CH2}$：3 个相邻 C—H → 3 个超共轭
- 异丙基自由基 $\ce{(CH3)2^.CH}$：6 个相邻 C—H → 6 个超共轭
- 叔丁基自由基 $\ce{(CH3)3^.C}$：9 个相邻 C—H → 9 个超共轭 → 最稳定
:::

::figure{src="/images/chemistry/svg/ch05/radical-hyperconjugation.svg" caption="四种自由基σ-p超共轭稳定性：甲基(0个)→伯(3个)→仲(6个)→叔(9个C-H)，超共轭越多越稳定" alt="四种自由基并排：甲基自由基0个超共轭，伯自由基3个，仲自由基6个，叔自由基9个C-H与p轨道重叠，稳定性依次升高"}

:::insight{label=数 C—H 法判断自由基/碳正离子稳定性**
σ-p 超共轭的数目 = 相邻 C—H 键的数目。所以快速判断自由基或碳正离子稳定性的方法：**数相邻 C—H 键**。

- $\ce{^.CH3}$：相邻碳数 = 0 → 0 个 C—H → 0 个超共轭
- $\ce{CH3-^.CH2}$：相邻 $\ce{CH3}$ 有 3 个 C—H → 3 个超共轭
- $\ce{(CH3)2^.CH}$：相邻 2 个 $\ce{CH3}$，共 6 个 C—H → 6 个超共轭
- $\ce{(CH3)3^.C}$：相邻 3 个 $\ce{CH3}$，共 9 个 C—H → 9 个超共轭

碳正离子的稳定性顺序与自由基相同（$\ce{3^{\circ} > 2^{\circ} > 1^{\circ} > CH3+}$），原因也是 σ-p 超共轭（空 p 轨道与相邻 C—H σ 键的部分重叠）。
:::

:::pitfall{label=把"超共轭"和"共轭"混淆**
- **共轭**（conjugation）：p 轨道与 π 轨道，或 π 与 π 之间的离域（如 1,3-丁二烯、苯胺）
- **超共轭**（hyperconjugation）：σ 轨道（C—H 或 C—C）与 p 轨道或 π 轨道的部分重叠

超共轭比共轭**弱**得多，因为它涉及"满的"σ 轨道与"空的/半满的"p 轨道之间的相互作用，重叠程度小。但累加起来仍能显著稳定自由基或碳正离子。

考试时不要把"超共轭"简写成"共轭"，两者是不同的概念。
:::

:::example{label=自由基稳定性比较}
**题目**：比较 $\ce{^.CH3}$、$\ce{CH3CH2^.}$、$\ce{(CH3)2CH^.}$、$\ce{(CH3)3C^.}$ 的稳定性。

**分析**：用 σ-p 超共轭数目判断。

**解**：稳定性从高到低：

$$
\ce{(CH3)3C^. > (CH3)2CH^. > CH3CH2^. > ^.CH3}
$$

- $\ce{(CH3)3C^.}$：9 个相邻 C—H → 9 个 σ-p 超共轭 → 最稳定
- $\ce{(CH3)2CH^.}$：6 个相邻 C—H → 6 个 σ-p 超共轭
- $\ce{CH3CH2^.}$：3 个相邻 C—H → 3 个 σ-p 超共轭
- $\ce{^.CH3}$：0 个相邻 C—H → 0 个 σ-p 超共轭 → 最不稳定

这就是为什么烷烃卤代时，叔氢最容易被夺取（生成最稳定的叔自由基），符合 $\ce{3^{\circ}H > 2^{\circ}H > 1^{\circ}H > CH4}$ 的活性顺序。
:::

## 动态诱导极化效应

:::definition{label=动态诱导极化效应}
**动态诱导极化效应**：当非极性分子与具有永久偶极矩的极性分子相互接近时，会发生诱导极化，形成瞬时诱导偶极。分子间的诱导极化产生的瞬时偶极矩，会引起原来不带电荷的反应中心原子的反应性的显著变化，这种作用称为动态诱导极化效应。

简单说：**极性环境（极性溶剂或极性试剂）可以使非极性分子"暂时极化"，改变其反应性**。
:::

:::theorem{label=动态诱导极化的典型例子}
苯酚与 $\ce{Br2}$ 的反应：

**弱极性溶剂（HOAc）**：
$$
\ce{C6H5OH + Br2 ->[\text{HOAc}] C6H4BrOH + HBr}
$$
只发生**一取代**（生成对溴苯酚）

**强极性溶剂（H2O）**：
$$
\ce{C6H5OH + 3Br2 ->[\text{H2O}] C6H2Br3OH + 3HBr}
$$
发生**三取代**（生成 2,4,6-三溴苯酚）

**原因**：在强极性溶剂水中，非极性的 $\ce{Br2}$ 分子发生动态诱导极化，$\ce{Br-Br}$ 键极化成 $\ce{Br^{\delta+}-Br^{\delta-}}$，使 $\ce{Br^{\delta+}}$ 的亲电性增强，反应活性提高，从而发生三取代。
:::

:::insight{label=溶剂极性如何影响反应**
动态诱导极化效应说明：**溶剂不仅是"背景"，它直接参与调控反应**。

- 弱极性溶剂（HOAc、$\ce{CCl4}$）：$\ce{Br2}$ 极化程度低，反应活性低，只能一取代
- 强极性溶剂（$\ce{H2O}$）：$\ce{Br2}$ 被极化，反应活性高，发生三取代

这在有机合成中很有用：通过选择溶剂，可以控制反应的程度和产物。例如苯酚在水中溴化是定性检测苯酚的经典方法（生成白色 2,4,6-三溴苯酚沉淀）。
:::

:::pitfall{label=把"动态诱导极化"和"诱导效应"混淆**
- **诱导效应**：分子**内部**固有的电负性差异引起的电子云偏移（静态）
- **动态诱导极化**：分子**外部**极性环境引起的瞬时偶极（动态）

两者都涉及电子云偏移，但诱因不同：诱导效应是分子自身的性质，动态诱导极化是环境的影响。考试时不要把"溶剂使 $\ce{Br2}$ 极化"说成"诱导效应"，应说"动态诱导极化效应"。
:::

## 立体效应

:::definition{label=立体效应}
**立体效应**（steric effect）：基团的空间位阻引起的效应，主要因素是基团的**空间位阻**。

立体效应引起：
1. **分子结构的变化**：大基团迫使分子采取特定构象
2. **分子物理化学性质的变化**：影响沸点、溶解性、反应性
3. **分子稳定构象的变化**：大基团"偏好"位阻小的位置
:::

### 立体效应改变分子构象

:::theorem{label=联苯的立体效应}
**联苯**（biphenyl）：两个苯环通过单键相连。

- **无取代基**：两个苯环**共平面**，相互共轭（共平面时 π 体系重叠最大，能量最低）
- **邻位有大基团**（如 $\ce{-NO2}$、$\ce{-COOH}$）：大基团互相排斥，破坏两个苯环的共平面构象，使两个苯环**正交**（互相垂直），降低大基团的位阻

这就是为什么某些邻位取代联苯能稳定存在为非平面构象——立体效应"战胜"了共轭效应。
:::

::figure{src="/images/chemistry/svg/ch05/biphenyl-steric-vs-conjugation.svg" caption="联苯立体效应 vs 共轭效应博弈：无取代（共平面，π重叠最大）vs 邻位大基团（正交90°，π重叠降低）" alt="左：无取代联苯两环共平面π体系最大重叠；右：邻位有NO2/COOH的联苯两环正交约90°避免位阻，立体效应战胜共轭效应"}

:::insight{label=立体效应 vs 共轭效应的"博弈"**
联苯的例子展示了立体效应和共轭效应的"博弈"：

- **共轭效应**：希望两个苯环共平面（π 体系重叠最大，能量最低）
- **立体效应**：希望两个苯环正交（避免邻位大基团互相排斥）

无取代基时，共轭效应主导 → 共平面
邻位有大基团时，立体效应主导 → 正交

这种"博弈"在有机化学中很常见。例如环己烷的椅式构象也是立体效应（避免 1,3-二轴向相互作用）的体现。
:::

### 立体效应影响反应速率

:::theorem{label=空间位阻与反应速率}
**位阻小 → 反应速率快；位阻大 → 反应速率慢**

空间位阻影响反应速率的机制：
1. **阻碍亲核试剂进攻**：大基团挡住反应中心，亲核试剂难以接近
2. **阻碍过渡态形成**：过渡态需要特定几何构型，大基团可能阻碍
3. **改变反应路径**：位阻大的位置不反应，反应发生在位阻小的位置

例如 $\mathrm{S_N2}$ 反应：
- 伯卤代烃：位阻小，$\mathrm{S_N2}$ 快
- 叔卤代烃：位阻大，$\mathrm{S_N2}$ 几乎不发生（改走 $\mathrm{S_N1}$）
:::

:::example{label=立体效应与反应选择性}
**题目**：某反应中，亲核试剂可以进攻 a 位或 b 位（如下图），a 位位阻小，b 位位阻大。预测主要产物。

**分析**：立体效应使位阻小的位置反应快。

**解**：
- a 位位阻小 → 亲核试剂容易进攻 → 反应速率快 → 主要产物
- b 位位阻大 → 亲核试剂难以接近 → 反应速率慢 → 次要产物

**规律**：当有多个可能的反应位点时，**位阻小的位点反应快，是主要产物**。这是有机合成中"区域选择性"的重要来源。
:::

### 立体效应在合成中的应用

:::theorem{label=立体效应的合成应用}
立体效应在有机合成中有广泛应用：

1. **保护基**：用大基团（如叔丁基）保护特定位置，使反应只发生在其他位置
2. **构象锁定**：叔丁基锁定环己烷构象（见第 4 章）
3. **区域选择性**：大位阻位置不反应，小位阻位置反应
4. **立体选择性**：亲核试剂从位阻小的一侧进攻

例如环己烷上若有叔丁基（锁定 e 键），其他位置的亲核进攻只能从 a 键方向（位阻小的一侧）进行，得到立体选择性产物。
:::

## 取代基效应的综合应用

:::theorem{label=取代基效应总结}
有机化学中的取代基效应包括：

1. **诱导效应（I）**：电负性差异引起，沿 σ 键传递，迅速衰减
2. **共轭效应（C）**：π 电子离域，沿 π 键传递，不衰减（只在共轭体系中）
3. **超共轭效应**：σ 键与 π/p 轨道部分重叠，稳定自由基和碳正离子
4. **动态诱导极化**：极性环境使非极性分子瞬时极化
5. **立体效应**：空间位阻影响构象和反应性

判断基团效应的步骤：
1. 是否在共轭体系中？是 → 考虑 C；否 → 只考虑 I
2. 基团有孤对电子？是 → 可能 +C（如 $\ce{-OH}$、$\ce{-NH2}$、$\ce{-X}$）
3. 基团有 π 键？是 → 可能 -C（如 $\ce{-NO2}$、$\ce{-CN}$、羰基）
4. 总效应 = I + C（共轭体系中）
5. 考虑立体效应（大基团）
:::

:::example{label=综合分析取代基效应}
**题目**：分析 $\ce{-NO2}$、$\ce{-CHO}$、$\ce{-OH}$ 对苯环的电子效应。

**解**：
- **$\ce{-NO2}$**：
  - -I（N 电负性大，吸电子）
  - -C（$\ce{N=O}$ π 键吸电子，与苯环共轭）
  - 总体：**强吸电子**

- **$\ce{-CHO}$**（醛基）：
  - -I（$\ce{C=O}$ 的 O 电负性大，使醛基碳带正电性，吸电子）
  - -C（$\ce{C=O}$ π 键与苯环共轭，吸电子）
  - 总体：**吸电子**

- **$\ce{-OH}$**：
  - -I（O 电负性大，吸电子）
  - +C（O 孤对电子给电子，p-π 共轭）
  - +C > -I
  - 总体：**给电子**

这三种基团的差异完美展示了诱导与共轭的协同与竞争。
:::

:::example{label=判断分子中各碳的电子云密度}
**题目**：判断 $\ce{p-NO2-C6H4-OH}$（对硝基苯酚）中各碳的电子云密度高低。

**分析**：$\ce{-NO2}$ 是吸电子基（-I + -C），$\ce{-OH}$ 是给电子基（+C > -I）。

**解**：
- $\ce{-OH}$ 通过 +C 向苯环给电子 → 邻、对位电子云密度高
- $\ce{-NO2}$ 通过 -C 从苯环吸电子 → 邻、对位电子云密度低
- 综合：$\ce{-OH}$ 的邻位（即 $\ce{-NO2}$ 的间位）电子云密度最高；$\ce{-NO2}$ 的邻位（即 $\ce{-OH}$ 的间位）电子云密度最低

具体到对硝基苯酚：
- C1（连 $\ce{-OH}$）：电子云密度较高（O 给电子）
- C4（连 $\ce{-NO2}$）：电子云密度较低（N 吸电子）
- C2、C6（$\ce{-OH}$ 邻位 = $\ce{-NO2}$ 间位）：电子云密度较高
- C3、C5（$\ce{-OH}$ 间位 = $\ce{-NO2}$ 邻位）：电子云密度较低

亲电取代反应发生在电子云密度最高的位置（C2、C6）。
:::

:::note{label=本节小结}
- **超共轭效应**（σ-π 或 σ-p）：C—H σ 键与 π/p 轨道部分重叠，使电子离域
  - σ-π：使 α-H 酸性增强（如醛、酮 α-H）
  - σ-p：稳定自由基和碳正离子（数相邻 C—H 法）
  - 自由基/碳正离子稳定性：$\ce{3^{\circ} > 2^{\circ} > 1^{\circ} > CH3}$
- **动态诱导极化效应**：极性环境使非极性分子瞬时极化
  - 例：苯酚 + $\ce{Br2}$ 在水中三取代，在 HOAc 中一取代
- **立体效应**：空间位阻影响构象和反应性
  - 大基团破坏共平面（联苯邻位取代）
  - 位阻小 → 反应快；位阻大 → 反应慢
  - 叔丁基可"锁定"环己烷构象
- **取代基效应综合**：诱导 + 共轭 + 超共轭 + 立体，需根据具体情况分析
