# 8.2 亲电取代与傅克反应

## 一、直觉：富电子的苯环招惹"带正电"的试剂

上一节我们得到一个核心结论：苯环有约 124 kJ/mol 的离域能要守护，所以它"难加成、易取代"。这一节就来看苯到底"怎样取代"。

先建立一个直觉。苯环上有 6 个 π 电子，形成了一个高密度的环状 π 电子云，整个苯环是一个**富电子**的体系。富电子的东西自然会吸引缺电子、带正电的对象。于是一个带正电的**亲电试剂**就可以来进攻苯环的 π 电子云——我们把苯的这类取代叫做**亲电取代反应**。

:::definition{label=亲电取代反应}
带正电荷的亲电试剂（或中性试剂带正电的一端）进攻苯环上的 π 电子云，暂时打断这个大 π 键并结合上去；为维持苯环大 π 键的稳定性，苯环最终舍弃一个氢、恢复芳香体系，整体表现为**用亲电试剂取代掉苯环上一个碳氢键的氢**，而不破坏 π 电子体系。这就是亲电取代反应。
:::

严格地说，进攻应当理解为"由电子发起"——是苯环的 π 电子去进攻亲电试剂；当然你也可以理解成亲电试剂去"招惹"苯环，结果是一样的。和我们之前学过的烷烃自由基取代不同，苯环走的是亲电取代这条路。

## 二、机理：两步走，关键在 σ 络合物这个能量高点

我们用 $\ce{E+}$ 代表亲电试剂，看反应是怎么发生的。

**第一步**，亲电试剂与苯环 π 电子云相互作用。以凯库勒式来理解，相当于把一个碳碳双键上的 π 键异裂，去结合亲电试剂。这一步破坏了原本极稳定的大 π 键，生成一个带正电的中间体，叫做 **σ 络合物**。这是整个反应的能量高点，是需要耗能的过程。

**第二步**，σ 络合物面临两条路：

- 一条是正电荷直接结合体系中的负离子（如 $\ce{Br-}$），生成**加成产物**。但这条路生成的产物能量非常高，甚至比反应物还高，在热力学上不被允许——苯环不愿意为了结合一个亲电试剂就把价值 120 多千焦离域能的稳定 π 电子永久破坏掉。
- 另一条是**抛掉一个质子**，重新恢复苯环的共轭大 π 键，得到**取代产物**。

苯环最终选择第二条路。所以整个亲电取代是两步完成的：先生成 σ 络合物（能量高点），再脱去质子完成取代。中间其实"犹豫"了一下——它本可以加成，但为了稳定还是选择了取代。

:::insight{label=取代而非加成的本质}
σ 络合物之后能不能加成，取决于它愿不愿意永久牺牲离域能。脱质子只是暂时失去一个氢、却能赎回整块离域能，是热力学上划算得多的选择。所以苯走取代，烯烃走加成，根子上是"有没有离域能要守护"的差别。
:::

## 三、四种经典亲电取代：卤代、硝化、磺化（傅克留到下文）

苯环的亲电取代形式主要有：卤代、硝化、磺化、傅克烷基化、傅克酰基化。它们的共同点是**先生成一个带正电的真正亲电试剂，再进攻苯环**。先看前三种。

### 1. 卤代

苯与溴、氯等卤素单质的亲电取代，一般要在**路易斯酸**催化下进行。和氯反应常用 $\ce{FeCl3}$，和溴反应用 $\ce{FeBr3}$，它们都是路易斯酸。有时也直接写催化剂为铁 $\ce{Fe}$——因为氯、溴有氧化性，会先把铁氧化成 $\ce{FeCl3}$ 或 $\ce{FeBr3}$，写铁是一样的。

机理上，路易斯酸诱导卤素异裂，生成一个卤正离子和一个卤负离子，由**卤正离子**进攻苯环生成 σ 络合物，再脱质子得取代产物。

$$\ce{C6H6 + Br2 ->[\ce{FeBr3}] C6H5Br + HBr}$$

碘的活性低，要用更强的酸（如硝酸）来诱导才能生成碘苯。所以溴、氯、碘都可发生卤代。

### 2. 硝化

硝化是苯与硝酸反应，把一个硝基引入苯环。真正起进攻作用的亲电试剂是**硝酰正离子 $\ce{NO2+}$**：硝酸在硫酸作用下生成 $\ce{NO2+}$，进攻苯环后生成硝基苯。反应要在硫酸催化下、并稍加温（约 50–60 ℃）进行。

$$\ce{C6H6 ->[HNO3,\ H2SO4][50\text{--}60\ ^\circ C] C6H5NO2}$$

硝化能继续进行，但条件越来越苛刻：一硝基苯继续硝化要用发烟硝酸和浓硫酸，得到间位取代产物（间二硝基苯）；要做到三取代则需要发烟硝酸、发烟硫酸，110 ℃ 反应 5 天。

:::insight{label=为什么硝化越往后越难}
硝基是强吸电子基。亲电取代本质上是亲电反应，吸电子基会把苯环上的 π 电子云吸走一部分，降低环的电子密度。第一个硝基上去后环就"穷"了一点，上第二个就更费劲，上第三个吸得更狠、更苛刻。所以条件越来越苛刻，恰恰是吸电子效应在"致钝"——这一点在下一节讲定位效应时会系统展开。
:::

### 3. 磺化

磺化是苯与硫酸反应，引入一个磺酸基生成苯磺酸，也需要加热。真正的亲电试剂是**三氧化硫 $\ce{SO3}$**，进攻生成 σ 络合物后快速脱质子，最后得到苯磺酸。

$$\ce{C6H6 ->[H2SO4][\Delta] C6H5SO3H}$$

:::note{label=磺化是可逆反应}
磺化是个可逆反应，在一定温度下可以"回去"（脱磺酸基）。这个可逆性后面在控制取代位置（如萘的 α/β 磺化）时是一个很有用的"开关"。
:::

卤代、硝化、磺化分别给苯装上了卤素、硝基、磺酸基。苯本身相对惰性，但通过这些反应等于给苯环装上了官能团，为后续转化打开了可能性。

## 四、傅克反应（一）：烷基化与碳正离子重排

亲电取代里还有一类很重要的反应叫**傅克反应**，分两种：傅克烷基化和傅克酰基化。

### 傅克烷基化反应

傅克烷基化的作用是给苯环装上一个烷基。条件是：苯环与**卤代烷**在 **$\ce{AlCl3}$**（路易斯酸）催化下反应。请牢牢记住这个特征条件——**卤代烷 + $\ce{AlCl3}$**。

机理：$\ce{AlCl3}$ 先把卤代烷中的卤原子拔下来，生成一个**碳正离子**；碳正离子作为亲电试剂进攻苯环，发生取代，生成烷基取代的苯。

$$\ce{C6H6 + CH3CH2Cl ->[\ce{AlCl3}] C6H5CH2CH3 + HCl}$$

:::pitfall{label=有碳正离子就要警惕重排}
只要反应中生成碳正离子，就要立刻想到一种可能——**碳正离子重排**。我们在烯烃亲电加成里见过它，傅克烷基化同样会发生。重排的条件有两条：第一，体系中有碳正离子生成；第二，向相邻碳重排后能生成**更稳定**的碳正离子（一级→二级、二级→三级、一级→三级）。
:::

要不要重排，要看具体结构。乙基碳正离子重排过去还是同样的碳，没必要也不会重排，所以乙基苯就是正常产物。但换成一氯丙烷就不同了：

$$\ce{C6H6 + CH3CH2CH2Cl ->[\ce{AlCl3}] C6H5CH(CH3)2\ (\text{主})\ +\ C6H5CH2CH2CH3\ (\text{少})}$$

理论上一氯丙烷生成正丙基碳正离子，进攻应得正丙苯；但正丙基碳正离子是一级碳正离子，相邻碳上的一个**负氢**带着一对电子迁移过来，就由一级碳正离子变成更稳定的二级（异丙基）碳正离子。这种迁移叫**负氢迁移**。重排后的异丙基碳正离子进攻苯环，主产物是异丙苯，正丙苯只占少数。

:::insight{label=两种重排：甲基迁移与负氢迁移}
碳正离子重排常见两种方式。一种是**甲基迁移**（在烯烃加成里见过），甲基带着一对电子迁过去；一种是**负氢迁移**，一个氢带着一对电子（即一个"负氢"）迁过去。两者都生成更稳定的碳正离子。判断时先看相邻碳重排后能否提升碳正离子级数，能提升就大概率重排。
:::

再看一个更极端的例子：底物是一个一级碳上取代、但相邻接着叔丁基的卤代烷。生成一级碳正离子后，会发生**甲基迁移**，由一级碳正离子直接变成三级碳正离子，稳定性提升极大，所以这种重排一定发生，最终几乎 100% 生成重排产物。

### 烷基化在合成上的局限

傅克烷基化最大的问题就是**容易重排**，常常得不到你想要的产物。比如你想在苯环上引入一个**正丙基**，用一氯丙烷做烷基化，多数产物却是异丙苯。此外傅克烷基化还是个**可逆反应**，并且容易生成多取代产物，选择性不好。

:::pitfall{label=傅克烷基化的三大软肋}
- **重排**：一级、二级碳正离子常重排成更稳定的碳正离子，得不到目标直链烷基。
- **多取代**：引入烷基后苯环被致活（烷基是给电子基），第二个、第三个烷基更容易接着上。
- **可逆**：一定条件下可逆回去。
综合下来，烷基化在合成上往往不是最优路径，需要直链烷基时尤其要小心。
:::

## 五、傅克反应（二）：酰基化与酰基正离子不重排

为了解决烷基化的重排问题，发展出了**傅克酰基化**。

### 酰化试剂：两类羧酸衍生物

酰化试剂一般有两种：**酰氯**（如乙酰氯 $\ce{CH3COCl}$）和**酸酐**（如乙酸酐 $\ce{(CH3CO)2O}$）。把羧酸 $\ce{RCOOH}$ 的羟基换成不同基团，就得到一系列羧酸衍生物：换成卤原子（通常是氯）得**酰卤**，换成烷氧基得**酯**，换成酰氧基得**酸酐**，换成氨基得**酰胺**。所以酰卤、酸酐、酯、酰胺四类都可看作羧酸羟基被取代后的衍生物。这里用作酰化试剂的是酰氯和酸酐。

### 酰基化机理与"不重排"优势

在 $\ce{AlCl3}$ 催化下，酰氯的氯、或酸酐的酰氧基被拔掉，生成**酰基正离子**；酰基正离子作为亲电试剂进攻苯环，发生酰基取代，得到酰化的苯。

$$\ce{C6H6 + CH3COCl ->[\ce{AlCl3}] C6H5COCH3 + HCl}$$

:::insight{label=酰基正离子为什么不重排}
酰基正离子的正电荷与羰基相连，结构上得到羰基的稳定（可写出 $\ce{R-C+=O <-> R-C#O+}$ 的共振，正电由碳、氧共担），本身已经足够稳定，没有重排的动力，所以**酰基化不发生重排**。这正是它相对烷基化的决定性优势。
:::

### 用"酰基化 + 还原"绕开重排做直链烷基苯

酰化后的苯环带一个含氧的羰基，可以进一步用还原方法把羰基还原成 $\ce{CH2}$，相当于多走一步就实现了"烷基化"，且全程不重排。

:::example{label=合成正丙基苯：选哪条路？}
目标：高收率合成正丙基苯。

**劣选——傅克烷基化**：用一氯丙烷 + $\ce{AlCl3}$，正丙基碳正离子会负氢重排成异丙基，主产物是异丙苯，收率大打折扣。

**优选——傅克酰基化 + 还原**：先用**丙酰氯 $\ce{CH3CH2COCl}$** 做酰基化，生成的丙酰基正离子不重排，可以近乎 100% 地得到丙酰化的苯（即苯基丙酮类结构）；再做一步还原，把羰基还原成 $\ce{CH2}$，就 100% 得到目标产物正丙基苯。

$$\ce{C6H6 ->[\ce{CH3CH2COCl},\ \ce{AlCl3}] C6H5COCH2CH3 ->[\text{还原}] C6H5CH2CH2CH3}$$

结论：凡是需要在苯环上引入**直链或易重排的烷基**，都优先走"酰基化 + 还原"两步，用酰基正离子的不重排来保证产物正确。
:::

### 酰化试剂对水敏感

酰氯和酸酐对水都非常敏感。酸酐相当于两分子羧酸脱水形成（乙酸酐相当于两分子乙酸脱水），一旦见水就极易水解回羧酸；酰氯遇水同样会水解。所以这两类试剂保存时一定要避免见水。

## 六、多取代合成路线设计：谁先谁后

掌握了这些亲电取代之后，就能着手设计简单的多取代合成路线。核心思路是先把要装的官能团一个个用对应的亲电取代装上去，再考虑顺序（顺序问题与定位效应密切相关，下一节系统讲，这里先建立"谁先谁后会影响结果"的意识）。

:::example{label=由苯合成多取代苯的次序意识}
比如由苯出发，想引入一个卤素和一个硝基（即做一次卤代、一次硝化），关键就是谁先谁后。这取决于你希望第二个基团进入什么位置——而第一个基团会决定第二个基团的去向。傅克反应中还要额外提防重排和多取代。具体的定位规律我们留到 8.3 展开，这里要记住：**多取代题里，反应次序是一个需要主动设计的变量。**
:::

## 七、易错点汇总

:::pitfall{label=亲电取代与傅克反应常见错误}
- **把真正的亲电试剂写错**：硝化是 $\ce{NO2+}$（不是 $\ce{HNO3}$ 直接进攻），磺化是 $\ce{SO3}$，卤代是卤正离子，傅克是碳正离子 / 酰基正离子。
- **傅克烷基化忘记重排**：一氯丙烷主产物是异丙苯不是正丙苯；一级碳正离子常负氢迁移成二级，接叔丁基的会甲基迁移成三级。
- **酰基正离子误判会重排**：酰基正离子受羰基稳定，**不重排**，这正是它的价值所在。
- **需要直链烷基苯却硬用烷基化**：应改走"酰基化 + 还原"两步。
- **忘了催化剂体系**：卤代 / 硝化 / 磺化用强酸或路易斯酸；傅克用 $\ce{AlCl3}$，烷基化配卤代烷、酰基化配酰氯或酸酐。
- **忽略酰氯、酸酐对水敏感**：实验条件要无水。
:::

:::note{label=本节小结}
1. 苯富电子，被带正电的**亲电试剂**进攻，发生**亲电取代**：为守护离域能而脱质子恢复芳香体系，取代而非加成。
2. 机理两步：先生成**σ 络合物**（能量高点），再脱质子得取代产物。
3. 四类经典反应及真正亲电试剂——**卤代**（卤正离子，路易斯酸催化）、**硝化**（$\ce{NO2+}$，硫酸催化，50–60 ℃）、**磺化**（$\ce{SO3}$，可逆）、**傅克反应**。
4. **傅克烷基化**：卤代烷 + $\ce{AlCl3}$，生成碳正离子，**易重排**（甲基迁移 / 负氢迁移）、易多取代、可逆，合成上局限大。
5. **傅克酰基化**：酰氯或酸酐 + $\ce{AlCl3}$，生成受羰基稳定的**酰基正离子，不重排**。需要直链烷基苯时走"**酰基化 + 还原**"两步可避开重排，近乎定量得到目标产物。
6. 多取代路线要主动设计**反应次序**，它由定位效应决定（详见下一节）。
:::
