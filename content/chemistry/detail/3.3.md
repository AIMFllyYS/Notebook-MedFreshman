# 3.3 分子间作用力与物理性质

## 为什么要研究"弱"作用对物性的影响

学完了范德华力与氢键，我们终于可以回答一个非常实际的问题：为什么甲烷是气体、汽油是液体、石蜡是固体？为什么羧酸的沸点总是异常地高？为什么药瓶上同一类分子，邻、间、对三种异构体性质差这么远？

本节用分子间弱相互作用去**解释**沸点、熔点、溶解性这三类物性的规律。请特别留意最后一节"适用边界"——这套工具很好用，但用错地方就会闹笑话。

## 一、弱相互作用与沸点

先把沸点的物理含义钉死：

:::definition{label=沸点}
**沸点**是在饱和大气压下把**液态**分子变成**气态**所需的温度。液态是连续相、气态是单分子相，加热提供的能量用来**打断分子间的弱相互作用**（范德华作用、氢键、各种偶极/静电作用）。所以分子间作用越强，沸点越高。
:::

### 1. 直链烷烃：碳数越多，沸点越高

:::example{label=甲烷→壬烷}
甲烷、丙烷、戊烷、庚烷、壬烷碳数递增，沸点也递增，状态随之变化：甲烷、丙烷常温为气态，到戊烷基本是液态，到壬烷变成固态。

**解释**：分子链增长意味着分子之间**接触面积增大**，范德华作用的作用范围随之增大、作用力增强，故沸点升高。甲烷之间接触面积有限、分子量又小；正壬烷之间接触面积大，分子间弱相互作用就大。
:::

:::insight{label=普遍规律}
许多物质在小分子阶段为气态，分子量增大变液态，再增大变固态——根子都在"接触面积增大 → 范德华作用增强"。
:::

### 2. 戊烷的同分异构体：支链越多，沸点越低

:::example{label=正戊烷 / 异戊烷 / 新戊烷}
三者都是五个碳、分子量相同。沸点：**新戊烷最低，正戊烷（直链）最高**。

**解释**：**支链越多，分子之间的接触面积越小**，范德华作用越弱，沸点越低。所以对同分异构体，支链越多沸点越低。
:::

### 3. 顺反异构：极性决定沸点

:::example{label=顺-2-丁烯 vs 反-2-丁烯}
顺式沸点 3.7 ℃，反式沸点 0.9 ℃。反式比顺式低，为什么？

**结构决定性质**：反式有对称中心；从极性看，顺式两个甲基在同侧、是**极性分子**，而反式的极性矢量相互抵消、是**非极性分子**。

反式只有色散力、范德华作用相对弱；顺式是极性分子、偶极矩更大、范德华作用相对强。所以对这类烯烃同分异构体，**顺式沸点高于反式**。
:::

### 4. 乙醇、甲醚、丙烷：氢键登场

:::example{label=分子量相当的三兄弟}
乙醇（沸点 78 ℃）、甲醚（−24 ℃）、丙烷（−42 ℃），都是三原子链状骨架、分子量相当，沸点却差很大。

**乙醇最高**：除范德华作用外还存在**分子间氢键**（羟基氢与另一分子羟基氧成键），所以沸点最高。

**甲醚为何高于丙烷？** 先判断能否成氢键：甲醚虽有氧，但**没有氢给体**——氢给体的氢必须连在电负性大的原子（氧、氟、氮）上，而 $\ce{C-H}$ 视作非极性键、碳上的氢不能当给体。所以甲醚、丙烷都**不能成氢键**，只有范德华作用。在只有范德华作用时，甲醚相当于水的两个氢被甲基取代、仍是角形结构、是**极性分子**，而丙烷是非极性分子，故甲醚范德华作用更大、沸点更高。
:::

### 5. 丙醛、丙酮、丁烷：极性与对称性

:::example{label=极性带来的差距}
丙醛（48 ℃）、丙酮（56 ℃）、丁烷（−0.5 ℃）。

丙醛、丙酮都高于丁烷——这是**极性分子 vs 非极性分子**的差别。丙酮与丙醛都不能成氢键，二者差距来自**极性大小**：丙酮极性更大（对称性更好），丙醛极性略小，故丙酮沸点更高。
:::

### 6. 羧酸：双氢键形成二聚体

:::theorem{label=羧酸的二聚体}
**所有羧酸的沸点都非常高**。原因：两分子羧酸之间可以形成**两个氢键（一对氢键）**，结合成**二聚体**；而两个甲醇或两个乙醇分子之间只能形成**一根氢键**。

正因为能形成这种分子间的**二聚体氢键**，羧酸与分子量相近的醇或酯相比，沸点显著偏高。
:::

### 7. 胺类：碳数、极性与位阻

:::example{label=胺的几组比较}
**碳数效应**：甲胺、乙胺、丙胺、丁胺随碳数增加沸点升高——仍是范德华作用（接触面积增大）。这些胺都是极性分子。

**位阻效应（二乙胺 vs 丁胺，同分异构体）**：二者都能成氢键（氮上氢连在电负性大的氮上，可当给体；另一分子的氮可当受体）。但**伯胺（丁胺）链状，分子间氢键更强**；**仲胺（二乙胺）有位阻，分子间氢键相对弱**。同分子量下，二级胺氢键更弱、沸点更低，一级胺氢键更强、沸点更高。

**电负性效应（乙烷 / 乙胺 / 乙醇，分子量相当）**：乙醇沸点最高（乙胺 17 ℃，乙烷为气态）。乙胺、乙醇都有氢键，乙烷没有。乙胺成的是氮—氢氢键，乙醇成的是氧—氢氢键，**氧电负性比氮大**，氧—氢氢键键能更高、氢键更强，故乙醇沸点更高。
:::

### 8. 硝基苯酚的邻、间、对：分子内 vs 分子间氢键

:::example{label=邻、间、对硝基苯酚}
邻硝基苯酚沸点 100 ℃，间硝基苯酚 194 ℃，对硝基苯酚加热不到沸点就分解（沸点相当高）。

硝基上的氧（电负性大）可作氢受体，羟基的氢可作氢给体，二者间能成氢键。三者差别在于**氢键的类型**：

- **对硝基苯酚**：硝基与羟基隔得最远，形成**链状的分子间氢键**——沸点**最高**。
- **邻硝基苯酚**：硝基与羟基离得近，形成**分子内氢键**（与水杨醛情形相同），分子间只剩范德华作用——沸点**最低**。
- **间硝基苯酚**：硝基离羟基有点远，难成分子内氢键，仍走**分子间氢键**，但这种分子间氢键**不是链状**的、作用较弱——沸点正好**居中**。
:::

:::insight{label=邻、间、对的判断套路}
看到邻/间/对取代，先问：取代基之间够不够近、能不能成分子内氢键（尤其能否闭合稳定的五/六元环）。
**邻位**常成分子内氢键 → 分子间作用弱 → 沸点低；
**对位**只能成分子间氢键、且常呈链状 → 沸点高；
**间位**走非链状分子间氢键 → 居中。
:::

## 二、弱相互作用与熔点

:::definition{label=熔点}
**熔点**是从**固态**到**液态**的转变温度。
:::

熔点比沸点多考虑一个因素：

:::theorem{label=熔点的两个决定因素}
沸点（液→气）打破的只是**分子间弱相互作用**。而熔点（固→液）不仅要打破分子间弱相互作用，还要打破固体的**晶格能**。

因此比较熔点要看两个因素：
1. **分子间弱相互作用**；
2. **分子的对称性**——对称性越好，晶体排列越紧实、晶格能越高、熔点越高。
:::

### 1. 直链烷烃熔点的奇偶折线规律

:::insight{label=偶数碳升得更猛}
直链烷烃熔点随碳数增加而升高，但**含偶数碳的烷烃熔点升幅大于奇数碳**，整体呈一条**折线**。原因：**偶数碳烷烃分子对称性更好**，故其熔点升高超过相邻的奇数碳。
:::

### 2. 对称性对熔点的影响

:::example{label=新戊烷的"反转"}
戊烷三种异构体：论**沸点**，看分子大小与支链多少即可（正戊烷最高、新戊烷最低）。但论**熔点**规律反过来——**新戊烷熔点最高**（而它的沸点最低）！因为新戊烷**高度对称**，结晶规整、晶格能高，熔点反而最高；正戊烷不对称。这里体现的正是**对称性**对熔点的影响。
:::

:::example{label=顺反异构的熔点反转}
顺反异构：**沸点**是顺式高、反式略低（极性问题）；但**熔点**反过来变成**反式高**——因为**反式对称性更好**。
:::

:::insight{label=记忆要点}
**比沸点看"分子间作用力（含极性、氢键、接触面积）"，比熔点要再加上"对称性"。** 很多"沸点低却熔点高"的反转（如新戊烷、反式异构体），都是对称性在起作用。
:::

## 三、弱相互作用与溶解性：相似相溶

:::theorem{label=相似相溶}
溶解性的经验规律就四个字：**相似相溶**。结构相似的溶质与溶剂一般能良好互溶——水易溶甲醇、乙醇（结构相似）；苯能较好地溶解萘、蒽这类芳香化合物。

相似相溶主要与分子的**极性**及分子的**大小、结构**有关：结构相似意味着分子大小与极性都高度相似，溶质与溶剂便能很好相溶。这是一条**经验规律**，只能从弱相互作用与结构特征上作解释。
:::

## 四、弱相互作用的适用边界：只能解释，不能预测

这是本节、也是全章最需要记住的一句话。

:::theorem{label=弱相互作用只能解释、不能预测}
分子间弱相互作用（范德华力、氢键）**不是一个能做预测的理论**。我们只能在**拿到沸点/熔点数据之后**，用它去**解释**为什么会这样；而**不能反过来凭它去预测**任意两个分子谁高谁低。
:::

:::pitfall{label=适用边界的红线}
1. **只能比较"相似的结构"。** 比如同系物，或分子量相同的、可类比的物质（醇与烷烃互比）。**不能把任意两个不相干的分子拉到一起互相比较**。
2. **范德华力与氢键只是众多弱相互作用中两种典型代表。** 还有许多作用力我们没学，甚至有些人类至今尚未认识。所以它只能对相似结构作**定性解释**，无法对随意挑出的两个分子给出谁大谁小的判断。
3. **判断氢键先认给体。** 醚里虽有氧却无氢给体（碳上的氢不算），故甲醚不能成氢键。别一看到电负性大的原子就以为能成氢键。
4. **沸点、熔点用不同的工具。** 沸点别忘了极性与氢键；熔点务必补上**对称性**，否则会在"新戊烷""反式异构体"这类题上栽跟头。
5. **数据以校正后为准。** 课本中沸点等数据存在印刷错误，应以课堂订正过的数据为准，避免被错印数据误导推理。
:::

:::note{label=本节小结}
1. **沸点**打断的是分子间弱相互作用：直链烷烃**碳数越多沸点越高**（接触面积增大）；同分异构体**支链越多沸点越低**；顺反异构中**顺式（极性大）沸点高于反式**；有分子间氢键者（醇、酸、胺）沸点更高，其中**羧酸因形成二聚体双氢键沸点特别高**，氧—氢氢键强于氮—氢氢键。
2. **氢键给体**的氢须连在 O、F、N 等电负性大的原子上；**碳上的氢不能作给体**，故醚、烃不能成氢键。
3. **邻/间/对**取代：邻位多成**分子内氢键**（沸点低），对位成**链状分子间氢键**（沸点最高），间位成**非链状分子间氢键**（居中）。
4. **熔点**比沸点多一个**对称性**因素（影响晶格能）：故出现"新戊烷沸点最低却熔点最高""反式沸点低却熔点高"的反转；直链烷烃熔点呈**奇偶折线**，偶数碳升幅更大。
5. **溶解性**遵循经验规律**相似相溶**（结构、极性、大小相似则易溶）。
6. **核心边界**：分子间弱相互作用**只能解释、不能预测**，且只适用于**相似结构之间**的定性比较。
:::
