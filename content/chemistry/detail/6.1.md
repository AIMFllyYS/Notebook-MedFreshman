# 6.1 烯烃的结构与催化加氢

## 从"一根活泼的键"说起

如果说烷烃是有机化学里"老实巴交"的一类分子——只会发生比较迟钝的自由基取代，那么烯烃就是性格活泼、爱"凑热闹"的那一类。它们的活泼，全都来自分子里那个标志性的结构：**碳碳双键**。一旦分子里出现了这根双键，烯烃的几乎所有化学性质就都围着它转了。

那么这根双键到底"特殊"在哪？我们先从直觉建立认识：双键看上去是两条线，但这两条线并不是一模一样的两根普通单键。它由一根 $\sigma$ 键和一根 $\pi$ 键组成，这两根键的"脾气"完全不同。把这一点想清楚，烯烃后面所有的加成、氧化反应就都有了根。

## 一、双键的成键本质：$\sigma$ 键与 $\pi$ 键

### 1. 两根键，两种性格

:::definition{label=σ键与π键}
碳碳双键 $\ce{C=C}$ 由两根键组成：
- **$\sigma$ 键**：成键电子云沿着两个碳原子核的连线方向"头碰头"重叠，电子云对称地集中在两核之间，束缚得很紧、键能较大、比较稳定。
- **$\pi$ 键**：由两个碳上方向相同的 p 轨道"肩并肩"侧面重叠形成，电子云分布在双键平面的上下两侧，对电子的束缚比 $\sigma$ 键弱，键能较小、电子云流动性强，因而比较活泼。
:::

录音里老师反复强调一个画面：$\pi$ 键是"直立的两个 p 轨道形成的键"，它对电子的束缚没有 $\sigma$ 键那么强，所以"流动性很好——你可以把它想成柔韧性很好"。这就是理解烯烃一切反应的钥匙。一根活泼、富电子、又容易被外界基团"推"或"拉"而变形的键，自然就成了各种试剂进攻的靶子。

:::insight{label=为什么反应发生在双键上}
$\pi$ 电子云裸露在双键平面的上下两侧，既"显眼"又"松散"。带正电的试剂从外面凑过来时，最先够得着、也最容易交出去的就是这对 $\pi$ 电子。所以烯烃的化学反应主要发生在双键上，而在双键上最主要的反应形式，就是**加成**——把试剂的两个部分分别接到双键的两个碳上，双键随之打开变成单键。
:::

### 2. 双键不能自由旋转

单键（$\sigma$ 键）可以绕键轴自由旋转，因为旋转时电子云的重叠程度几乎不变。但双键里多了一根 $\pi$ 键，情况就不一样了。

:::insight{label=旋转受限的来由}
$\pi$ 键依靠两个 p 轨道**侧面平行**的重叠才能成键。如果你强行把双键的一端相对另一端扭转，这两个 p 轨道就会从"平行"逐渐变成"垂直",侧面重叠被破坏，$\pi$ 键被拉断。拆断一根化学键需要很高的能量，常温下分子根本没有这么多能量去"硬扭"。所以**双键不能像单键那样自由旋转**，双键两端的相对取向被牢牢固定下来。
:::

正是这个"转不动"的特点，带来了烯烃一个重要的立体化学后果——顺反异构。

## 二、顺反异构与 Z/E 标记

### 1. 同一个连接方式，两种不同分子

既然双键转不动，那么双键两端各自连着的基团，它们的相对位置就被冻结了。

:::definition{label=顺反异构}
当双键的**每一个**碳原子上都连有两个不同的基团时，由于双键不能旋转，相同（或较大）的基团既可能落在双键平面的同一侧，也可能落在两侧，从而产生两种不能互相转化的异构体。这种因双键旋转受限而产生的立体异构，称为**顺反异构**。
:::

直觉上可以这样理解：把双键平面想象成一道篱笆，篱笆不能转动。两个甲基都站在篱笆的同一侧，是一种分子；一个站这边、一个站那边，又是另一种分子。它们的原子连接顺序完全相同，却因为空间取向不同而成为两个不同的化合物。

需要强调的前提：**双键每个碳上的两个基团都必须不同**，才有顺反之分。如果某个双键碳上连了两个相同的基团（比如两个氢），把它翻过来看还是一样的，就不存在顺反异构。

### 2. 顺、反与 Z/E

习惯上，当两个相同（或同类）的基团在双键同侧时叫**顺式**，在异侧时叫**反式**。但当双键碳上连的四个基团都不一样、无法简单说"哪两个相同"时，"顺/反"就说不清了，于是引入更严格的 **Z/E 标记法**。

:::definition{label=Z/E标记法}
对双键的**每一个**碳原子，分别按"顺序规则"比较它所连两个基团的大小，各自选出**较大**的那一个。
- 两个"较大基团"在双键**同侧**，记为 **Z**（来自德文 zusammen，"在一起"）。
- 两个"较大基团"在双键**异侧**，记为 **E**（来自德文 entgegen，"相反"）。
:::

:::note{label=顺序规则要点}
顺序规则比的不是基团"个头大小"，而是原子在**元素周期表里的位置**：原子序数大的优先。第一个原子相同时，再依次比较与之相连的下一层原子。这正是录音里判断绝对构型时强调的同一套规则——"顺序规则不是比你的个头大小，是比你的位置大小"。Z/E 与顺/反在很多简单分子里是一致的，但 Z/E 适用范围更广、更不容易出错，推荐优先使用。
:::

## 三、催化加氢

认识了结构，我们来看烯烃的第一个、也是最"温和"的加成反应——催化加氢。

### 1. 反应与催化剂的作用

:::definition{label=催化加氢}
在**贵金属催化剂**（铂 $\ce{Pt}$、钯 $\ce{Pd}$、镍 $\ce{Ni}$ 等）存在下，烯烃与氢气加成，双键打开，两个碳各结合一个氢，生成相应的烷烃。
:::

通式可写为：

$$\ce{R-CH=CH-R' ->[\ce{H2}][\text{Pt/Pd/Ni}] R-CH2-CH2-R'}$$

高中时我们只知道"氢加上去就变成烷烃"，似乎很简单。但这里有个值得追问的问题：**为什么非要有催化剂？** 直接把氢气和烯烃混在一起为什么不反应？

:::insight{label=催化剂到底做了什么}
氢气分子里的 $\ce{H-H}$ 键相当结实，要让它直接去和双键反应，活化能很高，常温下走不通。催化剂的作用是：氢气分子先吸附在贵金属表面，与金属作用、$\ce{H-H}$ 键被削弱甚至断开，形成活泼的"贵金属—氢"物种（金属氢化物）。与此同时，烯烃分子也靠近催化剂表面，双键的 $\pi$ 电子云与金属作用而被吸附、活化。两者都被"安置"在催化剂表面、并被预先活化之后，氢原子再一个接一个地转移到双键碳上。催化剂等于给反应铺了一条活化能更低的新路径，自己最后并不被消耗。
:::

### 2. 顺式加成机理

催化加氢最重要的立体化学特征，是它属于**顺式加成**。

:::theorem{label=顺式加成}
催化加氢时，烯烃双键的 $\pi$ 电子云吸附到催化剂表面，是"贴着"金属的某一面靠上去的。氢原子从催化剂表面、也就是从双键的**同一侧**逐个加到两个双键碳上。结果是：两个新加上的氢原子落在原来双键平面的**同一侧**。这种把两个原子从同一方向加到双键上的方式，叫做**顺式加成**。
:::

为什么是顺式？关键在机理：双键是平躺在催化剂表面被吸附的，氢原子来自下方的金属表面，自然只能从平面的"下方"这一侧加上去，分子另一侧根本接触不到氢。所以两个氢必然同侧加入。

:::pitfall{label=顺式加成是"推测"出来的，不是"规定"的}
录音里老师特别提醒："并不是我们规定烯烃加氢一定是顺式加成、所以产生了这样的结果。"逻辑顺序恰恰相反：化学是实验科学，是因为做反应时**发现**产物几乎全是同侧加成的立体结构，我们才**推测**它走的是顺式加成的机理。永远是先有实验事实，再用理论去解释，而不是先有规则去框住实验。
:::

### 3. 顺式加成的立体意义

顺式加成不是空谈，它会实实在在地决定产物的立体构型。以 1,2-二甲基环己烯为例（双键碳上各带一个甲基）：

:::example{label=1,2-二甲基环己烯的催化加氢}
$$\ce{1,2-\text{二甲基环己烯} ->[\ce{H2}][\text{Pt/Pd/Ni}] \text{顺-1,2-二甲基环己烷}}$$

两个氢从双键的同一面加上去，加成完成后，原来双键上的两个甲基就被一起"顶"到了新生成的环己烷环的**同一侧**，得到**顺式**的 1,2-二甲基环己烷。如果反应走的是反式加成（一个氢加在一面、另一个加在另一面），得到的将是反式产物。实验上得到的几乎都是顺式产物，这正是顺式加成机理的有力佐证。
:::

### 4. 位阻对加氢速率的影响

既然氢原子是从催化剂表面、从双键的同一侧加上去的，那么双键能不能"贴近"催化剂表面，就成了反应快慢的关键。双键周围如果挂着又多又大的取代基，它们会挡在双键和催化剂之间，妨碍双键接触表面——这就是**位阻效应**。

:::theorem{label=取代基越多越大，加氢越慢}
催化加氢的速率与双键上取代基的多少、大小密切相关：**双键上取代基越大、数目越多，加氢速率越慢。**
因此各类烯烃加氢的相对速率大致是：
- 乙烯（双键上四个全是氢，毫无位阻）——最快；
- 双键上带有取代基的烯烃——居中；
- 双键上四个位置都被取代——最慢。
:::

这个规律有一个非常实用的推论：**当氢气不足量时，催化加氢可以表现出区域选择性。**

:::example{label=不足量氢气下的选择性加氢}
某些含两个双键的天然烯烃分子，其中一个双键在环上、取代基较多、位阻较大，另一个双键在链端、取代基较少、位阻较小。若只提供 1 摩尔氢气（以氧化铂等作催化剂），氢气会**优先**加到位阻较小、取代基较少的那个双键上。

录音里的师生对话点明了原因：氢气不足时，它"优先加成在双键上取代基较少、位阻较小的碳原子上"，因为这里更容易贴近催化剂表面、活化能更低。当然，**若氢气足量，所有双键就都会被还原。**
:::

:::pitfall{label=位阻"看局部"，不是"比总数"}
判断哪个双键位阻更小，不能简单地数"两边各有几个取代基就比总数"。录音里强调："它的位阻不是打牌——不是说你这一组牌跟我这一组牌比总数。" 反应是从某个具体方向"贴"上去的，要看**那一侧实际有没有基团挡路**。比如端基双键虽然名义上也是两个取代基、两个氢，但氢气从端头方向接近时几乎没有阻挡，位阻就明显小于环内双键。
:::

## 四、把催化加氢的要点串起来

我们把这一节的核心拎清楚：

:::note{label=本节小结}
**结构层面：**
1. 碳碳双键 = 一根 $\sigma$ 键 + 一根 $\pi$ 键。$\sigma$ 键键能大、稳定；$\pi$ 键键能小、电子云活泼、易变形——烯烃的反应主要发生在 $\pi$ 键上。
2. $\pi$ 键依赖 p 轨道侧面平行重叠，强行扭转会拆断它，因此**双键不能自由旋转**。
3. 双键旋转受限 → 双键两碳上若各连两个不同基团，就产生**顺反异构**。用 **Z/E** 标记更严格：各碳按顺序规则取较大基团，同侧为 Z、异侧为 E。

**催化加氢层面：**
4. 条件：必须有贵金属催化剂（Pt / Pd / Ni 等），它通过吸附、活化氢气和双键，降低活化能。
5. 立体特征：**顺式加成**——两个氢从双键同一侧加入，落在原双键平面同侧；这一结论由实验产物推测得来。
6. 速率规律：双键上取代基**越多越大，加氢越慢**（位阻效应）；据此，**氢气不足时优先加成位阻小的双键**，氢气足量则全部还原。

记住一句话：催化加氢"没有亲电不亲电"的问题，它只是催化剂表面上的一次顺式加成。下一节我们将进入烯烃反应的主战场——亲电加成。
:::
