# 8.3 定位效应与多取代规则

## 一、直觉：先来的人会影响后来的人选哪张床

上一节我们讲的卤代、硝化、磺化、傅克反应，都是在**没有取代基**的苯环上做第一次取代。这时苯环上六个位置等价，引一个基团上去不分主次，取代在哪都一样。

可一旦苯环上**已经有一个取代基**，再引入第二个取代基时，大量研究发现：它**并不是随机**地落在剩下五个位置上的。有一类已有取代基会把后来者"指挥"到自己的**邻位或对位**，另一类会把后来者指挥到自己的**间位**。

:::insight{label=入学抢床位的类比}
这就像新生入学抢床位。第一个到的人四张床随便挑，但第二个人挑床位时一定会受第一个人的影响：有的是主动影响——"你住到我对面去/离我远点"；有的是被动影响——"看这哥们不好相处，我离他远点"。从博弈论看，只要有一个人先住进去，他必然影响第二个人的判断。苯环取代里也一样：**第一个取代基决定第二个取代基的位置**，这种性质就叫定位效应。
:::

:::definition{label=定位效应与两类定位基}
苯环上已有取代基会决定第二个亲电取代发生的位置，这种性质称为**定位效应**。据此把取代基分两类：
- **第一类定位基（邻对位定位基）**：使第二个取代基主要进入它的**邻位和对位**。
- **第二类定位基（间位定位基）**：使第二个取代基主要进入它的**间位**。

注意：邻对位 / 间位说的是**首先上去的那个基团**的定位效果，不是后来上去的那个基团。
:::

## 二、第二个维度：致活与致钝

定位效应说的是"位置"。还有另一个维度说的是"难易"。

:::definition{label=致活基与致钝基}
- **致活基**：第一个基团上去后，使第二次亲电取代变得**更容易**（反应条件更温和）的取代基。
- **致钝基**：第一个基团上去后，使第二次亲电取代变得**更难**（反应条件更苛刻）的取代基。
:::

为什么有的致活、有的致钝？关键看它对苯环电子云密度的影响。亲电取代要求苯环 π 电子云密度越高越容易发生。

:::theorem{label=电子效应决定致活致钝}
- **给电子基**让苯环电子密度升高 → **致活**。
- **吸电子基**把苯环电子密度拉低 → **致钝**（如硝基，强吸电子，使再次亲电取代更难）。
:::

## 三、两个维度组合，为什么只剩三类

把"定位（邻对位 / 间位）"和"难易（致活 / 致钝）"两两组合，理论上有四种：致活的邻对位、致钝的邻对位、致活的间位、致钝的间位。但实际上所有取代基只落在**三类**：

1. 致活的邻对位定位基
2. 致钝的邻对位定位基
3. 致钝的间位定位基

唯独**没有"致活的间位定位基"**。换句话说：**间位定位基全部是致钝的**。

:::insight{label=一条贯穿全局的主线}
给电子 → 致活 → 邻对位；吸电子 → 致钝 → 间位。这两条线几乎把所有取代基都归好了类。唯一打破这条对应的，是**卤素**这个特例——下文专门讲它。把这条主线记住，定位效应就不必死背了。
:::

## 四、各类取代基的归类

### 第一类：致活的邻对位定位基

- 与**饱和氮**相连：氨基 $\ce{-NH2}$、取代氨基、酰胺（指取代发生在氮这头、氮直接相连的情形）。
- 与**饱和氧**相连：羟基 $\ce{-OH}$、烷氧基 $\ce{-OR}$，以及酯基中与**饱和氧**直接相连的那一端。
- 与**烃基**相连：苯基、烷基及取代烷基。

致活强度有别：含氮一列**强活化**，含氧一列**中等活化**，烃基**弱活化**（活化程度不高，但至少不致钝）。

### 第二类：致钝的邻对位定位基（卤素特例）

就是**氟、氯、溴**这一组卤素，是一个特殊情况。

:::insight{label=卤素特例：致钝却邻对位定向}
卤素接到苯环上，总体的**诱导效应 + 共轭效应**表现为**吸电子**，所以它**致钝**。但它的致钝并不使它成为间位定位基——因为卤素的**共轭效应（p-π 共轭）是给电子的**，孤对电子能向邻、对位反馈电子，使邻对位的 σ 络合物中间体更稳定，所以它仍然是**邻对位定位基**。于是卤素成了"既吸电子致钝、又邻对位定向"的唯一特例。记住这个特例即可。
:::

### 第三类：致钝的间位定位基

- **直接和羰基相连**的酰基取代基（羰基是强吸电子基，诱导与共轭两种效应叠加，强致钝）。
- 强吸电子基：磺酸基 $\ce{-SO3H}$、氰基 $\ce{-CN}$、硝基 $\ce{-NO2}$。
- 被卤代的烷基，如三氟取代、三氯取代的烷基——原本的碳氢键变成强极性的碳氟键、碳氯键后，整体变成吸电子基。

:::pitfall{label="酯基"与"酰基"连接方式决定性质相反}
同样含羰基，连接方式不同，定位与致活致钝完全相反：
- **直接和羰基相连**（酰基，如 $\ce{-COR}$、$\ce{-CHO}$、$\ce{-COOH}$）→ 强吸电子 → **致钝的间位定位基**。
- 酯基中**与饱和氧相连**的那一端（$\ce{-O-CO-R}$ 里苯环接在饱和氧上）→ **致活的邻对位定位基**。
做题时务必看清苯环到底接在羰基上还是接在饱和氧上。
:::

## 五、定位规律一句话总结

:::theorem{label=定位效应速记}
- 所有**给电子基**都是**致活**的，致活就一定是**邻对位**定位。
- 所有**吸电子基**都是**致钝**的，致钝就一定是**间位**定位。
- 唯一例外是**卤素**：它是吸电子基（致钝），但又是**邻对位**定位基。
把这一个特例单独记住，其余按"给电子—致活—邻对位 / 吸电子—致钝—间位"推即可。
:::

## 六、二取代苯：第二次取代用定位效应

有了定位效应，就能预测第二次亲电取代的位置和大致条件了。要分清两件事：**致活 / 致钝指的是反应条件（难易）**，**邻对位 / 间位指的是反应位置**，二者是不同因素。

:::example{label=几个典型二取代预测}
- **氯苯硝化**：氯是**致钝的邻对位定位基**，所以硝基进入氯的邻位、对位；又因致钝，反应条件要比苯直接硝化更猛一些。
- **硝基苯氯代**：硝基是强吸电子基，是**致钝的间位定位基**，所以氯进入间位（间位只有一个）；又因致钝，条件比苯直接氯代更苛刻。
- **甲苯磺化**：甲基是**弱致活的邻对位定位基**，邻位、对位产物都有；改变温度会发现高温下**对位产物提升**——这说明对位是**热力学产物**（结构更稳定），温度足够时更多生成热力学产物。
- **氯苯再氯代**：氯是邻对位定位基，在邻对位取代，但因致钝，温度比第一次氯代要高一些。甲苯也是邻对位定位基但**致活**，邻对位都有且温度更温和。
:::

:::insight{label=对位 vs 邻位：动力学与热力学}
邻对位定位基往往同时给出邻位和对位两种产物。低温（动力学控制）邻位常更多；升高温度（热力学控制）更稳定的对位产物比例上升。甲苯磺化随温度变化对位升高，正是热力学产物在足够能量下占优的体现。这与共轭二烯 1,2- / 1,4- 加成的动力学 / 热力学之分是同一套思想。
:::

## 七、三取代苯：两个取代基竞争定位

苯环六个位置，已经住了两个取代基，再来第三个时"谁说了算"？分几种情况。

### 情况一：两个取代基定位一致

两个取代基的定位效应指向**同一个位置**，第三个取代基就由它们共同决定，落在那个位置。

:::example{label=定位一致的几例}
- 上方酯基（接在**饱和氧**上，邻对位）与下方酯基（接在**羰基**上，间位）：邻对位指向的位置与间位指向的位置恰好重合，第三个基团就进这里。
- 硝基（间位）与醛基 $\ce{-CHO}$（间位）：两者间位重合，取代在该处。
- 两个甲基（邻对位）：若某位置同时是一个甲基的邻位、又是另一个甲基的对位，两意见统一，取代在此。
- 羟基（邻对位）与硝基（间位）：羟基邻 / 对位与硝基间位重合时，取代在重合处。
:::

### 情况二：两个取代基同类（如都是邻对位定位基）

定位不一致时，**谁的活化作用强听谁的**（实力强者说了算）。

:::example{label=同类竞争看强弱}
- 两个都是邻对位定位基，一个强活化、一个弱活化 → 听**强活化**的。
- 甲基（弱活化）与另一邻对位基竞争 → 由更强者主导。
- 羟基 vs 甲基 → **羟基**是老大，以羟基的邻 / 对位定向。
- 酰胺（氮直接相连，强活化）vs 烷基（弱活化）→ 听**酰胺**的，进它的邻对位。
:::

### 情况三：两个取代基不同类

一个间位定位基、一个邻对位定位基（例如两者都致钝、分不出强弱），第三个取代基**主要由邻对位定位基决定**——由能把基团定到邻对位的那个说了算。例如某致钝间位基与致钝邻对位基（如氯）并存时，由氯决定，进它的邻位和对位。

:::theorem{label=三取代定位三原则}
1. **定位一致**：第三基进两者共同指向的位置。
2. **同类竞争**：进**活化作用较强**者所定的位置。
3. **不同类**：主要由**邻对位定位基**决定。
:::

## 八、位阻：被两个基团夹住的位置不好进

定位之外还要考虑**位阻**。当某个位置被两个取代基夹在中间时，第三个基团很难挤进去。

:::pitfall{label=1,3 位之间的"夹心位"取代率大幅降低}
当两个取代基处于 **1,3 位**时，它们之间的 **2 位**（被夹在中间的位置）虽然在定位上可能同时是两者的邻位、理论上能取代，但由于两侧都有取代基的空间阻碍，**进入 2 位的比例大幅降低**，实际几乎很少在此取代。
:::

:::example{label=位阻改变"理论上该取代"的位置}
- 酰胺（强活化，邻对位）与烷基并存时，酰胺的两个邻位中，**夹在两个取代基之间**的那个邻位因位阻难取代，一般在另一个邻位和对位取代。
- 甲基（邻对位）与氯（邻对位）处于 1,3 位时，中间位置虽是两者共同的邻位、理论上可取代，但因夹心位阻，实际很少在此发生，主要落在不受夹的邻位和对位。
:::

## 九、合成应用：用定位效应规划次序

定位效应最实用之处，是在多步合成中决定**谁先谁后**——通过改变基团或改变反应顺序来"借用"或"转换"定位属性。

:::example{label=由甲苯合成间 / 对硝基苯甲酸}
**补充知识**：甲苯在强氧化条件下（如 $\ce{KMnO4}$）会被氧化成苯甲酸；无论侧链多长，强氧化都会把它断成只含一个碳的苯甲酸（$\ce{-COOH}$）。

要做硝化和氧化两步，核心是先后顺序：

- **目标：间硝基苯甲酸。** 甲基本是邻对位定位基，直接在甲基间位上硝基很难。先把甲基**氧化**成苯甲酸——这一步把一个**邻对位定位基（甲基）转换成间位定位基（羧基）**——再硝化，硝基进入间位，得间硝基苯甲酸。

$$
\ce{C6H5CH3 ->[\ce{KMnO4}] C6H5COOH ->[\ce{HNO3}/\ce{H2SO4}] \textit{m}\text{-}NO2C6H4COOH}
$$

- **目标：对硝基苯甲酸。** 先利用甲基的**邻对位**属性做**硝化**，得邻、对硝基甲苯，分离出对位产物，再氧化成羧酸。

$$
\ce{C6H5CH3 ->[\ce{HNO3}/\ce{H2SO4}] \textit{p}\text{-}NO2C6H4CH3 ->[\ce{KMnO4}] \textit{p}\text{-}NO2C6H4COOH}
$$

要间位就**先氧化（把甲基变成间位定位基）后硝化**；要对位就**先硝化（保留甲基的邻对位定向）后氧化**。
:::

:::example{label=由苯合成邻 / 间 / 对硝基甲苯（卤代版同理）}
由苯引入一个卤素和一个硝基也是顺序问题：
- 要**邻 / 对位**产物：先**卤代**（氯是邻对位定位基），定好位再上硝基 → 邻对位硝基取代。
- 要**间位**产物：先**硝化**（硝基是间位定位基），上去后第二个进间位。
:::

## 十、易错点汇总

:::pitfall{label=定位效应常见错误}
- **混淆两个维度**：致活 / 致钝管"难易（条件）"，邻对位 / 间位管"位置"，别用一个去推另一个（卤素就是反例）。
- **忘记卤素特例**：卤素吸电子致钝，却是邻对位定位基。
- **酯基 / 酰基连接方式搞反**：接羰基→致钝间位；接饱和氧→致活邻对位。
- **三取代不分情况**：定位一致→共同决定；同类→强者决定；不同类→邻对位基决定。
- **忽略位阻**：1,3 位之间的夹心 2 位即使定位"合法"，实际很少取代。
- **合成题次序选反**：要转换定位属性（如甲基→羧基）必须安排好氧化 / 取代的先后。
:::

:::note{label=本节小结}
1. **定位效应**：第一个取代基决定第二个取代基的位置——分**邻对位定位基**与**间位定位基**。
2. 另一维度**致活 / 致钝**由电子效应决定：给电子致活、吸电子致钝。
3. 实际只有三类：**致活的邻对位、致钝的邻对位（仅卤素）、致钝的间位**；没有"致活的间位"。
4. 主线速记：给电子—致活—邻对位；吸电子—致钝—间位；唯一特例是**卤素**（致钝但邻对位定向，因诱导吸电子、共轭给电子）。
5. 二取代预测要分清位置（邻对位 / 间位）与条件（致活 / 致钝）；邻对位常给邻、对两种产物，高温利于更稳定的**对位（热力学产物）**。
6. 三取代三原则：定位一致看共同位置、同类看活化强弱、不同类看邻对位基；并要考虑**位阻**（1,3 位夹心的 2 位取代率大幅降低）。
7. 合成中用定位效应规划**反应次序**，必要时通过氧化等手段**转换定位属性**（如甲基→羧基由邻对位变间位）。
:::
