# 9.3 Fischer 投影式与 R/S 绝对构型

## 为什么需要一种新的画法

到目前为止，我们已经会用楔形式表示立体结构、用纽曼投影式判断构象。但当我们想简洁地写下一个手性碳的空间排布、并据此判断它的"绝对构型"时，楔形式画起来还是嫌麻烦。本节要学的**费歇尔（Fischer）投影式**，正是为此而生：第一，它能把立体分子结构大大简化；第二，配合顺序规则，它能让我们方便地判断手性碳的**绝对构型**（R 还是 S）。

打个比方，药品说明书上印的那些"十字交叉"结构，绝大多数就是 Fischer 投影式。学会读写它，再学会从它判断 R/S，就掌握了描述手性分子的通用语言。

## Fischer 投影式的书写规则

Fischer 投影式的本质，是把一个三维的四面体碳"压扁"投影到纸面上，用一个十字交叉的交点代表手性碳。它遵循三条规则：

:::definition{label=Fischer 投影式三原则}
1. **碳链竖直放置，氧化态高的基团在上、低的在下。** 把含手性碳的主碳链尽量放在垂直方向，并让氧化态高的基团位于顶端、氧化态低的基团位于底端。

2. **竖直方向的键伸向纸面后方，水平方向的键伸向纸面前方。** 即"竖键朝里、横键朝外"。

3. **投影到纸面，用横线与竖线的交叉点表示手性碳。** 把整个立体结构压到纸面上，横线、竖线各自拉通，交叉点就是手性碳（该碳本身可以隐去不写）。
:::

这套约定里**最要命的就是第 2 条**：横、竖两个方向的空间含义是被**人为规定死**的——只要看到一个 Fischer 投影式，就必须认定它的横线两端朝向纸面外（朝向你），竖线两端伸进纸面里（远离你）。这不是可商量的画法习惯，而是读懂它的前提。

## 以乳酸为例：怎么"压"出来

:::example{label=乳酸的 Fischer 投影}
1. **选主链、定方向**：乳酸的三个碳构成最长碳链，放在竖直方向。
2. **定上下**：比较手性碳上两个方向的氧化态——羧基含氧多、氧化态高，放上面；甲基氧化态低，放下面。
3. **补横向基团**：把剩下的羟基、氢按手性碳上的连接顺序补到水平方向。

这样摆好后，得到一个楔形式：**两根竖键朝纸面里（黑板里），两根横键朝纸面外（黑板外）**。最后做一次"拉通"——横线拉通、竖线拉通、把交叉处的碳隐去，就得到标准的 Fischer 投影式。十字交叉的中心即为手性碳。
:::

## 逆向：从投影式还原成立体结构

实际做题时，我们更常做相反的事：看到一个 Fischer 投影式，要在脑子里把它**还原成立体的楔形式**。方法就是把三条规则倒过来用——

:::insight{label=横出竖进}
看到 Fischer 投影式时，立刻在脑中执行："**横着的两根线往前伸（朝向你），竖着的两根线往后伸（远离你）**。" 横线两端变成朝外的楔形实线，竖线两端变成朝里的虚线方向。这样就把平面的投影式还原成了三维结构。这是规定死的，只要写成 Fischer 形式，就一律按"横出竖进"理解。
:::

平面投影式与立体楔形式之间能否在脑中**等价互换**，是后面判断 R/S 的基本功，必须练熟。

## 旋转操作的禁忌

Fischer 投影式不是普通的平面图，它携带着空间信息，所以**不能随意旋转**。

:::pitfall{label=两条旋转禁令}
- **不能向纸面外翻转 180°。** 把某个基团从纸面里"拉出来"翻过去，相当于改变了横竖键的朝向，得到的是原分子的**对映异构体**——绝对构型被翻转了，已经不是原来那个分子。
- **不能在纸面内旋转 90°（或 90° 的奇数倍）。** 像转方向盘那样转 90°，由于 Fischer 约定"横外竖里"不变，转完之后横键、竖键的空间含义被强行互换，结果也是原分子的对映体。

**唯一允许**的是在纸面内旋转 180°（即 90° 的偶数倍）：此时横还是横、竖还是竖，朝向不变，分子不变。
:::

记忆口诀：**纸面内转 180° 安全，转 90° 或翻出纸面，都会变成对映体（相当于加了一面镜子）。**

:::example{label=甘油醛投影式的规范操作}
把甘油醛的立体手性构型改写成 Fischer 投影式：
1. 竖线选三个碳构成的碳链；
2. 比较氧化态——醛基高于羟基（而羧基又高于醛基），所以把**醛基提到顶端**；这一步相当于在纸面内把分子向上旋转，使醛基朝上、碳链落在竖线、两端正好落在纸面内。

**关键禁忌**：在"把醛基提到顶上"的过程中，羟基和氢的左右位置**不能动**（羟基仍在右、氢仍在左）。Fischer 投影式是用来表示手性碳结构的，一旦在变换中打乱了基团的相对顺序，就破坏了手性结构、写出了错误的构型。只要变换中各基团相对位置不变，最后还原成标准写法即可。
:::

## 绝对构型 R/S 与相对构型 D/L

:::definition{label=R/S 与 D/L}
- **R/S 是绝对构型**：它直接反映手性碳上四个基团的真实空间连接顺序，是现代通用的标记法（药品说明书上的 R、S 即指此）。
- **D/L 是相对构型**：早期在无法测知绝对构型时，人为以甘油醛为标准规定出来的标记法。
:::

D/L 的由来值得了解：早期人们能测出甘油醛两种对映体里哪一种是右旋的，于是**人为规定**右旋甘油醛为 D 型（其手性碳上羟基画在右侧）。由 D-甘油醛经氧化得到 D-甘油酸、再还原得到 D-乳酸，D/L 标记就这样沿着反应链传递下去。

:::insight{label=D/L 与左右旋没有对应关系}
有意思的是：D-甘油醛是右旋（+），但氧化后的 D-甘油酸却是左旋（−）。这说明**左右旋（旋光方向）与 D/L（结构标记）之间没有必然关系**——左右旋是分子对偏振光的偏转方向（要靠实验测），D/L 是分子的结构标记。当年 D/L 是人为规定的，受限于表征技术还测不出碳上基团的真实连接顺序；但后来发现，人为规定的连法竟然真就和实际相符。等技术发展到能确定真实连接顺序后，才改用能反映真实空间排布的 R/S 绝对构型。D/L 作为传统沿用方式至今仍在用（如 D-葡萄糖、L-葡萄糖），但它不代表绝对构型。
:::

## 顺序规则：给四个基团排大小

判断 R/S 之前，必须先能给手性碳上的四个基团排出"大、中、小"的优先级，这要用到命名部分学过的**顺序规则**。

:::definition{label=顺序规则四条}
1. **单原子比序数**：直接相连的是单个原子时，按原子序数排，**有大则大**（序数大者优先）。
2. **第一原子相同则向外比**：若第一个原子相同，就比较与它相连的下一层原子，仍**有大则大**。例如都是碳直接相连，就看这些碳各自又连了哪些原子，连有更大原子者优先。
3. **双键、三键作等价替代**：含双键、三键的基团先做等价替换再比较，一般**三键 > 双键**。
4. **不足四个补 0 号原子**：参与比较的原子不到 4 个时，补一个原子序数为 0 的假想原子占位。
:::

特别提醒：顺序规则判断的是"规则意义上的大小"，**与基团真实的体积（位阻）不是一回事**。位阻看真实大小，顺序规则按规则排。比如羧基那个碳连了两个氧（$\ce{C}$ 上接 $\ce{O}$、$\ce{O}$），按规则它可能排在某个体积更大的支链之前——不能凭"看起来大"就判优先。

## R/S 判断步骤

:::theorem{label=判断绝对构型的标准流程}
1. **排序**：把手性碳上的四个基团按顺序规则排出大、中、小及最小（手性碳连四个不同基团，必能排出唯一次序）。
2. **隐最小基**：把**最小**的基团放到离眼睛**最远**的位置（伸进纸面内），不再管它；剩下三个基团朝向眼睛。
3. **看转向**：从大到中到小依次连线——**顺时针为 R 构型，逆时针为 S 构型**。
:::

如果给的是 Fischer 投影式，要先按"横出竖进"还原成楔形式，再找出最小基团、把它转到背离眼睛的方向，让其余三个正对眼睛，然后判断转向。**这一步是全流程中最容易出错的地方**：把最小基团转到背面时，剩下三个基团的空间相对顺序一定不能弄乱。

:::pitfall{label=好事要成双}
"把最小基团转到背面"这一步若搞错一次，本来是 S 的就会判成 R——满盘皆输。但如果你恰好**错了两次**，两次翻转互相抵消，最终答案反而对了（所谓"好事成双"）。结论：要么一次都不错，要么就错偶数次。最稳妥的办法是老老实实先还原楔形式，盯紧三个基团的空间顺序别乱。

**另一个救场技巧**：若题目给的结构中**最小基团（常是氢）恰好朝向你**（伸出纸面），可以直接按给定方向判断出 R 或 S，再**反过来**取相反结果即可——因为你是从"反方向"在看。
:::

## 从 Fischer 式到绝对构型：实例

:::example{label=D-甘油醛是 R 型}
给定 D-甘油醛的 Fischer 投影式：
1. **还原**：横向两基团伸出来、竖向两基团伸进去。
2. **隐最小基**：四个基团中氢最小，把氢往纸面里拉，使羟基、醛基、羟甲基三支朝外。
3. **排序看转向**：直接连氧的**羟基最大**，**醛基居中**，**羟甲基那支最小**。从大到小连线为**顺时针** → **R 构型**。

所以相对构型记为 D 的甘油醛，其绝对构型是 R。（再次印证 D/L 与 R/S 是两套不同的标记，不能直接互推。）
:::

:::example{label=两个相同手性碳的总体法（酒石酸型）}
判断含两个手性碳的分子时，用**总体法**：判断上面那个碳时，把下面整支当作一个基团整体；判断下面那个碳时，把上面整支当作一个基团整体。

以上面的碳为例：除氢外，**羟基最大**，**羧基次之**（羧基那个碳连了两个氧），**下面那支最小**。把氢转到背面后，大→中→小为逆时针 → **S**。

巧妙之处：这类含**两个相同手性碳**的手性分子，只需判断一个碳即可。若上面是 S，下面必然也是 S——因为一旦其中一个变成 R（即一个 S 一个 R），分子就成了内消旋体。它既然是手性分子（没有变成内消旋体），上下就一定同为 S。
:::

## 生物活性分子的构型

绝对构型不是纸面游戏——它直接决定生物活性。对这类分子，往往不必先做 Fischer 变换，只要看准那个最小基团（常是氢）是朝向你还是远离你即可。

:::example{label=青霉素：三个手性中心}
青霉素含**三个手性中心**。先按"连四个不同基团"找出手性碳，再逐个判断：

- 某手性碳连着氨基、羰基、一个碳、一个氢，且该氢已自然伸向纸面内。按给定角度判断：氨基最大，再比较两个碳——一个连硫、一个连氧，**连硫的大**（有大则大），剩下的小。大→中→小为顺时针 → **R**。
- 另一中心氢也在后面：硫最大、氮次之、碳最小，顺时针 → **R**。
- 还有一个中心的**氢朝向你**：氮最大、连氧的碳次之、连硫的碳……据给定方向判得顺时针为 R，但因氢朝向你是"反方向看"，需反过来 → **S**。

所以青霉素三个手性中心的绝对构型为 **R、R、S**。**只有这种绝对构型才具有杀菌活性。**
:::

:::example{label=氯霉素：两个手性中心}
氯霉素（一种抗生素）含**两个手性中心**，两个手性碳的氢都朝向纸面内，可直接判断：

- 第一个中心：大基团是碳，再到氮，再到另一支，顺时针 → **R**。
- 第二个中心：连氧的碳最大，那个碳里除氧外还有碳的支居中，顺时针 → **R**。

两个中心都是 R。
:::

:::insight{label=旋光符号与 R/S 无对应关系}
实例中分子标注的正号、负号都是**实验测出来的**，与 R/S 没有规定性的对应关系——并不是"S 就是负、R 就是正"。同理也与 D/L 无对应。记住三套标记各自独立：**左右旋（+/−）靠测、D/L 靠历史规定、R/S 靠真实空间顺序。**
:::

:::note{label=本节小结}
- **Fischer 投影式三原则**：碳链竖直、氧化态高者在上；竖键朝里、横键朝外；交叉点代表手性碳。读图时一律"**横出竖进**"还原成立体结构。
- **旋转禁忌**：纸面内转 180° 不变；转 90°（奇数倍）或翻出纸面 180° 都会得到对映体。书写变换中绝不能打乱基团相对位置。
- **R/S 是绝对构型**（真实空间顺序），**D/L 是相对构型**（以甘油醛为标准的人为规定）；二者与左右旋（实测）都无固定对应关系。
- **顺序规则**：单原子比序数、相同则向外比、双三键作等价替代、不足补 0 号原子；注意"规则大小"不同于"体积大小"。
- **R/S 判断三步**：排序 → 把最小基团隐到背面 → 看其余三基团大到小的转向，顺时针 R、逆时针 S；最小基团朝向你时按反方向判后取反。
- **生物活性分子**：构型决定活性，如青霉素为 R、R、S 才有杀菌活性；判断时看准最小基团（氢）的朝向即可，氢朝里直接判、氢朝外判后取反。
:::
