# 10.1 亲核取代反应 SN1/SN2

## 一、从一个名字出发：什么是亲核取代

学习有机化学，最忌讳把反应当成一堆要背的箭头。我们换一个思路：先把"亲核取代反应"这个名字拆开，几乎所有要点都藏在名字里。

"亲核"是什么？亲核试剂。亲核试剂长什么样？要么本身直接带着负电荷，要么是一个中性分子里那个带有负电端、带有孤对电子的部分。它的共同特点是：电子比较富裕，喜欢去找原子核（正电荷）。所以"亲核"两个字，描述的就是这个试剂的脾气——它专门去进攻分子中那些感应出正电荷、暴露出原子核的位置。

那它进攻谁？我们把目光转向卤代烃。在一个饱和卤代烷分子里，碳卤键就是它的官能团。由于卤原子的电负性比碳大，碳卤键是高度极化的：卤原子把成键电子往自己这边拉，于是卤原子带上部分负电荷，碳原子带上部分正电荷。

$$\ce{C^{\delta+}-X^{\delta-}}$$

现在画面清楚了：一个带负电的亲核试剂，盯上了这个带部分正电的碳。它扑上去——"亲核"这一步发生了；扑上去之后旧的碳卤键断裂、卤原子被赶走——"取代"这一步发生了。两步合起来，就是**亲核取代反应**（nucleophilic substitution reaction），我们用 $S_N$ 表示。如果被进攻的是一个饱和碳原子，就叫"发生在饱和碳原子上的亲核取代反应"。

:::definition{label=亲核取代反应}
亲核试剂（带负电荷或带孤对电子的负电端）进攻底物分子中带部分正电荷的碳原子，取代掉一个离去基团的反应，称为亲核取代反应，记作 $S_N$。

以溴甲烷与氢氧根的反应为例：
$$\ce{CH3Br + OH^- -> CH3OH + Br^-}$$
其中 $\ce{CH3Br}$ 是**底物**，$\ce{OH^-}$ 是**亲核试剂**，$\ce{Br^-}$ 是**离去基团**。在有机化学里，通常是带负电荷的试剂发起进攻。
:::

## 二、两条路径：SN2 与 SN1 的本质区别

同样是亲核取代，自然界里却走两条不同的路，我们用脚标 2 和 1 区分它们。这个 1 和 2，指的不是反应次数，而是**决速步里有几个分子参与**。

### SN2：双分子、一步完成

2 代表**双分子**反应。哪两个分子？底物分子和亲核试剂。它们共同参与了决速步——也就是说，反应速率既取决于底物的浓度，也取决于亲核试剂的浓度。这两个分子一起作用，形成一个特殊的过渡态：中心碳上同时连着五个基团（原来的三个基团、即将进入的亲核试剂、即将离开的离去基团），这是一个五配位的过渡态。

:::theorem{label=SN2 机理特征}
- **一步完成**：新键的生成（亲核试剂—碳）和旧键的断裂（碳—离去基团）在同一步、协同进行，没有中间体。
- **背面进攻**：亲核试剂必须从离去基团的**背面**扑向碳原子，才能把离去基团从正对面顶出去。
- **构型翻转**：背面进攻的结果，是中心碳的构型像一把被风吹翻的雨伞那样**翻转**（瓦尔登翻转）。若底物碳是手性的，产物构型会反转。
- **决速步双分子**：速率 $\propto$ [底物][亲核试剂]。
:::

### SN1：单分子、分步、碳正离子

1 代表**单分子**反应。这里的"单分子"指决速步只由一个分子完成——底物分子自己先发生碳卤键的异裂，生成一个碳正离子和一个离去基团（如 $\ce{Br^-}$）。这一步只跟底物有关，亲核试剂还没登场。

碳正离子一旦生成，第二步才轮到亲核试剂上前结合，给出产物。所以 $S_N1$ 是**分步**进行的。

:::theorem{label=SN1 机理特征}
- **分步进行**：第一步碳卤键异裂生成碳正离子（决速步、耗能步），第二步亲核试剂与碳正离子结合。
- **平面碳正离子**：生成的碳正离子是平面型（$sp^2$）。
- **外消旋化**：亲核试剂可以从平面的**两面**进攻，所以手性底物会得到外消旋产物（两种构型各半）。
- **可能重排**：碳正离子容易通过基团迁移重排成更稳定的碳正离子，从而出现重排产物。
- **决速步单分子**：速率只 $\propto$ [底物]，与亲核试剂浓度无关。
:::

:::insight{label=为什么 SN1 不看亲核试剂}
$S_N1$ 的决速步是底物自己异裂，亲核试剂根本没参与那一步。所以一旦碳正离子生成，无论亲核试剂强弱，反应都很容易完成——因为面前摆着一个赤裸裸的正电荷。**这就是为什么亲核试剂的亲核性只对 $S_N2$ 有意义，对 $S_N1$ 几乎不起作用。** 记住这一点，后面判断反应快慢时能少走很多弯路。
:::

## 三、四个抓手：什么决定走 SN1 还是 SN2

$S_N1$ 与 $S_N2$ 是一对竞争路径。判断一个具体反应主要走哪条，有四个抓手：底物的烷基结构、离去基团、亲核试剂、溶剂。

### 抓手一：底物的烷基结构（位阻与级数）

$S_N2$ 是要真贴上去背面进攻的，所以**位阻**是它的命门。卤代烷 α 碳（连卤素的碳）和 β 碳（旁边的碳）上取代基越多、越大，亲核试剂越难钻进去，$S_N2$ 越难。

从级数看，随着 α 碳上取代基增多：
- **伯卤代烷（一级）**：位阻小，倾向 $S_N2$。
- **叔卤代烷（三级）**：位阻大，但异裂后能生成较稳定的三级碳正离子，倾向 $S_N1$。
- **仲卤代烷（二级）**：居中，$S_N1$ 与 $S_N2$ 都可能，要看其他条件。

### 抓手二：离去基团（对两者同向）

不论 $S_N2$（被顶掉）还是 $S_N1$（自己先断），那根碳卤键终归都要断。所以一句话：**越容易离去，越容易反应；好的离去基团对 $S_N1$ 和 $S_N2$ 都有利，是"同向"的。**

什么样的离去基团好？离去基团是带着一对电子离去、生成一个带电物种的。它离去后越能稳定自身的电荷（"扔出去后不哭不闹、能安稳存在"），就越是好的离去基团。

:::insight{label=卤素离去能力为何 I > Br > Cl ≫ F}
氟、氯、溴、碘离去后都带一个负电荷，电荷数相同，但碘原子体积最大，负电荷在大原子上能更好地分散、稳定，所以碘是最好的离去基团。顺序为 $\ce{I- > Br- > Cl- \gg F-}$。

实验室里做亲核取代，常选**溴代烷**：碘太容易离去导致选择性差；氯不易离去往往要高温；溴居中，最实用。氟代烷几乎不发生亲核取代。
:::

:::example{label=把坏的离去基团变成好的}
亲核取代不是卤代烷的专利——醇的羟基也能被取代。但羟基是**差**的离去基团，直接顶掉它非常困难。两种常见处理：

1. **酸催化**：加酸让羟基质子化，离去时生成中性水分子，键也被高度极化，反应变易。
$$\ce{R-OH ->[H+] R-OH2+ ->[Nu^-] R-Nu + H2O}$$

2. **磺酰化**：把醇转化成对甲苯磺酸酯（$\ce{TsO-}$）或甲基磺酸酯（甲磺酸酯）。磺酸根是共轭体系，能很好地离域、稳定负电荷，是优秀的离去基团。于是"差离去基团羟基"就变成了"好离去基团磺酸根"，再被亲核试剂顶掉。
$$\ce{R-OH ->[\text{磺酰化}] R-OTs ->[Nu^-] R-Nu + TsO^-}$$
:::

### 抓手三：亲核试剂（只影响 SN2）

亲核试剂在 $S_N2$ 的过渡态中亲身参与，所以它的"亲核性"主要影响 $S_N2$。判断亲核性强弱，要综合四个因素：

1. **是否带电**：带负电荷的一定强于中性分子。例如 $\ce{CH3I}$ 被甲氧基取代，直接用 $\ce{CH3O^-}$（削尖了的）比用中性甲醇（还被质子保护着）容易得多。
2. **体积**：$S_N2$ 要贴上去，试剂越小越灵活越容易钻进位阻区。同是烷氧基负离子，甲氧基最小，亲核性最好。
3. **碱性**：与可极化性共同构成亲核性的内核（下面专门辨析）。
4. **可极化性**：非常关键，决定试剂遇到正电荷时能否"变形"伸过去。

### 抓手四：溶剂（极性有利于 SN1）

:::theorem{label=溶剂效应}
- **极性溶剂有利于 $S_N1$**：$S_N1$ 中间生成带电的碳正离子，极性溶剂能稳定这个带电物种、降低活化能。实验上，三级卤代烷在乙醇中速率设为 1，逐渐增加水的比例，$S_N1$ 速率不断上升，纯水中速率极大。
- **极性溶剂对 $S_N2$ 影响不大**：$S_N2$ 的过渡态没有生成独立的带电物种，对溶剂极性不敏感。**切勿想当然地认为"非极性溶剂有利于 $S_N2$"**——这是常见的错误推断。
:::

## 四、深挖亲核性：碱性 vs 可极化性

亲核性和碱性是两个极易混淆的概念，必须辨析清楚。

:::definition{label=碱性与亲核性}
- **碱性**：试剂与**质子**结合的能力。质子是元素周期表里最小的东西（氢失去电子），所以碱性**不需要考虑位阻**，只看结合质子的能力强弱。其规律我们高中就学过：同周期从左到右碱性减小，同族从上到下碱性减小。
- **亲核性**：试剂与带正电的**碳原子**结合的能力。碳原子上连着别的基团，**有位阻**，所以亲核性除了看结合能力，还要看体积和可极化性。
:::

本质上两者是相通的——都是去结合一个带正电荷的东西，所以亲核性在多数情况下与碱性同向：周期表中从左到右、从上到下，碱性和亲核性都减小。但亲核性额外要算上体积和可极化性，于是出现了偏差。

:::insight{label=什么是可极化性}
以 $\ce{I^-}$ 和 $\ce{Cl^-}$ 为例：两者都带一个负电荷，但碘的亲核性更好。原因在**可极化性**——碘原子体积大、原子核离外层负电荷远、对它的束缚弱，于是当碘靠近一个正电中心时，它的电子云容易**变形**，能"伸出去"主动接触并进攻。氯则原子核束缚强、电子云不易变形，进攻能力受限。

打个比方：一个试剂像蛋黄月饼，外壳硬、不能变形；另一个像青团，外皮软、能变形——内核都是一个蛋黄（电荷），但软硬不同，这就是可极化性的差异。
:::

:::insight{label=为何 I 碱性不强亲核性却强}
氟氯溴碘里，碘碱性不是最强的，但它的可极化性战胜了碱性的弱点：它更容易变形去接触碳原子。所以**亲核性 = 碱性 + 可极化性的综合**。在常见的水、醇等质子溶剂中做亲核取代，碘的亲核性反而更好。

为什么强调"质子溶剂"？因为同族从上到下碱性减弱、可极化性增大，最终亲核性顺序与溶剂有关；在我们常用的极性质子溶剂里，可极化性占了主导。
:::

:::note{label=质子溶剂中常见亲核试剂的亲核性顺序}
$$\ce{RS- (ArS-)} > \ce{CN-} > \ce{I-} > \text{氨/胺} > \ce{RO-(\approx OH-)} > \ce{Br-} > \ce{ArO-} > \ce{Cl-} \gg \ce{H2O} \gg \ce{F-}$$

烷硫基/芳硫基负离子最强，因为硫对负电荷的控制比氧弱得多，可极化性极好。
:::

## 五、底物活性的扩展：三类碳卤键

卤原子所处的环境不同，亲核取代的活性差别巨大：

- **乙烯型（卤代乙烯型）**：卤原子直接连在双键碳或苯环上。卤与共轭体系形成 p-π 共轭，碳卤键非常牢固，**很难**取代；要发生须高温，且常需邻位有强吸电子基。
- **孤立型**：最常见的饱和卤代烷，卤原子连在普通饱和碳上，活性居中。
- **烯丙型（卤代烯丙型）**：卤原子连在烯丙位或苄位。按 $S_N1$ 进行时生成的碳正离子能与双键或苯环共轭、特别稳定，所以**最活泼**。如在 $\ce{AgNO3}$ 作用下，银离子能直接把卤素拽下生成沉淀。

:::theorem{label=三类碳卤键的亲核取代活性顺序}
$$\text{烯丙型} > \text{孤立型（普通卤代烷）} > \text{乙烯型}$$
最不活泼的乙烯型并非不能反应——当苯环上带强吸电子基时，该位置也可被亲核试剂取代（与上一章亲电取代的知识点相互迁移）。
:::

## 六、例题

:::example{label=判断 SN2 反应活性顺序}
**离去基团不同、烷基相同**：比较一组只有卤素不同的卤代烷的 $S_N2$ 活性。

分析：烷基相同，只比离去基团好坏，越易离去越活泼，故
$$\ce{R-I} > \ce{R-Br} > \ce{R-Cl}$$
注意：离去基团对 $S_N1$ 和 $S_N2$ 同向，此顺序对两者都成立。

**烷基不同、离去基团相同**：只比碳卤键周围的位阻，α 碳与 β 碳位阻越小活性越高。
:::

:::example{label=同族亲核试剂的速率比较}
比较烷硫基负离子（$\ce{RS-}$）与烷氧基负离子（$\ce{RO-}$）发起 $S_N2$ 的速率。

硫与氧同族，硫在下方、可极化性更好，对负电荷控制更弱、更易变形进攻，故 $\ce{RS-}$ 亲核性更强、反应更快。

再比较苯氧基负离子与甲氧基负离子：甲氧基更快。原因有二——苯氧基中氧负电荷与苯环共轭被分散，进攻性减弱；且苯氧基体积大、甲氧基小更灵活。
:::

## 七、易错点

:::pitfall{label=四个常见陷阱}
1. **把亲核性当碱性**：碱性看结合质子（无位阻），亲核性看结合碳（有位阻、要算体积和可极化性）。$\ce{I-}$ 碱性弱但亲核性强，正是反例。
2. **以为亲核试剂对 $S_N1$ 也重要**：$S_N1$ 决速步亲核试剂不参与，其亲核性几乎不影响快慢，只对 $S_N2$ 有意义。
3. **误判溶剂对 $S_N2$ 的作用**：极性溶剂只明显有利于 $S_N1$，对 $S_N2$ 影响不大，不要反推"非极性利于 $S_N2$"。
4. **忽视 β 碳位阻**：$S_N2$ 不光看 α 碳，β 碳上的位阻同样会严重削弱反应。
:::

:::note{label=本节小结}
- 亲核取代 $S_N$：亲核试剂进攻碳卤键中带部分正电的碳，取代离去基团；底物—亲核试剂—离去基团是三要素。
- **$S_N2$**：双分子、一步协同、背面进攻、构型翻转、五配位过渡态；对位阻和亲核试剂敏感。
- **$S_N1$**：单分子、分步、生成平面碳正离子、外消旋化、可重排；只看底物，受极性溶剂促进。
- 四个抓手：①烷基结构（伯→$S_N2$，叔→$S_N1$）；②离去基团（越易离去越好，对两者同向，$\ce{I>Br>Cl\gg F}$）；③亲核试剂（只影响 $S_N2$，是带电、体积、碱性、可极化性的综合）；④溶剂（极性利于 $S_N1$）。
- 亲核性核心 = 碱性 + 可极化性；可极化性指电子云遇正电荷时的变形能力。
- 底物活性：烯丙型 > 孤立型 > 乙烯型。
:::
