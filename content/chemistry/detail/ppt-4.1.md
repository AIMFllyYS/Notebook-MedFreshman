:::insight{label=从"看似简单"的烷烃说起}
烷烃看起来是最"无聊"的有机物——只有 C 和 H，全是单键，没有官能团，化学性质不活泼。但恰恰是这种"无聊"，让烷烃成为有机化学的"地基"：理解了烷烃的结构和构象，你就掌握了所有 sp³ 杂化碳的行为规律，这为后续学习环烷烃、卤代烃、醇等打下基础。

更有趣的是，烷烃虽然"简单"，却隐藏着几个深刻的问题：为什么丁烷有 2 种异构体而戊烷有 3 种？为什么乙烷的交叉式比重叠式稳定？为什么 $\ce{C-C}$ 单键看似可以自由旋转，实际上却存在"构象偏好"？这些问题都指向本节的核心——**结构、构型与构象**的区分。
:::

## 烃的分类与烷烃定义

:::definition{label=烃与烷烃}
**烃**（hydrocarbon）是只含碳、氢两种元素的有机化合物，按结构分两大类：

- **脂肪烃**：开链烃（烷烃、烯烃、炔烃）+ 脂环烃
- **芳香烃**：含苯环等芳香结构的烃

**烷烃**（alkane）：分子中碳、氢原子都以单键相连的烃类化合物，称饱和碳氢化合物。按碳骨架分：
- **开链烷烃**（脂肪烃）：碳原子连接成链，通式 $\ce{C_nH_{2n+2}}$
- **环烷烃**：碳原子连接成环
:::

## 同分异构现象

:::definition{label=构造异构体}
**构造**（constitution）：分子中原子相互连接的次序。

**构造异构体**：分子式相同而构造式不同的现象。烷烃从丁烷开始有同分异构现象。

异构体数目随碳数增长极快：
- 丁烷（4C）：2 种
- 戊烷（5C）：3 种（正、异、新）
- 己烷（6C）：5 种
- 癸烷（10C）：75 种
- 十五烷（15C）：4347 种
:::

:::insight{label=为什么异构体数目爆炸式增长}
每增加一个碳原子，它可以选择连接到主链的不同位置，产生新的分支方式。这种"组合爆炸"使异构体数目随碳数增长极快。15 个碳就有 4347 种异构体——这就是为什么普通命名法（正/异/新）无法应付复杂烷烃，必须用系统命名法。
:::

## 碳原子的分类

:::definition{label=伯仲叔季碳}
按碳原子所连其他碳原子的数目分类：

| 类型 | 符号 | 定义 | 上面的氢 |
|------|------|------|----------|
| 伯碳（一级碳） | $\ce{1^{\circ}C}$ | 与 1 个碳相连 | 一级氢 $\ce{1^{\circ}H}$ |
| 仲碳（二级碳） | $\ce{2^{\circ}C}$ | 与 2 个碳相连 | 二级氢 $\ce{2^{\circ}H}$ |
| 叔碳（三级碳） | $\ce{3^{\circ}C}$ | 与 3 个碳相连 | 三级氢 $\ce{3^{\circ}H}$ |
| 季碳（四级碳） | $\ce{4^{\circ}C}$ | 与 4 个碳相连 | 无氢 |

例如新戊烷 $\ce{(CH3)4C}$：中心碳是季碳（$\ce{4^{\circ}C}$），4 个甲基碳都是伯碳（$\ce{1^{\circ}C}$）。
:::

:::pitfall{label=把"季碳"写成"季碳上有氢"}
季碳 $\ce{4^{\circ}C}$ 连 4 个碳，**没有氢**！这是与伯/仲/叔碳最大的区别。考试时不要写"季碳上的氢"，因为季碳上根本没有氢。
:::

## 烷烃的结构

:::definition{label=构型}
**构型**（configuration）：在一定构造的分子中，原子在空间的排列状况。

甲烷分子为**四面体构型**，碳为 sp³ 杂化，4 个 C—H σ 键夹角 $109.5^\circ$。
:::

:::theorem{label=σ键的特点}
烷烃中所有 C—C 和 C—H 键都是 **σ 键**——沿键轴方向"头对头"重叠形成的共价键。σ 键的特点：

1. 电子云沿键轴对称分布
2. **可沿键轴自由旋转**（这是构象变化的基础）
3. 重叠程度大，键较强（C—C 键能约 345 kJ/mol，C—H 约 415 kJ/mol）

乙烷 $\ce{CH3-CH3}$ 中 C—C σ 键可沿键轴旋转，导致甲基上的 H 在空间不同排布——这就是**构象**。
:::

## 构象——本节的核心

:::definition{label=构象}
**构象**（conformation）：一定构型的分子中，由于**单键的旋转**而导致原子或基团在空间的不同排布。

σ 键可沿键轴旋转，但旋转需要克服一定能垒（乙烷约 12.1 kJ/mol）。
:::

:::insight{label=为什么单键旋转有能垒}
理论上 σ 键可以自由旋转，但实际上旋转需要克服**扭转张力**（torsional strain）。当两个碳上的取代基处于"重叠"位置时，它们的电子云互相排斥，能量升高；处于"交叉"位置时，电子云互相远离，能量最低。

这种能垒不大（12 kJ/mol，远小于共价键能），所以在室温下分子可以通过热运动克服能垒，在不同构象间快速转换。但**优势构象**（能量最低的构象）占主导。
:::

### 表示构象的方法

:::theorem{label=构象的三种表示法}
1. **楔形结构式**：
   - 直线键：与纸共平面的键
   - 实楔形键：伸向纸面外的键
   - 虚楔形键：伸向纸面里的键

2. **锯架式（锲型式）**：从斜侧面观察 C—C 键，能直观看到两个碳上的取代基

3. **Newman 投影式**：把旋转的 σ 键轴放在垂直于纸面的位置，沿键轴由上至下垂直投影：
   - 靠近眼睛的 C 原子用**点**表示
   - 远离眼睛的 C 原子用**圆圈**表示
   - 三个键从点（或圆圈）呈 120° 角辐射出
:::

:::pitfall{label=Newman 投影式中前后碳混淆}
Newman 投影式中：
- **点**（中心）= 前端碳（靠近眼睛）
- **圆圈**= 后端碳（远离眼睛）

考试时一定要分清前后，否则会把交叉式和重叠式搞反。记忆："点在前，圈在后"。
:::

## 乙烷的构象

:::theorem{label=乙烷的两种极限构象}
乙烷 $\ce{CH3-CH3}$ 沿 C—C 键旋转，主要有两种极限构象：

1. **交叉式**（staggered）：前后两个碳上的 H 互相交错（夹角 60°）
   - 能量**最低**
   - 最稳定
   - 为**优势构象**

2. **重叠式**（eclipsed）：前后两个碳上的 H 互相重叠
   - 能量**最高**
   - 最不稳定

两者能量差约 12.1 kJ/mol（扭转能垒）。
:::

:::insight{label=为什么交叉式比重叠式稳定}
交叉式中，前后两个碳上的 H 距离最远（电子云排斥最小），能量最低。重叠式中，H 与 H 距离最近，电子云排斥最大，能量最高。

可以想象：两个人面对面站着，如果他们的胳膊（代表 H）对齐，就会互相碰撞（重叠式）；如果胳膊错开，就不会碰撞（交叉式）。所以乙烷分子"偏好"交叉式——这是能量最低的稳定构象。
:::

## 丁烷的构象

:::theorem{label=丁烷的四种极限构象}
丁烷 $\ce{CH3-CH2-CH2-CH3}$ 沿 C2-C3 键旋转，有四种极限构象（按能量从低到高）：

1. **对位交叉式**（anti-periplanar）：两个 $\ce{CH3}$ 在 180°（最远）
   - 能量**最低**，最稳定
   - 占 63%

2. **邻位交叉式**（gauche）：两个 $\ce{CH3}$ 在 60°
   - 能量略高于对位交叉
   - 占 37%

3. **部分重叠式**：两个 $\ce{CH3}$ 在 120°（一个交叉一个重叠）
   - 能量较高

4. **完全重叠式**：两个 $\ce{CH3}$ 在 0°（重叠）
   - 能量**最高**，最不稳定

室温下丁烷的优势构象分布：对位交叉式 63% + 邻位交叉式 37%（重叠式能量太高，占比极小）。
:::

:::example{label=丁烷构象能量分析}
**题目**：为什么对位交叉式比邻位交叉式更稳定？

**分析**：两者都是交叉式（无扭转张力），但 $\ce{CH3}$ 之间的空间关系不同。

**解**：
- **对位交叉式**：两个 $\ce{CH3}$ 在 180°，距离最远，空间位阻最小 → 能量最低
- **邻位交叉式**：两个 $\ce{CH3}$ 在 60°，距离较近，有一定空间位阻（gauche 相互作用）→ 能量略高

两者能量差约 3.8 kJ/mol（gaune 相互作用能）。室温下两者都在，但对位交叉式占主导（63% vs 37%）。
:::

## 构造、构型、构象的区分

:::theorem{label=构造/构型/构象的区别}
| 概念 | 定义 | 变化条件 | 例子 |
|------|------|----------|------|
| 构造（constitution） | 原子连接次序 | 须破坏化学键 | 正戊烷、异戊烷、新戊烷（碳骨架不同） |
| 构型（configuration） | 一定构造下原子在空间的排布 | 须破坏化学键 | sp³ 杂化 C 的四面体构型、顺反异构体 |
| 构象（conformation） | 一定构型下因单键旋转产生的空间排布 | 不破坏化学键 | 乙烷的交叉式和重叠式 |

**关键区别**：构型是固定的，构象一般是瞬间的。构型的变化须破坏化学键，而构象的变化则不会发生键的断裂。
:::

:::pitfall{label=把构象异构体当成不同的化合物}
构象异构体（如乙烷的交叉式和重叠式）**不是不同的化合物**——它们是同一分子的不同空间排布，可以通过单键旋转互相转化。在室温下，分子在不同构象间快速转换（每秒 $10^{10}$ 次以上），无法分离。

而构型异构体（如顺反异构体）是**不同的化合物**，需要破坏化学键才能互变，可以分离。这是构象与构型的根本区别。
:::

:::example{label=区分构造/构型/构象}
**题目**：判断下列变化属于构造、构型还是构象变化。

(a) 正戊烷 → 异戊烷  (b) 顺-2-丁烯 → 反-2-丁烯  (c) 乙烷交叉式 → 重叠式

**解**：
- (a) **构造变化**：碳骨架改变（直链变支链），须破坏 C—C 键
- (b) **构型变化**：双键不能自由旋转，须破坏 π 键才能顺反互变
- (c) **构象变化**：单键旋转，不破坏任何键

记忆：构造看"骨架"，构型看"固定空间排布"，构象看"旋转中的瞬间状态"。
:::

:::note{label=本节小结}
- 烷烃是饱和烃，开链烷烃通式 $\ce{C_nH_{2n+2}}$；
- 同分异构体数目随碳数爆炸式增长（15C 已有 4347 种），必须用系统命名法；
- 碳原子分四类：伯（$\ce{1^{\circ}}$）、仲（$\ce{2^{\circ}}$）、叔（$\ce{3^{\circ}}$）、季（$\ce{4^{\circ}}$）；季碳无氢；
- σ 键可沿键轴旋转，产生**构象**；
- **乙烷构象**：交叉式（能量低，优势）vs 重叠式（能量高），能差 12.1 kJ/mol；
- **丁烷构象**：对位交叉式（63%，最稳定）> 邻位交叉式（37%）> 部分重叠式 > 完全重叠式；
- **构造/构型/构象**：构造看连接次序，构型看固定空间排布，构象看旋转中的瞬间状态；构象变化不破坏键，构型/构造变化须破坏键；
- 下一节我们将学习烷烃的化学性质——主要是自由基卤代反应及其机理。
:::
