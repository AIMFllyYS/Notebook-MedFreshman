## 事件之间的"关系网"：为什么要研究事件的运算？

上一节我们知道：事件就是样本空间 $\Omega$ 的子集。掷骰子时，"出现偶数点"是子集 $\{2,4,6\}$，"点数大于4"是子集 $\{5,6\}$。

可实际问题远不止"单独一个事件发生不发生"这么简单：

- 买一张彩票，我们关心"中一等奖**或**中二等奖"；
- 做质量检验，我们关心"产品**既**合格**又**按时交货"；
- 投资决策，我们关心"股票涨**但**基金没跌"；
- 天气预报，我们想描述"**不**下雨"。

这里的"或""又""但""不"，在数学上对应**事件的并、交、差、对立**。因为事件是集合，所以集合的那套运算语言可以直接拿来用——这正是上一节把事件定义成子集的深意所在。

本节就来系统梳理事件之间的六种关系和四种运算，以及串起它们的运算律（含**德摩根定律**），为后续计算概率搭好语言脚手架。

---

## 一、事件之间的六种关系

### 1.1 包含（子事件）

:::definition{label=事件的包含}
设 $A, B$ 是同一样本空间 $\Omega$ 上的两个事件。若 $A$ 的每个样本点都属于 $B$，即

$$A \subseteq B$$

则称**事件 $A$ 包含于事件 $B$**（或"$B$ 包含 $A$"），也称"$A$ 发生必导致 $B$ 发生"。

特别地，对任意事件 $A$，始终有 $\varnothing \subseteq A \subseteq \Omega$。
:::

**掷骰子直观举例。** 设 $\Omega=\{1,2,3,4,5,6\}$，令

$$A = \{6\},\quad B = \{\text{点数}>4\} = \{5,6\}.$$

$A=\{6\}\subseteq B=\{5,6\}$：掷出6，必然点数大于4，所以 $A$ 发生 $\Rightarrow$ $B$ 发生。反之不然——掷出5，$B$ 发生，$A$ 不发生。

:::insight{label=包含的频率含义}
若 $A\subseteq B$，则在大量重复试验中，$A$ 发生的次数不超过 $B$ 发生的次数，因此频率满足

$$\frac{\text{A发生次数}}{n} \leqslant \frac{\text{B发生次数}}{n}$$

这就是后面**单调性**定理的频率直觉：$A\subseteq B \Rightarrow P(A)\leqslant P(B)$。
:::

### 1.2 相等

:::definition{label=事件相等}
若 $A \subseteq B$ 且 $B \subseteq A$，即两者包含彼此，则称事件 $A$ 与事件 $B$ **相等**，记作

$$A = B.$$
:::

相等意味着两个事件虽然"名字"不同，但包含完全相同的样本点。例如掷一颗骰子：

$$\{\text{点数为偶数}\} = \{2, 4, 6\} = \{\text{能被2整除的点数}\}.$$

用不同语言描述了同一个集合，所以它们作为事件是相等的。

## 二、四种基本运算

### 2.1 和事件（并）

:::definition{label=和事件（并事件）}
事件 $A$ 与 $B$ 的**和事件**（又称**并事件**）定义为

$$A \cup B = \{\omega \mid \omega \in A \text{ 或 } \omega \in B\}$$

即：$A \cup B$ 发生 $\Longleftrightarrow$ $A$ 发生**或** $B$ 发生（或两者同时发生）。

推广到有限个乃至可列个事件：

$$\bigcup_{i=1}^{n} A_i = A_1 \cup A_2 \cup \cdots \cup A_n, \qquad \bigcup_{i=1}^{\infty} A_i.$$
:::

**直觉图像。** 想象韦恩图（Venn diagram）：$A \cup B$ 是两个圆圈合在一起的区域，涂满阴影的整块。

**掷骰子例子。** $A=\{1,2,3\}$（点数$\leqslant 3$），$B=\{2,4,6\}$（偶数点），则

$$A \cup B = \{1,2,3,4,6\}.$$

掷出1、2、3、4或6，都算发生了 $A\cup B$（只有掷出5时，$A\cup B$ 不发生）。

::video{id=ch01-1.2-ops}

### 2.2 积事件（交）

:::definition{label=积事件（交事件）}
事件 $A$ 与 $B$ 的**积事件**（又称**交事件**）定义为

$$A \cap B = AB = \{\omega \mid \omega \in A \text{ 且 } \omega \in B\}$$

即：$AB$ 发生 $\Longleftrightarrow$ $A$ 与 $B$ **同时**发生。

推广：

$$\bigcap_{i=1}^{n} A_i = A_1 A_2 \cdots A_n, \qquad \bigcap_{i=1}^{\infty} A_i.$$
:::

**直觉图像。** $A \cap B$ 是韦恩图中两圆重叠的区域（"交叉地带"）。

**掷骰子续。** 承上，$A=\{1,2,3\}$，$B=\{2,4,6\}$，则

$$AB = A\cap B = \{2\}.$$

只有掷出2，才同时满足"点数$\leqslant 3$"和"偶数点"两个条件。

::interactive{id=ch01-1.2-venn}

### 2.3 差事件

:::definition{label=差事件}
事件 $A$ 与 $B$ 的**差事件**定义为

$$A - B = A \setminus B = \{\omega \mid \omega \in A \text{ 且 } \omega \notin B\}$$

即：$A-B$ 发生 $\Longleftrightarrow$ $A$ 发生**但** $B$ **不**发生。
:::

**直觉图像。** 从 $A$ 的圆圈里"抠掉"与 $B$ 重叠的部分，剩余的"$A$ 专属区域"。

**掷骰子续。** $A-B=\{1,3\}$（点数$\leqslant 3$ 且为奇数）。

:::insight{label=差事件的等价写法}
差事件有一个极其有用的等价表达：

$$A - B = A \cap \overline{B} = A\overline{B}$$

其中 $\overline{B}$ 是 $B$ 的对立事件（见下文）。

这个等价式在实际计算中频繁使用：当你需要算 $P(A-B)$，但不知道 $A$ 与 $B$ 的包含关系时，就把它转化为 $A\overline{B}$，再利用更多性质处理。老师课堂上专门强调过这一点！
:::

### 2.4 对立事件（补）

:::definition{label=对立事件}
事件 $A$ 的**对立事件**（又称**余事件**或**补事件**）定义为

$$\overline{A} = \Omega \setminus A = \{\omega \mid \omega \in \Omega, \omega \notin A\}$$

即：$\overline{A}$ 发生 $\Longleftrightarrow$ $A$ **不**发生。

对立事件满足：

$$A \cup \overline{A} = \Omega, \qquad A \cap \overline{A} = \varnothing, \qquad \overline{\overline{A}} = A.$$
:::

**掷骰子例。** 若 $A=\{1,3,5\}$（奇数点），则 $\overline{A}=\{2,4,6\}$（偶数点）。两者互不包含、合起来恰好是 $\Omega$。

## 三、互斥（互不相容）与对立

这两个概念初学者最容易混淆，需要专节厘清。

:::definition{label=互斥事件}
若 $A \cap B = \varnothing$，则称事件 $A$ 与 $B$ **互斥**（或**互不相容**），记作 $A \cdot B = \varnothing$。

含义：$A$ 与 $B$ **不能同时发生**；在韦恩图中，两圆不重叠。

推广：若 $n$ 个事件 $A_1, A_2, \ldots, A_n$ 两两互斥（即 $i\neq j$ 时 $A_i \cap A_j = \varnothing$），称它们**两两互不相容**。
:::

:::definition{label=对立事件（完整版）}
若 $A \cap B = \varnothing$ **且** $A \cup B = \Omega$，则称 $A$ 与 $B$ **互为对立事件**，记 $B = \overline{A}$。

含义：$A$ 与 $B$ 不仅不能同时发生，而且**必居其一**（非此即彼）。
:::

:::pitfall{label=互斥 vs 对立：最常见混淆}
**对立是互斥的特殊情形，但互斥不一定是对立。**

掷骰子，令 $A=\{1,2\}$，$B=\{3,4\}$。

- $A\cap B=\varnothing$，所以 $A$ 与 $B$ **互斥**。
- $A\cup B=\{1,2,3,4\}\neq\Omega=\{1,2,3,4,5,6\}$，所以 $A$ 与 $B$ **不对立**（因为掷出5或6，$A$、$B$ 都不发生）。

**结论：对立 $\Rightarrow$ 互斥；互斥 $\not\Rightarrow$ 对立。**

对立必须是整个 $\Omega$ 被两个事件"无缝覆盖"，缺一不可。
:::

## 四、完整示例与韦恩图练习

我们用一个贯穿例题，把六种关系和四种运算全部串联起来。

:::example{label=掷骰子综合运算}
掷一颗均匀骰子，$\Omega=\{1,2,3,4,5,6\}$。设

$$A=\{\text{出现奇数点}\}=\{1,3,5\},\quad B=\{\text{点数}\geqslant 4\}=\{4,5,6\}.$$

**求以下各事件并说明其含义：**

$A\cup B$，$AB$，$A-B$，$B-A$，$\overline{A}$，$\overline{B}$，$\overline{A}\cap\overline{B}$，$\overline{A\cup B}$。

**解：**

**第一步**，明确 $A$ 和 $B$ 的样本点：$A=\{1,3,5\}$，$B=\{4,5,6\}$，重叠处为 $\{5\}$。

**第二步**，逐一计算：

$$\begin{aligned}
A \cup B &= \{1,3,4,5,6\} && \text{（奇数点或点数}\geqslant 4\text{）}\\
AB &= \{5\} && \text{（奇数且点数}\geqslant 4\text{，仅有5）}\\
A - B &= \{1,3\} && \text{（奇数但点数}<4\text{）}\\
B - A &= \{4,6\} && \text{（点数}\geqslant 4\text{ 但为偶数）}\\
\overline{A} &= \{2,4,6\} && \text{（偶数点）}\\
\overline{B} &= \{1,2,3\} && \text{（点数}<4\text{）}\\
\overline{A}\cap\overline{B} &= \{2\} && \text{（偶数且点数}<4\text{，即仅2）}\\
\overline{A\cup B} &= \{2\} && \text{（同上，验证德摩根定律）}
\end{aligned}$$

**第三步**，验证结果自洽：

- $A$ 与 $B$ 是否互斥？$AB=\{5\}\neq\varnothing$，故**不互斥**。
- $A$ 与 $\overline{A}$ 是否对立？$A\cup\overline{A}=\{1,3,5\}\cup\{2,4,6\}=\Omega$ 且 $A\cap\overline{A}=\varnothing$，故**对立**。

注意最后一行：$\overline{A}\cap\overline{B}=\{2\}=\overline{A\cup B}$，这正是**德摩根定律**的体现，下节专讲。
:::

## 五、运算律与德摩根定律

事件的并、交、对立满足一组与集合论完全对应的运算律，是化简复杂事件表达式的工具。

:::theorem{label=事件运算律}
设 $A, B, C$ 为 $\Omega$ 上的任意事件，则：

**交换律：**

$$A \cup B = B \cup A, \qquad AB = BA.$$

**结合律：**

$$(A \cup B) \cup C = A \cup (B \cup C), \qquad (AB)C = A(BC).$$

**分配律：**

$$A(B \cup C) = AB \cup AC, \qquad A \cup (BC) = (A \cup B)(A \cup C).$$

**德摩根定律（对偶律）：**

$$\overline{A \cup B} = \overline{A} \cap \overline{B}, \qquad \overline{A \cap B} = \overline{A} \cup \overline{B}.$$

推广到 $n$ 个事件：

$$\overline{\bigcup_{i=1}^{n} A_i} = \bigcap_{i=1}^{n} \overline{A_i}, \qquad \overline{\bigcap_{i=1}^{n} A_i} = \bigcup_{i=1}^{n} \overline{A_i}.$$

进一步推广到可列个事件：

$$\overline{\bigcup_{i=1}^{\infty} A_i} = \bigcap_{i=1}^{\infty} \overline{A_i}, \qquad \overline{\bigcap_{i=1}^{\infty} A_i} = \bigcup_{i=1}^{\infty} \overline{A_i}.$$
:::

:::derivation{label=德摩根定律的严格证明}
以 $\overline{A\cup B}=\overline{A}\cap\overline{B}$ 为例，按集合相等的证明套路："双向包含"。

**方向一：$\overline{A\cup B}\subseteq\overline{A}\cap\overline{B}$。**

设 $\omega\in\overline{A\cup B}$，则 $\omega\notin A\cup B$，即 $\omega\notin A$ 且 $\omega\notin B$，即 $\omega\in\overline{A}$ 且 $\omega\in\overline{B}$，因此 $\omega\in\overline{A}\cap\overline{B}$。

**方向二：$\overline{A}\cap\overline{B}\subseteq\overline{A\cup B}$。**

设 $\omega\in\overline{A}\cap\overline{B}$，则 $\omega\in\overline{A}$ 且 $\omega\in\overline{B}$，即 $\omega\notin A$ 且 $\omega\notin B$，故 $\omega\notin A\cup B$，即 $\omega\in\overline{A\cup B}$。

两个方向都成立，故 $\overline{A\cup B}=\overline{A}\cap\overline{B}$。$\square$

$\overline{A\cap B}=\overline{A}\cup\overline{B}$ 的证明完全类似，留给读者练习。
:::

:::insight{label=德摩根定律的语言直觉}
德摩根定律有个超级好记的"翻译口诀"：

> **"不(A 或 B)" = "不A 且 不B"**  
> **"不(A 且 B)" = "不A 或 不B"**

对应到逻辑语言：
- "今天既没有下雨也没有刮风"＝"今天没有下雨，**并且**今天没有刮风"；
- "今天下雨和刮风没有**同时**发生"＝"今天没有下雨，**或者**今天没有刮风"。

德摩根定律的强大之处在于：它允许我们把"对立号搬进括号里"，同时把并号换成交号（或反过来）。这是化简含多重对立符号的事件表达式的核心手法。
:::

## 六、运算律的应用：化简复杂事件表达式

老师在课堂上展示了一道需要联合运用运算律的练习题，我们完整复现并详解。

:::example{label=化简练习：联合运用结合律与分配律}
设 $A, B$ 是两个事件，化简表达式

$$(A \cup \overline{A}B)(A \cup \overline{A}\,\overline{B}).$$

**解析（逐步演算，每步说明依据）：**

**第一步**，先处理括号内的子式。注意 $A\cup\overline{A}B$：

利用**分配律** $A\cup(XY)=(A\cup X)(A\cup Y)$，令 $X=\overline{A}$，$Y=B$：

$$A \cup \overline{A}B = (A\cup\overline{A})(A\cup B) = \Omega \cdot (A\cup B) = A\cup B.$$

（因为 $A\cup\overline{A}=\Omega$，而任何事件与 $\Omega$ 取交仍是自身。）

类似地处理第二个括号 $A\cup\overline{A}\,\overline{B}$：

$$A \cup \overline{A}\,\overline{B} = (A\cup\overline{A})(A\cup\overline{B}) = \Omega\cdot(A\cup\overline{B}) = A\cup\overline{B}.$$

**第二步**，代入原式：

$$(A\cup B)(A\cup\overline{B}).$$

再次用分配律，令 $X=B$，$Y=\overline{B}$：

$$(A\cup B)(A\cup\overline{B}) = A\cup(B\cdot\overline{B}) = A\cup\varnothing = A.$$

（因为 $B\cap\overline{B}=\varnothing$，$A\cup\varnothing=A$。）

**结论：**

$$(A \cup \overline{A}B)(A \cup \overline{A}\,\overline{B}) = A.$$

这道题的关键路径：看到"同类符号成群"时用**结合律**先归拢，再反向用**分配律**提取"公因式"，最后利用 $X\cap\overline{X}=\varnothing$ 或 $X\cup\overline{X}=\Omega$ 消掉多余项。
:::

:::example{label=用德摩根定律转化：老师课堂例题}
设 $A,B$ 是两个事件，已知 $P(A)=P(B)=0.5$，证明

$$P(AB) = P(\overline{A}\,\overline{B}).$$

**证明思路分析：**

等式右边含有两个对立符号加上交（积），无法直接用加法公式展开。注意到对立的交可以"翻转"成并的对立——这正是德摩根定律！

**第一步**，对右边施加德摩根定律：

$$P(\overline{A}\,\overline{B}) = P(\overline{A}\cap\overline{B}) = P(\overline{A\cup B}).$$

**第二步**，对立事件的概率公式：

$$P(\overline{A\cup B}) = 1 - P(A\cup B).$$

**第三步**，加法公式展开 $P(A\cup B)$：

$$P(A\cup B) = P(A)+P(B)-P(AB) = 0.5+0.5-P(AB) = 1-P(AB).$$

**第四步**，代入：

$$P(\overline{A}\,\overline{B}) = 1-(1-P(AB)) = P(AB). \quad \square$$

**关键步骤复盘：**先用**德摩根**把"交的对立"变成"并的对立"，再用**对立事件**公式翻转，最后用**加法公式**打开，一气呵成。这三板斧是处理含对立符号概率计算的标准套路。
:::

## 七、从集合语言看事件：完整对照表

事件关系和运算全部对应集合运算，下面这张对照表请务必熟记：

| 集合语言 | 事件语言 | 说明 |
|:---:|:---:|:---:|
| $A \subseteq B$ | $A$ 包含于 $B$ | $A$ 发生 $\Rightarrow$ $B$ 发生 |
| $A = B$ | $A$ 等于 $B$ | 样本点集合完全相同 |
| $A \cup B$ | $A$ 与 $B$ 的和事件 | $A$、$B$ 至少一个发生 |
| $A \cap B = AB$ | $A$ 与 $B$ 的积事件 | $A$、$B$ 同时发生 |
| $A - B = A\overline{B}$ | $A$ 与 $B$ 的差事件 | $A$ 发生但 $B$ 不发生 |
| $\overline{A} = \Omega - A$ | $A$ 的对立事件 | $A$ 不发生 |
| $A \cap B = \varnothing$ | $A$、$B$ 互斥（互不相容） | $A$、$B$ 不能同时发生 |
| $A \cap B = \varnothing$ 且 $A \cup B = \Omega$ | $A$、$B$ 互为对立事件 | 非此即彼，必居其一 |

:::insight{label=把事件画成集合的好处}
为什么要把事件语言翻译成集合语言？

1. **集合有图形**（韦恩图），可以直观验证结论是否正确；
2. **集合有成熟运算律**，复杂的"事件逻辑"能被代数化简；
3. **集合是概率论公理化的基础**——概率 $P$ 是定义在 $\sigma$ 域（事件域）上的函数，而 $\sigma$ 域就是满足特定条件的集合族（这是老师课堂上引入的概念：$\sigma$ 域对并、补运算封闭，保证我们所有感兴趣的运算结果仍是合法事件）。

学会用集合语言刻画事件，就是在学用**唯一正式语言**和概率论对话。
:::

## 八、综合例题：用集合语言表达生活事件

:::example{label=三个事件的综合表达}
某工厂生产一批零件，每个零件可能有三种缺陷：$A$="尺寸超差"，$B$="表面粗糙"，$C$="材料不合格"。用 $A, B, C$ 的运算表达以下事件：

1. 三种缺陷都没有（零件完全合格）；
2. 至少有一种缺陷；
3. 恰好有一种缺陷；
4. 至多有两种缺陷；
5. $A$ 有缺陷，但 $B$ 和 $C$ 都没有缺陷。

**解：**

**事件1**，"三种缺陷都没有"，即 $A$ 不发生、$B$ 不发生、$C$ 不发生，三者同时（取交）：

$$\overline{A} \cap \overline{B} \cap \overline{C} = \overline{A}\,\overline{B}\,\overline{C}.$$

或等价地，由德摩根：$\overline{A\cup B\cup C}$。

**事件2**，"至少一种缺陷"，即 $A, B, C$ 中至少一个发生，直接取并：

$$A \cup B \cup C.$$

也可理解为"不是三种都没有"，即事件1的对立：$\overline{\overline{A}\,\overline{B}\,\overline{C}}=A\cup B\cup C$（德摩根）。

**事件3**，"恰好一种缺陷"，枚举三种情况（$A$ 有但 $B,C$ 没有；$B$ 有但 $A,C$ 没有；$C$ 有但 $A,B$ 没有），三者互斥取并：

$$A\overline{B}\,\overline{C} \cup \overline{A}B\overline{C} \cup \overline{A}\,\overline{B}C.$$

**事件4**，"至多两种缺陷"，是"三种都有"的对立：

$$\overline{ABC}.$$

由德摩根等价地写成 $\overline{A}\cup\overline{B}\cup\overline{C}$（即"$A$、$B$、$C$ 中至少一个不发生"）。

**事件5**，"$A$ 有缺陷，$B$ 和 $C$ 都没有"，直接取交：

$$A \cap \overline{B} \cap \overline{C} = A\overline{B}\,\overline{C}.$$

**小提示：** 碰到"至少""至多""恰好"这三个词时，有一个通用策略：  
- "**至少**一个"→ 取并，或考虑对立（"全都不发生"的反面）；  
- "**恰好**$k$个"→ 枚举所有选 $k$ 个发生、其余不发生的组合，再取并；  
- "**至多**$k$个"→ 考虑对立（"超过$k$个发生"的反面）。
:::

:::example{label=老师课堂化简题再现}
已知 $A,B,C$ 是三个事件，化简

$$D = (A\overline{B}C)\cup(A\overline{B}\,\overline{C})\cup(\overline{A}\,\overline{B}C)\cup(\overline{A}\,\overline{B}\,\overline{C}).$$

**解：**

**第一步**，观察四项的共同特征：每一项都含 $\overline{B}$（即 $B$ 不发生）。提取公因子：

$$D = \overline{B}\bigl[(AC)\cup(A\overline{C})\cup(\overline{A}C)\cup(\overline{A}\,\overline{C})\bigr].$$

**第二步**，化简方括号内。这四项恰好覆盖了 $A, C$ 的所有组合（$AC, A\overline{C}, \overline{A}C, \overline{A}\,\overline{C}$ 是四个互不相容的基本组合），合在一起是 $\Omega$：

$$AC \cup A\overline{C} \cup \overline{A}C \cup \overline{A}\,\overline{C}$$
$$= A(C\cup\overline{C}) \cup \overline{A}(C\cup\overline{C}) \quad\text{（分配律）}$$
$$= A\Omega \cup \overline{A}\Omega = A\cup\overline{A} = \Omega.$$

**第三步**，代入：

$$D = \overline{B} \cap \Omega = \overline{B}.$$

**结论：** $D=\overline{B}$，即"$B$ 不发生"。

这道题教给我们的方法：**先提取公共因子，再识别内部是否覆盖了全部可能的组合。** 如果内部的所有情况合并是 $\Omega$，就可以直接消去。
:::

## 九、易错点汇总

:::pitfall{label=本节五大易错点}

**易错1：混淆"互斥"与"对立"。**  
互斥仅要求 $A\cap B=\varnothing$；对立还额外要求 $A\cup B=\Omega$。对立一定互斥，互斥不一定对立。

**易错2：对立号的作用范围。**  
$\overline{AB}$ 是先取交再取对立，等于 $\overline{A}\cup\overline{B}$（德摩根）。  
$\overline{A}\,\overline{B}$ 是先分别取对立再取交，等于 $\overline{A\cup B}$（德摩根）。  
这两者完全不同，千万别把 $\overline{AB}$ 误写成 $\overline{A}\,\overline{B}$！

**易错3：差事件 $A-B$ 不等于"$P(A)-P(B)$"。**  
差事件是集合运算的概念，与概率做差是不同的东西。$A-B$ 是一个事件（集合），$P(A)-P(B)$ 是一个数值，两者只在 $B\subseteq A$ 时才有关系 $P(A-B)=P(A)-P(B)$（见第1.3节）。

**易错4：忽视分配律的"两个方向"。**  
分配律有两个形式：  
$$A(B\cup C)=AB\cup AC \quad \text{（交对并的分配律）}$$  
$$A\cup(BC)=(A\cup B)(A\cup C) \quad \text{（并对交的分配律）}$$  
第二个形式初学者容易遗忘，但它在化简事件表达式时非常有用。

**易错5：把"$\omega\in A$"写成"$\omega\subset A$"。**  
$\omega$ 是样本点（元素），$A$ 是事件（集合），正确写法是 $\omega\in A$（属于），而非 $\omega\subset A$（包含，这是集合对集合的关系）。
:::

## 十、本节小结

:::note{label=本节小结与衔接}
本节用集合语言系统建立了事件之间的关系与运算体系：

**六种关系：**
- 包含 $A\subseteq B$：$A$ 发生 $\Rightarrow$ $B$ 发生；
- 相等 $A=B$：样本点集合完全相同；
- 互斥 $AB=\varnothing$：不能同时发生；
- 对立 $B=\overline{A}$：互斥且合并为 $\Omega$，非此即彼。

**四种运算：**
- 和（并）$A\cup B$：至少一个发生；
- 积（交）$AB$：同时发生；
- 差 $A-B=A\overline{B}$：$A$ 发生但 $B$ 不发生；
- 对立 $\overline{A}$：$A$ 不发生。

**核心运算律：**
- 交换律、结合律、分配律（两个方向）；
- **德摩根定律**：$\overline{A\cup B}=\overline{A}\cap\overline{B}$，$\overline{A\cap B}=\overline{A}\cup\overline{B}$；它是含对立符号的复杂表达式化简的终极武器。

**学习这一切的目的**是为概率计算做语言准备。现在我们能精确描述任意复杂的事件，下一步就是给这些事件赋予一个"可能性数值"——这就是第1.3节**概率的定义与性质**的任务：从公理化定义出发，推导出加法公式、减法公式、单调性等一系列计算工具，把本节的集合运算转化为实数运算。
:::
