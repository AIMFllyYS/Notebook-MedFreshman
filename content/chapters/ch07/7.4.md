## 从一个数到一段范围：为什么需要区间估计？

想象你是一名质检员，要估计某批灯泡的平均寿命 $\mu$。你随机抽了 20 只灯泡，测出样本均值 $\bar{x} = 1200$ 小时。于是你向工厂报告：**"平均寿命估计为 1200 小时。"**

工厂老板接着问：**"这个数字有多可信？误差有多大？"**

你哑口无言——因为点估计只给了你一个数字，却没有告诉你这个数字离真实值有多远，也没有说明你的信心有多大。

这正是**区间估计**要解决的问题：不只给出一个估计值，而是给出一个区间 $(L, U)$，并附上一句保证：**"我有 95% 的把握，真实参数就藏在这个区间里。"**

:::insight{label=点估计与区间估计的根本区别}
**点估计**：用统计量 $\hat{\theta}$ 直接代替未知参数 $\theta$，给出一个具体数值。优点是简洁，缺点是完全没有精度信息。

**区间估计**：给出一个随机区间 $(L, U)$，使得 $\theta$ 以足够高的概率落在其中。**区间既有宽度（精度），又有概率（可靠性）**，信息远比一个数丰富。

打个比方：
- 点估计像天气预报说"明天气温 25℃"；
- 区间估计像说"明天气温在 22℃ 到 28℃ 之间，置信度 95%"。
:::

## 置信区间的严格定义

:::definition{label=置信区间与置信水平}
设总体 $X$ 的分布含有未知参数 $\theta$，$X_1, X_2, \ldots, X_n$ 为样本。若存在两个统计量

$$L = L(X_1, X_2, \ldots, X_n), \quad U = U(X_1, X_2, \ldots, X_n)$$

满足

$$P(L \leq \theta \leq U) = 1 - \alpha \quad (0 < \alpha < 1),$$

则称随机区间 $(L, U)$ 是参数 $\theta$ 的**置信水平**（置信度）为 $1-\alpha$ 的**置信区间**；$L$ 称为**置信下限**，$U$ 称为**置信上限**，$1-\alpha$ 称为**置信水平**。
:::

这个定义有几个关键点需要仔细体会：

**第一，$L$ 和 $U$ 都是随机变量，不是固定数。** 它们由样本计算而来，每次抽样得到的数据不同，区间的两个端点就不同。换句话说，置信区间本身是随机的。

**第二，$\theta$ 是固定的未知常数，不是随机变量。** 说"$\theta$ 以概率 $1-\alpha$ 落入区间"并不意味着 $\theta$ 在飘移，而是说：在反复抽样中，大约有 $100(1-\alpha)\%$ 次构造出的区间会"罩住"真实的 $\theta$。

**第三，频率意义的正确解读。** 若 $1-\alpha = 0.95$，假设你进行了 100 次独立抽样，每次都计算出一个置信区间，那么在这 100 个区间中，大约有 95 个会包含真实参数 $\theta$，另外 5 个则不包含。

:::pitfall{label=置信区间最常见的误解}
**错误说法**："$\theta$ 有 95% 的概率落在区间 $(1150, 1250)$ 内。"

**为什么错**：一旦你算出具体数字 $(1150, 1250)$，这就是一个**固定区间**，而 $\theta$ 也是固定常数。一个固定常数要么在固定区间内（概率为 1），要么不在（概率为 0），谈不上 95%。

**正确说法**：采用这种方法构造的区间，有 95% 的概率在抽样时覆盖真实参数。或者说：这个具体的区间是用一种"成功率 95%"的方法造出来的。
:::

## 枢轴量法：构造置信区间的核心工具

如何系统地构造置信区间？最常用的方法是**枢轴量法**（Pivotal Quantity Method）。

:::definition{label=枢轴量}
若存在关于样本 $(X_1, \ldots, X_n)$ 与待估参数 $\theta$ 的函数

$$G = G(X_1, \ldots, X_n;\, \theta),$$

满足以下两个条件：

1. **分布已知且与 $\theta$ 无关**（可以查表求分位点）；
2. **包含待估参数 $\theta$（仅此一个未知量）**；

则称 $G$ 为**枢轴量**（枢轴统计量）。
:::

:::insight{label=枢轴量法的思路}
枢轴量法的核心逻辑分三步：

**第一步**：找一个分布已知的枢轴量 $G$。由于分布已知，可以找到两个分位点 $a, b$，使得

$$P(a \leq G \leq b) = 1 - \alpha.$$

**第二步**：对不等式 $a \leq G \leq b$ 进行代数变换，把 $\theta$ **夹到中间**，变成

$$P(L(X_1,\ldots,X_n) \leq \theta \leq U(X_1,\ldots,X_n)) = 1-\alpha.$$

**第三步**：读出置信下限 $L$ 和置信上限 $U$，得到置信区间 $(L, U)$。

整个过程的关键是**代数变形**——枢轴量里有 $\theta$，通过不等式变换把 $\theta$ 孤立出来。
:::

## 正态总体均值 $\mu$ 的置信区间

设 $X_1, X_2, \ldots, X_n$ 是来自正态总体 $X \sim N(\mu, \sigma^2)$ 的样本，$\bar{X}$ 为样本均值，$S^2$ 为样本方差（分母为 $n-1$）。我们分两种情况讨论。

### 情形一：方差 $\sigma^2$ 已知，用 $Z$ 统计量

**枢轴量的构造**：由于 $\bar{X} \sim N\!\left(\mu,\, \dfrac{\sigma^2}{n}\right)$（正态总体的样本均值精确服从正态），标准化后得到

$$Z = \frac{\bar{X} - \mu}{\sigma / \sqrt{n}} \sim N(0,1).$$

这个 $Z$ 分布已知（标准正态），且含有且仅含未知参数 $\mu$，满足枢轴量条件。

**找分位点**：对给定置信水平 $1-\alpha$，在标准正态分布中找上 $\alpha/2$ 分位点 $u_{\alpha/2}$，即 $P(Z > u_{\alpha/2}) = \alpha/2$。由对称性，

$$P\!\left(-u_{\alpha/2} \leq Z \leq u_{\alpha/2}\right) = 1 - \alpha.$$

**代数变换**：将 $Z$ 的表达式代入：

$$\begin{aligned}
&P\!\left(-u_{\alpha/2} \leq \frac{\bar{X} - \mu}{\sigma/\sqrt{n}} \leq u_{\alpha/2}\right) = 1-\alpha \\[6pt]
\Longleftrightarrow\quad &P\!\left(\bar{X} - u_{\alpha/2}\cdot\frac{\sigma}{\sqrt{n}} \leq \mu \leq \bar{X} + u_{\alpha/2}\cdot\frac{\sigma}{\sqrt{n}}\right) = 1-\alpha.
\end{aligned}$$

:::theorem{label=σ已知时 μ 的置信区间}
设样本来自 $N(\mu, \sigma^2)$（$\sigma^2$ 已知），置信水平为 $1-\alpha$，则 $\mu$ 的置信区间为

$$\left(\bar{X} - u_{\alpha/2}\cdot\frac{\sigma}{\sqrt{n}},\quad \bar{X} + u_{\alpha/2}\cdot\frac{\sigma}{\sqrt{n}}\right),$$

简写为 $\bar{X} \pm u_{\alpha/2}\cdot\dfrac{\sigma}{\sqrt{n}}$。

常用分位点：$u_{0.025} = 1.96$（$1-\alpha=0.95$）；$u_{0.005} = 2.576$（$1-\alpha=0.99$）。
:::

### 情形二：方差 $\sigma^2$ 未知，用 $t$ 统计量

实践中 $\sigma^2$ 往往未知。此时不能直接用 $Z$，需要用样本方差 $S$ 来代替 $\sigma$。但代替后的统计量不再是标准正态，而是 $t$ 分布。

**枢轴量的构造**：

$$T = \frac{\bar{X} - \mu}{S / \sqrt{n}} \sim t(n-1).$$

这个结论来自第六章定理：$\bar{X}$ 与 $S^2$ 独立，$\dfrac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)$，由 $t$ 分布的构造定义可推出上式（详见 6.3 节推论）。

**代数变换**：$t(n-1)$ 分布同样关于原点对称，其上 $\alpha/2$ 分位点记为 $t_{\alpha/2}(n-1)$，即 $P(T > t_{\alpha/2}(n-1)) = \alpha/2$，故

$$P\!\left(-t_{\alpha/2}(n-1) \leq T \leq t_{\alpha/2}(n-1)\right) = 1-\alpha,$$

代入 $T$ 的表达式并变换，得：

$$P\!\left(\bar{X} - t_{\alpha/2}(n-1)\cdot\frac{S}{\sqrt{n}} \leq \mu \leq \bar{X} + t_{\alpha/2}(n-1)\cdot\frac{S}{\sqrt{n}}\right) = 1-\alpha.$$

:::theorem{label=σ未知时 μ 的置信区间}
设样本来自 $N(\mu, \sigma^2)$（$\sigma^2$ **未知**），置信水平为 $1-\alpha$，则 $\mu$ 的置信区间为

$$\left(\bar{X} - t_{\alpha/2}(n-1)\cdot\frac{S}{\sqrt{n}},\quad \bar{X} + t_{\alpha/2}(n-1)\cdot\frac{S}{\sqrt{n}}\right),$$

其中 $S = \sqrt{\dfrac{1}{n-1}\sum_{i=1}^n(X_i - \bar{X})^2}$ 为样本标准差。
:::

:::insight{label=Z 区间 vs t 区间：哪个更宽？}
$t$ 分布比标准正态分布具有更厚的尾部（尖峰厚尾），因此同样置信水平下，$t_{\alpha/2}(n-1) > u_{\alpha/2}$。这意味着：

**$\sigma^2$ 未知时的置信区间比 $\sigma^2$ 已知时更宽。**

这很直观——未知的不确定性更多（既不知道 $\mu$，又不知道 $\sigma^2$），自然需要更宽的区间来保证同样的置信水平。

当 $n$ 足够大（通常 $n > 45$）时，$t_{\alpha/2}(n-1) \approx u_{\alpha/2}$，两个区间几乎相同。
:::

## 正态总体方差 $\sigma^2$ 的置信区间

有时我们关心的不是均值，而是方差——它度量了数据的离散程度（波动性）。例如，质检时既关心平均重量，也关心重量是否稳定。

**枢轴量的构造**：由第六章定理，

$$\chi^2 = \frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1).$$

这个量分布已知（卡方分布），且仅含未知参数 $\sigma^2$，是合法的枢轴量。

**注意**：$\chi^2$ 分布**不对称**，因此需要找两个不同的分位点。令 $\chi^2_{1-\alpha/2}(n-1)$ 和 $\chi^2_{\alpha/2}(n-1)$ 分别为 $\chi^2(n-1)$ 分布的下 $\alpha/2$ 和上 $\alpha/2$ 分位点，即

$$P\!\left(\chi^2 < \chi^2_{1-\alpha/2}(n-1)\right) = \frac{\alpha}{2}, \quad P\!\left(\chi^2 > \chi^2_{\alpha/2}(n-1)\right) = \frac{\alpha}{2},$$

则

$$P\!\left(\chi^2_{1-\alpha/2}(n-1) \leq \frac{(n-1)S^2}{\sigma^2} \leq \chi^2_{\alpha/2}(n-1)\right) = 1-\alpha.$$

**代数变换**：对不等式取倒数（注意不等号方向翻转），并乘以 $(n-1)S^2$：

$$\begin{aligned}
&\chi^2_{1-\alpha/2} \leq \frac{(n-1)S^2}{\sigma^2} \leq \chi^2_{\alpha/2} \\[6pt]
\Longleftrightarrow\quad &\frac{(n-1)S^2}{\chi^2_{\alpha/2}} \leq \sigma^2 \leq \frac{(n-1)S^2}{\chi^2_{1-\alpha/2}}.
\end{aligned}$$

:::theorem{label=σ² 的置信区间}
设样本来自 $N(\mu, \sigma^2)$（$\mu, \sigma^2$ 均未知），置信水平为 $1-\alpha$，则 $\sigma^2$ 的置信区间为

$$\left(\frac{(n-1)S^2}{\chi^2_{\alpha/2}(n-1)},\quad \frac{(n-1)S^2}{\chi^2_{1-\alpha/2}(n-1)}\right).$$

对应地，$\sigma$ 的置信区间为

$$\left(\sqrt{\frac{(n-1)S^2}{\chi^2_{\alpha/2}(n-1)}},\quad \sqrt{\frac{(n-1)S^2}{\chi^2_{1-\alpha/2}(n-1)}}\right).$$
:::

:::derivation{label=χ² 区间中分位点顺序的推导细节}
设 $c_1 = \chi^2_{1-\alpha/2}(n-1)$，$c_2 = \chi^2_{\alpha/2}(n-1)$，由于 $\alpha < 1$，有 $\alpha/2 < 1-\alpha/2$，故在卡方分布的右尾中，$c_1 < c_2$（$c_1$ 对应较大概率即靠左，$c_2$ 对应较小概率即靠右）。

从 $c_1 \leq \dfrac{(n-1)S^2}{\sigma^2} \leq c_2$ 解 $\sigma^2$：

$$\frac{1}{c_2} \leq \frac{\sigma^2}{(n-1)S^2} \leq \frac{1}{c_1}$$

$$\frac{(n-1)S^2}{c_2} \leq \sigma^2 \leq \frac{(n-1)S^2}{c_1}.$$

由于 $c_1 < c_2$，故 $\dfrac{1}{c_1} > \dfrac{1}{c_2}$，区间左端点用 $\dfrac{(n-1)S^2}{c_2}$（即除以较大的 $c_2 = \chi^2_{\alpha/2}$），右端点用 $\dfrac{(n-1)S^2}{c_1}$（即除以较小的 $c_1 = \chi^2_{1-\alpha/2}$）。
:::

## 区间长度、样本量与精度

置信区间有个天然的"矛盾"：置信水平越高，区间越宽，精度越差；而精度越高（区间越窄），置信水平越低。那如何在两者之间取得平衡？

### 以 $\sigma$ 已知时 $\mu$ 的区间为例

区间长度为

$$d = 2u_{\alpha/2} \cdot \frac{\sigma}{\sqrt{n}}.$$

若要求区间半宽不超过 $\delta$（即 $d/2 \leq \delta$），则

$$u_{\alpha/2} \cdot \frac{\sigma}{\sqrt{n}} \leq \delta \quad \Longrightarrow \quad n \geq \left(\frac{u_{\alpha/2} \cdot \sigma}{\delta}\right)^2.$$

:::insight{label=三条基本规律}
从公式 $d = 2u_{\alpha/2} \cdot \dfrac{\sigma}{\sqrt{n}}$ 中可以读出三条规律：

1. **$n$ 越大，区间越窄（精度越高）**：样本量加大，$\sqrt{n}$ 增大，分母变大，区间缩短。这是精度的代价。
2. **$1-\alpha$ 越高（$u_{\alpha/2}$ 越大），区间越宽（精度越低）**：要"更有把握"，就必须接受更宽的范围。
3. **$\sigma$ 越大（总体越分散），区间越宽**：原始数据越乱，估计自然越不精准。

在实际问题中，通常先确定置信水平（例如 0.95），再确定允许误差 $\delta$，最后由此计算所需最小样本量 $n$。
:::

## 由浅入深的例题

:::example{label=例1：σ已知，估计灯泡寿命均值}
某型号灯泡寿命 $X \sim N(\mu, \sigma^2)$，已知 $\sigma = 100$ 小时。随机抽取 $n = 25$ 只，测得样本均值 $\bar{x} = 1050$ 小时。求 $\mu$ 的 95% 置信区间。

**解**：

**第一步**：确认条件。$\sigma = 100$ 已知，$n = 25$，$\bar{x} = 1050$，置信水平 $1-\alpha = 0.95$，故 $\alpha = 0.05$，$\alpha/2 = 0.025$。

**第二步**：查表得 $u_{0.025} = 1.96$。

**第三步**：代入公式。区间半宽为

$$u_{0.025} \cdot \frac{\sigma}{\sqrt{n}} = 1.96 \times \frac{100}{\sqrt{25}} = 1.96 \times 20 = 39.2.$$

**第四步**：写出区间。

$$\left(1050 - 39.2,\quad 1050 + 39.2\right) = \left(1010.8,\; 1089.2\right).$$

**结论**：有 95% 的把握认为该型号灯泡的平均寿命在 1010.8 小时到 1089.2 小时之间。

**直觉验证**：$n = 25$ 时 $\sigma/\sqrt{n} = 20$，这是样本均值的标准差（标准误差）。区间半宽约为 2 个标准误差，符合正态分布 95% 对应"约 $\pm2$ 标准差"的直觉。
:::

:::example{label=例2：σ未知，估计某校学生身高均值}
从某校随机抽取 $n = 16$ 名男生，测得身高（单位：cm）数据，计算得 $\bar{x} = 172.5$，$s = 6.2$（样本标准差，分母为 $n-1=15$）。设身高服从正态分布，求均值 $\mu$ 的 99% 置信区间。

**解**：

**第一步**：确认条件。$\sigma^2$ 未知，$n = 16$，$\bar{x} = 172.5$，$s = 6.2$，置信水平 $0.99$，故 $\alpha = 0.01$，$\alpha/2 = 0.005$，自由度 $n-1 = 15$。

**第二步**：查 $t$ 分布表，得 $t_{0.005}(15) = 2.947$。

**第三步**：区间半宽为

$$t_{0.005}(15) \cdot \frac{s}{\sqrt{n}} = 2.947 \times \frac{6.2}{\sqrt{16}} = 2.947 \times 1.55 = 4.568.$$

**第四步**：写出区间。

$$\left(172.5 - 4.568,\quad 172.5 + 4.568\right) = \left(167.93,\; 177.07\right).$$

**结论**：有 99% 的把握认为该校男生平均身高在 167.93 cm 到 177.07 cm 之间。

**与 95% 区间对比**：若置信水平改为 95%，$t_{0.025}(15) = 2.131$，半宽变为 $2.131 \times 1.55 = 3.30$，区间为 $(169.20, 175.80)$——明显比 99% 的区间窄，但可靠性也更低。这正是"精度与可靠性之间的权衡"。
:::

:::example{label=例3：估计总体方差 σ²}
某零件加工过程中，随机抽取 $n = 10$ 件，测量其直径（单位：mm）。设直径服从正态分布，计算得样本方差 $s^2 = 0.04$（$mm^2$）。求 $\sigma^2$ 和 $\sigma$ 的 95% 置信区间。

**解**：

**第一步**：确认条件。$n = 10$，$s^2 = 0.04$，$1-\alpha = 0.95$，$\alpha = 0.05$，$\alpha/2 = 0.025$，自由度 $n-1 = 9$。

**第二步**：查 $\chi^2$ 分布表，得

$$\chi^2_{0.025}(9) = 19.023, \quad \chi^2_{0.975}(9) = 2.700.$$

（注意：$\chi^2_{0.975}$ 是下 $\alpha/2 = 0.025$ 分位点，即左尾概率为 0.025 的点。）

**第三步**：计算端点。

$$\text{下限} = \frac{(n-1)s^2}{\chi^2_{0.025}(9)} = \frac{9 \times 0.04}{19.023} = \frac{0.36}{19.023} \approx 0.0189,$$

$$\text{上限} = \frac{(n-1)s^2}{\chi^2_{0.975}(9)} = \frac{9 \times 0.04}{2.700} = \frac{0.36}{2.700} \approx 0.1333.$$

**第四步**：写出区间。

$$\sigma^2 \text{ 的置信区间：} (0.0189,\; 0.1333) \text{ (mm}^2\text{)}.$$

$$\sigma \text{ 的置信区间：} (\sqrt{0.0189},\; \sqrt{0.1333}) \approx (0.137,\; 0.365) \text{ (mm)}.$$

**结论**：有 95% 的把握认为加工过程中直径的标准差在 0.137 mm 到 0.365 mm 之间。

**特别注意**：$\chi^2$ 区间**不对称**——下限和上限分别除以不同的分位点值（19.023 和 2.700），这与 $Z$、$t$ 区间的对称形式 $\bar{X} \pm c$ 不同。原因是 $\chi^2$ 分布本身不关于任何轴对称。
:::

:::example{label=例4：确定所需样本量}
某工厂要估计产品的某项性能指标（正态总体），已知总体标准差 $\sigma = 5$。要求在置信水平 95% 下，估计误差不超过 $\delta = 1.5$（即区间半宽不超过 1.5）。问至少需要抽取多少件样品？

**解**：

**第一步**：置信水平 $0.95$，$\alpha/2 = 0.025$，$u_{0.025} = 1.96$。

**第二步**：由区间半宽条件：

$$u_{\alpha/2} \cdot \frac{\sigma}{\sqrt{n}} \leq \delta \quad \Longrightarrow \quad 1.96 \times \frac{5}{\sqrt{n}} \leq 1.5.$$

**第三步**：解方程：

$$\sqrt{n} \geq \frac{1.96 \times 5}{1.5} = \frac{9.8}{1.5} \approx 6.533,$$

$$n \geq 6.533^2 \approx 42.68.$$

**第四步**：由于样本量必须是整数，故 $n \geq 43$。

**结论**：至少需要抽取 43 件样品，才能在 95% 置信水平下将估计误差控制在 1.5 以内。

**实践意义**：这道题展示了"先做实验设计"的思维——在收集数据前，先预算好需要多少数据，避免过度浪费或样本不足。
:::

| $\mu$（$\sigma^2$ 已知） | $Z = \dfrac{\bar{X}-\mu}{\sigma/\sqrt{n}}$ | $N(0,1)$ | $\bar{X} \pm u_{\alpha/2}\dfrac{\sigma}{\sqrt{n}}$ |
| $\mu$（$\sigma^2$ 未知） | $T = \dfrac{\bar{X}-\mu}{S/\sqrt{n}}$ | $t(n-1)$ | $\bar{X} \pm t_{\alpha/2}(n-1)\dfrac{S}{\sqrt{n}}$ |
| $\sigma^2$（$\mu$ 未知） | $\chi^2 = \dfrac{(n-1)S^2}{\sigma^2}$ | $\chi^2(n-1)$ | $\left(\dfrac{(n-1)S^2}{\chi^2_{\alpha/2}},\, \dfrac{(n-1)S^2}{\chi^2_{1-\alpha/2}}\right)$ |

## 易错点整理

:::pitfall{label=五大高频错误}
**错误一：混淆随机区间与固定区间。**
置信区间 $(L, U)$ 在抽样前是随机变量，抽样后变为具体数字。概率 $1-\alpha$ 是对"这套构造方法"的评价，而不是对某一个具体区间的评价。

**错误二：搞错 $\chi^2$ 分位点的上下标。**
$\chi^2_{\alpha/2}(n-1)$ 是右尾面积为 $\alpha/2$ 的分位点（较大的数），$\chi^2_{1-\alpha/2}(n-1)$ 是右尾面积为 $1-\alpha/2$ 的分位点（较小的数，有时写作左尾 $\alpha/2$ 分位点）。置信区间下限除以大的分位点，上限除以小的分位点。

**错误三：$\sigma^2$ 未知时仍用 $Z$ 统计量。**
只有 $\sigma^2$ 已知才能用 $Z$。若 $\sigma^2$ 未知，必须用 $T = \dfrac{\bar{X}-\mu}{S/\sqrt{n}}$，服从 $t(n-1)$，而不是 $N(0,1)$。

**错误四：样本方差的分母搞错。**
本章所有公式中的 $S^2$ 均指**无偏样本方差** $S^2 = \dfrac{1}{n-1}\sum_{i=1}^n(X_i-\bar{X})^2$（分母是 $n-1$，不是 $n$）。

**错误五：提高置信水平却以为区间会变窄。**
置信水平从 95% 提高到 99%，对应的分位点变大（如 $u_{0.025} = 1.96 \to u_{0.005} = 2.576$），区间只会**变宽**，不会变窄。要想区间窄且置信水平高，只有一个办法：加大样本量 $n$。
:::

## 本节小结

:::note{label=7.4 区间估计核心要点}
**核心思想**：区间估计通过给出随机区间 $(L, U)$ 以及概率保证 $P(L \leq \theta \leq U) = 1-\alpha$，比点估计提供了更丰富的精度信息。

**方法论**：枢轴量法——找一个分布已知、仅含待估参数的函数，通过查分位点、解不等式得到置信区间。

**三大区间**（均基于正态总体）：

1. **$\mu$ 的区间（$\sigma$ 已知）**：$\bar{X} \pm u_{\alpha/2}\dfrac{\sigma}{\sqrt{n}}$，枢轴量为 $Z \sim N(0,1)$；

2. **$\mu$ 的区间（$\sigma$ 未知）**：$\bar{X} \pm t_{\alpha/2}(n-1)\dfrac{S}{\sqrt{n}}$，枢轴量为 $T \sim t(n-1)$；

3. **$\sigma^2$ 的区间**：$\left(\dfrac{(n-1)S^2}{\chi^2_{\alpha/2}(n-1)},\, \dfrac{(n-1)S^2}{\chi^2_{1-\alpha/2}(n-1)}\right)$，枢轴量为 $\chi^2 \sim \chi^2(n-1)$。

**区间宽度的决定因素**：置信水平 $1-\alpha$（越高越宽）、样本量 $n$（越大越窄）、总体方差 $\sigma^2$（越大越宽）。在精度要求和置信水平确定后，可以反解出所需最小样本量。

**下一节预告**：第七章至此结束了参数估计的两大分支（点估计与区间估计）。第八章将进入**假设检验**——我们不再是"估计参数是多少"，而是"检验参数是否等于某个假设值"。有趣的是，假设检验与区间估计有着深刻的对偶关系：对 $\mu$ 的双侧检验恰好与置信区间等价。学完第八章后，再回头看本节，你会有全新的理解。
:::
