## 从"标准化"到"三大分布"：为什么需要新的工具？

你已经学会了如何用标准正态分布处理单个正态总体的问题。但现实中有一类问题让人头疼：

- 我想比较两台机器产品质量的波动，涉及**两个方差**怎么办？
- 总体方差 $\sigma^2$ 未知时，用样本标准差 $S$ 替代 $\sigma$ 后，那个"标准化"得到的量还是不是标准正态？
- 抽了一组样本，$n$ 个独立标准正态的**平方和**到底服从什么分布？

为了系统回答这三类问题，统计学家构造了三个新的分布——**$\chi^2$ 分布、$t$ 分布、$F$ 分布**，合称"三大抽样分布"。它们不是凭空发明的玩具，而是从正态分布出发，通过特定的代数构造自然涌现的分布，是后续推断统计（区间估计、假设检验）的核心工具。

本节的学习策略：每种分布重点掌握三件事：**①构造定义**（它是怎么凑出来的）、**②数字特征与图像特征**、**③上 $\alpha$ 分位点**的含义与查表用法。

---

## 一、$\chi^2$ 分布

### 直觉：平方和为什么重要？

样本方差 $S^2 = \dfrac{1}{n-1}\sum_{i=1}^{n}(X_i - \bar{X})^2$ 里有一个平方和结构。若总体是正态分布，把每个 $X_i$ 标准化后，得到的恰好是若干独立标准正态随机变量的平方和。理解这个平方和的分布，就是理解 $\chi^2$ 分布的出发点。

生活中的类比：你扔了 $n$ 枚针，每枚落点偏差都是标准正态的，$\chi^2$ 就是这 $n$ 个偏差的平方之和，它刻画了整体的"偏离程度"。

::video{id=ch06-6.2-sampling-dist}

:::definition{label=χ² 分布}
设 $X_1, X_2, \ldots, X_n$ 相互独立，且均服从标准正态分布 $N(0,1)$，则称随机变量

$$\chi^2 = X_1^2 + X_2^2 + \cdots + X_n^2 = \sum_{i=1}^{n} X_i^2$$

服从自由度为 $n$ 的 **$\chi^2$ 分布**，记作

$$\chi^2 \sim \chi^2(n)$$

**构造要素（缺一不可）：**
1. **独立**：这 $n$ 个随机变量必须相互独立；
2. **标准正态**：每个 $X_i$ 必须服从 $N(0,1)$，不是普通正态！
3. **平方和**：取每个变量的平方再求和。

:::

:::insight{label=自由度的直觉含义}
"自由度"（degree of freedom）是 $\chi^2$ 分布唯一的参数。它等于参与求和的**独立**标准正态随机变量的个数，不是你看到了几个变量，而是**真正独立**的有几个。

后面你会看到，$\frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)$ 的自由度是 $n-1$ 而不是 $n$，就是因为 $n$ 个离差 $(X_i - \bar{X})$ 之间有一个约束条件 $\sum(X_i - \bar{X}) = 0$，只有 $n-1$ 个是自由的。
:::

### $\chi^2$ 分布的密度函数与图像

$\chi^2(n)$ 的概率密度函数（了解即可，不需要记忆）含有 Gamma 函数：

$$f(x) = \begin{cases} \dfrac{1}{2^{n/2}\,\Gamma(n/2)}\, x^{n/2-1} e^{-x/2}, & x > 0 \\ 0, & x \leqslant 0 \end{cases}$$

**图像关键特征（要记住）：**

- $\chi^2$ 分布只取**正值**，整个图像位于 $x > 0$ 的半轴；
- 图像是**右偏**（正偏）的，右边有一条长尾巴，不对称；
- 自由度越小，图像越向右侧倾斜、越集中在原点附近；随着 $n$ 增大，图像逐渐"展平"并向右移动，形状越来越像正态分布的形状；
- 自由度 $n=2$ 时，$\chi^2(2)$ 恰好是参数为 $\frac{1}{2}$ 的**指数分布**（这是一个有用的特殊情形）。

:::theorem{label=χ² 分布的数字特征}
若 $\chi^2 \sim \chi^2(n)$，则

$$E(\chi^2) = n, \qquad D(\chi^2) = 2n$$

即期望等于自由度，方差等于自由度的两倍。
:::

:::derivation{label=χ² 均值与方差的推导}
**期望：**

$$E(\chi^2) = E\!\left(\sum_{i=1}^n X_i^2\right) = \sum_{i=1}^n E(X_i^2)$$

由于各 $X_i \sim N(0,1)$，有 $E(X_i) = 0$，$D(X_i) = 1$。

利用方差公式 $D(X) = E(X^2) - [E(X)]^2$，得

$$E(X_i^2) = D(X_i) + [E(X_i)]^2 = 1 + 0 = 1$$

所以 $E(\chi^2) = n \cdot 1 = n$。

**方差：**

因为各 $X_i$ 独立，

$$D(\chi^2) = \sum_{i=1}^n D(X_i^2)$$

需要计算 $D(X_i^2) = E(X_i^4) - [E(X_i^2)]^2$。

对标准正态分布，$k$ 阶中心矩（偶数阶）为 $(k-1)!!$（双阶乘），且期望为 0，所以中心矩等于原点矩：

$$E(X_i^4) = 3!! = 3 \cdot 1 = 3$$

因此 $D(X_i^2) = 3 - 1^2 = 2$，故

$$D(\chi^2) = n \cdot 2 = 2n$$
:::

:::theorem{label=χ² 分布的可加性}
若 $\chi_1^2 \sim \chi^2(n_1)$，$\chi_2^2 \sim \chi^2(n_2)$，且二者**相互独立**，则

$$\chi_1^2 + \chi_2^2 \sim \chi^2(n_1 + n_2)$$

更一般地，若 $\chi_k^2 \sim \chi^2(n_k)$（$k = 1, 2, \ldots, m$）相互独立，则

$$\sum_{k=1}^m \chi_k^2 \sim \chi^2\!\left(\sum_{k=1}^m n_k\right)$$

独立的 $\chi^2$ 之和仍然是 $\chi^2$，自由度相加。
:::

:::insight{label=可加性的直觉与应用}
因为每个标准正态的平方都是 $\chi^2(1)$，而 $n$ 个独立 $\chi^2(1)$ 的和就是 $\chi^2(n)$，可加性正是将定义"反过来"表述的另一种方式。

**实际应用**：若 $X \sim E(\lambda)$（指数分布，参数为 $\lambda$），令 $Y = 2\lambda X$，可以证明 $Y \sim \chi^2(2)$。因此对来自指数总体的样本 $X_1, \ldots, X_n$，有

$$2\lambda \sum_{i=1}^n X_i = 2n\lambda\bar{X} \sim \chi^2(2n)$$

这将指数总体的样本均值与 $\chi^2$ 分布联系起来，在区间估计和假设检验中很有用。
:::

### $\chi^2$ 分布的上 $\alpha$ 分位点

:::definition{label=χ² 分布的上 α 分位点}
设 $\chi^2 \sim \chi^2(n)$，对给定的 $\alpha$（$0 < \alpha < 1$），称满足条件

$$P\!\left(\chi^2 > \chi^2_\alpha(n)\right) = \alpha$$

的点 $\chi^2_\alpha(n)$ 为 $\chi^2(n)$ 分布的**上 $\alpha$ 分位点**（也叫上侧临界值）。
:::

:::insight{label=分位点的本质}
分位点的求解方向与通常的"已知分布和区间求概率"**完全相反**：

- 正向：已知分布，已知区间 → 求概率
- **分位点**：已知分布，已知概率（右尾面积 $= \alpha$）→ **反求那个临界点**

图形理解：在 $\chi^2(n)$ 的密度曲线右尾处，阴影面积恰好等于 $\alpha$ 的那个 $x$ 值，就是 $\chi^2_\alpha(n)$。

影响分位点的两个因素：
1. **自由度 $n$**：决定是哪条密度曲线；
2. **$\alpha$**：决定右尾面积的大小，$\alpha$ 越小，临界点越向右移。
:::

**查表示例**：$\chi^2_{0.05}(10) = 18.307$ 的含义是：自由度为 $10$ 的 $\chi^2$ 分布中，随机变量超过 $18.307$ 的概率恰好为 $0.05$。

**大自由度近似**：当 $n$ 较大时，$\sqrt{2\chi^2} - \sqrt{2n-1}$ 近似服从 $N(0,1)$，从而

$$\chi^2_\alpha(n) \approx \frac{1}{2}\left(u_\alpha + \sqrt{2n-1}\right)^2$$

其中 $u_\alpha$ 是标准正态的上 $\alpha$ 分位点。

## 二、$t$ 分布

### 直觉：方差未知时标准化后的分布

当总体方差 $\sigma^2$ 已知时，样本均值标准化后是标准正态：

$$\frac{\bar{X} - \mu}{\sigma/\sqrt{n}} \sim N(0,1)$$

但实际问题中 $\sigma^2$ 往往**未知**，必须用样本标准差 $S$ 来替代 $\sigma$。替换后得到

$$\frac{\bar{X} - \mu}{S/\sqrt{n}}$$

这个量还是标准正态吗？**不是了！** 因为 $S$ 本身也是随机变量，它的随机性"感染"了整个表达式，使分布的尾部变得更厚。英国统计学家 Gosset 在 1908 年发现了这个量的精确分布，并以笔名 "Student" 发表，因此这个分布又称 **Student 分布**。

:::definition{label=t 分布}
设 $X \sim N(0,1)$，$Y \sim \chi^2(n)$，且 $X$ 与 $Y$ **相互独立**，则称随机变量

$$T = \frac{X}{\sqrt{Y/n}}$$

服从自由度为 $n$ 的 **$t$ 分布**，记作

$$T \sim t(n)$$

**构造要素（缺一不可）：**
1. 分子是**标准正态** $X \sim N(0,1)$；
2. 分母是：$\chi^2$ 随机变量除以自己的自由度，再取**平方根**；
3. 分子与分母必须**相互独立**。

$t$ 分布的自由度由**分母中 $\chi^2$ 的自由度**决定。
:::

:::insight{label=为什么叫 t 而不叫别的}
$t$ 分布的构造方式揭示了一件事：当我们用样本标准差 $S$ 替换总体标准差 $\sigma$ 时，分母中引入了一个额外的随机来源（$\chi^2$ 的随机性），导致整个比值不再是标准正态，而是具有更厚尾巴的 $t$ 分布。自由度 $n$ 越小，尾巴越厚；$n \to \infty$ 时，$t(n)$ 趋近于 $N(0,1)$。
:::

### $t$ 分布的密度函数与图像

$t(n)$ 的概率密度函数为

$$f(t) = \frac{\Gamma\!\left(\frac{n+1}{2}\right)}{\sqrt{n\pi}\,\Gamma\!\left(\frac{n}{2}\right)}\left(1 + \frac{t^2}{n}\right)^{-\frac{n+1}{2}}, \quad -\infty < t < +\infty$$

不需要记忆这个公式，但要牢记图像特征：

**图像关键特征（要记住）：**
- $t$ 分布关于 $y$ 轴**对称**，形状与标准正态非常相似；
- 与标准正态相比，$t$ 分布是**尖峰厚尾**的：尾部比正态分布衰减慢，更容易出现极端值；
- 自由度 $n$ 越小，尾部越厚，峰顶越高（相对而言）；
- 当 $n \geqslant 30$ 时，$t(n)$ 与 $N(0,1)$ 已经非常接近；$n > 45$ 时通常直接用标准正态近似；
- 特例：$n = 1$ 时，$t(1)$ 是**柯西分布**（期望不存在）。

```
自由度 n 增大 ↑
尾部厚度   ↓（越来越薄，趋近正态）
峰度       ↓（趋近正态峰度 0）
```

::interactive{id=ch06-6.2-sampling-dist-explorer}

:::theorem{label=t 分布的数字特征}
若 $T \sim t(n)$，则

- 期望：$E(T) = 0$（当 $n > 1$；$n = 1$ 时期望不存在）
- 方差：$D(T) = \dfrac{n}{n-2}$（当 $n > 2$；$n \leqslant 2$ 时方差不存在）

由于对称性，$t$ 分布的奇数阶矩（若存在）均为 $0$。

**渐近正态性**：当 $n \to \infty$ 时，

$$t(n) \xrightarrow{d} N(0,1)$$

即 $t$ 分布依分布收敛于标准正态分布。
:::

### $t$ 分布的上 $\alpha$ 分位点

:::definition{label=t 分布的上 α 分位点}
设 $T \sim t(n)$，对给定的 $\alpha$（$0 < \alpha < 1$），称满足

$$P(T > t_\alpha(n)) = \alpha$$

的点 $t_\alpha(n)$ 为 $t(n)$ 分布的**上 $\alpha$ 分位点**。
:::

**对称性带来的重要关系**：由于 $t$ 分布关于 $y$ 轴对称，

$$t_{1-\alpha}(n) = -t_\alpha(n)$$

这意味着：如果 $t_\alpha(n)$ 是右尾面积为 $\alpha$ 的临界值，那么左尾面积为 $\alpha$ 的临界值就是 $-t_\alpha(n)$，也就是 $t_{1-\alpha}(n)$。

**查表规则**：
- $n \leqslant 45$ 时：查 $t$ 分布表；
- $n > 45$ 时：$t_\alpha(n) \approx u_\alpha$（用标准正态分位点近似）。

**双侧临界值**：在双侧情形下，若右尾、左尾各占 $\alpha/2$ 的面积，则两个临界值分别为 $t_{\alpha/2}(n)$ 和 $-t_{\alpha/2}(n)$，满足

$$P\!\left(|T| > t_{\alpha/2}(n)\right) = \alpha$$

## 三、$F$ 分布

### 直觉：比较两个总体的方差

$F$ 分布由英国统计学家 Fisher 在研究方差分析时提出，因此以 F 命名。它用来回答这样的问题：两个独立正态总体的方差相等吗？

构造 $F$ 分布的原材料是**两个独立的 $\chi^2$ 随机变量**。

:::definition{label=F 分布}
设 $U \sim \chi^2(m)$，$V \sim \chi^2(n)$，且 $U$ 与 $V$ **相互独立**，则称随机变量

$$F = \frac{U/m}{V/n}$$

服从第一自由度为 $m$、第二自由度为 $n$ 的 **$F$ 分布**，记作

$$F \sim F(m, n)$$

**构造要素：**
1. 分子是 $\chi^2(m)$，除以自己的自由度 $m$；
2. 分母是 $\chi^2(n)$，除以自己的自由度 $n$；
3. 分子与分母必须**相互独立**；
4. **两个自由度有顺序**：$F(m,n)$ 中 $m$ 是分子自由度，$n$ 是分母自由度，$F(m,n) \neq F(n,m)$（一般情况下）。
:::

:::insight{label=F 分布两个自由度的顺序很重要}
$F(3, 10)$ 和 $F(10, 3)$ 是两个完全不同的分布！记法规则：**先分子，后分母**。

直觉上，$F$ 分布衡量的是"两个独立的平均平方和的比值"，分子的自由度和分母的自由度各自记录了对应 $\chi^2$ 的独立信息个数。
:::

### $F$ 分布的密度函数与图像

$F$ 分布的密度函数形式复杂（含 Beta 函数），不需要记忆。

**图像关键特征（要记住）：**
- $F$ 分布只取**正值**（因为是两个非负量的比）；
- 图像也是**右偏**的，类似 $\chi^2$ 分布；
- 两个自由度 $m, n$ 都影响图像形状；$m$ 和 $n$ 越大，图像越趋近于以 $1$ 为中心的对称形态。

:::theorem{label=F 分布的数字特征}
若 $F \sim F(m, n)$，则

$$E(F) = \frac{n}{n-2} \quad (n > 2)$$

$$D(F) = \frac{2n^2(m+n-2)}{m(n-2)^2(n-4)} \quad (n > 4)$$

注意期望只与**分母自由度** $n$ 有关，当 $n$ 较大时趋近于 $1$（两个方差相差不大时 $F$ 应在 $1$ 附近）。
:::

:::theorem{label=F 分布的重要性质}
**性质一：倒数关系**

若 $F \sim F(m, n)$，则

$$\frac{1}{F} \sim F(n, m)$$

即分子分母互换，自由度顺序也互换。

**性质二：与 $t$ 分布的关系**

若 $T \sim t(n)$，则

$$T^2 \sim F(1, n)$$

这是因为 $T = \dfrac{X}{\sqrt{Y/n}}$，故 $T^2 = \dfrac{X^2/1}{Y/n}$，其中 $X^2 \sim \chi^2(1)$，$Y \sim \chi^2(n)$，正好是 $F(1,n)$ 的构造。
:::

:::derivation{label=F 分布倒数关系的推导}
设 $F = \dfrac{U/m}{V/n}$，$U \sim \chi^2(m)$，$V \sim \chi^2(n)$，$U, V$ 独立。

则

$$\frac{1}{F} = \frac{V/n}{U/m}$$

分子是 $V/n$（$\chi^2(n)$ 除以自由度 $n$），分母是 $U/m$（$\chi^2(m)$ 除以自由度 $m$），且二者独立。

根据 $F$ 分布的定义，$\dfrac{1}{F} \sim F(n, m)$，与原来的 $F(m,n)$ 的两个自由度恰好互换。
:::

### $F$ 分布的上 $\alpha$ 分位点

:::definition{label=F 分布的上 α 分位点}
设 $F \sim F(m, n)$，对给定的 $\alpha$（$0 < \alpha < 1$），称满足

$$P(F > F_\alpha(m, n)) = \alpha$$

的点 $F_\alpha(m, n)$ 为 $F(m, n)$ 分布的**上 $\alpha$ 分位点**。
:::

**F 分布分位点的重要公式**：

$$F_{1-\alpha}(m, n) = \frac{1}{F_\alpha(n, m)}$$

这个公式非常实用！$F$ 分布表通常只给出 $\alpha$ 较小（如 $0.05, 0.025, 0.01$）时的分位点（右尾值），当需要左尾分位点 $F_{1-\alpha}(m,n)$（$1-\alpha$ 接近 $1$）时，利用此公式转化为右尾查表。

:::derivation{label=F 分位点公式的推导}
设 $F \sim F(m,n)$，由定义，$F_\alpha(m,n)$ 满足

$$P(F > F_\alpha(m,n)) = \alpha$$

等价地，

$$P(F \leqslant F_\alpha(m,n)) = 1 - \alpha$$

$$P\!\left(\frac{1}{F} \geqslant \frac{1}{F_\alpha(m,n)}\right) = 1 - \alpha$$

由倒数性质，$\dfrac{1}{F} \sim F(n,m)$，故

$$P\!\left(\frac{1}{F} > \frac{1}{F_\alpha(m,n)}\right) = \alpha$$

（因连续分布，$\geqslant$ 和 $>$ 概率相同）

对照 $F(n,m)$ 上 $\alpha$ 分位点的定义，得

$$F_\alpha(n,m) = \frac{1}{F_\alpha(m,n)} \implies F_{1-\alpha}(m,n) = \frac{1}{F_\alpha(n,m)}$$
:::

## 四、三大分布的统一视角：上 $\alpha$ 分位点

:::insight{label=分位点的统一理解}
不管是 $\chi^2$、$t$ 还是 $F$ 分布，"上 $\alpha$ 分位点"的定义都一样：

> 给定一个概率 $\alpha$（右尾面积），找出一个临界点，使随机变量落在该点右侧的概率恰好等于 $\alpha$。

用通用记号写，若随机变量 $X$ 服从某分布，则上 $\alpha$ 分位点 $x_\alpha$ 满足

$$P(X > x_\alpha) = \alpha$$

这是"正向"（已知分布和范围求概率）的**逆问题**。在假设检验和区间估计中，我们往往先给定显著水平 $\alpha$，然后通过分位点确定拒绝域或置信区间的边界。
:::

**三种分布分位点对比：**

| 分布 | 对称性 | 特殊关系 | 查表规则 |
|---|---|---|---|
| $\chi^2(n)$ | 非对称（右偏） | 大 $n$ 近似正态 | 直接查 $\chi^2$ 表 |
| $t(n)$ | 关于 $0$ 对称 | $t_{1-\alpha}(n) = -t_\alpha(n)$；$n>45$ 近似 $u_\alpha$ | 查 $t$ 表或标准正态 |
| $F(m,n)$ | 非对称（右偏） | $F_{1-\alpha}(m,n) = 1/F_\alpha(n,m)$ | 只需查右尾表 |

## 五、例题精讲

:::example{label=例1：识别χ²分布}
**题目**：设 $X_1, X_2, \ldots, X_{10}$ 是来自正态总体 $N(0, 0.3^2)$ 的样本，求

$$P\!\left(\sum_{i=1}^{10} X_i^2 > 1.44\right)$$

**解题过程：**

**第一步：识别分布**

注意 $X_i \sim N(0, 0.09)$，$\sigma = 0.3$，不是标准正态！所以 $X_i^2$ **不是** $\chi^2(1)$，不能直接求和成 $\chi^2(10)$。

**第二步：标准化**

$$\frac{X_i}{\sigma} = \frac{X_i}{0.3} \sim N(0,1)$$

从而 $\left(\dfrac{X_i}{0.3}\right)^2 \sim \chi^2(1)$，各项独立（样本独立），故

$$\sum_{i=1}^{10}\left(\frac{X_i}{0.3}\right)^2 = \frac{1}{0.09}\sum_{i=1}^{10}X_i^2 \sim \chi^2(10)$$

**第三步：转化概率**

$$P\!\left(\sum_{i=1}^{10}X_i^2 > 1.44\right) = P\!\left(\frac{1}{0.09}\sum_{i=1}^{10}X_i^2 > \frac{1.44}{0.09}\right) = P(\chi^2(10) > 16)$$

**第四步：查表**

查 $\chi^2$ 分布表，自由度 $n=10$，$\chi^2_{0.1}(10) = 15.987 \approx 16$，故

$$P(\chi^2(10) > 16) \approx 0.1$$

**答**：$P\!\left(\sum_{i=1}^{10}X_i^2 > 1.44\right) \approx 0.1$

**关键反思**：一定要先标准化，再识别 $\chi^2$ 分布。直接看到平方和就写 $\chi^2$ 是常见错误。
:::

:::example{label=例2：t 分布的构造识别}
**题目**：设 $X_1, X_2, \ldots, X_n$ 是来自正态总体 $N(\mu, \sigma^2)$ 的样本，$\bar{X}$ 为样本均值，$S^2$ 为样本方差，$\sigma^2$ 未知。证明

$$T = \frac{\bar{X} - \mu}{S/\sqrt{n}} \sim t(n-1)$$

**解题过程：**

构造 $t$ 分布需要：分子是标准正态，分母是 $\sqrt{\chi^2/\text{自由度}}$，且二者独立。

**第一步：找标准正态**

由于 $X_i \sim N(\mu, \sigma^2)$ 独立同分布，所以

$$\bar{X} \sim N\!\left(\mu, \frac{\sigma^2}{n}\right)$$

（正态的线性组合仍是正态，这是精确结果，不是近似！）

标准化得

$$U = \frac{\bar{X} - \mu}{\sigma/\sqrt{n}} \sim N(0,1)$$

**第二步：找 $\chi^2$**

由正态总体的定理（第6.3节将详证），

$$V = \frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)$$

**第三步：验证独立性**

正态总体中，$\bar{X}$ 与 $S^2$ 相互独立（这是正态分布的特殊性质），故 $U$ 与 $V$ 独立。

**第四步：凑出 $t$ 结构**

$$T = \frac{\bar{X} - \mu}{S/\sqrt{n}} = \frac{\bar{X} - \mu}{\sigma/\sqrt{n}} \cdot \frac{\sigma}{S} = U \cdot \frac{\sigma}{S}$$

注意到 $S^2 = \dfrac{\sigma^2 V}{n-1}$，故 $\dfrac{\sigma}{S} = \sqrt{\dfrac{n-1}{V}}$，因此

$$T = U \cdot \sqrt{\frac{n-1}{V}} = \frac{U}{\sqrt{V/(n-1)}}$$

这正好是 $t$ 分布的构造：分子是 $N(0,1)$，分母是 $\sqrt{\chi^2(n-1)/(n-1)}$，且二者独立。

$$T \sim t(n-1)$$

**意义**：这说明当 $\sigma$ 未知时，用 $S$ 替代 $\sigma$ 标准化后不是正态，而是 $t(n-1)$。小样本时这个差别很大，不能忽略。
:::

:::example{label=例3：F 分布的识别}
**题目**：设 $F \sim F(5, 10)$，已知 $F_{0.05}(5,10) = 3.33$，求 $P(F < 0.30)$。

**分析**：需要计算左尾概率，而 $F$ 分布表通常只给右尾。利用倒数关系转化。

**解题过程：**

**第一步：利用倒数性质**

由 $F \sim F(5,10)$，知 $1/F \sim F(10,5)$。

$$P(F < 0.30) = P\!\left(\frac{1}{F} > \frac{1}{0.30}\right) = P\!\left(\frac{1}{F} > 3.33\right)$$

**第二步：识别分位点**

$1/F \sim F(10,5)$，故

$$P\!\left(\frac{1}{F} > 3.33\right) = P(F(10,5) > 3.33)$$

查 $F$ 分布表（或已知条件），$F_{0.05}(10,5) = 4.74$……

等等，我们需要的是 $F_{0.05}(5,10)$ 的倒数关系：

$$F_{1-0.05}(5,10) = \frac{1}{F_{0.05}(10,5)}$$

反过来，$F_{0.05}(5,10) = 3.33$，所以

$$F_{1-0.05}(10,5) = F_{0.95}(10,5) = \frac{1}{F_{0.05}(5,10)} = \frac{1}{3.33} \approx 0.30$$

**第三步：得出结论**

由分位点定义，$F_{0.95}(10,5)$ 满足 $P(F(10,5) > F_{0.95}(10,5)) = 0.05$，即

$$P(F(10,5) \leqslant 0.30) = 0.95 \implies P(F(10,5) > 0.30) = 0.05$$

因此

$$P(F < 0.30) = P\!\left(\frac{1}{F} > 3.33\right) = P(F(10,5) > 3.33)$$

查 $F$ 表（已知 $F_{0.05}(10,5) = 4.74$，但实际上这里需要另行查表，核心方法是倒数变换）。

**简洁做法**：直接用公式 $P(F < c) = P(1/F > 1/c)$，将左尾转化为右尾问题，再利用 $1/F \sim F(n,m)$ 查表。
:::

:::example{label=例4：样本均值的概率计算（σ 已知与未知的对比）}
**题目**：设总体 $X \sim N(12, 4)$（即 $\mu = 12$，$\sigma^2 = 4$，$\sigma = 2$），抽取样本容量为 $n = 25$ 的样本。

（1）$\sigma^2 = 4$ 已知，求 $P(\bar{X} > 12.5)$；

（2）$\sigma^2$ 未知，已知样本方差观测值 $s^2 = 5.76$（即 $s = 2.4$），求 $P(\bar{X} > 12.5)$ 的近似。

**解题过程：**

**（1）$\sigma$ 已知的情形**

$$\bar{X} \sim N\!\left(12, \frac{4}{25}\right) = N(12, 0.16)$$

标准化：

$$\frac{\bar{X} - 12}{2/\sqrt{25}} = \frac{\bar{X} - 12}{0.4} \sim N(0,1)$$

故

$$P(\bar{X} > 12.5) = P\!\left(\frac{\bar{X}-12}{0.4} > \frac{12.5-12}{0.4}\right) = P(Z > 1.25) = 1 - \Phi(1.25) \approx 0.1056$$

**（2）$\sigma$ 未知的情形**

用 $S$ 替代 $\sigma$ 后，

$$T = \frac{\bar{X} - 12}{S/\sqrt{25}} = \frac{\bar{X} - 12}{S/5} \sim t(24)$$

注意：此时统计量服从 $t(n-1) = t(24)$ 分布，而非标准正态。

代入 $s = 2.4$：

$$P(\bar{X} > 12.5) = P\!\left(T > \frac{12.5 - 12}{2.4/5}\right) = P\!\left(T > \frac{0.5}{0.48}\right) = P(T > 1.042)$$

查 $t(24)$ 分布表，找到 $t_\alpha(24) = 1.042$ 对应的 $\alpha$ 值，可得 $P(T > 1.042) \approx 0.154$。

**对比**：
- $\sigma$ 已知时（用标准正态）：概率 $\approx 0.106$
- $\sigma$ 未知时（用 $t$ 分布）：概率 $\approx 0.154$

$t$ 分布尾部更厚，所以同样的超过某点的概率**更大**，这反映了由于 $\sigma$ 未知引入的额外不确定性。
:::

## 六、易错点汇总

:::pitfall{label=五大易错点}
**1. $\chi^2$ 分布忘记先标准化**

看到平方和就写 $\chi^2$，但 $\sum X_i^2$ 只有在 $X_i \sim N(0,1)$ 时才是 $\chi^2$。如果 $X_i \sim N(\mu, \sigma^2)$，必须先写成 $\left(\dfrac{X_i - \mu}{\sigma}\right)^2$ 再求和。

**2. 混淆 $F(m,n)$ 与 $F(n,m)$**

两个自由度有顺序！$F(3,10) \neq F(10,3)$。分子自由度在前，分母自由度在后，这个顺序决定了是完全不同的分布。

**3. $t$ 分布与标准正态混用**

样本量小且 $\sigma$ 未知时，必须用 $t(n-1)$，不能用 $N(0,1)$。只有 $n > 45$ 时才可以近似为标准正态。

**4. 忘记独立性条件**

$\chi^2(n) = \sum X_i^2$ 要求 $X_i$ 相互独立；$t(n)$ 构造中分子分母独立；$F(m,n)$ 构造中两个 $\chi^2$ 独立。独立性缺失，构造就不成立。

**5. 分位点查表方向**

上 $\alpha$ 分位点 $x_\alpha$ 满足 $P(X > x_\alpha) = \alpha$，是**右尾**面积等于 $\alpha$。查表时给的是 $\alpha$ 和自由度，找的是临界值。查出的数 $x_\alpha$ 越大，意味着 $\alpha$ 越小（右尾越小，临界点越往右）。
:::

## 七、本节小结

:::note{label=本节小结}
本节构造了三大抽样分布，它们都从标准正态出发：

| 分布 | 构造方式 | 参数 | 对称性 | 重要性质 |
|---|---|---|---|---|
| $\chi^2(n)$ | $n$ 个独立 $N(0,1)$ 的平方和 | 自由度 $n$ | 右偏 | 可加性；期望 $n$，方差 $2n$ |
| $t(n)$ | $N(0,1)$ 除以 $\sqrt{\chi^2(n)/n}$（独立） | 自由度 $n$ | 关于 $0$ 对称 | $n\to\infty$ 趋近标准正态 |
| $F(m,n)$ | $\chi^2(m)/m$ 除以 $\chi^2(n)/n$（独立） | 两个自由度 $m,n$ | 右偏 | $1/F \sim F(n,m)$；$T^2 \sim F(1,n)$ |

**上 $\alpha$ 分位点**是三种分布共同的工具概念：给定右尾面积 $\alpha$，逆向找临界点。

**下一节**（6.3 节）将利用本节的三大分布，给出正态总体中样本均值 $\bar{X}$ 与样本方差 $S^2$ 的抽样分布定理——这是第七章区间估计和第八章假设检验的直接基础。核心结论是：$\dfrac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)$，以及 $\bar{X}$ 与 $S^2$ 相互独立，由此可以构造出各种实用的 $t$ 统计量和 $F$ 统计量。
:::
