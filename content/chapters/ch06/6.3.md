## 正态总体的抽样分布：四个奠基性定理

### 从一个疑问出发

你在工厂的质检部门工作。流水线上生产的零件直径服从正态分布，均值 $\mu = 10$ mm，方差 $\sigma^2 = 0.04$（$\text{mm}^2$）。每天你随机抽取 $n = 16$ 个零件，测量它们的直径，计算样本均值 $\bar{X}$ 和样本方差 $S^2$。

问题来了——

- **$\bar{X}$ 是什么分布？** 它会偏离总体均值 $\mu$ 多少？
- **$S^2$ 是什么分布？** 它的随机性如何刻画？
- **$\bar{X}$ 与 $S^2$ 之间有没有关联？** 一个偏大的时候，另一个会偏大吗？
- **如果 $\sigma$ 未知**，用 $S$ 代替 $\sigma$ 之后，原来那套"标准化—查正态表"的方法还管用吗？

这四个问题，正好对应本节的四个核心定理。它们是后续**区间估计**和**假设检验**的数学基础——没有这四个定理，那些检验公式就只是凭空落下来的符号，无从推导，也无从理解。

## 预备：正态分布的线性稳定性

在学习四个定理之前，先回顾一个关键事实：**正态分布对线性运算是"封闭"的**。

:::theorem{label=正态分布的线性组合}
设 $X_1, X_2, \ldots, X_n$ 相互独立，且 $X_i \sim N(\mu_i, \sigma_i^2)$，则它们的线性组合

$$Y = a_1 X_1 + a_2 X_2 + \cdots + a_n X_n$$

仍然服从正态分布，且

$$Y \sim N\!\left(\sum_{i=1}^n a_i \mu_i,\; \sum_{i=1}^n a_i^2 \sigma_i^2\right)$$
:::

这个性质太重要了。它说的是：**独立正态的线性组合，还是正态**。这一点，和"中心极限定理给出的近似正态"有本质区别——这里是**精确的正态**，不需要 $n$ 很大。

## 定理一：样本均值的分布

### 直觉理解

想象你每次抽 16 个零件，算平均值。每次得到的 $\bar{X}$ 都是一个随机数。"把这个游戏重复无数次"，$\bar{X}$ 的这些随机值构成一个分布——这就是 $\bar{X}$ 的**抽样分布**。

直觉上：
- $\bar{X}$ 应该在总体均值 $\mu$ 附近波动（越多人投票，平均结果越稳定）；
- 16 个数一平均，比单个数波动要小（小很多）；
- 总体是正态，$\bar{X}$ 是它的加权平均，线性运算——所以 $\bar{X}$ 也应该是正态。

:::theorem{label=定理一：样本均值的分布}
设总体 $X \sim N(\mu, \sigma^2)$，$X_1, X_2, \ldots, X_n$ 是来自该总体的简单随机样本，$\bar{X} = \dfrac{1}{n}\sum_{i=1}^n X_i$ 是样本均值。则

$$\bar{X} \sim N\!\left(\mu,\; \frac{\sigma^2}{n}\right)$$

等价地，对 $\bar{X}$ 标准化后：

$$\frac{\bar{X} - \mu}{\sigma / \sqrt{n}} \sim N(0, 1)$$
:::

:::derivation{label=定理一推导}
由于 $X_1, X_2, \ldots, X_n$ 独立同分布，每个 $X_i \sim N(\mu, \sigma^2)$。

$$\bar{X} = \frac{1}{n}\sum_{i=1}^n X_i = \frac{1}{n}X_1 + \frac{1}{n}X_2 + \cdots + \frac{1}{n}X_n$$

这是独立正态随机变量的线性组合，系数均为 $\dfrac{1}{n}$，由正态分布线性稳定性：

$$\begin{aligned}
E(\bar{X}) &= \sum_{i=1}^n \frac{1}{n} \cdot \mu = \mu \\[6pt]
D(\bar{X}) &= \sum_{i=1}^n \left(\frac{1}{n}\right)^2 \cdot \sigma^2 = \frac{n \sigma^2}{n^2} = \frac{\sigma^2}{n}
\end{aligned}$$

因此 $\bar{X} \sim N\!\left(\mu, \dfrac{\sigma^2}{n}\right)$。这是**精确**结论，不是近似！
:::

:::insight{label=为什么方差缩小为 σ²/n}
方差 $\dfrac{\sigma^2}{n}$ 比总体方差 $\sigma^2$ 小了 $n$ 倍。

直观理解：你抽的人越多（$n$ 越大），样本均值就越稳定，随机波动越小。抽 1 个人和抽 1000 个人取平均，哪个平均值更"靠谱"？当然是后者。数学上，这正是"平均"在降低方差：$n$ 个随机量的误差"互相抵消"，方差以 $1/n$ 的速率衰减。

标准差从 $\sigma$ 变成 $\dfrac{\sigma}{\sqrt{n}}$，称为**均值的标准误差（Standard Error）**，简称**标准误**。
:::

## 定理二：样本方差服从卡方分布

### 直觉铺垫：自由度为何是 $n-1$？

样本方差定义为

$$S^2 = \frac{1}{n-1}\sum_{i=1}^n (X_i - \bar{X})^2$$

分母是 $n-1$，而不是 $n$。为什么？

关键在于求和式 $\sum_{i=1}^n(X_i - \bar{X})^2$ 中，$n$ 个偏差 $(X_i - \bar{X})$ **不是独立的**——它们受到约束：

$$\sum_{i=1}^n (X_i - \bar{X}) = 0$$

这意味着只要知道前 $n-1$ 个偏差，第 $n$ 个就完全确定了。所以这个平方和实际上只有 $n-1$ 个"自由的"信息，自由度为 $n-1$。

用 $\bar{X}$（样本估计量）代替了 $\mu$（总体真值），消耗了一个自由度，剩下 $n-1$ 个。

:::theorem{label=定理二：样本方差的分布}
设总体 $X \sim N(\mu, \sigma^2)$，$X_1, \ldots, X_n$ 是来自该总体的简单随机样本，$S^2$ 是样本方差，则

$$\frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)$$

即 $(n-1)S^2/\sigma^2$ 服从自由度为 $n-1$ 的卡方分布。
:::

:::insight{label=这个结论为什么重要}
我们来解读一下 $\dfrac{(n-1)S^2}{\sigma^2}$ 的意思。

把 $S^2$ 展开：

$$\frac{(n-1)S^2}{\sigma^2} = \frac{1}{\sigma^2}\sum_{i=1}^n(X_i - \bar{X})^2 = \sum_{i=1}^n \left(\frac{X_i - \bar{X}}{\sigma}\right)^2$$

如果用总体均值 $\mu$ 代替 $\bar{X}$，那么

$$\sum_{i=1}^n \left(\frac{X_i - \mu}{\sigma}\right)^2 \sim \chi^2(n)$$

因为每个 $\dfrac{X_i - \mu}{\sigma}$ 是独立的标准正态，平方和就是 $\chi^2(n)$。

但现在用 $\bar{X}$ 代替了 $\mu$，相当于加了一个约束（$\sum(X_i - \bar{X}) = 0$），自由度从 $n$ 降到 $n-1$。

这正是用样本均值估计总体均值需要"付出的代价"——每用一个估计量，就损失一个自由度。
:::

:::derivation{label=定理二推导思路（非严格版）}
从 $\chi^2(n)$ 到 $\chi^2(n-1)$ 的关键分解：

注意恒等式

$$\sum_{i=1}^n \left(\frac{X_i - \mu}{\sigma}\right)^2 = \sum_{i=1}^n \left(\frac{X_i - \bar{X}}{\sigma}\right)^2 + n\left(\frac{\bar{X} - \mu}{\sigma}\right)^2$$

即

$$\underbrace{\sum_{i=1}^n \left(\frac{X_i - \mu}{\sigma}\right)^2}_{\chi^2(n)} = \underbrace{\frac{(n-1)S^2}{\sigma^2}}_{\text{待确定}} + \underbrace{\left(\frac{\bar{X}-\mu}{\sigma/\sqrt{n}}\right)^2}_{\chi^2(1)}$$

可以证明右边两项相互独立（利用正态分布的特殊性质），又由卡方分布的可加性，

$$\chi^2(n) = \frac{(n-1)S^2}{\sigma^2} + \chi^2(1)$$

因此 $\dfrac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)$。

**严格证明**需要正交变换（Helmert 变换），将 $X_1 - \bar{X}, \ldots, X_n - \bar{X}$ 分解为 $n-1$ 个独立的标准正态的平方和，超出本课范围，了解结论即可。
:::

## 定理三：$\bar{X}$ 与 $S^2$ 的独立性

这是正态总体**独有的**一个美丽结论，在其他分布中未必成立。

:::theorem{label=定理三：独立性}
设总体 $X \sim N(\mu, \sigma^2)$，$X_1, \ldots, X_n$ 是来自该总体的简单随机样本。则

$$\bar{X} \text{ 与 } S^2 \text{ 相互独立}$$
:::

:::insight{label=为什么独立性如此珍贵}
$\bar{X}$ 描述样本的"中心位置"，$S^2$ 描述样本的"离散程度"。

直觉上，一组数据的位置信息和离散信息应该"没有必然联系"——平均值高不代表方差大。但对于**非正态总体**，$\bar{X}$ 和 $S^2$ 未必独立（例如，泊松分布的样本均值和方差就有相关性）。

正态分布由于其特殊的对称性和指数型的概率结构，使得 $\bar{X}$ 和 $S^2$ 完全独立。

这个独立性是构造第四个定理（$t$ 分布）的**关键前提**：我们需要把分子（涉及 $\bar{X}$）和分母（涉及 $S$）作为**独立的**量拼在一起。
:::

## 定理四：$t$ 统计量及其分布

### 用 $S$ 替换 $\sigma$ 会带来什么？

在实际问题中，总体方差 $\sigma^2$ 通常是**未知的**。定理一告诉我们

$$\frac{\bar{X} - \mu}{\sigma / \sqrt{n}} \sim N(0,1)$$

但 $\sigma$ 未知时，这个公式无法直接使用！怎么办？

自然的想法：**用样本标准差 $S$ 替换总体标准差 $\sigma$**。把 $\sigma$ 换成 $S$，得到

$$T = \frac{\bar{X} - \mu}{S / \sqrt{n}}$$

问题是：这个 $T$ 还是标准正态吗？

**不是了！** 分母从一个**常数** $\sigma$ 变成了一个**随机变量** $S$，引入了新的随机性。$T$ 比标准正态有更"厚的尾部"（更容易出现较大的偏差）。

:::theorem{label=定理四：t 统计量的分布}
设总体 $X \sim N(\mu, \sigma^2)$，$X_1, \ldots, X_n$ 是来自该总体的简单随机样本，$\bar{X}$ 是样本均值，$S^2$ 是样本方差。则

$$T = \frac{\bar{X} - \mu}{S / \sqrt{n}} \sim t(n-1)$$

即 $T$ 服从自由度为 $n-1$ 的 $t$ 分布（又称学生氏分布）。
:::

:::derivation{label=定理四推导——从定义出发凑构造}
$t$ 分布的定义：若 $U \sim N(0,1)$，$V \sim \chi^2(n)$，且 $U$、$V$ 相互独立，则

$$\frac{U}{\sqrt{V/n}} \sim t(n)$$

我们来把 $T = \dfrac{\bar{X}-\mu}{S/\sqrt{n}}$ 凑成这个形式。

**第一步：找 $U$（标准正态部分）**

由定理一知

$$U = \frac{\bar{X} - \mu}{\sigma / \sqrt{n}} \sim N(0,1)$$

**第二步：找 $V$（卡方部分）**

由定理二知

$$V = \frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)$$

**第三步：验证独立性**

由定理三，$\bar{X}$ 与 $S^2$ 独立，故 $U$（是 $\bar{X}$ 的函数）与 $V$（是 $S^2$ 的函数）也独立。

**第四步：拼出 $t$ 分布**

$$\begin{aligned}
T &= \frac{\bar{X} - \mu}{S/\sqrt{n}} \\[8pt]
&= \frac{\bar{X} - \mu}{\sigma/\sqrt{n}} \cdot \frac{\sigma/\sqrt{n}}{S/\sqrt{n}} \\[8pt]
&= U \cdot \frac{\sigma}{S} \\[8pt]
&= \frac{U}{\sqrt{S^2/\sigma^2}} \\[8pt]
&= \frac{U}{\sqrt{\dfrac{(n-1)S^2}{\sigma^2} \cdot \dfrac{1}{n-1}}} \\[10pt]
&= \frac{U}{\sqrt{V/(n-1)}}
\end{aligned}$$

这正好是 $t$ 分布的构造形式（分子标准正态，分母是卡方除以自由度再开方，且二者独立），故

$$T \sim t(n-1)$$

自由度是分母中卡方的自由度 $n-1$，不是 $n$！
:::

:::insight{label=t 分布与正态分布的区别}
$t(n-1)$ 分布有如下特征：

1. **关于 $y$ 轴对称**，期望为 0（$n > 1$ 时），形状酷似标准正态；
2. **尾部更厚**：$t$ 分布比标准正态有更大的概率出现"极端值"，这是因为分母 $S$ 也带有随机性（会偶然偏小，使 $T$ 偶然偏大）；
3. **自由度越大，越接近正态**：当 $n \to \infty$（等价于 $n-1 \to \infty$）时，$t(n-1) \to N(0,1)$。实际中，$n \geq 45$ 时，$t$ 分布与标准正态非常接近，可用正态表代替。

这解释了一个实践中的结论：**小样本时必须用 $t$ 分布，大样本时近似用正态分布**。

对称性带来一个有用公式：若 $T \sim t(n)$，则

$$t_{1-\alpha}(n) = -t_{\alpha}(n)$$

即下 $\alpha$ 分位点是上 $\alpha$ 分位点的相反数。
:::

## 四个定理的"全景图"

在进入例题之前，用一张示意图把四个定理整合起来：

$$\boxed{X \sim N(\mu, \sigma^2)}$$

$$\Downarrow \text{ 抽 } n \text{ 个样本}$$

$$\begin{aligned}
&\textbf{定理一：} & \bar{X} &\sim N\!\left(\mu,\, \frac{\sigma^2}{n}\right) \\[6pt]
&\textbf{定理二：} & \frac{(n-1)S^2}{\sigma^2} &\sim \chi^2(n-1) \\[6pt]
&\textbf{定理三：} & \bar{X} &\perp S^2 \quad (\text{独立}) \\[6pt]
&\textbf{定理四：} & \frac{\bar{X} - \mu}{S/\sqrt{n}} &\sim t(n-1)
\end{aligned}$$

定理四是在定理一、二、三的**联合作用**下推导出来的——三者缺一不可。

## 例题

:::example{label=例1：样本均值落在某区间的概率}
设总体 $X \sim N(12, 4)$（即 $\mu = 12$，$\sigma^2 = 4$，$\sigma = 2$）。从该总体中抽取容量为 $n = 25$ 的简单随机样本，求样本均值 $\bar{X} > 12.5$ 的概率。

**解：**

**第一步：确定 $\bar{X}$ 的分布。**

由定理一，

$$\bar{X} \sim N\!\left(\mu, \frac{\sigma^2}{n}\right) = N\!\left(12, \frac{4}{25}\right) = N(12, 0.16)$$

即 $\bar{X}$ 的标准差为 $\dfrac{\sigma}{\sqrt{n}} = \dfrac{2}{5} = 0.4$。

**第二步：标准化。**

$$P(\bar{X} > 12.5) = P\!\left(\frac{\bar{X} - 12}{0.4} > \frac{12.5 - 12}{0.4}\right) = P(Z > 1.25)$$

其中 $Z \sim N(0,1)$。

**第三步：查标准正态分布表。**

$$P(Z > 1.25) = 1 - \Phi(1.25) = 1 - 0.8944 = 0.1056$$

**结论：** 样本均值超过 $12.5$ 的概率约为 $10.56\%$。

**注意：** 这里 $\sigma = 2$ 已知，所以用正态分布，而不是 $t$ 分布。
:::

:::example{label=例2：σ 未知时用 t 分布}
同上题，设总体 $X \sim N(12, \sigma^2)$（$\sigma^2$ **未知**），从该总体中抽取 $n = 25$ 个样本，观测到样本方差 $s^2 = 3.8$（即 $s = \sqrt{3.8} \approx 1.949$）。求 $P(\bar{X} > 12.5)$。

**解：**

**第一步：$\sigma$ 未知，改用 $t$ 统计量。**

由定理四，

$$T = \frac{\bar{X} - \mu}{S/\sqrt{n}} \sim t(n-1) = t(24)$$

**第二步：把事件转化为关于 $T$ 的不等式。**

$$P(\bar{X} > 12.5) = P\!\left(\frac{\bar{X} - 12}{S/\sqrt{25}} > \frac{12.5 - 12}{S/\sqrt{25}}\right)$$

注意：在计算具体概率时，我们把 $S$ 用观测值 $s = \sqrt{3.8}$ 代入。

$$\frac{12.5 - 12}{s/5} = \frac{0.5}{\sqrt{3.8}/5} = \frac{0.5 \times 5}{\sqrt{3.8}} = \frac{2.5}{1.949} \approx 1.283$$

**第三步：查 $t(24)$ 分布表。**

查 $t(24)$ 分布上分位点表，可得

$$P(T > 1.059) = 0.15, \quad P(T > 1.318) = 0.10$$

由插值（或精确查表），$t \approx 1.283$ 对应的上尾概率约为

$$P(\bar{X} > 12.5) \approx 0.11$$

**与例1对比：** 当 $\sigma^2 = 4$（已知）时答案约为 $0.1056$；当 $\sigma^2$ 未知、用 $s^2 = 3.8$ 估计时答案约为 $0.11$。两者接近，因为 $s^2 \approx \sigma^2$，但使用的分布不同（正态 vs $t(24)$），处理方法有本质区别。
:::

:::example{label=例3：关于 S² 的概率计算}
设总体 $X \sim N(\mu, \sigma^2)$，$\sigma^2 = 1$（已知），从中抽取 $n = 10$ 个样本。求 $P(S^2 > 1.8)$。

**解：**

**第一步：将 $S^2$ 转化为卡方统计量。**

由定理二，

$$\frac{(n-1)S^2}{\sigma^2} = \frac{9 S^2}{1} = 9S^2 \sim \chi^2(9)$$

**第二步：转化事件。**

$$P(S^2 > 1.8) = P(9S^2 > 9 \times 1.8) = P(\chi^2(9) > 16.2)$$

**第三步：查 $\chi^2$ 分布表。**

查 $\chi^2(9)$ 上分位点表：

- $\chi^2_{0.05}(9) = 16.919$（即 $P(\chi^2(9) > 16.919) = 0.05$）
- $\chi^2_{0.10}(9) = 14.684$（即 $P(\chi^2(9) > 14.684) = 0.10$）

由于 $14.684 < 16.2 < 16.919$，所以

$$0.05 < P(S^2 > 1.8) < 0.10$$

若查更精细的表或使用软件，$P(\chi^2(9) > 16.2) \approx 0.063$。

**解释：** 样本方差超过总体方差 $1.8$ 倍的概率约为 $6.3\%$，属于小概率事件，不算太反常。
:::

:::example{label=例4：综合应用——识别统计量的分布}
设 $X_1, X_2, \ldots, X_9$ 是来自正态总体 $N(0, 9)$ 的简单随机样本（即 $\mu = 0$，$\sigma^2 = 9$，$\sigma = 3$）。

设 $\bar{X} = \dfrac{1}{9}\sum_{i=1}^9 X_i$，$S^2 = \dfrac{1}{8}\sum_{i=1}^9 (X_i - \bar{X})^2$。

问以下统计量各服从什么分布？

（1）$\dfrac{\bar{X}}{3/\sqrt{9}} = \dfrac{\bar{X}}{1}$

（2）$\dfrac{\bar{X} - 0}{S/\sqrt{9}} = \dfrac{3\bar{X}}{S}$

（3）$\dfrac{1}{9}\sum_{i=1}^9 X_i^2$

**解：**

**(1)** $\dfrac{\bar{X}}{1} = \bar{X}$

由定理一，$\bar{X} \sim N\!\left(0, \dfrac{9}{9}\right) = N(0, 1)$。

因此 $\dfrac{\bar{X} - 0}{\sigma/\sqrt{n}} = \dfrac{\bar{X}}{3/3} = \bar{X} \sim N(0,1)$。

**(2)** $\dfrac{3\bar{X}}{S}$

注意 $\dfrac{\bar{X} - \mu}{S/\sqrt{n}} = \dfrac{\bar{X} - 0}{S/3} = \dfrac{3\bar{X}}{S}$。

由定理四，

$$\frac{3\bar{X}}{S} = \frac{\bar{X} - 0}{S/\sqrt{9}} \sim t(n-1) = t(8)$$

**(3)** $\dfrac{1}{9}\sum_{i=1}^9 X_i^2$

每个 $X_i \sim N(0, 9)$，因此 $\dfrac{X_i}{3} \sim N(0,1)$（标准正态），于是 $\left(\dfrac{X_i}{3}\right)^2 \sim \chi^2(1)$。

由独立性和卡方的可加性，

$$\sum_{i=1}^9 \left(\frac{X_i}{3}\right)^2 = \frac{1}{9}\sum_{i=1}^9 X_i^2 \sim \chi^2(9)$$

因此 $\dfrac{1}{9}\sum_{i=1}^9 X_i^2 \sim \chi^2(9)$。

**注意：** 此处是 $\mu = 0$ 已知，所以是对 $\mu$ 而不是对 $\bar{X}$ 做标准化，得到 $\chi^2(9)$ 而非 $\chi^2(8)$。若 $\mu$ 未知，则 $\sum(X_i - \bar{X})^2/\sigma^2 \sim \chi^2(8)$。
:::

## 易错点

:::pitfall{label=5 个高频错误}

**1. 把定理四中的自由度弄成 $n$**

$$T = \frac{\bar{X} - \mu}{S/\sqrt{n}} \sim t(\mathbf{n-1}), \quad \text{不是 } t(n)$$

自由度是 $n-1$，因为 $S^2$ 的卡方分布自由度是 $n-1$。

---

**2. $\bar{X}$ 的正态性——"精确"还是"近似"傻傻分不清**

- 总体是正态 $\Rightarrow$ $\bar{X}$ **精确**服从正态（无论 $n$ 大小）。
- 总体不是正态 $\Rightarrow$ 只有当 $n$ 足够大时，由中心极限定理，$\bar{X}$ **近似**服从正态。

本节所有定理都基于**正态总体**，结论是精确的，不需要 $n$ 大。

---

**3. $\mu$ 已知 vs 未知影响卡方自由度**

$$\sum_{i=1}^n \left(\frac{X_i - \mu}{\sigma}\right)^2 \sim \chi^2(\mathbf{n}) \quad (\mu \text{ 已知})$$

$$\frac{(n-1)S^2}{\sigma^2} = \sum_{i=1}^n \left(\frac{X_i - \bar{X}}{\sigma}\right)^2 \sim \chi^2(\mathbf{n-1}) \quad (\mu \text{ 未知，用 } \bar{X} \text{ 代替})$$

区别就在于减的是 $\mu$（已知常数，不消耗自由度）还是 $\bar{X}$（估计量，消耗一个自由度）。

---

**4. $\sigma$ 已知用正态，$\sigma$ 未知用 $t$**

| 情况 | 使用分布 |
|---|---|
| $\sigma$ 已知 | $\dfrac{\bar{X}-\mu}{\sigma/\sqrt{n}} \sim N(0,1)$ |
| $\sigma$ 未知 | $\dfrac{\bar{X}-\mu}{S/\sqrt{n}} \sim t(n-1)$ |

这个区别在假设检验和置信区间构造中至关重要。

---

**5. $S^2$ 前面系数别丢——是 $(n-1)S^2/\sigma^2$，不是 $S^2/\sigma^2$**

正确结论是 $\dfrac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)$，前面必须有 $n-1$。若丢掉这个系数，分布的均值和方差全部出错。
:::

## 正态总体独有性质的深层含义

:::insight{label=为什么正态总体如此特殊}
本节的四个定理合在一起，展现了正态总体的一个"完美性质"：

1. 线性运算（$\bar{X}$）仍然正态；
2. 平方和运算（$(n-1)S^2/\sigma^2$）精确服从卡方；
3. $\bar{X}$ 和 $S^2$ 相互独立（这在非正态总体中几乎从不成立）；
4. 用 $S$ 替换 $\sigma$ 之后，统计量精确服从 $t$ 分布。

这种"精确可推导"的性质，使得正态总体成为数理统计中最重要、研究最透彻的总体。

在大样本理论中，很多非正态总体的统计量也可以近似用正态、卡方、$t$ 分布来处理，这背后正是中心极限定理在发挥作用——而正态分布是这些极限结论的"目标"。
:::

## 分位点与查表的衔接

本节的定理直接给出了统计量的分布，但在实际计算概率或构造区间时，需要查分布表（或用软件），涉及各分布的**上 $\alpha$ 分位点**：

- 标准正态：$u_\alpha = z_\alpha$，满足 $P(Z > u_\alpha) = \alpha$
- $t$ 分布：$t_\alpha(n)$，满足 $P(T > t_\alpha(n)) = \alpha$（其中 $T \sim t(n)$）
- $\chi^2$ 分布：$\chi^2_\alpha(n)$，满足 $P(\chi^2 > \chi^2_\alpha(n)) = \alpha$

**重要对称关系（$t$ 分布）：**

$$t_{1-\alpha}(n) = -t_{\alpha}(n)$$

例如，$t_{0.95}(10) = -t_{0.05}(10)$。这是因为 $t$ 分布关于 $y$ 轴对称。

当自由度 $n > 45$ 时，$t_\alpha(n) \approx u_\alpha$（用标准正态分位点近似 $t$ 分位点），两者差距可忽略。

---

:::note{label=本节小结}
**四个核心结论（全部基于正态总体 $X \sim N(\mu, \sigma^2)$）：**

$$\begin{aligned}
&\textbf{定理一（均值）：} && \bar{X} \sim N\!\left(\mu, \frac{\sigma^2}{n}\right) \quad \Leftrightarrow \quad \frac{\bar{X}-\mu}{\sigma/\sqrt{n}} \sim N(0,1) \\[8pt]
&\textbf{定理二（方差）：} && \frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1) \\[8pt]
&\textbf{定理三（独立性）：} && \bar{X} \perp S^2 \quad \text{（正态总体独有）} \\[8pt]
&\textbf{定理四（t 统计量）：} && \frac{\bar{X}-\mu}{S/\sqrt{n}} \sim t(n-1) \quad (\sigma \text{ 未知时使用})
\end{aligned}$$

**学习要点：**

- 定理一是"均值也正态"，关键在精确正态（不是近似）；
- 定理二的卡方自由度是 $n-1$，因为估计 $\mu$ 消耗了一个自由度；
- 定理三的独立性是正态总体的特权，其他总体不保证；
- 定理四是定理一、二、三联合作用的结果；当 $\sigma$ 未知时，用 $S$ 替换 $\sigma$，分布从 $N(0,1)$ 变为 $t(n-1)$。

**下一节（6.4）** 将利用这四个定理，学习两个正态总体的情形：两样本的均值差和方差比，分别引出涉及 $t$ 分布（均值差的推断）和 $F$ 分布（方差比的推断）的统计量。四个定理的思路将继续延伸——把"一个总体"推广到"两个总体"。
:::
