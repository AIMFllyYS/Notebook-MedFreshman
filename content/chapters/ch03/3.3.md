## 直觉与动机：已知一部分信息，如何重新描述另一个变量？

想象你在医院做健康检查，检验报告同时给出你的血糖值 $X$ 和体重 $Y$ 两个指标。这两个数字并不独立——体重偏高的人，血糖往往偏高。

如果我随机从人群中抽一个人，他的血糖 $X$ 会怎样分布？那是**边缘分布**，把全体人群都算进去。  
但如果我告诉你：这个人的体重 **已经确定**是 $Y = 80\,\text{kg}$，那么这时候血糖的分布会有所不同——因为"体重 80 kg"这个信息压缩了样本空间，**相当于我们把整个人群过滤到了体重恰好为 80 kg 的子群体**。这个在已知 $Y=y$ 的条件下，$X$ 的概率规律，就叫做 **条件分布**。

条件分布是联合分布在"某条线上"的截面——这是本节最核心的几何直觉，我们一步步展开。

## 离散型条件分布律

我们先从有限个点的情形出发，最容易看清楚结构。

:::definition{label=离散型条件分布律}
设 $(X,Y)$ 是二维离散型随机变量，其联合分布律为

$$p_{ij} = P(X = x_i,\ Y = y_j), \quad i,j=1,2,\dots$$

记 $Y$ 的边缘分布律为 $p_{\cdot j} = P(Y = y_j)$。若 $p_{\cdot j} > 0$，则称

$$P(X = x_i \mid Y = y_j) = \frac{P(X = x_i,\ Y = y_j)}{P(Y = y_j)} = \frac{p_{ij}}{p_{\cdot j}}$$

为在 $Y = y_j$ 条件下，$X$ 的**条件分布律**，简记为 $P(X=x_i \mid Y=y_j)$。
:::

这正是高中条件概率公式 $P(A\mid B) = P(AB)/P(B)$ 的直接应用，只不过这里的事件变成了 $A=\{X=x_i\}$，$B=\{Y=y_j\}$。

**验证它是合法的分布律：** 对固定的 $j$，我们需要检查

$$\sum_i P(X=x_i \mid Y=y_j) = \sum_i \frac{p_{ij}}{p_{\cdot j}} = \frac{1}{p_{\cdot j}}\sum_i p_{ij} = \frac{p_{\cdot j}}{p_{\cdot j}} = 1$$

最后一步用了边缘分布律的定义 $p_{\cdot j} = \sum_i p_{ij}$。非负性显然成立，所以它确实是一个分布律。

:::insight{label=条件分布是联合分布的"行/列归一化"}
把联合分布律写成表格形式：

|  | $Y=y_1$ | $Y=y_2$ | $\cdots$ | 行和 $p_{i\cdot}$ |
|:---:|:---:|:---:|:---:|:---:|
|$X=x_1$|$p_{11}$|$p_{12}$|$\cdots$|$p_{1\cdot}$|
|$X=x_2$|$p_{21}$|$p_{22}$|$\cdots$|$p_{2\cdot}$|
|$\vdots$|$\vdots$|$\vdots$|$\ddots$|$\vdots$|
|列和 $p_{\cdot j}$|$p_{\cdot 1}$|$p_{\cdot 2}$|$\cdots$|$1$|

**在 $Y=y_j$ 条件下，$X$ 的条件分布律**就是把第 $j$ 列的每个元素除以该列的列和 $p_{\cdot j}$。

这相当于：把视野缩到第 $j$ 列，把这一列的概率"重新归一化"为 1，得到的就是条件分布律。

几何上，等价于：在联合分布的二维"概率地图"上，沿 $Y=y_j$ 这条竖线截一刀，看这条线上 $X$ 的相对概率分布。
:::

:::example{label=摸球问题}
袋中有 3 个白球（编号 1、2、3）和 2 个黑球（编号 4、5）。不放回地抽两次，$X$ 记第一次抽到球的编号类型（1=白，2=黑），$Y$ 记第二次抽到球的编号类型（1=白，2=黑）。

联合分布律如下（可由古典概型直接计算）：

|  | $Y=1$（白） | $Y=2$（黑） |
|:---:|:---:|:---:|
|$X=1$（白）|$\dfrac{3\times 2}{5\times 4}=\dfrac{6}{20}$|$\dfrac{3\times 2}{5\times 4}=\dfrac{6}{20}$|
|$X=2$（黑）|$\dfrac{2\times 3}{5\times 4}=\dfrac{6}{20}$|$\dfrac{2\times 1}{5\times 4}=\dfrac{2}{20}$|

边缘分布：$P(Y=1)=\dfrac{6+6}{20}=\dfrac{12}{20}=\dfrac{3}{5}$，$P(Y=2)=\dfrac{6+2}{20}=\dfrac{8}{20}=\dfrac{2}{5}$。

**问：已知第二次摸出白球（$Y=1$），第一次摸出白球的概率是多少？**

$$P(X=1 \mid Y=1) = \frac{p_{11}}{p_{\cdot 1}} = \frac{6/20}{12/20} = \frac{6}{12} = \frac{1}{2}$$

**在 $Y=1$ 的条件下，$X$ 的完整条件分布律：**

$$P(X=1 \mid Y=1) = \frac{1}{2}, \quad P(X=2 \mid Y=1) = \frac{6/20}{12/20} = \frac{1}{2}$$

验证：$\frac{1}{2}+\frac{1}{2}=1$，正确。

直觉解读：已知第二次是白球，说明第一次可能拿走了一白一黑中的一个——给定第二次是白，第一次拿白或黑各一半。
:::

## 连续型条件密度函数

连续型的情况稍微复杂，因为 $P(Y=y)=0$ 对任何固定的 $y$ 都成立，直接代入条件概率公式会出现 $0/0$ 的不定形。我们需要用极限过程来定义。

### 从极限过程出发

对连续型 $(X,Y)$，联合密度为 $f(x,y)$，$Y$ 的边缘密度为

$$f_Y(y) = \int_{-\infty}^{+\infty} f(x,y)\,\mathrm{d}x$$

考虑条件概率

$$P(X \leq x \mid y < Y \leq y+\varepsilon) = \frac{P(X \leq x,\ y < Y \leq y+\varepsilon)}{P(y < Y \leq y+\varepsilon)}$$

当 $\varepsilon \to 0^+$ 时：

$$\text{分子} = \int_{-\infty}^{x}\int_{y}^{y+\varepsilon} f(u,v)\,\mathrm{d}v\,\mathrm{d}u \approx \varepsilon\int_{-\infty}^{x} f(u,y)\,\mathrm{d}u$$

$$\text{分母} = \int_{y}^{y+\varepsilon} f_Y(v)\,\mathrm{d}v \approx \varepsilon f_Y(y)$$

因此极限为

$$\lim_{\varepsilon\to 0^+} P(X\leq x \mid y < Y \leq y+\varepsilon) = \frac{\displaystyle\int_{-\infty}^{x} f(u,y)\,\mathrm{d}u}{f_Y(y)}$$

这个极限就定义为在 $Y=y$ 条件下 $X$ 的**条件分布函数** $F_{X|Y}(x|y)$，对其求导，就得到条件密度。

:::definition{label=连续型条件密度函数}
设 $(X,Y)$ 为二维连续型随机变量，联合密度为 $f(x,y)$，$Y$ 的边缘密度为 $f_Y(y)$。

若在点 $y$ 处 $f_Y(y)>0$，则称

$$f_{X|Y}(x \mid y) = \frac{f(x,y)}{f_Y(y)}$$

为在 $Y=y$ 条件下，$X$ 的**条件密度函数**（或条件概率密度）。

类似地，$X=x$ 条件下 $Y$ 的条件密度为

$$f_{Y|X}(y \mid x) = \frac{f(x,y)}{f_X(x)}, \quad f_X(x)>0$$

其中 $f_X(x) = \int_{-\infty}^{+\infty} f(x,y)\,\mathrm{d}y$ 是 $X$ 的边缘密度。
:::

**验证它是合法的密度函数：**

$$\int_{-\infty}^{+\infty} f_{X|Y}(x\mid y)\,\mathrm{d}x = \int_{-\infty}^{+\infty}\frac{f(x,y)}{f_Y(y)}\,\mathrm{d}x = \frac{1}{f_Y(y)}\cdot f_Y(y) = 1$$

非负性显然，所以它是合法的密度函数。

## 条件分布的几何意义

连续型条件密度有非常直观的几何解释。

:::insight{label=条件密度是联合密度曲面的截线归一化}
把联合密度 $f(x,y)$ 想象成 $xOy$ 平面上方的一个三维"山丘"曲面。

**在 $Y=y_0$ 这条线上截一刀**，得到的截线是

$$\text{截线函数}\quad g(x) = f(x, y_0)$$

这条截线反映的是固定 $Y=y_0$ 后，$x$ 方向上联合密度的相对大小。但它的面积（积分值）是

$$\int_{-\infty}^{+\infty} f(x,y_0)\,\mathrm{d}x = f_Y(y_0)$$

不一定等于 1，所以它还不是密度函数。把截线除以 $f_Y(y_0)$ 归一化后，就得到了条件密度

$$f_{X|Y}(x\mid y_0) = \frac{f(x,y_0)}{f_Y(y_0)}$$

**总结一句话**：条件密度 = 联合密度在某条竖（横）线上的截面，然后归一化。

这和离散型"第 $j$ 列除以列和"的操作完全一致，只是从有限个值变成了连续曲线。
:::

## 联合、边缘、条件三者的关系

三种分布之间存在非常对称的推导链条，掌握这条链条是本章的核心。

:::theorem{label=三种分布的相互转化（乘法公式）}
设 $(X,Y)$ 为二维连续型随机变量，则：

**由条件密度还原联合密度（乘法公式）：**

$$\boxed{f(x,y) = f_{X|Y}(x\mid y)\cdot f_Y(y) = f_{Y|X}(y\mid x)\cdot f_X(x)}$$

**由联合密度求边缘密度（积分）：**

$$f_Y(y) = \int_{-\infty}^{+\infty} f(x,y)\,\mathrm{d}x, \quad f_X(x) = \int_{-\infty}^{+\infty} f(x,y)\,\mathrm{d}y$$

**由联合密度求条件密度（相除）：**

$$f_{X|Y}(x\mid y) = \frac{f(x,y)}{f_Y(y)}, \quad f_{Y|X}(y\mid x) = \frac{f(x,y)}{f_X(x)}$$

这六个公式本质上是同一个等式 $f(x,y)=f_{X|Y}\cdot f_Y$ 的不同变形，学会任意两个可推出其余。
:::

:::derivation{label=乘法公式的推导与验证}
从条件密度的定义出发：

$$f_{X|Y}(x\mid y) = \frac{f(x,y)}{f_Y(y)}$$

两边同乘 $f_Y(y)$，即得

$$f(x,y) = f_{X|Y}(x\mid y)\cdot f_Y(y)$$

同理，由 $f_{Y|X}(y\mid x) = f(x,y)/f_X(x)$ 得

$$f(x,y) = f_{Y|X}(y\mid x)\cdot f_X(x)$$

因此

$$f_{X|Y}(x\mid y)\cdot f_Y(y) = f_{Y|X}(y\mid x)\cdot f_X(x)$$

这也是**贝叶斯公式**的连续型版本：

$$f_{X|Y}(x\mid y) = \frac{f_{Y|X}(y\mid x)\cdot f_X(x)}{f_Y(y)}$$
:::

## 例题：从联合密度求条件密度

:::example{label=三角形区域上的均匀分布}
设 $(X,Y)$ 在三角形区域 $D=\{(x,y)\mid x\geq 0,\ y\geq 0,\ x+y\leq 1\}$ 上服从均匀分布。求在 $X=x$（$0<x<1$）条件下，$Y$ 的条件密度 $f_{Y|X}(y\mid x)$。

**第一步：写出联合密度。**

三角形 $D$ 的面积为 $S = \frac{1}{2}\times 1\times 1 = \frac{1}{2}$，均匀分布的联合密度为

$$f(x,y) = \begin{cases}2, & (x,y)\in D,\\ 0, & \text{其他.}\end{cases}$$

**第二步：求 $X$ 的边缘密度。**

固定 $x\in(0,1)$，$y$ 的范围是 $0\leq y\leq 1-x$，因此

$$f_X(x) = \int_{-\infty}^{+\infty} f(x,y)\,\mathrm{d}y = \int_0^{1-x} 2\,\mathrm{d}y = 2(1-x), \quad 0<x<1$$

**第三步：求条件密度。**

$$f_{Y|X}(y\mid x) = \frac{f(x,y)}{f_X(x)} = \frac{2}{2(1-x)} = \frac{1}{1-x}, \quad 0<y<1-x$$

**解读：** 固定 $X=x$ 后，$Y$ 在 $(0, 1-x)$ 上服从**均匀分布** $U(0, 1-x)$。这很直观——在三角形中固定 $x$ 坐标，纵向截线长度是 $1-x$，条件分布就是该截线上的均匀分布。

**验证：**

$$\int_0^{1-x}\frac{1}{1-x}\,\mathrm{d}y = \frac{1}{1-x}\cdot(1-x) = 1 \quad \checkmark$$
:::

---

:::example{label=正态联合分布的条件密度}
设 $(X,Y)$ 的联合密度为

$$f(x,y) = \frac{1}{2\pi}e^{-\frac{x^2+y^2}{2}}, \quad (x,y)\in\mathbb{R}^2$$

（即二维标准正态，$X$ 与 $Y$ 独立）。求 $f_{X|Y}(x\mid y)$。

**第一步：** $Y$ 的边缘密度为

$$f_Y(y) = \int_{-\infty}^{+\infty}\frac{1}{2\pi}e^{-\frac{x^2+y^2}{2}}\,\mathrm{d}x = \frac{e^{-y^2/2}}{2\pi}\int_{-\infty}^{+\infty}e^{-x^2/2}\,\mathrm{d}x = \frac{e^{-y^2/2}}{2\pi}\cdot\sqrt{2\pi} = \frac{1}{\sqrt{2\pi}}e^{-y^2/2}$$

**第二步：** 条件密度为

$$f_{X|Y}(x\mid y) = \frac{f(x,y)}{f_Y(y)} = \frac{\frac{1}{2\pi}e^{-\frac{x^2+y^2}{2}}}{\frac{1}{\sqrt{2\pi}}e^{-y^2/2}} = \frac{1}{\sqrt{2\pi}}e^{-x^2/2}$$

**结论：** 条件密度就是 $X$ 的标准正态密度，与 $y$ 无关。这说明 $X$ 与 $Y$ 独立时，条件分布等于边缘分布——**已知 $Y=y$ 对 $X$ 的分布没有任何影响**。
:::

---

:::example{label=指数分布与条件密度}
设 $X\sim\text{Exp}(\lambda)$，$Y\sim\text{Exp}(\mu)$，且 $X$ 与 $Y$ 独立。记 $Z=\min(X,Y)$。已知 $Z=z$（$z>0$），求在此条件下 $X>z$ 的条件概率。

**分析：** $Z = \min(X,Y)$，联合密度为

$$f(x,y) = \lambda e^{-\lambda x}\cdot\mu e^{-\mu y}, \quad x>0,y>0$$

$Z$ 的密度：$f_Z(z)=(\lambda+\mu)e^{-(\lambda+\mu)z}$（独立指数分布取最小值仍为指数分布，参数为 $\lambda+\mu$）。

**用全概率分解：**

$$\{Z=z\} = \{X=z<Y\}\cup\{Y=z<X\}$$

（两个事件概率分别正比于 $\lambda$ 和 $\mu$）

$$P(X>z\mid Z=z) = P(Y\text{ 先坏}\mid Z=z) = \frac{\mu}{\lambda+\mu}$$

**直觉解释：** 在最短寿命 $z$ 处，"哪个先坏"的概率仅取决于各自的失效率之比，与具体 $z$ 无关——这正是指数分布无记忆性的体现。
:::

## 条件期望的引入

掌握了条件分布之后，我们可以自然地引入**条件期望**的概念——它是本章与第四章数字特征的桥梁。

:::definition{label=条件期望（初步）}
**离散型：** 在 $Y=y_j$ 的条件下，$X$ 的条件期望定义为

$$E(X\mid Y=y_j) = \sum_i x_i\cdot P(X=x_i\mid Y=y_j)$$

**连续型：** 在 $Y=y$ 的条件下，$X$ 的条件期望定义为

$$E(X\mid Y=y) = \int_{-\infty}^{+\infty} x\cdot f_{X|Y}(x\mid y)\,\mathrm{d}x$$

本质上，条件期望就是用**条件分布**代替普通分布，求出来的期望值。
:::

:::insight{label=条件期望是 y 的函数}
普通期望 $EX$ 是一个固定的数。但条件期望 $E(X\mid Y=y)$ 会随着 $y$ 的不同而取不同的值——它是 $y$ 的一个函数！

当我们把 $y$ 看成随机变量 $Y$ 时，$E(X\mid Y)$ 本身就是一个**随机变量**，称为"$X$ 关于 $Y$ 的条件期望"。

**重期望公式**（全期望公式）告诉我们：

$$E[E(X\mid Y)] = EX$$

即：先对 $Y$ 的每个可能值计算条件期望，再对 $Y$ 求期望，就还原了 $X$ 的期望。这是计算复杂期望的重要工具，第四章将详细展开。
:::

---

:::example{label=用条件期望计算候车时间}
某公交线路每 20 分钟一班。乘客到达时刻 $X$ 在 $[0,20]$ 上均匀分布，等候时间 $W$ 为到下一班车的时间。

**求 $E(W)$：**

直接法：$W = 20 - X$（若到达时刻为 $X$，则等 $20-X$ 分钟）。

$$E(W) = E(20-X) = 20 - E(X) = 20 - 10 = 10 \text{（分钟）}$$

这是最简洁的做法。如果 $W$ 的分布很复杂（比如来了两路车，到达时刻服从不同分布），条件期望就会特别有用——先固定某一路车的条件，计算条件期望，再加权平均。
:::

## 条件分布与独立性的关系

学过独立性之后，我们可以从条件分布的角度重新理解"独立"这个概念。

:::theorem{label=独立性的条件分布刻画}
**连续型：** $X$ 与 $Y$ 相互独立 $\iff$ 对几乎所有的 $y$，有

$$f_{X|Y}(x\mid y) = f_X(x)$$

即：$Y$ 的取值对 $X$ 的条件分布没有任何影响；条件密度就等于边缘密度。

**直觉：** 独立意味着"已知 $Y=y$ 不会改变 $X$ 的分布"，这正是条件分布与边缘分布相等的数学表述。
:::

:::derivation{label=从条件分布证明联合密度的分解}
若 $f_{X|Y}(x\mid y)=f_X(x)$ 对所有 $y$ 成立，则由乘法公式：

$$f(x,y) = f_{X|Y}(x\mid y)\cdot f_Y(y) = f_X(x)\cdot f_Y(y)$$

这正是独立性的等价条件：**联合密度等于两个边缘密度的乘积**。
:::

## 易错点

:::pitfall{label=三个常见错误}
**错误一：把条件密度当作联合密度使用。**

$f_{X|Y}(x\mid y)$ 只有固定 $y$ 后，对 $x$ 积分才为 1。若对 $x$ 和 $y$ 同时积分，结果不是 1，不可当做二维密度使用。

---

**错误二：忘记确认 $f_Y(y)>0$（或 $p_{\cdot j}>0$）。**

条件分布只有在 $Y=y$ 这个事件有正概率（或正密度）时才有意义。若 $f_Y(y)=0$，则 $f_{X|Y}(x\mid y)$ 无定义，不能硬算。

---

**错误三：混淆 $f_{X|Y}(x\mid y)$ 与 $f_{Y|X}(y\mid x)$。**

$$f_{X|Y}(x\mid y) = \frac{f(x,y)}{f_Y(y)} \neq \frac{f(x,y)}{f_X(x)} = f_{Y|X}(y\mid x)$$

条件方向不同，结果完全不同。类比：$P(A\mid B)\neq P(B\mid A)$，这是最常见的混淆，也是贝叶斯公式能够发挥作用的根本原因。

**错误四：条件期望是常数而非函数。**

$E(X\mid Y=y_0)$ 是一个常数（固定 $y_0$），但 $E(X\mid Y)$ 是 $Y$ 的函数，是随机变量。两者层次不同，书写时注意区分。
:::

## 综合例题：从条件已知还原联合

:::example{label=已知条件密度和边缘密度，求联合密度}
已知在 $X=x$（$0<x<1$）的条件下，$Y$ 的条件密度为

$$f_{Y|X}(y\mid x) = \frac{3y^2}{x^3}, \quad 0 < y < x$$

又知 $X$ 的边缘密度为 $f_X(x) = 4x^3$，$0<x<1$。

**求联合密度 $f(x,y)$，并求 $P(Y < 1/2)$。**

**第一步：由乘法公式求联合密度。**

$$f(x,y) = f_{Y|X}(y\mid x)\cdot f_X(x) = \frac{3y^2}{x^3}\cdot 4x^3 = 12y^2, \quad 0<y<x<1$$

（注意支撑域：$0<y<x<1$，即三角形区域，$y$ 在 $x$ 下方。）

**第二步：求 $P(Y<1/2)$。**

先求 $Y$ 的边缘密度：

$$f_Y(y) = \int_{y}^{1} 12y^2\,\mathrm{d}x = 12y^2(1-y), \quad 0<y<1$$

（积分下限是 $x=y$，因为要求 $y<x$；上限是 $x=1$。）

再求概率：

$$\begin{aligned}
P\!\left(Y < \tfrac{1}{2}\right) &= \int_0^{1/2} 12y^2(1-y)\,\mathrm{d}y\\
&= 12\int_0^{1/2}\left(y^2 - y^3\right)\mathrm{d}y\\
&= 12\left[\frac{y^3}{3} - \frac{y^4}{4}\right]_0^{1/2}\\
&= 12\left(\frac{1}{24} - \frac{1}{64}\right)\\
&= 12\cdot\frac{8-3}{192} = 12\cdot\frac{5}{192} = \frac{5}{16}
\end{aligned}$$

**解题逻辑总结：** 这道题展示了"由条件密度和边缘密度恢复联合密度"的过程。只要掌握乘法公式，这类题无论怎么变形都可以化解。
:::

## 综合例题：条件密度的计算与分析

:::example{label=二维正态的条件分布}
设 $(X,Y)$ 服从二维正态分布，参数为 $\mu_1,\mu_2,\sigma_1^2,\sigma_2^2,\rho$，联合密度为

$$f(x,y) = \frac{1}{2\pi\sigma_1\sigma_2\sqrt{1-\rho^2}}\exp\!\left\{-\frac{1}{2(1-\rho^2)}\left[\frac{(x-\mu_1)^2}{\sigma_1^2} - 2\rho\frac{(x-\mu_1)(y-\mu_2)}{\sigma_1\sigma_2} + \frac{(y-\mu_2)^2}{\sigma_2^2}\right]\right\}$$

已知 $X$ 的边缘分布 $X\sim N(\mu_1,\sigma_1^2)$，$Y$ 的边缘分布 $Y\sim N(\mu_2,\sigma_2^2)$。

**求在 $Y=y$ 条件下 $X$ 的条件密度，并说明其分布类型。**

**过程概述：**

$$f_{X|Y}(x\mid y) = \frac{f(x,y)}{f_Y(y)}$$

对 $x$ 进行配方（这里 $y$ 是固定的常数），可以验证结果仍然是正态分布：

$$X\mid Y=y \sim N\!\left(\mu_1+\rho\frac{\sigma_1}{\sigma_2}(y-\mu_2),\ \sigma_1^2(1-\rho^2)\right)$$

**核心结论：**

1. 条件均值 $E(X\mid Y=y) = \mu_1 + \rho\dfrac{\sigma_1}{\sigma_2}(y-\mu_2)$ 是 $y$ 的线性函数。
2. 条件方差 $\text{Var}(X\mid Y=y) = \sigma_1^2(1-\rho^2)$ 与 $y$ 无关。
3. 当 $\rho=0$ 时，条件均值等于 $\mu_1$，条件分布等于边缘分布——这正是 $X$ 与 $Y$ 独立的表现。
4. 当 $|\rho|$ 越大，条件分布的方差越小，说明"已知 $Y=y$"给出的信息越多，$X$ 的不确定性越小。

这是二维正态分布的一个极为优美的性质：**二维正态的条件分布仍然是正态**，且参数由第五个参数 $\rho$ 精确刻画。
:::

## 本节小结

:::note{label=本节小结}
**核心公式：**

- 离散型：$P(X=x_i\mid Y=y_j) = \dfrac{p_{ij}}{p_{\cdot j}}$（联合概率除以边缘概率）
- 连续型：$f_{X|Y}(x\mid y) = \dfrac{f(x,y)}{f_Y(y)}$（联合密度除以边缘密度）

**几何直觉：** 条件分布是联合分布在某条直线上的"截面+归一化"。

**三种分布的关系：**
$$f(x,y) = f_{X|Y}(x\mid y)\cdot f_Y(y) = f_{Y|X}(y\mid x)\cdot f_X(x)$$

**条件分布与独立性：** $X,Y$ 独立 $\iff$ 条件密度等于边缘密度 $f_{X|Y}(x\mid y)=f_X(x)$。

**条件期望引入：**
$$E(X\mid Y=y) = \int_{-\infty}^{+\infty} x\,f_{X|Y}(x\mid y)\,\mathrm{d}x$$

它是 $y$ 的函数；固定 $y$ 就是一个数，把 $y$ 换成随机变量 $Y$ 就是随机变量。

**下一节展望：** 3.4 节将研究二维随机变量的函数 $Z=g(X,Y)$ 的分布，包括卷积公式（求 $X+Y$ 的密度）、商的分布等，是本章最具挑战性的内容。掌握了条件密度之后，再看函数分布，你会发现它们在本质上都是"在某个约束条件下的积分"，思路完全相通。
:::
