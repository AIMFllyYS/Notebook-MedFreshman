## 一、选择题（每题2分，共24分）

### 第1题

:::callout{kind=note label="题目"}
下列原子杂化判断正确的是：乙腈 CH₃C≡N 中 C≡N 的碳和氮分别为

A. sp³、sp²    B. sp、sp    C. sp²、sp    D. sp、sp²
:::

:::callout{kind=insight label="解析"}
**【答案】B**

叁键两端原子为直线型，均采取sp杂化。

:::

:::callout{kind=note label="杂化轨道判断"}
**sp杂化**：直线型，键角180°，如炔烃、腈类
**sp²杂化**：平面型，键角120°，如烯烃、芳烃
**sp³杂化**：四面体，键角109.5°，如烷烃

**乙腈结构**：
- CH₃- 中的碳：sp³杂化
- -C≡N 中的碳：sp杂化
- 氮原子：sp杂化（直线型）
:::


### 第2题

:::callout{kind=note label="题目"}
下列化合物沸点最高的是

A. 邻硝基苯酚    B. 对硝基苯酚    C. 苯甲醚    D. 甲苯
:::

:::callout{kind=insight label="解析"}
**【答案】B**

对硝基苯酚能形成分子间氢键，沸点显著升高；邻硝基苯酚以分子内氢键为主。

:::

:::callout{kind=insight label="氢键与沸点"}
**分子间氢键**：提高沸点（对硝基苯酚）
**分子内氢键**：降低沸点（邻硝基苯酚）

**沸点排序**：
对硝基苯酚 > 邻硝基苯酚 > 苯甲醚 > 甲苯

**原因**：
- 对硝基苯酚：分子间氢键，二聚体
- 邻硝基苯酚：分子内氢键（六元环），不形成分子间氢键
- 苯甲醚：偶极-偶极作用
- 甲苯：仅范德华力
:::


### 第3题

:::callout{kind=note label="题目"}
反-1,2-二甲基环己烷最稳定构象中两个甲基的位置是

A. 均为a键    B. 均为e键    C. 一个a键一个e键    D. 构象能量完全相同
:::

:::callout{kind=insight label="解析"}
**【答案】B**

邻位反式二取代可同时为a,a或e,e；大基团均为e键时位阻最小。

:::

:::callout{kind=note label="环己烷构象分析"}
**反-1,2-二取代**：
- 两个取代基可同时处于e,e或a,a
- e,e构象更稳定（1,3-二轴相互作用小）

**顺-1,2-二取代**：
- 必须一a一e
- 稳定性介于反式的e,e和a,a之间

**规律**：大基团优先占据e键
:::


### 第4题

:::callout{kind=note label="题目"}
2-甲基丁烷在Br₂/hν条件下单溴代的主产物为

A. 1-溴-2-甲基丁烷    B. 2-溴-2-甲基丁烷    C. 1-溴-3-甲基丁烷    D. 2-溴-3-甲基丁烷
:::

:::callout{kind=insight label="解析"}
**【答案】B**

溴代自由基取代选择性高，优先取代三级氢，生成更稳定三级自由基来源的产物。

:::

:::callout{kind=insight label="自由基取代选择性"}
**氢原子活性**：3° > 2° > 1° > CH₃

**2-甲基丁烷结构**：
```
    CH₃
    |
CH₃-C-CH₂-CH₃
    |
    H
```

**氢原子类型**：
- 1个3°H（C2上）
- 2个2°H（C3上）
- 9个1°H（甲基上）

**溴代选择性**：Br·优先夺取3°H，生成3°自由基（最稳定）

**产物**：2-溴-2-甲基丁烷
:::


### 第5题

:::callout{kind=note label="题目"}
下列卤素取代基的吸电子诱导效应最强的是

A. -F    B. -Cl    C. -Br    D. -I
:::

:::callout{kind=insight label="解析"}
**【答案】A**

同族自上而下电负性降低，-I效应：F > Cl > Br > I。

:::

:::callout{kind=note label="诱导效应与电负性"}
**电负性顺序**：F(4.0) > Cl(3.0) > Br(2.8) > I(2.5)

**-I效应（吸电子诱导）**：
-F > -Cl > -Br > -I

**+I效应（给电子诱导）**：
-I > -Br > -Cl > -F（与电负性相反）

**注意**：
- 诱导效应：F最强
- 共轭效应（+C）：F也最强（2p-2p重叠最好）
- 但总效应：F是致钝基团（-I > +C）
:::


### 第6题

:::callout{kind=note label="题目"}
1-戊烯与HBr在过氧化物存在下反应的主要产物是

A. 1-溴戊烷    B. 2-溴戊烷    C. 1,2-二溴戊烷    D. 戊醇
:::

:::callout{kind=insight label="解析"}
**【答案】A**

HBr自由基加成给反马氏规则产物，Br最终连到端基碳。

:::

:::callout{kind=insight label="反马氏加成机理"}
**Kharasch反应**：
1. ROOR → 2RO·（自由基引发）
2. RO· + HBr → ROH + Br·
3. Br· + CH₂=CH-R → ·CH₂-CHBr-R（2°自由基，更稳定）
4. ·CH₂-CHBr-R + HBr → CH₃-CHBr-R + Br·

**产物**：1-溴戊烷（反马氏）

**注意**：仅HBr有此反应，HCl和HI不行！
:::


### 第7题

:::callout{kind=note label="题目"}
烯烃在冷、稀、碱性KMnO₄条件下通常生成

A. 反式邻二醇    B. 顺式邻二醇    C. 羧酸裂解产物    D. 环氧化物
:::

:::callout{kind=insight label="解析"}
**【答案】B**

弱氧化条件下高锰酸钾顺式羟基化，生成顺式邻二醇。

:::

:::callout{kind=note label="烯烃氧化条件对比"}
| 条件 | 产物 | 立体化学 |
|------|------|----------|
| 冷稀碱性KMnO₄ | 邻二醇 | 顺式 |
| 酸性KMnO₄（加热） | 羧酸/酮 | 双键断裂 |
| OsO₄ | 邻二醇 | 顺式（更专一） |
| O₃ + Zn/H₂O | 醛/酮 | 双键断裂 |
| 过氧酸 | 环氧化物 | 顺式 |

**机理**：MnO₄⁻与双键形成环状锰酸酯，顺式加成
:::


### 第8题

:::callout{kind=note label="题目"}
鉴别末端炔烃最典型的试剂是

A. Lucas试剂    B. FeCl₃水溶液    C. 硝酸银氨溶液    D. NaHCO₃溶液
:::

:::callout{kind=insight label="解析"}
**【答案】C**

末端炔氢具有弱酸性，可与银氨溶液形成炔银沉淀。

:::

:::callout{kind=note label="末端炔烃的鉴定"}
**反应**：R-C≡CH + [Ag(NH₃)₂]⁺ → R-C≡CAg↓ + NH₄⁺

**现象**：白色沉淀（炔银）

**其他试剂**：
- Cu₂Cl₂/NH₃·H₂O：红色沉淀（炔铜）
- 非末端炔无此反应

**应用**：区分末端炔和非末端炔
:::


### 第9题

:::callout{kind=note label="题目"}
下列芳香亲电取代反应活性由强到弱正确的是

A. 硝基苯 > 苯 > 甲苯 > 苯甲醚    B. 苯甲醚 > 甲苯 > 苯 > 硝基苯    C. 苯 > 甲苯 > 苯甲醚 > 硝基苯    D. 甲苯 > 苯甲醚 > 苯 > 硝基苯
:::

:::callout{kind=insight label="解析"}
**【答案】B**

-OCH₃强给电子活化，-CH₃弱活化，-NO₂强吸电子钝化。

:::

:::callout{kind=insight label="芳香亲电取代活性"}
**活性顺序**：
强致活 > 弱致活 > 苯 > 致钝

**具体排序**：
-OCH₃（强致活）> -CH₃（弱致活）> H > -NO₂（强致钝）

**原因**：
- -OCH₃：p-π共轭给电子，大幅活化
- -CH₃：超共轭给电子，弱活化
- -NO₂：π-π共轭吸电子，强钝化
:::


### 第10题

:::callout{kind=note label="题目"}
氯苯硝化时，主要得到邻、对位产物，其原因是Cl表现为

A. 强活化邻对位定位基    B. 致钝邻对位定位基    C. 强吸电子间位定位基    D. 无定位作用
:::

:::callout{kind=insight label="解析"}
**【答案】B**

卤素总效应吸电子致钝，但孤对电子p-π共轭使其定位为邻对位。

:::

:::callout{kind=note label="卤素的特殊定位效应"}
**矛盾现象**：
- 诱导效应（-I）：吸电子，致钝
- 共轭效应（+C）：给电子，邻对位定位

**结果**：
- 总体：致钝（反应比苯慢）
- 定位：邻对位（+C效应主导定位）

**对比其他基团**：
- -OH：+C > -I，强致活，邻对位
- -NO₂：-I和-C都吸电子，致钝，间位
:::


### 第11题

:::callout{kind=note label="题目"}
苯与1-氯丙烷/AlCl₃进行傅-克烷基化，主要产物通常是

A. 正丙苯    B. 异丙苯    C. 氯苯    D. 苯乙烯
:::

:::callout{kind=insight label="解析"}
**【答案】B**

一级丙基碳正离子会发生负氢迁移重排为更稳定的二级碳正离子。

:::

:::callout{kind=insight label="碳正离子重排"}
**反应机理**：
1. CH₃CH₂CH₂Cl + AlCl₃ → CH₃CH₂CH₂⁺ + AlCl₄⁻
2. CH₃CH₂CH₂⁺ → CH₃CH⁺CH₃（负氢迁移，1°→2°）
3. CH₃CH⁺CH₃ + C₆H₆ → 异丙苯

**重排驱动力**：生成更稳定的碳正离子

**避免重排**：使用傅-克酰基化（酰基正离子不重排）
:::


### 第12题

:::callout{kind=note label="题目"}
关于内消旋体，说法正确的是

A. 一定是外消旋混合物    B. 一定无手性碳    C. 由于分子内部对称而无旋光性    D. 一定有一对对映异构体等量混合
:::

:::callout{kind=insight label="解析"}
**【答案】C**

内消旋体是单一化合物，含手性中心但分子有内部对称因素，整体非手性。

| 旋光性 | 无 | 无（抵消） |

**示例**：内消旋酒石酸（meso-tartaric acid）

## 二、填空题（每空1分，共8分）

### 第1题

:::callout{kind=note label="题目"}
Hückel芳香性规则要求环状共轭体系的π电子数满足 __________。
:::

:::callout{kind=insight label="解析"}
**答案**：4n+2

**解析**：Hückel规则：环状、平面、共轭体系，π电子数为4n+2（n=0,1,2...）时具有芳香性。

:::

:::callout{kind=note label="Hückel芳香性规则"}
**条件**：
1. 环状结构
2. 平面或近平面
3. 完全共轭（每个原子有p轨道）
4. π电子数 = 4n+2

**示例**：
- 苯：6π电子（n=1），芳香性
- 环戊二烯负离子：6π电子，芳香性
- 环庚三烯正离子：6π电子，芳香性
- 环丁二烯：4π电子，反芳香性
:::


### 第2题

:::callout{kind=note label="题目"}
不对称烯烃与HX按经典亲电加成时，H通常加到含H较 __________ 的双键碳上。
:::

:::callout{kind=insight label="解析"}
**答案**：多

**解析**：马氏规则：H加在含氢较多的碳上，X加在含氢较少的碳上。

:::

:::callout{kind=note label="马氏规则"}
**马氏规则**：亲电加成中，H加到含H较多的碳上

**原因**：生成更稳定的碳正离子中间体

**示例**：
CH₃CH=CH₂ + HBr → CH₃CHBrCH₃（2-溴丙烷）
- 若H加在C1：生成2°碳正离子（稳定）
- 若H加在C2：生成1°碳正离子（不稳定）
:::


### 第3题

:::callout{kind=note label="题目"}
Br₂/CCl₄与烯烃加成通常先形成 __________ 离子，最终为 __________ 加成。
:::

:::callout{kind=insight label="解析"}
**答案**：环溴鎓；反式

**解析**：Br₂与烯烃形成环溴鎓离子中间体，Br⁻从背面进攻，最终为反式加成。

:::

:::callout{kind=note label="环溴鎓离子机理"}
**机理**：
1. Br₂接近双键，极化
2. Br⁺进攻，形成三元环溴鎓离子
3. Br⁻从背面进攻，开环
4. 结果：反式加成

**立体化学**：
- 顺式烯烃 → 外消旋产物
- 反式烯烃 → 内消旋产物
:::


### 第4题

:::callout{kind=note label="题目"}
Lindlar催化剂使炔烃选择性加氢为 __________ 烯烃；Na/NH₃(l)还原则得到 __________ 烯烃。
:::

:::callout{kind=insight label="解析"}
**答案**：顺式；反式

**解析**：Lindlar催化剂（Pd/CaCO₃，毒化）使炔烃部分加氢为顺式烯烃；Na/NH₃(l)还原使炔烃转为反式烯烃。

:::

:::callout{kind=insight label="炔烃选择性还原"}
| 试剂 | 产物 | 机理 |
|------|------|------|
| H₂/Lindlar | 顺式烯烃 | 顺式加成（表面催化） |
| H₂/Pd | 烷烃 | 完全还原 |
| Na/NH₃(l) | 反式烯烃 | 电子转移，自由基负离子 |
| (sia)₂BH | 醛 | 硼氢化-氧化 |

**Lindlar催化剂**：Pd/CaCO₃，用Pb(OAc)₂或喹啉毒化，降低活性
:::

C₆H₁₀的不饱和度为 __________。

### 第6题

:::callout{kind=note label="题目"}
对映异构体在非手性环境中熔点、沸点、溶解度通常 __________。
:::

:::callout{kind=insight label="解析"}
**答案**：相同

**解析**：对映异构体在非手性环境中物理性质相同（熔点、沸点、溶解度、密度等），仅旋光方向相反。

:::

:::callout{kind=note label="对映异构体的性质"}
**非手性环境中**：
- 熔点、沸点、溶解度：相同
- 旋光度：大小相等，方向相反

**手性环境中**：
- 与手性试剂反应：速率不同
- 在手性溶剂中：溶解度不同
- 在手性催化剂存在下：反应选择性不同
:::


### 第7题

:::callout{kind=note label="题目"}
苯环上-NO₂是 __________ 定位基；-OH是 __________ 定位基。
:::

:::callout{kind=insight label="解析"}
**答案**：间位；邻对位

**解析**：-NO₂强吸电子，间位定位，致钝；-OH强给电子，邻对位定位，致活。

:::

:::callout{kind=note label="定位基总结"}
**邻对位定位基（致活）**：
-OH, -OR, -NH₂, -NHR, -NR₂, -R

**间位定位基（致钝）**：
-NO₂, -CN, -COOH, -CHO, -COR, -SO₃H

**特殊（邻对位定位但致钝）**：
-Cl, -Br, -I, -F
:::


### 第8题

:::callout{kind=note label="题目"}
臭氧化还原处理可用于由产物倒推烯烃的 __________ 位置。
:::

:::callout{kind=insight label="解析"}
**答案**：双键

**解析**：臭氧化断裂双键，每个双键碳生成一个羰基。由羰基产物可推断原双键位置。

:::

:::callout{kind=note label="臭氧化反应的应用"}
**正向应用**：烯烃 → 醛/酮（合成）

**逆向应用**：
- 由醛/酮推断烯烃结构
- 将产物中的C=O还原为C=C
- 两个羰基碳原来是双键的两个碳

**示例**：
产物：CH₃CHO + CH₃COCH₃
→ 原烯烃：CH₃CH=C(CH₃)₂（2-甲基-2-丁烯）
:::

## 三、判断题（每题1分，共8分）

### 第1题

:::callout{kind=note label="题目"}
环己烷椅式构象通常比船式构象稳定。（　　）
:::

:::callout{kind=insight label="解析"}
**【答案】√**

解析：椅式构象键角和扭转张力小，取代基拥挤少。

:::

:::callout{kind=note label="环己烷构象能量"}
**椅式构象**：
- 键角接近109.5°
- 无角张力、无扭转张力
- 最稳定构象

**船式构象**：
- 存在旗杆氢相互作用
- 扭转张力
- 能量比椅式高~30 kJ/mol

**半椅式构象**：
- 能量最高
- 过渡态构象
:::


### 第2题

:::callout{kind=note label="题目"}
顺-1,4-二甲基环己烷可以同时让两个甲基均处于e键。（　　）
:::

:::callout{kind=insight label="解析"}
**【答案】×**

解析：1,4-顺式必为一a一e；反式才可e,e。

:::

:::callout{kind=note label="1,4-二取代环己烷"}
**顺式**：
- 两个取代基同向（都朝上或都朝下）
- 必须一a一e

**反式**：
- 两个取代基反向
- 可同时a,a或同时e,e
- e,e更稳定

**最稳定构象**：反-1,4-二取代的e,e构象
:::

**无手性碳但有手性**：

### 第4题

:::callout{kind=note label="题目"}
烯烃与HBr在过氧化物存在下的反马氏加成只适用于HBr，不能简单推广到HCl或HI。（　　）
:::

:::callout{kind=insight label="解析"}
**【答案】√**

解析：自由基链反应热力学只适合HBr。

:::

:::callout{kind=insight label="反马氏加成的限制"}
**仅HBr有效的原因**：

**HCl**：
- H-Cl键太强（432 kJ/mol）
- Cl·夺取H困难

**HI**：
- H-I键太弱（297 kJ/mol）
- I·是弱自由基，不活泼
- I₂反而能捕获自由基

**HBr**：
- H-Br键强度适中（366 kJ/mol）
- Br·活性适中，可选择性进攻
:::

## 五、鉴别/排序题（每题5分，共10分）

### 第1题

:::callout{kind=note label="题目"}
鉴别题：用简单化学方法鉴别：环己烷、环己烯、苯乙炔。写出试剂、现象和判断顺序。
:::

:::callout{kind=insight label="解析"}
**答案**：先加Br₂/CCl₄：不褪色者为环己烷；褪色者为环己烯或苯乙炔。再对褪色样品加硝酸银氨溶液：生成白色/浅色沉淀者为苯乙炔；无沉淀者为环己烯。

**解析**：烯烃和炔烃均能与溴加成使溴褪色；末端炔还可生成炔银沉淀。

:::

:::callout{kind=note label="鉴别策略"}
**第一步：Br₂/CCl₄**
- 褪色：含不饱和键（烯烃、炔烃）
- 不褪色：饱和烃（烷烃、环烷烃）

**第二步：银氨溶液**
- 白色沉淀：末端炔烃
- 无沉淀：烯烃或非末端炔

**第三步（如需进一步区分）**：
- KMnO₄/H⁺：烯烃氧化（褪色），炔烃也氧化
- 臭氧化：由产物推断结构
:::


### 第2题

:::callout{kind=note label="题目"}
排序题：按酸性由强到弱排序：对硝基苯酚、苯酚、对甲苯酚、乙醇、碳酸。
:::

:::callout{kind=insight label="解析"}
**答案**：碳酸 > 对硝基苯酚 > 苯酚 > 对甲苯酚 > 乙醇

**解析**：羧酸/碳酸强于酚；-NO₂吸电子稳定酚氧负离子，-CH₃给电子削弱酸性；醇酸性弱于酚。

:::

:::callout{kind=insight label="酸性排序依据"}
**pKa值**：
- 碳酸：6.4
- 对硝基苯酚：7.15
- 苯酚：10
- 对甲苯酚：10.3
- 乙醇：16

**影响因素**：
- 吸电子基团（-NO₂）：稳定共轭碱，增强酸性
- 给电子基团（-CH₃）： destabilize共轭碱，减弱酸性
:::

## 六、合成题（8分）

以苯为原料合成正丙苯，要求避免傅-克烷基化重排。写出路线。

:::callout{kind=insight label="解析"}
**路线**：

苯 --(CH₃CH₂COCl, AlCl₃)--> 苯丙酮（PhCOCH₂CH₃） --(Zn-Hg/HCl 或 NH₂NH₂/KOH, Δ)--> 正丙苯。

**解析**：直接用1-氯丙烷傅-克烷基化会重排成异丙苯；先傅-克酰基化生成不重排的酰基苯，再把C=O还原为CH₂。

:::

:::callout{kind=insight label="Clemmensen还原 vs Wolff-Kishner还原"}
**Clemmensen还原**：
- 试剂：Zn-Hg/HCl
- 条件：酸性，加热
- 适用范围：对酸稳定的酮

**Wolff-Kishner还原**：
- 试剂：NH₂NH₂/KOH，加热
- 条件：碱性
- 适用范围：对碱稳定的酮

**选择依据**：
- 分子含酸敏感基团 → Wolff-Kishner
- 分子含碱敏感基团 → Clemmensen
:::

## 七、推断题（10分）

A、B、C均为C₆H₁₀，均能使Br₂/CCl₄褪色。A与硝酸银氨溶液生成沉淀，热KMnO₄氧化得戊酸和CO₂；B不与银氨溶液反应，热KMnO₄氧化得乙酸和丁酸；C不与银氨溶液反应，热KMnO₄氧化得己二酸。推断A、B、C结构。

:::callout{kind=insight label="解析"}
**A = 1-己炔**（HC≡C(CH₂)₃CH₃）

**B = 2-己炔**（CH₃C≡CCH₂CH₂CH₃）

**C = 环己烯**

**解析**：

C₆H₁₀不饱和度2。A能生成炔银，说明为末端炔；末端炔热氧化给RCOOH + CO₂，R为戊基片段，故A为1-己炔。B氧化裂解为乙酸和丁酸，对应CH₃-C≡C-CH₂CH₂CH₃，即2-己炔。C氧化为己二酸是环己烯氧化开环的典型产物。

:::

:::callout{kind=insight label="推断题分析策略"}
**线索分析**：
1. C₆H₁₀，不饱和度2 → 炔烃、二烯烃或环烯烃
2. 能使Br₂褪色 → 含不饱和键
3. 银氨溶液反应 → 区分末端炔和非末端炔/烯烃

**氧化产物推断**：
- 末端炔 → RCOOH + CO₂
- 非末端炔 → 两种羧酸
- 环己烯 → 己二酸（二元羧酸）

**结构验证**：
- 1-己炔：HC≡C-CH₂-CH₂-CH₂-CH₃
  - 氧化：HOOC-(CH₂)₃-CH₃（戊酸）+ CO₂
- 2-己炔：CH₃-C≡C-CH₂-CH₂-CH₃
  - 氧化：CH₃COOH（乙酸）+ HOOC-CH₂-CH₂-CH₃（丁酸）
- 环己烯：氧化开环 → HOOC-(CH₂)₄-COOH（己二酸）
:::

9. **推断题**：氧化产物与结构的对应关系
