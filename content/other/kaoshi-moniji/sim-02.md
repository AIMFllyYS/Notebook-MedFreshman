## 一、选择题（每小题2分，共20分）

### 第1题

:::callout{kind=note label="题目"}
下列化合物中能形成分子间氢键的是（　　）

A. CH₃OCH₃　　B. CH₃CHO　　C. CH₃COOH　　D. CH₃COCH₃
:::

:::callout{kind=insight label="解析"}
**【答案】C**

能形成分子间氢键需要有既能作为氢键给体又能作为受体的官能团。CH₃COOH含有-OH（给体）和C=O（受体），可以形成强的分子间氢键（甚至形成二聚体）。CH₃OCH₃中的氧可以做受体但无活泼氢做给体。CH₃CHO中的C=O只能做受体。

:::

:::callout{kind=note label="氢键形成条件"}
**氢键给体**：含活泼氢（-OH、-NH、-COOH）
**氢键受体**：含孤对电子（O、N、F）

**常见可形成分子间氢键的化合物**：
- 羧酸（最强，可形成二聚体）
- 醇、酚
- 胺、酰胺
- 水

**不能形成分子间氢键**：醚、醛、酮（只有受体无给体）
:::


### 第2题

:::callout{kind=note label="题目"}
关于环己烷椅式构象的叙述，正确的是（　　）

A. 大基团处于a键（直立键）更稳定　　B. 大基团处于e键（平伏键）更稳定

C. a键和e键能量相同　　D. 环己烷只有一种椅式构象
:::

:::callout{kind=insight label="解析"}
**【答案】B**

环己烷椅式构象中，大基团处于e键（平伏键/赤道键）时，与邻位碳上的a键基团距离较远，1,3-二轴相互作用小，能量更低，构象更稳定。这是课堂重点讲解的内容，室温时椅式构象占99.9%。

:::

:::callout{kind=insight label="环己烷椅式构象"}
**a键（直立键）**：垂直于环平面，上下交替
**e键（平伏键）**：大致平行于环平面，向外伸展

**稳定性规律**：
- 大基团优先占据e键（1,3-二轴相互作用小）
- 反-1,2-二取代：两个取代基可同时处于e,e或a,a
- 顺-1,2-二取代：必须一a一e
- 反-1,4-二取代：可同时e,e（最稳定）

**构象翻转**：椅式翻转时，a键变e键，e键变a键
:::


### 第3题

:::callout{kind=note label="题目"}
下列化合物沸点最高的是（　　）

A. 正戊烷　　B. 异戊烷　　C. 新戊烷　　D. 三者沸点相同
:::

:::callout{kind=insight label="解析"}
**【答案】A**

正异新三种戊烷分子量相同，但沸点不同。正戊烷为直链，分子接触面最大，分子间范德华作用力最强，沸点最高。支链越多，分子越接近球形，接触面越小，沸点越低。

:::

:::callout{kind=note label="沸点与分子结构"}
**沸点影响因素**：
1. **分子间作用力**：氢键 > 偶极-偶极 > 范德华力
2. **分子接触面积**：直链 > 支链
3. **分子量**：越大沸点越高

**正戊烷异构体沸点**：正戊烷(36°C) > 异戊烷(28°C) > 新戊烷(9.5°C)

**注意**：熔点还受对称性影响（新戊烷熔点最高）
:::


### 第4题

:::callout{kind=note label="题目"}
甲苯苯环上进行亲电取代反应时，甲基是（　　）

A. 间位定位基，致活基团　　B. 邻、对位定位基，致钝基团

C. 邻、对位定位基，致活基团　　D. 间位定位基，致钝基团
:::

:::callout{kind=insight label="解析"}
**【答案】C**

甲基为给电子基团（超共轭效应），使苯环电子云密度增加（致活基团），且电子云密度增加主要体现在邻位和对位，因此甲基是邻、对位定位基，是致活基团。

:::

:::callout{kind=insight label="定位基分类"}
**致活基团（邻对位定位）**：
- 强致活：-NH₂、-OH、-OR、-NHR
- 中等致活：-R（烷基）、-C₆H₅
- 弱致活：-CH₂Cl

**致钝基团**：
- 间位定位：-NO₂、-CN、-COOH、-CHO、-COR、-SO₃H
- 邻对位定位（但致钝）：-Cl、-Br、-I、-F

**甲基的特殊性**：超共轭效应给电子，使邻对位电子云密度增加最多
:::


### 第5题

:::callout{kind=note label="题目"}
下列化合物中，具有手性的是（　　）

A. 2-丁醇　　B. 2-丙醇　　C. 正丁醇　　D. 叔丁醇
:::

:::callout{kind=insight label="解析"}
**【答案】A**

2-丁醇(CH₃CH(OH)CH₂CH₃)的2号碳连有四个不同基团(-CH₃, -OH, -H, -C₂H₅)，是手性碳原子，含一个手性碳的分子一定有手性。其他三个醇都没有手性碳原子。

:::

:::callout{kind=note label="手性判断"}
**手性碳**：连有四个不同基团的碳原子（不对称碳）

**分子手性的判断**：
- 含一个手性碳 → 一定有手性
- 含多个手性碳 → 可能有内消旋体（有对称面）
- 无手性碳 → 可能仍有手性（如丙二烯型、联苯型）

**2-丁醇的手性**：
C2连有：-CH₃、-OH、-H、-CH₂CH₃（四个不同基团）→ 手性碳

\>CC(O)C\<\/SvgDiagram\>

### 第6题

:::callout{kind=note label="题目"}
苯与溴在三溴化铁催化下反应生成的主要产物是（　　）

A. 苯溴化物（加成产物）　　B. 溴苯（取代产物）

C. 1,2-二溴环己烷　　D. 不反应
:::

:::callout{kind=insight label="解析"}
**【答案】B**

苯在FeBr₃（路易斯酸）催化下与溴发生亲电取代反应，生成溴苯。苯环的芳香性使其优先发生取代反应而非加成反应，以维持芳香环的稳定大π键体系。

:::

:::callout{kind=note label="苯的芳香性"}
**芳香性特点**：
- 难加成（破坏芳香性，需高能量）
- 易亲电取代（保持芳香性）
- 稳定（共振能约150 kJ/mol）

**苯与溴的反应**：
- FeBr₃催化：亲电取代 → 溴苯
- 光照/加热：加成（需特殊条件，如紫外线）

**对比**：环己烯 + Br₂ → 加成产物（1,2-二溴环己烷）
:::


### 第7题

:::callout{kind=note label="题目"}
下列醇脱水生成烯烃的反应活性顺序正确的是（　　）

A. 伯醇 > 仲醇 > 叔醇　　B. 叔醇 > 仲醇 > 伯醇

C. 仲醇 > 伯醇 > 叔醇　　D. 伯醇 > 叔醇 > 仲醇
:::

:::callout{kind=insight label="解析"}
**【答案】B**

醇的脱水反应是消除反应（E1），经历碳正离子中间体。叔醇最容易形成稳定的三级碳正离子，脱水最容易；伯醇难以形成稳定的碳正离子，脱水最困难。

:::

:::callout{kind=insight label="醇脱水反应机理"}
**反应机理**：E1（单分子消除）

**活性顺序**：叔醇 > 仲醇 > 伯醇

**原因**：碳正离子稳定性 3° > 2° > 1°

**反应条件**：
- 酸催化（H₂SO₄、H₃PO₄）
- 加热
- 可能伴随重排

**Saytzeff规则**：主要产物为双键上取代基最多的烯烃（更稳定）
:::


### 第8题

:::callout{kind=note label="题目"}
下列化合物酸性最强的是（　　）

A. C₂H₅OH（乙醇）　　B. C₆H₅OH（苯酚）　　C. H₂CO₃（碳酸）　　D. CH₃COOH（乙酸）
:::

:::callout{kind=insight label="解析"}
**【答案】D**

有机化合物酸性顺序：有机羧酸(pKa≈5) > 碳酸(pKa₁≈6.4) > 苯酚(pKa≈10) > 乙醇(pKa≈16)。这是课堂上老师强调的重要酸性顺序。

:::

:::callout{kind=insight label="有机酸酸性顺序"}
**酸性顺序**：羧酸 > 碳酸 > 苯酚 > 醇 > 水

**pKa值**：
- 乙酸：4.76
- 碳酸：6.4（第一级）
- 苯酚：10
- 乙醇：16
- 水：15.7

**影响因素**：
- 共轭碱稳定性（共轭碱越稳定，酸性越强）
- 吸电子基团增强酸性（-NO₂ > -Cl > -H）
- 给电子基团减弱酸性（-CH₃ < -H）
:::


### 第9题

:::callout{kind=note label="题目"}
关于R/S构型的判断，下列叙述正确的是（　　）

A. R构型一定是右旋的　　B. S构型一定是左旋的

C. R/S构型与旋光方向无必然联系　　D. R构型的对映体一定也是R构型
:::

:::callout{kind=insight label="解析"}
**【答案】C**

R/S构型是根据基团的空间排列（顺序规则）确定的绝对构型标记，而旋光方向（+/-）是实验测定的物理性质。两者没有简单的对应关系，需要实验测定。

:::

:::callout{kind=note label="R/S构型 vs 旋光方向"}
**R/S构型**：
- 根据Cahn-Ingold-Prelog顺序规则确定
- 是分子的绝对构型
- 与对映体关系：R的对映体是S

**旋光方向（+/-）**：
- (+)或d：右旋（顺时针旋转偏振光）
- (-)或l：左旋（逆时针旋转偏振光）
- 实验测定，与构型无必然联系

**示例**：
- (R)-2-丁醇可能是右旋也可能是左旋
- 需通过实验测定确定
:::


### 第10题

:::callout{kind=note label="题目"}
环氧乙烷在酸催化下与甲醇的开环反应，亲核试剂进攻的位置是（　　）

A. 位阻小的碳　　B. 位阻大的碳或连接给电子基多的碳

C. 随机进攻　　D. 不反应
:::

:::callout{kind=insight label="解析"}
**【答案】B**

环氧乙烷在酸催化开环时，第一步是质子化使环氧中的氧带正电荷，然后亲核试剂倾向于进攻位阻较大或连接给电子基团较多的碳原子（因为该碳带更多正电荷）。碱催化开环则相反，进攻位阻小的碳。

:::

:::callout{kind=insight label="环氧乙烷开环机理"}
**酸催化开环**：
- 质子化氧 → 碳带更多正电荷
- 亲核试剂进攻位阻大/给电子基多的碳（更缺电子）

**碱催化开环**：
- 亲核试剂进攻位阻小的碳（SN2机理）

**应用**：
- 酸催化：制备不对称醚（亲核试剂连在位阻大的碳上）
- 碱催化：制备不对称醚（亲核试剂连在位阻小的碳上）
:::

## 二、填空题（每空2分，共20分）

### 第1题

:::callout{kind=note label="题目"}
分子间弱相互作用包括范德华力和氢键。范德华力的三种形式为_______、_______和_______。
:::

:::callout{kind=insight label="解析"}
**答案**：色散力；诱导力；取向力

**解析**：范德华力由三种分子间作用力组成：色散力（瞬时偶极引起）存在于所有分子间；诱导力（永久偶极诱导瞬时偶极）；取向力（永久偶极间的相互作用）。色散力在大多数情况下占主导。

:::

:::callout{kind=note label="范德华力三种形式"}
| 类型 | 存在条件 | 特点 |
|------|----------|------|
| 色散力 | 所有分子 | 瞬时偶极引起，非极性分子间唯一作用力 |
| 诱导力 | 极性分子+非极性分子 | 永久偶极诱导瞬时偶极 |
| 取向力 | 极性分子之间 | 永久偶极定向排列 |

**注意**：色散力通常占主导，即使是极性分子也是如此
:::


### 第2题

:::callout{kind=note label="题目"}
烷烃的自由基卤代反应中，氢原子被取代的选择性顺序为：_______氢 > _______氢 > _______氢（按反应活性从高到低）。
:::

:::callout{kind=insight label="解析"}
**答案**：叔（三级）；仲（二级）；伯（一级）

**解析**：自由基取代反应中，生成更稳定的碳自由基越有利。碳自由基稳定性：叔碳自由基 > 仲碳自由基 > 伯碳自由基 > 甲基自由基，因此叔氢最容易被取代。

:::

:::callout{kind=note label="自由基稳定性与选择性"}
**碳自由基稳定性**：3° > 2° > 1° > CH₃·

**原因**：超共轭效应和诱导效应稳定自由基

**卤代选择性**：
- 溴代选择性高（主要取代叔氢）
- 氯代选择性低（各种氢都被取代）

**应用**：
- 溴代：选择性高，可得到特定产物
- 氯代：产物复杂，工业上常用
:::


### 第3题

:::callout{kind=note label="题目"}
苯环上的亲电取代反应中，给电子基团是_______定位基（邻对/间位），使苯环活性_______（增强/降低）。
:::

:::callout{kind=insight label="解析"}
**答案**：邻对（位）；增强

**解析**：给电子基团（如-CH₃、-OH、-NH₂等）使苯环电子云密度增大，是致活基团，使亲电取代更容易。电子云密度增大主要体现在邻位和对位，因此是邻对位定位基。

:::

:::callout{kind=insight label="定位基效应总结"}
**给电子基团（邻对位定位，致活）**：
- -OH、-OR、-NH₂、-NHR、-NR₂（强致活）
- -R、-Ar（中等致活）

**吸电子基团（间位定位，致钝）**：
- -NO₂、-CN、-COOH、-CHO、-COR
- -SO₃H、-CF₃

**卤素（邻对位定位，但致钝）**：
- 诱导效应吸电子（致钝）
- 共轭效应给电子（邻对位定位）
:::


### 第4题

:::callout{kind=note label="题目"}
R/S绝对构型的判定方法：按顺序规则排列四个基团的优先级，将最小基团放在远离观察者的位置，从大→中→小画圈，顺时针为_______构型，逆时针为_______构型。
:::

:::callout{kind=insight label="解析"}
**答案**：R；S

**解析**：这是课堂上详细讲解的R/S构型判定方法。将最小基团置于远离观察者的位置（在楔形式中为虚线方向），从最大到最小基团画圈，顺时针为R（rectus），逆时针为S（sinister）。

:::

:::callout{kind=note label="R/S构型判定步骤"}
**Cahn-Ingold-Prelog规则**：
1. **排序**：按原子序数排列四个基团（大→中→小→最小）
2. **定位**：将最小基团指向远离观察者（虚线）
3. **画圈**：从大→中→小画圈
4. **判断**：顺时针为R，逆时针为S

**特殊情况**：
- 最小基团指向观察者（实线）：判断结果取反
- 含双键/三键：视为连有两个/三个相同原子
:::


### 第5题

:::callout{kind=note label="题目"}
苯酚的酸性比醇_______（强/弱），原因是苯酚中氧的孤对电子与苯环形成_______共轭。
:::

:::callout{kind=insight label="解析"}
**答案**：强；p-π

**解析**：苯酚中羟基氧上的孤对电子与苯环的π电子形成p-π共轭，使O-H键极化，质子更易离去，因此苯酚(pKa≈10)的酸性远强于醇(pKa≈16)。离去后的苯酚氧负离子因共轭效应而稳定。

:::

:::callout{kind=insight label="p-π共轭效应"}
**p-π共轭**：
- 含孤对电子的原子（O、N、S、Cl等）与相邻π键的共轭
- 使电子离域，体系稳定

**对酸性的影响**：
- 苯酚：p-π共轭稳定酚氧负离子 → 酸性增强
- 羧酸：p-π共轭稳定羧酸根负离子 → 酸性强
- 醇：无p-π共轭 → 酸性弱

**常见p-π共轭体系**：
- 苯酚、烯醇、烯胺
- 羧酸、酯、酰胺
- 氯乙烯、氯苯
:::

## 三、判断题（每小题2分，共10分，对的画√，错的画×）

### 第1题

:::callout{kind=note label="题目"}
分子的对称性越好，熔点越高。（　　）
:::

:::callout{kind=insight label="解析"}
**【答案】√**

解析：分子对称性好→分子间排列紧密→晶格致密→熔点高。如新戊烷(高度对称)的熔点 > 正戊烷 > 异戊烷。这是老师课堂上强调的"熔点除看分子间作用力外还要看对称性"。

:::

:::callout{kind=note label="熔点与对称性"}
**熔点影响因素**：
1. 分子间作用力（主要）
2. 分子对称性（次要但重要）

**戊烷异构体熔点**：
- 新戊烷（高度对称）：-16.6°C
- 正戊烷：-129.8°C
- 异戊烷：-159.9°C

**原因**：对称分子在晶格中排列更紧密，晶格能更高
:::


### 第2题

:::callout{kind=note label="题目"}
苯的硝化反应中，亲电试剂是NO₂⁺（硝酰正离子）。（　　）
:::

:::callout{kind=insight label="解析"}
**【答案】√**

解析：苯的硝化反应中，浓硝酸在浓硫酸催化下生成硝酰正离子NO₂⁺（亲电试剂），NO₂⁺进攻苯环的π电子云，发生亲电取代反应生成硝基苯。

:::

:::callout{kind=note label="硝化反应机理"}
**反应条件**：浓HNO₃ + 浓H₂SO₄，50-60°C

**亲电试剂生成**：
HNO₃ + 2H₂SO₄ → NO₂⁺ + H₃O⁺ + 2HSO₄⁻

**反应步骤**：
1. NO₂⁺进攻苯环形成σ络合物（碳正离子中间体）
2. 失去H⁺恢复芳香性
3. 生成硝基苯

**注意**：NO₂⁺是直线型，氮带正电荷
:::


### 第3题

:::callout{kind=note label="题目"}
含有手性碳原子的分子一定是手性分子。（　　）
:::

:::callout{kind=insight label="解析"}
**【答案】×**

解析：含两个以上相同手性碳原子的分子可能是内消旋体，虽含手性碳但有对称面，分子整体无手性。如内消旋酒石酸含两个手性碳但有对称面，无手性。判断手性要看分子是否有对称因素（对称面、对称中心），不能仅看手性碳。

:::

:::callout{kind=insight label="内消旋体"}
**内消旋体（meso compound）**：
- 含手性碳但无手性
- 有对称面或对称中心
- 是单一化合物，不是混合物

**示例**：内消旋酒石酸
- 两个手性碳：一个R，一个S
- 分子有对称面 → 无手性
- 无旋光性

**对比**：外消旋体是R和S的等量混合物，有旋光性但总体抵消
:::


### 第4题

:::callout{kind=note label="题目"}
傅克烷基化反应中，烷基可能发生碳正离子重排。（　　）
:::

:::callout{kind=insight label="解析"}
**【答案】√**

解析：傅克烷基化反应中，卤代烷在AlCl₃催化下先生成碳正离子，碳正离子可能发生重排（甲基迁移或负氢迁移）使之变为更稳定的碳正离子，再与苯环发生取代。如1-氯丙烷可能发生负氢重排生成异丙基苯。

:::

:::callout{kind=note label="碳正离子重排"}
**重排类型**：
1. **负氢迁移**：H从相邻碳迁移到碳正离子中心
2. **甲基迁移**：CH₃从相邻碳迁移到碳正离子中心

**驱动力**：生成更稳定的碳正离子

**示例**：
CH₃CH₂CH₂⁺ → CH₃CH⁺CH₃（负氢迁移，1°→2°）

**避免重排的方法**：使用傅克酰基化（酰基正离子不重排）
:::


### 第5题

:::callout{kind=note label="题目"}
醇的酸性强弱与醇的级别无关，只与烷基的电子效应有关。（　　）
:::

:::callout{kind=insight label="解析"}
**【答案】×**

解析：醇的酸性还受到其他因素影响，如分子量、取代基的诱导效应等。但从级别角度看，叔醇的碳正离子较稳定而醇氧负离子相对不稳定，也会影响酸性。不过影响醇酸性的因素确实主要是烷基的电子效应（给电子/吸电子），此说法过于绝对，故判×。

:::

:::callout{kind=note label="醇的酸性"}
**醇酸性顺序**：CH₃OH > 1° > 2° > 3°

**原因**：
- 烷基是给电子基团，稳定醇氧负离子能力：CH₃O⁻ > 1° > 2° > 3°
- 但溶剂化效应也起作用

**增强酸性的方法**：
- 吸电子取代基（如CF₃CH₂OH，pKa≈12.4）
- 共轭效应（如苯酚，pKa≈10）
:::

## 四、反应题（每小题5分，共25分。写出主要产物或反应条件）

### 第1题

:::callout{kind=note label="题目"}
甲苯 + Br₂ ——(FeBr₃)——→ ？（写出主要产物及位置）
:::

:::callout{kind=insight label="解析"}
**答案**：邻溴甲苯 + 对溴甲苯（邻、对位产物混合物）

**解析**：甲基是邻对位定位基，致活基团。在FeBr₃（路易斯酸）催化下，溴的亲电取代反应发生在甲基的邻位和对位。主产物一般为邻位和对位的混合物。注意：光照条件下则是侧链的自由基取代反应。

:::

:::callout{kind=note label="定位效应与反应条件"}
**亲电取代（FeBr₃催化）**：
- 甲基定位：邻位 + 对位
- 邻位产物：约58%
- 对位产物：约37%
- 间位产物：约5%（次要）

**自由基取代（光照）**：
- 发生在侧链苄位
- 产物：苄基溴

**注意**：反应条件决定反应类型！
:::


### 第2题

:::callout{kind=note label="题目"}
(R)-2-溴丁烷 + NaOH ——(水溶液,加热)——→ ？（判断反应类型并写出产物）
:::

:::callout{kind=insight label="解析"}
**答案**：反应类型为SN1（仲卤代烷在水溶液中）；产物为(±)-2-丁醇（外消旋2-丁醇）

**解析**：仲卤代烃在极性质子溶剂（水溶液）中，加热有利于SN1反应。SN1经历碳正离子中间体（平面型），OH⁻可从两面进攻，得到外消旋化产物。同时可能伴有少量E1消除产物（2-丁烯）。

:::

:::callout{kind=insight label="SN1反应中的立体化学"}
**外消旋化的原因**：
- 碳正离子为平面sp²杂化
- 亲核试剂从两面进攻概率相等
- 产物为R和S的等量混合物

**注意**：
- 并非完全外消旋化（离子对效应可能导致部分构型保持）
- 通常观察到的是大部分外消旋化+少量构型翻转

**对比SN2**：构型完全翻转（Walden翻转）
:::


### 第3题

:::callout{kind=note label="题目"}
苯酚 + 3Br₂ ——(水溶液)——→ ？
:::

:::callout{kind=insight label="解析"}
**答案**：2,4,6-三溴苯酚↓（白色沉淀）+ 3HBr

**解析**：苯酚的-OH是强致活基团，使苯环活性大大增强。在水溶液中（极性溶剂），苯酚与过量溴水直接生成三取代产物（邻位两个+对位一个），生成白色的2,4,6-三溴苯酚沉淀。在非极性溶剂中则只生成一溴代产物。

:::

:::callout{kind=note label="苯酚的溴代反应"}
**反应条件与产物**：
| 条件 | 产物 |
|------|------|
| 水溶液（Br₂/H₂O） | 2,4,6-三溴苯酚（白色沉淀） |
| CS₂或CCl₄（低温） | 一溴代苯酚（对位为主） |

**原因**：
- -OH强致活基团，使苯环活性大增
- 水溶液中极性环境有利于多取代
- 非极性溶剂中控制条件可得一取代

**应用**：苯酚的定性鉴定（白色沉淀）
:::


### 第4题

:::callout{kind=note label="题目"}
环己醇 + KMnO₄ ——(H⁺)——→ ？
:::

:::callout{kind=insight label="解析"}
**答案**：环己酮（进一步可氧化为己二酸）

**解析**：仲醇在酸性KMnO₄（强氧化剂）条件下被氧化成酮。如果使用更温和的氧化剂（如柯林斯试剂、琼斯试剂），也可以选择性氧化为酮。

:::

:::callout{kind=note label="醇的氧化"}
**氧化产物**：
| 醇类型 | 氧化产物 | 条件 |
|--------|----------|------|
| 伯醇 | 醛 → 羧酸 | 强氧化剂直接到酸 |
| 仲醇 | 酮 | 温和氧化剂 |
| 叔醇 | 难氧化 | 强条件才反应 |

**常用氧化剂**：
- KMnO₄/H⁺：强氧化，伯醇→羧酸
- K₂Cr₂O₇/H⁺：强氧化
- PCC/CH₂Cl₂：温和，伯醇→醛
- Collins试剂：温和，伯醇→醛
:::


### 第5题

:::callout{kind=note label="题目"}
苯 + CH₃COCl ——(AlCl₃)——→ ？（写出反应类型及主要产物）
:::

:::callout{kind=insight label="解析"}
**答案**：C₆H₅COCH₃（苯乙酮）+ HCl；反应类型：傅-克酰基化反应

**解析**：酰氯在AlCl₃催化下生成酰基正离子（CH₃CO⁺），进攻苯环发生亲电取代反应。与傅克烷基化不同，酰基化不会发生碳正离子重排。

:::

:::callout{kind=insight label="傅-克酰基化 vs 烷基化"}
| 特征 | 酰基化 | 烷基化 |
|------|--------|--------|
| 重排 | 不重排 | 可能重排 |
| 产物 | 酮 | 烷基苯 |
| 可逆性 | 不可逆 | 可逆 |
| 多取代 | 不易多取代 | 易多取代 |

**酰基化优势**：
- 产物为酮，可进一步还原为烷基
- 避免重排，可制备直链烷基苯
:::

## 五、合成题（每小题7.5分，共15分。写出合成路线及所用试剂和条件）

### 第1题

:::callout{kind=note label="题目"}
以苯和乙烯为原料合成苯乙酮（C₆H₅COCH₃）。
:::

:::callout{kind=insight label="解析"}
**路线一（傅克酰基化）**：

苯 + CH₃COCl ——(AlCl₃)——→ C₆H₅COCH₃（苯乙酮）

**解析**：直接使用乙酰氯与苯进行傅-克酰基化反应即可。

**路线二（先烷基化再氧化）**：

① 苯 + CH₂=CH₂ ——(H₃PO₄或AlCl₃/HCl)——→ C₆H₅CH₂CH₃（乙苯）

② C₆H₅CH₂CH₃ ——(KMnO₄/H⁺)——→ C₆H₅COOH（苯甲酸）

注意：此路线氧化后得到苯甲酸而非苯乙酮，需要其他方法。正确思路应使用傅克酰基化。

**最佳路线**：苯 + CH₃COCl ——(AlCl₃)——→ 苯乙酮

:::

:::callout{kind=note label="合成策略选择"}
**目标分子分析**：
- 苯乙酮：芳环 + 乙酰基（-COCH₃）
- 直接引入乙酰基 → 傅克酰基化

**为什么不用烷基化**：
- 傅克烷基化会重排（1-氯乙烷→乙基苯）
- 氧化侧链得到苯甲酸（不是苯乙酮）
- 酰基化直接得到目标产物
:::


### 第2题

:::callout{kind=note label="题目"}
以乙醇为起始原料合成正丁醇。（提示：考虑格氏试剂与环氧乙烷的反应）
:::

:::callout{kind=insight label="解析"}
**路线**：

① CH₃CH₂OH + HBr ——→ CH₃CH₂Br（溴乙烷）

② CH₃CH₂Br + Mg ——(无水乙醚)——→ CH₃CH₂MgBr（乙基格氏试剂）

③ CH₃CH₂MgBr + 环氧乙烷 ——→ [烷氧基镁盐] ——(H₃O⁺)——→ CH₃CH₂CH₂CH₂OH（正丁醇）

**解析**：格氏试剂与环氧乙烷的反应可使碳链增长两个碳。格氏试剂中的碳负离子进攻环氧乙烷中位阻较小的碳（亲核取代），形成新的C-C键。乙基(2C) + 环氧乙烷(2C) = 正丁醇(4C)。这是课堂上强调的"多两个碳用环氧乙烷"的方法。

:::

:::callout{kind=insight label="格氏试剂增碳策略"}
**增碳方法**：
| 增碳数 | 试剂 | 产物类型 |
|--------|------|----------|
| +1 | CO₂ | 羧酸 |
| +1 | HCHO | 伯醇 |
| +2 | 环氧乙烷 | 伯醇（多两个碳） |
| +R | RCHO | 仲醇 |
| +R | RCOR | 叔醇 |

**注意**：格氏试剂需无水无氧条件！
:::

## 六、推断题（10分）

化合物A(C₈H₁₀)是一种芳香烃，A经KMnO₄/H⁺氧化后只得到对苯二甲酸。

**(1) 推断A的结构式。（3分）**

**(2) A进行一氯代反应（FeCl₃催化）时，可能的产物有哪些？哪个是主产物？说明理由。（4分）**

**(3) A进行侧链自由基溴代（光照条件）时，主要产物是什么？为什么？（3分）**

:::callout{kind=insight label="解析"}
**(1) A的结构式**

A的结构式为：对二甲苯（1,4-二甲苯，CH₃-C₆H₄-CH₃）

**推断过程**：A(C₈H₁₀)是芳香烃，不饱和度=(2×8+2-10)/2=4，恰好为一个苯环。减去苯环C₆H₅后剩余C₂H₅，可以是两个甲基取代。氧化后只得对苯二甲酸（HOOC-C₆H₄-COOH），说明两个甲基在对位。

（3分：结构式1.5分，推断过程1.5分）

**(2) 一氯代反应**

两个甲基都是邻对位定位基。由对称性分析，苯环上只有2号位（与甲基邻位）可被取代，生成2-氯-1,4-二甲苯。由于分子的对称性，所有可取代位置是等价的，只有一种产物，即主产物。

（4分：产物结构2分，定位分析2分）

**(3) 侧链自由基溴代**

主要产物是对甲基溴化苄 CH₂Br-C₆H₄-CH₃

**原因**：自由基溴代发生在苯环侧链的苄位（与苯环直接相连的碳上），因为苄基自由基因与苯环共轭而特别稳定。两个甲基等价，任一甲基上的氢被溴取代。

（3分：产物1.5分，理由1.5分）

:::

:::callout{kind=insight label="推断题解题策略"}
**由氧化产物推断取代基位置**：
- 对苯二甲酸 → 对位二取代
- 邻苯二甲酸 → 邻位二取代
- 间苯二甲酸 → 间位二取代

**苄位自由基稳定性**：
- 苄基自由基：p-π共轭稳定
- 烯丙基自由基：p-π共轭稳定
- 稳定性：苄基/烯丙基 > 3° > 2° > 1° > CH₃·

**反应条件识别**：
- FeCl₃/Cl₂ → 环上亲电取代
- 光照/Br₂ → 侧链自由基取代
:::
