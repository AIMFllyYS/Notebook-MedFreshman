## 一、选择题（每题2分，共24分）

### 第1题

:::callout{kind=note label="题目"}
下列碱性由强到弱正确的是

A. 苯胺 > 氨 > 甲胺    B. 甲胺 > 氨 > 苯胺    C. 氨 > 苯胺 > 甲胺    D. 苯胺 > 甲胺 > 氨
:::

:::callout{kind=insight label="解析"}
**【答案】B**

甲基给电子增强碱性；苯胺N孤对电子与苯环共轭，碱性弱于氨。

:::

:::callout{kind=insight label="胺类碱性比较"}
**碱性顺序**：甲胺 > 氨 > 苯胺

**原因**：
- 甲胺：甲基给电子，氮上电子云密度高
- 氨：无取代基效应
- 苯胺：p-π共轭分散孤对电子

**扩展顺序**：
脂肪胺 > 氨 > 苯胺 > 二苯胺 > 三苯胺 > 酰胺
:::


### 第2题

:::callout{kind=note label="题目"}
苯胺、乙胺、二乙胺、三乙胺可用来区分伯/仲/叔胺的经典反应是

A. 碘仿反应    B. 兴斯堡反应    C. Lucas反应    D. 银镜反应
:::

:::callout{kind=insight label="解析"}
**【答案】B**

苯磺酰氯/碱条件下，伯、仲、叔胺表现不同。

:::

:::callout{kind=note label="兴斯堡反应（Hinsberg反应）"}
**试剂**：苯磺酰氯/NaOH

**现象**：
| 胺类型 | 与苯磺酰氯反应 | NaOH中 | 酸化后 |
|--------|--------------|--------|--------|
| 伯胺 | 生成磺酰胺 | 溶解 | 析出 |
| 仲胺 | 生成磺酰胺 | 不溶 | 不溶 |
| 叔胺 | 不反应 | 溶解 | 溶解 |

**应用**：鉴别伯、仲、叔胺
:::


### 第3题

:::callout{kind=note label="题目"}
苯胺重氮化的典型条件是

A. NaNO₂/HCl，0-5℃    B. Br₂/FeBr₃，室温    C. NaOH/H₂O，加热    D. SOCl₂，回流
:::

:::callout{kind=insight label="解析"}
**【答案】A**

芳香伯胺在低温亚硝酸条件下生成重氮盐。

:::

:::callout{kind=note label="重氮化反应条件"}
**试剂**：NaNO₂ + HCl（或H₂SO₄）

**温度**：0-5℃（低温防止重氮盐分解）

**注意**：
- 脂肪伯胺：重氮盐不稳定，立即分解
- 芳香伯胺：重氮盐稳定（低温）
- 仲胺：生成N-亚硝基胺
- 叔胺：不反应或环上取代
:::


### 第4题

:::callout{kind=note label="题目"}
苯重氮盐与CuBr反应属于

A. Wittig反应    B. Sandmeyer反应    C. Cannizzaro反应    D. Claisen缩合
:::

:::callout{kind=insight label="解析"}
**【答案】B**

Sandmeyer反应将重氮基替换为Cl/Br/CN等。

:::

:::callout{kind=note label="Sandmeyer反应"}
**反应**：ArN₂⁺ + CuX → Ar-X + N₂↑

**试剂**：CuCl、CuBr、CuCN

**产物**：
- CuCl：氯代苯
- CuBr：溴代苯
- CuCN：氰基苯

**机理**：自由基取代（单电子转移）
:::


### 第5题

:::callout{kind=note label="题目"}
吡咯的碱性很弱，主要原因是

A. N上无孤对电子    B. N孤对电子参与芳香π体系    C. 环不是芳香性    D. N为sp3杂化
:::

:::callout{kind=insight label="解析"}
**【答案】B**

吡咯型氮孤对电子计入6π芳香体系，不易结合质子。

:::

:::callout{kind=insight label="吡咯的弱碱性"}
**吡咯结构**：
- 6π电子：4个来自双键，2个来自N孤对电子
- N孤对电子参与芳香体系
- 不能结合质子（否则破坏芳香性）

**对比吡啶**：
- N孤对电子在sp²轨道，不参与芳香体系
- 可结合质子，碱性较强
:::


### 第6题

:::callout{kind=note label="题目"}
五元富电子杂环吡咯、呋喃、噻吩发生亲电取代通常优先在

A. α位    B. β位    C. N/O/S上    D. 任意位置等量
:::

:::callout{kind=insight label="解析"}
**【答案】A**

α位取代的σ络合物共振稳定更多，通常为主。

:::

:::callout{kind=note label="五元杂环亲电取代定位"}
**活性顺序**：吡咯 > 呋喃 > 噻吩 > 苯

**定位**：α位优先于β位

**原因**：
- α位取代：σ络合物有3个共振式（更稳定）
- β位取代：σ络合物有2个共振式

**反应条件**：
- 吡咯：非常活泼，不需催化剂
- 呋喃：较活泼，温和条件
- 噻吩：较稳定，需催化剂
:::


### 第7题

:::callout{kind=note label="题目"}
吡啶亲电取代相对困难，通常主要发生在

A. 2位    B. 3位    C. 4位    D. N上    E. 任意位
:::

:::callout{kind=insight label="解析"}
**【答案】B**

N吸电子降低环电子云；β位σ络合物较少出现N邻位正电荷，较稳定。

:::

:::callout{kind=insight label="吡啶的亲电取代"}
**特点**：
- 缺电子芳香体系（6π全部来自碳）
- 亲电取代比苯困难
- 定位：β位（3位）

**原因**：
- 2位/4位取代：σ络合物中N邻位带正电荷（不稳定）
- 3位取代：N不直接邻位带正电（较稳定）

**对比**：
- 亲核取代：α/γ位（N活化邻位）
:::


### 第8题

:::callout{kind=note label="题目"}
下列糖中通常为非还原糖的是

A. 葡萄糖    B. 果糖    C. 纤维二糖    D. 蔗糖
:::

:::callout{kind=insight label="解析"}
**【答案】D**

蔗糖两个异头碳均参与糖苷键，无游离半缩醛羟基。

:::

:::callout{kind=note label="还原糖 vs 非还原糖"}
**还原糖**：含游离半缩醛/半缩酮羟基
- 单糖（葡萄糖、果糖）
- 还原性二糖（麦芽糖、乳糖）

**非还原糖**：无游离半缩醛/半缩酮羟基
- 蔗糖（两个苷羟基都参与糖苷键）
- 糖苷（甲基葡萄糖苷）
- 多糖

**判断依据**：是否能开环形成醛基/酮基
:::


### 第9题

:::callout{kind=note label="题目"}
葡萄糖、甘露糖、果糖可形成相同糖脎，原因是糖脎形成主要改变

A. C1和C2    B. C3和C4    C. 所有手性碳    D. C5构型
:::

:::callout{kind=insight label="解析"}
**【答案】A**

成脎涉及羰基碳和α-羟基碳，C1/C2以外构型相同则糖脎相同。

:::

:::callout{kind=note label="糖脎反应"}
**反应**：糖 + 3PhNHNH₂ → 糖脎（黄色结晶）

**特点**：
- 涉及C1和C2
- C1（醛基）→ 苯腙
- C2（α-OH）→ 苯腙

**相同糖脎的糖**：
- C3、C4、C5构型相同
- C1、C2不同（差向异构体）

**示例**：
- 葡萄糖、甘露糖、果糖 → 相同糖脎
- 因为C3、C4、C5构型相同
:::


### 第10题

:::callout{kind=note label="题目"}
α-氨基酸与茚三酮共热通常显

A. 绿色    B. 蓝紫色    C. 白色沉淀    D. 银镜
:::

:::callout{kind=insight label="解析"}
**【答案】B**

茚三酮是α-氨基酸灵敏显色反应。

:::

:::callout{kind=note label="茚三酮反应"}
**反应**：α-氨基酸 + 茚三酮 → 蓝紫色化合物 + 醛 + CO₂

**颜色**：蓝紫色（脯氨酸为黄色，因是亚氨基酸）

**应用**：
- 氨基酸的定性鉴定
- 纸层析显色
- 蛋白质水解液分析

**灵敏度**：可检测μg级别的氨基酸
:::


### 第11题

:::callout{kind=note label="题目"}
除半胱氨酸外，天然L-α-氨基酸α碳R/S构型通常为

A. R    B. S    C. 外消旋    D. 内消旋
:::

:::callout{kind=insight label="解析"}
**【答案】B**

L-氨基酸多数为S；半胱氨酸因硫优先级高例外为R。

:::

:::callout{kind=insight label="L-构型与R/S的关系"}
**L-氨基酸**：
- 氨基在Fischer投影中朝左
- 多数对应S构型

**半胱氨酸例外**：
- -CH₂SH中S优先级高于-COOH
- 导致R构型

**注意**：
- L/D：相对构型（与甘油醛比较）
- R/S：绝对构型（Cahn-Ingold-Prelog规则）
- 两者无必然对应关系
:::


### 第12题

:::callout{kind=note label="题目"}
酰胺经Br₂/NaOH发生Hofmann降级反应，产物碳数相比原酰胺

A. 增加1个    B. 不变    C. 减少1个    D. 减少2个
:::

:::callout{kind=insight label="解析"}
**【答案】C**

Hofmann重排使酰胺变为少一个羰基碳的胺。

:::

:::callout{kind=note label="Hofmann降级反应"}
**反应**：R-CONH₂ + Br₂ + 4NaOH → R-NH₂ + Na₂CO₃ + 2NaBr + 2H₂O

**特点**：
- 酰胺 → 伯胺
- 减少1个碳（羰基碳）
- 经历异氰酸酯中间体（R-N=C=O）

**应用**：
- 由羧酸制备少一个碳的胺
- 制备不能由硝基还原得到的胺

**对比Curtius重排**：
R-CO-N₃ → R-N=C=O → R-NH₂（同样减碳）
:::

## 二、填空题（每空1分，共8分）

### 第1题

:::callout{kind=note label="题目"}
芳香伯胺经重氮盐可进一步发生 __________ 反应生成偶氮化合物。
:::

:::callout{kind=insight label="解析"}
**答案**：偶合

**解析**：重氮盐与活化芳环（如苯酚、苯胺）在弱碱低温下偶合生成偶氮化合物。

:::

:::callout{kind=note label="偶合反应"}
**反应**：ArN₂⁺ + Ar'OH → Ar-N=N-Ar'OH（偶氮化合物）

**条件**：
- 弱碱（pH 4-7）
- 低温（0-5℃）
- 活化芳环（-OH、-NH₂）

**应用**：
- 合成偶氮染料
- 指示剂（甲基橙）
:::


### 第2题

:::callout{kind=note label="题目"}
重氮盐中的重氮正离子在偶合反应中表现为 __________ 试剂。
:::

:::callout{kind=insight label="解析"}
**答案**：亲电

**解析**：重氮正离子进攻富电子芳环（如苯酚），是弱亲电试剂。

:::

:::callout{kind=note label="重氮盐的亲电性"}
**重氮正离子**：ArN₂⁺

**亲电性**：弱（只能与强活化芳环反应）

**对比**：
- 硝酰正离子NO₂⁺：强亲电
- 重氮正离子ArN₂⁺：弱亲电

**原因**：
- 正电荷在N上，离芳环较远
- 需活化芳环（-OH、-NH₂）才能反应
:::


### 第3题

:::callout{kind=note label="题目"}
吡啶型氮孤对电子 __________ 芳香π体系，因此具有一定碱性。
:::

:::callout{kind=insight label="解析"}
**答案**：不参与

**解析**：吡啶氮孤对电子在sp²轨道，不参与6π芳香体系，可结合质子。

:::

:::callout{kind=insight label="吡啶型氮 vs 吡咯型氮"}
| 特征 | 吡啶型 | 吡咯型 |
|------|--------|--------|
| 孤对电子 | 不参与芳香体系 | 参与芳香体系 |
| 轨道 | sp² | p |
| 碱性 | 有 | 无（N-H显酸性） |
| 例子 | 吡啶、嘧啶 | 吡咯、咪唑（一个N） |
:::


### 第4题

:::callout{kind=note label="题目"}
吡咯、呋喃、噻吩亲电取代活性通常均高于苯，属于 __________ 电子芳香体系。
:::

:::callout{kind=insight label="解析"}
**答案**：富

**解析**：6π电子中4个来自双键，2个来自杂原子，电子密度比苯高。

:::

:::callout{kind=note label="富电子 vs 缺电子杂环"}
**富电子杂环**（五元）：
- 吡咯、呋喃、噻吩
- 亲电取代比苯容易
- α位优先

**缺电子杂环**（六元）：
- 吡啶、嘧啶
- 亲电取代比苯困难
- β位优先
- 但亲核取代比苯容易
:::


### 第5题

:::callout{kind=note label="题目"}
糖苷形成后，异头碳不再有半缩醛羟基，因此通常无 __________ 现象。
:::

:::callout{kind=insight label="解析"}
**答案**：变旋光

**解析**：糖苷是缩醛结构，异头碳被固定，不能开环，因此无变旋光现象。

:::

:::callout{kind=note label="变旋光现象"}
**原因**：α-构型 ↔ β-构型 ↔ 开链式（互变异构）

**有变旋光**：单糖、还原性二糖
**无变旋光**：糖苷、非还原性二糖、多糖

**糖苷**：
- 异头碳的-OH被-OR取代
- 缩醛结构，稳定
- 不能开环
:::


### 第6题

:::callout{kind=note label="题目"}
能被Tollens、Benedict或Fehling试剂氧化的糖称为 __________ 糖。
:::

:::callout{kind=insight label="解析"}
**答案**：还原

**解析**：还原糖含游离醛基或能生成醛基（如果糖在碱性条件下异构化）。

:::

:::callout{kind=note label="还原糖的判断"}
**还原糖**：含游离半缩醛/半缩酮羟基
- 可开环形成醛基/酮基
- 能被弱氧化剂氧化

**非还原糖**：无游离半缩醛/半缩酮羟基
- 不能开环
- 不被弱氧化剂氧化

**常见还原糖**：葡萄糖、果糖、麦芽糖、乳糖
**常见非还原糖**：蔗糖、糖苷
:::


### 第7题

:::callout{kind=note label="题目"}
氨基酸在等电点pH=pI时净电荷为 __________，溶解度常较 __________。
:::

:::callout{kind=insight label="解析"}
**答案**：零；小

**解析**：等电点时正负电荷相等，净电荷为零；分子间静电引力最小，溶解度最低。

:::

:::callout{kind=insight label="等电点（pI）"}
**定义**：氨基酸净电荷为零时的pH

**计算**：
- 中性氨基酸：pI = (pK₁ + pK₂)/2
- 酸性氨基酸：pI = (pK₁ + pK_R)/2
- 碱性氨基酸：pI = (pK₂ + pK_R)/2

**性质**：
- 溶解度最小（易结晶析出）
- 电泳不移动

**应用**：
- 分离氨基酸（调节pH至pI沉淀）
- 电泳分离
:::


### 第8题

:::callout{kind=note label="题目"}
蛋白质一级结构指多肽链中 __________ 的排列顺序。
:::

:::callout{kind=insight label="解析"}
**答案**：氨基酸残基

**解析**：一级结构是氨基酸的线性序列，由肽键连接。

:::

:::callout{kind=note label="蛋白质结构层次"}
| 结构层次 | 定义 | 维持力 |
|----------|------|--------|
| 一级 | 氨基酸序列 | 肽键（共价） |
| 二级 | α-螺旋、β-折叠 | 氢键 |
| 三级 | 整体三维构象 | 多种副键 |
| 四级 | 亚基组装 | 非共价键 |

**一级结构决定高级结构！**
:::

## 三、判断题（每题1分，共8分）

### 第1题

:::callout{kind=note label="题目"}
三苯胺比苯胺碱性强，因为苯环数目多。（　　）
:::

:::callout{kind=insight label="解析"}
**【答案】×**

解析：苯环使N孤对电子离域且位阻增大，碱性反而降低。

:::

:::callout{kind=note label="苯环对胺碱性的影响"}
**苯胺**：pKb ~9.4
**二苯胺**：pKb ~13.2（更弱）
**三苯胺**：pKb ~15（极弱）

**原因**：
- 苯环与N孤对电子p-π共轭
- 电子离域到苯环上
- N上电子云密度降低
- 位阻也影响质子接近
:::


### 第2题

:::callout{kind=note label="题目"}
重氮盐可通过Sandmeyer反应转化为芳基卤化物。（　　）
:::

:::callout{kind=insight label="解析"}
**【答案】√**

解析：CuCl/CuBr/CuCN等可替换重氮基。

:::

:::callout{kind=note label="重氮盐的转化"}
**Sandmeyer反应**：
ArN₂⁺ + CuX → Ar-X + N₂↑

**Schiemann反应**：
ArN₂⁺ + HBF₄ → Ar-F + N₂↑ + HF

**水解**：
ArN₂⁺ + H₂O → Ar-OH + N₂↑ + H⁺

**还原**：
ArN₂⁺ + H₃PO₂ → Ar-H + N₂↑ + H₃PO₃
:::


### 第3题

:::callout{kind=note label="题目"}
苯酚和苯胺都能作为偶合反应中的活化芳环组分。（　　）
:::

:::callout{kind=insight label="解析"}
**【答案】√**

解析：-OH和-NH₂给电子活化芳环，利于弱亲电性的重氮盐偶合。

:::

:::callout{kind=note label="偶合反应条件"}
**活化芳环**：
- 苯酚（弱碱，pH 4-7）
- 苯胺（弱酸，pH 2-5）
- N,N-二甲基苯胺

**反应位置**：
- 苯酚：对位（邻位被占据时）
- 苯胺：对位

**注意**：
- 强碱性条件：重氮盐分解
- 强酸性条件：苯胺质子化，不活泼
:::


### 第4题

:::callout{kind=note label="题目"}
呋喃在强酸强氧化条件下仍十分稳定，因此可用浓硝酸直接硝化。（　　）
:::

:::callout{kind=insight label="解析"}
**【答案】×**

解析：五元富电子杂环易被强酸/氧化剂破坏，反应条件需温和。

:::

:::callout{kind=note label="呋喃的反应条件"}
**特点**：
- 富电子，非常活泼
- 易被强酸破坏（开环聚合）
- 易被氧化

**硝化条件**：
- 不能用浓HNO₃（破坏呋喃）
- 用乙酰硝酸酯（CH₃COONO₂）
- 低温温和条件

**噻吩**：较稳定，可用温和条件硝化
:::


### 第5题

:::callout{kind=note label="题目"}
所有单糖都是还原糖。（　　）
:::

:::callout{kind=insight label="解析"}
**【答案】√**

解析：单糖存在开链醛/酮形式；酮糖可在弱碱下异构化而还原弱氧化剂。

:::

:::callout{kind=note label="单糖的还原性"}
**醛糖**：含醛基，还原性

**酮糖**：
- 含酮基，本身还原性较弱
- 在碱性条件下异构化为醛糖
- 因此也能还原Tollens、Fehling试剂

**结论**：所有单糖都是还原糖
:::


### 第6题

:::callout{kind=note label="题目"}
甲基葡萄糖苷由于无游离半缩醛羟基，不发生变旋光。（　　）
:::

:::callout{kind=insight label="解析"}
**【答案】√**

解析：糖苷是缩醛结构，异头碳被固定。

:::

:::callout{kind=note label="糖苷的性质"}
**结构**：异头碳的-OH被-OR取代

**特点**：
- 缩醛结构，稳定
- 不能开环
- 无变旋光
- 非还原糖

**水解**：
- 酸催化：恢复为单糖
- 碱稳定：可用于保护苷羟基
:::


### 第7题

:::callout{kind=note label="题目"}
蔗糖能形成糖脎且具有变旋光现象。（　　）
:::

:::callout{kind=insight label="解析"}
**【答案】×**

解析：蔗糖为非还原二糖，无游离半缩醛羟基。

:::

:::callout{kind=note label="蔗糖的结构"}
**组成**：葡萄糖 + 果糖

**糖苷键**：
- 葡萄糖的C1（α）
- 果糖的C2（β）
- 1,2-糖苷键

**特点**：
- 两个异头碳都参与糖苷键
- 无游离半缩醛羟基
- 非还原糖
- 无变旋光
- 不能形成糖脎
:::


### 第8题

:::callout{kind=note label="题目"}
调节pH至氨基酸等电点可用于分离氨基酸。（　　）
:::

:::callout{kind=insight label="解析"}
**【答案】√**

解析：等电点净电荷为零，溶解度较小，易析出。

:::

:::callout{kind=note label="等电点分离"}
**原理**：
- pH = pI：净电荷为零，溶解度最小
- pH < pI：带正电
- pH > pI：带负电

**方法**：
1. 电泳：根据电荷分离
2. 离子交换色谱：根据电荷分离
3. 等电点沉淀：调节pH至pI

**应用**：
- 蛋白质纯化
- 氨基酸分离
:::

## 五、鉴别/排序题（每题5分，共10分）

### 第1题

:::callout{kind=note label="题目"}
鉴别苯胺、乙胺、二乙胺、三乙胺。
:::

:::callout{kind=insight label="解析"}
**答案**：先加溴水：生成白色2,4,6-三溴苯胺沉淀者为苯胺。其余用兴斯堡反应（苯磺酰氯/NaOH）：乙胺生成可溶盐，酸化后析出；二乙胺生成不溶磺酰胺沉淀；三乙胺不反应但可溶于盐酸。

**解析**：苯胺环强活化，溴水取代；Hinsberg根据伯、仲、叔胺磺酰化后酸碱性差异鉴别。

:::

:::callout{kind=insight label="系统鉴别策略"}
**第一步：溴水**
- 苯胺：白色沉淀（2,4,6-三溴苯胺）
- 其他胺：无反应

**第二步：兴斯堡反应**
- 伯胺（乙胺）：磺酰胺溶于NaOH，酸化析出
- 仲胺（二乙胺）：磺酰胺不溶于NaOH
- 叔胺（三乙胺）：不反应，但溶于酸

**注意**：鉴别顺序很重要！
:::


### 第2题

:::callout{kind=note label="题目"}
判断蔗糖、纤维二糖、α-D-吡喃型甲基葡萄糖苷、2-脱氧核糖是否具有变旋光现象和还原性。
:::

:::callout{kind=insight label="解析"}
**答案**：
- 蔗糖：无变旋光、非还原
- 纤维二糖：有变旋光、还原
- α-D-吡喃型甲基葡萄糖苷：无变旋光、非还原
- 2-脱氧核糖：有变旋光、还原

**解析**：关键看是否有游离半缩醛羟基。多糖一般还原性很弱/不明显，考试多考典型二糖和糖苷。

:::

:::callout{kind=note label="变旋光与还原性判断"}
| 糖类 | 变旋光 | 还原性 | 原因 |
|------|--------|--------|------|
| 蔗糖 | 无 | 无 | 两个苷羟基都参与糖苷键 |
| 纤维二糖 | 有 | 有 | 保留一个游离苷羟基 |
| 甲基葡萄糖苷 | 无 | 无 | 异头碳被-OMe取代 |
| 2-脱氧核糖 | 有 | 有 | 单糖，有游离半缩醛羟基 |

**判断依据**：是否有游离半缩醛/半缩酮羟基

## 六、合成题（8分）

由苯合成对溴苯胺，要求说明为什么需要保护氨基。

:::callout{kind=insight label="解析"}
**路线**：

苯 --(HNO₃/H₂SO₄)--> 硝基苯 --(Fe/HCl)--> 苯胺 --((CH₃CO)₂O)--> 乙酰苯胺 --(Br₂/AcOH)--> 对溴乙酰苯胺为主 --(H₃O⁺, Δ)--> 对溴苯胺。

**解析**：直接溴化苯胺会因-NH₂强活化导致多溴代；先乙酰化降低活化性并保留邻/对定位，对位因位阻为主，最后水解脱保护。

:::

:::callout{kind=insight label="氨基保护策略"}
**保护原因**：
- -NH₂是强致活基团
- 直接溴化：多溴代（三溴苯胺）
- 无法控制单取代

**乙酰化保护**：
- (CH₃CO)₂O或CH₃COCl
- -NH₂ → -NHCOCH₃（乙酰氨基）
- 活化性降低（中等致活）
- 仍保持邻对位定位
- 对位位阻小，为主产物

**脱保护**：
- H₃O⁺, Δ 或 OH⁻, Δ
- 恢复-NH₂
:::

## 七、推断题（10分）

X为芳香族含氮化合物，分子式C₆H₇N；遇溴水迅速生成白色沉淀；与NaNO₂/HCl在0-5℃生成盐Y，Y与CuCl反应放出N₂并生成氯苯；Y与苯酚在弱碱低温下生成橙红色化合物Z。推断X、Y、Z。

:::callout{kind=insight label="解析"}
**X = 苯胺**；**Y = 氯化重氮苯**；**Z = 对羟基偶氮苯（主产物）**

**解析**：C₆H₇N且溴水白色沉淀指苯胺；芳香伯胺低温重氮化生成重氮盐；重氮盐与CuCl为Sandmeyer氯代；与苯酚偶合得偶氮染料类橙红色产物。

:::

:::callout{kind=insight label="推断题分析"}
**线索分析**：
1. C₆H₇N，不饱和度4 → 含苯环
2. 溴水白色沉淀 → 苯胺（2,4,6-三溴苯胺）
3. NaNO₂/HCl, 0-5℃ → 重氮化反应
4. Y + CuCl → 氯苯（Sandmeyer反应）
5. Y + 苯酚 → 橙红色（偶氮化合物）

**验证**：
- 苯胺：C₆H₅NH₂（C₆H₇N）
- 溴水：生成2,4,6-三溴苯胺（白色沉淀）
- 重氮化：C₆H₅N₂⁺Cl⁻（氯化重氮苯）
- Sandmeyer：C₆H₅N₂⁺ + CuCl → C₆H₅Cl + N₂↑
- 偶合：C₆H₅N₂⁺ + C₆H₅OH → C₆H₅-N=N-C₆H₄OH（橙红色）
:::
