## 一、单项选择题（共 10 题，每题 2 分，共 20 分）

### 第1题

:::callout{kind=note label="题目"}
1987 年召开的党的（　　），第一次对中国特色社会主义理论的主要内容作了系统概括。

A. 十二大

B. 十三大

C. 十四大

D. 十五大
:::

:::callout{kind=insight label="解析"}
**【答案】B**

1987 年 10 月，党的十三大召开，系统阐述了社会主义初级阶段理论，完整表述了党在社会主义初级阶段的基本路线，并概括了"一个中心、两个基本点"的核心内容，第一次对中国特色社会主义理论的主要内容作了系统概括。A 项十二大（1982）首次提出"建设有中国特色的社会主义"，但尚未形成系统理论概括；C 项十四大（1992）确立社会主义市场经济体制改革目标；D 项十五大（1997）正式把邓小平理论确立为党的指导思想。十三大是"系统概括"的关键节点。
:::

:::callout{kind=note label="知识卡片：邓小平理论形成的关键会议"}
| 会议 | 时间 | 核心贡献 |
|------|------|---------|
| 十二大 | 1982 | 首次提出"建设有中国特色的社会主义" |
| **十三大** | **1987** | **系统阐述初级阶段理论，概括基本路线，第一次系统概括中国特色社会主义理论** |
| 十四大 | 1992 | 确立社会主义市场经济体制改革目标 |
| 十五大 | 1997 | 正式确立邓小平理论为党的指导思想 |
:::

:::callout{kind=tip label="结论速记"}
系统概括中国特色社会主义理论 = 十三大（1987）。正确答案: B
:::

### 第2题

:::callout{kind=note label="题目"}
邓小平理论首要的基本的理论问题是（　　）。

A. 什么是社会主义，怎样建设社会主义

B. 什么是社会主义，为什么建设社会主义

C. 为什么建设社会主义，怎样建设社会主义

D. 为什么建设社会主义，建设什么样的社会主义
:::

:::callout{kind=insight label="解析"}
**【答案】A**

"什么是社会主义，怎样建设社会主义"是邓小平理论首要的基本的理论问题。这一问题的提出，源于对建国后 20 多年探索中经验教训的深刻总结——方向正确，但对社会主义本质认识不清，把许多束缚生产力的东西当作"社会主义原则"固守。A 项准确对应教材表述。B、C、D 项将"为什么建设"或"建设什么样的"混入题干，均非标准提法；"为什么"侧重动机，"建设什么样的"侧重目标形态，而邓小平理论的核心突破在于**本质论**（什么是）与**道路论**（怎样建）的统一。
:::

:::callout{kind=note label="知识卡片：首要基本理论问题的两层含义"}
- **什么是社会主义**：关键是认清社会主义本质——解放生产力，发展生产力，消灭剥削，消除两极分化，最终达到共同富裕
- **怎样建设社会主义**：在坚持社会主义基本制度基础上，走改革开放之路，建立社会主义市场经济体制
- **易混点**：不是"为什么建设"（动机问题），而是"是什么"和"怎么做"（本质与道路问题）
:::

:::callout{kind=tip label="结论速记"}
首要基本理论问题 = 什么是 + 怎样建设社会主义。正确答案: A
:::

### 第3题

:::callout{kind=note label="题目"}
党的十三大报告提出了党在社会主义初级阶段的基本路线："领导和团结全国各族人民，以经济建设为中心，坚持四项基本原则，坚持改革开放，自力更生，艰苦创业，为把我国建设成为富强、民主、文明的社会主义现代化国家而奋斗。"党的基本路线中，"自力更生，艰苦创业"是实现奋斗目标的（　　）。

A. 领导力量

B. 基本途径

C. 根本立足点

D. 实施纲领
:::

:::callout{kind=insight label="解析"}
**【答案】C**

基本路线包含五个要素：奋斗目标（富强民主文明的现代化国家）、基本途径（一个中心两个基本点）、领导力量（中国共产党）、依靠力量（全国各族人民）、根本立足点（自力更生，艰苦创业）。A 项"领导力量"对应"领导和团结全国各族人民"中的"领导"——中国共产党；B 项"基本途径"对应"以经济建设为中心，坚持四项基本原则，坚持改革开放"；C 项"自力更生，艰苦创业"体现改变不发达现状的需要和社会主义奋斗精神，是根本立足点；D 项"实施纲领"不是基本路线的标准表述要素。
:::

:::callout{kind=note label="知识卡片：基本路线五要素速记"}
| 要素 | 对应内容 |
|------|---------|
| 奋斗目标 | 富强、民主、文明的社会主义现代化国家 |
| 基本途径 | 一个中心（经济建设）+ 两个基本点（四项基本原则、改革开放） |
| 领导力量 | 中国共产党 |
| 依靠力量 | 全国各族人民 |
| **根本立足点** | **自力更生，艰苦创业** |
:::

:::callout{kind=tip label="结论速记"}
自力更生、艰苦创业 = 根本立足点；一个中心两个基本点 = 基本途径。正确答案: C
:::

### 第4题

:::callout{kind=note label="题目"}
1982 年邓小平在党的十二大开幕词中首次明确提出（　　）。

A. 社会主义初级阶段

B. 社会主义本质论

C. 中国特色社会主义

D. 社会主义市场经济
:::

:::callout{kind=insight label="解析"}
**【答案】C**

1982 年 9 月，邓小平在党的十二大开幕词中首次明确提出"建设有中国特色的社会主义"这一命题，成为整个理论和实践探索的主题。A 项"社会主义初级阶段"是 1987 年十三大系统阐述的，1981 年十一届六中全会决议已有"处于初级阶段"的表述但未作为全局国情；B 项社会主义本质论完整表述于 1992 年南方谈话；D 项社会主义市场经济是 1992 年十四大确立的改革目标。十二大的标志性贡献是"中国特色社会主义"命题的首次提出。
:::

:::callout{kind=note label="知识卡片：十二大的历史地位"}
- **核心命题**："把马克思主义的普遍真理同我国的具体实际结合起来，走自己的道路，建设有中国特色的社会主义"
- **与后续理论的关系**：十二大提出命题 → 十三大系统阐述初级阶段和基本路线 → 南方谈话回答本质问题 → 十四大确立市场经济目标 → 十五大确立指导思想
- **易混点**："初级阶段"在 1981 年决议中有提及，但"中国特色社会主义"是十二大首次明确提出
:::

:::callout{kind=tip label="结论速记"}
十二大开幕词 = 首次明确提出"中国特色社会主义"。正确答案: C
:::

### 第5题

:::callout{kind=note label="题目"}
邓小平理论形成的最重要的时代背景是（　　）。

A. 和平与发展成为时代的主题

B. 战争与革命成为时代的主题

C. 冷战的阴云仍然密布

D. 两极格局分化较为严重
:::

:::callout{kind=insight label="解析"}
**【答案】A**

邓小平理论是在和平与发展成为时代主题的历史条件下，在总结我国社会主义胜利和挫折的历史经验并借鉴其他社会主义国家兴衰成败历史经验的基础上形成和发展起来的。A 项准确。B 项"战争与革命"是列宁主义时代对 20 世纪初世界形势的判断，已不符合 20 世纪 70 年代以后的世界格局；C、D 项描述的是冷战时期国际格局的某些特征，但不是邓小平作出战略判断的"时代主题"——主题强调的是全局性、长期性的战略问题，即和平与发展。
:::

:::callout{kind=note label="知识卡片：时代主题判断的基本点"}
- **核心判断**：和平与发展是当今世界两大带有全球性的战略问题
- **与理论形成的关系**：时代主题为改革开放和现代化建设提供了有利的外部条件
- **易混点**："冷战阴云""两极格局"是具体国际形势，不等于"时代主题"；时代主题是对世界大势的战略性判断
:::

:::callout{kind=tip label="结论速记"}
邓小平理论时代背景 = 和平与发展成为时代主题。正确答案: A
:::

### 第6题

:::callout{kind=note label="题目"}
"一国两制"的伟大构想的提出首先是用于解决（　　），在实践中先后运用于解决（　　）（　　）。

A. 香港问题　澳门问题　台湾问题

B. 澳门问题　香港问题　台湾问题

C. 台湾问题　香港问题　澳门问题

D. 台湾问题　澳门问题　香港问题
:::

:::callout{kind=insight label="解析"}
**【答案】C**

"一国两制"构想主要是针对台湾问题提出的——1981 年叶剑英九条方针、1983 年"邓六条"均围绕台湾和平统一展开。但由于香港、澳门回归问题在时间上有紧迫性（英国、葡萄牙租约到期），构想率先在港澳成功实践：1997 年香港回归、1999 年澳门回归。因此顺序是：构想针对台湾 → 实践先香港 → 再澳门。A、B 项将构想提出对象搞错；D 项实践顺序错误（香港先于澳门回归）。
:::

:::callout{kind=note label="知识卡片：一国两制提出与实践顺序"}
| 阶段 | 对象 | 关键节点 |
|------|------|---------|
| 构想提出 | **台湾问题** | 1981 叶九条、1983 邓六条、1985 确立为基本国策 |
| 率先实践 | **香港** | 1984 中英联合声明，1997.7.1 回归 |
| 继而实践 | **澳门** | 1987 中葡联合声明，1999.12.20 回归 |
| 待解决 | 台湾 | 海峡两岸统一大业 |
:::

:::callout{kind=tip label="结论速记"}
一国两制：先针对台湾提出，先香港后澳门实践。正确答案: C
:::

### 第7题

:::callout{kind=note label="题目"}
邓小平理论的精髓是（　　）。

A. 解放思想，实事求是

B. "一个中心，两个基本点"

C. 坚持中国共产党的领导

D. 解放和发展生产力
:::

:::callout{kind=insight label="解析"}
**【答案】A**

精髓是贯穿理论始终、处于最本质位置的内容。解放思想、实事求是是邓小平理论的精髓，也是党的思想路线。A 项正确。B 项"一个中心，两个基本点"是社会主义初级阶段基本路线的核心内容，不是理论精髓；C 项党的领导是基本路线的领导力量，是政治保障而非精髓；D 项解放和发展生产力是社会主义本质论和根本任务的内容，不是精髓。精髓强调的是**认识论和方法论**层面——如何认识和改造世界。
:::

:::callout{kind=note label="知识卡片：精髓 vs 基本路线 vs 本质论"}
| 概念 | 内容 | 层面 |
|------|------|------|
| **精髓** | 解放思想，实事求是 | 思想路线、认识论 |
| 基本路线 | 一个中心，两个基本点 | 政治纲领 |
| 本质论 | 解放生产力、发展生产力、共同富裕 | 理论内核 |
| 首要问题 | 什么是社会主义，怎样建设社会主义 | 理论主题 |
:::

:::callout{kind=tip label="结论速记"}
邓小平理论精髓 = 解放思想 + 实事求是。正确答案: A
:::

### 第8题

:::callout{kind=note label="题目"}
党的十五大正式确立了邓小平理论作为我们党的指导思想，它的召开时间是（　　）。

A. 1982

B. 1987

C. 1992

D. 1997
:::

:::callout{kind=insight label="解析"}
**【答案】D**

1997 年 9 月，党的十五大正式把邓小平理论同马克思列宁主义、毛泽东思想一起确立为党的指导思想并写入党章。A 项 1982 是十二大；B 项 1987 是十三大；C 项 1992 是十四大——十四大对邓小平建设有中国特色社会主义理论的主要贡献作了系统概括，但尚未正式以"邓小平理论"命名并确立为指导思想。十五大是"正式确立"的标志性会议。
:::

:::callout{kind=note label="知识卡片：邓小平理论指导地位的确立过程"}
- **1982 十二大**：提出"建设有中国特色的社会主义"
- **1987 十三大**：系统概括中国特色社会主义理论
- **1992 十四大**：系统概括邓小平建设有中国特色社会主义理论的主要贡献
- **1997 十五大**：**正式确立"邓小平理论"为党的指导思想并写入党章**
:::

:::callout{kind=tip label="结论速记"}
邓小平理论正式确立为指导思想 = 十五大（1997）。正确答案: D
:::

### 第9题

:::callout{kind=note label="题目"}
我们党和国家生存发展的政治基石是（　　）。

A. 四项基本原则

B. 改革开放

C. 以经济建设为中心

D. 以人民为中心
:::

:::callout{kind=insight label="解析"}
**【答案】A**

四项基本原则——坚持社会主义道路，坚持人民民主专政，坚持中国共产党的领导，坚持马克思列宁主义毛泽东思想——是我们党和国家生存发展的政治基石，是立国之本。A 项正确。B 项改革开放是强国之路，是基本路线中"两个基本点"之一，提供发展动力；C 项以经济建设为中心是"一个中心"，回答根本任务；D 项"以人民为中心"是新时代的发展思想表述，不是基本路线中的标准用语。政治基石强调的是**方向性、制度性**的底线保障。
:::

:::callout{kind=note label="知识卡片：立国之本与强国之路"}
- **四项基本原则** = 立国之本 = 政治基石（保证社会主义方向）
- **改革开放** = 强国之路（提供发展动力）
- **以经济建设为中心** = 一个中心（明确根本任务）
- **关系**：四项基本原则为改革开放提供政治保证，改革开放为四项基本原则注入时代内涵，二者统一于基本路线
:::

:::callout{kind=tip label="结论速记"}
政治基石 = 四项基本原则（立国之本）。正确答案: A
:::

### 第10题

:::callout{kind=note label="题目"}
邓小平提出，四个现代化中，关键是（　　）的现代化。

A. 工业

B. 农业

C. 科学技术

D. 国防
:::

:::callout{kind=insight label="解析"}
**【答案】C**

1978 年 3 月，在全国科学大会上，邓小平指出："四个现代化，关键是科学技术的现代化，没有现代科学技术，就不可能有现代化的工业、农业、国防，就不可能有国民经济的高速度发展。"1988 年，他进一步提出"科学技术是第一生产力"。C 项正确。A、B、D 项工业、农业、国防的现代化都依赖科学技术作为先导和支撑，科学技术是"关键"——抓住了科技现代化，其他领域的现代化才有基础和动力。
:::

:::callout{kind=note label="知识卡片：科学技术是第一生产力"}
- **1978 年**：四个现代化关键是科学技术现代化
- **1988 年**：科学技术是第一生产力
- **逻辑链条**：科技现代化 → 带动工农业国防现代化 → 推动国民经济高速度发展
- **实践影响**：大批科技人员走上领导岗位，"863 计划"等科技战略相继推出
:::

:::callout{kind=tip label="结论速记"}
四化关键 = 科学技术现代化；科技是第一生产力。正确答案: C
:::

## 二、多项选择题（共 5 题，每题 2 分，共 10 分）

### 第11题

:::callout{kind=note label="题目"}
1992 年初，邓小平在南方谈话中对社会主义本质作出了总结性的理论概括。邓小平关于社会主义本质的概括，突出强调了（　　）。

A. 中国共产党的领导是中国特色社会主义最本质的特征

B. 解放和发展生产力在社会主义社会发展中的重要地位

C. 消灭剥削、消除两极分化，最终达到共同富裕的发展目标

D. 社会和谐是中国特色社会主义的本质属性
:::

:::callout{kind=insight label="解析"}
**【答案】BC**

1992 年南方谈话中，邓小平完整表述社会主义本质："解放生产力，发展生产力，消灭剥削，消除两极分化，最终达到共同富裕。"这一概括包含两个层面：生产力层面（解放和发展生产力）和生产关系层面（消灭剥削、消除两极分化、共同富裕）。B、C 项分别对应这两个层面，正确。A 项"党的领导是最本质的特征"是习近平新时代中国特色社会主义思想的重要论断，不是邓小平南方谈话的内容；D 项"社会和谐是本质属性"是十六大以来的表述。命题考查的是邓小平本人关于本质的"两个突出强调"。
:::

:::callout{kind=note label="知识卡片：社会主义本质的两个层面"}
| 层面 | 内容 | 对应选项 |
|------|------|---------|
| 生产力 | 解放生产力，发展生产力 | B |
| 生产关系 | 消灭剥削，消除两极分化，最终达到共同富裕 | C |
| **不属于邓小平本质论** | 党的领导是最本质特征（习近平） | A |
| **不属于邓小平本质论** | 社会和谐是本质属性（十六大） | D |
:::

:::callout{kind=tip label="结论速记"}
邓小平本质论突出强调：生产力（解放发展）+ 生产关系（共同富裕）。正确答案: BC
:::

### 第12题

:::callout{kind=note label="题目"}
以下关于社会主义市场经济理论的表述，说法正确的有（　　）。

A. 计划经济不等于社会主义，市场经济也不等于资本主义

B. 市场经济作为资源配置手段本身具有制度属性

C. 计划和市场都是经济手段，对经济活动的调节各有优劣

D. 坚持社会主义制度与市场经济的结合，是社会主义市场经济的特色所在、优势所在
:::

:::callout{kind=insight label="解析"}
**【答案】ACD**

1992 年南方谈话和十四大确立了社会主义市场经济理论。A 项正确——打破了"计划经济=社会主义、市场经济=资本主义"的传统观念。C 项正确——"计划和市场都是经济手段"，不是社会制度的本质区别。D 项正确——社会主义市场经济的"特色"和"优势"在于公有制为主体与市场经济相结合，发展成果由人民共享。B 项错误——南方谈话明确指出，计划多一点还是市场多一点，**不是**社会主义与资本主义的本质区别，市场经济作为资源配置手段**本身不具有制度属性**，制度属性来自它与什么基本制度相结合。
:::

:::callout{kind=note label="知识卡片：社会主义市场经济三个要点"}
1. **计划≠社会主义，市场≠资本主义**（A）
2. **计划和市场都是经济手段**（C），市场本身无制度属性（B 错）
3. **社会主义制度 + 市场经济结合 = 特色与优势**（D）
:::

:::callout{kind=tip label="结论速记"}
市场经济无制度属性；社会主义市场经济的优势在于制度与市场结合。正确答案: ACD
:::

以下哪些属于社会主义初级阶段的特征（　　）。

A. 是逐步摆脱贫穷、摆脱落后的阶段

B. 是由农业人口占多数的手工劳动为基础的农业国，逐步变为非农产业人口占多数的现代化的工业国的阶段

1987 年十三大报告对社会主义初级阶段基本特征作了五点概括，1997 年十五大进一步扩展为九大特征。题干四项均出自十三大报告的经典表述：A 对应"逐步摆脱贫穷、摆脱落后"；B 对应农业国向工业国转变；C 对应自然经济半自然经济向商品经济高度发达转变；D 对应通过改革和探索建立充满活力的体制。四项全部正确，体现了初级阶段在经济结构、产业结构、经济形态和体制改革等多方面的历史方位。

:::

十三大初级阶段特征五点，题干四项全对。正确答案: ABCD

### 第14题

:::callout{kind=note label="题目"}
以下关于社会主义改革开放理论的表述，说法正确的有（　　）。

A. 改革是一场深刻的社会变革，是中国的第二次革命，是实现中国现代化的必由之路

B. 改革是社会主义社会发展的直接动力

C. 改革是一项崭新的事业，是一个大试验

D. 对外开放要吸引先进外资，是针对西方资本主义发达国家的开放
:::

:::callout{kind=insight label="解析"}
**【答案】ABC**

A 项正确——邓小平明确指出改革是中国的第二次革命，是从广度和深度而言的深刻社会变革，是实现现代化的必由之路。B 项正确——改革是社会主义社会发展的直接动力，通过变革束缚生产力的体制来解放和发展生产力。C 项正确——改革没有现成模式可循，是一项崭新的事业和大试验，需要在实践中探索。D 项错误——对外开放是全方位的，面向所有国家，既包括西方发达国家，也包括周边国家和其他发展中国家，不是"针对"西方资本主义发达国家。对外开放的基本逻辑是引进资金、技术、管理经验，利用国际国内两个市场、两种资源。
:::

:::callout{kind=note label="知识卡片：改革开放理论要点"}
- **改革性质**：第二次革命（A），社会主义制度的自我完善
- **改革动力**：社会主义社会发展的直接动力（B）
- **改革特点**：崭新事业、大试验（C），"摸着石头过河"
- **开放范围**：全方位、多层次、宽领域，**不是**仅对西方国家（D 错）
:::

:::callout{kind=tip label="结论速记"}
改革 = 第二次革命 + 直接动力 + 大试验；开放是全方位而非只对西方。正确答案: ABC
:::

### 第15题

:::callout{kind=note label="题目"}
邓小平对时代主题的判断，其基本点是（　　）。

A. 世界大战在一个相当长的时期内可以避免

B. 和平与发展是当今世界两大带有全球性的战略问题

C. 世界和平是促进各国共同发展的前提条件，各国的共同发展则是保持世界和平的重要基础

D. 和平与发展成为时代主题，并不意味着这两个问题已经解决
:::

:::callout{kind=insight label="解析"}
**【答案】ABCD**

邓小平关于时代主题判断包含四个基本点，A、B、C、D 四项全部正确。A 项——世界大战在一个相当长的时期内可以避免，为改革开放提供了相对和平的国际环境。B 项——和平与发展是当今世界两大带有全球性的战略问题，是时代主题的核心判断。C 项——和平是发展的前提，发展是和平的基础，二者相互依存、相互促进。D 项——成为时代主题不等于问题已经解决，当今世界仍面临局部战争、发展不平衡等挑战，这一判断体现了辩证思维，避免盲目乐观或悲观。
:::

:::callout{kind=note label="知识卡片：时代主题判断四基本点"}
1. **大战可避**（A）：相当长时期内世界大战可以避免
2. **两大战略问题**（B）：和平与发展是全球性战略问题
3. **辩证关系**（C）：和平是发展前提，发展是和平基础
4. **问题未解**（D）：成为主题 ≠ 已经解决
:::

:::callout{kind=tip label="结论速记"}
时代主题四基本点：大战可避、两大战略问题、和平发展互为基础、问题尚未解决。正确答案: ABCD
:::
