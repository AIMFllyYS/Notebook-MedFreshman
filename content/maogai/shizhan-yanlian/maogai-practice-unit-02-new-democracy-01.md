## 一、单项选择题（每题 2 分，共 20 分）

### 第 1 题

:::callout{kind=note label="题目"}
在半殖民地半封建的近代中国，（　　）是各种矛盾中最主要的矛盾。

A. 地主阶级和资产阶级的矛盾　　B. 帝国主义和中华民族的矛盾

C. 封建主义和人民大众的矛盾　　D. 传统文化和西方文化的矛盾
:::

:::callout{kind=insight label="解析"}
**【答案】B**


半殖民地半封建社会的性质，决定了近代中国社会存在两对主要矛盾：帝国主义和中华民族的矛盾、封建主义和人民大众的矛盾。其中，**帝国主义和中华民族的矛盾是最主要的矛盾**——帝国主义把中国变成半殖民地，是近代中国一切灾难的总根源。

- A 错：地主阶级与资产阶级的矛盾不是近代中国社会的主要矛盾。
- C 错：封建主义与人民大众的矛盾是主要矛盾之一，但非最主要。
- D 错：文化冲突不是社会主要矛盾的正确表述。
:::

:::callout{kind=note label="知识卡片：近代中国社会主要矛盾"}
| 矛盾 | 性质 | 地位 |
|------|------|------|
| 帝国主义和中华民族的矛盾 | 民族矛盾 | **最主要矛盾** |
| 封建主义和人民大众的矛盾 | 阶级矛盾 | 主要矛盾之一 |

半殖民地半封建社会的本质：财富和权力被帝国主义与封建主义所控制。
:::

:::callout{kind=tip label="结论速记"}
近代中国最主要矛盾 = **帝国主义和中华民族的矛盾**；两对主要矛盾中，民族矛盾居首位。
:::

### 第 2 题

:::callout{kind=note label="题目"}
以（　　）为标志，中国进入新民主主义革命阶段。

A. 戊戌变法　　B. 辛亥革命

C. 五四运动　　D. 北伐战争
:::

:::callout{kind=insight label="解析"}
**【答案】C**


在俄国十月革命影响下，以**五四运动**为标志，中国无产阶级开始作为独立的政治力量登上历史舞台，马克思列宁主义逐步成为指导思想，中国革命进入新民主主义革命阶段。

- A 错：戊戌变法是改良主义运动，未改变革命性质。
- B 错：辛亥革命属于旧式资产阶级民主革命范畴。
- D 错：北伐战争发生在新民主主义革命阶段之内，不是进入该阶段的标志。
:::

:::callout{kind=note label="知识卡片：旧民主主义向新民主主义转变"}
| 标志事件 | 关键变化 |
|----------|----------|
| 十月革命 | 世界历史方向转变，中国革命成为世界无产阶级社会主义革命一部分 |
| **五四运动** | 无产阶级独立登上政治舞台，马克思主义广泛传播，进入新民主主义革命阶段 |

旧民主主义革命：鸦片战争至辛亥革命；新民主主义革命：五四运动至新中国成立。
:::

:::callout{kind=tip label="结论速记"}
中国进入新民主主义革命阶段的标志 = **五四运动**（无产阶级独立登上政治舞台 + 马克思主义成为指导思想）。
:::

### 第 3 题

:::callout{kind=note label="题目"}
关于新民主主义革命的总路线，下列理解中错误的是（　　）。

A. 新民主主义革命是无产阶级领导的革命

B. 新民主主义革命是资产阶级领导的革命

C. 新民主主义革命是人民大众的革命

D. 新民主主义革命反对帝国主义、封建主义和官僚资本主义的革命
:::

:::callout{kind=insight label="解析"}
**【答案】B**


1948 年毛泽东在晋绥干部会议上完整表述的总路线是：**无产阶级领导的，人民大众的，反对帝国主义、封建主义和官僚资本主义的革命**。区别新旧民主主义革命的根本标志，就是革命领导权是否掌握在无产阶级手中。

- A、C、D 均正确：分别对应领导力量、革命动力、革命对象。
- B 错：资产阶级领导的革命是旧民主主义革命，新民主主义革命必须由无产阶级及其政党——中国共产党领导。
:::

:::callout{kind=note label="知识卡片：新民主主义革命总路线"}
**完整表述**：无产阶级领导的，人民大众的，反对帝国主义、封建主义和官僚资本主义的革命。

| 要素 | 内容 |
|------|------|
| 领导力量 | 无产阶级（中国共产党） |
| 革命动力 | 人民大众（无产阶级、农民、小资产阶级、民族资产阶级） |
| 革命对象 | 帝国主义、封建主义、官僚资本主义（三座大山） |
| 革命性质 | 资产阶级民主主义革命 |
| 革命前途 | 社会主义 |
:::

:::callout{kind=tip label="结论速记"}
新民主主义革命总路线核心：**无产阶级领导** + 人民大众 + 反帝反封建反官僚资本；领导权问题是区分新旧民主主义革命的根本标志。
:::

### 第 4 题

:::callout{kind=note label="题目"}
在不同历史阶段，随着社会主要矛盾的变化，中国革命的主要对象有所不同，其具体体现不包括（　　）。

A. 在国共合作的大革命时期，革命的主要对象是帝国主义支持下的北洋军阀

B. 在土地革命战争时期，革命的主要对象是民族资产阶级

C. 在抗日战争时期，革命的主要对象是日本帝国主义

D. 在解放战争时期，革命的主要对象是美帝国主义支持下的国民党反动派
:::

:::callout{kind=insight label="解析"}
**【答案】B**


中国革命对象总体上是三座大山，但不同阶段主要打击对象有所侧重：

- 大革命时期（1924—1927）：北洋军阀
- 土地革命战争时期（1927—1937）：**国民党新军阀**（而非民族资产阶级）
- 抗日战争时期（1937—1945）：日本帝国主义
- 解放战争时期（1946—1949）：美帝支持下的国民党反动派

- B 错：土地革命战争时期主要对象是国民党新军阀，民族资产阶级是革命动力之一，不是主要革命对象。
:::

:::callout{kind=note label="知识卡片：各阶段革命主要对象"}
| 历史时期 | 主要革命对象 |
|----------|-------------|
| 大革命时期 | 帝国主义支持下的北洋军阀 |
| 土地革命战争时期 | **国民党新军阀** |
| 抗日战争时期 | 日本帝国主义 |
| 解放战争时期 | 美帝国主义支持下的国民党反动派 |

注意：民族资产阶级是革命动力，中国共产党对其采取保护政策，不是革命主要对象。
:::

:::callout{kind=tip label="结论速记"}
土地革命战争时期主要对象是**国民党新军阀**，不是民族资产阶级；各阶段对象随主要矛盾变化而调整。
:::

### 第 5 题

:::callout{kind=note label="题目"}
新民主主义革命与旧民主主义革命相比有新的内容和特点，其集中表现不包括（　　）。

A. 新民主主义革命的领导力量是中国无产阶级及其先锋队——中国共产党

B. 新民主主义革命的指导思想是马克思列宁主义

C. 新民主主义革命是世界无产阶级社会主义革命的一部分

D. 新民主主义革命的前途是资本主义
:::

:::callout{kind=insight label="解析"}
**【答案】D**


新民主主义革命"新"在四处：①领导力量——无产阶级及其先锋队中国共产党；②指导思想——马克思列宁主义；③革命前途——**社会主义**而非资本主义；④革命范畴——世界无产阶级社会主义革命的一部分。

- A、B、C 均为"新"的集中表现。
- D 错：新民主主义革命的前途是社会主义，建立各革命阶级联合专政的新民主主义共和国，为向社会主义过渡做准备。
:::

:::callout{kind=note label="知识卡片：新民主主义革命的新特征"}
| 比较维度 | 旧民主主义革命 | 新民主主义革命 |
|----------|---------------|---------------|
| 领导力量 | 资产阶级 | **无产阶级（中国共产党）** |
| 指导思想 | 资产阶级思想 | **马克思列宁主义** |
| 革命前途 | 资本主义 | **社会主义** |
| 世界意义 | 旧的世界资产阶级革命一部分 | **世界无产阶级社会主义革命一部分** |
:::

:::callout{kind=tip label="结论速记"}
新民主主义革命"四新"：无产阶级领导、马列主义指导、前途社会主义、属于世界无产阶级革命一部分；**不是**走资本主义道路。
:::

### 第 6 题

:::callout{kind=note label="题目"}
新民主主义革命与社会主义革命性质不同，其表现不包括（　　）。

A. 新民主主义革命在政治上争取和联合民族资产阶级去反对共同的敌人

B. 新民主主义革命在经济上保护民族工商业，容许有利于国计民生的私人资本主义发展

C. 新民主主义革命要建立的是资产阶级专政，而不是无产阶级专政

D. 新民主主义革命仍然属于资产阶级民主主义革命的范畴
:::

:::callout{kind=insight label="解析"}
**【答案】C**


新民主主义革命与社会主义革命的区别：新民主主义革命仍属资产阶级民主主义革命范畴，政治上联合民族资产阶级，经济上保护民族工商业；要建立的是**无产阶级领导的各革命阶级联合专政**，而非资产阶级专政，也非无产阶级专政。

- A、B、D 正确：分别体现政治联合、经济保护、革命性质定位。
- C 错：新民主主义革命要建立的是各革命阶级联合专政的新民主主义共和国，不是资产阶级专政；无产阶级专政的社会主义共和国是革命前途，不是新民主主义革命直接建立的目标。
:::

:::callout{kind=note label="知识卡片：新民主主义革命与社会主义革命的区别"}
| 比较项 | 新民主主义革命 | 社会主义革命 |
|--------|---------------|-------------|
| 革命性质 | 资产阶级民主主义革命 | 无产阶级社会主义革命 |
| 政权形式 | 各革命阶级**联合专政** | **无产阶级专政** |
| 经济政策 | 保护民族工商业 | 消灭资本主义剥削制度 |
| 相互关系 | 上篇（必要准备） | 下篇（必然趋势） |

两革命紧密衔接，中间不容横插资产阶级专政。
:::

:::callout{kind=tip label="结论速记"}
新民主主义革命建立**各革命阶级联合专政**（非资产阶级专政、非无产阶级专政）；与社会主义革命是"上篇与下篇"的关系。
:::

### 第 7 题

:::callout{kind=note label="题目"}
关于新民主主义的政治纲领，下列理解中错误的是（　　）。

A. 推翻帝国主义和封建主义的统治

B. 建立一个无产阶级领导的、以工农联盟为基础的、各革命阶级联合专政的新民主主义的共和国

C. 新民主主义共和国不同于欧美式的资产阶级专政的共和国

D. 新民主主义共和国和苏联式的无产阶级专政的社会主义共和国完全相同
:::

:::callout{kind=insight label="解析"}
**【答案】D**


新民主主义政治纲领：推翻帝国主义和封建主义统治，建立无产阶级领导的、以工农联盟为基础的、各革命阶级联合专政的新民主主义共和国。这一国体既不同于欧美式资产阶级专政共和国，也不同于苏联式无产阶级专政的社会主义共和国，是过渡性的国家形式。

- A、B、C 均正确。
- D 错：新民主主义共和国与苏联式社会主义共和国**有区别**——前者是各革命阶级联合专政的过渡形式，后者是无产阶级专政的社会主义国家。
:::

:::callout{kind=note label="知识卡片：新民主主义政治纲领"}
**国体**：无产阶级领导的、以工农联盟为基础的、各革命阶级联合专政

**政体**：民主集中制的人民代表大会制度

**三种国家形式对比**：
- （甲）资产阶级专政的共和国——在中国行不通
- （乙）无产阶级专政的社会主义共和国——将来目标
- （丙）**各革命阶级联合专政的共和国**——新民主主义国体
:::

:::callout{kind=tip label="结论速记"}
新民主主义共和国 = 各革命阶级联合专政 + 人民代表大会制；既非欧美资产阶级共和国，也非苏联式社会主义共和国。
:::

### 第 8 题

:::callout{kind=note label="题目"}
关于新民主主义的经济纲领，下列描述中错误的是（　　）。

A. 消灭民族资产阶级

B. 没收官僚资产阶级的垄断资本归新民主主义的国家所有

C. 保护民族工商业

D. 没收封建地主阶级的土地归农民所有
:::

:::callout{kind=insight label="解析"}
**【答案】A**


新民主主义经济纲领三项：**没收封建地主土地归农民所有**（主要内容）、**没收官僚资本归新民主主义国家所有**（题中应有之义）、**保护民族工商业**（极具特色的一项）。

- B、C、D 均为经济纲领的正确内容。
- A 错：新民主主义革命的对象是帝国主义、封建主义和官僚资本主义，**不是**一般地消灭资本主义和民族资产阶级；对民族资产阶级在经济上实行保护政策。
:::

:::callout{kind=note label="知识卡片：新民主主义经济纲领"}
| 内容 | 要点 |
|------|------|
| 没收封建地主土地归农民所有 | 革命主要内容，"耕者有其田" |
| 没收官僚资本归国家所有 | 具有新民主主义革命和社会主义革命双重性质 |
| **保护民族工商业** | 极具特色；民族资产阶级是革命动力之一 |

"两没收一保护"——只推翻三座大山，不破坏尚能参加反帝反封建的资本主义成分。
:::

:::callout{kind=tip label="结论速记"}
新民主主义经济纲领 = **两没收一保护**（土地、官僚资本、民族工商业）；消灭民族资产阶级是错误表述。
:::

### 第 9 题

:::callout{kind=note label="题目"}
关于新民主主义文化，下列描述中错误的是（　　）。

A. 反对帝国主义压迫，主张中华民族的尊严和独立

B. 尊重中国的历史，反对民族虚无主义

C. 反对封建时代创造的一切文化

D. 以人民群众的实践作为创作的源泉，坚持为人民大众服务
:::

:::callout{kind=insight label="解析"}
**【答案】C**


新民主主义文化是**民族的、科学的、大众的**文化，即无产阶级领导的人民大众的反帝反封建的文化。

- A 对：体现"民族的"——内容反对帝国主义压迫，主张民族尊严和独立。
- B 对：体现"科学的"——尊重历史，反对民族虚无主义。
- D 对：体现"大众的"——为绝大多数人民服务，以人民实践为创作源泉。
- C 错：新民主主义文化反对封建**思想和迷信思想**，对封建时代创造的文化应**剔除封建糟粕、吸收民主精华**，而非反对一切封建时代文化。
:::

:::callout{kind=note label="知识卡片：新民主主义文化纲领"}
| 特征 | 内涵 |
|------|------|
| **民族的** | 反对帝国主义压迫，有中国作风和中国气派；吸收外国进步文化 |
| **科学的** | 反对封建思想和迷信；剔除封建糟粕、吸收民主精华；反对民族虚无主义 |
| **大众的** | 为工人阶级和劳动群众服务；文化普及于大众又提高大众 |

文化革命是革命总战线中的一条必要和重要的战线。
:::

:::callout{kind=tip label="结论速记"}
新民主主义文化 = 民族的 + 科学的 + 大众的；对传统文化应**去糟粕取精华**，不是反对一切封建时代文化。
:::

### 第 10 题

:::callout{kind=note label="题目"}
中国革命必须走农村包围城市、武装夺取政权的道路，是由中国所处的时代特点和具体国情决定的，其具体表现不包括（　　）。

A. 中国的无产阶级根本不可能像在资本主义国家那样，先在城市经过长期的、公开的合法斗争，然后再组织武装起义，夺取政权

B. 中国革命的主要斗争形式只能是武装斗争，以革命的武装消灭反革命的武装，相应的主要组织形式必然是军队

C. 必须充分地发动农民，否则就无法摧毁帝国主义和封建地主阶级反动统治的基础

D. 农民作为同盟军是不可靠的，这就要求无产阶级及其政党必须深入农村，从解决农民的土地问题入手
:::

:::callout{kind=insight label="解析"}
**【答案】D**


中国革命走农村包围城市道路的必然性：内无民主制度、外无民族独立，不可能像资本主义国家那样先在城市长期合法斗争；主要斗争形式只能是武装斗争；农民占人口绝大多数，是可靠同盟军和革命主力军，必须充分发动农民。

- A、B、C 均为道路必然性的正确表述。
- D 错：农民是**可靠的**同盟军和革命主力军，不是"不可靠的"；党深入农村、解决土地问题，正是因为农民是中国革命的基本问题和主要动力。
:::

:::callout{kind=note label="知识卡片：农村包围城市道路的必然性"}
**时代特点**：内无民主、外无独立 → 不可能城市长期合法斗争 → 只能武装斗争

**具体国情**：农业大国，农民占人口 80% 以上 → 必须发动农民、深入农村

**三者关系**（道路核心内容）：
- **土地革命**——基本内容
- **武装斗争**——主要形式
- **农村革命根据地建设**——战略阵地
:::

:::callout{kind=tip label="结论速记"}
农村包围城市道路：农民是**可靠**同盟军（非不可靠）；处理好土地革命、武装斗争、根据地建设三者关系。
:::

在领导农村革命根据地的斗争实践中，毛泽东相继写下了（　　）等文章，提出了"工农武装割据"思想，初步形成了农村包围城市的革命道路理论。

A. 《中国的红色政权为什么能够存在?》

B. 《井冈山的斗争》

C. 《星星之火，可以燎原》

D. 《论持久战》
:::

:::callout{kind=insight label="解析"}
**【答案】ABC**


井冈山斗争时期，毛泽东相继撰写《中国的红色政权为什么能够存在？》《井冈山的斗争》《星星之火，可以燎原》，系统阐述"工农武装割据"思想，初步形成农村包围城市道路理论。

- A、B、C 正确：三篇均为井冈山时期论述红色政权和革命道路的核心文献。
- D 错：《论持久战》写于抗日战争时期（1938 年），论述的是抗日战争战略方针，不属于农村包围城市道路理论形成时期的著作。

1927 年秋收起义后，毛泽东创建井冈山革命根据地，工作重心开始转向农村。
:::

:::callout{kind=tip label="结论速记"}
"工农武装割据"三篇文献 = **《中国的红色政权为什么能够存在?》+《井冈山的斗争》+《星星之火，可以燎原》**；《论持久战》属抗战时期。

下列各项中，不能完成反帝反封建的革命任务的有（　　）。

A. 不触动封建根基的自强运动和改良主义

B. 旧式的农民战争

C. 照搬西方资本主义的其他种种方案

D. 资产阶级革命派领导的民主革命
:::

:::callout{kind=insight label="解析"}
**【答案】ABCD**


旧民主主义革命的失败证明，以下各种方案均不能完成反帝反封建的历史任务：

- A 对：洋务运动、戊戌变法等不触动封建根基的自强运动和改良主义。
- B 对：太平天国运动等旧式农民战争，缺乏科学理论指导和先进阶级领导。
- C 对：照搬西方资本主义的种种方案，不符合中国国情。
- D 对：辛亥革命等资产阶级革命派领导的民主革命，未能改变中国半殖民地半封建的社会性质。

这些失败呼唤新的阶级及其政党、新的革命理论——新民主主义革命理论应运而生。
:::

:::callout{kind=note label="知识卡片：旧民主主义革命的失败"}
| 尝试 | 代表事件 | 失败原因 |
|------|----------|----------|
| 自强运动和改良主义 | 洋务运动、戊戌变法 | 不触动封建根基 |
| 旧式农民战争 | 太平天国运动 | 缺乏科学理论和先进阶级领导 |
| 资产阶级民主革命 | 辛亥革命 | 未能完成反帝反封建任务，果实被窃取 |
| 照搬西方方案 | 各种改良方案 | 不符合中国半殖民地半封建国情 |

历史结论：只有无产阶级领导的新民主主义革命才能完成反帝反封建任务。
:::

:::callout{kind=tip label="结论速记"}
旧民主主义革命各种方案（改良、旧式农民战争、照搬西方、资产阶级革命派）**均不能**完成反帝反封建任务。

### 第 13 题

:::callout{kind=note label="题目"}
中国革命走农村包围城市、武装夺取政权的道路，根本在于处理好（　　）之间的关系。

A. 土地革命

B. 武装斗争

C. 议会民主斗争

D. 农村革命根据地建设
:::

:::callout{kind=insight label="解析"}
**【答案】ABD**


农村包围城市道路的核心内容，是处理好**土地革命、武装斗争、农村革命根据地建设**三者之间的关系：

- **土地革命**——中国革命的基本内容
- **武装斗争**——中国革命的主要形式，是农村根据地建设和土地革命的强有力保证
- **农村革命根据地建设**——中国革命的战略阵地，是进行武装斗争和开展土地革命的依托

- C 错：近代中国内无民主制度，不存在通过议会民主斗争夺取政权的可能，这不是中国革命道路的内容。

| 要素 | 定位 | 相互关系 |
|------|------|----------|

| 武装斗争 | 主要形式 | 保障土地革命和根据地建设 |
| 农村革命根据地建设 | 战略阵地 | 开展土地革命和武装斗争的依托 |

三者密切结合、有机统一，是中国共产党领导下的革命实践结晶。
:::

:::callout{kind=tip label="结论速记"}

:::

### 第 14 题

:::callout{kind=note label="题目"}
新民主主义革命时期，中国共产党领导的统一战线，先后经历了（　　）等几个时期，为新民主主义革命的胜利作出了重要贡献。

A. 第一次国共合作的统一战线

B. 工农民主统一战线

C. 抗日民族统一战线

D. 人民民主统一战线
:::

:::callout{kind=insight label="解析"}
**【答案】ABCD**


新民主主义革命时期，统一战线先后经历四个发展阶段：

- A 对：**第一次国共合作的统一战线**（1924—1927），国民革命联合战线，目标是打倒列强除军阀。
- B 对：**工农民主统一战线**（1927—1937），大革命失败后联盟范围缩小为工人、农民和小资产阶级。
- C 对：**抗日民族统一战线**（1937—1945），中日民族矛盾上升为主要矛盾，实现第二次国共合作。
- D 对：**人民民主统一战线**（1946—1949），解放战争时期范围最广泛，团结一切反蒋力量。

四个阶段联盟范围随主要矛盾变化而调整，体现了统一战线的灵活策略。
:::

:::callout{kind=note label="知识卡片：统一战线四个发展阶段"}
| 阶段 | 时期 | 联盟范围 | 主要任务 |
|------|------|----------|----------|
| 第一次国共合作 | 1924—1927 | 工、农、小资产阶级、民族资产阶级 | 反帝反军阀 |
| 工农民主 | 1927—1937 | 工人、农民、小资产阶级 | 土地革命 |
| 抗日民族 | 1937—1945 | 除汉奸外一切抗日力量 | 全民族抗战 |
| 人民民主 | 1946—1949 | 最广泛的反蒋联盟 | 推翻国民党统治 |

统一战线内部包含两个联盟：工农联盟（基础）+ 与非劳动人民的联盟（必不可少）。
:::

:::callout{kind=tip label="结论速记"}
新民主主义革命统一战线四阶段 = **第一次国共合作 → 工农民主 → 抗日民族 → 人民民主**；联盟范围随主要矛盾灵活调整。
:::

### 第 15 题

:::callout{kind=note label="题目"}
党在领导新民主主义革命的过程中，逐步形成了（　　）的优良作风，这是中国共产党区别于其他任何政党的显著标志。

A. 理论联系实际

B. 密切联系群众

C. 批评与自我批评相结合

D. 教条主义
:::

:::callout{kind=insight label="解析"}
**【答案】ABC**


1945 年党的七大上，毛泽东在《论联合政府》中概括了党的三大优良作风：**理论联系实际、密切联系群众、批评与自我批评相结合**。这是中国共产党区别于其他任何政党的显著标志，也是党战胜敌人的重要法宝之一（与统一战线、武装斗争合称"三大法宝"中的党的建设方面）。

- A、B、C 正确：三大优良作风的完整表述。
- D 错：教条主义是党在革命过程中需要克服的错误倾向（如延安整风反对的对象），不是党的优良作风。
:::

:::callout{kind=note label="知识卡片：党的三大优良作风与三大法宝"}
**三大优良作风**（1945 七大《论联合政府》）：
1. 理论联系实际
2. 密切联系群众
3. 批评与自我批评相结合

**三大法宝**（1939《〈共产党人〉发刊词》）：
1. 统一战线
2. 武装斗争
3. 党的建设

三大作风是党的建设的重要内容，是党区别于其他政党的显著标志。
:::

:::callout{kind=tip label="结论速记"}
党的三大优良作风 = **理论联系实际 + 密切联系群众 + 批评与自我批评**；教条主义是错误倾向，不是优良作风。
:::
