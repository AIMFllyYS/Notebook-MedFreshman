> 题型：单项选择题 10 题（每题 2 分）+ 多项选择题 5 题（每题 2 分）　满分：30 分

## 一、单项选择题（共 10 题，每题 2 分，共 20 分）

### 第1题

:::callout{kind=note label="题目"}
（　　）是我国从新民主主义到社会主义的过渡时期。

A. 从全面内战爆发到中华人民共和国成立

B. 从中华人民共和国成立到抗美援朝战争胜利

C. 从中华人民共和国成立到社会主义改造基本完成

D. 从解放战争胜利到土地改革基本完成
:::

:::callout{kind=insight label="解析"}
**【答案】C**

过渡时期是党在过渡时期总路线表述中的核心时间概念。1953 年 12 月形成的完整表述明确：从中华人民共和国成立到社会主义改造基本完成，这是一个过渡时期。A 项把起点定在全面内战爆发，混淆了民主革命与社会主义革命；B 项以抗美援朝胜利为终点，只是过渡期内的一项重大任务，不能界定整个过渡时期；D 项以土地改革完成为界，土地改革属于民主革命遗留任务，1952 年前后已基本完成，但向社会主义的过渡并未结束。
:::

:::callout{kind=note label="知识卡片：过渡时期的时间界定"}
- **起止**：1949 年新中国成立 → 1956 年底三大改造基本完成
- **社会性质**：新民主主义社会，属于社会主义体系，不是独立社会形态
- **核心任务**：逐步实现社会主义工业化，逐步完成对农业、手工业和资本主义工商业的社会主义改造
- **易混点**：1952 年是国民经济恢复、民主革命遗留任务完成的重要节点，但不是过渡时期终点
:::

:::callout{kind=tip label="结论速记"}
过渡时期 = 建国到三大改造基本完成。正确答案: C
:::

### 第2题

:::callout{kind=note label="题目"}
在新民主主义时期，随着土地改革的基本完成，（　　）逐步成为我国社会的主要矛盾。

A. 地主阶级和资产阶级的矛盾

B. 帝国主义和中华民族的矛盾

C. 工人阶级和资产阶级的矛盾

D. 封建主义和人民大众的矛盾
:::

:::callout{kind=insight label="解析"}
**【答案】C**

新民主主义社会三种基本经济成分（社会主义经济、个体经济、资本主义经济）对应三大阶级力量，其矛盾集中表现为社会主义道路与资本主义道路之争，实质是工人阶级与资产阶级的矛盾。土地改革完成后，地主阶级作为整体已被消灭，A、D 所反映的封建主义矛盾不再占据主要地位；B 项是近代中国社会的主要矛盾之一，但新中国成立、废除不平等条约后，帝国主义与中华民族的矛盾已不再是国内主要矛盾。
:::

:::callout{kind=note label="知识卡片：主要矛盾的转化"}
| 阶段 | 主要矛盾 | 说明 |
|------|---------|------|
| 土地改革前 | 地主阶级与农民阶级 | 封建剥削关系 |
| 土地改革后 | 工人阶级与资产阶级 | 决定向社会主义还是资本主义发展 |
| 注意 | 民族资产阶级具有两面性 | 矛盾既有对抗性，也有非对抗性一面 |
:::

:::callout{kind=tip label="结论速记"}
土改完成 → 主要矛盾转为工人阶级与资产阶级。正确答案: C
:::

### 第3题

:::callout{kind=note label="题目"}
到（　　）年，我国国民经济得以恢复，民主革命遗留任务已经完成，经济、政治及社会面貌发生巨大变化。

A. 1949

B. 1952

C. 1954

D. 1956
:::

:::callout{kind=insight label="解析"}
**【答案】B**

1949—1952 年，党领导人民集中力量恢复国民经济，完成土地改革、镇压反革命、"三反""五反"、抗美援朝等任务，没收官僚资本建立国营经济，为向社会主义过渡奠定基础。到 1952 年底，这些任务基本完成，社会面貌焕然一新，党由此重新思考向社会主义过渡的时间和步骤，1953 年正式提出过渡时期总路线。1954 年宪法通过、1956 年三大改造完成，均晚于"恢复与遗留任务完成"这一节点。
:::

:::callout{kind=note label="知识卡片：1949—1952 年的主要成就"}
- **经济**：没收官僚资本，建立社会主义国营经济；调整工商业，发展初级国家资本主义
- **农村**：土地改革完成，互助合作运动开始发展
- **政治社会**：镇压反革命、"三反""五反"、抗美援朝胜利
- **1952 年的意义**：恢复任务完成 → 党判断"开始向社会主义过渡的时机已经到来"
:::

:::callout{kind=tip label="结论速记"}
国民经济恢复、民主革命遗留任务完成的关键年份是 1952 年。正确答案: B
:::

### 第4题

:::callout{kind=note label="题目"}
党在过渡时期总路线的主要内容被概括为"一化三改"，其中"三改"不包括（　　）。

A. 对个体农业的社会主义改造

B. 对手工业的社会主义改造

C. 对资本主义工商业的社会主义改造

D. 对军工业的社会主义改造
:::

:::callout{kind=insight label="解析"}
**【答案】D**

"一化三改"中，"一化"是社会主义工业化（主体），"三改"是对农业、手工业和资本主义工商业的社会主义改造（两翼）。三者相互联系、不可分离：工业化需要集中资源，三大改造变私有为公有，为工业化创造物质条件。D 项"对军工业的社会主义改造"是干扰项——军工业属于重工业建设范畴，归"一化"而非"三改"；"三改"针对的是占很大比重的个体私有经济和资本主义经济。
:::

:::callout{kind=note label="知识卡片：一体两翼"}
- **主体（一化）**：逐步实现国家的社会主义工业化
- **两翼（三改）**：农业、手工业、资本主义工商业的社会主义改造
- **关系**：建设与改造同时并举，相辅相成
- **记忆口诀**：一化是鸟身，三改是双翼，缺一则飞不起来
:::

:::callout{kind=tip label="结论速记"}
三改 = 农 + 手 + 资，没有"军工业改造"。正确答案: D
:::

### 第5题

:::callout{kind=note label="题目"}
关于适合我国情况的农业社会主义改造道路，下列描述中错误的是（　　）。

A. 对农民不能采取剥夺的办法，只能引导、说服和教育，使其自愿地走合作化的道路

B. 只需要发展农民在互助合作上的积极性，不需要关注他们在个体经济上的积极性

C. 依靠贫下中农，发展互助合作，由逐步限制到最后消灭富农剥削

D. 高级社实行生产资料农民集体所有、按劳付酬，取消土地分红
:::

:::callout{kind=insight label="解析"}
**【答案】B**

农业改造的第一个特点就是"充分调动两个积极性"——既要发挥农民个体经济的积极性，又要发展互助合作的积极性，二者不是对立的。B 项说"只需要"合作积极性、"不需要"个体积极性，与党中央"一方面不能挫伤个体经济的积极性，另一方面在条件成熟时提倡组织起来"的方针直接矛盾。A 项体现自愿原则；C 项是正确阶级政策；D 项准确描述高级社特征（土地等生产资料归集体，取消土地入股分红，实行按劳分配）。
:::

:::callout{kind=note label="知识卡片：农业改造的四大特点"}
1. **两个积极性**：个体经济积极性 + 集体经济积极性
2. **三原则**：自愿互利、典型示范、国家帮助
3. **阶级政策**：依靠贫下中农，限制到消灭富农剥削
4. **三步骤**：互助组（萌芽）→ 初级社（半社会主义）→ 高级社（完全社会主义）
:::

:::callout{kind=tip label="结论速记"}
两个积极性都要抓，不能只讲合作、忽视个体。正确答案: B
:::

### 第6题

:::callout{kind=note label="题目"}
关于适合我国情况的对资本主义工商业进行社会主义改造的道路，描述中错误的是（　　）。

A. 用强制的方法改造资本主义工商业

B. 将工人阶级和民族资产阶级之间存在着的对抗性矛盾转化为非对抗性矛盾并按照人民内部矛盾来处理

C. 采取从低级到高级的国家资本主义的过渡形式

D. 把资本主义工商业者改造成为自食其力的社会主义劳动者
:::

:::callout{kind=insight label="解析"}
**【答案】A**

中国对资本主义工商业改造的最大特色是和平赎买，而非暴力没收。马克思恩格斯曾设想"如果能用赎买的办法变革所有制，将是最便宜不过的"，中国共产党结合国情将其变为现实。A 项"强制方法"与和平赎买方针根本对立——苏联最终采取强制没收，中国则创造性地走和平道路。B 项体现民族资产阶级两面性及矛盾转化；C 项是国家资本主义从初级到高级的三步走；D 项是"量才使用、适当照顾"对人的改造目标。
:::

:::callout{kind=note label="知识卡片：资本主义工商业改造道路"}
- **方法**：和平赎买（非强制没收）
- **形式**：国家资本主义（初级 → 个别公私合营 → 全行业公私合营）
- **矛盾处理**：对抗性矛盾 → 非对抗性矛盾，按人民内部矛盾处理
- **人的改造**：企业改造 + 人的改造相结合，变为自食其力的劳动者
:::

:::callout{kind=tip label="结论速记"}
资本主义工商业改造的核心方法是和平赎买，不是强制。正确答案: A
:::

### 第7题

:::callout{kind=note label="题目"}
对资本主义工商业实行和平赎买的优点不包括（　　）。

A. 有利于发挥私营工商业在国计民生方面的积极作用

B. 有利于争取和团结民族资产阶级

C. 有利于激化工人阶级和民族资产阶级之间的对抗性矛盾

D. 有利于发挥民族资产阶级中大多数人的知识、才能、技术专长和管理经验
:::

:::callout{kind=insight label="解析"}
**【答案】C**

和平赎买的"便宜"体现在经济、社会、人才三个层面：企业维持生产、避免动荡；争取民族资产阶级和民主党派；保留并转化资本家的管理经验与专业知识。C 项"激化对抗性矛盾"与和平赎买目的完全相反——改造的目标正是把对抗性矛盾转化为非对抗性矛盾，按人民内部矛盾处理，从而避免社会大动荡。A、B、D 三项均为教材明确列举的和平赎买优点。
:::

:::callout{kind=note label="知识卡片：和平赎买的三大优点"}
| 层面 | 具体优点 |
|------|---------|
| 经济 | 发挥私营工商业在国计民生中的积极作用，促进国民经济发展 |
| 政治 | 争取团结民族资产阶级，巩固统一战线 |
| 人才 | 发挥其知识、才能、技术专长和管理经验 |
:::

:::callout{kind=tip label="结论速记"}
和平赎买是化解矛盾、争取人心，不是激化对抗。正确答案: C
:::

### 第8题

:::callout{kind=note label="题目"}
我国能够采取赎买的方式对资本主义工商业进行和平改造的原因不包括（　　）。

A. 民族资产阶级既有剥削工人取得利润的一面，又有拥护宪法、愿意接受社会主义改造的一面

B. 中国共产党与民族资产阶级长期保持着统一战线的关系

C. 我国建立了强大的社会主义国营经济并掌握了国家的经济命脉

D. 我国工人阶级和民族资产阶级之间不存在矛盾
:::

:::callout{kind=insight label="解析"}
**【答案】D**

能够和平赎买，根本在于三个条件同时具备：改造对象是民族资产阶级（非官僚资产阶级）；认识到其两面性；社会主义力量已占压倒性优势。D 项说"不存在矛盾"是明显错误——工人阶级与民族资产阶级之间存在剥削与被剥削的对抗性矛盾，只是"如果处理得当，可以转变为非对抗性的矛盾"。正因为有矛盾才需要改造，正因为矛盾可转化才使和平方式成为可能。A、B、C 均为教材列举的三项原因。
:::

:::callout{kind=note label="知识卡片：和平赎买的三个前提"}
1. **对象不同**：民族资产阶级（官僚资本已没收，不存在赎买对象）
2. **两面性**：有剥削一面，也有拥护宪法、接受改造一面
3. **力量对比**：人民民主专政 + 强大国营经济 → 私人资本主义对社会主义依赖
4. **矛盾性质**：存在矛盾，但可转化为非对抗性矛盾
:::

:::callout{kind=tip label="结论速记"}
有矛盾，可转化，不是"无矛盾"。正确答案: D
:::

### 第9题

:::callout{kind=note label="题目"}
关于我国社会主义改造中出现的国家资本主义经济，下列描述中错误的是（　　）。

A. 在人民政府管理之下

B. 用各种形式和国营社会主义经济联系着

C. 主要是为了资本家的利润而存在

D. 受工人监督
:::

:::callout{kind=insight label="解析"}
**【答案】C**

毛泽东在《关于国家资本主义经济》中明确指出：这种资本主义经济"主要地不是为了资本家的利润而存在，而是为了供应人民和国家的需要而存在"，"是带着很大的社会主义性质的，是对工人和国家有利的"。A、B、D 三项均符合这一定义——在人民政府管理下，与国营经济联系，受工人监督。C 项把目的说成"主要为资本家利润"，混淆了普通资本主义与国家资本主义的本质区别。
:::

:::callout{kind=note label="知识卡片：国家资本主义的性质"}
- **定义**：在国家直接控制和支配下的资本主义经济
- **管理**：人民政府管理之下，受工人监督
- **联系**：用各种形式与国营社会主义经济相联系
- **目的**：供应人民和国家需要（非为资本家利润）
- **性质判断**：取决于所在国家性质，我国的国家资本主义带有很大社会主义性质
:::

:::callout{kind=tip label="结论速记"}
国家资本主义为国家和人民服务，不是为资本家利润服务。正确答案: C
:::

### 第10题

:::callout{kind=note label="题目"}
国家资本主义有初级形式和高级形式之分，下列各项中，属于高级形式的国家资本主义的是（　　）。

A. 统购包销

B. 公私合营

C. 委托加工

D. 计划订货
:::

:::callout{kind=insight label="解析"}
**【答案】B**

国家资本主义分初级和高级两种形式。初级形式是国家对私营工商业实行委托加工、计划订货、统购包销、经销代销等——主要是同资本家在企业外部的合作，企业所有权未变。高级形式是公私合营，包括个别企业公私合营和全行业公私合营——国家投资入股，生产资料国家和资本家共有（个别合营）或完全归国家（全行业合营），公方代表居领导地位。A、C、D 均属初级形式，只有 B 是高级形式。
:::

:::callout{kind=note label="知识卡片：国家资本主义两级形式"}
| 形式 | 具体方式 | 合作特点 |
|------|---------|---------|
| **初级** | 委托加工、计划订货、统购包销、经销代销 | 企业外部合作，四马分肥 |
| **高级** | 个别企业公私合营 → 全行业公私合营 | 企业内部合作，定息 |
:::

:::callout{kind=tip label="结论速记"}
初级四形式记"加订包销"，高级 = 公私合营。正确答案: B
:::

## 二、多项选择题（共 5 题，每题 2 分，共 10 分）

### 第11题

:::callout{kind=note label="题目"}
对个体农业社会主义改造采取的原则是（　　）。

A. 自愿互利

B. 公私兼顾

C. 典型示范

D. 国家帮助
:::

:::callout{kind=insight label="解析"}
**【答案】ACD**

1953 年 2 月《关于农业生产互助合作的决议》规定，处理互助组和生产合作社任何问题，必须绝对遵守**自愿**和**互利**两条原则；在此基础上，还要通过**典型示范**和**国家帮助**引导农民走合作化道路。A、C、D 三项构成农业合作化的完整原则体系。"公私兼顾"是调整工商业时处理公私关系的政策用语（如"五反"后调整公私关系），属于资本主义工商业改造范畴，不是农业合作化原则，故 B 不选。
:::

:::callout{kind=note label="知识卡片：农业合作化三原则"}
- **自愿互利**：入社自愿，利益分配合理，不得强迫命令
- **典型示范**：先办成功范例，用事实说服农民
- **国家帮助**：资金、技术、农具等方面给予支持
- **易混**："公私兼顾"用于工商业，不用于农业合作化
:::

:::callout{kind=tip label="结论速记"}
农业三原则 = 自愿互利 + 典型示范 + 国家帮助。正确答案: ACD
:::

### 第12题

:::callout{kind=note label="题目"}
对手工业的社会主义改造，党和政府采取了（　　）的方针。

A. 积极领导

B. 顺其自然

C. 稳步前进

D. 迅猛突进
:::

:::callout{kind=insight label="解析"}
**【答案】AC**

对手工业改造，党和政府采取"**积极领导、稳步前进**"的方针，与农业改造方法论一致：积极领导，防止放任自流；稳步前进，防止急躁冒进。方法上从供销合作入手，逐步走向生产合作（供销小组 → 供销合作社 → 生产合作社）。B 项"顺其自然"是放任主义，与"积极领导"矛盾；D 项"迅猛突进"违背"逐步过渡"的历史经验，1955 年后改造过急正是后来总结的历史教训之一。
:::

:::callout{kind=note label="知识卡片：手工业改造方针与步骤"}
- **方针**：积极领导、稳步前进
- **方法**：从供销合作入手 → 逐步走向生产合作
- **三步走**：供销小组（萌芽）→ 供销合作社（半社会主义）→ 生产合作社（完全社会主义）
- **1956 年底**：有组织手工业者占 91.7%，改造基本完成
:::

:::callout{kind=tip label="结论速记"}
手工业改造方针 = 积极领导 + 稳步前进。正确答案: AC
:::

### 第13题

:::callout{kind=note label="题目"}
初级形式的国家资本主义是国家对私营工商业实行（　　）等。

A. 委托加工

B. 计划订货

C. 统购包销

D. 经销代销
:::

:::callout{kind=insight label="解析"}
**【答案】ABCD**

初级形式的国家资本主义，是国家同资本家在企业**外部**的合作：国家提供原材料、包销产品，帮助私营企业克服困难，使其生产和经营纳入国家计划轨道，但企业所有权仍属资本家。四种具体形式——委托加工、计划订货、统购包销、经销代销——在私营工业和私营商业中分别运用，利润按"四马分肥"分配。四项全部正确，是初级国家资本主义的完整表述。
:::

:::callout{kind=note label="知识卡片：初级国家资本主义四形式"}
- **委托加工**：国家供料，私营厂加工
- **计划订货**：国家有计划地向私营厂订货
- **统购包销**：国家统一收购、统一销售
- **经销代销**：商业领域，国营批发、私营零售
- **利润分配**：四马分肥（国家税、公积金、福利费、资方红利）
:::

:::callout{kind=tip label="结论速记"}
初级四形式：委托加工、计划订货、统购包销、经销代销，全选。正确答案: ABCD
:::

### 第14题

:::callout{kind=note label="题目"}
初级形式的国家资本主义对私营工商业的利润，按（　　）等方面进行分配。

A. 国家所得税

B. 企业公积金

C. 工人福利费

D. 资方红利
:::

:::callout{kind=insight label="解析"}
**【答案】ABCD**

"四马分肥"是初级国家资本主义和个别公私合营阶段利润分配的基本形式。企业利润按四个方面分配：国家所得税（约 34.5%）、企业公积金（约 30%）、工人福利费（约 15%）、资方红利（约 20.5%）。资方红利大体占四分之一，资本主义剥削受到限制，企业具有了社会主义因素。全行业公私合营后改为定息，不再四马分肥。A、B、C、D 四项构成完整的"四马分肥"。
:::

:::callout{kind=note label="知识卡片：四马分肥"}
| 分配项目 | 大致比例 | 用途 |
|---------|---------|------|
| 国家所得税 | 34.5% | 上缴国家 |
| 企业公积金 | 30% | 扩大再生产 |
| 工人福利费 | 15% | 改善工人生活 |
| 资方红利 | 20.5% | 资本家分红（和平赎买） |
:::

:::callout{kind=tip label="结论速记"}
四马分肥 = 国家税 + 公积金 + 福利费 + 资方红利。正确答案: ABCD
:::

### 第15题

:::callout{kind=note label="题目"}
在资本主义工商业的社会主义改造中，国家对资方在职人员和资方代理人（　　），通过改造阶级成分的方式达到从整体上消灭资产阶级的目的。

A. 在政治上适当安排

B. 在工作上发挥作用

C. 在生产上排除在外

D. 在生活上妥善照顾
:::

:::callout{kind=insight label="解析"}
**【答案】ABD**

对资本主义工商业者改造，采取"包下来"政策，以企业为基地，按"量才使用、适当照顾"原则处理。具体是：政治上适当安排（如荣毅仁任上海市副市长）、工作上发挥作用（利用其管理经验与专业知识）、生活上妥善照顾（保证生活水平不因改造下降）。C 项"在生产上排除在外"与政策相反——改造的目的不是简单排斥，而是把剥削者变为自食其力的社会主义劳动者，继续为社会主义建设服务。A、B、D 三项是"量才使用、适当照顾"的完整内涵。
:::

:::callout{kind=note label="知识卡片：量才使用、适当照顾"}
- **政治上适当安排**：根据其能力和影响给予相应政治安排
- **工作上发挥作用**：安排合适岗位，继续发挥管理才能
- **生活上妥善照顾**：保证生活水平不因改造而下降
- **目标**：企业改造 + 人的改造相结合，从整体上消灭资产阶级
:::

:::callout{kind=tip label="结论速记"}
量才使用 = 政治安排 + 工作发挥 + 生活照顾，不是排除生产。正确答案: ABD
:::
