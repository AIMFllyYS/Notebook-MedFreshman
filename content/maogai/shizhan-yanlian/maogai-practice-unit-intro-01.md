# 导论·马克思主义中国化单元真题训练

> 题型：单项选择题 11 题（每题 2 分）+ 多项选择题 5 题（每题 2 分）　满分：32 分

---

## 一、单项选择题（每题 2 分，共 22 分）

### 第1题

:::callout{kind=note label="题目"}
一百年来，中国共产党团结带领中国人民进行的一切奋斗、一切牺牲、一切创造，归结起来就是一个主题：（　　）

A. 实现马克思主义中国化　　B. 实现中华民族伟大复兴

C. 建设中国特色社会主义　　D. 建设社会主义现代化强国
:::

:::callout{kind=insight label="解析"}
**【答案】B**

习近平总书记在庆祝中国共产党成立100周年大会上的重要讲话明确指出：一百年来，中国共产党团结带领中国人民进行的一切奋斗、一切牺牲、一切创造，归结起来就是一个主题——**实现中华民族伟大复兴**。

- **A 错**：马克思主义中国化是贯穿百年奋斗的一条**主线**和方法论，不是"归结起来的一个主题"。
- **C 错**：建设中国特色社会主义是改革开放以来的**道路选择**，是实现复兴的具体路径，不能概括百年奋斗的全部主题。
- **D 错**：建设社会主义现代化强国是**新时代**的奋斗目标，范围小于百年奋斗的整体主题。
:::

:::callout{kind=note label="知识卡片：百年奋斗的一个主题"}
| 概念 | 定位 | 易混辨析 |
|------|------|---------|
| **一个主题** | 实现中华民族伟大复兴 | 概括百年奋斗全部实践 |
| **一条主线** | 马克思主义中国化时代化 | 理论创新贯穿始终的方法 |
| **一条道路** | 中国特色社会主义 | 实现复兴的具体路径 |
| **一个目标** | 建设社会主义现代化强国 | 新时代的具体奋斗目标 |
:::

:::callout{kind=tip label="结论速记"}
百年奋斗"一个主题"= 实现中华民族伟大复兴；马克思主义中国化是"一条主线"，二者不可混淆。
:::

### 第2题

:::callout{kind=note label="题目"}
以毛泽东为主要代表的中国共产党人逐渐认识到，面对中国的特殊国情，不能教条式地对待马克思列宁主义，必须从中国实际出发，实现（　　）

A. 马克思主义时代化　　B. 马克思主义大众化

C. 马克思主义中国化　　D. 马克思主义本土化
:::

:::callout{kind=insight label="解析"}
**【答案】C**

教材导论原文：以毛泽东同志为主要代表的中国共产党人逐渐认识到，面对中国的特殊国情，不能教条式地对待马克思列宁主义，**必须从中国实际出发，实现马克思主义中国化**。

- **A 错**："时代化"是后来（尤其是党的十八大、二十大）与"中国化"并列使用的完整表述，此题语境是面对**中国特殊国情**的命题，核心是"中国化"。
- **B 错**："大众化"强调用通俗语言让理论为大众接受，是传播层面的要求，不是此句的核心概念。
- **D 错**："本土化"不是党的正式理论表述，标准用语是"中国化"。
:::

:::callout{kind=note label="知识卡片：马克思主义中国化的提出背景"}
| 背景 | 内容 |
|------|------|
| 中国国情 | 半殖民地半封建的落后农业国，与欧洲资本主义社会截然不同 |
| 血的教训 | 第五次反"围剿"失败、长征巨大牺牲，根因是"照搬本本" |
| 核心认识 | 马克思主义提供的是立场、观点、方法，不是可直接照搬的施工图纸 |
| 正式提出 | 1938年六届六中全会，毛泽东《论新阶段》报告 |
:::

:::callout{kind=tip label="结论速记"}
面对中国特殊国情 → 不能教条对待 → 必须实现**马克思主义中国化**（标准表述，不用"本土化"）。
:::

### 第3题

:::callout{kind=note label="题目"}
1938年，毛泽东在党的六届六中全会上作了（　　）的报告，这标志着"马克思主义中国化"这一命题的正式提出。

A. 《反对本本主义》　　B. 《新民主主义论》

C. 《〈共产党人〉发刊词》　　D. 《论新阶段》
:::

:::callout{kind=insight label="解析"}
**【答案】D**

1938年9—10月，党的六届六中全会在延安召开。毛泽东在会上作了题为**《论新阶段》**的报告，首次正式提出"马克思主义的中国化"命题，强调"没有抽象的马克思主义，只有具体的马克思主义"。

- **A 错**：《反对本本主义》（1930年）提出"没有调查，没有发言权"，是反对教条主义的早期文献，但未正式提出"中国化"命题。
- **B 错**：《新民主主义论》（1940年）系统阐述新民主主义革命理论，晚于命题提出。
- **C 错**：《〈共产党人〉发刊词》（1939年）提出统一战线、武装斗争、党的建设"三大法宝"，非命题提出处。
:::

:::callout{kind=note label="知识卡片：马克思主义中国化命题的提出"}
| 要素 | 内容 |
|------|------|
| **时间** | 1938年 |
| **会议** | 党的六届六中全会 |
| **报告** | 《论新阶段》 |
| **核心表述** | 没有抽象的马克思主义，只有具体的马克思主义；按照中国的特点去应用它 |
| **政治前提** | 1935年遵义会议确立毛泽东领导地位 |
:::

:::callout{kind=tip label="结论速记"}
1938 · 六届六中全会 · 《论新阶段》= "马克思主义中国化"命题正式提出（三要素必记）。
:::

### 第4题

:::callout{kind=note label="题目"}
（　　），为实现中华民族伟大复兴创造了根本社会条件。

A. 新民主主义革命时期　　B. 社会主义革命和建设时期

C. 改革开放和社会主义现代化建设历史新时期　　D. 新时代中国特色社会主义时期
:::

:::callout{kind=insight label="解析"}
**【答案】A**

党的十九届六中全会《决议》和教材导论对四个历史时期的贡献有明确对应：

| 时期 | 为实现民族复兴提供的条件 |
|------|------------------------|
| **新民主主义革命时期** | **根本社会条件** |
| 社会主义革命和建设时期 | 根本政治前提和制度基础 |
| 改革开放新时期 | 体制保证和快速发展的物质条件 |
| 新时代中国特色社会主义时期 | 更完善的制度保证、更坚实的物质基础、更主动的精神力量 |

- **B 错**：对应"根本政治前提和制度基础"（建立社会主义制度）。
- **C 错**：对应"充满新的活力的体制保证和快速发展的物质条件"（改革开放）。
- **D 错**：对应"更为完善的制度保证、更为坚实的物质基础、更为主动的精神力量"。
:::

:::callout{kind=note label="知识卡片：四个时期与民族复兴的关系"}
| 时期 | 伟大成就关键词 | 为复兴提供 |
|------|--------------|-----------|
| 新民主主义革命 | 建立新中国、民族独立、人民解放 | **根本社会条件** |
| 社会主义革命和建设 | 社会主义制度建立、一穷二白→社会主义 | 根本政治前提和制度基础 |
| 改革开放新时期 | 经济总量世界第二、人民生活小康 | 体制保证 + 物质条件 |
| 新时代 | 全面建成小康社会、历史性变革 | 制度 + 物质 + 精神力量 |
:::

:::callout{kind=tip label="结论速记"}
四期贡献速记：革命→**社会条件**；建设→**政治制度**；改革→**体制物质**；新时代→**制度物质精神**。
:::

### 第5题

:::callout{kind=note label="题目"}
（　　），为实现中华民族伟大复兴奠定根本政治前提和制度基础。

A. 新民主主义革命时期　　B. 社会主义革命和建设时期

C. 改革开放和社会主义现代化建设历史新时期　　D. 新时代中国特色社会主义时期
:::

:::callout{kind=insight label="解析"}
**【答案】B**

**社会主义革命和建设时期**，党领导人民完成社会主义改造、确立社会主义基本制度，推进社会主义建设，实现了中华民族有史以来最为广泛而深刻的社会变革，为实现中华民族伟大复兴**奠定了根本政治前提和制度基础**。

- **A 错**：新民主主义革命时期创造的是"根本社会条件"（结束半殖民地半封建、实现民族独立）。
- **C 错**：改革开放新时期提供的是"体制保证和物质条件"，不是"政治前提和制度基础"。
- **D 错**：新时代提供的是"更为完善的"三重保障，表述不同。
:::

:::callout{kind=note label="知识卡片：社会主义革命和建设时期的标志性成就"}
| 成就 | 意义 |
|------|------|
| 完成社会主义改造 | 确立社会主义基本制度 |
| 实现社会变革 | 有史以来最为广泛而深刻的社会变革 |
| 一穷二白→社会主义 | 东方大国大步迈进社会主义社会 |
| 理论成果 | 毛泽东思想（含社会主义建设理论） |
:::

:::callout{kind=tip label="结论速记"}
"根本政治前提和制度基础"= 社会主义革命和建设时期（建立社会主义制度）。
:::

### 第6题

:::callout{kind=note label="题目"}
（　　），为实现中华民族伟大复兴提供了充满新的活力的体制保证和快速发展的物质条件。

A. 新民主主义革命时期　　B. 社会主义革命和建设时期

C. 改革开放和社会主义现代化建设历史新时期　　D. 新时代中国特色社会主义时期
:::

:::callout{kind=insight label="解析"}
**【答案】C**

**改革开放和社会主义现代化建设新时期**，党解放思想、锐意进取，创造了改革开放和社会主义现代化建设的伟大成就：经济总量跃居世界第二，人民生活从温饱不足到总体小康，推进了从"站起来"到"富起来"的伟大飞跃，为实现中华民族伟大复兴提供了**充满新的活力的体制保证和快速发展的物质条件**。

- **A 错**：对应"根本社会条件"。
- **B 错**：对应"根本政治前提和制度基础"。
- **D 错**：对应"更为完善的制度保证、更为坚实的物质基础、更为主动的精神力量"——注意"更为""更为主动"等表述差异。
:::

:::callout{kind=note label="知识卡片：改革开放新时期的标志性成就"}
| 维度 | 内容 |
|------|------|
| 经济 | 从生产力相对落后→经济总量世界第二 |
| 生活 | 从温饱不足→总体小康→奔向全面小康 |
| 历史飞跃 | 中华民族从站起来→**富起来** |
| 关键决策 | 1978年十一届三中全会，把工作中心转移到经济建设 |
| 理论成果 | 中国特色社会主义理论体系（新的飞跃） |
:::

:::callout{kind=tip label="结论速记"}
"体制保证 + 物质条件" + "富起来" = 改革开放和社会主义现代化建设新时期。
:::

### 第7题

:::callout{kind=note label="题目"}
（　　），为实现中华民族伟大复兴提供了更为完善的制度保证、更为坚实的物质基础、更为主动的精神力量。

A. 新民主主义革命时期　　B. 社会主义革命和建设时期

C. 改革开放和社会主义现代化建设历史新时期　　D. 新时代中国特色社会主义时期
:::

:::callout{kind=insight label="解析"}
**【答案】D**

**新时代中国特色社会主义时期**，在习近平新时代中国特色社会主义思想指导下，党自信自强、守正创新，全面建成小康社会，推动党和国家事业取得历史性成就、发生历史性变革，为实现中华民族伟大复兴提供了**更为完善的制度保证、更为坚实的物质基础、更为主动的精神力量**。

- **A、B、C 错**：三个时期的表述均不含"更为完善""更为坚实""更为主动"这一组三个"更为/更为主动"的搭配。
- 注意与第6题对比：新时期是"充满新的活力的体制保证"，新时代是"**更为完善的**制度保证"——程度不同，不可互换。
:::

:::callout{kind=note label="知识卡片：新时代的历史性成就"}
| 维度 | 内容 |
|------|------|
| 历史飞跃 | 从"富起来"→**"强起来"** |
| 标志性成就 | 全面建成小康社会，解决绝对贫困问题 |
| 三重保障 | 更为完善的制度保证 + 更为坚实的物质基础 + 更为主动的精神力量 |
| 历史进程 | 实现中华民族伟大复兴进入**不可逆转**的历史进程 |
| 指导思想 | 习近平新时代中国特色社会主义思想 |
:::

:::callout{kind=tip label="结论速记"}
"更为完善的制度 + 更为坚实的物质 + 更为主动的精神" = 新时代中国特色社会主义时期（强起来）。
:::

### 第8题

:::callout{kind=note label="题目"}
马克思主义中国化第一次历史性飞跃的理论成果是（　　）

A. 中国特色社会主义　　B. 毛泽东思想

C. 邓小平理论　　D. 习近平新时代中国特色社会主义思想
:::

:::callout{kind=insight label="解析"}
**【答案】B**

马克思主义中国化时代化产生了两次历史性飞跃：

| 飞跃次序 | 理论成果 | 时期 |
|---------|---------|------|
| **第一次历史性飞跃** | **毛泽东思想** | 新民主主义革命和社会主义革命建设时期 |
| 新的飞跃 | 中国特色社会主义理论体系 | 改革开放新时期 |
| 新的飞跃 | 习近平新时代中国特色社会主义思想 | 新时代 |

- **A 错**："中国特色社会主义"是**道路/事业**的名称，不是具体理论成果的名称。
- **C 错**：邓小平理论是中国特色社会主义理论体系的**开篇之作**，属于"新的飞跃"，非第一次。
- **D 错**：习近平新时代中国特色社会主义思想是新时代"新的飞跃"，非第一次。
:::

:::callout{kind=note label="知识卡片：马克思主义中国化时代化的理论成果"}
| 理论成果 | 历史定位 |
|---------|---------|
| **毛泽东思想** | 第一次历史性飞跃；1945年七大写入党章 |
| 邓小平理论 | 中国特色社会主义理论体系开篇之作 |
| "三个代表"重要思想 | 跨世纪发展 |
| 科学发展观 | 新世纪新阶段 |
| 习近平新时代中国特色社会主义思想 | 新时代新的飞跃 |
:::

:::callout{kind=tip label="结论速记"}
第一次历史性飞跃 = **毛泽东思想**；改革开放后 = 新的飞跃（中特理论体系 + 习近平思想）。
:::

### 第9题

:::callout{kind=note label="题目"}
中国共产党和中国人民以英勇顽强的奋斗向世界庄严宣告，（　　）是决定当代中国前途命运的关键一招，中国大踏步赶上了时代。

A. 党的领导　　B. 新民主主义革命

C. 改革开放　　D. 新时代中国特色社会主义
:::

:::callout{kind=insight label="解析"}
**【答案】C**

习近平总书记在庆祝中国共产党成立100周年大会上的讲话："中国共产党和中国人民以英勇顽强的奋斗向世界庄严宣告，**改革开放**是决定当代中国前途命运的关键一招，中国大踏步赶上了时代！"

- **A 错**：党的领导是中国特色社会主义**最本质的特征**和**最大优势**，但不是"关键一招"的表述。
- **B 错**：新民主主义革命使中国人民"站起来"，对应的是根本社会条件，不是"赶上时代"。
- **D 错**：新时代中国特色社会主义是党的十八大以来的历史方位，"赶上时代"特指改革开放的历史贡献。
:::

:::callout{kind=note label="知识卡片：改革开放的历史定位"}
| 表述 | 含义 |
|------|------|
| 关键一招 | 决定当代中国前途命运 |
| 历史贡献 | 中国大踏步赶上了时代 |
| 伟大飞跃 | 从"站起来"到"富起来" |
| 开启时间 | 1978年党的十一届三中全会 |
| 核心决策 | 工作中心转移到经济建设、实行改革开放 |
:::

:::callout{kind=tip label="结论速记"}
"关键一招" + "大踏步赶上时代" = **改革开放**（100周年讲话原话，必背）。
:::

### 第10题

:::callout{kind=note label="题目"}
党的十八大以来，中国特色社会主义进入（　　）

A. 新阶段　　B. 新时期

C. 新方位　　D. 新时代
:::

:::callout{kind=insight label="解析"}
**【答案】D**

党的十九大报告明确指出：**中国特色社会主义进入了新时代**，这是我国发展新的历史方位。

- **A 错**："新阶段"不是标准表述。党的十四大曾用"新阶段"指社会主义初级阶段，但与"党的十八大以来"不对应。
- **B 错**："新时期"通常指**改革开放和社会主义现代化建设新时期**（1978—2012），已被新时代所接续。
- **C 错**："新方位"是对新时代的补充说明（"新的历史方位"），但空格处应填的核心概念是"**新时代**"。
:::

:::callout{kind=note label="知识卡片：历史方位相关概念辨析"}
| 概念 | 时间范围 | 标准表述 |
|------|---------|---------|
| **新时代** | 2012年（十八大）至今 | 中国特色社会主义进入新时代 |
| 新时期 | 1978（十一届三中全会）—2012 | 改革开放和社会主义现代化建设新时期 |
| 新方位 | 新时代的同义补充 | "我国发展新的历史方位" |
| 新阶段 | 非十八大标准用语 | 易与"初级阶段"混淆，不选 |
:::

:::callout{kind=tip label="结论速记"}
十八大以来 → 中国特色社会主义进入**新时代**（不是"新阶段""新时期"）。
:::

（　　）明确把"不断谱写马克思主义中国化时代化新篇章"作为当代中国共产党人的庄严历史责任，并提出继续推进马克思主义中国化时代化的新要求。

**【答案】D**

**党的二十大**报告明确把"不断谱写马克思主义中国化时代化新篇章"作为**当代中国共产党人的庄严历史责任**，并提出了继续推进马克思主义中国化时代化的新要求。同时，二十大强调"是中国化时代化的马克思主义行"（从"一个行"到"两个行"）。

### 第13题

:::callout{kind=note label="题目"}
十九届六中全会指出，新民主主义革命时期，党面临的主要任务是，反对（　　），争取民族独立、人民解放。

A. 帝国主义　　B. 封建主义

C. 民族资本主义　　D. 官僚资本主义
:::

:::callout{kind=insight label="解析"}
**【答案】ABD**

新民主主义革命时期，党面临的主要任务是：**反对帝国主义、封建主义、官僚资本主义**，争取民族独立、人民解放，为实现中华民族伟大复兴创造根本社会条件。

- **A、B、D 正确**：这就是著名的"三座大山"——帝国主义、封建主义、官僚资本主义。
- **C 错**：**民族资本主义**不是革命对象。新民主主义革命的对象是帝国主义、封建主义和官僚资本主义；民族资产阶级具有两面性，是统一战线的组成部分，属于"朋友"而非"敌人"。

注意区分：新民主主义革命**性质**是资产阶级民主革命，但**对象**不包括民族资本主义；对民族资本主义采取"利用、限制、改造"的政策（社会主义改造时期）。
:::

:::callout{kind=note label="知识卡片：新民主主义革命的对象与动力"}
| 类别 | 内容 |
|------|------|
| **革命对象（三座大山）** | 帝国主义、封建主义、官僚资本主义 |
| 革命动力 | 无产阶级、农民阶级、城市小资产阶级、民族资产阶级 |
| 领导力量 | 无产阶级（通过中国共产党） |
| 主要任务 | 反对三座大山，争取民族独立、人民解放 |
| 易错点 | 民族资本主义是动力/朋友，**不是**革命对象 |
:::

:::callout{kind=tip label="结论速记"}
新民主主义革命对象 = 帝国主义 + 封建主义 + 官僚资本主义（**不含民族资本主义**）。
:::

党的十八大以来，创造了新时代中国特色社会主义的伟大成就，为实现中华民族伟大复兴提供了（　　）

A. 更为完善的制度保证

B. 更为坚实的物质基础

D. 更为主动的精神力量
:::

:::callout{kind=insight label="解析"}

新时代为实现民族复兴提供了**三个"更为/更为主动"**：

- **A 正确**——更为完善的制度保证
- **B 正确**——更为坚实的物质基础
- **D 正确**——更为主动的精神力量

- **C 错**："**更为根本的社会条件**"表述不存在。正确的对应是：新民主主义革命时期创造了"**根本**社会条件"（无"更为"）；新时代的三个保障均含"更为"或"更为主动"。

这是高频易错题：注意区分"根本社会条件"（革命时期）与"更为完善的制度保证"等（新时代）的表述差异。
:::

:::callout{kind=note label="知识卡片：四个时期"提供"对照表（防混淆）"}
| 时期 | 标准表述 | 有无"更为" |
|------|---------|:--------:|
| 新民主主义革命 | **根本**社会条件 | 无 |
| 社会主义革命和建设 | **根本**政治前提和制度基础 | 无 |
| 改革开放新时期 | 充满新的活力的体制保证 + 快速发展的物质条件 | 无 |
| 新时代 | **更为**完善的制度 + **更为**坚实的物质 + **更为主动**的精神 | 有 |
:::

:::callout{kind=tip label="结论速记"}
新时代 = ABD（更为完善的制度 + 更为坚实的物质 + 更为主动的精神）；**没有**"更为根本的社会条件"。
