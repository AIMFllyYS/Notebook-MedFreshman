### 第1题

:::callout{kind=note label="题目"}
毛泽东思想是马克思主义中国化时代化的（　　）

A. 第一个理论成果　　B. 唯一理论成果

C. 最新理论成果　　D. 最高理论成果
:::

:::callout{kind=insight label="解析"}
**【答案】A**

**【考点】**导论/第一章·毛泽东思想的历史地位

毛泽东思想是马克思主义中国化时代化的**第一个重大理论成果**，是马克思列宁主义在中国的创造性运用和发展。此后又形成了邓小平理论、"三个代表"重要思想、科学发展观和习近平新时代中国特色社会主义思想等理论成果。

- **B 错**：并非"唯一"理论成果，后续还有多个重大理论成果。
- **C 错**："最新"对应习近平新时代中国特色社会主义思想。
- **D 错**：教材无"最高理论成果"这一表述。
:::

:::callout{kind=note label="知识卡片：马克思主义中国化时代化的理论成果"}
| 理论成果 | 历史定位 |
|---------|---------|
| **毛泽东思想** | 第一个重大理论成果；1945年七大写入党章 |
| 邓小平理论 | 中国特色社会主义理论体系开篇之作 |
| "三个代表"重要思想 | 跨世纪发展 |
| 科学发展观 | 新世纪新阶段 |
| 习近平新时代中国特色社会主义思想 | 新时代新的飞跃 |
:::

:::callout{kind=tip label="结论速记"}
马克思主义中国化时代化**第一个**理论成果 = **毛泽东思想**。
:::

### 第2题

:::callout{kind=note label="题目"}
毛泽东思想、邓小平理论、"三个代表"重要思想、科学发展观之间的关系是（　　）

A. 相互取代　　B. 相互对立

C. 一脉相承又与时俱进　　D. 完全等同
:::

:::callout{kind=insight label="解析"}
**【答案】C**

**【考点】**导论·理论成果的关系

毛泽东思想和中国特色社会主义理论体系，都是马克思列宁主义在中国的运用和发展，是**一脉相承又与时俱进**的科学理论，是党和人民必须长期坚持的指导思想，是全国各族人民团结奋斗的共同思想基础。

- **A、B 错**：各理论成果是继承发展关系，不是取代或对立。
- **D 错**：各成果所处时代和回答的问题不同，不能"完全等同"。
:::

:::callout{kind=note label="知识卡片：五大理论成果的关系"}
| 关系 | 内容 |
|------|------|
| 共同基础 | 都是马克思列宁主义在中国的运用和发展 |
| 核心关系 | **一脉相承又与时俱进** |
| 地位 | 党和人民必须长期坚持的指导思想 |
| 2018年 | 写入宪法 |
:::

:::callout{kind=tip label="结论速记"}
各理论成果关系 = **一脉相承又与时俱进**（不是取代、对立或等同）。
:::

### 第3题

:::callout{kind=note label="题目"}
党的历史上，第一次提出"毛泽东思想"这一概念的是（　　）

A. 刘少奇　　B. 王稼祥

C. 张闻天　　D. 周恩来
:::

:::callout{kind=insight label="解析"}
**【答案】B**

**【考点】**第一章·毛泽东思想概念的提出

1943年**王稼祥**在《中国共产党与中国民族解放的道路》一文中，第一次提出并使用了"毛泽东思想"这一概念。1945年党的七大上，刘少奇在《关于修改党章的报告》中对毛泽东思想作了系统阐述，七大正式把毛泽东思想确立为党的指导思想。

- **A 错**：刘少奇是在七大上**系统阐述**毛泽东思想，不是首次提出概念。
- **C、D 错**：与"首次提出概念"无关。
:::

:::callout{kind=note label="知识卡片：毛泽东思想概念与确立"}
| 节点 | 人物/事件 | 内容 |
|------|----------|------|
| **1943年** | **王稼祥** | 首次提出"毛泽东思想"概念 |
| 1945年七大 | 刘少奇 | 系统阐述，确立为指导思想写入党章 |
| 1938年六届六中全会 | 毛泽东《论新阶段》 | 首次提出"马克思主义中国化"命题 |
:::

:::callout{kind=tip label="结论速记"}
首次提出"毛泽东思想"概念 = **1943年王稼祥**；七大确立为指导思想 = 刘少奇系统阐述。
:::

### 第4题

:::callout{kind=note label="题目"}
中国新民主主义革命开始的标志是五四运动，其区别于旧民主主义革命的根本标志是（　　）

A. 革命任务不同　　B. 革命领导阶级不同

C. 革命对象不同　　D. 革命前途不同
:::

:::callout{kind=insight label="解析"}
**【答案】B**

**【考点】**第二章·新旧民主主义革命的区别

新旧民主主义革命都属于资产阶级民主革命范畴，革命任务都是反帝反封建，但**领导阶级不同**：旧民主主义革命由资产阶级领导，新民主主义革命由无产阶级领导。**无产阶级领导权**是二者区别的根本标志。

- **A 错**：革命任务相同，都是反帝反封建。
- **C 错**：革命对象基本相同（帝国主义、封建主义等）。
- **D 错**：革命前途不同是重要区别，但**根本标志**是领导权问题。
:::

:::callout{kind=note label="知识卡片：新旧民主主义革命的区别"}
| 对比项 | 旧民主主义革命 | 新民主主义革命 |
|--------|--------------|--------------|
| **领导阶级** | 资产阶级 | **无产阶级**（根本标志） |
| 革命任务 | 反帝反封建 | 反帝反封建 |
| 革命前途 | 资本主义 | 社会主义 |
| 开端 | — | 1919年五四运动 |
:::

:::callout{kind=tip label="结论速记"}
新旧民主主义革命区别的**根本标志** = **革命领导阶级不同**（无产阶级领导权）。
:::

### 第5题

:::callout{kind=note label="题目"}
新民主主义的政治纲领所要建立的国家政体是（　　）

A. 议会共和制　　B. 民主集中制的人民代表大会制度

C. 总统制　　D. 苏维埃代表大会制度
:::

:::callout{kind=insight label="解析"}
**【答案】B**

**【考点】**第二章·基本纲领·政治纲领

新民主主义的政治纲领：**国体**是无产阶级领导的、以工农联盟为基础的各革命阶级联合专政的民主共和国；**政体**是**民主集中制的人民代表大会制度**。它既不同于欧美资产阶级专政的共和国，也不同于苏联式无产阶级专政的共和国。

- **A、C 错**：属于西方资产阶级共和制政体形式。
- **D 错**：苏联式无产阶级专政共和国，非新民主主义政体。
:::

:::callout{kind=note label="知识卡片：新民主主义政治纲领"}
| 要素 | 内容 |
|------|------|
| **国体** | 各革命阶级联合专政的民主共和国 |
| **政体** | 民主集中制的人民代表大会制度 |
| 对比 | 不同于欧美资产阶级共和国，也不同于苏联式共和国 |
| 1954年宪法 | 确立人民代表大会制度为根本政治制度 |
:::

:::callout{kind=tip label="结论速记"}
新民主主义政体 = **民主集中制的人民代表大会制度**。
:::

毛泽东在《〈共产党人〉发刊词》中指出，党的建设是一项"伟大的工程"。中国共产党加强自身建设，始终放在首位的是（　　）

A. 组织建设　　B. 作风建设

C. 思想建设　　D. 制度建设

:::callout{kind=insight label="解析"}
**【答案】C**

- **A 错**：组织建设重要，但不是"始终放在首位"。
- **B 错**：作风建设是三大优良作风，但不是首位。
- **D 错**：制度建设是后来逐步完善的，非发刊词强调的首位。

| 出处 | 1939年《〈共产党人〉发刊词》 |

党的建设**始终放在首位** = **思想建设**（思想建党）。

### 第7题

:::callout{kind=note label="题目"}
中国共产党的三大优良作风是（　　）

A. 理论联系实际、密切联系群众、批评与自我批评　　B. 谦虚谨慎、艰苦奋斗、实事求是

C. 独立自主、自力更生、艰苦创业　　D. 民主集中、团结统一、纪律严明
:::

:::callout{kind=insight label="解析"}
**【答案】A**

**【考点】**第二章·三大优良作风

毛泽东在党的七大上概括了党的三大优良作风：**理论和实践相结合（理论联系实际）的作风**、**和人民群众紧密联系在一起（密切联系群众）的作风**、**自我批评（批评与自我批评）的作风**。这是中国共产党区别于其他任何政党的显著标志。

- **B 错**："两个务必"等表述，非三大作风。
- **C 错**：独立自强是路线方针，非三大作风。
- **D 错**：非标准表述。
:::

:::callout{kind=note label="知识卡片：三大优良作风"}
| 作风 | 简称 |
|------|------|
| 理论和实践相结合 | **理论联系实际** |
| 和人民群众紧密联系在一起 | **密切联系群众** |
| 自我批评 | **批评与自我批评** |
| 意义 | 党区别于其他任何政党的显著标志 |
:::

:::callout{kind=tip label="结论速记"}
三大优良作风 = **理论联系实际 + 密切联系群众 + 批评与自我批评**。
:::

### 第8题

:::callout{kind=note label="题目"}
新民主主义社会的经济成分中，处于领导地位的是（　　）

A. 个体经济　　B. 私人资本主义经济

C. 社会主义性质的国营经济　　D. 国家资本主义经济
:::

:::callout{kind=insight label="解析"}
**【答案】C**

**【考点】**第三章·新民主主义社会经济成分

新民主主义社会存在五种经济成分：社会主义性质的国营经济、半社会主义性质的合作社经济、农民和手工业者的个体经济、私人资本主义经济、国家资本主义经济。其中**社会主义性质的国营经济居于领导地位**，虽然比重不大，但决定着社会发展方向。

- **A 错**：个体经济数量大，但非领导地位。
- **B 错**：私人资本主义是被限制、改造的对象。
- **D 错**：国家资本主义是过渡形式，非领导地位。
:::

:::callout{kind=note label="知识卡片：新民主主义社会五种经济成分"}
| 经济成分 | 地位 |
|---------|------|
| **社会主义国营经济** | **领导地位**（决定发展方向） |
| 合作社经济 | 半社会主义性质 |
| 个体经济 | 数量大，非领导 |
| 私人资本主义经济 | 被限制、改造 |
| 国家资本主义经济 | 过渡形式 |
:::

:::callout{kind=tip label="结论速记"}
新民主主义社会经济成分中**居领导地位** = **社会主义性质的国营经济**。
:::

### 第9题

:::callout{kind=note label="题目"}
在对资本主义工商业的社会主义改造中，国家资本主义的高级形式是（　　）

A. 委托加工　　B. 计划订货

C. 统购包销　　D. 公私合营
:::

:::callout{kind=insight label="解析"}
**【答案】D**

**【考点】**第三章·国家资本主义形式

对资本主义工商业改造采取从低级到高级的国家资本主义过渡形式：**初级形式**（委托加工、计划订货、统购包销、经销代销等）和**高级形式**（个别企业的公私合营、全行业的公私合营）。**公私合营**特别是全行业公私合营是国家资本主义的高级形式。

- **A、B、C 错**：均属于国家资本主义的**初级形式**。
:::

:::callout{kind=note label="知识卡片：国家资本主义形式"}
| 级别 | 形式 |
|------|------|
| **初级** | 委托加工、计划订货、统购包销、经销代销 |
| **高级** | **公私合营**（个别企业→全行业） |
| 改造方式 | 和平赎买；四马分肥、定息 |
:::

:::callout{kind=tip label="结论速记"}
国家资本主义**高级形式** = **公私合营**；A/B/C 均为初级形式。
:::

### 第10题

:::callout{kind=note label="题目"}
1954年第一届全国人民代表大会通过的《中华人民共和国宪法》确立的我国根本政治制度是（　　）

A. 中国共产党领导的多党合作和政治协商制度　　B. 人民代表大会制度

C. 民族区域自治制度　　D. 基层群众自治制度
:::

:::callout{kind=insight label="解析"}
**【答案】B**

**【考点】**第三章·社会主义政治制度的确立

1954年宪法明确规定了我国人民民主专政的国体和**人民代表大会制度的政体**。**人民代表大会制度**是我国的**根本政治制度**；中国共产党领导的多党合作和政治协商制度、民族区域自治制度等是**基本政治制度**。

- **A、C、D 错**：均为基本政治制度，非根本政治制度。
:::

:::callout{kind=note label="知识卡片：根本政治制度 vs 基本政治制度"}
| 类型 | 制度 |
|------|------|
| **根本政治制度** | **人民代表大会制度** |
| 基本政治制度 | 多党合作和政治协商制度、民族区域自治制度、基层群众自治制度 |
| 1954年宪法 | 确立国体（人民民主专政）和政体（人民代表大会制度） |
:::

:::callout{kind=tip label="结论速记"}
我国**根本政治制度** = **人民代表大会制度**（基本政治制度勿混）。
:::

### 第11题

:::callout{kind=note label="题目"}
毛泽东关于社会主义社会矛盾学说中，指出社会主义社会的基本矛盾仍然是（　　）

A. 敌我矛盾和人民内部矛盾　　B. 生产力和生产关系、经济基础和上层建筑之间的矛盾

C. 无产阶级和资产阶级的矛盾　　D. 先进和落后的矛盾
:::

:::callout{kind=insight label="解析"}
**【答案】B**

**【考点】**第四章·社会主义社会基本矛盾

毛泽东在《关于正确处理人民内部矛盾的问题》中指出，社会主义社会的**基本矛盾**仍然是**生产力和生产关系、经济基础和上层建筑之间的矛盾**，但这种矛盾是非对抗性的，可以通过社会主义制度本身不断地得到解决。敌我矛盾和人民内部矛盾是**两类不同性质的社会矛盾**，不是基本矛盾。

- **A 错**：敌我矛盾和人民内部矛盾是两类社会矛盾，非基本矛盾。
- **C 错**：无产阶级和资产阶级矛盾在社会主义改造完成后已不再是主要矛盾。
- **D 错**：非基本矛盾表述。
:::

:::callout{kind=note label="知识卡片：社会主义社会矛盾学说"}
| 概念 | 内容 |
|------|------|
| **基本矛盾** | 生产力与生产关系、经济基础与上层建筑的矛盾（非对抗性） |
| 两类矛盾 | 敌我矛盾（对抗性）、人民内部矛盾（非对抗性） |
| 国家政治生活主题 | 正确处理人民内部矛盾 |
| 出处 | 1957年《关于正确处理人民内部矛盾的问题》 |
:::

:::callout{kind=tip label="结论速记"}
社会主义社会**基本矛盾** = **生产力与生产关系、经济基础与上层建筑的矛盾**。
:::

### 第12题

:::callout{kind=note label="题目"}
毛泽东在处理祖国统一问题上，提出的对台方针"一纲四目"中的"一纲"是指（　　）

A. 台湾必须统一于中国　　B. 台湾实行高度自治

C. 双方互不派人破坏对方　　D. 台湾社会改革从缓
:::

:::callout{kind=insight label="解析"}

### 第25题

:::callout{kind=note label="题目"}
**材料一：** 1938年，毛泽东在党的六届六中全会上指出："没有抽象的马克思主义，只有具体的马克思主义。所谓具体的马克思主义，就是通过民族形式的马克思主义，就是把马克思主义应用到中国具体环境的具体斗争中去。"

**材料二：** 习近平强调，"两个结合"是我们取得成功的最大法宝。只有坚持把马克思主义基本原理同中国具体实际相结合、同中华优秀传统文化相结合，坚持运用辩证唯物主义和历史唯物主义，才能正确回答时代和实践提出的重大问题。

（1）根据材料，说明什么是马克思主义中国化时代化，为什么必须推进马克思主义中国化时代化。（8分）

（2）结合材料二，谈谈你对"两个结合"特别是"同中华优秀传统文化相结合"的理解。（9分）
:::

:::callout{kind=insight label="解析"}
**【考点】**导论·马克思主义中国化时代化/两个结合（结合时事）

**（1）什么是马克思主义中国化时代化，为什么必须推进？**

①马克思主义中国化时代化，就是把马克思主义基本原理同中国具体实际相结合、同中华优秀传统文化相结合，坚持马克思主义的科学世界观和方法论，紧跟时代步伐、回应实践要求，不断形成中国化时代化的马克思主义理论成果，并用以指导实践。

②必须推进马克思主义中国化时代化，是因为：第一，这是**解决中国问题的需要**，马克思主义只有同中国实际相结合才能发挥指导作用，教条式地对待马克思主义只会导致失败；第二，这是**马克思主义理论的内在要求**，马克思主义是发展的、开放的科学，必须随实践发展而发展；第三，这是**马克思主义永葆生机活力的关键**，实践证明，中国化时代化的马克思主义行。

**（2）对"两个结合"特别是"同中华优秀传统文化相结合"的理解**

①"两个结合"即坚持把马克思主义基本原理同中国具体实际相结合、同中华优秀传统文化相结合，是推进马克思主义中国化时代化的**根本途径和成功密码**。

②"同中国具体实际相结合"解决的是运用马克思主义解答中国现实问题的问题；"同中华优秀传统文化相结合"解决的是马克思主义在中国**扎根的文化土壤**问题。

③马克思主义与中华优秀传统文化具有高度契合性（如天下为公、民为邦本、实事求是等）。同中华优秀传统文化相结合，一方面使马克思主义获得民族形式、深厚根基和广泛认同，另一方面使中华优秀传统文化实现创造性转化和创新性发展，二者相互成就。

④这启示我们，既要坚持马克思主义的指导地位，又要坚定文化自信，从中华文明中汲取智慧，不断开辟马克思主义中国化时代化新境界。
:::

### 第26题

:::callout{kind=note label="题目"}
**材料一：** 毛泽东在《论人民民主专政》中指出，资产阶级共和国的方案在中国行不通，中国人民经过反复比较和选择，最终选择了中国共产党、选择了社会主义道路。

**材料二：** 中华人民共和国的成立，实现了中国从几千年封建专制政治向人民民主的伟大飞跃。占世界人口四分之一的东方大国建立起人民当家作主的新中国，极大地改变了世界政治格局，鼓舞了世界被压迫民族和被压迫人民争取解放的斗争。

（1）结合材料，阐述新民主主义革命胜利的伟大意义。（8分）

（2）为什么说资产阶级共和国的方案在近代中国行不通，只有社会主义才能救中国？（9分）
:::

:::callout{kind=insight label="解析"}
**【考点】**第二章·新民主主义革命的意义/历史和人民的选择

**（1）新民主主义革命胜利的伟大意义**

①**理论意义**：新民主主义革命理论把马克思列宁主义同中国实际相结合，揭示了近代中国革命发展的客观规律，科学回答了中国革命向何处去的问题，极大地丰富和发展了马克思主义理论宝库。

②**实践意义**：新民主主义革命的胜利，结束了帝国主义、封建主义和官僚资本主义在中国的反动统治，建立了中华人民共和国，实现了中国从封建专制政治向人民民主的伟大飞跃，为实现从新民主主义到社会主义的转变创造了前提。

③**世界意义**：新中国的成立冲破了帝国主义的东方战线，壮大了世界社会主义和民族解放的力量，改变了世界政治格局，鼓舞和推动了世界被压迫民族和人民争取解放的斗争。

**（2）为什么资产阶级共和国方案行不通，只有社会主义才能救中国？**

①从**历史**看，近代中国的资产阶级革命派曾试图建立资产阶级共和国（如辛亥革命），但由于中国民族资产阶级的软弱性和妥协性，加之帝国主义、封建势力的强大，这些方案都以失败告终，中国依然处于半殖民地半封建社会。

②从**国情**看，帝国主义不允许中国成为独立的资本主义强国，中国缺乏发展独立资本主义的外部条件；国内封建势力、官僚资本又严重束缚生产力。

③从**人民选择**看，中国人民经过反复比较，选择了中国共产党的领导和社会主义道路。只有在无产阶级领导下先完成新民主主义革命、再走社会主义道路，才能实现民族独立、人民解放和国家富强。

④历史和实践证明：**只有社会主义才能救中国**，只有中国特色社会主义才能发展中国。
:::

### 第27题

:::callout{kind=note label="题目"}
**材料一：** 进入21世纪，我国经济社会发展面临一系列突出矛盾和问题：城乡、区域发展不平衡，经济增长的资源环境代价过大，社会事业发展相对滞后。以胡锦涛为主要代表的中国共产党人提出，必须坚持以人为本，全面、协调、可持续发展。

**材料二：** 习近平指出，绿水青山就是金山银山，要推动经济社会发展全面绿色转型，坚持人与自然和谐共生。中国式现代化是人与自然和谐共生的现代化。

:::

:::callout{kind=insight label="解析"}
**【考点】**第八章·科学发展观（结合生态文明、中国式现代化时事）

**（1）科学发展观的科学内涵**

①**第一要义是发展**。必须坚持发展这个党执政兴国的第一要务，用发展的办法解决前进中的问题。

②**核心立场是以人为本**。发展为了人民、发展依靠人民、发展成果由人民共享，尊重和保障人民的主体地位和根本利益。

③**基本要求是全面协调可持续**。全面推进经济、政治、文化、社会等各方面建设，统筹处理各种关系，实现可持续发展。

④**根本方法是统筹兼顾**。统筹城乡发展、区域发展、经济社会发展、人与自然和谐发展、国内发展和对外开放等重大关系。

**（2）对建设美丽中国的启示**

①科学发展观强调全面协调可持续，把可持续发展、人与自然和谐作为重要内容，为正确处理发展与资源环境的关系提供了指导。

②启示之一：必须**转变发展方式**，摒弃以牺牲资源环境为代价的粗放增长，推动经济社会发展全面绿色转型，实现高质量发展。

③启示之二：牢固树立**"绿水青山就是金山银山"**的理念，正确处理经济发展与生态保护的关系，把生态文明建设放在突出位置。

④启示之三：坚持**人与自然和谐共生**，走生产发展、生活富裕、生态良好的文明发展道路，建设美丽中国。这是对科学发展观可持续发展思想的继承和发展，也是中国式现代化的内在要求。
