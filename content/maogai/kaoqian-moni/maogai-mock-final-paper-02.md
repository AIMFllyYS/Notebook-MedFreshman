### 第1题

:::callout{kind=note label="题目"}
毛泽东思想被确立为党的指导思想并写入党章，是在（　　）

A. 遵义会议　　B. 党的六届六中全会

C. 党的七大　　D. 党的八大
:::

:::callout{kind=insight label="解析"}
**【答案】C**

1945年党的七大把毛泽东思想确立为党的指导思想并写入党章，刘少奇作《关于修改党章的报告》，对毛泽东思想作了系统阐述。遵义会议确立了毛泽东在党中央和红军的领导地位。
:::

:::callout{kind=note label="知识卡片：毛泽东思想指导地位的确立"}
| 会议/事件 | 意义 |
|----------|------|
| **遵义会议（1935）** | 确立毛泽东在党中央和红军的领导地位 |
| **党的七大（1945）** | 毛泽东思想写入党章，成为党的指导思想 |
| **易混项** | 六届六中全会提出"马克思主义中国化"；八大分析主要矛盾变化 |
:::

:::callout{kind=tip label="结论速记"}
毛泽东思想**写入党章、确立为指导思想** = 1945年**党的七大**（非遵义会议）。
:::

### 第2题

中国共产党第一次独立自主地运用马克思列宁主义基本原理解决自己的路线、方针和政策问题、开始形成以毛泽东为主要代表的党中央正确领导的会议是（　　）

A. 八七会议　　B. 古田会议

C. 遵义会议　　D. 瓦窑堡会议
:::

:::callout{kind=insight label="解析"}
**【答案】C**

1935年1月遵义会议集中解决了当时最紧迫的军事和组织问题，实际上确立了毛泽东在党中央和红军的领导地位，是党第一次独立自主地运用马克思列宁主义基本原理解决自身路线方针政策问题、生死攸关的转折点。
:::

:::callout{kind=note label="知识卡片：遵义会议的历史地位"}
| 要点 | 内容 |
|------|------|
| **时间** | 1935年1月 |
| **核心意义** | 第一次独立自主运用马列主义解决路线方针政策问题 |
| **领导地位** | 实际上确立毛泽东在党中央和红军的领导地位 |
| **历史定位** | 生死攸关的转折点 |
| **易混项** | 古田会议确立党对军队绝对领导；八七会议确定土地革命和武装反抗 |
:::

:::callout{kind=tip label="结论速记"}

### 第3题

:::callout{kind=note label="题目"}
毛泽东提出"没有调查，没有发言权"这一著名论断的著作是（　　）

A. 《实践论》　　B. 《反对本本主义》

C. 《改造我们的学习》　　D. 《矛盾论》
:::

:::callout{kind=insight label="解析"}
**【答案】B**

1930年毛泽东在《反对本本主义》中提出"没有调查，没有发言权"，强调调查研究的极端重要性，反对本本主义和教条主义，是实事求是思想路线形成的重要标志。
:::

:::callout{kind=note label="知识卡片：实事求是思想路线相关文献"}
| 著作 | 核心贡献 |
|------|---------|
| **《反对本本主义》（1930）** | "没有调查，没有发言权" |
| **《实践论》《矛盾论》（1937）** | 系统阐述认识论和辩证法 |
| **《改造我们的学习》（1941）** | 延安整风，批判主观主义 |
| **易混项** | 调查论断出自《反对本本主义》，非《实践论》 |
:::

:::callout{kind=tip label="结论速记"}
"没有调查，没有发言权" = **1930年《反对本本主义》**（实事求是思想路线的重要标志）。

### 第4题

:::callout{kind=note label="题目"}

| **封建主义** | 主要对象之一 |
| **官僚资本主义** | 主要对象之一（后纳入总路线） |
| **先决条件** | 推翻帝国主义压迫、争取民族独立 |
:::

:::callout{kind=tip label="结论速记"}

### 第6题

:::callout{kind=note label="题目"}
新民主主义革命的性质是（　　）

A. 无产阶级社会主义革命　　B. 资产阶级民主主义革命

C. 旧式的资产阶级民主革命　　D. 农民阶级的土地革命
:::

:::callout{kind=insight label="解析"}
**【答案】B**

新民主主义革命以反帝反封建为根本任务，任务尚未越出资产阶级民主革命的范畴，因此其性质仍是资产阶级民主主义革命，但它是无产阶级领导的、区别于旧式资产阶级民主革命的新式的资产阶级民主主义革命，其前途是社会主义。
:::

:::callout{kind=note label="知识卡片：新民主主义革命性质辨析"}
| 概念 | 内容 |
|------|------|
| **性质** | 资产阶级民主主义革命（任务未越出资产阶级民主革命范畴） |
| **"新"在哪里** | 无产阶级领导 + 前途是社会主义 |
| **区别于旧式** | 旧式由资产阶级领导；新式由无产阶级领导 |
| **易错项** | 不是社会主义革命（C/D错）；不是"旧式"（有"新"字修饰） |
:::

:::callout{kind=tip label="结论速记"}
新民主主义革命性质 = **资产阶级民主主义革命**（新式：无产阶级领导，前途社会主义）。
:::

### 第7题

:::callout{kind=note label="题目"}
在中国革命的前途问题上，主张"毕其功于一役"、把民主革命和社会主义革命合并为一次革命的错误是（　　）

A. 陈独秀的"二次革命论"　　B. 王明的"一次革命论"

C. 李立三的"左"倾冒险主义　　D. 张国焘的分裂主义
:::

:::callout{kind=insight label="解析"}
**【答案】B**

王明的"一次革命论"混淆民主革命和社会主义革命的界限，企图"毕其功于一役"，是"左"倾错误；陈独秀的"二次革命论"则割裂两个革命阶段的联系，主张民主革命由资产阶级领导，是右倾错误。二者都是错误的。
:::

:::callout{kind=note label="知识卡片：两种错误倾向对比"}
| 错误倾向 | 代表人物 | 错误实质 | 倾向 |
|---------|---------|---------|------|
| **一次革命论** | 王明 | 混淆两阶段，"毕其功于一役" | "左"倾 |
| **二次革命论** | 陈独秀 | 割裂两阶段，民主革命由资产阶级领导 | 右倾 |
| **正确观点** | 毛泽东 | 两阶段既区别又联系，由无产阶级领导 | — |
:::

:::callout{kind=tip label="结论速记"}
"毕其功于一役" = 王明**一次革命论**（"左"）；陈独秀**二次革命论** = 割裂两阶段（右）。
:::

### 第8题

:::callout{kind=note label="题目"}
抗日民族统一战线时期，中国共产党处理同顽固派关系的策略总方针是（　　）

A. 一切经过统一战线　　B. 发展进步势力、争取中间势力、孤立顽固势力

C. 又团结又斗争、以斗争求团结　　D. 一切服从统一战线
:::

**【答案】C**

抗日民族统一战线中，党提出发展进步势力、争取中间势力、孤立顽固势力的策略总方针，同时对顽固势力实行"又联合又斗争""以斗争求团结"的策略原则。A、D是王明右倾投降主义的错误主张。
:::

:::callout{kind=note label="知识卡片：统一战线策略体系"}
| 层次 | 内容 |
|------|------|

| **对顽固派原则** | 又联合又斗争、以斗争求团结 |
| **错误主张** | "一切经过统一战线""一切服从统一战线"（王明右倾） |
| **本题考点** | 问"处理同顽固派关系"→ 选"又团结又斗争" |

对顽固派 = **又团结又斗争、以斗争求团结**（C）；策略总方针（B）是宏观布局，非"对顽固派"的直接表述。

### 第9题

:::callout{kind=note label="题目"}
新中国成立初期，标志着我国进入由新民主主义向社会主义过渡时期的是（　　）

A. 1949年中华人民共和国成立　　B. 1952年国民经济恢复完成

C. 1954年第一部宪法颁布　　D. 1956年三大改造完成
:::

:::callout{kind=insight label="解析"}
**【答案】A**

1949年中华人民共和国成立，标志着新民主主义革命取得基本胜利，我国进入由新民主主义向社会主义过渡的时期。新民主主义社会是一个过渡性社会，不是独立的社会形态。1956年三大改造完成标志着过渡时期结束、进入社会主义社会。
:::

:::callout{kind=note label="知识卡片：过渡时期起止标志"}
| 时间节点 | 标志意义 |
|---------|---------|
| **1949年新中国成立** | 进入新民主主义→社会主义**过渡时期** |
| **1956年三大改造完成** | 过渡时期**结束**，进入社会主义社会 |
| **新民主主义社会** | 过渡性社会形态，非独立社会形态 |
| **易混项** | 1952恢复完成、1954宪法颁布均非过渡时期起点 |
:::

:::callout{kind=tip label="结论速记"}
过渡时期**起点** = 1949年新中国成立；**终点** = 1956年三大改造完成。
:::

### 第10题

:::callout{kind=note label="题目"}
我国对农业进行社会主义改造遵循的原则不包括（　　）

A. 自愿互利　　B. 典型示范

**【答案】D**

:::callout{kind=note label="知识卡片：农业社会主义改造原则与步骤"}

|------|------|
| **三大原则** | 自愿互利、典型示范、国家帮助 |
| **改造步骤** | 互助组 → 初级社 → 高级社（循序渐进） |
| **不包括** | 强制入社（违背自愿原则） |
| **教材位置** | 第三章·农业社会主义改造 |
:::

:::callout{kind=tip label="结论速记"}
农业改造三原则 = **自愿互利 + 典型示范 + 国家帮助**；"强制入社"不在其中。
:::

:::callout{kind=note label="题目"}
毛泽东在《论十大关系》中提出的处理中国工业化问题的基本方针是（　　）

:::callout{kind=note label="知识卡片：中国工业化道路"}

| **与苏联区别** | 不走片面强调重工业、忽视农轻的路子 |
| **易错项** | 不是"平均发展"或"只发展重工业" |
:::

:::callout{kind=tip label="结论速记"}

### 第14题

## 三、材料分析题（共 50 分）

### 第25题

:::callout{kind=note label="题目"}
阅读下列材料，回答问题。

> **材料一**：毛泽东在《中国社会各阶级的分析》开篇指出："谁是我们的敌人？谁是我们的朋友？这个问题是革命的首要问题。"
>
> **材料二**：在新民主主义革命进程中，中国共产党先后建立了国民革命联合战线、工农民主统一战线、抗日民族统一战线和人民民主统一战线，团结一切可以团结的力量，最终夺取了革命的胜利。

（1）为什么说"分清敌友"是中国革命的首要问题？（8分）

（2）结合材料二，说明统一战线为什么能够成为战胜敌人的重要法宝，以及党领导统一战线的基本经验。（9分）
:::

:::callout{kind=insight label="解析"}
**【答案要点】**

**（1）** ①中国革命要取得胜利，必须动员和团结一切革命力量，同时准确辨认和打击真正的敌人。分不清敌友，就会误把朋友当敌人、误把敌人当朋友，导致革命力量涣散甚至失败。②中国是一个阶级结构复杂的半殖民地半封建国家，各阶级对革命的态度不同：无产阶级是领导力量，农民是主力军，城市小资产阶级是可靠同盟军，民族资产阶级具有两面性，帝国主义、封建地主阶级和官僚资产阶级是革命对象。只有正确分析各阶级的经济地位和政治态度，才能制定正确的路线、方针和政策。③因此，分清敌友、团结真正的朋友以攻击真正的敌人，是中国革命的首要问题。

**（2）** ①统一战线成为法宝的根源在于中国国情：中国革命的敌人异常强大而凶残，无产阶级人数较少，必须团结农民、小资产阶级乃至民族资产阶级等一切可以团结的力量，才能以弱胜强、孤立和战胜主要敌人。②建立最广泛的统一战线，主要是解决两个联盟问题：一是工人阶级同农民及其他劳动者的联盟（基础性联盟），二是工人阶级同一切可以合作的非劳动人民尤其是民族资产阶级的联盟。③党领导统一战线的基本经验是：必须坚持无产阶级（共产党）在统一战线中的领导权；必须以工农联盟为基础；对资产阶级实行又联合又斗争的政策，以斗争求团结。只有坚持独立自主原则和正确的策略，统一战线才能真正发挥作用。

:::callout{kind=note label="知识卡片：分清敌友与统一战线"}
| 要点 | 内容 |
|------|------|
| **革命首要问题** | 分清敌友（《中国社会各阶级的分析》） |
| **法宝必要性** | 敌人强大 + 无产阶级人数少 → 必须团结一切可团结力量 |
| **两个联盟** | 工农联盟（基础）+ 与民族资产阶级等联盟 |
| **领导经验** | 坚持领导权 + 工农联盟为基础 + 又联合又斗争 |
| **四种统一战线** | 国民革命联合战线 → 工农民主 → 抗日民族 → 人民民主 |
:::

:::callout{kind=tip label="结论速记"}
革命首要问题 = **分清敌友**；统一战线法宝 = 国情决定 + 两个联盟 + 坚持领导权/工农联盟/又联合又斗争。

>
> **材料二**：习近平指出，坚持人民至上，一切为了人民、一切依靠人民，从群众中来、到群众中去。江山就是人民，人民就是江山。

（1）根据材料一，阐述实事求是的科学内涵及其在毛泽东思想中的地位。（8分）

（2）结合材料二，谈谈群众路线的基本内涵及其如何体现马克思主义的基本原理。（9分）
:::

:::callout{kind=insight label="解析"}
**【答案要点】**

**（1）** ①实事求是的科学内涵是：一切从实际出发，理论联系实际，在实践中检验真理和发展真理。"实事"是客观存在的一切事物，"是"是事物内部的规律性，"求"是去研究，即从实际出发探求并遵循客观规律。②实事求是是马克思主义的根本观点，是中国共产党人认识世界、改造世界的根本要求，是党的基本思想方法、工作方法、领导方法。③实事求是是毛泽东思想的精髓和活的灵魂之一，是马克思主义中国化时代化理论成果的精髓，贯穿于毛泽东思想的各个组成部分。

**（2）** ①群众路线的基本内涵是：一切为了群众，一切依靠群众，从群众中来，到群众中去，把党的正确主张变为群众的自觉行动。②坚持群众路线，要坚持人民是推动历史发展的根本力量，全心全意为人民服务，建立党同人民群众的血肉联系。③群众路线本质上体现了马克思主义关于人民群众是历史创造者的基本原理。人民群众是社会物质财富和精神财富的创造者、是社会变革的决定力量。"江山就是人民、人民就是江山"正是这一原理在新时代的生动表达。只有坚持人民至上、依靠人民、造福人民，党才能立于不败之地。
:::

### 第27题

:::callout{kind=note label="题目"}
阅读下列材料，回答问题。

> **材料一**：改革开放以来，我国经济持续快速发展，同时也出现了地区差距、城乡差距、收入差距扩大等问题。党中央强调，实现共同富裕是社会主义的本质要求，要在高质量发展中促进共同富裕。2021年，我国如期打赢脱贫攻坚战、全面建成小康社会。
>
> **材料二**：邓小平指出："社会主义的本质，是解放生产力，发展生产力，消灭剥削，消除两极分化，最终达到共同富裕。"

（1）根据材料二，说明邓小平关于社会主义本质的概括是如何从生产力和生产关系两个层面揭示社会主义本质的。（8分）

（2）结合材料一，谈谈如何理解"共同富裕是社会主义的本质要求"。（9分）
:::

:::callout{kind=insight label="解析"}
**【答案要点】**

**（1）** ①邓小平关于社会主义本质的概括包含两个层面。②在生产力层面："解放生产力，发展生产力"，突出了社会主义的根本任务是发展生产力，纠正了长期以来忽视生产力发展的倾向，回到了马克思主义把生产力发展作为社会主义首要任务的基本立场。③在生产关系层面："消灭剥削，消除两极分化，最终达到共同富裕"，指明了社会主义的发展方向和目标，只有坚持公有制主体地位、避免两极分化，才能实现共同富裕。④这一概括把发展生产力的手段与共同富裕的目标有机统一起来，深化了对科学社会主义的认识，是对马克思主义的丰富和发展。

**（2）** ①共同富裕是社会主义本质的内在规定和奋斗目标。社会主义之所以区别于资本主义，就在于它以消灭剥削、消除两极分化、实现全体人民共同富裕为目标。②共同富裕不是同步富裕、平均主义，而是在发展生产力、做大做好"蛋糕"的基础上，通过合理制度安排分好"蛋糕"，允许一部分人先富起来，先富带动后富，逐步实现共同富裕。③实现共同富裕必须坚持公有制为主体，发挥国家在二次分配中的调节作用，保障和改善民生。④我国打赢脱贫攻坚战、全面建成小康社会，是迈向共同富裕的重要一步；在高质量发展中促进共同富裕，正是对社会主义本质要求的自觉实践。
:::

:::callout{kind=note label="知识卡片：社会主义本质与共同富裕"}
| 层面 | 内容 |
|------|------|
| **生产力层面** | 解放生产力、发展生产力（根本任务） |
| **生产关系层面** | 消灭剥削、消除两极分化、最终共同富裕 |
| **共同富裕内涵** | 非同步富裕/非平均主义；先富带动后富 |
| **制度保障** | 公有制为主体 + 二次分配调节 |
| **实践进展** | 脱贫攻坚 + 全面小康 → 高质量发展中促进共同富裕 |
:::

:::callout{kind=tip label="结论速记"}
社会主义本质两层面：生产力=解放发展生产力；生产关系=消灭剥削消除两极分化→共同富裕；共同富裕≠同步/平均。
:::
