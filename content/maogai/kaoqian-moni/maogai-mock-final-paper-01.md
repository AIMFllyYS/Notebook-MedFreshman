### 第1题

:::callout{kind=note label="题目"}
毛泽东在党的六届六中全会所作的政治报告中首次提出了"马克思主义中国化"这一命题，这次全会召开的时间是（　　）

A. 1935年　　B. 1938年

C. 1942年　　D. 1945年
:::

:::callout{kind=insight label="解析"}
**【答案】B**

1938年党的六届六中全会上，毛泽东作《论新阶段》的政治报告，明确提出"使马克思主义在中国具体化"的任务，第一次正式提出"马克思主义中国化"命题。1935年是遵义会议，1945年是七大。
:::

:::callout{kind=note label="知识卡片：马克思主义中国化命题的提出"}
| 要素 | 内容 |
|------|------|
| **时间** | 1938年 |
| **会议** | 党的六届六中全会 |
| **报告** | 《论新阶段》 |
| **核心表述** | 使马克思主义在中国具体化 |
| **易混时间** | 1935年遵义会议；1945年七大 |
:::

:::callout{kind=tip label="结论速记"}
1938 · 六届六中全会 · 《论新阶段》= "马克思主义中国化"命题首次正式提出。
:::

### 第2题

:::callout{kind=note label="题目"}
马克思主义中国化时代化理论成果的精髓是（　　）

A. 解放思想　　B. 群众路线

C. 实事求是　　D. 独立自主
:::

:::callout{kind=insight label="解析"}
**【答案】C**

实事求是是马克思主义中国化时代化理论成果的精髓，也是毛泽东思想活的灵魂之一。其基本内涵是一切从实际出发，理论联系实际，在实践中检验真理和发展真理。
:::

:::callout{kind=note label="知识卡片：精髓与活的灵魂"}
| 概念 | 定位 |
|------|------|
| **精髓** | 实事求是（贯穿全部理论成果） |
| **活的灵魂** | 实事求是、群众路线、独立自主 |
| **易混项** | 解放思想是思想路线的要求；艰苦奋斗是精神品质，非"活的灵魂" |
:::

:::callout{kind=tip label="结论速记"}
理论成果精髓 = **实事求是**；活的灵魂三项 = 实事求是 + 群众路线 + 独立自主。
:::

### 第3题

:::callout{kind=note label="题目"}
近代中国社会的主要矛盾中，最主要的矛盾是（　　）

A. 封建主义和人民大众的矛盾　　B. 帝国主义和中华民族的矛盾

C. 资产阶级和无产阶级的矛盾　　D. 大地主大资产阶级和民族资产阶级的矛盾
:::

:::callout{kind=insight label="解析"}
**【答案】B**

近代中国半殖民地半封建社会的主要矛盾是帝国主义和中华民族的矛盾、封建主义和人民大众的矛盾，其中帝国主义和中华民族的矛盾是最主要的矛盾。这决定了反帝反封建是近代中国革命的主要任务。
:::

:::callout{kind=note label="知识卡片：近代中国社会矛盾"}
| 矛盾 | 地位 |
|------|------|
| 帝国主义和中华民族 | **最主要矛盾** |
| 封建主义和人民大众 | 主要矛盾之一 |
| 革命任务 | 反帝反封建（新民主主义革命） |
| 社会性质 | 半殖民地半封建社会 |
:::

:::callout{kind=tip label="结论速记"}
近代中国两对主要矛盾，**最主要** = 帝国主义 vs 中华民族 → 决定反帝反封建任务。
:::

### 第4题

:::callout{kind=note label="题目"}
毛泽东指出："认清中国的国情，乃是认清一切革命问题的基本依据。"这表明（　　）是解决中国革命问题的基本前提。

A. 认清中国的社会性质和国情　　B. 建立革命统一战线

C. 开展武装斗争　　D. 加强党的建设
:::

:::callout{kind=insight label="解析"}
**【答案】A**

认清中国国情是解决中国革命问题的基本前提。毛泽东强调，认清中国社会的性质即认清中国的国情，是认清一切革命问题的基本依据。近代中国的国情是半殖民地半封建社会，而非帝国主义与大地主阶级"权利财富各占一半"。
:::

:::callout{kind=note label="知识卡片：认清国情是基本前提"}
| 要点 | 内容 |
|------|------|
| **基本前提** | 认清中国国情（社会性质） |
| **基本依据** | 认清国情 → 认清一切革命问题 |
| **近代国情** | 半殖民地半封建社会 |
| **易错** | 统一战线、武装斗争、党的建设是三大法宝，非"基本前提" |
:::

:::callout{kind=tip label="结论速记"}
革命问题基本前提 = **认清国情**；国情 = 半殖民地半封建社会。
:::

### 第5题

:::callout{kind=note label="题目"}
标志着中国无产阶级开始作为独立的政治力量登上历史舞台、成为新民主主义革命开端的事件是（　　）

A. 辛亥革命　　B. 中国共产党成立

C. 五四运动　　D. 秋收起义
:::

:::callout{kind=insight label="解析"}
**【答案】C**

1919年五四运动中，中国无产阶级开始作为独立的政治力量登上政治舞台，成为新民主主义革命的开端。中国共产党成立是1921年，此后在革命实践基础上逐渐形成新民主主义革命理论。
:::

:::callout{kind=note label="知识卡片：新民主主义革命的开端"}
| 事件 | 意义 |
|------|------|
| **五四运动（1919）** | 无产阶级独立登上政治舞台；新民主主义革命开端 |
| 辛亥革命（1911） | 旧民主主义革命高峰，未改变社会性质 |
| 中共成立（1921） | 开天辟地大事变，此后逐步形成新民主主义革命理论 |
:::

:::callout{kind=tip label="结论速记"}
新民主主义革命开端 = **1919年五四运动**（无产阶级独立登上政治舞台）。
:::

### 第6题

:::callout{kind=note label="题目"}
毛泽东完整表述新民主主义革命总路线是在（　　）

A. 1939年《中国革命和中国共产党》　　B. 1948年在晋绥干部会议上的讲话

C. 1940年《新民主主义论》　　D. 1949年《论人民民主专政》
:::

:::callout{kind=insight label="解析"}
**【答案】B**

1939年毛泽东在《中国革命和中国共产党》中首次提出新民主主义革命是无产阶级领导的、人民大众的反帝反封建的革命；1948年在晋绥干部会议的讲话中完整表述了总路线，即"无产阶级领导的、人民大众的、反对帝国主义、封建主义和官僚资本主义的革命"，增加了反对官僚资本主义。
:::

:::callout{kind=note label="知识卡片：新民主主义革命总路线"}
| 阶段 | 时间/文献 | 内容 |
|------|----------|------|
| **首次提出** | 1939《中国革命和中国共产党》 | 反帝反封建（未含官僚资本主义） |
| **完整表述** | 1948晋绥干部会议 | 反帝反封建反官僚资本主义 |
| **总路线** | 无产阶级领导的、人民大众的、反对三座大山的革命 |
:::

:::callout{kind=tip label="结论速记"}
总路线完整表述 = **1948年晋绥干部会议**；1939年只是首次提出。
:::

### 第7题

:::callout{kind=note label="题目"}
新民主主义革命的中心问题，也是区别新旧民主主义革命的根本标志是（　　）

A. 农民问题　　B. 无产阶级领导权问题

C. 革命对象问题　　D. 革命前途问题
:::

:::callout{kind=insight label="解析"}
**【答案】B**

无产阶级领导权问题是中国革命的中心问题，也是新民主主义革命理论的核心问题，是区别新旧民主主义革命的根本标志。旧民主主义革命由资产阶级领导，新民主主义革命由无产阶级领导。
:::

:::callout{kind=note label="知识卡片：领导权是中心问题"}
| 概念 | 内容 |
|------|------|
| **中心问题** | 无产阶级领导权 |
| **根本标志** | 区别新旧民主主义革命 |
| **旧民主主义** | 资产阶级领导 |
| **新民主主义** | 无产阶级（通过共产党）领导 |
| **易混** | 农民问题是主力军问题，非中心问题 |
:::

:::callout{kind=tip label="结论速记"}
新旧民主主义革命根本标志 = **无产阶级领导权**（中心问题 + 核心问题）。
:::

### 第8题

:::callout{kind=note label="题目"}
在新民主主义革命的动力中，被称为中国革命"主力军"的是（　　）

A. 无产阶级　　B. 农民阶级

C. 城市小资产阶级　　D. 民族资产阶级
:::

:::callout{kind=insight label="解析"}
**【答案】B**

无产阶级是中国革命最基本的动力和领导力量；农民阶级是中国革命的主力军；城市小资产阶级是无产阶级可靠的同盟军；民族资产阶级具有两面性。毛泽东指出中国革命实质上是农民革命。
:::

:::callout{kind=note label="知识卡片：革命动力四阶层"}
| 阶级 | 定位 |
|------|------|
| 无产阶级 | 最基本动力 + 领导力量 |
| **农民阶级** | **主力军**（革命实质是农民革命） |
| 城市小资产阶级 | 可靠同盟军 |
| 民族资产阶级 | 具有两面性（又联合又斗争） |
:::

:::callout{kind=tip label="结论速记"}
革命动力速记：领导=无产阶级，主力=**农民**，同盟=城市小资产阶级，两面=民族资产阶级。
:::

### 第13题

:::callout{kind=note label="题目"}
党在过渡时期总路线的主体是（　　）

A. 对农业的社会主义改造　　B. 对手工业的社会主义改造

C. 社会主义工业化　　D. 对资本主义工商业的社会主义改造
:::

:::callout{kind=insight label="解析"}
**【答案】C**

过渡时期总路线的核心要义是"一化三改"，即逐步实现国家的社会主义工业化，并逐步实现对农业、手工业和资本主义工商业的社会主义改造。毛泽东用"一体两翼"比喻：工业化是主体，三大改造是两翼。
:::

:::callout{kind=note label="知识卡片：过渡时期总路线"}
| 要素 | 内容 |
|------|------|
| **主体（一体）** | 社会主义工业化 |
| **两翼（三改）** | 农业、手工业、资本主义工商业改造 |
| **比喻** | 一体两翼 |
| **关系** | 工业化与改造相互促进、相辅相成 |
:::

:::callout{kind=tip label="结论速记"}
过渡时期总路线主体 = **社会主义工业化**（一体）；三改是两翼。
:::

### 第14题

:::callout{kind=note label="题目"}
我国对资本主义工商业进行社会主义改造采取的具有中国特点的方式是（　　）

A. 没收　　B. 无偿剥夺

C. 和平赎买　　D. 强制合并
:::

:::callout{kind=insight label="解析"}
**【答案】C**

我国对资本主义工商业采取和平赎买的方式，通过国家资本主义的过渡形式（初级形式的国家资本主义、个别企业的公私合营、全行业的公私合营）逐步改造，把资本家改造成为自食其力的劳动者。这是马克思主义中国化的独创性贡献。
:::

:::callout{kind=note label="知识卡片：资本主义工商业改造"}
| 要点 | 内容 |
|------|------|
| **方式** | 和平赎买（中国独创） |
| **过渡形式** | 国家资本主义（初→个别公私合营→全行业公私合营） |
| **目标** | 把资本家改造为自食其力的劳动者 |
| **对比** | 苏联采取没收政策 |

**【答案要点】**

**（1）** ①"一体"是社会主义工业化，"两翼"是对农业、手工业和资本主义工商业的社会主义改造，二者相互促进、相辅相成、不可分割。②社会主义工业化是改造的物质基础和目的：只有发展工业，才能为改造提供技术装备和物质条件。③三大改造是工业化的必要条件和保证：只有变生产资料私有制为公有制，把分散的个体经济和资本主义经济改造为社会主义经济，才能使国家有计划地集中财力物力优先发展重工业、推进工业化。二者体现了发展生产力与变革生产关系的统一。

**（2）** 这一观点是片面的。①从实践看，50年代进行社会主义改造是历史的必然选择：它建立起社会主义基本制度，解决了国家工业化、巩固国防和新生政权等重大问题，且当时具备相应的经济政治条件。改革开放后允许多种所有制经济发展，是因为社会主义初级阶段生产力水平不高，必须建立与生产力发展相适应的生产关系，这是社会主义制度的自我完善，并未改变社会主义性质。二者都坚持了从实际出发、实事求是。②从理论看，社会主义改造创造性地回答了经济文化落后国家能否建立社会主义、如何用和平方法改造资产阶级、如何引导农民走社会主义道路等历史性课题，丰富和发展了马克思主义。③因此，社会主义改造与改革开放并不矛盾，都是围绕建设社会主义这一主题的探索，前者为后者奠定了制度基础和物质条件。
:::

:::callout{kind=note label="知识卡片：社会主义改造的必要性与评价"}
| 要点 | 内容 |
|------|------|
| **一体两翼** | 工业化（主体）+ 三大改造（两翼），生产力与生产关系统一 |
| **改造必要性** | 建立社会主义制度、推进工业化、巩固政权（历史必然） |
| **与改革开放关系** | 不矛盾：改造奠定基础，改革是制度自我完善 |
| **理论贡献** | 和平赎买、引导农民走社会主义道路等，丰富马克思主义 |
| **辨析题** | "早知今日何必当初"是片面观点（课堂重点） |
:::

:::callout{kind=tip label="结论速记"}
一体两翼 = 工业化（物质基础和目的）↔ 三改（必要条件和保证）；改造与改革不矛盾 = 改造奠基 + 改革完善，都体现实事求是。
:::

### 第27题

:::callout{kind=note label="题目"}
阅读下列材料，回答问题。

> **材料一**：邓小平说："计划多一点还是市场多一点，不是社会主义与资本主义的本质区别。计划经济不等于社会主义，资本主义也有计划；市场经济不等于资本主义，社会主义也有市场。计划和市场都是经济手段。"
>
> **材料二**：习近平在纪念毛泽东同志诞辰130周年座谈会上强调，要坚持独立自主开拓前进道路，把国家和民族发展放在自己力量的基点上，坚定不移走中国式现代化道路。当前，面对某些国家的"脱钩断链""小院高墙"，中国更加坚定地把发展的主动权牢牢掌握在自己手中。

（1）根据材料一，说明邓小平的这段话对建立社会主义市场经济体制的意义。（8分）

（2）结合材料二，谈谈毛泽东思想活的灵魂之一——独立自主，对今天走好中国式现代化道路的现实启示。（9分）
:::

:::callout{kind=insight label="解析"}
**【答案要点】**

**（1）** ①这段话从根本上破除了把计划经济等同于社会主义、把市场经济等同于资本主义的传统观念束缚，是重大的思想解放。②它科学揭示了计划和市场都是资源配置和发展经济的手段，不是社会制度的本质属性，社会主义也可以搞市场经济。③它为党的十四大确立建立社会主义市场经济体制的改革目标提供了理论依据，为社会主义与市场经济的结合扫清了思想障碍，是解放思想、实事求是思想路线的生动运用。

**（2）** ①独立自主就是坚持独立思考、走自己的路，把国家和民族发展放在自身力量的基点上。它是毛泽东思想活的灵魂之一，贯穿于中国革命、建设和改革的全过程。②启示之一：坚定道路自信，从本国国情出发探索适合自己的发展道路，中国的问题必须由中国人自己来解答，中国式现代化不能照搬他国模式。③启示之二：把发展主动权掌握在自己手中，面对外部"脱钩断链""技术封锁"，必须坚持自力更生、自主创新，突破关键核心技术，同时坚持对外开放、积极争取外援，二者不可偏废。④启示之三：坚决维护国家主权、安全和发展利益，在平等基础上发展对外关系，既不屈从外部压力，也不搞封闭僵化。
:::

:::callout{kind=note label="知识卡片：社会主义市场经济与独立自主"}
| 主题 | 要点 |
|------|------|
| **邓小平南方谈话** | 计划和市场都是经济手段，非制度本质属性 |
| **思想解放** | 破除计划=社会主义、市场=资本主义的传统观念 |
| **实践意义** | 为十四大确立社会主义市场经济体制目标提供理论依据 |
| **独立自主内涵** | 独立思考、走自己的路、放在自身力量基点上 |
| **当代启示** | 道路自信 + 自力更生与开放结合 + 维护主权安全发展利益 |
:::

:::callout{kind=tip label="结论速记"}
计划市场都是**手段**（非制度本质）→ 为社会主义市场经济扫障；独立自主 = 走自己的路 + 掌握主动权 + 自力更生与开放并举。
:::
