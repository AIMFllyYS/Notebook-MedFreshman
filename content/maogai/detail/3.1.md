# 3.1 毛泽东思想的形成和发展

## 引言：毛泽东思想何以成为马克思主义中国化的第一次历史性飞跃？

毛泽东思想是马克思主义中国化的第一个重大理论成果，是马克思列宁主义在中国的创造性运用和发展，是被实践证明了的关于中国革命和建设的正确的理论原则和经验总结，是中国共产党集体智慧的结晶。要理解毛泽东思想为什么能够成为这样伟大的理论成果，我们需要从两个维度来认识：一是毛泽东思想形成和发展的**历史条件**——什么样的时代和什么样的人孕育了这一思想；二是毛泽东思想形成和发展的**历史过程**——它是在怎样的革命实践中一步步成熟起来的。

:::definition{label=毛泽东思想}
**毛泽东思想**是马克思列宁主义在中国的创造性运用和发展，是被实践证明了的关于中国革命和建设的正确的理论原则和经验总结，是中国共产党集体智慧的结晶，是党必须长期坚持的指导思想。
:::

## 一、毛泽东思想形成和发展的历史条件

毛泽东思想形成和发展的历史条件，可以从**个人条件**和**社会条件**两个层面来理解。个人条件回答的是"为什么是毛泽东"，社会条件回答的是"为什么是那个时代"。

### （一）个人条件：毛泽东的四个突出特征

毛泽东思想以毛泽东命名，与毛泽东本人的品格、学识和革命经历密不可分。要理解毛泽东思想的形成，首先要了解毛泽东是怎样一个人。毛泽东1893年12月26日出生在湖南湘潭县韶山冲的一个农家，是一位农家子弟。从他身上，我们可以概括出四个鲜明的特征。

#### 1. 不畏强权，敢于反抗封建专制制度的压迫

毛泽东从小就表现出强烈的反抗精神，不惧怕任何形式的强权和不公。

在家庭中，毛泽东的父亲是一个封建专制式的家长。在毛泽东13岁那年，家中来了许多客人，少年毛泽东当着众人的面与父亲发生了争吵。父亲要打他，毛泽东转身就跑，跑到门前的池塘边时停了下来，转过身对父亲说：如果你再追过来打我，我就跳下这个池塘。面对儿子的威胁，父亲不得不压住怒火。在亲友的劝说下，毛泽东道了歉，但只跪下一条腿，并声明"跪下的一条腿是为母亲跪的，站着的一条腿才代表自己"。

在私塾读书时，课堂上要求学生背诵四书五经，并且有严格的礼节——学生要走到老师面前，反背双手站着背诵。毛泽东对此非常反感，对老师说："你听得清楚我背书，而且你坐在那里，我为什么要站起来背呢？"后来害怕老师告诉家长，毛泽东干脆跑了，在山林里转了三天才被村里人找回来。毛泽东后来对斯诺回忆说："回到家以后想不到情况有点改善……我的抗逆行为的效果给了我深刻印象，这是一次胜利的罢课。"

走向社会后，毛泽东同样敢于反抗不法分子。1924年，毛泽东和张国焘因劳动工会的事从上海乘船前往香港，在船上遇到一伙流氓抢劫。毛泽东挺身与之打斗，而张国焘则劝他不要为几个小钱拼上性命。最终因力量悬殊加上旁人劝解才作罢。

:::note{label=课堂启示}
毛泽东这种不畏强权、敢于反抗的性格，使他在后来的革命生涯中，面对帝国主义、封建主义和官僚资本主义的压迫，始终保持着斗争的勇气和决心。正如习近平总书记所评价的，毛泽东表现出"坚定不移的革命信念"和"炉火纯青的斗争艺术"。
:::

#### 2. 对社会穷苦人民的深深同情

毛泽东从小就对社会底层人民怀有深切的同情心，这种同情不是停留在口头上的，而是体现在日常生活的一个个具体行动中。

读书的时候，毛泽东常常把自己的午饭分给一位家境贫困的同学。他的母亲知道后，不仅没有责怪，反而专门给他带了双份的午饭。毛泽东因此说他的母亲是一位"有着菩萨心肠的中国的传统妇女"。有一次上学路上，遇到一位乞讨的大婶，毛泽东把自己上学的口粮——两担白米——全部送给了她。还有一年秋收时下大雨，毛泽东不先收自家的稻谷，而是跑去帮邻居抢收。当受到父亲责罚时，他说："人家是佃户，要交租，冲走一点就了不得了；我们是自己的田，又比别人多一些，冲走一些也不要紧。"

1906年，长沙因饥荒发生了大规模的抢米暴动，引发一系列流血事件。消息传到韶山，毛泽东的朋友们大多站在暴动者一边表示同情，而13岁的毛泽东却看到了更深一层。他说："我却从此把它记在心上。我觉得跟暴民在一起的，也是像我自己家里人那样的普通人，对于他们受到的冤屈我深感不平。"

:::insight{label=同情心与革命动力}
毛泽东对穷苦人民的深深同情，不仅仅是一种个人品德，更是他日后走上革命道路的根本动力。正是因为深切感受到人民的苦难，他才会义无反顾地投身于改变中国社会的革命事业，才会在《中国社会各阶级的分析》中认真分析各阶级的处境，才会得出"革命的成败取决于农民"的结论。
:::

#### 3. 坚定的民族独立关切

毛泽东是一位坚定的民族主义者，他一生致力于使中国成为一个真正意义上独立的国家。

1919年3月，毛泽东送蔡和森等朋友前往法国留学，但他自己没有去。表面上有家庭原因——母亲生了重病需要照顾，家庭经济状况也不足以支撑留学，同时他正与杨开慧热恋。但根本原因在于：毛泽东出国的愿望并不强烈。他说："我觉得我对自己的国家了解的还不够，把我的时间花在中国会更有用处。"

毛泽东所理解的民族独立，主要指的是像中国这样落后的民族，对外应该获得应有的独立权利。这一信念贯穿了他的一生——从领导新民主主义革命到建立新中国，从抗美援朝到发展国防科技，他始终将国家独立和民族尊严放在至高无上的位置。

1927年八七会议上，共产党代表罗伊这样描述他第一次见到毛泽东的印象："一个高个子走了进来，步伐沉重而显得高傲……鲍罗廷悄声对我说：'这个人很难对付，是个典型的中国人。'"这个"典型的中国人"的评价，恰恰说明了毛泽东身上那种深沉的民族气质。

#### 4. 坚定的马克思主义者

1918年6月，毛泽东从长沙第一师范毕业后，正在形成自己的思想。当时，传统的封建专制主义思想已经失去了现实意义，而西方的自由主义道路他又无法认同——"他不可能在西方化的中国人身上找到认同"。在这种思想困惑中，毛泽东说："在惊心动魄的社会变迁中需要新的东西产生。"

带着对新思想的渴望，毛泽东来到了北京——当时新文化运动的中心。在北京大学图书馆，他谋得了一份月薪8块钱的工作——清理书架、打扫房间、登记阅览者姓名。毛泽东后来回忆说："由于我的职位低下，人们都不愿意同我来往。我曾经试图同他们交谈政治和文化问题，可是他们都是大忙人，没有时间听一个图书管理员讲南方话。"

尽管如此，在此期间毛泽东迅速朝着马克思主义方向发展。对他影响最大的是李大钊。毛泽东后来回忆说："李大钊帮助我成为了马克思主义者。"1917年十月革命胜利后，毛泽东在天安门广场亲耳听到了李大钊关于十月革命的著名演说。李大钊的文章《布尔什维克的胜利》刊登在《新青年》杂志上，使毛泽东开始具体了解十月革命和马克思主义。

1919年4月，毛泽东回到长沙，投身新文化运动和反对帝国主义运动。7月，他主编的湖南省学联刊物《湘江评论》创刊，在创刊号上发表了《民众的大联合》一文，热情歌颂十月革命，要求工人、农民、学生等联合起来进行革命。

1919年冬天，毛泽东再次来到北京，读了三本马克思主义著作——马克思恩格斯的《共产党宣言》、考茨基的《阶级斗争》、柯卡普的《社会主义史》。这三本书使他树立了马克思主义信仰。**自1920年夏天开始，毛泽东认为自己已经是一个马克思主义者了，并且以后从未动摇过。**

:::definition{label=毛泽东成为马克思主义者的标志}
**1920年夏**，毛泽东通过阅读《共产党宣言》《阶级斗争》《社会主义史》三部著作，树立了马克思主义信仰，成为一名坚定的马克思主义者。**1921年7月**，毛泽东作为代表参加了中国共产党第一次全国代表大会，成为中国共产党的创始人之一。
:::

### （二）社会条件：时代呼唤新的理论

毛泽东思想的形成，离不开20世纪上半叶中国社会的深刻变革。近代以来，中国逐步沦为半殖民地半封建社会，民族危机空前深重。在这样的历史背景下，中国的先进分子迫切需要找到一条救国救民的道路。

1917年十月革命的胜利，给中国送来了马克思列宁主义。新文化运动的兴起，为马克思主义在中国的传播创造了思想条件。北京作为新文化运动的中心，成为各种新思想交汇碰撞的舞台。正是在这样的社会条件下，毛泽东等一批先进分子开始接受并传播马克思主义，并在与中国革命实践的结合中，逐步形成了具有中国特色的革命理论。

:::insight{label=个人条件与社会条件的交汇}
毛泽东思想的形成，是个人条件与社会条件共同作用的结果。没有毛泽东个人的品格、学识和革命经历，就不会有以他名字命名的思想体系；没有近代中国社会的深刻变革和新文化运动的思想启蒙，马克思主义也不可能在中国生根发芽。个人与时代的交汇，造就了毛泽东思想这一马克思主义中国化的伟大理论成果。
:::

## 二、毛泽东思想形成和发展的过程

### （一）从调查研究到理论创造：早期革命实践

毛泽东思想不是在书斋中产生的，而是在革命实践的烽火中一步步形成的。毛泽东有一个鲜明的特点——**注重调查研究，用脚来丈量祖国的大地**。

1925年，毛泽东从上海回到湖南开展农民运动，随后前往广州主持全国农民运动讲习所。在此期间，他深入社会进行调查研究，写出了《中国社会各阶级的分析》和《国民革命与农民运动》两篇重要著作。在这两篇文章中，他得出了一个至关重要的结论：**革命的成败取决于农民**。

毛泽东写道："若无农民从农村中奋起，打倒宗法封建的地主阶级之特权，则军阀和帝国主义势力总不会根本坍塌。"

:::insight{label=卡尔·马克思降临到亚洲的稻田上}
马克思的社会理论建立在城市工人阶级的基础之上——工厂里的工人奋起进行社会主义革命来建立社会主义。但毛泽东根据中国的实际情况，指出中国的社会主义革命虽然需要工人阶级领导，但必须依靠农民。正是从《中国社会各阶级的分析》和《国民革命与农民运动》开始，马克思主义理论与中国农村的实际情况结合起来。因此，西方学者评价说："卡尔·马克思降临到了亚洲的稻田上。"这是对毛泽东将马克思主义与中国实际相结合的生动概括。
:::

### （二）井冈山时期：农村包围城市道路的开辟

1927年10月，毛泽东率领秋收起义的部队上了井冈山，随后朱毛会师，创建了中国工农革命军——中国人民解放军的前身。毛泽东也因此成为中国人民解放军的主要缔造者。

井冈山是一个什么样的地方？穷乡僻壤，面对国民党的重重围攻。在这样艰苦的环境中，红色政权到底能不能存在下去？"红旗到底能够打多久？"这是当时林彪等人的疑问。

毛泽东在给林彪的回信——《星星之火，可以燎原》中，提出了中国革命和武装斗争的基本思想，标志着毛泽东关于**建立农村根据地，以农村包围城市，最后武装夺取政权**的道路理论开始形成。

:::definition{label=农村包围城市、武装夺取政权}
毛泽东在井冈山时期提出的革命道路理论：在敌强我弱的条件下，先在农村建立革命根据地，积蓄力量，以农村包围城市，最终武装夺取全国政权。这一理论是毛泽东思想形成的重要标志。
:::

### （三）延安时期：毛泽东思想的成熟

从井冈山到延安，毛泽东思想在艰苦的革命斗争中不断丰富和成熟。延安时期，毛泽东在陕北的窑洞里写出了大量重要著作，系统地回答了中国革命的一系列重大问题。

这一时期，毛泽东留下了许多经典话语，每一句话都凝聚着深刻的理论思考和对革命实践的精辟总结：

| 时间 | 经典话语 | 背景与意义 |
|------|---------|-----------|
| 1930年 | "星星之火，可以燎原" | 井冈山时期，指明中国革命方向，鼓舞革命信心 |
| 1939年 | "人不犯我，我不犯人；人若犯我，我必犯人" | 面对国民党顽固派的军事进攻，表达反抗决心 |
| 1939年 | "自己动手，丰衣足食" | 陕甘宁边区大生产运动，体现自力更生精神 |
| 1944年 | "为人民服务" | 纪念战士张思德，明确革命队伍的根本宗旨 |
| 1946年 | "一切反动派都是纸老虎" | 国共内战爆发，鼓舞革命人民战胜强敌的信心 |
| 1949年 | "夺取全国胜利，这只是万里长征的第一步" | 七届二中全会，提醒全党保持谦虚谨慎 |
| 1949年 | "中国人民从此站起来了" | 中国人民政治协商会议开幕词，宣告新中国的诞生 |
| 1957年 | "世界是你们的，也是我们的，但是归根结底是你们的" | 莫斯科大学会见中国留学生，对青年寄予殷切希望 |

:::note{label=从经典话语看思想脉络}
这些经典话语不是孤立的豪言壮语，而是毛泽东思想在不同历史节点上的集中表达。从"星星之火，可以燎原"到"中国人民从此站起来了"，我们可以清晰地看到一条线索：在极其艰苦的条件下，毛泽东始终以坚定的信念和科学的分析，为中国革命指明方向，最终领导中国人民实现了民族独立和人民解放。
:::

### （四）新中国成立后：社会主义建设的探索

新中国成立后，毛泽东思想在社会主义革命和建设的实践中继续发展。毛泽东领导了社会主义改造，确立了社会主义基本制度，并对社会主义建设道路进行了初步探索。

1957年，毛泽东第二次出访苏联。在与苏共领导人赫鲁晓夫的激烈争论中，他深刻认识到：一个国家如果落后了，在国际社会就没有话语权。因此，他对中国的发展寄予了深切期望，特别把希望寄托在青年一代身上。

然而，也要看到，1957年之后，社会主义建设的探索也走过了一些弯路，如1958年的大跃进、1959年的自然灾害等，这些都是探索过程中经历的曲折。正如习近平总书记所指出的，对毛泽东同志的评价应该放在其所处时代和社会历史条件下去认识，不能忽略历史条件的局限性。

## 三、毛泽东的革命牺牲与矢志不渝

理解毛泽东思想的形成，不能不了解毛泽东为革命付出的巨大牺牲。毛泽东一家为中国革命牺牲了六位亲人：夫人杨开慧（1930年牺牲，年仅29岁）、大弟毛泽民（1943年牺牲）、小弟毛泽覃（1935年牺牲）、堂妹毛泽健（全家牺牲）、侄子毛楚雄（1946年19岁时被敌人活埋），以及儿子毛岸英（1950年在朝鲜战场牺牲，年仅27岁）。

杨开慧被国民党逮捕后，面对劝降，她说："我死不足惜，惟愿润之革命早日成功。"1983年翻修杨开慧在长沙的旧居时，在天花板上发现了她写给毛泽东的信，信中写道："从此我有了一个新的意识，我觉得我为母亲所生之外，就是为了他。"

毛岸英在朝鲜战场牺牲后，消息直到1951年1月2日才由周恩来报告给毛泽东。听到消息后，毛泽东一声不响，过了许久才发出一声叹息："谁叫他是毛泽东的儿子。"随后一夜未眠，坐在藤椅上抽了一整夜的烟。毛岸英的遗物——一件衬衣、一双袜子、一条毛巾——被毛泽东珍藏在卧室的盒子里，直到他去世。

:::insight{label=伟人也是凡人}
毛泽东是伟大的革命领袖，但他也是一个父亲、一个丈夫、一个兄长。他对亲人怀有深沉的感情——为儿子遗物的珍视、对杨开慧的深深怀念、对弟弟毛泽覃又爱又恨的兄弟之情。正是这些革命者以自己的生命和亲情为代价，才换来了中国的独立和人民的解放。理解了这一点，我们才能更深刻地理解毛泽东思想的分量。
:::

正如习近平总书记在纪念毛泽东诞辰120周年大会上的讲话所评价的：**不管是翻江卷巨澜，还是雄关漫道真如铁，毛泽东同志都矢志不渝、执着追求**。他表现出一个伟大革命领袖高瞻远瞩的政治远见、坚定不移的革命信念、勇于开拓的非凡魄力、炉火纯青的斗争艺术、杰出高超的领导才能。他思想博大深邃，胸怀坦荡宽广，文韬武略兼备，领导艺术高超，心系人民群众，终身艰苦奋斗，为中华民族和中国人民建立了不朽的功勋。

:::definition{label=毛泽东的历史地位}
- **中国共产党的创始人之一**——1921年参加中共一大
- **中国人民解放军的主要缔造者**——创建中国工农革命军
- **中华人民共和国的主要缔造者**——领导新民主主义革命取得胜利
- **社会主义基本制度的奠基人**——领导社会主义改造和建设
- **马克思主义中国化的伟大开拓者**——毛泽东思想是马克思主义中国化的第一次历史性飞跃
:::
