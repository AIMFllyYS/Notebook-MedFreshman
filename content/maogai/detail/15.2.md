# 第十五章 第二节 · 祖国统一与对外关系

## 一、90年代以来对台方针政策的发展演变

:::definition{label=对台方针政策的三个里程碑}
- **江八条**（1995年）：江泽民提出发展两岸关系、推进祖国和平统一进程的八项主张。
- **胡四点**（2005年）：胡锦涛就新形势下发展两岸关系提出四点意见，以四个"绝不"为核心。
- **习五点**（2015年）：习近平就新形势下推进两岸关系发展提出五点重要主张。
:::

20世纪90年代以来，面对台独势力的上扬，中国共产党在"一国两制"基础上不断丰富和发展对台方针政策，形成了从江八条到胡四点再到习五点的政策演进脉络。

### （一）江八条（1995年）

1995年1月，江泽民发表了题为《为促进祖国统一大业的完成而继续奋斗》的重要讲话，提出现阶段发展两岸关系、推进祖国和平统一进程的八项主张。

**时代背景**：1995年前后，台独势力上扬。李登辉上台后推行分裂路线，同年美国总统克林顿邀请李登辉访美，引发中国政府强烈抗议，导致1995至1996年的第二次、第三次台海危机。

**核心内容**：江八条的前四条均围绕**一个中国原则**展开，强调一个中国是两岸关系不可动摇的基础。第五条涉及大陆发展与两岸经济交流合作。这体现了在台独势力抬头的严峻形势下，对台政策首先着眼于政治原则的坚守。

### （二）胡四点（2005年）

2005年，胡锦涛在看望参加全国政协十届三次会议的民革、台盟、台联界委员时，就新形势下发展两岸关系提出了四点意见。

**时代背景**：2000年民进党的陈水扁上台执政。民进党主张台独，陈水扁甚至声称上台后就要宣布独立。面对这一严峻局面，对台政策需要以更加坚定的态度予以回应。

:::note{label=四个"绝不"}
1. 坚持一个中国原则**绝不动摇**
2. 争取和平统一的努力**绝不放弃**
3. 贯彻寄希望于台湾人民的方针**绝不改变**
4. 反对分裂活动**绝不妥协**
:::

四个"绝不"以斩钉截铁的语气，表明了中国大陆在台湾问题上的坚定立场。也正是在2005年的全国人大会上，通过了《反分裂国家法》，首次以国家大法的形式维护国家统一。

### （三）习五点（2015年）

2015年，习近平总书记就新形势下推进两岸关系的发展提出了五点重要主张。

**核心内容**：
- **第一点**：坚持九二共识。
- **第二点**：深化两岸利益融合，共创两岸互利共赢。

:::insight{label=从江八条到习五点的政策重心变化}
习五点将经贸合作从江八条的第五点**上升到第二点**，体现了对台政策重心的与时俱进——在坚守一个中国原则的基础上，更加重视通过经济融合促进两岸关系和平发展。
:::

### （四）江八条、胡四点、习五点对比

| 维度 | 江八条（1995年） | 胡四点（2005年） | 习五点（2015年） |
|-----|---------------|---------------|---------------|
| 核心关切 | 一个中国原则（前四条） | 四个"绝不"坚守底线 | 九二共识 + 经济融合 |
| 经贸合作定位 | 第五条（较后位置） | 未单独列出 | 第二条（地位提升） |
| 应对的主要挑战 | 李登辉台独路线 | 陈水扁民进党执政 | 两岸关系新形势 |
| 政策基调 | 原则宣示为主 | 底线思维（绝不） | 融合发展 + 原则坚守 |

## 二、对台政策的四个核心维度

从江八条、胡四点到习五点，90年代以来对台方针政策可以概括为四个核心维度：

### （一）一个中国是原则

一个中国原则是两岸关系的**政治基础**。为维护这一原则，主要采取了以下措施：

- **坚持九二共识**：1992年10月，大陆的海峡两岸关系协会（海协会）与台湾的海峡交流基金会（海基会），就解决两岸事务性商谈中如何表明坚持一个中国原则的态度问题，达成了以口头方式表达的共识——海峡两岸均坚持一个中国原则。这就是"九二共识"。

:::definition{label=九二共识}
九二共识的核心意涵是：**海峡两岸均坚持一个中国原则**。十九大报告指出，一个中国原则是两岸关系的政治基础，体现一个中国原则的九二共识，明确界定了两岸关系的根本性质，是确保两岸关系和平发展的关键。二十大报告再次强调坚持一个中国原则和九二共识。
:::

- **主张和平统一，但不承诺放弃使用武力**：以警示和威慑的方式遏制台独势力。
- **通过《反分裂国家法》**（2005年）：这是全国人大通过的所有法律中，**唯一一个没有加上"中国"二字**的法律。之所以不加"中国"二字，是因为针对民进党"台湾不属于中国"的论调——反的是分裂"国家"这个行为，而这个国家就是中国，台湾是中国的一部分。该法首次以国家大法的形式宣示了中华人民共和国维护国家统一、领土完整的坚强意志。
- **大力促进国共两党交往**：在坚守九二共识的基础上，推动两岸政党交流。

### （二）经贸合作是基石

:::insight{label=两岸经贸数据}
- 中国大陆是台湾的**第一大贸易伙伴**。
- 大陆对台贸易长期处于**顺差**状态：
  - 2007年顺差达1201亿美元
  - 2021年顺差达1414亿美元
  - 2025年顺差达1426亿美元
- 台湾仅有2300万人口，2025年从大陆获得的贸易顺差**人均超过6000美元**。
:::

80年代改革开放以来，台湾同胞就开始到大陆投资。本世纪初以来，大陆继续欢迎台商投资，同时大力发展两岸经贸合作，推动两岸共同参加区域经济合作。通过发展两岸经贸关系，目标是建立**深度的相互依存关系**——尤其是台湾对大陆的依存。当这种依存达到"隔断彼此之间的联系，台湾失去的远远比大陆失去的"这一程度时，和平统一就有了坚实的经济基础。

### （三）人文交流是重心

新世纪以来，两岸人文交流主要体现在三个层面：

1. **发展"三通"**：通邮、通航、通商，便利两岸人员往来。
2. **拓展文化交流**：大陆高校招收台湾学生到大陆就读大学，台湾也接收大陆学生赴台学习。
3. **互帮互助**：在困难时刻两岸同胞相互支援。

:::definition{label=惠台31条措施（2018年）}
2018年2月28日，国台办发布《关于促进两岸经济文化交流合作的若干措施》，就台商经营及台湾民众的学习、创业、就业、生活等方面提出**31项惠台措施**。其核心要义是：给予在大陆学习、工作、经商的台湾同胞**与大陆同胞同等的待遇**。该措施在台湾民众调研中获得了**70%以上的认可度**。
:::

十九大报告明确提出：逐步为台湾同胞在大陆学习、创业、就业、生活提供与大陆同胞**同等的待遇**。这一政策体现了两岸同胞命运与共的血脉联系。

### （四）政治互信是关键

习近平总书记强调：**如果没有政治互信，那就地动山摇。** 政治互信是两岸关系和平发展的关键保障。在坚守九二共识、反对台独的共同政治基础上，两岸才能开展广泛深入的对话协商，共同推动两岸关系和平发展，推进祖国和平统一进程。

## 三、九二共识与国共交往的历史脉络

### （一）九二共识的形成

1992年10月，大陆海协会与台湾海基会在香港进行事务性商谈，就如何表明坚持一个中国原则的态度问题，达成了以口头方式表达的共识——"海峡两岸均坚持一个中国原则"。这一共识成为两岸关系和平发展的政治基础。

九二共识的另一面就是一个中国原则。二十大报告强调：在坚持一个中国原则和九二共识的基础上，推进同台湾各党派、各界人士就两岸关系和国家统一开展广泛深入协商，共同推动两岸关系和平发展，推进祖国和平统一进程。

### （二）国共领导人会面的历史脉络

| 时间 | 事件 | 意义 |
|-----|------|------|
| 1945年 | 重庆谈判：毛泽东与蒋介石会面 | 双方穿中山装面向记者，但**未握手**，预示内战不可避免 |
| 2005年 | 连战访问大陆，胡锦涛在人民大会堂会见 | 距上次领导人会面**60年后**的首次握手 |
| 2015年11月 | 习近平与马英九在新加坡会面 | 双方握手持续**80秒**，两岸领导人首次直接会面 |
| 2024年4月 | 国民党主席郑立文访京，习近平会见 | 延续国共交往传统 |

习近平总书记在会见郑立文时强调："我们愿在坚持九二共识、反对台独的共同政治基础上，同包括中国国民党在内的台湾各政党团体和社会各界人士一道，加强交流对话，为两岸谋和平、为同胞谋福祉、为民族谋复兴，把两岸关系的未来牢牢掌握在中国人自己手中。"

## 四、邓小平理论的历史地位与对外关系

### （一）邓小平理论的三大历史地位

:::note{label=邓小平理论历史地位的三个层面}
1. 马克思列宁主义、毛泽东思想的**继承和发展**
2. 中国特色社会主义理论体系的**开篇之作**
3. 改革开放和社会主义现代化建设的**科学指南**
:::

十九届六中全会通过的《中共中央关于党的百年奋斗重大成就和历史经验的决议》（2021年）对邓小平理论做出全面评价：以邓小平同志为主要代表的中国共产党人，深刻总结新中国成立以来正反两方面经验，围绕"什么是社会主义、怎样建设社会主义"这个根本问题，创立了邓小平理论，科学回答了建设中国特色社会主义的一系列基本问题，成功开创了中国特色社会主义。

### （二）作为开篇之作的系统回答

邓小平理论第一次比较系统地初步回答了中国社会主义的一系列基本问题：

| 基本问题 | 核心回答 |
|---------|---------|
| 发展道路 | 坚持社会主义基本制度基础上的改革开放 |
| 发展阶段 | 长期处于社会主义初级阶段 |
| 根本任务 | 以经济建设为中心 |
| 发展动力 | 改革开放 |
| 外部条件 | 和平与发展是时代主题 |
| 政治保障 | 四项基本原则 |
| 战略步骤 | 三步走 |
| 领导力量 | 党的领导 |
| 依靠力量 | 全国各族人民 |
| 祖国统一 | 一国两制 |

### （三）三步走战略的实践推进

:::insight{label=三步走战略的推进历程}
- **第一步**（90年代初）：解决温饱问题，全国人民吃饱了肚子。
- **第二步**（2000年）：进入小康社会，吃得好、穿得好、住得好、出行更方便。
- **第三步**：由全面建设小康社会到全面建成小康社会，再到社会主义强国建设的两个目标。
:::

三步走的发展战略一步步向前推进，改革开放使中国发生了翻天覆地的变化：思想得到解放，经济得到发展——"毛泽东让中国人站起来了，邓小平让中国人醒过来了、富起来了"。实践证明，邓小平理论是指导中国人民建设中国特色社会主义、保证中国在改革开放中实现国家繁荣富强和人民共同富裕的系统科学理论。

### （四）党的建设理论

邓小平理论的最后一个要点是党的建设理论。建设中国特色社会主义，关键在于坚持、加强和改善党的领导。具体包括五个方面：

1. 重视马克思主义理论学习
2. 加强党的组织建设
3. 加强领导班子建设
4. 加强党风建设
5. 重视制度建设

## 五、一国两制与两岸关系的未来展望

习近平总书记在纪念邓小平诞辰120周年座谈会上强调：针对台独图谋，我们"不能放弃用非和平方式"。同时，我们加强两岸经济文化交流合作，坚决反对和遏制分裂势力，牢牢掌握了两岸关系发展的**主导权和主动权**。

从"一纲四目"到"一国两制"，从江八条到胡四点再到习五点，对台方针政策始终在坚守一个中国原则的基础上与时俱进、开拓创新。一个中国是原则，经贸合作是基石，人文交流是重心，政治互信是关键——这四个维度构成了新时代对台政策的完整框架。我们期待宝岛台湾早日回到祖国的怀抱，共同实现中华民族的伟大复兴。
