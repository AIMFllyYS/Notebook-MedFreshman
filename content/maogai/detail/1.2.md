# 1.2 马克思主义中国化命题

## 引言：一个理论如何跨越万里改变了一个国家？

1848年，马克思和恩格斯在欧洲发表了《共产党宣言》。大约70年后，这个诞生于德国的理论经由俄国传入中国，从此深刻改变了这个东方古国的命运。这门课程要讲的核心故事，就是**马克思主义如何来到中国、如何与中国实际相结合、又如何指导中国走上了社会主义道路**。

要理解"马克思主义中国化"这个命题，我们需要先回到源头——弄清楚马克思主义到底是什么，然后才能理解它为什么必须"中国化"，以及"中国化"之后产生了什么样的理论成果。

:::insight{label=本节的核心问题}
马克思主义是关于**无产阶级和全人类解放**的学说。但它诞生于欧洲资本主义高度发达的社会背景之下，要在中国这样一个半殖民地半封建的落后农业国落地生根，就必须与中国的具体国情相结合——这就是"马克思主义中国化"命题的由来。
:::

---

## 一、什么是马克思主义？

### 1. 马克思主义的基本定义

:::definition{label=马克思主义}
由马克思恩格斯创立的，为他们的后继者所发展的，以**反对资本主义、建设社会主义和实现共产主义**为目标的科学理论体系。其核心是**无产阶级和全人类的解放**。
:::

"解放"换一个词就是**自由**。马克思主义所描绘的，是人类作为一个整体不断走向自由的历程。在马克思恩格斯的理论框架中，人类走向自由需要经历**四个阶段**。

### 2. 人类解放的四个阶段

| 阶段 | 解放的对象 | 历史对应 | 完成者 |
|:---:|-----------|---------|-------|
| 第一阶段 | 从**自然界**的束缚中解放 | 人类由原始共产主义社会进入奴隶社会 | 人类整体的生产力发展 |
| 第二阶段 | 从**人的束缚**中解放（封建等级制度） | 资产阶级革命，建立资本主义社会 | 资产阶级 |
| 第三阶段 | 从**资本**的束缚中解放 | 无产阶级革命，建立社会主义社会 | 工人阶级（无产阶级） |
| 第四阶段 | 从**自身**的束缚中解放 | "自由人的联合体"——彻底的自由 | 留给后人探索 |

:::insight{label=马克思主义的核心任务}
马克思恩格斯的学说主要解决的是**第三个阶段**的问题——从资本的束缚中解放出来，建立一个社会主义社会来取代资本主义社会。但在俄国和中国，第二阶段的任务（反封建）并未彻底完成，这就产生了马克思主义必须"俄国化""中国化"的现实需要。
:::

### 3. 资本主义制度的建立与批判

要理解马克思主义为什么主张建立社会主义，必须先理解它对资本主义制度的分析。

**资本主义制度的兴起**可以追溯到13、14世纪欧洲商人的出现。商人的根本目的是追求财富，但他们发现自己的财产权利受到了**封建贵族**的压制——贵族凭借出生身份享有政治特权，在社会财富分配中居于优先地位。

为了获得平等追求财富的权利，资产阶级思想家打出了一个响亮的旗号：**"所有人都是生而自由且平等的"**。他们以这个理念为武器，发动了资产阶级革命（以1789年法国大革命为代表），推翻了封建专制制度，建立了资本主义社会。

:::definition{label=资本主义制度的两大支柱}
- **市场经济**：赋予每个人平等追求财富的权利
- **民主政治**：通过公开选举产生掌握权力的人，防止权力之手伸入经济领域，维护市场的公平竞争
:::

**资本主义的积极性**是显而易见的。正如恩格斯在19世纪90年代所评价的：资本主义制度在过去100年内所创造的财富，比人类过去1000年创造的财富还要多。原因在于，市场经济把所有劳动（包括人自身）都变成了可以计量的价值，极大地调动了人的劳动积极性。中国的经验也印证了这一点——1992年党的十四大建立社会主义市场经济体制后，中国的GDP一路飙升，到2010年成为世界第二大经济体。

**但资本主义也带来了严重的问题**——社会贫富分化。对此有两种解释：

| 观点 | 核心主张 | 解决方案 |
|------|---------|---------|
| **自由主义** | 贫富分化是平等规则下个人能力差异的自然结果 | 二战后西方改良：征税→二次分配→福利国家 |
| **马克思主义** | 贫富分化的根源是掌握生产资料的资本家对无产者的**剥削** | 无产阶级革命→建立社会主义制度 |

:::note{label=社会主义制度的历史定位}
社会主义制度不是凭空设计的，而是建立在对资本主义制度**深刻揭露和批判**的基础之上的。它要解决的核心问题是：在资本主义创造了巨大财富、但也制造了严重不平等之后，如何建立一个更加公平的社会制度。
:::

## 二、马克思主义中国化时代化的提出

### 1. 马克思主义的传播路径

马克思主义并不是直接从德国飞到中国的，它经历了一条独特的传播路径：

**德国（马克思恩格斯创立）** → **俄国（列宁的发展与十月革命）** → **中国**

正如毛泽东所说："十月革命一声炮响，给我们送来了马克思列宁主义。"因此，"马克思主义中国化"中的"马克思主义"不仅指马克思恩格斯的原创理论，还包括列宁在俄国的发展。

:::definition{label=马克思主义俄国化}
列宁将马克思主义的基本原理与俄国的具体实际相结合，领导了1917年十月革命，建立了世界上第一个社会主义国家——苏联。俄国的实践证明，社会主义革命不一定非要发生在资本主义最发达的国家，这为马克思主义传入中国提供了重要的理论和实践基础。
:::

### 2. 为什么必须"中国化"？

马克思主义诞生于19世纪的欧洲，面对的是资本主义高度发达的工业社会。而20世纪初的中国，是一个半殖民地半封建的落后农业国，与欧洲的情况截然不同。

这种差异意味着，马克思恩格斯提出的许多具体结论不能机械地照搬到中国。马克思主义提供的是一种**立场、观点和方法**——即认识世界和改造世界的思维工具——而不是一张可以直接照搬的施工图纸。

:::insight{label="中国化"的本质}
"马克思主义中国化"并不是要修改马克思主义的基本原理，而是要**运用**马克思主义的立场、观点和方法来研究和解决中国的实际问题。它要求做到"两个结合"：同中国具体实际相结合、同中华优秀传统文化相结合。
:::

### 3. 从"中国化"到"中国化时代化"

马克思主义中国化不是一个一劳永逸的过程，而是一个持续推进的历史进程。中国革命、建设、改革的不同历史时期面临着不同的时代课题，马克思主义必须不断地吸收新的时代内容，回答新的时代问题。这就是为什么今天我们讲的是"马克思主义中国化**时代化**"——不仅要与中国实际相结合，还要与时代发展相同步。

## 三、中国式现代化：贯穿课程的核心主线

### 1. "中国式现代化"概念的提出

"中国式现代化"是2022年党的二十大报告正式提出的概念，但其思想源头可以追溯到更早。

| 时间 | 事件 | 关键表述 |
|------|------|---------|
| 1978年3月 | 全国科学大会 | 邓小平提出建设工业、农业、国防、科学技术的"四个现代化" |
| 1978年10-11月 | 邓小平出访日本、东南亚 | 参观日本全自动汽车生产线后感叹"我知道什么是现代化了" |
| 1979年1-2月 | 邓小平访美9天 | 参观NASA、福特汽车公司（计算机控制生产线每小时装配50辆汽车） |
| 1979年12月 | 邓小平会见日本首相大平正芳 | 正式提出"我们要实现的四个现代化是中国式的四个现代化" |
| 2022年10月 | 党的二十大报告 | 系统阐述中国式现代化的中国特色和本质要求 |

### 2. 邓小平的现代化探索：从"看世界"到"走自己的路"

1978年，邓小平密集出访了一系列国家，这段经历深刻影响了他对现代化道路的思考：

| 出访时间 | 出访国家 | 观察与感悟 |
|---------|---------|-----------|
| 1978年1月 | 缅甸、尼泊尔 | 这些国家的现代化水平不如中国 |
| 1978年10月 | 朝鲜 | 朝鲜的现代化水平比中国强不了多少 |
| 1978年10月 | **日本** | 参观全自动汽车生产车间："我知道什么是现代化了"；乘坐新干线，一个字——"快" |
| 1978年11月 | 泰国、马来西亚 | 现代化水平并不比中国高多少 |
| 1978年11月 | **新加坡** | 独立仅十年的"亚洲四小龙"，走出了一条新加坡式现代化道路 |

:::insight{label=邓小平出访的三个层次感悟}
从日本和美国的现代化经验中，邓小平形成了对中国式现代化的三个层次认识：

1. **短期内不可能达到西方的现代化水平**——先解决"小康之家"的问题
2. **西方的先进技术与经验可以借鉴**——引进资金、技术、管理经验，用中国的市场来交换
3. **绝不能照搬西方的现代化道路**——必须走出一条中国自己的现代化道路
:::

出访回国后，邓小平参加了长达36天的中央工作会议，做了"解放思想、实事求是、团结一致向前看"的重要发言，确立了以经济建设为中心、对内改革、对外开放的路线。紧接着，1978年12月18日至22日，十一届三中全会召开（仅4天），向全国人民宣布了改革开放的历史性决策。

### 3. 中国式现代化的五大中国特色

党的二十大报告系统阐述了中国式现代化的五大特色，每一个特色都有深厚的历史根基：

#### （一）人口规模最大的现代化

2021年建党100周年之际，中国全面建成小康社会，历史性地解决了绝对贫困问题。几个关键数据：

- 近**1亿**农村贫困人口全部脱贫
- 中等收入群体超过**4亿**人
- 基本医疗保险覆盖**13亿**人
- 基本养老保险覆盖近**10亿**人

:::definition{label=全面建成小康社会}
指在2020年现行标准下，使近1亿农村贫困人口全部脱贫，建成世界上规模最大的社会保障体系，历史性地解决中国的绝对贫困问题。这是中国式现代化人口规模巨大的集中体现。
:::

#### （二）全体人民共同富裕的现代化

这条道路的起点，是"吃饱肚子"。

1978年十一届三中全会后，农村改革率先启动。1979年家庭联产承包责任制在全国推行，但争议巨大。1980年邓小平表态：**"看来我们农村经济的形式可以多样化一点。像贫困区，过去不是有个包产到户吗？可以试试。将来要改还可以改，先吃饱饭要紧。"**——肚子都没有吃饱，争个啥？

从"吃饱肚子"到全面建成小康社会，这条路走了40多年。

#### （三）物质文明和精神文明相协调的现代化

从"一穷二白"到今天，中国的物质文明和精神文明实现了巨大的飞跃。

:::definition{label=一穷二白}
毛泽东在1956年《论十大关系》中对中国经济文化状况的概括："我们一个穷，二个白——穷就是没有多少工业，农业也不发达；白就是一张白纸，文化水平、科学水平都不高。"
:::

今天，人形机器人已经从"扭秧歌的显眼包"进化到能够在舞台上完成武术表演、参与小品和歌曲演出。扩展现实、超高清智播等前沿技术打造的视听盛宴，既是物质文明的结晶，也满足了人民群众的精神文化需求。

#### （四）人与自然和谐共生的现代化

中国的工业化进程曾付出巨大的环境代价。以下数据触目惊心：

| 时间 | 环境问题 |
|------|---------|
| 2001年 | 世界银行公布全球污染最严重的20个城市，中国占**16个** |
| 2002年 | 中国**2/3**的城市空气质量达不到世界卫生组织标准 |
| 长期 | 环境污染和恶化造成的损失相当于GDP的**8%-12%** |

2012年以后，生态文明建设被纳入"五位一体"总体布局，环境治理取得显著成效。以武汉为例，空气质量优良天数从2013年的162天提升到2017年的255天。华中科技大学的喻家湖也从一条臭水沟改造为校园景观。

:::insight{label=从喻家湖看美丽中国建设}
华中科技大学喻家湖的变迁是一个缩影：十多年前这里是污水横流的臭水沟，夏天连路都不敢走；如今改造为校园景观，一年四季鲜花不断。这背后是"五位一体"总体布局中生态文明建设的落地，也是"美丽中国"建设的具体成果。
:::

#### （五）走和平发展道路的现代化

西方国家的现代化历史上伴随着对外的殖民掠夺和扩张。邓小平明确指出：我们不走那条路。

中国的现代化走的是**和平发展道路**——通过与世界各国进行全方位、多层次、宽领域的合作来实现自身发展。2010年，中国成为世界第二大经济体，日渐走进世界舞台的中央。

## 四、当代国际局势：理解"中国化"的时代背景

### 1. 世界进入新的动荡变革期

2021年，习近平总书记指出："百年未有之大变局会同新冠肺炎疫情，世界进入新的动荡变革期。"近年来的一系列国际事件印证了这一判断：

- **俄乌战争**（2022年至今）：俄罗斯对乌克兰发起特别军事行动，至今未结束
- **中东局势**：伊朗问题持续紧张
- **技术革命**：从2003年伊拉克战争的信息化战争（一周），到当下人工智能驱动的军事变革（几秒到几分钟）

### 2. 中美关系的历史演变

理解今天的国际局势，必须理解中美关系的历史演变。这段历史清晰地展现了中国从"不被承认"到"被视为对手"的历程：

| 时间 | 事件 | 美国对华态度 |
|------|------|------------|
| 1949年 | 新中国成立 | **拒不承认**新生的社会主义政权 |
| 1950-1953年 | 朝鲜战争 | 武装对抗 |
| 1954年 | 日内瓦会议 | 虽同意中国参会，但在公报中注明"不含有外交承认之意" |
| 1970年代 | 中美关系改善 | 从对抗转向接触 |
| 1979年 | 中美建交，邓小平访美 | 合作开启，但会场嘈杂、嘉宾对中国"不在意" |
| 2015年 | 习近平访美演讲 | 会场鸦雀无声，讲毕长时期掌声——中国已走进世界舞台中央 |
| 2018年3月 | 中美贸易冲突爆发 | 从贸易摩擦升级为**技术遏制** |
| 2022年10月 | 拜登政府国家安全战略 | 将中国定位为**"公开的战略竞争对手"** |

:::insight{label=从1979到2015：一个意味深长的对比}
1979年邓小平访美时，800名嘉宾的招待会会场嘈杂，因为一方面展厅没有为演讲做声学设计，另一方面"这些嘉宾对中国不了解，更重要的是对中国不在意"。2015年习近平访美时，200多位嘉宾鸦雀无声地聆听，讲毕报以长时期掌声。同一个国家，36年间天壤之别——这背后是中国综合国力的巨大提升。
:::

### 3. 拜登政府的五点核心判断

2022年10月，拜登政府公布的国家安全战略中，对中美关系做出了系统性的战略判断：

1. 应对中国挑战是美国的**全球优先事务**
2. 中国是美国面对的**最严重的地缘政治挑战**，是唯一一个既有意愿也越来越有能力重塑国际秩序的竞争者
3. 美国与中国的战略竞争**覆盖全球**，包括经济、技术、安全、国际机制等方面的全领域竞争
4. **未来十年**是美国与中国进行较量的决定性十年
5. 美国对华战略总目标是**"竞而胜之"**

### 4. "新冷战"的阴影

:::definition{label=冷战陷阱}
习近平总书记在国际舞台上多次表示，中美之间不要陷入过去美苏那样对抗的"冷战陷阱"中。当这个词被提出时，意味着中美关系已经出现了新冷战的对峙状态——双方在意识形态、发展模式、地缘政治、国际秩序等方面利益冲突不可调和，陷入对抗，但又无法承受直接军事冲突的后果。
:::

:::note{label=回到历史寻找答案}
面对今天中美关系的严峻局面，我们不应忘记：在毛泽东时代，中国面临的困难比今天更大——美国拒不承认新中国、在朝鲜半岛兵戎相见、全面封锁禁运。但中国共产党人不仅走出了困境，还发展和巩固了社会主义制度，以至于美国主动寻求改善对华关系。**了解这段历史，正是本课程的重要价值所在。**
:::

## 小结

本节围绕"马克思主义中国化"这一核心命题，完成了三个层次的论述：

- **理论源头**：马克思主义是关于无产阶级和全人类解放的学说，它深刻揭露和批判了资本主义制度，为建立社会主义制度提供了理论基础
- **命题提出**：马克思主义经由俄国传入中国，但中国半殖民地半封建社会的国情决定了它必须与中国具体实际相结合——这就是"马克思主义中国化"的命题
- **主线展开**：中国式现代化是贯穿本课程的核心主线。从1979年邓小平提出"中国式的现代化"，到2022年党的二十大系统阐述其五大特色，这条道路的探索本身就是马克思主义中国化时代化最生动的实践
