# 1.1 课程介绍与教学安排

## 引言：为什么大学还要上"政治课"？

进入大学，很多同学可能会问：为什么我们还要上思想政治理论课？这门"毛泽东思想和中国特色社会主义理论体系概论"（简称"毛概"）到底讲什么？它和我们之前学的"中国近现代史纲要"有什么区别？学了这门课，对我的人生到底有什么用？

这些问题并不幼稚，恰恰是理解这门课程的起点。本节将从课程定位、学习意义、教学安排三个维度，帮助大家建立对这门课程的整体认知。

:::insight{label=一句话理解这门课}
这门课不是让你背诵政治口号，而是帮你回答一个根本问题：**你所生活的这个国家，是怎么走到今天的？它为你的人生提供了什么样的机遇和条件？**
:::

## 一、这是一门什么课程？

### 1. 课程全称与核心定位

本课程全称**"毛泽东思想和中国特色社会主义理论体系概论"**，是一门思想政治理论课。它的核心任务是讲述：1848年诞生于德国的马克思主义理论，在20世纪不远万里来到中国，使中国变成一个社会主义国家的历史。

:::definition{label=马克思主义中国化时代化}
立足中国国情和时代特点，坚持把马克思主义基本原理同**中国具体实际**相结合、同**中华优秀传统文化**相结合，深入研究和解决中国革命、建设、改革不同时期的实际问题，科学回答时代提出的重大理论和实践课题，不断创新的理论成果。
:::

### 2. "一二三四"：课程的总体框架

理解这门课程，可以用**"一二三四"**四个数字来把握：

| 数字 | 内容 | 说明 |
|:---:|------|------|
| **一** | 一条主线 | 马克思主义中国化时代化 |
| **二** | 两大理论 | 毛泽东思想与中国特色社会主义理论体系 |
| **三** | 三个阶段 | 社会主义在中国的建立、建设、改革 |
| **四** | 四个自信 | 道路自信、理论自信、制度自信、文化自信 |

其中，将毛泽东思想和中国特色社会主义理论体系连接在一起的**纽带**，正是"马克思主义中国化时代化"。

### 3. 两大理论成果的历史定位

| 理论成果 | 历史定位 | 出处 |
|---------|---------|------|
| 毛泽东思想 | 马克思主义中国化**第一次历史性飞跃**的理论成果 | 党的七大章程；纪念毛泽东诞辰130周年讲话 |
| 中国特色社会主义理论体系（邓小平理论、"三个代表"重要思想、科学发展观） | 马克思主义中国化**最新成果** | 党的十七大报告 |
| 习近平新时代中国特色社会主义思想 | 马克思主义中国化时代化**新的飞跃** | 党的二十大报告 |

:::note{label=两大理论的关系}
毛泽东思想是中国特色社会主义理论体系的**源头和基础**；中国特色社会主义理论体系是对毛泽东思想的**继承和发展**。二者通过"马克思主义中国化时代化"这条主线一脉相承。
:::

## 二、为什么要学习这门课程？

### 1. "四个正确认识"：思政课的总目标

2016年12月，新中国成立以来首次全国高校思想政治工作会议召开。习近平总书记在会上提出，要通过教育引导学生树立**"四个正确认识"**：

| 序号 | 正确认识 | 核心要义 |
|:---:|---------|---------|
| 1 | 正确认识**世界和中国发展大势** | 认识中国特色社会主义道路，把握人类社会发展规律 |
| 2 | 正确认识**中国特色和国际比较** | 全面客观认识当代中国，正确看待外部世界 |
| 3 | 正确认识**时代责任和历史使命** | 将个人理想融入国家和民族事业，用中国梦激扬青春梦 |
| 4 | 正确认识**远大抱负和脚踏实地** | 将远大抱负落实到实际行动中，让勤奋学习成为青春飞扬的动力 |

:::insight{label=四个认识的逻辑递进}
前两个认识解决的是**"看清楚"**的问题——看清楚中国走的是什么道路、中国特色"特"在哪里；后两个认识解决的是**"怎么做"**的问题——树立什么样的个人理想、以什么样的方式去实现。
:::

### 2. 五门思政课的逻辑关系

"四个正确认识"贯穿于大学本科的五门思想政治理论课中。这五门课之间存在着清晰的逻辑递进关系：

| 课程 | 定位 | 核心问题 | 逻辑关系 |
|------|------|---------|---------|
| 思想道德与法治 | **修身课** | 如何树立正确三观和个人理想？ | 起点：个人理想的实现需要国家社会条件 |
| 中国近现代史纲要 | **历史课** | 中国人为什么选择了马克思主义？ | 回答"选择"的必然性 |
| 马克思主义基本原理 | **理论课** | 马克思主义到底是什么？ | 了解"工具"本身 |
| 习近平新时代中国特色社会主义思想 | **新时代课** | 新时代中国特色社会主义怎么建设？ | 认识"当下" |
| 毛泽东思想和中国特色社会主义理论体系概论 | **治国课** | 中国社会主义革命、建设和改革的历程是怎样的？ | 追溯"历史逻辑" |

:::definition{label=修身课与治国课的关系}
"思想道德与法治"偏重**修身**——帮助你树立个人理想；"毛概"偏重**治国**——帮助你理解国家和社会为个人理想实现提供了什么样的条件和机遇。正如课堂上所说的：**"时代的一粒灰落到一个人身上就是一座山，时代的一缕阳光落到一个人身上就是阳光灿烂。"**
:::

:::note{label=课程改革预告}
从下一年度起，"毛概"与"中国近现代史纲要"将合二为一，实现**史论结合**——一个偏重历史（时间、地点、人物、事件），一个偏重理论（在诸多事件中寻找规律性并上升到理论层面）。武汉大学等高校已率先试点。
:::

### 3. 课程目的：四个层次层层递进

教材第12至13页对开设本课程的目的做了概括，其核心可归纳为**四个层层递进的层次**：

| 层次 | 关键词 | 含义 |
|:---:|:---:|------|
| 第一层 | **了解** | 对革命、建设、改革的历史进程有更全面的系统了解 |
| 第二层 | **理解** | 对马克思主义中国化时代化有更深刻的理性理解 |
| 第三层 | **把握** | 对形成的理论成果有更准确的整体把握 |
| 第四层 | **能力提升** | 运用马克思主义立场、观点、方法分析和解决问题的能力明显提升 |

:::insight{label=为什么是"回到历史"？}
本课程的核心方法是**立足今天的中国特色社会主义建设，回到历史中去**。了解我们是怎样从"一穷二白"走到今天的，才能理解今天面对的挑战并不比历史上更难，也才能从历史中找到应对当下困难的启示。
:::

## 三、教学安排与考核方式

### 1. 课程时间安排

本课程共**17周**，分为课堂教学和晚间活动两部分：

| 时间段 | 内容 | 说明 |
|-------|------|------|
| 第1周 | 导论 | 课程介绍、马克思主义中国化导论 |
| 第3-9周 | 第一至四章（毛泽东思想） | 先讲毛泽东思想产生的社会条件（第1章），再讲主要内容（第2-4章），最后回到第1章评价历史地位 |
| 第10-16周 | 第五至七章（邓三科） | 邓小平理论（约3周）、"三个代表"重要思想、科学发展观 |
| 第17周 | 总结与复习 | 课程结束，第18周为考试周 |
| 第10-11周（晚间） | 专业展演活动 | 以小组为单位完成 |
| 第15周（晚间） | 时事热点讨论 | 以小组为单位完成 |

### 2. 成绩构成

| 项目 | 分值 | 说明 |
|------|:---:|------|
| **平时成绩** | **50分** | 包含以下子项 |
| ├ 考勤 | 20分 | 多次考勤，缺勤一次扣分（×2计入）；**必须事先请假**，事后不认（不可抗力除外） |
| ├ 专业展演 | 20分 | 小组完成，主讲人可加1-2分 |
| ├ 时事讨论 | 10分 | 小组完成 |
| └ 课堂发言 | 加分项 | 每次发言加1分，上限不超过50分 |
| **期末考试** | **50分** | 单选题（数量增加）+ 多选题（数量减少）+ 材料分析题 |

:::note{label=考试趋势}
近年来考试的一个变化是**增加了单选题、减少了多选题**，目的是提分——因为单选题得分率更高。经验表明：选择题做得好的同学，材料分析题往往也做得好，二者呈正相关。每章讲完后老师会带着梳理重点和历年考题。
:::

### 3. 教学方法与课程特色

本课程采用**问题导读为主的互动教学方法**，具体特点包括：

- **理论导向**：不同于"史纲"侧重时间、地点、人物、事件的讲述，本课程偏重于从历史事件中**提炼规律性**，上升到理论层面
- **专题深入**：在对教材内容整体概括的基础上，结合教师自身的研究方向，对重要专题进行深入的理论分析
- **原文研读**：今年增加了对毛泽东选集等经典文献**原文**的引用和研读
- **现实关怀**：以**中国式现代化道路探索**为主线，分析中国共产党领导中国人民取得的成就和经验

:::insight{label=毛概与史纲的核心区别}
两门课讲的是同一段历史，但视角截然不同。**史纲**告诉你"发生了什么"——时间、地点、人物、事件；**毛概**告诉你"为什么会这样"——在诸多事件中寻找内在规律，将其上升为理论。正因为偏重理论，毛概在部分同学看来会比史纲更"枯燥"一些，但它能帮助你建立**分析社会现象的思维框架**。
:::

## 四、当代大学生的时代处境

在正式进入课程内容之前，老师引导同学们讨论了一个问题：**列举新时代（2012年以来）对你感触最深的国内外大事**。课堂上五位同学的发言涵盖了以下主题：

| 发言主题 | 具体内容 | 与课程的关联 |
|---------|---------|-------------|
| 俄乌战争 | 俄罗斯对乌克兰发起特别军事行动，至今未结束 | 世界进入新的动荡变革期 |
| 美国对华遏制 | 美国抛弃自身宣扬的自由民主规则 | 中美战略博弈与新冷战 |
| 人工智能技术 | AI改变了学习和生活方式，对各行业产生冲击 | 技术革命与个人命运 |
| 日本核废水排放 | 从福岛产品到核废水排海 | 人与自然和谐共生的现代化 |
| AI在医学领域的应用 | AI辅助诊断发现医生遗漏的肿瘤 | 科学与人文相融的时代要求 |

:::note{label=课堂总结}
老师将同学们的发言归结为一个共同主题——**中国式现代化**。无论是国际局势的动荡、技术革命的冲击，还是环境治理的挑战，都可以纳入"中国式现代化"这一框架来理解。这也是贯穿本课程始终的**核心主线**。
:::

## 小结

本节从三个维度完成了对这门课程的整体介绍：

- **定位**：这是一门关于"马克思主义中国化时代化"的思想政治理论课，可用"一二三四"概括其框架
- **意义**：它帮助我们理解国家和社会为个人理想提供了什么样的条件和机遇，实现从"了解"到"理解"到"把握"到"能力提升"的层层递进
- **安排**：17周教学、平时成绩与期末考试各占50%、理论导向与互动教学相结合

理解了"为什么学"和"怎么学"之后，下一节将正式进入课程的核心命题——**马克思主义中国化**，看看这个命题是如何提出的，以及它与"中国式现代化"之间的内在关联。
