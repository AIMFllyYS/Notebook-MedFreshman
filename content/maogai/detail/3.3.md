# 3.3 毛泽东思想的历史地位

## 引言：如何科学评价毛泽东思想？

毛泽东思想是马克思主义中国化的第一次历史性飞跃，也是马克思主义中国化的第一个重大理论成果。正确认识和评价毛泽东思想的历史地位，不仅关系到对毛泽东个人的公正评价，更关系到对党的历史、对中国特色社会主义道路的根本认识。本节将从三个维度来理解毛泽东思想的历史地位：对毛泽东思想的**科学评价**、毛泽东思想与中国特色社会主义理论体系的**内在关系**、以及毛泽东思想作为党必须长期坚持的**指导思想**的重大意义。

## 一、对毛泽东思想的科学评价

### （一）三句话的科学定性

党对毛泽东思想作出了明确的科学评价，可以概括为三句话：

:::definition{label=对毛泽东思想的科学评价}
1. **马克思列宁主义在中国的运用和发展**——说明了毛泽东思想的理论来源和创新性质；
2. **被实践证明了的关于中国革命和建设的正确的理论原则和经验总结**——说明了毛泽东思想的实践基础和科学性质；
3. **中国共产党集体智慧的结晶**——说明了毛泽东思想的形成方式和集体性质。
:::

第一句话揭示了毛泽东思想的"根"和"魂"——它的理论根基是马克思列宁主义，但它不是对马克思列宁主义的简单照搬，而是在中国土壤上的**创造性运用和发展**。毛泽东将马克思主义的基本原理与中国革命的具体实际相结合，提出了一系列独创性的理论，如农村包围城市的革命道路、新民主主义革命理论、人民民主专政理论等。这些理论既坚持了马克思主义的基本立场，又根据中国的特殊情况做出了重大创新。

第二句话强调了毛泽东思想的**实践性**。毛泽东思想不是脱离实际的空洞理论，而是从中国革命和建设的实践中总结出来的，又被实践反复证明是正确的理论。正如毛泽东本人所强调的，理论来源于实践，又反过来指导实践。从井冈山的"星星之火，可以燎原"，到延安窑洞中的《论持久战》，再到新中国成立后的社会主义建设探索，毛泽东思想始终扎根于中国革命和建设的实践之中。

第三句话说明了毛泽东思想虽然以毛泽东的名字命名，但**不仅仅属于毛泽东个人**。它是中国共产党在长期革命斗争中集体智慧的结晶，是全党全国人民共同奋斗经验的总结。毛泽东是这一思想的主要创立者，但周恩来、刘少奇、朱德等老一辈革命家也对毛泽东思想的形成和发展做出了重要贡献。

:::insight{label=为什么说毛泽东思想是集体智慧的结晶？}
毛泽东思想的形成过程，本身就是中国共产党集体探索的过程。从建党初期对中国革命道路的探索，到井冈山根据地的建立，从抗日战争时期全民族的抗战斗争，到解放战争的伟大胜利，每一个历史阶段的理论成果都凝聚着全党全国人民的集体智慧。毛泽东同志是这些集体智慧的集大成者和主要理论概括者，但正如毛泽东自己所说，他的思想是在党和人民的集体奋斗中产生的。
:::

### （二）"两个不能否定"

在科学评价毛泽东思想时，必须坚持"两个不能否定"的基本原则：

:::definition{label="两个不能否定"}
- **不能用改革开放后的历史时期否定改革开放前的历史时期**；
- **也不能用改革开放前的历史时期否定改革开放后的历史时期**。
:::

这一原则要求我们把毛泽东同志的功绩和晚年所犯的错误区别开来，把毛泽东思想的科学体系和毛泽东同志晚年的错误理论区别开来。毛泽东思想是经过实践检验的科学理论，它所蕴含的马克思主义立场、观点和方法具有长远的指导意义。

## 二、毛泽东思想与中国特色社会主义理论体系的关系

毛泽东思想与中国特色社会主义理论体系是马克思主义中国化的两大理论成果，二者之间既有联系又有发展。

### （一）毛泽东思想为中国特色社会主义理论体系提供基本遵循

毛泽东思想所蕴含的马克思主义立场、观点和方法，为中国特色社会主义理论体系提供了根本的遵循。这种基本遵循作用主要体现在两个方面：

**第一，毛泽东思想活的灵魂在中国特色社会主义理论体系中得到继承和发展。** 实事求是、群众路线、独立自主这三个活的灵魂，被中国特色社会主义理论体系全面继承。例如，邓小平理论强调"实践是检验真理的唯一标准"，这正是实事求是精神的体现；"三个代表"重要思想强调代表最广大人民的根本利益，这正是群众路线的体现；科学发展观强调走中国特色自主创新道路，这正是独立自主精神的体现。

**第二，毛泽东关于社会主义建设的一系列重要思想，为开创中国特色社会主义做了重要的理论准备。** 毛泽东在社会主义建设的初步探索中，提出了许多具有前瞻性的思想，如正确处理人民内部矛盾、统筹兼顾、"双百"方针等，这些思想为后来中国特色社会主义的开创提供了重要的理论资源。

:::note{label=两者的内在联系}
| 关系维度 | 具体内容 |
|---------|---------|
| 活的灵魂的传承 | 实事求是、群众路线、独立自主被中国特色社会主义理论体系全面继承和发展 |
| 理论准备 | 毛泽东关于社会主义建设的重要思想为开创中国特色社会主义提供了理论基础 |
| 方法论启示 | 毛泽东思想中蕴含的马克思主义立场、观点和方法为后续理论发展提供了根本遵循 |
:::

### （二）中国特色社会主义理论体系丰富和发展了毛泽东思想

中国特色社会主义理论体系不是对毛泽东思想的简单重复，而是在新的历史条件下对毛泽东思想的进一步丰富和发展。

中国特色社会主义理论体系的各个组成部分——邓小平理论、"三个代表"重要思想、科学发展观、习近平新时代中国特色社会主义思想——都是在认真总结中国特色社会主义建设的历史经验和改革开放最新实践经验的基础上，创造性地提出的一系列新思想、新观点、新论断，从而进一步丰富和发展了毛泽东思想。

:::insight{label=继承与发展的辩证统一}
毛泽东思想与中国特色社会主义理论体系的关系，体现了马克思主义中国化的继承与发展的辩证统一。继承，是继承了马克思主义的基本立场、观点和方法；发展，是根据新的历史条件和实践要求，不断推进理论创新。这种继承与发展的统一，正是马克思主义中国化时代化不断向前推进的内在逻辑。
:::

### （三）党的十七大的理论贡献

2007年，党的十七大提出了**中国特色社会主义理论体系**的科学概念，将邓小平理论、"三个代表"重要思想和科学发展观统一纳入中国特色社会主义理论体系之中。这一科学概念的提出，明确了毛泽东思想与中国特色社会主义理论体系的关系，为科学认识马克思主义中国化的两大理论成果提供了重要的理论框架。

## 三、毛泽东思想作为指导思想的重大意义

### （一）毛泽东思想是党必须长期坚持的指导思想

毛泽东思想不仅是历史的产物，更是具有长远指导意义的科学理论。它所蕴含的马克思主义立场、观点和方法，它所确立的实事求是的思想路线，它所坚持的群众路线和独立自主的原则，在任何时代都不会过时。

:::definition{label=毛泽东思想的当代价值}
毛泽东思想的当代价值体现在：
1. **思想方法的指导**：实事求是的思想路线为认识和解决新问题提供了根本方法；
2. **工作方法的启示**：群众路线为密切联系群众、服务群众提供了根本遵循；
3. **精神品格的传承**：自力更生、艰苦奋斗的精神为应对各种挑战提供了强大精神力量；
4. **历史经验的镜鉴**：毛泽东思想中关于社会主义建设的经验教训为今天的改革和发展提供了宝贵借鉴。
:::

### （二）伟人的历史定位

理解毛泽东思想的历史地位，还需要从"伟人"的高度来认识毛泽东的历史贡献。在中华民族近现代历史上，有三位伟人各自为民族的生存和发展指明了方向：

| 伟人 | 历史贡献 | 核心意义 |
|------|---------|---------|
| **孙中山** | 使中国人"醒过来"——引进西方自由主义理念，结束了中国几千年的封建专制制度，为专制制度的结束奠定了思想基础 | 启蒙与觉醒 |
| **毛泽东** | 使中国人民"站起来"——领导新民主主义革命胜利，建立了新中国，实现了民族独立和人民解放 | 独立与解放 |
| **邓小平** | 使中国人民"富起来"——领导改革开放，开辟了中国特色社会主义道路 | 富裕与发展 |

:::insight{label=伟人与政治家的区别}
伟人与政治家的区别在于：**伟人能够为国家和民族在特定的历史时期指出正确的方向**。当一个国家和民族面对"往哪里走"的根本性选择时，能够指明正确方向的人才能称为伟人。毛泽东就是这样一位伟人——在中国革命面临各种道路选择的关键时刻，他始终能够以高瞻远瞩的政治远见，为中华民族指明前进方向。正如习近平总书记所指出的，毛泽东"思想博大深邃，胸怀坦荡宽广，文韬武略兼备，领导艺术高超，心系人民群众，终身艰苦奋斗"。
:::

### （三）习近平总书记对毛泽东的高度评价

习近平总书记在纪念毛泽东同志诞辰120周年大会上的讲话中，对毛泽东给予了全面而崇高的评价：

:::definition{label=习近平对毛泽东的评价}
毛泽东同志表现出一个伟大革命领袖**高瞻远瞩的政治远见、坚定不移的革命信念、勇于开拓的非凡魄力、炉火纯青的斗争艺术、杰出高超的领导才能**。他思想博大深邃，胸怀坦荡宽广，文韬武略兼备，领导艺术高超，心系人民群众，终身艰苦奋斗，**为中华民族和中国人民建立了不朽的功勋**。

不管是翻江卷巨澜，还是雄关漫道真如铁，毛泽东同志都**矢志不渝、执着追求**。
:::

这一评价准确概括了毛泽东作为革命领袖的卓越品质和历史贡献。毛泽东的一生，是为中华民族独立和中国人民解放不懈奋斗的一生。他所创立的毛泽东思想，是留给中国共产党和中国人民最宝贵的精神财富。

### （四）毛泽东思想的深远影响

毛泽东思想的影响不仅限于中国国内，它对世界被压迫民族的解放运动也产生了深远影响。毛泽东关于人民战争的理论、关于农村包围城市的革命道路、关于帝国主义和一切反动派都是纸老虎的论断，鼓舞了亚非拉许多国家和地区的民族解放斗争。

在国内，毛泽东思想的精神深深融入了中华民族的文化基因之中。"为人民服务"成为共产党人的根本宗旨，"自力更生、艰苦奋斗"成为民族精神的重要组成部分，"星星之火，可以燎原"成为激励人们战胜困难的精神力量。这些思想遗产超越了具体的历史时空，成为中华民族共同的精神财富。

:::note{label=学习毛泽东思想的现实意义}
学习毛泽东思想，不仅是为了了解历史，更是为了把握当下、面向未来。毛泽东思想中蕴含的实事求是的思想方法、群众路线的工作方法、独立自主的民族精神，对于我们在新时代坚持和发展中国特色社会主义，实现中华民族伟大复兴的中国梦，具有重要的现实指导意义。建议同学们通过阅读《毛泽东传》和《毛泽东选集》，深入了解毛泽东思想的丰富内涵和精神实质。
:::

## 总结

毛泽东思想的历史地位，可以从以下几个方面来概括：

1. **理论地位**：毛泽东思想是马克思主义中国化的第一次历史性飞跃，是马克思列宁主义在中国的创造性运用和发展；
2. **实践地位**：毛泽东思想是被实践证明了的关于中国革命和建设的正确的理论原则和经验总结；
3. **组织地位**：毛泽东思想是中国共产党集体智慧的结晶；
4. **指导地位**：毛泽东思想是党必须长期坚持的指导思想；
5. **传承地位**：毛泽东思想为中国特色社会主义理论体系提供了根本遵循和理论准备。

这五个方面相互关联、有机统一，共同构成了毛泽东思想历史地位的完整图景。在新的历史条件下，我们既要珍视毛泽东思想的宝贵遗产，又要在实践中不断丰富和发展它，使之在新时代焕发出新的生机与活力。
