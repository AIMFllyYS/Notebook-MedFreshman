# 第一二节 · 马克思主义中国化导论 · 课堂纪要

## 关键词
马克思主义中国化时代化、中国式现代化、四个正确认识、四个自信、中美关系、新冷战、人类解放、改革开放

## 核心议题
1. 课程定位与学习意义——为什么要学习"毛概"这门思政课
2. 中国式现代化的历史脉络与五大特征
3. 当代国际局势：世界进入新的动荡变革期与中美战略博弈
4. 马克思主义基本原理：人类解放的四个阶段与资本主义制度批判
5. 中国特色社会主义革命、建设、改革的历史进程

## 教学要点

### 一、课程定位与总体框架
- 本课程全称"毛泽东思想和中国特色社会主义理论体系概论"，是一门思想政治理论课
- 课程可概括为"一二三四"：**一条主线**（马克思主义中国化时代化）、**两大理论**（毛泽东思想与中国特色社会主义理论体系）、**三个阶段**（社会主义在中国的建立、建设、改革）、**四个自信**（道路自信、理论自信、制度自信、文化自信）
- 将毛泽东思想和中国特色社会主义理论体系连接在一起的纽带是"马克思主义中国化时代化"
- 课程讲授的是1848年诞生的马克思主义理论在20世纪来到中国，使中国变成社会主义国家的历史
- 马克思主义中国化时代化的内涵（教材第四页）：立足中国国情和时代特点，坚持把马克思主义基本原理同中国具体实际相结合、同中华优秀传统文化相结合，深入研究和解决中国革命、建设、改革不同时期的实际问题

### 二、为什么要学习这门课程
- **习近平总书记提出的"四个正确认识"**（2016年12月全国高校思想政治工作会议）：
  1. 正确认识世界和中国发展大势——认识中国特色社会主义道路
  2. 正确认识中国特色和国际比较——全面客观认识当代中国，看待外部世界
  3. 正确认识时代责任和历史使命——用中国梦激扬青春梦，将个人理想融入国家和民族事业
  4. 正确认识远大抱负和脚踏实地——将远大抱负落实到实际行动中
- 开设本课程的目的（教材第12-13页）：对中国共产党领导的革命、建设、改革的历史进程有更全面的**了解**，对马克思主义中国化时代化有更深刻的**理解**，对形成的理论成果有更准确的**把握**，对运用马克思主义立场、观点、方法解决问题的能力有更明显的**提升**（了解→理解→把握→能力提升，层层递进）

### 三、五门思政课的逻辑关系
- **思想道德与法治**（修身课）：树立正确三观和个人理想→关注个人理想的实现需要国家社会条件
- **毛泽东思想和中国特色社会主义理论体系概论**（治国课）：了解中国社会革命、建设、改革为个人理想提供的社会条件和机遇→"时代的一粒灰落到一个人身上就是一座山，时代的一缕阳光落到一个人身上就是阳光灿烂"
- **中国近现代史纲要**：通过讲述鸦片战争以来的历史，回答"中国人对马克思主义的选择是自觉自愿的选择"
- **马克思主义基本原理**：了解马克思主义是什么——建立公有和公平社会制度的学说
- **习近平新时代中国特色社会主义思想**：讲授党的十八大以来新时代中国特色社会主义建设（2012年至今）
- 本课程立足于今天的中国特色社会主义建设，回到历史中去了解社会主义革命、建设、改革的历史进程、历史变革和历史成就
- 注：明年起"毛概"与"史纲"将合二为一（史论结合），武汉大学等已试点

### 四、中国式现代化——本课程的主线
- **中国式现代化的提出**：2022年党的二十大正式提出，但早在1979年邓小平会见日本首相大平正芳时就已提出"中国式的四个现代化"
- **中国式现代化是中国共产党领导的社会主义现代化**，既有各国现代化的共同特征，更有基于国情的中国特色

#### 邓小平的现代化探索历程
- 1978年1-11月邓小平密集出访：缅甸、尼泊尔、朝鲜（现代化水平不如中国）→日本（参观全自动化汽车生产车间、乘坐新干线，感叹"我知道什么是现代化了""快"）→泰国、马来西亚、新加坡（学习新加坡式现代化道路）
- 出访回国后参加36天中央工作会议，做"解放思想、实事求是、团结一致向前看"发言→确定以经济建设为中心，对内改革、对外开放
- 1978年12月15日中美建交谈判达成协议，12月18-22日十一届三中全会召开（仅4天）
- 1979年1-2月邓小平访美9天，参观NASA、休斯公司、福特汽车公司（计算机控制生产线每小时装配50辆汽车）
- 1979年12月会见大平正芳：提出"中国式的现代化"，核心含义——短期内不可能达到西方水平，先解决"小康之家"；可借鉴西方先进技术与经验；必须走中国自己的现代化道路

#### 中国式现代化的五大特色
1. **人口规模最大的现代化**：2021年建党100周年全面建成小康社会，历史性解决绝对贫困——近1亿农村贫困人口全部脱贫，中等收入群体超4亿人，基本医疗保险覆盖13亿人，基本养老保险覆盖近10亿人
2. **全体人民共同富裕的现代化**：从"吃饱肚子"起步——1979年家庭联产承包责任制推行，1980年邓小平表态"先吃饱饭要紧"→全面建成小康社会
3. **物质文明和精神文明相协调的现代化**：以2026年春晚为例——人形机器人从"扭秧歌显眼包"到武术表演+小品+歌曲+微电影全方位演出，扩展现实、超高清等前沿技术打造科技视听盛宴，"十二花神"节目展现传统文化之美
4. **人与自然和谐共生的现代化**：武汉空气质量从2013年优良天数162天改善到2017年255天；华中科技大学喻家湖从臭水沟改造为"马尔代夫"校园景观；2001年世界银行公布全球污染最严重20个城市中国占16个→2012年后生态文明建设纳入"五位一体"→美丽中国建设→绿色一带一路
5. **走和平发展道路的现代化**：不走西方殖民掠夺扩张的老路，通过与世界各国全方位、多层次、宽领域的合作崛起→2010年成为世界第二大经济体→日渐走进世界舞台中央

### 五、当代国际局势与中美关系
- **世界进入新的动荡变革期**：习近平2021年提出"百年未有之大变局会同新冠肺炎疫情，世界进入新的动荡变革期"→俄乌战争、委内瑞拉总统被抓、伊朗问题等印证
- **技术革命的冲击**：从2003年伊拉克战争（信息化战争，一周）到当下人工智能战争（几秒到几分钟）→AI对个人职业生涯、医学行业乃至整个世界的革命性变化
- **中美关系的历史演变**：
  - 1949年：美国拒不承认新中国
  - 1950-1953年：朝鲜战争
  - 1954年日内瓦会议：美国虽同意中国参加但在邀请公报中注明"不含有外交承认之意"
  - 1970年代：中美关系改善
  - 1979年：中美建交，邓小平访美
  - 2015年：习近平访美演讲，会场鸦雀无声、长时期掌声（对比1979年邓小平访美时会场嘈杂）
  - 2017年：特朗普访华→2018年3月中美贸易冲突爆发→孟晚舟被扣→从贸易冲突升级为技术遏制
  - 2022年：拜登政府国家安全战略将中国定位为"公开的战略竞争对手"——五点核心判断：应对中国挑战是美国全球优先事务；中国是最严重的地缘政治挑战；全领域竞争覆盖全球；未来十年是决定性十年；美国对华战略目标是"竞而胜之"
  - "冷战陷阱"：中美关系出现新冷战对峙状态，双方在意识形态、发展模式、地缘政治、国际秩序方面利益冲突不可调和

### 六、马克思主义基本原理——人类解放的四个阶段
- **马克思主义的定义**：由马克思恩格斯创立、后继者发展的，以反对资本主义、建设社会主义和实现共产主义为目标的科学理论体系，核心是无产阶级和全人类的解放（即自由）
- **人类走向自由的四个阶段**：
  1. **从自然界束缚中解放**→取得相对自然的自由（人类由原始共产主义社会进入奴隶社会）
  2. **从人的束缚中解放**→资产阶级打着"所有人都是生而自由且平等"的旗号进行革命，建立资本主义社会
  3. **从资本的束缚中解放**→工人阶级（无产阶级）起来革命，建立社会主义社会（马克思主义主要解决的是这个阶段）
  4. **从自身束缚中解放**→"自由人的联合体"（马克思留给后人想象的概念）
- **资本主义制度分析**：
  - 核心：市场（赋予每个人平等追求财富的权利）+ 民主（防止权力介入经济领域）
  - 积极性：极大调动劳动积极性，带来社会财富巨大增长——恩格斯："资本主义在过去100年创造的财富比人类过去1000年还多"
  - 问题：社会贫富分化——自由主义认为是个人能力差异所致，马克思主义者认为是资本对劳动的剥削
  - 二战后西方改良：征税→二次分配→福利国家
  - 中国案例：1992年十四大建立社会主义市场经济体制→GDP一路飙升至2010年世界第二
- **社会主义制度**建立在对资本主义制度深刻揭露和批判的基础上

## 课堂讨论/互动
教师引导学生列举"新时代（2012年以来）感触最深的国内外大事"，五位同学发言：

1. **俄乌战争**（同学3）：俄罗斯对乌克兰发起特别军事行动，造成大量牺牲，至今未结束；教师补充：乌克兰将智障孩子送上战场、无人机绞杀的残酷现实
2. **美国对华遏制**（同学2）：提及美国的霸权行为；教师展开：不讲任何规则，自由女神火炬宣扬的自由民主规则全部抛弃
3. **人工智能技术发展**（同学4）：无法想象以前没有AI的大学生活，AI在困难时给予帮助；教师回应：自己备课时用AI核实所有资料的准确性，防止课堂上被学生"搜"出错误，技术冲击对人文社科同样巨大
4. **日本核废水排放**（同学5）：从奥运会提供福岛产品到核废水排海，对水资源、空气污染、核安全的忧虑；教师评价：这是"反人类的罪行"，环境破坏最终落实到人的生命健康上，与医学生的职业密切相关
5. 教师补充AI在医学领域的案例：肺部肿瘤检查中AI比医生多发现两个小肿瘤

教师总结：同学们的发言都可汇集到"中国式现代化"这一主题上。
