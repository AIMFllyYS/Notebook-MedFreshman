:::example{label=事件关系综合判断（掷骰子）}
掷一颗均匀骰子一次，样本空间为 $\Omega = \{1, 2, 3, 4, 5, 6\}$。

定义以下事件：

- $A = \{1, 2, 3\}$（出现不超过3点）
- $B = \{2, 4, 6\}$（出现偶数点）
- $C = \{1, 3, 5\}$（出现奇数点）
- $D = \{4, 5, 6\}$（出现大于3点）

回答：

1. 写出 $AB$、$A \cup B$、$A - B$、$B - A$
2. 判断 $A$ 与 $C$ 的关系；$B$ 与 $C$ 的关系；$A$ 与 $D$ 的关系
3. 判断哪些事件对互为对立事件
4. 求 $\overline{A \cup B}$ 和 $\bar{A}\bar{B}$，并验证德摩根律
:::

:::insight{label=分析}
本题需要综合运用积事件、和事件、差事件、互不相容、对立事件等概念，逐项计算和判断。
:::

:::derivation{label=解答}
### 第(1)题：计算各运算结果

**积事件 $AB$：**

$$AB = \{1,2,3\} \cap \{2,4,6\} = \{2\}$$

> $A$ 发生（不超过3点）且 $B$ 发生（偶数点）即"出现2点"。

**和事件 $A \cup B$：**

$$A \cup B = \{1,2,3\} \cup \{2,4,6\} = \{1, 2, 3, 4, 6\}$$

> 出现不超过3点或出现偶数点，共5个样本点。

**差事件 $A - B$：**

$$A - B = A\bar{B} = \{1,2,3\} - \{2,4,6\} = \{1, 3\}$$

> $A$ 发生而 $B$ 不发生 = "不超过3点且为奇数" = 出现1或3。

**差事件 $B - A$：**

$$B - A = \bar{A}B = \{2,4,6\} - \{1,2,3\} = \{4, 6\}$$

> $B$ 发生而 $A$ 不发生 = "偶数且大于3" = 出现4或6。

### 第(2)题：判断事件关系

**$A$ 与 $C$ 的关系：**

$$AC = \{1,2,3\} \cap \{1,3,5\} = \{1, 3\} \neq \emptyset$$

$$A \cup C = \{1,2,3\} \cup \{1,3,5\} = \{1,2,3,5\} \neq \Omega$$

结论：$A$ 与 $C$ **既不互斥，也不对立**，有交叉关系（$AC = \{1,3\}$）。

**$B$ 与 $C$ 的关系：**

$$BC = \{2,4,6\} \cap \{1,3,5\} = \emptyset$$

$$B \cup C = \{1,2,3,4,5,6\} = \Omega$$

结论：$B$ 与 $C$ 满足 $BC = \emptyset$ 且 $B \cup C = \Omega$，**$B$ 与 $C$ 互为对立事件**，即 $C = \bar{B}$。

（偶数与奇数在掷骰子中恰好互补）

**$A$ 与 $D$ 的关系：**

$$AD = \{1,2,3\} \cap \{4,5,6\} = \emptyset$$

$$A \cup D = \{1,2,3\} \cup \{4,5,6\} = \{1,2,3,4,5,6\} = \Omega$$

结论：$AD = \emptyset$ 且 $A \cup D = \Omega$，**$A$ 与 $D$ 互为对立事件**，即 $D = \bar{A}$。

（不超过3与大于3恰好互补）

### 第(3)题：哪些事件对互为对立

由上面分析：

| 事件对 | 是否对立 | 原因 |
|--------|----------|------|
| $A$ 与 $D$ | **是** | $A = \{1,2,3\}$，$D = \{4,5,6\}$，恰好互补 |
| $B$ 与 $C$ | **是** | $B = \{2,4,6\}$，$C = \{1,3,5\}$，恰好互补 |
| $A$ 与 $B$ | 否 | $AB = \{2\} \neq \emptyset$ |
| $A$ 与 $C$ | 否 | $A \cup C \neq \Omega$ |

### 第(4)题：验证德摩根律

计算 $\overline{A \cup B}$：

$$A \cup B = \{1,2,3,4,6\}$$

$$\overline{A \cup B} = \Omega - \{1,2,3,4,6\} = \{5\}$$

计算 $\bar{A}\bar{B}$（用德摩根律，理论上应等于 $\overline{A \cup B}$）：

$$\bar{A} = \{4,5,6\}, \quad \bar{B} = \{1,3,5\}$$

$$\bar{A}\bar{B} = \{4,5,6\} \cap \{1,3,5\} = \{5\}$$

**验证：** $\overline{A \cup B} = \{5\} = \bar{A}\bar{B}$ ✓

德摩根律：$\overline{A \cup B} = \bar{A} \cap \bar{B}$ 得到验证。
:::

:::note{label=结果汇总}
| 运算 | 结果 |
|------|------|
| $AB$ | $\{2\}$ |
| $A \cup B$ | $\{1,2,3,4,6\}$ |
| $A - B$ | $\{1,3\}$ |
| $B - A$ | $\{4,6\}$ |
| $\overline{A \cup B} = \bar{A}\bar{B}$ | $\{5\}$ |
:::
