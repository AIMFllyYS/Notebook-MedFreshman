:::example{label=题目描述}

某药厂宣称其生产的某药物有效成分含量均值为 $\mu_0 = 500$ mg。质检部门随机抽取 $n = 16$ 片，测得样本均值 $\bar{x} = 492$ mg，样本标准差 $s = 20$ mg。假设有效成分含量服从正态分布，在显著性水平 $\alpha = 0.05$ 下，检验该药物有效成分含量均值是否为 $500$ mg。

**已知条件：**
- 总体标准差：$\sigma$ **未知**（这是本题与 Z 检验的核心区别）
- 样本量：$n = 16$
- 样本均值：$\bar{x} = 492$ mg
- 样本标准差：$s = 20$ mg
- 显著性水平：$\alpha = 0.05$
- 总体服从正态分布

:::

:::insight{label=分析}

本题属于**单总体均值的假设检验**，但与 Z 检验不同，本题**总体标准差 $\sigma$ 未知**，只能用样本标准差 $s$ 代替，此时检验统计量不再服从标准正态分布，而是服从**自由度为 $n-1$ 的 t 分布**，故应使用 **t 检验**。

解题框架与 Z 检验类似，但有以下关键差异：

1. 统计量构造中用 $S$（样本标准差）替换 $\sigma$（总体标准差）。
2. 临界值从 t 分布表查取，需要指定自由度 $\nu = n - 1$。
3. t 分布的尾部比正态分布更厚（胖尾），导致相同 $\alpha$ 下 t 检验的临界值通常大于 Z 检验。
4. 样本量越大，t 分布越接近正态分布；$n \to \infty$ 时 t 检验等价于 Z 检验。

此外，本题还将延伸讨论**单侧检验**的情形，分析在不同 $\alpha$ 下结论的变化。

:::

:::derivation{label=解答}

**第一步：建立假设**

根据题意，质检部门怀疑药物有效成分含量可能偏离 $500$ mg，检验是否等于声称值，建立如下假设：

$$H_0: \mu = 500 \quad \text{（原假设，认为含量符合标准）}$$

$$H_1: \mu \neq 500 \quad \text{（备择假设，认为含量不符合标准）}$$

由于 $H_1$ 是 $\mu \neq 500$，两侧均可拒绝，为**双侧检验**。

**第二步：选择检验统计量**

由于总体标准差 $\sigma$ 未知，用样本标准差 $S$ 代替，选用 **t 检验统计量**：

$$T = \frac{\bar{X} - \mu_0}{S / \sqrt{n}} \sim t(n-1)$$

在 $H_0$ 成立且总体为正态分布的条件下，统计量 $T$ 服从自由度为 $n - 1 = 15$ 的 t 分布。

**第三步：确定拒绝域**

自由度 $\nu = n - 1 = 16 - 1 = 15$，显著性水平 $\alpha = 0.05$，双侧检验查 t 分布表：

$$t_{\alpha/2}(\nu) = t_{0.025}(15) = 2.131$$

拒绝域为：

$$W = \left\{ |T| \geq t_{0.025}(15) \right\} = \left\{ |T| \geq 2.131 \right\}$$

**第四步：计算检验统计量的观测值**

将 $\bar{x} = 492$，$\mu_0 = 500$，$s = 20$，$n = 16$ 代入：

$$T = \frac{\bar{x} - \mu_0}{s / \sqrt{n}} = \frac{492 - 500}{20 / \sqrt{16}} = \frac{-8}{20 / 4} = \frac{-8}{5} = -1.6$$

所以 $T$ 的观测值为 $t = -1.6$，即 $|t| = 1.6$。

**第五步：作出统计判断**

比较 $|t| = 1.6$ 与临界值 $t_{0.025}(15) = 2.131$：

$$|t| = 1.6 < 2.131 = t_{0.025}(15)$$

观测值落在接受域 $(-2.131,\ 2.131)$ 内，**不拒绝 $H_0$**。

**结论：** 在显著性水平 $\alpha = 0.05$ 下，没有充分证据拒绝"该药物有效成分含量均值为 $500$ mg"的假设，认为含量符合声称标准。

**延伸讨论：单侧检验（$H_1: \mu < 500$）**

若质检部门认为该药物含量只可能偏低（疗效不足），则建立单侧检验：

$$H_0: \mu = 500, \quad H_1: \mu < 500 \quad \text{（左侧检验）}$$

此时拒绝域改为左侧：$W = \{T \leq -t_{\alpha}(15)\}$

当 $\alpha = 0.05$ 时，$t_{0.05}(15) = 1.753$，拒绝域为 $T \leq -1.753$。

$$t = -1.6, \quad -1.6 > -1.753$$

仍然**不拒绝 $H_0$**。

但若取 $\alpha = 0.10$（放宽显著性水平），则 $t_{0.10}(15) = 1.341$，拒绝域为 $T \leq -1.341$。

$$t = -1.6 < -1.341$$

此时**拒绝 $H_0$**，认为在 $\alpha = 0.10$ 显著性水平下有证据表明含量低于 $500$ mg。

这说明检验结论依赖于显著性水平 $\alpha$ 的选取，$\alpha$ 越大，拒绝 $H_0$ 越容易，但犯第一类错误（弃真）的风险也越大。

:::

:::theorem{label=关键公式}

**t 检验统计量（总体方差未知）：**

$$T = \frac{\bar{X} - \mu_0}{S / \sqrt{n}} \sim t(n-1)$$

其中 $S$ 为样本标准差：

$$S = \sqrt{\frac{1}{n-1} \sum_{i=1}^{n}(X_i - \bar{X})^2}$$

**双侧检验拒绝域：**

$$W = \left\{ |T| \geq t_{\alpha/2}(n-1) \right\}$$

**左侧单侧检验拒绝域：**

$$W = \left\{ T \leq -t_{\alpha}(n-1) \right\}$$

**右侧单侧检验拒绝域：**

$$W = \left\{ T \geq t_{\alpha}(n-1) \right\}$$

**t 分布与正态分布的关系（大样本近似）：**

$$\lim_{n \to \infty} t(n-1) \approx N(0,1)$$

**常用 t 分布临界值（自由度 $\nu = 15$）：**

| 显著性水平 $\alpha$ | $t_{\alpha/2}(15)$（双侧） | $t_{\alpha}(15)$（单侧） |
|:-----------------:|:------------------------:|:----------------------:|
| $0.10$ | $1.753$ | $1.341$ |
| $0.05$ | $2.131$ | $1.753$ |
| $0.02$ | $2.602$ | $2.249$ |
| $0.01$ | $2.947$ | $2.602$ |

:::

:::pitfall{label=易错点1}

**总体方差未知时仍使用 Z 检验**：当总体方差 $\sigma^2$ 未知时，**不能**使用 Z 检验，必须改用 t 检验。本题条件是"样本标准差 $s = 20$"而非"总体标准差 $\sigma = 20$"，两者含义不同。若误用 Z 检验，临界值为 $z_{0.025} = 1.96 < 2.131$，虽然本题结论恰好相同，但这是数学上的错误。

:::

:::pitfall{label=易错点2}

**自由度计算错误**：t 分布的自由度为 $\nu = n - 1$，而非 $\nu = n$。本题 $n = 16$，故自由度 $\nu = 15$，查表应用 $t_{0.025}(15) = 2.131$，而非 $t_{0.025}(16)$。自由度来源于：计算样本标准差 $S$ 时，分母用 $n-1$ 而非 $n$，因此有 $n-1$ 个自由变动量。

:::

:::pitfall{label=易错点3}

**单侧与双侧检验的混淆**：检验是双侧还是单侧，完全取决于**备择假设 $H_1$ 的形式**，而非问题的措辞。$H_1: \mu \neq \mu_0$ 对应双侧检验，临界值用 $t_{\alpha/2}(n-1)$。$H_1: \mu < \mu_0$ 对应左侧单侧检验，临界值用 $t_{\alpha}(n-1)$（左尾）。$H_1: \mu > \mu_0$ 对应右侧单侧检验，临界值用 $t_{\alpha}(n-1)$（右尾）。错误地将双侧 $\alpha$ 用在单侧检验（或反之），会导致临界值选取错误，判断结论可能翻转。

:::

:::pitfall{label=易错点4}

**忽视正态性假设**：t 检验的理论基础是总体服从正态分布（或样本量足够大以满足中心极限定理）。本题已明确指出"假设有效成分含量服从正态分布"，这是使用 t 检验的前提条件，不可忽略。若总体不服从正态分布且样本量较小（如 $n = 16$），t 检验的结论可靠性下降，需要非参数检验方法。

:::

:::pitfall{label=易错点5}

**样本标准差 $S$ 与总体标准差 $\sigma$ 的计算公式差异**：总体标准差：$\sigma = \sqrt{\frac{1}{N}\sum(X_i - \mu)^2}$，分母是 $N$（总体大小）。样本标准差：$S = \sqrt{\frac{1}{n-1}\sum(X_i - \bar{X})^2}$，分母是 $n-1$（贝塞尔校正）。t 检验统计量中用的是样本标准差 $S$（分母 $n-1$），使用分母为 $n$ 的版本会导致 $S$ 偏小，t 值偏大，可能错误地拒绝 $H_0$。

:::

:::pitfall{label=易错点6}

**大样本时 t 检验与 Z 检验的关系**：当样本量 $n$ 很大（通常 $n > 30$ 或 $n > 100$）时，$t(n-1)$ 分布趋近于 $N(0,1)$，此时 t 检验与 Z 检验的结果几乎相同。但小样本（如本题 $n = 16$）时，t 分布的胖尾效应显著，临界值明显大于正态分布，必须严格使用 t 检验，不能用正态分布近似替代。

:::
