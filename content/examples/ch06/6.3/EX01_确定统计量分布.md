:::example{label=题目描述}

设总体 $X\sim N(0,1)$（标准正态分布），$(X_1, X_2, \ldots, X_{10})$ 是来自总体的简单随机样本（各 $X_i$ 独立同分布于 $N(0,1)$）。

判断以下统计量服从何种分布，并说明理由：

(a) $Y_1 = \sum_{i=1}^{10} X_i^2$

(b) $Y_2 = \frac{X_1}{\sqrt{\dfrac{1}{9}\sum_{i=2}^{10}X_i^2}}$

(c) $Y_3 = \frac{\dfrac{1}{5}\sum_{i=1}^5 X_i^2}{\dfrac{1}{5}\sum_{i=6}^{10}X_i^2}$

(d) $Y_4 = \sqrt{9}\cdot\frac{X_1}{\sqrt{\sum_{i=2}^{10}X_i^2}}$（与 (b) 比较）

(e) $Y_5 = \frac{(X_1+X_2)/\sqrt{2}}{\sqrt{\dfrac{1}{8}\left(\sum_{i=3}^{10}X_i^2\right)}}$

:::

:::insight{label=分析}

本题考查如何将给定的统计量分解，识别出 $N(0,1)$、$\chi^2$、$t$、$F$ 分布的标准构造形式，从而确定其分布。

核心方法：
- **$\chi^2(n)$ 的构造**：$n$ 个独立 $N(0,1)$ 变量的平方和；
- **$t(n)$ 的构造**：$T=X/\sqrt{Y/n}$，$X\sim N(0,1)$，$Y\sim\chi^2(n)$，$X$ 与 $Y$ 独立；
- **$F(m,n)$ 的构造**：$F=(U/m)/(V/n)$，$U\sim\chi^2(m)$，$V\sim\chi^2(n)$，$U$ 与 $V$ 独立。

本题中总体为 $N(0,1)$，各 $X_i$ 本身就是标准正态变量，因此可以直接利用 $\chi^2$ 分布的定义。

:::

:::derivation{label=解答}

**（a）$Y_1 = \sum_{i=1}^{10}X_i^2$**

$X_1,\ldots,X_{10}$ 相互独立，均服从 $N(0,1)$，由 **$\chi^2$ 分布的定义**：

$$Y_1 = \sum_{i=1}^{10}X_i^2 = X_1^2+X_2^2+\cdots+X_{10}^2 \sim \chi^2(10)$$

$$\boxed{Y_1 \sim \chi^2(10)}$$

期望：$E[Y_1]=10$；方差：$D[Y_1]=20$。

**（b）$Y_2 = \dfrac{X_1}{\sqrt{\dfrac{1}{9}\sum_{i=2}^{10}X_i^2}}$**

**分解分子和分母**：

- **分子**：$X_1\sim N(0,1)$；
- **分母内的平方和**：$\sum_{i=2}^{10}X_i^2$，共 9 项（$X_2,\ldots,X_{10}$）独立同 $N(0,1)$，故 $\sum_{i=2}^{10}X_i^2\sim\chi^2(9)$；
- **独立性**：$X_1$ 与 $\sum_{i=2}^{10}X_i^2$ 相互独立（下标不重叠）。

将 $Y_2$ 改写为 $t$ 分布的标准形式：

$$Y_2 = \frac{X_1}{\sqrt{\dfrac{\sum_{i=2}^{10}X_i^2}{9}}} = \frac{X_1}{\sqrt{Y/9}}$$

其中 $Y=\sum_{i=2}^{10}X_i^2\sim\chi^2(9)$，$X_1\sim N(0,1)$，且独立。由 $t$ 分布定义：

$$Y_2 = \frac{N(0,1)}{\sqrt{\chi^2(9)/9}} \sim t(9)$$

$$\boxed{Y_2 \sim t(9)}$$

**（c）$Y_3 = \dfrac{\dfrac{1}{5}\sum_{i=1}^5X_i^2}{\dfrac{1}{5}\sum_{i=6}^{10}X_i^2}$**

**分解分子和分母**：

- **分子**：$\sum_{i=1}^5 X_i^2\sim\chi^2(5)$（5项独立 $N(0,1)$ 的平方和）；
- **分母**：$\sum_{i=6}^{10}X_i^2\sim\chi^2(5)$（另5项独立 $N(0,1)$ 的平方和）；
- **独立性**：两组下标不重叠，故相互独立。

将 $Y_3$ 改写为 $F$ 分布的标准形式：

$$Y_3 = \frac{\dfrac{1}{5}\sum_{i=1}^5 X_i^2}{\dfrac{1}{5}\sum_{i=6}^{10}X_i^2} = \frac{\left(\sum_{i=1}^5 X_i^2\right)/5}{\left(\sum_{i=6}^{10}X_i^2\right)/5} = \frac{\chi^2(5)/5}{\chi^2(5)/5} \sim F(5,5)$$

$$\boxed{Y_3 \sim F(5,5)}$$

期望：$E[Y_3]=\dfrac{5}{5-2}=\dfrac{5}{3}$（$n>2$）。

**（d）$Y_4 = \sqrt{9}\cdot\dfrac{X_1}{\sqrt{\sum_{i=2}^{10}X_i^2}}$**

改写为标准形式：

$$Y_4 = \frac{X_1}{\sqrt{\dfrac{\sum_{i=2}^{10}X_i^2}{9}}} = \frac{X_1}{\sqrt{\chi^2(9)/9}}$$

这与 (b) 中 $Y_2$ 完全相同，故：

$$\boxed{Y_4 \sim t(9)}$$

说明：$\sqrt{9}\cdot\frac{X_1}{\sqrt{\sum X_i^2}} = \frac{X_1}{\frac{1}{\sqrt{9}}\sqrt{\sum X_i^2}} = \frac{X_1}{\sqrt{\sum X_i^2/9}}$，即 $Y_4=Y_2$，两种写法等价。

**（e）$Y_5 = \dfrac{(X_1+X_2)/\sqrt{2}}{\sqrt{\dfrac{1}{8}\sum_{i=3}^{10}X_i^2}}$**

**关键步骤——分子的标准化**：

$X_1+X_2\sim N(0,2)$（两个独立 $N(0,1)$ 之和），故 $(X_1+X_2)/\sqrt{2}\sim N(0,1)$。

- **分子**：$(X_1+X_2)/\sqrt{2}\sim N(0,1)$；
- **分母**：$\sum_{i=3}^{10}X_i^2\sim\chi^2(8)$（8项），$\sqrt{\chi^2(8)/8}$ 是 $t$ 分布分母的标准形式；
- **独立性**：$(X_1,X_2)$ 与 $(X_3,\ldots,X_{10})$ 独立，故分子与分母独立。

$$Y_5 = \frac{N(0,1)}{\sqrt{\chi^2(8)/8}} \sim t(8)$$

$$\boxed{Y_5 \sim t(8)}$$

:::

:::theorem{label=关键公式}

$$X_i \overset{iid}{\sim} N(0,1) \implies \sum_{i=1}^n X_i^2 \sim \chi^2(n)$$

$$X\sim N(0,1),\; Y\sim\chi^2(n),\; X\perp Y \implies \frac{X}{\sqrt{Y/n}}\sim t(n)$$

$$U\sim\chi^2(m),\; V\sim\chi^2(n),\; U\perp V \implies \frac{U/m}{V/n}\sim F(m,n)$$

$$X_1,\ldots,X_n\overset{iid}{\sim}N(0,1) \implies \frac{X_1+\cdots+X_k}{\sqrt{k}}\sim N(0,1) \quad (1\leq k\leq n)$$

:::

:::pitfall{label=易错点1}

**$\chi^2$ 分布的平方和要求标准正态（均值为0）**：$\chi^2$ 分布定义为 $N(0,1)$ 的平方和，若 $X_i\sim N(\mu,\sigma^2)$（$\mu\neq 0$ 或 $\sigma\neq 1$），则 $\sum X_i^2$ 服从非中心 $\chi^2$ 分布，不是标准 $\chi^2$ 分布。本题总体为 $N(0,1)$，故可直接应用。

:::

:::pitfall{label=易错点2}

**独立性检查**：$t$ 和 $F$ 分布的定义要求分子和分母独立。若分子和分母的 $X_i$ 下标有重叠（如分子含 $X_1$，分母也含 $X_1$），则不独立，不能直接套用公式。

:::

:::pitfall{label=易错点3}

**$F$ 分布自由度顺序**：$Y_3\sim F(5,5)$，分子对应的 $\chi^2$ 自由度在前，分母在后。若分子有 5 项、分母有 3 项，则应为 $F(5,3)$，不能颠倒。

:::

:::pitfall{label=易错点4}

**忘记线性组合的标准化**：(e) 中，$X_1+X_2\sim N(0,2)$ 而非 $N(0,1)$，必须除以 $\sqrt{2}$ 才得到标准正态，然后再套 $t$ 分布定义。直接用 $\frac{X_1+X_2}{\sqrt{Y/n}}$ 不满足 $t$ 分布的构造要求，结果不正确。

:::
