:::example{label=题目描述}

查分位点表，求下列各分位点的值，或利用分位点的性质化简计算：

(a) 标准正态分布上 0.05 分位点：$z_{0.05}$

(b) $t$ 分布上 0.025 分位点：$t_{0.025}(10)$

(c) $\chi^2$ 分布上 0.05 分位点：$\chi^2_{0.05}(8)$

(d) $F$ 分布上 0.05 分位点：$F_{0.05}(5,10)$

(e) 利用对称性，求 $t_{0.975}(10)$ 和 $z_{0.95}$

(f) 利用倒数关系，求 $F_{0.95}(5,10)$（不得直接查表）

(g) 求满足 $P(|Z| > c) = 0.05$ 的 $c$ 值（$Z\sim N(0,1)$）

(h) 求满足 $P(\chi^2(10) < d) = 0.05$ 的 $d$ 值

:::

:::insight{label=分析}

分位点查表的关键是：
1. 明确"上 $\alpha$ 分位点"的定义：$P(X>x_\alpha)=\alpha$（右尾面积为 $\alpha$）；
2. 掌握**正态/t 分布的对称性**（减少查表次数）；
3. 掌握 **$F$ 分布的倒数关系**（将下分位点转化为互换自由度的上分位点）；
4. 注意 $\chi^2$ 分布无对称性，上下分位点须分别查表。

:::

:::derivation{label=解答}

**（a）标准正态上 0.05 分位点 $z_{0.05}$**

查标准正态分布表，满足 $P(Z>z_{0.05})=0.05$，等价于 $\Phi(z_{0.05})=0.95$：

$$\boxed{z_{0.05} = 1.645}$$

验证：$\Phi(1.645)\approx 0.95$，右侧面积 $1-0.95=0.05$。

**（b）$t$ 分布上 0.025 分位点 $t_{0.025}(10)$**

查 $t$ 分布表，自由度 $n=10$，$\alpha=0.025$，满足 $P(T>t_{0.025}(10))=0.025$：

$$\boxed{t_{0.025}(10) = 2.228}$$

注意：自由度为 10，查 $t(10)$ 行，$\alpha=0.025$ 列。

对比：$z_{0.025}=1.960 < t_{0.025}(10)=2.228$，$t$ 分布尾部更厚，相同尾部概率对应更大的临界值。

**（c）$\chi^2$ 分布上 0.05 分位点 $\chi^2_{0.05}(8)$**

查 $\chi^2$ 分布表，自由度 $n=8$，$\alpha=0.05$，满足 $P(\chi^2>\chi^2_{0.05}(8))=0.05$：

$$\boxed{\chi^2_{0.05}(8) = 15.507}$$

同时给出对称的下分位点：$\chi^2_{0.95}(8)=2.733$（满足 $P(\chi^2>2.733)=0.95$，即左尾面积 0.05）。

**（d）$F$ 分布上 0.05 分位点 $F_{0.05}(5,10)$**

查 $F$ 分布表，第一自由度 $m=5$，第二自由度 $n=10$，$\alpha=0.05$：

$$\boxed{F_{0.05}(5,10) = 3.33}$$

**（e）利用对称性求 $t_{0.975}(10)$ 和 $z_{0.95}$**

**$t$ 分布的对称性**：$t(n)$ 分布关于 0 对称，故 $t_{1-\alpha}(n)=-t_\alpha(n)$。

$$t_{0.975}(10) = -t_{0.025}(10) = -2.228$$

$$\boxed{t_{0.975}(10) = -2.228}$$

验证：$P(T > -2.228) = P(T < 2.228) = 1-0.025 = 0.975$，即右尾面积为 $0.975$，符合上 $0.975$ 分位点的定义。

**标准正态的对称性**：$z_{1-\alpha}=-z_\alpha$。

$$z_{0.95} = -z_{0.05} = -1.645$$

$$\boxed{z_{0.95} = -1.645}$$

**（f）利用倒数关系求 $F_{0.95}(5,10)$**

$F$ 分布不对称，利用倒数关系：$F_{1-\alpha}(m,n)=\dfrac{1}{F_\alpha(n,m)}$（注意自由度互换）：

$$F_{0.95}(5,10) = \frac{1}{F_{0.05}(10,5)}$$

查表得 $F_{0.05}(10,5)=4.74$（第一自由度 10，第二自由度 5），故：

$$F_{0.95}(5,10) = \frac{1}{4.74} \approx 0.211$$

$$\boxed{F_{0.95}(5,10) \approx 0.211}$$

验证：$F_{0.95}(5,10)<1$（符合直觉，因为 0.95 分位点是分布的左侧部分，$F$ 分布均值约为 $n/(n-2)=10/8=1.25>1$，故下分位点小于 1）。

**（g）求满足 $P(|Z|>c)=0.05$ 的 $c$ 值**

$P(|Z|>c)=0.05$ 等价于 $P(Z>c)+P(Z<-c)=0.05$。

由对称性 $P(Z>c)=P(Z<-c)=0.025$，故：

$$P(Z>c) = 0.025 \implies c = z_{0.025} = 1.960$$

$$\boxed{c = z_{0.025} = 1.960}$$

这正是双侧检验中 $\alpha=0.05$ 的临界值，对应置信区间 $[\bar{x}-1.960\sigma/\sqrt{n},\; \bar{x}+1.960\sigma/\sqrt{n}]$。

**（h）求满足 $P(\chi^2(10)<d)=0.05$ 的 $d$ 值**

$P(\chi^2(10)<d)=0.05$ 等价于 $P(\chi^2(10)>d)=0.95$，即 $d$ 是 $\chi^2(10)$ 的**上 0.95 分位点**：

$$d = \chi^2_{0.95}(10)$$

查 $\chi^2$ 分布表，自由度 10，$\alpha=0.95$（右尾面积 0.95，即左尾面积 0.05）：

$$\boxed{d = \chi^2_{0.95}(10) = 3.940}$$

注意：$\chi^2_{0.95}(10)=3.940$，而 $\chi^2_{0.05}(10)=18.307$，两者相差悬殊，体现了 $\chi^2$ 分布的非对称性（右偏）。

:::

:::theorem{label=关键公式}

**对称性**（$N(0,1)$ 和 $t(n)$ 适用）：

$$z_{1-\alpha} = -z_\alpha, \quad t_{1-\alpha}(n) = -t_\alpha(n)$$

**$F$ 分布倒数关系**：

$$F_{1-\alpha}(m,n) = \frac{1}{F_\alpha(n,m)} \quad \text{（自由度互换！）}$$

**双侧对称区间**（$Z\sim N(0,1)$）：

$$P(|Z| \leq z_{\alpha/2}) = 1-\alpha$$

即 $P(-z_{\alpha/2}\leq Z\leq z_{\alpha/2})=1-\alpha$。

:::

:::pitfall{label=易错点1}

**$F$ 分布倒数关系忘记互换自由度**：$F_{0.95}(5,10)=1/F_{0.05}(5,10)$ 是错误的（自由度没有互换），正确应为 $1/F_{0.05}(10,5)$。这是最常见的错误，务必记住"倒数同时自由度要互换"。

:::

:::pitfall{label=易错点2}

**$\chi^2$ 分布下分位点读表错误**：查 $\chi^2_{0.95}(10)$ 时，表中该行对应"右尾面积 0.95"，由于 $\chi^2$ 分布右偏，此时分位点值较小（3.940），不要与右尾面积 0.05 对应的大值（18.307）混淆。

:::

:::pitfall{label=易错点3}

**混淆 $t_{0.975}$ 的含义**：$t_{0.975}(10)=-2.228$ 是负数，代表"右尾面积为 0.975"（几乎整个分布在其右侧），这是分布的左侧极端值，不是常用的临界值。双侧检验通常用 $\pm t_{0.025}(n)$，即 $\pm2.228$。

:::

:::pitfall{label=易错点4}

**题（h）转化出错**：$P(\chi^2<d)=0.05$ 要求 $d$ 很小（左尾5%），须转化为 $P(\chi^2>d)=0.95$，查上 0.95 分位点（值很小），不要错误查上 0.05 分位点（值很大）。

:::
