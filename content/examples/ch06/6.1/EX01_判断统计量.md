:::example{label=题目描述}

设总体 $X\sim N(\mu, \sigma^2)$，其中 $\mu$ 和 $\sigma^2$ 均为未知参数。$(X_1, X_2, \ldots, X_n)$ 是来自总体 $X$ 的简单随机样本，$\bar{X}=\frac{1}{n}\sum_{i=1}^n X_i$ 为样本均值，$S^2=\frac{1}{n-1}\sum_{i=1}^n(X_i-\bar{X})^2$ 为样本方差。

判断以下各量是否为统计量，并说明理由：

(a) $X_1 - \mu$

(b) $\bar{X} = \frac{1}{n}\sum_{i=1}^n X_i$

(c) $\frac{(n-1)S^2}{\sigma^2}$

(d) $\max(X_1, X_2, \ldots, X_n)$

(e) $\frac{\bar{X} - \mu_0}{S/\sqrt{n}}$，其中 $\mu_0$ 是某个已知的常数

(f) $\frac{1}{n}\sum_{i=1}^n (X_i - \mu)^2$

:::

:::insight{label=分析}

判断一个量是否为统计量，只需检验两个条件：
1. 它是否是样本 $(X_1,\ldots,X_n)$ 的函数（即只含 $X_i$，不含其他随机量）；
2. 它是否**不含任何未知参数**（即不含 $\mu$、$\sigma^2$ 等值未知的参数）。

两个条件同时满足，才是统计量。本题中 $\mu$ 和 $\sigma^2$ 均为**未知参数**，含有它们的量不是统计量；$\mu_0$ 是**已知常数**，含有 $\mu_0$ 的量可以是统计量。

:::

:::derivation{label=解答}

**(a) $X_1 - \mu$**

$X_1 - \mu$ 虽然含有样本量 $X_1$，但同时含有**未知参数 $\mu$**。当我们拿到样本数据 $x_1=3$ 时，无法计算 $x_1-\mu$ 的值，因为 $\mu$ 未知。

**结论：不是统计量。**

**(b) $\bar{X} = \frac{1}{n}\sum_{i=1}^n X_i$**

$\bar{X}$ 是 $X_1,\ldots,X_n$ 的线性函数，不含任何参数（$n$ 是已知的样本量，不是未知参数）。拿到样本数据后，可以立即计算出 $\bar{x}$ 的值。

**结论：是统计量。**（样本均值，估计总体均值 $\mu$）

**(c) $\frac{(n-1)S^2}{\sigma^2}$**

虽然 $S^2=\frac{1}{n-1}\sum(X_i-\bar{X})^2$ 是统计量，但 $\frac{(n-1)S^2}{\sigma^2}$ 中含有**未知参数 $\sigma^2$**。该量虽然有重要的理论意义（正态总体下它服从 $\chi^2(n-1)$），但在实际计算中由于 $\sigma^2$ 未知无法得到其数值。

**结论：不是统计量。**

注意：$S^2$ 本身是统计量，但 $S^2/\sigma^2$ 不是（因为 $\sigma^2$ 未知）。这里考查学生能否识别统计量与含参量的区别。

**(d) $\max(X_1, X_2, \ldots, X_n)$**

$\max(X_1,\ldots,X_n)$ 是样本 $X_{(n)}$（最大顺序统计量），是 $n$ 个样本值的函数，不含任何未知参数。

**结论：是统计量。**（顺序统计量，在非参数统计中有重要应用）

**(e) $\frac{\bar{X}-\mu_0}{S/\sqrt{n}}$，$\mu_0$ 为已知常数**

$\mu_0$ 是**已知常数**（例如检验假设 $H_0:\mu=\mu_0$ 时指定的值），不是未知参数。$\bar{X}$ 和 $S$ 均是统计量，$n$ 和 $\mu_0$ 均已知，故整个表达式不含任何未知参数。

**结论：是统计量。**（这是检验假设 $H_0:\mu=\mu_0$ 时的 $t$ 检验统计量，在 $H_0$ 成立时服从 $t(n-1)$）

**(f) $\frac{1}{n}\sum_{i=1}^n(X_i-\mu)^2$**

该量含有**未知参数 $\mu$**，当 $\mu$ 未知时无法计算其值。

**结论：不是统计量。**（若 $\mu$ 已知，则它是 $\sigma^2$ 的无偏估计量 $\tilde{S}^2$，此时是统计量）

:::

:::theorem{label=关键公式}

统计量的定义：$T=T(X_1,\ldots,X_n)$ 是统计量当且仅当 $T$ 是样本的函数且不含任何未知参数。

本题中判断结果总结：

| 量 | 是否统计量 | 原因 |
|----|-----------|------|
| $X_1-\mu$ | 否 | 含未知参数 $\mu$ |
| $\bar{X}$ | 是 | 不含未知参数 |
| $(n-1)S^2/\sigma^2$ | 否 | 含未知参数 $\sigma^2$ |
| $\max(X_i)$ | 是 | 不含未知参数 |
| $(\bar{X}-\mu_0)/(S/\sqrt{n})$ | 是 | $\mu_0$ 为已知常数 |
| $\frac{1}{n}\sum(X_i-\mu)^2$ | 否 | 含未知参数 $\mu$ |

:::

:::pitfall{label=易错点1}

**混淆已知参数值和未知参数**：题目（e）中的 $\mu_0$ 是指定的已知数值（如"检验 $H_0:\mu=5$"中的 5），不是总体未知参数 $\mu$，因此 $(\bar{X}-\mu_0)/(S/\sqrt{n})$ 是统计量。许多学生看到 $\mu$ 字样就认为不是统计量，忽略了下标 $_0$ 表示已知值的含义。

:::

:::pitfall{label=易错点2}

**认为含参量就有理论分布则是统计量**：$(n-1)S^2/\sigma^2\sim\chi^2(n-1)$ 是有重要理论意义的量，但这个推断是在 $\sigma^2$ 已知时才能数值化。"有抽样分布"和"是统计量"不等价——量必须不含未知参数才是统计量。

:::

:::pitfall{label=易错点3}

**忽略顺序统计量是统计量**：$\max(X_i)$，$\min(X_i)$，中位数等顺序统计量都是统计量，因为它们只是对样本值的排序和选取，不引入任何参数。

:::
