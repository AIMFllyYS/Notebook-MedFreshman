**题目**：

1. 用乙烯为原料制备正丁醇。

2. 用 1-碘丙烷制备下列化合物：异丙醇、1,2-二溴丙烷、α-溴丙烯、二丙醚。

3. 用苯或甲苯制备下列化合物：1-苯基-1,2-二氯乙烷、1,2-二苯乙烷。

**解**：

**1. 乙烯制备正丁醇**：

合成路线：
- CH2=CH2 + HBr → CH3CH2Br（加成）
- CH3CH2Br + Mg/无水乙醚 → CH3CH2MgBr（格氏试剂）
- CH3CH2MgBr + CH2O（甲醛）→ CH3CH2CH2OMgBr → H2O → CH3CH2CH2OH（增碳）
- CH3CH2CH2OH + HBr → CH3CH2CH2Br
- CH3CH2CH2Br + Mg → CH3CH2CH2MgBr
- CH3CH2CH2MgBr + CH2O → CH3CH2CH2CH2OH（正丁醇）

或用乙醛 + 格氏试剂：
- CH2=CH2 + H2O/H+ → CH3CH2OH
- CH3CH2OH + [O] → CH3CHO
- CH3CHO + CH3CH2MgBr → CH3CH(OH)CH2CH3（2-丁醇，非正丁醇）

**2. 1-碘丙烷制备目标化合物**：

- **异丙醇**：CH3CH2CH2I + NaOH/醇 → CH2=CHCH3（消除）→ H2O/H+ → (CH3)2CHOH（马氏加成）
- **1,2-二溴丙烷**：CH3CH2CH2I → CH2=CHCH3（消除）→ Br2 → CH2Br-CHBr-CH3
- **α-溴丙烯**：CH3CH2CH2I → CH2=CHCH3（消除）→ NBS → CH2=CH-CH2Br（烯丙位溴代）
- **二丙醚**：2 CH3CH2CH2I + Na2O → (CH3CH2CH2)2O（威廉森醚合成）

**3. 苯/甲苯制备目标化合物**：

- **1-苯基-1,2-二氯乙烷**：苯 + CH2=CHCl/AlCl3（傅克烷基化）→ Ph-CH2-CH2Cl →（Cl2/光照）→ Ph-CHCl-CH2Cl
- **1,2-二苯乙烷**：苯 + CH2=CH2/AlCl3 → Ph-CH2-CH3（乙苯）→（Cl2/光照）→ Ph-CHCl-CH3 →（苯/AlCl3）→ Ph-CH2-CH2-Ph

或：2 Ph-CH2Cl + Na → Ph-CH2-CH2-Ph（武兹反应）

**要点**：
1. 格氏试剂 + 甲醛 = 增碳制备伯醇；
2. 消除反应可制备烯烃，再进行各种加成；
3. 威廉森醚合成是制备醚的重要方法；
4. 傅克反应是芳环上引入烷基/酰基的重要方法。
