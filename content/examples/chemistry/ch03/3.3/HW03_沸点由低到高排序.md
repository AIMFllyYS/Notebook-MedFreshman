**题目**：将下列化合物按沸点由低到高排序。

（1）（A）正己烷　（B）异己烷　（C）丁酮　（D）丁酸

（2）（A）丁醇　（B）丁酸　（C）丁二酸　（D）乙酸乙酯

（3）（A）丙酮　（B）丙醇　（C）丁烷　（D）乙醚　（E）乙酸

**解**：

（1）沸点由低到高：异己烷 < 正己烷 < 丁酮 < 丁酸

解析：烷烃沸点随支链增多而降低（异己烷 < 正己烷）；丁酮有偶极-偶极作用，沸点高于烷烃；丁酸能形成分子间氢键（二聚体），沸点最高。

（2）沸点由低到高：乙酸乙酯 < 丁醇 < 丁酸 < 丁二酸

解析：乙酸乙酯不能形成氢键，沸点最低；丁醇能形成氢键；丁酸形成更强的氢键二聚体；丁二酸为二元酸，能形成更多氢键，沸点最高。

（3）沸点由低到高：丁烷 < 乙醚 < 丙酮 < 丙醇 < 乙酸

解析：丁烷仅范德华力，最低；乙醚有弱偶极；丙酮偶极矩较大；丙醇能形成氢键；乙酸形成氢键二聚体，最高。

**要点**：沸点排序的依据是分子间作用力强弱：氢键 > 偶极-偶极 > 范德华力。同类作用力下，分子量大的沸点高；同分子量下，支链多的沸点低；能形成氢键二聚体的（如羧酸）沸点异常高。
