**题目**：写出下列试剂与环氧乙烷反应所得主要产物的结构。

**解**：

环氧乙烷由于**环张力大**，容易发生**开环反应**。

**开环规律**：

1. **碱性条件/中性条件**（亲核试剂如 RMgX、RO-、NH3 等）：
   - 亲核试剂进攻**位阻小**的碳（端碳）；
   - SN2 机理，反式开环；
   - 产物为亲核基团连在位阻小的碳上。

2. **酸性条件**（H+ 催化）：
   - 质子化后，亲核试剂进攻**取代基多**的碳（更稳定的碳正离子过渡态）；
   - 产物为亲核基团连在取代基多的碳上。

**常见反应**：
- 环氧乙烷 + H2O（酸催化）→ 乙二醇（HOCH2CH2OH）
- 环氧乙烷 + ROH → 醚醇（ROCH2CH2OH）
- 环氧乙烷 + NH3 → 乙醇胺（H2NCH2CH2OH）
- 环氧乙烷 + RMgX → 增碳醇（RCH2CH2OH，伯醇）
- 环氧乙烷 + HX → 卤代醇（XCH2CH2OH）

**要点**：
1. 环氧乙烷开环是制备双官能团化合物的重要方法；
2. 酸催化 → 进攻取代基多的碳；碱催化 → 进攻位阻小的碳；
3. 开环为反式（SN2）；
4. 与格氏试剂反应可增碳制备伯醇。
