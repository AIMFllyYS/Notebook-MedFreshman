**题目**：有三种化合物 A、B、C 都具有分子式 C5H8，都能使溴的 CCl4 溶液退色。A 与硝酸银的氨溶液作用生成沉淀，而 B、C 则不能；当用热的高锰酸钾氧化时，A 得到 CH3CH2CH2COOH 和 CO2，B 得到乙酸和丙酸，C 得到戊二酸。试指出 A、B、C 的构造式。

**解**：

**第一步：计算不饱和度**

C5H8 的不饱和度 = (2×5+2-8)/2 = 2

分子中存在两个双键，或一个叁键，或一个双键和一个环。均能使溴退色，说明三个分子中均含有不饱和键。

**第二步：判断 A**

A 与 [Ag(NH3)2]+ 生成沉淀 → A 为**端基炔**（含 -C≡CH）。

A 氧化得 CH3CH2CH2COOH + CO2 → 端基炔氧化断裂，CO2 来自端基 -C≡CH，CH3CH2CH2COOH 来自另一段。

故 A 为：**1-戊炔**（CH≡C-CH2-CH2-CH3）

**第三步：判断 B**

B 不与银氨溶液反应 → 非端基炔，可能为内炔或二烯烃。

B 氧化得 CH3COOH + CH3CH2COOH → 双键（或叁键）断裂为两段，一段含 2 碳（乙酸），一段含 3 碳（丙酸）。

故 B 为：**2-戊炔**（CH3-C≡C-CH2-CH3）

**第四步：判断 C**

C 不与银氨溶液反应，氧化得**戊二酸**（HOOC-CH2-CH2-CH2-COOH，5 碳）→ 氧化未断链，说明含一个双键和一个环。

故 C 为：**环戊烯**（含一个环和一个双键）

**要点**：
1. 端基炔可用银氨溶液鉴别（生成沉淀）；
2. 热高锰酸钾氧化使不饱和键断裂，根据氧化产物可逆推原结构；
3. 炔氧化断裂得羧酸，端基炔还产生 CO2；
4. 环烯氧化得二元羧酸（不断链）。
