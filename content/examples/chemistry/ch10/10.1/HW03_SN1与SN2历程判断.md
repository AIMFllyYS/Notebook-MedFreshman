**题目**：卤代烷与氢氧化钠在水与乙醇混合液中进行反应，根据下列实验事实，判断 SN 历程。

（1）一级卤代烷速率大于三级卤代烷
（2）碱的浓度增加，反应速率无明显变化
（3）两步反应，第一步是决定反应速率的步骤
（4）增加溶剂的含水量，反应速率明显加快
（5）产物构型 80% 消旋，20% 转化
（6）进攻试剂亲和性愈强，反应速率愈快
（7）有重排现象
（8）增加溶剂含醇量，降低含水量，反应速率加快

**解**：

| 实验事实 | SN 历程 | 理由 |
| --- | --- | --- |
| （1）一级 > 三级 | **SN2** | SN2 位阻敏感，一级位阻小 |
| （2）碱浓度不影响速率 | **SN1** | SN1 速率只与底物有关 |
| （3）两步反应，第一步决速 | **SN1** | SN1 先生成碳正离子（慢步） |
| （4）含水量增加，速率加快 | **SN1** | 极性溶剂稳定碳正离子 |
| （5）80% 消旋，20% 转化 | **SN1** | 平面碳正离子两面进攻，以外消旋为主 |
| （6）亲核性强，速率快 | **SN2** | SN2 速率与亲核试剂浓度和强度有关 |
| （7）有重排现象 | **SN2** | 实际应为 SN1（碳正离子可重排）* |
| （8）含醇量增加，速率加快 | **SN2** | 非极性溶剂有利于 SN2 |

*注：（7）有重排现象应为 **SN1**（碳正离子重排是 SN1 的特征）。

**SN1 与 SN2 对比**：

| 特征 | SN1 | SN2 |
| --- | --- | --- |
| 速率 | 仅与底物有关 | 与底物和亲核试剂有关 |
| 中间体 | 碳正离子 | 无（一步反应） |
| 立体化学 | 外消旋化 | 构型翻转 |
| 重排 | 有 | 无 |
| 底物活性 | 3° > 2° > 1° | 1° > 2° > 3° |
| 溶剂 | 极性溶剂有利 | 非极性溶剂有利 |

**要点**：判断 SN1/SN2 需综合考虑底物结构、亲核试剂、溶剂、立体化学和重排等因素。
