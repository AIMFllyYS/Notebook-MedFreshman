**题目**：卤代烃的分子式为 C6H11Cl (A)，构型为 R，A 经水解后得 C6H11OH (B)，其构型不变，A 经氢化饱和后可得卤代烃 C6H13Cl (C)，其旋光性消失。试写出 A、B、C 的结构式及其构型，并写出相关的反应式。

**解**：

**第一步：计算不饱和度**

C6H11Cl 的不饱和度 = (2×6+2-11-1)/2 = 1，存在 1 个双键或环。

**第二步：分析构型不变**

A 构型为 R，水解后 B 构型不变 → **Cl 没有连接在手性碳原子上**（若 Cl 在手性碳上，SN2 会翻转，SN1 会外消旋）。

**第三步：分析氢化后旋光消失**

A 氢化后 C 旋光性消失 → 存在双键，且双键氢化后生成的基团与另一基团相同 → 可确定双键为**乙烯基**（-CH=CH2），另一基团为 -C2H5。

**第四步：确定结构**

手性碳上连接的四个基团：
- -H
- -CH2Cl
- -CH=CH2（乙烯基）
- -CH2CH3（乙基）

氢化后 -CH=CH2 变为 -CH2CH3，与原有 -CH2CH3 相同，手性消失。

**第五步：确定 R 构型**

根据 R 构型，确定手性 C 原子上四个基团的连接顺序（按顺序规则排列）。

**结构**：
- A：(R)-3-氯甲基-1-己烯（C6H11Cl，含乙烯基和氯甲基）
- B：(R)-3-羟甲基-1-己烯（C6H11OH，构型不变）
- C：无手性的 C6H13Cl（氢化后两个乙基相同）

**反应式**：
- A + NaOH/H2O → B（构型保持，因 Cl 不在手性碳上）
- A + H2/Ni → C（双键加氢，手性消失）

**要点**：
1. 构型不变 → 反应未涉及手性碳上的键断裂；
2. 氢化后旋光消失 → 双键还原导致两个基团变为相同；
3. 不饱和度计算是推断的关键；
4. R/S 构型需按顺序规则确定。
