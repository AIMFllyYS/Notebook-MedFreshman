**题目**：

1. 名词解释：多肽的一级结构。

2. 多肽一级结构的测定包含哪些内容？分别用什么方法测定？

3. 蛋白质分子中的主键及主要副键有哪些？

**解**：

**1. 多肽一级结构**：

氨基酸在多肽链中的**排列顺序**为多肽的一级结构。

**2. 多肽一级结构的测定**：

测定内容：
- 组成多肽的氨基酸的**种类**；
- 组成多肽的氨基酸的**数目**；
- 氨基酸的**排列顺序**。

测定方法：
- **末端残基分析法**：确定肽链两端的氨基酸（N 端和 C 端）；
  - N 端：Sanger 法（2,4-二硝基氟苯法）、Edman 降解法（苯异硫氰酸酯法）；
  - C 端：羧肽酶法；
- **部分水解法**：将多肽部分水解为短肽片段，分别测定各片段的序列，再拼接出完整序列。

**3. 蛋白质的主键及副键**：

- **主键**：**肽键**（-CO-NH-，氨基酸之间通过酰胺键连接）；
- **副键**（维持高级结构）：
  - **氢键**（最重要，维持 α-螺旋和 β-折叠）；
  - **离子键**（盐键，酸性/碱性氨基酸侧链间）；
  - **疏水键**（非极性侧链间的疏水作用）；
  - **二硫键**（-S-S-，半胱氨酸之间，共价键）；
  - **酯键**（侧链 -OH 与 -COOH 之间）。

**要点**：
1. 一级结构 = 氨基酸序列；
2. 末端分析法确定 N 端和 C 端；
3. 部分水解法拼接完整序列；
4. 肽键为主键，氢键/离子键/疏水键/二硫键为副键。
