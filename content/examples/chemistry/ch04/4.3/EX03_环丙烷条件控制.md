环丙烷 $\ce{C3H6}$ 与氯气反应，在不同条件下结果迥异。请分别写出（1）光照下、（2）路易斯酸 $\ce{FeCl3}$ 催化下两种条件的反应方向（取代还是开环）。另外，写出环丙烷与 $\ce{Br2}$、$\ce{HBr}$、$\ce{H2O}$（酸催化）开环加成的产物。并说明小环为什么容易开环。
:::
:::insight{label=分析}
小环（三元环、四元环）的特殊性是容易开环加成。同一对反应物在不同条件下方向可能相反，所以"条件至上"。开环的根本原因要从三元环的真实键角与"香蕉键"去解释。
:::
:::note{label=解答}
**环丙烷 + 氯气，条件决定方向：**
- **光照** ⇒ 自由基**取代**（取代到环上，环不开）；
- **$\ce{FeCl3}$（路易斯酸）催化** ⇒ **开环加成**。

**环丙烷的其他开环加成产物：**
- 与 $\ce{Br2}$（室温）开环，溴加到两端碳上，生成 1,3-二溴丙烷：$\ce{C3H6 + Br2 -> BrCH2CH2CH2Br}$
- 与 $\ce{HBr}$ 开环加成，生成 1-溴丙烷：$\ce{C3H6 + HBr -> CH3CH2CH2Br}$
- 与 $\ce{H2O}$（浓硫酸催化）开环加成，相当于把一分子水加到环上（一头氢、一头羟基），生成丙醇：$\ce{C3H6 + H2O ->[H2SO4] CH3CH2CH2OH}$

**小环易开环的原因：**三元环真实键角约 **105.3°**，仅偏离标准 $\ce{sp^3}$ 键角 109.5° 约 4°（并非画成正三角形时想象的 60°）。三元环为尽量维持键角，牺牲的是**电子云的重叠程度**：成键电子云向外弯曲成弧形，形似香蕉，称为**香蕉键**。电子云重叠程度越小、键越弱、越容易断裂，这才是小环容易开环的真正原因。
:::
