:::example{label=甲烷氯代生成氯乙烷}
**题目**：解释甲烷与氯气进行氯代反应时会有少量氯代乙烷生成的原因。

**解**：

甲烷氯代反应是自由基链式反应：

**链引发**：
Cl2 →(光或热) 2 Cl·

**链增长**：
- Cl· + CH4 → HCl + ·CH3
- ·CH3 + Cl2 → CH3Cl + Cl·

**氯代乙烷的生成**：
在链增长过程中，生成的甲基自由基（·CH3）除了与 Cl2 反应外，还可与生成的氯甲烷（CH3Cl）发生反应：

·CH3 + CH3Cl → CH3-CH3 + Cl·

生成的乙烷（CH3-CH3）再次参与到反应中，与氯自由基反应：

Cl· + CH3CH3 → HCl + ·CH2CH3
·CH2CH3 + Cl2 → CH3CH2Cl + Cl·

因此会生成少量氯代乙烷（CH3CH2Cl）。

**要点**：自由基反应中，各种自由基可与各种含氢底物反应，导致副产物生成。这是自由基反应选择性较差的原因。
:::
