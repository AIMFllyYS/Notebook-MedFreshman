**题目**：写出下列反应的各步产物。

含氮化合物的多步转化反应

**解**：

**含氮化合物的多步反应**常涉及以下转化：

1. **硝基苯 → 苯胺**：
   - 硝基苯 + Fe/HCl（或 Sn/HCl）→ 苯胺（还原）

2. **苯胺 → 重氮盐**：
   - 苯胺 + NaNO2/HCl（0-5 ℃）→ 重氮盐（重氮化反应）

3. **重氮盐的各类反应**：
   - 重氮盐 + H2O/Δ → 苯酚（水解）
   - 重氮盐 + HCl/CuCl → 氯苯（Sandmeyer 反应）
   - 重氮盐 + HBr/CuBr → 溴苯
   - 重氮盐 + KCN/CuCN → 苯甲腈
   - 重氮盐 + H3PO2 → 苯（还原去氮）
   - 重氮盐 + 苯酚/胺 → 偶氮化合物（偶合反应）

4. **季铵碱的霍夫曼消除**：
   - 季铵盐 + Ag2O → 季铵碱
   - 季铵碱 + Δ → 烯烃 + 叔胺（霍夫曼消除）

**要点**：
1. 重氮盐是重要的合成中间体，可转化为多种官能团；
2. Sandmeyer 反应引入 -Cl、-Br、-CN；
3. 偶合反应生成偶氮染料；
4. 霍夫曼消除生成双键上烷基最少的烯烃。
