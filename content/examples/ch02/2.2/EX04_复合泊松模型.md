:::example{label=复合泊松模型——泊松分布稳定性}
**复合泊松模型**（Compound Poisson Model）是泊松分布在随机"稀释"或"筛选"下的行为研究。本例题的核心结论是：泊松分布在独立伯努利筛选（Bernoulli thinning）下仍然保持泊松分布，这被称为泊松分布的**稳定性**（Stability Property）。

**问题背景**：
- 到某商场购物的顾客数 $Y$ 服从泊松分布，$Y \sim P(\lambda)$（每小时平均 $\lambda$ 名顾客到达）
- 每位顾客以概率 $p$ 独立地购买电视机，各顾客的购买决策相互独立
- $X$ 为一小时内卖出的电视机台数

**核心结论**：$X \sim P(\lambda p)$，即销售量服从参数为 $\lambda p$ 的泊松分布。
:::

:::insight{label=分析}
**这是一个两层随机过程**：
1. 第一层（泊松过程）：顾客按泊松过程到达，人数 $Y \sim P(\lambda)$
2. 第二层（伯努利筛选）：每位到达的顾客以概率 $p$ 购买，相当于 $Y$ 次独立的伯努利试验

即 $X = Z_1 + Z_2 + \cdots + Z_Y$，其中每个 $Z_i \sim B(1, p)$，且 $Z_i$ 之间、$Z_i$ 与 $Y$ 之间相互独立。

**意义**：到达的顾客流（强度 $\lambda$）经过概率为 $p$ 的"筛选"（购买）后，得到的购买流的强度恰好为 $\lambda p$，仍然是泊松流。这正是泊松分布对独立稀释的封闭性。

**推广（可加性）**：若 $X_1 \sim P(\lambda_1)$，$X_2 \sim P(\lambda_2)$，且 $X_1$，$X_2$ 独立，则 $X_1 + X_2 \sim P(\lambda_1 + \lambda_2)$。这是泊松分布稳定性的另一种表现：多个独立泊松过程的叠加仍是泊松过程，参数相加。
:::

:::theorem{label=关键公式}
**设定**：$Y \sim P(\lambda)$，$X \mid Y = n \sim B(n, p)$，求 $P(X = k)$

**全概率公式展开**：

$$P(X = k) = \sum_{n=k}^{\infty} P(X = k \mid Y = n) \cdot P(Y = n)$$

**代入条件分布和泊松分布**：

$$P(X = k) = \sum_{n=k}^{\infty} \binom{n}{k} p^k (1-p)^{n-k} \cdot \frac{\lambda^n e^{-\lambda}}{n!}$$

**提取公因子**：

$$= \frac{e^{-\lambda} p^k}{k!} \sum_{n=k}^{\infty} \frac{\lambda^n (1-p)^{n-k}}{(n-k)!}$$

**换元** $m = n - k$：

$$= \frac{e^{-\lambda} p^k \lambda^k}{k!} \sum_{m=0}^{\infty} \frac{[\lambda(1-p)]^m}{m!} = \frac{e^{-\lambda} (\lambda p)^k}{k!} \cdot e^{\lambda(1-p)}$$

**化简指数部分**：$e^{-\lambda} \cdot e^{\lambda(1-p)} = e^{-\lambda + \lambda - \lambda p} = e^{-\lambda p}$

**最终结果**：

$$\boxed{P(X = k) = \frac{(\lambda p)^k e^{-\lambda p}}{k!}, \quad k = 0, 1, 2, \ldots}$$

即 $X \sim P(\lambda p)$。

**期望和方差**：

$$E(X) = \lambda p, \quad D(X) = \lambda p$$

**泊松分布稳定性（可加性）**：

$$X_1 \sim P(\lambda_1),\; X_2 \sim P(\lambda_2),\; X_1 \perp X_2 \implies X_1 + X_2 \sim P(\lambda_1 + \lambda_2)$$
:::

:::note{label=直观理解}
**"筛选"的比喻**：想象一条河流（到达的顾客流，强度 $\lambda$），经过一个"过滤网"（购买决策，通过概率 $p$），流出的"净水"（购买的顾客）的流量强度变为 $\lambda p$。泊松流经过伯努利筛选后，仍然是泊松流，参数按比例缩减。

**三个参数的物理意义**：
- $\lambda$：原始强度（顾客到达率）
- $p$：筛选概率（购买概率）
- $\lambda p$：筛选后的强度（购买率）

**可加性的直觉**：两个独立的泊松流合并（叠加）后，总流量是两者之和，强度也相加。例如，A 品牌手机每小时卖出 $P(3)$ 台，B 品牌每小时卖出 $P(4)$ 台（独立），则总销量 $P(7)$。

**为什么这个结论重要？**
1. 简化计算：不需要对每种顾客人数 $Y = n$ 分情况讨论，直接得到销售量的分布
2. 建模灵活性：在保险精算（理赔次数的筛选）、通信工程（数据包的传输与丢失）、流行病学（感染者中发病的比例）等领域广泛适用
3. 泊松分布的"优美性"：泊松分布在多种随机操作（叠加、筛选）下都保持泊松性，这是泊松过程成为随机过程中最重要基本模型的深层原因
:::

:::pitfall{label=易错点1}
**不知道使用全概率公式**。当随机变量 $X$ 依赖于另一个随机变量 $Y$（即已知 $Y$ 后 $X$ 的条件分布很简单，但边际分布未知），标准做法就是用全概率公式 $P(X=k) = \sum_n P(X=k|Y=n) P(Y=n)$，这是解决混合分布问题的通用工具。
:::

:::pitfall{label=易错点2}
**求和范围写错**。$P(X=k) = \sum_{n=k}^{\infty} \ldots$ 中，下界是 $n=k$（因为 $Y = n$ 时，$X$ 最多为 $n$，$X = k$ 要求 $n \geq k$）。如果从 $n=0$ 开始，则当 $n < k$ 时，$\binom{n}{k} = 0$，项为零，结果不变，但数学上应从 $n=k$ 开始。
:::

:::pitfall{label=易错点3}
**换元后范围计算错误**。令 $m = n-k$，当 $n$ 从 $k$ 到 $\infty$ 变化时，$m$ 从 $0$ 到 $\infty$ 变化，范围要正确转换。
:::

:::pitfall{label=易错点4}
**误认为结论不需要推导**。直接宣称"泊松分布筛选后还是泊松分布"而不给出推导过程，在考试中是不完整的。掌握全概率公式 + 换元 + 识别泰勒展开这三个步骤是必要的。
:::

:::pitfall{label=易错点5}
**混淆稳定性与可加性**。稳定性（筛选）：$Y \sim P(\lambda)$，每个成功以概率 $p$ 保留，结果 $X \sim P(\lambda p)$。可加性（叠加）：$X_1 \sim P(\lambda_1)$，$X_2 \sim P(\lambda_2)$，独立，则 $X_1 + X_2 \sim P(\lambda_1 + \lambda_2)$。两者是不同的操作，结论形式也不同。
:::

:::note{label=知识关联}
**与泊松分布基础（KP03）的关系**：本例深化了 KP03 的内容，从单纯描述泊松分布的 PMF 和期望，进一步展示了泊松分布在随机筛选操作下的封闭性，揭示了泊松分布的代数结构。

**与条件分布和全概率公式的关系**：本例的关键计算工具是全概率公式，将"条件已知时简单的分布"整合为"边际分布"。这是概率论中处理混合模型、层次模型的基本技术。

**与保险精算的关系**：保险公司收到的理赔次数 $N \sim P(\lambda)$（到达的索赔），每次索赔金额超过某阈值的概率为 $p$（被保险覆盖），则实际赔付次数 $X \sim P(\lambda p)$，直接应用了本例的结论。

**与随机过程（泊松过程）的关系**：复合泊松模型的结论是随机过程理论中"泊松流的独立稀释仍是泊松流"这一定理的离散版本，是泊松过程理论的基础。
:::
