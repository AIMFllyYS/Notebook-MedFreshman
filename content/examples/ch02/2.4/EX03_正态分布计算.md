:::example{label=正态分布计算}
设随机变量 $X \sim N(3, 4)$（即均值 $\mu = 3$，方差 $\sigma^2 = 4$，标准差 $\sigma = 2$）。

利用标准正态分布函数 $\Phi$ 表（已知 $\Phi(1) = 0.8413$，$\Phi(2) = 0.9772$），计算下列概率：

**(1)** $P(1 < X < 5)$

**(2)** $P(X > 7)$

**(3)** $P(|X - 3| < 4)$（即 $P(-1 < X < 7)$）

**(4)** 求常数 $c$，使得 $P(X > c) = 0.9$。
:::

:::insight{label=分析}
本题是正态分布标准化的综合练习，每一问都需要以下"三步走"流程：

**第一步**：确认参数，$\mu = 3$，$\sigma = 2$（注意 $N(3,4)$ 中 $4$ 是方差，$\sigma = \sqrt{4} = 2$）。

**第二步**：标准化——将 $X$ 的上下界转化为 $Z$ 分数：$z = (x - 3)/2$。

**第三步**：利用 $\Phi$ 函数及其对称性 $\Phi(-z) = 1 - \Phi(z)$ 计算概率。

**注意**：第(4)问是"反问题"——已知概率求分位点，需要利用 $\Phi$ 函数的反函数。
:::

:::derivation{label=解答}
**参数确认**：$X \sim N(3, 4)$，故 $\mu = 3$，$\sigma^2 = 4$，$\sigma = 2$。

标准化变量 $Z = \dfrac{X - 3}{2} \sim N(0, 1)$。

### 第(1)问：$P(1 < X < 5)$

将端点标准化：
$$
\frac{1 - 3}{2} = \frac{-2}{2} = -1, \qquad \frac{5 - 3}{2} = \frac{2}{2} = 1
$$

$$
P(1 < X < 5) = P\!\left(-1 < Z < 1\right) = \Phi(1) - \Phi(-1)
$$

利用对称性 $\Phi(-1) = 1 - \Phi(1) = 1 - 0.8413 = 0.1587$：

$$
P(1 < X < 5) = 0.8413 - 0.1587 = \boxed{0.6826}
$$

这正是 $3\sigma$ 原则中 $P(\mu - \sigma < X < \mu + \sigma) \approx 68.26\%$ 的验证（此处区间恰为 $\mu \pm \sigma = 3 \pm 2$，即 $[1, 5]$）。

### 第(2)问：$P(X > 7)$

将端点标准化：$\dfrac{7 - 3}{2} = 2$

$$
P(X > 7) = P(Z > 2) = 1 - \Phi(2) = 1 - 0.9772 = \boxed{0.0228}
$$

即约有 $2.28\%$ 的概率超过 $\mu + 2\sigma = 7$，符合 $3\sigma$ 原则的直觉（超出 $2\sigma$ 外的单侧概率约为 $2.28\%$）。

### 第(3)问：$P(|X - 3| < 4)$

首先展开绝对值不等式：$|X - 3| < 4 \iff -4 < X - 3 < 4 \iff -1 < X < 7$

将端点标准化：
$$
\frac{-1 - 3}{2} = -2, \qquad \frac{7 - 3}{2} = 2
$$

$$
P(-1 < X < 7) = P(-2 < Z < 2) = \Phi(2) - \Phi(-2)
$$

利用对称性 $\Phi(-2) = 1 - \Phi(2) = 1 - 0.9772 = 0.0228$：

$$
P(-1 < X < 7) = 0.9772 - 0.0228 = \boxed{0.9544}
$$

此结果对应 $3\sigma$ 原则中 $P(\mu - 2\sigma < X < \mu + 2\sigma) \approx 95.44\%$（区间 $[-1, 7] = [3-4, 3+4] = [3-2\times2, 3+2\times2]$）。

### 第(4)问：求 $c$ 使 $P(X > c) = 0.9$

由 $P(X > c) = 0.9$ 得 $P(X \leq c) = 1 - 0.9 = 0.1$，即：

$$
\Phi\!\left(\frac{c - 3}{2}\right) = 0.1
$$

由于 $0.1 < 0.5$，$c < \mu = 3$，所以 $\dfrac{c-3}{2} < 0$，令 $z_0 = \dfrac{c-3}{2}$：

$$
\Phi(z_0) = 0.1 \implies \Phi(-z_0) = 1 - 0.1 = 0.9
$$

查表：$\Phi(1.28) \approx 0.8997 \approx 0.9$，故 $-z_0 \approx 1.28$，即 $z_0 \approx -1.28$。

$$
\frac{c - 3}{2} = -1.28 \implies c = 3 + 2 \times (-1.28) = 3 - 2.56 = \boxed{0.44}
$$

**验证**：$P(X > 0.44) = P\!\left(Z > \frac{0.44-3}{2}\right) = P(Z > -1.28) = \Phi(1.28) \approx 0.9$ ✓
:::

:::theorem{label=关键公式}
**标准化变换**（$\mu = 3$，$\sigma = 2$）：
$$
Z = \frac{X - 3}{2} \sim N(0, 1)
$$

**各问结果汇总**：
$$
P(1 < X < 5) = \Phi(1) - \Phi(-1) = 2\Phi(1) - 1 \approx 0.6826
$$
$$
P(X > 7) = 1 - \Phi(2) \approx 0.0228
$$
$$
P(-1 < X < 7) = \Phi(2) - \Phi(-2) = 2\Phi(2) - 1 \approx 0.9544
$$
$$
P(X > c) = 0.9 \implies c = 3 - 2 \times 1.28 = 0.44
$$

**对称区间概率通式**：
$$
P(|X - \mu| < k\sigma) = 2\Phi(k) - 1
$$
:::

:::pitfall{label=易错点1}
**将 $N(3, 4)$ 中的 $4$ 误认为 $\sigma = 4$**

$N(\mu, \sigma^2)$ 的第二个参数是**方差** $\sigma^2 = 4$，标准差 $\sigma = \sqrt{4} = 2$。若误用 $\sigma = 4$，标准化公式 $z = (x-3)/4$ 将导致所有概率计算错误。拿到题目第一步先提取 $\sigma$（取方差的平方根），避免后续连锁出错。
:::

:::pitfall{label=易错点2}
**处理 $P(X > c) = 0.9$ 的反问题时方向搞混**

$P(X > c) = 0.9 > 0.5$ 意味着 $c < \mu$（$c$ 在均值左侧），对应负 $z$ 值。若直接查表 $\Phi^{-1}(0.9) \approx 1.28$ 后直接取正值代入，会得到 $c = 3 + 2 \times 1.28 = 5.56$（大于 $\mu$），而此时 $P(X > 5.56) = 1 - \Phi(1.28) \approx 0.1 \neq 0.9$，完全错误。必须先分析 $c$ 是在 $\mu$ 的左侧还是右侧。
:::

:::pitfall{label=易错点3}
**第(3)问展开绝对值出错**

$|X - 3| < 4$ 等价于 $-4 < X - 3 < 4$，即 $-1 < X < 7$（两侧各加 $\mu = 3$）。注意不是 $|X| < 7$，展开绝对值时要对称地减去和加上 $4$，不要漏掉加 $\mu$ 的步骤。
:::
