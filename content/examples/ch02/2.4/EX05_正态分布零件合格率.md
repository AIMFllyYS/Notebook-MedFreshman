:::example{label=正态分布零件合格率}
某工厂生产一批轴承，其直径（毫米）$X \sim N(50, 0.04)$（即均值 $\mu = 50$ mm，方差 $\sigma^2 = 0.04$，标准差 $\sigma = 0.2$ mm）。

规定直径在 $[49.6, 50.4]$ mm 范围内为合格品。

**(1)** 求该批产品的合格率（即一个零件为合格品的概率）。

**(2)** 从该批产品中随机抽取 $1000$ 件，期望合格品有多少件？

**(3)** 若希望合格率不低于 $99\%$，求合格区间 $[50 - \delta, 50 + \delta]$ 中 $\delta$ 的最小值（精确到 $0.001$ mm）。
:::

:::insight{label=分析}
本题综合考查正态分布标准化和 $3\sigma$ 原则的应用，是典型的质量控制类应用题。

**解题框架**：

- 第(1)(2)问是标准化概率计算：$P(49.6 < X < 50.4) = P\left(\frac{49.6-50}{0.2} < Z < \frac{50.4-50}{0.2}\right)$，标准化后识别这正是 $3\sigma$ 原则中 $\pm 2\sigma$ 区间的概率。
- 第(3)问是**反向推导**：从目标概率 $0.99$ 反推 $\delta$，用 $\Phi$ 函数的逆函数（查反表）。

**参数提取**：$\sigma^2 = 0.04 \implies \sigma = \sqrt{0.04} = 0.2$（这是关键一步，方差转标准差）。
:::

:::derivation{label=解答}
**参数确认**：$\mu = 50$ mm，$\sigma^2 = 0.04$，$\sigma = 0.2$ mm。

### 第(1)步：计算合格率

$$P(49.6 < X < 50.4) = P\!\left(\frac{49.6 - 50}{0.2} < Z < \frac{50.4 - 50}{0.2}\right)$$

标准化端点：
$$\frac{49.6 - 50}{0.2} = \frac{-0.4}{0.2} = -2, \qquad \frac{50.4 - 50}{0.2} = \frac{0.4}{0.2} = 2$$

$$P(49.6 < X < 50.4) = P(-2 < Z < 2) = \Phi(2) - \Phi(-2) = 2\Phi(2) - 1$$

代入 $\Phi(2) = 0.9772$：

$$= 2 \times 0.9772 - 1 = 1.9544 - 1 = \boxed{0.9544}$$

合格率约为 $\mathbf{95.44\%}$。这正是 $3\sigma$ 原则中 $P(\mu - 2\sigma < X < \mu + 2\sigma) \approx 95.44\%$ 的体现，因为合格区间 $[49.6, 50.4] = [\mu - 2\sigma, \mu + 2\sigma]$。

### 第(2)步：$1000$ 件中期望合格数

合格品数量 $Y \sim B(1000, 0.9544)$（二项分布），期望合格件数为：

$$E(Y) = 1000 \times 0.9544 = \boxed{954.4 \text{ 件}} \approx 954 \text{ 件}$$

即期望有约 $954$ 件合格品，约 $46$ 件不合格品。

### 第(3)步：求满足合格率 $\geq 99\%$ 的最小 $\delta$

合格区间为 $[50 - \delta, 50 + \delta]$，合格率为：

$$P(50 - \delta < X < 50 + \delta) = P\!\left(\frac{-\delta}{0.2} < Z < \frac{\delta}{0.2}\right) = P\!\left(-\frac{\delta}{0.2} < Z < \frac{\delta}{0.2}\right)$$

$$= \Phi\!\left(\frac{\delta}{0.2}\right) - \Phi\!\left(-\frac{\delta}{0.2}\right) = 2\Phi\!\left(\frac{\delta}{0.2}\right) - 1$$

要求合格率 $\geq 99\%$：

$$2\Phi\!\left(\frac{\delta}{0.2}\right) - 1 \geq 0.99$$

$$\Phi\!\left(\frac{\delta}{0.2}\right) \geq \frac{1 + 0.99}{2} = 0.995$$

查标准正态分布表，找 $z$ 使 $\Phi(z) = 0.995$：

$$\Phi(2.576) \approx 0.9950 \implies \frac{\delta}{0.2} \geq 2.576$$

$$\delta \geq 2.576 \times 0.2 = 0.5152 \text{ mm}$$

取精确到 $0.001$ mm 的最小值：

$$\delta_{\min} = \boxed{0.516 \text{ mm}}$$

**验证**：合格率 $= 2\Phi(0.516/0.2) - 1 = 2\Phi(2.58) - 1 \approx 2 \times 0.9951 - 1 = 0.9902 > 0.99$ ✓

**结论**：合格区间至少需要设置为 $[49.484, 50.516]$ mm，才能保证合格率不低于 $99\%$。
:::

:::theorem{label=关键公式}
**标准化（$\mu = 50$，$\sigma = 0.2$）**：
$$Z = \frac{X - 50}{0.2} \sim N(0, 1)$$

**合格率计算**：
$$P(49.6 < X < 50.4) = P(-2 < Z < 2) = 2\Phi(2) - 1 \approx 0.9544$$

**关于 $\mu$ 对称区间的通用公式**：
$$P(\mu - \delta < X < \mu + \delta) = 2\Phi\!\left(\frac{\delta}{\sigma}\right) - 1$$

**反推 $\delta$ 的步骤**：
$$2\Phi\!\left(\frac{\delta}{0.2}\right) - 1 = 0.99 \implies \Phi\!\left(\frac{\delta}{0.2}\right) = 0.995 \implies \frac{\delta}{0.2} = 2.576 \implies \delta = 0.515 \text{ mm}$$
:::

:::pitfall{label=易错点1}
**$\sigma^2 = 0.04$ 时忘记开方求 $\sigma$**

$N(50, 0.04)$ 中 $0.04$ 是方差，标准差是 $\sigma = \sqrt{0.04} = 0.2$，而非 $\sigma = 0.04$。若错误地用 $\sigma = 0.04$ 标准化，端点变成 $(49.6-50)/0.04 = -10$，这会得到 $P(-10 < Z < 10) \approx 1$，明显不合理，应当警觉并重新检查参数。
:::

:::pitfall{label=易错点2}
**$3\sigma$ 原则数字对应错误**

合格率 $0.9544$ 对应 $\pm 2\sigma$ 区间（不是 $\pm 1\sigma$ 或 $\pm 3\sigma$）。本题中 $[49.6, 50.4] = [50 - 2\times 0.2, 50 + 2\times 0.2]$，确认这是 $\pm 2\sigma$ 区间后，直接引用 $3\sigma$ 原则得 $0.9544$，无需重新积分。混淆三个区间的概率值（0.6826 / 0.9544 / 0.9974）是常见失误。
:::

:::pitfall{label=易错点3}
**第(3)问的方程列错**

$2\Phi(\delta/0.2) - 1 \geq 0.99$ 变形后应得 $\Phi(\delta/0.2) \geq 0.995$（两边加 $1$ 除以 $2$），不能错写为 $\Phi(\delta/0.2) \geq 0.99$。这一步对应着"将双侧概率转化为单侧 $\Phi$ 函数值"，$(1 + 0.99)/2 = 0.995$ 是固定算法，需熟练掌握。
:::
