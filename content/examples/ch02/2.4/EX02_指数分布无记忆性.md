:::example{label=指数分布无记忆性}
某电子元件的寿命 $X$（小时）服从指数分布，已知其平均寿命为 $1000$ 小时。

**(1)** 求参数 $\lambda$，写出PDF和CDF。

**(2)** 求该元件能正常工作超过 $1200$ 小时的概率 $P(X > 1200)$。

**(3)** 若已知该元件已经正常工作了 $500$ 小时，求它还能继续正常工作超过 $1200$ 小时的概率 $P(X > 1700 \mid X > 500)$。

**(4)** 解释(3)中结果的实际意义，并说明"无记忆性"的含义。
:::

:::insight{label=分析}
本题分四步递进，核心是**无记忆性的理解与应用**：

- 第(1)步是基础：由期望 $E(X) = 1/\lambda = 1000$ 得 $\lambda$；
- 第(2)步是指数分布右尾概率的标准计算：$P(X > x) = e^{-\lambda x}$；
- 第(3)步是本题关键：表面上看是条件概率，但无记忆性让计算化简为直接套用 $P(X > t)$；
- 第(4)步加深理解：无记忆性意味着"过去的使用历史对未来寿命没有影响"。

**关键洞察**：$P(X > 1700 \mid X > 500) = P(X > 1700 - 500) = P(X > 1200)$，无记忆性使条件概率等于无条件概率。
:::

:::derivation{label=解答}
### 第(1)步：确定参数 $\lambda$

指数分布的期望 $E(X) = \dfrac{1}{\lambda}$，由题意：

$$\frac{1}{\lambda} = 1000 \implies \lambda = \frac{1}{1000} = 0.001$$

**概率密度函数（PDF）**：
$$f(x) = 0.001 \cdot e^{-0.001x}, \quad x > 0$$

**累积分布函数（CDF）**：
$$F(x) = 1 - e^{-0.001x}, \quad x > 0$$

### 第(2)步：计算 $P(X > 1200)$

利用指数分布的右尾公式 $P(X > x) = e^{-\lambda x}$：

$$P(X > 1200) = e^{-0.001 \times 1200} = e^{-1.2} \approx 0.3012$$

即约有 $30.12\%$ 的元件寿命超过 $1200$ 小时。

### 第(3)步：计算 $P(X > 1700 \mid X > 500)$

**方法一：直接用无记忆性**

指数分布的无记忆性公式：$P(X > s + t \mid X > s) = P(X > t)$

令 $s = 500$，$t = 1200$，则 $s + t = 1700$：

$$P(X > 1700 \mid X > 500) = P(X > 1200) = e^{-1.2} \approx \boxed{0.3012}$$

**方法二：用条件概率定义验证**

$$P(X > 1700 \mid X > 500) = \frac{P(X > 1700 \text{ 且 } X > 500)}{P(X > 500)} = \frac{P(X > 1700)}{P(X > 500)}$$

$$= \frac{e^{-0.001 \times 1700}}{e^{-0.001 \times 500}} = \frac{e^{-1.7}}{e^{-0.5}} = e^{-1.7+0.5} = e^{-1.2} \approx 0.3012 \checkmark$$

两种方法结果一致，互相验证。

### 第(4)步：无记忆性的解释

计算结果显示：一个已经正常工作了 $500$ 小时的元件，**还能再工作超过 $1200$ 小时**的概率，与一个**全新元件直接能工作超过 $1200$ 小时**的概率完全相同（均为 $30.12\%$）。

这就是指数分布的**无记忆性**：元件"不会老化"，已经使用的历史对未来寿命没有任何影响。用过 $500$ 小时的旧元件，其"剩余寿命"的分布与新元件的寿命分布完全一致。

**现实意义**：这在实际中意味着，如果电子元件的失效是由随机的"突发性故障"引起（而非累积磨损），则无记忆性模型是合理的。纯电子器件（如二极管、电容）接近于这种模型；而机械零件因有磨损积累，更适合用威布尔分布建模。
:::

:::theorem{label=关键公式}
**参数关系**：
$$E(X) = \frac{1}{\lambda} \implies \lambda = \frac{1}{E(X)} = \frac{1}{1000} = 0.001$$

**右尾概率公式**：
$$P(X > x) = e^{-\lambda x}$$

**无记忆性公式（核心）**：
$$P(X > s + t \mid X > s) = P(X > t) = e^{-\lambda t}$$

**条件概率验证**：
$$P(X > 1700 \mid X > 500) = \frac{P(X > 1700)}{P(X > 500)} = \frac{e^{-1.7}}{e^{-0.5}} = e^{-1.2} \approx 0.3012$$
:::

:::pitfall{label=易错点1}
**将条件概率 $P(X > 1700 \mid X > 500)$ 误算为 $P(X > 1200 \mid X > 500)$**

题目问的是"已用 $500$ 小时，还能用超过 $1200$ 小时"，即总寿命超过 $500 + 1200 = 1700$ 小时，条件是"$X > 500$"。无记忆性下等于 $P(X > 1200)$，不要混淆 $1700$ 和 $1200$。
:::

:::pitfall{label=易错点2}
**在非指数分布中套用无记忆性**

无记忆性是指数分布的**特有性质**，仅对指数分布（连续）和几何分布（离散）成立。若题目给出的是正态分布或均匀分布，不能使用无记忆性公式。
:::

:::pitfall{label=易错点3}
**遗忘右尾公式，改用 $1 - F(x)$**

$P(X > x) = e^{-\lambda x}$ 是指数分布最常用的公式，直接给出右尾概率，无需通过 $1 - F(x) = 1 - (1 - e^{-\lambda x})$ 转化。直接记住右尾公式可节省计算步骤并减少错误。
:::
