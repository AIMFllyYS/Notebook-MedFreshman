:::note{label=核心概念}

本例题是**概率积分变换定理**的具体验证，通过指数分布这一最常见的连续分布直接演示定理的每一步推导。

**问题设定：** 设 $X \sim \text{Exp}(\lambda)$（参数为 $\lambda > 0$ 的指数分布），分布函数为 $F_X(x) = 1 - e^{-\lambda x}$（$x \geq 0$）。令

$$Y = 1 - e^{-\lambda X} = F_X(X)$$

即对 $X$ 做分布函数变换，验证 $Y \sim U(0, 1)$。

这道例题实现了两个目标：

1. **验证定理**：具体计算 $F_Y(y)$，证明它恰好等于 $y$（$U(0,1)$ 的分布函数），从而为一般性的概率积分变换定理提供具体的支撑。

2. **演示逆变换方法**：由 $Y = 1 - e^{-\lambda X}$ 可以反解出 $X = -\frac{\ln(1-Y)}{\lambda}$，若已知 $Y \sim U(0,1)$，则此式给出了从均匀分布生成指数分布样本的方法，是蒙特卡洛模拟的基础操作。

本题的技术要点包括：分布函数法的应用（对 $Y = g(X)$ 求 $F_Y$）、对数变换与不等式方向的处理、指数函数的单调性，以及最终结果的简洁验证。

:::

:::theorem{label=关键公式}

**指数分布的密度和分布函数：**

$$f_X(x) = \lambda e^{-\lambda x}, \quad x \geq 0$$
$$F_X(x) = 1 - e^{-\lambda x}, \quad x \geq 0$$

**变换定义：**

$$Y = 1 - e^{-\lambda X} = F_X(X)$$

**$Y$ 的取值范围：**

$$x \geq 0 \Rightarrow e^{-\lambda x} \in (0, 1] \Rightarrow 1 - e^{-\lambda x} \in [0, 1) \Rightarrow Y \in [0, 1)$$

**计算 $F_Y(y)$（分布函数法）：**

对 $0 \leq y < 1$：
$$F_Y(y) = P(Y \leq y) = P(1 - e^{-\lambda X} \leq y)$$
$$= P(e^{-\lambda X} \geq 1 - y) \quad \text{（不等号翻转！）}$$
$$= P(-\lambda X \geq \ln(1-y)) \quad \text{（取对数，因为 } e^t \text{ 单调递增）}$$
$$= P\!\left(X \leq -\frac{\ln(1-y)}{\lambda}\right) \quad \text{（除以 } -\lambda \text{ 再翻转不等号）}$$
$$= F_X\!\left(-\frac{\ln(1-y)}{\lambda}\right) = 1 - e^{-\lambda \cdot \left(-\frac{\ln(1-y)}{\lambda}\right)} = 1 - e^{\ln(1-y)} = 1 - (1-y) = y$$

**结论：**

$$F_Y(y) = y, \quad y \in [0, 1)$$

这正是 $U(0,1)$ 的分布函数，故 $Y \sim U(0, 1)$。$\blacksquare$

**逆变换公式（由此生成指数分布样本）：**

$$U \sim U(0,1) \Rightarrow X = -\frac{\ln(1-U)}{\lambda} \sim \text{Exp}(\lambda)$$

由于 $1-U$ 与 $U$ 同为 $U(0,1)$，也可以等价地写为 $X = -\frac{\ln U}{\lambda}$。

:::

:::note{label=直观理解}

**直觉一：分布函数"均匀化"的必然性**
指数分布在靠近 $0$ 的地方概率密度很高（$f_X(0) = \lambda$），在较大的 $x$ 处密度很低。当我们对 $X$ 做分布函数变换 $Y = F_X(X)$ 时：
- 对于密度高的区域，分布函数上升快，相同的 $X$ 变化量对应更大的 $Y$ 变化量，将"密集"的概率"分散"开了；
- 对于密度低的区域，分布函数上升慢，将"稀疏"的概率"压缩"回来。

这两种效果恰好抵消，使得 $Y$ 的分布变得均匀。

**直觉二：不等号翻转的物理意义**
$e^{-\lambda X} \geq 1-y$ 等价于 $X \leq -\frac{\ln(1-y)}{\lambda}$，不等号翻转的原因是：$e^{-\lambda x}$ 是关于 $x$ 的**严格递减函数**（$x$ 越大，$e^{-\lambda x}$ 越小），所以"$e^{-\lambda X}$ 偏大"等价于"$X$ 偏小"，不等号方向因此反转。

**直觉三：模拟的便利性**
已知 $U \sim U(0,1)$，通过 $X = -\ln(U)/\lambda$ 可以生成指数分布样本。如果 $\lambda = 1$，每次只需取 $-\ln U$ 即可，计算极为简单。这在统计模拟、随机事件生成（如模拟排队等待时间）中被广泛使用。

:::

:::pitfall{label=易错点1}

**不等号翻转遗漏**：在 $1 - e^{-\lambda X} \leq y$ 两边减 $1$ 得到 $-e^{-\lambda X} \leq y-1$，再乘以 $-1$ 时（得到 $e^{-\lambda X} \geq 1-y$），忘记翻转不等号，写成 $e^{-\lambda X} \leq 1-y$，导致后续所有推导方向错误。

:::

:::pitfall{label=易错点2}

**对数变换后不等号错误**：$e^{-\lambda X} \geq 1-y$ 两边取 $\ln$，因为 $\ln$ 是单调递增函数，不等号**不翻转**，得到 $-\lambda X \geq \ln(1-y)$。但随后除以 $-\lambda$（负数）时，不等号**必须翻转**。学生常常在两步之一遗漏翻转。

:::

:::pitfall{label=易错点3}

**$1-y$ 的定义域问题**：$1-y$ 必须为正才能取对数，即要求 $y < 1$。当 $y = 1$ 时 $\ln(1-y) \to -\infty$，这对应 $F_Y(1) = 1$，是 $U(0,1)$ 分布函数在端点的行为，应单独处理。

:::

:::pitfall{label=易错点4}

**混淆 $-\ln(1-U)$ 和 $-\ln U$**：两者都服从指数分布（因为 $1-U \sim U(0,1)$），但原推导给出的是 $-\ln(1-U)/\lambda$。若题目要求"写出生成指数分布的逆变换公式"，需要从 $Y = 1 - e^{-\lambda X}$ 出发反解，直接写 $-\ln U / \lambda$ 需要先说明 $U$ 与 $1-U$ 同分布。

:::

:::pitfall{label=易错点5}

**忘记验证 $F_Y$ 是合法分布函数**：得到 $F_Y(y) = y$ 后，需要验证：$F_Y(0) = 0$，$F_Y(1) = 1$，$F_Y$ 单调不减，$F_Y$ 右连续，这四条才能确认它是一个合法的分布函数。

:::

:::note{label=知识关联}

- **KP06（概率积分变换）**：本例题是KP06的一次完整的具体演示，所有推导步骤都来自KP06的定理框架。理解本题等于理解了KP06定理的"活" 版本。

- **KP04（分布函数法）**：本题的推导过程是分布函数法的标准操作：写 $F_Y(y) = P(g(X) \leq y)$，转化不等式，用 $F_X$ 计算。本题的特殊性在于 $g(x) = F_X(x)$，最终结果有特殊的简洁性。

- **KP05（公式法）**：变换 $Y = 1 - e^{-\lambda X}$ 是关于 $X$ 的严格递增函数（因为 $e^{-\lambda x}$ 单调递减，取负再加$1$后单调递增），可以用公式法验证。反函数 $x = h(y) = -\ln(1-y)/\lambda$，$|h'(y)| = 1/(\lambda(1-y))$，$f_Y(y) = f_X(h(y))|h'(y)| = \lambda e^{-\lambda \cdot (-\ln(1-y)/\lambda)} \cdot \frac{1}{\lambda(1-y)} = 1$，同样得 $Y \sim U(0,1)$。

- **KP02（混合型）**：指数分布是混合型变量（KP02）最常用的基础分布，指数分布的无记忆性 $P(X \geq s+t | X \geq s) = P(X \geq t)$ 保证了截断混合变量的原子概率有简洁的 $e^{-\lambda V_0}$ 形式，与本例题中 $e^{-\lambda x}$ 的计算呼应。

:::
