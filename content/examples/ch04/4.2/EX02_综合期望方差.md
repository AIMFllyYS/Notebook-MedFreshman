:::example{label=题目描述}

进行 $n$ 次独立重复的 Bernoulli 试验，每次试验成功的概率为 $p$（$0 < p < 1$），失败的概率为 $q = 1-p$。设第 $i$ 次试验的结果为

$$X_i = \begin{cases} 1, & \text{第 } i \text{ 次成功} \\ 0, & \text{第 } i \text{ 次失败} \end{cases}$$

令 $S_n = X_1 + X_2 + \cdots + X_n$ 为 $n$ 次试验中成功的总次数（$S_n \sim B(n, p)$）。

1. 求每个 $X_i$ 的期望 $E[X_i]$ 和方差 $D(X_i)$。
2. 利用期望的线性性求 $E[S_n]$。
3. 利用独立性求 $D(S_n)$。
4. 推导频率 $S_n/n$ 的期望和方差，并利用切比雪夫不等式估计 $P\left(\left|\dfrac{S_n}{n} - p\right| \geq \varepsilon\right)$ 的上界，从而看到大数定律的雏形。

:::

:::insight{label=分析}

本题是一道综合题，依次应用本章的核心知识点：

- **第1步**：直接用 $B(1,p)$ 的期望方差公式（或从定义推导）。
- **第2步**：期望线性性，将 $n$ 个 $X_i$ 的期望求和，无需独立性。
- **第3步**：方差加法，利用 $X_i$ 相互独立（题目已给出），将 $n$ 个 $X_i$ 的方差求和。
- **第4步**：将结果推广到频率 $S_n/n$，应用切比雪夫不等式，体会当 $n\to\infty$ 时频率如何收敛到概率 $p$。

:::

:::derivation{label=解答}

**第一步：求 $X_i$ 的期望和方差**

$X_i$ 服从两点分布 $B(1,p)$：$P(X_i=1)=p$，$P(X_i=0)=1-p=q$。

$$E[X_i] = 1 \times p + 0 \times q = p$$

$$E[X_i^2] = 1^2 \times p + 0^2 \times q = p$$

$$D(X_i) = E[X_i^2] - (E[X_i])^2 = p - p^2 = p(1-p) = pq$$

**第二步：利用期望的线性性求 $E[S_n]$**

$$E[S_n] = E[X_1 + X_2 + \cdots + X_n] = E[X_1] + E[X_2] + \cdots + E[X_n]$$

（线性性对任意随机变量成立，无需独立性）

$$= \underbrace{p + p + \cdots + p}_{n \text{ 个}} = np$$

$$\boxed{E[S_n] = np}$$

这与二项分布 $B(n,p)$ 的期望公式完全一致，且推导过程更直观。

**第三步：利用独立性求 $D(S_n)$**

由于 $X_1, X_2, \ldots, X_n$ **相互独立**（题目条件）：

$$D(S_n) = D(X_1 + X_2 + \cdots + X_n) = D(X_1) + D(X_2) + \cdots + D(X_n)$$

（独立变量之和的方差等于各方差之和）

$$= \underbrace{pq + pq + \cdots + pq}_{n \text{ 个}} = npq = np(1-p)$$

$$\boxed{D(S_n) = np(1-p)}$$

**第四步：频率的期望、方差与切比雪夫不等式**

设频率 $\hat{p} = S_n / n$，利用方差性质（$D(aX) = a^2 D(X)$）：

$$E\left[\frac{S_n}{n}\right] = \frac{1}{n} E[S_n] = \frac{np}{n} = p$$

$$D\left(\frac{S_n}{n}\right) = \frac{1}{n^2} D(S_n) = \frac{np(1-p)}{n^2} = \frac{p(1-p)}{n}$$

频率 $S_n/n$ 是 $p$ 的无偏估计，且其方差随 $n$ 增大而趋于零，说明频率越来越集中在 $p$ 附近。

**应用切比雪夫不等式**：令 $Z = S_n/n$，$E[Z]=p$，$D(Z)=p(1-p)/n$：

$$P\left(\left|\frac{S_n}{n} - p\right| \geq \varepsilon\right) \leq \frac{D(S_n/n)}{\varepsilon^2} = \frac{p(1-p)}{n\varepsilon^2}$$

因为 $p(1-p) \leq 1/4$（当 $p=1/2$ 时取等，这是 $p(1-p)$ 在 $[0,1]$ 上的最大值），所以：

$$P\left(\left|\frac{S_n}{n} - p\right| \geq \varepsilon\right) \leq \frac{p(1-p)}{n\varepsilon^2} \leq \frac{1}{4n\varepsilon^2}$$

当 $n \to +\infty$ 时，右端趋于 $0$，因此：

$$\lim_{n \to +\infty} P\left(\left|\frac{S_n}{n} - p\right| \geq \varepsilon\right) = 0$$

这就是**（弱）大数定律**：频率依概率收敛到概率 $p$。

:::

:::theorem{label=关键公式}

**两点分布**：
$$E[X_i] = p, \quad D(X_i) = p(1-p)$$

**期望线性性**（无需独立）：
$$E[S_n] = \sum_{i=1}^n E[X_i] = np$$

**独立变量方差可加**：
$$D(S_n) = \sum_{i=1}^n D(X_i) = np(1-p)$$

**频率的方差**：
$$D\left(\frac{S_n}{n}\right) = \frac{p(1-p)}{n} \xrightarrow{n\to\infty} 0$$

**切比雪夫不等式（大数定律预告）**：
$$P\left(\left|\frac{S_n}{n} - p\right| \geq \varepsilon\right) \leq \frac{p(1-p)}{n\varepsilon^2} \leq \frac{1}{4n\varepsilon^2} \to 0$$

:::

:::pitfall{label=易错点1}

**方差加法需要独立性**：$D(S_n)=\sum D(X_i)$ 依赖于 $X_i$ 相互独立，而期望线性性 $E[S_n]=\sum E[X_i]$ 不需要独立性。两者条件不同，不能混淆。

:::

:::pitfall{label=易错点2}

**$D(S_n/n)$ 中系数的处理**：$D(S_n/n) = D(S_n)/n^2$（因为 $D(aX)=a^2D(X)$，$a=1/n$），不是 $D(S_n)/n$。许多学生漏掉了分母的平方，写成 $\frac{np(1-p)}{n}=p(1-p)$（少了一个 $n$）。

:::

:::pitfall{label=易错点3}

**切比雪夫不等式应用的对象**：不等式应用于频率 $S_n/n$，其均值为 $p$，方差为 $p(1-p)/n$。不能把 $S_n$ 的均值 $np$ 和方差 $np(1-p)$ 直接代入，因为关注的偏差是 $S_n/n - p$，而非 $S_n - np$。

:::

:::pitfall{label=易错点4}

**$p(1-p)\leq 1/4$ 的使用**：这是放大上界的一步，用 $1/4$ 替换 $p(1-p)$ 是为了得到与 $p$ 无关的统一上界，但原始上界 $p(1-p)/(n\varepsilon^2)$ 更精确，不应盲目替换。

:::

:::pitfall{label=易错点5}

**大数定律的适用范围**：此处推导的是**弱大数定律**（依概率收敛），而**强大数定律**（几乎必然收敛）需要更强的工具，切比雪夫不等式仅能证明弱收敛。

:::
