:::example{label=题目描述}

设某总体的分布未知，但已知其均值 $\mu = 2$，方差 $\sigma^2 = 0.04$（标准差 $\sigma = 0.2$）。从该总体中独立抽取 $n = 100$ 个样本 $X_1, X_2, \ldots, X_{100}$，设样本均值为 $\bar{X} = \dfrac{1}{100}\sum_{i=1}^{100} X_i$。

用中心极限定理近似计算 $\bar{X}$ 落在区间 $[1.96,\; 2.04]$ 内的概率，即求 $P(1.96 \leq \bar{X} \leq 2.04)$。

:::

:::insight{label=分析}

$X_1, \ldots, X_{100}$ 独立同分布，均值 $\mu=2$，方差 $\sigma^2=0.04$，满足莱维-林德伯格定理条件。

样本均值 $\bar{X}$ 的期望和方差为：
$$E[\bar{X}] = \mu = 2, \qquad D(\bar{X}) = \frac{\sigma^2}{n} = \frac{0.04}{100} = 0.0004$$

标准化后，$\bar{X}$ 近似服从 $N(0,1)$（标准化后），或等价地：

$$\bar{X} \overset{\text{近似}}{\sim} N\!\left(2,\; \frac{0.04}{100}\right) = N(2,\; 0.0004)$$

**步骤：**
1. 计算 $\bar{X}$ 的均值和标准差；
2. 标准化区间 $[1.96, 2.04]$；
3. 查 $\Phi$ 表得近似概率。

:::

:::derivation{label=解答}

**第一步：计算 $\bar{X}$ 的均值和标准差**

$$E[\bar{X}] = \mu = 2$$

$$D(\bar{X}) = \frac{\sigma^2}{n} = \frac{0.04}{100} = 0.0004$$

$$\text{标准差：} \sqrt{D(\bar{X})} = \frac{\sigma}{\sqrt{n}} = \frac{0.2}{\sqrt{100}} = \frac{0.2}{10} = 0.02$$

**第二步：建立标准化变量**

由中心极限定理（$n=100$ 足够大）：

$$Z = \frac{\bar{X} - \mu}{\sigma/\sqrt{n}} = \frac{\bar{X} - 2}{0.02} \overset{\text{近似}}{\sim} N(0,1)$$

**第三步：标准化区间端点**

$$\frac{1.96 - 2}{0.02} = \frac{-0.04}{0.02} = -2$$

$$\frac{2.04 - 2}{0.02} = \frac{0.04}{0.02} = 2$$

**第四步：计算概率**

$$P(1.96 \leq \bar{X} \leq 2.04) \approx P(-2 \leq Z \leq 2)$$

$$= \Phi(2) - \Phi(-2) = \Phi(2) - [1 - \Phi(2)] = 2\Phi(2) - 1$$

查表：$\Phi(2) = 0.9772$，故

$$P(1.96 \leq \bar{X} \leq 2.04) \approx 2 \times 0.9772 - 1 = \boxed{0.9544}$$

**结论：** 样本均值 $\bar{X}$ 落在 $[1.96, 2.04]$ 内的概率约为 $95.44\%$。

**直观解释：** 区间 $[1.96, 2.04]$ 以 $\mu=2$ 为中心，宽度为 $0.08 = 2 \times 0.04$。而 $\bar{X}$ 的标准差为 $0.02$，故区间宽度恰好为标准差的 $\pm 2$ 倍，对应 $2\sigma$ 概率 $\approx 95.44\%$。

**推广：** 若问 $P(\bar{X}$ 落在 $[1.951, 2.049])$ 呢？

$$\frac{1.951-2}{0.02}=-2.45, \quad \frac{2.049-2}{0.02}=2.45$$

$$P \approx 2\Phi(2.45)-1 = 2\times0.9929-1=0.9858$$

:::

:::theorem{label=关键公式}

**样本均值的分布（中心极限定理）：**

$$\bar{X} = \frac{1}{n}\sum_{i=1}^n X_i \overset{\text{近似}}{\sim} N\!\left(\mu,\; \frac{\sigma^2}{n}\right) \quad (n \text{ 足够大})$$

**样本均值的标准化：**

$$Z = \frac{\bar{X} - \mu}{\sigma/\sqrt{n}} \overset{\text{近似}}{\sim} N(0,1)$$

**对称区间概率：**

$$P\!\left(\mu - k \cdot \frac{\sigma}{\sqrt{n}} \leq \bar{X} \leq \mu + k \cdot \frac{\sigma}{\sqrt{n}}\right) \approx 2\Phi(k) - 1$$

本题：$k = 0.04 / 0.02 = 2$，故概率 $\approx 2\Phi(2)-1=0.9544$。

**常用临界值对应的概率：**

| $k$ | $P(\bar{X}$ 在 $\mu \pm k\sigma/\sqrt{n}$ 内$)$ |
|---|---|
| $1.645$ | $90\%$ |
| $1.960$ | $95\%$ |
| $2.576$ | $99\%$ |

:::

:::pitfall{label=易错点1}

**分母用 $\sigma$ 代替 $\sigma/\sqrt{n}$**：对样本均值 $\bar{X}$ 标准化时，分母是 $\sigma/\sqrt{n}=0.2/10=0.02$，而非 $\sigma=0.2$。若用0.2，则区间端点标准化为 $(-0.2, 0.2)$，结果将严重偏低。

:::

:::pitfall{label=易错点2}

**混淆 $D(\bar{X})$ 与 $\sigma^2$**：$D(\bar{X}) = \sigma^2/n = 0.04/100 = 0.0004$，标准差是 $0.02$，不是 $0.04$（$\sigma$）或 $0.2$（$\sqrt{\sigma^2/n}$ 的错误写法）。

:::

:::pitfall{label=易错点3}

**区间宽度不对称时忘记分开计算**：若题目改为 $P(1.97 \leq \bar{X} \leq 2.05)$，标准化端点不对称（$-1.5$ 和 $2.5$），不能用 $2\Phi(k)-1$，需 $\Phi(2.5)-\Phi(-1.5)$ 分开查表。

:::

:::pitfall{label=易错点4}

**忘记"分布未知"不影响CLT的应用**：本题特意说明"分布未知"，是为了强调CLT的威力——只要知道 $\mu$ 和 $\sigma^2$，$n$ 足够大时就可以应用CLT，无需知道原始分布的具体形式。

:::

:::pitfall{label=易错点5}

**将近似结果误当精确值**：本题结果0.9544是近似值，精确值依赖原始分布（未知）。近似的质量由Berry-Esseen界保证，$n=100$ 时一般误差很小。

:::
