:::example{label=题目描述}

某工厂生产一批产品，每件产品的重量为随机变量 $X$，已知：
- 均值 $E[X] = 100$（克）
- 方差 $D(X) = 4$（克²）
- 产品重量的分布类型未知

利用切比雪夫不等式：
1. 估计 $P(|X - 100| \geq 3)$ 的上界
2. 估计 $P(97 \leq X \leq 103)$ 的下界
3. 若已知 $X \sim N(100, 4)$（正态分布），计算上述两个概率的精确值，并与切比雪夫上界作比较

:::

:::insight{label=分析}

切比雪夫不等式是"在未知分布情况下，利用均值和方差估计概率"的工具。

公式：$P(|X - \mu| \geq \varepsilon) \leq \dfrac{\sigma^2}{\varepsilon^2}$

其等价形式为：$P(|X - \mu| < \varepsilon) \geq 1 - \dfrac{\sigma^2}{\varepsilon^2}$

注意 $97 \leq X \leq 103$ 等价于 $|X - 100| \leq 3$，即偏差小于3。

:::

:::derivation{label=解答}

已知：$\mu = E[X] = 100$，$\sigma^2 = D(X) = 4$，$\sigma = \sqrt{4} = 2$。

**第一问：估计 $P(|X - 100| \geq 3)$ 的上界**

令 $\varepsilon = 3$，直接代入切比雪夫不等式：

$$P(|X - 100| \geq 3) \leq \frac{D(X)}{\varepsilon^2} = \frac{4}{3^2} = \frac{4}{9} \approx 0.444$$

即产品重量偏离100克超过3克的概率**不超过 $44.4\%$**。

**第二问：估计 $P(97 \leq X \leq 103)$ 的下界**

注意：
$$\{97 \leq X \leq 103\} = \{|X - 100| \leq 3\}$$

利用切比雪夫不等式的等价形式：

$$P(|X - 100| < 3) \geq 1 - \frac{D(X)}{\varepsilon^2} = 1 - \frac{4}{9} = \frac{5}{9} \approx 0.556$$

**注意等号问题：** 切比雪夫不等式给出的是 $P(|X - \mu| < \varepsilon)$ 的下界（严格小于号），而题目要求 $P(97 \leq X \leq 103) = P(|X - 100| \leq 3)$。

若 $X$ 是连续型随机变量，则 $P(|X - 100| = 3) = 0$，故：
$$P(97 \leq X \leq 103) = P(|X - 100| \leq 3) = P(|X - 100| < 3) \geq \frac{5}{9}$$

即产品重量在97克到103克之间的概率**至少为 $55.6\%$**。

**第三问：正态分布精确值与比较**

设 $X \sim N(100, 4)$，即 $X \sim N(\mu = 100, \sigma^2 = 4)$，$\sigma = 2$。

将 $X$ 标准化：$Z = \dfrac{X - 100}{2} \sim N(0, 1)$。

**精确计算 $P(|X - 100| \geq 3)$：**

$$P(|X - 100| \geq 3) = P\!\left(|Z| \geq \frac{3}{2}\right) = 2\left[1 - \Phi(1.5)\right]$$

查标准正态分布表：$\Phi(1.5) = 0.9332$

$$P(|X - 100| \geq 3) = 2(1 - 0.9332) = 2 \times 0.0668 = 0.1336$$

**精确计算 $P(97 \leq X \leq 103)$：**

$$P(97 \leq X \leq 103) = P\!\left(-\frac{3}{2} \leq Z \leq \frac{3}{2}\right) = 2\Phi(1.5) - 1 = 2 \times 0.9332 - 1 = 0.8664$$

**对比汇总：**

| 概率 | 切比雪夫结果 | 正态分布精确值 |
|---|---|---|
| $P(\|X-100\| \geq 3)$ | $\leq \dfrac{4}{9} \approx 44.4\%$ | $\approx 13.4\%$ |
| $P(97 \leq X \leq 103)$ | $\geq \dfrac{5}{9} \approx 55.6\%$ | $\approx 86.6\%$ |

**观察：** 切比雪夫给出的界相当宽松——对于正态分布，实际的尾部概率（13.4%）比切比雪夫上界（44.4%）小得多，实际的区间内概率（86.6%）比切比雪夫下界（55.6%）大得多。这说明切比雪夫不等式适用于"分布未知"的保守估计，当分布已知时应使用精确计算。

:::

:::theorem{label=关键公式}

**切比雪夫不等式：**
$$P(|X - \mu| \geq \varepsilon) \leq \frac{\sigma^2}{\varepsilon^2}$$

**等价形式（区间内概率下界）：**
$$P(\mu - \varepsilon < X < \mu + \varepsilon) \geq 1 - \frac{\sigma^2}{\varepsilon^2}$$

**正态分布标准化：**
$$P(a \leq X \leq b) = \Phi\!\left(\frac{b-\mu}{\sigma}\right) - \Phi\!\left(\frac{a-\mu}{\sigma}\right)$$

:::

:::pitfall{label=易错点1}

**$\varepsilon$ 代入错误：** 本题 $\varepsilon = 3$（偏差的阈值），$\sigma^2 = 4$（方差），切比雪夫上界为 $4/9$，不是 $\sqrt{4}/3 = 2/3$。注意分母是 $\varepsilon^2$，不是 $\varepsilon$。

:::

:::pitfall{label=易错点2}

**误以为切比雪夫给出精确值：** 切比雪夫给出的是**上界/下界**，不是精确值。上界/下界可以非常宽松。

:::

:::pitfall{label=易错点3}

**混淆方差和标准差：** $D(X) = 4$（方差），$\sigma = 2$（标准差）。切比雪夫不等式用方差 $\sigma^2$，正态分布标准化用标准差 $\sigma$。

:::

:::pitfall{label=易错点4}

**等号方向搞反：** $P(|X - \mu| \geq \varepsilon) \leq$（不超过，是上界），$P(|X - \mu| < \varepsilon) \geq$（至少，是下界）。

:::

:::pitfall{label=易错点5}

**连续型与离散型的细节差异：** 对连续型随机变量，$P(|X - \mu| \leq \varepsilon) = P(|X - \mu| < \varepsilon)$（端点概率为零），计算下界时可以忽略端点。对离散型，需注意不等号是否包含端点。

:::
