:::example{label=题目描述}

设二维连续型随机变量 (X, Y) 的联合密度函数为：

$$f(x, y) = e^{-y}, \quad 0 < x < y < +\infty$$

（其他区域 f(x, y) = 0）

求：
1. Y 的边缘密度函数 fᵧ(y)；
2. 在 Y = y（y > 0）条件下，X 的条件密度函数 f_{X|Y}(x|y)；
3. 说明该条件密度对应什么分布。

:::

:::insight{label=分析}

本题考查连续型条件密度函数的完整计算流程。关键步骤包括：第一，明确联合密度的支撑集为 0 < x < y（即对固定的 y，x 的范围是 (0, y)，而不是 (-∞, +∞)）；第二，计算 Y 的边缘密度时，对 x 积分，积分上下限由支撑集确定（0 到 y）；第三，用联合密度除以边缘密度得到条件密度，并写出条件密度的有效范围；第四，识别条件密度的分布类型。支撑集非矩形（0<x<y 中 x 的范围依赖 y）是本题的难点，必须准确处理积分范围。

:::

:::derivation{label=解答}

**第一步：确定 Y 的边缘密度函数**

对固定的 y > 0，x 的有效范围为 0 < x < y。对 x 积分：

$$f_Y(y) = \int_{-\infty}^{+\infty} f(x, y)\, dx = \int_0^y e^{-y}\, dx$$

由于 e^{-y} 不含 x，积分结果为：

$$f_Y(y) = e^{-y} \cdot [x]_0^y = e^{-y} \cdot y = y e^{-y}, \quad y > 0$$

这是参数为 2 的 Gamma 分布（即 Γ(2,1)）的密度函数，也可以验证：∫₀⁺∞ ye^{-y}dy = Γ(2) = 1! = 1 ✓

**第二步：计算条件密度函数**

由条件密度公式：

$$f_{X|Y}(x \mid y) = \frac{f(x, y)}{f_Y(y)} = \frac{e^{-y}}{y e^{-y}} = \frac{1}{y}, \quad 0 < x < y$$

（当 x ≤ 0 或 x ≥ y 时，f_{X|Y}(x|y) = 0）

**第三步：识别分布类型**

条件密度函数 f_{X|Y}(x|y) = 1/y（0 < x < y）是定义在区间 (0, y) 上的均匀分布密度函数（因为该区间长度为 y，密度值恒为 1/y）。

**结论：**

$$\boxed{f_{X|Y}(x \mid y) = \frac{1}{y}, \quad 0 < x < y}$$

即在 Y = y 的条件下，X 服从区间 (0, y) 上的均匀分布：X | Y=y ~ U(0, y)。

:::

:::theorem{label=关键公式}

- **连续型条件密度公式：**
$$f_{X|Y}(x \mid y) = \frac{f(x, y)}{f_Y(y)}, \quad f_Y(y) > 0$$

- **边缘密度计算：**
$$f_Y(y) = \int_{-\infty}^{+\infty} f(x, y)\, dx = \int_0^y e^{-y}\, dx = y e^{-y}$$

- **均匀分布密度识别：**
$$U(a, b): f(x) = \frac{1}{b-a}, \quad a < x < b$$

:::

:::note{label=验证}

**验证归一性：**
$$\int_0^y \frac{1}{y}\, dx = \frac{1}{y} \cdot y = 1 \quad \checkmark$$

:::

:::pitfall{label=易错点1}

**边缘密度积分范围错误：** 计算 fᵧ(y) 时，x 的积分范围是 (0, y) 而非 (-∞, +∞) 或 (0, +∞)，支撑集约束必须严格遵守。

:::

:::pitfall{label=易错点2}

**忽略条件密度的有效范围：** 只写 f_{X|Y}(x|y) = 1/y，忘记注明 0 < x < y，导致答案不完整。

:::

:::pitfall{label=易错点3}

**e^{-y} 积分失误：** 将 ∫₀ʸ e^{-y}dx 误当成关于 y 的积分，忘记此处 y 是参数，积分变量是 x，结果直接是 y·e^{-y}。

:::

:::pitfall{label=易错点4}

**未识别分布类型：** 得到 1/y 后未进一步说明这是均匀分布 U(0,y)，答案不够完整。

:::
