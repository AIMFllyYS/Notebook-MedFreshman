:::example{label=题目描述}

设二维连续型随机变量 (X, Y) 的联合密度函数为：

$$
f(x, y) = x + y, \quad 0 < x < 1,\ 0 < y < 1
$$

（其他区域 f(x, y) = 0）

判断 X 与 Y 是否相互独立。

:::

:::insight{label=分析}

判断连续型随机变量独立性的方法是：计算 X 和 Y 各自的边缘密度函数，然后检验 f(x,y) 是否等于 fₓ(x)·fᵧ(y)。本题支撑集是单位正方形（矩形区域），满足矩形支撑集的必要条件。但仍需计算边缘密度后进行验证，不能仅凭支撑集是矩形就直接判断独立。关键点在于：f(x,y) = x+y 是否可以写成 g(x)·h(y) 的形式——显然 x+y 不能分离变量（是加法而非乘法），因此预期结论是不独立。但必须通过计算边缘密度并验证不等式来严格证明。

:::

:::derivation{label=解答}

**第一步：计算 X 的边缘密度函数**

$$
f_X(x) = \int_{-\infty}^{+\infty} f(x, y)\, dy = \int_0^1 (x + y)\, dy
$$

$$
= \left[xy + \frac{y^2}{2}\right]_0^1 = x \cdot 1 + \frac{1}{2} - 0 = x + \frac{1}{2}, \quad 0 < x < 1
$$

**第二步：计算 Y 的边缘密度函数**

$$
f_Y(y) = \int_{-\infty}^{+\infty} f(x, y)\, dx = \int_0^1 (x + y)\, dx
$$

$$
= \left[\frac{x^2}{2} + xy\right]_0^1 = \frac{1}{2} + y - 0 = y + \frac{1}{2}, \quad 0 < y < 1
$$

**第三步：计算边缘密度的乘积**

$$
f_X(x) \cdot f_Y(y) = \left(x + \frac{1}{2}\right)\left(y + \frac{1}{2}\right)
$$

展开得：

$$
= xy + \frac{x}{2} + \frac{y}{2} + \frac{1}{4}
$$

**第四步：与联合密度比较**

$$
f(x, y) = x + y
$$

$$
f_X(x) \cdot f_Y(y) = xy + \frac{x}{2} + \frac{y}{2} + \frac{1}{4}
$$

显然 $x + y \neq xy + \dfrac{x}{2} + \dfrac{y}{2} + \dfrac{1}{4}$（例如取 x=0.5, y=0.5：左边=1，右边=0.25+0.25+0.25+0.25=1，偶然相等不代表恒等；再取 x=0, y=0：左边=0，右边=1/4≠0）。

**结论：** 由于 f(x,y) ≠ fₓ(x)·fᵧ(y)，所以 **X 与 Y 不相互独立**。

**直观解释：** 联合密度 f(x,y) = x+y 是加法形式，无法分解为纯粹关于 x 的函数与关于 y 的函数的乘积（变量分离失败），因此 X 与 Y 之间存在依赖关系，不相互独立。

:::

:::theorem{label=关键公式}

- **连续型独立判断条件：**
$$
f(x, y) = f_X(x) \cdot f_Y(y), \quad \text{a.e.}
$$

- **边缘密度计算：**
$$
f_X(x) = \int_0^1 (x+y)\, dy = x + \frac{1}{2}
$$

$$
f_Y(y) = \int_0^1 (x+y)\, dx = y + \frac{1}{2}
$$

- **变量分离充分条件（反用）：**
$$
f(x,y) = x+y \text{ 不能写成 } g(x) \cdot h(y) \Rightarrow \text{不独立}
$$

:::

:::pitfall{label=易错点1}

**支撑集矩形就判断独立：** 看到支撑集是正方形就认为独立，忽略还需验证密度函数可分离的条件，支撑集矩形只是独立的必要条件之一。

:::

:::pitfall{label=易错点2}

**加法形式误认为可分离：** 将 x+y 误认为是某种乘积形式而判断独立，实际上只有纯乘积 g(x)·h(y) 才能分离变量。

:::

:::pitfall{label=易错点3}

**边缘密度积分计算失误：** 积分 ∫₀¹(x+y)dy 时，将 x 当作变量（而非参数）处理，导致结果错误。

:::

:::pitfall{label=易错点4}

**仅取特殊值验证相等：** 偶然取到使 f(x,y)=fₓ(x)·fᵧ(y) 成立的点（如 x=y=0.5），便误判为独立，必须验证对所有 (x,y) 恒等，或找到一个不等的反例。

:::
