:::example{label=题目描述}

设二维离散型随机变量 (X, Y) 的联合分布律如下表所示，其中 X 的取值为 {0, 1}，Y 的取值为 {0, 1, 2}：

| X \ Y | 0 | 1 | 2 |
|-------|---|---|---|
| 0 | 2/9 | 1/9 | 0 |
| 1 | 1/9 | 2/9 | 3/9 |

求在 Y = 1 的条件下，X 的条件分布律，并验证归一性。

:::

:::insight{label=分析}

本题考查离散型条件分布律的计算。解题的核心步骤是：第一，从联合分布表中读取 Y=1 对应列的联合概率 p₀₁ 和 p₁₁；第二，计算边缘概率 P{Y=1} = p·₁，即将 Y=1 那一列所有联合概率相加；第三，用各联合概率除以边缘概率，得到条件分布律；第四，验证所有条件概率之和为1，确认结果正确。整个过程本质上是对联合分布表中的某一列进行归一化操作，体现了条件概率公式 P(A|B) = P(AB)/P(B) 的直接应用。

:::

:::derivation{label=解答}

**第一步：读取 Y=1 列的联合概率**

从分布表中读取 Y=1 对应列：
$$P\{X=0, Y=1\} = p_{01} = \frac{1}{9}$$
$$P\{X=1, Y=1\} = p_{11} = \frac{2}{9}$$

**第二步：计算 Y=1 的边缘概率**

$$P\{Y=1\} = p_{\cdot 1} = p_{01} + p_{11} = \frac{1}{9} + \frac{2}{9} = \frac{3}{9} = \frac{1}{3}$$

**第三步：计算条件分布律**

由条件分布律公式 $P\{X=x_i \mid Y=y_j\} = \dfrac{p_{ij}}{p_{\cdot j}}$，得：

$$P\{X=0 \mid Y=1\} = \frac{p_{01}}{p_{\cdot 1}} = \frac{1/9}{1/3} = \frac{1}{9} \times 3 = \frac{1}{3}$$

$$P\{X=1 \mid Y=1\} = \frac{p_{11}}{p_{\cdot 1}} = \frac{2/9}{1/3} = \frac{2}{9} \times 3 = \frac{2}{3}$$

**第四步：列出条件分布律表**

| X | 0 | 1 |
|---|---|---|
| P{X=·\|Y=1} | 1/3 | 2/3 |

**第五步：验证归一性**

$$P\{X=0 \mid Y=1\} + P\{X=1 \mid Y=1\} = \frac{1}{3} + \frac{2}{3} = 1 \quad \checkmark$$

**结论：** 在 Y=1 的条件下，X 的条件分布律为 P{X=0|Y=1} = 1/3，P{X=1|Y=1} = 2/3。

:::

:::theorem{label=关键公式}

- **条件分布律：**
$$P\{X=x_i \mid Y=y_j\} = \frac{P\{X=x_i, Y=y_j\}}{P\{Y=y_j\}} = \frac{p_{ij}}{p_{\cdot j}}$$

- **边缘概率：**
$$p_{\cdot j} = \sum_i p_{ij}$$

- **归一性验证：**
$$\sum_i P\{X=x_i \mid Y=y_j\} = 1$$

:::

:::pitfall{label=易错点1}

**分母选错：** 将分母写成总概率1而非边缘概率 p·ⱼ，导致条件分布律不归一。

:::

:::pitfall{label=易错点2}

**读错对应列：** 在复杂分布表中，混淆行列，将 P{X=xᵢ|Y=yⱼ} 和 P{Y=yⱼ|X=xᵢ} 所用的行/列搞反。

:::

:::pitfall{label=易错点3}

**漏加归一性验证：** 计算结束后忘记验证所有条件概率之和为1，导致错误结果无法被发现。

:::

:::pitfall{label=易错点4}

**分数运算失误：** 分数除法时，将 (1/9)/(1/3) 误算为 1/27，正确算法是乘以倒数：(1/9)×3=1/3。

:::
