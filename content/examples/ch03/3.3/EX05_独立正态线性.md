:::example{label=题目描述}

设 X ~ N(1, 4)，Y ~ N(2, 9)，且 X 与 Y 相互独立。

(1) 求 Z = 2X - Y + 3 的分布；

(2) 求 P{Z ≤ 3}；

(3) 求 P{|Z| ≤ 5}，即 P{-5 ≤ Z ≤ 5}。

（其中 Φ 为标准正态分布函数，所需数值：Φ(0) = 0.5，Φ(0.4) ≈ 0.6554，Φ(1.6) ≈ 0.9452）

:::

:::insight{label=分析}

本题综合考查独立正态分布线性组合的三个层次：确定分布、计算单侧概率、计算双侧概率。解题路径清晰：首先利用独立正态变量线性组合仍服从正态分布的性质（再生性），分别用期望的线性性和方差的独立可加性（注意系数要平方）计算 Z 的均值和方差，确定 Z 的分布；然后将概率计算标准化，转化为标准正态分布函数 Φ 的查表问题。特别需要注意：减去 Y 时，Y 的系数为 -1，(-1)²=1，方差贡献为正；加常数3只影响均值，不影响方差。

:::

:::derivation{label=解答}

**第一部分：确定 Z 的分布**

**均值（期望）计算：**

由期望的线性性（无论是否独立均成立）：

$$E(Z) = E(2X - Y + 3) = 2E(X) - E(Y) + 3 = 2 \times 1 - 2 + 3 = 3$$

**方差计算：**

由于 X 与 Y 相互独立，D(X±Y) = D(X) + D(Y)。注意常数不影响方差：

$$D(Z) = D(2X - Y + 3) = D(2X) + D(-Y) + D(3)$$

$$= 4 \cdot D(X) + 1 \cdot D(Y) + 0 = 4 \times 4 + 1 \times 9 = 16 + 9 = 25$$

**分布类型：**

由于 X, Y 独立且均服从正态分布，它们的线性组合仍服从正态分布，因此：

$$\boxed{Z = 2X - Y + 3 \sim N(3, 25)}$$

即 Z 服从均值为 3、方差为 25（标准差为 5）的正态分布。

---

**第二部分：计算 P{Z ≤ 3}**

将 Z 标准化，令 $Z^* = \dfrac{Z - 3}{5}$，则 Z* ~ N(0,1)：

$$P\{Z \leq 3\} = P\left\{\frac{Z-3}{5} \leq \frac{3-3}{5}\right\} = P\{Z^* \leq 0\} = \Phi(0) = \boxed{0.5}$$

**直观理解：** Z 的均值为3，P{Z≤3} = P{Z≤均值} = 0.5，正态分布关于均值对称，这一结果是必然的。

---

**第三部分：计算 P{|Z| ≤ 5}**

$$P\{|Z| \leq 5\} = P\{-5 \leq Z \leq 5\}$$

标准化：

$$= P\left\{\frac{-5-3}{5} \leq Z^* \leq \frac{5-3}{5}\right\} = P\left\{-\frac{8}{5} \leq Z^* \leq \frac{2}{5}\right\} = P\{-1.6 \leq Z^* \leq 0.4\}$$

利用标准正态分布函数：

$$= \Phi(0.4) - \Phi(-1.6) = \Phi(0.4) - [1 - \Phi(1.6)]$$

$$= 0.6554 - (1 - 0.9452) = 0.6554 - 0.0548 = \boxed{0.6006}$$

:::

:::theorem{label=关键公式}

- **独立正态线性组合：**
$$aX + bY + c \sim N(a\mu_1 + b\mu_2 + c,\ a^2\sigma_1^2 + b^2\sigma_2^2)$$

- **方差计算（独立时）：**
$$D(aX + bY + c) = a^2 D(X) + b^2 D(Y)$$

- **标准化：**
$$Z^* = \frac{Z - \mu_Z}{\sigma_Z} \sim N(0,1)$$

- **标准正态分布函数性质：**
$$\Phi(-x) = 1 - \Phi(x)$$

- **区间概率：**
$$P\{a \leq Z \leq b\} = \Phi\!\left(\frac{b-\mu}{\sigma}\right) - \Phi\!\left(\frac{a-\mu}{\sigma}\right)$$

:::

:::pitfall{label=易错点1}

**减法方差相减：** 计算 D(-Y) = (-1)²D(Y) = D(Y) = 9，而非 -D(Y)，方差永远非负。这是最高频的错误。

:::

:::pitfall{label=易错点2}

**系数未平方：** D(2X) = 2²D(X) = 4×4 = 16，误写成 2×4 = 8，忘记系数要平方。

:::

:::pitfall{label=易错点3}

**常数影响方差：** 误认为 D(2X-Y+3) = D(2X-Y) + 3，实际上常数不影响方差，D(Z+c) = D(Z)。

:::

:::pitfall{label=易错点4}

**Φ(-x) 公式应用错误：** Φ(-1.6) = 1-Φ(1.6)，而非 Φ(1.6)，符号处理时务必细心。

:::

:::pitfall{label=易错点5}

**忘记独立性条件：** 方差加法公式 D(X+Y)=D(X)+D(Y) 仅在 X 与 Y 独立时成立，非独立情形需加协方差项。

:::
