:::example{label=题目描述}

设 $X$ 与 $Y$ 相互独立，均服从参数为 $\lambda > 0$ 的指数分布，即

$$f_X(x) = \lambda e^{-\lambda x}, \quad x > 0; \qquad f_Y(y) = \lambda e^{-\lambda y}, \quad y > 0$$

求 $Z = X + Y$ 的概率密度函数，并指出其所属分布族。

:::

:::insight{label=分析}

本题是**独立随机变量之和**的分布问题，适合使用**卷积公式**。

**方法**：由于 $X$ 与 $Y$ 独立，$Z = X + Y$ 的密度为

$$f_Z(z) = \int_{-\infty}^{+\infty} f_X(x) \cdot f_Y(z - x) \, dx$$

**关键步骤**：

1. 确定 $f_X(x)$ 和 $f_Y(z-x)$ 同时非零的积分区间（支撑分析）。
2. 在有效区间上计算积分。
3. 认识结果所属的分布族（Gamma分布 / Erlang分布）。

**支撑分析**：
- $f_X(x) \neq 0$ 要求 $x > 0$。
- $f_Y(z-x) \neq 0$ 要求 $z - x > 0$，即 $x < z$。
- 因此，积分区间为 $0 < x < z$（当 $z > 0$ 时）。
- 当 $z \leq 0$ 时，两个条件无法同时满足，$f_Z(z) = 0$。

:::

:::derivation{label=解答}

**当 $z \leq 0$ 时**：$f_Z(z) = 0$（$X > 0$ 且 $Y > 0$，故 $Z = X+Y > 0$ 必然成立，$Z$ 不可能取非正值）。

**当 $z > 0$ 时**：代入卷积公式，积分区间为 $x \in (0, z)$：

$$f_Z(z) = \int_0^z f_X(x) \cdot f_Y(z - x) \, dx$$

代入指数分布的密度函数：

$$f_Z(z) = \int_0^z \lambda e^{-\lambda x} \cdot \lambda e^{-\lambda(z-x)} \, dx$$

展开指数部分：

$$= \int_0^z \lambda^2 e^{-\lambda x} \cdot e^{-\lambda z + \lambda x} \, dx$$

$$= \int_0^z \lambda^2 e^{-\lambda z + \lambda x - \lambda x} \, dx$$

$$= \int_0^z \lambda^2 e^{-\lambda z} \, dx$$

注意：$e^{-\lambda x} \cdot e^{-\lambda(z-x)} = e^{-\lambda x - \lambda z + \lambda x} = e^{-\lambda z}$，与 $x$ 无关！

因此 $e^{-\lambda z}$ 可提到积分号外：

$$f_Z(z) = \lambda^2 e^{-\lambda z} \int_0^z dx = \lambda^2 e^{-\lambda z} \cdot z$$

**最终结果**：

$$\boxed{f_Z(z) = \lambda^2 z \, e^{-\lambda z}, \quad z > 0}$$

**验证归一性**：

$$\int_0^{+\infty} \lambda^2 z e^{-\lambda z} \, dz$$

用分部积分，令 $u = \lambda^2 z$，$dv = e^{-\lambda z} dz$：

$$= \lambda^2 \left[-\frac{z e^{-\lambda z}}{\lambda}\right]_0^{+\infty} + \lambda^2 \int_0^{+\infty} \frac{e^{-\lambda z}}{\lambda} \, dz = 0 + \lambda \cdot \frac{1}{\lambda} = 1 \checkmark$$

**分布族识别**：

$f_Z(z) = \lambda^2 z e^{-\lambda z}$ 是 **Gamma 分布** $\Gamma(\alpha, \lambda)$ 在 $\alpha = 2$ 时的密度：

$$f(z; \alpha, \lambda) = \frac{\lambda^\alpha}{\Gamma(\alpha)} z^{\alpha - 1} e^{-\lambda z}$$

当 $\alpha = 2$，$\Gamma(2) = 1! = 1$：

$$f(z; 2, \lambda) = \frac{\lambda^2}{1} z^1 e^{-\lambda z} = \lambda^2 z e^{-\lambda z} \checkmark$$

因此 $Z \sim \Gamma(2, \lambda)$，也称为 **Erlang(2, λ) 分布**（爱尔朗分布），是 2 阶爱尔朗分布。

**期望与方差**（Gamma分布性质）：

$$E[Z] = \frac{\alpha}{\lambda} = \frac{2}{\lambda} = E[X] + E[Y] = \frac{1}{\lambda} + \frac{1}{\lambda} \checkmark$$

$$D[Z] = \frac{\alpha}{\lambda^2} = \frac{2}{\lambda^2} = D[X] + D[Y] = \frac{1}{\lambda^2} + \frac{1}{\lambda^2} \checkmark$$

:::

:::theorem{label=关键公式}

**卷积公式（独立连续随机变量之和）**：

$$f_Z(z) = \int_{-\infty}^{+\infty} f_X(x) \cdot f_Y(z - x) \, dx$$

**指数分布密度**（参数 $\lambda > 0$）：

$$f(x) = \lambda e^{-\lambda x}, \quad x > 0$$

**Gamma分布密度**（形状参数 $\alpha > 0$，尺度参数 $\lambda > 0$）：

$$f(z; \alpha, \lambda) = \frac{\lambda^\alpha}{\Gamma(\alpha)} z^{\alpha-1} e^{-\lambda z}, \quad z > 0$$

**指数分布的再生性（推广）**：

若 $X_1, \ldots, X_n$ 独立，$X_i \sim \text{Exp}(\lambda)$，则

$$Z = X_1 + X_2 + \cdots + X_n \sim \Gamma(n, \lambda)$$

**指数积分**（计算中用到）：

$$\int_0^z e^{-\lambda z} \, dz = z e^{-\lambda z}\bigg|_{\text{const w.r.t. }x} = z \cdot e^{-\lambda z}$$

:::

:::pitfall{label=易错点1}

**积分下限写成 $-\infty$**：不分析支撑就直接写 $\int_{-\infty}^{+\infty}$，对 $x < 0$ 或 $x > z$ 的区域，$f_X$ 或 $f_Y$ 为零，虽然结果不变但过程不严谨。应明确写出 $\int_0^z$，说明积分区间的理由。

:::

:::pitfall{label=易错点2}

**指数合并计算错误**：$e^{-\lambda x} \cdot e^{-\lambda(z-x)} = e^{-\lambda z}$ 而非 $e^{-2\lambda z}$。注意 $-\lambda x + (-\lambda)(z-x) = -\lambda x - \lambda z + \lambda x = -\lambda z$，$x$ 项恰好抵消，这是指数分布卷积计算的核心关键步骤，务必仔细展开。

:::

:::pitfall{label=易错点3}

**结论写 $Z \sim \text{Exp}(2\lambda)$ 而非 $\Gamma(2, \lambda)$**：$Z$ 的密度 $\lambda^2 z e^{-\lambda z}$ 不是指数分布（指数分布密度不含 $z$ 因子），$Z \sim \Gamma(2, \lambda)$，这两个分布不同。（注：$\min(X,Y) \sim \text{Exp}(2\lambda)$，是最小值，不是和。）

:::

:::pitfall{label=易错点4}

**没有验证 $z > 0$ 的条件**：最终结果应写成分段函数：$f_Z(z) = \lambda^2 z e^{-\lambda z}$（$z > 0$），$f_Z(z) = 0$（$z \leq 0$），漏掉后半段是不完整的。

:::

:::pitfall{label=易错点5}

**Gamma分布参数记法混淆**：不同教材中 Gamma 分布参数记法不同，有些写 $\Gamma(\alpha, \beta)$ 其中 $\beta = 1/\lambda$ 为尺度参数，有些写 $\Gamma(\alpha, \lambda)$ 其中 $\lambda$ 为速率参数。使用时需确认教材约定，计算期望时 $E[Z] = \alpha/\lambda$（速率参数记法）或 $E[Z] = \alpha\beta$（尺度参数记法），两者数值相同。

:::
