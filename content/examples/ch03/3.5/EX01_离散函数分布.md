:::example{label=题目描述}

设随机变量 $X$ 的分布律如下：

| $X$ | $-1$ | $0$ | $1$ | $2$ |
|:---:|:---:|:---:|:---:|:---:|
| $P$ | $\dfrac{1}{4}$ | $\dfrac{1}{3}$ | $\dfrac{1}{4}$ | $\dfrac{1}{6}$ |

求 $Y = X^2$ 的分布律，并计算 $E[Y]$ 和 $D[Y]$。

:::

:::insight{label=分析}

本题属于**离散型随机变量函数的分布**问题，使用枚举合并法：

1. 将 $X$ 的每个取值 $x_i$ 代入 $g(x) = x^2$，得到对应的 $y_i$ 值。
2. 注意 $g(x) = x^2$ 不是单射：$x = -1$ 和 $x = 1$ 均映射到 $y = 1$，需要合并。
3. 将映射到同一 $y$ 值的概率相加，得到 $Y$ 的分布律。
4. 利用分布律计算期望和方差。

首先验证 $X$ 的概率之和：$\dfrac{1}{4} + \dfrac{1}{3} + \dfrac{1}{4} + \dfrac{1}{6} = \dfrac{3}{12} + \dfrac{4}{12} + \dfrac{3}{12} + \dfrac{2}{12} = \dfrac{12}{12} = 1$ ✓

:::

:::derivation{label=解答}

**第一步：枚举映射**

列出每个 $x_i$ 的值、$y_i = x_i^2$ 的值以及对应概率：

| $x_i$ | $y_i = x_i^2$ | $P(X = x_i)$ |
|:---:|:---:|:---:|
| $-1$ | $1$ | $\dfrac{1}{4}$ |
| $0$ | $0$ | $\dfrac{1}{3}$ |
| $1$ | $1$ | $\dfrac{1}{4}$ |
| $2$ | $4$ | $\dfrac{1}{6}$ |

**第二步：识别 $Y$ 的不同取值**

$Y$ 的可能取值为 $\{0, 1, 4\}$（注意 $y = 1$ 出现两次，来自 $x = -1$ 和 $x = 1$）。

**第三步：合并相同取值的概率**

- $P(Y = 0) = P(X = 0) = \dfrac{1}{3}$

- $P(Y = 1) = P(X = -1) + P(X = 1) = \dfrac{1}{4} + \dfrac{1}{4} = \dfrac{1}{2}$

  （因为 $\{Y=1\} = \{X=-1\} \cup \{X=1\}$，两事件互斥，概率相加）

- $P(Y = 4) = P(X = 2) = \dfrac{1}{6}$

**第四步：写出 $Y$ 的分布律**

| $Y$ | $0$ | $1$ | $4$ |
|:---:|:---:|:---:|:---:|
| $P$ | $\dfrac{1}{3}$ | $\dfrac{1}{2}$ | $\dfrac{1}{6}$ |

**验证**：$\dfrac{1}{3} + \dfrac{1}{2} + \dfrac{1}{6} = \dfrac{2}{6} + \dfrac{3}{6} + \dfrac{1}{6} = \dfrac{6}{6} = 1$ ✓

**第五步：计算 $E[Y]$**

$$E[Y] = 0 \times \frac{1}{3} + 1 \times \frac{1}{2} + 4 \times \frac{1}{6} = 0 + \frac{1}{2} + \frac{2}{3} = \frac{3}{6} + \frac{4}{6} = \frac{7}{6}$$

**验证（用 $X$ 直接计算）**：$E[X^2] = (-1)^2 \cdot \dfrac{1}{4} + 0^2 \cdot \dfrac{1}{3} + 1^2 \cdot \dfrac{1}{4} + 2^2 \cdot \dfrac{1}{6} = \dfrac{1}{4} + 0 + \dfrac{1}{4} + \dfrac{4}{6} = \dfrac{1}{2} + \dfrac{2}{3} = \dfrac{7}{6}$ ✓

**第六步：计算 $D[Y]$**

$$E[Y^2] = 0^2 \times \frac{1}{3} + 1^2 \times \frac{1}{2} + 4^2 \times \frac{1}{6} = 0 + \frac{1}{2} + \frac{16}{6} = \frac{1}{2} + \frac{8}{3} = \frac{3}{6} + \frac{16}{6} = \frac{19}{6}$$

$$D[Y] = E[Y^2] - (E[Y])^2 = \frac{19}{6} - \left(\frac{7}{6}\right)^2 = \frac{19}{6} - \frac{49}{36} = \frac{114}{36} - \frac{49}{36} = \frac{65}{36}$$

:::

:::theorem{label=关键公式}

**离散型函数分布**：
$$P(Y = y) = \sum_{\{i : g(x_i) = y\}} P(X = x_i)$$

**期望（直接利用 $X$ 的分布）**：
$$E[g(X)] = \sum_i g(x_i) \cdot P(X = x_i)$$

**方差**：
$$D[Y] = E[Y^2] - (E[Y])^2$$

:::

:::pitfall{label=易错点1}

**漏掉合并步骤**：将 $Y=1$ 写成两个取值（各概率 $1/4$），而非一个取值（概率 $1/2$）。这是最常见的错误，会导致 $Y$ 的分布律有4行而非3行，且概率不归一（$1/4 + 1/3 + 1/4 + 1/6 = 1$，但写成4行会误导读者认为 $Y$ 取4个值）。

:::

:::pitfall{label=易错点2}

**忽视负数平方**：$x = -1$ 时 $g(-1) = (-1)^2 = 1$，不是 $-1$。对含负数的 $X$ 取值，代入 $g$ 时需谨慎计算。

:::

:::pitfall{label=易错点3}

**误用 $E[Y] = g(E[X])$**：计算 $E[Y] = E[X^2]$ 时，错误地用 $E[X]$ 先算，再平方：$E[X] = (-1)\cdot\frac{1}{4} + 0\cdot\frac{1}{3} + 1\cdot\frac{1}{4} + 2\cdot\frac{1}{6} = 0 + 0 + \frac{1}{3} = \frac{1}{3}$，$(E[X])^2 = \frac{1}{9} \neq \frac{7}{6}$。正确做法是用 $E[X^2] = \sum x_i^2 p_i$（或先求 $Y$ 的分布律再算期望）。

:::

:::pitfall{label=易错点4}

**漏掉某个 $x$ 的取值**：若 $X$ 有较多取值，容易在枚举时漏掉某个 $x$，导致 $Y$ 的某个概率偏小，可通过验证概率总和为1来发现。

:::

:::pitfall{label=易错点5}

**混淆 $Y$ 的分布律行数**：$Y$ 的不同取值个数等于 $g$ 在 $X$ 支撑上的**像集的基数**，不必等于 $X$ 的取值个数（非单射时会减少）。

:::
