:::example{label=题目描述}

设 $X_1, X_2, X_3$ 相互独立，且均服从区间 $(0, 1)$ 上的均匀分布 $U(0, 1)$。令

$$M = \max(X_1, X_2, X_3), \qquad L = \min(X_1, X_2, X_3)$$

求：
1. $M$ 的概率密度函数 $f_M(z)$；
2. $L$ 的概率密度函数 $f_L(z)$；
3. $E[M]$ 和 $E[L]$；
4. 验证 $E[M] + E[L]$ 的直觉合理性。

:::

:::insight{label=分析}

本题使用**最大最小值分布**的标准公式。

$X_i \sim U(0,1)$ 的 CDF 和 PDF 为：

$$F(z) = z \quad (0 \leq z \leq 1), \qquad f(z) = 1 \quad (0 < z < 1)$$

对 $n = 3$ 个独立同分布变量：

**最大值**：$F_M(z) = [F(z)]^3 = z^3$，对 $z$ 求导得 $f_M(z) = 3z^2$。

**最小值**：$F_L(z) = 1 - [1 - F(z)]^3 = 1 - (1-z)^3$，对 $z$ 求导得 $f_L(z) = 3(1-z)^2$。

:::

:::derivation{label=解答}

### 第一部分：$M = \max(X_1, X_2, X_3)$ 的密度函数

**Step 1：求 $F_M(z)$**

事件 $\{M \leq z\}$ 等价于"三个变量都不超过 $z$"：

$$F_M(z) = P(M \leq z) = P(X_1 \leq z, X_2 \leq z, X_3 \leq z)$$

由于 $X_1, X_2, X_3$ 相互独立：

$$F_M(z) = P(X_1 \leq z) \cdot P(X_2 \leq z) \cdot P(X_3 \leq z) = [F(z)]^3$$

代入 $F(z) = z$（对 $z \in [0,1]$）：

$$F_M(z) = z^3, \quad 0 \leq z \leq 1$$

边界：$F_M(z) = 0$ 对 $z < 0$；$F_M(z) = 1$ 对 $z \geq 1$。

**Step 2：求 $f_M(z)$**

$$f_M(z) = \frac{d}{dz} F_M(z) = \frac{d}{dz} z^3 = 3z^2$$

完整写法：

$$\boxed{f_M(z) = 3z^2, \quad 0 < z < 1; \qquad f_M(z) = 0 \text{ 其他}}$$

---

### 第二部分：$L = \min(X_1, X_2, X_3)$ 的密度函数

**Step 1：求 $F_L(z)$**

利用补事件。事件 $\{L > z\}$ 等价于"三个变量都超过 $z$"：

$$P(L > z) = P(X_1 > z, X_2 > z, X_3 > z) = [1 - F(z)]^3 = (1 - z)^3$$

因此：

$$F_L(z) = 1 - P(L > z) = 1 - (1 - z)^3, \quad 0 \leq z \leq 1$$

**Step 2：求 $f_L(z)$**

$$f_L(z) = \frac{d}{dz} \left[1 - (1-z)^3\right] = -3(1-z)^2 \cdot (-1) = 3(1-z)^2$$

完整写法：

$$\boxed{f_L(z) = 3(1-z)^2, \quad 0 < z < 1; \qquad f_L(z) = 0 \text{ 其他}}$$

---

### 第三部分：计算 $E[M]$ 和 $E[L]$

**期望 $E[M]$**：

$$E[M] = \int_0^1 z \cdot f_M(z) \, dz = \int_0^1 z \cdot 3z^2 \, dz = \int_0^1 3z^3 \, dz = 3 \cdot \frac{z^4}{4}\bigg|_0^1 = \frac{3}{4}$$

**期望 $E[L]$**：

$$E[L] = \int_0^1 z \cdot f_L(z) \, dz = \int_0^1 z \cdot 3(1-z)^2 \, dz$$

展开 $(1-z)^2 = 1 - 2z + z^2$：

$$= 3\int_0^1 z(1 - 2z + z^2) \, dz = 3\int_0^1 (z - 2z^2 + z^3) \, dz$$

$$= 3\left[\frac{z^2}{2} - \frac{2z^3}{3} + \frac{z^4}{4}\right]_0^1 = 3\left(\frac{1}{2} - \frac{2}{3} + \frac{1}{4}\right)$$

$$= 3 \cdot \frac{6 - 8 + 3}{12} = 3 \cdot \frac{1}{12} = \frac{1}{4}$$

---

### 第四部分：验证直觉合理性

$$E[M] + E[L] = \frac{3}{4} + \frac{1}{4} = 1$$

**直觉解释**：每个 $X_i \sim U(0,1)$，$E[X_i] = 1/2$。三个均匀分布的点将 $(0,1)$ 分成4段（2个端点固定，3个随机内点），由对称性，期望将区间均匀分割：

$$0, \frac{1}{4}, \frac{1}{2}, \frac{3}{4}, 1$$

其中3个随机点的期望排序值恰为 $1/4, 1/2, 3/4$，即 $E[L] = 1/4$，$E[\text{中位数}] = 1/2$，$E[M] = 3/4$。这与计算结果完全吻合 ✓

**一般规律**：$n$ 个 $U(0,1)$ 独立变量，第 $k$ 小的期望值为 $\dfrac{k}{n+1}$。本题 $n=3$，最小值期望 $= 1/4$，最大值期望 $= 3/4$。

:::

:::theorem{label=关键公式}

**i.i.d. 最大值分布（$n$ 个变量）**：

$$F_M(z) = [F(z)]^n, \qquad f_M(z) = n[F(z)]^{n-1} f(z)$$

**i.i.d. 最小值分布（$n$ 个变量）**：

$$F_L(z) = 1 - [1-F(z)]^n, \qquad f_L(z) = n[1-F(z)]^{n-1} f(z)$$

**均匀分布 $U(0,1)$**（代入 $n=3$）：

$$f_M(z) = 3z^2, \quad f_L(z) = 3(1-z)^2, \quad 0 < z < 1$$

**顺序统计量期望**（$U(0,1)$ 的 $n$ 个独立样本）：

$$E[X_{(k)}] = \frac{k}{n+1}$$

:::

:::note{label=验证}

**验证 $f_M$ 归一性**：$\int_0^1 3z^2 \, dz = [z^3]_0^1 = 1$ ✓

**验证 $f_L$ 归一性**：$\int_0^1 3(1-z)^2 \, dz = [-(1-z)^3]_0^1 = 0 - (-1) = 1$ ✓

**验证直觉**：$E[M] + E[L] = \frac{3}{4} + \frac{1}{4} = 1$，与顺序统计量理论一致 ✓

:::

:::pitfall{label=易错点1}

**最小值公式写错**：$F_L(z) = 1-(1-z)^3$ 容易误写为 $(1-z)^3$ 或 $1-z^3$。记忆口诀：最小值大于 $z$ 等于所有值都大于 $z$，生存函数为 $(1-z)^3$，取补得CDF。

:::

:::pitfall{label=易错点2}

**求导时链式法则遗漏**：$\dfrac{d}{dz}(1-z)^3 = 3(1-z)^2 \cdot (-1) = -3(1-z)^2$，取负号后得 $f_L(z) = +3(1-z)^2 > 0$。忘记乘 $(-1)$ 会得到负的"密度"，是显然错误。

:::

:::pitfall{label=易错点3}

**混淆 $f_M$ 和 $f_L$ 的表达式**：两者形式相似但底数不同，$f_M$ 含 $z^2$（向右集中），$f_L$ 含 $(1-z)^2$（向左集中）。可通过检验单调性和众数位置来区分。

:::

:::pitfall{label=易错点4}

**期望积分展开错误**：计算 $E[L] = \int_0^1 z \cdot 3(1-z)^2 \, dz$ 时，展开 $(1-z)^2$ 后逐项积分，容易算错各项系数，建议逐步展开并检验各项积分结果。

:::

:::pitfall{label=易错点5}

**忘记考虑分布支撑**：$f_M$ 和 $f_L$ 仅在 $z \in (0,1)$ 上非零，最终写密度函数时应标注支撑范围，超出范围取值为0，不能仅写代数表达式而不注明有效区间。

:::
