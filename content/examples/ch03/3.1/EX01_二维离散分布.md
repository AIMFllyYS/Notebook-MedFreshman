:::example{label=核心概念}
本例题涵盖二维离散型随机变量的三项基本操作：**构造联合分布表、计算边缘分布、验证独立性**，并可能进一步计算条件分布。这三项操作是处理离散型多维随机变量问题的完整流程，相互依赖、层层递进。

**典型题型设置**：已知 $X$ 与 $Y$ 各自的边缘分布，以及关于 $(X,Y)$ 联合分布的若干约束条件（如某些格子的概率值，或者 $X$ 与 $Y$ 独立的条件），要求：
1. 写出完整的联合分布表（含所有 $p_{ij}$）
2. 由联合分布表读出各变量的边缘分布
3. 验证 $X$ 与 $Y$ 是否独立（检验 $p_{ij} = p_{i\cdot} \cdot p_{\cdot j}$）
4. 计算某条件下的条件分布律

**另一类典型**：直接给出联合分布表（含参数），要求利用规范性条件（所有概率之和为1）确定参数，再进行后续计算。
:::

:::theorem{label=关键公式}
**规范性条件（确定未知参数的核心方程）：**

$$\sum_i \sum_j p_{ij} = 1$$

**边缘分布律（行和与列和）：**

$$p_{i\cdot} = P(X = x_i) = \sum_j p_{ij}$$

$$p_{\cdot j} = P(Y = y_j) = \sum_i p_{ij}$$

**独立性验证条件（必须对所有 $i,j$ 均成立）：**

$$p_{ij} = p_{i\cdot} \cdot p_{\cdot j}, \quad \forall\, i, j$$

**条件分布律（缩减样本空间）：**

$$P(X = x_i \mid Y = y_j) = \frac{p_{ij}}{p_{\cdot j}}$$

$$P(Y = y_j \mid X = x_i) = \frac{p_{ij}}{p_{i\cdot}}$$

**条件期望：**

$$E(X \mid Y = y_j) = \sum_i x_i \cdot P(X = x_i \mid Y = y_j) = \sum_i x_i \cdot \frac{p_{ij}}{p_{\cdot j}}$$
:::

:::derivation{label=解题过程}
**例**：设 $(X, Y)$ 的联合分布表如下（$a$ 为未知参数）：

|       | $Y=0$ | $Y=1$ | $Y=2$ |
|:-----:|:-----:|:-----:|:-----:|
| $X=0$ | $0.1$ | $0.2$ | $a$   |
| $X=1$ | $0.2$ | $0.1$ | $0.3$ |

**第一步：利用规范性确定参数**

$$0.1 + 0.2 + a + 0.2 + 0.1 + 0.3 = 1 \Rightarrow a = 0.1$$

**第二步：写出完整分布表并计算边缘分布**

|       | $Y=0$ | $Y=1$ | $Y=2$ | $p_{i\cdot}$ |
|:-----:|:-----:|:-----:|:-----:|:------------:|
| $X=0$ | $0.1$ | $0.2$ | $0.1$ | $0.4$        |
| $X=1$ | $0.2$ | $0.1$ | $0.3$ | $0.6$        |
| $p_{\cdot j}$ | $0.3$ | $0.3$ | $0.4$ | $1$    |

**第三步：验证独立性**

检验 $(i=0, j=0)$：$p_{00} = 0.1$，$p_{0\cdot} \cdot p_{\cdot 0} = 0.4 \times 0.3 = 0.12 \neq 0.1$。

不满足独立性条件，故 $X$ 与 $Y$ **不相互独立**。

**第四步：求条件分布（例：在 $Y=2$ 条件下 $X$ 的分布）**

$$P(X=0 \mid Y=2) = \frac{0.1}{0.4} = 0.25, \quad P(X=1 \mid Y=2) = \frac{0.3}{0.4} = 0.75$$
:::

:::note{label=直观理解}
联合分布表可以理解为一张"概率热力图"：每个格子的颜色深浅代表该取值对 $(x_i, y_j)$ 发生的概率大小。

**边缘化**就是把热力图的每一行（或列）的所有颜色强度加总，投影到边框上，得到 $X$（或 $Y$）单独的概率分布。

**独立性验证**的直觉：若 $X$ 与 $Y$ 独立，则热力图中每一行的颜色模式（相对比例）应该完全相同。如果某行比其他行更"偏左"或更"偏右"，说明 $X$ 的分布受 $Y$ 取值影响，即不独立。

**条件分布**是"把某列（或行）单独放大"——只看 $Y=y_j$ 这一列，重新归一化，就得到在 $Y=y_j$ 条件下 $X$ 的分布。
:::

:::pitfall{label=易错点1}
**规范性条件列方程时漏项**

应将所有 $p_{ij}$（包括已知的和未知的）全部相加等于1，漏掉任何一格会导致参数计算错误。
:::

:::pitfall{label=易错点2}
**边缘分布求和方向记错**

$p_{i\cdot}$ 是固定 $i$ 对所有 $j$ 求和（行和），$p_{\cdot j}$ 是固定 $j$ 对所有 $i$ 求和（列和）。行和给出 $X$ 的边缘分布，列和给出 $Y$ 的边缘分布，方向与直觉有时相反（行的汇聚得到行变量 $X$ 的分布）。
:::

:::pitfall{label=易错点3}
**独立性验证只检查一个格子就下结论**

只要存在一个 $(i,j)$ 使 $p_{ij} \neq p_{i\cdot} p_{\cdot j}$，就不独立；必须找到不满足的格子才能说不独立，而不是找到满足的格子就说独立。
:::

:::pitfall{label=易错点4}
**条件分布不归一化**

计算条件分布律后，各条件概率之和必须等于1，未验证的结果可能有计算错误。
:::

:::pitfall{label=易错点5}
**混淆联合概率与条件概率**

$p_{ij} = P(X=x_i, Y=y_j)$ 是联合概率，$P(X=x_i \mid Y=y_j) = p_{ij}/p_{\cdot j}$ 是条件概率，两者数值一般不同。
:::
