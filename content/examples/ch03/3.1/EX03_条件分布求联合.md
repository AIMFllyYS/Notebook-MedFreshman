:::example{label=核心概念}
本例题与前两道例题的方向相反——**已知条件分布和边缘分布，反向求联合分布**。这是概率论中一类重要的"逆问题"，利用条件密度与联合密度的关系：

$$f(x, y) = f_{X|Y}(x \mid y) \cdot f_Y(y)$$

即"联合 = 条件 × 边缘"（乘法公式）。

**典型题型设置**：
- **形式一**：已知 $Y$ 的边缘密度 $f_Y(y)$，以及在 $Y=y$ 条件下 $X$ 的条件密度 $f_{X|Y}(x \mid y)$，利用乘法公式求联合密度 $f(x,y)$，进而求 $X$ 的边缘密度 $f_X(x)$ 或条件期望 $E(X \mid Y=y)$。
- **形式二**：涉及二维正态分布，利用条件分布仍为正态的性质（条件均值为另一变量的线性函数），推导联合分布的参数或计算条件期望。
- **形式三**：由两个嵌套条件分布（分层模型），通过乘法公式求边缘分布，这在贝叶斯统计中极为重要。

**知识点地位**：本例题深刻体现了联合分布、边缘分布、条件分布三者的关系——它们是同一个概率结构的三种不同"切面"，任意两个可以推出第三个（在某些条件下）。乘法公式是三者之间最核心的桥梁。
:::

:::theorem{label=关键公式}
**乘法公式（联合密度由条件密度和边缘密度构造）：**

$$f(x, y) = f_{X|Y}(x \mid y) \cdot f_Y(y)$$

等价地：

$$f(x, y) = f_{Y|X}(y \mid x) \cdot f_X(x)$$

**由联合密度求 $X$ 的边缘密度（全概率公式的连续推广）：**

$$f_X(x) = \int_{-\infty}^{+\infty} f(x, y)\, dy = \int_{-\infty}^{+\infty} f_{X|Y}(x \mid y) \cdot f_Y(y)\, dy$$

**条件期望（连续型）：**

$$E(X \mid Y = y) = \int_{-\infty}^{+\infty} x \cdot f_{X|Y}(x \mid y)\, dx$$

**二维正态分布的条件期望（特殊公式）：**

若 $(X,Y) \sim N(\mu_1, \mu_2, \sigma_1^2, \sigma_2^2, \rho)$，则：

$$E(X \mid Y = y) = \mu_1 + \rho \frac{\sigma_1}{\sigma_2}(y - \mu_2)$$

$$D(X \mid Y = y) = \sigma_1^2(1 - \rho^2)$$

**贝叶斯公式（连续型，由联合求另一方向的条件密度）：**

$$f_{Y|X}(y \mid x) = \frac{f_{X|Y}(x \mid y) \cdot f_Y(y)}{f_X(x)}$$
:::

:::derivation{label=解题过程}
**例**：设 $Y \sim \text{Exp}(\lambda)$（参数为 $\lambda > 0$ 的指数分布），且在 $Y=y$ 条件下，$X \mid Y=y \sim U(0, y)$（即在 $[0,y]$ 上的均匀分布）。

**第一步：写出已知条件**

$$f_Y(y) = \lambda e^{-\lambda y}, \quad y > 0$$

$$f_{X|Y}(x \mid y) = \frac{1}{y}, \quad 0 < x < y$$

**第二步：利用乘法公式求联合密度**

$$f(x, y) = f_{X|Y}(x \mid y) \cdot f_Y(y) = \frac{1}{y} \cdot \lambda e^{-\lambda y} = \frac{\lambda e^{-\lambda y}}{y}$$

其中支撑区域为 $\{0 < x < y\}$，即 $0 < x < y < +\infty$。

**第三步：求 $X$ 的边缘密度**

固定 $x > 0$，$y$ 的范围为 $y > x$（由 $x < y$ 决定）：

$$f_X(x) = \int_x^{+\infty} \frac{\lambda e^{-\lambda y}}{y}\, dy$$

此积分无初等表达式，但可以验证 $\int_0^{+\infty} f_X(x)\,dx = 1$。

**第四步：求条件期望 $E(X \mid Y=y)$**

由于 $X \mid Y=y \sim U(0,y)$，其条件期望为均匀分布的均值：

$$E(X \mid Y = y) = \frac{0 + y}{2} = \frac{y}{2}$$

**第五步：验证全期望公式**

$$E(X) = E[E(X \mid Y)] = E\!\left[\frac{Y}{2}\right] = \frac{1}{2} E(Y) = \frac{1}{2\lambda}$$

这与直接用 $f_X(x)$ 计算 $E(X)$ 的结果一致，可作为验证。
:::

:::note{label=直观理解}
"由条件分布和边缘分布求联合分布"这一逆问题有鲜明的**概率建模直觉**：

想象一个**两阶段随机实验**：
1. 第一阶段：从边缘分布 $f_Y(y)$ 中抽取 $Y = y$（如先抽取某个"尺度参数"）
2. 第二阶段：已知 $Y=y$，再从条件分布 $f_{X|Y}(x \mid y)$ 中抽取 $X$（如在确定的范围内均匀选取一点）

这一两阶段过程的最终结果 $(X,Y)$ 的联合分布，正好就是乘法公式 $f(x,y) = f_{X|Y}(x|y) \cdot f_Y(y)$ 所描述的。

这在**贝叶斯统计**中极为自然：$f_Y(y)$ 是参数 $Y$ 的先验分布，$f_{X|Y}(x|y)$ 是给定参数 $Y=y$ 时数据 $X$ 的似然，而 $f(x,y)$ 是数据与参数的联合分布。

**条件期望的直觉**：$E(X \mid Y=y)$ 是"知道 $Y=y$ 之后，对 $X$ 的最优预测（均方意义下）"。全期望公式 $E(X) = E[E(X \mid Y)]$ 说明无条件期望可以通过先条件再期望得到，计算上往往比直接积分更简便。
:::

:::pitfall{label=易错点1}
**乘法公式方向搞反**

$f(x,y) = f_{X|Y}(x|y) \cdot f_Y(y)$，不是 $f_{X|Y}(x|y) \cdot f_X(x)$。乘的是$y$的边缘密度（条件所在的那个变量的边缘密度）。
:::

:::pitfall{label=易错点2}
**支撑区域描述不清**

求 $f_X(x)$ 的积分时，必须明确 $y$ 的积分下限随 $x$ 变化（如 $y > x$），而不是从 $-\infty$ 或 $0$ 积分到 $+\infty$。
:::

:::pitfall{label=易错点3}
**混淆条件密度和边缘密度**

$f_{X|Y}(x|y)$ 是固定 $y$ 后关于 $x$ 的函数，是 $x$ 的密度；$f_Y(y)$ 是 $y$ 的密度。两者乘积 $f(x,y)$ 是关于 $(x,y)$ 的二元函数。
:::

:::pitfall{label=易错点4}
**全期望公式应用错误**

$E(X) = E_Y[E(X|Y)]$ 中，第一个期望是对 $Y$ 求的（$E_Y$ 表示对 $Y$ 的边缘分布取期望），不是固定某个值计算。
:::

:::pitfall{label=易错点5}
**二维正态条件期望公式记错**

$E(X|Y=y) = \mu_1 + \rho\frac{\sigma_1}{\sigma_2}(y-\mu_2)$，注意系数是 $\sigma_1/\sigma_2$（不是方差比 $\sigma_1^2/\sigma_2^2$），且减去的是 $\mu_2$。
:::

:::note{label=知识关联}
- **KP05（条件分布）**：本例题是KP05知识的"逆应用"——KP05介绍由联合密度求条件密度（除法），本题则由条件密度和边缘密度求联合密度（乘法），两个方向合称"乘法公式"，是条件密度理论最核心的操作。

- **KP04（二维正态分布）**：二维正态的条件分布仍为正态，条件期望是 $y$ 的线性函数，是本例题"条件期望"计算的最重要具体案例。二维正态的条件均值公式是线性回归（最小二乘回归）的理论基础。

- **KP03（二维连续型密度）**：本例题求出联合密度后，求 $X$ 的边缘密度仍需用KP03的积分方法，且涉及非标准积分区域的处理。

- **KP06（独立性）**：若条件密度 $f_{X|Y}(x|y)$ 与 $y$ 无关（即对所有 $y$ 相同），说明 $X$ 与 $Y$ 独立，此时联合密度就分解为 $f_X(x) \cdot f_Y(y)$。通过本例题可以体会独立性在"分层建模"中的含义。

- **EX02（三角形均匀分布）**：EX02中三角形均匀分布的条件分布 $X|Y=y \sim U(0,y)$（或类似形式），实际上正是本例题所假设的条件分布。两道例题可以互相关联：EX02从联合分布出发求条件分布，EX03从条件分布出发重建联合分布，形成完整的闭环。

- **贝叶斯统计（数理统计部分）**：本例题的思路与贝叶斯推断完全一致：先验 $\times$ 似然 $=$ 联合，然后边缘化得到后验。掌握本例题是理解贝叶斯统计思想的重要桥梁。
:::
