:::example{label=题目描述}
设二维随机变量 $(X, Y)$ 在三角形区域 $D = \{(x, y) : 0 \leq y \leq x \leq 1\}$ 上服从均匀分布。

(1) 写出 $(X, Y)$ 的联合密度函数；

(2) 求 $P\{Y \leq 1/2\}$；

(3) 求 $P\{X + Y \leq 1\}$。
:::

:::insight{label=分析}
本题是二维均匀分布的典型应用。首先计算三角形区域 $D$ 的面积 $S(D)$，写出联合密度函数（在 $D$ 上为常数 $1/S(D)$，在 $D$ 外为 $0$）。

对于(2)和(3)，利用二维均匀分布的核心性质：$P\{(X,Y)\in A\} = S(A\cap D)/S(D)$，即只需计算满足附加条件的子区域面积与 $D$ 的面积之比。这将积分计算转化为简单的几何面积计算，是二维均匀分布最大的计算优势。
:::

:::derivation{label=解答}
**第(1)部分：联合密度函数**

区域 $D = \{(x,y): 0\leq y \leq x \leq 1\}$ 是以 $(0,0)$, $(1,0)$, $(1,1)$ 为顶点的直角三角形。

面积：$S(D) = \frac{1}{2} \times 1 \times 1 = \frac{1}{2}$

联合密度函数：
$$f(x, y) = \begin{cases} \dfrac{1}{S(D)} = 2, & (x, y) \in D \\ 0, & \text{其他} \end{cases}$$

**第(2)部分：求 $P\{Y \leq 1/2\}$**

需要计算区域 $A_1 = D \cap \{(x,y): y \leq 1/2\}$ 的面积。

区域 $A_1$ 是 $D$ 中满足 $y \leq 1/2$ 的部分。在 $D$ 中，$y$ 的范围为 $0 \leq y \leq x \leq 1$，所以 $y \leq 1/2$ 的限制将 $D$ 分成两部分：
- 当 $x \leq 1/2$ 时，$y$ 从 $0$ 到 $x$（完整保留）
- 当 $x > 1/2$ 时，$y$ 从 $0$ 到 $1/2$（被截断）

$$S(A_1) = \int_0^{1/2} x\,dx + \int_{1/2}^1 \frac{1}{2}\,dx = \left[\frac{x^2}{2}\right]_0^{1/2} + \frac{1}{2} \cdot \frac{1}{2} = \frac{1}{8} + \frac{1}{4} = \frac{3}{8}$$

$$P\{Y \leq 1/2\} = \frac{S(A_1)}{S(D)} = \frac{3/8}{1/2} = \frac{3}{4}$$

**第(3)部分：求 $P\{X+Y \leq 1\}$**

需要计算区域 $A_2 = D \cap \{(x,y): x+y \leq 1\}$ 的面积。

在 $D$ 中 $0 \leq y \leq x \leq 1$，而 $x+y \leq 1$ 即 $y \leq 1-x$。

当 $0 \leq x \leq 1$ 时，$y$ 的上界为 $\min(x, 1-x)$：
- 当 $x \leq 1/2$ 时，$x \leq 1-x$，上界为 $x$（$x+y\leq 1$ 不起限制）
- 当 $x > 1/2$ 时，$x > 1-x$，上界为 $1-x$

$$S(A_2) = \int_0^{1/2} x\,dx + \int_{1/2}^1 (1-x)\,dx = \frac{1}{8} + \left[x - \frac{x^2}{2}\right]_{1/2}^1 = \frac{1}{8} + \left(1-\frac{1}{2}\right) - \left(\frac{1}{2}-\frac{1}{8}\right) = \frac{1}{8} + \frac{1}{2} - \frac{3}{8} = \frac{1}{4}$$

$$P\{X+Y\leq 1\} = \frac{S(A_2)}{S(D)} = \frac{1/4}{1/2} = \frac{1}{2}$$
:::

:::theorem{label=关键公式}
$$f(x,y) = \frac{1}{S(D)} = 2,\quad (x,y)\in D$$

$$P\{(X,Y)\in A\} = \frac{S(A\cap D)}{S(D)}$$
:::

:::pitfall{label=易错点1}
**三角形面积计算错误**

三角形 $D$ 的面积计算：$S(D)=1/2$，不要错算为 $1$（以为是单位正方形面积）。
:::

:::pitfall{label=易错点2}
**分段积分上界判断错误**

第(2)部分分段积分时，注意 $y \leq 1/2$ 与 $y \leq x$ 的交互：只有当 $x > 1/2$ 时，$y=1/2$ 才是真正的上界。
:::

:::pitfall{label=易错点3}
**约束条件遗漏**

第(3)部分中，$x+y \leq 1$ 在 $x \leq 1/2$ 时不对 $y$ 产生额外约束（因为 $y \leq x \leq 1/2 \leq 1-x$）。
:::

:::pitfall{label=易错点4}
**忘记除以总面积**

最终概率用 $S(A)/S(D)$ 而非 $S(A)$，忘记除以 $S(D)=1/2$（等价于乘以 $2$）。
:::
