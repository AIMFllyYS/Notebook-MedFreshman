:::example{label=核心概念}
本例题是二维连续型随机变量边缘密度计算的经典案例。其特殊性在于：联合密度函数定义在一个**三角形区域**（而非矩形区域）上，使得边缘密度积分的上下限必须随另一个变量的值动态变化，不能使用固定常数作为积分边界。

**题目设置**：设 $(X, Y)$ 在三角形区域 $D = \{(x,y) : 0 \leq x \leq 1, 0 \leq y \leq x\}$（或类似的三角形区域）上服从均匀分布，即：

$$
f(x, y) = \begin{cases} c, & (x,y) \in D \\ 0, & \text{其他} \end{cases}
$$

首先由规范性条件 $\iint_D c\, dx\, dy = 1$ 确定常数 $c$（$c$ 等于三角形面积的倒数）；然后计算 $X$ 和 $Y$ 各自的边缘密度；最后判断 $X$ 与 $Y$ 是否独立。

**核心挑战**：对于非矩形支撑区域，求边缘密度时必须正确确定积分限，这是连续型边缘化计算中最具技巧性的步骤。
:::

:::theorem{label=关键公式}
**三角形区域的规范性常数（以 $D = \{0 \leq y \leq x \leq 1\}$ 为例）：**

$$
\iint_D c\, dx\, dy = c \cdot \frac{1}{2} \cdot 1 \cdot 1 = \frac{c}{2} = 1 \Rightarrow c = 2
$$

故联合密度为 $f(x,y) = 2$（在三角形 $D$ 上），$f(x,y) = 0$（在 $D$ 外）。

**$X$ 的边缘密度**（固定 $x \in [0,1]$，对 $y$ 从 $0$ 到 $x$ 积分）：

$$
f_X(x) = \int_{-\infty}^{+\infty} f(x, y)\, dy = \int_0^x 2\, dy = 2x, \quad 0 \leq x \leq 1
$$

**$Y$ 的边缘密度**（固定 $y \in [0,1]$，对 $x$ 从 $y$ 到 $1$ 积分）：

$$
f_Y(y) = \int_{-\infty}^{+\infty} f(x, y)\, dx = \int_y^1 2\, dx = 2(1-y), \quad 0 \leq y \leq 1
$$

**独立性检验**：

$$
f_X(x) \cdot f_Y(y) = 2x \cdot 2(1-y) = 4x(1-y)
$$

$$
f(x, y) = 2 \neq 4x(1-y) \text{（一般情形下）}
$$

故 $X$ 与 $Y$ **不相互独立**。

**区域概率计算（例）：**

$$
P(X + Y \leq 1) = \iint_{\{(x,y)\in D:\ x+y\leq 1\}} 2\, dx\, dy
$$

需将积分区域与 $D$ 取交集，再计算该交集的面积（面积 $\times 2$）。
:::

:::derivation{label=解答}
**第一步：画出支撑区域**
在 $xOy$ 坐标系中画出三角形 $D$（如 $0\leq y \leq x \leq 1$ 对应的直角三角形，顶点为 $(0,0), (1,0), (1,1)$）。这一步极为关键，几何图形直接决定积分限。

**第二步：确定归一化常数**
三角形面积为 $\frac{1}{2}$，故 $c \cdot \frac{1}{2} = 1$，$c = 2$。

**第三步：求 $f_X(x)$**
固定 $x \in [0,1]$，在三角形内，$y$ 的范围是 $[0, x]$（由三角形的不等式 $0 \leq y \leq x$ 读出）。积分：$f_X(x) = \int_0^x 2\,dy = 2x$，$0 \leq x \leq 1$。

**第四步：求 $f_Y(y)$**
固定 $y \in [0,1]$，在三角形内，$x$ 的范围是 $[y, 1]$（由 $y \leq x \leq 1$ 读出）。积分：$f_Y(y) = \int_y^1 2\,dx = 2(1-y)$，$0 \leq y \leq 1$。

**第五步：验证边缘密度合法性**
$$
\int_0^1 f_X(x)\,dx = \int_0^1 2x\,dx = 1 \checkmark
$$
$$
\int_0^1 f_Y(y)\,dy = \int_0^1 2(1-y)\,dy = 1 \checkmark
$$

**第六步：判断独立性**
$f(x,y) = 2 \neq f_X(x) \cdot f_Y(y) = 4x(1-y)$（在 $D$ 内），故不独立。

注意：即使 $f(x,y)$ 在形式上是"常数"（看起来"可分离"），但支撑区域是三角形（不可分离），所以 $X$ 与 $Y$ 不独立。
:::

:::note{label=直观理解}
在三角形均匀分布中，随机点 $(X,Y)$ 均匀散布在三角形内。

**为什么 $X$ 与 $Y$ 不独立？** 直觉上，$X$ 越大，$Y$ 的可取范围（$[0,x]$）就越大，说明知道 $X$ 的大小会影响对 $Y$ 的判断——$X$ 大时，$Y$ 倾向于也大（因为范围更宽）。这种依赖关系完全来自支撑区域的形状，而非密度函数本身的形状。

**边缘密度的直觉**：$f_X(x) = 2x$ 是一个线性递增函数——$X$ 越大，对应的"竖条截面"（$y$ 的范围为 $[0,x]$，宽度为 $x$）越宽，包含的均匀密度越多，因此 $X$ 取大值的可能性越大。$f_Y(y) = 2(1-y)$ 是线性递减函数——$Y$ 越大，对应的"横条截面"（$x$ 的范围为 $[y,1]$，宽度为 $1-y$）越窄，密度越小，因此 $Y$ 倾向于取小值。
:::

:::pitfall{label=易错点1}
**积分限用固定常数代替变量**

最常见错误是将 $f_X(x)$ 的积分写成 $\int_0^1 2\,dy = 2$（积分限固定为 $[0,1]$），而忽略了固定 $x$ 时 $y$ 只能在 $[0,x]$ 内取值。正确积分限依赖于 $x$。
:::

:::pitfall{label=易错点2}
**积分上下限方向搞反**

求 $f_Y(y)$ 时，固定 $y$，$x$ 的范围是 $[y, 1]$（$x$ 从 $y$ 到 $1$），容易误写为 $[0, y]$ 或 $[0,1]$。画图是避免此错误的最有效方法。
:::

:::pitfall{label=易错点3}
**以为常数密度就意味着独立**

在矩形区域上的均匀分布确实独立，但在三角形（或其他非矩形）区域上的均匀分布一般不独立。独立性不仅取决于密度函数的形式，还取决于支撑区域的形状。
:::

:::pitfall{label=易错点4}
**漏写边缘密度的定义域**

$f_X(x) = 2x$ 必须指明 $0 \leq x \leq 1$（其余为0），否则密度函数不完整。
:::

:::pitfall{label=易错点5}
**验证步骤跳过**

求出边缘密度后，应验证积分等于1，这既是检验计算正确性的手段，也是解题规范性的要求。
:::

:::note{label=知识关联}
- **KP03（二维连续型联合密度与边缘密度）**：本例题是KP03核心理论（边缘密度积分法）在非矩形区域上的典型应用，重点练习积分限随另一变量变化的技巧。

- **KP06（独立性）**：通过本例清晰展示：即使联合密度是常数，只要支撑区域不可分离，$X$ 与 $Y$ 也不独立。这是理解"密度可分离 $+$ 区域可分离 $\Rightarrow$ 独立"这一完整条件的关键例子。

- **KP01（联合分布函数）**：若需进一步计算 $F(x,y)$，需要对联合密度在三角形区域内进行二重积分，涉及分区域讨论（$(x,y)$ 在三角形内外的不同情形）。

- **EX01（离散型例题）**：EX01与EX02形成离散—连续对比，边缘化的"求和"与"积分"操作在形式上完全类比，学习两者可相互印证。

- **EX03（条件分布例题）**：在本例题的基础上，进一步计算条件密度 $f_{X|Y}(x|y) = f(x,y)/f_Y(y) = 2/[2(1-y)] = 1/(1-y)$（$y \leq x \leq 1$），即在三角形区域上的条件均匀分布，是EX03相关内容的铺垫。
:::
