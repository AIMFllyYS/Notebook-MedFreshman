:::example{label=题目描述}
设二维随机变量 $(X, Y)$ 的联合密度函数为：
$$f(x, y) = \begin{cases} 6xy^2, & 0 < x < 1,\ 0 < y < 1 \\ 0, & \text{其他} \end{cases}$$

(1) 求 $X$ 的边缘密度函数 $f_X(x)$；

(2) 求 $Y$ 的边缘密度函数 $f_Y(y)$；

(3) 判断 $X$ 与 $Y$ 是否相互独立。
:::

:::insight{label=分析}
求边缘密度函数的方法是对联合密度函数进行"边缘化"积分。求 $f_X(x)$ 时，固定 $x$，对 $y$ 在其有效范围上积分；求 $f_Y(y)$ 时，固定 $y$，对 $x$ 在其有效范围上积分。

本题支撑区域为正方形 $\{0<x<1, 0<y<1\}$，积分范围较简单。积分完成后，还需注明 $f_X(x)$ 和 $f_Y(y)$ 在支撑区域外等于 $0$。

判断独立性时，验证 $f(x,y) = f_X(x) \cdot f_Y(y)$ 是否对所有 $(x,y)$ 成立。本题中若 $f(x,y)$ 可以分解为仅依赖 $x$ 的函数与仅依赖 $y$ 的函数的乘积，且支撑区域为矩形，则 $X$ 与 $Y$ 独立。
:::

:::derivation{label=解答}
**第(1)部分：求 $f_X(x)$**

当 $0 < x < 1$ 时，$y$ 的有效范围为 $(0, 1)$：
$$f_X(x) = \int_{-\infty}^{+\infty} f(x,y)\,dy = \int_0^1 6xy^2\,dy = 6x \cdot \left[\frac{y^3}{3}\right]_0^1 = 6x \cdot \frac{1}{3} = 2x$$

因此：
$$f_X(x) = \begin{cases} 2x, & 0 < x < 1 \\ 0, & \text{其他} \end{cases}$$

**验证：** $\int_0^1 2x\,dx = [x^2]_0^1 = 1$ ✓

**第(2)部分：求 $f_Y(y)$**

当 $0 < y < 1$ 时，$x$ 的有效范围为 $(0, 1)$：
$$f_Y(y) = \int_{-\infty}^{+\infty} f(x,y)\,dx = \int_0^1 6xy^2\,dx = 6y^2 \cdot \left[\frac{x^2}{2}\right]_0^1 = 6y^2 \cdot \frac{1}{2} = 3y^2$$

因此：
$$f_Y(y) = \begin{cases} 3y^2, & 0 < y < 1 \\ 0, & \text{其他} \end{cases}$$

**验证：** $\int_0^1 3y^2\,dy = [y^3]_0^1 = 1$ ✓

**第(3)部分：判断独立性**

验证 $f(x,y) = f_X(x) \cdot f_Y(y)$：

$$f_X(x) \cdot f_Y(y) = 2x \cdot 3y^2 = 6xy^2 = f(x,y), \quad 0 < x < 1,\ 0 < y < 1$$

在支撑区域外，$f(x,y) = 0$，$f_X(x) \cdot f_Y(y) = 0$，等式同样成立。

又因为支撑区域是矩形 $(0,1)\times(0,1)$，满足独立性的几何条件。

**结论：$X$ 与 $Y$ 相互独立。**
:::

:::theorem{label=关键公式}
$$f_X(x) = \int_{-\infty}^{+\infty} f(x,y)\,dy = \int_0^1 6xy^2\,dy = 2x$$

$$f_Y(y) = \int_{-\infty}^{+\infty} f(x,y)\,dx = \int_0^1 6xy^2\,dx = 3y^2$$

**独立性：** $f(x,y) = f_X(x) \cdot f_Y(y) = 2x \cdot 3y^2 = 6xy^2$ ✓
:::

:::pitfall{label=易错点1}
**积分方向搞混**

求 $f_X(x)$ 时对 $y$ 积分，求 $f_Y(y)$ 时对 $x$ 积分，方向容易搞混。
:::

:::pitfall{label=易错点2}
**常数处理错误**

积分时将 $6xy^2$ 中的 $y^2$ 当成常数对 $y$ 积分（应将 $x$ 视为常数，对 $y^2$ 积分得 $y^3/3$）。
:::

:::pitfall{label=易错点3}
**遗漏定义域说明**

边缘密度函数遗漏"其他情况为 $0$"的说明。
:::

:::pitfall{label=易错点4}
**独立性判断不完整**

独立性判断时未检验支撑区域是否为矩形（本题为矩形，满足条件）；若支撑区域非矩形，即使密度函数形式可分，也不独立。
:::
