:::example{label=题目描述}
设二维随机变量 $(X, Y)$ 的联合密度函数为：
$$f(x, y) = \begin{cases} k e^{-(2x+y)}, & x > 0,\ y > 0 \\ 0, & \text{其他} \end{cases}$$

(1) 确定常数 $k$；

(2) 求概率 $P\{X + Y \leq 1\}$。
:::

:::insight{label=分析}
第(1)部分利用联合密度函数的**归一性条件**：$\int_{-\infty}^{+\infty}\int_{-\infty}^{+\infty}f(x,y)\,dx\,dy=1$，将积分计算出来，令其等于 $1$ 即可求出常数 $k$。积分区域为 $x>0, y>0$，两个方向的积分相互独立，可以分离计算。

第(2)部分需要确定积分区域 $D = \{(x,y): x>0, y>0, x+y\leq 1\}$，这是第一象限内位于直线 $x+y=1$ 左下方的三角形区域。在此区域上对密度函数进行二重积分，可以选择先对 $y$（从 $0$ 到 $1-x$）再对 $x$（从 $0$ 到 $1$）的积分次序。
:::

:::derivation{label=解答}
**第(1)部分：确定常数 $k$**

由归一性：
$$\int_0^{+\infty}\int_0^{+\infty} k e^{-(2x+y)}\,dy\,dx = 1$$

由于 $e^{-(2x+y)} = e^{-2x} \cdot e^{-y}$，积分可分离：
$$k \int_0^{+\infty} e^{-2x}\,dx \cdot \int_0^{+\infty} e^{-y}\,dy = 1$$

$$k \cdot \frac{1}{2} \cdot 1 = 1 \implies k = 2$$

**第(2)部分：求 $P\{X+Y \leq 1\}$**

积分区域为 $D = \{(x,y): 0 < x,\ 0 < y,\ x+y \leq 1\}$，即三角形区域。

$$P\{X+Y \leq 1\} = \iint_{D} 2e^{-(2x+y)}\,dx\,dy = \int_0^1\int_0^{1-x} 2e^{-(2x+y)}\,dy\,dx$$

**先计算内层积分（对 $y$）：**
$$\int_0^{1-x} 2e^{-(2x+y)}\,dy = 2e^{-2x}\int_0^{1-x}e^{-y}\,dy = 2e^{-2x}\left[-e^{-y}\right]_0^{1-x} = 2e^{-2x}(1 - e^{-(1-x)})$$

$$= 2e^{-2x} - 2e^{-2x} \cdot e^{-(1-x)} = 2e^{-2x} - 2e^{-(x+1)}$$

**再计算外层积分（对 $x$）：**
$$\int_0^1 \left(2e^{-2x} - 2e^{-(x+1)}\right)dx = \left[-e^{-2x}\right]_0^1 - \left[-2e^{-(x+1)}\right]_0^1$$

$$= \left(-e^{-2}+1\right) + 2\left[e^{-(x+1)}\right]_0^1 = (1 - e^{-2}) + 2(e^{-2} - e^{-1})$$

$$= 1 - e^{-2} + 2e^{-2} - 2e^{-1} = 1 + e^{-2} - 2e^{-1}$$

$$\approx 1 + 0.1353 - 0.7358 \approx 0.3995$$

故 $P\{X+Y\leq 1\} = 1 + e^{-2} - 2e^{-1} \approx 0.400$。
:::

:::theorem{label=关键公式}
**归一性：** $\iint_{\mathbb{R}^2} f(x,y)\,dx\,dy = 1$（用于确定常数 $k$）

$$P\{X+Y\leq 1\} = \int_0^1\int_0^{1-x} f(x,y)\,dy\,dx$$

$$\int_0^{+\infty}e^{-ax}\,dx = \frac{1}{a}\quad (a>0)$$
:::

:::pitfall{label=易错点1}
**归一化积分区间错误**

归一化时积分区间应为 $x>0, y>0$，不要扩展到全实数（其余区域密度为零）。
:::

:::pitfall{label=易错点2}
**积分区域约束不完整**

求 $P\{X+Y\leq 1\}$ 时，积分区域须同时满足 $x>0, y>0, x+y\leq 1$，缺少任一约束条件都会导致积分范围错误。
:::

:::pitfall{label=易错点3}
**内层积分上限写错**

内层积分上限为 $1-x$（依赖 $x$），不能错写为固定的常数。
:::

:::pitfall{label=易错点4}
**指数函数积分符号遗漏**

指数函数积分中注意符号：$\int e^{-y}dy = -e^{-y} + C$，负号容易遗漏。
:::
