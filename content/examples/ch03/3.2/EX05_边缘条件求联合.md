:::example{label=题目描述}
设随机变量 $X$ 在区间 $(0,1)$ 上服从均匀分布，即 $X \sim U(0,1)$。在给定 $X=x$ 的条件下，$Y$ 在区间 $(0, x)$ 上服从均匀分布，即 $Y \mid X=x \sim U(0,x)$。

（1）求 $(X,Y)$ 的联合密度函数 $f(x,y)$；

（2）求 $Y$ 的边缘密度函数 $f_Y(y)$。
:::

:::insight{label=分析}
本题是"由边缘分布和条件分布反推联合分布"的经典问题，利用的核心公式是：

$$f(x,y) = f_X(x) \cdot f_{Y|X}(y|x)$$

这与条件概率的定义完全对应。得到联合密度后，再通过积分求 $Y$ 的边缘密度。本题的难点在于：联合密度的定义域是一个三角形区域 $0 < y < x < 1$，这决定了求 $Y$ 的边缘密度时，对 $x$ 的积分下限是 $y$（而非 $0$）。最终 $f_Y(y) = -\ln y$，是一个在 $(0,1)$ 上取正值的递减函数。
:::

:::derivation{label=解答}
**第一步：写出 $X$ 的边缘密度**

$X \sim U(0,1)$，故：

$$f_X(x) = \begin{cases} 1, & 0 < x < 1 \\ 0, & \text{其他} \end{cases}$$

**第二步：写出条件密度**

给定 $X=x$（其中 $0 < x < 1$），$Y \mid X=x \sim U(0,x)$，故：

$$f_{Y|X}(y|x) = \begin{cases} \dfrac{1}{x}, & 0 < y < x \\ 0, & \text{其他} \end{cases}$$

**第三步：由乘法公式求联合密度**

$$f(x,y) = f_X(x) \cdot f_{Y|X}(y|x) = 1 \cdot \frac{1}{x} = \frac{1}{x}$$

有效范围为 $f_X(x) > 0$ 且 $f_{Y|X}(y|x) > 0$ 同时成立，即：

$$0 < y < x < 1$$

这是平面上 $0<x<1$, $0<y<1$ 的单位正方形中 $y<x$ 的三角形区域。

完整表达式：

$$f(x,y) = \begin{cases} \dfrac{1}{x}, & 0 < y < x < 1 \\ 0, & \text{其他} \end{cases}$$

**第四步：求 $Y$ 的边缘密度**

$$f_Y(y) = \int_{-\infty}^{+\infty} f(x,y)\, dx$$

当 $y \leq 0$ 或 $y \geq 1$ 时，$f_Y(y) = 0$。

当 $0 < y < 1$ 时，需要 $x > y$（且 $x < 1$），故：

$$f_Y(y) = \int_y^1 \frac{1}{x}\, dx = \Big[\ln x\Big]_y^1 = \ln 1 - \ln y = -\ln y$$

**第五步：写出完整结果**

$$f_Y(y) = \begin{cases} -\ln y, & 0 < y < 1 \\ 0, & \text{其他} \end{cases}$$

验证：$\displaystyle\int_0^1 (-\ln y)\, dy = \Big[-y\ln y + y\Big]_0^1 = (0+1)-(0+0) = 1$ ✓

（注意：$\lim_{y\to 0^+} y\ln y = 0$）
:::

:::theorem{label=关键公式}
- 联合密度与条件密度的关系：$f(x,y) = f_X(x) \cdot f_{Y|X}(y|x)$

- 均匀分布 $U(a,b)$ 的密度：$f(t) = \dfrac{1}{b-a}$，$a < t < b$

- $Y$ 的边缘密度：$f_Y(y) = \displaystyle\int_{-\infty}^{+\infty} f(x,y)\, dx$

- 常用积分：$\displaystyle\int_y^1 \frac{1}{x}\, dx = -\ln y$（$0 < y < 1$）

- 验证用：$\displaystyle\int_0^1 (-\ln y)\, dy = 1$（可用分部积分验证）
:::

:::pitfall{label=易错点1}
**联合密度的定义域**

$f(x,y)=1/x$ 仅在 $0<y<x<1$ 上成立，不能写成 $0<x<1, 0<y<1$（那包含了 $y>x$ 的部分，在那里 $f=0$）。
:::

:::pitfall{label=易错点2}
**$f_Y(y)$ 的积分下限**

对 $x$ 积分时，下限是 $y$（因为需要 $x>y$），不是 $0$，这是最容易出错的地方。
:::

:::pitfall{label=易错点3}
**$-\ln y$ 是正值**

当 $0<y<1$ 时 $\ln y < 0$，故 $-\ln y > 0$，符合密度函数非负性，不要因为看到负号就怀疑结果。
:::

:::pitfall{label=易错点4}
**$f(x,y)=1/x$ 在 $x\to 0^+$ 时趋于无穷**

密度函数可以无界，只要积分（总概率）为 $1$ 即可，不要误认为密度函数必须有界。
:::

:::pitfall{label=易错点5}
**验证归一性的积分技巧**

$\int_0^1(-\ln y)\,dy$ 需要分部积分，令 $u=-\ln y$，$dv=dy$，得 $\int_0^1(-\ln y)\,dy=[-y\ln y+y]_0^1=1$。
:::
