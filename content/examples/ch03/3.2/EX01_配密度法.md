:::example{label=题目描述}
已知二维随机变量 $(X,Y)$ 服从二维正态分布，其联合密度函数为：

$$f(x,y) = \frac{1}{2\pi\sigma_1\sigma_2\sqrt{1-\rho^2}} \exp\left\{-\frac{1}{2(1-\rho^2)}\left[\frac{(x-\mu_1)^2}{\sigma_1^2} - \frac{2\rho(x-\mu_1)(y-\mu_2)}{\sigma_1\sigma_2} + \frac{(y-\mu_2)^2}{\sigma_2^2}\right]\right\}$$

其中 $\mu_1, \mu_2 \in \mathbb{R}$，$\sigma_1, \sigma_2 > 0$，$-1 < \rho < 1$。求 $X$ 的边缘密度函数 $f_X(x)$。
:::

:::insight{label=分析}
本题的核心方法是"配密度法"。求边缘密度需要对联合密度关于 $y$ 从负无穷到正无穷积分。直接计算这个积分涉及复杂的指数函数，难度较大。配密度法的思路是：将被积函数中含 $y$ 的部分凑成某个已知概率密度函数（这里是正态分布密度）的形式，然后利用密度函数的规范性（即任意正态密度在全实轴上的积分等于1）直接得到积分结果，从而避免繁琐的显式积分计算。这一技巧充分利用了正态分布的结构特性，是处理二维正态边缘分布的标准方法。
:::

:::derivation{label=解答}
**第一步：写出边缘密度公式**

$$f_X(x) = \int_{-\infty}^{+\infty} f(x,y)\, dy$$

**第二步：整理指数中关于 $y$ 的表达式**

记指数部分的方括号内为 $Q$：

$$Q = \frac{(x-\mu_1)^2}{\sigma_1^2} - \frac{2\rho(x-\mu_1)(y-\mu_2)}{\sigma_1\sigma_2} + \frac{(y-\mu_2)^2}{\sigma_2^2}$$

令 $u = x - \mu_1$，$v = y - \mu_2$，则：

$$Q = \frac{u^2}{\sigma_1^2} - \frac{2\rho u v}{\sigma_1\sigma_2} + \frac{v^2}{\sigma_2^2}$$

**第三步：对 $v$（即 $y$）做配方**

将 $Q$ 关于 $v$ 配方：

$$Q = \frac{1}{\sigma_2^2}\left(v - \frac{\rho\sigma_2}{\sigma_1}u\right)^2 + \frac{u^2}{\sigma_1^2}\left(1-\rho^2\right)$$

**第四步：代入并提出与 $y$ 无关的因子**

$$f_X(x) = \frac{1}{2\pi\sigma_1\sigma_2\sqrt{1-\rho^2}} \exp\left\{-\frac{u^2}{2\sigma_1^2}\right\} \int_{-\infty}^{+\infty} \exp\left\{-\frac{\left(v - \frac{\rho\sigma_2}{\sigma_1}u\right)^2}{2\sigma_2^2(1-\rho^2)}\right\} dv$$

**第五步：识别并应用配密度法**

被积函数恰好是均值为 $\frac{\rho\sigma_2}{\sigma_1}u$、方差为 $\sigma_2^2(1-\rho^2)$ 的正态密度（差一个归一化系数）。即：

$$\int_{-\infty}^{+\infty} \exp\left\{-\frac{\left(v - \frac{\rho\sigma_2}{\sigma_1}u\right)^2}{2\sigma_2^2(1-\rho^2)}\right\} dv = \sqrt{2\pi}\cdot\sigma_2\sqrt{1-\rho^2}$$

**第六步：得到最终结果**

$$f_X(x) = \frac{1}{\sqrt{2\pi}\,\sigma_1} \exp\left\{-\frac{(x-\mu_1)^2}{2\sigma_1^2}\right\}$$

即 $X \sim N(\mu_1, \sigma_1^2)$，$X$ 的边缘分布为一维正态分布。
:::

:::theorem{label=关键公式}
- 边缘密度公式：$f_X(x) = \displaystyle\int_{-\infty}^{+\infty} f(x,y)\, dy$

- 正态密度规范性：$\displaystyle\int_{-\infty}^{+\infty} \frac{1}{\sqrt{2\pi}\,\sigma} e^{-\frac{(t-\mu)^2}{2\sigma^2}} dt = 1$

- 配方公式：$\dfrac{v^2}{\sigma_2^2} - \dfrac{2\rho u v}{\sigma_1\sigma_2} = \dfrac{1}{\sigma_2^2}\left(v - \dfrac{\rho\sigma_2}{\sigma_1}u\right)^2 - \dfrac{\rho^2 u^2}{\sigma_1^2}$

- 结论：二维正态分布的边缘分布仍为一维正态分布，参数为 $(\mu_1, \sigma_1^2)$ 和 $(\mu_2, \sigma_2^2)$。
:::

:::pitfall{label=易错点1}
**配方时忘记交叉项**

$Q$ 中有交叉项 $-2\rho uv/(\sigma_1\sigma_2)$，配方时必须完整处理，不能遗漏。
:::

:::pitfall{label=易错点2}
**忽略 $(1-\rho^2)$ 因子**

配方后提出的与 $y$ 无关的部分包含 $(1-\rho^2)$，若忽略会导致指数系数错误。
:::

:::pitfall{label=易错点3}
**配密度法的适用条件**

被积函数必须能凑成完整的概率密度形式（含归一化系数），否则不能直接令积分为1。
:::

:::pitfall{label=易错点4}
**注意积分变量**

提出与 $y$ 无关的因子后，留在积分号内的只有含 $y$ 的部分，不要将常数因子也积掉。
:::

:::pitfall{label=易错点5}
**结论的推广**

二维正态边缘为正态，但反之不成立（两个正态边缘不一定构成二维正态），见 EX02。
:::
