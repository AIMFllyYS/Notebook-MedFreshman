:::example{label=题目描述}
一名射手独立重复射击，每次击中目标的概率为 $p$（$0 < p < 1$），未击中概率 $q = 1-p$。射击持续进行，直到恰好击中两次为止。

- 设 $X$ 为第一次击中时的射击次数（即首次成功所在第几次射击）；
- 设 $Y$ 为总共射击的次数（即恰好完成两次击中时所需的总次数）。

（1）求 $(X,Y)$ 的联合分布律；

（2）求 $X$ 和 $Y$ 各自的边缘分布律；

（3）求条件分布律 $P(X=m \mid Y=n)$。
:::

:::insight{label=分析}
本题综合考察几何分布、负二项分布、联合分布、边缘分布和条件分布的计算。关键是正确刻画 $X$ 和 $Y$ 的取值范围及联合分布的参数范围：$X=m$ 表示前 $m-1$ 次未击中、第 $m$ 次击中；$Y=n$ 表示恰在第 $n$ 次射击时完成第二次击中，这意味着前 $n-1$ 次中恰好有 $1$ 次击中，且第 $n$ 次击中。利用独立性将联合概率分解，然后通过求和得边缘分布，最后用条件概率公式得条件分布。条件分布结果为离散均匀分布，有清晰的组合意义。
:::

:::derivation{label=解答}
**第一步：确定取值范围**

- $X$ 的取值：$m = 1, 2, 3, \ldots$（首次击中可在任何一次）
- $Y$ 的取值：$n = 2, 3, 4, \ldots$（至少需要2次射击才能击中2次）
- 联合约束：第一次击中在第 $m$ 次，第二次击中在第 $n$ 次，故必须 $m < n$

综合：$m \geq 1$，$n \geq m+1$（等价于 $1 \leq m \leq n-1$，$n \geq 2$）

**第二步：计算联合分布律**

事件 $\{X=m, Y=n\}$ 表示：
- 第 $m$ 次恰好是第一次击中：前 $m-1$ 次全未中（$q^{m-1}$），第 $m$ 次击中（$p$）
- 第 $n$ 次恰好是第二次击中：$m$ 到 $n-1$ 次中全部未击中（$q^{n-1-m}$），第 $n$ 次击中（$p$）

由独立性：

$$P(X=m, Y=n) = q^{m-1} \cdot p \cdot q^{n-1-m} \cdot p = p^2 q^{n-2}$$

其中 $1 \leq m \leq n-1$，$n \geq 2$。

**第三步：求 $X$ 的边缘分布**

$$P(X=m) = \sum_{n=m+1}^{\infty} P(X=m, Y=n) = \sum_{n=m+1}^{\infty} p^2 q^{n-2}$$

令 $k = n-m-1$，从 $0$ 到 $\infty$：

$$P(X=m) = p^2 q^{m-1} \sum_{k=0}^{\infty} q^k = p^2 q^{m-1} \cdot \frac{1}{1-q} = p \cdot q^{m-1}$$

故 $X \sim G(p)$（几何分布），$P(X=m) = pq^{m-1}$，$m=1,2,3,\ldots$

**第四步：求 $Y$ 的边缘分布**

$$P(Y=n) = \sum_{m=1}^{n-1} P(X=m, Y=n) = \sum_{m=1}^{n-1} p^2 q^{n-2} = (n-1)p^2 q^{n-2}$$

故 $Y$ 服从负二项分布（Pascal分布），$P(Y=n) = (n-1)p^2 q^{n-2}$，$n=2,3,4,\ldots$

**第五步：求条件分布律**

$$P(X=m \mid Y=n) = \frac{P(X=m, Y=n)}{P(Y=n)} = \frac{p^2 q^{n-2}}{(n-1)p^2 q^{n-2}} = \frac{1}{n-1}$$

其中 $m = 1, 2, \ldots, n-1$（共 $n-1$ 个等可能值）。

故在 $Y=n$ 条件下，$X$ 在 $\{1,2,\ldots,n-1\}$ 上服从**离散均匀分布**。
:::

:::theorem{label=关键公式}
- 几何分布：$P(X=m) = p(1-p)^{m-1}$，$m = 1,2,3,\ldots$

- 负二项分布（第 $r$ 次成功所需次数）：$P(Y=n) = \binom{n-1}{r-1}p^r q^{n-r}$

  本题 $r=2$：$P(Y=n) = (n-1)p^2 q^{n-2}$，$n=2,3,\ldots$

- 等比级数求和：$\displaystyle\sum_{k=0}^{\infty} q^k = \frac{1}{1-q}$（$|q|<1$）

- 条件分布：$P(X=m \mid Y=n) = \dfrac{P(X=m, Y=n)}{P(Y=n)}$
:::

:::pitfall{label=易错点1}
**范围约束**

必须保证 $m < n$（首次击中在总次数之前），若写成 $m \leq n$ 会包含不合理情形。
:::

:::pitfall{label=易错点2}
**联合分布与 $m$ 无关**

$P(X=m,Y=n) = p^2q^{n-2}$ 中不含 $m$，这是因为给定总次数 $n$，第一次击中的位置在前 $n-1$ 次中等可能，初学者可能对此感到意外。
:::

:::pitfall{label=易错点3}
**边缘分布的求和技巧**

对 $X$ 求边缘时需将 $n$ 从 $m+1$ 到 $\infty$ 求和，注意提出 $q^{m-1}$ 后用等比级数。
:::

:::pitfall{label=易错点4}
**负二项分布的参数形式**

本题 $Y$ 的边缘 $(n-1)p^2q^{n-2}$ 是 $r=2$ 的负二项分布，不要与 $r=1$ 的几何分布混淆。
:::

:::pitfall{label=易错点5}
**条件分布的验证**

$\sum_{m=1}^{n-1} \frac{1}{n-1} = 1$，可以验证条件分布的归一性。
:::
