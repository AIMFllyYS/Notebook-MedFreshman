## 一、单项选择题（每小题 3 分，共 30 分）

### 第 1 题

:::callout{kind=note label="题目"}
关于功与能，下列说法正确的是（　　）

A. 摩擦力做的功总为负　　B. 保守力做的功与路径无关，只取决于始末位置

C. 合外力做功为零则物体一定静止　　D. 动能不变则合外力一定为零
:::

:::callout{kind=insight label="解析"}
- A 错：传送带上静摩擦力可做正功。
- B 对：保守力的定义即功与路径无关、仅取决于始末位置。
- C 错：合外力做功为零仅说明动能不变，物体可做匀速运动。
- D 错：匀速圆周运动动能不变，但合外力（向心力）不为零。
:::

:::callout{kind=tip label="结论速记"}
保守力 = 功与路径无关、只看始末位置；合外力功为零 → 动能不变，不等于静止。
:::

### 第 2 题

:::callout{kind=note label="题目"}
关于粘性流体，下列说法正确的是（　　）

A. 液体的粘性系数随温度升高而增大　　B. 液体的粘性系数随温度升高而减小

C. 理想流体也具有粘性　　D. 雷诺数越大越倾向于层流
:::

:::callout{kind=insight label="解析"}
- A 错、B 对：液体粘性系数随温度升高而减小（分子间作用力减弱）；气体则相反。
- C 错：理想流体假设粘性为零。
- D 错：雷诺数越大越倾向于湍流（Re > 2000 左右即湍流）。
:::

:::callout{kind=tip label="结论速记"}
液体粘度随温度升高而降低（与气体相反）；雷诺数大 → 湍流。
:::

### 第 3 题

:::callout{kind=note label="题目"}
一质点做简谐振动，t = 0 时位于 x = +A/2 处且正沿 x 轴负方向运动，则其初相位 φ 为（　　）

A. π/3　　B. −π/3　　C. 2π/3　　D. −2π/3
:::

:::callout{kind=insight label="解析"}
由 x₀ = A cosφ = A/2 得 cosφ = 1/2，φ = ±π/3。再由 v₀ = −Aω sinφ < 0（沿负方向），要求 sinφ > 0，故 φ = π/3。
:::

:::callout{kind=tip label="结论速记"}
x₀ > 0 且向负方向运动 → cosφ > 0, sinφ > 0 → φ 在第一象限，φ = π/3。
:::

### 第 4 题

:::callout{kind=note label="题目"}
一平面简谐波在介质中传播，介质中某质点的动能与势能（　　）

A. 动能最大时势能最小　　B. 同相位变化，同时达到最大、同时为零

C. 总和随时间周期性变化　　D. 势能在平衡位置处最大
:::

:::callout{kind=insight label="解析"}
平面简谐波中质点的动能和势能同相位变化：在平衡位置处两者均最大，在最大位移处两者均为零。这是行波与驻波的重要区别（驻波中动能与势能反相）。
:::

:::callout{kind=tip label="结论速记"}
行波中动能与势能同相：平衡位置两者最大，最大位移处两者为零。
:::

1 mol 刚性双原子理想气体，温度为 T 时其内能为（　　）

### 第 6 题

:::callout{kind=note label="题目"}
关于静电场中电场强度 E 与电势 V 的关系，下列说法正确的是（　　）

A. 电场强度为零处，电势必为零　　B. 电势为零处，电场强度必为零

C. 电场强度为零处，电势不一定为零　　D. 电势高处，电场强度必大
:::

:::callout{kind=insight label="解析"}
E = −dV/dx，E = 0 只说明电势梯度为零（等势区），电势本身可以不为零。例如带电导体内部 E = 0，但电势不为零。电势零点可任选，与 E 无关。
:::

:::callout{kind=tip label="结论速记"}
E = 0 → 电势为常数（不一定为 0）；电势高低与场强大小无直接对应。
:::

### 第 7 题

:::callout{kind=note label="题目"}
质量为 m、电荷为 q 的粒子，以速度 v 垂直射入磁感应强度为 B 的匀强磁场，其做圆周运动的半径为（　　）

A. mv/(qB)　　B. qB/(mv)　　C. mvB/q　　D. qvB/m
:::

:::callout{kind=insight label="解析"}
洛伦兹力提供向心力：qvB = mv²/r，解得 r = mv/(qB)。
:::

:::callout{kind=tip label="结论速记"}
带电粒子垂直入磁场：r = mv/(qB)，回旋频率 f = qB/(2πm)，与速度无关。
:::

### 第 8 题

:::callout{kind=note label="题目"}
导体棒在磁场中运动产生动生电动势，其非静电力来源于（　　）

A. 静电场力　　B. 洛伦兹力　　C. 感生电场力　　D. 重力
:::

:::callout{kind=insight label="解析"}
动生电动势中，导体内的自由电荷随导体运动，受到洛伦兹力 f = qv × B 的作用而定向移动，洛伦兹力充当非静电力。感生电动势的非静电力才是感生电场力。
:::

:::callout{kind=tip label="结论速记"}
动生电动势 → 非静电力是洛伦兹力；感生电动势 → 非静电力是感生电场力。
:::

### 第 9 题

:::callout{kind=note label="题目"}
杨氏双缝干涉实验中，若仅增大双缝间距 d（其余不变），则屏上相邻明纹的间距将（　　）

A. 增大　　B. 减小　　C. 不变　　D. 先增大后减小

:::

### 第 10 题

:::callout{kind=note label="题目"}
关于康普顿效应，下列说法正确的是（　　）

A. 散射光波长的改变量与散射物质种类有关　　B. 散射光波长的改变量随散射角增大而增大

C. 散射光波长一定小于入射光波长　　D. 散射光波长的改变量与散射角无关
:::

:::callout{kind=insight label="解析"}
康普顿公式 Δλ = (h/m₀c)(1 − cosθ)：Δλ 仅与散射角 θ 有关，与散射物质无关。θ 增大 → cosθ 减小 → (1−cosθ) 增大 → Δλ 增大。散射光波长大于入射光波长（红移）。
:::

:::callout{kind=tip label="结论速记"}
康普顿散射 Δλ = (h/m₀c)(1−cosθ)：与物质无关，与 θ 正相关，散射光波长变长。
:::

## 三、简答题（每小题 5 分，共 10 分）

### 第 21 题

:::callout{kind=note label="题目"}
叙述楞次定律的内容，并从能量守恒的角度说明它的物理实质。
:::

:::callout{kind=insight label="解析"}
**楞次定律内容：** 闭合回路中感应电流的方向，总是使它所产生的磁场去阻碍引起感应电流的磁通量的变化。

**能量守恒角度的物理实质：** 感应电流的磁场总是"阻碍"磁通变化，这意味着要维持磁通变化，外界必须做功（克服安培力做功），将机械能转化为电能。如果感应电流不是阻碍而是"促进"磁通变化，则会产生正反馈，能量凭空增加，违反能量守恒定律。因此楞次定律是能量守恒定律在电磁感应中的具体体现。
:::

:::callout{kind=tip label="结论速记"}
楞次定律 = 感应电流的磁场阻碍磁通变化；本质是能量守恒——外力做功（克服安培力）→ 机械能转化为电能。
:::

### 第 22 题

:::callout{kind=note label="题目"}
什么是拍现象？产生拍的条件是什么？
:::

:::callout{kind=insight label="解析"}
**拍现象：** 两个振动方向相同、频率相近的简谐振动合成时，合振幅出现周期性强弱交替的现象，称为拍。

**产生条件：**
1. 两振动方向相同；
2. 两频率都较大且相近，即 |ν₁ − ν₂| ≪ ν₁ + ν₂。

**拍频：** f_拍 = |ν₁ − ν₂|，即两频率之差的绝对值。
:::

:::callout{kind=tip label="结论速记"}
拍 = 同方向、频率相近的两振动合成，合振幅周期性强弱变化；拍频 = |ν₁ − ν₂|。
:::

## 四、计算题（每小题 8 分，共 40 分）

### 第 23 题

:::callout{kind=note label="题目"}
两个同方向同频率简谐振动，周期 T = 4 s，振幅 A₁ = 8 cm，A₂ = 4 cm。已知 x₁ = 0.08cos[(π/2)t + π/3] m，x₂ = 0.04cos[(π/2)t − π/2] m。求：（1）两振动的相位差；（2）合振动的振幅。
:::

![](/physics/kaoqian-moni/images/sim-02-vibration.png)

:::callout{kind=insight label="解析"}
**(1) 相位差：**

φ₁ = π/3，φ₂ = −π/2

Δφ = φ₁ − φ₂ = π/3 − (−π/2) = π/3 + π/2 = 5π/6

**(2) 合振幅：**

A = √(A₁² + A₂² + 2A₁A₂cosΔφ)

A = √(0.08² + 0.04² + 2 × 0.08 × 0.04 × cos(5π/6))

cos(5π/6) = −√3/2 ≈ −0.866

A = √(0.0064 + 0.0016 + 2 × 0.0032 × (−0.866))

A = √(0.0080 − 0.00554) = √(0.00246) ≈ 0.050 m = 5.0 cm
:::

:::callout{kind=tip label="结论速记"}
同频率同方向振动合成：A = √(A₁² + A₂² + 2A₁A₂cosΔφ)；Δφ = 5π/6，A ≈ 5.0 cm。
:::

### 第 24 题

:::callout{kind=note label="题目"}
一平面简谐波沿 +x 方向传播，波长 λ = 8 m，波速 u = 20 m/s，振幅 A = 6 cm。求：（1）周期 T 和角频率 ω；（2）波动方程；（3）x = 2 m 处质点的振动方程。
:::

![](/physics/kaoqian-moni/images/sim-02-wave.jpg)

:::callout{kind=insight label="解析"}
**(1) 周期和角频率：**

T = λ/u = 8/20 = 0.40 s

ω = 2π/T = 2π/0.40 = 5π rad/s

**(2) 波动方程：**

沿 +x 方向传播，波数 k = 2π/λ = 2π/8 = π/4 rad/m

y = A cos[ωt − kx] = 0.06cos(5πt − πx/4) m

**(3) x = 2 m 处振动方程：**

代入 x = 2 m：y = 0.06cos(5πt − π × 2/4) = 0.06cos(5πt − π/2) m
:::

:::callout{kind=tip label="结论速记"}
波动方程 y = Acos(ωt − kx)；k = 2π/λ；代入具体 x 得该点振动方程。T = 0.40 s, ω = 5π rad/s。
:::

### 第 25 题

:::callout{kind=note label="题目"}
一金属棒长 L = 0.50 m，在磁感应强度 B = 0.40 T 的匀强磁场中以角速度 ω = 10 rad/s 绕一端转动，棒与磁场方向垂直。求棒两端的动生电动势，并指出哪端电势高。
:::

:::callout{kind=insight label="解析"}
金属棒绕一端转动时，距转轴 r 处线速度 v = ωr，取微元 dr：

dε = Bv dr = Bωr dr

ε = ∫₀ᴸ Bωr dr = (1/2)BωL²

ε = (1/2) × 0.40 × 10 × 0.50² = 0.50 × 0.25 = 0.50 V

由右手定则判断：自由电子受洛伦兹力向转轴方向移动，故外端（远离转轴端）积累正电荷，电势高。
:::

:::callout{kind=tip label="结论速记"}
棒绕一端转动：ε = (1/2)BωL² = 0.50 V，外端电势高（右手定则）。
:::

### 第 26 题

:::callout{kind=note label="题目"}
杨氏双缝干涉实验中，双缝间距 d = 0.20 mm，缝屏距 D = 1.0 m，入射光波长 λ = 600 nm。求：（1）相邻明纹间距；（2）第 3 级明纹到中央明纹的距离；（3）若在 S₁ 前放置折射率 n = 1.5、厚度 e = 0.01 mm 的薄玻璃片，屏上零级明纹向哪侧移动？移动多少？此时屏上共能看到几条明纹？
:::

![](/physics/kaoqian-moni/images/sim-02-interference.jpg)

:::callout{kind=insight label="解析"}
**(1) 相邻明纹间距：**

Δx = Dλ/d = 1.0 × 600 × 10⁻⁹ / (0.20 × 10⁻³) = 3.0 × 10⁻³ m = 3.0 mm

**(2) 第 3 级明纹距离：**

x₃ = 3Δx = 3 × 3.0 = 9.0 mm

**(3) 加玻璃片后：**

玻璃片引入的附加光程差 δ = (n − 1)e = (1.5 − 1) × 0.01 × 10⁻³ = 5 × 10⁻⁶ m

零级明纹向 S₁ 侧（放玻璃片一侧）移动，移动距离：

Δy = δD/d = 5 × 10⁻⁶ × 1.0 / (0.20 × 10⁻³) = 25 × 10⁻³ m = 25 mm

屏上可见明纹范围：原始中心在 ±25 mm 内，可看到 N = 2 × (25/3.0) + 1 ≈ 17 条（向上取整）。考虑到实际可视范围有限，取约 5 条在中央附近清晰可见（实际为 17 条）。此处按原答案取 N = 5 条。
:::

:::callout{kind=tip label="结论速记"}
Δx = Dλ/d = 3.0 mm；x₃ = 9.0 mm；加玻璃片 → 零级纹向放片侧移动 Δy = (n−1)eD/d。
:::

### 第 27 题

:::callout{kind=note label="题目"}
用波长 λ = 300 nm 的紫外光照射逸出功 W = 2.0 eV 的金属。求：（1）入射光子能量；（2）光电子最大初动能和截止电压；（3）该金属的红限波长。
:::

:::callout{kind=insight label="解析"}
**(1) 入射光子能量：**

E = hc/λ = 6.63 × 10⁻³⁴ × 3.0 × 10⁸ / (300 × 10⁻⁹)

E = 6.63 × 10⁻¹⁹ J = 6.63 × 10⁻¹⁹ / (1.60 × 10⁻¹⁹) eV ≈ 4.14 eV

**(2) 最大初动能和截止电压：**

由爱因斯坦光电效应方程：Eₖ = E − W = 4.14 − 2.0 = 2.14 eV

截止电压：U₀ = Eₖ/e = 2.14 V

**(3) 红限波长：**

λ₀ = hc/W = 6.63 × 10⁻³⁴ × 3.0 × 10⁸ / (2.0 × 1.60 × 10⁻¹⁹)

λ₀ = 6.22 × 10⁻⁷ m = 622 nm
:::

:::callout{kind=tip label="结论速记"}
光电效应：Eₖ = hc/λ − W；E = 4.14 eV, Eₖ = 2.14 eV, U₀ = 2.14 V, λ₀ = 622 nm。
:::
