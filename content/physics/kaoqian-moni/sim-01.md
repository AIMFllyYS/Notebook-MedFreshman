## 一、单项选择题（每小题 3 分，共 30 分）

### 第1题

:::callout{kind=note label="题目"}
无风的雨天，雨滴竖直下落。乘客坐在匀速直线行驶的火车中，透过车窗看到雨滴的下落方向（　　）

A. 仍然竖直向下　　B. 偏向火车前进方向　　C. 偏向火车前进的反方向　　D. 无法判断
:::

:::callout{kind=insight label="解析"}
这是经典的相对速度问题。地面参考系中雨滴速度 $\vec{v}_{雨}$ 竖直向下，火车参考系中需要将雨滴速度减去火车速度：$\vec{v}_{雨/车} = \vec{v}_{雨} - \vec{v}_{车}$。

由于 $\vec{v}_{车}$ 水平向前，$\vec{v}_{雨/车}$ 在竖直向下分量基础上叠加了水平向后的分量，因此乘客看到雨滴偏向火车前进的反方向。这就是为什么下暴雨时打开车窗，雨水会斜着飞进来。
:::

:::callout{kind=tip label="结论速记"}
相对速度叠加：观察者运动时，所见物体速度需减去观察者自身速度。匀速直线运动的火车中，竖直下落的雨滴看起来偏向火车前进的反方向。
:::

### 第2题

:::callout{kind=note label="题目"}
人体血液在循环系统中流动，主动脉总截面积小、毛细血管总截面积大。根据连续性方程，血液在毛细血管中的平均流速与主动脉中相比（　　）

A. 更大　　B. 更小　　C. 相等　　D. 无确定关系
:::

:::callout{kind=insight label="解析"}
理想流体的连续性方程：$S_1 v_1 = S_2 v_2$，即截面积与流速成反比。

主动脉总截面积 $S_A$ 小，毛细血管总截面积 $S_C$ 大，且 $S_C \gg S_A$，由连续性方程 $S_A v_A = S_C v_C$ 得：

$$v_C = \frac{S_A}{S_C} v_A$$

由于 $S_C > S_A$，故 $v_C < v_A$，毛细血管中血液流速更小。这也解释了为什么毛细血管中血液与组织有充分时间进行物质交换。
:::

:::callout{kind=tip label="结论速记"}
连续性方程 $Sv = \text{常数}$：截面积越大处流速越小。毛细血管总截面积远大于主动脉，因此血液在毛细血管中流速显著减小。
:::

### 第3题

:::callout{kind=note label="题目"}
质点做简谐振动，当其位移达到最大值时（　　）

A. 速度最大，加速度最大　　B. 速度最大，加速度为零　　C. 速度为零，加速度最大　　D. 速度为零，加速度为零
:::

:::callout{kind=insight label="解析"}
简谐振动的基本关系：$x = A\cos(\omega t + \varphi)$

- 速度：$v = \frac{dx}{dt} = -A\omega\sin(\omega t + \varphi)$
- 加速度：$a = \frac{d^2x}{dt^2} = -A\omega^2\cos(\omega t + \varphi) = -\omega^2 x$

当位移最大（$|x| = A$）时：
- $\cos(\omega t + \varphi) = \pm 1$，此时 $\sin(\omega t + \varphi) = 0$，故 $v = 0$
- $|a| = \omega^2 A$，加速度达到最大值

物理图像：质点运动到最大位移处瞬间"折返"，速度为零；但回复力最大（$F = -kx$），加速度最大。
:::

:::callout{kind=tip label="结论速记"}
简谐振动中位移最大时：速度为零、加速度最大。加速度与位移反向：$a = -\omega^2 x$。速度最大出现在平衡位置。
:::

### 第4题

:::callout{kind=note label="题目"}
关于机械波，下列说法正确的是（　　）

A. 声波在空气中传播是横波　　B. 波传播时介质质点随波一起向前迁移　　C. 横波的振动方向与传播方向垂直　　D. 机械波可以在真空中传播
:::

:::callout{kind=insight label="解析"}
逐项分析：

A. 声波在空气中是**纵波**（疏密波），质点振动方向与传播方向平行。错误。

B. 波传播时，介质质点只在平衡位置附近振动，**不随波迁移**。这是波动的基本特征——传播的是振动状态和能量，不是物质本身。错误。

C. 横波的定义正是振动方向与传播方向**垂直**的波。正确。

D. 机械波需要介质传播，**不能在真空中传播**。电磁波才可以在真空中传播。错误。
:::

:::callout{kind=tip label="结论速记"}
机械波核心辨析：①横波振动方向⊥传播方向，纵波振动方向∥传播方向；②声波在空气中是纵波；③质点只振动不随波迁移；④机械波需要介质，不能在真空中传播。
:::

### 第5题

:::callout{kind=note label="题目"}
一定量理想气体，分子的平均平动动能 $\bar{\varepsilon}$ 取决于（　　）

A. 气体的压强　　B. 气体的体积　　C. 气体的温度　　D. 分子数密度
:::

:::callout{kind=insight label="解析"}
根据气体分子运动论，分子的平均平动动能与温度的关系为：

$$\bar{\varepsilon} = \frac{3}{2} k_B T$$

其中 $k_B$ 为玻尔兹曼常数，$T$ 为热力学温度。

这是统计力学的基本结论：温度是分子热运动剧烈程度的量度。分子的平均平动动能**仅取决于温度**，与气体的压强、体积、分子数密度无关。
:::

:::callout{kind=tip label="结论速记"}
分子平均平动动能 $\bar{\varepsilon} = \frac{3}{2}k_BT$，仅由温度决定。温度是分子热运动剧烈程度的宏观量度。
:::

### 第6题

:::callout{kind=note label="题目"}
一均匀带正电的球面，总电荷为 $Q$，半径为 $R$。则在球面内部（$r < R$）任一点的电场强度（　　）

A. 处处为零　　B. 指向球心　　C. 背离球心　　D. 与 $1/r^2$ 成正比
:::

:::callout{kind=insight label="解析"}
利用高斯定理求解。取半径为 $r$（$r < R$）的同心球面为高斯面：

$$\oint \vec{E} \cdot d\vec{A} = \frac{Q_{内}}{\varepsilon_0}$$

由于电荷全部分布在球面上（$r = R$ 处），对于 $r < R$ 的高斯面，内部包围的电荷 $Q_{内} = 0$。

由球对称性，$E \cdot 4\pi r^2 = 0$，故 $E = 0$。

**球壳内部电场强度处处为零**，这是静电平衡下导体内部电场为零的等价表述。
:::

:::callout{kind=tip label="结论速记"}
均匀带电球面内部（$r < R$）电场强度处处为零。高斯面内无净电荷。球面外部等效于点电荷，$E = Q/(4\pi\varepsilon_0 r^2)$。
:::

### 第7题

:::callout{kind=note label="题目"}
带电粒子在匀强磁场中运动，磁场力（洛伦兹力）对粒子（　　）

A. 总是做正功　　B. 总是做负功　　C. 不做功　　D. 做功正负取决于速度方向
:::

:::callout{kind=insight label="解析"}
洛伦兹力公式为：

$$\vec{F} = q\vec{v} \times \vec{B}$$

洛伦兹力的方向始终垂直于速度 $\vec{v}$ 和磁场 $\vec{B}$ 所确定的平面，即 $\vec{F} \perp \vec{v}$。

功率 $P = \vec{F} \cdot \vec{v} = 0$（因为力与速度始终垂直），所以洛伦兹力**永远不做功**。

这是洛伦兹力的重要特性：它只改变粒子运动方向，不改变粒子速率（动能）。
:::

:::callout{kind=tip label="结论速记"}
洛伦兹力 $\vec{F} = q\vec{v} \times \vec{B}$ 始终垂直于速度方向，永远不做功。它只改变带电粒子运动方向，不改变速率和动能。
:::

### 第8题

:::callout{kind=note label="题目"}
将一条形磁铁的 N 极插入闭合线圈，则线圈中感应电流产生的磁场，在面对磁铁一侧（　　）

A. 为 N 极，阻碍磁铁插入　　B. 为 S 极，吸引磁铁插入　　C. 为 N 极，吸引磁铁插入　　D. 方向与磁铁运动无关
:::

:::callout{kind=insight label="解析"}
根据楞次定律：感应电流的效果总是**阻碍**引起感应电流的原因。

N 极插入线圈时，穿过线圈的磁通量增加。感应电流产生的磁场要**阻碍磁通量增加**，因此在面对磁铁一侧应为 **N 极**，与磁铁 N 极相斥，阻碍插入。

也可以从"来拒去留"的角度理解：磁铁靠近时产生排斥力（来拒），磁铁离开时产生吸引力（去留）。本质都是电磁感应中的能量守恒。
:::

:::callout{kind=tip label="结论速记"}
楞次定律：感应电流的磁场总是阻碍原磁通量变化。N 极插入时面对侧为 N 极（阻碍插入）；S 极插入时面对侧为 S 极（阻碍插入）。口诀："来拒去留"。
:::

### 第9题

:::callout{kind=note label="题目"}
近视眼看不清远处物体，矫正所用的镜片以及其作用是（　　）

A. 凸透镜，使像后移到视网膜上　　B. 凹透镜，使像后移到视网膜上　　C. 凸透镜，使像前移到视网膜上　　D. 凹透镜，使像后移到视网膜上
:::

:::callout{kind=insight label="解析"}
近视眼的成因：眼球前后径过长或晶状体折射力过大，远处物体的像成在视网膜**前方**。

矫正方法：佩戴**凹透镜**（发散透镜），使进入眼睛的光线先发散一些，这样经过晶状体折射后，像会**后移**到视网膜上。

- 凹透镜 = 发散透镜 = 矫正近视
- 凸透镜 = 会聚透镜 = 矫正远视

注意：选项 C（凸透镜使像前移）是远视眼的矫正方式。
:::

:::callout{kind=tip label="结论速记"}
近视眼：像成在视网膜前方，用凹透镜（发散透镜）矫正，使像后移到视网膜上。远视眼相反：像成在视网膜后方，用凸透镜矫正。
:::

### 第10题

:::callout{kind=note label="题目"}
关于光电效应，下列说法正确的是（　　）

A. 遏止电压与入射光强度成正比　　B. 只要光足够强就一定能产生光电效应　　C. 光电子的最大初动能与入射光频率有关　　D. 存在光电流的时间随光强增大而明显滞后
:::

:::callout{kind=insight label="解析"}
光电效应方程（爱因斯坦）：

$$E_k = h\nu - W_0$$

其中 $E_k$ 为光电子最大初动能，$h\nu$ 为光子能量，$W_0$ 为逸出功。

逐项分析：

A. 遏止电压 $U_c$ 满足 $eU_c = E_k = h\nu - W_0$，与入射光强度无关。错误。

B. 光电效应的发生条件是入射光频率 $\nu > \nu_0$（截止频率），与光强无关。光再强，若频率低于截止频率也不能产生光电效应。错误。

C. 由 $E_k = h\nu - W_0$，最大初动能与入射光频率有关。正确。

D. 光电效应的响应时间极短（约 $10^{-9}$ s 量级），几乎无滞后，与光强无关。错误。
:::

:::callout{kind=tip label="结论速记"}
光电效应关键结论：①最大初动能 $E_k = h\nu - W_0$，与频率有关，与光强无关；②存在截止频率 $\nu_0$，低于它无论多强的光都不能产生光电效应；③遏止电压由频率决定；④光电效应几乎是瞬时的。
:::

## 二、填空题（每空 2 分，共 20 分）

### 第11题

:::callout{kind=note label="题目"}
大容器侧壁开一小孔，孔在液面下方深度为 $h$，由托里拆利定理，小孔流速 $v =$ ______。
:::

:::callout{kind=insight label="解析"}
根据伯努利方程，液面处（流速近似为零）和小孔处：

$$P_0 + \rho g h = P_0 + \frac{1}{2}\rho v^2$$

解得：

$$v = \sqrt{2gh}$$

这就是托里拆利定理（Torricelli's theorem），本质上是伯努利方程的特例。
:::

:::callout{kind=tip label="结论速记"}
托里拆利定理：$v = \sqrt{2gh}$，液面下深度 $h$ 处小孔的出流速度等于自由下落高度 $h$ 所获得的速度。
:::

### 第12题

:::callout{kind=note label="题目"}
无限大均匀带电平面，面电荷密度为 $\sigma$，其两侧产生的电场强度大小 $E =$ ______。
:::

:::callout{kind=insight label="解析"}
利用高斯定理。取柱形高斯面，一端在平面一侧，另一端在另一侧，轴线垂直于带电平面：

$$\oint \vec{E} \cdot d\vec{A} = 2E \cdot A = \frac{\sigma A}{\varepsilon_0}$$

解得：

$$E = \frac{\sigma}{2\varepsilon_0}$$

无限大均匀带电平面的电场是**匀强电场**，与距离无关。两侧电场方向均垂直于平面，正电荷向外，负电荷向内。
:::

:::callout{kind=tip label="结论速记"}
无限大均匀带电平面电场：$E = \sigma/(2\varepsilon_0)$，为匀强电场，与距离无关，方向垂直于平面两侧。
:::

### 第13题

:::callout{kind=note label="题目"}
真空中一无限长直导线通有电流 $I$，距导线 $r$ 处的磁感应强度大小 $B =$ ______。
:::

:::callout{kind=insight label="解析"}
由安培环路定理，取以导线为轴、半径为 $r$ 的圆形安培环路：

$$\oint \vec{B} \cdot d\vec{l} = B \cdot 2\pi r = \mu_0 I$$

解得：

$$B = \frac{\mu_0 I}{2\pi r}$$

磁感应强度与距离 $r$ 成反比，方向沿圆周切线方向（右手螺旋定则判断）。
:::

:::callout{kind=tip label="结论速记"}
无限长直导线磁场：$B = \mu_0 I/(2\pi r)$，与距离 $r$ 成反比。右手螺旋定则：拇指指向电流方向，四指弯曲方向为磁场方向。
:::

### 第14题

:::callout{kind=note label="题目"}
两相干波在某点相遇，发生相消干涉（振动减弱）的条件是波程差 $\delta =$ ______。
:::

:::callout{kind=insight label="解析"}
两列相干波（频率相同、振动方向相同、相位差恒定）在空间某点相遇时：

- 相长干涉（振动加强）：$\delta = k\lambda$（$k = 0, \pm 1, \pm 2, \ldots$）
- 相消干涉（振动减弱）：$\delta = (2k+1)\frac{\lambda}{2}$（$k = 0, \pm 1, \pm 2, \ldots$）

相消干涉意味着两列波在该点的振动方向相反，合成振幅为两振幅之差。
:::

:::callout{kind=tip label="结论速记"}
相消干涉条件：波程差 $\delta = (2k+1)\frac{\lambda}{2}$（半波长的奇数倍）。相长干涉条件：$\delta = k\lambda$（波长的整数倍）。
:::

### 第15题

:::callout{kind=note label="题目"}
气体分子的平均自由程 $\bar{\lambda}$ 与分子直径 $d$、分子数密度 $n$ 的关系为 $\bar{\lambda} =$ ______。
:::

:::callout{kind=insight label="解析"}
在理想气体分子运动论中，分子平均自由程的公式为：

$$\bar{\lambda} = \frac{1}{\sqrt{2} \pi d^2 n}$$

其中：
- $d$ 为分子有效直径
- $n$ 为分子数密度（单位体积内分子数）
- $\sqrt{2}$ 因子来自分子相对运动的统计修正

物理含义：分子数密度越大或分子直径越大，分子碰撞越频繁，平均自由程越短。
:::

:::callout{kind=tip label="结论速记"}
气体分子平均自由程：$\bar{\lambda} = 1/(\sqrt{2}\pi d^2 n)$。分子数密度 $n$ 越大或分子直径 $d$ 越大，碰撞越频繁，平均自由程越短。
:::

### 第16题

:::callout{kind=note label="题目"}
自然光由折射率 $n_1$ 的介质射向折射率 $n_2$ 的介质，当入射角等于布儒斯特角 $i_0$ 时反射光为完全线偏振光，此时 $\tan i_0 =$ ______。

![布儒斯特角入射图](/physics/kaoqian-moni/images/sim-01-brewster.jpg)
:::

:::callout{kind=insight label="解析"}
布儒斯特定律：当入射角满足

$$\tan i_0 = \frac{n_2}{n_1}$$

时，反射光为完全线偏振光（振动方向垂直于入射面）。

此时反射光线与折射光线恰好垂直（$i_0 + r = 90°$）。布儒斯特角的存在是因为在特定角度下，平行于入射面振动的分量完全透射，反射光中只剩下垂直分量。

应用：激光器中的布儒斯特窗就是利用这一原理。
:::

:::callout{kind=tip label="结论速记"}
布儒斯特定律：$\tan i_0 = n_2/n_1$，入射角等于布儒斯特角时反射光为完全线偏振光。此时反射光线与折射光线垂直。
:::

### 第17题

:::callout{kind=note label="题目"}
动量为 $p$ 的微观粒子，其德布罗意波长 $\lambda =$ ______。
:::

:::callout{kind=insight label="解析"}
德布罗意关系式（de Broglie relation）：

$$\lambda = \frac{h}{p} = \frac{h}{mv}$$

其中 $h$ 为普朗克常量，$p$ 为粒子动量。

这是量子力学的基本假设之一：一切微观粒子都具有波粒二象性，其波动性由德布罗意波长描述。动量越大，波长越短；质量越大，波长越短，因此宏观物体的波动性无法观测。
:::

:::callout{kind=tip label="结论速记"}
德布罗意波长：$\lambda = h/p$，动量越大波长越短。宏观物体因质量大、动量大，德布罗意波长极短，波动性不可观测。
:::

### 第18题

:::callout{kind=note label="题目"}
X 射线管的管电压为 $U$，则其连续 X 射线谱的最短波长 $\lambda_{\min} =$ ______。
:::

:::callout{kind=insight label="解析"}
当高速电子撞击靶材时，电子动能全部转化为光子能量时产生最短波长：

$$eU = h\nu_{\max} = \frac{hc}{\lambda_{\min}}$$

解得：

$$\lambda_{\min} = \frac{hc}{eU}$$

其中 $h$ 为普朗克常量，$c$ 为光速，$e$ 为电子电荷量。

$\lambda_{\min}$ 与管电压 $U$ 成反比，电压越高，最短波长越短，X 射线穿透力越强。
:::

:::callout{kind=tip label="结论速记"}
X 射线连续谱最短波长：$\lambda_{\min} = hc/(eU)$，与管电压成反比。电压越高，最短波长越短，穿透力越强。
:::

## 三、简答题（每小题 5 分，共 10 分）

### 第21题

:::callout{kind=note label="题目"}
写出两列光波形成稳定干涉所必须满足的相干条件，并简要说明为什么两个独立的普通光源发出的光不能产生稳定干涉。

![比结合能曲线](/physics/kaoqian-moni/images/sim-01-binding-energy.png)
:::

:::callout{kind=insight label="解析"}
**相干条件**（三个条件缺一不可）：

1. **频率相同**：两列波的振动频率必须一致，否则相位差会随时间变化，无法形成稳定干涉图样。

2. **振动方向相同（或有平行分量）**：两列波的电矢量振动方向必须有平行分量，才能在相遇点进行矢量叠加。

3. **相位差恒定**：两列波在相遇点的相位差必须不随时间变化，这是形成稳定干涉图样的关键。

**普通光源不能产生稳定干涉的原因**：

普通光源发光来自大量原子的自发辐射。每个原子独立发光，各原子发光的初相位彼此独立且随机变化。两独立光源之间没有固定的相位关系，相位差随时间剧烈波动（变化周期约 $10^{-8}$ s），人眼无法分辨如此快速变化的干涉图样，因此观测不到稳定的干涉。

**解决方案**：将同一光源发出的光分成两束（分波前法如杨氏双缝，或分振幅法如薄膜干涉），使两束光具有固定的相位关系，才能产生稳定干涉。
:::

:::callout{kind=tip label="结论速记"}
相干三条件：频率相同、振动方向有平行分量、相位差恒定。独立光源因原子自发辐射的随机性，相位差不恒定，故不能产生稳定干涉。常用分波前法或分振幅法将同一光源分为两束相干光。
:::

### 第22题

:::callout{kind=note label="题目"}
什么是原子核的比结合能？说明为什么重核裂变和轻核聚变都能够释放核能。
:::

:::callout{kind=insight label="解析"}
**比结合能的定义**：

$$\text{比结合能} = \frac{\text{结合能}}{\text{核子数}} = \frac{\Delta m \cdot c^2}{A}$$

比结合能表示将原子核拆散成自由核子时，平均每个核子需要提供的能量。比结合能越大，原子核越稳定。

**比结合能曲线**：

比结合能随质量数 $A$ 的变化曲线呈"山峰"形：
- 轻核（$A < 30$）：比结合能随 $A$ 增大而增大
- 中等质量核（$A \approx 56$，铁附近）：比结合能达到最大值（约 8.8 MeV/核子）
- 重核（$A > 200$）：比结合能随 $A$ 增大而缓慢减小

**能量释放的原理**：

- **重核裂变**：重核（如 ${}^{235}\text{U}$）分裂成中等质量的碎片核，产物的比结合能大于反应前，每个核子释放约 1 MeV 能量。
- **轻核聚变**：轻核（如 ${}^2\text{H}$、${}^3\text{H}$）聚合成较重的核，产物的比结合能也大于反应前，每个核子释放的能量更大（约 3~4 MeV/核子）。

本质都是：反应后产物的比结合能增大，反应前后存在质量亏损 $\Delta m$，根据 $\Delta E = \Delta m \cdot c^2$ 释放能量。
:::

:::callout{kind=tip label="结论速记"}
比结合能 = 结合能/核子数 = $\Delta mc^2/A$，反映原子核稳定性。铁附近（$A\approx56$）比结合能最大。重核裂变和轻核聚变都使产物比结合能增大，差额以核能形式释放。聚变单位质量释放能量比裂变更大。
:::

## 四、计算题（每小题 8 分，共 40 分）

### 第23题

:::callout{kind=note label="题目"}
**（力学）** 质量 $m = 2.0\;\text{kg}$ 的滑块在光滑水平面上以初速度 $v_0 = 6.0\;\text{m/s}$ 滑行，受到阻力 $f = -kv$（$k = 4.0\;\text{kg/s}$）。

(1) 求速度随时间的变化关系 $v(t)$；

(2) 求滑块从开始运动到停止的总路程 $x$。
:::

:::callout{kind=insight label="解析"}
**(1) 求 $v(t)$**

由牛顿第二定律：

$$m\frac{dv}{dt} = -kv$$

分离变量：

$$\frac{dv}{v} = -\frac{k}{m}dt$$

两边积分（$t=0$ 时 $v=v_0$）：

$$\int_{v_0}^{v} \frac{dv'}{v'} = -\frac{k}{m}\int_0^t dt'$$

$$\ln\frac{v}{v_0} = -\frac{k}{m}t$$

$$\boxed{v(t) = v_0 \, e^{-\frac{k}{m}t} = 6.0 \, e^{-2.0t} \;\text{(m/s)}}$$

**(2) 求总路程 $x$**

$$x = \int_0^{\infty} v(t)\,dt = v_0 \int_0^{\infty} e^{-\frac{k}{m}t}\,dt = v_0 \cdot \frac{m}{k}$$

$$\boxed{x = \frac{mv_0}{k} = \frac{2.0 \times 6.0}{4.0} = 3.0 \;\text{m}}$$
:::

:::callout{kind=tip label="结论速记"}
阻力与速度成正比（$f = -kv$）的减速运动：速度按指数衰减 $v = v_0 e^{-kt/m}$，总路程 $x = mv_0/k$，有限值。与匀减速运动不同，理论上永远不会完全停止，但总路程有限。
:::

### 第24题

:::callout{kind=note label="题目"}
**（流体力学）** 水箱水面高度 $H = 1.25\;\text{m}$，侧壁距箱底 $h = 0.45\;\text{m}$ 处开一小孔。

(1) 求小孔处水的出流速度 $v$；

(2) 求水从小孔射出后落到地面的水平射程 $s$；

(3) 小孔开在距箱底多高时水平射程最大？最大射程是多少？
:::

:::callout{kind=insight label="解析"}
**(1) 出流速度 $v$**

水面（流速近似为零）和小孔处列伯努利方程，小孔距液面深度为 $H - h$：

$$v = \sqrt{2g(H - h)} = \sqrt{2 \times 9.8 \times (1.25 - 0.45)} = \sqrt{2 \times 9.8 \times 0.80} = \sqrt{15.68}$$

$$\boxed{v \approx 3.96 \;\text{m/s}}$$

**(2) 水平射程 $s$**

水从小孔射出后做平抛运动，小孔距地面高度为 $h$：

$$s = v \cdot t = v \cdot \sqrt{\frac{2h}{g}} = 3.96 \times \sqrt{\frac{2 \times 0.45}{9.8}} = 3.96 \times \sqrt{0.0918}$$

$$\boxed{s \approx 3.96 \times 0.303 \approx 1.20 \;\text{m}}$$

**(3) 最大射程**

$$s = \sqrt{2g(H-h)} \cdot \sqrt{\frac{2h}{g}} = 2\sqrt{h(H-h)}$$

令 $s$ 对 $h$ 求导并令其为零：

$$\frac{ds}{dh} = 2 \cdot \frac{H - 2h}{2\sqrt{h(H-h)}} = 0$$

解得 $h = \frac{H}{2}$，即小孔开在水箱正中间时射程最大：

$$\boxed{h = \frac{H}{2} = 0.625 \;\text{m}, \quad s_{\max} = H = 1.25 \;\text{m}}$$
:::

:::callout{kind=tip label="结论速记"}
托里拆利定理出流速度 $v = \sqrt{2g(H-h)}$。平抛射程 $s = 2\sqrt{h(H-h)}$，当 $h = H/2$（小孔在水面与地面正中间）时射程最大，$s_{\max} = H$。
:::

### 第25题

:::callout{kind=note label="题目"}
**（静电场）** 无限长均匀带电圆柱体，半径 $R$，体电荷密度 $\rho$。求距轴线 $r$ 处的电场强度 $E$。

(1) $r > R$（圆柱体外）；

(2) $r < R$（圆柱体内）。

![带电圆柱体截面](/physics/kaoqian-moni/images/sim-01-cylinder.jpg)
:::

:::callout{kind=insight label="解析"}
由于圆柱体无限长且均匀带电，电场具有轴对称性，方向沿径向。取半径为 $r$、长度为 $l$ 的同轴圆柱面为高斯面：

$$\oint \vec{E} \cdot d\vec{A} = E \cdot 2\pi r l = \frac{Q_{内}}{\varepsilon_0}$$

**(1) $r > R$（圆柱体外）**

高斯面内包围的全部电荷：

$$Q_{内} = \rho \cdot \pi R^2 l$$

$$E \cdot 2\pi r l = \frac{\rho \pi R^2 l}{\varepsilon_0}$$

$$\boxed{E = \frac{\rho R^2}{2\varepsilon_0 r}}$$

与 $1/r$ 成正比，相当于线电荷密度 $\lambda = \rho \pi R^2$ 的无限长直导线产生的电场。

**(2) $r < R$（圆柱体内）**

高斯面内包围的电荷：

$$Q_{内} = \rho \cdot \pi r^2 l$$

$$E \cdot 2\pi r l = \frac{\rho \pi r^2 l}{\varepsilon_0}$$

$$\boxed{E = \frac{\rho r}{2\varepsilon_0}}$$

与 $r$ 成正比，圆柱体内部电场从轴线处的零值线性增加到表面的最大值。
:::

:::callout{kind=tip label="结论速记"}
无限长均匀带电圆柱体：外部 $E = \rho R^2/(2\varepsilon_0 r)$，与 $1/r$ 成正比（等效线电荷）；内部 $E = \rho r/(2\varepsilon_0)$，与 $r$ 成正比（从轴线零值线性增至表面）。高斯定理求解的关键是利用对称性。
:::

### 第26题

:::callout{kind=note label="题目"}
**（稳恒磁场）** 载流直导线中部弯成半径 $R = 0.10\;\text{m}$ 的半圆环，电流 $I = 2.0\;\text{A}$。求半圆环圆心处的磁感应强度 $B$ 的大小和方向。

![半圆环电流](/physics/kaoqian-moni/images/sim-01-semicircle.jpg)
:::

:::callout{kind=insight label="解析"}
圆心处的磁感应强度由两部分贡献：半圆环弧和两段直导线。

**半圆环弧的贡献**：

完整圆环在圆心处的磁场为 $B_{圆} = \mu_0 I/(2R)$，半圆环为其一半：

$$B_{弧} = \frac{\mu_0 I}{4R}$$

方向：由右手螺旋定则，四指沿电流方向弯曲，拇指指向为磁场方向。若电流沿逆时针方向，磁场垂直纸面向外。

**两段直导线的贡献**：

每段直导线的延长线通过圆心，根据毕奥-萨伐尔定律，直导线在其延长线上产生的磁场为零。因此两段直导线在圆心处的贡献为零。

**总磁场**：

$$\boxed{B = \frac{\mu_0 I}{4R} = \frac{4\pi \times 10^{-7} \times 2.0}{4 \times 0.10} = \frac{8\pi \times 10^{-7}}{0.40} = 2\pi \times 10^{-6} \approx 6.28 \times 10^{-6} \;\text{T}}$$

方向：**垂直纸面向外**（假设电流沿逆时针方向弯曲）。
:::

:::callout{kind=tip label="结论速记"}
半圆环圆心磁场：$B = \mu_0 I/(4R)$，方向由右手螺旋定则判断。直导线延长线通过场点时贡献为零。完整圆环为 $B = \mu_0 I/(2R)$。
:::

### 第27题

:::callout{kind=note label="题目"}
**（几何光学）** 共轴两薄透镜，$f_1 = 10\;\text{cm}$，$f_2 = 20\;\text{cm}$，两透镜间距 $d = 40\;\text{cm}$。在第一透镜左侧 $u_1 = 15\;\text{cm}$ 处放一物体。

(1) 求第一透镜所成像的位置 $v_1$；

(2) 求第二透镜所成最终像的位置 $v_2$ 和总放大率 $m$。

![共轴薄透镜组](/physics/kaoqian-moni/images/sim-01-lens.png)
:::

:::callout{kind=insight label="解析"}
**(1) 第一透镜成像**

透镜公式：

$$\frac{1}{u_1} + \frac{1}{v_1} = \frac{1}{f_1}$$

代入 $u_1 = 15\;\text{cm}$，$f_1 = 10\;\text{cm}$：

$$\frac{1}{v_1} = \frac{1}{10} - \frac{1}{15} = \frac{3-2}{30} = \frac{1}{30}$$

$$\boxed{v_1 = 30 \;\text{cm}}$$

$v_1 > 0$，为**实像**，在第一透镜右侧 $30\;\text{cm}$ 处。

**(2) 第二透镜成像**

第一透镜所成的像作为第二透镜的物。两透镜间距 $d = 40\;\text{cm}$，第一像距第一透镜 $30\;\text{cm}$，因此该像距第二透镜：

$$u_2 = d - v_1 = 40 - 30 = 10 \;\text{cm}$$

$u_2 > 0$，为实物。由透镜公式：

$$\frac{1}{v_2} = \frac{1}{f_2} - \frac{1}{u_2} = \frac{1}{20} - \frac{1}{10} = \frac{1-2}{20} = -\frac{1}{20}$$

$$\boxed{v_2 = -20 \;\text{cm}}$$

$v_2 < 0$，为**虚像**，在第二透镜左侧 $20\;\text{cm}$ 处（即两透镜之间，距第一透镜 $20\;\text{cm}$ 处）。

**总放大率**：

$$m = m_1 \times m_2 = \left(-\frac{v_1}{u_1}\right) \times \left(-\frac{v_2}{u_2}\right) = \left(-\frac{30}{15}\right) \times \left(-\frac{-20}{10}\right) = (-2) \times (2) = \boxed{-4}$$

$|m| = 4$，最终像是放大像；$m < 0$，最终像是倒立的。
:::

:::callout{kind=tip label="结论速记"}
共轴透镜组逐次成像法：前一透镜的像作为后一透镜的物。物距 $u_2 = d - v_1$（注意正负号）。总放大率 $m = m_1 \times m_2$。透镜公式：$1/u + 1/v = 1/f$，实像 $v > 0$，虚像 $v < 0$。
:::
