# 3.1 简谐振动

## 什么是振动


::figure{src="/images/physics/3-大学物理学习指导-第三章-振动/47081837e8a3ef3923e7e9ba5836538ed174eba2441a508196a0e968b4107cf6.jpg" alt="" caption=""}

:::definition{label=振动}
一物理量在某一定值附近**周期性变化**的现象称为**振动**（vibration）。最基本、最简单、也最重要的是**简谐振动**（simple harmonic motion, SHM）。
:::

:::insight{label=简谐振动的地位}
弹簧振子、单摆（小角度）、音叉、心电图中的周期性信号，都可近似或分解为简谐振动。掌握 SHM，等于掌握了理解一切复杂振动的基础"积木块"。
:::

## 简谐振动的动力学与运动学特征

以**弹簧振子**为例：质量为 $m$ 的物体连在劲度系数 $k$ 的弹簧上。

**动力学特征**：物体在**回复力** $F = -kx$ 作用下作来回运动，力始终指向平衡位置，大小与位移成正比。

**运动学特征**：加速度 $a = -\omega^2 x$，与位移成正比、方向相反。

**微分方程**：

$$
\frac{d^2 x}{dt^2} + \omega^2 x = 0, \quad \omega = \sqrt{\frac{k}{m}}
$$

**简谐振动方程**（位移随时间的变化）：

$$
x = A\cos(\omega t + \varphi)
$$

**频率的物理意义**：频率 $\nu$（或角频率 $\omega$）描述振动**快慢**。同样的振幅 $A$，频率越高，单位时间内完成的振动次数越多，能量 $\propto A^2$ 与频率无关，但速度最大值 $v_{\max} = \omega A$ 随 $\omega$ 增大——高频振动"更急"。弹簧振子 $\omega = \sqrt{k/m}$：劲度系数大、质量小 → 振动快；这是设计振动仪器（如加速度计、地震仪）时的基本参数关系。

**简谐振动的判定**：若物体所受回复力可写成 $F = -kx$（或加速度 $a = -\omega^2 x$），则运动必为 SHM。小角度单摆、某些电路中的 LC 振荡，在适当近似下均满足这一形式。判定步骤：① 找平衡位置；② 求位移 $x$ 处的回复力；③ 检验是否正比于 $-x$。

**速度与加速度的再推导**：对 $x = A\cos(\omega t + \varphi)$ 求导，$v = -\omega A\sin(\omega t + \varphi)$，最大速率 $v_{\max} = \omega A$ 出现在平衡位置；$a = -\omega^2 A\cos(\omega t + \varphi) = -\omega^2 x$，最大加速度 $|a|_{\max} = \omega^2 A$ 出现在端点。端点处 $v=0$ 但 $|a|$ 最大——这是"变加速"运动的典型特征：加速度随位置变，而非随时间线性变。

## 简谐振动的特征量

:::definition{label=振幅}
**振幅** $A$：振动物体离开平衡位置的最大位移的绝对值，决定振动幅度。
:::

:::definition{label=周期、频率与角频率}
- **周期** $T$：完成一次全振动所需时间；
- **频率** $\nu$：单位时间内全振动次数，$\nu = 1/T$；
- **角频率** $\omega$：$2\pi$ 秒内振动次数，$\omega = 2\pi\nu = 2\pi/T$。

弹簧振子：$\omega = \sqrt{k/m}$，$T = 2\pi\sqrt{m/k}$，$\nu = \dfrac{1}{2\pi}\sqrt{k/m}$。
:::

:::definition{label=相位与初相位}
**相位** $\omega t + \varphi$ 决定 $t$ 时刻的振动状态 $(x, v, a)$。$t = 0$ 时的 $\varphi$ 叫**初相位**（initial phase）。两振动相位之差 $\Delta\varphi = \varphi_2 - \varphi_1$：

- $\Delta\varphi = 2k\pi$：**同相**；
- $\Delta\varphi = (2k+1)\pi$：**反相**。
:::

**位移、速度、加速度的相位关系**：

| $\omega t + \varphi$ | $x$ | $v$ | $a$ |
|----------------------|-----|-----|-----|
| $\pi/2$ | 0 | $-\omega A$ | 0 |
| $3\pi/2$ | 0 | $\omega A$ | 0 |
| 0 | $A$ | 0 | $-\omega^2 A$ |
| $\pi$ | $-A$ | 0 | $\omega^2 A$ |

**速度超前位移 $\pi/2$，但落后加速度 $\pi/2$**；位移与加速度**反相**。

## 初始条件决定 $A$ 与 $\varphi$

由 $t = 0$ 时的 $x_0$、$v_0$ 可确定 $A$、$\varphi$：

$$
A = \sqrt{x_0^2 + \left(\frac{v_0}{\omega}\right)^2}, \quad \tan\varphi = -\frac{v_0}{\omega x_0}
$$

:::example{label=四种初始条件的初相位}
设向上为正，$x = A\cos(\omega t + \varphi)$。

1. 平衡位置向上：$x_0=0$，$v_0>0$ → $\varphi = -\pi/2$ 或 $3\pi/2$；
2. 平衡位置向下：$x_0=0$，$v_0<0$ → $\varphi = \pi/2$；
3. $x_0 = A/2$ 向上：$\varphi = -\pi/3$；
4. $x_0 = A/2$ 向下：$\varphi = \pi/3$。

竖直振动，$t=0$ 在 $A/2$ 处向下，$y = A\cos(\omega t + \varphi)$ 中 $\varphi = 2\pi/3$（选 B）。
:::

:::example{label=由条件写振动方程}
已知 $A = 10$ cm，$T = 2$ s，$t = 0$ 时 $x = -5$ cm 且向 $x$ 负向运动。$\omega = \pi$ rad/s。由旋转矢量法得 $\varphi = 2\pi/3$，故

$$
x = 10\cos\left(\pi t + \frac{2\pi}{3}\right) \text{ cm}
$$

当 $x = 5$ cm 且向正向运动时，可求该点 $v$、$a$ 及回到平衡位置的最短时间（相位扫过 $\pi/3$ 对应 $T/6 = 1/3$ s）。
:::

## 旋转矢量法

:::definition{label=旋转矢量法}
作矢量 $\vec{A}$，长度 $A$，以角速度 $\omega$ 逆时针匀速绕原点转。$t$ 时刻矢量与 $x$ 轴夹角为 $\omega t + \varphi$。**矢量在 $x$ 轴上的投影** $x = A\cos(\omega t + \varphi)$ 即为简谐振动；投影的速度、加速度对应 $v$、$a$。
:::

相位 $\omega t + \varphi$ 直观表示运动状态：$\varphi$ 为初相位；相位增加，质点沿 $x$ 轴往复。旋转矢量法便于**合成**多个同频振动、**求相位差**。

:::example{label=由 x-t 曲线写方程并求相差}
两振动周期相同，从曲线读 $A$、$T$、$t=0$ 时的 $x$ 与运动方向，写出 $x_1$、$x_2$，相位差 $\Delta\varphi = \varphi_2 - \varphi_1$ 决定超前滞后关系。
:::

## 简谐振动的能量

以弹簧振子为例，由 $m\omega^2 = k$：

$$
E_k = \frac{1}{2}mv^2 = \frac{1}{2}kA^2\sin^2(\omega t + \varphi)
$$

$$
E_p = \frac{1}{2}kx^2 = \frac{1}{2}kA^2\cos^2(\omega t + \varphi)
$$

$$
E = E_k + E_p = \frac{1}{2}kA^2 = \frac{1}{2}m\omega^2 A^2 = \text{常数}
$$

**要点**：

1. 动能、势能随时间变化，**总机械能守恒**；
2. $E \propto A^2$：振幅反映振动强度与总能量；
3. 设 $\varphi = 0$，最大位移处势能最大、动能为零；平衡位置反之——能量在动能与势能间转化。

这些结论**适用于一切简谐振动**。

**相位与状态的对应关系**（旋转矢量法的文字版）：当 $\omega t + \varphi = 0$ 时，质点在正向位移极大处，速度为零，下一时刻将向 $x$ 轴负方向运动；当 $\omega t + \varphi = \pi/2$ 时，质点正越过原点，以最大速率向负方向运动；当 $\omega t + \varphi = \pi$ 时，在负向位移极大处；当 $\omega t + \varphi = 3\pi/2$ 时，再次过原点，以最大速率向正方向运动。熟记这四个相位点，写方程、求速度加速度时可直接读出结果，不必每次求导。

**由 $x$-$t$ 曲线读初相位**：曲线在 $t=0$ 的位移给出 $x_0$，曲线斜率 $\dfrac{dx}{dt}\big|_{t=0}$ 给出 $v_0$ 的符号与相对大小；结合 $x = A\cos(\omega t + \varphi)$，用 $x_0 = A\cos\varphi$、$v_0 = -\omega A\sin\varphi$ 可唯一确定 $\varphi$（在 $0 \sim 2\pi$ 内）。若曲线在 $t=0$ 从最大位移开始向平衡位置运动，则 $\varphi = 0$（向负向）或 $\varphi = \pi$（向正向），需看参考方向。

:::example{label=小角度单摆}
单摆摆角 $\theta$ 很小时，$F = -mg\sin\theta \approx -mg\theta/L = -(mg/L)s$，等效 $k = mg/L$，振动方程 $s = A\cos(\omega t + \varphi)$，$\omega = \sqrt{g/L}$。动能、势能、总能量形式与弹簧振子相同。摆长约 1 m 的立钟，$T \approx 2$ s。

**立钟的设计**：老式立钟摆长约 1 m，$\omega = \sqrt{g/L} \approx \pi$ rad/s，$T = 2\pi/\omega \approx 2$ s，与日常"嘀嗒"节奏一致。摆长随温度略有热胀冷缩，精密钟用补偿摆（不同金属组合）减小 $L$ 漂移。**能量转化过程**：设 $\varphi = 0$，$t=0$ 在 $x=A$，势能最大、动能为零；过平衡位置时 $E_k$ 最大、$E_p$ 最小；到 $x=-A$ 又全为势能。一个周期内完成两次动能势能的完全交换——这与匀速圆周投影的 $v$、$a$ 相位关系完全对应。
:::

:::pitfall{label=简谐振动易错点}
1. 运动至正方向端点：$x = A$，$v = 0$，$a = -A\omega^2$（选 D），不是 $a = +A\omega^2$；
2. 简谐振动是**变加速**运动（选 D），不是匀速或匀加速；
3. 相位与初相位的单位是**弧度**，写方程时注意 $v$ 的正负与参考方向一致。
4. **旋转矢量法**画矢量时，逆时针旋转对应 $x = A\cos(\omega t + \varphi)$ 的标准形式；若改用 $\sin$ 形式，初相差 $\pi/2$，勿混用。
:::

:::note{label=本节小结}
- **SHM 方程** $x = A\cos(\omega t + \varphi)$，$\omega = \sqrt{k/m}$，$T = 2\pi/\omega$；
- **振幅** $A$、**相位** $\omega t + \varphi$、**初相位** $\varphi$ 由初始条件确定；
- **旋转矢量法**：投影得 $x$，便于合成与相位分析；
- **能量** $E = \frac{1}{2}kA^2$ 守恒，$E_k$ 与 $E_p$ 相互转化；
- 小角度单摆、弹簧振子均为典型 SHM。

下一节讨论多个简谐振动的**合成**——同向同频、拍现象、互相垂直及李萨如图形。

**学习建议**：先熟练 $x$、$v$、$a$ 的三角函数形式与相位关系，再练旋转矢量作图；初相位题务必画 $t=0$ 时刻的矢量在 $x$ 轴上的投影方向，避免只凭代数符号猜 $\varphi$。
:::
