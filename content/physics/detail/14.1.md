## X 射线的产生与性质

:::insight{label=电子"急刹车"打出 X 射线}
X 射线是波长约 0.005–10 nm 的电磁波，能量高、穿透力强。医用 X 射线管中，热阴极发射的电子经高压加速轰击钨靶，约 1% 动能转化为 X 射线（轫致辐射），其余为热能——理解产生机制，才能理解强度、硬度与谱线的区别。
:::

:::definition{label=X射线}
**X 射线**：伦琴 1895 年发现，波长约 1–10 nm（医学常用范围），能量高、穿透力强，具有波粒二象性：干涉、衍射、偏振（波动性）；光电效应、康普顿效应（粒子性）。X 射线不带电，不受电磁场偏转。
:::

## X 射线产生装置

产生 X 射线需：（1）**高速电子流**；（2）**阳极靶**阻止电子运动。

**X 射线管结构**：
- **阴极**：灯丝发射电子，钼制聚焦杯聚焦并收集二次电子
- **阳极（靶）**：钨或钼（高 $Z$、耐高温）；固定式用于小管电流；旋转式高速旋转散热，用于大功率
- **管电压**：阴阳极间几十至几百 kV 直流高压
- **玻璃外壳**：高真空（$< 10^{-4}$ Pa）

:::definition{label=实际焦点与有效焦点}
**实际焦点**：电子轰击阳极靶的面积。**有效焦点**：实际焦点在垂直于管轴方向的投影面积。诊断用小焦点，治疗用大焦点。有效焦点 = 实际焦点 × $\sin\theta$（靶角 15°–19°）。
:::

高速电子能量损失：**碰撞损失**（~99%，变热）与**辐射损失**（~1%，产生 X 射线）。

## X 射线的强度与硬度

:::definition{label=强度与硬度}
**强度** $I$：单位时间垂直通过单位面积的辐射能量（W/m²），常用管电流 mA（毫安率）表示。总辐射能量 $\propto$ 管电流 × 时间（mA·s）。

影响因素：靶原子序数越高、阴极温度越高（管电流越大）、管电压越高 → 强度越大。

**硬度**：X 射线的**贯穿本领**，只取决于单个光子能量（波长），与光子数无关。用管电压 kV 表示（千伏率）。
:::

| 名称 | 管电压(kV) | 主要用途 |
|------|-----------|---------|
| 极软 | 40 以下 | 软组织 |
| 软 | 40–80 | 透视、摄影 |
| 硬 | 80–120 | 透视、摄影 |
| 极硬 | 120–300 | 深部组织 |

## X 射线谱

**连续 X 射线（轫致辐射）**：电子受核电场急减速，辐射光子，波长连续分布。短波极限：

$$
\lambda_{\min} = \frac{hc}{eU} = \frac{1.24}{U(\mathrm{kV})}\,\mathrm{nm}
$$

管电压升高 → 各波长强度增强，短波极限向短波移动。靶 $Z$ 越大，连续谱强度越大。

**标识 X 射线**：当 $U > 70$ kV，在内壳层空位填充时产生。波长仅取决于靶元素（K 系、L 系），与管电压无关（需超过激发电压）。

## X 射线的衰减

与物质相互作用三种主要形式（低能区）：**光电效应**、**康普顿散射**、**电子对效应**（高能）。

:::theorem{label=朗伯定律}
单色 X 射线穿过物质，强度指数衰减：

$$
I = I_0 e^{-\mu x}
$$

$\mu$ 为**线性衰减系数**（cm⁻¹）。**质量衰减系数** $\mu_m = \mu/\rho$（cm²/g）。

**半价层** $x_{1/2}$：强度减为一半时的厚度，$x_{1/2} = (\ln 2)/\mu$。经 $n$ 个半价层，$I = I_0/2^n$（12.5% 需 3 个半价层）。
:::

::plot{fn="exp(-0.5*x)" xmin=0 xmax=10 ymin=0 ymax=1.1 label="朗伯定律：I/I₀ = e^(-μx)"}

**滤过与硬化**：铝/铜滤波板吸收软 X 射线，平均硬度提高（线质变硬）。产生硬 X 射线：高管电压、高 $Z$ 靶、厚滤波板；软 X 射线：低管电压、钼靶、铍窗。

## X 射线衍射

1912 年劳厄证实 X 射线波动性。晶体原子间距 ~$10^{-10}$ m，可作立体光栅。

:::theorem{label=布喇格定律}
$$
2d\sin\theta = k\lambda, \qquad k = 1,2,3,\ldots
$$

$d$ 为晶面间距，$\theta$ 为掠射角。满足布喇格条件的晶面方向发生相长干涉——用于测晶格常数、DNA 双螺旋结构（Watson & Crick）等。
:::

:::example{label=布喇格衍射求晶格常数}
KCl 晶体二级布喇格角 $\theta = 30°$，$\lambda = 1.57$ Å，求晶格常数。

**解**：$d = k\lambda/(2\sin\theta) = 2 \times 1.57/(2 \times 0.5) = 3.14$ Å（B）。
:::

## X 射线的基本效应与医学应用概要

1. **电离作用**：测量、治疗、防护
2. **荧光作用**：透视、增感屏
3. **光化学（感光）作用**：X 射线摄影
4. **穿透本领**：透视、摄影、CT
5. **生物效应**：诊断与放疗的基础

**普通透视**：实时观察，剂量较大，无客观记录。**摄影**：胶片感光，剂量较小，可存档（黑白与荧光屏相反）。

**数字化成像**：DF、DR、CR；**DSA** 减影血管造影（时间/能量减影）；**X-CT** 体层重建，CT 值以水为 0 Hu，气 -1000 Hu，骨 +1000 Hu。

:::pitfall{label=易错辨析}
- X 射线波长比可见光**短**（D），比 γ 射线**长**。
- 连续谱最短波长只取决于**管电压**（B），与靶材料无关。
- **贯穿本领**取决于硬度/波长（B），非强度。
- **强度**与管电压、管电流均有关（D）。
- 连续 X 射线由**轫致辐射**产生（A）。
:::

## X 射线与物质的相互作用（详）

**光电效应**：光子能量全部交给内层电子，光子消失；概率 $\propto Z^4/\lambda^3$，低能、高 $Z$ 物质显著。

**康普顿散射**：光子与外层（近似自由）电子碰撞，波长变长；中等能量区主导。

**电子对效应**：$h\nu > 2m_0 c^2 = 1.022\,\mathrm{MeV}$，光子在近核场产生 e⁺e⁻ 对；高能区主导。

质量衰减系数 $\mu_m$ 与 $\lambda$、$Z$ 有关：波长越长越易吸收；$Z$ 越大吸收越强。以上公式仅适用于**单色** X 射线；诊断 X 射线为连续谱，需按谱加权平均。

## X 射线医学应用

**透视**：实时动态观察，剂量较大，无永久记录；荧光屏由氰化钡、硫化锌等荧光物质组成，铅玻璃防护。

**摄影**：胶片感光，剂量较小，可存档；影像黑白与荧光屏明暗**相反**（负片）。

**体层摄影、造影**：造影剂（碘、钡）增强对比度。

**数字化**：CR（成像板）、DR（平板探测器）、DSA（数字减影血管造影，时间减影消除骨骼软组织，显示血管）。

**X-CT**：多角度投影 + 重建算法（Radon 变换）得断层图像。CT 值 $= 1000 \times (\mu - \mu_\text{水})/(\mu_\text{水} - \mu_\text{气})$ Hu。螺旋 CT、多层 CT 提高扫描速度与分辨率；64 层 CT 空间分辨 ~0.4 mm。

:::example{label=半价层计算}
强度减至 12.5% = $(1/2)^3$，需 **3 个半价层**。质量半价层 $x_{m,1/2} = (\ln 2)/\mu_m$。若 $\mu_m = 0.05\,\mathrm{cm^2/g}$，则 $x_{m,1/2} = 0.693/0.05 = 13.86\,\mathrm{g/cm^2}$（A）。
:::

**X-CT**：多角度投影 + 重建算法得断层图像。CT 值以水为 0 Hu，空气 -1000 Hu，密骨 +1000 Hu。螺旋 CT、多层 CT 提高扫描速度；64 层 CT 空间分辨约 0.4 mm，可用于肺栓塞等血管疾病的无创诊断（准确率可达 90% 以上）。

:::note{label=本节小结}
- X 射线管：热阴极 + 高压 + 高 $Z$ 靶 + 真空。
- **强度** ∝ mA·s；**硬度**由 kV/波长决定。
- 谱线：连续（轫致辐射）+ 标识（内层跃迁）。
- **朗伯定律** $I = I_0 e^{-\mu x}$；布喇格定律 $2d\sin\theta = k\lambda$。
- 透视、摄影、CT、DSA——医学影像的物理根基。

:::
