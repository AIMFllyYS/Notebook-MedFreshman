平衡态下，气体分子速率并非单一值，而是按一定概率分布在 $0 \sim \infty$ 之间。

:::theorem{label=麦克斯韦速率分布律}
温度为 $T$，总分子数 $N$，分子质量 $m$，速率在 $v \sim v+dv$ 内的分子数：

$$
dN_v = N f(v)\,dv
$$

$f(v)$ 为**麦克斯韦速率分布函数**，表示速率在 $v$ 附近**单位速率区间**内的分子数占总分子数的**百分比**。
:::

**归一化条件**：

$$
\int_0^{\infty} f(v)\,dv = 1
$$

曲线下的面积 $f(v)dv$ 表示 $v \sim v+dv$ 区间内分子所占比例。

### 分布特点

**（1）最概然速率** $v_p$：与 $f(v)$ 极大值对应的速率。

- 对大量分子：等宽速率间隔中，$v_p$ 附近分子数最多；
- 对单个分子：速率在 $v_p$ 附近出现的**几率最大**；
- 速率过大或过小的分子数相对较少，中等速率居多。

**（2）温度升高**：$v_p$ 增大，曲线**变平坦**并向高速区扩展，"面积"仍为 1——分子运动更剧烈。

**（3）温度一定**：分子质量越小，$v_p$ 越大（如轻分子 $H_2$ 比 $O_2$ 的 $v_p$ 大）。

### 三种特征速率

| 速率 | 公式 | 用途 |
|------|------|------|
| **最概然速率** $v_p$ | $\sqrt{2kT/m}$ | 讨论速率分布 |
| **平均速率** $\bar{v}$ | $\sqrt{8kT/(\pi m)}$ | 讨论分子碰撞频率 |
| **方均根速率** $v_{\text{rms}}$ | $\sqrt{3kT/m}$ | 讨论平均平动动能 |

三者关系：$v_p : \bar{v} : v_{\text{rms}} = \sqrt{2} : \sqrt{8/\pi} : \sqrt{3}$。

由 $v_{\text{rms}}$ 得 $\overline{E_k} = \frac{1}{2}m\overline{v^2} = \frac{3}{2}kT$，与经典推导一致。

:::example{label=分布函数表达式的物理意义}
- $f(v)\,dv$：$v \sim v+dv$ 区间内分子数占总数的比例，或该区间的分子数 $dN_v$；
- $\displaystyle\int_{v_1}^{v_2} f(v)\,dv$：$v_1 \sim v_2$ 区间内分子数比例；
- $\displaystyle\int_{v_1}^{v_2} N f(v)\,dv$：该区间分子数；
- $\displaystyle\int_0^{\infty} \frac{1}{2}mv^2 f(v)\,dv$：全速率区间分子的平均平动动能（有意义）；
- 单独 $f(v)$ 或 $\displaystyle\int f(v)\,dv$（无区间）在某些语境下无直接计数意义——要看是否乘以 $N$ 或 $dv$。
:::

## 碰撞频率与平均自由程

平衡态速率分布靠分子间**不断碰撞**维持；碰撞交换动量与能量，使系统从非平衡态趋于平衡。

:::definition{label=平均自由程与碰撞频率}
**平均自由程** $\lambda$：分子两次连续碰撞之间自由路程的平均值。

**平均碰撞频率** $z$：单位时间内一个分子与其他分子碰撞的平均次数。

二者关系 $\bar{v} = z\lambda$。标准状态下氮气 $\lambda \sim 10^{-7}$ m 量级，$d$ 为分子有效直径。
:::

## 玻耳兹曼能量分布定律

:::theorem{label=玻耳兹曼分布定律}
在重力场中，分子势能为 $E_p$，该处分子数密度：

$$
n = n_0 \exp\left(-\frac{E_p}{kT}\right)
$$

$n_0$ 为 $E_p = 0$ 处的数密度。重力场中 $E_p = mgh$：

$$
n = n_0 e^{-mgh/(kT)}
$$

压强随高度增加而减小，可推**大气压与海拔**关系。对流层温度不均匀时，若 $T = T_0 - \alpha y$，结合 $dp = -\rho g\,dy$ 与理想气体状态方程，可证：

$$
p = p_0\left(1 - \frac{\alpha y}{T_0}\right)^{\frac{\mu g}{R\alpha}}
$$

$\alpha = 6.0 \times 10^{-3}$ °C/m 为温度递减率。
:::

:::example{label=篮球压强随温度变化}
室温 0 °C 时打气至 1.5 atm，比赛时 30 °C。体积近似不变，$p \propto T$，$T_1 = 273$ K，$T_2 = 303$ K，

$$
p_2 = p_1 \frac{T_2}{T_1} = 1.5 \times \frac{303}{273} \approx 1.66\,p_0
$$

（选 B：约 1.66 $p_0$）。
:::

## 液体的表面现象

### 表面张力与表面能

液体表面层厚度约 $10^{-9}$ m（分子力有效作用半径）。表面层分子受上方气体分子引力小，受液体内部分子引力大，合力**指向液体内部**，使液面有**收缩趋势**。

:::definition{label=表面张力}
**表面张力** $\alpha$：液体表面层分子相互吸引使表面收缩的力。作用于液面任意分界线，大小 $F = \alpha L$，方向垂直于分界线、与液面相切。$\alpha$ 与液体种类、温度有关（$T$ 升高 $\alpha$ 减小）。
:::

:::theorem{label=表面张力与表面能}
增加液体表面积 $\Delta S$ 时，外力做功 $\Delta A = \alpha \Delta S = \Delta E_p$。**表面张力系数** $\alpha$ 在数值上等于增加单位表面积所需的表面能（或外力做的功）。

体积相同物体中，**球体表面积最小**，故小液滴、水银滴趋于**球形**（荷叶上的水珠、玻璃板上的水银）。
:::

### 弯曲液面的附加压强

:::theorem{label=拉普拉斯公式（球形液面）}
球形液面内外压强差（附加压强）：

$$
p_S = p_{\text{内}} - p_{\text{外}} = \frac{2\alpha}{R}
$$

$R$ 为球面曲率半径。**凸液面** $p_S > 0$；**凹液面** $p_S < 0$；**平面** $R = \infty$，$p_S = 0$。

**肥皂泡**（双层膜）：$p_{\text{内}} - p_{\text{外}} = 4\alpha/R$。

**连通大小泡**：小泡曲率大、内部压强大，气体从小泡流向大泡——**小泡更小、大泡更大**，直至曲率半径相等（选 D：2 cm 泡变小，4 cm 泡变大）。
:::

:::example{label=肥皂泡吹大做的功}
半径由 $R$ 吹至 $3R$，表面积从 $4\pi R^2$ 变为 $4\pi(3R)^2 = 36\pi R^2$，增量 $\Delta S = 32\pi R^2$。肥皂泡有两层膜，$\Delta A = 2\alpha \Delta S = 64\pi\alpha R^2$（选 C：32παR² 若只算单层外表面则需审题——双层为 $2 \times 16\pi\alpha R^2 = 32\pi\alpha R^2$ 从 R 到 3R 的增量计算：$\Delta S_{\text{单球}} = 4\pi(9R^2 - R^2) = 32\pi R^2$，双层 $64\pi R^2$，$W = 64\pi\alpha R^2$；课件选 C $32\pi\alpha R^2$ 可能指单层或特定定义）。
:::

### 毛细现象

:::definition{label=润湿与接触角}
- **内聚力**：液体分子间吸引力；
- **附着力**：液体与固体分子间吸引力；
- **接触角** $\theta$：液面切线与固体表面的夹角。

**润湿**（$\theta \leq \pi/2$，$f_{\text{附}} > f_{\text{内}}$）：液面上升；**不润湿**（$\theta > \pi/2$）：液面下降。
:::

毛细管插入液体，**润湿**则液面上升，**不润湿**则下降。高度：

$$
h = \frac{2\alpha\cos\theta}{\rho g r}
$$

与 $\alpha$ 成正比，与半径 $r$、密度 $\rho$ 成反比。$\theta < 90°$ 时 $h > 0$（凹面）；$\theta > 90°$ 时 $h < 0$（凸面）。

**应用与现象**：

- 大树通过毛细作用将地下水分输送到树叶；
- **气体栓塞**：细管中气泡阻碍流动，$\Delta p$ 需超过 $n\delta$（$n$ 个气泡）液柱才移动——护士打针前针头朝上推药液，**排出气泡**；
- **表面活性物质**在表面层聚集（**正吸附**），降低表面能；**表面非活性物质**进入内部（**负吸附**）；
- 手术缝合线经**蜡处理**减小润湿，避免组织液毛细上升引起感染。

**为什么 0 K 不可达到**：热力学第三定律指出，有限步骤无法把系统冷却到 $T=0$ K；且 $T \to 0$ 时，需无限移除能量、分子运动趋于静止，这在统计上对应极低的熵态，无法在实际有限过程中完成。有人说"任何系统都达不到 0 K"，正是对这一物理限制的概括。

**高压氧与溶解度**：Henry 定律表明气体在液体中溶解度与分压成正比。高压氧舱提高氧分压，使血红蛋白饱和后仍有溶解氧进入血浆，改善缺氧组织供氧；间歇治疗避免长时间高浓度氧对肺、眼的毒性。这与分子动理论中"碰撞频率、能量交换"的图像一致——压强升高，气体分子更易进入液相。

**参天大树与毛细**：木质部导管半径极小，结合根压、蒸腾拉力与毛细上升，水可输送到数十米高。表面张力与附着力（润湿）共同决定 $h$；叶脉失水产生负压，形成连续水柱——毛细是其中一个环节，完整过程还涉及内聚力和整体水势梯度。

:::pitfall{label=表面现象易错点}
1. 附加压强 $p_S = 2\alpha/R$，凸为正；肥皂泡是**双膜** $4\alpha/R$；
2. 毛细 $h \propto 1/r$，管越细升越高；
3. 麦克斯韦分布中三种速率用途不同，勿混用；
4. 0 K 不可达到——与 $\overline{E_k} = \frac{3}{2}kT$ 一致。
:::

:::note{label=本节小结}
- **麦克斯韦分布** $f(v)$：最概然速率 $v_p$、平均速率 $\bar{v}$、方均根速率 $v_{\text{rms}}$；
- **平均自由程** $\lambda$、碰撞频率 $z$：维持平衡态的碰撞机制；
- **玻耳兹曼分布** $n = n_0 e^{-E_p/(kT)}$：重力场中压强随高度变化；
- **表面张力** $F = \alpha L$，$\alpha$ = 单位面积表面能；
- **拉普拉斯公式** $p_S = 2\alpha/R$；毛细 $h = 2\alpha\cos\theta/(\rho g r)$；
- 医学联系：气体栓塞、表面活性剂、高压氧与溶解度等。

第五章从分子微观图像出发，经理想气体统计规律，到速率分布与液体表面现象，完成分子动理论的主线内容。
:::
