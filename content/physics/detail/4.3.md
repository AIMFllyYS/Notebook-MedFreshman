# 4.3 多普勒效应与声波

## 多普勒效应

:::definition{label=多普勒效应}
当**波源**或**接收者**相对于**介质**运动时，接收者测到的**频率**与波源发出的频率**不同**，这种现象称为**多普勒效应**（Doppler effect）。
:::

火车进站鸣笛音调变高、出站变低，是日常体验。分析时设波在介质中速度 $u$，波源速度 $v_S$，观测者速度 $v_R$（接近为正），源频率 $\nu$，观测频率 $\nu'$。

### 各种情形的频率公式

**1. 波源与观测者均静止**：$\nu' = \nu$。

**2. 观测者运动，波源静止**（$v_S = 0$）：

- 观测者**靠近**波源：$\nu' = \dfrac{u + v_R}{u}\nu$（频率**变大**）；
- **远离**：$\nu' = \dfrac{u - v_R}{u}\nu$（频率**变小**）。

**3. 波源运动，观测者静止**（$v_R = 0$）：

波源运动导致**波长**变化。靠近时前方波面压缩、后方拉长。

- 波源**靠近**观测者：$\nu' = \dfrac{u}{u - v_S}\nu$（**变大**）；
- **远离**：$\nu' = \dfrac{u}{u + v_S}\nu$（**变小**）。

**4. 两者都运动**——统一公式（**靠近取上方符号，远离取下方**）：

$$
\nu' = \frac{u \pm v_R}{u \mp v_S}\nu
$$

分子 $+$ 表示观测者向源运动；分母 $\mp$ 中源向观测者运动取 $-$（频率升高）。

**5. 速度不共线**：取沿视线方向的速度分量代入。

:::example{label=超声波测速}
固定波源发射频率 $\nu_0$ 的超声波，汽车向波源行驶，接收器收到反射波频率 $\nu'$。反射过程等价于"动目标作二次多普勒"，联立 $\nu'$ 与 $\nu_0$、声速 $u$ 可求车速。警察雷达测速即基于此原理。
:::

:::example{label=警车追小偷的鸣笛频率}
警车速度 $v_1$、小偷车速 $v_2$，警笛 $\nu_0$，声速 $u$。小偷听到的频率需考虑源（警车）与接收者（小偷）相对运动，用 $\nu' = \dfrac{u \pm v_R}{u \mp v_S}\nu_0$，代入沿视线方向的分量。
:::

## 超声多普勒血流仪

医学诊断的核心应用之一：

1. 以速度 $v_R$ 运动的红细胞接收到的频率（ blood 作为"观测者"）；
2. 静止探头接收红细胞**反射**回的频率；
3. **多普勒频移** $\Delta\nu = \nu' - \nu$；
4. 由此反推**血流速度** $v_R$。

**彩超**用颜色编码：朝向探头 $v > 0$ 红色，背离蓝色；亮度表示 $|v|$；绿色可表示 $\dfrac{dv}{dx}$（速度梯度）；湍流方向多变。

## 冲击波与声暴

当波源速度 $v_S < u$ 时，前述多普勒公式成立。若 $v_S$ 接近 $u$，波前堆积形成**马赫锥**；$v_S = u$ 时锥面退化为平面，能量极度集中，称**声暴**（sonic boom）——超音速飞机必须尽快越过"音障"。

高速快艇舷波、超音速飞机、子弹激波均属**冲击波**，常由非线性效应引起，使介质密度、速度、温度急剧变化。

## 声波的基本量

:::definition{label=声压与声强}
**声场**：声波传播的空间。**声压** $\Delta p = p - p_0$，有声波与无声波时的压强差，单位 Pa；**声压幅值** $p_m$ 简称声幅。

**声阻抗** $Z = \rho u$（媒质密度 × 声速），单位 kg·m⁻²·s⁻¹，表征媒质传声能力。

**声强** $I$：单位时间内通过垂直于传播方向单位面积的声波能量，即平均能流密度。
:::

关系：

$$
I = \frac{p_m^2}{2Z} = \frac{1}{2}\rho u \omega^2 A^2
$$

**声速**：纵波 $u = \sqrt{E/\rho}$（固体）或 $u = \sqrt{B/\rho}$（流体）。空气中 $u = (331 + 0.6 t_{^\circ\text{C}})$ m/s；一般 $u_{\text{固}} > u_{\text{液}} > u_{\text{气}}$。

垂直入射两种介质界面：**声强反射系数**与**透射系数**由两介质 $Z$ 决定。

## 声强级与响度级

人耳可听声强范围约 $10^{-12} \sim 1$ W/m²（相差 $10^{12}$ 倍），故用对数标度。

:::definition{label=声强级}
**声强级**（dB）：

$$
L_I = 10\lg\frac{I}{I_0} \quad \text{dB}, \quad I_0 = 10^{-12}\,\text{W/m}^2
$$

1000 Hz 附近，人耳听觉域约 0～120 dB。
:::

:::example{label=张飞怒吼相当于多少士兵}
张飞声强级 140 dB，士兵 90 dB。由 $L = 10\lg(I/I_0)$，

$$
\frac{I_1}{I_2} = 10^{(140-90)/10} = 10^5
$$

即张飞一声相当于**十万**名士兵同时大喝。
:::

低语 $10^{-8}$ W/m² 与飞机噪声 10 W/m²，声强级差 $10\lg(10/10^{-8}) = 90$ dB（选 B）。

:::definition{label=响度级}
**响度**与声强、**频率**均有关。同一声强级 78 dB，50 Hz 与 1000 Hz 听起来不等响；50 Hz、78 dB 与 1000 Hz、60 dB 可能等响，响度级均为 60 phon（方）。
:::

**听觉阈**：1000 Hz 时 $I_{\min} = 10^{-12}$ W/m²；**痛觉阈**：约 1 W/m²。可听频率约 16～20000 Hz。

两声波频率比 2:1、振幅比 5:1，同介质中声强比 $(A_1/A_2)^2 \times (\nu_1/\nu_2) = 25/2$（与阻抗无关，选 C 声强比 10:1 需具体计算：$I \propto \rho\omega^2 A^2 u$，$\omega$ 差 2 倍则 $I$ 差 4 倍，$A$ 差 5 倍则差 25 倍，合计 100:4 量级——以课件"声强之比为 10:1"为准时检查振幅频率组合）。

## 声波分类与医学

声波按频率可分为 infrasound、可听声、超声波等。*超声波*在医学成像、治疗中有专门章节。听诊、超声心动图、多普勒血流均利用声波携带的振动信息。

**听觉的物理链**：声源振动 → 介质传播 → 外耳收集 → 鼓膜、听小骨放大 → 耳蜗基底膜对不同频率选择响应 → 神经冲动 → 大脑感知。你听到我讲话，本质是声带振动经空气传到耳蜗的**纵波**过程。**声强级计算技巧**：两声源级差 $\Delta L = L_1 - L_2$，则 $I_1/I_2 = 10^{\Delta L/10}$。差 10 dB → 强度比 10 倍；差 20 dB → 100 倍；差 60 dB → $10^6$ 倍。**多普勒 blood flow**：超声探头固定，红细胞运动使反射波频移 $\Delta f \propto v_R \cos\theta$，$\theta$ 为血流方向与声束夹角——因此测量时需尽量使声束与血管平行，否则 $\cos\theta$ 减小会导致速度低估。

**冲击波与临床**：体外碎石机利用聚焦冲击波击碎肾结石；声暴则是有害效应——超音速飞机突破音障时，地面听到类似爆炸的巨响，是马赫锥扫过观测者所致。

**声阻抗与透射**：垂直入射时，声强透射系数 $T_I = \dfrac{4Z_1 Z_2}{(Z_1+Z_2)^2}$，反射系数 $R_I = \dfrac{(Z_1-Z_2)^2}{(Z_1+Z_2)^2}$。$Z$ 相差越大，反射越强——超声在软组织与骨骼界面反射明显，故 B 超能显示骨轮廓。**听阈曲线解读**：1000 Hz 听阈最低（人耳最敏感）；低频需更大声强才听到，故同样 dB 下 100 Hz 不如 1000 Hz 响。Pain threshold 约 120 dB 以上，长期暴露可致听力损伤——这与声强 $I \propto p_m^2$ 对组织的机械与热效应相关。

:::pitfall{label=多普勒与声学的易错点}
1. 多普勒公式中 $v_S$、$v_R$ 是**相对介质**的速度，不是相对彼此；
2. **靠近→频率升高**，分子分母符号勿反；
3. 声强级差 90 dB 对应声强比 $10^9$，不是 90 倍；
4. 响度级与声强级数值可不同（频率影响响度）。
:::

:::note{label=本节小结}
- **多普勒效应**：$\nu' = \dfrac{u \pm v_R}{u \mp v_S}\nu$；靠近频率升高；
- **超声多普勒**：测血流速度；彩超用红/蓝/绿编码；
- **冲击波、声暴**：$v_S \to u$ 时能量集中；
- **声压、声阻抗** $Z = \rho u$、**声强** $I = p_m^2/(2Z)$；
- **声强级** $L_I = 10\lg(I/I_0)$ dB；**响度级** phon 考虑频率；
- 应用：测速、医学血流监测、听觉与噪声评价。
