## 眼的光学结构与调节

:::insight{label=人眼是一台"可变焦距相机"}
人眼可近似为共轴光学系统：角膜、房水、晶状体、玻璃体构成屈光介质，视网膜是"底片"。看近看远时像距（视网膜位置）基本不变，靠的是**调节**——改变晶状体曲率半径，使不同物距都能清晰成像。理解这一点，近视、远视、老花的矫正就一目了然了。
:::

## 眼的光学结构

:::definition{label=示意眼}
实际人眼可近似为由五种介质组成的共轴光学系统，称为**示意眼**，包括：空气、角膜、房水、晶状体、玻璃体及视网膜。各结构元素有平均的曲率半径、厚度和折射率参数。
:::

:::definition{label=简约眼}
**简约眼**是眼睛的简化单球面折射系统。人眼看近处和远处物体时，**像距不变**，需根据物距**调节曲率半径** $r$，使之满足成像公式。
:::

## 眼的调节

:::definition{label=调节、远点、近点、明视距离}
**调节**：眼睛改变焦度（焦距）的过程。

**远点**：不经调节时，能看清最远物体与眼睛的距离。正常视力者远点为无穷远。

**近点**：最大调节时，能看清最近物体与眼睛的距离。正常视力者近点约 10–12 cm。

**明视距离**：日常最适宜、不易疲劳的观察距离，正常约为 **25 cm**。
:::

:::definition{label=正视眼与屈光不正}
**正视眼**：不调节时，平行光入眼经折射后刚好在视网膜上成清晰像。

否则为**非正视眼**（屈光不正），包括：近视、远视、老花、散光。
:::

## 眼的分辨本领与视力

:::definition{label=视角与分辨本领}
**视角**：从物体两端射到眼中节点的光线所夹角度。**分辨本领**是眼睛能分辨两物点间最小距离的能力，与视角有关——视角越小，越难分辨。

**小数法视力**（1990年5月前国际标准）与**五分法视力**（1990年5月后对数视力）是两种常用表示。标准视力对应视角约 1′（1 分）。
:::

## 屈光不正及其矫正

### 近视眼

:::definition{label=近视眼}
眼睛不调节时，平行光入眼后会聚于**视网膜前**，视网膜上成像模糊，称为**近视眼**。

**成因**：（1）长时间近距离用眼，角膜或晶状体曲率半径过小，屈光能力加强；（2）眼轴过长或某介质折射率过大。

**轴性近视**：眼前后径过长；**屈光性近视**：屈光系统能力过强。

**特点**：远点与近点都向眼移近；越近视，远点越近。
:::

**矫正**：用**凹透镜**矫正远点。凹透镜（近视镜）将无穷远物点成像于患者能看清的远点处。

:::theorem{label=近视镜焦度}
对近视镜，若远点为 $d$（眼前距离），则

$$
\Phi = -\frac{1}{d}, \qquad f = -d
$$

（远点在眼前，$d>0$，$\Phi<0$，凹透镜）
:::

:::example{label=近视眼配镜度数}
某近视眼远点在眼前 50 cm，欲看清无穷远物体，应佩戴多少度眼镜？

**解**：眼镜须使无穷远物体在 50 cm 处成虚像，$f = -0.50\,\mathrm{m}$，

$$
\Phi = \frac{1}{f} = -2\,\mathrm{D} = -200\ \text{度}
$$

应佩戴 **200 度凹透镜**。
:::

### 远视眼

:::definition{label=远视眼}
不调节时，远处平行光会聚于**视网膜后**，称为**远视眼**。

**轴性远视**：眼球前后径过短；**屈光性远视**：屈光能力过弱。成因包括眼病、遗传等。近点比正视眼更远。
:::

**矫正**：用**凸透镜**矫正近点，将近处物点（常取明视距离 25 cm）成像于患者能看清的近点处。

:::example{label=远视眼配镜度数}
某远视眼近点在眼前 1.2 m，欲看清眼前 12 cm 处物体，应佩戴多少度眼镜？

**解**：$u = -0.12\,\mathrm{m}$，$v = -1.2\,\mathrm{m}$，

$$
\frac{1}{f} = \frac{1}{-0.12} + \frac{1}{-1.2} = -9.17\ \mathrm{m}^{-1}
$$

$\Phi \approx +7.5\,\mathrm{D}$，即 **750 度凸透镜**。
:::

### 老花眼与散光眼

:::definition{label=老花眼}
**老花眼**：随年龄增长，晶状体弹性下降、睫状肌不灵，属正常生理现象。看远正常，**近点远移**。矫正：**凸透镜**（老花镜）。
:::

:::definition{label=散光眼}
**散光眼**：各子午线曲率不完全相同，点光源经眼折射后不能会聚于一点。有近视散光、远视散光之分。成因：眼病、遗传。矫正：**凹柱面透镜**或**凸柱面透镜**。
:::

:::pitfall{label=角膜在水中"失焦"}
角膜是眼最主要的聚焦部分。在水中丧失大部分聚焦本领，主要因为**角膜与水的折射率接近**（而非全反射或吸收）。这是游泳时看不清的重要光学原因。
:::

## 示意眼的结构参数

示意眼将角膜、房水、晶状体、玻璃体简化为共轴折射系统。角膜是眼最主要的聚焦部分，在水中丧失大部分聚焦本领，因为**角膜与水的折射率接近**（折射率差减小 → 折射能力减弱），这是潜水时视力模糊的光学原因。

:::definition{label=简约眼的调节机制}
简约眼把复杂的眼屈光系统简化为单一折射球面。看远处时曲率半径较大（屈光力较弱）；看近处时睫状肌收缩，晶状体变凸，曲率半径减小，屈光力增强，使近物仍成像于视网膜。像距（视网膜位置）基本不变，**调节的本质是改变焦度**。
:::

视力表测量的是分辨两相邻点所需的最小视角。五分法视力 $L = \lg(1/\theta)$（$\theta$ 以分为单位），视角 1′ 对应五分记录 5.0。小数视力 1.0 与五分 5.0 大致对应标准视力。

:::example{label=近视镜度数的计算通式}
远点在眼前 $d$（米）的近视眼，要看清无穷远，需 $1/f + 1/(-d) = 0$（物在无穷，像在目镜远点），故 $f = -d$，$\Phi = -1/d$（D）。$d = 0.25\,\mathrm{m}$ 时 $\Phi = -4\,\mathrm{D}$，即 400 度近视镜。

远视矫正同理：将近处物（如 $u = -0.25\,\mathrm{m}$）成像于患者近点 $v = -d_\text{近}$，由 $\dfrac{1}{f} = \dfrac{1}{u}+\dfrac{1}{v}$ 求 $\Phi$。
:::

散光眼各子午线曲率不同，点光源成像为**弥散圆**而非像点，故远处点物可能被看成一条模糊线——这是散光与单纯近视的重要区别。矫正需柱面透镜，只在某一子午面方向提供所需焦度。

:::note{label=本节小结}
- 人眼通过**调节**改变晶状体焦度，使不同物距在视网膜上成像。
- **远点、近点、明视距离**（25 cm）是衡量眼屈光状态的基本量。
- **近视**用凹透镜、**远视/老花**用凸透镜、**散光**用柱面透镜——核心是"把物点成像移到患者能看清的位置"。
- 眼镜度数 $\Phi = 1/f$（D），凹透镜度数为负，凸透镜为正。

:::
