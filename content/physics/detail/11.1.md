:::insight{label=两束"步调一致"的光才会稳定干涉}
普通光源各原子发光随机、间歇，没有稳定相位差，**不能干涉**。要看到稳定的明暗条纹，必须让两列光**频率相同、振动方向相同、相位差恒定**——这就是相干光。杨氏双缝的巧妙之处：把同一光源的一束光分成两束再叠加，天然满足相干条件。
:::

## 光源与相干性

:::definition{label=可见光与光源}
**可见光**：能引起视觉的电磁波，真空中波长约 400 nm–760 nm。

**光源**按机理分：**热辐射**（太阳、白炽灯，热能激发）与**非热辐射**（日光灯、LED，电能/化学能激发）。发光本质：原子能级跃迁——吸收能量到高能态，不稳定，跃迁辐射光子回到低能态。
:::

原子发光特点：**间歇性**（波列持续 ~$10^{-8}$ s）、**随机性**（频率、相位、振动方向各不相干）。因此**普通光源不是相干光源**。

- **单色光**：单一波长；**复色光**：多种波长（白光是复色光，激光接近单色光）
- **色散**：将复色光中不同波长分开

:::definition{label=相干条件}
两列光**相干**需满足：
1. 振动方向相同
2. 频率相同
3. 有**恒定的相位差**

获得相干光的方法：用单色性好的点光源，把光线分成两部分再叠加（如杨氏双缝、分波面法）。
:::

## 光程与光程差

:::definition{label=光程}
光在介质中传播几何路程 $r$，相位变化与真空中传播 $nr$ 相同。**光程** $L = nr$，好处是统一用真空中波长 $\lambda$ 计算。

真空中波长 $\lambda$，折射率 $n$ 的介质中波长 $\lambda' = \lambda/n$。
:::

:::theorem{label=理想薄透镜等光程性}
**理想薄透镜**改变光路而不增加光程差；成像具有**等光程性**——物点到像点的各光线光程相等。
:::

## 杨氏双缝干涉

1801 年托马斯·杨完成双缝实验，屏上出现明暗相间等间距条纹，证明光的波动性，并测出波长。

:::theorem{label=杨氏双缝明暗条件（近轴）}
设双缝 $S_1$、$S_2$ 到屏上 $P$ 点的光程差为 $\delta$：

**明纹**（干涉加强）：$\delta = k\lambda$，$k = 0,1,2,\ldots$（$k=0$ 为中央明纹）

**暗纹**（干涉相消）：$\delta = (2k+1)\dfrac{\lambda}{2}$

近轴时 $\delta \approx d\sin\theta \approx \dfrac{dx}{D}$（$d$ 缝距，$D$ 缝到屏距离，$x$ 为屏上位置）。
:::

:::theorem{label=条纹间距}
相邻明纹（或暗纹）间距：

$$
\Delta x = \frac{D\lambda}{d}
$$

**特点**：（1）已知 $D$、$d$ 测 $\Delta x$ 可求 $\lambda$；（2）$d$ 越小、$D$ 越大，条纹越疏、干涉越明显；（3）$x_k \propto \lambda$，白光入射得彩色条纹。
:::

:::example{label=杨氏双缝条纹间距计算}
杨氏实验中 $D = 1\,\mathrm{m}$，钠光 $\lambda = 589.3\,\mathrm{nm}$。（1）$d_1 = 0.2\,\mathrm{mm}$ 和 $d_2 = 1.0\,\mathrm{mm}$ 时明纹间距各多大？（2）装置置于水中（$n = 1.33$）时条纹间距？

**解**：（1）$\Delta x_1 = \dfrac{1 \times 589.3 \times 10^{-9}}{0.2 \times 10^{-3}} = 2.95\,\mathrm{mm}$；$\Delta x_2 = 0.59\,\mathrm{mm}$。

（2）水中有效波长 $\lambda' = \lambda/n$，$\Delta x = \dfrac{D\lambda}{nd} = \dfrac{2.95}{1.33} = 2.22\,\mathrm{mm}$（以 $d = 0.2\,\mathrm{mm}$ 为例）。
:::

:::example{label=薄膜遮住一缝后的级次}
双缝间距 0.60 mm，缝屏距 1.50 m，相邻明纹间距 1.50 mm。（1）求波长。（2）用 $n = 1.30$、厚度 $l = 0.01\,\mathrm{mm}$ 的薄膜遮住一缝，原中央明纹处变为第几级明纹？

**解**：（1）$\lambda = \dfrac{d\Delta x}{D} = \dfrac{0.60 \times 10^{-3} \times 1.5 \times 10^{-3}}{1.5} = 600\,\mathrm{nm}$。

（2）遮住 $S_1$ 后，原中央明纹处光程差增加 $(n-1)l = 0.003\,\mathrm{mm}$。由 $(n-1)l = k\lambda$，$k = 5$，变为**第 5 级明纹**。
:::

## 劳埃德镜与半波损失

:::definition{label=半波损失}
**半波损失**：光从**光疏**介质（折射率小）射向**光密**介质（折射率大）表面反射时，反射光有半个波长的附加光程差（附加 $\pi$ 相位）。**透射光无半波损失**。反之（光密→光疏反射）无半波损失。
:::

劳埃德镜实验：屏移近镜面一端，接触处为**暗纹**，说明镜面反射光相对直射光有 $\pi$ 相位突变，与杨氏双缝互补。

## 薄膜干涉

:::theorem{label=薄膜干涉明暗条件}
设薄膜折射率 $n$，厚度 $e$，入射角 $i$，反射光光程差 $\delta = 2ne\cos r$（$r$ 为膜内折射角）。

**反射光加强**（$k \neq 0$）：$\delta = k\lambda$（需根据 $n_1,n,n_2$ 关系判断是否加 $\lambda/2$）

**反射光减弱**：$\delta = (2k+1)\dfrac{\lambda}{2}$

**适用条件**：
- $n_1 < n > n_2$ 或 $n_1 > n < n_2$：**不加** $\lambda/2$
- $n_1 < n < n_2$ 或 $n_1 > n > n_2$：**要加** $\lambda/2$

**透射光与反射光互补**：反射加强处透射减弱，反之亦然。
:::

:::example{label=肥皂膜颜色}
空气中厚度 $e = 0.32\,\mu\mathrm{m}$ 的肥皂膜（$n = 1.33$），白光垂直入射，呈现什么颜色？

**解**：垂直入射，$n_1 < n > n_2$，有半波损失。反射加强：$2ne + \lambda/2 = k\lambda$。$k=4$ 时 $\lambda \approx 530\,\mathrm{nm}$（**绿色**）。
:::

:::example{label=增透膜}
照相机镜头镀 MgF$_2$（$n = 1.38$），玻璃 $n' = 1.5$，要使 $\lambda = 550\,\mathrm{nm}$ 反射最小。求最小膜厚。

**解**：$n' > n > n_0$，无半波损失。相消：$2ne = (2k+1)\lambda/2$，$k=0$ 时 $e_{\min} = \lambda/(4n) = 550/(4 \times 1.38) \approx 99.6\,\mathrm{nm}$。
:::

## 等厚干涉：劈尖与牛顿环

:::theorem{label=劈尖干涉}
劈尖角 $\theta$ 很小，垂直入射，两束反射光程差 $\delta = 2ne$。同一条纹对应相同厚度 → **等厚条纹**（与棱边平行的直条纹）。

- 棱边 $e=0$ 对应暗纹（有半波损失时）
- 相邻明（暗）纹厚度差 $\Delta e = \lambda/(2n)$
- 明纹间距 $L = \lambda/(2n\sin\theta)$；$\theta$ 越大，$L$ 越小
:::

:::theorem{label=牛顿环}
空气隙牛顿环，第 $k$ 级暗环半径 $r_k = \sqrt{kR\lambda}$（$R$ 为凸透镜曲率半径）。中心为暗点；边缘条纹愈密。可用于测 $R$ 或 $\lambda$。
:::

::video{id=phys-ch11-11.1-kp4-thin-film-equal-thickness}

:::example{label=牛顿环求曲率半径}
氦氖激光 $\lambda = 633\,\mathrm{nm}$，第 $k$ 暗环半径 5.63 mm，第 $k+5$ 暗环 7.96 mm，求 $R$。

**解**：$r_{k+5}^2 - r_k^2 = 5R\lambda$，$R = \dfrac{7.96^2 - 5.63^2}{5 \times 633 \times 10^{-6}} \approx 1.00\,\mathrm{m}$。
:::

## 迈克耳孙干涉仪

两相干光束空间分开，可通过移动反射镜或插入介质改变光程差。$M_1$ 移动 $\Delta d$ 时，条纹移动（吞吐）数目 $N = 2\Delta d/\lambda$。两镜面严格垂直 → 等倾干涉；不严格垂直 → 等厚（劈尖）干涉。迈克耳孙-莫雷实验是否定以太，为相对论重要实验基础。

## 光学发展简史

波动光学是光学史第三阶段（19 世纪）：杨氏双缝、菲涅耳衍射确立了光的波动说。本章三节——干涉、衍射、偏振——完整呈现光的波动侧；与第 12 章光的粒子性共同构成波粒二象性。

:::definition{label=迈克耳孙干涉仪}
两束相干光在空间分开，通过分束镜、反射镜形成可变光程差。$M_1$ 移动 $\Delta d$，条纹吞吐数 $N = 2\Delta d/\lambda$，可用于精密测长、测折射率。镜面严格垂直 → 等倾干涉（同心圆环）；略有倾角 → 等厚干涉（劈尖条纹）。迈克耳孙-莫雷实验（1881）否定以太，为相对论奠基。迈克耳孙因此获 1907 年诺贝尔物理学奖。
:::

等倾条纹特征：同心圆环，愈近中心级次愈高；增加膜厚，条纹自里向外**冒出**；反之**缩入**。可根据冒出条纹数测定微小长度变化。

劈尖应用：测波长（已知 $\theta$、$n$ 测 $L$）；测微小厚度、金属丝直径；检测表面平整度——有"凸起"或"凹槽"时条纹弯曲方向相反。

:::example{label=油膜厚度（例3）}
油膜 $n_1 = 1.30$ 在玻璃 $n_2 = 1.50$ 上，波长连续变化时 500 nm 与 700 nm 反射同时消失（干涉极小）。上下表面反射均有半波损失，相消：$2n_1 e = k\lambda$。联立 $k=4$、$k'=3$ 得 $e = 0.673\,\mu\mathrm{m}$。
:::

:::note{label=本节小结}
- **相干条件**：同频、同向、恒定相位差；普通光源需分波面。
- **光程** $L=nr$；杨氏条纹 $\Delta x = D\lambda/d$。
- **半波损失**只发生在光疏→光密反射。
- **薄膜、劈尖、牛顿环**是等厚/等倾干涉的典型应用；增透膜、测厚、测曲率半径均源于此。

下一节：光不再"严格走直线"——衍射打开波动光学另一扇大门。干涉习题常考半波损失判断、薄膜厚度与劈尖条纹移动方向，需结合 $n_1,n,n_2$ 三层介质关系仔细分析。
:::
