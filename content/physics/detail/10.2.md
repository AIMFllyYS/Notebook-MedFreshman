## 薄透镜成像与透镜组合

:::insight{label=两个球面"合成"一个透镜}
透镜本质上是两个折射球面的组合。当透镜厚度远小于曲率半径时，可忽略中间光程差，把两个球面的逐次成像合并为一个**薄透镜公式**——这正是相机、显微镜物镜、眼镜片最常用的模型。
:::

## 透镜的分类

按外形：凸透镜（双凸、平凸、弯月凸）与凹透镜（双凹、平凹、弯月凹）。按厚度：薄透镜与厚透镜。凸透镜一般起会聚作用，凹透镜起发散作用。

照相机、投影仪、显微镜物镜等都基于薄透镜成像原理：物距大于焦距时成实像，可用于胶片或传感器接收；物距小于焦距时成虚像，需用眼睛在镜后观察——这一区分在解题时首先要判断。

:::derivation{label=薄透镜公式的推导思路}
对第一折射面：$\dfrac{1}{u} + \dfrac{n}{v_1} = \dfrac{n-1}{r_1}$

对第二折射面（第一面的像作第二面的物，$u_2 = -v_1$）：$\dfrac{n}{-v_1} + \dfrac{1}{v} = \dfrac{1-n}{r_2}$

两式相加，$v_1$ 消去，即得 $\dfrac{1}{u}+\dfrac{1}{v}=(n-1)\left(\dfrac{1}{r_1}-\dfrac{1}{r_2}\right)$。薄透镜条件保证了中间光程差可忽略。
:::

## 薄透镜成像公式

:::definition{label=薄透镜}
**薄透镜**指厚度远小于曲率半径的透镜。设第一折射面曲率半径 $r_1$、第二面 $r_2$，折射率 $n$，外侧介质折射率为 1（空气）时，对第一、第二折射面分别用逐次成像，并令 $u = u_1$、$v = v_2$，两式相加得：
:::

:::theorem{label=薄透镜成像公式}
$$
\frac{1}{u} + \frac{1}{v} = (n-1)\left(\frac{1}{r_1} - \frac{1}{r_2}\right)
$$

当透镜置于空气中时，焦距 $f$ 满足

$$
\frac{1}{f} = (n-1)\left(\frac{1}{r_1} - \frac{1}{r_2}\right)
$$

薄透镜**高斯公式**：

$$
\frac{1}{u} + \frac{1}{v} = \frac{1}{f}
$$
:::

:::theorem{label=薄透镜焦度}
薄透镜的**焦度** $\Phi = 1/f$（单位 D）。$f > 0$ 时为实焦点，透镜会聚光线；$f < 0$ 时为虚焦点，透镜发散光线。
:::

:::example{label=平凸透镜焦距（双向验证）}
求薄平凸透镜焦距。设 $n = 1.5$，凸面曲率半径 $R = 30\,\mathrm{cm}$，置于空气中。

**解**：（1）光线从凸面入射：$r_1 = +30\,\mathrm{cm}$，$r_2 = \infty$，

$$
\frac{1}{f} = (1.5-1)\left(\frac{1}{30} - 0\right) = \frac{1}{60} \quad \Rightarrow \quad f = 60\,\mathrm{cm}
$$

（2）光线从平面入射：$r_1 = \infty$，$r_2 = -30\,\mathrm{cm}$，结果相同。说明薄透镜焦距与入射方向无关（近轴、薄透镜近似下）。
:::

## 薄透镜组合

:::definition{label=透镜组}
两个或两个以上薄透镜组成的共轴系统称为**薄透镜组合**（透镜组）。
:::

**1. 紧密接触透镜**

设两透镜焦距分别为 $f_1$、$f_2$，透镜组焦距 $f$，对第一透镜 $\dfrac{1}{u} + \dfrac{1}{v_1} = \dfrac{1}{f_1}$，对第二透镜 $\dfrac{1}{-v_1} + \dfrac{1}{v} = \dfrac{1}{f_2}$，相加得：

:::theorem{label=密接透镜组焦距与焦度}
$$
\frac{1}{f} = \frac{1}{f_1} + \frac{1}{f_2}, \qquad \Phi = \Phi_1 + \Phi_2
$$

透镜组成像公式仍为 $\dfrac{1}{u} + \dfrac{1}{v} = \dfrac{1}{f}$。
:::

:::example{label=水-玻璃弯月组合透镜}
汇聚弯月形透镜，$n = 1.5$，两表面曲率半径分别为 5 cm 和 10 cm，凹面朝上盛满水（$n_\text{水} = 4/3$），置于空气中。求组合焦距。

**解**：设光线从上方入射。玻璃透镜 $r_1 = +10\,\mathrm{cm}$，$r_2 = +5\,\mathrm{cm}$：

$$
\frac{1}{f_1} = 0.5\left(\frac{1}{10} - \frac{1}{5}\right) = -0.1 \quad \Rightarrow \quad f_1 = -10\,\mathrm{cm}
$$

水透镜（弯月形）：$r_1 = -5\,\mathrm{cm}$，$r_2 = -10\,\mathrm{cm}$：

$$
\frac{1}{f_2} = \left(\frac{4}{3}-1\right)\left(\frac{1}{-5} - \frac{1}{-10}\right) = \frac{1}{3} \cdot \left(-\frac{1}{10}\right) \quad \Rightarrow \quad f_2 = -30\,\mathrm{cm}
$$

两透镜密接：$\dfrac{1}{f} = \dfrac{1}{-10} + \dfrac{1}{-30} = -\dfrac{2}{15}$，故 $f = -7.5\,\mathrm{cm}$。
:::

**2. 不紧密接触——逐次成像法**

两透镜间距较大时，不能再用密接公式，必须逐次成像。

:::example{label=凸凹透镜组逐次成像}
凸透镜 $L_1$ 焦距 $f_1 = 20\,\mathrm{cm}$，凹透镜 $L_2$ 焦距 $f_2 = -40\,\mathrm{cm}$，$L_2$ 在 $L_1$ 右侧 40 cm 处。物置于 $L_1$ 左侧 30 cm，求最终像。

**解**：**$L_1$ 成像**：$\dfrac{1}{30} + \dfrac{1}{v_1} = \dfrac{1}{20}$，得 $v_1 = 60\,\mathrm{cm}$（实像）。

**$L_2$ 成像**：该实像作 $L_2$ 的虚物，$u_2 = -(60-40) = -20\,\mathrm{cm}$：

$$
\frac{1}{-20} + \frac{1}{v_2} = \frac{1}{-40} \quad \Rightarrow \quad v_2 = -40\,\mathrm{cm}
$$

最终在 $L_2$ 左侧 40 cm 处成**虚像**。
:::

## 厚透镜与柱面透镜

:::definition{label=厚透镜三对基点}
**厚透镜**不能忽略厚度，可用**三对基点等效光路法**：
- **两焦点** $F_1$、$F_2$（物方、像方焦点）
- **两主点** $H_1$、$H_2$（第一、第二主平面）
- **两节点** $N_1$、$N_2$（入射角不变地出射）

用任意两条特殊光线（过 $F_1$ 折射后平行光轴；平行光轴折射后过 $F_2$；过 $N_1$ 从 $N_2$ 同向出射）即可作图求像。
:::

:::definition{label=柱面透镜}
**柱面透镜**的两个折射面为圆柱面的一部分，各方向曲率半径（或焦距、焦度）不同，属**非对称折射系统**，常用于矫正散光。
:::

## 透镜的像差

:::definition{label=像差}
由于物点发出的光偏离理想光路，造成成像不完善的现象称为**像差**。**单色像差**包括球面像差、像散、彗形像差等；**色像差**由介质折射率随波长变化引起。
:::

- **球面像差**：近轴与远轴光线不能会聚于同一点，点物成像为圆斑。矫正：加光阑、会聚与发散透镜组合。
- **色像差**：不同波长不能同点成像，透镜越厚越明显。矫正：折射率不同的会聚、发散透镜适当组合。

提高镜头质量：增透膜、偏振片、透镜组消球差与色差。实际镜头还采用非球面、特殊玻璃（消色差双合透镜）等工艺，在像差与成本之间折中。

## 作图法与透镜成像性质

薄透镜成像作图三光线：（1）平行光轴 → 过像方焦点；（2）过光心（薄透镜中心）→ 方向不变；（3）过物方焦点 → 平行光轴出射。实像在透镜另一侧，可用屏接收；虚像在物同侧，只能用眼观察。

凹透镜只能成虚像，无论物在何处；凸透镜物在 $f$ 外成实像，$f$ 内成虚像。这一规律与 $f$ 正负、$u$、$v$ 符号法则一致，可相互验证。

:::example{label=密接透镜组焦度相加}
两薄透镜密接，$f_1 = 20\,\mathrm{cm}$，$f_2 = -30\,\mathrm{cm}$，求组合焦距与焦度。

**解**：$\dfrac{1}{f} = \dfrac{1}{20} + \dfrac{1}{-30} = \dfrac{1}{60}$，$f = 60\,\mathrm{cm}$。$\Phi = 1/0.6 \approx +1.67\,\mathrm{D}$。一凸一凹密接时，组合透镜可能仍为会聚或发散，取决于两者焦度代数和。
:::

## 柱面透镜与像差矫正

柱面透镜只在某一子午面方向有屈光力，用于矫正散光眼。球差使点物成像为弥散圆，可通过缩小孔径（光阑）或透镜组合部分校正；色差使各色光焦点分离，可用柯克双合透镜（冕牌玻璃 + 火石玻璃）补偿。现代镜头设计是几何光学、波动光学（衍射极限）与材料科学的综合。

:::note{label=本节小结}
- **密接组合**：$\dfrac{1}{f} = \dfrac{1}{f_1} + \dfrac{1}{f_2}$；**非密接**用逐次成像。
- 厚透镜用三对基点作图；柱面透镜用于散光矫正。
- **像差**是实际系统与理想高斯光学之间的偏差，医用与照相镜头需专门校正。
- 厚透镜在精密设计中不可忽略；弯月透镜、非球面透镜等仍是折射定律的组合应用。

下一节把透镜模型用于人眼——你每天都在用的"生物相机"。作图时记住：实像倒立、虚像正立；放大率 $|m| = |v/u|$，结合高斯公式可快速验算。
:::
