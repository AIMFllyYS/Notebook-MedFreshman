:::insight{label=只有横波才能"挑方向"}
偏振是**横波**特有的现象：若光通过偏振片后强度随片轴旋转而变化，说明光振动有方向性，光是横波而非纵波。自然光各方向振动均等；线偏振光只有一个振动方向——偏振片、布儒斯特角、双折射都是操控"振动方向"的工具。
:::

:::definition{label=偏振}
**偏振**：横波振动方向相对于传播方向的不对称性。纵波（如声波）不能偏振；光能偏振，证明光是横波。
:::

**演示实验对比**：
- **自然光**通过偏振片，旋转偏振片光强不变
- **线偏振光**通过检偏器，旋转一周出现两次最强、两次消光

## 自然光与偏振光

:::definition{label=偏振光的分类}
（1）**线偏振光**（完全偏振）：振动只有一个方向

（2）**部分偏振光**：各方向都有振动，某一方向占优势

（3）**椭圆（圆）偏振光**：振动方向随时间沿椭圆（或圆）变化

（4）**自然光**：大量原子发光，无数振动方向机会均等，可分解为两个垂直、等振幅的独立振动
:::

:::definition{label=起偏器与检偏器}
**起偏器**（偏振片）：从自然光获得偏振光。**检偏器**：检验某束光是否偏振。偏振片由拉伸的聚乙烯醇薄膜掺碘形成"碘链"，沿链方向吸收光振动，垂直方向透过。
:::

## 马吕斯定律

:::theorem{label=马吕斯定律}
自然光通过偏振片后光强为原来的一半。线偏振光（强度 $I_0$）再通过与振动方向夹角 $\theta$ 的偏振片，出射光强：

$$
I = I_0\cos^2\theta
$$

这是**马吕斯定律**（Malus' law）。
:::

::plot{fn="(cos(x))^2" xmin=0 xmax=6.28 label="马吕斯定律 I/I₀=cos²θ"}

:::example{label=自然光与线偏振光混合}
自然光与线偏振光混合，通过偏振片旋转，最强与最弱光强比 6:1，求两者强度比。

**解**：设线偏振光强度 $I_p$，自然光强度 $I_n$。自然光过偏振片后 $I_n/2$；线偏振光最强 $I_p + I_n/2$，最弱 $I_n/2$（线偏振垂直时）。由 $(I_p + I_n/2)/(I_n/2) = 6$，得 $I_p/I_n = 5$。
:::

:::example{label=插入第三块偏振片}
自然光通过夹角 60° 的两偏振片，透射光强 $I$。其间再插入与前两者均成 30° 的偏振片，求新光强。

**解**：插入前 $I = \dfrac{I_0}{2}\cos^2 60° = I_0/8$。

插入后：$I' = \dfrac{I_0}{2}\cos^2 30° \cdot \cos^2 30° = \dfrac{I_0}{2} \times \left(\dfrac{3}{4}\right)^2 = \dfrac{9I_0}{32} = \dfrac{9}{4}I$。
:::

## 布儒斯特定律

:::definition{label=反射光的偏振}
自然光在两种介质表面反射和折射时，反射光和折射光一般是**部分偏振光**。当入射角 $i = i_0$（**起偏角**）时，反射光为**完全线偏振光**，振动方向垂直于入射面。
:::

:::theorem{label=布儒斯特定律}
当 $i_0 + r = \pi/2$（反射光与折射光垂直）时，

$$
\tan i_0 = \frac{n_2}{n_1}
$$

此时反射光为振动垂直于入射面的线偏振光；**透射光仍为部分偏振光**。

例：空气（$n_1=1$）→ 玻璃（$n_2=1.50$），$i_0 = 56.3°$；玻璃→ 空气，$i_0 = 33.7°$。
:::

**实质**：以布儒斯特角入射时，若**没有反射光**，则入射光为振动平行于入射面的线偏振光——平行分量全部透射。

**应用**：
1. **玻片堆**：多次反射-折射累积，获得线偏振折射光
2. **布儒斯特窗**：激光器谐振腔端面
3. 测不透明介质折射率
4. 摄影/相机偏振片消反光、太阳镜防眩目

:::pitfall{label=不能把自然光变成偏振光的器件}
**光栅**只能色散，不能起偏。起偏器、检偏器、玻璃片堆可以。
:::

## 光的双折射

:::definition{label=双折射}
光进入**各向异性晶体**（如方解石）后分裂为两束折射光，沿不同方向传播，称为**双折射**（birefringence）。两束光均为线偏振光。
:::

- **寻常光（o 光）**：服从折射定律，各方向折射率、速度相同
- **非常光（e 光）**：不服从普通折射定律，各方向折射率不等

**主折射率**：$n_o = c/v_o$，$n_e = c/v_e$。**正晶体** $v_o > v_e$，$n_o < n_e$（如石英）；**负晶体** $v_o < v_e$，$n_o > n_e$（如方解石）。

光沿**光轴**入射时不发生双折射。惠更斯原理可解释 o 光、e 光波面在晶体中的不同传播。

## 偏振光的应用（拓展）

:::definition{label=光弹效应与克尔效应}
**光弹效应**：透明介质受机械应力产生各向异性 → 双折射，$n_o - n_e \propto p$（$p$ 为应力）。

**克尔效应**：透明介质受电场产生各向异性 → 双折射，$n_o - n_e = kE^2$。
:::

:::definition{label=旋光性}
**旋光现象**：偏振光通过某些物质后，振动面以传播方向为轴转过一定角度。

- **左旋物质**：对着入射方向观察，逆时针旋转
- **右旋物质**：顺时针旋转（如葡萄糖溶液）

固体：$\varphi = \alpha l$；溶液：$\varphi = \alpha c l$（$\alpha$ 为旋光率）。应用：测溶液浓度、糖量计。
:::

## 液晶与偏振应用（拓展）

**液晶**：介于晶体与液体之间，具有各向异性，可显示双折射、选择反射和光电效应，广泛用于显示面板。

**防眩目偏振片**：汽车前罩灯、挡风玻璃加偏振片，调节至合适角度可大幅减弱反射眩光（摄影中同样用 CPL 偏振镜消除水面、玻璃反光）。

:::example{label=马吕斯定律应用}
线偏振光强度 $I_0$ 通过检偏器，透射光强 $I = I_0\cos^2\theta$。$\theta = 45°$ 时 $I = I_0/2$；$\theta = 90°$ 时消光。检偏器旋转一周，线偏振入射时出现两次极大、两次消光——这是鉴定线偏振光的实验标志。
:::

方解石双折射：o 光与 e 光振动方向相互垂直，叠加可产生偏振光。波片（1/4 波片、1/2 波片）利用双折射改变偏振态，是激光器和偏振测量中的常用元件。旋光色散指旋光率随波长变化，糖量计利用葡萄糖等物质的旋光性测定浓度。偏振在三维电影、LCD 显示、应力分析（光弹效应）中也有广泛应用。

:::note{label=本节小结}
- **自然光**各向均等；**线偏振光**只有一个振动方向。
- **马吕斯定律** $I = I_0\cos^2\theta$；自然光过偏振片强度减半。
- **布儒斯特角** $\tan i_0 = n_2/n_1$，反射光振动垂直入射面。
- **双折射**源于晶体各向异性，o 光、e 光偏振方向相互垂直。
- 偏振片消反光、玻片堆起偏、旋光测浓度——偏振在光学仪器与化学分析中应用广泛。

波动光学三章（干涉、衍射、偏振）完整呈现了光的波动性；下一章将进入光的粒子性与量子世界。
:::
