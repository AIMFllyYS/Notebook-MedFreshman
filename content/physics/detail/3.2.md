# 3.2 简谐振动的合成

## 为什么要研究振动合成


::figure{src="/images/physics/3-大学物理学习指导-第三章-振动/47f22ca2ed06bbebac091906ef00e891dd988e0ca9f723efc603e98624f41106.jpg" alt="" caption=""}

实际振动往往是多个简谐分量叠加的结果：两束光干涉、两音叉同时发声、心电与脑电的频谱分析，都建立在"合成"与"分解"之上。本节先学**合成**，再简要介绍**傅里叶分解**——合成的逆过程。

:::insight{label=同频合成的核心结论}
同方向、同频率的两个简谐振动合成后，**仍是简谐振动**，频率不变，只是振幅和初相位改变。这一结论使大量问题可化为矢量加法一步解决。
:::

## 同振动方向、同频率的合成

设两振动：

$$
x_1 = A_1\cos(\omega t + \varphi_1), \quad x_2 = A_2\cos(\omega t + \varphi_2)
$$

合振动 $x = x_1 + x_2 = A\cos(\omega t + \varphi)$，仍为**同频率**简谐振动。由**旋转矢量法**（平行四边形法则）：

$$
A = \sqrt{A_1^2 + A_2^2 + 2A_1 A_2 \cos(\varphi_2 - \varphi_1)}
$$

$$
\tan\varphi = \frac{A_1\sin\varphi_1 + A_2\sin\varphi_2}{A_1\cos\varphi_1 + A_2\cos\varphi_2}
$$

### 讨论

**1. 同相** $\varphi_2 - \varphi_1 = 2k\pi$：

$$
A_{\max} = A_1 + A_2
$$

合振幅**最大**。

**2. 反相** $\varphi_2 - \varphi_1 = (2k+1)\pi$：

$$
A_{\min} = |A_1 - A_2|
$$

合振幅**最小**；若 $A_1 = A_2$，则 $A = 0$（完全抵消）。

**3. 其他相位差**：$|A_1 - A_2| < A < A_1 + A_2$。

:::example{label=同向同频 SHM 的合成}
$y_1 = 5\cos(\pi t - \pi/2)$ cm，$y_2 = 15\cos(\pi t + \pi/2)$ cm。相位差 $\varphi_2 - \varphi_1 = \pi$（反相），$A = |15 - 5| = 10$ cm（选 B）。
:::

## 同方向、不同频率的合成

两个或更多**同方向、不同频率**的简谐振动合成，合振动**一般不再是简谐振动**，但仍是**周期性运动**（频率为各分频的最小公倍数时）。可用图示法（旋转矢量相对转动）理解波形。

### 拍（beat）

:::definition{label=拍}
两同方向、**振幅与初相位相同**、频率**非常接近**的振动合成，合振动振幅周期性**强弱交替**，这种现象叫**拍**。
:::

设 $x_1 = A\cos(\omega_1 t + \varphi)$，$x_2 = A\cos(\omega_2 t + \varphi)$，当 $|\omega_1 - \omega_2| \ll \omega_1, \omega_2$ 时，合振幅缓变、载波频率较高。**拍频**（振幅变化频率）：

$$
\nu_{\text{拍}} = |\nu_1 - \nu_2|
$$

拍在声学（调音）、光学、无线电中广泛应用：两音叉频率接近时听到"嗡—嗡"的强弱变化。

::video{id=phys-ch03-3.2-kp2-beat-phenomenon}

**拍现象的数学简述**：利用和角公式，$x_1 + x_2 = 2A\cos\dfrac{(\omega_1 - \omega_2)t}{2}\cos\dfrac{(\omega_1 + \omega_2)t}{2}$。后一因子为"快振动"（载波），前一因子为"慢变化"的振幅包络，其频率为 $\dfrac{|\omega_1 - \omega_2|}{2\pi} = |\nu_1 - \nu_2|$。调音时，待调弦与标准音叉的拍频趋近于零，表示两弦频率一致。

**不同频同向合成**：合振动仍周期，但波形复杂；只有当各分频之比为有理数时，周期才有限。傅里叶级数告诉我们，任何此类周期波都可再分解为 SHM 之和——这是下一小节与频谱分析的桥梁。

**同相与反相的物理图像**：两扬声器同相发声，某点声压振幅相加，听感更响；反相时，若路径差为零且 $A_1=A_2$，该点几乎无声——噪声抵消耳机即利用反相干涉。**互相垂直合成小结**：$\varphi_2 - \varphi_1 = 0$ 或 $\pi$ 时轨迹为直线段（往返振动）；$\pm\pi/2$ 时为椭圆（或圆）；一般值为斜椭圆。轨迹形状只由 $A_1$、$A_2$ 和 $\Delta\varphi$ 决定，与绝对相位无关——这简化了许多作图题。

## 谐振分析（傅里叶级数）

:::theorem{label=傅里叶理论}
任何一个**复杂的周期性振动**，可分解为一系列简谐振动之和，各分振动频率为基频 $\nu$ 的整数倍：$\nu$、$2\nu$、$3\nu$……

$$
f(t) = A_0 + \sum_{n=1}^{\infty} A_n\cos(n\omega t + \varphi_n)
$$

称为**傅里叶级数**。基频 $\nu_{\text{分}} = \nu_{\text{合}}$；$n\nu$ 为 $n$ 次**谐频**（倍频）。
:::

**频谱分析**：以频率为横轴、振幅为纵轴作图，得**频谱图**。锯齿波、脉搏波形、心电、脑电均可作频谱分析——Quiet 状态下脉搏及其频谱即为一例。

:::insight{label=合成与分解是一对}
合成：多个 SHM → 复杂周期运动；分解（傅里叶）：复杂周期运动 → 多个 SHM。现代信号处理的根基就在于此。
:::

## 同频率、互相垂直的合成

质点同时参与：

$$
x = A_1\cos(\omega t + \varphi_1), \quad y = A_2\cos(\omega t + \varphi_2)
$$

联立消去 $t$ 得**合轨迹方程**。

| $\varphi_2 - \varphi_1$ | 轨迹 |
|-------------------------|------|
| 0 | 斜率 $A_2/A_1$ 的线段 |
| $\pi$ | 斜率 $-A_2/A_1$ 的线段 |
| $\pm\pi/2$ | 椭圆；$A_1 = A_2$ 时为**圆** |
| 其他 | 斜椭圆 |

**激光合成**、偏振光等用到互相垂直振动的叠加。

## 互相垂直、不同频率——李萨如图形

:::definition{label=李萨如图形}
$x$、$y$ 方向均为简谐振动，频率比为**简单整数比**时，合轨迹为稳定的**李萨如图形**（Lissajous figures）。图形与水平轴、竖直轴的**切点数**与两方向频率成反比。
:::

例如 $f_x : f_y = 2 : 3$，图形在 $x$ 方向切 3 次、$y$ 方向切 2 次。频率比不同，图形形状各异，可用于演示或校准。

::video{id=phys-ch03-3.2-kp3-perpendicular-lissajous}

**垂直同频合成的应用**：当 $\varphi_2 - \varphi_1 = \pm\pi/2$ 且 $A_1 = A_2$ 时，轨迹为圆——这是**圆偏振**的基础图像。

**锯齿波与傅里叶级数**：PPT 中锯齿波可展开为 $A_0 + \sum (A_n\cos n\omega t + B_n\sin n\omega t)$，频谱在基频 $\nu$ 及其整数倍 $2\nu, 3\nu, \ldots$ 处有峰，振幅随 $n$ 增大而减小。理解这一点，就明白为何任意复杂周期波形都可由简谐分量"拼"出来——合成与分解是同一枚硬币的两面。

**同频合成公式速记**：$\Delta\varphi = 0$ 时 $A = A_1 + A_2$；$\Delta\varphi = \pi$ 时 $A = |A_1 - A_2|$；一般情况用旋转矢量平行四边形法则最不易错。写合振动方程 $x = A\cos(\omega t + \varphi)$ 时，$A$、$\varphi$ 均由分振动唯一确定，频率 $\omega$ 与分振动相同——这是"同频合成仍为 SHM"的代数体现。

**拍频应用**：调钢琴时，校准音叉与琴弦同时发声，拍频 $|\nu_1 - \nu_2|$ 越接近零，两音越准；当拍频为零时音高一致。无线电超外差接收机也利用本振与外来信号的差频（中频）实现选频放大——拍现象从声学延伸到电子技术，原理相同。

**小结提示**：振动合成章节的核心工具是旋转矢量法与相位差；遇到不同频率时先判断能否用拍或傅里叶分解，垂直方向则联立 $x(t)$、$y(t)$ 消去 $t$ 得轨迹方程。

:::pitfall{label=合成问题易错点}
1. 同频合成必用**相位差** $\varphi_2 - \varphi_1$，不是单独 $\varphi_1$ 或 $\varphi_2$；
2. 反相时 $A = |A_1 - A_2|$，注意是绝对值；
3. 不同频合成一般**不是** SHM，勿强行用 $A = A_1 + A_2$；
4. 互相垂直合成消去 $t$ 时，注意 $\cos^2 + \sin^2$ 的化简。
5. **李萨如图形**频率比 $f_x : f_y = n : m$（整数）时，图形闭合；切点数与频率成反比，示波器上数切点可反推频率比。
:::

:::note{label=本节小结}
- **同向同频**：合振动仍为 SHM，$A = \sqrt{A_1^2 + A_2^2 + 2A_1A_2\cos\Delta\varphi}$；同相加强、反相减弱；
- **拍**：频率接近、振幅相同 → 拍频 $|\nu_1 - \nu_2|$；
- **傅里叶分解**：周期振动 = 基频 + 谐频之和；**频谱分析**显示各分量的贡献；
- **垂直同频**：轨迹为线、圆或椭圆，由 $\Delta\varphi$ 决定；
- **李萨如图形**：频率整数比 → 稳定闭合曲线。

:::
