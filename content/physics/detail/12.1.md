:::insight{label=经典物理在"紫外区"撞墙}
19 世纪末，人们用经典电磁理论和能量均分原理去解释黑体辐射，短波部分预测能量趋于无穷大——"紫外灾难"。普朗克 1900 年提出能量**不连续**，一份一份地交换，才与实验曲线完美吻合。这是量子概念的起点：能量有最小单元 $h\nu$。
:::

:::definition{label=热辐射与黑体}
**热辐射**：物体以电磁波形式向外传播能量，辐射能的多少及波长分布与温度有关。

**单色辐出度** $M_\lambda$：单位波长间隔内的辐出度。**辐出度** $M = \int M_\lambda\,\mathrm{d}\lambda$。

**黑体**：只吸收不反射的物体；带小孔的空腔可近似黑体。吸收能力最强 → 辐射能力最强。
:::

实验规律：
1. 辐射能随温度升高而增加——**斯特藩-玻尔兹曼定律** $M = \sigma T^4$
2. 单色辐出度曲线有峰值，温度升高峰向短波移动——**维恩位移定律** $\lambda_m T = b$

:::theorem{label=普朗克黑体辐射公式}
普朗克公式在全波段与实验惊人符合：

$$
M_\lambda(\lambda, T) = \frac{2\pi h c^2}{\lambda^5} \cdot \frac{1}{e^{hc/(\lambda kT)} - 1}
$$

其中 $h = 6.626 \times 10^{-34}\,\mathrm{J\cdot s}$ 为**普朗克常数**。
:::

:::definition{label=能量量子化假设}
物体辐射或吸收能量 $E = n\varepsilon = nh\nu$（$n = 1,2,3,\ldots$），$\varepsilon = h\nu$ 为**能量子**——能量不连续，存在最小单元。必须抛弃能量连续取值的经典观念。
:::

经典理论：维恩公式只适用于短波；瑞利-金斯公式只适用于长波——在黑体辐射研究中**经典物理失败**。

:::example{label=人体热辐射功率}
人体近似黑体，表面积 1.73 m²，温度 306 K，求辐射总功率。

**解**：由斯特藩-玻尔兹曼定律 $P = \sigma S T^4$，$\sigma = 5.67 \times 10^{-8}\,\mathrm{W/(m^2\cdot K^4)}$，

$P \approx 5.67 \times 10^{-8} \times 1.73 \times 306^4 \approx 900\,\mathrm{W}$（需考虑与环境交换，净辐射功率更小）。
:::

## 光电效应

:::definition{label=光电效应实验规律}
（1）光电流 $I$ 随电压增加趋于饱和；**饱和光电流** $I_S$ 与入射光**强度**成正比

（2）存在**遏止电压** $U_a$，$E_{k,\max} = eU_a$

（3）$E_{k,\max}$ 与光**频率**线性相关，与强度无关

（4）存在**红限频率** $\nu_0$，$\nu < \nu_0$ 时不发生光电效应

（5）响应时间极短（$< 10^{-9}$ s）
:::

经典电磁理论困难：光强应决定电子最大动能；电子需积累能量克服逸出功，不可能瞬时发生。

:::theorem{label=爱因斯坦光子理论与光电方程}
光是一束以光速运动的光子流，每个频率 $\nu$ 的光子能量 $\varepsilon = h\nu$。

**爱因斯坦光电方程**：

$$
h\nu = W + E_{k,\max}
$$

$W$ 为**逸出功**；红限频率 $\nu_0 = W/h$，红限波长 $\lambda_0 = hc/W$。

解释：（1）光强大 → 光子多 → 饱和电流大；（2）一个光子被一个电子一次性吸收；（3）$\nu < \nu_0$ 时单个光子能量不足以逸出。
:::

::plot{fn="x-1" xmin=0 xmax=4 ymin=-1.5 ymax=3 label="光电效应：eUa = h(ν-ν₀)，截止频率 ν₀ 处 Ek=0"}

:::theorem{label=光子的质量与动量}
光子静止质量为零。由相对论：$E = pc$，故 $p = h/\lambda = h\nu/c$。

**波粒二象性**：干涉衍射证明波动性；光电效应证明粒子性。
:::

:::example{label=钾的光电效应}
钾的红限 $\lambda_0 = 6.2 \times 10^{-7}\,\mathrm{m}$，（1）求逸出功；（2）$\lambda = 3.0 \times 10^{-7}\,\mathrm{m}$ 紫外线照射，求遏止电压；（3）电子初速度。

**解**：（1）$W = hc/\lambda_0 = 6.63 \times 10^{-34} \times 3 \times 10^8 / (6.2 \times 10^{-7}) \approx 3.2 \times 10^{-19}\,\mathrm{J}$

（2）$E_{k,\max} = hc/\lambda - W$，$U_a = E_{k,\max}/e \approx 1.6\,\mathrm{V}$

（3）由 $E_{k,\max} = \frac{1}{2}mv^2$ 求 $v$。
:::

## 康普顿效应

:::definition{label=康普顿效应}
X 射线与物质散射时，除波长 $\lambda_0$ 成分外，还有 $\lambda > \lambda_0$ 的成分；波长改变量 $\Delta\lambda$ 仅与散射角 $\theta$ 有关，与物质无关（对同一 $\theta$）。
:::

**量子解释**：X 射线光子与**静止自由电子**弹性碰撞，光子损失能量 → 波长增大。光电效应是光子与**束缚电荷**作用；康普顿是光子与**自由电子**的弹性碰撞。

:::theorem{label=康普顿公式}
$$
\Delta\lambda = \lambda - \lambda_0 = \frac{h}{m_0 c}(1 - \cos\theta) = \lambda_C(1 - \cos\theta)
$$

$\lambda_C = h/(m_0 c) = 2.43 \times 10^{-12}\,\mathrm{m}$ 为**康普顿波长**。$\theta$ 越大，$\Delta\lambda$ 越大。

光子与**内层束缚电子**（相当于与整个原子）碰撞，原子质量远大于光子，波长几乎不变。
:::

::video{id=phys-ch12-12.1-kp4-compton-scattering-geometry}

:::example{label=康普顿散射计算}
波长 2.0 Å 的 X 射线在碳上散射，波长改变 0.04%，求散射角及反冲电子能量。

**解**：（1）$\Delta\lambda = 0.0004 \times 2.0 \times 10^{-10} = 8 \times 10^{-14}\,\mathrm{m}$

$1 - \cos\theta = \Delta\lambda/\lambda_C$，解得 $\theta \approx 28°$

（2）反冲电子动能 = 光子损失能量 $\Delta E = hc(1/\lambda - 1/\lambda')$。
:::

::video{id=phys-ch12-12.1-ex1-compton-scatter-90deg}

:::pitfall{label=光电效应 vs 康普顿效应}
- **光电效应**：电子**吸收**光子，一般不讨论动量守恒
- **康普顿效应**：光子与电子**弹性碰撞**，能量与动量均守恒
- 可见光康普顿效应极弱，因 $\Delta\lambda$ 相对原波长太小
:::

## 量子力学发展历程

1900 普朗克能量子 → 1905 爱因斯坦光子 → 1913 玻尔原子模型 → 1924 德布罗意物质波 → 1925–1927 海森堡、薛定谔、狄拉克建立量子力学 → 1926 玻恩统计解释 → 1927 电子衍射验证。1927 年索尔维会议汇集爱因斯坦、玻尔、薛定谔、海森堡等大师，讨论量子力学诠释。

维恩位移定律 $\lambda_m T = b$（$b = 2.898 \times 10^{-3}\,\mathrm{m\cdot K}$）可估算星体表面温度：测出辐射峰值波长，反推 $T$。

:::example{label=维恩定律应用}
某星体辐射峰值 $\lambda_m = 290\,\mathrm{nm}$，则 $T = b/\lambda_m \approx 10^4\,\mathrm{K}$ 量级（具体数值依 $b$ 与 $\lambda_m$ 计算）。
:::

普朗克公式在长波极限 $\to$ 瑞利-金斯，短波极限 $\to$ 维恩公式，统一了经典两极端。

太阳能电池、光电倍增管、光电器件均基于光电效应——光子能量必须大于逸出功，因此存在红限；提高转换效率需匹配材料禁带宽度与太阳光谱。

康普顿效应中，可见光 $\Delta\lambda \sim 10^{-12}\,\mathrm{m}$ 相对 $\lambda \sim 10^{-7}\,\mathrm{m}$ 可忽略，故用可见光难以观测；X 射线 $\lambda \sim 10^{-11}\,\mathrm{m}$，$\Delta\lambda$ 相对显著。康普顿效应除揭示光子动量外，还表明微观碰撞中能量、动量守恒依然成立——守恒定律是物理学的普遍基石。

普朗克常数 $h$ 是量子力学与近代物理的基石，其数值需与 $c$、$e$、$m_e$ 等常数配合使用。斯特藩常数 $\sigma = 5.67 \times 10^{-8}\,\mathrm{W/(m^2\cdot K^4)}$，维恩常数 $b \approx 2.898 \times 10^{-3}\,\mathrm{m\cdot K}$，在热辐射估算题中频繁出现。

:::pitfall{label=光电 vs 康普顿系统守恒}
例3 结论：光电效应中电子吸收光子，**不严格遵循动量守恒**（原子核参与但常被忽略）；康普顿效应为光子与自由电子弹性碰撞，**能量与动量均守恒**（答 D）。
:::

:::note{label=本节小结}
- 普朗克：**能量量子化** $E = nh\nu$，$h = 6.626 \times 10^{-34}\,\mathrm{J\cdot s}$。
- 光电方程 $h\nu = W + E_{k,\max}$；红限 $\nu_0 = W/h$。
- 康普顿公式 $\Delta\lambda = \lambda_C(1-\cos\theta)$，进一步证实光子动量。
- 三大实验（黑体、光电、康普顿）共同建立光的**粒子性**图景。

下一节：玻尔用"量子化轨道"解释氢原子光谱——旧量子论的高峰。黑体辐射题常考维恩位移（峰值波长与温度反比）与斯特藩-玻尔兹曼（总功率与 $T^4$ 成正比），光电效应题则重点用 $h\nu = W + E_k$ 求红限与遏止电压。
:::
