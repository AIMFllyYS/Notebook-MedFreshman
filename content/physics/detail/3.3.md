# 3.3 阻尼振动、受迫振动和共振

## 从理想 SHM 到真实世界

弹簧振子在真空中可永远振动；在空气或液体中，摩擦阻力使振幅逐渐减小。这种能量不断损耗、振幅衰减的振动叫**阻尼振动**（damped vibration）。若再施加周期性外力，则出现**受迫振动**；当驱动频率接近系统固有频率时，可能发生**共振**——振幅急剧增大，既有应用也有危害。

:::insight{label=为什么阻尼与共振都重要}
汽车减震（临界阻尼）、乐器共鸣（共振）、桥梁风振与飞机颤振（共振危害）、微波炉（受迫吸收）——从日常到工程，离开阻尼和共振几乎无法讨论真实振动系统。
:::

## 阻尼振动

振动系统不在真空中，而在媒质中运动时受到**摩擦阻力**，振动能量（振幅）逐渐减小。

阻尼振动的运动方程（线性阻尼）：

$$
m\frac{d^2 x}{dt^2} + \gamma \frac{dx}{dt} + kx = 0
$$

引入 $\beta = \gamma/(2m)$，$\omega_0 = \sqrt{k/m}$，解的形式取决于 $\beta$ 与 $\omega_0$ 的关系：

:::definition{label=阻尼振动的三种状态}
1. **欠阻尼**（underdamping）$\beta < \omega_0$：振幅按指数衰减，仍作往复振动，周期略大于无阻尼时的 $T_0$；

2. **临界阻尼**（critical damping）$\beta = \omega_0$：最快回到平衡位置而**不越过**——汽车减震器、电表指针的理想设计；

3. **过阻尼**（overdamping）$\beta > \omega_0$：缓慢回到平衡，无往复。
:::

欠阻尼时位移可写为 $x = A e^{-\beta t}\cos(\omega' t + \varphi)$，其中 $\omega' = \sqrt{\omega_0^2 - \beta^2}$。**振幅 $A$ 随时间越来越小**（选 A），不是时大时小或不变。

## 受迫振动

:::definition{label=受迫振动}
系统在**周期性外力** $F = F_0\cos\Omega t$ 作用下的振动，称为**受迫振动**（forced vibration）。$F_0$ 为外力幅度，$\Omega$ 为驱动角频率。
:::

运动方程：

$$
m\frac{d^2 x}{dt^2} + \gamma \frac{dx}{dt} + kx = F_0\cos\Omega t
$$

稳态时质点作与驱动力**同频率**的受迫振动 $x = A\cos(\Omega t + \varphi)$，振幅 $A$ 与 $\Omega$、阻尼 $\beta$、固有频率 $\omega_0$ 有关。当阻尼较小且 $\Omega \approx \omega_0$ 时，**振幅显著增大**。

## 共振

:::theorem{label=共振}
当驱动频率 $\Omega$ 接近系统**固有频率** $\omega_0$ 时，受迫振动**振幅达到极大**，这种现象称为**共振**（resonance）。数学上对振幅 $A(\Omega)$ 求极值，令 $\dfrac{dA}{d\Omega} = 0$ 得共振条件。
:::

**共振的应用**：

- 乐器：弦、空气柱在固有频率处共鸣；
- 微波炉：使水分子在特定频率下受迫吸收能量；
- 核磁共振：原子核在特定磁场频率下共振吸收。

**共振的危害**：

- 1940 年塔科马海峡大桥风振；
- 飞机、桥梁设计须使固有频率避开常见激励频率；
- 雪崩、机械损坏等。

演示：360° 秋千、鱼洗（搓盆使水溅起）——在合适频率下反复施力，能量累积。

**阻尼振动的周期**：欠阻尼时 $x = A e^{-\beta t}\cos(\omega' t + \varphi)$，$\omega' = \sqrt{\omega_0^2 - \beta^2} < \omega_0$，故周期 $T' = 2\pi/\omega' > T_0$——阻尼使"表观周期"略变长，振幅包络 $A e^{-\beta t}$ 指数衰减。品质因数 $Q = \omega_0/(2\beta)$ 越大，衰减越慢、共振峰越 sharp。

**受迫振动稳态振幅**：$A(\Omega) = \dfrac{F_0/m}{\sqrt{(\omega_0^2 - \Omega^2)^2 + (2\beta\Omega)^2}}$。当 $\Omega = \omega_0$ 且 $\beta$ 很小时，分母最小，$A$ 最大——共振。若 $\beta$ 较大，即使 $\Omega = \omega_0$，振幅也不会无穷大；实际系统总有阻尼，共振是"相对极大"而非数学奇点。

**共振的工程考量**：设计桥梁、飞机机翼时，需计算结构固有频率，避免与风振、发动机振动、步行频率（如士兵齐步走）重合。增加阻尼（减振器）、改变刚度或质量分布，都是常用的调频与抑振手段。临界阻尼在仪表指针、门窗闭门器中常用：快速回位、不来回摆动。

**三种阻尼状态的对比**：欠阻尼——汽车弹簧若阻尼过小，过坑会多次弹跳；过阻尼——门后阻尼铰链关闭缓慢、无回弹；临界阻尼——电梯水平仪、电表指针一次到位。受迫振动中，驱动力频率远离 $\omega_0$ 时，系统响应接近驱动振幅 $F_0/k$；接近 $\omega_0$ 时响应放大，放大倍数由 $Q$ 值决定。**生活联系**：推秋千每周期在相同相位推一把，即"共振式"能量输入；若随机乱推，能量耦合弱，振幅不易维持。

**从 SHM 到受迫振动**：自由 SHM 是理想、无耗散、无驱动的特例；阻尼振动加入 $-\gamma v$ 项；受迫振动再加入 $F_0\cos\Omega t$。三者方程逐级推广，解的形式从简谐衰减到稳态振幅 $A(\Omega)$——这一脉络贯穿振动学全章，复习时建议并排比较三种方程与解的图像。

**共振实例补充**：微波炉中水分子本征频率与微波频率匹配，能量高效吸收；乐器共鸣箱使空气柱在特定频率驻波加强，音色更丰满；1940 年塔科马海峡大桥因风致涡振频率接近桥体固有频率而垮塌，是共振危害的典型案例。设计任何振动系统时，都应列出可能的激励频率，并与固有频率对比。

**减振措施**：增大阻尼（$m$ 大、$k$ 小——如大理石板 vs 充气轮胎）、改变固有频率、避免长期处于共振频率驱动。

:::example{label=阻尼与受迫的判断}
阻尼振动：振幅单调减小。受迫振动稳态：振幅恒定（由 $F_0$、$\Omega$、$\beta$ 决定）。若 $\Omega = \omega_0$ 且 $\beta$ 小，该恒定振幅可以很大——即共振。
:::

## 与前两节的关系

| 类型 | 方程特点 | 振幅 |
|------|----------|------|
| 自由 SHM | 无阻尼、无外力 | 恒定 |
| 阻尼振动 | 有阻尼、无外力 | 衰减 |
| 受迫振动 | 有阻尼、有周期外力 | 稳态恒定 |
| 共振 | 受迫 + $\Omega \approx \omega_0$ | 稳态极大 |

:::pitfall{label=共振相关易错点}
1. 共振不是"任意频率都振幅大"，只在 $\Omega$ 接近 $\omega_0$ 时显著；
2. **阻尼越大，共振峰越宽、越低**——完全无阻尼时理论振幅趋于无穷（理想情况）；
3. 临界阻尼是**最快回归平衡且不振荡**，不等于"振幅最大"或"振幅最小"。
4. **受迫振动**稳态频率恒等于驱动力频率，与系统固有频率无关——只有振幅随 $\Omega$ 变化。
:::

:::note{label=本节小结}
- **阻尼振动**：媒质摩擦 → 振幅衰减；欠阻尼、临界阻尼、过阻尼三种状态；
- **受迫振动**：周期外力 $F_0\cos\Omega t$ → 稳态同频振动；
- **共振**：$\Omega \approx \omega_0$ 时振幅极大；应用（乐器、微波炉）与危害（桥梁、飞机）并存；
- 减振：增大阻尼、调整 $m$ 与 $k$、避开危险频率。

至此第三章完成：从 SHM 基本方程，到合成与分解，再到阻尼与共振，构成振动学的完整链条。
:::
