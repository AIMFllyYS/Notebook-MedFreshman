说话人 1 01:36

说话人 1 02:37
X 射线波长大概在 1 Å 到 10 Å（0.1 纳米到 1 纳米）。伦琴第一次发现了 X 射线。

说话人 1 03:00
X 射线穿透力强，在医用里应用很多。这是一个简单的 X 射线管，有整流电路、低压电源和高压电源。要有高速移动的电子或电子流，怎么产生？比如一根钨丝，接一个 220V 电压通过变压器变成低压。

说话人 1 03:42
低压电源相当于灯泡一样把电阻丝烧热，加热后一些自由电子（尤其外层电子）就变成自由电子。还有一个高压电路，通过 220 伏电压升压变成高压（这是整流电路）。有自由电子后加高压，电子就打到阳极上。为什么有整流电路？因为变压过来是交流电，电压方向一直变，加速到一半会往回加速。整流电路只让电子往一个方向加速（这边正这边负），把交流高压变成脉冲直流高压。这个高压能到几十千伏到几百千伏，非常高。

说话人 1 05:07
加速电子后，电子速度很快打到阳极靶上产生 X 射线。这是医用的：阴极发射电子（灯丝），阳极靶受高速电子轰击产生 X 射线，用一个 filter 让固定方向的出来。射线管里的电压叫管电压。X 射线管里一般是超高真空，因为电子很轻，不真空的话会和气体分子碰撞，速度一下降下来，而且保证灯丝不被氧化。

说话人 1 06:20
最早白炽灯里加惰性气体（虽没抽真空）免得氧化。X 射线也是加热灯丝产生自由电子，通过高压加速撞击阳极靶产生 X 射线，透过人体成像。产生自由电子后通过管高压加速时通常需要把电子聚焦到一个区域，能量更集中。有圆形聚焦和线焦点聚焦，出来的 X 射线分辨率不一样。

说话人 1 07:33
阴极产生的电子各个方向都有，加压时不一定直接往阳极跑，可能往各方向跑。所以阴极背后像电筒一样。

说话人 1 08:00
电筒后面做成凹形把光聚焦。阴极后面有一个物质的聚焦杯，把所有电子聚焦后再通过高压加速。阳极这边也有一个聚焦杯，把散射的或没撞到阳极靶的电子吸收，免得高速电子直接打到玻璃罩或壁上损坏仪器（超高真空，稍微打漏真空度就保持不住）。

说话人 1 08:47
钨质的聚焦杯主要聚焦电子，还有收集二次电子防止危害。二次电子就是电子直接打到阳极时能量没完全损失反射回来的，聚焦杯也可以收集住，免得电子乱撞损害仪器。

说话人 1 09:24
阴极、阳极靶有什么要求？高速电子打阳极靶，电子动能转化为：一部分变成 X 射线（有能量），绝大部分变成热。所以阳极靶材要求熔点高、导热性好、原子序数大。为什么原子序数大？X 射线产生机理是原子序数大时电子打进去产生 X 射线概率高，更容易产生 X 射线。

说话人 1 10:06
阳极导热性好、熔点高，通常用钨和钼（原子序数高、耐高温）。把钨和钼贴在铜质衬底上（铜导热性好），钨的热传到铜，铜很大一块可以把热散出去。

说话人 1 10:37
功率不太高的（如牙科 X 射线，剂量低）被动散热就可以。功率高的 X 射线（电流密度大，阳极加热厉害），阳极可能要旋转起来，让电子束不集中在一处，或在阳极后面制冷（水冷、油冷）。这就是阳极。

说话人 1 11:27
X 射线强度就是 X 射线的量：单位时间内通过与 X 射线方向垂直的单位面积的辐射量（瓦每平方米）。通常用管电流毫安数间接表示（称为毫安率），因为不容易直接测 X 射线强度。

说话人 1 11:56
垂直于 X 射线的单位面积辐射能量不太容易测，但电流毫安数比较好测。加高压后电子从阴极流向正极形成电流，可以测出来，相当于自由电子的多少和速度。可以间接用这个电流表示 X 射线强度。

说话人 1 12:47
X 射线就是波长极短的光，每个光子能量 $h\nu_i$。X 射线波长（频率）不一定固定，有各种频率。$h\nu_1$ 是 $\nu_1$ 频率单个光子能量，这个频率有 $n_1$ 个光子，乘起来求和就是 X 射线整个强度。用波长表示就是从最小波长到正无穷积分，就是 X 射线整个强度。要增加 X 射线强度有什么方法？

说话人 1 14:03
第一增加 $N$（光子数），第二增加单个光子能量。增加光子数就是增加电流密度（一个电子换一个光子，电流密度越大光子数越多）。单个光子能量和电子能量有关，电子能量越大换出来的光子能量越大。

说话人 1 14:51
X 射线强度与阳极原子序数有关。同样速度的电子打到原子序数大的阳极靶，X 射线强度大（$N$ 多）。原子序数小的靶材可能 10 万个电子才打出一个 X 射线。

说话人 1 15:22
X 射线管热阴极温度越高、管电流越大，就是增加 $N$（阴极温度越高自由电子越多，管电流越大）。管压越大单个电子能量越高（$eU$），放出的单个光子能量越大。这说的是强度。X 射线总辐射量不仅与强度有关，还与时间有关，与管电流成正比，也与辐照时间成正比（辐照总量是毫安乘以时间秒）。

说话人 1 16:22
X 射线还有一个度叫硬度——穿透物体的能力（贯穿本领），只取决于 X 射线波长及单光子能量。单个光子能量越强穿透效率越高（能量越强波长越短），所以波长越短穿透能力越强，与光子数量无关，这是硬度。

说话人 1 17:03
硬度只与单个光子能量有关，与光子数量无关。单个光子能量与单个电子能量有关，单个电子能量与管电压有关。管电压越高电子速度越快，产生的 X 射线波长越短，被物体吸收越小，穿透能力越强。可以用管电压表示。

说话人 1 17:40
用管电压表示，比如较软的 X 射线管电压在 5~20 千伏，波长 0.25 到很短，软组织摄影。软的 X 射线 20~100 千伏，拍片（骨科、牙科）。

说话人 1 18:12
较硬的 X 射线用于较深组织治疗（不是成像）。X 射线有能量，但 X 射线治疗现在可能被替代，因为有辐射，而且聚焦没有其他技术好（具有波动性，聚焦不好）。

说话人 1 18:58
X 射线谱：连续 X 射线和标识 X 射线。X 射线怎么形成的？叫韧致辐射。韧致辐射是 X 射线管中高速电子撞击靶时受电场作用，原子核带正电，即使没撞到原子核，库仑力也对它做功，绕一下后能量降低，能量降低以 X 射线形式辐射出去。

说话人 1 19:48
因为受核电场作用急剧减速，电子能量降低转化为光子（X 射线）辐射出来。减速不是固定的，和电子离原子核的距离有关：近一点减速多，远一点减速少。所以韧致辐射出来的 X 射线波长是连续的。

说话人 1 20:38
这也回答了为什么原子序数越大 X 射线强度越大。不是因为原子核大小（$10^{-15}$ 量级，都差不多）。

说话人 1 21:09
是核电场和质子数有关。原子核内全是带正电的，序数小带电量小。序数大原子核带正电大，核电场大，作用半径大，转化效率高。

说话人 1 21:47
一个电子进来，离原子核很远可能一点速度都不减就出去，离得近就减速，能量损失以 X 射线发射。管电压不同，辐射出的 X 射线最低波长往更小的波长走（能量更高），辐射出的 X 射线强度（曲线下面积）越来越大。对任意管电压都有一个截止波长。这个波长怎么理解？韧致辐射中电子能量损失转化成 X 射线，能转化的最多的 X 射线就是电子能量全部损失。

说话人 1 23:00
最大 X 射线能量等于电子能量 $eU$，所以最大频率 $h\nu_{max} = eU$，最小波长 $\lambda_{min} = \frac{hc}{eU}$，$hc$ 和 $e$ 都是常数带进去。

说话人 1 23:39
得到 $\lambda_{min} = \frac{1.22}{U}$（书上有这个公式），注意单位，$U$ 取千伏时 $\lambda$ 单位是纳米。要激发更短波长的 X 射线，管电压要特别大。增加管电压，截止波长往低频推。这是连续 X 射线谱，加到 50 kV 还是连续谱，加到 65 kV 基本还是连续谱。

说话人 1 24:50
加到 100 kV 后就有独立的峰出来（四个峰）。对钨靶加到 70 kV 以上就会看到特征 X 射线。特征 X 射线对应的波长是钨的，特征是不随管电压变化而变化。截止的最小 $\lambda$ 随管电压增加往更短波长移动，但特征谱一旦出现就不动。

说话人 1 25:40
怎么解释？电压增加到 70 千伏左右，X 射线在波长 0.02 纳米附近有 4 个比较高的峰。标识 X 射线波长仅取决于阳极靶材，换不同靶材波的位置不一样，但增加管电压峰的位置不变，所以称为标识 X 射线谱。每种元素的 X 射线波长固定，这几个标识线对应不同元素不变。

说话人 1 26:39
产生原因：高能电子从原子核里走一圈没带走东西。如果能量更高，能把内层电子打掉。如果能量大于内层电子逃逸成自由电子的能量，外来电子就把最内层（K 层）的电子打掉，里面就是一个空穴。最内层能量最低，外层电子（高能态）往里面跃迁，从 L 层、M 层、N 层跃迁过来。

说话人 1 27:52
所以看到一些峰，是外层电子向内层被打掉的电子跃迁导致的，和原子光谱非常类似。壳层间能量差大，发出光的能量高、频率大、波长短。看这张图，当管电压的最小波长刚好小于特征峰时就开始激发特征峰。最小能激发的 X 射线波长对应电子的最高能量。

说话人 1 29:10
当电子能量大于内层电子跃迁（变成自由电子）的能量，就会产生特征峰。X 射线衰减：X 射线有穿透性但还是会和物质相作用衰减。X 射线衰减主要形式（应该是上一章内容）：光电效应、康普顿散射和电子对效应。

说话人 1 30:00
光电效应：X 射线就是光子，光子到固体里去会打一个电子出来。光电效应和康普顿散射区别：光电效应把光子能量全部吸收，放一个自由电子出来；康普顿散射只拿走一部分，光子损失一部分能量波长变了。电子对效应：光子不带电，到里面去可能变成两个电子（一个正电子一个负电子），相互作用和能量有关——能量高时电子对效应占主导，康普顿效应在中间区间。纵轴是吸收物质原子序数，有分界线：一个区间光电效应占优势，下面康普顿占优势，右边电子对占优势。

说话人 1 31:26
X 射线衰减是因为和物质有相互作用。单色 X 射线衰减满足朗伯定律：强度对传播距离的导数等于 $-\mu I$，积分得 $I = I_0 e^{-\mu x}$，$x$ 是传播距离，是指数衰减。$\mu x$ 就是 X 射线在物质里传播 $x$ 距离被吸收了多少。

说话人 1 32:44
$\mu$ 叫线性衰减系数：X 射线穿过单位厚度物质薄层时强度减弱的百分数（单位厚度的作用概率），单位是厘米分之一。质量衰减系数：X 射线穿过冰、水、水蒸气衰减系数不一样（密度不一样）。同一种物质衰减系数一样，这个衰减系数叫质量衰减系数 $\mu_m = \frac{\mu}{\rho}$（线性系数除以密度）。

说话人 1 34:12
线性衰减系数在固态变气态、液态变气态时密度变化大、$\mu$ 变化大，但 $\frac{\mu}{\rho}$（质量衰减系数）值不变。引入质量厚度 $x_m$：厚度为 $x$ 的吸收体单位面积的质量，$x_m = \rho x$，单位千克每平方米。

说话人 1 35:16
质量厚度是单位面积的质量（千克每平方米）。朗伯定律 $I = I_0 e^{-\mu x}$，$\mu$ 除以 $\rho$ 再乘 $\rho$，$\frac{\mu}{\rho}$ 是质量衰减系数，$\rho x$ 是质量厚度，可以用质量衰减系数和质量厚度写朗伯定律。

说话人 1 36:12
半价层和质量半价层定义和半衰期非常类似。半价层是 X 射线走过多长距离强度变成一半（物质对 X 射线强度吸收一半的厚度）$x_{1/2}$。

说话人 1 37:00
半价层 $x_{1/2} = \frac{0.693}{\mu}$，质量半价层把 $\mu$ 换成 $\mu_m$。

说话人 1 37:36
和原子衰减一样，用半价层把 $\mu$ 代替，强度可写成 $I = I_0 (\frac{1}{2})^{x/x_{1/2}}$。

说话人 1 38:20
伦琴最早研究 X 射线不是故意的，是研究荧光发光时发现这种穿透力很强的射线，不知道是什么就命名为 X 射线。后面劳厄通过衍射实验证明 X 射线具有波动性，拿了诺贝尔物理学奖（1914 年），证实 X 射线是频率很高、波长很短的电磁波，波长范围相当于可见光的 10 万分之一（短的）到 $\frac{1}{50}$（长的）。

说话人 1 39:16
劳厄通过 X 射线衍射。难点在哪？要产生衍射（如单缝衍射），缝宽要和波长差不多才能产生明显衍射条纹。X 射线即使长的也是 10 纳米，通常是 Å 级别。当年（1910 年代）没有先进光刻技术，做不出这种缝。

说话人 1 40:09
所以他直接用自然界存在的东西做衍射——晶体。晶体原子之间的距离差不多就是纳米甚至 Å 量级，刚好和 X 射线波长匹配。实验：产生 X 射线，用一个缝让 X 射线平行，照到硫化铜晶体（原子规则排布）。你能用连续谱做吗？能看到很好的干涉、衍射条纹吗？就像用白光不能做衍射。

说话人 1 41:26
得用特征谱（一个波长）。不同波长形成的干涉条纹、衍射条纹位置不一样，连续谱得到的也是连续的。利用自然界中晶体（规则排列的原子）可看作立方光栅，原子线宽约 Å 量级（$10^{-10}$）。根据光栅衍射，只要 X 射线波长与晶体间隔同量级或更小就可以得到衍射。

说话人 1 42:23
劳厄得到衍射图证明 X 射线有波动性。证明波动性后，X 射线衍射的理论解释主要是布拉格父子。布拉格儿子二十岁就拿了诺奖，当时科学界觉得他抱了他爸大腿，但后面证明自己很厉害。他们用 X 射线做衍射，尤其是理论方面。晶格间距（原子间距）大约 $10^{-10}$，但不同物质原子间距不知道。

说话人 1 50:15
