# 第二十七讲 · X射线·激光及医学应用 · 课堂纪要

## 教学要点

### X 射线的产生
- 三种产生方式：受主辐射（医用，高速电子撞靶）、同步辐射（高速离子自发辐射）、受激辐射。
- 受主产生必备：高速电子流 + 阳极靶。X 射线波长约 1–10 Å（0.1–1 nm）；伦琴首次发现。
- X 射线管结构：钨丝经低压电源加热产生自由电子，高压电源（几十至几百千伏，称**管电压**）加速电子撞击阳极靶；需**整流电路**（把交流高压变脉冲直流，只让电子单向加速）；管内**超高真空**（防电子与气体碰撞减速、防灯丝氧化）。
- 阴极后置聚焦杯（聚焦电子）；阳极聚焦杯收集二次电子防损坏仪器。
- **阳极靶要求**：熔点高、导热好、原子序数大（序数大产生 X 射线概率高），常用钨、钼贴铜衬底散热；高功率时阳极旋转或水冷/油冷。

### X 射线的强度与硬度
- 强度：单位时间通过垂直单位面积的辐射量（W/m²），$I = \sum n_i h\nu_i$（从最短波长到正无穷积分）；不易直测，常用**管电流毫安数**（毫安率）间接表示。
- 增大强度：① 增加光子数（增大电流密度/灯丝温度→自由电子多）；② 增加单光子能量（增大管电压）；亦与阳极原子序数有关。
- 总辐射量 = 强度 × 时间（与管电流、辐照时间成正比）。
- **硬度**：穿透能力（贯穿本领），只取决于 X 射线波长与单光子能量（能量越大、波长越短、穿透越强），**与光子数量无关**；由管电压决定。较软（5–20 kV）软组织摄影，软（20–100 kV）拍片，较硬用于深部治疗。

### X 射线谱
- **连续谱（韧致辐射）**：高速电子受核电场（库仑力）作用急剧减速，能量以 X 射线辐射；减速程度与离核距离有关，故波长连续。原子序数大→核电场强→转化效率高→强度大。
- 截止（最短）波长：电子能量全部转化时 $h\nu_{max} = eU$，$\lambda_{min} = \dfrac{hc}{eU} = \dfrac{1.24}{U}$（U 取 kV 时 $\lambda$ 单位 nm）；管电压越高截止波长越短。
- **标识（特征）X 射线**：管电压足够高（如钨靶 >70 kV）时，电子打掉内层（K 层）电子留下空穴，外层电子（L/M/N）跃迁填补放出特征峰；波长**仅取决于靶材**、不随管电压变化，可用于元素标识。

### X 射线的衰减
- 衰减三机制：光电效应（光子能量全被吸收、放出自由电子）、康普顿散射（光子失部分能量、波长变长）、电子对效应（高能光子转化为正负电子对）。
- 朗伯定律：$I = I_0 e^{-\mu x}$，$\mu$ 为**线性衰减系数**（单位 cm⁻¹）。
- **质量衰减系数** $\mu_m = \dfrac{\mu}{\rho}$（同一物质不随物态/密度变化）；**质量厚度** $x_m = \rho x$（kg/m²）。
- **半价层** $x_{1/2} = \dfrac{0.693}{\mu}$（强度减半的厚度，类比半衰期）；$I = I_0(\tfrac{1}{2})^{x/x_{1/2}}$。

### X 射线衍射与布拉格方程
- 劳厄用晶体（原子间距约 Å，与 X 射线波长匹配）做衍射，证明 X 射线的波动性（1914 年诺奖）；需用特征谱（单一波长），连续谱得不到清晰条纹。
- 布拉格父子给出**布拉格方程** $2d\sin\theta = k\lambda$（相邻晶面反射光程差 $2d\sin\theta$ 等于波长整数倍时相长）。
- 原理：X 射线使晶格点电子受迫振动成为新波源（散射），满足条件时干涉相长。
- 应用：已知 $\lambda$、扫 $\theta$ 测峰即可求晶面间距 $d$（XRD 测晶格常数），不同晶面对应不同峰；反之知 $d$ 可求 $\lambda$。

### 激光

#### 特点
- 方向性极好（几千米外光斑只增大几厘米）、亮度（强度）极高（聚焦可达几百万度高温，用于氢聚变点火）、单色性好（氦氖激光波长变化 <$10^{-7}$ Å）、相干性好（相干长度可达几百公里，普通钠光灯仅几厘米）、频率高（信息容量大）。

#### 发光三过程
- **自发辐射**：电子自发由 $E_2$ 跃迁到 $E_1$ 放出光子 $h\nu = E_2 - E_1$；各原子独立、非相干，方向/波长/偏振各异，难应用。
- **受激辐射**：能量恰为能级差的光子诱导高能态电子跃迁，放出与入射光子**频率、相位、振动方向、传播方向全相同**的光子，光子数呈几何级数（$2^n$）增长——这是激光的基础。
- **受激吸收**：基态电子吸收光子跃迁到 $E_2$，光子数减少（与激光相反）。

#### 粒子数反转与泵浦
- 玻尔兹曼统计下低能态粒子数多于高能态；要让受激辐射占优，必须使高能态粒子数远超低能态，即**粒子数反转**。
- 用**泵浦**技术持续供能实现反转，需有合适能级结构的**激活物质**。
- 氦氖激光器：氦为辅助物质（1S、2S 亚稳态聚集大量原子），氖为激活物质（4S、5S）；氦氖碰撞产生能量共振转换，使氖达激发态形成粒子数反转；氖 5S 跃迁辐射 **632.8 nm 红光**（另有 1.15 μm、3 μm）。

#### 光学谐振腔
- 两反射镜（一面 100%、一面约 98%）：只有严格平行于轴的光子来回反射几百次后从 98% 镜透出，偏离轴的被淘汰，从而准直。
- 谐振腔长度精心设计使所需波长形成驻波共振、能量增强，其他波长相互抵消；配合布儒斯特窗统一偏振方向。
- 谐振腔作用：维持光振荡放大、保证方向性、保证单色性、产生合适偏振态、选择控制增强。
- 产生激光必要条件：受激辐射（需粒子数反转维持）+ 合适激活物质 + 光学谐振腔。

#### 激光应用
- 激光测绘、激光雷达（自动驾驶测距，可重构三维世界但点云数据量大、算力需求高）、激光切割、激光防伪、聚变点火。特斯拉采用视觉（摄像头）方案，国内多数新能源车加装激光雷达（成本已大幅下降）。

## 考试重点标注
- ⚠️ 截止波长公式 $\lambda_{min} = \dfrac{hc}{eU} = \dfrac{1.24}{U}$（注意单位，U 取 kV → $\lambda$ 取 nm）；只与管电压有关。
- ⚠️ 区分**连续谱（韧致辐射）**与**标识谱（外层电子向内层跃迁）**：标识谱波长只取决于靶材、不随管电压变。
- ⚠️ X 射线强度与管电压、管电流均有关（与靶面积一般无关）；调强度主要调灯丝电流（调管电压会改变整个波长范围）。
- ⚠️ 硬度（穿透本领）只取决于 X 射线波长/单光子能量，与光子数无关。
- ⚠️ 朗伯定律 $I = I_0 e^{-\mu x}$、质量衰减系数 $\mu_m = \frac{\mu}{\rho}$、半价层 $x_{1/2} = \frac{0.693}{\mu}$；强度变 $\frac{1}{8}$ 需经 3 个半价层。
- ⚠️ 布拉格方程 $2d\sin\theta = k\lambda$（已知 $\lambda$、$\theta$、级次 $k$ 求 $d$，注意 $k$ 代入级次）。
- ⚠️ 管电压 100 kV 时光子最大能量 $eU$（$1.6\times10^{-19}\times10^5$）与最短波长。
- ⚠️ X 射线波长比可见光短、与伽马射线比不一定（伽马穿透更强、波长更短）。
- ⚠️ 激光产生必要条件、受激辐射特性（同频同相同方向同偏振）、粒子数反转与谐振腔作用为简答常考点。

## 课堂讨论/互动
- Q（教师设问）：能用连续谱做 X 射线衍射吗？
  A：不能，就像白光不能做衍射；不同波长形成的条纹位置不同会叠成连续，需用特征谱（单一波长）。
- 教师设问：为什么原子序数越大 X 射线强度越大？
  A：不是因核尺寸（都约 $10^{-15}$），而是序数大→核带正电多→核电场强、作用半径大→韧致辐射转化效率高。
- 教师讨论：自动驾驶用激光雷达还是摄像头？——摄像头极限类似人眼，激光雷达可精确测距重构三维但点云数据量大、对算力要求高。
- 课间闲聊：6 点下课、"这个李子有点酸酸的还挺好喝"。
