## 教学要点

### 安培力与洛伦兹力的关系

- 电流元安培力 $dF = I\,dl\,B\sin\alpha$，矢量式 $d\vec{F} = I\,d\vec{l}\times\vec{B}$；整根导线受力 $\vec{F} = \displaystyle\int I\,d\vec{l}\times\vec{B}$。
- **本质联系**：$I = q/t$，$dl/t = v$，故 $I\,d\vec{l}\times\vec{B} = q\vec{v}\times\vec{B}$，即洛伦兹力。安培力起源于小区域内所有电荷所受的洛伦兹力之和。
- 高斯单位制与SI单位制之间相差 $4\pi$ 因子（比例系数 $k$ 用于统一单位）。

### 平行载流导线的相互作用

- 无限长导线 $I_1$ 与有限长导线 $I_2$（长 $l$、近端距 $a$、夹角 $\theta$）：$B_1 = \dfrac{\mu_0 I_1}{2\pi r}$ 为位置函数，建立 $dl = dr/\sin\theta$，从 $a$ 积分到 $a+l\sin\theta$。
- 特例：$\theta=90°$ 直接代入；$\theta=0$（平行）出现 $0/0$，用**洛必达**取极限。
- $a\ll l$ 时单位长度受力 $f = \dfrac{\mu_0 I_1 I_2}{2\pi a}$，**同向电流相吸、反向相斥**。
- 历史定义：2019年前，真空中两无限长平行导线相距1 m、每米受力 $2\times10^{-7}$ N 时，电流为1 A；现用 $I=q/t$ 定义（1 s 通过约 $6.24\times10^{18}$ 个电子为 1 A）。

### 载流线圈的力矩与磁偶极矩

- 矩形线圈在沿 $x$ 轴均匀磁场中：$F_1$、$F_2$ 大小相等方向相反 → **合力为零**；但二者不共线 → **合力矩不为零**，线圈转动。
- 定义**磁偶极矩** $\vec{p}_m = I\vec{S}$，方向沿线圈法线（右手法则：右手顺电流，拇指为法线）。
- **磁力矩** $\vec{M} = \vec{p}_m\times\vec{B}$，即 $M = p_m B\sin\theta$（$\theta$ 为法线与 $B$ 夹角），结论对任意形状线圈成立。
- $n\parallel B$（$\theta=0°$ 或 $180°$）→ $M=0$（稳定平衡）；$n\perp B$（$\theta=90°$）→ $M$ 最大。磁力矩总使线圈转向稳定平衡态（$p_m$ 平行于外磁场）。
- 关键结论：均匀磁场对线圈的作用只取决于 $p_m$（电流×面积），与形状无关。

### 霍尔效应

- 原理：导体通电流并加垂直磁场，载流子受洛伦兹力偏转，正负电荷在上下表面聚集形成横向电势差。
- 实验式 $U = R_H\dfrac{IB}{d}$，$d$ 为厚度，霍尔系数 $R_H = \dfrac{1}{ne}$（$n$ 为载流子数密度）。平衡时洛伦兹力与霍尔电场力相等：$qvB = qE_H$，$U = E_H d$。
- 应用：判断载流子浓度与正负（金属为电子负载流子；P型半导体为空穴正载流子）、测磁场（霍尔探头）、做传感器。霍尔效应发现于1879年，低场时 $U$ 与 $B$ 线性。
- **量子霍尔效应**（冯·克利青，1980）：高场（>5–10 T）、低温（1.5 K）时出现平台，$R_H = \dfrac{h}{ne^2}$（$n$ 为整数），1985年诺奖。
- **分数霍尔效应**（崔琦、斯托默，1982）：更低温（约85 mK）、更高场（28 T），出现 $4/3$、$3/5$ 等分数。
- 反常量子霍尔效应：磁性拓扑材料（内部绝缘、表面导体），电阻-磁场关系呈回线。

### 带电粒子的应用

- **质谱仪**：先用电场+磁场（速度选择器，$qE = qvB_1$）筛选速度，再进磁场区由 $qvB_2 = \dfrac{mv^2}{r}$ 得 $m = \dfrac{qB_1 B_2 r}{E}$；半径不同区分同位素。
- **回旋加速器**：周期恒定，仅在 D 形盒缝隙处加交变电场加速，半径 $r = \dfrac{mv}{qB}$ 随速度增大；最大速度受半径限制，接近光速时质量变化失效。例：劳伦斯首台可达1 MeV（质子约5%光速）。

### 电磁实际应用
- 电磁泵：磁场+电流推动导电液体流动，用于核电站冷却液循环（无旋转部件）。
- 磁流体发电：高温电离 + 磁场分离正负电荷，效率高（少能量转换环节），但需几千度高温，未大规模铺开。
- 电磁流量计：利用流体自由电子的霍尔效应测流速，无需与流体接触（如测血流）。

## 考试重点标注
- ⚠️ 安培力本质即洛伦兹力（$I\,d\vec{l}\times\vec{B} = q\vec{v}\times\vec{B}$）——老师明确为"理解电磁学中力的作用机制的关键"。
- ⚠️ 平行导线单位长度受力 $f = \dfrac{\mu_0 I_1 I_2}{2\pi a}$、同向相吸反向相斥，及 $\theta=0$ 用洛必达取极限。
- ⚠️ 磁偶极矩 $\vec{p}_m = I\vec{S}$ 与磁力矩 $\vec{M} = \vec{p}_m\times\vec{B}$——"PM 相同的线圈受到的磁场作用完全相同"。
- ⚠️ 霍尔电压 $U = R_H\dfrac{IB}{d}$，$R_H = \dfrac{1}{ne}$——含多薄片浓度/厚度比例计算题。
- ⚠️ 弯曲导线 $O$ 点磁感强度（半无限长 + 1/4 圆叠加）习题、正方形排列导线中心场抵消、挖孔圆柱体填补法。
- ⚠️ **均匀磁场中安培力与导线形状无关，只与端点位置有关** $\vec{F} = I\vec{l}\times\vec{B}$，闭合回路受力为 $0$——老师"这个结论挺重要"。
- ⚠️ 回旋加速器相关参数计算（由频率求 $B = \dfrac{2\pi mf}{q}$、最大速率与能量 $\dfrac{1}{2}mv^2$）。
- ⚠️ 本章重点小结：磁感应强度概念、毕奥-萨伐尔定律、磁场高斯定理与安培环路定理（尤其用于计算）、洛伦兹力与安培力受力、回旋加速器。

## 课堂讨论/互动
- Q: 一根直载流导线与一个环形电流之间有无相互作用？
  A: 没有，因为电流方向与磁场方向处处平行。
- Q: 半导体薄片中空穴（正电）与电子（负电）所受洛伦兹力方向是否相同？
  A: 相同。电流方向相同时两者运动方向相反，洛伦兹力方向一致（习题选向下，选A）。
- Q: 挖去小圆柱体的载流圆柱体，求空心中心 $O'$ 点磁感强度？
  A: 用填补法，将挖去部分视为反向负电流；该负电流对 $O'$（位于其轴心）无贡献，只算补全后大圆在 $O'$ 处的场。若求任意点则须叠加负电流贡献。
- 科学史讨论：欧洲LHC可达6.7 TeV，曾讨论建100公里半径对撞机；杨振宁撰文"party is over"反对建大型对撞机（经费高、削弱其他科研），亦有人支持（工程衍生成果，如万维网最早源于高能物理内网）。
