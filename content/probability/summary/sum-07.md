## 教学要点

### 一、正态分布的标准化与查表计算
- 若 $X \sim N(\mu, \sigma^2)$，则标准化变换 $Z = \frac{X-\mu}{\sigma} \sim N(0,1)$（标准正态分布）
- 任意正态分布的分布函数可通过标准正态分布函数 $\Phi(\cdot)$ 表示：$F_X(x) = \Phi\left(\frac{x-\mu}{\sigma}\right)$
- 标准正态分布对称性：$\Phi(-x) = 1 - \Phi(x)$
- ⚠️ 给定分布函数形式 $\Phi\left(\frac{x-a}{b}\right)$，可反推参数：$\mu = a$，$\sigma = |b|$（注意分母为标准差，非方差）

### 二、3σ原则（正态分布的集中性）
- $k=1$：$P\{|X-\mu| < \sigma\} = 2\Phi(1)-1 \approx 0.6826$（约68.26%概率落在1σ范围内）
- $k=2$：$P\{|X-\mu| < 2\sigma\} = 2\Phi(2)-1 \approx 0.9545$（约95.45%概率落在2σ范围内）
- $k=3$：$P\{|X-\mu| < 3\sigma\} = 2\Phi(3)-1 \approx 0.9974$（约99.74%概率落在3σ范围内）
- 3σ原则的实际意义：正态随机变量取值几乎全部集中在 $(\mu-3\sigma, \mu+3\sigma)$ 区间内，超出该范围的概率不到0.3%
- 质量控制应用：3σ以外区域视为异常区域
- σ越小，分布越集中，相同区间内概率越大

### 三、随机变量类型总结与混合型
- 离散型：分布函数为右连续阶梯函数，分布列刻画
- 连续型：分布函数连续，密度函数刻画，单点概率为0
- ⚠️ 注意：分布函数连续 $\neq$ 连续型随机变量（存在奇异型）
- **混合型随机变量**：分布函数可表示为离散型与连续型分布函数的凸组合：
  $$F(x) = pF_1(x) + (1-p)F_2(x), \quad 0 < p < 1$$
  其中 $F_1$ 对应离散型，$F_2$ 对应连续型
- 电压表例题：$X = \min(V, V_0)$，$V \sim \text{Exp}(\lambda)$，$X$ 在 $V_0$ 处有跳跃 $P\{X=V_0\} = e^{-\lambda V_0}$，为混合型
- 判断非连续型的方法：若存在某点概率不为0，则不是连续型随机变量

### 四、随机变量函数的分布（离散情形）
- 若 $X$ 为离散型，则 $Y = g(X)$ 必为离散型
- 求法：
  1. 确定 $Y$ 的所有可能取值
  2. 对一一对应情形：$P\{Y = g(x_i)\} = P\{X = x_i\}$
  3. 对多对一情形：合并同类项，将对应 $X$ 值的概率相加

### 五、随机变量函数的分布（连续情形）
#### 1. 分布函数法（万能方法）
- 步骤：
  1. 确定 $Y = g(X)$ 的取值范围
  2. 由定义求分布函数：$F_Y(y) = P\{Y \leq y\} = P\{g(X) \leq y\}$
  3. 将不等式转化为 $X$ 的显式范围
  4. 利用 $X$ 的分布计算概率
  5. 对 $y$ 求导得密度：$f_Y(y) = F_Y'(y)$
- 例：$Y = X^2$，$f_Y(y) = \frac{1}{2\sqrt{y}}[f_X(\sqrt{y}) + f_X(-\sqrt{y})], \quad y > 0$

#### 2. 公式法（严格单调函数）
- 定理条件：$g(x)$ 严格单调，反函数 $g^{-1}(y)$ 有连续导数
- 密度公式：
  $$f_Y(y) = f_X(g^{-1}(y)) \cdot \left|(g^{-1}(y))'\right|, \quad y \in R_Y$$
- 单调递增时反函数导数为正，单调递减时为负，取绝对值保证密度非负
- ⚠️ 非严格单调或分段单调情形，建议使用分布函数法，避免套公式出错

### 六、正态分布的线性变换不变性
- 若 $X \sim N(\mu, \sigma^2)$，$Y = aX + b$（$a \neq 0$），则：
  $$Y \sim N(a\mu + b, a^2\sigma^2)$$
- 标准化变换是线性变换的特例：$a = \frac{1}{\sigma}$，$b = -\frac{\mu}{\sigma}$，得到 $N(0,1)$
- ⚠️ 重要结论：正态分布经线性变换后仍为正态分布，类型不变

### 七、指数分布变换与概率积分变换
- 指数分布例题：$X \sim \text{Exp}(\lambda)$，$Y = e^{-X}$，则 $Y \sim U(0,1)$（(0,1)上均匀分布）
- **概率积分变换**：若连续型随机变量 $X$ 的分布函数 $F(x)$ 严格单调连续，则 $Y = F(X) \sim U(0,1)$
- 逆变换：若 $X \sim U(0,1)$，$F^{-1}$ 为某分布函数的反函数，则 $Y = F^{-1}(X)$ 服从该分布
- 应用：均匀分布是生成其他随机数的基础

| 正态标准化 | $Z = \frac{X-\mu}{\sigma} \sim N(0,1)$，$X \sim N(\mu,\sigma^2)$ |

| 标准正态对称性 | $\Phi(-x) = 1 - \Phi(x)$ |
| 3σ原则(k=1) | $P\{\|X-\mu\| < \sigma\} = 2\Phi(1)-1 \approx 0.6826$ |
| 3σ原则(k=2) | $P\{\|X-\mu\| < 2\sigma\} = 2\Phi(2)-1 \approx 0.9545$ |
| 3σ原则(k=3) | $P\{\|X-\mu\| < 3\sigma\} = 2\Phi(3)-1 \approx 0.9974$ |

## 考试重点标注
- ⚠️ 正态分布标准化计算与查表：必须熟练掌握 $F_X(x) = \Phi\left(\frac{x-\mu}{\sigma}\right)$ 的应用
- ⚠️ 3σ原则的三个数值：0.6826、0.9545、0.9974，记住σ越小分布越集中
- ⚠️ 连续型随机变量的判断：单点概率不为0则不是连续型
- ⚠️ 分布函数法是求随机变量函数分布的万能方法，必须掌握
- ⚠️ 公式法仅适用于严格单调函数，注意绝对值
- ⚠️ 正态线性变换不变性：线性变换后仍为正态，参数计算要正确（方差乘 $a^2$）
- ⚠️ $Y = X^2$ 的密度公式推导与应用

## 课堂讨论/互动
- 车门高度设计问题：按男子身高 $N(170, 6^2)$ 设计，碰头概率小于0.01，求临界高度 $h$，通过 $\Phi\left(\frac{h-170}{6}\right) \geq 0.99$ 查表得 $h \approx 184$ cm
- 两个正态比较题：$P\{|X-\mu_1|<1\} > P\{|Y-\mu_2|<1\}$，通过图像分析得 $\sigma_1 < \sigma_2$（越集中σ越小）
- 电压表读数例题：$X = \min(V, V_0)$，学生容易忽略混合型的判断，需通过分布函数间断点跃度求单点概率
- $Y = \max(X, 1/X)$ 例题：非单调函数，必须用分布函数法，先分析取值范围 $Y \geq 1$，再转化为 $1/y \leq X \leq y$ 计算
