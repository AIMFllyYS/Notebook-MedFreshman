## 教学要点

### 一、总体与样本
#### 1. 总体与个体
- **总体（Population）**：研究对象全体组成的集合，本质上对应一个随机变量 $X$。研究总体等价于研究随机变量 $X$。
- **个体**：组成总体的每个元素，个体与总体是元素与集合的属于关系。
- **总体与随机变量的对应**：研究某一数量指标（如身高、体重）时，该数量指标就是一个随机变量 $X$，随机抽取一个个体测得的数值是 $X$ 的一个观测值，带有随机性。

#### 2. 简单随机抽样
- **抽样的必要性**：总体庞大无法全部观测，且某些检验具有破坏性（如测试产品寿命、粉笔质量）。
- **简单随机抽样满足两个条件**：
  1. **随机性**：每次抽取每个个体机会均等，无偏见
  2. **独立性**：每次抽取互不影响，有放回抽样属于简单随机抽样，无放回抽样破坏独立性
- **注**：本书中"抽样"均指简单随机抽样。此外还有分层抽样、整群抽样、阶段抽样等其他方式。

#### 3. 简单随机样本
- **定义**：若 $X_1, X_2, \dots, X_n$ 相互独立且与总体 $X$ 同分布，则称其为来自总体 $X$ 的容量为 $n$ 的简单随机样本。
- **两个核心条件（看到"样本"就默认满足）**：
  1. **独立性**：$X_1, X_2, \dots, X_n$ 相互独立
  2. **同分布性**：每个 $X_i$ 与总体 $X$ 分布相同
- **样本容量 $n$**：独立随机变量的个数，可理解为信息量大小。
- **观测值**：一次抽样得到的具体数值 $x_1, x_2, \dots, x_n$，是常数（实验前是随机变量，实验后是具体数值）。

#### 4. 样本的联合分布
- **连续型总体**：若总体 $X$ 的密度函数为 $f(x)$，则样本的联合密度为边缘密度的乘积：
  $$f(x_1, x_2, \dots, x_n) = \prod_{i=1}^n f(x_i), \quad x_i \in \text{定义域}$$
  例如：指数总体 $f(x) = \lambda e^{-\lambda x} (x > 0)$，联合密度为 $\lambda^n e^{-\lambda \sum x_i}$，要求所有 $x_i > 0$。
- **离散型总体**：联合分布列为各点概率的乘积，对两点分布 $P(X=1)=p, P(X=0)=q$：
  $$P(X_1 = x_1, \dots, X_n = x_n) = \prod_{i=1}^n p^{x_i} q^{1-x_i}, \quad x_i \in \{0,1\}$$
- **意义**：第七章参数估计的基础，必须正确写出联合分布（注意定义域范围）。

### 二、经验分布函数
#### 1. 构造方法
- **步骤**：设 $x_1, x_2, \dots, x_n$ 是样本观测值，将其从小到大排序为 $x_{(1)} \leq x_{(2)} \leq \dots \leq x_{(n)}$（顺序统计量观测值），定义经验分布函数：
  $$F_n(x) = \begin{cases}
  0, & x < x_{(1)} \\
  k/n, & x_{(k)} \leq x < x_{(k+1)}, \quad k = 1,2,\dots,n-1 \\
  1, & x \geq x_{(n)}
  \end{cases}$$

#### 2. 性质
- $F_n(x)$ 是**单调非降、右连续**的函数，函数值介于 $0$ 和 $1$ 之间，且 $F_n(-\infty)=0, F_n(+\infty)=1$
- 因此 $F_n(x)$ **本身就是一个分布函数**，对应一个离散型均匀分布
- 对应的分布列：在每个排序后的观测值 $x_{(k)}$ 处概率为 $1/n$（重复数据合并同类项，概率为 $m/n$，$m$ 是重复次数）

#### 3. 直观理解
- **频率估计概率**：$F_n(x) = \frac{k}{n}$ 表示 $n$ 个观测值中不超过 $x$ 的频率，用频率估计概率 $P(X \leq x) = F(x)$，理论依据是伯努利大数定律
- **随机函数**：$F_n(x)$ 依赖于样本观测值，是一个随机函数
- **例子**：样本观测值为 0.3, 0.4, 0.4, 0.5, 0.6（$n=5$），则 $F_5(x)$ 在 $[0.3,0.4)$ 为 $1/5$，在 $[0.4,0.5)$ 为 $3/5$，对应分布列为 $P(0.3)=1/5, P(0.4)=2/5, P(0.5)=1/5, P(0.6)=1/5$。

#### 4. 格里文科定理
- 当 $n \to \infty$ 时，$F_n(x)$ 以概率1关于 $x$ 一致收敛于总体分布函数 $F(x)$，即经验分布函数一致逼近理论分布函数
- 直观上：随着样本量增大，阶梯形的经验分布函数"缠绕"在连续的理论分布函数曲线上

### 三、统计量
#### 1. 定义
- 设 $X_1, X_2, \dots, X_n$ 是来自总体的样本，若 $T = T(X_1, X_2, \dots, X_n)$ 是样本的**连续函数**（不要求严格记忆）且**不含任何未知参数**，则称 $T$ 为统计量。
- **本质**：统计量是随机变量（样本的函数），将观测值代入得到的具体数值 $t = T(x_1, \dots, x_n)$ 称为统计量的观测值。
- **核心要求**：函数中**不能含有未知参数**——代入观测值必须能算出具体数值，不能有未知量。

#### 2. 判断示例
- $X_1 + \mu$（$\mu$ 已知）：是统计量（无未知参数）
- $\frac{X_1 - \mu}{\sigma}$（$\sigma$ 未知）：不是统计量（含未知参数 $\sigma$）
- $\min\{X_1, \dots, X_n\}$：是统计量（仅是样本函数）
- 注：不要求用到所有样本，只要是样本的函数且无未知参数即可。

#### 3. 统计量的意义
- 统计量是对样本信息的**加工、提炼、集中**过程，从零散的数据中提取有用信息。

### 四、常用统计量
#### 1. 样本均值
$$\bar{X} = \frac{1}{n}\sum_{i=1}^n X_i$$
- 与总体期望 $E[X] = \mu$ 对比：$\bar{X}$ 是**随机变量**，$\mu$ 是**常数**；两者都表示"平均"含义
- 期望和方差：$E[\bar{X}] = \mu$，$D(\bar{X}) = \frac{\sigma^2}{n}$

#### 2. 样本方差与样本标准差
- **样本方差**：
  $$S^2 = \frac{1}{n-1}\sum_{i=1}^n (X_i - \bar{X})^2$$
  - 注意：除以 $n-1$ 而非 $n$（后续章节会说明原因，$S^2$ 具有无偏性等良好性质）
- **样本二阶中心矩**（记为 $S_1^2$ 或 $B_2$）：
  $$B_2 = S_1^2 = \frac{1}{n}\sum_{i=1}^n (X_i - \bar{X})^2$$
  - 关系：$S^2 = \frac{n}{n-1} S_1^2$
- **样本标准差**：$S = \sqrt{S^2}$
- 与总体方差 $D(X) = \sigma^2$ 对比：$S^2$ 刻画样本的分散程度，$\sigma^2$ 刻画总体的分散程度
- **平方和分解公式**（计算常用）：
  $$\sum_{i=1}^n (X_i - \bar{X})^2 = \sum_{i=1}^n X_i^2 - n\bar{X}^2$$
  推导：展开平方和后利用 $\sum X_i = n\bar{X}$ 化简。

#### 3. 样本矩
- **样本 $k$ 阶原点矩**：$A_k = \frac{1}{n}\sum_{i=1}^n X_i^k$，对应总体 $k$ 阶原点矩 $\mu_k = E[X^k]$
- **样本 $k$ 阶中心矩**：$B_k = \frac{1}{n}\sum_{i=1}^n (X_i - \bar{X})^k$，对应总体 $k$ 阶中心矩
- 特例：$A_1 = \bar{X}$（一阶原点矩即样本均值），$B_2$ 是样本二阶中心矩（不是 $S^2$）

### 五、顺序统计量
#### 1. 定义
将样本 $X_1, \dots, X_n$ 按观测值从小到大排序为 $X_{(1)} \leq X_{(2)} \leq \dots \leq X_{(n)}$，称 $X_{(k)}$ 为第 $k$ 个顺序统计量。
- $X_{(1)} = \min\{X_1, \dots, X_n\}$：最小顺序统计量
- $X_{(n)} = \max\{X_1, \dots, X_n\}$：最大顺序统计量
- **重要注意**：$X_{(1)}$ 和 $X_{(n)}$ **不是**原来某个固定的 $X_i$，而是新的随机变量（参数可能改变）
  - 例：$X_1, X_2 \sim B(1,p)$ 独立，则 $X_{(1)} \sim B(1,p^2)$，$X_{(2)} \sim B(1,1-q^2)$，参数不是 $p$。

#### 2. 常用顺序统计量
- **样本中位数 $M_e$**：
  $$M_e = \begin{cases}
  X_{(\frac{n+1}{2})}, & n \text{ 为奇数} \\
  \frac{1}{2}\left(X_{(\frac{n}{2})} + X_{(\frac{n}{2}+1)}\right), & n \text{ 为偶数}
  \end{cases}$$
  位于数据最中间位置，当分布偏斜时比期望更有代表性。

- **样本极差 $R$**：$R = X_{(n)} - X_{(1)}$（最大值减最小值），反映样本分散程度，但极易受极端值影响（如考试成绩极差100意味着有人满分有人0分）。

### 六、三大抽样分布引入（预备）
- 第六章将介绍三大抽样分布：$\chi^2$ 分布（卡方分布）、$t$ 分布（学生氏分布）、$F$ 分布
- 每种分布需掌握三点：**定义（构造方式）**、**性质**、**分位点**
- 这三大分布是统计推断的基础，均通过构造方式定义（不需记密度函数）。

## 核心公式汇总
| 名称 | 公式 |
|------|------|
| 样本均值 | $\displaystyle \bar{X} = \frac{1}{n}\sum_{i=1}^n X_i$ |
| 样本方差（除 $n-1$） | $\displaystyle S^2 = \frac{1}{n-1}\sum_{i=1}^n (X_i - \bar{X})^2$ |
| 样本二阶中心矩（除 $n$） | $\displaystyle B_2 = S_1^2 = \frac{1}{n}\sum_{i=1}^n (X_i - \bar{X})^2$ |
| 平方和分解公式 | $\displaystyle \sum_{i=1}^n (X_i - \bar{X})^2 = \sum_{i=1}^n X_i^2 - n\bar{X}^2$ |
| 样本 $k$ 阶原点矩 | $\displaystyle A_k = \frac{1}{n}\sum_{i=1}^n X_i^k$ |
| 样本 $k$ 阶中心矩 | $\displaystyle B_k = \frac{1}{n}\sum_{i=1}^n (X_i - \bar{X})^k$ |
| 样本极差 | $\displaystyle R = X_{(n)} - X_{(1)}$ |
| 连续型样本联合密度 | $\displaystyle f(x_1,\dots,x_n) = \prod_{i=1}^n f(x_i)$ |

## 考试重点标注
- ⚠️ **简单随机样本的两个核心条件**：独立、同分布，这是后续所有推导的基础
- ⚠️ **统计量的判断**：必须是样本函数且**不含未知参数**，这是常考概念题
- ⚠️ **样本方差除以 $n-1$**：不要与二阶中心矩（除以 $n$）混淆，注意 $S^2$ 与 $B_2$ 的区别和联系
- ⚠️ **样本联合分布的书写**：连续型总体要写对密度乘积和定义域范围，这是第七章最大似然估计的基本功
- ⚠️ **顺序统计量不是原样本中的固定一个**：最小/最大值是新的随机变量，参数可能改变

## 课堂讨论/互动
- **学生提问**：为什么要抽样？
  **教师回答**：总体太庞大无法观察全部，且有些检验具有破坏性（如测试粉笔质量）。
- **课堂练习**：给定0-1分布总体容量为2的样本，求 $X_{(1)}$ 和 $X_{(2)}$ 的分布。
  - 方法提示：最小值取1当且仅当两个都取1，故 $P(X_{(1)}=1) = p^2$，即 $X_{(1)} \sim B(1,p^2)$；最大值取0当且仅当两个都取0，故 $P(X_{(2)}=0) = q^2$。
