## 教学要点

### 一、二维连续型的边缘密度
- 若 $(X,Y)$ 为二维连续型，联合密度为 $f(x,y)$，则 $X, Y$ 均为一维连续型随机变量
- **边缘密度公式**：
  $$f_X(x) = \int_{-\infty}^{+\infty} f(x,y) \, dy$$
  $$f_Y(y) = \int_{-\infty}^{+\infty} f(x,y) \, dx$$
- 积分要点：求 $f_X(x)$ 时，$x$ 视为常数，对 $y$ 积分（消去 $y$）；求 $f_Y(y)$ 时，$y$ 视为常数，对 $x$ 积分（消去 $x$）
- 直观理解（球的累加）：固定 $x$，将该竖直线上所有点的"概率质量"累加投影到x轴上，得到边缘密度
- ⚠️ 难点：被积函数常分区域定义，需正确确定积分上下限

### 二、二维正态分布的边缘分布
- **重要性质**：若 $(X,Y) \sim N(\mu_1, \mu_2, \sigma_1^2, \sigma_2^2, \rho)$，则
  $$X \sim N(\mu_1, \sigma_1^2), \quad Y \sim N(\mu_2, \sigma_2^2)$$
- 二维正态的两个边缘分布都是一维正态分布，且参数仅由前4个参数决定，与 $\rho$ 无关
- **计算技巧：配密度法**
  1. 做标准化变换 $u = \frac{x-\mu_1}{\sigma_1}, v = \frac{y-\mu_2}{\sigma_2}$
  2. 对被积函数中的指数部分关于 $v$ 配方
  3. 识别出正态密度形式，利用规范性 $\int_{-\infty}^{+\infty} \frac{1}{\sqrt{2\pi}} e^{-t^2/2} dt = 1$ 直接得到结果，无需显式积分
- 配密度法核心：将被积函数凑成某个已知分布的密度，利用规范性=1快速计算积分
- ⚠️ 记住结论：二维正态的边缘必为正态

### 三、二维均匀分布的边缘分布
- **重要反例**：二维均匀分布的边缘分布**不一定**是均匀分布
- 三角形区域反例：区域 $G: x\geq0, y\geq0, 2x+y\leq2$（面积S=1），联合密度 $f(x,y)=1$ 在G上
  - $f_X(x) = \begin{cases} 2(1-x), & 0\leq x\leq1 \\ 0, & \text{其他} \end{cases}$（线性递减，非均匀）
  - $f_Y(y) = \begin{cases} 1-\frac{y}{2}, & 0\leq y\leq2 \\ 0, & \text{其他} \end{cases}$（线性递减，非均匀）
- 直观解释：三角形区域中，靠近原点处竖直线更长，累加球更多，密度更大；远离原点处竖直线短，累加球少，密度小
- **特例**：矩形区域上的二维均匀分布，其边缘分布是均匀分布
- ⚠️ 注意：非矩形区域的均匀分布，边缘通常不是均匀的

### 四、边缘正态与联合正态的关系
- 结论1：二维正态 $\Rightarrow$ 边缘正态（已证）
- 结论2：边缘正态 $\nRightarrow$ 联合正态（**反之不成立**）
- 反例1：构造联合密度 $f(x,y) = \frac{1}{2\pi}e^{-(x^2+y^2)/2}(1+\sin x\sin y)$
  - 计算边缘密度：$f_X(x) = \frac{1}{\sqrt{2\pi}}e^{-x^2/2}$（奇函数积分=0），同理 $f_Y(y)$ 也是标准正态
  - 但联合密度不是二维正态形式
- 反例2：$X \sim N(0,1)$，$Y = 2X+1$（Y也是正态），但 $(X,Y)$ 以概率1落在直线 $y=2x+1$ 上，线上概率≠0，故不是二维连续型，更不是二维正态
- ⚠️ 考试易错点：两个边缘正态拼起来不一定是二维正态

### 五、二维离散型的条件分布
- 实际背景：给定身高范围（如170~180cm），体重的分布与无条件分布不同
- 定义：已知联合分布列 $p_{ij} = P\{X=x_i, Y=y_j\}$，在 $Y=y_j$ 条件下 $X$ 的**条件分布列**：
  $$P\{X = x_i | Y = y_j\} = \frac{p_{ij}}{p_{\cdot j}}, \quad i = 1,2,\dots$$
  在 $X=x_i$ 条件下 $Y$ 的条件分布列：
  $$P\{Y = y_j | X = x_i\} = \frac{p_{ij}}{p_{i\cdot}}, \quad j = 1,2,\dots$$
- 条件：$p_{\cdot j} > 0$（或 $p_{i\cdot} > 0$），分母不能为0
- 规范性验证：$\sum_{i} P\{X=x_i|Y=y_j\} = \frac{1}{p_{\cdot j}}\sum_i p_{ij} = \frac{p_{\cdot j}}{p_{\cdot j}} = 1$，确实是分布列

### 六、经典例题：射击问题
- 问题：射击进行到击中目标两次为止，$X$ 为首次击中的射击次数，$Y$ 为总射击次数，每次击中概率为 $p$
- 联合分布列：$P\{X=m, Y=n\} = p^2(1-p)^{n-2}, \quad n=2,3,\dots, \, m=1,2,\dots,n-1$
  - 解释：前n-1次恰好击中1次（首次在第m次），第n次击中
- $X$ 的边缘分布：$P\{X=m\} = p(1-p)^{m-1}$（几何分布，首次成功的试验次数）
- $Y$ 的边缘分布：$P\{Y=n\} = C_{n-1}^1 p^2(1-p)^{n-2}$（负二项分布/帕斯卡分布，成功2次所需试验次数）
- 条件分布：
  - $P\{X=m | Y=n\} = \frac{1}{n-1}, \, m=1,\dots,n-1$（均匀分布：已知第n次第二次击中，首次击中在前面n-1个位置等可能）
  - $P\{Y=n | X=m\} = p(1-p)^{n-m-1}, \, n=m+1,m+2,\dots$（几何分布：无记忆性，从第m次后重新开始等待下一次击中）

### 七、经典例题：婴儿出生问题
- 模型：$X$ 为总婴儿数，$Y$ 为男婴数，$P\{X=n,Y=m\} = \frac{e^{-14}(6.86)^m(7.14)^{n-m}}{m!(n-m)!}$
- 边缘分布：
  - $X \sim P(14)$（泊松分布，参数14，总婴儿数）
  - $Y \sim P(7.14)$（泊松分布，参数7.14，男婴数）
  - 计算技巧：配二项式定理 $\sum_{m=0}^n C_n^m a^m b^{n-m} = (a+b)^n$，配泰勒展开 $e^x = \sum_{k=0}^\infty \frac{x^k}{k!}$
- 条件分布：$Y|X=n \sim B(n, 0.51)$（二项分布：已知总出生n个，男婴数服从n重伯努利，每次生男概率0.51）

### 八、二维连续型的条件密度
- 问题：直接用条件概率 $P\{X \leq x | Y=y\}$ 会出现分母为0（连续型 $P\{Y=y\}=0$），需用密度处理
- 定义：设 $f_Y(y) > 0$，在 $Y=y$ 条件下 $X$ 的**条件概率密度**：
  $$f_{X|Y}(x|y) = \frac{f(x,y)}{f_Y(y)}$$
  同理，在 $X=x$ 条件下 $Y$ 的条件密度：
  $$f_{Y|X}(y|x) = \frac{f(x,y)}{f_X(x)}$$
- 条件概率计算：$P\{a < X \leq b | Y=y\} = \int_a^b f_{X|Y}(x|y) \, dx$
- 几何理解：固定 $X=x$ 切一刀，截口曲线为 $f(x,y)$（关于y的一元函数），该曲线下的面积为 $f_X(x)$，将截口曲线除以 $f_X(x)$ 归一化（使面积=1），即得到条件密度 $f_{Y|X}(y|x)$
- 乘法公式（由条件密度反推联合密度）：
  $$f(x,y) = f_X(x) f_{Y|X}(y|x) = f_Y(y) f_{X|Y}(x|y)$$

### 九、二维正态的条件分布
- 优良性质：若 $(X,Y) \sim N(\mu_1, \mu_2, \sigma_1^2, \sigma_2^2, \rho)$，则条件分布仍为正态：
  $$X|Y=y \sim N\left( \mu_1 + \rho\frac{\sigma_1}{\sigma_2}(y-\mu_2), \, \sigma_1^2(1-\rho^2) \right)$$
  $$Y|X=x \sim N\left( \mu_2 + \rho\frac{\sigma_2}{\sigma_1}(x-\mu_1), \, \sigma_2^2(1-\rho^2) \right)$$
- 条件均值（回归均值）：是给定值的线性函数，$\rho$ 控制线性关系的斜率
- 条件方差：$\sigma^2(1-\rho^2)$ 与给定值无关，且小于无条件方差 $\sigma^2$（已知Y的信息减少了X的不确定性）
- 三维几何解释：用平面 $Y=y$（或 $X=x$）去截二维正态的钟形曲面，截口曲线形状与正态密度成比例，归一化后即为正态条件密度

### 十、条件密度的应用：由条件生成联合
- 例题：$X \sim U(0,1)$，给定 $X=x$（$0<x<1$）时，$Y|X=x \sim U(0,x)$，求联合密度
- 解法：用乘法公式 $f(x,y) = f_X(x) f_{Y|X}(y|x)$
  - $f_X(x) = 1$（0&lt;x&lt;1）
  - $f_{Y|X}(y|x) = 1/x$（0&lt;y&lt;x）
  - $f(x,y) = 1/x$（0&lt;y&lt;x&lt;1）
- 注意：这种由边缘+条件生成联合的模式是构建概率模型的常用方法

## 核心公式汇总

| 名称 | 公式 |
|------|------|
| 边缘密度（X） | $f_X(x) = \int_{-\infty}^{+\infty} f(x,y) \, dy$ |
| 边缘密度（Y） | $f_Y(y) = \int_{-\infty}^{+\infty} f(x,y) \, dx$ |
| 二维正态边缘 | $X \sim N(\mu_1,\sigma_1^2), Y \sim N(\mu_2,\sigma_2^2)$ |
| 离散条件分布 | $P\{X=x_i\|Y=y_j\} = \frac{p_{ij}}{p_{\cdot j}}$ |
| 连续条件密度 | $f_{X\|Y}(x\|y) = \frac{f(x,y)}{f_Y(y)}$ |
| 乘法公式 | $f(x,y) = f_X(x)f_{Y\|X}(y\|x)$ |
| 二维正态条件均值 | $E(X\|Y=y) = \mu_1 + \rho\frac{\sigma_1}{\sigma_2}(y-\mu_2)$ |
| 二维正态条件方差 | $\text{Var}(X\|Y=y) = \sigma_1^2(1-\rho^2)$ |
| 条件概率计算 | $P\{a<X\leq b\|Y=y\} = \int_a^b f_{X\|Y}(x\|y)dx$ |

## 考试重点标注
- ⚠️ 边缘密度积分公式：求谁的边缘就保留谁做变量，对另一个变量积分，特别注意分区域积分的上下限确定
- ⚠️ 二维正态边缘必为正态，但边缘正态不一定推出联合正态（反例要知道）
- ⚠️ 二维均匀分布（非矩形区域）的边缘通常不是均匀分布，三角形区域边缘密度是线性函数
- ⚠️ 配密度法：利用正态密度规范性=1，避免复杂积分计算
- ⚠️ 条件分布公式：分子是联合，分母是对应边缘（给定谁就除以谁的边缘）
- ⚠️ 射击问题：联合分布→边缘（几何/负二项）→条件（均匀/几何），推导过程要掌握
- ⚠️ 条件密度的几何理解：截口曲线归一化，面积=1
- ⚠️ 二维正态条件分布仍为正态，条件均值是线性的，条件方差小于无条件方差

## 课堂讨论/互动
- 二维正态边缘分布推导：用配密度法将被积函数凑成标准正态密度，利用规范性=1直接得到结果，避免显式积分
- 三角形区域均匀分布：学生先通过"球累加"的直观图像猜测边缘密度是递减函数，再通过积分验证
- 射击问题：学生需要识别X为几何分布、Y为负二项分布，并正确写出联合分布的范围（m &lt; n）
- 婴儿出生问题：需要配二项式定理和e^x泰勒展开来计算边缘分布，技巧性较强
- 条件密度的几何解释：用"切蛋糕"的比喻，每一刀切下去的截口曲线不是密度（面积≠1），除以边缘密度后归一化才是条件密度
