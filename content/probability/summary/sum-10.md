## 教学要点

### 一、随机变量独立性的概念
- 直观含义：一个随机变量的取值不影响另一个随机变量的分布，即条件分布等于无条件分布
- 分布函数定义：对任意实数 $x,y$，有
  $$F(x,y) = F_X(x) F_Y(y)$$
  即联合分布函数等于边缘分布函数的乘积
- 事件理解：对任意 $x,y$，事件 $\{X \leq x\}$ 与 $\{Y \leq y\}$ 相互独立
- 二维离散型独立条件：对所有 $i,j$，有
  $$p_{ij} = p_{i\cdot} p_{\cdot j}$$
  即联合分布列等于边缘分布列的乘积
- 二维连续型独立条件：对几乎所有 $x,y$，有
  $$f(x,y) = f_X(x) f_Y(y)$$
  即联合密度等于边缘密度的乘积
- 条件分布角度：独立等价于 $f_{X|Y}(x|y) = f_X(x)$ 或 $P\{X=x_i|Y=y_j\} = P\{X=x_i\}$

### 二、二维正态的独立性
- **核心定理**：若 $(X,Y) \sim N(\mu_1, \mu_2, \sigma_1^2, \sigma_2^2, \rho)$，则
  $$X \text{ 与 } Y \text{ 相互独立} \iff \rho = 0$$
- 充分性（$\rho=0 \Rightarrow$ 独立）：当 $\rho=0$ 时，二维正态密度可拆分为两个一维正态密度的乘积：
  $$f(x,y) = \frac{1}{\sqrt{2\pi}\sigma_1}e^{-\frac{(x-\mu_1)^2}{2\sigma_1^2}} \cdot \frac{1}{\sqrt{2\pi}\sigma_2}e^{-\frac{(y-\mu_2)^2}{2\sigma_2^2}} = f_X(x)f_Y(y)$$
- 必要性（独立 $\Rightarrow \rho=0$）：特取 $x=\mu_1, y=\mu_2$，代入 $f(x,y)=f_X(x)f_Y(y)$，对比常数项直接得 $\rho=0$
- ⚠️ $\rho$ 是刻画 $X$ 与 $Y$ 线性相关程度的参数，$\rho=0$ 表示不线性相关；对二维正态而言，不相关等价于独立

### 三、三角形区域均匀分布的独立性判断
- 例题：区域 $G: x\geq0, y\geq0, x+y\leq1$（三角形，面积S=1/2），联合密度 $f(x,y)=2$ 在G上
- 求得边缘密度：
  - $f_X(x) = 2(1-x), \, 0\leq x\leq1$
  - $f_Y(y) = 2(1-y), \, 0\leq y\leq1$
- 判断：$f(x,y) = 2 \neq f_X(x)f_Y(y) = 4(1-x)(1-y)$，故 $X$ 与 $Y$ 不独立
- 直观理解：三角形区域中，$X$ 的取值范围受 $Y$ 约束（$X \leq 1-Y$），两者不独立
- ⚠️ 判断独立的方法：验证联合密度是否等于边缘密度乘积

### 四、n维随机变量的独立性
- 定义：若对任意实数 $x_1,\dots,x_n$，有
  $$F(x_1,\dots,x_n) = F_{X_1}(x_1) \cdots F_{X_n}(x_n)$$
  则称 $X_1,\dots,X_n$ 相互独立
- n维离散独立：$P\{X_1=x_1,\dots,X_n=x_n\} = \prod_{i=1}^n P\{X_i=x_i\}$
- n维连续独立：$f(x_1,\dots,x_n) = \prod_{i=1}^n f_{X_i}(x_i)$
- 重要性质：整体独立 $\Rightarrow$ 任意k个分量独立（$2\leq k\leq n$）
- 验证技巧：只需验证n维联合等于n个边缘的乘积，则任意低维的联合自动等于对应边缘的乘积（通过将其余变量取 $+\infty$ 推导）
- 独立的好处：已知各边缘分布，独立条件下可直接相乘得到联合分布（统计中抽样的基础）

### 五、独立随机变量函数的独立性
- 定理：若 $X_1,\dots,X_n$ 相互独立，$g_1,\dots,g_n$ 是一元连续函数，则
  $$g_1(X_1), g_2(X_2), \dots, g_n(X_n)$$
  也相互独立
- 注意：必须是**一元函数**，每个函数仅作用于一个随机变量；多元函数不保证
- 应用：独立正态变量的线性组合仍为正态（见后文）

### 六、二维离散型随机变量函数的分布
- 若 $(X,Y)$ 为二维离散型，则 $Z = g(X,Y)$ 必为一维离散型
- 求法（分布函数法思想）：
  1. 确定 $Z$ 的所有可能取值
  2. 对每个取值 $z_k$，找出所有满足 $g(x_i,y_j) = z_k$ 的 $(x_i,y_j)$ 组合
  3. 将对应概率相加（合并同类项）：$P\{Z=z_k\} = \sum_{g(x_i,y_j)=z_k} p_{ij}$
- 独立离散情形：$p_{ij} = p_i p_j$

### 七、泊松分布的可加性
- 定理：若 $X \sim P(\lambda_1)$，$Y \sim P(\lambda_2)$，且 $X,Y$ 独立，则
  $$X + Y \sim P(\lambda_1 + \lambda_2)$$
- 证明（离散卷积）：
  $$P\{X+Y=k\} = \sum_{i=0}^k P\{X=i\}P\{Y=k-i\} = \sum_{i=0}^k \frac{e^{-\lambda_1}\lambda_1^i}{i!} \cdot \frac{e^{-\lambda_2}\lambda_2^{k-i}}{(k-i)!}$$
  提取公因子后配二项式：$= \frac{e^{-(\lambda_1+\lambda_2)}}{k!} \sum_{i=0}^k C_k^i \lambda_1^i \lambda_2^{k-i} = \frac{e^{-(\lambda_1+\lambda_2)}(\lambda_1+\lambda_2)^k}{k!}$
- 推广：$n$ 个独立泊松变量之和仍为泊松，参数为各参数之和：
  $$X_1+\dots+X_n \sim P(\lambda_1+\dots+\lambda_n), \quad X_i \sim P(\lambda_i) \text{ 独立}$$
- 应用：不同时间段独立泊松流的合并仍为泊松流（如每月销量之和）

### 八、二维连续型随机变量和的分布（卷积公式）
- 问题：已知 $(X,Y)$ 的联合密度 $f(x,y)$，求 $Z = X+Y$ 的密度
- 分布函数法：$F_Z(z) = P\{X+Y \leq z\} = \iint_{x+y\leq z} f(x,y) \, dx dy$
- **卷积公式**：
  $$f_{X+Y}(z) = \int_{-\infty}^{+\infty} f(x, z-x) \, dx = \int_{-\infty}^{+\infty} f(z-y, y) \, dy$$
- 独立情形：$f(x,y) = f_X(x)f_Y(y)$，则
  $$f_{X+Y}(z) = \int_{-\infty}^{+\infty} f_X(x) f_Y(z-x) \, dx = \int_{-\infty}^{+\infty} f_X(z-y) f_Y(y) \, dy$$
- 记忆方法：被积函数为联合密度，两个变量相加等于 $z$；对哪个变量积分，另一个变量用 $z$ 减它
- ⚠️ 使用关键：正确确定被积函数非零的积分区间（需分z讨论）

### 九、正态分布的可加性
- 特例：$X \sim N(0,1)$，$Y \sim N(0,1)$ 独立，则 $X+Y \sim N(0, 2)$
  - 推导：用卷积公式，配密度法（凑正态密度规范性=1）
- **一般定理**：若 $X_1,\dots,X_n$ 相互独立，$X_i \sim N(\mu_i, \sigma_i^2)$，则对任意常数 $a_1,\dots,a_n$（不全为0），线性组合
  $$\sum_{i=1}^n a_i X_i \sim N\left( \sum_{i=1}^n a_i \mu_i, \, \sum_{i=1}^n a_i^2 \sigma_i^2 \right)$$
- 重要结论：独立正态变量的任意线性组合仍为正态分布（比泊松可加性更强，不限于和）
- 应用：多个独立正态误差叠加后仍为正态

### 十、两个均匀分布和：三角分布
- 例题：$X \sim U[0,1]$，$Y \sim U[0,1]$ 独立，求 $Z=X+Y$ 的密度
- 分析：被积函数非零条件 $0\leq x\leq1$ 且 $0\leq z-x\leq1$，即 $\max(0,z-1) \leq x \leq \min(1,z)$
- 分段讨论得三角分布：
  $$f_{X+Y}(z) = \begin{cases} z, & 0\leq z < 1 \\ 2-z, & 1\leq z < 2 \\ 0, & \text{其他} \end{cases}$$
- 取值范围：$Z \in [0,2]$，密度图像为三角形，在 $z=1$ 处取最大值1

### 十一、商的分布与柯西分布
- 商的密度公式：$Z = X/Y$，则
  $$f_{X/Y}(z) = \int_{-\infty}^{+\infty} |y| f(yz, y) \, dy$$
- 记忆：一变量为 $yz$，二变量为 $y$，乘 $|y|$（处理正负区间），对二变量积分
- **柯西分布**：若 $X,Y \sim N(0,\sigma^2)$ 独立，则 $Z = X/Y$ 的密度为
  $$f_Z(z) = \frac{1}{\pi(1+z^2)}, \quad z \in \mathbb{R}$$
- 特点：柯西分布密度函数尾部很重，数学期望不存在（积分不绝对收敛）
- 物理别名：洛伦兹分布，在光谱学、物理学中出现

### 十二、极大值与极小值分布
- 背景：$n$ 个独立随机变量 $X_1,\dots,X_n$，各自分布函数为 $F_i(x)$
- 极大值 $M = \max\{X_1,\dots,X_n\}$ 的分布函数：
  $$F_M(x) = P\{M \leq x\} = P\{X_1\leq x, \dots, X_n\leq x\} = \prod_{i=1}^n F_i(x)$$
  （最大值≤x 等价于每个都≤x，独立则概率乘积）
- 极小值 $m = \min\{X_1,\dots,X_n\}$ 的分布函数：
  $$F_m(x) = 1 - P\{m > x\} = 1 - P\{X_1>x, \dots, X_n>x\} = 1 - \prod_{i=1}^n (1-F_i(x))$$
  （对立事件法：最小值&gt;x 等价于每个都&gt;x）
- 独立同分布（i.i.d.）情形：$F_i(x) = F(x)$，则
  - $F_M(x) = [F(x)]^n$
  - $F_m(x) = 1 - [1-F(x)]^n$
- 若为连续型，对分布函数求导即得密度
- ⚠️ 理解推导过程，不要死记公式

### 十三、分段函数与全概率公式应用
- 例题：$Z = \begin{cases} X, & X\leq 1/2 \\ Y, & X>1/2 \end{cases}$，$X\sim U[0,1]$，$Y\sim\text{Exp}(\lambda)$ 独立
- 解法：用全概率公式，按 $\{X\leq1/2\}$ 和 $\{X>1/2\}$ 划分样本空间
  $$F_Z(z) = P\{X\leq1/2\}P\{X\leq z|X\leq1/2\} + P\{X>1/2\}P\{Y\leq z\}$$
- 分段计算得到分段的分布函数

## 核心公式汇总

| 名称 | 公式 |
|------|------|
| 独立（分布函数） | $F(x,y) = F_X(x)F_Y(y)$ |
| 独立（密度） | $f(x,y) = f_X(x)f_Y(y)$ |
| 二维正态独立条件 | $X,Y$独立 $\iff \rho=0$ |
| 泊松可加性 | $X\sim P(\lambda_1), Y\sim P(\lambda_2)$独立 $\Rightarrow X+Y\sim P(\lambda_1+\lambda_2)$ |
| 和的卷积公式 | $f_{X+Y}(z) = \int_{-\infty}^{+\infty}f(x,z-x)dx$ |
| 独立卷积公式 | $f_{X+Y}(z) = \int_{-\infty}^{+\infty}f_X(x)f_Y(z-x)dx$ |
| 正态线性组合 | $\sum a_iX_i \sim N(\sum a_i\mu_i, \sum a_i^2\sigma_i^2)$（独立正态） |
| 商的密度 | $f_{X/Y}(z) = \int_{-\infty}^{+\infty}\|y\|f(yz,y)dy$ |
| 柯西分布 | $f(z) = \frac{1}{\pi(1+z^2)}$ |
| 极大值分布 | $F_{\max}(x) = \prod_{i=1}^n F_i(x)$ |
| 极小值分布 | $F_{\min}(x) = 1-\prod_{i=1}^n(1-F_i(x))$ |
| 三角分布（两个U[0,1]和） | $f(z) = z$（$0\leq z<1$），$2-z$（$1\leq z<2$） |

## 考试重点标注
- ⚠️ 二维正态独立的充要条件是 $\rho=0$：必须记住，这是考试高频考点
- ⚠️ 判断独立性：验证联合是否等于边缘乘积，三角形区域均匀分布不独立
- ⚠️ 卷积公式的使用：关键是正确确定积分上下限，需对z分段讨论
- ⚠️ 泊松可加性和正态可加性：独立泊松之和为泊松（参数相加），独立正态线性组合仍为正态（均值线性、方差加权平方和）
- ⚠️ 极大值、极小值分布的推导方法：最大值≤x等价于所有都≤x；最小值用对立事件
- ⚠️ 两个独立U[0,1]之和为三角分布，密度形式要记住
- ⚠️ 独立随机变量的一元函数仍独立（注意是一元函数）
- ⚠️ 柯西分布：由两个独立正态之比产生，期望不存在

## 课堂讨论/互动
- 三角形区域均匀分布独立性判断：学生计算边缘后验证 $f(x,y) \neq f_X(x)f_Y(y)$，确认不独立
- 二维正态 $\rho=0$ 推出独立：学生观察到当 $\rho=0$ 时密度可拆分为x部分和y部分的乘积
- 泊松可加性证明：关键步骤是配二项式定理 $(\lambda_1+\lambda_2)^k = \sum C_k^i \lambda_1^i\lambda_2^{k-i}$
- 两个U[0,1]和的密度：通过画被积函数非零区域（平行四边形），按z在0-1和1-2分段积分，得到三角分布
- 极大极小分布记忆技巧：不要死记公式，记住"最大的都小于等于x等价于每个都小于等于x"，"最小的大于x等价于每个都大于x"
