## 教学要点

### 一、σ域（事件域）
- **定义**：设$\Omega$是样本空间，$\mathcal{F}$（花写F）是$\Omega$的子集构成的集合族（集合的集合），满足：
  1. $\Omega\in\mathcal{F}$（必然事件在其中）
  2. 若$A\in\mathcal{F}$，则$\overline{A}\in\mathcal{F}$（对立事件封闭）
  3. 若$A_1,A_2,\cdots\in\mathcal{F}$（可列个），则$\bigcup_{n=1}^\infty A_n\in\mathcal{F}$（可列并封闭）
- 满足以上三点的集合族称为$\sigma$-域（$\sigma$-代数），也叫事件域，因为其中的元素就是我们关心的事件
- **σ域的作用**：对所有常见事件运算封闭——不可能事件$\emptyset=\overline{\Omega}\in\mathcal{F}$，交运算$A\cap B=\overline{\overline{A}\cup\overline{B}}\in\mathcal{F}$，差运算、对称差、事件序列极限都在其中
- **为什么不用所有子集作为σ域**：
  1. 理论上样本空间复杂时所有子集太大，无法在其上定义概率
  2. 实际上不需要关心所有事件，只要关心的事件在里面且运算封闭即可
- **例子**：$\Omega=\{1,2,3\}$时，最大σ域是所有子集族（共$2^3=8$个元素），最小σ域是$\{\emptyset,\Omega\}$（平凡σ域）

### 二、概率的公理化定义
- 概率是**集合函数**：对每个事件$A\in\mathcal{F}$，对应一个实数$P(A)$，满足三条公理：
  1. **非负性**：$P(A)\geq0$
  2. **规范性**：$P(\Omega)=1$
  3. **可列可加性**：若$A_1,A_2,\cdots$两两互不相容，则
     $$P\left(\bigcup_{n=1}^\infty A_n\right)=\sum_{n=1}^\infty P(A_n)$$
- **函数三要素**：
  - 定义域：事件域$\mathcal{F}$（自变量是事件$A\in\mathcal{F}$）
  - 值域：$[0,1]$（概率值在0到1之间，算出大于1肯定错了，通常是排列组合算重了）
  - 对应法则：满足三条公理
- 无论什么概率模型（古典、几何、统计），都必须与这三条公理兼容

### 三、概率的基本性质（从公理推导）
1. **不可能事件概率为0**：$P(\emptyset)=0$
   - ⚠️ 反过来不成立：概率为0的事件不一定是不可能事件。例如在$[0,1]$上任取一个实数，取到$\frac{1}{2}$的概率是0，但它可能发生；取到有理数的概率也是0（有理数可列但测度为0）
   - 同理：概率为1的事件也不一定是必然事件

2. **有限可加性**：若$A_1,\cdots,A_n$两两互不相容，则
   $$P\left(\bigcup_{i=1}^n A_i\right)=\sum_{i=1}^n P(A_i)$$

3. **对立事件公式**：$P(\overline{A})=1-P(A)$
   - 技巧：正面分析种类多时，从反面计算往往更简单

4. **减法公式**：
   - 若$B\subset A$（B是A的子集），则$P(A-B)=P(A)-P(B)$
   - 对任意$A,B$，常用形式：$P(A-B)=P(A)-P(AB)$（因为$AB\subset A$）

5. **单调性**：若$B\subset A$，则$P(B)\leq P(A)$
   - ⚠️ 反过来不成立：$P(B)\leq P(A)$不能推出$B\subset A$

6. **加法公式**：
   - 两个事件：$P(A\cup B)=P(A)+P(B)-P(AB)$（减去重复计算的交集）
     - 若$A,B$互斥，$P(AB)=0$，退化为有限可加性$P(A\cup B)=P(A)+P(B)$
   - n个事件的**容斥原理**：
     $$P\left(\bigcup_{i=1}^n A_i\right)=\sum P(A_i)-\sum_{i<j}P(A_iA_j)+\sum_{i<j<k}P(A_iA_jA_k)-\cdots+(-1)^{n+1}P(A_1\cdots A_n)$$
     规律：奇数个事件交集加，偶数个减，"加多了减，减多了加"

7. **概率的连续性**：若事件序列$\{A_n\}$单调递增或单调递减，则
   $$P\left(\lim_{n\to\infty}A_n\right)=\lim_{n\to\infty}P(A_n)$$
   即极限运算与概率运算可交换（类似级数中求和与求导在一定条件下可交换）

### 四、概率计算典型例题
- **例题1**：已知$P(A)=\frac{2}{3}$，$P(B)=\frac{1}{2}$，求$P(B\overline{A})$
  - 解：$B\overline{A}=B-A$，$P(B-A)=P(B)-P(AB)$
    - 若$A,B$互斥：结果为$\frac{1}{2}$
    - 若$B\subset A$：结果为0
    - 若$P(AB)=\frac{1}{4}$：结果为$\frac{1}{4}$

- **例题2**：已知$P(A)=P(B)=\frac{1}{2}$，证明$P(AB)=P(\overline{A}\overline{B})$
  - 证：用对偶律$\overline{A}\overline{B}=\overline{A\cup B}$，再用对立事件和加法公式：
    $$P(\overline{A}\overline{B})=1-P(A\cup B)=1-P(A)-P(B)+P(AB)=P(AB)$$

- **例题3**：已知$P(A)=0.7$，$P(A-B)=0.3$，求$P(\overline{AB})$
  - 解：$P(AB)=P(A)-P(A-B)=0.7-0.3=0.4$，故$P(\overline{AB})=1-0.4=0.6$

- **例题4**：若$P(A_i)=1\ (i=1,\cdots,n)$，证明$P(\bigcap_{i=1}^n A_i)=1$
  - 证：从对立面出发，$\overline{\bigcap A_i}=\bigcup\overline{A_i}$（德摩根律），$P(\overline{A_i})=0$。用容斥原理，所有项概率都为0（利用单调性：子集概率不超过全集），故$P(\bigcup\overline{A_i})=0$，得证。

### 五、经典概率模型
1. **统计定义（频率定义）**
   - $n$次重复试验中事件$A$发生$n_A$次，频率$f_n(A)=\frac{n_A}{n}$
   - n很大时频率稳定在常数附近，该常数即为概率$P(A)\approx f_n(A)$
   - 两个理论问题留到第五章大数定律解决：频率是否收敛？收敛到的是否为概率？

2. **古典概型（等可能概型）**
   - 两个条件：样本点有限、每个样本点等可能发生
   - 计算公式：$P(A)=\frac{m}{n}$，其中$n$是总样本点数，$m$是$A$包含的样本点数
   - 关键：数数（排列组合）
     - 排列：$A_n^r=P_n^r=\frac{n!}{(n-r)!}$（讲顺序，全排列$A_n^n=n!$）
     - 组合：$C_n^r=\binom{n}{r}=\frac{n!}{r!(n-r)!}=\frac{A_n^r}{r!}$（不讲顺序，性质$C_n^r=C_n^{n-r}$）
     - 不全相异元素全排列：n个元素分k类（第i类$n_i$个，$n_1+\cdots+n_k=n$），排列数为$\frac{n!}{n_1!n_2!\cdots n_k!}$
   - **生日问题**：n个人中至少两人生日相同的概率
     $$P(\text{至少两同天})=1-\frac{A_{365}^n}{365^n}$$
     结果：30人时概率约0.7，55人时约0.99，90人时几乎为1（但概率为1仍不是必然事件）

3. **几何概型**
   - 样本空间是可度量区域（一维长度、二维面积、三维体积），每个点等可能出现
   - 计算公式：$P(A)=\frac{\text{度量}(A)}{\text{度量}(\Omega)}$
   - 与古典概型区别：样本点无限，每个基本事件概率为0
   - **例题**：从$[0,1]$任取两数$x,y$，求$xy<\frac{1}{4}$的概率
     - 样本空间：单位正方形（面积1）
     - 事件区域面积：$\frac{1}{4}+\int_{1/4}^1\frac{1}{4x}dx=\frac{1}{4}+\frac{\ln2}{2}\approx0.5966$
     - 注：$xy<\frac{1}{4}$和$xy\leq\frac{1}{4}$概率相同，因为边界度量为0
   - **蒲丰投针问题**：等距离$2a$平行线，投长$2l(l<a)$的针，相交概率$P=\frac{2l}{\pi a}$，结果含$\pi$，可通过投针实验估计$\pi$值（蒙特卡洛方法思想）
   - **蒙特卡洛方法估计π**：单位正方形第一象限内投点，点落在1/4圆内概率为$\frac{\pi}{4}$，故$\pi\approx4\times\frac{\text{圆内点数}}{\text{总点数}}$

### 六、条件概率初步
- **问题背景**：研究某个事件发生后，对另一个事件发生概率的影响
- **定义**：设$P(B)>0$，则"在B发生条件下A发生"的条件概率为
  $$P(A|B)=\frac{P(AB)}{P(B)}$$
- 直观理解（几何概型）：B发生后样本空间从$\Omega$缩减到B，在B内A发生即$AB$发生，概率为$AB$度量比$B$度量，上下同除$\Omega$度量即得公式
- **两种计算方法**：
  1. 按定义在原样本空间计算$P(AB)$和$P(B)$
  2. 缩减样本空间：直接在B构成的新样本空间中计算A的概率（常更简便）
- **垃圾邮件过滤例子**：含"免费""折扣"等词汇概率0.35，既含这些词又是垃圾邮件概率0.3，则看到这些词时是垃圾邮件概率为$P(B|A)=\frac{0.3}{0.35}\approx0.86$，可用于垃圾邮件过滤

## 核心公式汇总
| 名称 | 公式 |
|------|------|
| σ域三条件 | $\Omega\in\mathcal{F}$；$A\in\mathcal{F}\Rightarrow\overline{A}\in\mathcal{F}$；$A_i\in\mathcal{F}\Rightarrow\bigcup_{i=1}^\infty A_i\in\mathcal{F}$ |
| 概率三公理 | $P(A)\geq0$；$P(\Omega)=1$；$P(\bigcup A_n)=\sum P(A_n)$（两两互斥） |
| 对立事件公式 | $P(\overline{A})=1-P(A)$ |
| 减法公式（一般） | $P(A-B)=P(A)-P(AB)$ |
| 减法公式（子集） | $B\subset A\Rightarrow P(A-B)=P(A)-P(B)$ |
| 加法公式（两个事件） | $P(A\cup B)=P(A)+P(B)-P(AB)$ |
| 容斥原理（n个事件） | $P(\bigcup_{i=1}^n A_i)=\sum P(A_i)-\sum_{i<j}P(A_iA_j)+\cdots+(-1)^{n+1}P(A_1\cdots A_n)$ |
| 排列数 | $A_n^r=\frac{n!}{(n-r)!}$ |
| 组合数 | $\binom{n}{r}=\frac{n!}{r!(n-r)!}$ |
| 古典概型公式 | $P(A)=\dfrac{m}{n}=\dfrac{A\text{包含样本点数}}{\text{总样本点数}}$ |
| 几何概型公式 | $P(A)=\dfrac{\text{度量}(A)}{\text{度量}(\Omega)}$ |
| 条件概率定义 | $P(A|B)=\dfrac{P(AB)}{P(B)}\quad(P(B)>0)$ |
| 蒲丰投针概率 | $P=\dfrac{2l}{\pi a}$（针长$2l$，线距$2a$） |

## 考试重点标注
- ⚠️ 概率的三条公理是整个概率论的理论基础，必须牢记并理解其含义
- ⚠️ 概率性质中"反过来不成立"的两个结论是概念题常考点：$P(\emptyset)=0$但概率为0不一定是不可能事件；$B\subset A\Rightarrow P(B)\leq P(A)$但概率小不一定是子集
- ⚠️ 减法公式$P(A-B)=P(A)-P(AB)$是计算常考公式，不要记错成$P(A)-P(B)$（后者仅当$B\subset A$成立）
- ⚠️ 加法公式（容斥原理）是计算必考内容，两个事件和三个事件的加法公式必须熟练
- ⚠️ 古典概型是计算题基础，排列组合（尤其是组合数$C_n^r=C_n^{n-r}$简化计算）必须掌握
- ⚠️ 条件概率定义$P(A|B)=\frac{P(AB)}{P(B)}$中注意竖线顺序：后面是条件，前面是目标事件
- ⚠️ 生日问题、蒲丰投针、蒙特卡洛方法是经典例题，理解其思想即可

## 课堂讨论/互动
- 学生问：为什么不直接把所有子集作为σ域？教师解答：理论上复杂样本空间所有子集太大无法定义概率，实际上也不需要关心所有事件，σ域只要包含我们关心的事件且运算封闭即可
- 教师强调：概率是一个函数（集合函数），理解函数三要素（定义域、值域、对应法则）对理解概率本质至关重要
- 教师通过$[0,1]$取点例子说明"概率为0的事件可能发生"：取到任何一个具体点概率都是0，但总有一个点会被取到，帮助学生建立对连续样本空间的直觉
- 教师提醒：排列组合计算概率时容易算重导致结果大于1，算出大于1肯定错了，要检查计数是否重复
