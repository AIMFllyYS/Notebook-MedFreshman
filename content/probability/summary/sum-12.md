## 教学要点

### 一、期望性质回顾与深化
- **线性性**：$E[aX+bY] = aE[X]+bE[Y]$，无条件成立，是简化计算的核心
  - 线性含义：仅涉及数乘与加法，随机变量之间相乘不属于线性运算
- **独立乘积性质**：$X,Y$ 独立 $\implies E[XY] = E[X]E[Y]$，可推广到n个独立变量
  - 条件可减弱（后续讲不相关即可成立）
- **单调性**：若 $X\geq Y$（以概率1），则 $E[X]\geq E[Y]$
- **绝对值不等式**：$|E[X]| \leq E[|X|]$
- **常数期望**：$E[c] = c$

### 二、期望性质的应用：分解思想
- **二项分布期望（分解法）**：
  - $X\sim B(n,p)$，引入指示变量 $X_i$：第i次试验成功记1，失败记0
  - 每个 $X_i\sim B(1,p)$（0-1分布），$E[X_i]=p$，且 $X = X_1+X_2+\dots+X_n$
  - 由线性性：$E[X] = \sum_{i=1}^n E[X_i] = np$，比直接用定义计算简单
- **巧合问题（匹配问题）**：
  - n个数字任意排列，$X$ 为巧合（数字k在第k位）个数
  - 引入 $X_k$：第k个数字巧合记1，否则0，$P(X_k=1)=\frac{1}{n}$，$E[X_k]=\frac{1}{n}$
  - $X = \sum_{k=1}^n X_k$，故 $E[X] = n\cdot\frac{1}{n}=1$，与n无关
- **负二项分布期望（分解法）**：
  - $X\sim NB(k,p)$（k次成功所需试验次数）
  - 分解为k个独立几何分布之和：$X = X_1+X_2+\dots+X_k$，$X_i\sim G(p)$
  - 几何分布期望 $E[X_i]=\frac{1}{p}$，故 $E[X]=\frac{k}{p}$
- **核心思想**：复杂随机变量可拆分为简单指示变量之和，利用线性性大幅简化计算

### 三、柯西-施瓦茨不等式
- **概率形式**：对任意随机变量 $X,Y$，有 $(E[XY])^2 \leq E[X^2]E[Y^2]$
- **高中形式**：$(\sum_{i=1}^n a_i b_i)^2 \leq (\sum_{i=1}^n a_i^2)(\sum_{i=1}^n b_i^2)$，是概率形式取离散均匀分布的特例
- **线性代数联系**：对应内积空间中 $|\langle u,v\rangle|^2 \leq \|u\|^2\|v\|^2$，是定义夹角余弦的基础
- **特例推论**：取 $Y=1$，得 $(E[X])^2 \leq E[X^2]$，即 $E[X^2] - (E[X])^2 \geq 0$，为方差非负性提供依据
- **证明思路**：构造二次函数 $E[(tX-Y)^2]\geq0$，利用判别式非正证明

### 四、离散-连续混合型随机变量的期望
- **方法一（分布函数拆分法）**：
  1. 先求混合型随机变量的分布函数 $F_X(x)$
  2. 将 $F_X$ 拆分为连续型分布 $F_1$ 和离散型分布 $F_2$ 的凸组合：$F_X = A F_1 + B F_2$，$A+B=1$
  3. 分别求连续部分期望 $E[X_1]$ 和离散部分期望 $E[X_2]$
  4. $E[X] = A E[X_1] + B E[X_2]$
- **方法二（函数期望定理法，推荐）**：
  1. 将混合型随机变量表示为已知分布随机变量的函数：$X=g(V)$，其中 $V$ 是连续型
  2. 直接用定理4.1：$E[X] = E[g(V)] = \int_{-\infty}^{+\infty} g(v) f_V(v) dv$
  3. 分段积分即可，无需拆分分布函数
- **典型例题**：电压表读数问题（指数分布电压超过量程 $V_0$ 时截断读数），结果为 $E[X]=\frac{1}{\lambda}(1-e^{-\lambda V_0})$

### 五、方差的定义与直观含义
- **实际背景**：刻画随机变量取值的分散/集中程度（如仪器测量精度、产品质量稳定性、运动员发挥稳定性）
- **定义**：若 $E[(X-E[X])^2]$ 存在，则称 $D(X) = E[(X-E[X])^2]$ 为 $X$ 的方差（也记为 $Var(X)$）
  - 本质：计算"随机变量与期望距离平方的平均"
  - 几何含义：离均差平方的期望，反映取值围绕均值的分散程度
- **推导公式（常用）**：$D(X) = E[X^2] - (E[X])^2$（平方期望减期望平方）
  - 推导：展开 $(X-E[X])^2 = X^2 - 2XE[X] + (E[X])^2$，逐项取期望化简
- **标准差**：$\sigma(X) = \sqrt{D(X)}$，量纲与 $X$ 一致（方差量纲是 $X$ 量纲的平方）
- **计算基础**：所有方差计算本质上都是用定理4.1/4.2计算函数期望

### 六、常见分布的方差计算
- **均匀分布 $U(a,b)$**：
  - $E[X]=\frac{a+b}{2}$，$E[X^2]=\frac{a^2+ab+b^2}{3}$
  - $D(X)=\frac{(b-a)^2}{12}$，仅与区间长度有关，与位置无关
- **泊松分布 $P(\lambda)$**：
  - 技巧：$k^2 = k(k-1)+k$，分别计算期望
  - $E[X^2]=\lambda^2+\lambda$，故 $D(X)=\lambda$，参数 $\lambda$ 既是期望也是方差
- **指数分布 $E(\lambda)$**：
  - $E[X]=\frac{1}{\lambda}$，$E[X^2]=\frac{2}{\lambda^2}$
  - $D(X)=\frac{1}{\lambda^2}$，方差是参数平方的倒数
- **正态分布 $N(\mu,\sigma^2)$**：第二个参数 $\sigma^2$ 就是方差

### 七、切比雪夫不等式（⚠️考试重点）
- **定理**：设随机变量 $X$ 的期望 $E[X]=\mu$ 和方差 $D(X)$ 存在，则对任意 $\varepsilon>0$，有
  $$P(|X-\mu|\geq\varepsilon) \leq \frac{D(X)}{\varepsilon^2}$$
  等价形式：$$P(|X-\mu|<\varepsilon) \geq 1-\frac{D(X)}{\varepsilon^2}$$
- **证明思路**（连续型为例）：
  1. $P(|X-\mu|\geq\varepsilon) = \int_{|x-\mu|\geq\varepsilon} f(x)dx$
  2. 在积分区域上 $(x-\mu)^2\geq\varepsilon^2$，故被积函数放大：$f(x) \leq \frac{(x-\mu)^2}{\varepsilon^2}f(x)$
  3. 积分区域扩大到全实轴：$\leq \frac{1}{\varepsilon^2}\int_{-\infty}^{+\infty}(x-\mu)^2f(x)dx = \frac{D(X)}{\varepsilon^2}$
- **意义与应用**：
  - 无需知道分布，仅用期望和方差即可估计概率的上界/下界
  - 理论价值：大数定律证明的核心工具
  - 实际应用：医学参考值范围估计（如白细胞计数在均值±3倍标准差内的概率至少8/9）

### 八、方差的基本性质
- **非负性**：$D(X)\geq0$，由柯西-施瓦茨不等式推论 $(E[X])^2\leq E[X^2]$ 保证
- **方差为0的充要条件**：$D(X)=0$ $\iff$ $P(X=c)=1$（单点分布，$c=E[X]$），即随机变量以概率1取常数值
- **常数倍性质**：$D(cX) = c^2 D(X)$（注意常数要平方）
  - 推论：$D(-X)=D(X)$（取负号不改变分散程度）
- **和的方差**：$D(X+Y) = D(X) + D(Y) + 2E[(X-E[X])(Y-E[Y])]$，多出的项为协方差（下节内容）
  - 若 $X,Y$ 独立，则 $D(X+Y)=D(X)+D(Y)$，可推广到n个独立变量
- **常数平移不变**：$D(X+c) = D(X)$（平移不改变分散程度）

## 核心公式汇总
| 名称 | 公式 |
|------|------|
| 柯西-施瓦茨不等式 | $(E[XY])^2 \leq E[X^2]E[Y^2]$ |
| 方差定义 | $D(X) = E[(X-E[X])^2]$ |
| 方差推导式 | $D(X) = E[X^2] - (E[X])^2$ |
| 切比雪夫不等式 | $P(\|X-\mu\|\geq\varepsilon) \leq \frac{D(X)}{\varepsilon^2}$ |
| 均匀分布方差 | $X\sim U(a,b) \implies D(X)=\frac{(b-a)^2}{12}$ |
| 泊松分布方差 | $X\sim P(\lambda) \implies D(X)=\lambda$ |
| 指数分布方差 | $X\sim E(\lambda) \implies D(X)=\frac{1}{\lambda^2}$ |
| 方差常数倍 | $D(cX)=c^2D(X)$ |
| 独立和的方差 | $X,Y$独立$\implies D(X+Y)=D(X)+D(Y)$ |

## 考试重点标注
- ⚠️ **切比雪夫不等式**：高频考点，需牢记公式形式，会用于概率估计，是第五章大数定律证明的基础
- ⚠️ **常见分布的期望与方差**：泊松（$\lambda$既是期望也是方差）、指数（期望$1/\lambda$，方差$1/\lambda^2$）、均匀（方差为区间长度平方/12）必须熟记
- ⚠️ **方差推导公式**：$D(X)=E[X^2]-(E[X])^2$，所有方差计算的核心公式
- ⚠️ **分解思想**：二项分布拆分为0-1分布之和、负二项分布拆分为几何分布之和，利用线性性简化计算
- ⚠️ **柯西-施瓦茨不等式**：需掌握概率形式，会用于证明相关结论
