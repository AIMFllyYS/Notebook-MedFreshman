## 教学要点

### 一、多维随机变量的概念
- 定义：设随机试验 $E$ 的样本空间为 $\Omega$，$X_1, X_2, \dots, X_n$ 是定义在 $\Omega$ 上的随机变量，则有序组 $(X_1, X_2, \dots, X_n)$ 称为 **n维随机变量** 或 **随机向量**
- 引入原因：单个随机变量不足以描述复杂现象（如平面上粒子位置需横、纵坐标）
- 几何意义：一维随机变量将样本空间映射到实轴；二维随机变量将样本空间映射到平面
- 注意：$(X,Y)$ 与 $(Y,X)$ 是不同的二维随机变量（有序性）

### 二、联合分布函数
- 定义：对二维随机变量 $(X,Y)$，二元函数
  $$F(x,y) = P\{X \leq x, Y \leq y\}$$
  称为 $(X,Y)$ 的 **联合分布函数**
- 定义域：$x, y \in \mathbb{R}$（整个平面），值域：$[0,1]$
- 几何含义：对平面上任意点 $(x,y)$，$F(x,y)$ 表示 $(X,Y)$ 落在以 $(x,y)$ 为顶点的左下方无穷矩形区域（含边界）内的概率
- 事件理解：逗号表示"且"，即 $\{X \leq x\}$ 与 $\{Y \leq y\}$ 同时发生的概率

### 三、联合分布函数的性质
1. **单调非降**：$F(x,y)$ 分别对 $x$ 和 $y$ 单调非降
2. **有界性**：
   - $F(-\infty, y) = 0$，$F(x, -\infty) = 0$，$F(-\infty, -\infty) = 0$
   - $F(+\infty, +\infty) = 1$
   - ⚠️ 注意：单个变量取 $+\infty$ 不等于1，而是边缘分布函数
3. **右连续性**：$F(x,y)$ 分别对 $x$ 和 $y$ 右连续
4. **矩形不等式**：对任意 $x_1 < x_2, y_1 < y_2$，有
   $$P\{x_1 < X \leq x_2, y_1 < Y \leq y_2\} = F(x_2,y_2) - F(x_2,y_1) - F(x_1,y_2) + F(x_1,y_1) \geq 0$$
   这是二维随机变量落在矩形区域（左开右闭、下开上闭）内的概率
- ⚠️ 判定二元函数能否作为联合分布函数，必须验证上述4条性质，特别是第4条（非负性）

### 四、边缘分布函数
- 定义：单个分量 $X$（或 $Y$）的分布函数称为 **边缘分布函数**，记为 $F_X(x)$（或 $F_Y(y)$）
- 本质：边缘分布函数就是普通的一维分布函数
- 由联合求边缘：
  $$F_X(x) = P\{X \leq x\} = P\{X \leq x, Y < +\infty\} = F(x, +\infty)$$
  $$F_Y(y) = P\{Y \leq y\} = P\{X < +\infty, Y \leq y\} = F(+\infty, y)$$
- 直观理解：求 $X$ 的边缘分布，即让 $Y$ 取遍所有可能值（$Y < +\infty$ 是必然事件，不改变概率）
- 推广到n维：保留自己的变量，将其余变量趋向 $+\infty$

### 五、二维离散型随机变量
- 定义：若 $(X,Y)$ 的所有可能取值为有限对或可列无限多对，则称其为 **二维离散型随机变量**
- 几何直观：平面上的散点分布
- **联合分布列**：
  $$P\{X = x_i, Y = y_j\} = p_{ij}, \quad i,j = 1,2,\dots$$
- 联合分布列的性质：
  - 非负性：$p_{ij} \geq 0$
  - 规范性：$\sum_{i=1}^\infty \sum_{j=1}^\infty p_{ij} = 1$
- 表格表示：行对应 $X$ 的取值 $x_i$，列对应 $Y$ 的取值 $y_j$，交叉点为 $p_{ij}$
- 概率计算：$(X,Y)$ 落在区域 $G$ 内的概率等于 $G$ 内所有点的 $p_{ij}$ 之和
- ⚠️ 注意：即使 $p_{ij}=0$ 也不能省略单元格，因为其他组合可能有概率

### 六、二维连续型随机变量
- 定义：若存在非负二元函数 $f(x,y)$，使得联合分布函数可表示为
  $$F(x,y) = \int_{-\infty}^x \int_{-\infty}^y f(u,v) \, du \, dv$$
  则称 $(X,Y)$ 为 **二维连续型随机变量**，$f(x,y)$ 称为 **联合概率密度函数**
- 直观理解：二维连续型随机变量在平面上**成片成区域**地取值（非单点、非单线聚集）
- 联合密度的性质：
  1. 非负性：$f(x,y) \geq 0$
  2. 规范性：$\int_{-\infty}^{+\infty} \int_{-\infty}^{+\infty} f(x,y) \, dx \, dy = 1$
- 密度与分布函数关系：在 $f(x,y)$ 的连续点处，$\frac{\partial^2 F}{\partial x \partial y} = f(x,y)$
- 概率计算：$(X,Y)$ 落在平面区域 $G$ 内的概率
  $$P\{(X,Y) \in G\} = \iint_G f(x,y) \, dx \, dy$$
- 几何意义：$z = f(x,y)$ 是空间曲面，概率等于曲顶柱体体积；整个空间上的体积为1
- ⚠️ 重要性质：二维连续型随机变量落在任意曲线上的概率为0（线面积为0），因此区域边界是否包含不影响概率

### 七、二维均匀分布
- 定义：设 $D$ 为平面上的有界区域，面积为 $S$，若 $(X,Y)$ 的联合密度为
  $$f(x,y) = \begin{cases} \frac{1}{S}, & (x,y) \in D \\ 0, & \text{其他} \end{cases}$$
  则称 $(X,Y)$ 在区域 $D$ 上服从 **二维均匀分布**
- 均匀性：落在 $D$ 内任一子区域 $D_1$ 的概率仅与 $D_1$ 的面积有关，与位置和形状无关
  $$P\{(X,Y) \in D_1\} = \frac{S_{D_1}}{S_D}$$
- 与几何概型的关系：二维均匀分布是二维几何概型的概率模型
- 几何直观：空间中高度为 $1/S$ 的平顶柱体

### 八、二维正态分布
- 定义：若 $(X,Y)$ 的联合密度为
  $$f(x,y) = \frac{1}{2\pi\sigma_1\sigma_2\sqrt{1-\rho^2}} \exp\left\{ -\frac{1}{2(1-\rho^2)} \left[ \left(\frac{x-\mu_1}{\sigma_1}\right)^2 - 2\rho\frac{(x-\mu_1)(y-\mu_2)}{\sigma_1\sigma_2} + \left(\frac{y-\mu_2}{\sigma_2}\right)^2 \right] \right\}$$
  则称 $(X,Y)$ 服从 **二维正态分布**，记为 $(X,Y) \sim N(\mu_1, \mu_2, \sigma_1^2, \sigma_2^2, \rho)$
- 参数：共5个参数
  - $\mu_1, \mu_2$：位置参数（分别为 $X, Y$ 的均值）
  - $\sigma_1^2 > 0, \sigma_2^2 > 0$：尺度参数（分别为 $X, Y$ 的方差）
  - $|\rho| < 1$：相关系数（刻画 $X$ 与 $Y$ 的线性相关程度）
- 几何形态：单峰钟形曲面（丘陵状），峰值在 $(\mu_1, \mu_2)$ 处

### 九、二维离散型的边缘分布列
- 已知联合分布列 $p_{ij} = P\{X=x_i, Y=y_j\}$，则：
  - $X$ 的边缘分布列：$P\{X = x_i\} = \sum_{j=1}^\infty p_{ij} \triangleq p_{i\cdot}$（**第 $i$ 行的行和**）
  - $Y$ 的边缘分布列：$P\{Y = y_j\} = \sum_{i=1}^\infty p_{ij} \triangleq p_{\cdot j}$（**第 $j$ 列的列和**）
- 记号：$p_{i\cdot}$ 的点表示对 $j$ 求和，$p_{\cdot j}$ 的点表示对 $i$ 求和

## 核心公式汇总

| 名称 | 公式 |
|------|------|
| 联合分布函数定义 | $F(x,y) = P\{X \leq x, Y \leq y\}$ |
| 边缘分布函数（X） | $F_X(x) = F(x, +\infty)$ |
| 边缘分布函数（Y） | $F_Y(y) = F(+\infty, y)$ |
| 矩形概率公式 | $P\{x_1 < X \leq x_2, y_1 < Y \leq y_2\} = F(x_2,y_2) - F(x_2,y_1) - F(x_1,y_2) + F(x_1,y_1)$ |
| 联合密度规范性 | $\int_{-\infty}^{+\infty}\int_{-\infty}^{+\infty}f(x,y)dxdy = 1$ |
| 区域概率公式 | $P\{(X,Y)\in G\} = \iint_G f(x,y)dxdy$ |
| 二维均匀密度 | $f(x,y) = \frac{1}{S_D}, (x,y)\in D$ |
| 离散边缘（X） | $p_{i\cdot} = \sum_{j=1}^\infty p_{ij}$ |
| 离散边缘（Y） | $p_{\cdot j} = \sum_{i=1}^\infty p_{ij}$ |

## 考试重点标注
- ⚠️ 联合分布函数的4条性质，特别是矩形不等式（第4条），用于判断函数是否为联合分布函数
- ⚠️ 由联合分布函数求边缘分布函数：$F_X(x) = F(x, +\infty)$，$F_Y(y) = F(+\infty, y)$，注意变量位置不能写反
- ⚠️ 二维均匀分布的密度是区域面积的**倒数**，不是面积本身（规范性要求）
- ⚠️ 二维正态分布有5个参数（不是4个），记住参数顺序 $\mu_1, \mu_2, \sigma_1^2, \sigma_2^2, \rho$，且 $|\rho| < 1$
- ⚠️ 二维连续型随机变量在任意曲线上的概率为0，边界不影响概率计算
- ⚠️ 二维离散型已知联合求边缘：行和得X的边缘，列和得Y的边缘
- ⚠️ 联合分布函数求待定常数（如$A,B,C$），利用$F(+\infty,+\infty)=1$和$F(-\infty,y)=F(x,-\infty)=0$

## 课堂讨论/互动
- 联合分布函数待定系数例题：$F(x,y) = A(B+\arctan x)(C+\arctan y)$，利用极限性质解得 $A = 1/\pi^2, B = C = \pi/2$
- 有放回取数例题：$X$ 在1,2,3中等可能取值，$Y$ 在1到$X$中等可能取值，用乘法公式 $P\{X=i,Y=j\} = P\{X=i\}P\{Y=j|X=i\}$ 计算联合分布列
- 指数型联合密度例题：$f(x,y) = e^{-(x+y)}, x>0,y>0$，分区域讨论求分布函数和概率 $P\{Y \leq X\}$
- 二维均匀分布与几何概型的联系：落在子区域的概率为面积比
