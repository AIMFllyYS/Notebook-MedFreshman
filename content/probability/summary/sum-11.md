## 教学要点

### 一、第三章内容回顾：随机变量函数的分布
- **串联系统寿命**：$N = \min(X_1,X_2,\dots,X_n)$，相互独立指数分布取较小仍为指数分布，参数为各子系统参数之和 $\lambda = \sum_{i=1}^n \lambda_i$
- **并联系统寿命**：$M = \max(X_1,X_2,\dots,X_n)$，需自行推导分布与期望
- **备用系统寿命**：$S = X_1+X_2$，为独立随机变量和的分布，使用卷积公式计算，需注意被积函数非零范围
- **离散+连续混合型**：一个离散、一个连续的二维随机变量，既非二维离散也非二维连续，使用**全概率公式**按离散变量取值分类讨论，万能方法是从分布函数定义出发 $P(Z\leq z)$

### 二、数字特征概念引入
- 数字特征是与随机变量分布有关的常数，刻画随机变量某一方面的突出特征
- 第四章主要数字特征：期望、方差、协方差、相关系数
- **核心观点**：期望是第四章最重要的概念，随机变量函数期望定理（定理4.1/4.2）是计算所有数字特征的基础

### 三、离散型随机变量的数学期望
- **定义**：设离散型随机变量 $X$ 的分布列为 $P(X=x_k)=p_k$，若级数 $\sum_{k=1}^\infty x_k p_k$ 绝对收敛，则称 $E[X] = \sum_{k=1}^\infty x_k p_k$ 为 $X$ 的数学期望
- **本质**：期望仅依赖于分布，分布相同则期望相同，与随机变量符号无关
- **计算结果**：期望是常数，带有量纲（与 $X$ 同单位）
- **直观含义**：
  - 加权平均：$p_k$ 为权重，$E[X]$ 是加权平均分
  - 质心位置：类比物理中质点系的质心，期望是概率分布的平衡点
- **注意事项**：
  - 期望值不一定是随机变量的可能取值（如平均成绩1.8分无人得此分）
  - 期望不一定存在，要求级数**绝对收敛**（保证结果与求和顺序无关）
- **常见离散分布期望**：
  - 泊松分布 $X\sim P(\lambda)$：$E[X]=\lambda$（参数即平均发生次数）
  - 0-1分布：$E[X]=p$

### 四、连续型随机变量的数学期望
- **定义推导（离散化思想）**：
  1. 将实轴分割为充分小的区间 $\Delta x_i$
  2. 用区间端点 $x_i$ 代表整个区间取值，概率近似为 $f(x_i)\Delta x_i$
  3. 离散化后期望为黎曼和 $\sum x_i f(x_i)\Delta x_i$
  4. 取极限得积分定义
- **定义**：设连续型随机变量 $X$ 的概率密度为 $f(x)$，若积分 $\int_{-\infty}^{+\infty} x f(x) dx$ 绝对收敛，则 $E[X] = \int_{-\infty}^{+\infty} x f(x) dx$
- **绝对收敛条件**：$\int_{-\infty}^{+\infty} |x| f(x) dx < +\infty$，否则期望不存在
- **反例：柯西分布**：$f(x)=\frac{1}{\pi(1+x^2)}$，期望不存在；数值实验显示其样本均值不稳定，不收敛到固定值；尾部较厚（厚尾分布）
- **常见连续分布期望**：
  - 指数分布 $X\sim E(\lambda)$：$E[X]=\frac{1}{\lambda}$（参数的倒数），串联越多平均寿命越短
  - 均匀分布 $U(a,b)$：$E[X]=\frac{a+b}{2}$（区间中点，即质心）

### 五、随机变量函数的期望定理（全章核心）
- **问题提出**：已知 $X$ 的分布，$Y=g(X)$，求 $E[Y]$，无需先求 $Y$ 的分布
- **定理4.1（一维）**：
  - 离散情形：$E[g(X)] = \sum_{k=1}^\infty g(x_k) p_k$
  - 连续情形：$E[g(X)] = \int_{-\infty}^{+\infty} g(x) f(x) dx$
- **定理4.2（二维）**：$Z=g(X,Y)$，已知联合分布
  - 离散情形：$E[g(X,Y)] = \sum_{i}\sum_{j} g(x_i,y_j) p_{ij}$
  - 连续情形：$E[g(X,Y)] = \int_{-\infty}^{+\infty}\int_{-\infty}^{+\infty} g(x,y) f(x,y) dxdy$
- **核心优势**：跳过求函数分布的繁琐步骤（2.4节/3.4节内容），直接用原始分布计算
- **计算技巧**：若二重积分不熟练，可先求边缘密度再用一维公式计算，但线性组合问题用定理4.2更直接

### 六、期望性质初探
- **常数的期望**：$E[c] = c$（常数的期望是自身）
- **线性性**：$E[aX+bY] = aE[X]+bE[Y]$，该性质**无条件成立**，无需独立假设
  - 推论：$E[X+Y]=E[X]+E[Y]$（和的期望=期望的和）
- **独立情形乘积期望**：若 $X,Y$ 独立，则 $E[XY] = E[X]E[Y]$，可推广到n个独立随机变量

## 核心公式汇总
| 名称 | 公式 |
|------|------|
| 离散型期望定义 | $E[X] = \sum_{k=1}^\infty x_k p_k$（绝对收敛） |
| 连续型期望定义 | $E[X] = \int_{-\infty}^{+\infty} x f(x) dx$（绝对收敛） |
| 一维函数期望定理 | $E[g(X)] = \int_{-\infty}^{+\infty} g(x) f(x) dx$ |
| 二维函数期望定理 | $E[g(X,Y)] = \iint_{\mathbb{R}^2} g(x,y) f(x,y) dxdy$ |
| 期望线性性 | $E[aX+bY] = aE[X]+bE[Y]$（无条件成立） |
| 独立乘积期望 | $X,Y$ 独立 $\implies E[XY] = E[X]E[Y]$ |
| 泊松分布期望 | $X\sim P(\lambda) \implies E[X]=\lambda$ |
| 指数分布期望 | $X\sim E(\lambda) \implies E[X]=\frac{1}{\lambda}$ |

## 考试重点标注
- ⚠️ **定理4.1/4.2（随机变量函数期望）**：全章核心，所有数字特征计算基础，必须熟练掌握
- ⚠️ **期望线性性**：无条件成立，是简化计算的关键，二项分布/负二项分布期望计算的核心技巧
- ⚠️ **常见分布的期望**：泊松、指数、均匀等分布的期望与参数关系需牢记
- ⚠️ **期望的本质**：仅依赖分布，分布相同则期望相同
