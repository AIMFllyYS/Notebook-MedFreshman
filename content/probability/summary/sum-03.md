# 第3节 · 条件概率·乘法公式·全概率·贝叶斯 · 课堂纪要

## 关键词
条件概率，乘法公式，划分（完备事件组），全概率公式，贝叶斯公式，先验概率，后验概率，事件独立性

## 核心议题
1. 条件概率的两种计算方法：定义公式法与缩减样本空间法
2. 乘法公式：由条件概率变形得到，计算积事件概率
3. 划分（完备事件组）概念与全概率公式：由因导果，计算结果总概率
4. 贝叶斯公式（逆概率公式）：由果溯因，是机器学习的核心
5. 贝叶斯思想：先验概率通过新数据迭代更新为后验概率
6. 事件独立性：独立与互斥的区别

## 教学要点

### 一、条件概率回顾
- **定义**：设$P(B)>0$，则$P(A|B)=\frac{P(AB)}{P(B)}$
- **两种计算方法**：
  1. 定义法：在原样本空间计算$P(AB)$和$P(B)$，按公式求比值
  2. 缩减样本空间法：B发生后样本空间缩减为B，直接在B上计算A发生的概率（更直观简便）
- 注意竖线顺序：竖线后是条件，前是目标事件

### 二、乘法公式
- **两个事件乘法公式**：由条件概率定义恒等变形得
  $$P(AB)=P(A)P(B|A)=P(B)P(A|B)$$
  - 与条件概率区别：条件概率是已知积事件和条件事件概率求条件概率；乘法公式是已知单个事件概率和条件概率求**积事件（同时发生）**的概率
- **n个事件乘法公式**：
  $$P(A_1A_2\cdots A_n)=P(A_1)P(A_2|A_1)P(A_3|A_1A_2)\cdots P(A_n|A_1A_2\cdots A_{n-1})$$
  - 共n个因子，每个条件概率的条件是前面所有事件都发生
  - 证明：将每个条件概率按定义展开，分子分母连锁相消即得
  - 虽然交运算有交换律，但实际问题按过程发生顺序写不容易漏
- **波利亚摸球模型例题**：袋中3红2白，每次取后放回并加2个同色球，$A_i$为第i次取红球
  - $P(A_1A_2)=P(A_1)P(A_2|A_1)=\frac{3}{5}\times\frac{5}{7}=\frac{3}{7}$（两次都红）
  - $P(A_1A_2A_3)=\frac{3}{5}\times\frac{5}{7}\times\frac{7}{9}=\frac{1}{3}$（三次都红）
  - $P(A_2)=P(A_1)P(A_2|A_1)+P(\overline{A_1})P(A_2|\overline{A_1})=\frac{3}{5}\times\frac{5}{7}+\frac{2}{5}\times\frac{3}{7}=\frac{3}{5}$（第二次取红球，分类思想即全概率公式雏形）

### 三、划分（完备事件组）
- **定义**：n个事件$A_1,A_2,\cdots,A_n$构成样本空间$\Omega$的一个划分，满足：
  1. **两两互斥**：$i\neq j$时$A_iA_j=\emptyset$（不重）
  2. **和为全集**：$\bigcup_{i=1}^n A_i=\Omega$（不漏）
- 直观理解：把样本空间"蛋糕"切成n块，块与块不重叠，拼起来是完整蛋糕
- 划分不唯一，切法可根据问题灵活选择

### 四、全概率公式
- **公式**：若$A_1,\cdots,A_n$是划分，且$P(A_i)>0$，则对任意事件B
  $$P(B)=\sum_{i=1}^n P(A_i)P(B|A_i)$$
  （可列无穷个划分也成立）
- **证明**：$B=B\cap\Omega=B\cap(\bigcup A_i)=\bigcup(BA_i)$，由$A_i$互斥知$BA_i$互斥，用有限可加性和乘法公式即得
- **直观理解**：$A_i$是导致结果B发生的n种"原因"或"途径"，全概率公式把每条途径导致B发生的概率加起来，得到B发生的总概率——**由因导果**
- **关键**：找对划分，即哪些原因会导致B发生
- **典型例题**：
  1. **迟到问题**：多种交通方式（校车、骑车、走路等）是原因，每种方式有不同迟到概率，全概率公式算总迟到概率
  2. **MAX单词脱落问题**：单词MAX（2M、2A、1X）随机脱落两个字母再放回，按"脱落两个字母是否相同"划分，得拼写正确概率$\frac{3}{5}$
  3. **敏感问题调查**：调查早恋比例等敏感问题，盒中放A类（敏感问题，概率0.4）和B类（无关问题"电话末尾是否奇数"，概率0.6，回答"是"概率0.5）问题，学生抽题只答是/否，老师不知道抽中哪类，用全概率反解出真实比例$p=\frac{P(\text{是})-0.3}{0.4}$
  4. **电子管次品问题**：三家工厂市场份额15%、80%、5%，次品率0.02、0.01、0.03，全概率算总次品率$P(B)=0.15\times0.02+0.8\times0.01+0.05\times0.03=0.0125$

### 五、贝叶斯公式（逆概率公式）
- **问题背景**：结果B已经发生了，反推是哪个原因$A_i$导致的——**由果溯因**（电子管次品问题第二问：已知是次品，问来自哪家厂可能性最大）
- **公式**：若$A_1,\cdots,A_n$是划分，$P(A_i)>0,P(B)>0$，则
  $$P(A_i|B)=\frac{P(A_i)P(B|A_i)}{\sum_{j=1}^n P(A_j)P(B|A_j)}=\frac{P(A_i)P(B|A_i)}{P(B)}$$
  - 分子：第i条途径的概率
  - 分母：全概率（所有途径的和，即结果B总概率）
- **证明**：条件概率定义 + 乘法公式 + 全概率公式
- **两类概率**：
  - $P(A_i)$：**先验概率**——观测到结果B之前，对原因$A_i$发生可能性的认知，来自历史数据、经验或主观判断（无信息时可假设等概率）
  - $P(A_i|B)$：**后验概率**——观测到结果B之后，对原因$A_i$发生可能性的新认知
- **贝叶斯思想核心——迭代更新**：
  新数据（结果B）到来时，用贝叶斯公式把先验概率更新为后验概率，这个后验又可作为下一次观测的先验，随数据不断流入迭代修正认知——这就是机器学习的基本逻辑
  - 例："狼来了"——第一次喊大家信（先验高），发现被骗后"可信"后验降低，多次后没人信
  - 例：信用评分——新用户无信息先假设好坏各半，按时还款提升好信用概率，逾期则降低，多次记录后评级越来越准确
- **电子管次品问题续**：算得$P(A_1|B)=0.24$，$P(A_2|B)=0.64$，$P(A_3|B)=0.12$，虽然三厂次品率最高(0.03)，但因市场份额太小(5%)，次品来自二厂可能性最大(64%)

### 六、医疗检测经典例题（贝叶斯反直觉案例）
- **设定**：某种疾病患病率0.1%（$P(C)=0.001$），检测真阳性率95%（$P(A|C)=0.95$），真阴性率95%（$P(\overline{A}|\overline{C})=0.95$，假阳性率5%）
- **问题**：检测呈阳性，真有病的概率是多少？
- **计算**：
  $$P(C|A)=\frac{P(C)P(A|C)}{P(C)P(A|C)+P(\overline{C})P(A|\overline{C})}=\frac{0.001\times0.95}{0.001\times0.95+0.999\times0.05}\approx0.0187$$
- **结果反直觉**：95%准确率的检测，阳性真患病概率仅约1.87%！原因是病太罕见（先验0.1%太低），大量假阳性淹没真阳性
- **检测的意义**：阳性群体患病率从0.1%提升到1.87%，提高了18倍。对阳性者做第二次检测，此时先验不再是0.001而是0.0187，再次阳性后$P(C|A_2)\approx0.27$，第三次阳性后约0.78，多次迭代后概率很高——这就是重大疾病要复查的原因

### 七、事件独立性
- **定义**：若$P(AB)=P(A)P(B)$，则称A、B相互独立。独立是地位对等的，没有顺序
- **特殊结论**：
  - 不可能事件（概率0）与任意事件独立
  - 必然事件（概率1）与任意事件独立
- **等价条件**（设$P(A)>0,P(B)>0$）：
  $$P(AB)=P(A)P(B)\iff P(B|A)=P(B)\iff P(A|B)=P(A)$$
  - 直观解释：$P(B|A)=P(B)$即"A发生"这个条件不影响B发生的概率，有没有A都一样
- **互斥 vs 独立**（必须区分！）：
  - 互斥：$AB=\emptyset$（不能同时发生）
  - 独立：$P(AB)=P(A)P(B)$（发生与否不影响概率）
  - ⚠️ 若$P(A)>0,P(B)>0$，则互斥一定不独立，独立一定不互斥！因为互斥时$P(AB)=0\neq P(A)P(B)>0$

## 核心公式汇总
| 名称 | 公式 |
|------|------|
| 条件概率定义 | $P(A|B)=\dfrac{P(AB)}{P(B)}\quad(P(B)>0)$ |
| 乘法公式（两个事件） | $P(AB)=P(A)P(B|A)=P(B)P(A|B)$ |
| 乘法公式（n个事件） | $P(A_1\cdots A_n)=P(A_1)P(A_2|A_1)\cdots P(A_n|A_1\cdots A_{n-1})$ |
| 划分两条件 | $A_iA_j=\emptyset\ (i\neq j)$；$\displaystyle\bigcup_{i=1}^n A_i=\Omega$ |
| 全概率公式 | $P(B)=\displaystyle\sum_{i=1}^n P(A_i)P(B|A_i)$ |
| 贝叶斯公式 | $P(A_i|B)=\dfrac{P(A_i)P(B|A_i)}{\displaystyle\sum_{j=1}^n P(A_j)P(B|A_j)}=\dfrac{P(A_i)P(B|A_i)}{P(B)}$ |
| 两事件独立定义 | $P(AB)=P(A)P(B)$ |
| 独立等价条件 | $P(B|A)=P(B)=P(B|\overline{A})$（$P(A)>0$时） |

## 考试重点标注
- ⚠️ 乘法公式、全概率公式、贝叶斯公式是第一章三大核心公式，**必考计算题**，必须熟练掌握
- ⚠️ 全概率公式是"由因导果"，贝叶斯公式是"由果溯因"，要能根据问题判断用哪个公式
- ⚠️ 用全概率和贝叶斯的关键是**正确划分样本空间**，找对"原因"
- ⚠️ 贝叶斯公式中先验概率与后验概率的概念，以及贝叶斯"迭代更新"的思想是机器学习基础，概念题常考
- ⚠️ 医疗检测阳性预测值问题是经典反直觉例题，掌握贝叶斯计算方法
- ⚠️ 独立与互斥的区别是概念题必考：$P(A)>0,P(B)>0$时互斥一定不独立，独立一定不互斥，不要混淆
- ⚠️ 敏感问题调查是全概率公式的巧妙应用，理解其设计原理

## 课堂讨论/互动
- 学生问：为什么全概率公式划分必须"不重不漏"？教师解答："不漏"保证所有原因都考虑到了，B不会被漏掉；"不重"保证用可加性时不会重复计算，二者缺一不可
- 教师通过"狼来了"和信用评分的例子解释贝叶斯迭代思想：不是一次更新就结束，而是随着新数据不断修正认知，这就是人脑学习和机器学习的共同逻辑
- 医疗检测例题引发讨论：直觉认为95%准确率很高，阳性应该大概率有病，但计算结果只有约1.87%，教师解释这是因为基础患病率（先验）太低，提醒学生不要靠直觉做概率题，一定要按公式计算
- 教师强调：全概率和贝叶斯看起来是两个公式，实际上是一个问题的两个方面——全概率算"结果有多大概率发生"，贝叶斯算"结果发生了是哪个原因导致的"，二者分母相同
