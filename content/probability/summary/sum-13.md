## 教学要点

### 一、期望与方差性质对比总结
- **期望线性性**：$E\left[\sum_{i=1}^n a_i X_i\right] = \sum_{i=1}^n a_i E[X_i]$，**无条件成立**，无需独立假设
- **方差性质**：
  - 常数提出需平方：$D(cX) = c^2 D(X)$，注意负号平方后为正
  - 常数平移不变：$D(X+c) = D(X)$
  - 和的方差：$D(X+Y) = D(X)+D(Y)+2Cov(X,Y)$，多交叉项（协方差），仅当独立/不相关时才有 $D(X+Y)=D(X)+D(Y)$
- **独立条件的作用**：
  - 独立 $\implies$ $E[XY] = E[X]E[Y]$（乘积期望等于期望乘积）
  - 独立 $\implies$ $Cov(X,Y)=0$ $\implies$ 和的方差等于方差和

### 二、随机变量的标准化变换
- **标准化公式**：对随机变量 $X$，若 $E[X]=\mu$，$D(X)=\sigma^2$，令 $Y = \frac{X-\mu}{\sigma}$，则 $E[Y]=0$，$D(Y)=1$
- **标准化的意义**：
  - 期望变为0，方差变为1
  - **消除量纲**：原始变量 $X$ 带有量纲（如米、千克），标准化后 $Y$ 为纯数，便于不同量纲变量间比较
- **应用场景**：不同单位/尺度数据的比较、正态分布概率计算（查表前必须标准化）

### 三、独立同分布样本均值的数字特征
- **背景**：$X_1,X_2,\dots,X_n$ 独立同分布（i.i.d.），$E[X_i]=\mu$，$D(X_i)=\sigma^2$
- **样本均值**：$\bar{X} = \frac{1}{n}\sum_{i=1}^n X_i$
- **期望不变性**：$E[\bar{X}] = \mu$（多次测量求平均不改变真实值）
- **方差缩小**：$D(\bar{X}) = \frac{\sigma^2}{n}$（误差缩小为原来的 $1/n$，精度提高）
- **原理解释**：物理/化学实验中"多次测量求平均"的统计依据——不改变真实值，但降低测量误差，提高稳定性
  - 注意：$\frac{1}{n}$ 提出来时要平方，故为 $\frac{1}{n^2}\cdot n\sigma^2 = \frac{\sigma^2}{n}$

### 四、二项分布方差（分解法）
- **分解**：$X\sim B(n,p)$，$X = X_1+X_2+\dots+X_n$，$X_i$ 独立同服从0-1分布 $B(1,p)$
- **0-1分布方差**：$D(X_i) = pq = p(1-p)$（由 $E[X_i]=p$，$E[X_i^2]=p$，$D(X_i)=p-p^2=p(1-p)$）
- **二项分布方差**：由独立性，$D(X) = \sum_{i=1}^n D(X_i) = np(1-p) = npq$
- **对比超几何分布**：无放回抽样破坏独立性，不能直接用方差和公式，需额外计算协方差

### 五、正态分布线性组合应用：活塞装入气缸问题
- **问题**：活塞直径 $X\sim N(\mu_1,\sigma_1^2)$，气缸直径 $Y\sim N(\mu_2,\sigma_2^2)$，$X,Y$ 独立，求活塞能装入气缸的概率 $P(X<Y)$
- **转化**：$P(X<Y) = P(X-Y<0)$，令 $Z=X-Y$
- **正态分布优良性质**：独立正态变量的线性组合仍为正态分布
- **计算Z的参数**：
  - 期望：$E[Z] = E[X]-E[Y] = \mu_1-\mu_2$
  - 方差：$D(Z) = D(X)+D(Y) = \sigma_1^2+\sigma_2^2$（注意：$-1$ 平方为+，是加不是减！）
- **标准化查表**：$Z\sim N(\mu_1-\mu_2, \sigma_1^2+\sigma_2^2)$，标准化后查标准正态分布表
- **注意事项**：二维正态联合密度形式复杂，但当 $\rho=0$（独立）时，联合密度为边缘密度乘积

### 六、随机变量的矩
- **定义（设各阶矩存在）**：
  - **k阶原点矩**：$E[X^k]$，记为 $\alpha_k$
  - **k阶中心矩**：$E[(X-E[X])^k]$，记为 $\mu_k$
  - k阶绝对原点矩：$E[|X|^k]$
  - k阶绝对中心矩：$E[|X-E[X]|^k]$
- **本质**：矩是期望和方差的推广，本质仍是函数期望（用定理4.1计算）
- **特殊矩的含义**：
  - 一阶原点矩 $\alpha_1 = E[X]$：期望（均值）
  - 二阶中心矩 $\mu_2 = D(X)$：方差
  - 一阶中心矩 $\mu_1 = E[X-E[X]] = 0$（恒为0，只要存在）
- **计算方法**：所有矩都是随机变量函数的期望，统一用定理4.1/4.2计算
- **对称分布性质**：若密度函数关于期望 $\mu$ 对称（$f(\mu-x)=f(\mu+x)$），则所有奇数阶中心矩为0（正负抵消，如正态分布）
- **正态分布的中心矩**：
  - k为奇数：$\mu_k = 0$
  - k为偶数：$\mu_k = \sigma^k \cdot (k-1)!!$，其中双阶乘 $(k-1)!! = 1\times3\times5\times\cdots\times(k-1)$
  - 特例：$k=2$ 时，$\mu_2=\sigma^2$（方差），验证成立

### 七、偏度（Skewness）
- **定义**：$S_X = \frac{E[(X-E[X])^3]}{(D(X))^{3/2}} = \frac{\mu_3}{\sigma^3}$
- **实质**：三阶中心矩除以标准差的三次方，为无量纲纯数
- **作用**：刻画分布的不对称（偏斜）程度
  - 对称分布（如正态）：$S_X=0$（奇数阶中心矩为0）
  - $S_X>0$（正偏/右偏）：分布右侧有长拖尾，峰偏左，如彩票收益（极小概率获大额收益）
  - $S_X<0$（负偏/左偏）：分布左侧有长拖尾，峰偏右，如保险损失（极小概率大额损失）
- **为什么用三阶**：一阶中心矩恒为0，三阶是最简单的能反映不对称性的奇数阶矩

### 八、峰度（Kurtosis）
- **定义（ excess kurtosis，超额峰度）**：$K_X = \frac{E[(X-E[X])^4]}{(D(X))^2} - 3 = \frac{\mu_4}{\sigma^4} - 3$
- **减3的原因**：正态分布的 $\frac{\mu_4}{\sigma^4}=3$，减3使正态分布峰度恰好为0，便于与正态比较
- **注意**：部分教材/软件不减3（与3比较），需注意区分
- **作用**：刻画分布的峰尖峭程度和尾部厚度
  - $K_X=0$：与正态分布峰度相同（正态分布）
  - $K_X>0$：尖峰厚尾（比正态峰更尖、尾部更厚），金融收益率分布常见
  - $K_X<0$：峰度不足（肩部较宽、尾部薄），如均匀分布
- **重要提示**：实际数据常呈尖峰厚尾特征，盲目假设正态会低估尾部风险

### 九、协方差定义（引入）
- **问题提出**：和的方差公式中多出的交叉项 $E[(X-E[X])(Y-E[Y])]$ 需要专门研究
- **定义**：$Cov(X,Y) = E[(X-E[X])(Y-E[Y])]$，刻画两个随机变量之间的协同变化关系
- **推导公式（常用）**：$Cov(X,Y) = E[XY] - E[X]E[Y]$（乘积期望减期望乘积）
  - 推导：展开乘积后用期望线性性化简
- **直观含义**：
  - $Cov(X,Y)>0$：$X$ 与 $Y$ 同向变化（X增大时Y有增大趋势）
  - $Cov(X,Y)<0$：$X$ 与 $Y$ 反向变化（X增大时Y有减小趋势）
  - $Cov(X,Y)=0$：无线性协同变化趋势
- **注意**：协方差的符号表示方向，但**大小受量纲影响**，不能直接用于比较关系紧密程度
- **特殊情形**：
  - $Cov(X,X) = D(X)$（自身协方差即方差）
  - $Cov(X,c) = 0$（与常数的协方差为0）
- **计算基础**：用定理4.2（二维函数期望）计算

## 核心公式汇总
| 名称 | 公式 |
|------|------|
| 标准化变换 | $Y = \frac{X-\mu}{\sigma}$，则 $E[Y]=0,D(Y)=1$ |
| 样本均值期望 | $E[\bar{X}]=\mu$（$\bar{X}=\frac{1}{n}\sum X_i$） |
| 样本均值方差 | $D(\bar{X})=\frac{\sigma^2}{n}$（i.i.d.条件下） |
| 二项分布方差 | $X\sim B(n,p) \implies D(X)=np(1-p)$ |
| k阶原点矩 | $\alpha_k = E[X^k]$ |
| k阶中心矩 | $\mu_k = E[(X-E[X])^k]$ |
| 正态分布偶数阶中心矩 | $\mu_k = \sigma^k (k-1)!!$（k偶） |
| 偏度 | $S_X = \frac{\mu_3}{\sigma^3}$ |
| 峰度 | $K_X = \frac{\mu_4}{\sigma^4} - 3$ |
| 协方差定义 | $Cov(X,Y) = E[(X-E[X])(Y-E[Y])]$ |
| 协方差推导式 | $Cov(X,Y) = E[XY] - E[X]E[Y]$ |

## 考试重点标注
- ⚠️ **标准化变换**：基本操作，正态分布计算查表必需
- ⚠️ **样本均值的方差**：$D(\bar{X})=\sigma^2/n$，注意1/n要平方，是统计推断的基础
- ⚠️ **二项分布方差**：$np(1-p)$，与期望 $np$ 一起记忆
- ⚠️ **常见分布的数字特征汇总**：泊松（期望=方差=λ）、指数（E=1/λ, D=1/λ²）、均匀（D=(b-a)²/12）、二项（E=np, D=npq）、正态（μ和σ²为参数）
- ⚠️ **独立正态线性组合**：仍为正态，方差是加不是减（系数平方）
- ⚠️ **偏度峰度含义**：正态偏度=0、峰度=0；右偏/左偏、尖峰厚尾的含义
