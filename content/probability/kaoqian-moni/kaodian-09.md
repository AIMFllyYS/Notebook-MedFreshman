# 考点九·随机变量的独立性

> 来源：华科《概率论与数理统计》习题集（第2版）·分类练习
> 对应章节：ch03 独立性

### 考点说明

:::callout{kind=note label="考点注记"}
本考点为高频考点之一，但通常不单独出题，相关题型可参考上面第13题等（考点八中的相关题目）。
:::

:::callout{kind=insight label="解析"}
独立性的核心概念：

**定义**：随机变量 $X$ 与 $Y$ 独立，当且仅当联合分布函数等于边缘分布函数的乘积：
$$F_{X,Y}(x, y) = F_X(x) F_Y(y)$$

对于离散型：$P(X = x, Y = y) = P(X = x) P(Y = y)$ 对所有 $x, y$ 成立

对于连续型：$f_{X,Y}(x, y) = f_X(x) f_Y(y)$ 几乎处处成立

**重要性质**：
1. 独立 ⟹ 不相关（协方差为0），但不相关 ⟹ 独立（仅对正态分布成立）
2. 若 $X, Y$ 独立，则 $g(X)$ 与 $h(Y)$ 也独立（$g, h$ 为Borel可测函数）
3. 独立变量的函数分布可通过卷积等方法计算

**判断方法**：
- 定义法：验证联合分布是否可分解
- 物理意义：一个变量的取值不影响另一个变量的分布
- 特殊分布：正态分布中不相关等价于独立

**相关题型参考**：
- 考点八第7题：判断 $X, Y$ 是否独立（通过联合密度是否可分解）
- 考点八第10题：边缘正态但联合不正态，不独立
:::

:::callout{kind=tip label="结论速记"}
独立性判断：验证 $f(x, y) = f_X(x) f_Y(y)$。不相关 $\nRightarrow$ 独立（正态除外）。
:::
