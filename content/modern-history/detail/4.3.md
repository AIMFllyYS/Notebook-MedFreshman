## 维新派与守旧派的论战

甲午战败之后，求新求变的呼声在知识界如山崩海啸般涌来。康有为、梁启超、谭嗣同、严复等人总结了王韬、郑观应等早期改良派的思想和实践，明确提出了向西方学习制度的主张，将变法思潮推向了新的阶段，并发展成为一场政治运动。然而，既然维新派首先要拿中国传统思想开刀、为政治改革做舆论准备，那么他们在思想上也必然遭到反对者和守旧派的激烈反扑。整个改革过程中，维新派和守旧派的论战从未停歇，其核心围绕三个问题展开：**要不要变法**、**要不要兴民权设议院**、**要不要废八股改科举**。

---

## 一、论战的思想基础：康有为的理论武器

在进入三大论战之前，必须首先理解维新派领袖康有为构建的一整套思想理论体系，因为这是维新派在论战中的核心武器。

### 1.1 康有为其人

:::definition{label=康有为}
康有为（1858—1927），广东南海人，维新派的精神领袖和思想导师。30多岁时已成为广东名儒，学问精深但极度骄傲自负，是有名的"狂生"。他是**中国历史上第一个给自己修年谱**的人——《康南海自编年谱》，且字号"南海圣人"，自己称自己为圣人。维新变法运动中他与光绪皇帝实际只见过一面，但他自称见过很多面，甚至伪造了与光绪的合影照片（由梁启超找高手在底片上用针尖绘制）。
:::

### 1.2 公车上书：政治资本的积累

:::definition{label=公车上书}
1895年，《马关条约》签订的消息传来，全国震动。38岁的康有为和学生梁启超在北京松筠庵起草万言书，抗议清帝在条约上加盖玉玺，提出拒和、抗战、迁都变法等主张，18省举人集会响应，上千人签名。这一事件被称为"公车上书"（"公车"是汉代以来对高级士子的尊称）。

然而，后来的学者研究普遍认为：这份万言书实际上**从未送到光绪皇帝手上**，光绪帝当时根本不知道有康有为这个人。但康有为在返回广东后大肆宣扬自己"领导了公车上书"，万言书得到了光绪赞赏，由此积累了巨大的政治资本。
:::

### 1.3 今文经学与进化论的结合

康有为思想的核心武器，是将中国传统的**今文经学**与严复引进的**西方进化论**相结合。

:::definition{label=今文经学}
今文经学是相对于"古文经学"而言的学术流派。秦始皇焚书之后，汉初由伏生等老儒生用当时的隶书默写先秦经典，形成"今文经"；后来从孔子旧宅墙壁中发现用先秦篆书书写的经典，形成"古文经"。两派之争的实质不仅是学术真伪问题，更是"学术如何为政治服务"的问题。

今文经学的核心特征是"**微言大义**"——认为孔子不是学者而是"素王"（没有称王的王），六经不是孔子编的而是孔子写的，描述的不是上古社会的真实状况而是孔子的理想。因此，今文经学强调**变革**，强调应顺应时代去重新阐述儒家经典，历来是变革者的思想武器。
:::

:::insight{label=康有为的理论创新}
康有为先后写出两部关键著作：

1. **《新学伪经考》**——"破"：断定全部古文经是汉代学者刘歆父子伪造的，目的是为王莽登基做理论准备。既然清政府信奉的儒家经典是假的，那么传统认识中那个要恢复周礼的保守孔子形象，自然也是假的。
2. **《孔子改制考》**——"立"：重新塑造孔子形象——孔子是**世界上最早的改革家**，核心主张是政治制度"**应因时而变，要不断进化**"。

结论呼之欲出：**不是我要改革，是孔老夫子要我改革**。要当正统的中国人、正统的儒学家，就应当学习孔子精神，大搞改革。
:::

### 1.4 "三世说"：历史进化论的中国版本

康有为进一步将今文经学的"三世说"与严复翻译的《天演论》中的进化论融合。

:::definition{label=三世说的重新诠释}
东汉今文经学提出"据乱世、升平世、太平世"的历史循环论。康有为将其与进化论结合，宣称在孔子那里，历史**不是循环的，而是上升的**：

- **据乱世** → 君主专制之治；
- **升平世** → 君主立宪之治；
- **太平世** → 民主共和之治。

康有为晚年在《大同书》中更进一步，认为人类未来的社会"一定是共产主义社会"——这说明他的思想在某些维度上甚至比后来的革命者更为激进。
:::

## 二、论战焦点一：要不要变法？

这是维新派与守旧派之间最根本的分歧。

### 2.1 维新派的论证

维新派利用孔子这个"神主牌"来论证变法的正当性。通过将孔子重塑为"改革家"，将传统儒家学说与西方政治社会理论附会起来，试图削弱反对者的反对力度。

### 2.2 论证的内在困境

:::insight{label=孔子神主牌的双刃剑效应}
维新派的论证策略存在一个根本性的困境：既然要把孔子拉出来当神主牌，就必须承认孔子的正统地位、承认儒学的正统地位。那么奉儒学为思想支柱的清政府，完全可以"不认你这个孔子"——你既然不能砸毁孔子的神圣性，问题就始终纠结在"你说的这个孔子到底对不对"的层面上。

更深层的问题在于：维新派本质上是要进行政治改革，而不是要进行纯粹的思想论战。从思想进入实践领域，最关键的不是"有没有道理"的问题，而是**"有没有利益"**的问题。正如毛泽东所说："政治就是把我们的敌人搞得少少的，把我们的朋友搞得多多的。"你拿孔子出来当神主牌得到的支持，远远不如你说一句"我要废除科举制度"得到的反对来得多。
:::

## 三、论战焦点二：要不要兴民权、设议院？

这是维新变法运动中最具政治敏感性的议题。

### 3.1 维新派的主张

从王韬开始，尽管对立宪政治进行了种种中国化、本土化的思想阐释——宣称是"君民共主""君民共治"——但无论如何，立宪政治的实质就是要**制约最高统治者的权力**。不管英国还是日本的君主立宪制，虽然君主仍有实权，但宪法的存在意味着君王不再是独一无二、不受约束的最高存在。

### 3.2 各方立场的复杂博弈

:::definition{label=三方立场}
围绕"兴民权、设议院"的论战，实际上形成了一个三方博弈的格局：

- **维新派**：高喊要兴民权、设议院、搞君主立宪，限制君权；
- **守旧派**：坚持"君本"，宣称"民权之说无一益而有百害"；
- **光绪皇帝**：**沉默不语**。
:::

### 3.3 光绪帝的微妙角色

:::insight{label=光绪帝的真实动机}
光绪帝参与维新变法的动机，并非真正希望限制自己的权力。他的目的是**借助维新派去挑战既得利益集团**——说白了，就是想通过维新派从慈禧太后手中夺回权力。

这就形成了一个悖论：维新派的理想是通过立宪来制约君权，而支持他们的唯一实权人物——光绪帝——恰恰是因为想扩大自己的权力才支持改革。双方"互相利用"：维新派利用皇帝来推行政治主张、改变中国命运；皇帝利用维新派来挑战既得利益、夺回权力。

这种微妙的合作关系，注定是脆弱的。
:::

## 四、论战焦点三：要不要废八股、改科举？

这是三大论战中**最要害、最核心**的问题，因为它直接触及了帝国数百万读书人的切身利益。

### 4.1 八股文的僵化

:::definition{label=八股文的弊病}
八股文经过明清近600年的发展，日益僵化。其核心弊病在于：

- 知识分子读书只读"时文汇编"——考上举人、进士的范文合集，即"全国百篇优秀作文合集"；
- 四书五经中只读四书和自己选定的一本经，其余四经都不读；
- 更不读《史记》等史书，很多人连基本的历史常识都没有。

这种制度已经无法产生真正的人才，其不好的一面已经完全压过了好的一面。
:::

### 4.2 废除科举的合理性与危险性

维新派认为，要让人们接受新思想、接受改革，废除八股、改科举、兴西学几乎是必然的。

:::insight{label=政策的"最优解"与"次优解"}
从政治实践的角度看，废除科举制度面临一个根本性的矛盾：

- **正当性**上，废除僵化的科举制度无疑是正确的；
- **可行性**上，它会把帝国**300万读书人**的前途一刀斩断，而维新派没有提供任何可以替代的方案。

政治实践中，能推行下去的往往不是最优解，而是**次优解**。最优解总是在理想状态下才会出现，而现实中必然面临种种阻碍和不利因素。维新派坚持要一步到位，结果是——光绪皇帝下了诏之后，**地方根本就不接诏**，废八股改科举的政策根本推行不下去。
:::

### 4.3 科举制度的本质辨析

:::definition{label=科举不是封建专制的核心}
一个值得深思的问题：科举制度只有约1500年的历史，而1500年前中国同样存在君主专制——这说明科举制度**并非封建专制的本质属性**，它本质上只是一种人才选拔的形式。

今天中国的公务员考试制度（而非高考）才是科举制的现代继承者。现代公务员考试制度源于18世纪英国建立的文官考试制度，而英国的文官制度正是从中国学的科举制——启蒙思想家伏尔泰曾狂热地称赞科举制为"世界上最完美的制度"。

因此，问题的核心**不在于废不废除科举制度**，而在于**谁控制着中央权力、谁通过科举考试来分配政治资源**。控制科举的人是传统派，用传统内容来分配权力，这才是西学传播的真正障碍。
:::

## 五、论战的实效与局限

### 5.1 舆论战的开创性意义

维新派开创了近代中国政治史上一个极为重要的先例——**舆论战**。

:::example{label=维新派的舆论攻势}
维新派通过以下方式展开舆论攻势：

- **创办学会**：强学会、保国会、南学会等；
- **开设学堂**：康有为的万木草堂、梁启超的时务学堂等；
- **创办报纸**：在长沙、北京、上海、广州等各地大规模创办报纸。

这使得国家政策讨论从上层大知识分子下降到了一般知识分子层面。过去只有高级士人才有资格讨论的"国家秘闻"，现在一个秀才也可能通过新式报纸和学堂参与到全国性的政治讨论中。
:::

### 5.2 影响力的双重局限

然而，维新派舆论战的影响范围存在明显的双重局限：

:::pitfall{label=上下两头都够不到}
- **对上层**：王公贵族、核心大臣们对"这些臭读书的"不屑一顾，维新派在政治上层缺乏真正的支持者；
- **对下层**：谁可以上学堂、看报纸？还是知识分子。最底层的老百姓没有参与到其中。

最终能支持维新派的，只有本身没有实际权力的光绪皇帝——"他已经没有什么好失去了"。"光脚的不怕穿鞋的"，无论兴民权、设议院还是废八股兴西学，都切实伤害到了一大批既得利益阶层，而中国传统统治的中坚力量——地主、士大夫和官僚这三个群体本质上是同一个群体。
:::

### 5.3 慈禧的真实态度

:::insight{label=慈禧并非铁板一块的顽固派}
人们通常认为慈禧是极端顽固保守、食古不化的人物。但历史细节显示：光绪颁布《明定国是诏》的当天傍晚就向慈禧请安并呈阅了诏书，慈禧只说"清皇帝做主"，**并未提出反对意见**。包括光绪的老师翁同龢——过去常被认为保守的人物——实际上就是《明定国是诏》的起草者。在变法之初，慈禧和翁同龢对变法虽然谈不上多赞同，但至少是**默认**的，因为"中国已经到了这一步了，大家都觉得不变法问题很大"。

真正使慈禧走向改革对立面的，是百日维新的急速和激烈——103天内下了100多份诏书，每一份都切切实实地伤害着既得利益集团。这不是变法的内容有问题，而是变法的**节奏和策略**出了问题。
:::

## 六、因果分析：论战何以未能转化为改革实效

:::definition{label=论战局限的三重原因}
维新派与守旧派的论战最终未能转化为实际的改革成效，其原因可从三个层面分析：

1. **思想层面**：维新派借用孔子神主牌的论证策略有其内在局限——不能彻底否定儒学正统，就无法从根本上动摇守旧派的思想根基；
2. **利益层面**：废八股改科举直接威胁300万读书人的前途，维新派未能提供替代方案，招致最广泛的反对；
3. **政治层面**：维新派是一群"布衣"，在上层缺乏实权支持者，在下层未能触及普通百姓。他们唯一的盟友光绪帝本身也是"光脚的"，改革的正当性被改革的危险性所压倒。

毛泽东所言"政治就是把敌人搞得少少的，把朋友搞得多多的"，维新派恰恰相反——在短短103天内，以激烈的速度把潜在的盟友推向了对立面。
:::

:::note{label=本节小结}
- 维新派与守旧派的论战围绕三大核心问题展开：**要不要变法**、**要不要兴民权设议院**、**要不要废八股改科举**。这三大问题层层递进，从思想正当性到政治体制再到具体利益分配。
- 康有为以今文经学与进化论相结合为理论武器，将孔子重塑为改革家，试图为变法提供正统性依据。但这一策略因必须维护孔子权威而存在内在局限。
- "兴民权设议院"的论战暴露了维新派与光绪帝之间微妙的利益交换关系——维新派要限制君权，光绪帝要借改革夺权，双方的合作注定脆弱。
- "废八股改科举"是最要害的议题。科举制度本身不是封建专制的核心属性，但废除它会直接切断300万读书人的前途，维新派未能提供替代方案，导致政策在地方层面根本推行不下去。
- 维新派开创了近代中国政治史上的舆论战先例，但其影响力仅局限于知识分子阶层，对上层权贵和下层百姓均触及有限。论战在思想层面开阔了视野、解放了思想，但在政治实践层面收效甚微。
:::
